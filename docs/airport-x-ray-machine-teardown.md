# 机场 x 光机拆卸

> 原文：<https://hackaday.com/2013/03/09/airport-x-ray-machine-teardown/>

谁的车库里有一台机场随身携带的 x 光机？显然[Mike]知道，他分享了这个乐趣，他发布了一个真正深入机器硬件和操作系统的视频拆卸系列。

在这一点上，该系列包括六个冗长的片段。第一集，你会发现在休息后嵌入，从硬件的外部概述开始。[Mike]提到它在这一点上不起作用。他猜测这要么与启用机器的安全设置有关(毕竟它确实能产生 x 射线)，要么与 EPROM 芯片中的损坏内存有关。后来，当他查看代码反汇编并找到请求用户名和密码以访问某些菜单的字符串时，密码锁定得到了确认。第二部分包括更多的反汇编，以找出密码并获得对机器的完全访问权。在第四段视频中，他用 x 光检查了商店里的一些随机物品。

有很多值得看的，但看到他用稀有的设备走多远是令人兴奋的。
