# 线性书籍扫描仪用 Arduino 来做

> 原文：<https://hackaday.com/2015/04/23/linear-book-scanner-does-it-with-arduino/>

大约两年半前，谷歌图书团队开源了他们的图书扫描设备 T1 的计划，人们欢欣鼓舞。正如[Dany Qumsiyeh]在[我们当时](http://hackaday.com/2012/11/16/google-books-team-open-sources-their-book-scanner/)链接到的 Google Tech talk 中解释的那样，扫描仪使用真空从堆叠中取出下一页并翻转它，节省了几个小时的人力，并且不可否认地，将书籍置于一点危险之中。

[Chris]向我们透露了由[Forssa1]开发的线性书籍扫描仪的不同之处，它使用服务器风扇翻页。[Forssa1]的设备由激光切割的丙烯酸树脂制成，并采用了两个由 Arduino Mega 驱动的手持扫描仪。我们没有太多关于这个版本的信息，但是你可以在休息后查看一下。
