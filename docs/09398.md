# Hackaday 奖参赛作品:用 Vinduino 节约用水

> 原文:[https://hack aday . com/2015/06/27/hack aday-prize-entry-saving-water with-the-vinduino/](https://hackaday.com/2015/06/27/hackaday-prize-entry-saving-water-with-the-vinduino/)

[Reinier van der Lee]在南加州拥有一个葡萄园——这个州有点水危机。[Reinier van der Lee]还拥有一台 arduino 和一个烙铁。他组织了一个项目，减少了 25%的用水量，并将其转移到开源之地。它叫做[葡萄酒。](https://hackaday.io/project/6444-vinduino-a-wine-growers-water-saving-project)

它的操作非常简单。你在泥土里放了一个水传感器。你打开水。当水碰到传感器时，你就把水关掉。然而，这不是最有效的方法。问题是，当传感器关闭时，土壤已经饱和，植物无法吸收，水被浪费了。

通过使用三个传感器解决了这个问题。最低的传感器放置在根部下方。所以它永远不会爆炸。如果是这样，植物没有吸收所有的水分，你可以减少产量。它上面的两个传感器监测水在土壤中的流动，因此它知道何时减少水量和浇水周期。

请务必查看项目详情。根据 GNU 通用公共许可证 3.0，所有代码和编译文件都可以在他的 github 上获得
