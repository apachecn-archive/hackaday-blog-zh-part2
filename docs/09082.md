# 如何判断你是否反向安装箔式电容器

> 原文:[https://hack aday . com/2015/05/20/how-to-tell-if-you-installing-foil-capacitors-backward/](https://hackaday.com/2015/05/20/how-to-tell-if-youre-installing-foil-capacitors-backwards/)

只需一个错误就能意识到电解电容有极性，但如果你正在使用旧的电子管装置、电子管放大器或任何带有旧的橙色 dip、棕色 dip 或绿色 dip 箔电容的旧设备[，你还必须注意你的极性](https://www.youtube.com/watch?v=BnR_DLd1PDI)。这些旧电容由箔屏蔽构成，总有一面应该连接到机箱接地。如果不这样做，就会受到干扰，这不是放大器电路所希望的。

早已被淘汰的旧帽子通常会有一条黑带，标明帽子的哪一面是“箔底”。这是应该接地的一侧。如果你看看现代的锡箔帽，你可能还会在帽的一侧看到一条黑带，如果我们生活在一个公正的世界，这也应该标明锡箔地。[情况并非总是如此](https://www.youtube.com/watch?v=BnR_DLd1PDI&t=480)。

为了正确测试箔片电容并确定哪一侧应该更靠近地，您可以构建一个小型测试盒，它差不多是一个 h 桥，中间有一个开关和一对鳄鱼夹。将盖子连接到夹子上，将电路的输出放入示波器中，轻弹开关:干扰最小的方向表示盖子的箔接地。用一个新的、方向正确的帽子替换你老式设备中的旧帽子，你就拥有了一个伟大的音响放大器。
