# 切纸刀变成了激光雕刻机

> 原文:[https://hack aday . com/2015/01/31/切纸刀成为激光雕刻师/](https://hackaday.com/2015/01/31/paper-cutter-becomes-a-laser-engraver/)

小而强大的激光二极管越来越便宜，有一些商业产品可以让任何人用电脑控制的切割机切割纸张和乙烯。两者结合会怎么样？[一个拼凑起来的激光雕刻机的开端](https://petervojtek.github.io/diy/2015/01/30/silhouette-laser-engraver.html)。

在这次制作中，[Peter]使用了一个[剪影肖像](http://www.silhouetteamerica.com/shop/machines/portrait)，这是一个桌面数控切割机，通常用于乙烯贴花和复杂的剪纸工艺品。这台机器不仅仅局限于装饰工艺品——它已经被用于[切割 PCB 模板](http://hackaday.com/2013/03/11/pcb-stencils-for-200/)和其他伪工业任务。

因为剪影肖像有一个界面，允许几乎任何 CAM 软件控制它，所以[Peter]在激光雕刻实验中唯一需要做的就是安装激光二极管。幸运的是，激光与机器的切割刀片有着相似的外形，一点胶带将所有东西粘在一起。

通过拧开透镜来聚焦激光，通过一点点的尝试和错误，[彼得]能够在他选择的材料上做一些标记。这不是一台激光切割机，,但只要再多做一点工作，它就能成为一台神奇的激光雕刻机。
