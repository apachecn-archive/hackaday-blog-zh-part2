# 手工丙烯酸骨架时钟是一个令人印象深刻的卷轴锯壮举

> 原文:[https://hackaday . com/2014/04/29/manufactured-acrylic-skeleton-clock-is-a-impressive-feat-of-scroll-saw/](https://hackaday.com/2014/04/29/handmade-acrylic-skeleton-clock-is-an-impressive-feat-of-scroll-sawing/)

![Handmade Acrylic Skeleton Clock](../Images/d55495d40934646f0f174a36b05820ee.png)

为了他的一个机械工程学校项目，[Ben Murton]决定从头开始设计和建造一个时钟——虽然它看起来像是激光切割的… [但他完全是手工切割的。](http://www.instructables.com/id/Acrylic-Skeleton-Clock)

这是一个古老的落地钟和一个骨架钟的机械运作的结合——目标是让所有的运动都可见，以观察时钟是如何运作的。他使用 Autodesk Inventor 设计了它，并在网上提供了[文件供任何人使用](https://drive.google.com/folderview?id=0B4V1Ma-qByctOGprQ0NWV3Nac00&usp=drive_web)——他指出，如果你有激光切割机或 CNC 路由器，制作起来会特别容易！

无论如何，这个钟是由 3 毫米厚的丙烯酸树脂，5 毫米的黄铜轴，尼龙线，一些重物(铅)，一些螺母和螺钉制成的。在打印出他的 CAD 模板后，[Ben]小心翼翼地用线锯切割出每一个**齿轮和联动装置——我们印象深刻。**
