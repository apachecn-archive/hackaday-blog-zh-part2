# 跟我来:让伺服跟踪手的运动与飞跃

> 原文:[https://hack aday . com/2015/08/22/follow-me-making-servos-track-hand-motion-with-leap/](https://hackaday.com/2015/08/22/follow-me-making-servos-track-hand-motion-with-leap/)

Leap 控制器是那些可能更好的小工具之一，因为它的酷因素而不是它的实用性。~~飞行时间~~光学传感器读取手势，但在许多情况下它很难替代鼠标。我们看到的它的最佳用途似乎是需要知道你的手在哪里的专用系统。例如，[贾斯汀·普拉茨]和[库尔特·科洛希尔]有一个[有趣的演示](http://www.pubnub.com/blog/motion-controlled-servos-with-leap-motion-raspberry-pi/)，它使用一个跳跃来控制树莓派。Pi 命令伺服电机移动 LED 块来跟踪你的手的动作。他们的代码是 GitHub 上的[。](https://github.com/pubnub/LeapMotionServoBots)

该项目正在寻求一个视频，但他们确实提供了一个动画图像:

[![pubnub_leapmotion_RaspberryPi_servo_iot_DIY](../Images/beca094c200a9cf2f83878ae6f12e173.png)T2】](https://hackaday.com/wp-content/uploads/2015/08/pubnub_leapmotion_raspberrypi_servo_iot_diy.gif)

该演示展示了 PubNub 可能实现的低延迟，pub nub 是另一个物联网代理(显然，每个人都想成为统治他们的人)。

真正吸引眼球的一点是使用这些 8×8 RGB 模块作为机器人的“眼睛”。你不得不承认这刷新了使用相机作为机器人眼睛的范例！

这并不是我们已经介绍过的的第一次[实际应用。更不用说，如果你有 3D 打印机，你应该看看下面的视频。](http://hackaday.com/2013/09/03/finally-a-practical-use-for-the-leap/)
