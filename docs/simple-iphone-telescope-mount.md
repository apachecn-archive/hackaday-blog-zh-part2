# 简单的 IPhone 望远镜支架

> 原文：<https://hackaday.com/2013/01/23/simple-iphone-telescope-mount/>

这个便宜又简单的软件可以让你用你的旧智能手机通过望远镜拍摄照片和视频。[Xobmo]只花了 55 美分就造出了这个连接器。除了我们担心将手机插入支架时会刮伤镜头之外，我们喜欢这个想法。

去年圣诞节，妻子送给他一台 Celestron Powerseeker 70AZ 作为礼物。他在网上搜索了一下，发现已经有一些使用 iPhone 3GS 录制视频的解决方案。Thingiverse 上的这个设计将是完美的，但他没有 3D 打印机，从服务中订购它将花费近 50 美元。但当他开始思考时，他需要的只是一个适合望远镜的戒指和一个连接 iPhone 的方法。他去五金店买了一个 PVC 耦合器。在用锯子和钻孔机工作后，他完成了一个有两个翅膀的槽。把手机放进去，然后把环套在目镜上。在休息后的剪辑中，你可以看到一些动作镜头，并看到坐骑本身。
