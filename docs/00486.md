# 电吉他听起来很美妙；因为复杂而让我们流口水

> 原文:[https://hackaday . com/2012/05/21/player-guitar-sounds-wonderful-makes-us-droop-cause-as-the-complexity/](https://hackaday.com/2012/05/21/player-guitar-sounds-wonderful-makes-us-drool-because-of-the-complexity/)

![](../Images/2189e6d2c3ce82e8be204c916262f50a.png "solenoid-driven-guitar")

当你听到吉他弹奏者的声音时，很明显这不是一个人在弹奏。但唯一的原因是歌曲播放的精准性。除了准确性之外，有趣的是，这首曲子通常是由一群吉他手演奏的，但在这里，机器设法在一种乐器上演奏。我们认为这听起来很棒！

这来自于【Vladimir Demin】，一个之前[制造过一个自动贝扬](http://hackaday.com/2010/09/05/self-playing-bayan-built-nearly-22-years-ago/)的制造者(类似于手风琴，但是在键盘通常被找到的地方有按钮)。这一次，吉他的六根弦和许多品都配备了一个螺线管。在上图中，你可以看到安装在音孔附近的弹奏装置。六个拨片被固定在适当的位置，看起来每个拨片都有两个螺线管。从视频中我们可以观察到，其中一个螺线管用于拨弦，另一个倾斜拨片支架，这样当拨片回到起始位置时就不会有第二次拨弦。
