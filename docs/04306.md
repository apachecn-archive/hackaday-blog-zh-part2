# 更多驱动器托架、冷却和电源，适合 DIY Raid 机箱

> 原文:[https://hackaday . com/2013/10/14/more-drive-bays-cooling-and-power-for-a-DIY-raid-box/](https://hackaday.com/2013/10/14/more-drive-bays-cooling-and-power-for-a-diy-raid-box/)

事实上，我们一直在寻找适合家庭使用的网络附加存储解决方案。我们想要一个嵌入式选项，仅仅是为了省电，但是您见过商用嵌入式 RAID 系统的价格吗？更好的办法是找到一个节能的 PSU，并把它用在电脑机箱 RAID 转换上，就像[萨米]做的这个。他从一个旧的电脑机箱开始，改装成可以容纳更多的硬盘。

上面的图像显示了他的安装方案。我们大多数人都把废弃的光驱扔进了垃圾箱。很多时候，它们最终成为了与 CNC 一起玩的一种方式，但在这种情况下，[Samimy]去掉了内脏，使用了几个尖括号来在机箱内安装硬盘。由于 PSU 没有足够的 SATA 电源连接器，现在他可以将更多的驱动器安装到他需要供电的机箱中。他从一个坏掉的电源上剪下一串连接器，把它连接到这个电源上。最后，他在外壳顶部切了一个洞，为系统增加一点冷却。

他使用 Windows 7 为使用四个驱动器的 RAID0 和 RAID1 阵列提供支持。为了帮助提高系统的性能，他还使用 USB 拇指驱动器作为缓存。这是我们不熟悉的东西，我们很高兴他给[提供了 ReadyBoost](http://en.wikipedia.org/wiki/ReadyBoost) 的链接，该软件使这成为可能。
