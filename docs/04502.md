# 遥控 FPV 驾驶舱

> 原文:[https://hackaday.com/2013/11/07/remote-control-fpv-cockpit/](https://hackaday.com/2013/11/07/remote-control-fpv-cockpit/)

![cockpit](../Images/2186fffd4852d3e0820e5a6500202a97.png)

FPV 飞行，不管它实际上有多棒，仍然包括摆弄遥控发射器，或者戴上视频护目镜，或者眯着眼看屏幕。不错，但没有[【布雷特·海斯】的封闭式驾驶舱地面站](http://flitetest.com/articles/fpv-full-scale-cockpit-ground-station-w-42-tv)酷。这是一个可牵引的飞行模拟系统，让你在不离开地面的情况下，在当地的地形上驾驶飞机。

这个建筑的中心是一台 42 英寸的平板电视，售价 160 美元。它被放置在一个大胶合板和 2×2 盒子的前面，还有一个计算机操纵杆、油门和方向舵控制。

控制器内的罐子需要被换出来，以匹配旧双叶发射器内的罐子的电阻。从那里开始，完成驾驶舱只是为视频切换器，起落架收回杆，襟翼制造一些面板的问题。和遥控收音机设置。

这是一个真正令人惊叹的建筑，当放在由[布雷特]的吉普车牵引的拖车上时，它有可能成为最接近驾驶载人飞机的东西，而无需飞行员执照。
