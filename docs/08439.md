# 最聪明的巫师不会喝罐头

> 原文：<https://hackaday.com/2015/03/02/the-wisest-wizard-doesnt-drink-from-cans/>

“巫师杖”或“最聪明的巫师”是一种在聚会上玩的喝酒游戏，参加者把他们喝过的空饮料罐一个接一个地放在一起，组成一个喝醉了的力量杖。一个拥有较长法杖的人被认为是处于较高的水平，因此可以命令较低的巫师将他们当前的饮料捣到他们认为合适的程度。然而，并不是每个参加聚会的人都一定要喝罐装的美味饮料。因此，[Ahmed]和他的团队为我们这些可能会选择挥舞一杯*品脱玻璃杯代替*的人想出了一个解决方案。

在他们 2015 年伊利诺斯州黑客大会的硬件项目中，[布雷迪·萨尔兹]、[艾哈迈德·苏希尔]、[达里奥·阿兰吉斯]和[卡舍夫·达尔米亚]决定给游戏添加一种科技的热情。对于他们的更新版本，眼镜配备了移动电池组，火花微控制器和不同颜色的发光二极管作为指示器。几根电线伸入每个玻璃杯的底部，测量电导率，并记录装满和倒空的次数。在由铝制外壳和管道胶带组成的 leu 塔中，该小组开发了一个简单的 Android 应用程序，供参与者登录，该应用程序将跟踪并可视化注册到其中一个眼镜的每个球员的排名。他们甚至创建了一个 pebble 版本的应用程序，它将显示所有相同的信息，以防你不想冒险在喝酒时处理手机…嘿。

为了增加乐趣，一旦玩家达到比别人高一个等级，他们就可以选择“挑战”弱小的对手。通过在应用程序中选择这个人的用户名，他们杯子上的 LED 和蜂鸣器就会激活，让他们知道他们已经被选中喝完剩下的饮料。如果你想知道他们是如何做到的，你可以在 Github 上查看这个团队的代码，也许可以尝试给这个游戏一个你自己的改造。
