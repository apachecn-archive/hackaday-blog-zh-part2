# 建造一座 Strandbeest

> 原文:[https://hackaday.com/2013/06/13/building-a-strandbeest/](https://hackaday.com/2013/06/13/building-a-strandbeest/)

![flexing-jansen-mechanism](../Images/e2d39578c08b4cd2ce2d05fb2a47c942.png)

[杰里米]可能已经放弃了他的大型六足机器人项目，但这并不意味着他已经脱离了腿式机器人的世界。他开始了另一个项目，比简单的六足机器人更加优雅和美丽。这一次，[他正在建造一台 Strandbeest](http://www.jcopro.net/2013/06/12/making-a-beest-leg-linkage-system/) ，这台机器是由非凡的步行机器【西奥·詹森】设计的。

想出 Strandbeest 腿连杆的正确长度和关节不是你可以凭空想象出来的，所以在[Jeremy] [找到他新项目的灵感](http://www.jcopro.net/2013/05/24/making-inspiration-the-strandbeest/)后，他钻研了 Strandbeest 腿的相关文献。他在[找到了【多米尼克·斯图德】](http://www.armure.ch/WALKING.htm)的工作，并着手制作自己的机械腿。

现在，[杰里米]有一个木制的 Strandbeest 腿连杆的原型。它仍然需要一点点的工作，但很快就会有一只 PVC 管登山蜂在[杰里米]家附近的丛林中游荡。
