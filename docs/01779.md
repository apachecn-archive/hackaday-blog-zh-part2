# 智能自动驾驶汽车参加制造商博览会

> 原文:[https://hack aday . com/2012/10/03/intelligent-autonomous-vehicle-make-it-to-maker-faire/](https://hackaday.com/2012/10/03/intelligent-autonomous-vehicle-makes-it-to-maker-faire/)

![](../Images/ab2cba9d8c53ef081953316bcc812f27.png "NAVI")

几个来自罗格斯大学的家伙带着 Navi 出现在 Maker Faire，他们的车参加了 [2012 年智能地面车辆竞赛。](http://ieee.rutgers.edu/)Navi 由两个巨大的铅酸电池供电，配备了足够的高端硬件，有望通过或绕过任何地形。

装载了激光测距仪，立体摄像机设置，指南针，GPS 接收器和一对能够拉动 40A 的电机，Navi 拥有所有需要的硬件传感器，可以在没有人工干预的情况下绕轨道行驶。一切都由控制面板下的一个小上网本控制，它本身装载了足够多的开关和一个 8×32 LED 矩阵，令人完全无法理解。

在休息后的视频中，来自罗格斯大学的家伙们展示了进入 Navi 的系统。还有一个展示 Navi 悬挂的视频，这是一个令人印象深刻的定制叉骨设置，有望在整个比赛中保持 Navi 平稳。

同样值得注意的是:一份为纳维准备的 [PDF 设计报告](http://www.igvc.org/design/2012/Rutgers%20University.pdf)和纳维自己的博客。
