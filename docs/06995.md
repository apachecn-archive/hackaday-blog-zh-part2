# DIY 紫外线灯是治疗指甲和更多

> 原文:[https://hacka day . com/2014/09/05/DIY-uv 灯-is-the-cure-for-nail-and-more/](https://hackaday.com/2014/09/05/diy-uv-lamp-is-the-cure-for-nails-and-more/)

我们必须承认，我们想知道阿达果的(贝基·斯特恩)是如何用她的指甲做任何事情的。它们总是又长又漂亮，没有任何碎片、凹痕或凹痕。原来，她用的是 UV 凝胶指甲油。它比标准的风干抛光剂耐用得多，但是它需要紫外光来固化。[Becky]买了一盏灯在家里使用，但它非常笨重，必须插在墙上。她知道有更好的方法，于是设计了她的 DIY 紫外线迷你美甲灯。

她真的什么都想到了。开源 3D 打印外壳包括顶部的一个小隔间，用于存放角质层棒、金刚砂板和镊子。Li-poly 电池可通过 USB 与 Adafruit 的 [PowerBoost 500c](http://www.adafruit.com/product/1944) 一起充电。灯本身由 30 个紫外发光二极管和 100ω电阻组成。[Becky]在她的内衣里衬上银色的粘纸，帮助紫外线均匀分布。
