# 一种快速简单的多色印花接丝机

> 原文:[https://hacka day . com/2014/01/20/a-快速简单的多色印刷用接丝机/](https://hackaday.com/2014/01/20/a-quick-and-simple-filament-joiner-for-multi-color-prints/)

![Filament splicer](../Images/17ac94787cf3286d3f5cad7ce9539999.png)

[Malcolm]和他的新 3D 打印机玩得很开心。不过，他已经厌倦了单色印花。由于没有多台挤出机，他开始寻找一种连接细丝的方法。在 Thingiverse 上有一些设计，但是它们需要磨制的零件，而他没有工具来重新制作。与其投资一家工厂，[马尔科姆]决定[建造自己的细丝接合机](http://www.3dhacker.com/category/551/Other/listings/564/Filament-joiner-for-multicoloured-prints.html)。他从抢夺妻子的护发工具开始。他的第一个测试是卷发棒。它有热量，但缺乏一个良好的表面来连接灯丝。[马尔科姆的]下一个测试是陶瓷直发器，他发现这是一个完美的工具。

拼接过程很简单。从一个热熨斗开始，然后在熨斗的短端放两根细丝。它们很快变软并融化在一起。[马尔科姆]真正的诀窍是，一旦两部分连接起来，就轻轻拉动接头。牵拉导致细丝拉伸，稍微减小接头的直径。较薄的接头有助于防止挤出机在接头通过时堵塞。这个方法对解放军很管用。我们很想看看它是否也适用于 ABS。
