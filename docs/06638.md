# 红牛创造奖得主:创客双胞胎

> 原文:[https://hack aday . com/2014/07/23/红牛-创造-赢家-制造者-双胞胎/](https://hackaday.com/2014/07/23/red-bull-creation-winners-maker-twins/)

今年红牛的创作主题“重新发明轮子”相当广泛，但制造商 Twins 设法将其与他们的获奖项目紧密结合，该项目在不到 72 小时内完成。他们采纳了城市农业的想法，并想出了一种方法，通过帮助消除浪费和美化产品展示，使农贸市场更加可行。

这个项目相当于一个摩天轮。代替乘客车厢的是模块化板条箱，每个板条箱都有一个木制托盘。轮子本身是链条驱动的，允许系统跟踪每个板条箱在旋转中的位置。这些数据有几种不同的用途。其中一个让顾客在平板电脑应用程序上选择他们的产品，板条箱会旋转到位，这样他们就可以挑选他们想要的单个商品。该机器还将负责自动浇水，以确保展示的产品不会变干。蛋糕上的糖衣是一个单独的站，用于清洗和切割购买的蔬菜。
