# 柔软的硅树脂机器人

> 原文：<https://hackaday.com/2015/07/08/soft-and-squishy-silicone-robotics/>

这个机器人手臂和抓手几乎完全由硅树脂制成。手工铸造零件，【Mike】组装这个工作的，[遥控机械臂抓手。](http://www.instructables.com/id/Soft-Robots-Make-An-Artificial-Muscle-Arm-And-Gri/)

我们会让它在一分钟内消失。他把一个超大的硅胶牙膏管变成了一个气动机械臂。天啊。我们之前已经看过[软机器人](http://hackaday.com/2012/12/04/simplifying-fabrication-of-soft-robots/)的 **[很多](http://hackaday.com/2014/03/24/soft-robotics-silicone-rubber-and-amazing-castings/)** ，但这是一些真正酷的东西！

你看，[迈克]实际上正计划利用这项技术制造一种廉价的机器人假手。这仅仅是一个测试，看看他能做多好的硅树脂空气肌肉——我们可以说这是相当成功的！这个机器人附件中的每个硅树脂圆盘内部都有四个密封的口袋。当空气通过它们流入时，它们膨胀，导致整个附肢向一侧伸展。有了四个这样的按钮和不同的压力，就可以向任何方向移动附肢了！
