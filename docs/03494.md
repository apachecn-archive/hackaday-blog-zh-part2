# 为残障人士构建红外鼠标界面

[克里斯·扬]有身体残疾，这意味着他不能很好地使用鼠标。他通常使用 Dragon Naturally Speaking 通过语音命令来移动鼠标，但他发现它缺少一些他需要的功能，有时会崩溃。

他对这个问题的解决方案是创造一种设备，它将把来自简单遥控器的红外信号转化为鼠标的动作和移动。他使用 Arduino micro 来完成这项任务，正如你在视频中看到的那样，这似乎对他很有帮助。他的网站上有代码和原理图，如果你想自己重现的话。

[Chris]实际上已经为自己和他人建造了几个无障碍设备。你应该[看看他的博客](http://tech.cyborg5.com/)了解更多，包括他对商业无障碍设备与 DIY 的[成本的看法。如果你想尝试制作一种设备来帮助身体残疾的人使用电脑，那就去 thecontrollerproject.com](http://tech.cyborg5.com/2013/01/06/assistive-technology-at-both-ends-of-the-price-spectrum/)[加入论坛吧。](http://thecontrollerproject.com)
