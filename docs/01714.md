# Wii 双截棍控制机器人展示坚如磐石的平衡

> 原文:[https://hack aday . com/2012/09/25/Wii-nuncchuck-controlled-robot-exhibits-rock-solid-balancing/](https://hackaday.com/2012/09/25/wii-nunchuck-controlled-robot-exhibits-rock-solid-balancing/)

![](../Images/21cbb1d549f39421f3b26eb222d697be.png "rock-solid-self-balancing-robot")

威利·万帕正在展示他的自动平衡机器人。这个版本给我们的印象是他的反馈回路看起来是如此的和谐。在休息后的视频中，你会看到绝对没有用于保持平衡的可见振荡。

所用的零件很容易获得。丙烯酸安装板是他妻子的设计，并通过 Pololu 服务定制切割。他们也是齿轮马达的来源。他用的是带 Arduino 和电机护罩的 SparkFun IMU。大约一个月前，他第一次发布了这个版本，但新版本切换到 Pololu 电机驱动盾，他说这种盾工作得更好，并通过无线 Wii 双节棍增加了控制。【T2

PID 循环给了它非常坚实的直立姿态，它来自[Brett Beauregard]编写的库。开源的概念再次让我们站在别人的肩膀上创造伟大的东西。【T2
