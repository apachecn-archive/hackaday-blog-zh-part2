# 本周失败:这根无生命的钛棒

> 原文：<https://hackaday.com/2014/09/11/fail-of-the-week-this-inanimate-titanium-rod/>

几周前，你看到了【克里斯】用动能砂[廉价铸铝，是吗？他最近通过现代交易方法的魔力得到了一些钛，并被其强度、硬度和差的热传递所折服。](http://hackaday.com/2014/08/12/and-so-castings-made-of-kinetic-sand-turn-out-pretty-well-actually/)

他想他会把它铸造成一个漂亮、结实的开瓶器。正如你可能猜到的，那并不顺利。首先，要锯断这根细杆并不容易。一旦他真的把它一分为二，除了顶端，摸起来出奇的凉。这是令人讨厌的伏笔，不是吗？

[Chris]花一点时间来帮助我们吸收他将要做的事情的重力，当然是使用 DC 弧焊机通过那个可怜的杆发送几百安培。由于氧气和加热的钛之间的反应，需要特别的预防措施。他可靠的石墨坩埚被固定在一个大铝盆的底部，一层来自 TIG 焊机的舒适的氩气会保护钛不被烧伤。

好吧。。。钛没有熔化。此外，坩埚是烤面包。从好的方面来看，用老虎钳对坩埚进行的横截面检查证明，坩埚壁上仍然有黄金。
