# 用丙酮蒸汽平滑 3D 打印

> 原文：<https://hackaday.com/2013/03/23/smoothing-3d-prints-with-acetone-vapor/>

如果你曾经使用过挤压式 3D 打印机，你就会知道最终的打印效果并不完全平滑。在 Southackton hackerspace [James]和[Bracken]研究出一种使用蒸汽使零件光滑的方法。该方法包括加热丙酮直到它形成蒸汽，然后将 ABS 部件暴露在蒸汽中。该方法仅适用于 ABS，但会产生一些好看的结果。

丙酮非常易燃，所以他们开始做一些安全测试。这包括获得一个良好的丙酮混合燃料空气，并测试什么是最坏的情况下，如果它被点燃。测试表明，他们使用的丙酮量是相当安全的，即使它着火了，这是上次我们看到方法时几个人提到的一个问题[。](http://hackaday.com/2013/02/26/giving-3d-printed-parts-a-shiny-smooth-finish/)

休息之后，[詹姆斯]和[布雷肯]对过程进行了详细的解释。
