# 双通道可变电压试验箱是忙碌的控制台修改者的梦想

> 原文:[https://hack aday . com/2012/08/07/双通道可变电压测试盒繁忙控制台修改者梦想/](https://hackaday.com/2012/08/07/dual-channel-variable-voltage-test-box-is-a-busy-console-modders-dream/)

![multi-voltage-test-box](../Images/26e040aa4b9d41dc0279fbd4ce71d534.png "multi-voltage-test-box")

似乎[克里斯·唐宁]总是在尝试新的东西。如果他不忙于[开发前代游戏机的便携版](http://hackaday.com/2012/03/11/geneboy-is-the-portable-sega-genesis-youve-always-wanted/)，他就在想办法让他的改装生活变得更简单。

最近，在开发一个用于控制三种不同游戏机的任天堂控制器时，[唐宁]发现他的桌子被埋在一堆电源和 A/V 电缆中。对自己越来越大的老鼠窝感到恼火，他决定[建造一个通用电源](http://www.made-by-bacteria.com/viewtopic.php?f=20&t=1925)，这样他就可以毫不费力地在游戏机之间快速切换。

他挖出一个旧的电脑电源，并把它输入一个为汽车制造的 LED 控制箱。[唐宁]然后在盒子上安装了九个摇臂开关，沿途增加了 A/V 输入和输出。隐藏在内部的一组电压调节器允许[唐宁]拨入他此刻可能需要的任何定制电压。

随着[唐宁]在接下来的几个月里追求更多的改装项目，这个测试箱应该会很方便。与此同时，一定要看看下面的视频，他在那里涵盖了设备设计的细节。
