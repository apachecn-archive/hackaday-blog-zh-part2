# 绕线机

> 原文:[https://hackaday.com/2014/01/30/the-filawinder/](https://hackaday.com/2014/01/30/the-filawinder/)

![filawinder](../Images/2b79aa19c8b9d75f52974df78207d4c1.png)

3D 打印机配件系列的最新成员是[绕线机](http://www.filastruder.com/collections/spare-parts/products/filawinder)，这是一种将细丝整齐缠绕在线轴上的工具。如果你已经放弃了一卷一卷地购买细丝，而是喜欢用更便宜的颗粒来制作自己的细丝——例如[的莱曼挤压机](http://hackaday.com/2013/03/05/finally-a-machine-that-makes-cheap-3d-printer-filament/)、[的 Filabot Wee](http://hackaday.com/2013/11/22/diy-filament-the-filabot-wee/) 或[的其他替代品](http://hackaday.com/2013/11/01/low-cost-filament-extruder/)，包括绕线机的配套产品[FilaStruder](http://www.filastruder.com/)——那么你很可能不得不用手来卷起所有东西，也许是在它们先在地板上扑腾之后。

细丝卷绕器在细丝挤出时为您卷绕，使用传感器调整卷绕速度以匹配挤出速度，并使其通过一些 PTFE 管，在移动时轻轻地卷绕。也许最重要的是，绕线机提供了一个导向臂，在绕线轮上来回引导细丝，以保持细丝均匀分布。点击[他们的页面](http://www.thingiverse.com/thing:174383)获取可打印件列表，他们的[组装指南可以在这里](http://solidoodletips.wordpress.com/2013/10/15/winder-2/)找到，也可以在 YouTube 上[找到。休息后，您可以看到 FilaWinder 卷绕的概述视频。](http://www.youtube.com/channel/UCAH5DBuFoaHegDD11qYkSWQ)
