# 3D 打印门户炮塔搜索和摧毁

> 原文：<https://hackaday.com/2014/06/12/3d-printed-portal-turret-searches-and-destroys/>

啊，对了，传送门——这是 Valve 过去十年中最具代表性的益智游戏之一。[Yvo]只是对他的全 3D 打印，工作，[传送门炮塔进行最后的润色。嗯——它没有枪——但它有几乎所有的功能！](http://ytec3d.com/turret)

这个复杂道具复制品对所有细节都很关注。[Yvo 的]精心设计，使其尽可能接近视频游戏模型。它的眼睛和手臂可以移动，有一个瞄准激光，甚至有一个基于颜色的物体跟踪摄像头。你也可以使用操纵杆来玩它——但让它处于自主模式会有趣得多。

因为整个东西都是 3D 打印的，[Yvo]还[制作了一个可指导你自己制作的](http://www.instructables.com/id/Building-a-moving-and-tracking-Portal-Turret/)，据他说，制作起来并不难！这也不是他的第一次牛仔竞技表演，如果你记得我们去年分享的令人敬畏的格拉多斯——那也是他的。
