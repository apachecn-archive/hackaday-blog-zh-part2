# Hackaday 奖参赛作品:挑战传统的管弦乐发明

> 原文:[https://hack aday . com/2015/07/17/hack aday-prize-entry-orchestra-invention-defies-conventi on/](https://hackaday.com/2015/07/17/hackaday-prize-entry-orchestral-invention-defies-convention/)

像我们许多人一样，[劳伦斯]喜欢电子游戏音乐和随心所欲地使用硬件。用一台打印机、几个软驱和一些旧硬盘，他组建了一个[非常规乐器乐队](https://hackaday.io/project/6485-unconventional-instrument-ochestra)。这位 2015 年 Hackaday 奖的竞争者通过一个 Java 程序在步进器和基于螺线管的硬件上播放任何 MIDI 文件。

不久前，[Laurens] [在我们的比赛中赢得了 Fubarino](http://hackaday.com/2013/12/22/fubarino-contest-minecraft-zelda-arduinos-and-hackaday/) 奖，他用 MIDI 键盘和 Arduino 控制了《我的世界》的环境，演奏了《塞尔达传说:时光之笛》。非传统乐器管弦乐队使用胜利的 Fubarino 来控制两个软驱的步进器。他只需要三个引脚来控制驱动器——一个启用，一个设置磁头的方向，一个让它每个脉冲走一步。

如果你曾经接触过 3D 打印机，你就会知道它们会把音乐作为一种自然的副作用。问题是让印刷者遵守音乐中的休止符。为了做到这一点，[Laurens]使用他的软件来控制打印机，本质上是保留下一个命令，直到歌曲中的适当时间。

这个管弦乐队的打击元素是由一个硬盘驱动器以头撞墙提供的。由于基本上不可能让硬盘做到这一点(谢天谢地)，[Laurens]用单个晶体管取代了控制板，以驱动移动磁头的线圈。

[Laurens]已经为音乐会上的管弦乐队制作了几个视频，这本身就是一种乐趣。每个视频的大部分可视空间都被软件产生的音乐的实时可视化所占据。仍然有足够的空间来展示管弦乐队本身，歌曲特定的游戏，以及 16 段显示中的文本评论。休息之后，请查看我们嵌入的播放列表。
