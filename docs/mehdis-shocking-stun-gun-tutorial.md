# [迈赫迪的]震撼电击枪教程

> 原文：<https://hackaday.com/2014/10/25/mehdis-shocking-stun-gun-tutorial/>

[Mehdi Sadaghdar]从不让火、电击或烧伤手指之类的小事妨碍他的项目。他的最新作品是关于制作简单电击装置的教程。眩晕武器产生非常高的电压，在执法中用于暂时使人丧失能力。[Mehdi]反复强调不要对任何人使用这个。如果你这么做了，他就不会再喜欢你了。当然，如果你看过[以前的任何视频](http://hackaday.com/2013/03/16/ac-vs-dc-human-pain-test/)，你就会知道他会[在项目完成前电击自己](http://hackaday.com/2013/12/17/fixing-christmas-lights-and-shocking-yourself-silly/)并点燃什么东西。

为了创造他的尤物，[迈赫迪]用汽车点火产生高压。点火线圈是一种特殊的变压器，可以产生击晕器所需的 10000 伏以上的电压。线圈的比例为 60:1，由 12V DC 电源供电。由于线圈在 DC 短路，系统只有在断电时才会产生高压脉冲。然而，脉冲太短，无法产生令人满意的电弧。[Mehdi]添加了一个电容器，产生了一个随着电荷衰减而振荡的 LC 电路，产生了一个更好的火花。然后他用一个 RC 电路和一个继电器制造了一个简单的振荡开关。为了画龙点睛，他用两个钉子在变压器的次级上制造了一个火花隙。以典型的[迈赫迪]风格，他在这个过程中差点烧掉了他的数字卡尺。

最终结果是一个美好的火花，温暖了[迈赫迪]颤动的心。我们赞扬他以科学的名义成为如此勇敢的受虐狂。休息之后看看他的教程吧！
