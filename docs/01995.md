# 南瓜俄罗斯方块的灵感来自我们自己的 LED 南瓜灯

> 原文:[https://hack aday . com/2012/10/29/pumpkin-Tetris-灵感来自我们自己的 led 杰克灯/](https://hackaday.com/2012/10/29/pumpkin-tetris-inspired-by-our-own-led-jack-o-lantern/)

![](../Images/fa5cb78c7efd3e50113d691d85c27f01.png "pumpktris_finished")

孩子们(或者他们的父母)会在[内森的]前廊排队，轮到他们玩南瓜俄罗斯方块。没错，[他把一款俄罗斯方块游戏打造成了一个真正的南瓜](http://www.hahabird.com/2012/10/pumpktris/)。当我们第一次看到它时，我们觉得这看起来很熟悉，事实上他的灵感来自于两年前万圣节[我们自己的 LED 矩阵南瓜](http://hackaday.com/2010/10/26/70-led-matrix-in-a-jack-o-lantern/)。我们喜欢看到衍生作品，而[内森]肯定对流程做了一些很大的改进。

矩阵本身的连线方式与我们使用的非常相似，但他增加了额外的 58 个 led，使显示器的大小几乎增加了一倍。他用纸网格和电钻为这些洞腾出空间，但通过在水果皮上雕刻方形像素来提高灯光的可见度。但是一个人如何控制游戏呢？南瓜的茎实际上是一个操纵杆。物理构建中最具创新性的部分之一是在内部使用干墙锚来安装操纵杆硬件。
