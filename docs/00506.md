# 工程师解释了 MEMS 加速度计芯片的工作原理

> 原文:[https://hack aday . com/2012/05/22/the-engineer-guy-explains-how-MEMS-accelerometer-chips-work/](https://hackaday.com/2012/05/22/the-engineer-guy-explains-how-mems-accelerometer-chips-work/)

你很有可能每天都在使用 MEMS 加速度计。就是这个小芯片让你的智能手机自动调整屏幕方向。它们是很棒的芯片，因为它们是大规模生产的，所以你可以把它们添加到你的项目中(如果你能忍受小包装的话)。但是我们不知道它们是如何制造的，只是对它们的工作原理略有所知。[Bill Hammack]解释了【MEMS 加速度计是如何制造的，以及它们是如何工作的，填补了这一知识空白。

我们的基础知识来自首字母缩略词:微机电系统。芯片里有东西在动(对固态电子设备来说就这么多了；这让我们想知道这些是否会磨损)。[Bill]在休息后在他的视频中加入了一个图表，显示了硅基系统在重力的影响下移动。这改变了结构的电容特性，可以测量并报告给微控制器以供进一步使用。这种结构是用复杂的蚀刻工艺建造的，我们不想在家里尝试。
