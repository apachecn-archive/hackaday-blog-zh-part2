# 逆转术:哈夫拉·汉默福治痊愈

> 原文：<https://hackaday.com/2015/09/02/retrotechtacular-hafla-hammerforge-healed/>

今年秋天去美国任何一个文艺复兴博览会，你都会发现一个铁匠。他会在帐篷里挥汗如雨，敲打放在铁砧上的一块发光的钢。虽然单个铁匠的做法延续至今，但过去的高产工作需要越来越多的肌肉。肌肉越不知疲倦越好。铁匠的手工努力被巨大的锤子取代，铁匠只需要在压痕之间转动工件，并保持对机器令人敬畏的压碎力的健康尊重。

上周，铁匠爱好者在瑞典芬斯潘完成了哈夫拉锤子的修复工作。这台有着 333 年历史的液压锤自 1924 年以来就再也没有使用过，当时哈夫拉哈默福吉工厂停止了运营。这家工厂建于 1682 年，使用德国的锻造方法，这种方法在 16 世纪被引入瑞典。钢铁生产在 19 世纪通过[贝塞麦炼钢法](http://hackaday.com/2015/04/07/retrotechtacular-the-bessemer-converter/)发生了革命性的变化，生产出更坚固的产品。
