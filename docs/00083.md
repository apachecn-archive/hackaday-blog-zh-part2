# 为围棋或国际象棋制作游戏时钟

> 原文：<https://hackaday.com/2012/04/11/building-a-game-clock-for-go-or-chess/>

[马蒂亚斯]刚刚进入电子爱好领域，并决定通过[建造这个游戏时钟](http://matias.blog.br/2012/04/07/gameclock/)(现场试玩[互联网档案馆](https://web.archive.org/web/20120507003148/http://matias.blog.br/2012/04/07/gameclock/))来挑战自己的技能极限。他来自软件设计背景，这一点在休息后的视频中的 UI 设计中得到了体现。我们喜欢他的原型制作过程，从 Arduino 和试验板开始，到这个独立的计时器结束。

在建造了第一个带有四个按钮和一个字符 LCD 的工作原型后，他转向了一个塑料冰淇淋容器作为外壳。这已经足够好了，但是脆弱的外壳需要升级。当他展望下一个版本时，他决定转向 Arduino Nano 板以节省空间。其余的组件被焊接到某个原型板上，带有一对引脚头来接收 Nano。成品电路板的长度与 Nano 相同，宽度只有它的两倍。

这个盒子是在电脑上模拟的(对我们来说它看起来像 SketchUp，因为我们可能是错的)，然后从 Masonite 的碎片上切割下来。它有一个角色 LCD 和一对街机按钮，每个玩家都可以把时间负担转移给对手。中间的按钮可以暂停游戏，背面有一个 trimpot 可以调节屏幕对比度。[马蒂亚斯]设法包括了惊人数量的设置，这将使这个小盒子适用于广泛的游戏类型。
