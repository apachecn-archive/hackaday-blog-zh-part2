# 问 Hackaday:导电墨水会成功吗？

> 原文：<https://hackaday.com/2015/02/13/ask-hackaday-are-conductive-inks-going-to-make-it/>

令人惊讶的是，PCB 制造成本变得如此低廉。长期以来，在家里制作自己的单面和双面板是经济的(尽管并不总是简单的)，专业制作变得越来越普遍。成本和周转时间继续下降。但是人们对非传统的兴趣越来越大。

在过去的一年里，我们看到了基于导电墨水的 PCB 技术的巨大推动力。这些技术的目标是小批量的原型制作，利用悬浮在液体(比如胶水)中的金属(通常是银)来绘制轨迹，而不是在一层薄薄的铜上蚀刻出轨迹。我们的问题:你认为传导 in 会成为一个可行的原型选项吗？

## Voltera V-One 电路板原型机

[https://www.youtube.com/embed/exRxh0f0hpI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/exRxh0f0hpI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

我在 2015 年 CES 上录制了这篇采访，但被要求在他们的众筹活动上线之前不要发表。如果你没有注意到，Voltera 在剩下的 26 天里几乎完成了他们 7 万美元目标的 400%。这台打印机肯定能用。你可以印刷电路，焊接元件~~或回流它们~~，甚至还有第二种不导电的墨水可以用来在迹线交叉时绝缘。在视频中，[Alroy]建议 Voltera 进行 10-20 块电路板的小批量生产。你会把这个用在 10-20 块板上吗？

就个人而言，我认为我可以在更短的时间内焊接点对点原型。考虑到这一点:V-One 将打印你的痕迹，但你仍然必须自己焊接元件。如果电路板设计达到较高的复杂程度，时间可能会改变，但与铜走线相比，油墨电阻的增加如何影响电路板的可行性？我认为太复杂而不能点对点焊接的东西将会深入到高频通信中(想想 LCD 显示器的并行总线，等等)。).我的假设正确吗？你认为导电墨水会比蚀刻你自己的原型既可行又可取吗？我们需要多长时间才能实现？

现在，我确实看到了 Voltera 的一些完美用例。比如电路设计概论课。如果你在初中或高中就有一台这样的打印机，它会激发你对电子工程的兴趣。不需要在手边准备氯化铜或氯化铁之类的化学试剂，你可以带学生们完成简单的电路板设计和组装，最终产品可以带回家。这是一个我绝对可以支持的愿景，我认为它将开启下一代硬件黑客的大门。

**更正:**【Arachnidster】在评论中指出，Voltera 仍在致力于能够回流由 V-One 打印的电路板。在他们的 Kickstarter 页面上，他们提到:“(Voltera 印刷电路板的回流目前正在开发中)”
