# 3 DOF 开源机械臂只是一个开始

> 原文:[https://hack aday . com/2014/03/27/3-DOF-开源机器人手臂只是个开始/](https://hackaday.com/2014/03/27/3-dof-open-source-robot-arm-is-just-the-beginning/)

![Arm3-v1](../Images/bf4b5db4352c51e744c201032d96fb0d.png)

边际聪明的丹·罗耶做了一个梦。建造一个任何人都可以制造的开源 6 自由度机器人的梦想！为此，他在过去两年中一直在学习机器人技术，并且刚刚完成了第一步——他设计并制造了一个开源的 3 自由度码垛机器人！

他的这个小家伙是基于商用 [ABB 460 码垛机器人](http://www.abb.ca/product/seitp327/ff492162d8dbaf44c1257861005de2fe.aspx)的，这是一个久经考验的真正的工业机器人。它的特点是所有的激光切割部件，一些螺母和螺栓，一些步进电机和一个 Arduino UNO 大脑。他已经在 Thingiverse 上发布了所有的[设计文件，在](http://www.thingiverse.com/thing:280263) [GitHub](https://github.com/MarginallyClever/Arm3) 上发布了固件——这是另一个项目，如果我们有一台激光切割机就好了！

别担心，Arduino UNO 只用于第一个原型，他已经开始为第二版的 RUMBA(带快板驱动的 Reprap 通用 Mega Board)控制器编写代码。

留下来看它用记号笔写下它的第一个问候——你好，世界！
