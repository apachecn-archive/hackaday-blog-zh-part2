# 智能心肺复苏假人确保你做得正确

> 原文：<https://hackaday.com/2014/12/22/smart-cpr-dummy-makes-sure-you-do-it-right/>

你参加过急救和 CPR 培训课程吗？难道你不喜欢你用来练习的假人模特的真实感吗？康奈尔大学的学生 Park，Qurashi 和 Chen 认为这些假人可以使用智能升级。

这是他们电子和计算机工程课程 ECE 4760 的期末项目。他们所做的是使用古老的 ATmega1284 微控制器成功地创造了一个预算友好的 CPR *不那么愚蠢的*假人。

当进行胸部按压时，假人可以感觉到，当进行呼吸时，鼻子是否被正确堵塞，头部是否向后倾斜以打开气道，它甚至利用麦克风来检测呼吸是否正确！在这样做的同时，它使用 LED 眼睛和 LCD 屏幕向学生提供培训反馈。一旦学生充分练习，它也有一个“真实”模式，不会给你任何反馈，以确保学生真正学到了技术。
