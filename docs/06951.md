# WiFi Raspberry Pi 触摸屏摄像头

> 原文:[https://hack aday . com/2014/08/30/wifi-raspberry-pi-触摸屏-摄像头/](https://hackaday.com/2014/08/30/wifi-raspberry-pi-touchscreen-camera/)

![raspberry_pi_open-case-parts](../Images/d6ee34f3ec4d798e66bd7b5c8de66413.png)

Adafruit 在他们的网站上有一个教程，展示如何组装一个[云连接的傻瓜相机](https://learn.adafruit.com/diy-wifi-raspberry-pi-touch-cam/overview)。这个项目最好的部分是，它可以根据心脏的内容进行定制，不像传统的数码相机或智能手机。集成的触摸屏和开源计算允许类似 Instagram 的过滤器可以轻松滚动。不需要外壳，但可以附上 3D 打印的外壳，以获得更完美的效果。

这个支持 Raspberry Pi 的设备的备份系统无线连接到互联网，并通过使用 Dropbox API 上传照片。这项功能非常适合将相机同步到基于云的服务器上，然后可以将它变成一个网站的临时图片数据库。这种相机可能也适合记录延时摄影，程序可以从备份照片中自动创建 gif。这看起来也不难，尤其是因为 Adafruit 几乎总能提供很好的文档。他们的视频通常也很好。下面发布的内容相对较短，但提供了足够的信息来了解它是如何工作的。
