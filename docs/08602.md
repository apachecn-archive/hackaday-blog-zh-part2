# MRRF 的比格波恩斯

> 原文:[https://hackaday.com/2015/03/22/beaglebones-at-mrrf/](https://hackaday.com/2015/03/22/beaglebones-at-mrrf/)

[杰森·克里德纳]——比格伯恩的家伙—[本周末前往中西部说唱音乐节](https://www.youtube.com/watch?v=CdxCB11XkHA)。有很多单板计算机，但 BeagleBoard 和 Bone 非常适合控制打印机和运动控制系统，这要归功于板上的实时 PRUs。如果你想玩复古电子游戏或建立一个媒体中心，它不是你的董事会；这是建筑材料的板子。

在 BeagleBooth 上感兴趣的是一些专门为 CNC 和 3D 打印工作设计的斗篷。[有痉挛](http://reprap.org/wiki/CRAMPS)，一个非常受欢迎的 RAMPS 3D 打印机电子板的克隆，为小猎犬号制造。如果你试图控制一台只能通过并口控制的旧磨机，[这里有块板给你](http://karltech.com/2013/11/announcing-the-first-in-a-series-of-new-products-io-1/)。有 3D 打印机板[和荒谬的布局](http://reprap.org/wiki/T-Bone)，它们既能作为打印机控制器板工作，也是为什么你永远不应该在建造它之前想出名字的原因。

[杰森]的 MRRF 之行不仅仅是为了颂扬普鲁斯的美德；一个伟大的运动控制软件 Machinekit 也在那里，在几只小猎犬上运行。BeagleBooth 的打印机正在运行 Machinekit，除了几行 GCode 导致打印头撞上零件之外，一切都运行良好。
