# 刮刮乐马达采用了巧妙的设计

> 原文:[https://hack aday . com/2012/10/23/scratch-build-motor-uses-a-clever-design/](https://hackaday.com/2012/10/23/scratch-built-motor-uses-a-clever-design/)

![](../Images/cb838f787f6db7bee609359423283694.png "lou-builds-a-motor")

[Lou]在有趣的建造方面一直处于领先地位。这一次，他用木头、PVC、一些紧固件和一束漆包线制作了自己的马达。

他制造换向器的方法很有趣。他首先制作了一个转子，方法是切掉 PVC 四通接头的两个相对面，将一根短镀锌管穿过剩下的部分。在添加了两个带有盖子和钉子作为支点的 PVC 小块后，他用一段连续长度的漆包线包裹 PVC 和金属管。然后用砂纸打磨掉 PVC 周围绕组的珐琅，一半用绝缘胶带覆盖。旋转的转子将导致电刷在旋转的一半期间接触裸线，并在另一半期间被胶带绝缘。休息后的视频显示了运行中的电机，然后带你走过构建的每一步。

如果你喜欢这个视频，你应该看看[Lou]的[水瓶火箭发射器](http://hackaday.com/2012/10/10/build-your-own-water-rocket-launcher/)，或者他的[自动乒乓球台顶端](http://hackaday.com/2012/09/24/garage-door-opener-used-to-automatically-lower-a-game-table-top/)，它把游戏存储在天花板上。
