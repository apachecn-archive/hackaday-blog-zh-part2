# 有了这个床弹射系统，醒来很容易

> 原文：<https://hackaday.com/2015/07/31/waking-up-is-easy-with-this-bed-ejection-system/>

醒来很难。当你醒来的时候，床实在是太舒服了…这就是为什么你需要升级你的床。由我们唯一信任的人制造的这种设备，[柯林·福尔泽]真的提高了他的项目的赌注——他称这个为[高压喷射床。](http://www.youtube.com/watch?v=EVzn1pl4nlo)

在一家咖啡公司的赞助下，[科林]真的全力以赴搞这个项目——如果这是产品营销的未来——把我们也算上。这种床使用两个巨大的气压缸来翻转床尾，就像自动倾卸卡车——或者像[科林]说的，卡车。全金属框架肯定花了一大笔钱——但这东西看起来会永远用下去——毕竟，他也是一名焊接专家。

休息后留下来看它把[科林]投入意识。
