# 无人机救援行动

> 原文：<https://hackaday.com/2015/07/29/operation-drone-rescue/>

当[哈里森·霍维斯]在邻居的屋顶上丢失了他的 Syma X5C 无人机时，他认为所有的希望都破灭了。很难到达那里，邻居对他试图营救也不感兴趣。几个月过去了，哈里森有了一辆新的四轮车。不是一般的四轴飞行器，而是 DJI 幻影 3 专业版。是时候尝试一次空中救援行动了！

利用一些旧衣架和一些绿色油漆工的胶带，哈里森制作了两个挂钩挂在幻影的下面。他还将 FPV 模块垂直向下倾斜，以获得救援的最大可见度。

随着汉斯·季默(来自*国际零售商*)的《没时间小心了的配乐响起，看着我们的英雄熟练地将他的旧四轮车从屋顶上抬下来，回到安全的地方。
