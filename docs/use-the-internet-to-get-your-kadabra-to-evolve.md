# 使用互联网让你的卡达布拉进化

> 原文：<https://hackaday.com/2015/04/01/use-the-internet-to-get-your-kadabra-to-evolve/>

如果你是玩《口袋妖怪红》或《口袋妖怪蓝》长大的，你现在可能已经远离你童年的朋友了。如果你还在玩红色或蓝色口袋妖怪，你现在可以使用[Pepijn]新改进的 Game Boy 链接与这些朋友重新联系，该链接[允许玩家通过互联网](http://pepijndevos.nl/TCPoke/)交易口袋妖怪。

基于[Pepijn]之前的工作，构建了一个基于 [Arduino 的口袋妖怪存储系统](http://hackaday.com/2015/02/13/bills-arduino/)(其灵感来自于能够欺骗交易的[独立项目](http://hackaday.com/2014/01/02/spoofing-pokemon-trades/))，该设备允许游戏男孩(包括口袋版、彩色版和高级版)通过一个小盾连接到互联网。在线等候室软件叫做 TCPoke，它方便了游戏迷们的上网。从那里，你所要做的就是通过项目的[维基](https://github.com/pepijndevos/TCPoke/wiki)连接！

TCPoke 软件可在项目网站上获得。此外，请务必查看下面的视频，该视频展示了该软件是如何工作的。与 Game Boys 之间的直接链接相比，有明显的延迟，但它的功能非常好。我们没有看到这个链接系统在战斗中起作用，但看看它是否可能会很有趣。如果是这样，你可能再也不用去参加口袋妖怪联盟会议了！
