# 秘密键盘储藏

> 原文：<https://hackaday.com/2015/07/18/secret-keyboard-stash/>

藏在众目睽睽之下是一条古老的公理，也是[基普凯]铭记在心的一条。他的[偷偷摸摸的键盘黑客](http://kipkay.com/episodes/sneaky-keyboard-hack/)将很少使用的数字键盘转换成一个方便(和秘密)的隐藏孔，用于放置你想避开窥探的小物体。

你可能必须根据你的具体型号来调整黑客，但[Kipkay]切割出薄膜键盘，用热熔胶固定数字键盘键，然后用 Dremel 切割出来。一些纸板制成隔间，一旦假键盘就位，没人会知道。

正如你在休息后的视频中看到的，隔间不是很大。你不会把手机藏在里面，但它的大小正好可以放一些应急现金、信用卡或者一两张 SD 卡。

隐藏在众目睽睽之下并不是黑客的新想法。我们已经讲述了[秘密的 LCD 信息](http://hackaday.com/2015/07/15/trick-google-used-hides-secret-messages-on-lcd-screens/)、[秘密的阁楼门](http://hackaday.com/2014/12/25/secret-attic-library-door/)，以及一个(不完全成功的)[藏在书里的洞](http://hackaday.com/2014/03/27/fail-of-the-week-secret-agent-style-book-hideaway/)。更不用说通过隐写术在众目睽睽之下隐藏数据的许多方法了。
