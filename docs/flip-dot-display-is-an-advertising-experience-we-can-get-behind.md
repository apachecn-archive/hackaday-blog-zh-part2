# 翻转点显示是我们可以获得的一种广告体验

> 原文：<https://hackaday.com/2012/07/19/flip-dot-display-is-an-advertising-experience-we-can-get-behind/>

这个[巨大而迷人的互动展示](http://breakfastny.com/2012/06/reactive-super-speed-electromagnet-dot-display-per/)只是一个大广告。这是一个翻转点显示器。每个像素都是一个机械圆盘，一边是白色，另一边是黑色。早餐网的团队破解了显示模块，并编写了自己的软件，这样它就可以以闪电般的速度刷新。在左边，你可以看到高对比度的文本，但在右边，它显示的是基于摄像头的交互性。几秒钟后，这位先生向一边挥动手臂，所有的像素都散开了，就像被大风吹走了一样。休息之后你可以直接跳到视频。

这个展示是 TNT 网络上一个新节目的广告，名为 *Perception。*我们不得不说，如果你要做广告，这是最好的方式。做一些很酷的东西，然后分享细节。我们开始享受所有这些点翻转到位的咔嗒声，他们让我们至少认识了网络并说出了节目的名称。每个人都赢了。
