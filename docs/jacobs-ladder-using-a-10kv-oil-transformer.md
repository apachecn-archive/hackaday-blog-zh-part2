# 使用 10kV 油变压器的天梯

> 原文：<https://hackaday.com/2014/08/21/jacobs-ladder-using-a-10kv-oil-transformer/>

![Jacob's ladder using an Oil Furnace Transformer](img/5d2ba99e8fb0e103e546d57de4598196.png)

雅各布之梯是任何自尊的疯狂科学家的巢穴——呃，黑客工作室——中的一个主要实验。为什么不呢？高压，电弧，甚至足以杀死你——太棒了！但是尽管雅各布之梯很棒，但它并没有那么复杂。

在凯文·达拉的最新教程中，他向我们展示了如何用从油炉中取出的变压器制作一个。为什么油炉会有高压变压器呢？它们实际上被用作点火源，就像指示灯一样！

[Kevin]拥有的是一个 110VAC 到 10,000VAC 的变压器，它输出大约 20mA(可能足以杀死你)。要把它变成一个雅各布的梯子，你只需要两根长而硬的电线(铜是一个很好的选择)。电线最靠近变压器容易产生电弧的底部，电弧电离并加热空气，使空气上升，携带电弧。随着电弧沿着梯子向上延伸，它变得越来越长，因为电线变得越来越远，变得越来越不稳定，直到它断裂。当这种情况发生时，电弧在电阻的最低点——底部再次形成。

如果操作得当，这当然是一个有趣的实验，只要确保你在使用任何高压设备时都是安全的——不要像这个人一样使用类似的变压器给一些自制的狼獾爪子通电。
