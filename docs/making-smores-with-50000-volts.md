# 用 50000 伏的电压做馅饼

> 原文：<https://hackaday.com/2013/08/26/making-smores-with-50000-volts/>

[Skyy]给我们发了一段他用电弧烹饪 s'mores 的视频。他使用一个带有零电压开关(ZVS)驱动器的反激式变压器。这产生了大约 50 千伏的电压，足以烘烤棉花糖。

ZVS 是一种当半导体开关上的电压为零时触发半导体开关的技术。这确保了开关产生的热量最小，因为它们在切换时不会中断任何电流。ZVS 也用于照明调光器，以切断电源而不产生干扰。

如果你对细节感兴趣，有一个关于构建驱动程序的很棒的教程。如果你有兴趣了解它是如何工作的，看看这个[模拟视频](http://www.youtube.com/watch?v=0ONq3DxXE2w "ZVS Driver Simulation")。

[Skyy]承认他的装置并不十分安全，因为它使用的是试验板，不能承受高电压和高电流。请记住，这些电路可能会杀死你。休息之后，看一个棉花糖在 50 千伏的电弧中炸。
