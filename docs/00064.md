# 钟罩里的“点唱机”

> 原文:[https://hackaday.com/2012/04/10/a-jukebox-in-a-bell-jar/](https://hackaday.com/2012/04/10/a-jukebox-in-a-bell-jar/)

[![jukebox in bell jar](../Images/5adf8877dac2123fd28338be79a1ae4c.png "jukebox30")T2】](http://hackaday.com/2012/04/10/a-jukebox-in-a-bell-jar/jukebox30/)

[Tim]写信告诉我们他的 [Mark III 硬盘](http://www.youtube.com/watch?v=XGPPSzfo0aI "hard drive in bell jar")在一个钟罩里，我们印象深刻！使用钟罩保护里面的硬盘驱动器以便全世界都能看到它旋转的原理确实是一个很酷的想法，但他对这个项目的执行非常出色。据报道，它是如此之好，以至于一所不知名的大学竟然要求他为他们建造一个驾驶显示器。

马克三是一个很好的建设，小硬盘已被交换到它的前面，以便更好地可视化。用于 Mark III 构建的 iPod Mini 的缺点是，它实际上有足够的缓冲，因此读取头在整首歌曲中不必旋转。正因为如此，我们认为[他的第一个版本](http://www.youtube.com/watch?v=Q5un_GkwjCs "first hard drive in bell jar")，昵称为【pink】，甚至更有趣，如果稍微不那么精致的话。尽管以我们的许多黑客标准来看，两者都是令人难以置信的艺术精品！

这两个建筑都有过多的发光二极管让你娱乐，休息后可以看到演示！
