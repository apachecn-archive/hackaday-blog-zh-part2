# Hackaday 奖品入口:多肉猪肉

> 原文：<https://hackaday.com/2015/05/20/hackaday-prize-entry-the-multispork/>

如果你正在做一个移动项目——一个机器人，外面的东西，甚至是你的汽车——你不希望带一个示波器，逻辑分析仪，信号发生器，或者任何其他应该放在工作台上的设备。为了他的 Hackaday 奖参赛作品，[皮尔斯·尼科尔斯] [正在研发一款电子版的皮衣人](https://hackaday.io/project/1577-multispork):小巧便携，在紧要关头也能应付自如。

正如[Pierce]对他的设备的称呼，MultiSpork 是一个完全便携的支持 WiFi 的单板。加上一个 50 美元的 Android 平板电脑，它非常接近于一个盒子里的完整电子实验室。

MultiSpork 的核心是 Maxim 的新芯片 [MAX 11300](http://www.maximintegrated.com/en/products/analog/data-converters/analog-to-digital-converters/MAX11300.html) 。该芯片有 20 个引脚，可用作 12 位 ADC、12 位 DAC 或 GPIOs。它是一个逻辑分析仪，信号发生器，示波器，和一个单芯片的总线盗版。至于董事会的其余部分，[皮尔斯]正在放弃任何硬件冻结的概念，并将 Atmel 微控制器改为 TI CC3200 芯片，这将很快推出。
