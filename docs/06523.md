# Pi 供电的反猫陷阱浸泡猫科动物和其他动物

> 原文:[https://hack aday . com/2014/07/10/pi-powered-anti-cat-trap-soaks-felis-and-other-animals/](https://hackaday.com/2014/07/10/pi-powered-anti-cat-trap-soaks-felines-and-other-animals/)

[约书亚]有猫的问题。他们喜欢在他的花园里便便。他决定采取行动，使用这个令人敬畏的自动化且人性化的捕猫器。

现在只是澄清一下，在全力为猫辩护之前，他确实尝试了一些其他的选择——最简单的解决办法是自己养一只猫，但是，唉，他不是一个喜欢猫的人。

该系统在防水外壳中使用了树莓皮，并配有 PiCam。他写了一些基本的代码来利用 PiCam Python 库，该库也允许他记录捕捉猫之前的片段，这让我们很开心。当感应到运动时，Pi 触发一个 24v 交流电磁水阀，打开洒水器，迅速浸湿入侵者。

休息后留下来看几个视频，看看这些毛茸茸的小家伙在干什么！
