# 离 Collabatorium 还有 3 个小时

> 原文：<https://hackaday.com/2015/07/08/3-hours-until-collabatorium-2/>

这是你的 90 分钟警告。本周的 Collabatorium 将在 1.5 小时后开始，您将受到邀请。要加入，只需点击黑客频道页面左侧栏的[中的“请求加入这个项目”。](https://hackaday.io/project/5373-hackaday-prize-hacker-channel)

加入后，您可以[打开该项目的群发消息](https://hackaday.io/messages#/conversation/2369)，这是 Hackaday.io 上众多精彩协作功能之一。从世界协调时 17:00 开始，我们将推出 Collabatorium 来庆祝、讨论、鼓励和寻找 2015 Hackaday 奖参赛作品的合作伙伴。本期现场活动由[ [索菲·克拉维茨](https://hackaday.io/sophikravitz)和[ [茉莉·布莱凯特](https://hackaday.io/jasmine)主持。

[![vote-short](img/cd7faa9049541bafdc4fa2b5c90a919b.png)](https://hackaday.io/prize/vote) 当我们引起你的注意时，这里有另一个提醒，请前往[投票决定是否成为宇航员](https://hackaday.io/prize/vote)。每周我们都会随机抽取一个黑客号码作为 1000 美元的赠品，*，但前提是你已经投票了！*。下一次抽奖就在今天，所以请马上至少投一票以获得资格。
