# 树莓派成为吉他效果处理器

> 原文:[https://hack aday . com/2013/01/28/raspberry-pi-becoms-a-吉他效果处理器/](https://hackaday.com/2013/01/28/raspberry-pi-becomes-a-guitar-effects-processor/)

Raspberry Pi 的一个更有趣的用例是以有趣的方式开发其 DSP 功能。树莓派内部有很多马力，足以用音频做一些非常有趣的事情，同时由一个小的壁式适配器供电。纯数据邮件列表上的[Pierre]有一个概念验证工作,它使用 Raspi 作为吉他效果处理器。[结果](http://www.youtube.com/watch?v=NwJNeouLqgQ&feature=youtu.be)非常令人鼓舞——【Pierre】能够使用他的 Raspi 作为延迟、音高移位器，当然还有延迟约 16 毫秒的经典侧翼、移相器和合唱

Raspi 的音频接口需要几个步骤来获得低延迟。[Pierre]正在无头运行他的 Pi，[给 CPU](http://guitarextended.wordpress.com/2012/09/11/setting-up-the-raspberry-pi-for-pd-1-allocate-more-ram-to-cpu/) 分配了更多的 RAM。

如果你想自己尝试一下，[Pierre]有一个教程，教你如何用 Raspberry Pi 设置纯数据。他将很快更新他的博客，稍后会有更多的教程和经过验证的 USB 音频接口。

休息之后，请查看运行中的处理器。
