# 用硅树脂封装电子设备

> 原文：<https://hackaday.com/2012/06/04/potting-electronics-with-silicone/>

如果你想提高你的电子产品的可靠性，就用硅树脂或环氧树脂覆盖它们。这种技术被称为封装，可以保护你的项目免受冲击和松散的电线，但大多数情况下最终都是一团糟。[查尔斯]想出了一个绝妙的方法，用硅树脂和急诊室的一些用品来制作看起来很专业的罐装电子产品。

在[查尔斯]的一次急诊室之旅中，一名护士将[tega derm](http://solutions.3m.com/wps/portal/3M/en_US/3MSWC/Skin-Wound-Care/BrandsDirectory/Tegaderm/)放在伤口上，他注意到他的皮肤可以呼吸。考虑到 Tegaderm 还能让硅胶呼吸，他要了一份样本，然后回到实验室。在 PCB 上放上几滴硅胶后，[查尔斯]用 Tegaderm 把整个东西包起来。令他惊讶的是，硅胶固化后在他的板上形成了一层光滑、有光泽的保护层。

[Charles]放了一张他的 tega derm/硅树脂封装电子产品的 Picassa 相册,我们不得不说它们看起来非常专业。这种技术也可以应用于【查理】’[玻璃 PCB 触摸传感器](http://hackaday.com/2011/09/12/glass-pcbs)，允许创建清晰(或至少透光)的 PCB。

休息过后，你可以看看[Charles]用 Tegaderm 将他的 ATtinyISP 板封装在硅胶中。不过，有一个警告:不要使用使用醋酸固化的硅树脂——[那是对电子产品的死刑判决。幸运的是，有一个简单的测试来看看你的特定品牌的硅树脂在固化时是否会产生乙酸。只是在一块废料上挤一点硅胶，如果过一会儿有醋味，就不要用了。](http://www.logwell.com/tech/servtips/RTV.html)
