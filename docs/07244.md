# 用一个腕部调谐发射机给肩扛式音箱馈电

> 原文：<https://hackaday.com/2014/10/05/feed-that-shoulder-boom-box-with-a-wrist-tune-transmitter/>

下次当你肩扛高保真音响昂首阔步走过街区时，不要让自己受限于电台紧凑的播放列表或磁带短暂的播放时间。把你的音乐装在手腕上，用这个可佩戴的调频发射机直接传送出去。没有电线…就像未来就是现在！

树莓派已经证明了自己是一个可靠的调频发射机。这个项目跟随这些脚步，但将目标线向前推进了几大步。该建筑有一个完整的用户界面，这将使它很容易适应几乎任何你能想象的应用程序。在休息之后，视频的后三分之一显示了增加的扭曲。[Navic209]在设计中加入了一个麦克风，佩戴者可以将声音传送到一个适当调谐的收音机中。这给了这个设备一种非常迪克特蕾西风格的感觉。
