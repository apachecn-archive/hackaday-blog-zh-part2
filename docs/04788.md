# 本周失败:物理像素显示器

> 原文:[https://hack aday . com/2013/12/12/fail-of-the-week-physical-pixel-display/](https://hackaday.com/2013/12/12/fail-of-the-week-physical-pixel-display/)

![fotw-physical-pixel-display](../Images/9a7e73b63b96b6341bc08bec6f7e4521.png)

这种物理像素显示器让我们想起了多年来我们看到的一些不同的黑客攻击。它看起来令人印象深刻，但[马特]不能完全让它工作。这不是 Kinect 传感器和图像解释的问题。让上面看到的硬件组件可靠地执行是[失败。](http://electronicsfluff.blogspot.com/2013/11/kinect-pin-art-fail-back-in-winter-of.html)

如果你不知道这应该做什么，看看去年现代博览会上安装的[信息变形台](http://hackaday.com/2013/11/07/inform-mits-morphing-table/)或[像素墙](http://hackaday.com/2012/09/18/thousands-of-physical-pixels-turn-these-walls-into-a-huge-display/)。[Matt]的尝试要小得多，只有 10×6 的网格。像素本身就是圆珠笔(他拿便宜易得的材料加分)。由于一些[鲍登电缆](http://en.wikipedia.org/wiki/Bowden_cable)连接到业余爱好伺服系统，笔可以进出。机械工程师可能已经找出了故障…像素似乎被挂起，尽管使用的材料进行了几次修改，但无法修复。

选择业余爱好伺服系统是因为它们比合适的线性致动器便宜得多。我们认为也许[Matt]应该[建造他自己的螺线管](http://hackaday.com/2010/12/02/make-your-own-solenoids-then-play-the-xylophone/),但是那不是一个好主意，因为你不能用那种方式有可变的深度(你能吗？).也许钢笔应该是垂直的，伺服系统可以通过重力[拉动通过滑轮连接到钢笔](http://hackaday.com/wp-content/uploads/2013/12/physical-pixel-gravity-concept.png)的绳子，使它们回到开始位置？一定有一种便宜且相对简单的方法让这东西工作起来。请在下面留下评论，让我们知道你如何让项目回到正轨。
