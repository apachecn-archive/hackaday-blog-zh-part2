# 帮助 Hackaday 为一个好的理由建立一个定制的游戏控制器

> 原文:[https://hack aday . com/2013/02/12/help-hack aday-build-a-custom-gaming-controller-for-a-good-cause/](https://hackaday.com/2013/02/12/help-hackaday-build-a-custom-gaming-controller-for-a-good-cause/)

[![jstick-R-1](../Images/c4099b647222266c7b7a55faddeb2694.png)T2】](http://hackaday.com/wp-content/uploads/2013/02/jstick-r-1.jpg)

我们将为患有肌肉萎缩症的儿童定制游戏控制器。他叫托马斯，喜欢 minecraft。这是我多年来一直想做的一个项目。我现在才刚刚开始，你可以[加入这个项目，在我们的论坛](http://forums.hackaday.com/viewtopic.php?f=3&t=3080)上发表你的想法。我们从托马斯开始，但最终，我们想开发一个相当简单的开源游戏控制器的集合。

对于身体残疾的人来说，游戏会对心理产生深远的影响。有时候，这是他们唯一可以去的地方，可以和他们的同龄人公平竞争。定制游戏控制器通常非常昂贵，尤其是当你开始离开标准的 xbox/playstation 外形时。

现在，[手里拿着 3d 打印机](http://hackaday.com/2013/02/09/adventures-in-3d-printing-our-first-week-with-the-lulzbot-ao-100/)，我能够比我想象的更快更容易地做出原型。我可以在几分钟内设计出一些东西，然后打印出来，看看*感觉如何*。一旦我确定了一些东西，我就可以改进设计，使其更容易组装，并适合你可以从 xbox 控制器中获得的库存零件(他们有官方的 windows 驱动程序，这很好)。

这是我玩原型创意的一个快速视频。你可以看到，我首先从控制器中布局出所有的标准游戏输入，然后我把一块假想的塑料扔在上面，开始雕刻。你会想把 velcro 放在底部，这样它就能以一个完美舒适的角度呆在膝盖板上。

[https://www.youtube.com/embed/DzvTJbPxDqc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/DzvTJbPxDqc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)T2】

这绝不是一个完成的原型。这只是一个快速的造型，让我的想法下来。虽然我只做了控制器的一半，但你可以想象你可以很容易地像这样放置两个单元，这样你的手就可以放在上面了。

[加入论坛](http://forums.hackaday.com/viewtopic.php?f=3&t=3080)帮我想出更多可能有用的设计！
