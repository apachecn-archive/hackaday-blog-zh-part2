# 现代木工工作台

> 原文：<https://hackaday.com/2014/11/21/a-modern-woodworking-workbench/>

这是一个关于工作台的帖子，但不是你可能会想到的工作台。用于电子开发的工作台很简单——几乎任何平面、几个设备架和一个防静电垫都可以满足任何可能的用途。木工工作台完全是另一回事。木工工作台的发展背后其实有相当多的历史，但基本想法是一个厚的层压木材顶部，集成虎钳，工作面上的孔用于工作台狗，以及人体工程学，允许舒适地使用手动工具。几百年来，这些长椅的基本设计没有太大变化，[和【Dirk】认为这种设计已经可以进行现代更新了](http://www.21stcenturywoodworking.com/finished-project-video/)。

是的。这个会自己移动。这是一个机器人木工工作台，可以将工件和桌子提升到舒适的位置。提升机构由四个 Acme 螺纹杆组成，全部由 Arduino 控制的步进电机驱动，通过链轮和链条连接在一起。有一个遥控器来升高和降低工作台，还有一些电池藏在机械后面，以允许离网运行。

长凳必须坚固，这种长凳的框架上有夹子，可以将“电梯轿厢”牢固地夹在长凳上。调平脚轮使得这种长凳可以移动，使[Dirk]能够在商店里移动它，或者从一个地方移动到另一个地方。一体式端面虎钳和双螺杆端面虎钳可将工件牢牢固定在工作台上，亚麻子油涂层可轻松修复划痕和擦伤。

框架的大部分是由数控切割的桦木胶合板制成的，所以如果你在当地的 hackerspace 有一个 Shopbot 或其他大型路由器，为自己建造这个长凳比更传统的木工工作台的榫眼和榫接合要简单得多。如果你最终建立了这个板凳，一定要拿起脚轮[德克]使用；这东西有 800 磅重。巨大，沉重，是一个优秀的长椅，可以传递给你的孙子。下面视频。
