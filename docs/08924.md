# 建造一辆方形自行车

> 原文：<https://hackaday.com/2015/05/01/building-a-square-bicycle/>

受到[迈克尔·乌贝森·雅各布森的 BauBike 设计的启发，[亚伦·塞特]决定尝试[建造自己的](http://imgur.com/a/YHn4T)。

整个框架由 1 英寸的方形钢管制成，很快就焊接在一起。实际上，为了骑上这辆自行车，他必须对 BauBike 的原始设计做一些改动。最重要的是，他抬高了座位，以便在踩踏板时可以适当伸展腿部。

有趣的是，Reddit 上的大多数[人都认为这是行不通的，所以在将图库发布到 imgur 后不久，[Aaron]也上传了一个短视频到 YouTube，以证明，是的，他实际上可以骑它。休息后留下来看看它的运行。](https://www.reddit.com/r/DIY/comments/3467x9/how_i_built_a_square_bicycle_out_of_square_tubing/)
