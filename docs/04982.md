# [Fran]的 led、Nixies 和 VFD。

> 原文:[https://hackaday.com/2014/01/04/frans-leds-nixies-and-vfds/](https://hackaday.com/2014/01/04/frans-leds-nixies-and-vfds/)

[![FRAN LED](../Images/cb6a44de8263a7d4d08997f25d6ce784.png)T2】](http://hackaday.com/wp-content/uploads/2014/01/fran-led.png)

凭借对 blinky 和 glowey 东西的热爱，这些年来[Fran]收集了很多电子显示设备。现在，她正在对她(和我们)最喜欢的部分进行一些拆卸和指导:[led](http://www.youtube.com/watch?v=yusrrOPM43o)和 [VFD 和谢妮管](http://www.youtube.com/watch?v=Z_Ga15np_Lk)

或许，拥有土星五号飞行计算机硬件的人也拥有大量老式组件并不令人惊讶，但我们只是惊讶于[弗兰]的收藏如此完整。她拥有有史以来最早的商用发光二极管之一。这是一个非常小的红色 LED，由孟山都公司(是的，公司)生产，包装在一个非常奇怪的铅杯包装中。

在她的 LED 系列中还有一个奇怪的西方电气部件，它是绿色的，但不是你期望从 LED 中看到的绿色。这种 LED 更像是一种翡翠色，而不是这种颜色，更像是 CMYK 过程中得到的绿色。如果能看到其中一个与红色、绿色和蓝色 LED 封装在一起，那就太棒了，考虑到 RGB LED 的色彩空间，它可能有一些有趣的应用。

除了她的发光二极管，[弗兰]还收集了大量的 VFD 和谢妮管。尽管易贝·塞勒斯相信，这两种技术是不一样的:VFD 是真正的真空管，有磷光涂层，工作原理类似于阴极射线管。另一方面，Nixies 充满了一种气体(通常是氖)，当电流流过其中一个数字时，这种气体就会变成等离子体。[Fran]有一大堆 VFD 和 Nixies——大部分是军事剩余——还送了几个给[达夫·琼斯]让他摆弄。

这都是非常酷的东西，也是我们听到的[弗兰]接下来要看的东西的一个很好的引子:在阿波罗导航计算机中发现的电致发光显示器。
