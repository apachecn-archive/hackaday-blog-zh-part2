# 问 Hackaday:谁来造这个气压传动的东西？

> 原文:[https://hackaday . com/2014/10/10/ask-hackaday-who ' s-to-build-this-pneumatic-transmission-thing/](https://hackaday.com/2014/10/10/ask-hackaday-who-is-going-to-build-this-pneumatic-transmission-thing/)

![fluid_transmission](../Images/44eb1d3b7203408355b98b6f59217e12.png)

迪士尼研究公司正在做他们最擅长的事情，为迪士尼制造非常酷的东西*，并告诉世界其他人他们有多酷。这一次，它是一个非常低摩擦的流体传动装置[，为电子动画](http://www.disneyresearch.com/publication/fluid-soft-actuator/)设计。*

通过测试一些玩具机械臂，我们可以毫无疑问地说，如果你正在设计需要逼真运动的机器人和电子动画，伺服系统和电机不是合适的选择。为了解决这个问题，迪士尼匹兹堡分校的一些研究人员转向了气动和液压，其中一个关节由两组活塞控制。它与气动乐高非常相似，但更精确，也更逼真。

该系统在机器人的每个关节上使用一对气缸。迪士尼使用滚动隔膜来密封其管道和气缸中的工作流体。这是一个极低摩擦的装置，没有简单的 o 型环气动和液压所发现的任何晃动或抖动。

该系统是可反向驱动的，这意味着一个机械臂可以控制另一个机械臂，*以及反过来的*。因为我们处理的是液压系统，液压缸(和机器人/电子设备)不需要相同的尺寸；一个小设备可以很容易地控制一个更大的自身副本，反之亦然。

这些装置相当简单，中间有齿轮、齿形带和一些塑料。这些机器人唯一独特的部分是滚动隔膜，我们不知道从哪里获得。它看起来很适合一些机器人或钢铁侠式的机甲套装，但能够找到组件将是一个挑战。
