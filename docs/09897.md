# Hammond Novachord

> 原文:[https://hack aday . com/2015/08/26/retro tech tacular-the-Hammond-nova chord/](https://hackaday.com/2015/08/26/retrotechtacular-the-hammond-novachord/)

就在我们以为已经听说了所有很酷的早期合成器的时候，一个线人告诉我们一个消息，有人完全修复了一个 Novachord。哈蒙德仅用了四年时间就制作出了这些钢琴形状的原型合成器。大约有 1000 辆是在 1942 年销量下滑、零部件短缺之前制造的。据估计，只有 200 只左右仍然存在。

Novachord 的声音由一组 12 个单稳态真空管振荡器产生。每一个都被调整到半音阶的一个音高，这就是所谓的分频结构。[Hammond]和他的共同创造者[John Hanert]和[C.N. Williams]利用将频率一分为二的特性来产生相同的音调，但低了一个八度。这种设计意味着所有 72 个音符可以同时演奏。可调共振峰滤波器塑造了通常超凡脱俗的声音，然后通过基于软管的包络传递。

[Phil]知道在任何情况下修复 Novachord 都是一项艰巨的任务。数以千计的无源元件都必须更换。这个柜子上有一个常用的客厅乐器的所有特征——鸡尾酒的水响、抓痕和大量香烟燃烧的痕迹。[Phil]说木工真的不是他的菜，但他做得很出色，打磨了每个角落和缝隙，还涂了几层染色剂。在他的项目网站上有大量令人垂涎的图片，还有几个[菲尔]真正测试它的片段。
