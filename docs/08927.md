# BB-8 开始制造

> 原文:[https://hackaday.com/2015/05/01/the-bb-8-builds-begin/](https://hackaday.com/2015/05/01/the-bb-8-builds-begin/)

是的，将于 12 月上映的新《星球大战》电影有 x 翼，通过星际驱逐舰的引擎进行的空战，太空战，一个死罐子罐子，还有[R2D2]。它还将有迄今为止最酷的机器人，[BB-8]，滚动球体主角，肯定会在 2016 年圣诞节前作为遥控玩具出售。

XRobots 的[James]不想等到[BB-8]玩具到来，[所以他正在建造自己的](https://www.youtube.com/watch?v=dlwcXgZYImU)。现在，它只是一个原型，但它是一个非常非常酷的机器人的机械和控制系统的开端。

我们已经看到了第一个 BB-8 机器人[，它基本上是在一个 Sphero 机器人](http://hackaday.com/2015/04/19/mini-bb-8-droid-made-from-a-sphero/)上放置一些磁铁，但这个机器人并不完全具有现实生活中的[BB-8]的相同功能。[真正的【BB-8】其实是两部分](http://hackaday.com/2015/04/19/bb-8-is-real-but-how-did-they-do-it/)，一个遥控球体，和一个独立的遥控头。[詹姆斯]正专注于他的原型的头部，用一个假人代替遥控身体，一个大的聚苯乙烯泡沫球。

[詹姆斯]' [BB-8]的头部是 3D 打印的，有一些特别聪明的设计特点。电子设备只有四个 DC 电机、一个 Arduino 和一些电机驱动器。在未来，[詹姆斯]可能会着眼于步进电机或伺服电机，但现在他的[BB-8]机器人可以站直，并作为一个伟大的平台来测试控制方案。
