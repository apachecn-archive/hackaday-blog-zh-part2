# 滑动式相机支架对于业余摄影来说已经足够好了

> 原文:[https://hacka day . com/2012/04/03/滑动式相机支架足以满足业余摄影爱好者的需求/](https://hackaday.com/2012/04/03/sliding-camera-mount-is-good-enough-for-amateur-photography/)

![](../Images/0a76c71c55f7ac507dfd2701de14fa5d.png "camera-sliding-rig")

[Unihopper]建造了这个[滑动相机支架](http://unitips.ca/?p=385)来为他的自由式独轮车视频添加一些动作。这非常简单，但仍然取得了非常好的效果，正如你在休息后的剪辑中看到的那样。

上图显示了没有连接相机的底座。你可以看到在前景中的块上的螺纹钉是用于那个目的的。毛毡被包裹在木块的底部，木块安放在一个木制的通道中。绳子与木块上的一个吊环挂钩相连，系在木板远端的线轴上。一个 K'nex 马达驱动卷轴，慢慢地将相机滑向卷轴。

与其他基于玩具的雪橇不同，轨道系统的使用有助于保持相机的正确方向。很明显，这不会实现你从像[这种轨道和线性轴承版本](http://hackaday.com/2011/12/20/motorized-camera-mount-unexpectedly-popular-for-cnc-aimed-hardware/)的雪橇系统中获得的完美平滑和精确的运动。但老实说，我们大多数人都没有质量足以保证这种高端系统的相机。 [https://www.youtube.com/embed/SNmHhpgobfY?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/SNmHhpgobfY?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)
