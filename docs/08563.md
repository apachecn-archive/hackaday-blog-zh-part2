# SXSW 创建:ATX 黑客空间区

> 原文:[https://hack aday . com/2015/03/17/sxsw-create-atx-hacker space-area/](https://hackaday.com/2015/03/17/sxsw-create-atx-hackerspace-area/)

我们周末在 2015 SXSW Create 度过了一段美好的时光。我真的很兴奋地看到，奥斯汀地区有一个非常大的区域留给了黑客空间，他们充分利用了这一点。最引人注目的是，ATX·哈克空间公司有多张桌子，吸引了一大群人。

[![sxsw-atx-theremin](../Images/a1184fa42287769715186ad138b8039f.png)](https://hackaday.com/wp-content/uploads/2015/03/sxsw-atx-theremin.png) 这张桌子就是手头演示的好例子。主要是超声波特雷门琴的集合。经典的特雷门琴使用基于振荡器的声音产生(我们已经在这个概念上运行了[系列)，带有一组天线，利用你身体的接近来调整信号。这个版本模仿了用户界面，但通过将天线换成超声波测距仪并以数字方式生成音频，大大简化了制作仪器所需的技能。更敏锐的观察者会注意到被拿着的乐器。我忘了问这个，但它看起来确实像](http://hackaday.com/2015/03/09/logic-noise-sawing-away-with-analog-waveforms/)[一个全息照相仪，这是你下一个项目的另一个伟大的种子想法](http://hackaday.com/2013/08/22/unplayable-holophonor-replica-is-unplayable/)。 **更新:**是个[葫芦丝](https://en.m.wikipedia.org/wiki/Hulusi)。

我认为值得注意的是，ATX 也留出了很多桌子，让他们的成员在活动中实际参与建筑项目。我们大力提倡这一点，而不仅仅是展示完成的项目。你在做什么并不重要；看到桌子上摆满了有趣的零件和工具，被有趣的人一起工作，显然享受彼此的陪伴是黑客空间的核心信息…对吗？

休息之后，我和[Gardner]在视频中讨论了 ATX，并在展示桌上快速浏览了一圈。
