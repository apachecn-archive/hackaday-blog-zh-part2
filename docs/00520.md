# 机器人猎鹰:有翼单位降落在你身上！

> 原文:[https://hack aday . com/2012/05/23/robotic-falconry-winged-unit-lands-on-you/](https://hackaday.com/2012/05/23/robotic-falconry-winged-unit-lands-on-you/)

![](../Images/cfa1ccadd1e1cd5f3ead5f2caa595ec8.png "perching-robot-lands-on-you")

它没有四个旋翼，但这种先进的滑翔机与最复杂的四旋翼飞行器一样令人印象深刻。这是第一架能成功停在你手臂上的滑翔机。我们不禁回想起原版电影《T2》中的猫头鹰。

伊利诺伊大学厄巴纳-香槟分校航天机器人和控制实验室的团队很高兴展示他们一直在进行的试飞。我们在休息后嵌入了其中的两个画面，显示了单元落在这个人的手臂上，和椅子的座位上。上面的图像显示了飞行中几帧的蒙太奇，让我们很好地看了一下铰接的机翼。你可以看到他们都在飞行中途弯下腰去瞄准着陆区。除此之外，在机翼和尾巴的后缘还有副翼。由于滑翔机没有垂直尾翼来稳定飞行，所以飞行路线有点偏离。

现在，如果他们能从高架电线上获取能量，他们就有了一只可以从飞机上(或无人机上)投下的间谍鸟。
