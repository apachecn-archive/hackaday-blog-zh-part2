# 我们有一个问题:淋浴反馈回路

> 原文：<https://hackaday.com/2015/03/18/we-have-a-problem-shower-feedback-loop/>

黑客日，我们有个问题。干净的水很珍贵，我们想提出一些想法来保护它。今天的话题是淋浴时浪费的水。让我们讨论一些想法，并为 Hackaday 奖推荐一些新的构建。

我们都做过。你会走神，在你意识到之前，你已经在浴室里站了太久了。你浪费了多少水？谁知道呢，洗澡的时候没有反馈回路。但是我们认为增加一点反馈是一个帮助打击浪费习惯的极好途径。

## 变色淋浴头

![shower-feedback-color-idea](img/1492aca75e631b6e895f855b07a874f4.png)

如果莲蓬头根据流过的水量改变颜色会怎样？我们已经看到了内置 led 的消费淋浴喷头，流量计也很容易买到。从绿色开始淋浴，当你打出泡沫时，泡沫会穿过蓝色、紫色、橙色、红色，最后变成闪烁的红色。它不一定是令人讨厌的，但足以帮助量化有多少被倒入下水道。

## 淋浴节拍

过去，我们是 SSX 诡计游戏的忠实粉丝。你越擅长变戏法，音乐就越好听。如果你撞得很重，你只会听到高帽和低沉的低音。将此应用于淋浴时间。如果你安装在淋浴头上的流量计连接到淋浴收音机会怎么样？从世界上最好的音乐[开始](https://www.youtube.com/watch?v=YJVmu6yttiw)，当你跑干水库时，进展到最烂的(当然是 ymmv 上的这些选曲)。

## 现在你试试

如果你洗澡时减少 5 秒钟，这对你的家庭影响很小。但是想象一下世界上所有家庭都这样做。
