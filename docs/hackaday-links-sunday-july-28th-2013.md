# 黑客日链接:2013 年 7 月 28 日，周日

[Chris Gammell]告诉我们他正在建立一个学习电子学的在线培训项目。十节课的课程需要付费，但是如果你是 beta 测试者之一的话，你可以免费得到这些东西。我们喜欢听 [*的安培小时*](http://www.theamphour.com/) 播客，这只是【克里斯】许多努力中的一个。

你这周买了一个 Chromecast 吗？我们有，但是我们手头还没有(通过亚马逊订购的)。你仍然可以从[iFixit 拆除](http://www.ifixit.com/Teardown/Chromecast+Teardown/16069/1)中看到里面。

通过用它编写一个乒乓游戏来练习你的处理技能。

一点打火机油和一个被砍的插入物会让你得到一个燃烧的钱包。我们猜测这是一种不同类型的[反扒手装置](http://hackaday.com/2013/03/08/quick-wallet-hack-adds-pickpocket-alarm/)。[谢谢斯蒂芬]

[Brain]使用 1.50 美元的放大镜帮助他的 [Raspberry Pi 相机模块更好地读取二维码](http://unfocusedbrain.com/site/2013/07/22/corrective-lenses-for-the-raspberry-pi-camera-module/)。

我们真的很喜欢【亚伦克里斯托夫的】 [LED 矩阵时钟](http://atcnetz.blogspot.de/2013/04/groe-led-matrix-uhr.html) ( [译](http://translate.google.com/translate?sl=auto&tl=en&js=n&prev=_t&hl=en&ie=UTF-8&u=http%3A%2F%2Fatcnetz.blogspot.de%2F2013%2F04%2Fgroe-led-matrix-uhr.html))。他从一个至少有十年历史的帐篷开始做起。他把它拆了下来，想出了如何用 Sanguino 作为控制器来驾驶它。
