# 最后，飞跃的实际用途

> 原文：<https://hackaday.com/2013/09/03/finally-a-practical-use-for-the-leap/>

腹腔镜手术中使用的机器人相当普遍，但控制它们远非易事。通常的设置类似于瓦尔多风格的机械手，允许外科医生在房间内进行切割、烧灼和缝合。由于一些新的硬件，还有另一种方法可以实现这一点，正如[Sriranjan] [用他的跳跃控制手术机器人](http://www.youtube.com/watch?v=JLqLmvL75B0)向我们展示的那样。【T2

[Sriranjan]在他的实验中没有使用真正的腹腔镜手术机器人。相反，他使用 Le-Sur 模拟器，将两个虚拟机器人手臂放在训练中的外科医生面前。每个机械臂都有七个自由度，通过使用两个 Leap 控制器(虚拟机中各一个)，[Sriranjan]能够用手控制它们。

我们已经看到了许多 Leap 传感器的创造性应用，如[控制四轴飞行器](http://hackaday.com/2012/11/18/controlling-a-quadcopter-with-a-leap-motion/)、[控制六足机器人](http://hackaday.com/2013/08/06/leap-motion-controls-hexapod-with-hand-signals/)、[控制更多四轴飞行器](http://hackaday.com/2013/08/22/crazyflie-control-with-leap-and-kinect/)，但这是我们第一次看到 Leap 做了其他控制器做不到的事情——模仿外科医生的手的微妙触摸
