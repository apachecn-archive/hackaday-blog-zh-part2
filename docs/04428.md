# 令人印象深刻的激光竖琴

> 原文:[https://hackaday.com/2013/10/30/impressive-laser-harp/](https://hackaday.com/2013/10/30/impressive-laser-harp/)

![laser harp](../Images/56472853ef1ee1d902e50af4850f4a6b.png)

在深夜随意浏览 YouTube 视频后，我们偶然发现了这个令人印象深刻的激光竖琴装置。除了做一把很棒的激光竖琴，[艾瑞克]甚至[也会弹！](http://www.youtube.com/watch?v=5d8P-k6MNIQ)

如果你在享受美食之前从未见过这些！激光竖琴是一种需要合成器来创作音乐的数字乐器。主要有两种，框架式和开放式。框架型使用光束末端的光传感器产生数字信号，转换成各种音调。开放式的要复杂得多，但要酷得多——它依靠从玩家手中反射回来的激光来产生信号。这允许根据到传感器的距离改变音调。

休息后留下来看看它的行动，因为[埃里克]打破它，激光风格。
