# 蜡烛动力风扇使用热电发电机让你保持凉爽

> 原文:[https://hackaday . com/2014/05/21/蜡烛供电的风扇使用热电发电机保持凉爽/](https://hackaday.com/2014/05/21/candle-powered-fan-keeps-you-cool-using-a-thermoelectric-generator/)

这是在项目中使用热电发电机的一个很好的例子。[Joohansson]用电脑上的组件制作了一个实用又美观的风扇。

热电发电机(用于高温的 TEG，用于低温的更便宜的 tec)也被称为 peltier 元件，它看起来像是一小块方形的陶瓷，上面伸出两根电线。如果你给它供电，一边会变热，另一边会变冷。朱汉森使用的 tec 希望两边的温差为 68 摄氏度。它们通常用于冷却电子产品，甚至一些便宜的迷你冰箱也会使用热端有巨大散热器的冰箱。

此外，由于塞贝克效应，它们还可以用作发电机。如果你能在两边制造温差，你就能发电。使用 CPU 散热器、冷却器和风扇，[Joohansson]能够只用一支蜡烛为一个小型直流风扇供电。这是塞贝克效应的精彩演示。
