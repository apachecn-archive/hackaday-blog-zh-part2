# 在红牛创造 2013 上制造麻烦

> 原文：<https://hackaday.com/2013/06/17/causing-trouble-at-redbull-creation-2013/>

[https://www.youtube.com/embed/9BsR_ePFLkk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/9BsR_ePFLkk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

Skullduggery systems(第一次参与)为他们的仪器提出了一个非常大胆的想法。这是一个罐子，它会根据泵入和泵出的水来改变音调。最终，他们遇到了一些问题，这意味着效果不是他们想要的。他们不应该感到太糟糕，一堆团队在他们参与的第一年没有成功地做出东西。

然而，他们的乐器所做的事情，我觉得简直令人愉快。水壶发出的声音就像是音响系统的反馈。比赛在一个名为“北边音乐节”的大型音乐节上举行。大部分时间都有一个舞台和一大群人。他们正在测试他们的水壶，我们注意到舞台上的乐队(当时挤满了人)，已经停止演奏，正在检查他们所有的设备。录音师们像疯了一样跑来跑去。他们认为他们得到了反馈。

我们推迟了一段时间，决定我们将等到声音检查来扰乱他们。虽然我们想制造麻烦，但我们不想毁了整个人群的表演。正如你在视频中看到的，声音非常像反馈。他们会开始玩，然后停下来追踪这个神秘的反馈。我们甚至骗过了在红牛帐篷里通过本地音响系统说话的播音员(我向你保证，这是意外)。
