# 体积电路

> 原文:[https://hackaday.com/2014/09/13/volumetric-circuits/](https://hackaday.com/2014/09/13/volumetric-circuits/)

用少量的铜和焊料建造曼哈顿风格的电路本身就是一项技术，而建造一个原型死虫风格接近于一种黑色艺术。[Anderson]用他的体积电路[将它带到了一个新的水平。他不仅建立了一个自由形式的电路，也是一位 ALU，他还设计了软件，使这种电路易于设计和建立。](http://hackaday.io/project/2396-Volumetric-Circuits!)

[Anderson]称他的 3D 电路设计软件为黄铁矿，它确实如罐头上所说的那样:创造三维的、网格排列的物理电路。自动构建电路不是一件简单的事情，将所有这些元件焊接在一起更是如此。

随着他的软件的第一个原型，[安德森]进入了一个简单的一位算术逻辑单元的示意图。然后用焊料和热胶小心地将最终的布局拼凑起来。它没有工作，但那只是因为原理图是错的。设计软件仍然是一项令人难以置信的成就，现在[Anderson]有了一个自动设计自由形式和死 bug 电路的初步系统，有很多有趣的可能性。有没有想过老式收音机中的点对点布线是否是最有效的布局？[安德森]可能会告诉你。
