# 对 6502 计算机的又一次尝试

> 原文:[https://hack aday . com/2013/02/08/another-take-on-the-6502-computer/](https://hackaday.com/2013/02/08/another-take-on-the-6502-computer/)

![another-take-on-the-6502-computer](../Images/0250af4590dfce3286c56a45cff078f6.png)

[马克]刚刚开始他自己的 6502 电脑旅程。他受到了我们在这里看到的其他一些项目的启发，像[奎因·邓基的][维罗妮卡项目](http://hackaday.com/2012/01/04/backplane-and-mainboard-for-a-6502-computer/)，但利用现代处理器来减轻一些混乱的工作。正如你在上面看到的，在 65C02 处理器上面有一个 Atmel 芯片。该芯片不仅向处理器提供数据(通过所有那些略呈对角线的黄线)，还提供时钟信号，并操作复位和总线使能线。

对[Mark]来说，这更像是一篇问候世界的帖子。芯片现在只是运行 NOP 命令。但它表明基本想法是可行的，休息后的视频让我们看到了电路的另一个节省时间的方面。他用字符液晶显示器来显示存储位置和数据值。他的计划是开一个博客，但他不太愿意这么做，因为这会占用他宝贵的黑客时间。我们不同意。一旦项目开始变得非常棘手，这篇文章(尽管我们读起来非常有趣)最终会成为他的参考手册。
