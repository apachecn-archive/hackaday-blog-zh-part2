# 用于延时目的的旋转设备

> 原文:[https://hack aday . com/2014/09/12/出于延时目的旋转植物/](https://hackaday.com/2014/09/12/rotating-plants-for-time-lapse-purposes/)

在园艺爱好者和业余爱好者中，植物和蔬菜种植过程的自动化是一种日益增长的趋势。这也不足为奇，有了微控制器、湿度传感器、二氧化碳探测器，甚至有了带旋转木架的定时相机，这些相机掌握在全世界数百万业余园艺爱好者手中。

[Liz]的这个项目有助于记录她在芝加哥的公寓里开始茁壮成长的小葡萄柚盆景树的发芽过程。

类似的设备可以用于几乎任何类型的室内植物。它们也可以被改造成根据光照的多少来移动植物和蔬菜。更进一步，只需添加一些代码将照片拼接在一起，您就拥有了一个自定义设置，可以生成动画 GIF 文件并轻松上传到互联网。一页又一页快乐健康成长的植物从地下破土而出，贴满了整个发芽过程的网页。下面是[Liz]的一个视频示例。
