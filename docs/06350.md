# 一个电钻的巧妙再利用产生了一个迷你木头车床

> 原文:[https://hack aday . com/2014/06/19/clever-re-purposing-of-a-power-drill-results-in-a-mini-wood-lathe/](https://hackaday.com/2014/06/19/clever-re-purposing-of-a-power-drill-results-in-a-mini-wood-lathe/)

![Power Drill Wood Lathe](../Images/fdd3c098607f811491b8be3996214df0.png)

用过车床吗？没有吗？[杰克·豪韦林]也没有，但这并没有阻止他建造自己的车并学习如何操作！

车床非常有趣，尤其是对于小型木材加工项目。使用大部分木材和一些小硬件，[杰克]能够在一个下午建造一个非常好的工作！

他正在使用一个便宜的绳索电钻来驱动工件，但我们真正喜欢的是他如何用几根螺纹杆和一个支架来制作骨刺中心和骨刺活中心。这是一个简单的系统，通过拧紧活顶尖的螺纹轴，他可以很容易地固定工件。

在休息后的视频中，他经历了整个构建过程，甚至展示了他第一次使用车床的尝试——他实际上在第三次尝试时能够做出非常好的工具握持！
