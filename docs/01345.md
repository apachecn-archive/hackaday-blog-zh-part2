# Knex 制造的机械滑雪球

> 原文：<https://hackaday.com/2012/08/15/mechanical-skeeball-built-from-knex/>

我们真的不知道该说什么。这个 [Skeeball 橱柜完全由 Knex](http://www.instructables.com/id/Knex-Skeeball-Machine/) 制成。它的工作方式完全符合你对 Skeeball 的预期。你插上 25 美分硬币，它就会分发球并记分。

[Shadowman39]在这个版本上工作了一年多。除了一些橡皮筋和记分牌上使用的纸数字，所有的东西都是 Knex 部件。有六个马达驱动这台机器。其中四个负责转动记分牌上的数字，其他人处理球的返回。

顶部的链接以一堆不同部分的图像开始，但你会想在休息后观看视频以进行更仔细的观察。它更详细地显示了硬币漏斗。它只接受 25 美分硬币，拒绝所有其他硬币。
