# 德尔塔咖啡冲泡机器人

> 原文：<https://hackaday.com/2014/10/09/delta-coffee-brewing-bot/>

倒冲式咖啡冲泡是使用自动咖啡机的一种简单而便宜的替代方法。虽然咖啡爱好者经常选择倒酒只是为了手工冲泡体验，但这并没有阻止[Elias]用一个精心制作的 delta 机器人自动化他的倒酒咖啡设置。

咖啡冲泡机器人是围绕由 BeagleBone Black 控制的 3d 打印机的 delta 组件构建的。BeagleBone 驱动步进电机，在小型开源硬件 [HDMI LCD 显示器](http://hackaday.com/2013/07/08/a-very-small-hdmi-display/)上显示信息，并提供网络界面来控制机器。浇注模式的半径、水温和*去毛刺*的研磨粗糙度都可以通过其网络界面定制。

对于那些想要建造自己的倾倒机器人的人来说，[Elias]已经绘制了设计草图，并且他的所有 Python 源代码[都可以作为开源](https://bitbucket.org/intelligentagent/debrew/overview)获得。休息之后，请观看视频，其中[Elias]解释了他的 delta 倾倒机器人是如何工作的。
