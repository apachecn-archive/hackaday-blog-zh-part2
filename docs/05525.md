# 几乎免费制作高清投影仪！

> 原文:[https://hackaday . com/2014/03/13/make-an-HD-projector-for-next-to-nothing/](https://hackaday.com/2014/03/13/make-an-hd-projector-for-next-to-nothing/)

需要高清投影仪吗？没有 1000 块闲钱？那么这个黑客是给你的！[TheJinxster]组装了这台[超棒的高清投影仪，](https://www.youtube.com/watch?v=BDA6Kfy_lKo)而且他完全没有花一分钱就造出来了！

他从从 Freecycle 上拿起一台旧的高射投影仪开始。Freecycle 是一个很棒的草根网络，有点像 Kijiji 或者 Craigslist——但是**一切都是免费的。**这一切都是为了重复利用，让有用的东西远离垃圾填埋场。无论如何，他也有一个旧的液晶显示器坐在周围积灰…

他把液晶显示器拆开，把光秃秃的屏幕放在玻璃上，嗯，就是这样！就这么简单。最困难的部分是拆开显示器——高射投影仪和固定镜头负责剩下的部分。像这样的系统的优点是，你可以从相对较小和便宜的液晶面板获得高清晰度！唯一的问题是它不会永远持续下去——液晶显示器(尤其是显示器液晶显示器)的设计不能透过那么多光。
