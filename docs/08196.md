# 包装小小的 SDR

> 原文:[https://hackaday.com/2015/01/30/casing-up-the-teensy-sdr/](https://hackaday.com/2015/01/30/casing-up-the-teensy-sdr/)

[Rich，VE3MKC]在基于 Teensy 的软件定义无线电(SDR)方面取得了很大进展。他的最新更新[展示了机箱中的硬件](http://rheslip.blogspot.ca/2015/01/teesy-sdr-project-update.html)和一些新功能。

当[我们在去年 4 月](http://hackaday.com/2014/04/25/building-a-software-defined-radio-with-a-teensy/)看到这一点时，它几乎是一个概念验证，组件从跳线上松散地悬挂下来。新的机箱将所有东西都安装在一个塑料哈蒙德外壳中，前后面板都包有铜。SoftRock SDR 单元被从其外壳中拉出，并加装了连接器，使其可与其他单元交换。

一点点帮助可以走很长的路，[Rich]感谢他的朋友[Loftur，VE2LJX]贡献了许多代码改进和功能添加，这些内容可以在库中查看[。看看下面展示这些功能的视频。](https://github.com/rheslip/Teensy_SDR)

在当前状态下，收音机在 12V 接收模式下消耗 80 mA。它还没有传输，但我们会继续关注另一个更新。[Rich]计划填充输入电路，然后编写传输代码。
