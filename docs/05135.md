# 参观 Deezmaker，世界上第一家*好* 3D 打印机商店

> 原文：<https://hackaday.com/2014/01/23/touring-deezmaker-the-first-good-3d-printer-store-in-the-world/>

本周早些时候，当我们访问 Crash Space hackerspace 时，我们注意到我们距离 T2 Deezmaker T3 只有几个街区，这是一家 3D 打印机商店，位于美丽的帕萨迪纳，是 Bukobot 和 Bukito 打印机的大本营，有许多令人敬畏的打印塑料制品，以及 T4 RepRap Wallace T5 和 T6 T7 的创造者 Rich。显然，一次旅行是必要的。

Deezmaker 内部是一个印刷小玩意的宝库和一个装满人字形行星齿轮系统的鱼缸，可以免费取用。他们有一直运转的打印机，一个非常好的实验室，足够每周举办几次研讨会的工作空间。

在下面的视频中，[Diego]，Deezmaker 的大奶酪带我们逛商店，展示他的商品。[whosawhatsis]还展示了他的最新发明 Bukito 打印机。这是一个非常小的难以置信的便携式打印机，可以由电池供电。他们在 Bukito 露营时使用的是 3 芯 5000 毫安时的锂电池。我不知道电池能持续多长时间，但足够喷出一些他们赠送的齿轮系统了。
