# 将三星 Galaxy Note 10.1 变成一台合适的 Linux 机器

> 原文：<https://hackaday.com/2013/03/05/turning-the-samsung-galaxy-note-10-1-into-a-proper-linux-box/>

在 xda 开发者论坛上，[exception13]向我们展示了他为让 Debian 在他的三星 Galaxy Note 10.1 上运行所做的工作，这使他能够在一台设备上双重启动 Android 和 Linux。

该项目仍然处于相当早期的状态，但是到目前为止[exception13]已经具备了运行一个体面的 Linux 体验所需的大部分好东西。有 WiFi、蓝牙、声音、usb-otg 和触摸屏支持，以及对 Note 的笔的支持，Wacom 数字化仪基本上将 Galaxy Note 10.1 变成了 Intuos 触摸板。

还有很多工作要做，包括启动和运行摄像机，以及启用 GPS 接收器。尽管如此，这仍然是一个非常酷的项目，它将适当的桌面界面的力量融入到平板电脑中，有足够的马力来完成一些有用的事情。

如果你想让这个在你的 Galaxy Note 上运行，[exception13]可以在 Google Code 上下载。还有一个视频[exception13]展示了他的笔记能做的所有很酷的事情，你可以在休息后看看。
