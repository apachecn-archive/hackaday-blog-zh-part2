# 3D 打印 HOG 驱动器

> 原文:[https://hackaday.com/2013/02/28/3d-printed-hog-drive/](https://hackaday.com/2013/02/28/3d-printed-hog-drive/)

![3d-printed-hog-drive](../Images/edb618921a33f9c8e65d9b75a900ee13.png)

这里有一个 3D 打印的半球形全向万向驱动系统，你可以在家里制作。这是一个拗口的说法，这就是为什么它通常被称为猪驱动器。没听说过吗？你需要跟上你的黑客时代，因为大约 20 个月前我们报道了[这个神奇的机器人项目](http://hackaday.com/2011/07/09/amazing-hemispherical-omnidirectional-gimbaled-wheel-robot/)，它使用了一个。该设计是一个三轮车的方向与猪驱动器作为唯一的动力“车轮”。但它并不是一个真正的轮子，而是一个半球(上图中没有这个半球，但它附着在电机主轴上)，可以根据电机的旋转方向和万向支架的方向提供任何方向的推力。

不幸的是[丹]还没有展示由该设备驱动的车辆。但是从我们在跳跃后的演示中看到的来看，它是完全起作用的。他的系统目标项目是一个循线机器人，我们希望当他达到这个目标时，可以发布一个后续项目。
