# 最简单的蒸汽机

> 原文：<https://hackaday.com/2015/07/28/the-simplest-steam-engine/>

[RimstarOrg]发布了一篇关于他的[英雄的蒸汽机](http://rimstar.org/science_electronics_projects/heros_steam_engine_soda_can.htm)的精彩评论。Hero 的发动机是一个来自第一世纪的希腊设计，是已知最早的蒸汽机。令人惊讶的是，他在工业革命前 17 个世纪就开发出了发动机，然而它却在很大程度上被忽视了。虽然你可以找到更多的[忠实的复制品](https://www.youtube.com/watch?v=IXf7jcHn0Hg)，这个里程碑式的机器【RimstarOrg】的钻机不能被击败的简单，他做了一个伟大的工作，解释操作和建设的原则。

使用装满水的汽水罐和丙烷火炬[RimstarOrg]能够通过从罐子侧面的两个孔喷射蒸汽来使罐子快速旋转。打捞转环用于提供枢转接头，并允许罐自由旋转。

虽然我们之前已经介绍过[蒸汽机](http://hackaday.com/2013/03/20/second-grade-science-project-a-steam-engine/)，但我们喜欢这个简单的设计，迫不及待地想看看【RimStarOrg】接下来会推出什么。
