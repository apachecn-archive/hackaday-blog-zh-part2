# 黑客空间简介:新在上海

[https://www.youtube.com/embed/SGx5hm4jwNQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/SGx5hm4jwNQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)
【保罗】在向上海[新车健黑客空间](http://www.xinchejian.com/)提交此视频简介时写道。似乎他们在那里有一个相当活跃的空间，有几个正在进行的项目。他们展示了从智能无线电源板到水族箱的一系列东西。这个空间似乎有一些常见的便利设施，比如项目仓库和一个机械加工车间。有人解释说一些工具“几乎可以工作了”，这种熟悉的声音提醒我们，这是一个小世界，我们都差不多。
