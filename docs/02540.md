# 逆行追踪:差异

> 原文：<https://hackaday.com/2013/01/15/retrotechtacular-the-differential/>

任何以两分钟的摩托车编队骑行开始的视频都必须是好的。如果粒状黑白视频没有提示你这是在一个不同的时间，站在移动的摩托车座位上的无头盔骑手肯定会。但是这种阐述是有目的的。一列并排骑着的摩托车绕着一个弯道行驶，说明了[为什么差速器是必要的，很快你就会知道差速器是如何工作的](http://www.youtube.com/watch?v=yYAw79386WI)。

当车辆转弯时，安装在一个轴上的两个车轮需要以不同的速度转动，或者其中一个车轮必须滑动以适应速度差。差速器是必要的，以允许这些不同的转速，同时仍然让两个车轮连接到动力传动系。我们惊讶地从休息后的视频中得知，早期的汽车通过只给四个轮子中的一个提供动力来解决这个问题。

这个教学视频是对我们在 Retrotechtacular 的上一期中看到的[流体耦合视频的完美补充。](http://hackaday.com/2012/12/31/retrotechtacular-fluid-coupling/)
