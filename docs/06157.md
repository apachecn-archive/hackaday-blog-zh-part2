# Lightgame 项目:多人 Arduino 游戏

> 原文:[https://hack aday . com/2014/05/28/the-light game-project-a-multiplayer-arduino-game/](https://hackaday.com/2014/05/28/the-lightgame-project-a-multiplayer-arduino-game/)

夏天即将来临。光明游戏项目是一款基于 Arduino 的多人反应时间游戏。对于那些不安分的孩子(和成年人)来说，这是一个完美的雨天项目！).由两个本科生[Efstathios]和[Thodoris]为一个学期的项目设计的，所有的艰苦工作已经为你做了。

我们喜欢[游戏](http://hackaday.com/2014/02/27/gamebuino-a-handheld-arduino-gaming-console-ready-for-prime-time/)的原因有很多，你可以[打造自己](http://hackaday.com/2014/03/01/the-credit-card-sized-gameboy/)。首先，这是一个让孩子们对爱好电子产品、制作和黑客感兴趣的惊人方法。尤其是当他们可以和朋友一起玩游戏(并向他们炫耀)的时候。另一个原因是，这是一个与朋友和家人分享你的项目的完美方式，展示你一直在学习的东西。这个游戏是基于你的反应时间和当另一个玩家的颜色出现时你是否按下按钮。该项目围绕两个通过 I2C 连接[的 Arduinos 构建。主机处理游戏的机制，而从机处理](http://arduino.cc/en/Tutorial/MasterWriter) [TFT LCD](http://arduino.cc/en/Main/GTFT) 并通过蜂鸣器播放音乐。

I2C 是一个很好的通信协议，这是一个很好的尝试项目。[Efstathios]和[Thodoris]在撰写他们的帖子方面做得非常好，另外他们还包括了构建自己的帖子所需的所有代码和原理图。如果能看到更多的大学教授和他们的学生一起培养开源硬件和软件，那就太好了。特别感谢 Dasygenis 博士]向我们提交了他学生的作品！
