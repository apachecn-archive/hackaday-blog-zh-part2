# 黑掉你自己的模拟摄像机

> 原文：<https://hackaday.com/2015/08/20/hack-your-own-analog-camera/>

我们记得小时候用雪茄盒制作针孔摄像机。焦点相机网站想让你用精选的积木制作复杂的相机。我们说的是带胶卷的相机，而不是数码相机(虽然我们想知道你是否可以安装图像传感器…但那是另一种黑客行为)。

这些模块确实需要使用激光切割机，而且你还需要寻找或以其他方式获得镜子和透镜之类的东西。该网站就如何用丙烯酸镜等廉价物品制作第一面镜子给出了建议。

目的是能够从模块中建立你自己的照相机。他们有一个针孔相机，以防你怀旧，但你也可以建立单反相机，大画幅相机，甚至立体相机。并不是所有的模块都准备好了，但是网站上有几个示例相机和用它们拍摄的照片。像大多数积木一样，真正的乐趣在于用户开始以意想不到的方式组合它们。

以前也有制作自己的单反相机套件(见视频)，但这些都是组装一个单一的相机设计，而不是一个模块化的积木系统来开始你自己的设计。虽然针孔相机可以做得令人惊讶的好，但建造一个真正复杂的相机的前景在几个层面上很有吸引力。
