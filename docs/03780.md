# 从头开始做弓

> 原文：<https://hackaday.com/2013/06/06/making-a-bow-from-scratch/>

![bow](img/56cc30d6bb8b358669e2607f5e1dc765.png)

有了 Hackaday 新的手工制作类别，我们可以选择覆盖广泛的制作——从纸上设计和手工机器制作的喷气式发动机到用最原始的工具制作的旧世界工艺品。这一次，我们将看到[从零开始制作长弓](http://www.youtube.com/watch?v=6eWDgNrLYps&feature=youtu.be)，这是【比利·伯杰】的作品，这个项目涵盖了从选择一棵树到制作一把弓以制造最好的武器的所有事情。

受欧洲启发的长弓通常由紫杉制成，但在[比利]的家乡东德克萨斯，紫杉有点难找。他最终选择了一棵小奥萨吉橙树作为他的弓，剥去树皮，劈开原木，开始制作他的手工弓。

制作弓最重要的部分是确保弓的背面只有一个生长环。用一把拉刀，[比利]小心翼翼地刨平弓的背面，这样只有一个树的年轮可见，然后开始塑造弓的腹部和侧面。

木材是一种天然材料，新切割的木材含有大量水分。当[比利]在做弓的时候，一些湿气离开了他的木材，导致木材有些扭曲。有一个人类已经做了几千年的解决办法——火弯曲木头。通过在木头上涂上某种动物脂肪([Billy]用过的橄榄油)，你可以把一块木头放在一个小框架上而不会烧焦。用一棵树的弯曲作为虎钳，[比利]扭曲了木头，给了他一个完美的直弓。

这把弓的制作量惊人，考虑到[比利]只用手工工具和原始的木工方法，这并不奇怪。尽管如此，完成的弓仍然是一件艺术品，是工艺的杰作。你可以在下面查看[Billy]演示的所有四个部分。
