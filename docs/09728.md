# Hackaday 奖参赛作品:照片和 Arduinos，生活在一起的猫和狗

> 原文：<https://hackaday.com/2015/08/06/hackaday-prize-entry-pics-and-arduinos-cats-and-dogs-living-together/>

我们在互联网上的小角落有一半抱怨 Arduino，Arduino 标准的 pin 标头如何没有任何意义，Arduino IDE 如何是垃圾，给出了为什么 Arduino 语言阻碍下一代嵌入式程序员的充分理由，并哀叹一切都商品化到 Arduino 兼容包中的事实。我们互联网小角落的另一半使用微芯片图片。

[Jarrett]很固执，他想使用具有独特 Arduino 引脚布局的 PIC。[由此诞生了 PIC-On-Go](http://hackaday.com/2014/08/22/fear-and-loathing-at-defcon-2014/)。这是一款 PIC18F4520，采用熟悉的高飞引脚封装，专为那些只想认真工作的人制作。

这不是唯一的 PIC-before-Arduino 板；Fubarino 是一款支持 Arduino 的出色主板，但它并没有利用我们最喜欢的 Arduino 屏蔽。不管怎样，我们都很惊讶像[Jarrett]的项目还不存在，这使得它成为 Hackaday 奖的一个很好的参赛作品。
