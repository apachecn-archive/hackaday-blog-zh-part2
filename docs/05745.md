# VCF 东部:旧电脑，新游戏

> 原文:[https://hack aday . com/2014/04/08/vcf-east-old-computers-new-games/](https://hackaday.com/2014/04/08/vcf-east-old-computers-new-games/)

虽然新泽西州沃尔的复古计算机节几乎拥有你能想象到的所有复古应用程序——多种多样的*NIXes、前 Zork 地下城、所有功能的基本解释器，以及几乎所有为 8 位 Commodore 系统发布的游戏——但显然缺乏为这些复古系统编写的现代程序，这一点或许并不令人惊讶。是的，尽管仍有人蜷缩在键盘前为老式系统编写游戏，但现代软件在上周末却是一个奇怪的异类。

然而，有两个奇妙的例外。第一个是 Fahrfall，一个为 TRS-80 彩色电脑设计的游戏。[我们在](http://hackaday.com/2012/02/02/creating-a-game-for-the-coco/)之前已经看过 Fahrfall，当时【约翰·林维尔】为 2012 年的反挑战冬季热身写的。游戏本身是对 Atari Jaguar 垮台的重新想象，图形被大幅缩小。这个游戏的基本思想是在垂直滚动的屏幕上从一个窗台跳到另一个窗台。撞到墙壁或底部，你就死定了。这是一个伟大的游戏，如果它是当代发行，可能会卖得很好。

接下来是 TI-99 相当令人印象深刻的 Flappy Bird 港。该视频没有做到这一点游戏公正，虽然其中一部分可能只是可怕的 Amiga 显示器用于显示。这款游戏是由 Vintage Volts 公司的杰夫·萨尔兹曼带来的，他并不是这款游戏的作者。老实说，这个视频没有把图像做得很好。这真的是一个很棒的端口，就像 Android/iDevice 原版一样令人上瘾。
