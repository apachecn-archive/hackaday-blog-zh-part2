# 价值 100 万美元的示波器拆卸

> 原文：<https://hackaday.com/2015/02/10/the-one-million-dollar-scope-teardown/>

[Labmaster 10-100zi 示波器](http://teledynelecroy.com/100ghz/)是世界上最快的示波器之一，速度高达 100GHz，每秒实时采样高达 240。示波器由 Teledyne LeCroy 制造，并使用由 LeCroy 完善的频率交错技术，这允许它提供单个 100GHz 信道，或两个 33GHz 信道和单个 65GHz 信道。价格标签？一百万美元。

【Shahriar】[带我们进入](http://thesignalpath.com/blogs/2015/02/10/experiments-and-teardown-of-the-teledyne-lecroy-labmaster-10-100zi-100ghz-240gss-oscilloscope/)纽约栗岭的 Teledyne Lecroy 工厂，这些瞄准镜就是在这里制造的，并带我们参观一下。首先，一名工程师描述了交错频率技术，这种技术可以实现闪电般的采样速率。然后他们真的把价值百万美元的瞄准镜拆下来给我们看。如果你还想要更多，他们把它放回一起，并运行一些测试，以推动范围达到其深远的极限。最后，[Shahriar]带我们参观了制造望远镜的工厂。

这是一个冗长的视频，所以拿起你最喜欢的饮料大吃一顿吧！技术进步之快令人震惊。就在大约 18 个月前，[Shahriar] [带我们参观了当时的示波器冠军](http://thesignalpath.com/blogs/2013/08/13/experiments-and-demo-of-an-agilent-dsa-x-96204q-160gss-62ghz-oscilloscope/)Agilent DSA-X 96204 q，它在 62GHz 下获得了 160GS/s 的速度。
