# [Fran]PCB 蚀刻技术

> 原文:[https://hack aday . com/2013/03/04/Frans-PCB-etching-techniques/](https://hackaday.com/2013/03/04/frans-pcb-etching-techniques/)

![frantronics-pcb-tutorials](../Images/7ca071fd42b5cb0194f52a398f6c5507.png)

我们认为任何在家制作 PCB 的人都会喜欢[Fran]在蚀刻过程中保持的整洁。她最近发布了一个由三部分组成的视频教程，展示了她的技巧。正如你在上面的截图中看到的，她的习惯散发着顶尖实验室技能的味道。

普通读者大概能猜到她在蚀刻什么电路。这是她的 LVDC 逆向工程的测试板。她使用的是墨粉转移法，但与大多数家用蚀刻机略有不同。她使用为这项工作制作的蓝色转印纸，但在将它转移到覆铜之前，她使用一个灯箱(有点像医生办公室的 x 光胶片浏览器)来检查碳粉未粘附的任何间隙。在那里，她用热压机涂上保护层。这比用熨斗要简单得多，但是当然你手头必须有一个这样的东西。

教程的第二部分在休息后嵌入。我们选择这一部分是因为它展示了[Fran]如何建造她自己的化学罩。这是一个倒置的透明塑料储物容器。一个工作窗口已经削减了前侧，一个 4 英寸的排气软管添加到顶部。[Fran 的]实验室有一个大容量低速风扇，它连接到风扇，将烟雾抽到外面。
