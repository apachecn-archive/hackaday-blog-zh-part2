# 用光学录音照亮有声电影的道路

> 原文：<https://hackaday.com/2014/03/04/retrotechtacular-lighting-the-way-for-talkie-pictures-with-optical-sound-recording/>

本周的回顾片是一部 1943 年的大英百科全书电影，聚焦于电影的光学声音复制。声音和图像都记录在胶片上，胶片只受光的影响。因此，声波必须转换成光的变化。

这是以你可能期望的方式完成的:声波击中麦克风，电流的变化被放大，并用于控制落在薄膜上的光的强度。影片结尾描述并精彩演示了三种类型的配乐。

这三种类型都是由一系列细光柱组成，相应的电流值由它们的长度或宽度的变化来表示。在单侧可变区域录音中，小节从音轨的右侧延伸。双侧可变区域记录条从中心向边缘均匀发散。在可变密度录音中，所有的条从左向右延伸，但是它们的厚度不同。

可变密度记录是通过一个[光阀](http://www3.alcatel-lucent.com/bstj/vol08-1929/articles/bstj8-1-173.pdf)完成的，光阀包含一对位于磁场中的精致金属带，当声音电流流过它们时，它们像百叶窗一样移动。透过胶片的光因色带之间的缝隙而变化。光模式通过光电管变回声音，光电管将光的变化转换回变化的电流。这些变化被放大并通过扬声器传播。请务必观看到最后，观看录制方法的演示，设置为我们非常确定的卡米尔·圣桑的 *Danse Macabre* 。
