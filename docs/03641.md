# 苹果 II 上的动画 gif

> 原文：<https://hackaday.com/2013/07/22/animated-gifs-on-an-apple-ii/>

在互联网出现之前，计算机爱好者需要以某种方式获得他们的猫图片、图像宏和动画 gif。要是[Nate]在 80 年代写代码就好了:他为 Apple II 创造了一个视频播放器，本质上是将经典电脑变成了一台可以播放一两张 gif 动画的机器。

老式微型计算机并不以内存容量大或速度快而闻名，所以[Nate]需要引入一些额外的硬件，给他最近购买的 Apple II+64k 内存磁盘来存储 gif。

gif 在 PC 上用 Python 脚本转换后从软盘驱动器上下载，分辨率和颜色降低到 280 x 192 像素，呈现出令人惊叹的四色彩虹。对于下面看到的一些 gif 图，Apple II 能够完成这一技巧确实令人印象深刻。令人惊讶的是[内特]也能让这东西工作。

如果你有一个苹果 II 设置，你真棒。你应该去仔细阅读一下[【Nate】的 git](https://github.com/nategri/apple2/tree/master/animation_2color) ，为你的经典电脑制作你自己的动画 gif。
