# 你的客厅成为下一个马里奥赛车课程

> 原文：<https://hackaday.com/2014/10/04/your-living-room-becomes-next-mario-kart-course/>

[Ken]喜欢他的客厅，他一直致力于让它变得更有趣。最近，他在一个被他称为 [RomoCart](http://kawalabo.blogspot.jp/2014/10/romocart-hack-to-turn-your-living-room.html) 的赛车游戏项目上取得了巨大的进步。把它想象成一个部分物理的马里奥赛车游戏。你可以在赛道上和其他人比赛，同时仍然有能力发射射弹或放弃防御措施来赢得比赛！

首先，让我们谈谈所需的硬件。参赛者是标准的罗莫教育机器人。无线游戏控制器为驾驶员提供了控制 Romos 的手段。天花板上悬挂着一台 [Xtion](http://www.asus.com/Multimedia/Xtion_PRO/) 动作感应摄像机和一台视频投影仪，两者都指向地面。

首先，该系统扫描地板，并根据房间布局和附近的任何物理对象确定比赛路线。然后生成一条路线以避开障碍物，并投射到地板上。在这一点上，它仍将是一个非常整洁的项目，但[肯]走得更远。安装在天花板上的摄像机跟踪 Romos 赛车在赛道上的运动，视频投影仪显示每个赛车后面的烟雾轨迹。赛道上随机显示的是帮助你赢得比赛的项目，包括一个加速项目，让你的罗莫在短时间内快两倍。

有尾随者吗？没问题，只要捡起一些香蕉，把它们丢在轨道上。如果后面的竞争对手撞上了，他们就会失控。如果你想变得超级粗鲁，拿起一些导弹，向你前面的参赛者开火。直接命中会让他们立刻停下来。

[Ken]对 HaD 并不陌生，他以前在这里介绍过他的一些项目。看看他的[测温镜](http://hackaday.com/2012/11/15/an-extemely-unique-weather-display/)、[移动窗](http://hackaday.com/2013/08/27/a-facelift-for-the-view-out-your-livingroom-window/)和他的[自主照明系统](http://hackaday.com/2014/02/01/autonomous-lighting-with-intelligence/)。
