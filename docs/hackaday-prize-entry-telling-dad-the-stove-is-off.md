# Hackaday 奖参赛作品:告诉爸爸炉子已经关了

> 原文：<https://hackaday.com/2015/05/26/hackaday-prize-entry-telling-dad-the-stove-is-off/>

一个月前，Hackaday 登陆了纽约市 TechCrunch Disrupt，这是一个人们通宵使用 MacBooks 和大量咖啡因和维生素 B12 的堡垒。20 个小时里，每个人都在打字，试图打造下一个会被谷歌、亚马逊或脸书收购的伟大服务。Hackaday 团队藏在房间的一个小角落里，向 Disrupt 的几十名硬件黑客分发开发板、组件和建议。这些 Hackaday 爱好者之一[在 Disrupt](https://hackaday.io/project/4688/logs) 获得了 Twilio 赞助奖，现在[成为了 Hackaday 奖的参赛作品](https://hackaday.io/project/4688-an-iot-device-that-tells-dad-the-stove-is-off)。

[大卫]的爸爸有一点点不小心忘了关炉子的妄想症。这通常在离开房子几分钟后表现出来，这意味着要把车转过来，以确保炉子是关着的。在 TechCrunch 黑客马拉松上，[David]建造了一个小型物联网设备来自动读取炉子的温度，并将其发送到互联网上，最后通过 Twilio 以短信形式发送。

[David]使用的硬件非常少——一个热电堆、一个气体传感器、一个 WiFi 模块和一个微控制器。[这个项目](https://hackaday.io/project/4688-an-iot-device-that-tells-dad-the-stove-is-off/log/17599-how-fast-can-i-iterate)有很多迭代，从 TI MSP430s 到 Teensys 到 Arduinos 再到 ESP8266 模块，【David】都在考虑。尽管如此，在 20 小时内拼凑出粗略的原型是赢得 Disrupt 的 Twilio 奖所需要的一切，这对于一个非常好的 Hackaday 奖参赛作品来说绰绰有余。
