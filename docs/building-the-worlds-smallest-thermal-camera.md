# 建造世界上最小的热感相机

> 原文：<https://hackaday.com/2014/09/07/building-the-worlds-smallest-thermal-camera/>

electricstuff.co.uk 背后的策划者[Mike Harrison]刚刚完成了热成像相机中发现的轻子模块的逆向工程——然后他创造了自己的，可能是世界上最小的热成像相机。

他拆开了 Flir One iPhone 热成像单元，取出了让这一切成为可能的神奇部分——轻子模块。它只有 80×60 像素的分辨率，但在热成像领域，这已经相当不错了。你可以直接从 Flir 以 250 美元的价格购买 1000 个。

他的博客有关于如何与模块接口的所有细节，这真的很令人印象深刻。一旦他完全理解了这一点，他就着手把它做成一个小型热感相机。外壳由黑色丙烯酸树脂加工而成，iPod nano 屏幕用作显示屏，80×60 可以很好地放大到 iPod 的 320×240 分辨率。一个自制的 PCB 连接到模块，有一个电压调节器和锂离子电池的充电电路——然后连接到一个原型 iPod nano PCB，其中一些功能被删除——他说连接这一切是一个噩梦，我们不怪他，这是一些严重的黑客技术！
