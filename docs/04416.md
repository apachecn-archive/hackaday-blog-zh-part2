# 微型切片:微型 Arduino 激光切割机

> 原文:[https://hack aday . com/2013/10/27/microslice-the-tiny-arduino-laser-cutter/](https://hackaday.com/2013/10/27/microslice-the-tiny-arduino-laser-cutter/)

[SilverJimmy]已经有了一个全尺寸的 50 瓦激光切割机，但他决定试着组装一些更小的微控制器驱动的东西。结果就是这个[可爱的小雕刻师:微型切片](http://www.instructables.com/id/MicroSlice-A-tiny-Arduino-laser-cutter/)。

为了保持设计简单，[SilverJimmy]选择了固定切割台，这意味着切割头和 X 轴作为一个单元沿 Y 轴移动。解决方案是从[龙门起重机](https://en.wikipedia.org/wiki/Gantry_crane)中获取灵感。他找到了几个带螺纹轴的步进电机，在 [Inkscape](http://inkscape.org/) 中设计了零件，然后启动他的全尺寸切割机切割出零件。一个 Arduino Uno 和用于激光和风扇的继电器位于 MicroSlice 的底部平台上，两个 EasyDriver 电机控制器位于它们上方的下一层。

浏览 Instructables 以获得包括源代码在内的更多细节，并查看下面的雕刻师视频。[SilverJimmy]的激光来自易贝，但看看今年早些时候使用 DVD 二极管的[雕刻机。](http://hackaday.com/2013/05/06/dvd-laser-diode-used-to-build-a-laser-engraver/)
