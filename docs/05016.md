# 一台耀眼的拉森扫描仪

> 原文:[https://hack aday . com/2014/01/08/a-blindly-bright-Larson-scanner/](https://hackaday.com/2014/01/08/a-blindingly-bright-larson-scanner/)

![craylarsonscanner](../Images/fdcd851fc561a78a3f278ce98ce04f5a.png)

这个巨大的照明纪念碑是朱尔斯的 6 英尺长的终极拉森扫描仪。如果你在困惑中挠头，拉森扫描仪是在凯特(汽车)上看到的永远受欢迎的滚动 led[，来自*霹雳游侠*和](http://en.wikipedia.org/wiki/KITT)[在*太空堡垒卡拉狄加* (1978)中的赛昂](http://en.wikipedia.org/wiki/Cylon_(1978))，以系列的创作者[命名。](http://en.wikipedia.org/wiki/Glen_A._Larson)

[DJJules 的]迭代由 16 个 10W 的 led 组成，每个 led 安装在一个散热器上，该散热器固定在一个 6 英尺长的角形铝片上，该铝片来自当地的五金店。他为每个 LED 使用一个基本的 MOSFET 恒流驱动器，用 4-40 个螺钉将 MOSFET 连接到散热器上。每个 LED 模块然后连接到来自 [Sparkfun](http://www.sparkfun.com/products/10616) 的 TLC5940 LED 驱动器分线板，该分线板插入 Arduino Pro Mini。

查看 Instructables 页面，了解源代码和其他重要细节，包括安全地为建筑供电(每个 led 可以在 1A 周围画出一圈)，然后带上你的太阳镜，留下来观看下面的快速视频。
