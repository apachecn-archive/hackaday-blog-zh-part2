# 磁力螺旋桨平衡器消除震动

> 原文:[https://hack aday . com/2014/03/20/磁力推进器平衡器带走震动/](https://hackaday.com/2014/03/20/magnetic-propeller-balancer-takes-away-the-shakes/)

![prop-balancer](../Images/dcc9a4c276a8d0eb84bec7a6cc255ee0.png)

朋友们可能会说他精神不正常，但他的螺旋桨总是准确无误。他平衡良好的道具要归功于这个[磁性平衡器，它是他用废弃的个人电脑零件和偷来的办公用品](http://www.instructables.com/id/Magnetic-propeller-balancer/)制成的。螺旋桨平衡对于无线电控制的模型来说一直很重要。带着一个平衡不好的螺旋桨飞行几个小时会对飞机和里面的无线电设备造成严重破坏。随着今天的无人机和遥控飞行器携带相机、加速度计和陀螺仪，螺旋桨平衡变得更加重要。振动的螺旋桨最多能毁掉一张照片。在最坏的情况下，它会干扰陀螺仪和加速度计的读数，导致撞车。

[danije l0 frk]平衡器基于几种商业设计。CD-ROM 驱动器的钢轴被切割，两端被打磨成点。待平衡的螺旋桨安装在轴上。螺旋桨并不都有相同的孔大小，所以两个笔尖压入服务作为自定心安装。[Danijel0frk 的]图片显示一个尖端指向螺旋桨毂，而另一个尖端指向远处。如果两个尖端都指向内，平衡器会工作得更好，确保轴直接位于螺旋桨毂孔的中心。硬盘驱动器磁铁将支柱和轴悬挂在驱动导轨和木架上。我们应该注意到，这种类型的平衡器只会执行一个[静态平衡](http://en.wikipedia.org/wiki/Tire_balance#Static_balance)。动态平衡需要螺旋桨在装有力传感器的马达上旋转。为了执行静态平衡，材料被添加到螺旋桨或从螺旋桨移除，直到它在任何方向上平衡。点击休息时间观看关于平衡的视频教程。
