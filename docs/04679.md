# WiFi 和蓝牙机器人控制

> 原文:[https://hackaday.com/2013/11/29/wifi-bluetooth-bot-control/](https://hackaday.com/2013/11/29/wifi-bluetooth-bot-control/)

![MasterFeatureGraphic1024x500New](../Images/da64a242b9514b7ad8d7e8cd38b74778.png)

正在寻找一种简单的方法来控制你制作的 Arduino 驱动的机器人吗？我们刚刚听说了一对看起来非常方便的 Android 应用程序！ [WiFi 机器人控制](http://www.plastibots.com/?p=2971)和[蓝牙机器人控制。](http://www.plastibots.com/index.php/2013/03/07/btbotcontrol/)

那么，它们是什么？它们是可配置的应用程序，能够接收 IPCam 视频信号，并通过 URL 字符串(或蓝牙信号)发送各种命令，供 Arduino 解释。它标配了一个操纵杆控制协议和 8 个可定制的命令按钮。在机器人方面，你看到的是一个相当基本的设置，包括一个 Arduino，一个 WiFi 模块，一个电机护罩和电机，以及一个 IP 摄像头。剩下的就看你自己了！

它看起来非常容易设置，并且 [lite 版本是免费的](https://play.google.com/store/apps/details?id=appinventor.ai_picoware.WiFiBotControlLite)！那是我们尝试它的足够好的理由。完整版的价格是 1.99 美元，对于你的项目来说，这是一个不错的价格。休息之后，请观看名为 Eddie 的机器人的实际操作。
