# 构建您自己的独立网络电台

> 原文：<https://hackaday.com/2014/10/11/build-your-own-stand-alone-web-radio/>

如果你喜欢通过播客或收听网站来打发空闲时间，但不想总是占用你的笔记本电脑或平板电脑，这款基于 Arduino 的无线电播放器将为音乐提供一个基站。

由[Vassilis Serasidis]设计的网络电台项目以令人愉悦的细节概括了如何将四个模块连接起来。这些包括一个以太网屏蔽，液晶显示屏，MP3 解码器和 USB 串行转换器，以及一个 Arduino Mini，以便标记和播放您最喜欢的 14 个频道。他的手工焊接板将所有东西连接成一个整齐堆叠的封装。[教学视频](http://www.youtube.com/watch?v=g-jGM48xUrs#t=23)展示了这一点，他甚至解释了如何在[internet-radio.com](http://www.internet-radio.com/)上找到你最喜欢的电台，并将它们的端口和 IP 号码直接复制到提供给你使用的示例草图中。如果你一直想为你的办公桌构建一个独立的无线电节点，而没有额外的负担，这对于精通硬件和更注重代码编写的人来说都是一个轻而易举的项目。

如果你打算建造自己的收音机，将你的高科技创造伪装成更质朴的东西总是很酷的。看看[Dominic Buchstaller]的这个项目,这是一个老式收音机的绝佳例子。
