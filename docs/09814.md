# Hackaday 奖参赛作品:第六感

> 原文：<https://hackaday.com/2015/08/13/hackaday-prize-entry-a-sixth-sense/>

环境中还有更多人类感官无法察觉的事情。鸟类在地球磁场的帮助下迁徙，某些种类的鱼类可以探测到电场。对于他的黑客奖参赛作品，[【塞巴斯蒂安】给了我们第六感](https://hackaday.io/project/5648-thesixthsense)。他正在建造一种装置，可以让任何人探测到地球磁场，并在夏天找到去北方的路。

[塞巴斯蒂安]项目的最初想法是在他父亲的内耳受损后产生的。医生告诉他，他的大脑需要被训练成只用一个内耳工作。如果它对平衡起作用，[Sebastian]想知道，为什么大脑不能被训练来听一个额外的感官输入？

[Sebastian]的设备是一个加速度计和磁力计，连接到一个驱动几个振动电机的微控制器。通过在脚踝带上安装这些马达，[Sebastian]希望训练他的大脑倾听磁场。

到目前为止，[Sebastian]有一个设备可以感应地球磁场，并嗡嗡作响最接近磁北的电机。还有很多工作要做，包括过滤磁力计输入，添加“睡眠”模式，以及将蓝牙功能放在板上，但这已经是一个设计非常好的项目。
