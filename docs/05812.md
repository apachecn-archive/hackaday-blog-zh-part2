# Kyub MIDI 键盘将钢琴放在你的口袋里

> 原文:[https://hack aday . com/2014/04/16/kyu b-midi-keyboard-puts-a-piano-in-your-pocket/](https://hackaday.com/2014/04/16/kyub-midi-keyboard-puts-a-piano-in-your-pocket/)

基思·巴克斯特喜欢制作电子仪器。他的最新愿景是成为一款开源 MIDI 键盘的 [Kyub。[Keith]之前曾光临我们的网站，用他的](http://servoelectricguitar.com/kyubmusic.html)[伺服电吉他](http://hackaday.com/2009/11/06/servoelectric-guitar-is-a-keytar-with-strings)破解了 [*科普*](http://www.popsci.com/entertainment-amp-gaming/article/2009-03/homemade-guitar-hero?nopaging=1) 。

[Keith]想做一个完全开源的乐器，优雅、有用，比伺服电吉他更容易使用，所以他和一个黑客/电子音乐专家和一个工业设计师合作。他围绕 Arduino Uno 制造了早期原型。目前的迭代使用 Teensy 2.0，并通过 [Kickstarter](https://www.kickstarter.com/projects/kyub/the-kyub-a-three-dimensional-feather-touch-midi-ke) 以各种形式提供。[Keith]将 Kyub 开放给大众融资，以获得一些器件的批量定价以及 Eagle 许可证，使 PCB 文件可以在商业上使用。

Kyub 在立方体的五个面上有 11 个压敏电容键盘。加速度计可以用来改变音符音量，弯曲音高，或者你编程让它做的任何事情。当然，你需要一台装有合成器程序的电脑，但是[Keith]说它可以兼容大多数软件合成器程序，其中一些是免费的。

休息后有一个早期原型的演示视频。Kickstarter 页面上提供了当前形式的 Kyub 视频。
