# 家用微型机器人，谁需要 DARPA 资助？

> 原文:[https://hack aday . com/2014/06/07/微型机器人-家庭制造-谁需要-DARPA-资助/](https://hackaday.com/2014/06/07/micro-robots-made-at-home-who-needs-darpa-funding/)

我们喜欢这种情况发生。有时，我们会发布一些公司正在开发的很酷的技术，这可能会引起“不是黑客”的强烈抗议，但有时，它仍然会激发一些读者。[耶鲁安·多姆伯格]——每个人都知道他是[Sprite_TM]，[是 Hackaday Prize]](http://hackaday.io/prize/judges#Sprite_TM) 的评委——看到了最近一个关于微型机器人的视频，这是一个由 DARPA 资助的项目，他决定尝试制作自己的 **[。](http://spritesmods.com/?art=magnmicrobots)**

几个月前，我们分享了关于这些类似[复制器的微型机器人](http://hackaday.com/2014/04/23/micro-robots-are-scary-awesome/)的原始故事，它们可以在表面上爬行，使用工具，制造微型零件。机器人本身实际上只是愚蠢的小磁铁——魔力就在它们下面的表面。

这引起了一些人的兴趣，于是[Mike]将[做成了它的一个非常简单的版本](https://www.youtube.com/watch?v=7_Wh6i8Hb6c)，能够进行一维运动。这激发了[Sprite_TM]更上一层楼，推出了他自己的二维版本。他设计了自己的 PCB，允许他通过电路板的轨迹向各个方向传输电流——这有效地允许他控制一个小钕磁铁向他想要的方向移动。
