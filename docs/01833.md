# 青少年在驾驶他们的木制跑车时靠边停车

> 原文:[https://hack aday . com/2012/10/09/青少年驾驶他们的木制跑车时靠边停车/](https://hackaday.com/2012/10/09/teens-pulled-over-while-driving-their-wooden-roadster/)

![](../Images/77a84c50fab8818bf3ae4d90fc2873ed.png "teens-pulled-over-while-driving-their-wooden-roadster")

从这个角度来看，我们会认为手工制作的木制跑车是合法的。但是它缺少一些在高速公路上需要的东西。建造了这辆[的青少年前几天晚上被拦下](http://www.nu.nl/opmerkelijk/2928801/politie-haalt-zelfgemaakte-houten-cabrio-van-weg.html) ( [翻译为](http://translate.google.com/translate?sl=auto&tl=en&js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&u=http%3A%2F%2Fwww.nu.nl%2Fopmerkelijk%2F2928801%2Fpolitie-haalt-zelfgemaakte-houten-cabrio-van-weg.html))并被指控驾驶时没有时速表或侧面指示灯。

上图显示了他们设计的 mark II。可悲的是，他们破坏了第一个版本，这给了他们一个机会来彻底检查整个设计。现在他们有了一个合适的框架，它是由钢方管焊接而成的。它有一个令人印象深刻的齿轮齿条式转向系统和前后减震悬架。安装在座位后面的越野车发动机通过链条驱动后轮。他们使用 Arduino 来增加转向信号，并为夜间驾驶提供前灯。

[Gerrit]提供了这方面的提示，他认为由于 Arduino 已经在车辆中使用，所以添加速度表并重新上路应该是一种快速解决方案。
