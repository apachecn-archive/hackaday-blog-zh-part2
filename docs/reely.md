# 希望 X:用 ReelyActive 创造智能空间

> 原文：<https://hackaday.com/2014/07/22/reely/>

[https://www.youtube.com/embed/JKHxZdGM-ck?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/JKHxZdGM-ck?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

当我们听到物联网时，我们会认为它是一种带有某种传感器、无线电模块和将数据上传到互联网的能力的便携式设备。然而，正如来自 ReelyActive 的杰夫(Jeff)上周末在 HOPE X 上向我们展示的那样，并没有说把数据放到互联网上的设备必须是便携的。

[【杰夫】的初创公司](http://context.reelyactive.com/index.html)正在研发一种能将每个空间都变成智能空间的设备。这是通过连接到电脑的无线电模块来实现的，这些模块可以监听蓝牙以及 868、915 和 2400MHz 频段。这些模块将每个地方变成一个智能空间，识别谁刚刚走进一个房间，谁在特定的位置。把它想象成任何真正智能的房子的无形基础。

无线电模块本身用 Cat5 电缆连接，能够插入集线器或现有的以太网分支。使整个事情运转的软件可以在任何东西上运行；如果你想让一个 Raspi 在你进入房间时打开灯，或者在你离开大楼时关闭恒温器，那只是几行代码和一个继电器。

软件是开源的，[Jeff]和他的团队正在考虑开放硬件。这是一个很好的想法，这将是 Hackaday 奖的一个很好的参赛作品，但 ReelyActive 位于蒙特利尔，像叙利亚和朝鲜一样，我们不允许在魁北克举办比赛。
