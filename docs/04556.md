# 遥控飞机上的总距推力矢量

> 原文：<https://hackaday.com/2013/11/15/collective-pitch-thrust-vectoring-on-a-rc-plane/>

上面显示的遥控飞机是悬停在该位置。这大概是它能做的最不令人印象深刻的事情了。这就是[在飞机上集体俯仰推力矢量的力量。](http://www.youtube.com/watch?v=6YkkZw9GVaU)

那么究竟什么是总距推力矢量呢？简单来说，这就像把直升机旋翼绑在飞机前面。我们认为这背后的基本机制被称为[旋转斜盘](http://en.wikipedia.org/wiki/Swashplate_(helicopter))(在直升机旋翼上发现的)，它允许推力矢量，这意味着螺旋桨叶片实际上可以周期性地改变它们的螺距，同时仍然高速旋转！这就是为什么直升机能做出像滚桶这样疯狂的动作。

一架普通的遥控飞机只能随着发动机的速度增加或减少推力。但有了这个，推力可以随着叶片旋转而周期性变化，从而实现推力转向(高级转向)。加上一些巨大的控制面和机翼稳定器，这意味着一些非常疯狂的空气动力学壮举。
