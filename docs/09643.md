# 用硬盘制作麦克风

> 原文:[https://hack aday . com/2015/07/27/用硬盘制作麦克风/](https://hackaday.com/2015/07/27/make-a-microphone-out-of-a-hard-drive/)

[Rulof Maker]喜欢用旧电子产品制作漂亮的项目。引起我们注意的是一个由旧硬盘部件制成的麦克风。驱动器的臂和磁铁被放在一边，而铝底座被对角切成两块。其中一块后来被用来将硬盘的磁铁和臂重新组装到一个木制平台上。

驱动臂和音圈致动器是这个项目的关键部分。它被改装成一个金属延伸部分，这样就可以连接一个从音频扬声器上剪下的纸盆，这个想法被用在[麦克风项目](http://hackaday.com/2015/04/18/diy-bass-drum-microphone-uses-woofer-cone-as-diaphragm/)中，我们已经[展示过](http://hackaday.com/2007/10/27/speaker-as-a-microphone/)。从扬声器中取出的铜线被焊接到手臂上的音圈和音频插孔上。在硬盘麦克风的第一个版本中，手臂通过一对弹簧保持直立，当锥体捕捉到声音时会振动。

虽然麦克风工作正常，但[Rulof]看到了改进的空间。在第二个版本中，他用磁铁取代了机械弹簧，以保持手臂在高处。一对粘在底座的两侧，而另一对从旧光驱中回收的粘在臂上。他制作了一个更大的纸锥，并添加了一个由裤袜制成的流行过滤器。更高的音质是显而易见的。如果你对更多的[Rulof]项目感兴趣，可以看看他的[YouTube 频道](https://www.youtube.com/channel/UC5eVAqIMYef7YQ9QrXKpfjA)。
