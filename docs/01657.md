# Popinator 向你的嘴里发射爆米花，可能是一个公关噱头

> 原文：<https://hackaday.com/2012/09/19/popinator-fires-popcorn-into-your-mouth-is-probably-a-pr-stunt/>

[https://www.youtube.com/embed/b1cz8IasV4w?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/b1cz8IasV4w?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

印第安纳爆米花公司，也就是你可能在便利店找到的那家生产袋装玉米的公司，在他们的网站上发布了一个名为[爆米花机](http://www.popcornindiana.com/popinator-project)的项目。这是一个装置，你把它装满爆米花，打开，对“砰”的一声做出反应，把一片爆米花放进嘴里。关于这个构建的细节很少，很可能是因为 Popinator 并不完全按照 Popinator 项目的[视频演示](http://www.youtube.com/watch?v=b1cz8IasV4w)中描述的那样运行。

我们将称之为构建一个想象中的印第安纳爆米花营销无人机，但这个项目背后的想法实际上相当有趣。使用语音识别来确定何时听到“砰”的一声打开机器的想法非常酷。使用飞行时间来计算某人的头部位置，这使得这个构建成为我们以前很少见过的令人敬畏的类别。

尽管很酷，但我们不禁认为这个项目只是试图“像病毒一样传播”并从随机的技术博客中获得大量宣传，只使用一个摄像机和几个小时的 Final Cut Pro。不客气，印第安纳爆米花。

肯定会有大量的评论来讨论这个版本的优点。你可以在休息后查看这些评论，以及官方的 Popinator 视频。
