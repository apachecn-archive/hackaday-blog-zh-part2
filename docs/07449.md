# 按下按钮，接受糖果(或死亡)

> 原文:[https://hack aday . com/2014/10/30/push-button-receive-candy-or-death/](https://hackaday.com/2014/10/30/push-button-receive-candy-or-death/)

你会在万圣节分发糖果吗？也许你有一个聚会要参加，或者带着孩子在附近转，而不能回家接电话。你不想成为没有糖果的黑房子，因为那是一个很好的方式。除了试图避免用门廊上无人看管的碗诱惑潜在的小偷，我们还不确定[本]的催化剂是什么。不管是什么原因，我们很高兴向大家展示[糖果或死亡](http://www.benrady.com/2014/10/candy-or-death-the-automatic-halloween-candy-dispenser.html)，他的游戏化糖果(或死亡)分发机。

好吧，现在它只分发糖果。[本]还没有完全摆脱他的死亡射线。他把它设计在面向门廊的窗户后面，这样它就不会被弄乱。不给糖就捣蛋的人所能做的就是按下按钮，拿走糖果。它是围绕一台谷物分配器建造的，该分配器被修改为由一根圆杆驱动，该圆杆由一台 NEMA-17 步进电机和一台带电机护罩的 Arduino Uno 驱动。糖果沿着一段铝制雨水沟滑入一个塑料堆积箱，整个东西被装进一个漂亮的木头框架里。

有必要做一些调整来防止它卡住。分配器的漏斗使用橡胶刀片来控制流量，他最终移除了一些并修剪了其他的。[本]有一个相册，里面有他所有的[建造图片](http://imgur.com/a/8OJBE)，并把[的代码](https://github.com/benrady/candyBot)放在 gits 上。留下来从正面和背面观看机器的视频。
