# NFC 戒指解锁您的手机

> 原文：<https://hackaday.com/2014/03/17/nfc-ring-unlocks-your-phone/>

[![NFC Ring](img/e13864c79f4a1195cc197354c87b98e9.png)](http://hackaday.com/2014/03/17/nfc-ring-unlocks-your-phone/nfcring/)

这个小戒指包含了 NFC 钥匙链的核心，允许[Joe]用手指触摸解锁他的手机。

NFC 戒指的灵感来自于 [Kickstarter 项目](https://www.kickstarter.com/projects/mclear/nfc-ring)的一个类似设备。[Joe]支持那个项目，但是后来决定建立他自己的版本。他拆开了一个 NFC 钥匙链，拆下了用于通信和电源的线圈。接下来，他在一根管子上缠绕了一个新线圈，与他的戒指尺寸相匹配。装配完成后，环氧树脂被用来铸造环形。

在将戒指切割成合适的尺寸，并做了一些抛光后，[乔]最终得到了一件实用的极客珠宝。为了照顾 NFC 解锁，他安装了 [NFC 锁屏关闭](http://repo.xposed.info/module/pk.qwerty12.nfclockscreenoffenabler)。它使用 [Xposed](http://repo.xposed.info/module/de.robv.android.xposed.installer) ，所以需要一个根 Android 设备。

我们必须等待，看看[乔]的自制解决方案与他的 Kickstarter 戒指相比如何。在那之前，你可以在休息后观看一个用戒指解锁手机的快速视频。
