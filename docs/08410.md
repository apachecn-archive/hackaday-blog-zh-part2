# 拉小提琴的机器人 Ro-Bow

> 原文：<https://hackaday.com/2015/02/26/ro-bow-the-violin-playing-robot/>

有些机器人会用吸尘器打扫你的房子，修剪你的草坪，在数百英尺的空中盘旋时，它们一眨不眨的电子眼会一直盯着你。拉小提琴的机器人怎么样？这就是(塞斯·戈尔茨坦)所建立的。他称之为“动态雕塑”，但有足够多的电子和机电一体化技术让最挑剔的修补匠也保持兴趣。

[Seth]的小提琴演奏动态雕塑有三个主要部分。第一种是弓架，用电磁铁将弓压在弦上，拉动弓穿过弦。每根琴弦都用四个橡胶圆盘拨弄，一个倾斜机构转动小提琴，这样期望的琴弦总是在琴弓和机械手指的下面。

就软件而言，Ro-Bow 将 MIDI 文件转换成机器人机械化，让小提琴唱歌。据我们所知，它还不如一个人类玩家；一次只能弹奏一根弦。然而，它的工作非常出色，是一件令人惊叹的机械雕塑。
