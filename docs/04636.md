# 用 Arduino 开自助洗衣店

> 原文:[https://hack aday . com/2013/11/25/running-a-laundromat-with-an-arduino/](https://hackaday.com/2013/11/25/running-a-laundromat-with-an-arduino/)

![Wasch_140](../Images/23bb09bc0daa49a58c0cc8af03f995dd.png)

[Hubert]给我们发了一个关于一个朋友的项目的提示，该项目旨在从一家自助洗衣店的故障电子设备中拯救出来。我们不完全确定旧的控制中心出了什么问题，但考虑到更换一个会花费近 25，000 欧元，我们认为[斯特凡]找到了完美的解决方案:他给它做了一个 Arduino 和 Android 大修 ( [翻译](http://translate.google.com/translate?sl=auto&tl=en&js=n&prev=_t&hl=en&ie=UTF-8&u=http%3A%2F%2Fedv-dompteur.de%2Fforum%2Findex.php%3Fpage%3DThread%26postID%3D613%23post613))。

虽然[Stefan]解释说电路板有缺陷，但也许我们的一位德国读者可以帮助我们进行更具体的翻译。然而，更清楚的是升级系统的步骤。自助洗衣店的情况有点紧急:顾客没有办法支付机器的使用费。因此，[斯特凡]可以随心所欲地进行他认为合适的改革。他决定取消复杂的按钮设置，转而使用触摸屏 Android 平板电脑，为用户提供一个简单的界面来进行选择。平板电脑仅用作输入设备。繁重的工作由 Arduino Mega 2560 处理，它连接到原始系统的剩余部分，并控制自助洗衣店的 27 台机器。

[Stefan]承认他不是 Arduino 的特别粉丝，但就价格而言，这是一个很难击败的解决方案。他不是唯一一个对 Arduinos 进行彻底改造的人。看看其他升级机器的例子，比如增强 Arduino 的 PopCARD 自动售货机。

> 出问题的，是两块主板。
> 
> 一位新顾客称他为“什么都不管用！现在需要帮助！”控制中心不再工作了。打开门后，可以看到开关和 led 的一些组合矩阵电路。一般来说，共享列驱动程序并不困难，但是调试有点困难，尤其是在这样的共享设置中。
> 
> 这两个矩阵板都有故障，但由于大多数简单的原则，很容易修复它们。在对电路板进行更多研究后，发现了更多错误。低发光的发光二极管，没有任何光的发光二极管，但电完全工作，一些在高电阻和其他纯粹的短路。相当气人，因为主人前几天肯定所有的开关都工作得很好。
> 
> 搭载了一些晶体管后，矩阵电路在模拟输入下正常工作，所以下一步是带处理器的主板。顺便提一下，还有另一个类似的主板。首先，简单地检查硬币接收器单元，服务/操作开关矩阵，并操作四个相同的中继卡。
> 
> 这些都是很基本的。只是一些由时钟移位寄存器驱动的光电耦合器，主要用于切换洗衣机，也用于获取机器的完成状态。
> 
> 这些在后来的设置中完全没有修改。
> 
> 接下来的照片是关于洗衣机和洗衣粉分配器的一些板。
> 
> 因为他在组装的时候够不到一些零件，所以他不得不整夜拆除整个控制中心，只为了确定主板完全坏了。他们有未定义的逻辑电平，浮动信号，这里不需要修正。
> 
> 他没有花 25.000 欧元买一个新的，而是买了一个 Arduino Mega2560 R3，他并不喜欢，但 Arduino 非常便宜，所以不管怎样。于是他在原型板上加了一个自制的简易继电器屏蔽罩，差不多就完成了。
> 
> 很有趣，但是用 Android 平板电脑替换所有的开关和 led 要比保留它并将其连接到 Arduino 容易得多，Arduino 已经有足够的端口，所以他需要再添加一个端口扩展等等。但他试图保持简单和可替换。起初，原型电路板 shied 上没有任何半导体，但他必须添加半导体，以消除一些额外的旧电路板，因此他只能保留继电器卡和两条主电源电路。
