# 逆向工程:傅立叶级数

> 原文：<https://hackaday.com/2013/04/04/retrotechtacular-the-fourier-series/>

这里有一个很快的视频，它用一种不同于我们习惯的方式来理解傅立叶级数。如果你是一个经常阅读的人，我们肯定你听说过傅立叶级数(通常被称为 FFT 或快速傅立叶变换)，但你很可能对它知之甚少。该系列允许您将复杂的信号(比如声波)分解成简单的正弦或余弦方程的组合，这些方程可以由微控制器处理。

长期以来，我们已经有了这种基本的理解。但是当你开始深入挖掘时，我们发现它变成了一个不那么直观的数学练习。中断后嵌入的视频剪辑改变了这一点。它从显示一个旋转矢量开始。水平绘制该向量的尖端将绘制波形。然后利用傅立叶级数，将谐波的旋转矢量添加到最后一个矢量的顶端。这些谐波相加的结果产生了上面看到的基于正弦的方波近似值。

这有点拗口，我们相信您会同意视频演示更容易理解。但是三分钟的短片只是触及了表面。如果你决心掌握傅立叶级数，试试这个主题为的庞大的斯坦福系列讲座。
