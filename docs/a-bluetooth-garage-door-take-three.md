# 蓝牙车库门，第三次

> 原文：<https://hackaday.com/2015/01/02/a-bluetooth-garage-door-take-three/>

几年前，[楼]想出了一个非常聪明的建立，用他的手机打开他的车库门。他只是拿了一个蓝牙耳机，用一个晶体管代替了扬声器，并将晶体管连接到他车库开门器的几根电线上。当蓝牙耳机连接时，扬声器输出的短哔哔声打开了门。

这款手机的最新版本取消了简单的蓝牙耳机，代之以蓝牙 4.0 芯片。原因是苹果和他们的应用程序商店围墙花园永远不会允许三星蓝牙耳机与他们的设备一起使用。

最新的构建就像使用蓝牙耳机一样简单。一块看起来使用了 TI 的 CC2540 芯片的电路板通过几个无源器件和一个晶体管连接到车库开门器上。将新电路与手机配对就像短接一对引脚一样简单，新的 iOS 应用程序做了它应该做的事情——按下非按钮打开车库门。

虽然它不能和垃圾抽屉里的碎片放在一起，但它仍然是用手机打开车库门的一个非常简单的解决方案。下面视频。
