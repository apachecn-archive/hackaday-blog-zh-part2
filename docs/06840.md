# 机器人秤

> 原文:[https://hackaday.com/2014/08/17/robotic-scalextrics/](https://hackaday.com/2014/08/17/robotic-scalextrics/)

![scalectrics](../Images/83f4c82170eb1aa0110defd692aff445.png)

在大众汽车的工厂里，有两座塔楼——auto türme——装满了巨大的机器人，将汽车提升到停车位。这是迄今为止将大量汽车放入小空间的最有效方式。老爷车是存在的，所以[为 1:32 比例的汽车建造一个完全过度紧张但完全令人敬畏的机器人车库](http://lapresaslot.blogspot.com.es/2012/10/ubicando-el-parking.html)怎么样？(.es，谷歌 [translatrix](https://translate.google.com/translate?sl=es&tl=en&js=y&prev=_t&hl=en&ie=UTF-8&u=http%3A%2F%2Flapresaslot.blogspot.com.es%2F2012%2F10%2Fubicando-el-parking.html&edit-text=)

这个项目[是围绕着几个“架子”](https://translate.googleusercontent.com/translate_c?depth=1&hl=en&ie=UTF8&prev=_t&rurl=translate.google.com&sl=es&tl=en&u=http://lapresaslot.blogspot.com.es/2012/10/el-parking.html&usg=ALkJrhi3u2F4EDpoe60HsfN6-EovcxOmGg)建造的，用来放置围绕着中央电梯的轿厢。Arduino 负责移动所有的电机和读取所有的传感器，该项目的基本思想是能够选择一辆汽车，并在几分钟后让它出现在赛道上。

虽然这只是已经令人印象深刻的老虎机轨道的一小部分，但它是最电子化的。其他独特的增加包括一个非常独特的[悬臂/吊桥](https://translate.googleusercontent.com/translate_c?depth=1&hl=en&ie=UTF8&prev=_t&rurl=translate.google.com&sl=es&tl=en&u=http://lapresaslot.blogspot.com.es/2012/11/el-puente-colgante.html&usg=ALkJrhjM5jlW82pOV99L_juNvqztBXiiFQhttps://translate.googleusercontent.com/translate_c?depth=1&hl=en&ie=UTF8&prev=_t&rurl=translate.google.com&sl=es&tl=en&u=http://lapresaslot.blogspot.com.es/2012/11/el-puente-colgante.html&usg=ALkJrhjM5jlW82pOV99L_juNvqztBXiiFQ)和常用的建模技术，只用纸板和胶水创造一个景观。
