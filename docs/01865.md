# 在 Mac 上还有更多软件定义的无线电乐趣

> 原文:[https://hack aday . com/2012/10/12/still-more-software-defined-radio-fun-on-the-MAC/](https://hackaday.com/2012/10/12/still-more-software-defined-radio-fun-on-the-mac/)

![](../Images/0b7323a3fe116a9b0144153c856480ae.png "Cocoa Radio")

尽管软件定义无线电的世界开始时只是 Linux 的努力，但最近几个软件版本已经把球完全踢给了 OS X 用户。[hpux735]的新 [Cocoa Radio release](http://www.alternet.us.com/?p=1806) 为任何拥有 USB 电视调谐器和 mac 的人提供了(几乎)全功能的软件定义无线电。

本周早些时候，我们看了(并测试了)[Elias]'[gqrx](http://dekar.wc3edit.net/2012/09/30/osx-port-of-the-awesome-gqrx-sdr-software/)的港口，印象相当深刻。[hpux735]的应用程序做同样的工作，也提供源代码，所以你可以自己编译。

之前，[hpux735] [将这些基于 RTL2832U 的 USB 电视调谐器加密狗的 osmocom 驱动程序](http://hackaday.com/2012/05/12/putting-a-software-defined-radio-on-a-mac/)移植到 Mac 上，并编写了一个小的 Cocoa 驱动程序。新的 Cocoa Radio 软件使用了这个驱动程序，并增加了你期望从软件无线电包中得到的所有特性；在这篇文章的标题图片中，你可以看到我家附近排名前 40 的广播电台和他们对动态范围的平淡的仇恨。

[hpux735]贴了几个他开发过程的视频。休息之后你可以去看看。
