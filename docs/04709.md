# 顺从的机器人抓手不会搅乱你的鸡蛋

> 原文:[https://hack aday . com/2013/12/03/compliant-robot-gripper-wont-scramble-your-eggs/](https://hackaday.com/2013/12/03/compliant-robot-gripper-wont-scramble-your-eggs/)

[Chiprobot]已经[创造了一个惊人的柔顺手爪](http://letsmakerobots.com/node/39387)。设计机器人手(或末端执行器)可能是一项危险的任务。给机器人[一双又大又好又强壮的手](http://www.youtube.com/watch?feature=player_detailpage&v=aj-OpTHixpU#t=25)是很容易的。强大的夹具必须由传感器控制。然而，不能总是依赖传感器来确保这些手不会压碎它们接触到的任何东西。硬件故障，软件有 bug。有时候，最好的解决方案是一个巧妙的机械设计，它能确保夹子与它所夹的物体相一致。在之前，我们已经看到过[“卡住”夹子。(之所以这样命名，是因为他们使用了一种颗粒物质，这种物质会堵塞在被抓住的物体周围)。](http://hackaday.com/2013/06/04/dead-simple-jamming-gripper-design/)

(Chiprobot 的)抓手是完全不同的东西。他在搅拌机中设计了他的夹子，并用他的 Ultimaker 3D 打印机打印了出来。材料是柔性 PLA。三个塑料“手指”缠绕在被抓住的物体上。手指由两条印刷塑料带组成，通过电线连接。手指的柔性塑料创造了一个板簧设计。手指连接到夹持器中心点的线性致动器上。线性驱动器本身是另一个伟大的黑客。[Chiprobot] [用一个伺服系统和一个空胶棒](http://www.youtube.com/watch?v=sI8tvjdhYxw)创造了它。当线性致动器被拉进时，手指在其抓持下拉动任何物体。最终的结果是，握的力度足够大，可以在摇晃鸡蛋时握住鸡蛋，但不足以打碎鸡蛋。

我们希望看到抓取器抓取其他物体，因为鸡蛋可能会出奇的坚固。我们都见过[物理把戏](http://www.youtube.com/watch?v=XwgAKW4LrKU)徒手挤压鸡蛋不会打破它，但是戴着戒指挤压鸡蛋会导致它破裂……就像鸡蛋一样。
