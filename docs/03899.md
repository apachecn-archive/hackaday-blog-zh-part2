# 用经典条件反射把自己从社交媒体中解放出来

> 原文:[https://hack aday . com/2013/08/22/将自己从经典条件下的社交媒体中解放出来/](https://hackaday.com/2013/08/22/free-yourself-from-social-media-with-classical-conditioning/)

![pavlov](../Images/fe622b39404f531412dd1947368cdffc.png)

当[罗伯特]和[丹]应该继续写论文的时候，他们发现他们实际上花了很多时间在脸书和其他社交网站上消磨时光。他们从一个剥皮器盒子中获得灵感，安装了他们的电脑[，以便在他们每次冲浪到脸书](http://www.robertrmorris.org/pavlovpoke)时对他们进行电击。

他们的构建使用了 OS X 的 UI 检查器和一个 Python 脚本来激活一个 Arduino，该 Arduino 与那些“令人震惊的口香糖”恶作剧之一相关联。这种令人震惊的接触是附在键盘手腕休息，提供了一个美妙的刺痛的感觉，每当家伙冲浪到脸书。

因为震动还不够极端，[罗伯特]和[丹]通过唤起机械土耳其人用户的愤怒，将他们的构建更进了一步。他们编写了一个 Python 脚本来查看他们的 UI 检查器，并在他们登录脸书时向 Mechanical Turk 提交一个作业。结果是一个随机的人得到 1.40 美元，在电话里对[罗伯特]或[丹]大喊大叫，因为他们在脸书身上浪费了时间。
