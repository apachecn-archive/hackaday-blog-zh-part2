# 摩托车爱好者的自动调平陀螺摄像机

> 原文:[https://hackaday . com/2014/08/18/an-auto-leveling-gyro-camera-for-motor-favorites/](https://hackaday.com/2014/08/18/an-auto-leveling-gyro-camera-for-motorcycle-enthusiasts/)

![FB2XXIQHF23RIUW.LARGE](../Images/c3d60f6b44f140e8dbfb97fb46c37005.png)

[Saftari]受到 MotoGP 世界锦标赛中用于捕捉视频的技术的启发，创作了这些关于如何建造自动调平陀螺摄像机的[指南](http://www.instructables.com/id/Gyro-Camera-for-Motorcycle/?ALLSTEPS)。无论摩托车转弯时与地面成多大角度，他开发的设置都可以使摄像机保持垂直于地面的水平。

涉及的组件包括一个 Arduino Uno、一个三轴加速度计、一个数字伺服系统和一个陀螺仪分线板。建造了一个支架来容纳和固定摄像机到车辆的侧面。2mm 的丙烯酸用于此，并通过加热材料而弯曲。一旦完成，测试运行就完成了，展示了这种类型的自己动手钻机的能力。

休息后的视频质量有点模糊，但它证明了陀螺相机设置可以在家里建立:
