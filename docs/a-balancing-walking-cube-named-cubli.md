# 一个名为 Cubli 的平衡行走立方体

> 原文：<https://hackaday.com/2013/12/19/a-balancing-walking-cube-named-cubli/>

见见 Cubli！Cubli 是一个 15 x 15 x 15cm 厘米的机器人立方体，它可以通过一系列陀螺仪滚动和平衡。

该项目自 2011 年 2 月在苏黎世动力系统与控制研究所开始蓬勃发展，并开始给人留下深刻印象。今年早些时候，我们第一次分享了 Cubli，当时它只是一个小小的二维和三维原型。

从那时起，它已经成为一个完全封闭的无线立方体，能够直立跳跃，保持平衡，并控制下落——它可以行走和滚动！为此，它在每个轴上都有三个大的反作用轮，当刹车时，可以将它们的角速度传递给整个立方体，允许它向任何方向移动。

根据休息后的视频，该团队只是在建造立方体，“因为他们可以”，然而其他研究人员对自组装机器人的技术应用，甚至行星探索感兴趣。
