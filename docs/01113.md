# 测量组件对极低温度的反应

> 原文:[https://hack aday . com/2012/07/22/measuring-how-components-react-to-extremely-cold-temperature/](https://hackaday.com/2012/07/22/measuring-how-components-react-to-extremely-cold-temperatures/)

![](../Images/0c29106c04c6b867fb1fb94a7e88ecb6.png "how-components-react-to-the-cold")

[Shahriar Shahramian]正在玩一些液氮，以便观察各种成分如何对极低的温度作出反应。休息过后，你会看到 41 分钟的视频，他在视频中指导并解释了每个实验。这确实有实际应用。如果你正在设计用于太空的硬件，你肯定需要知道硬件将会受到怎样的影响。我们实际上已经看到了[为此建造的试验台](http://hackaday.com/2008/04/09/thermal-testing-electronics-for-outer-space/)。

在演示过程中，他没有淡化观察到的概念，包括控制每个温度变化反应的方程。如果你想吃清淡一点的食物，你应该看看一些液氮烹饪技巧，比如这个超冷鸡尾酒项目。
