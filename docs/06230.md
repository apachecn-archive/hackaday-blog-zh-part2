# 廉价的热感相机获得了令人印象深刻的回炉

> 原文:[https://hack aday . com/2014/06/04/cheap-thermo cam-gets-an-impressive-re haul/](https://hackaday.com/2014/06/04/cheap-thermocam-gets-an-impressive-rehaul/)

[Max Ritter]21 岁，是德国 Weingarten 应用科学大学信息技术专业的学生。三年前，他给我们带来了 DIY 的廉价热感相机，这是一款用于热成像的工具，他已经做了一些升级。

最初的廉价热摄像机使用了 Arduino，温度计枪的传感器和一些 XY 伺服系统。在大约 2 分钟内，XY 伺服系统可以使用温度计的传感器扫描和测量 1344 个点，创建一个 42 x 32 像素的热视觉地图——不惊人，但它很有效——而且很便宜！

新版本(V3)拥有自己的 ARM Cortex M3 处理器，它可以在 2 分钟内测量 3072 个点，从-70 C 到 380 C，精度为 0.5 C，并以 640 x 480 的分辨率导出图像——接近商业产品！它不具备实时扫描的能力，但对于大多数用途来说，你需要一个这样的设备——这真的没有必要。

这是第一代的视频，你可以从他的网站上免费获得这些计划。
