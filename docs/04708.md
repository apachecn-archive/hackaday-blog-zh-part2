# 手机充电感应钱包

> 原文:[https://hack aday . com/2013/12/03/手机-充电-感应-钱包/](https://hackaday.com/2013/12/03/cellphone-charging-inductive-purse/)

![projects_cell-phone-charging-purse-00](../Images/efdef6d0c4537d1373c2c3ba3c586dd9.png)

不管出于什么原因，手机公司似乎真的不在乎给你的手机提供一个好电池。如果你碰巧有一个钱包，这里有一个很好的方法——把它变成一个感应充电器！男包也算，我们不做评判。

Adafruit 的[Becky]想出了一个关于这个可穿戴黑客的好主意。如果你的手机长时间放在钱包里，为什么不给它充电呢？这是一个非常简单的方法，利用一对感应充电回路。一个藏在你选择的包包底部，另一个安装在办公室或家里的固定装置上。她正在使用磁铁将她的钱包吸到工作中的架子上——这确保了线圈排成一行，这样就可以传输全部额定电荷。

另一个选择是把整个感应充电电路放在你的钱包里，然后用一个有专门口袋的电池组来装你的手机——这样手机就可以在安全存放的时候一直充电！
