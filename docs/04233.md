# JJ·达舍:修补匠(纪录片)

> 原文：<https://hackaday.com/2013/10/05/jj-dasher-the-tinkerer-documentary/>

![ScreenShot038](img/d9194732402c5ebbeefc3ae0bbdfca6a.png)

这不是我们通常的集会，这是一部关于黑客的非常酷的纪录片。[JJ·达舍]是来自犹他州泰勒斯维尔的一名修补匠，这是他的故事。

像许多机械黑客一样，他开始和他父亲一起拆卸引擎，他父亲当时拥有一家摩托车修理店。上面的封面照片是他的微型自行车项目，可以让他走得相当快！他还制作了不少特斯拉线圈，喜欢从旧货店捡东西来做黑客。他现在有一个孩子，这占用了他很多时间，但他开玩笑说，他的儿子只是他等待完成的下一个大项目。

事实上，我们之前已经多次介绍过(JJ 的)项目。他给我们带来了 [Doombox](http://hackaday.com/2010/03/01/doombox-classic-keys-meet-tiny-screen/) (只能毁灭的手持电脑)[、令人敬畏的强力 GPS 撞针器](http://hackaday.com/2013/01/05/brute-forcing-a-gps-pin/)，以及在万圣节的精神中我们最喜欢的东西之一:[一个特斯拉线圈，它能提供令人震惊的糖果！](http://hackaday.com/2011/11/08/a-tesla-coil-that-delivers-shocking-candy/)

休息后留下来观看这部拍摄精良的纪录片——别担心，它只有 8 分钟长！
