# 制作自己的激光切割 PSU

> 原文：<https://hackaday.com/2015/05/27/making-your-own-laser-cut-psu/>

[Csaba]和他的朋友不久前在易贝买了一个 600W 的开关实验室式电源装置，经过大约一年的电线缠绕和混乱，最终决定将它封装在一个别致的盒子里。

PSU 本身需要一些修改，因为它只是一个控制器和一个电源板——所以他们增加了一个专用的电源变压器和一个缓冲电容器。外壳由 3 毫米厚的胶合板制成，这是他们专门为 PSU 设计和激光切割的，看起来棒极了。

它包括一个冷却风扇，一个小型数字显示器和一整套控制装置，用于微调您的电子设备功率要求——休息后看看演示视频。
