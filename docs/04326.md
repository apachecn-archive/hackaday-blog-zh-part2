# 圆锯到台锯的转换

> 原文：<https://hackaday.com/2013/10/19/circular-saw-to-table-saw-conversion/>

绳状圆锯在任何旧货店里都不稀罕，然而台锯是一项更昂贵的投资——你有没有想过能否用圆锯做一个？[马提亚斯万德尔]做了，他只是碰巧非常擅长木工！他制作了许多很棒的木工视频在 YouTube 上分享，在他最新的博客帖子中，他向我们展示了一种相当复杂的方法来将一把普通的老式圆锯转变成一把实用的台锯。

虽然这个概念看起来很简单，但除非你做了很多木工活，否则你可能会惊讶于制造东西对[马提亚斯]来说是多么容易——我们知道我们做到了。在视频的最后，他有了一个全功能的台锯，可以升高和降低高度，并以不同的角度切割。

如果你对自己制作一个感兴趣，他会在视频中详细解释制作过程——休息后看看吧！
