# 建立一个支持 WiFi 的谢妮柜台

> 原文:[https://hack aday . com/2013/06/11/building-a-wifi-enabled-谢妮-counter/](https://hackaday.com/2013/06/11/building-a-wifi-enabled-nixie-counter/)

![wifi-nixie-counter](../Images/23bd2aabfce315013c4c363e2189e10d.png)

[凯文·巴拉德]用公司的钱建了这个谢妮柜台。像这样的管子越来越难找到了，因为它们已经不再生产了。但是，当老板递给你一张公司信用卡时，这种担心就会让位于你的购物冲动。开始到完成[这个支持 WiFi 的计数器](http://tellaparteng.tumblr.com/post/51805124205/in-the-nix-of-time)花了六个星期才建成。

由于驱动它的电动 Imp，将板连接到互联网非常容易。困难的部分来自建立一个驱动板和电子管插座。我们没有看到他如何产生高电压的很多细节。但是你可以从图片中很好的感受到管道连接器。他正在使用 Kosbo 的适配器 PCB，该适配器 PCB 将管脚分成两排 0.1 英寸间距的管脚头。丙烯酸基底有一个端口，每个端口由一大块丙烯酸隔开的插脚插座构成。线束缠绕在底座的背面，与驱动硬件紧密配合。它被编程来计算某种类型的公司指标(毕竟它是由公司资助的)。他们一定相当成功，因为这些数字在演示视频中飞快地闪过。
