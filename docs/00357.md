# 因此，基于树脂的 3D 打印机开始泛滥

> 原文:[https://hacka day . com/2012/05/10/于是，基于树脂的 3d 打印机泛滥成灾——begins/](https://hackaday.com/2012/05/10/and-so-the-deluge-of-resin-based-3d-printers-begins/)

![](../Images/4b86bda91bcdc8d18f8001bd1938d4de.png "resin")

看起来 2012 年将成为树脂基 3D 打印机的一年。最新的来自[迈克尔·乔伊斯]，被称为[B9 创造者](http://b9creator.com/)。像其他树脂打印机一样，[迈克尔]使用 DLP 投影仪一次固化一层打印。层高约为 100 微米，这对于基于套件的打印机来说太疯狂了。

B9Creator 有一个 [Kickstarter](http://www.kickstarter.com/projects/b9creations/b9creator-a-high-resolution-3d-printer) ，套件售价 2400 美元。这个工具包里什么都有，包括 DLP 投影仪和一公斤树脂。2400 美元甚至比 Makerbot 或 RepRap 等最高档的熔融塑料 3D 打印机都贵得多，但这是你为高质量打印付出的代价。

当然，这个项目是在一个月前一个更早的，类似的，更神秘的叫做[贝洛索 3D 打印机](http://hackaday.com/2012/04/05/3d-printer-with-insane-accuracy-uses-a-dlp-projector/)的项目之后。B9Creator 承诺，一旦所有 Kickstarter 机器出货，它将是开源的，而且[迈克尔]对他的设计和树脂配方非常开放——这是一个制造商令人钦佩的品质。
