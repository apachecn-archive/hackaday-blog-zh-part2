# 越野四轮车使用…气动驱动？

> 原文:[https://hack aday . com/2014/05/02/off-road-quad-uses-a-pneumatic-drive/](https://hackaday.com/2014/05/02/off-road-quad-uses-a-pneumatic-drive/)

![AirQuad_Outside](../Images/22987a7f0d543f186d90a5bbfb99e7a4.png)

这是一个非常酷的工程学位项目——来自维也纳一所大学的学生团队制造了一个气动马达，并将其安装在一个四轴飞行器上！ ( [译](http://translate.google.ca/translate?hl=en&sl=de&u=http://www.airquad.at/&prev=/search%3Fq%3Dairquad.at%26es_sm%3D122%26biw%3D1920%26bih%3D955))

该团队由[西蒙·弗里斯彻]、[西蒙·谢德尔]、[克里斯托夫·西贝尔]和[曼努埃尔·斯特雷思]组成，他们与为我们带来了[VoLumen display](http://hackaday.com/2014/04/21/volumen-the-most-advanced-persistence-of-vision-display-yet/)和[Max]Ripper CNC 的[马克西米利安]和[塞巴斯蒂安]是同班同学！

对于他们的主要项目，目标是创造一种替代和创新的发动机——一种使用压缩空气的发动机，使用消防员的气罐。一旦他们想通了这一点，他们决定找点乐子，组装空中四轴飞行器。它只有几公里的范围，并没有它原来的汽油同行表现得那么好，但我们不得不承认，这是一个非常光滑的概念证明！
