# GoPro 弹弓

> 原文：<https://hackaday.com/2013/08/21/gopro-slingshot/>

想尝试航拍，却买不起四轴飞行器？[杰里米]装配了一个低成本的 GoPro 弹弓，拍了一些非常漂亮的湖面照片。

弹弓本身是为水球准备的，但很容易有足够的能量来点燃相机。为了获得良好的视频，需要一些稳定。[Jeremy]用包装泡沫制作了一个稳定鳍，并用吊环螺栓将其连接到 GoPro 的螺纹三脚架上。由泡沫和拉链组成的简单尾翼实际上很好地稳定了相机。

当你在湖边时，这看起来像是一个有趣的实验，因为你可以用房子周围的东西来建造它。对杰里米来说，这也是让他的狗开心的一种方式，因为每次拍照后她都会取回相机。休息之后，看看 GoPro 吊索装置的视频片段。
