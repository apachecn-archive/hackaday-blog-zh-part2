# 动手操作 FlexyDualie 3D 打印机挤出机

> 原文:[https://hack aday . com/2014/05/06/hands-on-the-the-flexy dualie-3d-printer-extruder/](https://hackaday.com/2014/05/06/hands-on-with-the-flexydualie-3d-printer-extruder/)

![Flexydualie Extruder](../Images/49997ae917a20276849d6ce934df3881.png)

听说过 FlexyDualie 挤压机吗？这是来自 Aleph Objects(Lulzbot 和 TAZ 的制造商)的[新型开源双挤出机](http://devel.alephobjects.com/lulzbot/TAZ/accessories/Dual_extruder_w_flexy/)，专门设计用于在硬材料和柔性材料中打印！

XRobots 的[James Bruton]刚刚拿到了一台新的 TAZ 打印机，他分享了他的想法。他正在用 ABS 塑料和 Ninjaflex 橡胶细丝进行测试。当然，你有时可以使用常规挤出机用 Ninjaflex 打印…但要做好头疼的准备，因为它会卡在你的挤出机导轨上。FlexyDualie 是专为挤出柔性细丝而设计的，这样可以大大减少麻烦，提高生产率！

在下面的演示视频中，[James]向我们展示了用硬材料和软材料打印零件的惊人可能性。他甚至详细讲述了如何为多材质 STL 文件设置 Slic3r。

从带硬轮缘的橡胶轮，到单向柔性接头，甚至是机器人的坦克履带……可能性是惊人的。事实上，他正计划制作一整套 3D 打印的外星人服装！
