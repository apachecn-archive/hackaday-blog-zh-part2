# 外在动机:雏菊风筝机载风力涡轮机

> 原文：<https://hackaday.com/2014/08/29/extrinsic-motivation-daisy-kite-airborne-wind-turbine/>

又有一个 THP 的词条没有入选，但值得分享。这一次我们的特色是[一个空中风力涡轮机](http://hackaday.io/project/1743)，正如项目描述所述，“可以安全有效地收获强劲和膨胀的风。”

冲压空气风筝旋转降落伞，反过来传递扭矩，可以在地面上捕获。以一种真正的黑客精神的方式，由[Rod]开发的装备在设计中利用了自行车车轮和直排轮。这个自制的发电机需要很大的空间来部署，但它看起来像是一个很好的空中能量收集的解决方案。[Rod]在 Hackaday.io 页面上查看了整个构建日志中的项目规范，并包括几个视频，描述了它是如何创建的，并显示了当它被释放到外面的气流中时会发生什么。还包括开源机载风力发电设备的图表和模型。

以下是他的几个视频。观看它们，并让我们知道你的想法。
