# 磁盘-O-Mat:放唱片的照相亭

> 原文:[https://hack aday . com/2013/12/10/disk-o-mat-a-photo booth-for-records/](https://hackaday.com/2013/12/10/disk-o-mat-a-photobooth-for-records/)

![record](../Images/5f5d5a0dfb6bb8658a9dce33ee584849.png)

照相亭是一个简单的概念——投入一枚硬币，在几分钟内拍几张照片。不过，这只是一个视觉记录。如果你想录制音频呢？因此，磁盘就诞生了。

光盘是[flo]的项目之一。代替照相亭里的小型暗室，[flo]安装了一个记录切割装置。7″的唱片是聚碳酸酯片，每个都通过真空夹具转移到转盘上。当塑料盘装入时，唱针放在盘上，唱片灯亮了。

没有电脑或任何其他数字手段来保存音频并在以后播放。一切都是按照 45——或者更确切地说，真正老的 78——的切割方式进行的；进入麦克风的任何东西都被直接切割成塑料。

这张唱片最初是在 2009 年制作的，已经在几个场馆巡回演出。[flo]正在努力加快这一过程，并使机器变得更加可靠。尽管如此，一个可怕的建筑和一个可怕的概念。
