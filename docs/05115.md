# 梅卡诺弹球机

> 原文:[https://hackaday.com/2014/01/21/a-meccano-pinball-machine/](https://hackaday.com/2014/01/21/a-meccano-pinball-machine/)

这个弹球桌几乎完全是由 Meccano 建筑公司制造的。[布莱恩·利奇]的[梅卡诺弹球机](http://www.selmec.org.uk/gallery_model_view.aspx?ID=75982872-ab34-45dd-b2f3-54e88f87b76d)有一个数字计数器，一个开球孔，脚蹼和一个计时器。

数字计数器可能是构建中最复杂的部分。通过给它发送一个电信号，一位、十位或百位数字都可以递增。电信号与电磁体接合，电磁体将电机连接到轮子上以增加分数。当一个数字从 9 滚动到 0 时，一种机制确保下一个数字递增，并允许计数器归零。

在翻转开关组上滚动球会增加分数。使用一种机制来确保开关将以较小的重量触发。电弧放电是一个问题，通过添加一个[缓冲器](http://en.wikipedia.org/wiki/Snubber)来抑制瞬态，这个问题得到了缓解。

弹球机在[东南伦敦梅卡诺俱乐部](http://www.selmec.org.uk/)进行了演示，这是一个很好的演示，展示了可以用建筑工具包建造什么。休息之后，看看弹球机的视频。
