# 雷达干扰

> 原文：<https://hackaday.com/2015/04/28/retrotechtacular-radar-jamming/>

有人说，最好的防守是有效的进攻。当在高空和途中向敌人投放有害载荷时，最好的防御是仔细计划你的接近和退出，并干扰他们的探测方法。如果他们找不到你，他们就不能开枪打你。

截至 1962 年 5 月，美国军方使用了三种主要的雷达干扰技术，如本周的电影中所描述的:安/ALQ-35 多目标中继器，安/ALQ-55 通信链路干扰机，以及安/ALQ-41 和-51 轨道断路器。这些装备最重要的作用是争取时间，这是各种战争中的宝贵资源。

安/ALQ-35 目标中继器由一个调谐器、脉冲发生器、发射机和控制面板组成，它们协同工作，在敌人的 PPI 示波器上显示多个误报。这个单位接收到敌人传来的脉冲，将其放大，重复，然后随机延迟发送回来。

安/ALQ-55 通信干扰机工作在 100-210 兆赫波段。它将威胁敌人的通信波段与信标和平民的通信波段区分开来，对它们进行评估，并用不连续的信号干扰它们，这有助于避免被发现。

最后，安/ALQ-41 和-51 反跟踪装置是用来破坏敌人的锁定并给出假情报的。它根据每种方法的角度和范围，以不同的方式提供对脉冲测距、FM-CW、圆锥和单脉冲雷达的同时保护。
