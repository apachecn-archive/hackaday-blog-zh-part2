# THP 参赛作品:经济实惠的金属 3D 打印机

> 原文：<https://hackaday.com/2014/06/09/thp-entry-an-affordable-metal-3d-printer-2/>

多年来，人们一直试图开发一种负担得起的、RepRap 衍生的 3D 打印机，这种打印机可以用金属制造物体。人们已经对高功率激光器和电子束等疯狂的设备做了很多工作，但迄今为止还没有人开发出一种可以轻松、廉价和安全地打印金属物体的机器。对于 Hackaday 奖，[【萨加尔】对他的金属 3D 打印机](http://hackaday.io/project/998-Affordable-Metal-3d-Printer)采取了不同的策略:他挤压低温合金，就像普通的 3D 打印机挤压塑料一样。

[Sagar]的打印机几乎是众多“纯塑料”3D 打印机的翻版，唯一的变化是挤出机和热端。作为一种材料，他在一个 3 毫米直径的线轴上使用了 95.8%锡，4%铜和 0.2%银的合金。这种合金的熔点为 235 摄氏度，大约与这些打印机通常使用的 ABS 塑料的温度相同。

唯一真正的问题是挤出机和喷嘴。[Sagar]正在用不锈钢铣削自己的喷嘴和热端；这是一项具有挑战性的加工，但仍在业余爱好者的范围内。他对 RepRap 衍生的塑料齿轮挤出机是否能够处理金属有些怀疑，所以他也在考虑设计一种新的版本，并用不锈钢进行铣削。

这是一个令人敬畏的项目，我们希望我们将很快看到这个项目的一些更新。虽然用低温合金生产物体的 3D 打印机短期内不会制造火箭发动机，但它可能是在家里制造一些合理的高强度零件的好方法。
