# 又一架 PCB 四轴飞行器

> 原文:[https://hackaday.com/2012/06/01/yet-another-pcb-quadcopter/](https://hackaday.com/2012/06/01/yet-another-pcb-quadcopter/)

![](../Images/78b2b9b7ce9488658d6a47dc24ac0681.png "pcbquad")

像大多数人一样，我们被这些嗡嗡作响、盘旋的四旋翼飞行器迷住了。我们真的很喜欢最近的小型化趋势，并使用 PCB 作为框架本身。正如[scolton]在他的说明书中指出的，他的不是第一个，最小的，或者最轻的，但是[他确实有一些很棒的信息](http://www.instructables.com/id/PCB-Quadrotor-Brushless/)和一些锦囊妙计，使这个项目成为一个有趣的检验项目。4pcb 从电机轴到电机轴的对角线长度为 6.5 英寸。它重 138 克，使用目前的电池，可以飞行大约 8 分钟。

4pcb 的一个优点是设计中包含无刷电机驱动器。显然这并不常见。不过，我们真正喜欢的是[斯科尔顿]提供的大量参考资料。你不仅可以下载重建他的作品所必需的一切，他还链接了许多其他项目和资源，你可以随心所欲地向任何方向扩展。如果你考虑在不久的将来建造一架四轴飞行器，你可能会想要收藏这一架。
