# 737 自动驾驶仪，由 Arduino 提供

> 原文：<https://hackaday.com/2013/07/10/737-autopilot-courtesy-of-an-arduino/>

![737](img/b859a9636557fa382d9e7a0286e957b8.png)

首先，*不*，我们没有看到一个实际的飞行硬件。这是安德里亚·朱迪奇(Andrea Giudici)的项目，将现实世界的硬件与飞行模拟器 X(T3)联系起来。这是模拟飞机的自动驾驶仪，所以那些想在不久的将来驾驶波音 737 的人不用担心电脑会驾驶你的飞机。不过，空客的乘客…

[Andrea]不想在 FSX 使用笨重的点击式界面，所以他创建了一个虚拟自动驾驶仪，配有一个 2×16 LCD 显示屏和一个 Arduino，用于交互和设置最常见的自动驾驶仪设置，如高度、速度、航向和接合。物理接口只有三个轻触开关和一个锅，而与 FSX 的接口是一个定制的驱动程序，可以将 Arduino 的 USB 转换为实际的飞行命令。

这不是车库里的 737 驾驶舱，但它仍然是在完全由计算机控制的界面中闲逛的一个很好的选择。
