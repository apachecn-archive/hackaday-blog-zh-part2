# 痛苦机器也带来快乐

> 原文:[https://hack aday . com/2014/09/30/pain-machine-bring-pleasure-too/](https://hackaday.com/2014/09/30/pain-machine-brings-pleasure-too/)

痛苦是件好事。它告诉我们把手从炉子上拿开，远离扭伤的脚踝。但是我们都有不同的疼痛体验，慢性疼痛会降低我们的生活质量。基于许多因素，一个人对疼痛的报告每天都会有所不同，因此 1-10 级在确定一个人的疼痛程度方面并不是普遍有效的。[Scott]的[参赛作品 hack aday Prize](http://hackaday.io/project/1860)基于经典的[冷加压测试](http://www.ncbi.nlm.nih.gov/pubmed/15820909)设备，该设备测量病人的手浸入冰水中一分钟时心率和血压的变化。

[斯科特]暂时将他的装置命名为疼痛机器，但它不仅仅是典型的冷加压装置；当阀门反转时，它还以温水的形式提供模拟疼痛缓解。此外，受测者可以在受够了的时候按下按钮。虽然他最初的计划使用外部冷热水源，但[Scott]从一些葡萄酒冷却器中取出了几个珀耳帖冷却器，以实现更具包容性的设计。

疼痛机器使用 Arduino ATMega 2560 来控制重力流螺线管，收集温度数据，并将数据发送到云端。几个 110 伏的水泵循环水。[Scott]将会在他完成注释和用用例充实代码后开放代码。现在，休息之后你可以看看他两分钟的参赛视频。
