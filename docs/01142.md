# 将蓝牙音频回放添加到丰田矩阵中

> 原文：<https://hackaday.com/2012/07/25/adding-bluetooth-audio-playback-to-a-toyota-matrix/>

![](img/a8fddd6e494c23ecb03f5d0d6ca0c1bb.png "adding-bluetooth-audio-to-toyota-matrix")

在这个项目中，[Ryan]展示了他如何将蓝牙音频添加到他的丰田 Matrix 的普通立体声系统中。他用附加硬件做的工作相当不错。安装出奇的简单。例如，挂在这张照片前景中的仪表板边框无需任何工具就可以轻松取下。此外，音响的 CD 换碟机输入是他用来接入系统的。它刚好有一个 0.1 英寸的插头，所以找到一个可以工作的连接器不是问题。

至于附加硬件，他围绕 ATmega168 微控制器和 Bluegiga WT32 蓝牙模块构建了自己的电路板。为了连接到汽车的数据系统，他使用了一个 RS485 驱动芯片。这不是完全正确的部分，但对他的目的来说已经足够了。到目前为止，他可以让音频播放工作，并计划增加免提电话和显示音轨信息的支持。嘿，也许他甚至会[添加一些额外的基于 shake 的自动化](http://hackaday.com/2012/06/18/shake-phone-to-start-scooter/)；谁知道呢？
