# 绘制时间的钟

> 原文:[https://hackaday.com/2014/02/13/a-clock-that-plots-time/](https://hackaday.com/2014/02/13/a-clock-that-plots-time/)

[Johannes]刚刚给我们发了一个关于他的小绘图仪的提示，这个绘图仪可以绘制出当前时间。

[Johannes]小型时钟绘图仪使用干擦笔在一小块写字板上写出时间。该设计由三个小型 9g 伺服系统组成，其中一个用于将笔抬离书写表面，另外两个用于控制一对连接的 x 轴和 y 轴关节臂。

这个小机器人费力地抹去之前的时间，然后在它的位置上潦草地写下当前时间(精确到分钟)。

[Johannes]在创建这个项目时就考虑到了可攻击性，确保保持标准部件，并使代码和设计文件可用。构建的硬件可以是激光切割或 3D 打印的。Arduino 草图可以在 [GitHub](https://github.com/9a/plotclock/blob/master/plotclock.ino) 上找到，设计文件可以在 [Thingiverse](http://www.thingiverse.com/thing:248009) 上找到。在[纽伦堡的私人实验室页面](http://wiki.fablab-nuernberg.de/w/Ding:Plotclock) ( [翻译](http://translate.google.com/translate?sl=auto&tl=en&js=n&prev=_t&hl=en&ie=UTF-8&u=http%3A%2F%2Fwiki.fablab-nuernberg.de%2Fw%2FDing%3APlotclock))上有更详细的构建说明。
