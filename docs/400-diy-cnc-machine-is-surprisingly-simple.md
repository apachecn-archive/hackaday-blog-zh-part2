# 400 美元的 DIY 数控机床出奇的简单！

> 原文：<https://hackaday.com/2014/07/11/400-diy-cnc-machine-is-surprisingly-simple/>

一旦你去了 CNC，你就再也回不去了——它们实在是太方便了！[Drez20001]向我们展示了他如何用大约 400 美元做了一个。当你可以使用抽屉滑轨时，谁还需要昂贵的滚柱轴承？

没错——这台数控机床的大部分成本是你真的不能再便宜的东西——伺服系统(或步进器)、皮带(或丝杠)，当然还有电机控制器和计算机接口。其他的呢？胶合板、抽屉滑轨和一堆紧固件几乎可以免费获得！

他多年来一直想建造一台 CNC，但由于成本和明显的复杂性，他在这样做时犹豫不决，但当他开始认真研究时，他发现事实并非如此！它基于基本的龙门系统设计，其中 X 轴驱动包含 Y 轴和 Z 轴的桥。无论如何，这不是一台重型机器(他现在只有一个类似 dremel 的小工具)，但对于他的目的来说，这已经足够了。

他节省了几美元的一个颇具创意的方法是使用他的马达耦合器——他实际上采用了橡胶输气管线，并将其紧固在两个轴上，他说这种方法效果很好！
