# 可印刷灯组件本身

> 原文:[https://hack aday . com/2014/06/07/printable-lamp-assemblies-self/](https://hackaday.com/2014/06/07/printable-lamp-assembles-itself/)

![Printable-Self-Assembling-Lamp](../Images/da38a805d752dddbce57705022431e5d.png)

随着 3D 打印机、可打印电路和其他“一机多用”自动化系统的出现，打印全功能项目正迅速成为现实。上图中的[灯最初是一块印有电路的平板纸板——通电后，它会自行折叠，形成一盏灯——它甚至有一个电容式触摸传感器来打开它！](http://spectrum.ieee.org/automaton/robotics/diy/harvard-self-folding-printable-lamp)

这种独创性直接来自哈佛大学的研究人员，他们在上周二举行的 IEEE 机器人和自动化国际会议(ICRA)上展示了“可打印机器的自组装传感器”项目。这不是完全可打印的(电线和 LED 必须手工焊接)，但这是一个惊人的概念证明——还有一个 Arduino 屏幕外的控制它。

纸板是形状记忆聚合物的夹层，由热量触发，通过薄铜迹层通电产生。这需要很长时间来组装，所以下面的视频已经加快了 32 倍的速度。
