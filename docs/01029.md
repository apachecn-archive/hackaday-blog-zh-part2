# 车库开门器现在是卧室关门器

> 原文:[https://hack aday . com/2012/07/13/车库开门器-现在是卧室关门器/](https://hackaday.com/2012/07/13/garage-door-opener-now-a-bedroom-door-closer/)

![](../Images/5e6c9c6fbf25c06a8673b129b1a13eee.png "garage-door-opener-closer")

[Roy]手头有一个额外的车库门开启器，他决定把它用作卧室门的遥控关闭装置。我们推测他有一些吵闹的室友，因为这个项目的灵感来自于当吵闹声打断他看电视时，他不想起床关门。

上图显示了铰链系统，该系统将车库开门器轨道的线性运动转化为将门关闭所需的旋转力。我们会说他真的做到了，因为这个系统与门框的角度完全匹配，当门完全打开时，角支架几乎贴着墙。我们当然没有关门的相同需求，但机制是要记住的。

开瓶器的马达藏在他的桌子下面。休息后你将无法在视频中看到它，因为他在它周围建立了一个匹配的围栏。现在，他只需要[添加一些 WiFi 连接](http://hackaday.com/2012/01/31/wifi-garage-door-opener-makes-forgotten-keys-a-thing-of-the-past/)，他就可以抛弃单任务射频遥控器，转而使用智能手机应用程序。
