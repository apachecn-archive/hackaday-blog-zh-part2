# 将角磨机用作台式研磨机

> 原文:[https://hack aday . com/2013/06/19/将角磨机用作台式研磨机/](https://hackaday.com/2013/06/19/using-an-angle-grinder-as-a-bench-grinder/)

![angle-grinder-bench-grinder](../Images/6d62b0cbca5b47f27aa576eb3117b030.png)

[Saeid Momtahan]制作了几个附件，让他能够将角磨机用作台式研磨机。这可能是更好地将其称为一个工作台电机，因为他在展示他的项目时使用了砂轮和钢丝刷。

附件分为两部分。第一个是一根方形的管子，在手柄的对面，与研磨机的主体平行。当使用它作为角磨机时，这是一个更大的抓握区域，也给他一些东西夹在他的台钳上。第二附件用作工件的支架。上图中你可以看到他正在用铁丝网刷一些生锈的木头。

拥有加倍工具任务的选项，而不是购买另一个可能不会经常使用的项目，这很好。我们也喜欢构建自己的工具的想法。如果你没有焊接设备来制造这些附件，这里有一个基于锂离子电池的系统来让你思考。
