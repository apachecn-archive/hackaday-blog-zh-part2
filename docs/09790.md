# DEF CON 的所有非官方电子徽章

> 原文:[https://hack aday . com/2015/08/10/all-the-official-electronic-badges-of-def-con/](https://hackaday.com/2015/08/10/all-the-unofficial-electronic-badges-of-def-con/)

2015 年是 DEF CON 23 非官方硬件徽章年。有一吨不同的硬件徽章设计的爱定制电子产品，我试图赶上每个不同的徽章的设计师。这是我这个周末看到的图片、视频演示和每一个细节的集合。

### 威士忌海盗

 [![DSC_0481](../Images/3af4c4dda877f97ac0a412d268667752.png "DSC_0481")](https://hackaday.com/2015/08/10/all-the-unofficial-electronic-badges-of-def-con/dsc_0481/)  [![DSC_0529](../Images/94c62f636e9cb4f477be9c5ddc361a15.png "DSC_0529")](https://hackaday.com/2015/08/10/all-the-unofficial-electronic-badges-of-def-con/dsc_0529/)  [![Gorgeous text treatment on back of this badge is indicative of [True's] mastery](../Images/e98aeb854fb712a31c294cb611ef8d6b.png "DSC_0528")](https://hackaday.com/2015/08/10/all-the-unofficial-electronic-badges-of-def-con/dsc_0528/) Gorgeous text treatment on back of this badge is indicative of [True’s] mastery [![DSC_0480](../Images/ce31e8a1b775fa4c8225c7fb719b48e6.png "DSC_0480")](https://hackaday.com/2015/08/10/all-the-unofficial-electronic-badges-of-def-con/dsc_0480/)  [![DSC_0477](../Images/09133cd2166e6d1fc95e31ed97cf7640.png "DSC_0477")](https://hackaday.com/2015/08/10/all-the-unofficial-electronic-badges-of-def-con/dsc_0477/)  [![DSC_0476](../Images/fece174c5aea647f7678409b0436cb38.png "DSC_0476")](https://hackaday.com/2015/08/10/all-the-unofficial-electronic-badges-of-def-con/dsc_0476/) 

[TrueControl]今年为威士忌海盗团设计了一个很棒的徽章。这是对[的重大更新，他去年设计的徽章](https://hackaday.com/2015/08/03/the-right-way-to-do-a-hacker-conference/)，保留了头骨和骨骼轮廓。它使用 PSOC4 芯片来控制一吨的 led。眼睛是 RGB 像素，每个都在自己的 PCB 上，焊接在徽章的背面，有开口供 LED 显示。两个 AA 电池为具有表面贴装 LED 矩阵的电路板供电。用户控制都是电容式触摸。一只眼睛周围有一个旋转器，以及用于选择和返回的衬垫。NRF24L01 无线电的工作频率为 2.4GHz。这个徽章是去年徽章上的命令的奴隶。当两者在同一区域时，2015 年徽章将滚动它“看到”的 2014 年徽章的昵称。压电元件还根据与不同徽章的相互作用发出许多不同的声音。

[正确]使设计成为一种艺术形式。哑光黑色阻焊膜看起来棒极了，他在字体、大小、对齐以及让铜露出来等方面都非常用心，打造出了一件令人惊叹的硬件艺术品。

继续阅读周末看到的十个更好的徽章。

### 密码徽章

 [![DSC_0526](../Images/1a537adfc346fc15c942daf4328afc44.png "DSC_0526")](https://hackaday.com/2015/08/10/all-the-unofficial-electronic-badges-of-def-con/dsc_0526/)  [![DSC_0486](../Images/475de115776f4546c9672ea37858ba52.png "DSC_0486")](https://hackaday.com/2015/08/10/all-the-unofficial-electronic-badges-of-def-con/dsc_0486/)  [![DSC_0490](../Images/05d80b92932f10fd7b581464af3482c0.png "DSC_0490")](https://hackaday.com/2015/08/10/all-the-unofficial-electronic-badges-of-def-con/dsc_0490/)  [![DSC_0488](../Images/b45efd985776a87ef76904bfed872ad5.png "DSC_0488")](https://hackaday.com/2015/08/10/all-the-unofficial-electronic-badges-of-def-con/dsc_0488/) 

加密徽章以 25 美元的价格出售。它有 12 个 led 环绕在圆圈周围，有两个 4 字符气泡显示器和两个按钮作为用户界面。当你把它插上电源时，不会发生什么事情。你需要在界面中输入单词和代码，才能获得徽章来做事。除了那个神秘的接口，在板子的背面还有一个密码，在挂绳上也有一个。

基于 MSP430g2955 的电路板的固件由[ [Karl Koscher](https://twitter.com/supersat) ]编写，硬件由[ [Jorge Lacoste](https://twitter.com/lacosteaef) 设计。他们向我展示的最酷的惊喜之一是一种将徽章变成超低功率调幅发射机的模式。把它放在收音机上，你就会听到俄罗斯方块的歌曲在播放！

[https://www.youtube.com/embed/jwpgpRm15kM?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/jwpgpRm15kM?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)T2】

### 汽车黑客徽章

 [![DSC_0483](../Images/1a718fb76ceaf7a8c87f7b510fb8c184.png "DSC_0483")](https://hackaday.com/2015/08/10/all-the-unofficial-electronic-badges-of-def-con/dsc_0483/)  [![DSC_0484](../Images/11e2ffd8813f00f5007d2245f73d3e83.png "DSC_0484")](https://hackaday.com/2015/08/10/all-the-unofficial-electronic-badges-of-def-con/dsc_0484/)  [![DSC_0485](../Images/e2343fde3a83e81f75f0aa1bb389ae0e.png "DSC_0485")](https://hackaday.com/2015/08/10/all-the-unofficial-electronic-badges-of-def-con/dsc_0485/) 

该徽章可在汽车黑客村购买，包括一个 ODB2 插头。这组接头和跳线允许用户选择不同的 CAN 总线连接(低速、高速、单线)。STM32 皮质-M0 芯片驱动该设备，并用卒语言编写脚本。

### DC801

 [![DSC_0453](../Images/c5298f3b8eea660778e98afac43e3462.png "DSC_0453")](https://hackaday.com/2015/08/10/all-the-unofficial-electronic-badges-of-def-con/dsc_0453/)  [![DSC_0456](../Images/2aa24e148519f8ef82424b031523cc63.png "DSC_0456")](https://hackaday.com/2015/08/10/all-the-unofficial-electronic-badges-of-def-con/dsc_0456/)  [![DSC_0455](../Images/4fccb40987e73820fdbaa874aabdcf11.png "DSC_0455")](https://hackaday.com/2015/08/10/all-the-unofficial-electronic-badges-of-def-con/dsc_0455/)  [![DSC_0457](../Images/9e1bf674395fb32c839a971ad75c1ed9.png "DSC_0457")](https://hackaday.com/2015/08/10/all-the-unofficial-electronic-badges-of-def-con/dsc_0457/)  [![Mike Szczys with some of the DC801 crew](../Images/a67b42ad50242a0a3c914980d548c6a3.png "DSC_0463")](https://hackaday.com/2015/08/10/all-the-unofficial-electronic-badges-of-def-con/dsc_0463/) Mike Szczys with some of the DC801 crew

来自盐湖城的 DC801 机组人员有一个巨大的徽章，装在一个数控铣削的外壳中。它的两侧都有 RGB LED 条，并配有一个巨大的 LCD 显示屏，两个模拟棒，两个按钮和一个旋转编码器。他们每年都做预售，这个徽章可以让你进入他们的派对

有一个螺旋桨芯片运行屏幕和 XBee 收音机的无线插座。ATmega328p 处理模拟棒、旋转编码器和按钮。两个黑客空间([晶体管](http://thetransistor.com/)和 [801 实验室](https://www.801labs.org/))中的 10-15 个人在这个项目上工作，前者处理大部分硬件，后者组织团队在展会上亮相的后勤工作

液晶显示器供应商出了点问题。屏幕到货时使用了错误的芯片组，无法及时投入使用。这将是固定的，因为单元中的 FTDI 芯片有一个总线选择器开关，允许 Prop 和 AVR 的固件更新。

### DC503

 [![DSC_0506](../Images/faf463d69aee492254e0d10e14a59962.png "DSC_0506")](https://hackaday.com/2015/08/10/all-the-unofficial-electronic-badges-of-def-con/dsc_0506/)  [![DSC_0519](../Images/1f54fc190c9450d6e8bf35336a6cf18b.png "DSC_0519")](https://hackaday.com/2015/08/10/all-the-unofficial-electronic-badges-of-def-con/dsc_0519/)  [![DSC_0520](../Images/c692ad3fb7b798dbc5164dc07ea67640.png "DSC_0520")](https://hackaday.com/2015/08/10/all-the-unofficial-electronic-badges-of-def-con/dsc_0520/)  [![DSC_0517](../Images/3ae90dd3d07b7b390e4d1031b5a233a5.png "DSC_0517")](https://hackaday.com/2015/08/10/all-the-unofficial-electronic-badges-of-def-con/dsc_0517/) 

DC503 徽章的形状像一辆自行车，创作者[Joe Fitz] ( [@securelyfitz](http://twitter.com/securelyfitz) )在设计中很好地运用了这一主题。板上的 14 个 led 安装在背面，通过基板照射到前面。ATtiny85 使用两个 PWM 通道驱动 led。该设置将 4 个 led 从 VCC 驱动到 PWM1，另外 4 个从 PWM1 驱动到 GND。它们被反相驱动，另一个 PWM 引脚上使用相同的技巧。控制由电容按钮提供。齿轮上的一个加速旋转(就像你在踩踏板)，把手(刹车的地方)减速。

这是你加入 DC503 派对的通行证。这些徽章是由众筹支持者获得的。纺 100 块板；他们为支持者组装了 50 个，为组织者现场组装了另外 10 个。DC503 是波特兰的一群朋友，他们中的许多人都参与了 CTRL-H 黑客空间。

### DEFCON DarkNet 徽章

 [![DSC_0524](../Images/dd711ce0da71e2e0402adf4b0449b486.png "DSC_0524")](https://hackaday.com/2015/08/10/all-the-unofficial-electronic-badges-of-def-con/dsc_0524/)  [![DSC_0525](../Images/b08bebaf5c9e125de8f20f3a255b9fa9.png "DSC_0525")](https://hackaday.com/2015/08/10/all-the-unofficial-electronic-badges-of-def-con/dsc_0525/)  [![DEF CON DarkNet Badge](../Images/e5021428ff99b5f64f841cddd8f59432.png "IMG_20150808_165515")](https://hackaday.com/2015/08/10/all-the-unofficial-electronic-badges-of-def-con/img_20150808_165515/) DEF CON DarkNet Badge

这不是真正的“非官方”，暗网徽章已经成为每年 DEF CON 的热门项目。去年我无法得到一个，今年我只是勉强得到一个。它以 25 美元一套的价格出售，当我周五下午试图购买时，它们已经卖光了。周六早上又卖出了 200 台，我等了大约 90 分钟，可能排在第 150 位。

徽章是由[Smitty]和[Krux](他是 Hackaday 的好朋友)设计的。它基于 Arduino，使用 IR 与其他徽章通信。你找到一个戴着组装好的徽章的人，让他们互相对着看几秒钟，交换徽章标识符。这是 DarkNet challenge 的一部分，鼓励与会者在 con 上解决谜题并尝试不同的活动。今年的设计与去年非常相似，所以[看看那个 repo](https://github.com/thedarknet/hhvkit) 了解更多信息。

### 杂项徽章

 [![PipMan](../Images/c939f8fe018dbc9221c3f114ee5bb2ae.png "IMG_20150809_220553")](https://hackaday.com/2015/08/10/all-the-unofficial-electronic-badges-of-def-con/img_20150809_220553/) PipMan [![NSL Cylon Soldering Kit](../Images/47d5f92c5666559e4b2408e3f7f5803e.png "IMG_20150809_220546")](https://hackaday.com/2015/08/10/all-the-unofficial-electronic-badges-of-def-con/img_20150809_220546/) NSL Cylon Soldering Kit

有一吨各种各样的硬件产品，不可否认，我没有得到他们中的很多照片。上面你可以看到[ [Christian](https://hackaday.io/hacker/24619-christian) ]在大会上穿的两件。 [PipMan](https://hackaday.io/project/2503-pipman-gps-watch) 是 Hackaday.io 上的一个项目。它的特点是一个彩色屏幕有机发光二极管屏幕，螺旋桨驱动的可穿戴设备挂在一个皮革腕带上。它显示时间使用 GPS 同步保持准确，包括温度，指南针，和一些辅助功能。他还戴着一个 [NSL 赛昂](http://wiki.032.la/cylon_iii) v2，这是一个用于表面贴装焊接练习的套件。
