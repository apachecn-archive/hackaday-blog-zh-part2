# Thurber Feeder 5000 有助于降低 Fido 的速度

> 原文：<https://hackaday.com/2014/09/24/the-thurber-feeder-5000-helps-to-slow-fido-down/>

你的狗吃得太快吗？[瑟伯]确实如此，他甚至在吃狗粮时偶尔会窒息——这自然会让他的主人担心，所以[杰森]挺身而出，挑战让他慢下来。介绍 [Thurber Feeder 5000。](https://www.youtube.com/watch?v=Hnxe4OCr2Ps)

[Jason]是一个经验丰富的制造商，在他的日子里建造了一些数控机器——他甚至用步进电机自动蚀刻草图。制作 Thurber Feeder 5000 简直易如反掌。他用 3D CAD 设计了整个东西，然后用他自制的 CNC 机器切割出所有零件，3D 打印出一些更复杂的安装支架。

这是一个相当简单的设备，由一个食物漏斗(可以密封以防止瑟伯靠近)、一个步进电机和一个从巧克力火锅喷泉借来的螺旋钻头组成。步进机通过 6:1 的皮带轮比，这给了它巨大的 200 盎司-英寸的扭矩来推动这些粗颗粒和小块通过饲管。速度可以通过编程视差推进器来调整，所以一旦他们找到一个可接受的进食速度[Jason]就将其设置为默认。一个简单的按钮就可以打开它，当机器运行时，它就会亮起来——当小瑟伯完成时，它就会熄灭。
