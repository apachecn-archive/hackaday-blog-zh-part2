# RPi 的《我的世界》发布

> 原文:[https://hackaday.com/2013/02/11/minecraft-for-rpi-released/](https://hackaday.com/2013/02/11/minecraft-for-rpi-released/)

[![Minecraft for RPi](../Images/b3c9bcca4902b4e59e0847947d968627.png)T2】](http://hackaday.com/2013/02/11/minecraft-for-rpi-released/minepi-2/)

《我的世界》背后的人 Mojang 已经正式发布了[《《我的世界》:圆周率》版本](http://mojang.com/2013/02/minecraft-pi-edition-is-available-for-download/ "Minecraft: Pi Edition Available for Download")。这款流行游戏的免费版本经过优化，可以在 Raspberry Pi 硬件上运行，并且有一个 API，可以将游戏的内部结构暴露给各种编程语言。

Mojang 希望这个版本成为教和学编程的教育工具。由于 API 在游戏中提供即时反馈，这可能是一种有趣的方式，让所有年龄的人都可以学习编码。

访问 RPi 上的 API 也意味着游戏可以连接到现实世界。例如，使用 Python 和 [RPi。GPIO](http://pypi.python.org/pypi/RPi.GPIO "RPi.GPIO") ，GPIO 头上的引脚可用于输出或输入。这就产生了一系列与游戏相关的可能的黑客攻击。

你想和《我的世界》一起在 RPi 上做什么？请在评论中告诉我们。此外，我们碰巧在 3d 打印机上打印了一个 minecraft pick。休息之后有一段延时视频！
