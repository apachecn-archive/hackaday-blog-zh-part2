# 从臀部焊接

> 原文：<https://hackaday.com/2012/05/25/soldering-from-the-hip/>

![](img/4cbd9d7f6de89da71d2f76150936c309.png "solder-gun-at-your-hip")

你可以成为这些地方的治安官，但前提是你要有一把焊枪和一个皮套。[米卡龙的]最新建造当然很有趣，即使它不那么实用。我们不会很快放弃我们的 Weller knock-off，但这种快速热重新包装肯定会成为您下一次 Hackerspace 活动的对话起点。

构建的商业端取自一个便宜的四节电池烙铁。[Mike]将每个组件分开，然后抓起一把玩具枪，看看每个组件可能适合的位置。电池的大小正好适合枪的弹匣。他要做的就是给枪加上触点，给弹匣装上弹簧。瞬时按钮开关位于触发器后面，用于将电池组连接到焊接头。

休息后，你会发现一点点过度建模，和一些焊料融化。这将与你制作的那款[9 毫米蓝牙耳机](http://hackaday.com/2008/07/12/igiveup-bluetooth-handset/)完美搭配。别把错的塞耳朵里就行。
