# RGB 自行车轮圈灯

> 原文：<https://hackaday.com/2014/12/26/rgb-bike-rim-lights/>

[Yvo]向我们发送了他的最新创作，这款令人敬畏的 [POV RGB 自行车轮圈灯构建](http://www.instructables.com/id/RGB-Bicycle-Rim-Lights/)，其特点是普通 RGB LEDs 沿轮圈向外呈圆形交织，因为它们根据车轮的转速显示不断变化的动画。

像许多 POV 车轮一样，[Yvo]利用霍尔效应传感器和固定磁铁来确定车轮旋转的速度。不同于大多数 [视点](http://hackaday.com/2009/06/11/rgbike-pov/) [构建](http://hackaday.com/2008/11/15/spoke-o-dometer-bicycle-pov-speed-display/)，然而【Yvo】的创作不仅仅是将一两根 RGB 棒夹在辐条上。相反，他的车轮拥有几个单独的 RGB LED 模块安装在轮辋上。

每个轮子有六个模块，每个模块都包含一个临时构建的 LED 控制器(74HC595 移位寄存器的菊花链),它可以装入定制的 3D 打印外壳中。外壳与 RGB LEDs 一起安装在一些铝条上，铝条跨在轮辋上安装在车轮上。

在速度下，灯光进入 POV 模式以模拟前部为白色、后部为红色的前灯/刹车。在下面的视频中，看看这些定制的圆形模块在骑行和休息时的不同之处。
