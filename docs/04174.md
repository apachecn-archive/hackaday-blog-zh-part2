# 试验板俄罗斯方块是电线艺术品

> 原文:[https://hack aday . com/2013/09/28/bread board-Tetris-is-wire-artwork/](https://hackaday.com/2013/09/28/breadboard-tetris-is-wire-artwork/)

![RGB-module-breadboard-tetris](../Images/a80395c82db30bcaffd7ed99e2bb67bb.png)

仔细观察这对 8×8 RGB LED 模块上的彩色像素，你将能够挑出一些熟悉的俄罗斯方块形状。令人印象深刻的是[李佳楠] [制作了他自己的彩色俄罗斯方块，包括主题音乐](http://jiananli.wordpress.com/2013/09/23/now-it-plays-the-tetris-theme-song/)，但是看看这个试验板！他的电路布局和他编写的让游戏开始运行的代码一样令人印象深刻。将如此复杂的电路放入他使用的空间需要相当多的计划，对吗？

有两个微控制器在工作，每个都运行 Arduino 引导加载程序。主芯片是 ATmega328，负责监控按钮和控制游戏。另一个是 ATmega85。八针芯片听它的哥哥，当游戏开始时播放主题曲，暂停或恢复以匹配用户输入，所以这个项目的下一站是[在建筑物的一侧玩俄罗斯方块](http://hackaday.com/2012/04/23/mit-students-take-tetris-to-a-grand-scale/)？

休息之后不要错过演示视频。我们还推出了他基于 Arduino 的钢琴的视频。它建立在一个几乎和这个一样令人印象深刻的试验板上。但让我们高兴的是他在双八度触觉开关键盘上演奏口袋妖怪主题的技巧。很明显，他父母为他支付的钢琴课真的很值得！
