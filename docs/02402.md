# 一台更加无用的机器

> 原文:[https://hack aday . com/2012/12/22/an-even-more-used-machine/](https://hackaday.com/2012/12/22/an-even-more-useless-machine/)

这是我们迄今为止见过的最没用的机器。它来自于[for4x]的车间，当它的八个开关中的任何一个被轻弹到 on 位置时，它就会自动关闭。

当[forn]的佳能 850i 打印机因为打印头损坏而放弃时，构建就开始了。所有其他的电子设备和机械设备仍然是可回收的，所以决定把这台打印机变成更没用的东西。

打印机使用带有光学编码器的常规 DC 马达来移动打印头。[for]很容易找到这个光学传感器的原理图，因为 TTL 输出能够读出滑块的位置。其余的构建是一个 ATMega8，一个伺服，和一个八位拨动开关。[forn]已经能够将伺服控制臂的精度降低到大约 0.1 毫米，足以精确地关闭所有开关。
