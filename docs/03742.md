# 声音球随着你的音乐颠簸

> 原文:[https://hack aday . com/2013/08/07/sound ball-bumps-to-your-tunes/](https://hackaday.com/2013/08/07/soundball-bumps-to-your-tunes/)

![soundball-blinks-to-the-music](../Images/0811c7855002735ecf6a3f1e0a62f789.png)

认识一下 soundball，这是一个业余电子项目，当时[用一个发光二极管](http://www.jarzebski.pl/blog/2013/08/04/soundball-equalizer-i-oswietlenie-relaksacyjne.html) ( [翻译为](http://translate.google.com/translate?sl=auto&tl=en&js=n&prev=_t&hl=en&ie=UTF-8&u=http%3A%2F%2Fwww.jarzebski.pl%2Fblog%2F2013%2F08%2F04%2Fsoundball-equalizer-i-oswietlenie-relaksacyjne.html))制成的球代替了一个迪斯科球。此图显示了注入外壳之前的设备。最后提供的是一个白色的项目箱，顶部有一个孔，通过扩散器覆盖的 blinky 球得到支持。

主板上有一系列常见的疑点:一个 ATmega328、一个 MSGEQ7 均衡器、几个 TLC5940 LED 驱动器和一个蓝牙屏蔽。均衡器芯片为[科尼利厄斯]提供音频分析，用于生成伴随音乐的光模式。但他仍然可以通过外壳上的按钮或通过蓝牙连接来手动控制灯光。

把 LED 驱动器换成固态继电器，你就可以随着音乐闪烁圣诞彩灯了。
