# 克鲁格的 Zippo 遥控器

> 原文:[https://hackaday.com/2014/08/23/krugers-zippo-remote/](https://hackaday.com/2014/08/23/krugers-zippo-remote/)

![FWHLN99HZ2IV5DF.MEDIUM](../Images/b9035482a5512eabc2d7aa2a91b08647.png)

受《美国队长:第一部复仇者联盟》电影中雷管的启发，[乔恩] [改装了一个普通的 Zippo 打火机，以激活接收器模块上的继电器](http://www.instructables.com/id/Krugers-Zippo-Remote/?ALLSTEPS)。他的 instructables 展示了如何通过调整插件来创建这样一个设备，如果有人把它打开，他们只会看到一个打火石轮、打火石、灯芯和所有这些东西；没有什么是不正常的。为了做到这一点，部件必须完全隐藏起来。

为了获取远程信号，[Jon]使用整个金属外壳作为天线，而不是用一个替换灯芯。将射频模块上的天线引脚连接到插件上，以获得必要的效果。然后火石轮变成了一个按钮，并安装了一个通知 LED。一旦代码被上传，接收器模块被组装起来，最终产品在另一端产生了火花。

这个黑客是为教育使用，只是为了演示的目的。
