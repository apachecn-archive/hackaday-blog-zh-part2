# 带宜家鸡蛋定时器的 GoPro 平移延时

> 原文:[https://hack aday . com/2013/06/04/go pro-panning-time-lapse-with-IKEA-egg-timer/](https://hackaday.com/2013/06/04/gopro-panning-time-lapse-with-ikea-egg-timer/)

![ikea-go-pro-time-lapse](../Images/68cb45cbabb2eaa7fcd4370c00087c9d.png)

[塞巴斯蒂安·舒斯特的]周末项目是将他的 GoPro 相机变成一个平移延时装置。你会注意到它在一个防水的盒子里，因为他的黑客演示是在户外进行的，现在天气不是很好。多亏了宜家鸡蛋计时器[和 3D 打印相机支架](http://www.zipfelmaus.com/blog/3d-print-job-gopro-ikea-kitchen-timer-mount/)，他快速、轻松、廉价地将这些组装起来。

鸡蛋计时器是淘洗黑客的一个流行选择。任何类型，包括一个在水平轴上旋转的表盘都可以。Ikea Stam egg timer 具有凸起的手柄，3D 打印部件可以轻松抓握。你可以通过一个对我们来说是新的网络服务来获得设计文件。[Sebastian]使用了[的 tinker CAD](https://tinkercad.com/)进行设计，并在上面链接的他的项目帖子中分享了它。

这只是他收集的相机黑客中的又一个工具。几年前，我们研究了他建造的电动云台。
