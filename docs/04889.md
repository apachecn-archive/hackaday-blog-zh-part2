# Fubarino 竞赛:单 PCB Synth

> 原文:[https://hack aday . com/2013/12/23/fuba Rino-contest-single-PCB-synth/](https://hackaday.com/2013/12/23/fubarino-contest-single-pcb-synth/)

![fubarino-contest-synth](../Images/7133c4c24a520160964dcfbdc8c583b8.png)

这里有一个参加 Fubarino 比赛的音乐作品,结果相当令人愉快。首先，[Mats]展示了他的音乐知识，用 H、A 和 D 音符弹奏打开复活节彩蛋的和弦。你说什么？你的键盘上没有 H？你错了。在德国音乐传统中，B 自然音被称为 h。这是[允许巴赫写一首曲子拼写他的名字](http://en.wikipedia.org/wiki/BACH_motif)。

[Mats]正在黑他的 PlingPlong 合成器。第一个复活节彩蛋——你可以在休息后的剪辑中看到——以较低八度的 H-A-D 和弦开始，在 7 段显示屏上拼出我们的 URL。但我们更喜欢第二个彩蛋，以高八度的和弦展开，如上图所示。它使用 3×5 的 LED 栅格滚动显示地址；在该静止图像中，显示了 H。
