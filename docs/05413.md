# 巨型俄罗斯方块为你的房间增添了一些复古气息

> 原文:[https://hack aday . com/2014/02/26/giant-Tetris-adds-some-retro-to-your room/](https://hackaday.com/2014/02/26/giant-tetris-adds-some-retro-to-your-room/)

![giant tetris](../Images/9e14980997865b739474324ca2ca1076.png)

[山姆]刚刚完成这个令人敬畏的 [6 英尺高的俄罗斯方块游戏](https://decibel.ni.com/content/docs/DOC-35435)使用国家仪器 [myRIO 与 FPGA。](http://www.ni.com/myrio/what-is/)

该建筑使用了由 myRIO 控制的 10 x 20 的 RGB LEDs 网格。只要你有 WebSockets 的支持，你就可以在任何设备上使用 web 界面来玩这个游戏。[Sam]最初是使用 Arduino 作为核心来构建它的，但是想要一个独立的设备来做所有的事情——不需要额外的计算机或用于 web 界面的 Raspberry Pi。就在那时，他发现了 myRIO——这是一款非常酷的硬件，除了最近的[毕加索彩弹枪项目……](http://hackaday.com/2014/01/31/paintball-picasso-purposely-misses-its-prey/)
