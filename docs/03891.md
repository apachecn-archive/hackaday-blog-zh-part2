# 追溯艺术:ANSI 和 ASCII 艺术的历史

> 原文：<https://hackaday.com/2013/08/20/retrotechtacular-the-history-of-ansi-and-ascii-art/>

这些幻灯片可能不是你所记得的 2400 波特调制解调器时代的字符艺术风格；他们比一开始的时候更先进。事实证明，这种艺术亚文化还有一些生命力。在本周的 Retrotechtacular 中，我们来看看今年 BSides 大会上 Doug Moore 关于 ANSI 和 ASCII 艺术的历史和生存的演讲。

ASCII 仍然是一种常见的字符编码，所以您可能已经熟悉它了。另一方面，ANSI 是一个相当令人困惑的术语，因为当提到字符集时，它已经在晦涩中消失了。在这种情况下，它指的是一组扩展字符，最好将其描述为 [Windows 代码页](http://en.wikipedia.org/wiki/Windows_code_pages)。

我们对 ANSI 艺术现场的大部分了解来自观看 [BBS:纪录片](http://www.bbsdocumentary.com/)(它在[我们的十大最佳黑客视频列表](http://hackaday.com/2013/04/25/the-10-best-hacking-videos))。我们当然记得以前连接到拨号 BBS 后看到的垂直滚动艺术。但是理解围绕《这就是艺术》的创作、捆绑和发行而形成的派别是令人着迷的。[道格]在报道这段历史方面做得很好，并列分享了避免“剽窃”其他艺术家作品的例子。这个图像实际上不是 rip。在他的演讲的后面，他讨论了亚文化的持续存在，展示了现代人对同一主题的看法。

如果你只是对技术感兴趣，下面视频的前半部分值得一看。但我们敢打赌，很难不继续到最后，顺便了解一下艺术史。
