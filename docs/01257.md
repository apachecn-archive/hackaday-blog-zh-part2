# 用巴别鱼学习一门新语言

> 原文:[https://hackaday . com/2012/08/06/用巴别塔鱼学习新语言/](https://hackaday.com/2012/08/06/learn-a-new-language-with-the-babel-fish/)

《银河系漫游指南》中的巴别鱼是宇宙中最奇怪的东西之一。将巴别鱼插入你的耳朵后，它以脑电波能量为食，并将意识频率的矩阵分泌到大脑的语言区域。作为一个通用翻译器，它是非常宝贵的，但在地球被定为毁灭目标之前，我们将不得不用阿达果公司[的巴别鱼语言玩具](http://learn.adafruit.com/babel-fish/overview)中的【贝基】来凑合。

[Becky]的巴别鱼仍然能够以语言释放的能量为食，但在这种情况下，能量来自一组写有汉字的 RFID 卡。在巴别鱼面前挥舞这些 RFID 闪存卡后，连接到 Arduino 的防波板会播放一段录音，记录闪存卡上的徽标发音时应该发出的声音。

虽然不是一条生物工程鱼同时证明和否定上帝的存在，但人类的每一项努力——包括学习语言——都需要更多的[道格拉斯·亚当斯]参考。休息之后你可以看看[Becky]的巴别鱼演示视频。
