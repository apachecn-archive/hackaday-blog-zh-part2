# 激光蚀刻表面重新定义干燥

> 原文：<https://hackaday.com/2015/01/22/laser-etched-surface-redefines-dry/>

就在几天前，我们站在厨房里做鸡蛋，怀疑地盯着煎锅中央的一道长长的划痕。随着社交媒体上的所有文章提醒我们警惕环境中可能会杀死我们的东西，包括特氟隆，我想知道人类是否会开始找到解决日常问题的方法……比如粘鸡蛋，它不涉及复杂化学物质的使用。唉，宇宙以不可思议的时机做出了回应。由罗切斯特大学光学研究所的[Chunlei Guo]领导的一组研究人员最近发表了他们开发的激光纹理化表面，这种表面像橡胶球一样排斥流体……不涉及任何化学处理。你真的需要在下面的视频中看到这种情况。

这种物理魔术的灵感来自大自然，模仿排斥水的生物如荷叶或蝴蝶翅膀的表面张力特性。为了达到类似的效果，一种精密的激光被用来在金属上蚀刻出纳米级的图案，这种图案以一种流体分子不喜欢粘附的方式改变了表面特性。将材料表面纹理化的好处是，图案成为表面结构的固有部分，不会像化学密封那样随时间褪色。这种疏水技术可以改善我们保持表面卫生的方式，也有助于新的防冻方法。更不用说其他几十个不太重要的应用程序，我们只是想到自己的娱乐。

除了创造疏水表面，光学研究所还采用了类似的策略，开发出一种能够吸收液体并克服重力快速将其向上运送的材料。凭借物理学知识和激光的力量，我们很高兴看到人类想出了更聪明的方法来操纵我们生活的世界，让我们的日常生活更加舒适。
