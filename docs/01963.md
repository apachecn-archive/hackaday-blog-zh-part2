# 电子恶魔服装令人惊讶地令人不安

> 原文:[https://hackaday . com/2012/10/25/电子恶魔-服装-令人惊讶地-令人不安/](https://hackaday.com/2012/10/25/electronic-demon-costume-is-surprisingly-unnerving/)

![](../Images/ce600f135a58a409ca2e850acf5a16c7.png "yeahAnimated")

[Phil]在 Adafruit 上周日的展示和讲述中，一件令人惊叹的恶魔服装，包括变声器和用于眼睛和嘴巴的动画 LED 矩阵。他刚刚上传了他是如何制作这套服装的，但是你真的应该去[看视频](http://www.youtube.com/watch?feature=player_embedded&v=8HZFY4Xti7k)看看这个造型有多棒。

每个恶魔都需要一个可怕的声音，所以[菲尔]在这个版本中重新使用了他的基于 Arduino 的变声器。通过旋转旋钮来调节恶魔声音的音调，[菲尔]从来自地狱深渊的咆哮变成了一个声音有点烦人的恶魔。

眼睛利用了[阿达果 I2C 的 LED 矩阵背包](http://learn.adafruit.com/animating-multiple-led-backpacks/overview)。眼睛连接到同一个 I2C 地址，以防止 derping，但三个红色口 LED 矩阵能够显示适合 8×24 LED 矩阵的任何内容。

这个建筑的电子部分安装在一块树脂玻璃上，而树脂玻璃又安装在一个面具上[Phil],面具是从工艺品商店买来的。考虑到万圣节商店现在全年都在营业，这并不是最好的选择，但它确实发挥了作用。

morph suit——一种氨纶紧身衣——与一些恶魔翅膀和角一起完成了这一造型。在 Adafruit 的展示和讲述过程中，[Phil]的桌子上到处都是电子零件。为了把它变成一套服装，他将在胸部安装一个小型电池驱动的扬声器，并把所有的电子设备塞进一个腰包。

这是一个非常非常酷的建筑，它真正提升了 Arduino 驱动服装的游戏性。休息之后请看视频。
