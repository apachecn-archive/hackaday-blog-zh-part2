# THP 参赛作品:Tinusaur AVR 平台教新手，玩生活游戏

> 原文:[https://hack aday . com/2014/07/29/THP-entry-tinu saur-AVR-platform-teaches-noobs-plays-game-of-life/](https://hackaday.com/2014/07/29/thp-entry-tinusaur-avr-platform-teaches-noobs-plays-game-of-life/)

![tinusaur](../Images/5d6f19c7c8bfcf46f44874bd0eab18a5.png)【Neven Boyanov】说 [Tinusaur](http://hackaday.io/project/2069-The-Tinusaur) 没有什么特别的，这个小小的平台学习和教授 AVR 编程的乐趣。但是如果你渴望对你的 Arduino 有更深入的了解，或者想教别人一些基础知识，你可能不同意这个评价。

Tinusaur 易于组装，仅包含 ATTiny13/25/45/85 操作所需的组件(该套件带有‘85)。[Neven]通过放弃 USB 电压调节器节省空间和内存。套件中包括一个可选的纽扣电池支架和跳线。

[Neven]正在通过 Tinusaur 网站销售主板和套件，或者您可以从一些第三方供应商那里获得主板。他的网站上有一些组装和驾驶 Tinusaur 的项目和有用的指南。他最近[给它编程，让它在 8×8 的 LED 矩阵上玩康威的生命游戏](http://hackaday.io/project/2159)。如果您正在寻找 AVR 游泳池的零入口端，您可以从 Arduino IDE 中对其进行编程。不过，要小心；他们并不完全兼容。
