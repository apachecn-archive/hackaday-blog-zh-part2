# 制造轨道炮

> 原文:[https://hackaday.com/2013/10/15/building-a-rail-gun/](https://hackaday.com/2013/10/15/building-a-rail-gun/)

[瓦伦丁]向我们透露了他的最新项目:自制轨道炮。对于少数可能还不知道的人来说，轨道炮是一种电动电磁射弹发射器。它由一对平行的导电轨组成，沿导电轨滑动的电枢受到电流电磁效应的加速，电流沿一条导电轨向下流入电枢，然后沿另一条导电轨返回。[Valentin]的文章从详细解释这一原理开始，然后展示了一个简单的概念证明，其中一个两端带有两个小圆磁铁的金属棒沿着由 9V 电池供电的两个铝条加速。

上面显示的最后一个构建由一个电容器组供电，该电容器组由三个 400V 2200uF 电容器并联组成。[Valentin]选择了热轨设计，电力总是存在于铁轨上。抛射体通过弹簧加载的杠杆插入组件中。插播一段视频。如果你觉得这很有趣，你会爱上[全自动高斯枪](http://hackaday.com/2013/08/07/a-full-auto-gauss-gun-probably-wont-hurt-much/)。
