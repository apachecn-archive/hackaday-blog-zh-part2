# 用镍铬合金丝修复破损的塑料零件

> 原文:[https://hack aday . com/2013/02/05/using-nichrome-wire-to-repair-broken-plastic-parts/](https://hackaday.com/2013/02/05/using-nichrome-wire-to-repair-broken-plastic-parts/)

![nichrome-wire-plastic-repair](../Images/9be049efeca2b74fba35835d5c9300b0.png)

当注射成型的塑料部件破裂时，这真是一件令人沮丧的事情。我们从来没有发现一种胶合技术可以像相机三脚架上的夹子那样对受力的零件起作用。但是(马蒂亚斯·万德尔)可能会有所发现。这是他在用镍铬合金线加固破损的塑料部件。

休息后的视频中完整演示了修复过程。他从坏掉的吹风机的加热元件上清理电线。想法是将电线缠绕在断片上，然后从工作台电源供电。这加热了电线，然后电线可以被拉到塑料表面之下。[马提亚斯]把它比作在混凝土中使用钢筋。

他的实现还可以改进一点。让电线均匀嵌入是一个问题，但使用钳子而不是鳄鱼夹可能会产生更好的结果。
