# 参数化扬声器外壳的 3D 打印

> 原文:[https://hack aday . com/2014/09/30/3d-printing-of-parameterized-speaker-enclosures/](https://hackaday.com/2014/09/30/3d-printing-of-parameterized-speaker-enclosures/)

尽管你会从本田思域掀背车后面一堆乱七八糟的电线、地毯和中密度纤维板中收集到一些信息，但制造扬声器音箱是一件相当复杂的事情。为了获得正确的频率响应，您需要考虑驱动器的谐振频率、任何内部组件的音量以及扬声器在达到谐振频率时的工作情况。令人兴奋的东西，但当 NothingLabs 的[Rich]开始 3D 打印他自己的扬声器外壳时，[他意识到他可以自动计算出理想的外壳。啊，OpenSCAD 的乐趣。](http://nothinglabs.blogspot.com/2014/09/speakergen-parametric-3d-printed.html)

[Rich]写了一点 OpenSCAD，并把它放在 Thingiverse 定制器上[，允许任何人手动输入盒子体积、高度和宽度比、扬声器孔的大小，甚至低音端口。](http://www.thingiverse.com/apps/customizer/run?thing_id=451253)

这种构造音箱的方式有一些非常酷的功能；组装很容易，而且从打印机出来后很可能是气密的。[Rich]打印了一个 3 英寸驱动器的外壳，其频率响应低至 66Hz，令人印象深刻。下面视频。
