# 精确度惊人的 3D 打印机使用 DLP 投影仪

> 原文:[https://hack aday . com/2012/04/05/3d-printer-with-anazy-accuracy-uses-a-DLP-projector/](https://hackaday.com/2012/04/05/3d-printer-with-insane-accuracy-uses-a-dlp-projector/)

![](../Images/9f2380bcf1f545072e1b608cceba25c4.png "printer")

经过多年的努力，【小贝洛索】终于将他的 3D 打印机项目公之于众。与我们通常看到的 Makerbots 和 repraps 不同，[Junior]的打印机使用光固化树脂和 DLP 投影仪来构建具有难以置信的精细细节的对象。

[Junior]项目的一个亮点是开发低成本树脂。通常情况下，光固化树脂非常昂贵，但是[Junior]正在努力将树脂的价格降到每公斤 150 美元。一个快速的逆向计算告诉我们，用一公斤树脂可以打印大约 7-800 立方厘米。它比其他 3D 打印机使用的塑料细丝贵得多，但这是你为质量付出的代价。

有一个非常受欢迎的 [Indiegogo 活动](http://www.indiegogo.com/veloso3dprinter)试图筹集资金来大规模生产树脂和这个套件的一些组件。我们对这次活动的回报没有什么印象——59 美元购买一份没有任何尺寸的. PDF 格式的打印机说明，159 美元购买一份 BOM、尺寸和配方来制作自己的树脂，400 美元购买封闭源代码软件[Junior]devleoped——但希望 Indiegogo 能把便宜的树脂推向市场。关于这款打印机有一个[简短的常见问题解答](http://www.wix.com/3dprinterfaq/veloso3d#!kit|mainPage)，所以我们会让读者在下面的评论中机智地讨论这款打印机的优点。
