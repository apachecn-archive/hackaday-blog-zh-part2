# 数控禅宗花园

> 原文：<https://hackaday.com/2014/05/31/cnc-zen-garden/>

[https://www.youtube.com/embed/Ge4H4Uw630o?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/Ge4H4Uw630o?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

【大卫】第二版[遇见沙美甲数控机床](https://www.youtube.com/watch?v=Ge4H4Uw630o)。大约六个月前，我们看到了[的第一版](http://hackaday.com/2013/12/08/four-cable-drawing-machine-pulls-our-strings/)，它是他为加拿大的一个科学博物馆建造的。除了控制器之外，该产品基本相同。最初的版本需要一个完整的计算机来驱动它，但现在它已经被替换成了一个比格犬骨黑色。

软件对绘图仪的位置没有反馈，绘图仪是[大卫]在[卡尔加里原型空间](http://protospace.ca/)加工的铝块。当机器启动时，它需要在一个特定的位置，从那里通过计算四根线的缠绕或退绕量来跟踪图案。

在[大卫的]网页上还有一堆其他[非常棒的艺术装置和项目](http://davidbynoe.com)，值得点击浏览！
