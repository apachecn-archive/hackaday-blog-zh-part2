# 斯托克监控市场

> 原文:[https://hack aday . com/2013/01/24/stocker-monitors-the-markets/](https://hackaday.com/2013/01/24/stocker-monitors-the-markets/)

![stocker-monitors-the-market](../Images/69be152782eadb4434c8b8e92e5a8dfc.png)

如果你忍不住花一天时间查看你的股票价格，这个环境设备可以帮助你应对。它[监控交易的进展，并点亮一个 LED 作为反馈](http://alirezakohani.com/stocker/)。这里苹果股票正在上涨，所以绿灯亮了。休息后的视频显示其他股票交易下跌，导致它切换到红色。

Arduino 通过 USB 与自定义应用程序连接。目前看来，这两种颜色是它所能做到的，但我们认为还有更大的潜力。一些创造性的编码可以使用诸如股票波动幅度、交易量、波动性或其他大量数据等因素来提供反馈。我们可以看到用来提高反馈水平的一系列颜色(就像在温度图上一样)。如果市场真的下跌，总有能力增加闪光！

这个项目的扩散器对我们来说很有趣。[阿里·雷萨·科哈尼]用剩下的丙烯酸废料做的。气泡表面是在将板材弯曲成弧形之前用热风枪产生的。
