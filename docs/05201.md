# 巧妙的迷你火柴枪

> 原文：<https://hackaday.com/2014/02/02/clever-mini-matchstick-gun/>

寻找一个快速有趣的小项目，你可以做什么呢？为什么不自己做一个火柴棍/牙签发射器呢！

[Grant Thompson ][随机之王]又开始了另一个有趣的项目，任何人都可以做——我们只是希望你对它负责！你所需要的只是一些晒衣夹、一把美工刀和一些弹药——我们喜欢燃烧的火柴！

通过在晒衣夹上切几个凹槽，将它粘在一起，并重新配置弹簧布局，你可以制作一个强大的迷你枪，可以射击 20 英尺以上。用一根尖尖的牙签也能非常有效地将无害的水果串起来！

为了看到它的运行并学习如何自己做一个，休息后留下来。
