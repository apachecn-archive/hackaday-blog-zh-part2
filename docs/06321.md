# 旋转式手机

> 原文:[https://hackaday.com/2014/06/16/the-rotary-cell-phone/](https://hackaday.com/2014/06/16/the-rotary-cell-phone/)

已经发生了。终于发生了。如今，旋转手机已经成为现实，潮人穿上他们的好法兰绒，打开他们为特殊场合准备的 Genesee 的盒子。

[Jaromir]创造了这款令人惊叹的复古未来设备，作为参加[恩智浦 LPC810 挑战赛](http://www.nxp.com/campaigns/lpc800-go/)的参赛作品，该比赛旨在利用 8 引脚封装的 ARM Cortex M0+微控制器发挥最大作用。只有六个 I/O 引脚来控制 GSM 模块，显示几个按钮，花哨的旋转转盘意味着[Jaromir]需要以某种方式扩展他的 I/O。他选择了一个移位寄存器来处理按钮和显示，这是一个令人印象深刻的演示，展示了如何将移位寄存器同时用作输入和输出扩展器。

从[Jaromir]上传的视频来看，旋转式手机还没有为 Think Geek 做好生产准备。他需要输入 SIM 卡的 PIN 码，GSM 模块的 AT 命令，当然，对于只在老电影中见过旋转式电话的年轻人来说，这是一种可怕的用户输入方法。也就是说，这是一部运行在 8 针微控制器上的旋转式手机。你还想要什么？
