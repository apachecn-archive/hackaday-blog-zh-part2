# 制作镍钛合金丝尺蠖

> 原文：<https://hackaday.com/2014/06/28/making-a-nitinol-wire-inchworm/>

[史蒂文]又做了一个很酷的科学实验，这个实验并不难。这一次，他给自己做了一个[镍钛合金丝尺蠖，](http://rimstar.org/science_electronics_projects/nitinol_wire_inchworm_shape_memory_alloy_homemade.htm)当你给它施加一个开关电流时，它实际上会在桌子上移动！

镍钛诺是一种形状记忆合金，具有冷却特性，使其在加热(或通电)时保持(并返回)预设形状。)被应用。这其实很简单——他将镍钛诺金属丝紧紧地缠绕在一个钉子上，然后加热使其成为线圈形状。现在镍钛诺弹簧可以伸直了，但是一旦加热，它就会试图恢复到盘绕状态！

他利用这种记忆效应，用一些轻木和其他一些零碎的东西制作了一条电动尺蠖——休息后看看吧！
