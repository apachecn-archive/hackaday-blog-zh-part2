# 黑客日十周年:奎因·邓基和维罗妮卡

> 原文:[https://hack aday . com/2014/10/05/hack aday-10 周年纪念-奎因-邓基-和-维罗妮卡/](https://hackaday.com/2014/10/05/hackaday-10th-anniversary-quinn-dunki-and-veronica/)

如果你在过去几周没有阅读 Hackaday，我们本周末在帕萨迪纳举办了一场令人惊叹的十周年派对，充满了研讨会、讲座和派对，据说派对一直持续到凌晨四点。我们邀请来演讲的令人惊叹的黑客之一是[奎因·邓基]，Veronica 的创造者，塞在旧收音机里的现代 6502 计算机。

我们第一次见到维罗妮卡是在几年前，但是奎恩认为她已经做了五年的电脑了。她的职业是软件开发人员，她决定有一天涉足硬件开发的黑暗海洋，从头开始建造一台计算机。她选择了 6502 作为她的装置的大脑，将所有东西都放在厨房蚀刻的单面板上，并用背板连接所有东西。现在它有一个 USB 键盘，(技术上是一个带 USB 插头的 PS/2 键盘)，NES 控制器，VGA 显示器，以及一个显示器和只读存储器。[Quinn]的目标是建造一台可以自我编程的计算机，五年后，她完成了这个目标。

[Quinn]承认她的软件背景是她一些公认的糟糕设计选择的原因；VGA 是由 ATMega 微控制器生成的，其工作原理是，如果她能够以足够快的速度为微控制器计时，她就可以实现 VGA。她现在认为 FPGA 本来是视频输出的更好选择，但现在视频电路已经完成，她可能不会重新考虑这个问题。

维罗妮卡还缺少一样东西，也是[奎因]未来将会努力的东西:海量存储。现在，维罗妮卡能运行的每一个程序不是存储在只读存储器中，就是通过键盘输入。硬盘是下一个要解决的问题，无论是 SD 卡，还是紧凑型闪存或 IDE 硬盘。
