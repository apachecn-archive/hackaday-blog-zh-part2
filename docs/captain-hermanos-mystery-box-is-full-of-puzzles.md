# 赫曼诺船长的神秘盒子里装满了谜题

> 原文：<https://hackaday.com/2014/12/28/captain-hermanos-mystery-box-is-full-of-puzzles/>

[Raffi]需要一个生日礼物创意，但他想做一些特别的东西。他意识到送礼的很大一部分是打开礼物的期待和兴奋。为了延长这种体验，【Raffi】搭建了一个[电子拼图盒。盒子里装着最后的礼物，但首先必须解开一系列谜题才能打开盒子。](http://en.code-bude.net/2014/12/17/captain-herrmanos-mystery-box-a-reverse-geocache/ "Electronic puzzle box")

该项目在 Arduino Mega 上运行。它与几个传感器相连，包括温度传感器、GPS 单元和一氧化碳传感器。还有一个 LCD 屏幕和数字键盘供用户输入和输出。项目页面包含一个流程图，显示了所有的难题及其解决方案。其中一个更有趣的谜题要求使用者将烟草烟雾吹入一根管子。一氧化碳传感器检测到烟雾并解开下一个谜题。

有些谜题需要与外部系统互动。例如，一个谜题要求用户向虚构的 Hermano 船长的电子邮件地址发送电子邮件。如果电子邮件中包含正确的关键字，用户将收到一封回复，其中包含要输入到框中的代码。另一个谜题要求用户拨打一个特定的电话号码，然后听另一个谜语。我们在下面提供了视频演示。

这不是我们看到的第一个谜题盒子，但是每一个都有它自己特殊的天赋。这个做得很好，看起来好像花了很多心思。我们已经看到了另一种仅使用[分立元件](http://hackaday.com/2014/05/14/electronic-puzzle-box-uses-only-discrete-components/ "Puzzle box")的产品。我们已经看到了另一个使用[莫尔斯电码](http://hackaday.com/2010/01/21/buzzle-a-morse-code-puzzle-box/ "Morse code puzzle box")的。
