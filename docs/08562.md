# DIY Hololens 用的是 Pepper 的装在盒子里的幽灵！

> 原文：<https://hackaday.com/2015/03/17/diy-hololens-uses-peppers-ghost-in-a-box/>

对于微软的 Hololens，DIY 社区过于兴奋，在真正的交易出来之前，他们已经跳过了制造一些硬件的挑战。[肖恩·霍尔]有一个出色的 3D 打印原型，它利用[佩珀的幽灵幻觉](http://hackaday.com/2013/10/06/peppers-ghost-halloween-ghosting/)为这副独特的虚拟现实护目镜创建了一个“全息图”。

与我们见过的其他 DIY 虚拟现实护目镜类似，[Sean]已经 3D 打印了外壳——但它不是在你的眼前拍打智能手机，而是安装在护目镜上方，从镜子和一块透明的有机玻璃反射，由于 Pepper 的幽灵幻觉的概念，它产生了类似全息图的效果。

这些基于反射的全息图的问题是它们并不总是那么容易被看到，所以[Sean]正计划尝试一些单向反射汽车色调，以获得更明显的反射，同时仍然能够看穿图像。他还计划用一些名为 [Project Haytham](http://sourceforge.net/projects/haytham/?source=navbar) 的开源软件增加视线追踪功能。这是一个使用 Kinect 的深度传感器，使用 Playstation Move 的头部跟踪，甚至可能是一个用于虚拟对象操作的 leap 运动控制器。
