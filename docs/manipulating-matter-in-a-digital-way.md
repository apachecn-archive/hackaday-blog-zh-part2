# 以数字方式操纵物质

> 原文：<https://hackaday.com/2015/06/26/manipulating-matter-in-a-digital-way/>

基本上，计算机的处理器是由逻辑门组成的。这些门利用有电和无电来表示一个 1 和 0 的二进制系统。你说“我们已经知道了！”但是你有没有考虑过用电以外的东西来进行二进制计算？斯坦福大学的一个团队。他们正在用微小的水滴和条形磁铁制造逻辑门。

他们的目标不是操纵信息或与现代“电子”计算机竞争。相反，他们的目标是以逻辑的方式操纵物质。水滴就像可以携带各种其他分子的小袋子，使其应用范围更广。以生物学为例，信息是通过动作电位来交换的——动作电位是电和化学的尖峰信号。我们把电气部分拆了。这项技术也可以利用化学成分。
