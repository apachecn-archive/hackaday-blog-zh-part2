# 第一个 Arduino 雷达防护罩

> 原文:[https://hack aday . com/2014/06/14/the-first-arduino-radar-shield/](https://hackaday.com/2014/06/14/the-first-arduino-radar-shield/)

第一个完全可操作的雷达 Arduino shield [最近在湾区 Maker Faire](http://www.youtube.com/watch?v=nHDUjJCvMVo) 进行了演示。它是由加州大学戴维斯分校的本科生丹尼尔和大卫建造的。

许多人已经谈论过这样做，有些人甚至已经做出了原型，但是这些本科生完成了它。这是刘教授的[整学期高级设计课程](http://ucdart.net/seniordesign)在[麻省理工学院咖啡罐雷达短训班](http://ocw.mit.edu/resources/res-ll-003-build-a-small-radar-system-capable-of-sensing-range-doppler-and-synthetic-aperture-radar-imaging-january-iap-2011/)基础上的成果。*明年这门课程将有 30 名学生*，向世界展示基于项目的学习的兴趣和市场。

查看高分辨率测距演示，其中使用了更宽带的啁啾声，产生了惊人的效果。下面视频。
