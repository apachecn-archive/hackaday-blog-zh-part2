# VCF 东部:Swyft 卡

> 原文:[https://hackaday.com/2014/04/06/vcf-east-the-swyft-card/](https://hackaday.com/2014/04/06/vcf-east-the-swyft-card/)

95%的人会说苹果电脑是史蒂夫·乔布斯的发明。另外的 5%是对的，但是几乎没有人知道麦金塔项目最初是由杰夫·拉斯金构思的。他有幸将 Mac 变成了一种“信息设备”，并且是第一批认真考虑数百万人将如何与计算机互动的人之一。

不过，Mac 并不是杰夫在苹果的第一个项目。在 Mac 项目之前，他正在开发一个叫做 Swyft 的东西——一个易于使用的*命令行*系统，最初是作为 Apple IIe 的固件卡实现的。[Mike Willegal]好心地带了一张 Swyft 卡到这个周末的老式电脑节上，并为我们[做了一个演示](https://www.youtube.com/watch?v=sIk_XyiwCWQ)。

Swyft 卡背后的基本想法是有一个集成的文字处理器，计算器，并访问 Applesoft Basic。按住“leap”键——在 Apple IIe 插件的情况下，打开 Apple 键——允许用户搜索文本并对任何结果执行操作。这很奇怪，但它只是以某种奇怪的方式有意义。

[Mike]今天在 VCF 上构建课，任何参加的人都可以构建自己的 Swyft 卡。如果你想尝试用户界面设计中的一个“本来可以”的东西，他也有关于[构建你自己的](http://willegal.net/superproto/index.php?title=Swyft_Card)的指导。
