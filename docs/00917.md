# 噪音污染针锋相对地使用巴哈男孩作为武器

> 原文:[https://hack aday . com/2012/07/03/noise-pollution-tit-tat-uses-the-Baha-boys-as-a-weapon/](https://hackaday.com/2012/07/03/noise-pollution-tit-for-tat-uses-the-baha-boys-as-a-weapon/)

![](../Images/b79dff0e0d9d44adc03ebb44bb340f62.png "baha-boys-sound-weapon")

在这里[Matthew Br]解释了他和邻居的情况，他们共用他公寓的这堵墙。当他们听音乐时，他们喜欢声音大，所以他也能“享受”这种体验。但是他不能再忽视它了，他决定使用音量检测器向他们播放一些音乐。

他在墙上贴了一个麦克风，并将其连接到他的 Arduino 上。它监控传入的声音，并使用一个可调的阈值，当邻居太吵时它就会触发。我们认为他明智地加入了一些时间过滤，以确保巨响是持续的，而不仅仅是有人撞到墙上的结果。当系统在一段时间内检测到嘈杂的音乐时，它会触发[马修]自己的 CD 播放器播放出*谁把狗放出来了？*巴哈派男孩。它会播放一段时间，然后关掉听，看看邻居是否仍然吵闹。

休息之后，他在视频的后半部分记录了一次实际的跑步。我们当然希望他住在一栋只有两个单元的房子里，否则其他邻居也会发疯的！
