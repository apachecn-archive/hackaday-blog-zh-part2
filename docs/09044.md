# 气动钢笔枪适用于詹姆斯·邦德

> 原文：<https://hackaday.com/2015/05/15/pneumatic-pen-gun-is-fit-for-james-bond/>

《詹姆斯·邦德》系列因许多东西而闻名，但对我们黑客来说，最重要的或许是这些小玩意。邦德总是有一个很棒的小玩意，不知何故正是他摆脱困境所需要的东西。[hw97 kar bine]的最新项目将适合一个老邦德电影。他成功制造了一把单发弹丸枪，看起来像一支钢笔。

[hw97karbine]一开始是从一管碳纤维上切下身体。他用钢锯切割，然后在车床上清理边缘。从一根直径较小的铜管上切下一个桶。这两根管子最终会一根套在另一根里。定制的前端盖由黄铜加工而成。一端有棱纹，粘在碳纤维管里。枪管也粘在前盖的这一端，尽管它粘在盖子的内侧。盖子的另一端有 1/8”BSP 螺纹切入其中，以允许连接。

后端盖由 Delrin 加工而成。这件作品也有一个 Delrin 活塞放在里面。活塞有一小块橡胶用作垫圈。这个活塞阀使枪能够工作。后盖粘合到位并连接到 Schrader 气门上，从汽车轮胎气门杆上取下。

为了给系统加压，Schrader 阀上连接了一个自行车泵。这将活塞推向活塞筒，防止空气逸出。活塞没有形成完美的密封，因此空气在其周围泄漏，并对碳纤维管加压。Schrader 阀防止空气从笔体中泄漏出来。一个特殊的机械按钮被拧在 Schrader 阀门上。按下按钮，空气逸出；突然的压力不平衡导致活塞向后射出，为空气通过枪管逸出开辟了一条通道。逸出的空气发射出射弹。用一个[动画](http://www.spudfiles.com/files/howpneumaticpengunworks_868.gif "piston valve")更好地解释了整个过程。

现在，留在我们脑海中的问题是:这是几年前我们看到的真空压力火箭筒中使用的同一个压力不平衡概念吗？
