# 激光很容易制造，任何人都可以烧坏自己的眼睛

> 原文：<https://hackaday.com/2012/04/27/laser-so-easy-to-build-anyone-can-burn-their-eyes-out/>

![](img/7866049f73cce1ab63e1a0aa84013b54.png "burning-laser-build")

北街实验室的男孩们[制造了一个手持式燃烧激光器](http://www.northstreetlabs.org/laser.html)并让它看起来超级简单。其实不是。我们也不认为这很难，但它看起来如此容易的唯一原因是因为他们真的知道自己在做什么。

第一步是为应用寻找最好的部件。他们使用的是手持式手电筒，机身很小，但仍有足够的空间放置部件。接下来，他们订购了一个针对二极管波长的优质透镜，以及一个预制的驱动板。

现在真正的构建开始了。他们敲打金属车床，用一些铝原料加工出二极管外壳。为了将零件结合在一起，他们使用了一些导热膏，并使用扳手套筒来保护二极管免受虎钳施加的压力。它滑到位，整个东西完全适合手电筒的外壳。如果没有它燃烧东西的视频证据，这个项目就不完整。休息过后你会发现。
