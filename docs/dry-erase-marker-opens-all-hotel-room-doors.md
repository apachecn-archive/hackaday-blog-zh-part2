# 干擦记号笔可以打开所有酒店房间的门

> 原文：<https://hackaday.com/2012/10/02/dry-erase-marker-opens-all-hotel-room-doors/>

如果你带着一块暴露在外的电路板和一堆电线，人们会注意到你的。但是一个干擦记号笔不会让任何人回头。这个有着自己的小秘密。[充当酒店房间门锁](http://blog.spiderlabs.com/2012/10/pentesting-hotels-with-pens.html)的万能钥匙。

这实际上更像是一个重新包装的黑客。这个漏洞已经很出名了。酒店中最常用的 Onity 品牌钥匙卡锁底部有一个电源插孔，可兼作单线通信端口。第一个公布的概念验证使用 Arduino 板和简单的适配器在一秒钟内打开任何门。现在，硬件的尺寸已经减小，这样它就可以装入干擦记号笔的空心外壳中。更好的是，毡尖已被大小合适的桶形千斤顶所取代。休息后，看看它的超快和不起眼的使用。我们认为使用这个并不比实际拥有钥匙卡更明显。
