# THP 参赛作品:万能遥控发射器

> 原文：<https://hackaday.com/2014/07/16/thp-entry-the-everything-rc-transmitter/>

除了少数例外，大多数黑客大奖都是我们以前没见过的东西:已经被归入历史书的基数为 3 的计算机，极其古怪的 3D 打印机，以及新奇的新物联网设备是常态。[OSRC 对我们来说并不是一个新项目](http://hackaday.io/project/1839-OSRC---Open-Source-Remote-Control)。(更新:看起来他们删除了他们的项目页面。这是来自互联网档案馆的[快照。我们在 2011 年](https://web.archive.org/web/20140819093543/http://hackaday.io/project/1839)见过一次[，一年后](http://hackaday.com/2011/07/19/rc-transmitter-does-everything/)又见过一次[。让 OSRC 成为 Hackaday 奖的一个有趣项目的原因并不是它是有史以来最先进的 RC 发射器。创造这一点显然很容易。OSRC 可以用一个大的财政踢在裤子上，如果[德米特里斯]赢了，我们猜测他不会乘坐太空。相反，他将获得现金奖励，将他的终极发射器投入大规模制造，并投入野外。](http://hackaday.com/2012/09/10/remote-control-does-everything/)

乍一看，OSRC 的基本型号似乎很贵，大约在 6-700 美元之间，请考虑一下这个:一个优秀品牌的六通道发射机大约需要 120 美元。九个频道大约要花你 400 美元。OSRC 是一台 40 频道的收音机。操纵杆能够进行力反馈，当然,“专业”型号的 OSRC 也有那种奇妙的屏幕，能够显示来自 FPV 摄像机的视频、GPS/地图覆盖图或令人难以置信的大范围遥测显示。有几千美元的真正的飞机航空电子设备有一个较小的功能集，这不是夸张。

几个月前，[Demetris]在 Flite Test 接受了牛人的采访。下面嵌入了那个(强烈建议)视频。
