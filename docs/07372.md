# 固态特斯拉线圈播放音乐

> 原文：<https://hackaday.com/2014/10/20/solid-state-tesla-coil-plays-music/>

如果你曾经想制造一个特斯拉线圈，但发现它们非常昂贵和/或复杂，不要再找了！[Richard]制造了一个固态特斯拉线圈,它的零件最少，也相对容易制造。

这个特斯拉线圈是围绕一个空心变压器建立的，该变压器将低 DC 电压提升到非常高的交流电压。磁芯可以手动缠绕，也可以作为一个整体购买。驱动电路是特斯拉线圈建立的地方，与其他线圈分开。特斯拉线圈通常利用火花隙，但[Richard]使用的是功率脉冲调制器 PWM-OCXi v2，它用晶体管进行开关。特斯拉线圈将与一个驱动电路一起工作，但[理查德]指出，它与两个更稳定。

然而，这种构建并没有止步于固态电路。[Richard]使用 Arduino 和通常用于驱动扬声器的软件来让他的特斯拉线圈播放音乐。休息后请务必观看视频。如果你正在寻找一个更适合万圣节的特斯拉线圈，你可以看看[这个电击南瓜的特斯拉线圈](http://hackaday.com/2013/11/03/professor-kill-a-volt-shocks-pumpkins-with-his-tesla-coil/)！
