# Phoenard:和 Arduino Mega 一样小的 Arduino 手机

> 原文：<https://hackaday.com/2014/05/29/phoenard-arduino-phone-as-small-as-an-arduino-mega/>

[https://www.youtube.com/embed/_hXgwTRqwRU?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/_hXgwTRqwRU?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

在创客大会上，在 Atmel 展台的一边闲逛的是[帕蒙卡斯·苏马斯塔]，他正在炫耀他的名为 phoe nard 的 Arduino 手机。我们真的很喜欢它的外形，但它的易用性才是它真正的亮点。[Sumasta]展示了菜单系统，它非常简洁，可以让你很容易地添加自己的应用程序。软件不是唯一可以定制的东西，因为手机底部有一个连接器。他展示了一个试验板附件，上面有各种颜色的发光二极管。它们的强度可以通过触摸屏上的一个简单的滑动应用程序来改变。但是如果你想要的话，还有更多的力量。展出的还有一个自平衡机器人身体，它的顶部有一个电话连接器。

[Sumasta] [赢得了 Atmel 英雄大赛](http://atmelcorporation.wordpress.com/2014/02/25/and-the-avr-hero-design-contest-winners-are/)，我们假设他就是这样从荷兰一路来到旧金山参加 Maker Faire 的。你可以在脸书页面上了解更多关于 Phoenard 的技术细节。
