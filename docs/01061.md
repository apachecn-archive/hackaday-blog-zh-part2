# Kinect 上的手指识别

> 原文：<https://hackaday.com/2012/07/17/finger-recognition-on-the-kinect/>

![](img/e58cbf56f3f75275baeb0c3f2437d14d.png "vulcan")

Kinect 很棒，但如果你想以更高的分辨率检测一个人的肢体，你就没那么幸运了。CogniMem 的 Chris McCormick 有一个很好的解决方案:[使用芯片上的神经网络来识别手指](http://www.youtube.com/watch?v=NqjopQmqWAE)使用已经连接到 XBox 的硬件。

这个构建使用了非常酷的芯片上的 [CogniMem CM1K](http://www.cognimem.com/products/chips-and-modules/CM1K-Chip/index.html) 神经网络，该网络经过训练，可以区分单手从一数到四，以及“a-ok”手势、Vulcan 问候(如上所示)和在[Dio]音乐会上摇摆。正如[Chris]在视频中向我们展示的那样，这些手指手势可以用来在屏幕上绘画，只需张开手掌和握紧拳头就可以移动物体；离*少数派报告*和*钢铁侠* UIs 不太远。

如果你想复制这个版本，我们在这里找到了 CM1K 神经网络芯片，价格比我们愿意支付的要高一点。芯片上的神经网络是一个*非常*酷的设备，但看起来这一构建将不得不等待 Kinect 2 进入消费者和业余爱好者的舞台。
