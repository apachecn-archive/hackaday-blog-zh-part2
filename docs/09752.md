# 可爱的倒计时器提醒你即将到来的厄运

> 原文：<https://hackaday.com/2015/08/05/cute-countdown-timer-reminds-you-of-impending-doom/>

随着事情变得繁忙，无论是即将到来的产品发布会，怀孕，甚至是你最喜欢的游戏(或电影)的发布！)有时候有个小小的倒计时器也挺好的。不是你手机上的一个应用程序，而是放在你桌子上的一个有形的物理计时器。这就是为什么 SevenSeg 是一个如此可爱的想法。

[Mohit]想设计一些简单但美观的东西——他以前见过自由形式的电子项目，想尝试一下。他想出来的东西相当优雅！一个七段显示器通过 1/32”黄铜棒连接到控制器，一个粒子光子-有点像物联网的 WiFi Teensy。在将几个电阻与显示屏对齐后，再进行一点令人沮丧的电线弯曲，SevenSeg 就完成了。

因为它连接到了互联网，[Mohit]决定给它一些情感动画，在倒计时之间显示——他是不是很可爱？
