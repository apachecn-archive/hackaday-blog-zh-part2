# 史上最差的电子阅读器

> 原文:[https://hackaday.com/2014/06/14/the-worst-e-reader-ever/](https://hackaday.com/2014/06/14/the-worst-e-reader-ever/)

![oled](../Images/f6e0f87c5b574166a37b2a05446c59b5.png)

在项目网站上，【Jaromir】创造了一个微型设备，它有一个有机发光二极管显示屏，三个按钮和一个 USB 端口，用于存储文本文件，可以用来保存购物清单，备忘单，或者是有史以来最不可用的电子阅读器。

该设备的正面是一个 96×32 像素的有机发光二极管和三个按钮，分别用于“向上”、“向下”和“打开/关闭”。另一方面，PIC24 微控制器在芯片上建立了一个文件系统，允许[Jaromir]在容量少得可怜的闪存驱动器上写入 64kB 的数据。文本文件可以在有机发光二极管上查看，下面的视频显示维基百科的首页以 16×4 的文本模式显示。

无论如何，这都不是一个非常有用的设备，但出于某种原因，它在 Hackaday 项目上获得了许多头骨和追随者。作为回应，[Jaromir]正在开发第二版，使用新的 PCB 和 3D 打印外壳的设计。对于[Jaromir]自己所描述的比任何手机或平板电脑都糟糕的东西来说，这还算不错。
