# DIY 万用表，Arduino 单独出售

> 原文:[https://hack aday . com/2014/04/24/DIY-multimeter-arduino-sold-separately/](https://hackaday.com/2014/04/24/diy-multimeter-arduino-sold-seperately/)

![Arduino-based Multimeter](../Images/dd14c8a92037a439322d87f5c1801115.png)

你不能否认 Arduinos 在创客/黑客社区中非常受欢迎。有些人会说，肯定不缺少使用它们的项目。[Milen]不这么认为，并认为是时候创造一种基于 Arduino 的万用表了。

这个项目的核心是一个普通的 Arduino Uno。为了降低总成本和项目复杂性，额外的部件被保持在最低限度。成品可测量 0-100v 的电压、高达 500mA 的安培数和 0-250 kohm 的电阻。如果您需要检查连续性，它也可以做到这一点。

制造万用表所需的所有零件都装在一个直接插入 Arduino 的屏蔽罩上。香蕉插头允许连接测试引线。测量值显示在 LCD 屏幕和/或(如果连接)Arduino IDE 串行监视器上。如果只使用串行显示器，液晶显示屏可以省略，以节省一些钱。
