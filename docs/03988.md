# 定制汽车键盘输入

> 原文:[https://hackaday.com/2013/09/05/custom-car-keypad-entry/](https://hackaday.com/2013/09/05/custom-car-keypad-entry/)

![jeepKeyless](../Images/de5bc8f63f9724972aa17f6dc2d83ac5.png)

[EdsJunk]喜欢户外活动，用他的吉普牧马人带他去那里，但藏钥匙只是为了去游泳让他紧张。在一位朋友向他展示了通过键盘进入他的汽车是多么方便之后，[EdsJunk] [决定是时候建造自己的](http://www.wranglerforum.com/f33/custom-keyless-keypad-entry-272969.html)了。

该建筑使用了一个附加在 Arduino Micro 上的备用防水键盘。[EdsJunk]通过拆卸他的额外无钥匙进入钥匙链来简化事情；如果驾驶员从键盘上接收到正确的代码，他会按下遥控钥匙上的解锁按钮以允许进入。[EdsJunk]承认牧马人的软顶很容易进入，但解释说，这个项目的目标是保持警报激活，如果有人试图通过软顶闯入，警报可能会响起。您可以在下面观看键盘访问的视频演示。这是他在一辆汽车上进行的众多黑客攻击的又一个伟大补充。

然而，我们确实希望代码中内置某种锁定机制来防止暴力破解:在多次尝试失败后，应该很容易激活汽车的紧急按钮。汽车黑客今年夏天很受欢迎:如果你错过了[真正的汽车遥控器](http://hackaday.com/2013/09/01/a-real-car-remote-controlled-with-an-arduino-what-could-go-wrong/ "A real car remote controlled with an Arduino… what could go wrong?")就来看看吧。
