# 用你的意识聚集单细胞生物

> 原文:[https://hack aday . com/2013/05/23/hod-单细胞生物-用你的思维/](https://hackaday.com/2013/05/23/herd-single-cell-organisms-with-your-mind/)

![mind-controlled-paramecium](../Images/4a3c51414269a5ed6c93592cf0cc0643.png)

大多数涉及脑波的项目建议都被忽略了，因为很难从这种控制中获得更多。这个项目不一定会在控制方面取得进展，但你必须承认用你的思想来聚集生命形式值得仔细观察。

[杰瓦]搭建了一个平台，让他能够与草履虫互动——这是一种微小的单细胞生物，整天快乐地游来游去。就像脊椎动物一样，它们不喜欢电击。让电流流过液体，它们会游向负极。

这个实验用四根铅笔芯做电极。这些由 Arduino 驱动，Arduino 对玩具脑电波设备的输入做出反应。以正确的方式集中注意力，它们就会游到你想让它们去的任何地方。

这并不像[蟑螂精神控制](http://hackaday.com/2012/10/23/controlling-a-cockroach-leg-with-your-mind/)那么复杂，但却和
