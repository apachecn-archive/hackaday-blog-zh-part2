# 由 DIY 线性致动器驱动的简易卡特彼勒进料器

> 原文：<https://hackaday.com/2012/04/21/an-easy-to-build-cat-feeder-driven-by-a-diy-linear-actuator/>

叛逆实验室/疯狂科学博客的[Will Finucane]正在寻找一种方法，让他的猫在短暂的假期中快乐地喂食，所以他组装了一个便宜又简单的自动喂食器，以确保他离开时他的宠物不会挨饿。

我们以前在这里见过不同版本的自动宠物喂食器，[一些依靠旋转假底](http://hackaday.com/2010/01/31/recycled-cat-feeder/)，而另一些使用[曲柄式喂食器来完成工作](http://hackaday.com/2009/12/03/internet-enabled-cat-feeder/)。[Will 的]解决方案有点不同，它采用了一种廉价的线性致动器来输送饲料。

他倒空了一根胶棒，用一根铜管代替胶水。这给了他胶水所缺乏的刚性，让他可以轻松地上下移动一个装满猫粮的平台。他将胶棒安装在一个连续旋转的伺服系统上，将致动器和进料平台安装在一个纸板盒内。

使用 Arduino，他每隔 12 小时降低一次可移动平台，让一点猫粮从他在盒子侧面切割的孔中落下。虽然他的发明可能经不起多年的使用，但这是一个快速的解决方案，花费很少，取决于你身边有什么。
