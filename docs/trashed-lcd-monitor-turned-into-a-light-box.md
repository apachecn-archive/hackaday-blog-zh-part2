# 废弃的液晶显示器变成了灯箱

> 原文：<https://hackaday.com/2013/05/08/trashed-lcd-monitor-turned-into-a-light-box/>

给了[x2jiggy]一个不能正常工作的液晶显示器。他做了一点小小的努力来排除它不能在屏幕上显示任何东西的故障，但却无法让它再次工作。失败后，他决定将它重新用作灯箱，而不是仅仅送去回收。

显示器制造商投入了大量精力，以确保背光尽可能均匀地分配亮度，这将使这成为一个非常好的灯箱。[x2jiggy]通过移除外壳开始转换。当它脱落时，他给它涂了一层新漆。LCD 面板、驱动它的 PCB 和光扩散片都被移除，只留下背光和逆变器 PCB。用万用表探测了一下，他甚至找到了一个地方连接一个拨动开关来启动逆变器的电源。你可以在上面的图片中看到他加在表圈上的银色开关。完整的构建视频在休息后嵌入。
