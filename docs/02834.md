# 一个遥控的、自主的风筝产生能量

> 原文:[https://hack aday . com/2013/02/24/a-遥控-自主-风筝-发电/](https://hackaday.com/2013/02/24/a-remote-controlled-autonomous-kite-generates-power/)

![kite](../Images/48c2a0b7aeb3e00de0cc0ae6d0c463d6.png)

利用风力发电很容易——只要把风车挂在杆子上，再接上发电机。不过这并不特别酷，所以[Adrien]和他的高级设计项目团队正在使用自主控制的风筝来发电

用风筝发电的基本想法是让它绕着 8 字形飞行，同时从线轴上松开风筝线。风筝线上非常强的力可以用来驱动发电机，该发电机为以较低的迎角卷回风筝提供动力。你可以查看[Adrien]的[风筝动力理论页面](http://adrienemery.com/kite-power/),了解更多关于这是如何工作的细节。

现在，[阿德里安]和他的团队有一个基本的发电装置，并通过操纵杆放风筝。更新即将到来，你可以在休息后看看他们的遥控风筝的视频。
