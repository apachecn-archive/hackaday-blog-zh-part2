# 黑客空间之旅:如澳大利亚维多利亚的创客空间

> 原文:[https://hack aday . com/2014/06/23/hacker space-tour-eg-makers space-in-Victoria-Australia/](https://hackaday.com/2014/06/23/hackerspace-tour-eg-makerspace-in-victoria-australia/)

![EG MakerSpace](../Images/983b7709360764617055f005bb5cbf99.png)

我们刚刚听说位于澳大利亚 Bairnsdale 的东吉普斯兰创客空间需要更多会员！他们给我们发了一个精彩的旅游视频，他们的地方看起来简直太棒了。

这是一个非常大的设施(看起来像一所老学校),甚至可以与我们在欧洲的[黑客空间](http://hackaday.com/2013/11/06/hackerspacing-in-europe-conclusion/)之旅中看到的一些最大的黑客空间相媲美——说真的，他们有一个容纳一切的房间！

他们有所有的基本设施，如电子实验室、木工区、社区休息室、厨房、金属加工区、一般艺术和手工艺区。但他们也有一个声音亭(正在进行中)，一个芳香疗法和按摩室，一个陶器室，一个雕塑室，一个多功能黑客室，网络服务器室，一个复古的拱廊和计算机培训实验室，以及大量的存储空间！

留下来参观创始人斯科特·拉姆谢德的官方参观！
