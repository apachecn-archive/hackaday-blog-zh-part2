# 通过深度感应相机看世界

> 原文:[https://hack aday . com/2014/11/10/通过深度感应相机看世界/](https://hackaday.com/2014/11/10/seeing-the-world-through-depth-sensing-cameras/)

如果你想用立体视觉探索虚拟世界，Oculus Rift 和所有其他 3D 视频护目镜解决方案都很棒，但直到现在我们还没有看到任何人用数字立体观众探索现实生活。[pabr]将华硕 Xtion 中类似 Kinect 的传感器与智能手机结合在类似谷歌 Cardboard 的设置中[以获得人眼无法自然体验的 3D 视图](http://www.pabr.org/depthvision/depthvision.en.html)如第三人称视角，类似雷达的显示器，以及在你的眼睛相距 20 英寸的情况下看到世界会是什么样子。

[pabr]正在使用通过 USB OTG 端口连接到 Galaxy SIII 的 ASUS Xtion 深度传感器。只需一点点代码，深度传感器的输出就可以推送到手机的显示屏上。硬件设置由一个虚拟仪器组成，这是一种相当昂贵的塑料，但如果考虑到正确的机械因素，一块纸板或一些泡沫板和热熔胶就可以很好地完成。

[pabr]将他构建的视频演示与这个项目可以做的一些例子放在一起。这很奇怪，而且令人惊讶的是，这并不是一种多余的 3D 观看方式。你可以看看下面的视频。
