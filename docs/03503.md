# Raspberry Pi 自动化您的番茄农场

> 原文:[https://hack aday . com/2013/07/01/raspberry-pi-automates-your-tomato-farm/](https://hackaday.com/2013/07/01/raspberry-pi-automates-your-tomato-farm/)

![rpi-tomato-farming](../Images/4b45e001633be833a94030f3851ebf9d.png)

用一个他自己建造的监控系统检查一下[Devon]种植的番茄。它是基于一个树莓派。至于成长控制器，它还没有完全自动化。这是因为它唯一能启动的东西是盘旋在植物上方的黑色水线。但是[Devon]在监测和收集传感器数据方面的工作应该会使未来增加功能变得容易。

上图中的湿度传感器监测植物生长的土壤。但他也有温度和光线传感器。当从种子开始生长时，这些是非常重要的，并且可以与需要更高土壤温度的植物(如胡椒植物)的加热垫结合使用。西红柿也很修长。现在他在监控光线水平，用生长灯来增加设置会很好。一个长期的目标甚至可以是一个电动床，它可以把植物抬高到灯泡的高度，这样它们就不会碰到光了。

不要让我们眼中的星星分散你的注意力。他在这个项目上做了大量的工作，包括物理构建和绘制系统收集的数据。干得好！
