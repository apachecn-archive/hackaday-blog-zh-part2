# Hackaday 奖参赛作品:一辆轻型电动多功能车

> 原文:[https://hack aday . com/2015/06/07/hack aday-prize-entry-a-light-electric-utility-vehicle/](https://hackaday.com/2015/06/07/hackaday-prize-entry-a-light-electric-utility-vehicle/)

[克里斯]生活在南苏丹，那里有许多基础设施糟糕的贫困地区。该地区面临的一个更大的挑战是让人员和物资通过路况很差或根本不存在的道路。普通的交通工具并不适合这项任务，Hilux 或 Land Cruiser 太贵了。作为 Hackaday 奖的参赛作品，[Chris]正在为发展中国家建造一个坚固耐用的低成本多功能车平台。

这辆电池驱动的四轮车是由克里斯能找到的东西制成的。框架由 50x50mm 毫米的角铁焊接而成，车身面板由当地采购的 1200 x 2400 x 1.2 毫米板材制成。虽然[克里斯]想要更好的车轮，但便宜的中国摩托车车轮随处可见，而且便宜——65 美元，包括轴承、刹车和链轮。它甚至比陆地巡洋舰有更高的离地间隙。

[克里斯]已经有了他的项目的原型，它正在滚动。你可以看看下面的视频。
