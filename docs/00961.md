# 3D DLP 打印机构建橙色 TARDIS

> 原文：<https://hackaday.com/2012/07/05/3d-dlp-printer-builds-an-orange-tardis/>

![](img/81f8bb1b32f48d209ad49aa11184fede.png "TARDIS")

这个微型 TARDIS 是【罗恩莱特】的 [Sedgwick 3D DLP 打印机](http://sedgwick3d.com/)的最新打印品。是的，它是橙色的，但如此小的物体的打印质量是相当惊人的。

Sedgwick 3D 打印机目前在 Kickstarter 上作为套件[提供。500 块骨头，塞奇威克提供了所有的部件——除了 DLP 投影仪和树脂——来制作你自己的微型 40 型变色龙电路。这台打印机还能做更多的事情，从](http://www.kickstarter.com/projects/sedgwick/open-source-3d-dlp-printer)[的微型教堂到](http://s3.amazonaws.com/ksr/assets/000/027/069/317d578854e34ae008dc70fc8c029db3_large.jpg?1339538213)的空心短程球体。

这是 DLP 投影仪/树脂 3D 打印机产品线中的最新产品，也是迄今为止最实惠的产品。我们看到的最后一个是一台价值 2400 美元的机器，包括一个投影仪和树脂。500 美元的无投影仪套件，Sedgwick 仍然轻松击败了迄今为止我们见过的最便宜的选择。

[罗恩·莱特]来自堪萨斯城，我们的老板[凯莱布]几周前在堪萨斯城的创客集会上碰到了他。你可以看看这个小采访和几个塞奇威克在休息后做它的事情的视频。
