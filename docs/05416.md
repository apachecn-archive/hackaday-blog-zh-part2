# Gamebuino:一款手持 Arduino 游戏主机，准备好迎接黄金时代

> 原文:[https://hack aday . com/2014/02/27/gamebuino-a-hand held-arduino-gaming-console-ready-for-prime-time/](https://hackaday.com/2014/02/27/gamebuino-a-handheld-arduino-gaming-console-ready-for-prime-time/)

[Rodot]来信告诉我们关于[Gamebuino](http://gamebuino.com/)的情况，这是一款设计精美、易于复制的手持式 Arduino 游戏机。一年前，我们最初推出了基于 [Arduino 的游戏主机](http://hackaday.com/2012/10/01/fantastic-programming-makes-this-arduino-gaming-device-something-special/)。

有了 Gamebuino，你也可以围绕 Arduino 构建自己的游戏和游戏硬件。虽然该网站的 Wiki 目前缺少很多信息，如布局和游戏代码，[Rodot]计划将一切都开源。该控制台包括一个可充电锂电池、一个微型 SD 卡和 [I2C](http://hackaday.com/2013/05/17/reading-game-boy-carts-with-i2c/) 扩展连接器。这是一个需要密切关注的项目，尤其是因为一个完整的游戏库将会提供，让你可以轻松地创建自己的游戏。休息之后，在演示视频中看看控制台能做些什么。

这将是惊人的，所以看到一些旧的游戏男孩游戏移植到 Arduino 上运行，或者也许我们的一个读者可以为 Arduino 制作一个[游戏男孩模拟器](http://hackaday.com/2013/07/31/how-to-play-a-game-boy-emulator-on-chromecast/)！
