# 建立自己的雷达系统

> 原文：<https://hackaday.com/2012/04/28/build-your-own-radar-system/>

谁也不知道我们是如何错过这一次的，但去年 DEFCON 上的一次展示涵盖了 DIY 雷达构建。[Michael Scarito]讲述了 radar 背后的概念，然后继续展示了构建自己的设置并不太困难或昂贵。休息之后，我们嵌入了他 45 分钟的讲话。

上面这两大块硬件应该看着眼熟。他们是最受欢迎的黑客项目 cantenna 的后代。基于 can 的远程天线在 WiFi 应用中最受欢迎，但[我们已经看到它也用于蓝牙](http://hackaday.com/2010/04/23/wifi-and-bluetooth-sniffing-rifle/)，在这里看到它并不奇怪。剩下的就是大量的传感硬件和足够的数学知识塞进编码里，让你的耳朵耷拉下来。

如果你走得够远(正好 39 分钟的演讲)[Michael]分享了一些关于构建的更多信息的链接。我们认为虚拟生活对我们来说已经足够了，但是如果你想建立自己的生活，不要忘记发布一个项目日志！
