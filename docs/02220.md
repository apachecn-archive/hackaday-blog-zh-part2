# 国内通孔电镀和铣削

> 原文:[https://hack aday . com/2012/11/29/通孔电镀和在家铣削/](https://hackaday.com/2012/11/29/through-hole-plating-and-milling-at-home/)

![](../Images/cc7772cf6a2abf0ad4259d970fee3fc3.png)

下面是[一个让我们羡慕的 PCB 制作工艺](http://www.youtube.com/watch?v=yqjKohZX2t8)。这是相当接近 fab-house 质量在家里。[Cpirius]正在使用 CNC 轧机和通孔电镀技术生产他的双面电路板。

休息后嵌入的视频从头到尾显示了一个板。它始于轧机钻孔通过一些双面覆铜股票。一旦磨屑被清理干净，这些孔就被涂上防水墨水和碳的混合物。这通过使孔本身导电来为电镀做准备。然后，根据该指南对电路板进行[电镀处理。](http://hackaday.com/2012/10/03/through-hole-electroplating-in-your-home-lab/)

可能这个过程中最有趣的部分开始于剪辑的第 52 秒。该工厂使用导电探针来生成整个电路板的高度图。这使得它可以改变布线深度，以完美切割隔离走线。最终的路由过程如上图所示。
