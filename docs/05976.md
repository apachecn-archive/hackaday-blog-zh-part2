# 炸弹时钟吓醒你！

> 原文:[https://hack aday . com/2014/05/05/bomb-clock-scares-you-awake/](https://hackaday.com/2014/05/05/bomb-clock-scares-you-awake/)

![Bomb Clock Scares You Awake](../Images/a581103479b0d8e5a2d4129cb7c55d5e.png)

如果你没有按正确的贪睡组合，还有什么比担心自己即将爆发更好的醒来方式呢！这是[deve 定子](http://mikrokontrolery.blogspot.ca/2011/03/DIY-Devastator-Bombowy-zegarek-ATmega8.html) ( [翻译](https://translate.google.com/translate?hl=en&sl=pl&tl=en&u=http%3A%2F%2Fmikrokontrolery.blogspot.ca%2F2011%2F03%2FDIY-Devastator-Bombowy-zegarek-ATmega8.html))，【亚采克】最新的有趣项目，直接来自波兰。

作为一个狂热的彩弹爱好者和气枪爱好者，[亚采克]想要建造一个独特的时钟——所以他决定制作自己的经典炸药棒炸弹…时钟。他使用 ATmega8 微控制器作为项目的核心，同时使用 DS1307 RTC 和 DS1820 温度传感器，因为只是为了好玩，时钟还可以监控环境温度！

为了增加项目的真实性，他还使用 Eagle CAD 从头开始设计 PCB，这使他能够使整个事情看起来更具威胁性。为了实际制作 PCB，他使用了层压[热转印法。](http://hackaday.com/2013/09/12/diy-heatsinking-pcbs/)PCB 上的四个按钮允许您滚动浏览日期、时间、温度和设置闹钟时间。

哦，那“炸药”呢？裹着红头文件的纸巾卷。
