# 制造一扇双向开启的门

> 原文:[https://hack aday . com/2015/09/03/做一扇双开门/](https://hackaday.com/2015/09/03/making-a-door-that-opens-both-ways/)

你有多少次走错门了？有多少次，你会一时糊涂，不知道是该推还是该拉那个模糊的门把手，因为它的用途并不明显？

如果你再也不用担心门会怎么样？如果我们能有一扇全方位的门呢？[TVMiller]找不到任何这方面的例子，所以他决定建立自己的原型。他称之为[任意门。](https://hackaday.io/project/7483-any-way-door)

“任意门”只是一个 1:12 比例的版本，但正如你在下面的视频中看到的，它工作得非常好——如果有任何东西可以成为室内设计师/建筑设计师喜欢的非常酷的门。

问题是，它能有效地全尺寸完成吗？

[TVMiller]很想知道，但没有时间或资源来建立一个全尺寸模型，所以他邀请了黑客观众来尝试一下。他将这个项目发布在 Hackaday.io 上，欢迎任何人加入并做出贡献。
