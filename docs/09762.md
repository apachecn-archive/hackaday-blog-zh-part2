# 机器人，到处都是机器人，发音庸俗

> 原文:[https://hackaday . com/2015/08/11/retro tech tacular-robots-robots-everywhere-with-kitschy-innuation/](https://hackaday.com/2015/08/11/retrotechtacular-robots-robots-everywhere-with-kitschy-pronunciation/)

人类智慧的一个伟大之处在于，我们有能力制造各种复杂程度的机器来执行我们的命令。作为技术的主要支持者，雪佛兰汽车公司曾经梦想着这样一个未来，美国家庭主妇最平凡的事情都可以通过按下按钮来处理——一个让机器人管家行动起来的按钮。

在这部短片中，雪佛兰向我们展示了这个未来的样子，这部短片是他们在 1940 年世界博览会上展示的。一个家庭主妇的忠实“机器人”，在整幅画中发音为“robe-it”，已经出了故障。很自然，她打电话给修理工。我们从控制台控制器上看到，Roll——哦，袍子——可以处理各种家务:他可以应门和接电话、洗碗、打扫房间、铺床、取帽子、做晚饭、修炉子(而且只能修炉子)。还有那个滚开！功能？从来没有解释过。我们倾向于认为它与让孩子们离开草坪有关，或者可以与“get door”结合使用来赶走潜在的窃贼。当 Roll-Oh 开门时，我们看到了这一幕，吓得一个年轻的[加里·西尼斯*]穿着警察制服送花。

Roll-Oh 的上肢有几个类似瑞士军刀的器具在里面。他用一把锋利的剪刀把花盒子上的丝带剪了下来。一看到花，他就用喷雾器给它们轻轻喷雾。掉落的花瓣对 Roll-Oh 来说不是问题。他立刻用他的大金属脚把它们从薄薄的工业摄影棚地毯上吸了起来。Roll-Oh 的任务是得到晚餐。这相当于他费力地打开几个易拉罐，用藏在脸上的火炬点燃蜡烛。

虽然滚动-哦，大管道管家只是一个梦想，雪佛兰想让你知道，较小的长袍-它已经在我们身边。他们调节我们炉子的温度，不用烧焦就能烤熟我们的面包，用凉爽的双球形玻璃渗滤器煮咖啡。这些微小的仆人能够执行其他任务，比如当人类太近时关闭机器，或者感应热量并启动灭火系统。这里简单提到了一种叫做 Petomat 的东西，这是一种自动喂狗系统，本质上是一碗藏在一个锁着的盒子里的食物。当与之相连的闹钟响起时，门闩会猛烈地打开。

Robe-its 也在执行更重要的任务，比如保持飞机水平并朝着正确的方向前进。当然，它们也大量存在于雪佛兰汽车中。汽化器中的一个小阀门将“汽油热量和新鲜空气维生素”的适当混合物输送给发动机。很少能达到这种程度的技术细节，你知道的。其他人监视火花、进气歧管和电压调节。在驾驶室里，友好的长袍-它会很高兴地在一个预设的推动下穿越 AM dial。

*实际上可能不是[加里·西尼斯]。
