# Retrotechtacular:昔日的斧头工厂

> 原文:[https://hack aday . com/2013/12/03/retro techtacular-an-axe-factory-of-yore/](https://hackaday.com/2013/12/03/retrotechtacular-an-axe-factory-of-yore/)

当你的脑海中想到斧头工厂时，你可能会想象工人们将毛坯装入一台机器，在送往退火炉之前对形状和轮廓进行加工。但这是复古的，我们很高兴能看到制造史上不同的时代。这次 [ax 工厂之旅着眼于缅因州奥克兰工厂制造过程的每一步](http://www.youtube.com/watch?v=Qr4VTCwEfko)。它是在 1965 年工厂关闭前几个月拍摄的。[Peter Vogt]在拍摄和编辑这卷视频方面做得很好，同样出色的是将它转换成数字格式，以便我们可以在他的 YouTube 频道上欣赏。

上面你可以看到自动锤——被称为[一个跳闸锤](http://en.wikipedia.org/wiki/Trip_hammer)——它由凸轮动作驱动。在这一点上，已经做了很多工作。两名工人从钢筋上切下坯料。这些在被折成两半以形成斧柄的环之前在锤上成形。从那里添加一块高碳钢来形成切割面。这就把我们带到了上面的步骤，将两个发光发热的部分合二为一。

我们不想破坏工艺水平，以及这里展示的劳动密集型工艺。但是，我们不能在结束这篇文章的时候，至少提一提在工作中吸烟和抽烟斗的庸俗行为。有一次，一名工人用炽热的斧头点燃了烟斗。

为了让你了解这与现代制造业有何不同，这里有一段关于斧头如何制造的[一段*《T2》插曲*](http://www.youtube.com/watch?v=XmTY6-uL58Y)(尽管我们认为正在制造的东西被称为斧头更合适)。
