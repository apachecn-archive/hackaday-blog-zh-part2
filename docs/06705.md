# 激光投影圣诞灯

> 原文：<https://hackaday.com/2014/08/01/laser-projected-christmas-lights/>

现在是八月，当然这意味着是零售商店挂上圣诞装饰品的时候了！但是说真的，如果你想比邻居做得更好，你需要从现在开始。[Joey]已经提前开始装饰了，去年圣诞节他用 LED 灯和激光投影仪制作了一个[房子大小的灯光秀。](http://imgur.com/a/EfVRq)

最初的想法是在车库上方悬挂一个花环会很好，但很快就变成了一台激光投影仪，可以在房子的正面显示节日主题的动画。该项目还包括一些 RGB LED 灯条，可以匹配投影仪显示的颜色。led 由笔记本电脑控制的定制电源供电，运行在计算机上的程序将视频信号的颜色平均到投影仪，投影仪点亮 LED 灯条以匹配投影的图像。这创造了一个有趣的效果，类似于一些以[家庭影院环境照明](http://hackaday.com/2011/02/17/more-ambient-lighting-monitor-hacks/)为特色的项目。

[Joey]遇到的唯一主要问题是不得不考虑投射图案中的激光运动，这导致计算机读取错误的值。这一点以及其他一些与激光相关的问题都通过一点编程来解决，以确保系统正常运行。之后，很简单的事情就是把投影仪固定在屋顶上，然后把 LED 灯条用拉链绑在屋檐上。

投影仪不受天气影响，已经度过了一个严冬，可以在任何假期正常运行。万圣节就要到了，这可能是给“不给糖就捣蛋”增添趣味的好方法。休息之后，请观看视频，了解这一设置的运行情况。
