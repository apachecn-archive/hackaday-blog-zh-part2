# 纳米机器人在非牛顿流体中像扇贝一样游动

> 原文：<https://hackaday.com/2014/11/08/nanobots-swim-like-scallops-in-non-newtonian-fluids/>

使用纳米机器人治疗疾病的想法已经存在多年，尽管它还没有以任何重要的方式实现。受珀塞尔扇贝定理的启发，[来自马普智能系统研究所的科学家](http://www.engadget.com/2014/11/07/tiny-robotic-scallops-swim-blood-eyeballs/?utm_source=Feed_Classic_Full)[创造了他们自己的版本](http://www.nature.com/ncomms/2014/141104/ncomms6119/full/ncomms6119.html)。他们设计了一种“微型扇贝”，可以推动自己通过[非牛顿流体](http://en.wikipedia.org/wiki/Non-Newtonian_fluid)，这正是大多数生物流体。

科学家们决定建造一个相对简单的机器人，一个有两个刚性“外壳”和一个柔性连接铰链的机器人。他们 3D 打印了该结构的负模，并用混有荧光粉的聚二甲基硅氧烷(PDMS)溶液填充，以便进行检测。一旦固化，纳米机器人的尺寸为 800 微米宽，300 微米厚。值得注意的是，它没有马达。一旦模具完成，两个钕磁铁粘在每个外壳的外面。当施加无梯度的外部磁场时，磁铁使纳米机器人的外壳打开和关闭。这些往复运动导致了它在非牛顿介质中的净推进力。科学家们还在甘油中测试了它，甘油是牛顿流体的一个例子。证实了 Purcell 的扇贝定理，纳米机器人没有穿过甘油。他们使用立体镜、带有色玻璃滤光片的数码相机和紫外线 LED 拍摄纳米机器人运动的视频，以使荧光纳米机器人可以被检测到。

科学家们没有指出任何关于这种设计的进一步研究。相反，他们希望这将有助于未来的研究人员设计能够在血管和体液中游动的纳米机器人。我们不知道这需要多少年才能成为主流医学科学，但我们知道这一点:我们再也不会以同样的方式看待扇贝了！
