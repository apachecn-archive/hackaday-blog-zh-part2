# 欧洲核子研究中心竞赛——弗兰肯斯坦博士会感到骄傲的

> 原文:[https://hack aday . com/2015/05/28/caption-CERN-contest-dr-Frankenstein-poor-to-proud/](https://hackaday.com/2015/05/28/caption-cern-contest-dr-frankenstein-would-be-proud/)

欧洲核子研究中心竞赛第 16 周刚刚过去，但在每个写评论的人的脑海中，并不是没有激发一些宇宙漫画天才。感谢所有进入的人！如果你关注了上周的[博客文章](http://hackaday.com/2015/05/21/caption-cern-contest-not-your-fathers-pov-display/)，你已经知道这张图片不是早期的视点显示，或者某种奇怪的数据显示技术。它实际上是一个[火花室](http://en.wikipedia.org/wiki/Spark_chamber)。火花室使用高电压和惰性气体来产生可见的宇宙射线轨迹。由于这张照片拍摄于 1979 年，远在火花室被用于硬科学之后，我们猜测它是 CERN 实验室演示的一部分。

搞笑的是:

*   "在这里，我们看到道格和他隔壁楼的同伴玩大型多人乒乓游戏."–[[约翰·金尼斯顿](https://hackaday.io/JohnKiniston)
*   它说“你想玩游戏吗?”？“我就说好。那些是导弹发射轨道吗？”- [ [琼斯米尔](https://hackaday.io/jonsmirl)
*   "在 Arduino 出现之前，你需要一整个房间的设备来闪烁发光二极管！"–[[mjrippe](https://hackaday.io/mjrippe)]

在连续两周获得亚军后，本周的冠军是绿色绅士，他的口号是“等一下，让我调整一下垂直姿势，然后准备一场最正义的 3D 乒乓比赛！”可悲的是，这个欧洲核子研究中心的副产品从未进入市场”

我们确信[绿色绅士]在分享他在 Hackaday 商店的新巴士盗版时会对他的黑客伙伴非常客气！恭喜【绿先生】！
