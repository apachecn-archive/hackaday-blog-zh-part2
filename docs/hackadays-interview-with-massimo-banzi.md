# Hackaday 对 Arduino 首席执行官[Massimo Banzi]的采访

> 原文：<https://hackaday.com/2015/07/07/hackadays-interview-with-massimo-banzi/>

我在深圳创客大会上采访了[Massimo Banzi],谈论了中国的制造业、Arduino 的现在和未来，以及最近的事件可能如何塑造开放硬件的格局。

[https://www.youtube.com/embed/q7DhCxjW9OI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/q7DhCxjW9OI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

来自 SZMF 的 Arduino 的大新闻是[与 Seeed Studio](http://www.seeedstudio.com/blog/2015/06/26/the-new-partnership-arduino-llc-seeedstudio/) 合作制作 Genuino。这是中国为中国市场制造的官方 Arduino 板。知道董事会是官方的，并与创始人有联系是让制造商采用这种硬件的关键点。(马西莫)对“自豪的中国制造”的理想提出了很好的观点，我可以将它视为中国新兴制造商市场的一个卖点。在中国，这可能是一个日益增长的原则，但在克隆板的海洋中，这听起来像是一条艰难的前进道路。另一方面，他们的摊位被下新订单的人团团围住。

[马西莫]相信目前的阿杜伊诺冲突实际上有助于推进该项目。他引用 arduino.cc 和 arduino.cc 之间的分歧来催化 Adafruit Industries 和 Seeed Studios 之间的制造伙伴关系。这使得 Arduino 的官方硬件不仅仅是在意大利制造，而是在该硬件将被使用的地区制造；美国订单在纽约，中国订单在深圳。

我们的讨论以[Massimo]请求 Hackaday 社区对使用 Arduino 的项目不要那么善变而结束。那个让我有点窃笑！
