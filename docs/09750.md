# 适合苗条身材的木制自动扶梯

> 原文:[https://hack aday . com/2015/08/09/wooden-ladder-fit-for-a-slinky/](https://hackaday.com/2015/08/09/wooden-escalator-fit-for-a-slinky/)

我们最喜欢的木工机械大师【马蒂亚斯·万德尔】又来了，这次[为一个苗条的](https://www.youtube.com/watch?v=5UP_njBPpZ4)做了一个无尽的楼梯。当浓缩到一个两分钟半的视频时，用 2×4 和其他木材碎片制作一个自动扶梯看起来相当容易。事实上，像这样的工作需要大量的切割、破洞和大量的计划。

这种构造的难点似乎在于马达的配置。当谈到苗条的自动扶梯速度时，有一个甜蜜点。太快了，你会超过苗条的。太慢了，苗条的人会从自动扶梯的尽头飞出去。对于钻机触发器的粗略速度控制，保持速度在检查中被证明是一项困难的任务。解决办法是扔掉钻孔机，建造一个简单的手动曲柄机构。现在，只要你伸出手臂，Slinky 就可以从楼梯上滚下来。

休息后加入我们，观看 3 个视频，自动扶梯的制作，140 步的演示视频，以及后续视频(针对我们这样的极客),解释创意的来源，机器的问题以及可能的改进。
