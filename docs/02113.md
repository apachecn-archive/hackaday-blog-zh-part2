# 将 3D 快门眼镜变成自动太阳镜

> 原文:[https://hackaday . com/2012/11/14/将 3d 快门眼镜转换为自动太阳镜/](https://hackaday.com/2012/11/14/turning-3d-shutter-glasses-into-automatic-sunglasses/)

![](../Images/103adb922ee00188be9cdf7fbfb2748f.png "automatic-sunglasses")

本周，迪诺的黑客们试图创造出一种能够根据环境光线的强度而变暗的太阳镜。想法是，这应该给你最好的照明水平，即使是在亮度变化的情况下，比如当太阳在云后或从里到外行走时。他从一副 3D 快门眼镜开始。这些镜头每个都是一个液晶面板。这款眼镜会监控来自 3D 电视的红外信号，然后交替关闭镜片，以便每只眼睛看到不同的视频帧，从而产生立体效果。在休息后的视频中，他拆除了硬件，并用自己的环境光传感器电路重新组装起来。

只需 6V 电压就能立即使其中一个液晶面板变暗。有趣的是，它们需要几秒钟才能再次变得清晰。事实证明，为了在两个方向上都有快速的响应，你需要用一个电阻来释放窗格中的电压。上面你可以看到光敏电阻在框架的桥上，用来触发窗格。[迪诺]在他的视频结尾显示，他们的工作。但太阳镜的主要保护功能是过滤紫外线，他不确定这些是否有这种能力。
