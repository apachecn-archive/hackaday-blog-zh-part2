# 没有回音的语音命令

> 原文：<https://hackaday.com/2015/08/30/voice-command-with-no-echo/>

[Naran]对亚马逊 Echo 控制家用电器的能力很感兴趣，但决定开发自己的。通过在 beta Prota OS 上使用树莓 Pi，他成功控制了一些 Phillips Hue 灯泡和一个家酿智能插座。

Prota 有一个语音应用程序，这使得工作变得更加简单。不过，他确实指出，他的项目不会取代 Echo 通过搜索互联网回答问题的能力。不过，它的优点是很容易根据您的具体应用进行定制。此外，如果你有一个树莓皮挂在周围，你不能击败的价格。

我们不禁想知道这是不是最简单的方法。我们总是有几部没有服务的旧安卓手机。连接到 WiFi，他们用 Google Now 回答问题做得很好，并且具有出色的语音识别能力。一个简单的语音应用程序可以在这里创造奇迹，即使它最终仍然向 Pi 发送电子控制信号。另一方面，很难编写一个 Android 应用程序，它总是在监听，不需要你点击一个按钮(或者至少说“OK Google”)来启动(或重启)监听。

如果你想用另一种方法让树莓派听，试试 Jasper 。或者你也可以一直用——还有什么—[一个 Arduino 和 uSpeech](http://hackaday.com/2013/12/31/an-arduino-with-better-speech-recognition-than-siri/) 来完成这一切。
