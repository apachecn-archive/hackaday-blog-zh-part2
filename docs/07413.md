# 反技术:海军舰船上的火控计算机

> 原文:[https://hack aday . com/2014/10/28/retro tech tacular-fire-control-computers-in-navy-ships/](https://hackaday.com/2014/10/28/retrotechtacular-fire-control-computers-in-navy-ships/)

这是一部 1953 年的分为两部分的海军训练电影，描述了机械火控计算机的内部工作原理。它涵盖了七种机制:轴、齿轮、凸轮、差速器、组件解算器、积分器和乘法器，并以那个时代典型的良好执行方式完成。

火控系统依赖于许多同时发生的因素，不仅仅是本舰的速度和航向，到目标的距离，方位，目标的速度和航向(如果不是静止的)，初始炮弹速度，风速和风向。

该机制介绍了二维的齿条和齿轮演示。原则上讲，轴承载基于转数的值。由此，一个系统可以以不同的比率调整。

凸轮进一步发展了这一思想，将旋转等规则运动转化为不规则运动。它们使用工作表面作为输入，使用从动件作为输出。我们看到了凸轮是如何把旋转运动变成直线运动的。虽然最简单的例子限于单次旋转，但是通过延伸工作表面可以获得额外的旋转。这通常是用一个球在一个凹槽中完成的。

这部电影继续描述了消防系统中的这些机制。它很好地解释了几个不同的凸轮如何将船速的旋转输入作为线性运动传递给从动轮，再输出到其他系统。大多数是根据交付的产出类型恰当命名的；倒数凸轮的输出计算为输入的倒数，方形凸轮的输出为输入的平方。

![superelevations](../Images/251903c46ff7c22320a19db95c8f2eaa.png)切线凸轮的输入是 40°到 70°之间的角度，输出是该角度的切线。飞行时间凸轮将射程作为输入，给出射弹的飞行时间。或许最复杂的是，桶形凸轮获取目标的前进范围和前进高度，并使用它们来计算射弹的超高。它实际上包含无限数量的凸轮，每个凸轮计算不同的超高。通过对伞齿轮种类的直观分解，差速器得到了很好的解释。其中，端齿轮对为星形齿轮的小齿轮提供了环形齿条。

第二部分从组件解算器开始，它解决向固定目标开火的矢量问题。这些通过形成矢量图来提供连续的解决方案，矢量图基于在任何给定点自己的船的速度和到目标的方位。解算器使用凹槽凸轮计算相对于视线的速度矢量，并使用两个开槽齿条计算沿视线的距离变化率和垂直于视线的方位变化率。

圆盘式积分器用于距离保持，其中当前距离等于初始距离和距离变化的代数和。盘式积分器连续计算距离变化，并将其输出到微分器，微分器与初始距离一起计算当前距离。它使用时间盘和从组件解算器发送的距离变化率输出来完成此操作。这种机构就像一个变速齿轮，具有无限的传动比。

最后，乘数用于将两个连续变化的值相乘，这两个值可能是正的，也可能是负的。如果我们可以这么说的话，这个装置非常迷人。所描述的齿条类型由两个相互成直角的输入齿条、一个输出齿条和一个帮助确定零点的固定销组成。两个输入架都沿着秤移动，并在输出架上提供两个输入的乘积。

尽管这些系统很重，占地面积很大，需要大量的电力，但它们的优雅和可靠性还是有很多值得称道的地方。
