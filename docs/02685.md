# 制造线性马达

> 原文:[https://hackaday.com/2013/02/04/building-a-linear-motor/](https://hackaday.com/2013/02/04/building-a-linear-motor/)

![linear-motor](../Images/4abfffe18e2ef244b0a7cbd5b617160c.png)

我们承认这个项目没有太多可用的细节，但它太简洁了，我们不能错过。这是[ligonapproduction]在看到一个商业版本的简短描述后建造的一个小型线性马达。

休息后的视频显示他在测试马达。在这张截图中，他拿着中心轴，而线圈组件前后移动。但是它也可以和移动杆的固定线圈一起工作。马达基本上是一个改良的螺线管。轴内有 16 块钕磁铁。这组四个线圈由一个 ATtiny44 驱动。就像步进电机一样，以正确的顺序给线圈通电会推动稀土磁体产生运动。

我们不确定他对这个建筑是否有任何用处。对我们来说，我们只是想看到实践中的概念(我们对[单极电机制造](http://hackaday.com/2009/11/10/homopolar-motor/)有同样的感觉)。
