# Gamecube 机器人不仅仅是看上去那么简单

> 原文:[https://hack aday . com/2014/01/21/game cube-robot-is than-meet-the-eye/](https://hackaday.com/2014/01/21/gamecube-robot-is-more-than-meets-the-eye/)

[约书亚]有他的旧游戏方块踢来踢去。为了不让它积满灰尘，他把它带到哈维·马德学院的机械车间，用它的身体作为移动机器人的外壳。稍加思考，你就会发现你可以在相当小的 Gamecube 盒子里放很多东西。[Joshua]从几个遥控飞机式无刷外转子马达开始。这些马达通常比内燃马达扭矩更大，转速更慢。几千转/分钟的速度仍然太快，无法直接驱动乐高轮胎。他需要减速。

齿轮和狭小的空间通常会让人们跑去 [SDP/SI](http://www.sdp-si.com/) 网站。我们以前使用过 SDP/SI 零件，发现它们能制造出极其精确的齿轮和组件。然而，当你每样东西都买两个的时候，东西会变得很贵。为了在他的大学生预算范围内寻找更多的解决方案，[约书亚]研究了无线电控制伺服系统。遥控伺服有一些相当强大的输出齿轮，尤其是金属齿轮品种。即使有坚固的齿轮，零件也会在碰撞中损坏，所以替换齿轮组是可以买到的，也很便宜。[Joshua]选定了 Hitec 伺服系统的齿轮。他的下一个问题是为他的马达找到一个小齿轮。事实证明这很容易，因为 64 节齿轮通常用于钢筋混凝土汽车啮合公制伺服齿轮。最后的结果很棒。他的机器人扭矩大，速度快。它唯一缺少的是一个大脑。休息后的视频。
