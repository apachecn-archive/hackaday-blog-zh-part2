# 复古版:本周末 VCF 东 X

> 原文：<https://hackaday.com/2015/04/15/retro-edition-vcf-east-x-this-weekend/>

现在是四月中旬，又到了举办东方电脑节的时候了。X 表示 10。这是一个为期三天的周末，充满了有趣的人、酷的技术和你只听说过的电脑。我们会在那里呆三天，如果你在纽约或者不能自己加油(俄勒冈州除外)，这是一个度过周末的好方法。

本周五的会议将包括从 chiptunes 到 retr0bright 到在 FPGAs 上仿真老式计算机的所有内容。值得注意的会议包括我们自己的[Bil Herd]关于系统架构的演讲。想象一下，一群工程师在一个有白板的房间里。你怎么能不乐在其中。由吉姆·布雷恩领导的四分之一网络委员会也将召开第一次会议。本次会议将讨论在不同型号和不同品牌的老式计算机上实施老式网络协议。迷茫？这是一个“八位世界的两位解决方案。”这就是我们所知道的，我很确定这就是所有人所知道的。T2 将会很有趣。

周六和周日将展出数量惊人的展品，包括从 Atari 8 位、Hollerith 卡、大型机、业余电台(KC1CKV)到 Fairlight CMI 的所有东西。由于这是 PDP-8 的 50 周年纪念，将会有一些这样的古老机器展出。一架刚刚修复的直-8 将开始工作，[来自 hackaday.io](https://hackaday.io/project/4434-pidp-8i) 的令人难以置信的模拟也将开始工作。

仅仅因为有展品并不意味着会谈在星期五结束。周六的演讲嘉宾将是[[Brian Kernighan](http://en.wikipedia.org/wiki/Brian_Kernighan)]，他是 c .上这本书的作者。周日的演讲嘉宾将是 VisiCalc 的联合开发者[Bob Frankston]。整个周末这里都会有非常重要的人物。

即使你不喜欢老式电脑，会场上还是有很多东西值得一看。信息时代科学中心拥有可以追溯到一个世纪前的技术珍品，最近他们修复了一个旧的碟形卫星天线，把它变成了一个射电望远镜。[登记在这里进行](http://www.vintage.org/2015/east/)，如果最后几段没有说服你参加这次活动，你可以看看下面【上师冥想】的 VCF 预览视频。当然，我们会发布很多活动中的东西。
