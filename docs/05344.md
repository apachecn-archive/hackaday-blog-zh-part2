# Arduino 用 Flappy Bit 得到家禽

> 原文：<https://hackaday.com/2014/02/18/arduino-gets-fowl-with-flappy-bit/>

![flappy-bit](img/d42d07d382d68cad8382dbf8c067aaff.png)

我们必须吞下我们的骄傲，把它交给[Dan200]。他可能最终找到了一个所有人都认为非常适合 Arduino 的应用程序。Flappy Bit 是基于【丹】Arduino Uno 的 [Flappy Bird](http://en.wikipedia.org/wiki/Flappy_Bird) 的克隆体。[丹]本质上是个搞软件的人，但他最近迷上了电子产品。Flappy Bit 只是他学习 Arduino 编程的一个有趣的附带项目。硬件包括一个 8×8 LED 矩阵、限流电阻和一个按钮。

[Dan 的]实现并不是 100%忠实于 iOS/Android 的原始版本。他不是简单地鹦鹉学舌，而是做了一点改变。用户按住按钮爬升，松开按钮下降。这似乎让游戏变得更加宽容。我们也不会错过 Flappy Bird 所有可爱的音效。虽然 Flappy Bit 中的拍打声减少了，但它确实让我们更加怀念那些我们在 80 年代玩过的桌面 LCD/LED 游戏，并且今天无法停止吹嘘。

【丹】已经发布了项目的[完整源代码(Pastebin 链接)，在](http://pastebin.com/yjeuTcCL)[的 reddit 线程](http://www.reddit.com/r/arduino/comments/1y6ffx/flappy_bit_flappy_bird_on_arduino/)上有更多可用信息。试试 flappy bit。你不会白鹭吧！
