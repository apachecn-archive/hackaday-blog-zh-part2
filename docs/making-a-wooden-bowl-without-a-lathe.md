# 不用车床制作木碗

> 原文：<https://hackaday.com/2015/07/11/making-a-wooden-bowl-without-a-lathe/>

通常，当制作一个木碗时，工匠会在车床上完成。一大块木头会用螺栓固定在头架上，碗的底部会变成合适的形状。然后将半碗状的木头在车床上翻转，这样碗内侧的材料就可以去除了。这种传统的碗车削方法需要车床、车削工具以及完成任务所需的严格的技术和技能。

怪异木质加工工具的制作大师【伊西】，决定不用车床制作一个[木碗。他创造了一种独特的夹具，在台锯上切割碗的形状，这种设备对于普通 DIY 者来说更常见一些。夹具本身由木材制成，并在垂直位置支撑标准手钻。即将诞生的碗用螺栓固定在钻头上，悬浮在台锯片上方。打开台锯，夹具允许工件来回摇摆，形成碗的外部形状。钻头旋转工件，使碗周围的轮廓一致。](http://lumberjocks.com/projects/93761)

然后将碗翻转过来，重新连接到钻头上。这一次是为了切割碗的内部，夹具被锁定在垂直位置，木材在旋转的同时垂直落在旋转刀片上。锯条在木头上切出一个完美的半球形空腔。经过一点打磨和涂油后，最终的碗看起来棒极了。休息之后请看视频。

这不是[伊西的]项目第一次在 Hackaday 上出现，看看他的 [DIY 带锯](http://hackaday.com/2015/06/21/hand-drill-to-band-saw-conversion/)和[木制球体切割器](http://hackaday.com/2014/05/06/circular-saw-cuts-balls-wooden-balls-dont-worry/)。
