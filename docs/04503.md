# 自由边的无限门户

> 原文:[https://hackaday.com/2013/11/08/freesides-infinity-portal/](https://hackaday.com/2013/11/08/freesides-infinity-portal/)

![sidebyside](../Images/ae158cd34690192a37ba59b878861c6a.png)

如果无限远镜不够酷，10 英尺高的无限远*入口* 会让你大吃一惊。严格来说，镜子本身只有 7'x4 '，但你仍然会发现自己被拱门吞没。这个门户最初是一个简单的原型[，我们在今年夏天早些时候报道过](http://hackaday.com/2013/07/23/building-a-blinky-infinity-mirror/)，它只是一个 2×4 的框架，一些丙烯酸和 LED 条。它的工作原理是将光线放在一面双向镜子和另一面镜子之间，从内部反射大部分光线，创造出深度的错觉。

这座巨大的拱门也是作为一个小规模的原型开始的，它的形状和雕刻是由激光切割机雕刻出来的。一旦他们对它的设计感到满意，就该扩大规模了。全尺寸的入口需要极大的稳定性，所以 Freeside 的人用木制调色板建造了底座。他们需要这个入口去几个不同的场地，所以框架的其余部分被分解成组件，包括一个可移动的木制框架，丙烯酸悬挂在上面。Teensy 3.0 运行所有 WS2812 LED 灯条，选择这些灯条是因为它们的每个 LED 都是可单独寻址的。
