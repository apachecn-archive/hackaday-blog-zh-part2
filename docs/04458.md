# 用喷墨打印机完美印刷电路板

> 原文:[https://hack aday . com/2013/11/02/perfect-PCB-with a-inkjet-printer/](https://hackaday.com/2013/11/02/perfect-pcbs-with-an-inkjet-printer/)

与用调色剂转移法制造 PCB 或为感光板制作掩模不同，家用电路板制造的圣杯是一种直接喷墨蚀刻法。[Don]还没有完全实现，但是他在家里生产电路板的方法是我们见过的最简单的方法之一。

[Don]的电路板首先从 Eagle 获取输出，然后用 Epson Artisan 50 喷墨打印机进行打印。通过在铜板之前将一张卡片粘贴到打印机中，他能够精确地将迹线和焊盘对准铜板。

当纸板从打印机里出来的时候，上面只有墨水。虽然一些特种墨水足以作为抗蚀剂，但[Don]想出了一个聪明的方法来确保酸不会侵蚀所需位置的铜——他只需在复印机或激光打印机的墨粉上撒上灰尘，吹掉多余的墨粉，然后在烤箱中烘烤整个电路板。

如上所述，结果是电路板上完美的轨迹，而不需要在铜板上熨烫照片纸。

至于“为什么没有人早点想到这个”的想法，这是最重要的。[Don]说这种方法应该适用于印刷焊膏掩模的铝片。令人印象深刻的工作，现在唯一要做的事情是让两层板轻拍。要想更直接地了解铜版印刷，请查看过去几年我们报道过的黑客攻击。
