# 悬挂式玻璃音箱看起来超级酷

![FWXDXNDHRCK3KUJ.MEDIUM](../Images/e358733cffd64b838a125e997316450e.png)

寻找一种现代的方式来为你的公寓增添情趣？如果你不是一个发烧友，这些悬挂式玻璃扬声器看起来棒极了！

首先，我们知道你已经在问的问题——它们听起来怎么样？嗯，说实话，没那么糟糕！你可以把它描述成玻璃状的(哈哈)，但是你会惊讶于它的低音有多好。谈到高音时，扬声器受到影响，因为它出来时有点低沉。这可以通过几个策略性放置的隐藏高音喇叭来纠正！

那么它们是如何工作的呢？嗯，像任何扬声器一样，声音来自振动——在这种情况下，玻璃振动产生声音。为了实现这一点，[Evan]正在使用一对 HiWave HIAX32C20-8 触觉传感器，它们实际上是为了将大多数表面变成扬声器而设计的。这个建筑的棘手之处在于如何悬挂它们。

由于房间空间有限，[Evan]选择用电线将扬声器悬挂在天花板上，唯一的问题是钻玻璃不是那么容易。他分享了一些技巧，并最终成功地使用了 Dremel 工具。从那里开始，只需要在天花板上安装一些钩子，然后把它们串在一起。
