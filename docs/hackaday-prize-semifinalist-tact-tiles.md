# Hackaday 奖半决赛:Tact Tiles

> 原文：<https://hackaday.com/2015/08/31/hackaday-prize-semifinalist-tact-tiles/>

世界上有 360 万聋哑人，到目前为止，他们最大的问题是交流。作为 Hackaday 奖的参赛作品，我们自己在 hackaday.io 上的奇迹工作者正在创建一个能够为各种设备实现触觉通信的系统。它被称为 Tact-Tiles，而不是创造一个单一的设备，[Anderson]正在建立一个完整的系统，为聋人提供多种通信设备。

Tact Tile 系统的基本单元是一个小型的触摸感应振动垫。这些微型印刷电路板可以安装在任何东西上，包括有线手套，或者任何人可以想象的任何触觉接口。该设备的核心是一个小型 PCB，可以控制 32 个振动垫，并通过蓝牙连接与智能手机或计算机通信。

通过一点点软件，可以以任何想象得到的方式配置 Tact Tiles，将单个 Tiles 映射到字母表的字母，将手势映射到字母，或者两者之间的任何组合。[Anderson]有一个很棒的视频演示了他的设备的可能性，你可以在下面查看。
