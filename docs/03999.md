# 加工车间

> 原文:[https://hackaday.com/2013/09/06/machining-an-orrery/](https://hackaday.com/2013/09/06/machining-an-orrery/)

什么是地狱？如果你正在看上面的图片，我们肯定你已经想明白了(感谢那些知道这个单词的大脑)。对于那些不明白的人来说，天文望远镜是一个代表行星和卫星运动的机械装置。我们从没想过自己造一个。在看到上面显示的加工过程后，我们不确定是兴奋还是害怕所有的工作。

你可能想拿出 Chromecast，然后坐在沙发上看这个。YouTube 上有几十个视频展示了这一过程。从将板材切割成圆形废料，到制造齿、齿、齿和更多的齿，这不仅仅是齿轮。你也需要球体本身。

我们对第一次在 *[【黑暗水晶】](http://www.youtube.com/watch?v=9KPMc2VowQM)中看到作为场景一部分的奥瑞里有着美好的(也许是可怕的)回忆。*
