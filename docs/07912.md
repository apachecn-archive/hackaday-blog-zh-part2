# 韦斯利时钟上的圣诞村旋转

> 原文:[https://hack aday . com/2014/12/26/Christmas-village-spin-on-the-sley-clock/](https://hackaday.com/2014/12/26/christmas-village-spin-on-the-weasley-clock/)

我们以前见过圣诞村的黑客吗？一个都没想起来，我们的谷歌搜索也没找到。没关系，即使有几个，这个排名也会很高。[凯尔·安德森]建造了他每个亲人居住的房屋模型。当主人在家时，每个型号的灯都会亮起。

这让我们想起了韦斯莱钟，它本身就是一个流行的破解概念。这个想法是，每个家庭成员的位置都用一个独特的时钟指针和钟面上的一组异想天开的位置来显示。

[凯尔]称之为以太屋，执行类似的动作。每个亲人家中的 WiFi 接入点都会被监控，以监控他们的智能手机。当检测到它时，他们家用模型的灯被点亮。因为每个人都有自己的村庄副本，所以每个人都知道谁在家，谁不在家。
