# 复古游戏机，现在带网络收音机

> 原文:[https://hack aday . com/2014/07/03/retro-gaming-console-now-with-internet-radio/](https://hackaday.com/2014/07/03/retro-gaming-console-now-with-internet-radio/)

![RaspPi Retro Gaming Console with Internet Radio](../Images/c93a19da4fb8d39183d17013da7815d4.png)

你曾经怀念过你过去的游戏系统吗？[yv3]他做了一个复古游戏机。尽管[yv3]喜欢他以前的学校游戏，但他不想在玩这些游戏的时候听着以前学校的 8 轨磁带。对他来说，解决方案是建造一个带有集成互联网收音机的复古游戏控制台。

构建的游戏部分依赖于 [RetroPie](http://blog.petrockblock.com/retropie/) 。RetroPie 磁盘映像包含将 Raspberry Pi 转变为专用复古游戏系统所需的所有软件和模拟器。RetroPie 项目支持许多游戏系统，[yv3]选择包括雅达利、世嘉主系统和创世纪、NES、SNES 和 Turbografx-16。

[Raspberry Pi 网络电台](http://bobrathbone.com/raspberrypi_radio.htm)管理该项目的电台部分，并设置为在设备通电时自动开始播放。有 5 个按钮可以改变电台、音量和设置。电台由 SD 卡上的文本文件管理。来自收音机的音频可以被引导至 HDMI 或 RaspPi 的模拟输出。

谷歌 Sketchup 是用来设计的情况下，借鉴雅达利 2600 少年的灵感。所有的内部组件也进行了建模，以确保它们不仅适合，而且箱子上的开口也在正确的位置。一旦箱子设计完成，[yv3]将面板放在一些胶合板上，将它们剪下，然后组装在一起。

如果你仍然渴望复古项目，那么看看这个嵌入咖啡桌的[、这个](http://hackaday.com/2013/11/17/retropie-gaming-table/)[便携式 Pi](http://hackaday.com/2014/06/03/retropie-portable-build-uses-no-fancy-tools/) 或者这个被扔在原始 NES 盒子里的。
