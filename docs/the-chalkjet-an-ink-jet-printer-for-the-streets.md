# ChalkJET:街头用的喷墨打印机

> 原文：<https://hackaday.com/2014/08/29/the-chalkjet-an-ink-jet-printer-for-the-streets/>

需要做一些游击式的街头广告？你需要的是 ChalkJET 9000，一台带轮子的喷墨打印机。

这个小推车使用两个 Arduino Duemilanoves 作为大脑，装有八罐可以单独启动的喷雾粉笔。小螺线管拉下控制杆，以喷洒罐子。推车轮子上的编码器在推车前进时记录每个像素之间的间距。

安装在手柄上的一个小 LCD 可以让你选择你想要打印的文本，但看起来手动输入新单词是不可能的——在上街之前，你需要在连接到计算机的情况下加载一个库！
