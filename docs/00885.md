# 机器人谅你也不敢从这朵花上抢走鹅卵石

> 原文:[https://hack aday . com/2012/06/29/robot-dares-you-skate the pebble from this-flower/](https://hackaday.com/2012/06/29/robot-dares-you-to-snatch-the-pebble-from-this-flower/)

![](../Images/7a2e06b49ca7549e5610955bb2fa80f6.png "robotic-flower-says-snatch-the-pebble-from-my-bloom")

如果你不小心的话，这种看起来很可爱的植物可能会把你的手拿掉。[机器人花](http://www.grobot.gr/index.php?option=com_content&view=article&id=248) ( [翻译](http://translate.google.com/translate?sl=auto&tl=en&js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&u=http%3A%2F%2Fwww.grobot.gr%2Findex.php%3Foption%3Dcom_content%26view%3Darticle%26id%3D248))包括传感器，使花瓣对外部刺激做出反应。

你可以在两片花瓣上辨认出黑色矩形的距离传感器。这些让花通过旋转花茎来追踪物体。但是，如果他们确定物体离得太近，会不舒服，每个花瓣背面的伺服电机会使花朵突然夹紧。

休息后的视频从深入了解项目中使用的硬件开始。一个名为 GRoboduino 的 Arduino 克隆使这个项目变得容易得多，因为它在板上有一堆额外的东西，旨在像传感器和伺服电机这样的东西。花瓣动力伺服系统的安装技术非常吸引人，我们喜欢 Snapple 盖子(可能不是实际的品牌，但你可以看到图片)，它的中心涂有黄色毛毡。最终的外观足够正常，可以适应家居装饰，但它仍然有足够的极客来融化我们的黑客心。
