# Hackaday 奖参赛作品:Python 驱动的科学仪器

> 原文:[https://hack aday . com/2015/06/05/hack aday-prize-entry-python-powered-scientific-instrumentation/](https://hackaday.com/2015/06/05/hackaday-prize-entry-python-powered-scientific-instrumentation/)

Hackaday Prize 和 Hackaday.io 的一个共同主题是工具，以制造更多的工具。有很多人试图制造下一个公交车海盗，简单地测量事物是实现房屋自动化或创造下一个伟大发明的第一步。

在可能是今年 Hackaday 奖最有能力的测量系统中，[jithin]正在开发一个由 Python 驱动的科学仪器工具。这是一个由微控制器供电的盒子，包含几乎所有可以想象的台式电子工具，从恒流源、LCR 计、波形发生器、频率计数器和逻辑分析仪。

这个项目塞满了几乎所有能想到的电子工具；有可编程增益放大器、基准电压源、DAC 和恒流源、运算放大器和比较器，所有这些都连接到一串香蕉插孔。所有这些组件都被捆绑在一个漂亮的 Python 框架中，允许在一个盒子中进行一系列测量。

如果这还不够的话，[jithin]还在为这个盒子研究[无线扩展节点，以便从多个有线不可行的采集点获取数据。该功能使用 NRF24L01+无线电模块；对于许多传感器来说，这是绰绰有余的带宽，并且有足够的空间来放置所有您需要的无线传感器。](https://hackaday.io/project/5971-python-powered-scientific-instrumentation-tool/log/19005-wireless-nodes)
