# 如何将内燃机转换成蒸汽动力

> 原文：<https://hackaday.com/2012/10/04/how-to-convert-an-internal-combustion-engine-to-run-from-steam-power/>

我们不知道将内燃机转化为蒸汽动力所需的东西实际上是相当微不足道的。[David Nash]向我们展示了如何通过对琴弦修剪器的引擎进行改造来实现这一点。这些是用来修剪你院子里障碍物周围的植物的工具。只要是二冲程发动机，发动机的来源并不重要。

这台发动机有一个火花塞，它被拧入气缸体的顶部。[David]拆下这个，装上他的替换硬件。目前，他使用压缩空气进行开发，但将最终版本连接到锅炉上。

在发动机和锅炉之间只有几个重要的零件。有一个内置储油器来帮助对抗蒸汽的腐蚀性。还有一个止回阀。在广告之后的视频中，大卫展示了他用来启动止回阀的一大块圆珠笔。它实际上只是一个活塞向上推动以打开阀门的垫片。在最终版本中，这将被金属杆取代。
