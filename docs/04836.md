# 粒子系统忙箱让宝宝有事可做

> 原文:[https://hack aday . com/2013/12/17/particle-system-busy-box-keep-baby-occupated/](https://hackaday.com/2013/12/17/particle-system-busy-box-keeps-baby-occupied/)

![Busybox](../Images/c156aa6c0a9b1b22126a144461eb9726.png)

任何今天出生的孩子都有一个光明的未来——大部分是看着发光的矩形 80 或 90 年。为了让他的后代能够在一生中看到发光二极管闪烁，Dan[创造了一个忙碌的盒子。它实际上只是一个 Arduino、RGB LED 矩阵和一个编程粒子系统，但如果我们对它着迷，它可能会让婴儿大吃一惊。](https://www.youtube.com/watch?v=fXydJwxM2DY)

这个忙碌盒子的想法源于[一个早期的黑客帖子](http://hackaday.com/2013/05/28/arduino-particle-light-box-generates-animations-from-sound/)，它使用 8×8 的 RGB LEDs 矩阵来创建一个移动的彩色云。[Dan]把这个项目作为一个起点，创造了一个婴儿的忙碌盒子，有四种模式，一定会很有趣。

内部是一个 Rainboduino(一个 Arduino 兼容板),能够驱动 8×8 RGB LED 矩阵。忙碌盒里还塞了一块 9V 电池、电源摇臂开关和四个在各种模式间循环的街机按钮。第一种模式是某种“等离子云”模拟，下一种是“画家”灯光显示。最后两种模式拼出了[丹]的后代的名字，以及字母表中的所有数字和字母。
