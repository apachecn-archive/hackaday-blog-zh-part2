# 外在动机:你认为倒立摆很难

> 原文：<https://hackaday.com/2014/09/11/extrinsic-motivation-and-you-thought-inverted-pendulums-were-hard/>

控制系统课程中一个非常常见的项目是倒立摆。基本上，它是一个安装在线性轨道上的机器人，一个铰链，和一个直立在空中的钟摆。把你的算法做好，你就有了一个钟摆，它似乎可以抵抗地心引力的无情拉力，并且对赛格威、平衡机器人和四轴飞行器的工作原理有了很好的理解。

随着[进入 Hackaday 奖](http://hackaday.io/project/996-GimbalBot)，zakowy 将这一活动推向了一个新的高度。这是一个倒立摆，在万向风扇中有两个反向旋转的螺旋桨，是我们见过的最不稳定的无人机设计。

该建筑的机械结构由碳和环氧树脂框架组成，带有一个可以在 X 轴和 Y 轴上移动的电机支架。这个安装持有两个无刷电机，是相当大的俯仰和滚动伺服驱动。电子设备由四轴飞行器中常见的传感器套件组成——陀螺仪、加速度计、磁力计和气压高度计。一切都由 Arduino Due 控制，从 RC 接收器获取命令，并将遥测数据发送回计算机

[zakowy]的项目没有进入四分之一决赛，但他没有被吓倒。他建造这个奇怪的装置是因为他*能*，而不是因为我们在他的鼻子前面悬挂一些大奖。现在，[扎科维]正在[的测试平台上工作。这东西会飞起来的，这一点毋庸置疑。](http://hackaday.io/project/996/log/9230-planning-for-a-few-instrumented-tests)
