# 透明的 7 段显示器

> 原文:[https://hack aday . com/2014/01/09/a-transparent-7-segment-display/](https://hackaday.com/2014/01/09/a-transparent-7-segment-display/)

虽然[Connor]称它是一项正在进行的工作，但我们对他的透明 7 段显示器的外观留下了深刻的印象。这看起来也很简单。

这个建筑使用了一叠七种不同的丙烯酸玻璃，一个在另一个的前面，每个玻璃的表面都刻有不同的部分。窗格组件位于一个小支架上，该支架放置在七排发光二极管上，每排 5 个发光二极管。[Connor]在七个单独的丙烯酸玻璃之间留有空气间隙，以清楚地区分哪个被点亮，并与 LED 行的间隔相匹配。为了显示一个数字，他只需点亮相应的 LED 行，这些行将光散射到雕刻的部分，而不会溢出到另一个窗格中。
