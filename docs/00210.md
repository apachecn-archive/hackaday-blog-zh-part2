# 自制焊料罐对我们来说也太危险了

> 原文:[https://hackaday . com/2012/04/25/home brew-solder-pot-is-too-dangerous-even-for-us/](https://hackaday.com/2012/04/25/homebrew-solder-pot-is-too-dangerous-even-for-us/)

![](../Images/22b258ae4aa3f6e5cefdedbdcca1a1ee.png "solder pot")

[rue_mohr]正在建造一个六足机器人，这意味着他需要用焊料给一整束带状电缆上锡。使用烙铁来完成这项任务需要太长时间，所以他[制作了一个自制的焊料罐](http://eds.dyndns.org/~ircjunk/tutorials/elex/solderpot/main.html)来快速给所有的电线上锡。虽然[rue]能够快速地在所有这些电线上获得焊料，但我们需要质疑他的方法——他使用卤素灯和反射器来熔化所有的焊料。

这个建筑从一个可回收的卤素灯开始。在拆开整个组件后，[rue]把它重新组装成一个类似焊料锅的东西；凹面反射镜和卤素灯泡完全平放在桌子上，准备接受焊料。

在扳动开关并在反射镜中放入一些焊料后，焊料锅出人意料地工作了。[rue]能够快速给他的带状电缆上锡，卤素灯泡和反射器还没有坏。

这是我们见过的最不安全的焊料锅之一——灯泡很容易爆炸，熔化的焊料随时可能从反射器中涌出。[rue]意识到安全隐患，并确保佩戴护目镜。如果它有效，我们真的不能抱怨。

休息之后，看看[rue]的焊料锅(焊料池中央有一个令人敬畏的温度指示灯)的视频。
