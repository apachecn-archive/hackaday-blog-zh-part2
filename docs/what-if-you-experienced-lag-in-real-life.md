# 如果你在现实生活中经历了滞后怎么办？

> 原文：<https://hackaday.com/2014/04/29/what-if-you-experienced-lag-in-real-life/>

如果你是一个游戏玩家，滞后是你最大的敌人之一。但是如果你在现实生活中经历了滞后会是什么样的呢？想象一下那将是多么令人沮丧！

介绍“滞后生活”——这是一个由互联网提供商阳月华发起的可爱实验。使用 Oculus Rift 开发套件、树莓 Pi、降噪耳机和网络摄像头，阳月华拼凑了一个有趣的社交实验。网络摄像头捕捉音频和视频，并通过 Pi 以可变的延迟向 Oculus Rift 重复，以显示慢速互联网速度的影响。

他们尝试了四种不同的场景。乒乓球几乎是不可能的。舞蹈课就是尴尬。尝试烹饪或吃东西是绝对令人捧腹的。他们甚至尝试保龄球，这也证明比你想象的更难！
