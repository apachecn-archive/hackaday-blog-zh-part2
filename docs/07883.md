# 用乐高开发电影

> 原文：<https://hackaday.com/2014/12/22/developing-film-with-lego/>

现在在家冲洗胶卷肯定是一门几乎被遗忘的艺术，但是仍然有一些非常敬业的人愿意花时间学习这门手艺。[简]是其中的一个例外。他正在用乐高开发 35mm 胶片(荷兰语，[谷歌翻译](https://translate.google.com/translate?sl=nl&tl=en&js=y&prev=_t&hl=en&ie=UTF-8&u=http%3A%2F%2Fwww.janvandenbroekfotografie.nl%2Findex.php%2Fblog%2Fentry%2Fdiy-filmontwikkelen-met-behulp-van-lego&edit-text=&act=url))。

对于构建，[简]使用的是乐高 RCX 1.0，乐高头脑风暴的第一代，在 90 年代末发布。根据易贝的说法，对于可编程乐高来说，这是一个非常便宜的选择。乐高胶片开发者的机械装置由多个化学药品罐组成。电影被装在一个卷轴上，悬挂在乐高式龙门架上，浸泡在每个水箱中一段特定的时间。

硬件的第二个版本 ( [翻译](https://translate.google.com/translate?sl=nl&tl=en&js=y&prev=_t&hl=en&ie=UTF-8&u=http%3A%2F%2Fwww.janvandenbroekfotografie.nl%2Findex.php%2Fblog%2Fentry%2Fik-stop-met-ontwikkelen-film-m-b-v-lego-tenzij&edit-text=&act=url))被设计出来，胶片被装入一个旋转的圆筒中。一系列的化学物质将被泵入这个装置，希望减少所需的化学物质的数量。[这个系统最终是用一辆汽车的雨刷液泵建成的](http://www.janvandenbroekfotografie.nl/index.php/blog/entry/laatste-update-diy-film-ontwikkel-machine)。显然，从用这个系统冲洗的照片来看，这个系统工作得很好。它是简单还是高效完全是另一回事。
