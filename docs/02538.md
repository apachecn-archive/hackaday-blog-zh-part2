# LTC3105 和 LTC3109 能量采集芯片

> 原文:[https://hack aday . com/2013/01/15/LTC 3105-和-LTC 3109-能量采集芯片/](https://hackaday.com/2013/01/15/ltc3105-and-ltc3109-energy-harvesting-chips/)

![a-look-at-energy-harvester-chips](../Images/b2268ec06429945fb16c2ec5d6b440c7.png)

[Shahriar]将最新一集的*信号路径*献给了[查看能量收集芯片](http://thesignalpath.com/blogs/2013/01/03/tutorial-and-experiments-on-energy-harvesting-ics/)。这些部件旨在尽可能有效地从非传统来源收集能量。完整的一集大约有一个小时长，在休息之后嵌入。首先介绍这些器件的一些背景知识，并简要概述各种可用的芯片。每一种都适用于不同类型的能源。

他接着测试并解释了 LTC3105 和 LTC3109。前者显示在开发板上。[Shahriar]将它挂在他的工作台设备上，将其性能与公布的规格进行比较。这最终形成了一个电路，该电路使用太阳能电池作为电源，超级电容器作为储能。后者连接到珀耳帖冷却器，用于将冰块的势能转化为电能，为他的 iPhone 充电约三十秒。这可能对我们上周看到的帕尔贴发电机有用。
