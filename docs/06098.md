# 将乐高变成一台老虎机

> 原文:[https://hack aday . com/2014/05/20/turning-Lego-into-a-groove-machine/](https://hackaday.com/2014/05/20/turning-lego-into-a-groove-machine/)

![lego](../Images/b1371f3b0104480985f8886b2c203b48.png)

上周末不仅仅是关于创造者大会；在斯德哥尔摩，还有另一个 DIY 节，庆祝使电子音乐成为可能的协议。这是 MIDI Hack 2014，而[克里斯蒂安]、[迈克尔]、[布拉姆]和[托拜厄斯]组装了一些非常酷的东西:[一个乐高序列器](http://bauhausinteraction.org/2014/05/midihack2014/)

该系统建立在一个半透明的乐高基板上，悬挂在一个网络摄像头上方，为 OpenCV 和 Python 提供一些好处。从那里，数据被发送到本地仪器机器。有一个使用普通乐高积木的步进音序器，一个控制节拍延迟的推子，和一个用于混响的旋转编码器。

尽管限于螺柱和挂钩，下面视频中的简短演示实际上听起来不错，在推子和基于块的旋转编码器中发现了很多精度。[Kristian]将很快发布代码和更多细节。希望有足够的信息在不同音符的音序器部分使用不同的颜色块。
