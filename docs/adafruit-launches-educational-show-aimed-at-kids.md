# Adafruit 推出针对儿童的教育节目

> 原文：<https://hackaday.com/2013/04/02/adafruit-launches-educational-show-aimed-at-kids/>

![adafruit-kids-show](img/5ef66c4f70391715f675d438a90179df.png)

Adafruit Industries 刚刚发布了一个新的教育系列的第一集，该系列旨在教孩子们关于电子产品的知识。这一集的题目是“A 代表安培”,讲授电流背后的基本理论。这个主题似乎是 A-to-Z 主题电气教程的常见主题。[杰瑞·埃尔斯沃斯]为[做了一集类似的](http://hackaday.com/2011/01/13/let-paper-dolls-teach-you-science/)，但她的节目更针对电子爱好人群。

[利莫尔]和一伙人(科林·坎宁安扮成[安德鲁·玛立·安培])似乎全身心投入到这个项目中。这一集的特色是蓝色木偶 ADABOT，它在这一集扮演学生的角色。在演示了电源断路器跳闸后，本集继续讨论电子流和如何测量电流。

我们都很重视这种教育机会。这个系列的目标年龄组从来不知道没有触摸屏的日子，他们应该至少知道一些关于这些设备实际上是如何工作的。
