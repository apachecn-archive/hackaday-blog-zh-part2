# 学生巨魔反 Arduino 教授与寄生虫单片机

> 原文：<https://hackaday.com/2014/12/25/student-trolls-anti-arduino-prof-with-parasite-mcu/>

像我们的一些爱发牢骚的读者一样，[PodeCoet]的数字子系统教授讨厌每个人把一个 Arduino 捆绑到一个项目上，而一个功能不那么强大、复杂十倍的东西就可以了。一名学生问，他们是否可以用 Arduino 取代整个电路板上的“上行计数器”电路，但由于课程围绕基本逻辑门展开，教授拒绝了他的要求。我们的巨魔没有被吓倒，他将一个微控制器偷偷放入一个芯片中，并用它来拼出粗糙的信息。

没有 Arduino？没问题。他花了 4 次尝试，但[PodeCoet]从所需的电路中掏空了 SN74LS47N 显示驱动器，使其成为 PIC16F1503 控制器的傀儡。PIC 以各种方式模拟驱动芯片——按照命令，它显示上下计数——除了无人看管的 15 秒钟。然后，照片上写的不是数字，而是“你好”，接下来是泳衣通常覆盖的三样东西，最后是身体功能。

对于这样一个简单的黑客，它是精彩而幽默的记录。有注释的进展/失败的图片和视频的黑客工作。

它不像臭名昭著的不可能的 3 LED 电路中的[微观欺骗那样复杂，但它更快地抓住了要点。](http://hackaday.com/2011/12/19/ask-hackaday-troll-physics-edition/)
