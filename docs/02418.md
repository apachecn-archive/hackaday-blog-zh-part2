# 让 TARDIS 内部变大

> 原文:[https://hack aday . com/2012/12/27/making-a-tardis-bigger-on-the-inside/](https://hackaday.com/2012/12/27/making-a-tardis-bigger-on-the-inside/)

![tardis-pan](../Images/45a0b7c1dc7003373c8f26d16b0f4a9f.png)

感恩节期间，[格雷格]有了一点空闲时间，他决定需要一个下午项目。手头上有几块胶合板、一把 xacto 小刀和一些蓝色油漆意味着一个微型 TARDIS 将在他的工作台上成形。完成模型后，【格雷格】继续用一个闪亮的 LED 灯改进它，这时[他想到了给 TARDIS 增加一个内部装饰的想法](http://www.kumparak.com/2012/12/my-lil-tardis-its-bigger-on-the-inside-no-really/)。一个不能错过的好主意，真的。

当然，TARDIS 的外部较小，所以[Greg]需要一种方法来虚拟模拟[9]和[10]家的内部。在玩了几天 Blender 之后，[Greg]有了一个 TARDIS 内部的合理的 3D 复制品。现在唯一的问题是把它展示在前门后面。

[Greg]为他的手机开发了一个小应用程序，它可以读取门后的斑马纹图案，并覆盖 3D 建模的 TARDIS 内部。是的，它只能通过增强现实查看，但从一侧到另一侧倾斜桌面 TARDIS 可以看到整个控制台房间。休息之后，你可以在视频中查看[Greg]的 TARDIS 内部。
