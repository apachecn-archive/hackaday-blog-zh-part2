# 火箭炉燃料压块机

> 原文：<https://hackaday.com/2013/03/13/briquette-press-for-rocket-stove-fuel/>

![briquette-press-for-rocket-stove](img/dd090d599ab6948e682123404313bb1e.png)

外面冷的时候，他用火箭炉取暖。他一直在尝试各种不同的材料作为燃料，当他想到用废料制作煤球时。显然，这个项目是可行的。正如你在上面的图片中看到的，他刚刚用报纸纸浆和锯屑的混合物制成了一块燃料。

背景中看到的带有斧柄的橙色装置是他自己的创作。休息之后，您可以在视频中看到该设备的运行情况。在视频评论中，他还链接了一个 CAD 文件，如果你有兴趣自己制作的话。

如果你对火箭炉感兴趣，总可以选择[建造自己的](http://hackaday.com/2013/03/05/quick-and-easy-rocket-stove/)。
