# CD 执行室砰的一声送走了旧光盘

> 原文:[https://hack aday . com/2015/07/10/CD-execution-chamber-send-old-discs-off-a-bang/](https://hackaday.com/2015/07/10/cd-execution-chamber-sends-old-discs-off-with-a-bang/)

焊接钢安全笼？检查！聚碳酸酯防爆挡板？检查！真空吸尘器电机连接到一个诱人的红色按钮？仔细检查！尽可能破坏性地处理一堆 CD？[Firas Sirriyeh]用他的 [CD Terminator 1.0](https://www.facebook.com/media/set/?set=a.10155718416200137.1073741826.603965136&type=1&l=28a1328e4b) 覆盖了你。

虽然[Firas]的构建日志在描述性文本方面有点欠缺，但确实没有必要。他的照片讲述了这个故事。媒体碎纸机和互动行为艺术作品的组合是一个坚固的建筑，你会希望任何能够投掷锋利的塑料和聚酯薄膜弹片的东西都是这样。[Firas]在 2015 年耶路撒冷迷你创客博览会上展示了他的 CD 执行室(希伯来语；[英语链接](http://makerfairejerusalem.org/english))和 Musara Mix Festival，休息后必看的视频就是在那里拍摄的(略带 NSFW 语)。有些 CD 很快就断气了，但有一种却坚持了很长时间才最终爆炸；你可以看到它以一种几乎像慢动作一样的方式弯曲变形。

对于那些不想为那些烦人的安全问题大惊小怪的人来说，总有这个[自动 CD 发射器](http://hackaday.com/2015/07/08/cd-launcher-looks-dangerously-fun/)可以玩。
