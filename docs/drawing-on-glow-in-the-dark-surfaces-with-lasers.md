# 用激光绘制黑暗表面的辉光

> 原文：<https://hackaday.com/2014/12/17/drawing-on-glow-in-the-dark-surfaces-with-lasers/>

当你有一个电脑控制的激光笔和一大片在黑暗中发光的物质时，你会得到什么？显然是非常酷的东西。[Riley]将远远超出简单激光二极管和伺服构建的伟大构建放在一起。他使用步进电机和一个适当的运动控制软件。

该设备背后的理论很简单——将激光对准黑暗表面的一些辉光——但[赖利]做对了这个项目。激光笔的 X 轴和 Y 轴是步进电机，而不是抖动伺服系统。这些由 Arduino Due 和 [TinyG 运动控制软件](https://github.com/synthetos/TinyG)控制。这不是[莱利]和 TinyG 的第一次竞技；我们在纽约的 Maker Faire 上看到他做钟摆演示[，那绝对是惊人的](http://hackaday.com/2014/09/24/the-tinyg-motion-controller/)。

现在，[Riley]正在拍摄 SVG 图像，将它们转换成 Gcode，并把它们放在黑暗的乙烯树脂上。由于 Hackaday Skull 'n 扳手是 SVG 格式的，所以很容易就能决定用怪异的磷光绿色显示什么。你可以在下面看到一段视频和其他一些视频。
