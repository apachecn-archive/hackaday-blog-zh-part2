# Hackaday 奖参赛作品:3D 打印参数 Motrs

> 原文：<https://hackaday.com/2015/06/06/hackaday-prize-entry-3d-printed-parametric-motrs/>

如果你在建造一个会动的东西，你很可能会使用电动马达。有成千上万种不同的马达，每一种都有自己的特性、速度、扭矩和尺寸。如何选择合适的电机？大多数时候，这是一个受过高等教育的猜测，但[螺线管]有一个更好的想法:[只是 3D 打印一个由计算器设计的马达](https://hackaday.io/project/4593-calculator-for-3d-printed-parametric-motor)，它会给你你需要的属性

Hackaday 奖的参赛作品只是一个基于网络的电机设计计算器，它将扭矩、速度、尺寸或外形作为输入，并给出一个完整的电机设计。当然，你需要在 3D 打印的框架上缠绕线圈，但这个计算器消除了计算电感、线圈电容和构建高效电机所需的所有其他比特的需要。

虽然数百万的实际产品仍将使用现成的电机，但这个项目是一次性的完美选择。如果你想机动化一个望远镜座，这个项目将设计一个电机给定所需的功率和分辨率每一步。如果你想建造一个风力涡轮机，这个计算器会把叶片放在无刷电机的外转子上。这是一个伟大的项目，我们迫不及待地想看到它的成果。
