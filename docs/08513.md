# 用苹果遥控器控制任何东西

> 原文:[https://hack aday . com/2015/03/12/control-any-with-a-apple-remote/](https://hackaday.com/2015/03/12/control-anything-with-an-apple-remote/)

如果你像我们一样，你可能有不止一个苹果遥控器放在零件抽屉里，如果你更像我们，你可能真的很讨厌苹果动不动就使用专有硬件和软件的趋势(有人是 lightning connector 吗？).但是苹果遥控器现在有了希望:[Sourcery]已经完成了一个项目，允许[一个苹果遥控器控制你想要的任何东西](http://duinos.net/show/?id=206)。

这个想法相当简单:一个设备解释来自苹果遥控器的红外信号，然后输出另一个红外信号，可以在非苹果产品上做一些有用的事情。[Sourcery]使用 Arduino 来进行红外转换，以及一组红外发射器和探测器，现在苹果遥控器可以控制任何东西，从立体声系统到电视，到你能想象到的任何东西。它也不会删除苹果遥控器控制苹果产品的功能，如果你需要你的遥控器也能做到这一点的话。

[Sourcery]指出，有时处理原始红外信号可能有点困难，但他们的项目和 25 分钟视频中的信息讨论了如何处理这一问题，所以请务必在休息后查看。没有苹果遥控器？你可以用 PS3 控制器做类似的事情[。](http://hackaday.com/2014/03/03/remote-control-anything-with-a-ps3-controller/)
