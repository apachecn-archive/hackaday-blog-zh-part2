# 刺绣 Nyan 猫给现实世界带来了一个迷因

> 原文：<https://hackaday.com/2015/05/09/embroidered-nyan-cat-brings-a-meme-to-the-real-world/>

你有没有遇到过网络迷因，然后对自己说，“我必须把它带到现实世界中来！”嗯[0xb3nn]和[Knit Knit]做到了。他们决定采用经典的 nyan 猫迷因，并赋予它生命。

这个框架是 24 英寸 x 36 英寸。编织过程花了很多时间，但结果显然非常好。星星包括 24 个 LED 亮片，增加了闪闪发光的动画效果。这些是用导电线缝在作品背面的。它们足够亮，可以照到前面需要的地方。这些连接回 Arduino Pro Mini 5V 板。

Arduino 还连接到电容式触摸传感器。这允许用户简单地将他们的手放在 nyan 猫图像上来开始动画。不需要物理按钮或开关来脱离视觉设计。Adafruit AudioFX 声卡用于通过几个扬声器播放保存的 nyan cat 主题曲。这个项目的源代码可以在 [github](https://github.com/b3nn/NyanCat "github") 上找到。请务必观看下面的演示视频。
