# LED 百叶窗将窗户变成了显示屏

> 原文：<https://hackaday.com/2013/10/10/led-blinds-turn-windows-into-displays/>

[Dinofizz]几乎完成了他的垂直 LED 百叶窗。该构建使用 768 个漫射白光 LED(10mm 大小)，分辨率为 48×16，并且它只需要一个 16 通道 LED 驱动器(MBI5026)，该驱动器使用 3 个 4 至 16 解复用器。我们提到过它也有 16 种灰度吗？

许多精心焊接的电线的核心是 ATmega644A 微控制器，它负责为显示器解释数据。虽然固件不是他自己写的，但这要归功于[Jay Clegg](http://www.planetclegg.com/projects/QC-Peggy.html)，他和[Evil Mad Science 的](http://shop.evilmadscientist.com/)Peggy 2.0 LED 驱动程序一起做了一些非常酷的工作。

我们真正不得不佩服的是他为这个项目付出的努力。他使用定制的 PCB 将百叶窗菊花链在一起，300 英尺的 16 路带状电缆，以及大约 4000 个单独的焊点！你以为会有更简单的方法！

利用他的高层窗户，他现在有能力向全世界传播信息。休息之后，看看他们的视频！
