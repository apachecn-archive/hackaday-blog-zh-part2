# 由继电器和磁铁组成的简易 LED 手电筒

> 原文：<https://hackaday.com/2014/01/13/a-simple-led-flashlight-composed-of-a-relay-and-a-magnet/>

在我们的提示热线中，我们有时会收到令人惊讶的黑客攻击，仅仅是因为他们的独创性。这个[继电器供电的手电筒](http://www.instructables.com/id/Simple-squeeze-led-flashlight/?ALLSTEPS)绝对是其中之一。它的创造者[Berto]将其命名为 RattleGen，显然他经常制作日常生活中使用的简单黑客工具(看看他的 [YouTube 频道](http://www.youtube.com/user/Bertoaus))。

为了理解这个技巧，你首先需要知道(如果你还不知道的话)在导体(这里是线圈)附近移动的磁铁会在其终端感应出电压。这叫做[电磁感应](http://en.wikipedia.org/wiki/Electromagnetic_induction)。在上图中，你可以通过杠杆末端的磁铁来区分一个拆开的继电器。由于铁磁性金属已经放置在线圈内，杠杆默认“卡”在这个位置。通过持续按压后者的另一端，会在线圈端子处产生重要的电压尖峰。[Berto]因此使用桥式整流器将交流电转换成 DC，并使用一个 1000uF 的电容器来平滑发送到他的超亮 LED 的电源。在休息之后嵌入了该系统运行的视频。
