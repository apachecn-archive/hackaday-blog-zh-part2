# THP 词条:SatNOGS

> 原文：<https://hackaday.com/2014/06/13/thp-entry-satnogs/>

这里有一个有趣的想法:由于美国国家航空航天局的推广计划，花费一万美元建造一个立方体卫星，并免费搭乘一次发射是可能的。跟踪这颗卫星的整个轨道需要几十个地面站，所有地面站都配备有天线、USB 电视调谐器和互联网连接。实际上，建造和发射一颗立方体卫星的成本比建造一个地面站网络以从立方体卫星获得合理的实时遥测数据的成本更高。看起来，未来是令人敬畏的和不可思议的。

这是 SatNOGS 背后的人所做的观察。他们正在为一个完全开源的地面站网络开发一个平台，这个平台的想法是每个经度和纬度的人们和机构将建造一个简单的卫星跟踪天线支架，将其连接到互联网，并成为开源近空间网络的一部分，能够接收来自任何一个在低地球轨道上呼啸而过的小立方体卫星的遥测数据。

尽管这可能是开源硬件中最雄心勃勃和意义深远的项目之一，但该系统的设计相对简单:[硬件](https://satnogs.org/hardware.html)是一个 3D 打印的 alt-az 支架，能够将一对天线指向天空中的任何地方。[步进电机驱动板](https://github.com/satnogs/satnogs-arduino)基于 Arduino，运行各天线节点的计算机由 BeagleBone Black 或 WR703N 路由器供电。当然，天线接收器是一个 RTL-SDR 加密狗，能够收听所有常见的立方体卫星波段。甚至[软件](https://satnogs.org/software.html)也是来源于开源项目。通过 GPredict 可以计算出在空中跟踪一颗卫星，该团队正在开发一个观测调度和管理系统，该系统将多个地面站结合起来，覆盖全球。

这是一个伟大的想法，从全球各地的人们那里众包卫星跟踪，这可以被数百个足够幸运的机构用来将一小块电子设备发射到轨道上。
