# POV Display 以低廉的成本做到了这一点

> 原文:[https://hack aday . com/2014/07/23/POV-display-do-it-on-the-cheap/](https://hackaday.com/2014/07/23/pov-display-does-it-on-the-cheap/)

![lowBudgetPOV](../Images/604e4acec7e2182c3f5bb2956beebd27.png)

[Sholto]一起黑了这个超[低预算旋转显示器](http://www.instructables.com/id/Digital-Zoetrope/)。他称之为 [zoetrope](http://hackaday.com/2012/08/02/digital-zoetrope-uses-18-lcd-displays/) ，但我们认为它实际上是基于 LED 的[视觉暂留](http://hackaday.com/2013/03/06/pov-pong-game-uses-all-kinds-of-smart-design/) (POV)事件。过去我们已经见过很多 POV 设备,但是这一个证明了黑客并不一定要昂贵或者漂亮才能工作！

POV 显示的主要部分是[Sholto]放在周围的东西。几个糖果罐，一个简单的刷式业余爱好电机，一个 Arduino Pro Mini，7 个绿色 led，一个旧的霍尔效应传感器都是必需的。花哨的显示器可能会使用商业滑环来传输电力，但[Sholto]让它以低廉的价格工作！

这两个罐为显示器提供了底座，并为 Arduino 提供了负电源。锡焊接在一起，并与电机绝缘，电机热粘在较低的锡。一个回形针接触盖子的内部，使整个组件成为 Arduino 电源负极的滑环。一些铜编织摩擦电机的金属外壳形成积极的一面。

[Sholto]选择他的电阻来稍微加速他的绿色发光二极管。这使得显示器在使用 POV 时显得更亮。在正常操作期间，led 不会被驱动足够长的时间而导致损坏。如果软件锁定时 led 灯亮着，那么一切都完了！

[Sholto]包括一个非常酷的“saw wave”演示软件和一个简单的数字显示。只要再多做一点工作，这就可以成为一个非常酷的视点时钟，至少只要电机刷还能用！
