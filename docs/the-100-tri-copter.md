# 100 美元的三架直升机

> 原文：<https://hackaday.com/2012/08/24/the-100-tri-copter/>

[https://www.youtube.com/embed/tqHwqWaw1o4?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/tqHwqWaw1o4?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

我们见过很多廉价的三驾直升机，[但 100 美元对我们来说似乎是一笔大买卖](http://www.youtube.com/watch?v=tqHwqWaw1o4&feature=plcp)！看这个视频，你可以看到这种国产的直升机非常灵活，似乎操控得很好。令人惊讶的是，[hallstudio]声称它的成本大约为 100 美元。与最便宜的直升机相比，这个价格真的很不错。

与这种东西相关的大部分制造成本已经被去除，因为车身只是当地五金店的廉价木材。他甚至为他的尾桨做了一个公认的马虎的装备，这看起来并没有影响他的表现。一个很酷的特点是，你可以向后折叠前臂，使三架直升机可以塞进包里，便于运输。

你可以在他的视频中找到完整的零件清单，但看起来他的成本可能没有计入无线电控制器的成本。没有构建说明，但快速的谷歌搜索将我们带到了 rcexplorer tricopter ，这似乎是他使用的模板。那里有完整的构建细节。
