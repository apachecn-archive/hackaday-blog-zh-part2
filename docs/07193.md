# THP 半决赛选手:OSHWatch

> 原文：<https://hackaday.com/2014/09/28/thp-semifinalist-oshwatch/>

不，这不是 Cupertino 精心制作的腕表配件，而是[【Jared】的 OSHWatch](http://hackaday.io/project/2263-OSHWatch) ，但由于开放式设计和合理的手工焊接布局，你实际上可以制作这款腕表。

围绕在 DealExtreme 上发现的一个看起来可疑地类似于用来容纳 iPod Nano 的外壳的外壳而构建的，[Jared]的智能手表包括 128×128 RGB 有机发光二极管显示屏、磁力计、加速度计、蓝牙 4.0 收发器以及锂离子充电器和调节电路。一切都由 PIC24 控制，这意味着这款手表有足够的处理能力来处理手表应该处理的任何事情。

至于用户界面和这款手表的实际功能*【Jared】为这款手表重新设计了一些 Android 图形。现在，手表可以显示时间(natch)、日程安排中即将到来的约会、加速度计和磁力计数据，以及来自 CPU 的调试数据。这是非常非常好的组合，重新利用现有的手表外壳是一个非常巧妙的想法。下面的视频。*
