# 8X8X8 立方体入侵者

> 原文:[https://hackaday.com/2014/02/08/8x8x8-cube-invaders/](https://hackaday.com/2014/02/08/8x8x8-cube-invaders/)

![F98YM6YHQQLNHN7.LARGE](../Images/4af0462dc0109c51f6591a45e9df9a07.png)

信不信由你，[Anred Zynch]在开始这个项目之前没有焊接技能！我们在这里看到的是一个 8x8x8 的 LED 立方体，被设置为一个带有 Playstation 1 控制器的太空入侵者风格的游戏。

他受到了其他几个立方体的启发，比如[Chr's]的，以及[Das-Labour]的[博格立方体](http://www.das-labor.org/wiki/Borg3d_Bauanleitung)。该项目利用 Arduino Mega 2560 R3 来驱动 512 LED 阵列，并利用 Arduino Uno 来处理游戏过程中的声音效果。这有点像《太空入侵者》——但是是 3D 的！

撇开构建和布线的复杂性不谈，[Anred]已经为整个项目提供了很好的说明和代码，所以如果你想重新创建它或类似的东西，你可以！它现在也进入了 Instructable 的比赛，所以如果你喜欢它，我们肯定他会感谢投票。
