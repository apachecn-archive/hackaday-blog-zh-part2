# 气动狼獾爪子很可能是有史以来最好的东西

> 原文：<https://hackaday.com/2014/05/16/pneumatic-wolverine-claws-are-quite-possibly-the-best-thing-ever/>

随着新的《x 战警》电影即将上映，[柯林·福尔泽]决定从电影中制作一些真正的工作道具——从一些血腥辉煌的全功能可伸缩金刚狼爪子开始。

在之前，我们已经看过[狼獾的爪子，甚至](http://hackaday.com/2005/09/24/retractable-wolverine-claws/)[电狼獾的爪子](http://hackaday.com/2013/05/31/already-impressive-wolverine-claws-now-energized-with-high-voltage/)，但我们从未见过任何人把它们带到【科林】的水平。他不只是想要逼真的狼獾爪子。他不只是想要可以展开的爪子。他想要既能张开又能缩回的逼真的爪子——要快！他希望他们像真正的交易一样扩展业务。

他开始集思广益，用不同的方法来做这件事。马达、弹簧、齿轮齿条、电缆、气压缸……没有一样东西真正符合这个要求。气动动力似乎是性能的最佳选择，但问题是他需要一个 12 英寸的气缸来坐在他的爪子后面——这会彻底破坏外观——这是他项目的主要标准之一。

就在那时，他灵光一现。如果他用 o 型密封圈，把叶片变成导轨上的气动抛射物呢？在他的工作室里使用了一系列令人敬畏的玩具，包括 TIG 焊机、台式研磨机、研磨机等等，他制造了一个令人敬畏的双作用无活塞气压缸，可以根据需要高速来回喷射他的三叶爪子！

唯一的缺点是他需要携带一个装有阀组和储气罐的背包——但是一旦你看到下面的视频，你就会发现优点很快超过缺点。这真是太棒了。
