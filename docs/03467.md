# 一款迷人的 Arduino 眼镜王蛇游戏

> 原文:[https://hack aday . com/2013/06/22/a-charming-arduino-king-cobra-game/](https://hackaday.com/2013/06/22/a-charming-arduino-king-cobra-game/)

![cobra](../Images/3de01ebe67eb2867ded77763ba0cc0a7.png)

[这里有一个来自【A.J .】实验室的有趣游戏](http://www.youtube.com/watch?v=MpuxtmJ7BqA)。这是由 Arduino，超声波传感器和伺服系统制成的模拟眼镜王蛇。该游戏的目的是在设备前面抓住一个球，而不被连接到由伺服系统控制的吸管的图钉“咬”到。就像真的眼镜王蛇一样。

这个项目没有示意图或建造计划，但看起来很容易拼凑起来。尽管简单，这个游戏看起来非常有趣，而且很可能通过使用机器而不是简单地建造它来提供更多的娱乐。

没有文字描述这个游戏可以做到公正，所以看看下面的视频。这看起来很有趣，如果你已经在某个地方的箱子里有了零件，这将是一个很好的周末构建。
