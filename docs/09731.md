# 机器人在轨道上进行延时摄影

> 原文:[https://hack aday . com/2015/08/05/机器人在轨道上进行延时摄影/](https://hackaday.com/2015/08/05/robot-on-rails-for-time-lapse-photography/)

当你把一个摄影师和一个 Arduino 黑客交叉时，你会得到什么？如果有问题的十字是[nukevoid],你会得到一个聪明的相机轨道,它可以平滑地移动，并具有移动和旋转功能。令人印象深刻的构建使用了 Arduino Pro 迷你板和两个步进电机。一个步进器使用一些 Delrin 滑轮作为轮子在挤压铝轨道上滚动，在轨道上移动设备。另一个步进器旋转摄像机平台。

旋转平台很酷。这是一个塑料圆盘，边缘有一条 GT2 运动带。步进电机有一个匹配的滑轮，可以很容易地旋转平台。GT2 带只绕了磁盘的一半，大概软件知道什么时候根据步数在两边停下来。甚至还有一个支架可以在操作时稳定相机的镜头。

一些 AA 电池提供电力(因此可能不会运行一整天而不更换电池)。从视频来看，整个布景是防猫的(虽然没有什么是完全防猫的)。摄影师是一群创新的人，我们以前见过安卓驱动的轨道和 T2 旋转的转盘。
