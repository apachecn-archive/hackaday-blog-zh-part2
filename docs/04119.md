# Talkbot:面向初学者的 Arduino 驱动机器人

> 原文:[https://hack aday . com/2013/09/21/talk bot-an-arduino-driven-robot-for-初学者/](https://hackaday.com/2013/09/21/talkbot-an-arduino-driven-robot-for-beginners/)

![talkbotguts](../Images/6c094c4e96c724e46f10a402b038d013.png)

这并不完全是瓦力，而是[Bithead]对机器人的平价介绍——talk bot—*是由垃圾桶制成的*。这个小家伙使用 Arduino，并配备了各种功能，包括语音芯片、电机护罩和一对碰撞传感器。Talkbot 将四处巡游，直到一个碰撞传感器撞上一个障碍物。当他倒车、转向并试图找到一条更清晰的道路时，他的一条预先录制的信息将通过扬声器播放。

根据[Bithead]的构建日志，寻找合适的议价语音芯片有点麻烦；他跳过了文本到语音的选项，只是因为供应商的问题而停滞不前。他最终选定了来自易贝的 Sparkfun 的 WTV020SD 芯片的克隆产品，这种芯片可以让你访问存储在微型 SD 卡上的预先录制的 WAV 文件。机器人的身体直接从五金店的货架上下来，用 PVC 管做手臂，用聚苯乙烯底座来固定所有部件。以 110 美元的低价,[Bithead 的]学生将有一次真正的黑客体验，将说话机器人拼凑在一起，而不是使用预制套件。

一定要在下面的视频中看到 Talkbot，表演他的绿眼“友好模式”或红眼“暴躁模式”，这决定了他如何愉快地应对障碍。需要更高级的吗？看看触手机器人，正好赶上万圣节。
