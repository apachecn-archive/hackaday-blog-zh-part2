# 仅用镜头创建的断层摄影图像

> 原文：<https://hackaday.com/2014/12/17/lumographic-images-created-with-lens-only/>

发光图像是你在游泳池底部看到的那些图案。当水作为一个透镜时，明暗的光图案是随机的，并基于上面的波浪而游荡。【马修】想出了一个办法[仅凭镜头形状创造固定的图像](http://www.youtube.com/watch?v=nP6mHIB8Wqo)。只有当光线以合适的角度照射时，图像才能清晰地进入视野。在近角处，会出现一种怪异的有趣的镜面效果，但是离得太远，它会散射得无法辨认。

设计光学系统的确切方法没有被解释，尽管我们确信我们的读者中有人能弄清楚。这位艺术家声称这是一个有百年历史的百万变量数学问题。[镜片](http://www.zintaglio.com/lens.html)通常很厚，不像任何东西。然而，效果是尖锐的，清晰的和详细的。

起初，他怀疑自己需要天文数字般昂贵的军用级 50 纳米(0.000002 英寸)透镜精密加工，但一些汽车行业的朋友给了他一些提示，让他从更实惠的工业机器中挤出足够好的精度。该技术还允许图像从镜子和内部反射中出现。这可能不是你可以自己 [3D 打印](http://hackaday.com/2014/12/13/3d-printed-lenses-open-up-possibilities/)或[机器](http://hackaday.com/2014/07/12/one-small-step-for-magnification-one-giant-leap-for-home-lens-manufacturing/)的东西，但看到有人尝试会很有趣。

[马修]的作品在纽约国家数学博物馆的“[综合](http://www.zintaglio.com/)”画廊展出。休息之后，请观看视频，了解他用来操纵镜头以增强展览视觉效果的机器。
