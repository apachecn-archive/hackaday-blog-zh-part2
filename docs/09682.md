# 偏振照相机观察不可见的东西

> 原文：<https://hackaday.com/2015/07/28/polarization-camera-views-the-invisible/>

光的偏振是一种有趣的现象，在许多情况下都非常有用…但是人类的眼睛无法察觉任何偏振。幸运的是，[大卫]已经使用树莓皮和一些现成的组件建造了一个偏振敏感相机，允许任何人查看偏振。[David]列出了以下应用程序:

> 一个偏振成像仪，可以探测看不见的污染物，定位地雷，识别癌组织，甚至可以观察隐形的不明飞行物！

该建筑使用标准的 Raspberry Pi 2 和 500 万像素的摄像头，摄像头位于软件控制的电光偏振调制器后面，电光偏振调制器是从自动变暗焊接掩模中提取的。该面罩实质上是一个专用的液晶显示屏，易于电子控制。[David]在控制屏幕的 Pi 上编写了一些脚本，这就是相机如何能够查看光的各种偏振。由于偏振调制器是由软件控制的，基本上任何角度的光都可以通过计算机以任何方式进行分析。

在项目网站上有大量关于这个项目的信息，在[项目的官方博客](http://www.diyphysics.com/2015/07/27/dolpi-a-low-cost-raspi-based-polarization-camera/)上也有。已经有其他项目将[偏振光用于特定应用](http://hackaday.com/2014/01/07/homemade-polariscope-is-super-easy-to-make/)，但这是我们第一次看到软件控制的偏振相机，旨在用于一般用途，几乎任何人都可以制造。
