# LayerOne Demoscene Demoboard 党

> 原文：<https://hackaday.com/2015/05/26/layerone-demoscene/>

LayerOne 会议结束了，这意味着上周末见证了美国最大的演示派对之一。谁赢了？一支欧洲球队。我们应该预见到这一点。

LayerOne 演示组件分为两类，第一类仅使用 [LayerOne 演示板](http://store.hackaday.com/products/layerone-demoscene-board)。这是一个带有 PIC24F 微控制器、VGA 输出和 1/8”单声道音频输出的板。就是这样；从这个板上出来的所有东西都是手工编码在图片上的。几个月前，【JKing】[写了一个演示来演示这个演示板能做什么](https://www.youtube.com/watch?v=ub-F2Tb9r0o)。据他所说，这是 Hackaday 在商店里卖出一块演示板的唯一原因:

[https://www.youtube.com/embed/kJ4ZoxQQiKU?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/kJ4ZoxQQiKU?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

Demoscene 板竞赛的第一名被一个名为 COINE 的远程参赛团队获得。视频和房间里每个人的最初反应:

[https://www.youtube.com/embed/yShLibHRJ-Q?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/yShLibHRJ-Q?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

这个想法中没有人知道这是如何实现的。硬件应该不能做到这一点。分辨率和颜色数量太高。这是迄今为止 LayerOne 最令人印象深刻的演示。这并不意味着 Demoscene 板竞赛的其他提交内容被忽略了。[jamisnemo]的参赛作品很受欢迎，尽管他已经没时间写了:

[https://www.youtube.com/embed/Cu7p3W2zKgY?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/Cu7p3W2zKgY?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

LayerOne 演示比赛的第二个类别是“秘密”板。这种板子只制作了 10 或 12 块，但仍有一些令人印象深刻的作品。该板本身是围绕 atmega 88–8k 闪存、1K RAM 和 512 字节 EEPROM 构建的。如果使用 ATMega88 作为演示平台听起来很熟悉，[你可能是对的](http://www.linusakesson.net/scene/craft/)。[lft]早在 2008 年就围绕这个芯片制作了 Craft demo。Secret Board 旨在运行该演示，并作为在“Mega88:

[https://www.youtube.com/embed/5j2SQGi14CI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/5j2SQGi14CI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

总之，这是一场精彩的比赛。它受到了所有与会者的好评，明年的展会肯定会更大。如果任何人有任何关于大欧洲如何捕捉这些视频演示的想法，请在评论中留言。LayerOne 没人能搞清楚。
