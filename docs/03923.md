# [更新]弗拉基米尔的机器人吉他

> 原文:[https://hack aday . com/2013/08/26/update-Vladimir s-robot-guitar/](https://hackaday.com/2013/08/26/update-vladimirs-robot-guitar/)

![vladGuitar2013](../Images/84c1bc21b1870753e72b0bd084f1e8d3.png)

[弗拉基米尔·戴明]对我们来说是一个传奇；在业余时间，他一直在掌握乐器的自动化。这一次[他带着升级版](https://www.youtube.com/watch?v=7Xlm8LFImno)和四个新视频回来了。[Vladimir]的首要任务是重新设计早先在螺线管上运行的弹奏机制。他改进了音质，并通过更换步进电机和修改软件减少了咔哒声。

与他早期的设置相比，这个听起来更有感情，自动化程度更低，但[Vladimir]承认它仍然不够好，他正在开发一个新的，出色的实现。在那之前，花几分钟看看下面剩下的视频，然后和我们一起惊讶地挠头:一切都是用简单的手工工具建造的。

[Vladimir]走过了漫长的道路，它从这个 Bayan(按钮手风琴)开始。去年的[吉他打造](http://hackaday.com/2012/05/21/player-guitar-sounds-wonderful-makes-us-drool-because-of-the-complexity/)也值得一看，还有一个[深度访谈](http://hackaday.com/2012/05/25/interview-vladimir-denin-the-creator-of-the-player-guitar/)。
