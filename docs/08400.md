# Openhand 将 3D 打印与聚氨酯铸造相结合

> 原文:[https://hack aday . com/2015/02/25/open hand-combines-3d-printing-with-urethane-casting/](https://hackaday.com/2015/02/25/openhand-combines-3d-printing-with-urethane-casting/)

耶鲁大学给我们带来了他们的开放式项目。

如果你曾经操作过机械臂，你就会知道最笨重的部件之一总是末端执行器。它会很快让你意识到人类的手是一件多么神奇的工程，一个简单的开合手爪最终是多么糟糕。

[耶鲁大学]正致力于将肌腱驱动的机器人手带到大众面前，这是一种结合 3D 打印和树脂/聚氨酯铸造的有趣技术。这被称为混合沉积制造(HDM)，它允许团队 3D 打印机器人手指，其中也包含手指垫和关节的模具，所有这些都内置在 3D 部件中。肌腱驱动的手指允许非常简单的设计，不仅易于制造，而且部件数量也很少。由于类似人类的肌腱，手指自然地弯曲在物体周围，更加均匀和自然地分配力量，就像人手一样。在休息后的[视频](http://www.youtube.com/watch?v=C340gbK3sZc)中，你可以看到建造过程，以及手的动作。

最好的消息是[都是开源的](https://github.com/grablab)。它们还包括一些 python 库，因此您可以自定义 CAD 文件以满足您的需求。
