# 科林的实验室回来了

> 原文:[https://hackaday.com/2014/02/03/collins-lab-is-coming-back/](https://hackaday.com/2014/02/03/collins-lab-is-coming-back/)

[![Collin](../Images/44a5203adff360f5710293152cf32990.png)T2】](http://hackaday.com/wp-content/uploads/2014/02/collin1.jpg)

我们想分享一点好消息；科林的实验室重新出现在互联网上。如果你不知道，[科林·坎宁安]以前为电子元件和基础电子学杂志制作过优秀的短视频，有时是半开玩笑的。2012 年，他被 Adafruit 聘为创意工程师，协助软件开发和视频制作。

展望未来，Collin 的实验室视频将成为 Adafruit 的博客和他们的 T2 YouTube 频道的定期专题。我们相信，科林独特的视频制作技巧和演讲方式将会给我们带来无穷的乐趣。

这是 Collin 在 Adafruit 上的实验室的第一次发布，是一次初级回顾，涵盖了基本的万用表功能和测量。这里没有太多的中级到高级电子黑客，但对于初学者来说，这是一个很好的快速学习使用万用表的基础知识的方法。

如果你想看看科林以前的视频作品，你可以在他的 Narbotic Instruments 网站上找到它们，标题是-[“制作礼物”](http://narbotic.com/video/MakePresents/MakePresents.html)和[“科林的实验室”](http://narbotic.com/video/CollinsLab/CollinsLab.html)，或者通过这个[方便的播放列表](http://www.youtube.com/watch?v=ZYH9dGl4gUE&list=PLECC9C55893C696A7)观看它们。休息过后，你还可以观看他最新版的《科林的实验室》。
