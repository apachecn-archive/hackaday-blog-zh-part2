# 使用宜家 Mylonit 打造 RGB 情调灯

> 原文:[https://hack aday . com/2012/06/03/building-an-RGB-mood-lamp-using-an-IKEA-mylonit/](https://hackaday.com/2012/06/03/building-an-rgb-mood-lamp-using-an-ikea-mylonit/)

![](../Images/400d47e53626767508856aadbe45c632.png "ikea_final1_th")

[Toon Beerten]一直在试验 LED 灯的结构。他已经建造了一个非常整洁的 led 金字塔作为情绪灯，但他希望他的客厅有更高质量的东西。他跑出去拿起一盏宜家灯，你可以在上面看到。商店里的灯只能显示一种颜色，而且结构清晰。[Toon]想要 RGB 和不透明的表面，所以[他打磨了灯并建造了一个定制电路](http://tobe.nimio.info/project/moodlamp)。

他撕下了灯电路，并用自己的电路替换了它，包括一个 3W LED，一个散热器和一个 PIC 16F628(和支持组件)。主电路实际上非常适合放在灯的下面。如果你想复制它，你可以从他的网站下载完整的原理图和代码。

视频中可以看到，效果相当不错。
