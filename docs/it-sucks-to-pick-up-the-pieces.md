# 收拾残局真糟糕

> 原文：<https://hackaday.com/2015/08/01/it-sucks-to-pick-up-the-pieces/>

拼图游戏是一种有趣的互动方式，可以让你度过一个下午或十二个下午，这取决于拼图的数量和你的技能水平。找到完成一个部分或连接两个区域所需的部件是令人兴奋的，但如果你的灵活性差，当你移动捡起它们时，兴奋会变成沮丧。[thomasgruwez]在设计这个[拾取和放置拼图玩具辅助工具](http://www.instructables.com/id/Pick-place-jigsaw-puzzle-aid/?ALLSTEPS)时，考虑到了残疾人和其他笨手笨脚的人，它使用吸力来拾取和运输拼图块。

吸力来自反向运行的水族箱泵，[我们经常看到这种情况](http://hackaday.com/2010/12/09/quick-cheap-and-simple-vacuum-tweezers/)，【thomasgruwez】在另一份说明书中对此进行了解释。一个大的、吸引人的按钮连接在电线上，用来开关水泵。一个同样大而迷人的瞬时开关会暂时关闭真空，这样就可以放置物品了。

在这个黑客的商业端是一个便宜的真空笔的小吸头。为了将笔头与泵连接起来，[thomasgruwez]设计并打印了一根刚性吸管来弥合这一差距。考虑到实用性，[thomasgruwez]还设计了一个可以栓在吸管上的戒指，用来容纳你选择的稳定手指，就像理发师剪刀上的小手指钩。

我们最喜欢的部分是可选的附件——一个小平台，可以在不切断真空的情况下快速翻转棋子。休息之后来看看。
