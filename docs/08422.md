# 像异形一样走路

> 原文:[https://hackaday.com/2015/02/28/walk-like-a-xenomorph/](https://hackaday.com/2015/02/28/walk-like-a-xenomorph/)

[詹姆斯·布鲁顿]正忙于他的最新项目，一件受“废金属雕塑”启发的外星异形套装。然而，他想得到一个高度的提升以及数字化的姿态。为此，[詹姆斯] [3D 打印了一双定制的高跷](http://xrobots.co.uk/alien2/index_feet.html)。每个支柱由一个升降机和几个用丙酮层压在一起的部分组成。他把一双旧鞋栓在高跷上，在脚趾上加了带子以防止鞋子翘起来。

虽然高跷非常好用，但[詹姆斯]想给它们加上鞋底，让他走路时有一些牵引力——穿着由锋利塑料制成的异形服装摔倒听起来足够痛苦！他决定用 ABS[和](hackaday.com/2014/05/06/hands-on-with-the-flexydualie-3d-printer-extruder/) [Ninjaflex](http://www.ninjaflex3d.com/products/ninjaflex-filaments/) 混合打印鞋底。鞋底的 ABS 部分然后被丙酮焊接到高跷的底部。

[詹姆斯]希望增加一些爪子的效果，只要它们不会过多地妨碍他的行走。他已经完成了相当数量的 3D 打印西装。我们知道完成后的项目将会是惊人的:[詹姆士]创造了从[达雷克斯](hackaday.com/2014/05/31/wiimote-controlled-extermination-dalek-style/)到[钢铁侠](hackaday.com/2014/04/25/sci-fi-contest-roundup-i-am-iron-man/)的一切！
