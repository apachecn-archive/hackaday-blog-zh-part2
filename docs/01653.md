# 如何建立一个极其强大的 Nerf 枪

> 原文：<https://hackaday.com/2012/09/18/how-to-build-an-extremely-powerful-nerf-gun/>

[TopCityGear]在尝试将一块 PVC 用作吹气枪管时，他认为他应该尝试给它一点比他的肺所能发出的更大的力量。他想出的是[这种气动 Nerf 枪肯定会留下痕迹](http://www.youtube.com/watch?v=pXOcJ_KXo2U)。休息后的视频是一个展示和讲述，一个构建日志，最后是它的力量的展示。他在一支耐夫镖上加了一颗钉子，并把它穿过一块木板，然后用一支普通的旧泡沫镖在他可怜的朋友的胸口上留下了一个巨大的伤痕。这让我们想起了那些射豆袋的防暴枪。

空气储存在那个 12 英寸的聚氯乙烯容器中。在后盖上有一个 Schrader 阀，用于通过压缩机甚至自行车打气筒给油箱加压。手柄是一个带内脏的无绳电钻，它的电池可以作为电动喷水阀的电源。手柄前面的螺丝配件让他可以取下枪管，这样就可以插入射弹。

这让我们想起了[那个发射充水乒乓球的枪](http://hackaday.com/2010/12/03/water-blob-launcher/)。
