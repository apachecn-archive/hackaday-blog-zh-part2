# THP 半决赛:连续波雷达

> 原文:[https://hack aday . com/2014/09/24/THP-semi-finaller-a-continuous-wave-radar/](https://hackaday.com/2014/09/24/thp-semifinalist-a-continuous-wave-radar/)

除了少数使用现成无线电模块的项目之外，没有多少 Hackaday 奖参赛作品在 RF 领域发挥作用。这让我们有点惊讶，因为无线电是车库修理工一直非常活跃的领域之一。[Luke]正在用[一种 FM 连续波雷达](http://hackaday.io/project/1682)反潮流，这种雷达将用于自主飞行器、高度探测和合成孔径雷达成像的实验中。

[Luke]的雷达工作在 5.8-6 GHz 左右，应该是微波电子学的入门。这是一个非常模块化的系统，围绕来自 Hittite 的几个 VCO、混频器和放大器构建，所有这些都通过同轴电缆连接。

到目前为止，[卢克]已经把他所有的模块组装好了，[一对很棒的罐子](http://hackaday.io/project/1682/log/4334-some-simple-quick-antennas)用于天线，所有的东西都被确认在他的范围内工作，还有很多对他的 git repo 的提交[。](https://github.com/lukeweston/SimpleFMCWRadar)
