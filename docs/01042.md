# 一个复制的 DJ 控制器来统治他们

> 原文：<https://hackaday.com/2012/07/15/a-replica-dj-controller-to-rule-them-all/>

所以像许多 Hackaday 的追随者一样，也许你对电子音乐感兴趣。虽然我们敢说你们中很少有人有[【亚当扎克】](http://adamdzak.blogspot.com/2012/07/introducingapollo-13.html "Apollo 13")的奉献精神，因为他决定为[【阿贝尔顿现场】](http://www.ableton.com/live "Abelton Live")软件包复制[【萨沙】](http://www.djsasha.com/ "dj sasha")的定制控制器。显然，这比拆开你的 DJ Hero 控制器，然后把它黑进你的电脑来连接歌曲要困难得多。

“新”控制器被命名为阿波罗，是萨沙的 Maven 控制器的复制品。构建过程在他的文章中有很好的展示，到目前为止结果看起来相当不可思议。特别有意思的是，在无法使用这个设备的情况下，对它进行逆向工程的努力。无论是从机械的角度来看，还是试图弄清楚如何使用按钮来控制软件，都一定是一个相当大的挑战。休息之后，请观看视频，了解新控制器的运行情况。 [https://www.youtube.com/embed/cc1G0IYgrOA?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/cc1G0IYgrOA?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)
