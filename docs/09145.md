# 50 度灰水回用

> 原文：<https://hackaday.com/2015/05/30/50-shades-of-gray-water-reuse/>

进入今年的 Hackaday 奖，[TVMiller]建造了一个超级便宜的 Arduino 驱动的[灰水回收系统。](https://hackaday.io/project/5711-arduino-shades-of-grey-water)

该系统非常简单，可以很容易地为几乎任何浴室。通过在水槽下面制作一个之字形的 PVC 管，他创造了一个简单的灰色蓄水池，大小适合他的马桶的冲洗能力。如果你用了太多的水，它只会回流到下水道——把它想象成一个巨大的害虫陷阱！冲完马桶后，12V 螺线管和 240 升/小时的水泵打开——用重复使用的灰水重新装满水箱！他还增加了一个 Arduino 和一个 LCD 屏幕来记录节约的水；当然，还有一个不错的 HaD 标志。

我们喜欢[TVMiller 的]项目简要构建日志——他没有任何保留。

> 管子被粘上了，吸入的毒素穿过我的肺部，渗透到细胞里，把我变成了一个增强的人类，现在能够非常轻松地举起小东西，比如订书机或大口大口。
