# 简单的 LED 项目为你的万圣节派对增添趣味

> 原文:[https://hack aday . com/2014/10/26/simple-led-project-to-spice-up-your ' s Halloween-party/](https://hackaday.com/2014/10/26/simple-led-project-to-spice-up-your-halloween-party/)

[Paul 的]项目是一个很好的例子，说明如何将一个简单的项目变成更有趣的东西。他给自己做了一个内置互联网控制的 RGB LED 灯的南瓜灯。

[Paul]首先连接一个 RGB LED 到一个树莓 Pi。他肯定会用 100 欧姆的电阻来连接每种颜色，以防止 LED 烧毁。web 界面是用 Python 编写的。界面非常简单。它由三个文本字段组成。用户为三种 LED 颜色中的每一种输入 0 到 255 之间的值。然后程序相应地点亮 LED。

[保罗]意识到他需要一个发光二极管的漫射器，以便真正看到混合的颜色。他选择用塑料南瓜灯来庆祝，而不是使用乒乓球这样的普通解决方案。[Paul]从提灯上取下原来的白炽灯泡，将 LED 安装在里面。南瓜的内部被漆成白色，所以它很容易扩散光线。结果是一个南瓜灯，会根据派对客人的要求发出不同的颜色。请一定查看下面的演示视频。
