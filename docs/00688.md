# Pingbot:可爱且完整记录

> 原文:[https://hack aday . com/2012/06/10/ping bot-萌-全-录/](https://hackaday.com/2012/06/10/pingbot-adorable-and-fully-documented/)

![](../Images/0fa78f49c1ef84c04edc7976d7b60622.png "Pingbot-Micro-RC-Rechargable-Musical-Robot-Pal")

在构建了许多功能性的但看起来有些不完整的机器人之后，[Tomdf]真的想生产一些感觉“完整”的东西。 [Pingbot 是结果](http://www.instructables.com/id/Pingbot-Micro-RC-Rechargable-Musical-Robot-Pal/)，下面是[Tomdf]对它的描述:

> Pingbot 是一个非常小的(直径 38 毫米)，usb 充电，可编程，音乐，遥控机器人，旨在最大限度的乐趣和舞蹈。

虽然我不想跳舞，但我发现自己在看视频时笑了。Pingbot 看起来确实完成了，也很有个性。Pingbot 内部的大脑是一个 Picaxe 08m2 。有了 110 毫安时的锂电池，Pingbot 也可以快乐地跳舞好一阵子。当电量不足时，只需插入 USB 端口进行充电。您可以在说明书中找到所有信息和图表，以及打印您自己的 Pingbot shell 的文件。
