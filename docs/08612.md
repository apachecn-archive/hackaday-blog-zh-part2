# 加热拖鞋让你的脚保持温暖，也让你的房子保持温暖，因为它会烧到地面。

> 原文：<https://hackaday.com/2015/03/23/heated-slippers-keep-your-feet-warm-also-your-house-as-it-burns-to-the-ground/>

冬天可能要结束了，但这并不意味着脚不会冷。当然，袜子或拖鞋可能会有一点帮助，或者你甚至可以打开家里的暖气，但这些选择都没有[柯林·福尔泽的]解决方案有趣或危险。他想出了一些疯狂的 DIY 加热拖鞋。

当然，有电池供电的加热拖鞋，甚至可以用微波炉加热，但这些都很无聊。这些小狗实际上是气体驱动的，更像一个小型便携式野营炉，而不是任何种类的鞋。每只拖鞋下面都有一个定制的燃烧室，里面有一个小燃烧器。每个滑靴从其自带的汽油罐中获得恒定的燃油流量。暖脚再容易不过了，只要打开煤气，用火柴点燃炉子就行了。

不言而喻，但我们应该重申…不要在家里尝试这个！你能尝试的就是休息后看这些拖鞋的视频。

[柯林·福尔泽]在 Hackaday 上发布他的项目并不新鲜。更多他基于火焰的项目，请看他的[火焰手套](http://hackaday.com/2014/05/30/furze-sets-fire-to-everything-with-pyro-gloves/)。当然，谁会忘记这些神奇的可伸缩的[狼獾爪子](http://hackaday.com/2014/05/16/pneumatic-wolverine-claws-are-quite-possibly-the-best-thing-ever/)。
