# 遥控岩石履带车准备成为爬楼梯机器人

> 原文:[https://hack aday . com/2014/01/15/RC-rock-crawler-prepped-to-be-make-stair-climbing-robot/](https://hackaday.com/2014/01/15/rc-rock-crawler-prepped-to-become-stair-climbing-robot/)

![rc-robot-frame](../Images/3f20ef7224a4c7384f44b7a917204251.png)

[Starlino]正在研发一种[自主移动机器人](http://www.starlino.com/rc-crawler-robotic-platform-increase-size-mod.html)。像他之前的许多人一样，他在无线电控制的汽车世界里寻找一个基本框架。他在一个叫做“疯狂扭矩”的岩石履带车模型中找到了一个很好的候选人。爬虫已经存在多年了，但最近变得越来越受欢迎。像往常一样，受欢迎程度会导致入门级型号的价格降低，这使得这款爬虫的机器人框架价格合理。顾名思义，攀岩者就是在爬行。相对较低的速度、锁定的差速器、四轮驱动和(可选)四轮转向。

当然，[斯塔里诺]在拆下来安装电子设备之前，必须试开他的框架。作为长期的 R/C 建模者，我们不能责怪他。测试发现了一个主要问题。疯狂的扭矩还没有疯狂到爬上他家的楼梯。前轮胎会抓住第一个台阶并停下来，但轴距不够长，后轮抓不住。

[斯塔里诺的]解决方案是延长轴距。对于大多数四轮驱动遥控汽车或卡车，这将是一个主要问题，因为电机安装在船中部。轴距加长意味着传动轴或皮带也要加长。这不是岩石爬行动物的问题。履带需要支撑大量的悬挂铰接。通常的设计是在每个车轴上安装一个电动机，而不是制造复杂的驱动连杆。就[非簧载质量](http://en.wikipedia.org/wiki/Unsprung_mass)而言，这不是最棒的主意，但它确实有助于轻松改变轴距。[Starlino]发现这种设计非常模块化，他可以将第二个底盘栓接到原来的底盘上。新的后底盘用螺栓固定在顶部防震座上。一套额外的电池支架形成了一个较低的支架。新的加长卡车足够长，可以清理台阶，但从视频中可以看出，它仍然有点挣扎。我们认为更大直径的轮胎可能会有所帮助。下一步是扔掉遥控装置，给这个机器人一个大脑！
