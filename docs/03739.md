# Leap Motion 通过手势控制六足机器人

> 原文:[https://hack aday . com/2013/08/06/leap-motion-controls-hexapod-with-hand-signals/](https://hackaday.com/2013/08/06/leap-motion-controls-hexapod-with-hand-signals/)

![leap-motion-hexapod-hand-control](../Images/994c4f783a306e5fdc87bdfcca878d37.png)

移动你的手[会让这个六足动物像一个没有线的木偶一样跳舞](http://qbotics.co.uk/one-giant-leap-for-robotkind/)。好的，很明显有一根线，实际上是一根线，但是你知道我们的意思。地板上的设备是一个跳跃运动传感器，它正在监控(Queron Williams 的)手势。这是使用[一个利用 Leap Motion API 的处理库](https://github.com/voidplus/leap-motion-processing)完成的。

现在，手势只影响六足动物身体的俯仰、滚动和偏航。但[Queron]确实计划增加对双手监控的支持，以增加更多的控制。我们在休息后看演示，认为这非常接近于[汤姆·克鲁斯]在*少数派报告*中展示的操纵。添加谷歌眼镜作为平视显示器，你可以在外围渲染辅助控件。

当你在看[Queron 的]项目帖子时，点击他的“hexapod”标签，瞥一眼机器人的建造过程。
