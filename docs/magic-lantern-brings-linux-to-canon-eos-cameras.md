# 神灯将 Linux 引入佳能 EOS 相机

> 原文：<https://hackaday.com/2015/04/04/magic-lantern-brings-linux-to-canon-eos-cameras/>

4 月 1 日，Magic Lantern 团队宣布了一个概念验证，可以让你在佳能 EOS 相机上运行 Linux。因为帖子的日期，我们已经对此进行了大量的讨论，并且确信这不是一个玩笑。发展潜力巨大。

最近发现可以从引导加载程序访问摄像头[上的 LCD 屏幕，这为黑客攻击提供了便利。如果你不认识这个名字，](http://www.magiclantern.fm/forum/index.php?PHPSESSID=phvg6o2rsnbckj7de04gptpr12&topic=14732)[魔灯](http://www.magiclantern.fm/)是一个开源项目，它通过利用 SD 卡上的二进制文件引导加载程序为这些高端相机添加功能。这是[长期以来黑进更多功能](http://hackaday.com/2009/06/24/magic-lantern-for-canon-5d/)的一种方式，但一直很复杂，因为你必须弄清楚如何很好地与现有固件配合。命令液晶显示器是硬件的最后一部分，以前不是直接由 Magic Lantern 驱动的。

既然 Linux 内核已经出现，就可以构建全新的特性，而不用以任何方式处理现有的固件(也不用覆盖它)。我们很期待这一次的结果。目前它只是证明你可以引导 Linux，它实际上还没有功能。现在，您有机会提高那些您一直保留的内核移植技能。
