# 用迪士尼大厦道具装饰你的鬼屋

> 原文:[https://hackaday . com/2013/10/26/class-up-your ' s haunted-house with a-Disney-mansion-prop/](https://hackaday.com/2013/10/26/class-up-your-haunted-house-with-a-disney-mansion-prop/)

![raspiDisneyPortrait](../Images/9873f98698b2ae81d7e9b36492ce5d13.png)

在每年的这个时候，迪斯尼的铁杆粉丝[布兰登·埃托]一定拥有这个街区最酷的房子之一，尤其是现在他已经建造了自己的大师格雷西变化肖像。如果你不熟悉迪士尼鬼屋景点，在世界各地的主题公园有几个不同版本的；奥兰多那家的特点是壁炉上方的一幅肖像[奇迹般地变老成了骷髅。](https://en.wikipedia.org/wiki/List_of_Haunted_Mansion_characters#Aging_Man)

[Brandon 的]娱乐使用一个装有[视频循环标清图像](http://www.timschwartz.org/raspberry-pi-video-looper)的 Raspi，循环播放一个老人图像的剪辑。他制作了一个盒子来装 19 英寸的液晶显示器，并在前面安装了一个[便宜的宜家框架](http://www.ikea.com/us/en/catalog/products/60232813/)。魔法隐藏在窗户贴膜中，贴膜将镜框的玻璃变成了一面双向镜子:这是【布兰登】从[那里借来的一项技术，今年万圣节很有指导意义](http://www.instructables.com/id/Spooky-Face-in-Mirror-Special-Effect/)。

对于一步一步的教程，你会想去看看[布兰登的] [关于 MAKE](http://makezine.com/projects/disney-inspired-changing-portrait-using-a-raspberry-pi/) 的报道，但在休息后留下来看一个快速的视频演示，并查看另一个闹鬼的豪宅黑客:[唱歌的头](http://hackaday.com/2011/01/10/halloween-hangover-talking-heads/)。
