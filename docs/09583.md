# Hackaday 奖品入口:Arduino 报警系统

> 原文:[https://hack aday . com/2015/07/19/hack aday-prize-entry-an-arduino-alarm-system/](https://hackaday.com/2015/07/19/hackaday-prize-entry-an-arduino-alarm-system/)

在过去的几年里，家庭自动化设备的市场有了惊人的增长。为什么这项技术现在才兴起，这是我们永远也不会理解的——家庭自动化系统已经存在了几十年，主要是以安全系统的形式存在。对于他的 Hackaday 奖参赛作品，[IngGaro] [正在构建一个基于 Arduino 的安全系统](https://hackaday.io/project/2519-arduino-anti-theft-alarm-shield)，它可以做你对家庭自动化系统所期望的一切，从关闭百叶窗到温度监控。

[英加罗]的系统是围绕一个 Arduino Mega 的护盾建造的。该屏蔽包括与报警系统、GSM 调制解调器、温度和湿度传感器、以太网模块和红外运动传感器的连接。这款 Arduino Mega 连接到安装在前门附近的控制盒上，该控制盒装有一个 LCD、一个 NFC 和 RFID 读取器以及几个按钮来启动和解除系统。

自从这个项目在去年的 Hackaday 奖中出现以来，它已经取得了很大的进步。从那时起，多亏了以太网库的改变,[IngGaro]终于完成了这个项目。它现在稳定多了，而且有能力完全控制一所房子里的一切，那*应该*是自动化的。现在【英加罗】需要做的就是为这个项目创造一个很酷的 PCB，但在我们看来，你不可能比[这个项目已经掌握的性能板](https://hackaday.io/project/2519/gallery#bfbc16f7d11c766777fd1554b8b32ff3)做得更好了。
