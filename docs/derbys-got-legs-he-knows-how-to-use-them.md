# 德比有腿，他知道如何使用它们

> 原文：<https://hackaday.com/2014/12/17/derbys-got-legs-he-knows-how-to-use-them/>

关于节日和人类最好的朋友，有一些东西能激发出人们最好的一面。3D Systems 公司 CJP 产品管理总监塔拉·安德森(Tara Anderson)培育了一只名叫德比的哈士奇混血狗。德比出生时就有先天性缺陷:它的前腿不发达，没有爪子。这使得这个可怜的家伙不能到处乱跑，做所有狗喜欢做的事情。[塔拉]给它配了一辆手推车，但她仍然觉得德比应该有更多的机动性和自由。她认为 3D 打印假肢是解决问题的答案，于是她求助于她的同事，并与认证动物矫形师德里克·坎帕纳合作，为德比制作了一套新的前腿。

该设计不同于典型的腿部假肢；泰拉觉得典型的“跑男”设计对狗不适用，因为它们会陷进地下。取而代之的是“环”的设计，允许更多有趣的犬类动作。它们是在 3DS 的 ProJet 5500X 上使用[多喷嘴打印](http://www.3dsystems.com/quickparts/learning-center/mjp-anatomy)构建的。多喷射打印使假肢能够打印出坚硬和柔软的部分，既舒适又耐用。

他们设计的第一套较低，以帮助狗轻松使用假肢并增强其肌肉。随着他越来越习惯于他的新发现的流动性，新的将被打印出来，将逐渐增加高度。德比很喜欢它的新腿，快乐地四处闲逛，有时甚至跑得比它的新主人还快！

这是假肢的一个非常巧妙的变化。我们对新颖的设计和功效印象深刻。我们很乐意看到这种多喷射概念在制造商社区中得到探索，使用[开源平台](hackaday.com/2014/11/24/a-uv-lightbox-for-curing-prints/)和材料，以及通过 3D 打印实现的更多非传统设计。我们永远不会厌倦看到人们已经创造的许多方式[为](hackaday.com/2014/03/15/e-nableing-shea-to-build-a-prosthetic-hand-for-herself/)[男人](hackaday.com/2013/09/09/3d-printed-prosthetic-hand/)[女人](hackaday.com/2014/08/27/3d-printing-a-beautiful-prosthetic-hand-for-a-stranger/)[孩子](hackaday.com/2014/12/07/kid-designs-his-own-prosthetic-arm-at-a-summer-camp/)制作的假肢。不管你个人对 3D 系统的看法如何，你都不能否认 Derby 正和他的令人敬畏的游戏玩得很开心。

如果你有兴趣了解他的故事，德比有一个脸书页面。你还可以去看看[和平与爪子](http://peaceandpaws.org/site/)拯救德比犬免于安乐死。
