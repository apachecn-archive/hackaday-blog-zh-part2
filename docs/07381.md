# 这不是时钟

> 原文::1230【https://hacked ay . com/2014/10/21/CEC-nest-pas-un-clock/中

[Justin]向我们透露了他的[光滑的定制 OBD-II 仪表](http://www.ft86club.com/forums/showthread.php?t=75181)很容易被当作 OEM 模块。他能够使用他的斯巴鲁 BRZ 的时钟区域显示一系列信息，包括机油和冷却液温度以及电池电压。

上面链接的论坛帖子对他的所作所为有一个很好的基于 FAQ 的解释，但是很多人告诉他闭嘴，拿走他们的钱，所以他为它创造了一个指令。基本上，他有一个 Sparkfun OBD-II UART 板与一个专业饰品通信。展示的是阿达弗洛·有机发光二极管，他发现这是普通车内各种各样光线条件下的理想选择。

[Justin]能够重复使用(H)小时和(M)分钟按钮，并将它们重新分配给(H)igh 以显示峰值读数和(M)ode，嗯，在模式之间切换。(:00)现在重置峰值读数。他建议为您的汽车获取特定的 CAN 代码，以使数据更有意义。【Justin】的代码在 [Octocat](https://github.com/stirobot/arduinoModularTFTgauges) 的众多触角中是安全的，你可以查看下面他的演示视频。
