# 8×8 LED 阵列构成一个令人毛骨悚然的动画南瓜

> 原文:[https://hack aday . com/2014/10/27/8x8-led-arrays-make-for-one-creepy-animated-pumpkin/](https://hackaday.com/2014/10/27/8x8-led-arrays-make-for-one-creepy-animated-pumpkin/)

[Michal Janyst]来信告诉我们他为他的侄子在万圣节做的一个小项目——一个有面部表情的南瓜灯。

南瓜眼使用两个 MAX7219 LED 阵列，一个 Arduino nano 和一个 USB 电源。是的，这很简单——但是看完视频后，你可能也会想做一个。就是太可爱了！或者令人毛骨悚然。我们无法决定。他还在 GitHub 上为感兴趣的人提供了代码。

当然，如果你想要一个更高级的项目，你可以制作一个[俄罗斯方块南瓜灯](http://hackaday.com/2012/10/29/pumpkin-tetris-inspired-by-our-own-led-jack-o-lantern/)，它的特点是将一个巨大的 8×16 的 led 阵列直接嵌入南瓜中……或者如果你是一个万圣节纯粹主义者，并且相信电子产品在南瓜中没有一席之地，你至少可以做的是[让你的南瓜灯喷火。](http://hackaday.com/2013/10/25/fire-breathing-jack-o-lantern-of-death/)
