# 将 Linux 放在 Vtech InnoTab 上

> 原文:[https://hack aday . com/2012/05/24/puting-Linux-on-the-vtech-innotab/](https://hackaday.com/2012/05/24/putting-linux-on-the-vtech-innotab/)

Vtech InnoTab 是一款儿童尺寸的平板电脑，专为儿童打造。除了是让侄子和侄女脏兮兮、黏糊糊的手远离合适的“成人尺寸”平板电脑的理想解决方案，它还可以作为 Linux 平板电脑，完美地用于一些自制应用程序。[Mick]为他的儿子买了一个 InnoTab，但是在让 BusyBox 工作之后，我们认为它已经成为了父亲的玩具，而不是儿子的。

[Mick]敲开 InnoTab，然后[将几根电线焊接到一对引脚](http://i.imgur.com/eBG5p.jpg)上，这对引脚连接到一个 TTL 电平转换器，然后连接到一台电视。[米克]的新平板电脑上运行着完整的 Linux 外壳，这足以鼓励他开始移植[scum VM](http://www.scummvm.org/)，这是 90 年代初卢卡斯艺术公司著名的点击冒险游戏背后的引擎。

现在，它仍然是一个正在进行中的工作，但是[Mick]有全屏支持和虚拟键盘工作；绰绰有余享受 *[触手的日子](http://www.youtube.com/watch?v=0HaYZsc66Bk)* 和 *[山姆& Max 上路](http://www.youtube.com/watch?v=4yWnnk9fyJE)。*休息之后，你可以看到[Mick]的 InnoTab 运行更适合儿童的 SCUMM 冒险 *[Putt Putt 去月球](http://en.wikipedia.org/wiki/Putt-Putt_(series)#Putt-Putt_Goes_to_the_Moon_.281993.29)，*我们相信他的儿子会喜欢的东西。
