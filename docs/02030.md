# 电脑不是极客群体的糖果

今年的万圣节，极客团体决定采取一种非常不同的方法来拓展。他们没有制作电子动画，也没有将他们巨大的空间改造成鬼屋，而是举办了一场名为“电脑不是糖果”的活动，他们与当地一家大型公司合作，为 100 名年轻人带来了 100 台平板电脑。

如果你还不熟悉极客群体，你应该去看看他们。他们是一个巨大的黑客空间，似乎真的在一起行动。他们定期发布非常酷的视频，我们最喜欢的是 T2 的设备解剖。

虽然我们真的很钦佩他们向这些孩子介绍黑客空间的工作，但我们也喜欢糖果和愚蠢的怪异项目。明年电脑和糖果怎么样！
