# Fubarino 竞赛:游戏男孩打印机

> 原文:[https://hack aday . com/2013/12/17/fuba Rino-contest-game-boy-printer/](https://hackaday.com/2013/12/17/fubarino-contest-game-boy-printer/)

![fubarino-contest-gameboy-printer](../Images/d5507fd288734b9d34cbbd883eb90d7f.png)

[Dave]有一台 Game Boy 打印机，喜欢在开机时按住进纸键打印马里奥主题的复活节彩蛋。当他听说 Microchip 为我们的复活节彩蛋比赛提供了一些 Fubarino 板时，他立即想到了这个硬件，并着手为打印机添加一个普通的复活节彩蛋。

为了修补硬件，[Dave]在[几年前在这里介绍的[furtek]的工作基础上建立了](http://hackaday.com/2010/10/08/game-boy-printer-usb-cable-and-software/)，该工作简化了直接从 Arduino 板打印的过程。连接 Arduino 的 TX 和 RX 线路会触发新的复活节彩蛋。他在下面的视频中演示了打印这两个隐藏消息。
