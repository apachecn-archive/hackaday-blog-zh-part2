# 飞行机器人在机器人管弦乐队中演奏

> 原文:[https://hack aday . com/2014/04/28/flying-robots-jam-out-in-a-robotic-orchestra/](https://hackaday.com/2014/04/28/flying-robots-jam-out-in-a-robotic-orchestra/)

![Flying Robot Symphony](../Images/b31479cfd54fe03aae65a0938b6f33b0.png)

[KMel Robotics]又来了，这次用飞行机器人乐队偷走了我们的心。这是技术和音乐结合的惊人壮举。

我们第一次看到这样的事情，一群人演奏了[詹姆斯·邦德主题](http://hackaday.com/2012/03/04/quadrotor-squadron-plays-the-james-bond-theme-song-in-preparation-for-world-domination/)，这本身就令人印象深刻——管弦乐队在这一最新作品中变得更加先进。

大多数乐器都可以通过添加带特殊着陆垫的加重杆来演奏，六轴转子可以反弹，但我们认为我们最喜欢的乐器是弦乐器——另一个机器人可以调整线的张力来改变音高，很像吉他上的颤音。再加上一个电子放大器，你就能听到非常优美的声音。

另一种值得注意的乐器是鼓，它使用解构和拼凑的钢琴动作来演奏音符。留下来听听我们新的机器人霸主的声音吧！
