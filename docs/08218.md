# [迈克]向我们展示如何使用电枢咆哮者

> 原文：<https://hackaday.com/2015/02/02/mike-shows-us-how-to-use-an-armature-growler/>

[Mike]在他的[SmallEngineMechanic] YouTube 频道上放了一个很棒的视频，是关于一种我们现在不常见到的工具的。他正在用电枢咆哮器测试发电机的电枢。[电枢咆哮者](http://en.wikipedia.org/wiki/Growler_%28electrical_device%29)(或简称咆哮者)在几年前已经司空见惯。当汽车有发电机的时候，几乎每个汽车修理工手头都有一台发电机。他们执行三个简单的测试:检查电枢绕组是否对其他绕组短路、绕组开路和对电枢体短路。[迈克的]特别咆哮者对他来说是一个废人。电线损坏了，生锈了，通常需要精心打理。他把它修复成新的状态，并用它来帮助他的古董发动机和发电机组~~瘾~~爱好。

咆哮者本质上是一个带有 V 形框架的变压器初级。初级线圈连接到交流电源。待测试的电枢位于“V”形，通过感应的魔力，一些绕组成为次级线圈(稍后将详细介绍)。这意味着一些相当高的电压将暴露在测试电枢的换向器上，所以使用时要小心！

对接地或电枢铁芯短路的测试是一个简单的连续性测试。然而，不是压电嘟嘟声，而是短路将触发咆哮者开启，这意味着电枢将跳动一点，一切都将发出响亮的交流嗡嗡声。这无疑使测试更加有趣！

检查绕组是否开路就是给咆哮者的线圈通电，然后探测换向器上的触点对。绕组中感应的电压显示在咆哮者的仪表上。开路绕组将显示 0 伏。并非所有的电枢绕组都会同时出现在“咆哮者”的磁场中——所以全面测试电枢意味着要旋转几次，正如[Mike]在他的视频中所展示的那样。

最后一项测试是针对短路的线圈。这就是事情变得非常酷的地方。咆哮者被打开，一片薄的黑色金属——通常是一个旧的钢锯刀片——沿着电枢的核心运行。如果存在短路，钢锯刀片将振动短路绕组上方的电枢铁芯。我们不是 100%清楚咆哮者的初级绕组和两个绕组之间的耦合是如何导致叶片振动的，所以请随意在评论中插话解释。

大多数商店不再对电枢进行故障诊断，他们只是把新零件塞进去，直到一切都恢复正常。因此,“咆哮者”不像以前那样受欢迎了。不过，如果你使用 DC 汽车或发电机，这是一个很好的工具，它的操作本身就是一个非常酷的黑客。
