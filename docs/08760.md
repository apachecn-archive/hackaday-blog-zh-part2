# hacklet 42–鼠标项目

> 原文:[https://hackaday.com/2015/04/10/hacklet-42-mouse-projects/](https://hackaday.com/2015/04/10/hacklet-42-mouse-projects/)

自从[道格拉斯·恩格尔巴特]和他的团队发明了电脑鼠标，黑客、制造商和工程师们就一直在创造改变和改进设计的方法。即使是最初的鼠标也是一种黑客，由一块木头、一个按钮和两个编码器轮组成。电线伸向用户的手腕，使设备看起来像有尾巴。即使过了这么多年，人们仍在努力制造完美的指点设备。本周的 Hacklet 在 Hackaday.io 上重点介绍了一些最好的鼠标项目！

[![mouseballz](../Images/5e24bf40b153445ed595278080f1c347.png)](https://hackaday.io/project/4340) 我们从【s_sudhar】和[ORB——一款 3D 游戏鼠标](https://hackaday.io/project/4340)开始。Orb 使用加速度计和陀螺仪来跟踪其在 3D 空间中的位置。流行的 MPU-6050 芯片提供所有传感器，以创建惯性测量单元(IMU)。控制器是一个 Arduino Micro，它借助 Arduino 的[鼠标键盘库](http://arduino.cc/en/Reference/MouseKeyboard)为 PC 提供 USB 接口。两个微型开关控制按钮的功能。最初的球体是在一个纸板箱里建造的。[S_sudhar]创造了一个更先进的版本，装在一个带有两个按钮的 3D 打印球体中。球体两半之间的半透明接合处只是乞求一些 RGB LEDs。我们已经可以看到他们闪烁的红色，当你在堡垒 2 队被击中！

任何使用过 X-Windows 三键鼠标的人都知道现代的可点击中心滚轮有多疯狂。你不能在轮子没有滚动的情况下点击它，这会导致各种各样的混乱。有很多软件解决方案和窗口管理器模块可以解决这个问题，但是[mclien]想要一个真正的带侧滚轮的三键鼠标。然而，他并不想要任何鼠标——它必须是一个硅图形国际(SGI) 3 按钮单元。他的项目[带独立滚轮的 3 键鼠标](https://hackaday.io/project/3779)使用 dremel、钻床和胶水将 3 键滚动鼠标的电子设备移植到经典的 SGI 塑料中。最终的车轮放置确实有效，但不太适合[麦克莱恩]的手。不过，它确实非常适合他的一个朋友的手。事实上，这位朋友借用了[麦克莱恩]的创作。从那以后，老鼠和它的朋友都不见了！

[![jimmy](../Images/7bfbc9907b417017ab2ea2a91f88d183.png)](https://hackaday.io/project/887) 【杰伊-t】认为老鼠不仅仅是用来指指点点的，所以他建造了[老鼠机器人吉米](https://hackaday.io/project/887)。吉米是一个机器人，由一个老式的阿米加双按钮鼠标改装而成。他的大脑是一个视差推进器处理器。两个安装在支架上的齿轮马达帮助吉米四处行驶。Jimmy 有很多传感器，包括红外物体探测器、开关和 Adafruit 的 GPS 模块。吉米可能是世界上第一个自动导航的老鼠机器人。[Jay-t]在道具上用超光速粒子做交互测试。拥有 8 核处理器的好处是有足够的扩展空间。即使使用了所有这些传感器，Jimmy 仍然只使用了 3 个内核！

[![clovis](../Images/2da62eeda94f888c290937d903e04461.png)T2】](https://hackaday.io/project/5165)

最后，我们看看[克洛维斯·弗里岑]和[无线无电池鼠标](https://hackaday.io/project/5165)。这是我们最喜欢的项目类型——刚刚上传的那种。[克洛维斯]计划使用一种基于运动的系统为超级电容器充电——无需电池或电线。他还希望使用加速度计来检测鼠标的位置，而不是耗电的光学系统。细节还是稀稀拉拉的，因为他才刚刚开始这个项目！这些正是让我们思考的项目类型。[克洛维斯]将如何把运动转化为能量？会是像自动上弦手表那样的砝码吗？也许按钮上有 pizeo 元素。一旦电容器放电，人们会介意必须摇动鼠标才能让它工作吗？我们可以肯定的一件事是，[克洛维斯]在实施像他的[气象站](https://hackaday.io/project/4564)这样的项目方面有着良好的记录。进去帮助你自己的想法，或者简单地跟随我们，看看这个结果如何。
