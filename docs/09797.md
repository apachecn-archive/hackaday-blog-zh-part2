# 准将 C16 复活与树莓皮

> 原文：<https://hackaday.com/2015/08/16/commodore-c16-resurrection-with-a-raspberry-pi/>

[lactobacillusprime]有一台不能工作的 Commodore C16 和太多的 Raspberry Pi 计算机，所以他决定通过在 Pi 上模拟 C16 来让它起死回生。该项目的核心是 Pi，以及一个将旧式 Commodore 键盘(和操纵杆)转换为 USB 端口的小电路板。

一旦你有了作为 USB 接口的键盘，剩下的工作或多或少就是机械和软件了。[lactobacillusprime]做了很好的工作，把所有的东西都放进了新的盒子里，所有的 I/O 线都通过现有的端口。对于软件，[仿真站](http://emulationstation.org/)负责在 Pi 上启动 Commodore 仿真。

当然，没有理由仅仅局限于 Commodore 模拟器。模拟站和右后端模拟器可以让这台机器玩真正的 C16 准将玩不了的游戏。

当然，我们最高兴的是看到他启动准将 64 BASIC。也许我们应该完成我们在 20 世纪 80 年代开始的所有那些半成品 C64 基础项目。总的来说，我们不愿意看到旧电脑被损坏而不是被修复，但至少这台电脑将继续运行它的软件。如果你对看到机器被掏空感到不安，你可以随时切换到我们之前的报道[将 Commodore 的内脏放入一个新的盒子](http://hackaday.com/2015/03/11/hackaday-retro-edition-a-new-commodore-64-case/)。
