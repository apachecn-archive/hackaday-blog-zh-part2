# 带自制电子琴的电磁竖琴

> 原文:[https://hack aday . com/2012/05/09/电磁竖琴与自制电子书/](https://hackaday.com/2012/05/09/electromagnetic-harp-with-home-made-ebows/)

![](../Images/a9ddb474716455e31f55581090f68932.png "harp")

来自 Arduino subreddit 的是[dataplex]的 [26 弦电磁竖琴](http://www.reddit.com/r/arduino/comments/tbcv3/this_is_a_26_string_electromagnetic_harp_the/)。这是一个非常酷的环境乐器，通过[dataplex]的空间回声和 Fender Twin rig 听起来非常棒。

没有太多的构建细节，但从几个月前【data plex】的这篇帖子来看，这款仪器的工作原理与[的 Ebow](http://www.ebow.com/home.php) 相同。有一个很棒的[原型视频](http://vimeo.com/32643008)展示了硬件和软件。一个小音圈读出每一串。一个微小的运算放大器将这个信号发送到另一个音圈，然后返回到琴弦。有大量的反馈，但这是一个伟大的空灵的氛围。

原型由 Arduino 通过计算机控制，但最终版本[dataplex]转移到竖琴前面的铜/锌触摸界面。

这是一个非常非常酷的工具，我们正在耐心等待建造细节的发布。休息之后你可以看看[dataplex]的工作视频。
