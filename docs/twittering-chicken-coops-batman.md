# 叽叽喳喳的鸡窝，蝙蝠侠！

> 原文：<https://hackaday.com/2014/10/06/twittering-chicken-coops-batman/>

到目前为止，你已经看到了几乎所有的推文。但是你见过(法国)叽叽喳喳的鸡舍吗？ ( [谷歌翻译链接](http://translate.google.com/translate?sl=fr&tl=en&js=y&prev=_t&hl=en&ie=UTF-8&u=http%3A%2F%2Fwww.digitalspirit.org%2Fblog%2Findex.php%2Fpost%2F2014%2F09%2F30%2FLe-poulailler-connect%25C3%25A9&act=url))【雨果】养了两只鸡，作为减少家庭垃圾运动的一部分，然后开始工作。

即使你不懂法语，[鸡的 twitter 饲料](https://twitter.com/LaVieDePoule)基本上讲述了这个故事。

该设置可以拍摄睡觉的鸡的红外照片，并通知[Hugo]何时该收集鸡蛋。当然，还有大量其他传感器可用。鸡舍可以根据环境温度、鸟巢温度、光照水平、运动传感器状态或剩余鸡饲料量发出鸣叫。你可以很容易地跟踪这两只鸡是在鸡笼里还是在院子里。就像老大哥一样，只针对鸟。

坦白地说，这个申请很荒谬。但是如果你对家庭自动化感兴趣，有很多东西需要学习，而且这个项目有很好的文档记录。[Hugo]使用 [OpenCV](http://opencv.org) 进行视觉鸡蛋检测，并使用定制 Python 代码稍微随机化推文文本。所有这些细节都在他的 [Github 账户](https://github.com/hugokernel/LaVieDePoule)上。

如果你没有足够的鸡舍出租工具，一定要看看[这个移动鸡舍](http://hackaday.com/2013/01/28/mobile-chicken-coop-includes-wireless-sensors/)、[这个金色螺旋形状的鸡舍](http://hackaday.com/2012/05/11/golden-spiral-chicken-coop/)，或者[这个有蓝牙功能、会说话的鸡舍](http://hackaday.com/2011/02/28/bluetooth-enabled-talking-chicken/)等等。你会认为我们的名字是一天合作。
