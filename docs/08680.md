# 用 EL 线可视化数字逻辑

> 原文:[https://hack aday . com/2015/03/31/visualizing-digital-logic-with-El-wire/](https://hackaday.com/2015/03/31/visualizing-digital-logic-with-el-wire/)

[Bob]和[Aubrey]在马里兰州巴尔的摩以北一点的地方经营系统资源计算机博物馆。对于一个展览，他们想出了数字逻辑的可视化表示，并提出了两位二进制加法器。是的，它只是一个全加器，你可以在任何数字逻辑教科书的第二或第三章中找到。他们演示全加器如何工作的方式是这里的杀手锏:他们使用 EL 线连接所有的门。

全加器是用 Arduino Mega 实现的，但接口才是真正的亮点。在显示器的左侧有四个发光的拨动开关，显示虚拟电子流过 EL 线，通过门，最终到达七段显示器。EL 线由一个 [EL Escudo Dos shield](https://www.sparkfun.com/products/10878) 控制——这是一件好事，因为在开关、门和输出之间有许多线路。

你可以看看[Aubrey]的演示视频，它也展示了他们是如何建造它的。如果你在巴尔的摩附近，你可以去参观博物馆的展览。
