# 世界上第一个智能滑雪板根据你的动作改变音乐

> 原文：<https://hackaday.com/2014/12/21/worlds-first-smart-snowboard-changes-music-according-to-your-actions/>

想要一首你生活的配乐吗？一次几分钟，Signal Snowboards 用一个[智能滑雪板创造了这种体验，它会根据你在下山途中表演的技巧来改变你的音乐](http://www.youtube.com/watch?v=RC26DMal7TA)。

门上的标志写着“天才黑客学校”。在里面,[马特·戴维斯]帮助用加速度计连接音频——他经常用各种各样的黑客设备做这件事。最初的原型是一部模仿滑雪板运动的 iPhone，就像战斗机飞行员用手描述空战一样。将这些声音转换成声音的音频引擎是开源的，任何人都可以进行自己的调整。

一旦音频被弄清楚，男孩们就把它带回他们的商店，并把传感器嵌入一个新的滑雪板。该板配有 GPS、加速度计、几排 led 和蓝牙板，用于连接手机应用程序。这一切都是由一个车载脂电池和一个桶插孔来充电。手工用路由器切割通道，然后用环氧树脂将电子设备密封到位。不想“只是将一些[圣诞灯绑在滑雪板](http://hackaday.com/2014/05/17/led-snowboards-light-up-the-night/)上”，灯光也连接到传感器，并且是可编程的。
