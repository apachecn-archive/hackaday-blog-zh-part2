# 用声音雕刻粘土

> 原文:[https://hackaday.com/2012/04/23/sculpting-clay-with-sound/](https://hackaday.com/2012/04/23/sculpting-clay-with-sound/)

![](../Images/60b942bead763757b8ebead52d9f1eeb.png "user-shot")

邓迪大学的一群学生创造了这个有趣的原型，叫做[声音雕塑](http://soundbydesign.wordpress.com/about/)。目标是使用声音文件驱动雕刻手臂来雕刻粘土。理想情况下，你最终会得到对每首音乐都独一无二的艺术作品。正如你在视频中看到的(休息后)，他们在建造这个东西和让手臂对音乐做出反应方面做得非常好。这几乎是催眠观看。

我们不禁注意到有一点设计上的问题。由于 4 个臂是垂直固定的，粘土在同一轴上旋转，它们可以移动，你的变化将非常有限。我们认为这并没有减损项目，但确实提供了很大的改进空间。

你会如何改变雕塑手臂或它们的动作，让每件作品更独特？
