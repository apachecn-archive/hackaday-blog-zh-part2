# [Peter]还有神奇的彩色电话线手镯

> 原文:[https://hack aday . com/2014/10/15/Peter-and-the-amazing-technicolor-phone-wire-手镯/](https://hackaday.com/2014/10/15/peter-and-the-amazing-technicolor-phone-wire-bracelet/)

当一份工作留给他一些多余的电话线时，[彼得]没有把它扔进废品堆。他爆出浇铸树脂[做了一个很牛逼的手环](http://imgur.com/gallery/nXGnB) (Imgur 链接)。[彼得]正在成为一个相当多才多艺的珠宝商！当我们最后一次检查他的时候，他正在[用彩色铅笔做戒指](http://hackaday.com/2014/05/17/a-ring-of-colored-pencils/)。

用树脂浇铸导线就像建造一个正方形模型，放置导线，然后用适量的环氧树脂和硬化剂填充模型一样简单。一旦环氧树脂固化，[彼得]用锋利的[福斯特纳钻头](http://en.wikipedia.org/wiki/Benjamin_Forstner)钻出中心。带锯使石块的角更接近圆柱体。

从那里，它被转到车床，在那里[彼得]使用一个果酱卡盘举行手镯在适当的位置。一旦他成型手镯[彼得]开始湿砂。表里都用了很多很多的砂光来完成手镯。结果是一个镜面光滑的完成，与绝缘亮铜位刚刚弹出的树脂。

有人可能会认为铜手镯会很粗糙，但[Peter]在他的 Reddit 帖子上提到，它摸起来像塑料，尽管在打磨之前铜的碎片“非常微小”。我们建议涂上一层透明涂层来保护暴露在外的铜。戴在手腕上，所有暴露在外的金属会立刻开始氧化。

这个黑客给了我们许多铸造可穿戴电路的想法。一些 WS2812 和一个 teensy 将使一个相当浮华的设置！对一个项目有想法吗？在评论中告诉我们，或者发布到 [Hackaday.io！](http://hackaday.io/)
