# 终极便携式锐舞的 LED 灯杆

> 原文:[https://hack aday . com/2014/08/28/led-light-staffs-for-the-ultimate-portable-rave/](https://hackaday.com/2014/08/28/led-light-staffs-for-the-ultimate-portable-rave/)

![3531621392887203462](../Images/ef916c02668894f64503d9b1424d6b0c.png)

[risknc]和[mpinner]已经研究了一段时间的 LED 灯杆设计，并提出了一个可以用一系列流动颜色照亮夜晚的原型。甚至还有一个可以调高和调低亮度的刻度盘。

最初，[risknc]开始在 SpaceX 开发自己的项目，并在《燃烧的人》之前进一步深入这个想法。当在空中旋转时，这种视觉效果产生了一种非常明亮的能量流，可以看到它在用户周围盘旋。

8 英尺长的碳纤维棒的边缘塞满了 RGB 发光二极管。每米 60 个 LED 的 Neopixel 条带用于颜色之间的交替，一整束具有白色能力的 LED 也被嵌入到五线谱中。一个早期的设计被故意留在加州卡尔弗城一个叫做 Crashspace 的地方。社区成员试用的照片出现在黑客空间的网站上。此外，工作人员的描述和一些“闪光棒”的高质量照片被上传到 [Suprmasv 项目页面](https://www.suprmasv.com/projects/156/led-staff-for-spinning-crowd-con)。通过搜索图片，我们发现了一个实例，显示了在背景中有一个消防 poi 旋转器的流动会议期间使用的 LED 灯杆。也许有一种方法可以将发光二极管和火结合起来？不管怎样，后来的版本在 2014 年旧金山的创客大会上进行了测试。

该项目的完整规格和日志可以在 Hackaday.io 上找到。休息后会有一个关于[mpinner]的光之杖旋转的快速视频。在视频中，看起来他们正在 Crashspace 外面测试它，因为他们穿过后面小巷的黑暗，用漂亮的 LED 灯照亮了这个区域。未来的计划包括建造一堆这样的设备，并无线同步它们。CAD 模型也将很快上传。
