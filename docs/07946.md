# 新型 3D 打印技术——摩擦焊接

> 原文：<https://hackaday.com/2014/12/30/3d-printing-technique-friction-welding/>

尽管 3D 打印机可以制造几乎不可能铣削的复杂形状，但它们并不太适合需要桥接或有大空间的设计。为了克服这一点，[斯科奇]应用了一种简单的塑料焊接技术，这种技术既适用于 ABS，也适用于 PLA。你只需要一个旋转工具。

摩擦焊接是将两个表面摩擦到一起的过程，直到摩擦产生足够的热量将它们连接起来。在工业上，该方法适用于连接大型金属工件，否则将需要耗时的焊接。2012 年，[Fran]让我们想起了几十年前的一个玩具，它让孩子们利用摩擦力用塑料焊接苯乙烯[。这种改进的方法类似于焊条焊接，在熔化的接头中加入可消耗的填充棒。受我们对[Fran]报道的启发，[Scorch]进行了实验，发现一根安装在 Dremel 上的细丝同样适用于连接 3d 打印。](http://hackaday.com/2012/12/31/make-your-own-plastic-friction-welder/)

这就是事情的全部。剪下一点细丝，把它放入你的旋转工具中，用珠子连接零件和形状或者进行修理。摩擦焊接塑料的强度惊人，在某些接头上远远优于胶粘塑料。工具箱的另一个工具。参见下面的视频，了解[斯科奇]的演示。
