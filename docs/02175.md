# 一对公文包内置音箱

> 原文:[https://hack aday . com/2012/11/23/a-pair-of-公文包-boombox-builds/](https://hackaday.com/2012/11/23/a-pair-of-briefcase-boombox-builds/)

![](../Images/d6f4942baaa2d8515e2c15f440976beb.png "two-briefcase-boombox-builds")

这里有两个不同的公文包扬声器项目。[戴尔]早在高中时就开发了右边的产品，而[最近开发了升级版 2.0](http://www.youtube.com/watch?v=66fz1XfnK2I)。在看到昨天装满音乐的[手提箱](http://hackaday.com/2012/11/22/a-suitcase-full-of-tunes/)后，他受到启发，给这些项目发了一份小费。

第一个版本使用一对扬声器，它们是从垃圾场的一辆汽车里拿出来的。它们被安装在一些增强塑料公文包侧面的碎料板上。驱动它的放大器安装在机箱内，电池为系统供电。[Dale]包括一个用于输入电缆的原始存储支架，因为放大器可以驱动四个扬声器，所以外部有两个连接器。

第二版更加完美。他没有太多地展示那个，但你可以看到外壳上有一个 LED 灯，用作 VU 计，以及一个可能是电池电压的数字显示？他提到，这将吹走他的同事带到工作现场的任何商用系统。

休息后可以找到两个钻机的视频。
