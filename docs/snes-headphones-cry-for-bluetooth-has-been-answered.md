# SNES 耳机为蓝牙哭泣已被回答

> 原文：<https://hackaday.com/2015/03/26/snes-headphones-cry-for-bluetooth-has-been-answered/>

一年半前，我们发表了一篇关于 SNES 控制器改装成耳机的帖子。他们当然是好看和有创意的耳机，但是按钮，虽然存在，却没有功能。原帖子的标题是(可能是对抗性的)叫:' *[SNES 耳机为蓝牙控制尖叫](http://hackaday.com/2013/04/01/snes-headphones-scream-out-for-bluetooth-control/)* '。

嗯，耳机调制解调器[lyberty5]卷土重来。他响应了这一号召，为他的 SNES 耳机制作了[第二版……猜猜看，它们确实是蓝牙耳机！不仅如此，A、B、X 和 Y 按钮这一次是功能性的，并且已经连接到施主蓝牙模块上的控件。](https://www.youtube.com/watch?v=wZoeEILjvSc)

为了让这个项目开始，SNES 控制器被拆开，塑料外壳被切开，以分开两个圆形的侧面。一个纸板模型被粘在适当的位置，这样环氧油灰可以粗略地形成，以使每个部分完全圆。一旦固化，腻子是打磨和缺陷填补汽车车身填料。为了安装到头带上，钻了一些孔，并为蓝牙模块的 USB 端口做了一个插槽，这样耳机就可以充电了。在快速涂上一层任天堂灰色油漆后，耳机被重新组装起来。我们必须说这些东西看起来很棒。

如果你想自己制作一套 SNES 蓝牙耳机，休息后看看制作视频。
