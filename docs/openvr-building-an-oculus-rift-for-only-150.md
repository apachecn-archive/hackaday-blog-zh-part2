# OpenVR:只需 150 美元就能构建一个 Oculus Rift

> 原文：<https://hackaday.com/2014/06/13/openvr-building-an-oculus-rift-for-only-150/>

Oculus Rift 是一个非常酷的套件，但它的未来掌握在脸书手中，谁知道它现在会变成什么样。那么，为什么不自己建造呢？当 Oculus 第一次问世时,【Ahmet】立刻就被吸引住了——他开始研究虚拟现实和 Oculus 提供的体验——但是对于 300 美元的价格来说，光有好奇心是不够的。他推迟了，直到他有了一个有用的目的，事实证明他做到了——他制造并驾驶多翼飞机，为此，FPV 装置将会非常方便！

其他 FPV 系统的价格也接近 300 美元，所以拥有更多功能的设备是有意义的。意识到这一点后，他马上面临着制造者的困境:**买，还是建？**为了测试水域，他决定订购一些非球面透镜，用一部智能手机和一个简易纸箱装置做一些快速测试——结果出乎意料地好。现在不能回头！
硬件包括:

*   5.6 英寸 1280×800 液晶显示器
*   [3D 打印外壳](http://www.thingiverse.com/Croccy22/designs)
*   12V 电源适配器
*   USB 转 TTL
*   50 毫米非球面透镜(5X 变焦)
*   Arduino 迷你专业版
*   GY-85 九自由度惯性测量组合
*   各种电线、泡沫衬垫和胶水

这里的大部分成本是在液晶显示器上，其他东西都很便宜。一旦全部完成(细节在他的博客上)，是时候从 GitHub 下载软件了。剩下的就不言自明了——看看结果就知道了！我们现在甚至想建造一个。下面的视频。
