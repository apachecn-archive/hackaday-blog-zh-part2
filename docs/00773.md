# NC 2012 制造商博览会:电动汽车

> 原文:[https://hack aday . com/2012/06/26/maker-faire-NC-2012-electrical-vehicles/](https://hackaday.com/2012/06/26/maker-faire-nc-2012-electrical-vehicles/)

[![lab306-fox](../Images/4ceaa67a971bb045ec04329cd7092900.png "lab306-fox")T2】](http://hackaday.com/?attachment_id=77974)

虽然我不知道会在 NC Maker Faire 上看到什么，但我惊喜地看到了几辆制作精良的电动汽车。其中值得注意的是[【lab 306】](http://lab306.com/ "lab306")的福克斯车身电动野马。虽然它本身会给人留下深刻的印象，但它是由一个高中班级制作的，并在一些出版物中出现过。一定要看看他们的优秀网站，或者休息后的短视频！你不希望你去那所高中吗？

同样值得注意的是休息后看到的一些非常酷的汽车，包括一辆由[【绿色循环设计集团】](http://www.greencycledesigngroup.com/gcdg/ "Green Cycle Design Group")的套件制造的汽车。另外两辆按照传统汽车标准来看非常小，设计非常独特。
