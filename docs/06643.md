# 电动“微型赛车”有很大的马力

> 原文:[https://hack aday . com/2014/07/25/electric-micro kart-has-tons-of-kick/](https://hackaday.com/2014/07/25/electric-microkart-has-tons-of-kick/)

![Go Kart with Independent Suspension](../Images/6fae7c400f531127eb04a60b45433305.png)

当你建造一辆电动卡丁车时，你真的有两种选择。转换一个正常的汽油动力的交换了电厂…或从头开始建立它！[甘哈尔]选择了后者，以节省一些钱，并按照他想要的方式设计它。

现在你可能已经注意到它看起来有点小——因为它确实很小。这实际上更像是一辆微型卡丁车，但没关系，因为[甘哈尔]为他的孩子建造了这辆车，并因此获得了年度父亲奖！

它配备了两个 2kW (~3HP)无刷电机，独立驱动后轮。这些由两个 48V 50A 连续(100A 峰值)速度控制器供电。[Ganharr]在电池上也不遗余力，选择了由 Headway 电池组成的 48V 锂离子电池组(每个电池容量为 3.2V 15aH，40152 型)。

在 1500 美元的预算中，有 1000 多美元花在了电池和电机上，[甘哈尔]选择简化车架和传动系统。木材，车轮和其他一些基本组件构成了车辆的其余部分，但圣牛它工作得很好！
