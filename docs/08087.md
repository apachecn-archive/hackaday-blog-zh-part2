# 一次重大故障后的真空管修复

> 原文：<https://hackaday.com/2015/01/17/vacuum-tube-repair-after-a-spectacular-failure/>

[Eric]有一台 20 世纪早期的 Atwater Kent 55C AM 收音机。他一直试图将收音机恢复到正常工作状态。他最近的疼痛是与整流管有关。管子内部应该是完全真空的，但这里的情况并非如此。当电子管通电时，它会发出美丽的紫罗兰色。它可能看起来很漂亮，但这表明气体已经泄漏到管道中。它需要被替换。

[Eric]有一根管子可以作为很好的替代品，但它的插头与插座不匹配。他不得不用这个旧的破管子来做一个适配器。他决定先玩玩它，而不是把旧管子拆开。他把它连接到自耦变压器、安培计和电压表上。然后他慢慢增加电压，看看会发生什么。结果是视觉震撼。

该管开始与以前经历的相同的紫色/蓝色发光[埃里克]。随着电压的增加，它变得越来越强烈。最终我们开始看到一些绿色和紫罗兰色混合在一起。[Eric]对这个意外结果的反应是无价的。随着电子管变得越来越热，阳极开始发出橙红色的光。最后，灯丝开始像火花一样噼啪作响，然后电子管就放弃了，彻底失败了。

灯光秀结束后，[埃里克]继续更换灯管。他开始用螺丝刀头敲旧管子的插口。多次敲击后，玻璃开始从插座中脱落。扭动了一会儿后，管子终于从插座里出来了。[Eric]幸运的是，我有一个未使用的八进制插座，可以完美地插入旧插座。他所要做的就是将旧适配器的四个引脚连接到八进制插座的正确引脚上。小菜一碟。

……或者说[埃里克]是这样认为的。在用试管测试器测试了一些新试管后，他意识到他焊接了所有四个错误的管脚。最重要的是，他用强力胶把适配器粘在了一起。他最终把两块碎片分开了。这一次，他从八进制插座中移除了所有未使用的引脚，这样他就不会弄错了。试管测试器上的另一次运行证实了一切看起来都很好。把电子管插入收音机后，它就像预期的那样工作了
