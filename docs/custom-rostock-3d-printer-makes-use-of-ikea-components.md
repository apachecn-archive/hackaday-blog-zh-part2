# 定制 Rostock 3D 打印机使用宜家组件

> 原文：<https://hackaday.com/2013/12/18/custom-rostock-3d-printer-makes-use-of-ikea-components/>

在发现 3D 打印机后，[Turi]不得不制作一台。去年夏天他做到了，而且看起来棒极了。

他选择了罗斯托克的设计，不仅因为它可以快速打印大型零件，还因为它令人着迷的操作。3D 打印机通常在最初几分钟很有趣，但 Rostocks 在其末端执行器的运动中更具机器人吸引力(机器人行话，工具头)。

这个建筑最酷的部分是[Turi]选择的围墙。他的地下室有一个宜家橱柜积满了灰尘，所以他决定利用它的抽屉作为罗斯托克的主体结构。一点木材工作和一些哑光黑色喷漆后，他有一个很棒的外壳！构建的其余部分非常标准，使用 3D 打印部件，安装在 Arduino Mega 上的 RAMPS 1.4 控制板，以及计算机电源。不过，他确实用碳纤维箭制作了自己的控制臂！
