# 最近的娱乐周刊上有个该死的手机

> 原文:[https://hackaday . com/2012/10/05/theres-a-friendly-a-friendly-the-friendly-entertainment-weekly/](https://hackaday.com/2012/10/05/theres-a-friggin-cellphone-in-the-most-recent-entertainment-weekly/)

![](../Images/6bc8a5c01b7b01d56a9b3d4cfef0e9d8.png "cellphone-in-entertainment-weekly")

Hackaday 的读者们跌跌撞撞地互相发送一个关于娱乐周刊杂志内的这款 Android 手机的链接。感谢所有把它送进来的人，让他们继续来。我们宁愿得到太多的小费，也不愿一点小费都没有！

我们首先要解决的是休息后观看视频时的不适感。如果你有过拆开电子产品看看里面是什么的经历，看着这一对笨拙地穿过它，你会感到身体不适。这让我们想对视频做一些 MST3K 风格的过度渲染，但他们在描述中的内容所有权声明让我们确信，如果我们这样做，我们会立即收到撤下通知。

无论如何，我们这里有一些非常酷的技术，你几乎肯定会得到。在上图中，你可以看到右边的小液晶显示屏。当页面打开时，由于图像中滑动开关的指向，它变得栩栩如生。在设备开始滚动 CW Twitter feed 的项目之前，会播放一些电视节目的宣传片。当硬件脱离页面时，会有一些有趣的修补工作要做。缩短键盘覆盖上的联系人(视频开始大约 8 分钟)会弹出一个亚洲语言菜单，很明显是 Android。

这已经不是杂志第一次做这样的事情了。几年前，哥伦比亚广播公司嵌入了视频，但我们很确定，这并没有使用手机的全部功能。可惜这几期很少(两个城市只有 1000 期)，因为我们很开心[用 epaper 黑掉了那期《君子》杂志](http://hackaday.com/2008/10/14/how-to-make-an-e-paper-clock-and-hack-esquire-magazine/)。
