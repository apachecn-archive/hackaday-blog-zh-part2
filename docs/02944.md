# DIY 照明解决方案

> 原文:[https://hackaday.com/2013/03/10/diy-lighting-solutions/](https://hackaday.com/2013/03/10/diy-lighting-solutions/)

[![pvc-lights](../Images/82cc32b5c9bcd16253eb19d133e66095.png)T2】](http://hackaday.com/?attachment_id=95907)

随着夏令时的开始，你可能不太需要照明，但这对黑客应该保持一切良好的照明，无论是室外还是室内。休息之后，请观看两人的视频。

[第一个照明解决方案](http://www.youtube.com/watch?v=PaxP72L8QeI&feature=youtu.be "PVC light stand")来自[【本】的 Youtube 频道](http://www.youtube.com/user/kelna2?feature=watch "Youtube Channel")。这是一个简单的解决方案，将一个夹钳灯压入一个 1 英寸的 PVC 三通中，将灯连接到管道上。底座由聚氯乙烯制成，形状为三英尺，用于(希望)坚固的休息。根据需要，可以使用几个灯，可能会很好地用于制作他的下一个视频。

第二盏灯也来自 Youtube ，是关于将普通的 LED 灯转换成更亮的灯。跳到 7 点左右看室外对比。你可能想也可能不想做这样的事情，但是你永远不知道什么时候你可能会想要换掉[你的闪光灯](http://en.wikipedia.org/wiki/Blinkenlights "blinkenlights Wikipedia")来吓唬邻居！
