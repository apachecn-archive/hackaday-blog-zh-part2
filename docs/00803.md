# 自动进给笔打印机

> 原文:[https://hackaday.com/2012/06/21/self-feeding-pen-printer/](https://hackaday.com/2012/06/21/self-feeding-pen-printer/)

![](../Images/5cc83a754c2ed244904117e9a3008970.png "self-feeding-pen-plotter")

乍一看，我们认为这是一个绘图仪，但它实际上是一个点阵(或线矩阵)打印机。[Bruno] [用 DVD 光驱](http://forums.modretro.com/viewtopic.php?f=3&t=10069)的零件做出来的。它能够沿 Z 轴和 X 轴移动笔，并沿 Y 轴送纸。

休息后的视频显示机器正在打印 Megaman，这是一个非常适合低分辨率像素的图像。但是，即使没有你可能从热敏打印机得到的高像素计数[，我们也只是喜欢这个的外观。谁没有一个光驱坐在那里等着被黑呢？看起来你需要采购的一个部件是步进电机和齿轮传动送纸轮。](http://hackaday.com/2012/04/15/printing-anything-with-a-thermal-receipt-printer/)
