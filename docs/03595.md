# 你将拥有的最先进的微波炉

> 原文:[https://hack aday . com/2013/07/12/the-most-advanced-microwave-you-ever-own/](https://hackaday.com/2013/07/12/the-most-advanced-microwave-youll-ever-own/)

![raspberry-pi-microwave](../Images/1731ee463bc88b71dafa768547ab00c9.png)

语音激活、一键式烹饪、网络控制、基于 UPC 的烹饪设置……你见过具备所有这些功能的微波炉吗？我们确实没有。我们觉得我们的有三种不同设置的再加热按钮很不错。但是天啊，如果你真的可以把你的微波炉设定到你选择的精确设置呢？你可以，如果你[让树莓酱来烹饪](http://madebynathan.com/2013/07/10/raspberry-pi-powered-microwave/)。

这种破解进行得很深入，导致最终产品具有高[晶片](http://en.wikipedia.org/wiki/Wife_acceptance_factor)。内森从拆开他的旧微波炉开始。他拍摄了组成控制按钮矩阵的柔性薄片的照片，以便对它们的设计进行逆向工程。这让他蚀刻了自己的电路板，将输入连接到 Raspberry Pi 板上，并控制该设备的所有其他硬件。因为它还驱动七段显示，你再也不会在这台设备上看到错误的时间。它是根据 [NTP](http://en.wikipedia.org/wiki/Network_Time_Protocol) 设定的。

我们提到你可以调整特定食物的设置。演示视频中显示了实现这一点的最佳方式。网络界面用于编程设置。召回它们就像使用条形码阅读器扫描 UPC 一样简单。太神奇了。

现在你可以让那台旧微波炉继续工作，而不仅仅是为了零件而报废它。
