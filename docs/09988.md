# 变形金刚，伪装的生日蛋糕

> 原文：<https://hackaday.com/2015/09/01/transformers-birthday-cakes-in-disguise/>

[Russel Munro]决定全力以赴为他儿子的生日蛋糕:他做了一个[变形金刚机器人蛋糕，嗯，从一辆卡车变成一个机器人，擎天柱风格](http://russellmunro.blogspot.com.au/2015/08/transforming-birthday-cake-animated.html)。他令人印象深刻的建立有原来的行动:首先，前面抬起头，然后背部抬起形成身体，头部和手臂弹出顶部。在薄薄的软糖外表下面是一个 3D 打印的身体，由底部的一个机制驱动。他用钓鱼线吊起零件，由从 CD 播放器中回收的马达拉动，由 Sparkfun 的 [EasyDriver 板驱动。](https://www.sparkfun.com/products/12779)

他必须克服的主要问题是重量:显然他低估了覆盖蛋糕的软糖的重量，不得不做一些最后的工作来加强驱动机制，并跳过他妻子计划的装饰更华丽的版本。但是当他在派对上操作它时，他儿子脸上喜悦的表情是最精彩的部分。在电脑动画和电脑游戏盛行的今天，提醒孩子们在独创性和大量的热熔胶中仍然可以找到很多乐趣是很好的。
