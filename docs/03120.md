# 通过 USB 为 Oculus Rift 供电

> 原文:[https://hack aday . com/2013/04/03/用 usb 为 oculus-rift 供电/](https://hackaday.com/2013/04/03/powering-the-oculus-rift-with-usb/)

![rift](../Images/ea92f2e813f8c71a0e2c3fe5efd68305.png)

Oculus Rift 开发套件开始出现在全球制造商的工作台上。有些人甚至会拆开硬件，看看他们如何改进它。[Mike]不喜欢 Oculus Rift 需要壁式电源的事实，所以他修改了它，使它可以通过 USB 端口供电。

这个 mod 的关键洞察力来自于[Mike]在他的插座和 Rift 的电源适配器之间放置了一个 Kill-a-Watt。他发现，假设适配器的效率为 100%，Rift 只使用了 600 mA 的电流。USB 端口应该提供 500 毫安的电流，所以用焊枪，[迈克]桥接了 DC 输入插孔和 Rift 上的 USB 端口。也许不出所料，一切都运行得很完美。

如果你已经忘记了， [Hackaday 正在获得这些 Oculus Rift 开发套件中的一个](http://hackaday.com/2012/08/17/were-getting-an-oculus-rift-dev-kit/)。当[Caleb]学会分享的时候，我们会发布一个拆卸的消息。休息之后，你可以看看[迈克]改造的裂缝的视频和一些 dolla dolla billz。
