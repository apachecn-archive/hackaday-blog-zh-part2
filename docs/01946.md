# 自动化你的喝茶时间

> 原文:[https://hackaday.com/2012/10/22/automate-your-tea-time/](https://hackaday.com/2012/10/22/automate-your-tea-time/)

一杯好茶包含几样东西。一个是将水保持在正确的温度，另一个是浸泡适当的时间。该产品通过在精心选择的浸泡时间后[取出茶包来解决后者。](http://www.doityourselfgadgets.com/2012/10/automatic-tea-timer.html)

很难想象还有比这更简单的了。计时装置是一个便宜的鸡蛋计时器，它被修改成包括一个随着分针移动的回形针。当计时器归零时，回形针接触一个固定的电极，给马达供电。那个马达是一个死的光盘驱动器上的激光滑板。由于这些通常是因为镜头(而不是机械装置)而死，这是内部零件的一个很好的重用。雪橇滑向一边，直到碰到一个切断电源的限位开关。同时，这个动作使用木制杠杆来提取茶袋。所有这些都在跳转后的剪辑中进行了解释。

因为煮蛋计时器已经有了自己的铃声，你甚至会被提醒到喝茶时间了！
