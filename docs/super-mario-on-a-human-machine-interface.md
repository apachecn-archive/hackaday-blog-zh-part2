# 人机界面上的超级马里奥！

> 原文：<https://hackaday.com/2014/04/05/super-mario-on-a-human-machine-interface/>

![super mario](img/74b5ca3e1987f933c226cc83dba92a78.png)

让超级马里奥在你的 TI-83 计算器上工作几乎是年轻极客的成年礼，所以我们真的很喜欢这个项目，[Chad Boughton]设法让它在 PLC 的 HMI 屏幕上运行！

他使用的是丹佛斯 DP600LX 微控制器，带有 HMI 显示器和 CAN 总线操纵杆。这种设备通常用于控制液压系统，以及显示传感器数据— [Chad]很好奇他是否也可以用它制作动画—看起来他成功了！有趣的是，我们以前见过那些“操纵杆”,看到它们用于这样的事情很酷——就像[Chad]说的，它们通常用于启动液压缸和气压缸。

休息后留下来看马里奥吃蘑菇。
