# 伙计，我的车呢？

> 原文:[https://hackaday.com/2014/09/10/dude-wheres-my-car/](https://hackaday.com/2014/09/10/dude-wheres-my-car/)

有人偷了你的车。他们就在你眼皮底下把它拿走了，而你却不知道它在哪里。幸运的是，你安装了 GPS 追踪器，可以精确定位小偷开走的车辆。

当一些意想不到的事情发生时，在你的车上安装一个 GPS 追踪器变得非常有用。如果汽车突然消失，采取必要的预防措施以确保安全的跟踪系统可以节省大量时间和金钱。

帮助解决消失车辆问题的是 Cooking Hacks 的一个聪明、年轻的团队，他们创建了一个循序渐进的教程，展示如何创建一个自制的 GPS 追踪器。他们的设计是基于 Arduino 的，有一个 GPS+GPRS 的屏蔽，并附有一个天线，以持续获取车辆的位置。打电话给 Arduino inside 会触发一条 SMS 消息，其中包含追踪器所在位置的特定 GPS 数据。然后，信息被设置到服务器，并被插入到数据库中，打开一个专门的 Android 应用程序就可以访问该数据库。

我们以前也见过类似的想法，比如这个[被盗自行车 GPS 追踪器](http://hackaday.com/2014/09/02/gps-tracker-tracks-your-stolen-bike/)，但是这个由 Cooking Hacks 开发的项目是独一无二的，因为它的手机集成了谷歌地图。更不用说，他们为这个项目制作的视频棒极了。

如果你开发了这样的系统，一定要在评论里让我们知道；休息之后别忘了看看他们的视频。
