# 动力双摆是一个混乱的展示

> 原文:[https://hack aday . com/2015/02/25/powered-double-pendulum-is-a-chaos-display/](https://hackaday.com/2015/02/25/powered-double-pendulum-is-a-chaotic-display/)

如果你以前没见过双摆，那基本上就是一个摆的末端连着另一个摆。你可能不认为这有什么特别的，但是如果有足够的能量进入系统，这些设备会表现出极其混乱的行为。结果通常是吸引注意力的展示。[David]想建造自己的双摆显示器，但他想让它自己驱动。结果是一个[驱动的双摆](https://www.youtube.com/watch?v=7fDaeCDDVH0&feature=youtu.be "Powered double pendulum")。

这里没有太多的构建细节，但该设备非常简单，我们可以从演示视频中推断出它是如何工作的。它分为两个主要部分:框架和钟摆。这个框架似乎主要是由木头制成的。前板由夹在一起的三层制成。中间切了一个槽，可以让轨道直线上下滑动。轨道的设计方式是，它像轨道一样安装在前板的外层之间。

钟摆附在线性轨道上。轨道上下移动，将能量输入钟摆。这导致钟摆实际上移动并产生混沌行为。安装在底座上的电动马达使导轨上下滑动。机械工作类似于曲轴上的活塞。马达看起来好像是安装在一个木制支架上，这个支架是用激光切割机精确切割的。最终产品运行良好，尽管有点嘈杂。我们还想知道，如果马达的旋转增加了一点随机性，这个系统会不会看起来更有趣。或者他也可以在末端安装一个喷漆器。
