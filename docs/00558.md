# 迷你弹球 MAME 机小到足以适合任何游戏室

> 原文:[https://hackaday . com/2012/05/28/mini-pinball-mame-machine-is-small-to-fit in-any-game-room/](https://hackaday.com/2012/05/28/mini-pinball-mame-machine-is-small-enough-to-fit-in-any-game-room/)

![](../Images/846d0b084ce161f011122aa44762ad50.png "pinmame")

在你的地下室里有一大堆弹球机是一个极客值得骄傲的成就之一，但是如果你有一个巨大的物理弹球机图书馆在你的指尖会怎么样呢？[veriix]在 [reddit 帖子](http://www.reddit.com/r/gaming/comments/u5vvb/a_build_project_i_finished_warning_lots_of_pics/)中分享了一个 [imgur gallery](http://imgur.com/a/tAId9) ，记录了他白手起家建造的小小弹球机。

弹球柜内有两台显示器。一个 4:3 的三星显示器作为机器的后玻璃，而一个 23 英寸的高清电视提供了游戏场地。在软件方面，[veriix]使用了运行在他闲置的旧主板上的[PINMAME](http://sourceforge.net/projects/vpinball/)和[Visual Pinball 9](http://sourceforge.net/projects/vpinball/)。结果令人印象深刻。高清监视器游戏场提供了正确的视角来欺骗[veriix]的大脑，使其认为他正在玩一台真正的弹球机。

在之前，我们已经看到过[PINMAME build，但是它们被装在全尺寸的弹球柜中，占据了太多的空间。[veriix]的机器要小得多，非常适合车库、书房、地下室或任何你想建立一个令人敬畏的游戏室的地方。](http://www.minhembio.com/Tordyvel/301314#389811)

休息之后，你可以看看[veriix]在玩他的迷你弹球机。谢谢[Johnny]送来这个。
