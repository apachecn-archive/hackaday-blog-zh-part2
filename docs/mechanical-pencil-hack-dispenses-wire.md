# 自动铅笔黑客分发电线

> 原文：<https://hackaday.com/2015/08/10/mechanical-pencil-hack-dispenses-wire/>

制作 PCB 原型时，您是否在寻找一种简单的电线管理方法？有没有考虑过用自动铅笔来分配它？事实证明，它工作得非常好——你所需要的就是一个 3D 打印的附件！

[Proto G]正在使用 Papermate 0.5mm 毫米自动铅笔，这意味着如果你有 0.5 毫米的金属丝(或焊料)，你可以用它来分配金属丝，而不会缠绕线轴。在演示中，他使用了 0.5 毫米的电磁线，上面有一层薄薄的搪瓷涂层，焊接时很容易融化。

3D 打印的线轴和导轨卡在自动铅笔的背面，让您可以装载大量的电线进行原型制作。他有所有的。STL 文件可在他的 Instructable 的情况下，你想添加这个工具到您的车间。
