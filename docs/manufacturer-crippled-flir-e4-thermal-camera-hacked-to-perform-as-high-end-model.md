# 制造商瘫痪的前视红外 E4 热感相机被黑成高端型号

> 原文：<https://hackaday.com/2013/11/04/manufacturer-crippled-flir-e4-thermal-camera-hacked-to-perform-as-high-end-model/>

上个月，[迈克]看了一下前视红外 E4 热成像相机。当你需要捕食者的视觉时，这是一个很好的工具，[但他在](http://www.youtube.com/watch?v=NtqUE67BUDI)内部发现的东西要酷得多:似乎前视红外 E4 背后的工程师通过让 1000 美元的 E4 内部的电路与 6000 美元的前视红外 E8 相同，让他们的生活变得更容易。

这只能说明一件事，[【麦克】已经送来了](http://www.eevblog.com/forum/testgear/flir-e4-thermal-imaging-camera-teardown/msg321956/#msg321956)。他将前视红外 E4 的固件升级到了 E8，大大提高了分辨率——E4 从 80×40 提高到了 E8 的 320×240。

黑客本身就像把 E4 的序列号放在一个配置文件中，压缩几个文件，然后用 Flir 工具安装它一样简单。一个非常简单的 mod(有一个很棒的拆卸视频),可以把一个 1000 美元的热成像相机变成 6000 美元的高端型号。
