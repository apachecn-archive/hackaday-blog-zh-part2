# 作为警察和消防无线电扫描仪的 SDR

> 原文：<https://hackaday.com/2013/04/25/sdr-as-a-police-and-fire-radio-scanner/>

如果你对你买的 DVB 加密狗失去了兴趣，想尝试一下软件无线电，你应该把它拿出来。[Harrison Sand]刚刚完成了关于如何使用 SDR 监听警察和消防无线电波段的指南。

该项目在休息后可以听到清晰的音频接收，它使用一个 Windows 盒子上的一整套包列表来访问紧急波段。SDRSharp 处理硬件工作，[在其他 DVB 加密狗黑客](http://hackaday.com/2013/01/12/listening-in-on-weather-balloons-with-rtl-sdr/)中很受欢迎。在这种情况下，加密狗是一个 Newsky TV28T v2 模块，他花了几美元买的。他还使用了一些支持程序，包括将数据转换成音频的数字语音解码器。

我们想知道这将适用于多少地区。我们的理解是执法部门正在转向加密通信系统。但是我们真正知道的是，你可以用一个儿童玩具来干扰这个系统。
