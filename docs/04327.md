# 自动控制

> 原文:[https://hackaday.com/2013/10/29/retrotechtacular-automata/](https://hackaday.com/2013/10/29/retrotechtacular-automata/)

![automatonWriter](../Images/5a0df48e2a0dd6fd419ccb9b1644c99d.png)

暂时放下你的世界观，接受笛卡尔的机械论解释，即生物本质上是复杂的机器:一组内在不相关的部件移动和碰撞。于是，自动机代表了机械宇宙中成就的顶峰，需要相当多的技巧来建造。他们的大多数发明者，如皮埃尔·贾凯-德罗兹，都是钟表匠或钟表匠，而像 240 岁的少年作家这样的自动机则挤满了移动部件来实现自动运动。

考虑到它所有的活动部件——近 6000 个——完全适合这个男孩的身体，而且人们可以“编程”这个男孩创作的文本，贾凯-德罗兹的作家尤其令人印象深刻。声称这些钟表娱乐是计算机的前身听起来有点夸张，但它们影响了发明家和工程师几个世纪。

你可能听说过另一个著名的机器人:[土耳其人](http://www.youtube.com/watch?v=lfDDcaewlZU)(这实际上是一个骗局，在其基地内安置了一名操作员)。然而，这个土耳其人设法[启发了查尔斯·巴贝奇](http://www.wired.com/wired/archive/10.03/turk_pr.html)去建造一个能够执行数学功能的机械装置:即[差异引擎](http://en.wikipedia.org/wiki/Difference_engine)。

休息之后，请观看贾凯-德罗兹的其他钟表杰作的视频。像罗伯特-胡丁这样的魔术师负责建造许多自动机，所以我们建议你通过查看另一位魔术师的古怪表演来保持神秘的气氛。
