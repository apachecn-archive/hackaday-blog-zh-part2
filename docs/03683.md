# 心形心脏模拟器

> 原文:[https://hack aday . com/2013/07/27/heart-shaped-heart-simulator/](https://hackaday.com/2013/07/27/heart-shaped-heart-simulator/)

![Heart](../Images/b3ab46d13abefaca3c921fa83463a35b.png)

几年前，Tymkrs 的[Addie]组织了一个怪异的万圣节小项目:[一个模拟心脏电信号的小螺旋桨板](http://hackaday.com/2011/11/04/mimicing-a-heartbeat-in-sound-and-electrical-pathways/)。作为一名心脏科护士，她认为她的项目需要一点改进，两年后[她终于完成了](https://www.youtube.com/watch?v=VLxsLt2oI4A&feature=youtu.be&hd=1)。这是一个心形的板，模拟电信号在心脏中的流动。

有几个关键区域通过心脏传导电信号——窦房结、房室结和束支都像管弦乐队中的演员一样工作，以保持心脏正常跳动。如果其中一个出了问题，心脏就会出现心动过速或纤维性颤动——无论如何都不是好事。[Addie]的板模拟了一个心脏可能出现问题的所有不同方式，用 led 代替真实心脏中的电信号。这个游戏的名字是看着发光二极管，告诉心脏处于什么状态。 **T2】**

PCB 心脏只是[Addie]心脏模拟器的一部分。模拟心脏也可以插入一个整洁的心形项目盒，用螺线管、LCD 显示器、耳机插孔和其他电子设备连接起来，将这个电子心脏变成一个完整的心律研究工具。[Addie]病房的护士喜欢这个东西，看起来[Addie]可能有一个真正的心脏训练工具。
