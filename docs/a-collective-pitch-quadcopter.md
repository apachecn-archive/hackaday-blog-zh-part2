# 集体俯仰四轴飞行器

> 原文：<https://hackaday.com/2013/12/04/a-collective-pitch-quadcopter/>

四旋翼飞行器并不是什么新鲜事物，但尽管多旋翼飞行器取得了进步，它们仍然属于“在发动机上固定一个支柱，然后重复三次”的模式。世界顶级遥控直升机飞行员之一[Curtis Youngblood]为 quad 想出了一个非常酷的驱动系统，只需要一个马达，每个叶片的总螺距允许绝对疯狂的杂技能力。

[柯蒂斯]称他的新玩具为“黄貂鱼 500 ”,里面只有一个马达。它在这个四人组的 H 形框架的后面，用一对滑轮连接到沿着脊柱向下的轴上。所有四个转子都由这个旋转轴驱动。

因为[柯蒂斯]是一名杂技飞行员，他需要一种方法在多个方向上控制他的直升机。为了做到这一点，他在四轴飞行器的每个臂上添加了四个伺服系统，给每个旋翼提供集体俯仰，就像真正的直升机的尾旋翼一样。其结果是一架四轴飞行器可以非常轻松地上下颠倒飞行，进行桶滚，以及真正的 3D 遥控飞行器可以做的所有其他机动动作。

Flite Test 的牛逼家伙让[Curtis] [参观了他们的机库](http://www.flitetest.com/articles/stingray-500-review)，让他做了一次牛逼的演示飞行。你可以看看下面的视频。
