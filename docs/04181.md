# 50 英寸多点触控桌价格昂贵，不可破坏

> 原文:[https://hack aday . com/2013/09/29/50-multi touch-table-is-贵-不可破坏/](https://hackaday.com/2013/09/29/50-multitouch-table-is-expensive-indestructable/)

![50inchMultiTouch](../Images/7716287a4806f0934751dd74dafb6881.png)

如今，逛逛一个资金充足的博物馆，你很可能会发现四处散落着互动展品，比如这个光滑的 [50 英寸投影多点触控桌子](http://www.instructables.com/id/DIY-MT-50-Multitouch-Table/)。负责这一美景的公司 [Ideum](http://ideum.com/) 已经停产 MT-50 型号，转而采用 LCD 版本，并发布了旧型号的计划，作为[开放展览倡议](http://openexhibits.org/)的一部分。这对…嗯，所有人来说都是好事！

框架由交叉穿过全钢车身的铝制支柱组成，车身安装在脚轮上以方便移动。电脑规格似乎可以与现代游戏装备相媲美，并且依赖于 IEEE1394 摄像头输入。多排高强度红外 LED 灯条的成本开始增加，每卷可达 200 美元。玻璃是定制的，10 毫米厚，一侧有投影膜，并经过[微蚀刻](http://touchinternational.com/touchblog/sputtering-and-micro-etching-touch-screen-performance/)以减少反射，并将视角增加到近 180 度。投影仪是一台 InFocus IN-1503，它的[投影投比](https://en.wikipedia.org/wiki/Throw_(projector))非常短，最终分辨率为 1280×720。

评论中提到的估计价格相当高:12k-16k 美元。请告诉我们您自己的意见，哪些替代部件可能会降低成本，并观看下表的视频概述，以及其耐用性的视频演示。对于另一个自己动手建造博物馆，看看比尔·波特的"[反应时间挑战。](http://hackaday.com/2013/08/26/reaction-time-challenge/ "Reaction time challenge")
