# 计算器+ MSP430 +红外 LED =电视遥控器？

> 原文：<https://hackaday.com/2014/12/08/calculator-msp430-ir-led-tv-remote/>

避开商店购买的解决方案，[Stefan]设法用一个旧计算器制作了一个[电视遥控器。计算器的大脑被丢弃，换成了 MSP430，只留下了按钮矩阵和外壳。在被红外代码计时难住之前，他没有去查，而是成功地手工绘制了矩阵图。一些研究向他指出了三星红外代码的一个特点，在开源远程控制库的帮助下，他让它工作了。](http://codingequalsuperpower.tumblr.com/post/104241248678/building-a-calculator-ir-remote "TV remote out of an old calculator")

当范围太有限而不能满足他时，他增加了一个升压电路和一个 LED 驱动器，这是他从一个旧遥控器上拆下来的；现在它能在 30 英尺外工作。一些电工胶带和热胶后，它都适合回到原来的情况。

它不能[拍照](http://hackaday.com/2014/11/29/articam-interfaces-game-boy-camera-with-ti-calculators/)或[玩超级粉碎兄弟](http://hackaday.com/2014/11/17/super-smash-bros-on-a-calculator/)，但它能做遥控器需要做的事情:浏览指南中的频道、控制音量、打开或关闭电视。考虑到所有这些计算器都是用来做无聊的基本算术，这是一个拖延升级。

休息后看视频笑一笑。
