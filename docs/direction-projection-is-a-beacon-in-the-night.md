# 方向投影是黑夜中的灯塔

> 原文：<https://hackaday.com/2015/05/13/direction-projection-is-a-beacon-in-the-night/>

用手机导航可能会很麻烦:手机会显示一张小小的地图，你开车时根本不应该看它，但你当然会看。【Mikeasaurus】有终极解决方案:[方向投射！](http://www.instructables.com/id/Direction-projection)迈克创造了一个增强现实系统，没有眼镜平视显示器，也没有护目镜 ala [微软 Hololens](https://www.microsoft.com/microsoft-hololens) 。前方的路是他的画布。安装在他汽车顶部的标准投影仪显示地图和转向指示器，所有这些都来自他的手机。连接电话和投影系统通常需要穿过屋顶的 HDMI 或模拟视频电缆。[Mikeasaurus]通过使用 Chromecast 简化了这一过程，它允许他通过 WiFi 传输手机屏幕。

投影仪本身是 HD25-LV，Optima 的 3500 流明型号。HD25-LV 支持 1080p，尽管在这种情况下，亮度比分辨率更重要。[Mikeasaurus]安装了投影仪以及凝胶电池和 900 瓦 DC 到交流逆变器为其供电。一个移动 WiFi 热点填补了屋顶工具包。把这样一个昂贵的装置放在汽车顶上会招致灾难——无论是雨水、石头还是盗窃。[Mikeasaurus]提前想好，把他的装置绑在一个车顶货箱里。盒子前面有一个塑料覆盖的孔，允许投影仪在路上拍摄，同时保护其镜头。我们希望增加一个通风口和风扇，以确保投影仪也能获得一些气流。

在路上，该系统实际上是有效的。可以理解的是，在白天它不会工作得很好，但在晚上系统真的会发光！不要尾随——你不希望你前面的司机知道你要去哪里，对吗？
