# 保时捷 911 电动版

> 原文：<https://hackaday.com/2012/11/16/porsche-911-made-electric/>

[Kurt]想要一辆电动车，一直想开一辆保时捷。一举两得，他决定将这些愿望结合起来[将一辆 2002 款保时捷 911 改装成电动车](http://kurtsprojects.blogspot.com/)。

在拆除了发动机、燃料箱、排气管、散热器和所有其他使内燃机工作的东西后，[库尔特]安装了一个[大功率电机、控制器](http://kurtsprojects.blogspot.com/2012/08/this-weekend-i-tested-warp11-hv.html)和 72 个重量超过 500 磅的磷酸锂电池。在过去的几个月里，他在解决问题的同时已经让赛车跑了 300 多英里，但现在他终于解决了系统中的问题，让他可以达到相对较高的速度。

[Kurt]已经把他的新座驾开到 100 英里/小时，并做了一点点[范围测试](http://kurtsprojects.blogspot.com/2012/10/range-testing-ha.html)，这告诉他他应该预计他的新座驾每次充电大约 40 英里。这不完全是他所希望的，但足够在城里骑着时髦的车四处逛逛了。

休息之后是一段视频[Kurt]第一次试驾他的电动保时捷。
