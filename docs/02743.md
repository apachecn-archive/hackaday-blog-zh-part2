# 手握超级小阿鲁迪诺:FemtoDuino

> 原文:[https://hack aday . com/2013/02/11/hands-on-the-super-tiny-Aru dino-femto duino/](https://hackaday.com/2013/02/11/hands-on-with-the-super-tiny-arudino-femtoduino/)

[![photo(30)](../Images/cbb08878e4f5cd0d0ba99fb873f56aac.png)T2】](http://hackaday.com/wp-content/uploads/2013/02/photo30.jpg)

我们最初是在去年听说 FemtoDuino 的。它看起来足够好，足够小，但我们真的不需要它。不过最近，我们开始了一个新项目([，你可以在论坛上关注](http://forums.hackaday.com/viewtopic.php?f=3&t=3052)！)需要对现有电路进行简单的修改。空间和重量非常重要，所以我们决定以每只 25 美元的价格买几只 femtoduinos，让它们试一试。

这些东西很小。他们的脚印是 20.7×15.2mm，下图可以看到，有四分之一可供参考。很小。坦白说，他们没什么好说的。它们是兼容 arduino 的 Atmega328。我计划用我的红牛 Arduino 来编写这个东西，因为你需要带你自己的串行接口。
T3![photo(31)](../Images/d385adaf75fe97582ef20d673bfce64c.png)T5】

如果你和我一样，你有糟糕的焊接和颤抖的手。我有点担心在没有桥接焊盘的情况下焊接这些导线。不过，我还是成功了。这里有一个视频，你可以看到我的焊接技术和设备是多么可怕。

[https://www.youtube.com/embed/6t-I71TqoTA?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/6t-I71TqoTA?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)T2】

我真的对这个东西没有任何抱怨，它的工作方式就像一个 arduino，但更小。最接近投诉的是，丝网印刷有点模糊，很难看出哪个引脚是什么。它并不可怕，但也不完美。我们实在想不出其他什么，所以我们决定用俳句来表达。

> 一个 arduino
> 比四分之一
> Femtoduino 小得多
