# 拉斯皮游戏机给我们其余的人

> 原文:[https://hack aday . com/2014/07/06/the-raspi-game boy-for the-rest-of-the-rest-of-us/](https://hackaday.com/2014/07/06/the-raspi-gameboy-for-the-rest-of-us/)

我们已经看到了相当多的 casemods，它们将树莓 Pi 填充到 Game Boy 中，将它变成一个非常酷的便携式 Pi 和复古游戏设备。这些建筑中的大部分使用一个经过改造的 20 岁的游戏男孩作为围栏，如果你对你的绿色屏幕老朋友有感情，你可能不想为了一个 Pi 项目而放弃它。[注意 Adafruit 有一个解决方案—[一个 3D 打印的 Game Boy 外壳](https://learn.adafruit.com/pigrrl-raspberry-pi-gameboy?view=all),它可以将 Pi 和 TFT 屏幕变成一个几乎可以放入口袋的 Raspberry Pi，带有在路上安装 RetroPi 所需的所有按钮和电池。

这个版本被称为 PiGRRL，它为 Pi 使用了 [Adafruit 触摸屏 TFT 套件](http://www.adafruit.com/product/1601)，有效地将 Pi 变成了一个非常小的平板电脑。这允许与 Pi 进行正常的桌面交互，而且它也足够小，可以放在最小的机箱中。

[3D 打印外壳](https://www.thingiverse.com/thing:382485)是这里的明星，允许完全访问大多数 Pi 的端口，同时在外壳的其余部分留出足够的空间来放置较大的电池、充电电路和从 SNES 控制器取出的按钮。

最终的结果是一个非常有用的便携式 Pi，恰好是加载一些 rom 和玩一些经典视频游戏的完美形式。下面视频。
