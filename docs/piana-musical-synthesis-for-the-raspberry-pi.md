# 钢琴——树莓派的音乐合成

> 原文：<https://hackaday.com/2015/02/24/piana-musical-synthesis-for-the-raspberry-pi/>

在过去 15 年左右的时间里，软件合成器缓慢但肯定地取代了那些打击乐、架子鼓和真正的合成器。这对老硬件爱好者来说是个损失，但至少现在每个拥有 MacBook 的人都是音乐家，对吗？

Raspberry Pi 和 Pi2 已经比 99 年的台式机拥有更强的处理能力，所以毫不奇怪，Moog 的所有这些经典合成器。雅马哈 DX，卡西欧 CZ，Linn 鼓机，Fairlight，和一个 mellotron，都可以塞进一个 Pi，这要感谢[Phil Atkin] [和他的树莓 Pi 合成器](http://raspberrypisynthesizer.blogspot.nl/)的工作。

[Phil]将音频合成引入 Pi 的努力可以归结为三种技术:减法合成、相位失真合成和基于样本的合成，这是从 Akai MPCs、MacBooks 和《考斯比秀》一集中发现的东西。[Phil]正在将所有这些技术结合到一个能够在 Pi 上无缝运行的软件中，给任何一个拥有 35 美元计算机的人一个在 1985 年价值几千美元的工具。

该项目进展顺利，但最近发布的《覆盆子 Pi 2》[让【菲尔】陷入了困境](http://raspberrypisynthesizer.blogspot.nl/2015/02/raspberry-pi-2.html)。一方面，Pi 2 在硬件方面比原来的 Pi 更强大，这有助于提供更多的声音和更好的 GUI。另一方面，有数以百万计的原始圆周率 1 仍然有助于卓越的合成器。不管怎样，[菲尔]的作品是一个很好的例子，说明你可以用音频作品把 Pi 推向多远。
