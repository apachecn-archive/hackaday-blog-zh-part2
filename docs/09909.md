# 面包板坏掉了

> 原文:[https://hackaday.com/2015/08/24/breadboards-go-to-pot/](https://hackaday.com/2015/08/24/breadboards-go-to-pot/)

无焊试验板非常适合集成电路和分立元件，如电阻、电容和晶体管(至少是通孔型)。他们不太擅长固定电位计之类的大型元件。当然，你可以把修剪器塞进去。你也可以把引线焊接到一个罐子上，但是这样不美观，而且在拿的时候容易拔出来。[PaulStoffregen]对此感到厌倦，所以他组装了一些好看的 PC 板，将一个 6mm 的轴锅牢固地安装在试验板上。

[Paul]注意到，在试验板上进行精细的调整会抑制人们玩演示电路。新的设置邀请人们做出调整。图片和视频显示了一个有六个引脚的早期版本，但[Paul]在最近的一批中增加了两个引脚，以增加试验板的抓地力。

这是具有巨大效用的简单事物之一。你可以很容易地自己设计一个这样的电路板，甚至手工布线，但是[Paul]也把电路板贴在了 OSH Park 上，所以你可以从那里订购。

与由通用组件组成的面板[相比，这有利有弊。这种技术快速且易于重新配置。另一方面，面板可以容纳很多物品，而不仅仅是罐子。](http://hackaday.com/2015/08/14/panel-mounted-breadboard-accessories/)
