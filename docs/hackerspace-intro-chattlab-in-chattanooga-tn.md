# 黑客空间简介:查塔努加总氮查塔实验室

> 原文：<https://hackaday.com/2012/06/06/hackerspace-intro-chattlab-in-chattanooga-tn/>

[https://www.youtube.com/embed/ODfUV1hViEo?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/ODfUV1hViEo?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

在我们穿越田纳西州的旅途中，我们得以一窥尚未开放的黑客空间。位于查塔努加的查特*实验室刚刚获得了一个空间，并开始放入设备。查塔努加曾经被称为“南方的发电机”,因为那里有大量的工业，一段时间以来一直需要一个志同道合的黑客聚集的地方。他们已经通过各种方式在网上交谈了一段时间，但最终找到了一个适合他们需求的空间。

[Tim Youngblood]带我们参观了他们目前拥有的设备，并分享了一些他们未来的计划。正如你在视频中看到的，那里还没有很多东西。有几个不错的工具和一个非常漂亮的看起来相当大的数控机床。然而，他们有坚实的计划和坚定的人，所以我们预计这将很快扩大。说到扩张，这个空间可能看起来很小，但那只是因为我们分享了活跃的空间。该设施的其余部分是空的，可供他们在成长过程中扩展！

让这作为一个提醒，我们不在乎你的黑客空间是大是小，给我们发一个参观，我们会发布一个关于它的故事。几乎在每一个黑客空间的介绍中，我们都看到人们不知道他们附近有一个黑客空间的评论。
