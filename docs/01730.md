# 如何为索尼 NEX-5 系列 DSLR 相机添加音频

> 原文:[https://hackaday . com/2012/09/27/how-to-add-audio-in-the-Sony-NEX-5-line-of-of-dslr-camera/](https://hackaday.com/2012/09/27/how-to-add-audio-in-to-the-sony-nex-5-line-of-dslr-cameras/)

![](../Images/4ab613c5f0ef1279c67c8051dc881e06.png "add-audio-in-to-sony-dslr")

[Tynan]喜欢他的索尼 NEX-5 相机，但他受够了在录制视频时无法选择任何外部麦克风。最近，他开始着手补救这一点，并设法在不修改相机本身的情况下在插孔中添加音频。

这里真正的技巧是修改麦克风配件与摄像头的连接方式。在[Tynan 的]教程视频(插播后)中，他使用闪存模块的外壳作为连接器。在移除电子设备后，他留下了足够的空间来放置索尼麦克风配件。这些包括 PCB 和布线，但不包括麦克风元件本身。闪光灯外壳上增加了一个 3.5 毫米音频插孔，并焊接到麦克风电缆上。现在他有了一个模块化音频输入插孔。唯一的问题是他的修补只导致了单核细胞增多症。如果你不介意多花一点时间对废弃的麦克风进行逆向工程，我们打赌你可以把它变成一个真正的立体声选项。
