# 为 Raspberry Pi 摄像机添加夜视功能

> 原文：<https://hackaday.com/2013/05/27/adding-night-vision-to-the-raspberry-pi-camera/>

在几个月的承诺之后，树莓 Pi 相机终于走向了世界各地的黑客和制造商。*当然*用 Pi cam 美化 Hackaday 页面的第一个构建将是移除 IR 过滤器，而这恰好发生在 Reading hackerspace [的【加里】和他的团队是第一个这么做的](http://rlab.org.uk/wiki/Remove_IR_filter_from_Raspberry_Pi_Camera)。

正如[Gary] [在他的视频](http://www.youtube.com/watch?feature=player_embedded&v=8nl_qJ2XCLU)中所展示的，移除 Pi 摄像头的红外滤镜的过程*极其*复杂。将过滤器从相机中取出需要移除传感器，用手术刀轻轻切开它，最后用一点点强力胶将整个东西粘回一起。心脏不好的人不能用，没有像样的台式显微镜的人更不能用。

如果你正在寻找一个 Raspberry Pi 驱动的安全摄像头，游戏摄像头，或一些天文学应用程序，这是实现它的方法。移除红外过滤器时，您可能需要小心；[Gary]第一次尝试就弄坏了一台摄像机。不过，他们让它工作了，图像质量看起来很好，正如下面的视频所示。
