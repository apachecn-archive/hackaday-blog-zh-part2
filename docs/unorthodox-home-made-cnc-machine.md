# 非正统的国产数控机床

> 原文：<https://hackaday.com/2013/10/17/unorthodox-home-made-cnc-machine/>

这里有一个相当[非正统的国产数控机床](http://nullclan.wordpress.com/)我们以前从未见过！

[ModHappy]最近接受了一项挑战，用五金店里便宜易得的部件设计和制造一台数控机床。

在商场闲逛时，他注意到了管道岛，并开始摆弄地板上的 PVC 管，这让商场员工非常困惑。他建造了一个框架，但仍然需要一种方法来实现线性运动。他的电话在电气岛得到了回应，在那里他使用了导管壁挂支架！他们只需要一个小的改动，从抓住管子到沿着管子滑动。几个紧固件之后，一些来自损坏的扫描仪的步进电机，一个用于 z 轴的 CD-ROM 驱动器，一些使用固定螺钉组装的耦合器，机械构建完成了。

在控制方面，他使用了易贝的廉价中国 TB6560 数控驱动板，并对其进行了轻微修改以增加可靠性。整个过程在他的旧笔记本电脑上运行——值得注意的是，如果你还不了解 Linux 操作系统，它也可以实时运行。

休息后留下来看看它的运行，画出一个令人敬畏的网站的标志！
