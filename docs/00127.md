# (模型)直升机物理学

> 原文：<https://hackaday.com/2012/04/20/model-helicopter-physics/>

[![sideways helicopter](img/e74edef39f042e5a0477efa3d2fa6257.png "sideways-helicopter")](http://hackaday.com/2012/04/20/model-helicopter-physics/sideways-helicopter/)

如果你曾经想知道直升机是如何飞行的，或者只是想看看一些令人敬畏的遥控驾驶，休息后的四个[视频](http://www.youtube.com/watch?feature=player_embedded&v=WdEWzqsfeHM "smarter every day")应该正是你想要的！虽然一个人如何工作的基本物理学在最后三个中解释了，但是一个人仍然很难解释[卡尔]如何能够以他的方式驾驶他的遥控直升机。视频必须被看到才会被相信，甚至解释，但其中一个更简单的技巧涉及到起飞几英尺，做一个向前翻转，向后倒飞！

正如在[其他](http://www.youtube.com/watch?feature=player_embedded&v=MaB4k1SgfUg "smarter every day") [视频](http://www.youtube.com/watch?feature=player_embedded&v=d2mITQ53kr8 "smarter every day")中详细解释的那样，直升机由主旋翼上一个叫做旋转斜盘的东西控制，简而言之，它将线性动作转化为旋转动作。同样的事情也发生在尾桨上，但是你必须在休息后查看视频以获得完整的解释！在计算机出现之前，有人能想出这种模拟控制系统来使用，真是太巧妙了。

物理爱好者特别感兴趣的是，第四个视频中给出了对[陀螺进动](http://en.wikipedia.org/wiki/Precession "Gyroscopic precession")的解释。控制直升机可能不像你想象的那样工作！
