# 铰接式电脑灯点亮你的生活

> 原文：<https://hackaday.com/2015/05/17/articulated-computer-lamp-lights-up-your-life/>

[Samimy]从他的零件箱中拿出这个[铰接灯](https://www.youtube.com/watch?v=VtmtX4Wggpg) (YouTube 链接)用于他的电脑工作站。两块铝角形成灯的主体。几个支架用于形成两个铰链，使灯可以放置在[Samimy 的]显示器上方。在这种情况下，光线来自一对 4 瓦的 LED 灯泡。

[Samimy]在活动部件上使用了双螺母，以确保不会松动。外部螺母是橡子，确保没有人会被螺杆的外露部分割伤。[萨米]把两个灯泡连接在一个合适的并联电源电路上。开关是一个简单的开关，安装在灯末端的一块有机玻璃上。

我们想在这个建筑上看到的一件事是地线。由于所有暴露在外的铝和钢，一个松动的连接或一点绝缘磨损都可能使整个灯体带电。
