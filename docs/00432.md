# 惠普触摸板上的自供电 USB 主机模式

> 原文:[https://hack aday . com/2012/05/16/self-powered-USB-host-mode-on-the-HP-touchpad/](https://hackaday.com/2012/05/16/self-powered-usb-host-mode-on-the-hp-touchpad/)

![touchpad-usb-host-power](../Images/37a7633da5617003fc0bdb997bd0f220.png "touchpad-usb-host-power")

[ften]在他的惠普触摸板上运行 Android 非常有趣，但他很快发现，当在主机模式下运行时，平板电脑的微型 USB 端口不能为他的外围设备提供足够的电力。他开始四处挖掘，找到了在保持设备外观的同时提供额外电力的完美方法。

他把平板电脑撬开，在触摸板虚拟 SIM 卡插槽后面的空白区域安装了一个小型 DC 升压转换器。在将转换器连接到电池端子后，他在空槽中安装了一个微型 USB 适配器，经过一点打磨后，它完全适合。

他拼凑了一根 USB Y 形电缆，从他的新 USB 插孔中获取电源，同时通过原来的 USB 接口保留现有的数据连接。你可以在下面的视频中看到他的工作成果，虽然[ften]没有说他的 mod 对触摸板的电池寿命有多大影响，但他已经证实，一旦你不可避免地耗尽电池，它仍然会优雅地关闭。
