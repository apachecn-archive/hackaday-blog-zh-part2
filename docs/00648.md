# 蓝光数控看起来很适合品牌和雕刻

> 原文:[https://hack aday . com/2012/06/05/blu-ray-CNC-looks-great-for-branding-and-engraving/](https://hackaday.com/2012/06/05/blu-ray-cnc-looks-great-for-branding-and-engraving/)

![](../Images/285ef030d037d91431bfa79a2a2ad93d.png "blu-ray-CNC-branding")

[导航]得到了一个小激光切割机的 bug。他完成了构建，并且刚刚完成了第二个版本，做了一些很好的改进。他一直希望有一个激光切割机，但我们认为[这真的在标记物体](http://41j.com/blog/2012/06/bluray-laser-cnc-version-0-2/)时大放异彩，比如上面看到的碎木头。

这加入了一个漫长的光驱部件生产线。例如，我们看到这个[绘图仪使用了一些 CD-ROM 驱动器](http://hackaday.com/2010/11/08/cnc-machine-from-pc-parts/)中的镜头滑板。你可能认为[导航]不需要担心 Z 轴，因为这是一个激光器，但你错了。光的焦点需要打在正确的地方才能有效切割，这往往是激光切割机的麻烦。当材料被烧掉时，如果你不调整透镜的垂直位置，激光的效率就会降低。这就是为什么我们认为它是最好的雕刻师，但是[最初为他的切割师](http://41j.com/blog/2012/05/laser-cnc-from-dvd-parts/)制作的特写确实显示了一些在深色纸上切割字母的成功。
