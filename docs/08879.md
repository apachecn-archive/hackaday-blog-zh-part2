# 使用链锯制榫机的大块木头接头

> 原文：<https://hackaday.com/2015/04/25/massive-wood-joints-with-chainsaw-mortiser/>

木材加工中一种常见的细木工方法是榫眼和榫头。榫眼基本上是一块木头上的一个洞，而榫头是另一块木头，被切割成与那个洞紧密配合。榫通常用胶水、木钉或楔子固定到位。

[WayOutWest]的人正在建造一个围栏，他们需要在 4×4 英寸的柱子上切割一些榫眼，以容纳 2×6 英寸的栏杆。虽然他们有电锯，但试图用它手工切割榫眼被证明是超级危险的，因为每次刀刃接触木头时电锯都会发出声音。该团队有一些零件被踢来踢去，所以他们制作了一个[夹具来固定链锯](https://www.youtube.com/watch?v=h2CGslzGdB4&)，因为它被插入 4×4 的。

这个装置的框架是由一个旧的脚手架支架制成的，而滑道只是管道中的管道。链锯用螺栓固定在滑道上，一个杠杆使它前后移动。第二根杠杆上下移动要开榫眼的木头，这样榫眼就可以被切割成任何宽度。这是一个非常巧妙的建筑，只需要一点点努力，最终将节省大量的时间来榫眼无数的栅栏柱。
