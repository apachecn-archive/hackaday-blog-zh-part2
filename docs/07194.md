# 隔音盒使基于客厅的 CNC 布线变得可以忍受

> 原文:[https://hack aday . com/2014/09/28/sound-isolation-box-makes-living-room-based-CNC-routing-tolerable/](https://hackaday.com/2014/09/28/sound-isolation-box-makes-living-room-based-cnc-routing-tolerable/)

数控机床可能会很吵，尤其是如果它们配备了高速路由器主轴。不幸的是，如此大的噪音对外面的公寓居民来说可能是个问题。不要害怕！[佩特里]想出了一个解决办法。这是他的迷你 CNC 路由器的隔音外壳，兼作家具。它保留声音和灰尘，同时抽出一些很酷的部分…..在他的客厅里。

看起来像一个带软垫的盒子，实际上在设计上花了很多心思。前部 MDF 面板向下折叠，平放在地板上，这样用户可以跪在上面操作机器，而不会对门铰链施加不必要的压力。顶部还装有铰链，允许自上而下访问或快速查看作业状态。盒子的所有内角都被填缝成气密的，即使一点点空气通道也会让声音和灰尘逸出。两厘米厚的隔音线盒子的整个内部和两个检修盖有橡胶密封条，以确保关闭时的气密密封。

由于步进电机、主轴电机和控制电子设备都在一个封闭的盒子内运行，因此会产生一些热量。[Petteri]到目前为止没有任何问题，但他仍然安装了一个由 GFCI 插座和恒温器温度开关组成的超温断电装置。如果温度超过 50 ℃,该装置将通过有意断开 GFCI 插座来切断电源。内部零件在 300°C 以下都不会着火，所以有相当的安全缓冲。

虽然隔离盒做得相当不错，但[Petteri]承认还有改进的空间；当切割木材或铝时，噪音水平有点烦人。如果让他重来一次，他会用更厚的中纤板，20mm 而不是 5mm。然而，在切割塑料的一般使用过程中，刳刨机仍然比他的洗碗机更安静。
