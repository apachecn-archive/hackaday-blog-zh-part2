# 逆向工程:了解 SAGE 系统

> 原文：<https://hackaday.com/2015/03/31/retrotechtacular-wising-up-with-the-sage-system/>

超音速喷气式飞机的诞生让美国的空袭防御看起来过时了。因此，在冷战期间，政府与许多机构和供应商签订了合同，由西方电气公司作为项目经理来创建和维护[半自动地面环境(SAGE)飞机探测系统](http://www.youtube.com/watch?v=06drBN8nlWg)。

SAGE 是麻省理工学院林肯实验室在 IBM 制造的计算机上开发的。它使用了安/FSQ-7 实际上，这是有史以来最大的计算机。SAGE 作为一个国防部门网络运作，将美国大陆和加拿大分开。每个区域都有一个指挥中心，这是一个四层的混凝土碉堡，通过它自己的专用电站保护和操作 Q7。SAGE 计算机采用热备用处理器，以最大限度地延长正常运行时间，并在必要时故障转移到附近的指挥中心。

信息从陆地、空中和海上的许多雷达源输入到每个定向中心。研究结果在前端光线昏暗的房间里的观测仪上进行评估，并存储在后端的磁芯上。在定向中心的空中监视室处理的不可识别的飞机轨迹被发送到 ID 室，在那里它们被判断为友好。如果发现不友好，他们会被送到武器指挥室，以应对可能的后果。
