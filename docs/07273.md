# 斯克罗比在你的屋顶上，清洁你的太阳能电池板

> 原文：<https://hackaday.com/2014/10/08/scrobbys-on-your-roof-cleaning-your-solar-panels/>

太阳能电池板是一个伟大的，可持续的除了你家的能源计划。他们肯定会变脏，但他们不能承受苛刻的化学物质，仍然有效。虽然有些公司会一年来清洁你的设备几次，但这项服务是一项经常性的费用，会很快增加。[Stefan]通过参加 Hackaday 奖的 Scrobby，寻求建立一个高度可承受和可持续的解决方案，在安装后，不需要回到屋顶的危险旅程。

Scrobby 是太阳能驱动的，使用雨水清洁。用户可以通过手机蓝牙设置和更改清洁时间表。[Stefan]的原型是围绕 Teensy 3.0 构建的，但他最终将使用基于 Freescale KL26 的定制板。除了蓝牙模块，还有六个超声波传感器，雨水和温度传感器，以及用于拴系运动的电机驱动线轴。

在[Stefan]的演示视频中，观看 Scrobby 安装他的原型猪鬃并展示他的能力。要注册更新，请访问 Scrobby 的网站。如果你抓紧时间，你可以向[斯克罗比的 Kickstarter](https://www.kickstarter.com/projects/stefanhamminga/scrobby-solar-keeping-your-solar-panels-clean) 活动捐款。问题是，谁来清洁斯克罗比的太阳能电池板？
