# 制作自己的智能手表

> 原文：<https://hackaday.com/2014/02/17/make-your-own-smart-watch/>

可穿戴设备最近风靡一时。你最近是否一直在关注 Pebble 或一款新的智能手表，但不确定它是否适合你？有了[GodsTale]的[“复古手表”](http://www.instructables.com/id/Make-your-own-smart-watch/?ALLSTEPS)您现在可以打造自己的手表，让您无需巨额投资就能试用智能手表。

这款智能手表使用了非常常见且容易获得的部件: [Arduino Pro Mini](http://arduino.cc/en/Main/ArduinoBoardProMini) ，HC-06 蓝牙模块，Adafruit 的 [0.96 ' '有机发光二极管显示屏](http://www.adafruit.com/products/326)，以及一个锂电池。令人惊讶的是，这么少的零件可以用来制作这样一个功能性的项目。虽然所示的示例包装边缘有点粗糙，但它完成了工作。拥有如此简单的硬件使[GodsTale]能够专注于软件。这个项目最酷的一个方面是[GodsTale]提供的 Android 应用程序。该应用程序提供基本功能，如查看 RSS 订阅源和 Android 通知。查看 [GitHub](https://github.com/godstale/retrowatch) 和[更详细的报道](http://www.hardcopyworld.com/ngine/aduino/index.php/archives/670)了解更多信息。

看到这个项目在未来的发展将会很棒，它有如此大的潜力。我们希望看到定制电路板，或者这款出色的智能手表的 3D 打印外壳模型。休息之后，观看复古腕表的视频。如果你认为这很酷，看看[的](http://hackaday.com/2014/02/16/fixing-the-unfixable-pebble-smartwatch-screen-replacement/)和[最近的](http://hackaday.com/2014/01/12/squarewear-2-0-a-wearable-opensource-arduino/)黑客吧。
