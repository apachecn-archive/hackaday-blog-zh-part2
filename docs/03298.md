# 向订婚戒指添加发光二极管

> 原文:[https://hack aday . com/2013/05/20/adding-LEDs-to-an-engagement-ring/](https://hackaday.com/2013/05/20/adding-leds-to-an-engagement-ring/)

![ring](../Images/944c382d4cbf7867d9787e0314325168.png)

从前，一个书呆子遇到了一个女孩。事情发生了*就像他们做的*一样，最终【本】想要[创造出最重要、最终极的订婚戒指](http://www.kokes.net/projectlonghaul/projectlonghaul.htm)。([这里有一个宝藏](http://www.kokes.net.nyud.net/projectlonghaul/projectlonghaul.htm))这是一个简单的钛金属盒子，周围有 23 块石头。然而，让这个戒指如此酷的是，每当[本]和他的女朋友手牵手时，它就会发光。

建造的金属加工部分就像你所期望的加工钛一样简单。戒指从棒料上切下后，[本]把它带到一个磨坊，在那里为每颗宝石钻 23 个孔。宝石用珠宝环氧树脂固定在戒指上，整个戒指被打磨得闪闪发光。

电子设备是这个项目真正的亮点。将一个电容器电池放入一个环中几乎是不可能的，所以[Ben]决定用一个感应充电电路给 led 供电。缠绕在 kapton 胶带上的线圈充当电感，一个小型 SMD 电容为三个非常亮非常小的 led 供电。

感应充电单元本身就是黑客的杰作；[本]想让戒指在他和他的女朋友牵手的时候发光。为了做到这一点，[本]的感应充电器也是一种可穿戴设备:一个大线圈是充电器的变压器，可以戴在[本]的手腕上。整个充电电路可以很容易地隐藏在夹克袖子下，形成一个近乎神奇的发光戒指。

一件了不起的作品，也是我们很久以来见过的最好的珠宝作品之一。你可以在下面的视频中看到电感耦合和发光二极管。
