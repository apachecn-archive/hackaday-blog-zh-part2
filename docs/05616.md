# 这台机器吸球

> 原文：<https://hackaday.com/2014/03/23/this-machine-sucks-balls/>

除了赢得超级碗的最富有的宇航员之外，任何人都可以做出的最佳职业选择是在科学中心发现的动力学艺术装置的设计师，这些装置在物理学和力学的永恒展示中沿着轨道、绕圈和通过传送带滚动台球。[Niklas Roy]还没有达到那个水平，[但是他在一个旧想法的基础上提出了一个新的想法](http://www.niklasroy.com/project/170/suck-the-balls):一个从球坑中吸取球到透明管中的机器，让它们在安装空间周围呼啸而过。

这个装置由悬挂在克拉科夫波托基宫楼梯上的八十米长的塑料管组成。只要有人在球坑里，PIR 传感器就会打开真空吸尘器。它通过软管将球吸上来，绕过空间，进入坑上方悬挂的垃圾箱。拉一个杠杆，储存在箱子里的球就会被分配到下面用吸尘器吸起成千上万个球的人身上。
