# 信息时代之前升级列车信号

> 原文:[https://hack aday . com/2013/11/12/retro tech tacular-upgrading-train-signaling-before-the-information-age/](https://hackaday.com/2013/11/12/retrotechtacular-upgrading-train-signaling-before-the-information-age/)

![retrotechtacular-british-railway-signaling](../Images/ad53ec7995bdb4ba1fd4caf0dbce1048.png)

本周回顾的主题令人惊讶的是，这个主题并不是很久以前的。但是，看看这项工作的完成方式，就会觉得它已经过去很久了。1974 年，英国铁路委员会着手对信号系统进行现代化改造和互联。你在上面看到的是数百个老旧的信号控制室中的一个，它们将被一个互联系统所取代。

如今，我们认为这种事情是理所当然的。但是从项目一开始，很明显当时可用的技术限制了开发过程的效率。我们不是在讨论视频制造部分中显示的所有机电部件。不，从一开始，大量的大型纸质原理图和逻辑图就让人望而生畏。房间里满是工程师，堆着装订好的规划文件，这让我们感到陌生，因为如今，甚至连打印登机牌都显得过时了。

有了像这样精彩的半小时视频，谁还需要电视呢？我们建议将此添加到您的观察列表中，以便您可以适当地享受它。他们炫耀一切。制造电缆、在信号塔之间架设电缆、组装控制面板、测试等等。
