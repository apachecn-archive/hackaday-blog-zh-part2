# 阿特拉斯在毁灭之球中幸存

> 原文：<https://hackaday.com/2013/10/08/atlas-survives-the-wrecking-ball/>

[波士顿动力]在过去的几天里充满了视频。除了我们已经推出的的[野猫更新，他们已经放弃了 Atlas](http://hackaday.com/2013/10/05/boston-dynamics-takes-wildcat-outside/) 和 LS3 的更新。

这个视频展示了 Atlas 机器人的更新。与 DARPA 机器人挑战赛中发送给竞争对手的机器人相比，这将是 atlas 的简化版本。手臂已经换成了重物。看来这次更新的重点是 Atlas 在崎岖地形上的平衡和操控。阿特拉斯小心翼翼地走过一些压碎的煤渣砖—[可能就是我们最近看到大狗扔来扔去的那种](http://hackaday.com/2013/03/29/bigdog-throwing-cinder-blocks/)。有几个紧张的时刻，但阿特拉斯设法通过毫发无损。

(对我们来说)真正可怕的是看着阿特拉斯被一个加重的球击中。我们假设球上的 20 表示磅。想象一下被一个 20 磅的重物从侧面击中。你能站起来吗？我们有没有提到阿特拉斯是用一只脚站立的？然而，阿特拉斯却泰然自若——以非常人性化的方式挥动手臂来保持平衡。值得注意的是平衡系统。阿特拉斯的脚从未离开地面。这与我们已经了解并喜爱的[波士顿动力公司]其他机器人的弹跳/跳跃平衡系统略有不同。
