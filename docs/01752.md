# 数控提花织机什么都会织

19 世纪早期发明的提花织机使用穿孔卡片来制造相对复杂的纺织品，如锦缎和缎。这些穿孔卡最终被人口普查人员、机械计算器、早期模拟计算机、最早的数字计算机，甚至是[不起眼的 Arduino](http://hackaday.com/2012/07/30/reading-punch-cards-with-an-arduino-and-digital-camera/) 所使用。

然而，这并不意味着提花织机留在了 17 世纪。这款产品上周在纽约举行的开放硬件峰会上亮相，它太酷了，创客节的组织者很大方地为它找了个地方。

整个织布机由电脑控制，不需要打孔卡，由廉价的铝挤压材料制成。它还可以将任何两色图形制作成纺织品(是的，甚至是 Hackaday 的标志)。在 Maker Faire 的一天里，织布机并没有完全投入使用，但我们已经得到承诺，将来会更新。
