# [本·克拉斯诺]造了一台 CT 扫描仪

> 原文:[https://hack aday . com/2013/01/09/Ben-krasnow-builds-a-CT-scanner/](https://hackaday.com/2013/01/09/ben-krasnow-builds-a-ct-scanner/)

![gif](../Images/ebcdf2476054989431516c0ce65c8053.png)

在建造了自制的 x 射线反向散射成像仪后，[Ben Krasnow]意识到他几乎拥有了建造自己的 CT 扫描仪的所有组件，能够制作冷冻鸡肉内部的 3D 模型。

基本上，CT 扫描仪获取一个物体的数十张 x 光片，并在复杂算法的帮助下将它们重新组合，以便医生能够窥视人体内部。你会在当地医院找到的 CT 扫描仪是巨大的设备，在一些重型机电工程的帮助下，围绕病人旋转 x 射线管和传感器。[Ben]想要保持他的体型相当小，所以他没有围绕一个物体旋转 x 射线管和屏幕，而是简单地制作了一个步进电机驱动的 lazy suzan 来旋转他的冷冻鸟。

[Ben]将一台数码相机放在他的建筑旁边，拍下了 45 张旋转鸡的照片。在校正透视失真后，这些图像被扔进 [3D 切片器](http://www.slicer.org/)中，以创建一个经过 x 光检查的鸡的真实 3D 表示。
