# 当你不吃药的时候，自动售药机会向互联网告密

> 原文:[https://hack aday . com/2013/08/09/pill-dispenser-tattles-to-the-internet-when you-don-take-you-pills/](https://hackaday.com/2013/08/09/pill-dispenser-tattles-to-the-internet-when-you-dont-take-your-pills/)

![pill-taking-reminder-hardware](../Images/688a74c08d8fe75afa0225dd1dc72959.png)

这是 2013 年视差微医疗竞赛的另一个参赛作品。[Tim]称之为他的[基于推进器的互联网记录药丸分配器](http://pcig.info/uploads/Reviews/microMedicPILBoxSubmission2.pdf) (PDF 文件)，简称 P.I.L. Box。

硬件是一个基本单元，普通塑料药丸组织器放在其中。我们喜欢这种设计，因为许多药店会为你填充药盒，这不会使过程变得复杂。你只需在一周开始时拿出正确的盒子，放入基本单元中。您可以看到盒子中的星期一插槽上有一个白色 LED 在发光。该灯在设定的服药时间前一小时开始亮起。如果你走过这条路，它会提醒你。还有一个专门用来吓唬老人的声音从扬声器里出来，还有一个简单的信息在七段显示器上拼写出来。当您按下按钮打开药盒时，基本单元会进行检测，并将其计算为正确的给药剂量。

如果你忘了吃药，那可不是什么好事。运行在笔记本电脑上的服务器会出卖你。它使用 Twitter API 来提醒任何关注它的人——这意味着一个亲戚或护理人员——错过了一个剂量。让我们希望他们善于跟上他们的 Twitter 饲料！
