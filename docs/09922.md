# Ubuntu 核心支持树莓 Pi 2 I/O

> 原文:[https://hack aday . com/2015/08/27/Ubuntu-core-supports-raspberry-pi-2-io/](https://hackaday.com/2015/08/27/ubuntu-core-supports-raspberry-pi-2-io/)

虽然不是官方的，Ubuntu Core——Ubuntu 的小型物联网版本—[现在运行在 Raspberry Pi 2](https://developer.ubuntu.com/en/snappy/start/#snappy-raspi2) 上。如果你愿意的话，还有预构建的二进制文件以及如何运行自己的二进制文件的说明。你甚至可以访问 GPIO

Ubuntu Core 抛弃了旧式的 Debian 软件包，转而支持 Snap，这是 Ubuntu 手机点击软件包管理器的新版本。Snap 提供事务性更新。这个想法是，物联网上的所有这些“东西”都需要更新，以修补安全漏洞或解决其他问题。

正如你可能从一个非官方的版本中期待的，有一些粗糙的边缘。然而，正如你在下面的视频中看到的，这是可行的，在所有关于同一块板上 [Windows 10 的炒作中，尽早推出这个版本是一个好的举措。希望最终版本不会太远。Pi 2 当然足够强大，](http://hackaday.com/2015/08/13/raspberry-pi-and-windows-10-iot-core-a-huge-letdown)[它的规格足以让几年前的普通台式机相形见绌](http://hackaday.com/2015/02/02/introducing-the-raspberry-pi-2/)。
