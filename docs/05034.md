# 我是午夜留言板午夜有什么消息

> 原文:[https://hackaday . com/2014/01/11/I-am-the-midnight-message-board-what-messages-in-midnight/](https://hackaday.com/2014/01/11/i-am-the-midnight-message-board-what-messages-at-midnight/)

![](../Images/a462c9b8b849d8b99ee6e0f7823c7ad1.png)

卧室墙上或天花板上的光致发光星星非常酷，尽管静止的形状会变得很无聊。[Adi]也有这种感觉。当他在黑暗的乙烯基上用明亮的白光涂鸦时，他想到这可以成为一个有趣的展示。他着手制作 [GLO、午夜留言板和 RSS 显示屏](http://www.instructables.com/id/GLO-The-Midnight-Message-Board)。

[Adi]的 light writer 在一个由步进电机驱动的线性轴上使用 12 个 UV LEDs，在墙上书写 RSS 标题、Twitter 趋势或自定义文本。他发现文本的缓慢淡入淡出非常有助于入睡，原因显而易见。LED 阵列打印由 6×5 位模式组成的字符的一部分。12 个发光二极管被分成两组，所以它可以写两行，每行 45-50 个字符。[Adi]为此项目设计了自己的像素字体，并建议只使用大写字母形式。

【Adi】写的挺让人佩服的，很全面。在电路构建部分，他建议 led 必须非常靠近乙烯基以获得最佳效果，但它们应该比移位寄存器突出得更远，这样芯片就不会摩擦乙烯基。当然，你可以选择更强的光源，比如激光。休息之后看看它的实际效果。
