# WISP 为物联网增加了 Wifi

> 原文:[https://hack aday . com/2012/06/08/wisp-adds-wifi-to-the-internet-of-things/](https://hackaday.com/2012/06/08/wisp-adds-wifi-to-the-internet-of-things/)

![](../Images/50d5f57b88c26e5a44bb7ed99b14a129.png "wisp")

embdSocial 的人送来了一个他们已经做了一段时间的项目。这是一个用于 Arduino 或其他名为 Wisp 的微控制器的小型 wifi 模块。与我们见过的许多其他 wifi 分线板不同，Wisp 具有不可思议的潜力。embedSocial 团队开发了一个 API，允许 Arduino 向 Twitter 发布消息、发送文本消息，甚至具有远程管理功能，他们开发了一些非常酷的东西。

我们已经看到了很多使用 wifi 的项目，但是 Wisp 在如何控制项目方面非常聪明。每个 Wisp 都是通过互联网管理的。一旦 Wisp 注册到你的在线 embdSocial 帐户，你就可以上传新代码，而无需将微控制器物理连接到你的计算机上。

为了展示 Wisp 的远程管理能力，embdSocial 的人将 Arduino 和 Wisp [放在一个电气接线盒](http://www.embdsocial.net/projects/outlet/)内。通过他们的设置，这些人拥有了我们见过的最简单和最小的互联网连接电源插座。

休息之后，你可以看到一个 Wisp 打开车库门和一个远程操作的网络气枪炮塔的演示。我们喜欢炮塔将视频从枪发送到互联网上的任何设备，令人印象深刻的是[克里斯]和[艺术]在一个周末就完成了这两个项目。Wisp 也有一个 [Kickstarter](http://www.kickstarter.com/projects/1762774051/embdsocial-wisp-embedded-social-media-gateway) ,所以希望我们能很快找到其中的一个。
