# 机器人灯太可爱了！

> 原文：<https://hackaday.com/2015/05/12/robo-lamp-is-too-damn-cute/>

还记得皮克斯灯吗？嗯，来自[粘液质原型]的[Rick]决定自己做一个。

这实际上是该项目的第三次迭代，[第一次](http://phlegmatic-prototyping.tumblr.com/post/48230396351/i-was-playing-with-a-laser-cutter-lately-and-this)是由一个廉价的摄像头传感器、一个 ATMega328、一个电脑鼠标和激光切割部件制成的。第二次迭代……听起来像是皮克斯的灯——由于可能侵犯版权,[Rick]决定不展示它。这就引出了第三个问题，第三个是 Robolamp。

很明显**不是**现在的皮克斯灯，[Rick]用 3D CAD 设计了整个东西，并在 Shapeways 打印出来。它使用一个 CMUCAM5 视觉传感器来检测颜色色调，使灯能够跟踪彩色物体，如橙子，最终看起来非常可爱。看看视频就知道了。

[https://www.youtube.com/embed/grMq7Hd4aG8?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/grMq7Hd4aG8?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

他甚至从头开始设计印刷电路板，并用非常便宜的脏电路板制作。这意味着整个项目是非常可复制的——也许如果我们很好地请求，他会发布源文件？我们可以在办公室里使用~~皮克斯~~机器人！
