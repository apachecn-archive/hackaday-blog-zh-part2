# 在你测量之前想一想——旧的测试设备以及它为什么如此棒

> 原文:[https://hacka day . com/2014/10/20/在你测量之前想一想旧的测试设备和它为什么如此棒/](https://hackaday.com/2014/10/20/think-before-you-measure-old-test-gear-and-why-it-is-awesome/)

好的、可行的测试设备是实现我们爱好的关键。在本帖中，我们将讨论在哪里以最低的价格获得它，寻找什么，如何修复它，以及如何围绕可能不可靠的设备定制您的实验室实践。

我们很幸运生活在一个拥有大量高质量测试设备的时代。自第二次世界大战以来，过剩的测试设备以低廉的价格大量出现，使得爱好者、创新者和学者能够进行实验和建造伟大的东西。如果你愿意在测量前思考，你可以省下一大笔钱，并且在家里有一个专业的实验室。

在哪里买
显而易见的答案是易贝，但测试设备的交易却是最划算的。不要被名字迷惑了。Hamfests 卖的远不止业余无线电设备。Hamfests 是业余爱好者交易各种电子产品的交易会。查看 [ARRL 狂欢节日历](http://www.arrl.org/hamfests-and-conventions-calendar)，找到离你最近的下一个当地节日！不过，我建议你早点到。hamfests 的文化倾向于门一开就出现，并在正式结束前大约两个小时离开。早起的鸟儿有虫吃！

在你找到你所需要的一切之前，你将不得不去参加许多狂欢节。耐心是关键。只要坚持去参加 hamfests，最终你会找到它或类似的东西。如果你现在需要它，你可以在易贝上花更多的钱来快速装备自己。

当然，还有所有 hamfests 之母，代顿 Hamvention。正如我多次听到的，如果你在代顿找不到它，你就不需要它。

**寻找什么——品牌&质量时代**
如果你正在寻找一个范围，那么买一个 Tektronics 品牌，任何时代。有些‘电视修理工’的瞄准镜也不错。我的 Heath-Zenith 从 80 年代末就开始有好运了。如果示波器不是 Tek 或 HP，那么根据清洁度来判断设备。如果非常干净，那就值得一试；如果脏，那么买两个和一个可能会工作！

对于其他任何东西，只要有可能就买惠普。根据我的经验，50 年代和 60 年代的电子管 HP 信号发生器非常可靠。不足为奇的是，当你打开一个盖子时，它们似乎是按照非常高的规格制造的。70 年代和 80 年代的一些更复杂的齿轮可能很难维修，需要很难找到的卡扩展器等。我只买干净的装备，而且只要有可能，我都会买两套。

**修理还是更换？**
随着时间的推移，你的工作测试装置会出故障。你要么换一个，要么自己修。这只是一种时间成本的交易。花在维护设备上的时间越多，购买和维持实验室运行的费用就越少。在研究生院的时候，我的政策是花不超过 3 个小时修理设备，如果我超过这个时间没有任何实质性的进展，我会更换它。我目前的方法是在 hamfests 上购买二手设备，并运行它直到它报废。如果它坏了，我用另一个旧齿轮代替它。为了弥补更换设备的时间差距，我所有的设备都有重叠功能，所以我永远不会没有合适的工具来工作。

有[组](https://groups.yahoo.com/neo/groups/TekScopes/info)致力于 Tektronics 示波器的修理和恢复，可能还有其他 HP gear 的修理和恢复。

对于一些电子管设备，如希斯基特和其他低端电视和收音机维修设备，您可以使用之前讨论过的[古董收音机维修程序](http://hackaday.com/2014/09/09/welcome-to-the-old-school-restoring-antique-radios/),使旧希斯基特和其他旧设备在未来几十年内非常容易维修。

**三思而后行——使用旧设备的一些准则**
不要相信任何东西！当使用任何类型的旧测试装置时，您必须定制您的测量程序，以便始终进行验证。当您的电路不按预期运行时，您应该做的第一件事是验证电源(所有电压和接地)，验证电缆，验证测试设备的输出，并验证输入在您的测试设备上工作。不要让你的 hamfest 示波器上一个松动的 BNC 输入连接器或一根坏的测试电缆欺骗你，让你以为你的放大器坏了。这是扩展到您的测量设置的基本故障排除技能(许多人可能已经直观地做到了)。

**举例——微波实验室**
微波测试设备并不便宜。从高中开始，我开始采购测试设备，一开始什么都有，随着时间的推移，慢慢升级到更好、更可靠的设备。这个实验室开始生产一些严肃的项目，并在我的博士论文中达到顶峰，其中包括两个合成孔径雷达轨道成像系统和一个近场相控阵。这个实验室继续支持我作为业余爱好者和专业人士的所有工作。如你所见，它配备了 50 年代、60 年代、70 年代的装备，最新的装备是 80 年代的。

**价格，基于我之前购买和观察的内容**

它越重就越便宜，例如，工作状态下的电子管射频信号发生器通常售价不到 100 美元。

好的示波器(Tek 品牌，50-100 MHz 带宽 2 个或更多通道)价格在 50-200 美元之间。带宽越高，示波器就能卖得越多，尽管我曾经花 100 美元买了一台带宽为 2.5 GHz 的旧示波器。

函数生成器可以卖到 20-50 美元。

好的频谱分析仪，如基于 HP 141T 主机的，在工作状态下运行约 400-1200 美元。

惠普微波信号发生器售价 300 美元以上。

板凳用品，如功率较低的惠普哈里森型，每台 20-50 美元。

好的数字电压表，如福禄克或辛普森 260 的，20-50 美元。

总结
我们生活在一个充斥着旧测试工具的世界，我们所要做的就是在测量前思考。处理越来越复杂的项目。不要等到你可以去拜访朋友或者去工作的时候才去衡量一些事情。相反，要有内部能力。参观你当地的 hamfest，今天就在你的地下室或车库里建立一个专业的实验室！
