# 把寻找复活节彩蛋变成猎狐

> 原文:[https://hack aday . com/2013/04/04/将复活节彩蛋狩猎变成狐狸狩猎/](https://hackaday.com/2013/04/04/turning-an-easter-egg-hunt-into-a-fox-hunt/)

![fox](../Images/3ad52b59f9abfc6efcc0013ef6d79f36.png)

我们以前看过几次[托德·哈里森]的作品，但他以前从未让他的儿子如此彻底地参与进来。去年复活节，他认为让他的儿子和几个朋友参加寻找复活节彩蛋的活动是个好主意。作为一个业余爱好者，他决定将寻找复活节彩蛋变成一场无线电测向的冒险，或者业余无线电操作员称之为猎狐。

[Todd]用几根 PVC 管和几根铝棒和黄铜棒制作了一个很棒的八木(一种简单的定向天线)教程。有了这个和一套手持火腿，托德在当地公园附近藏了一只狐狸、一只复活节兔子和一篮糖果。在父亲的指导下，[托德]的儿子和他的朋友们最终找到了狐狸。将糖果放在亚利桑那州的阳光下可能不是[托德]的最佳主意——狐狸和糖果被发现时浑身都是蚂蚁——但这是对业余无线电的一个很好的介绍。
