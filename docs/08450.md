# 介绍焊钳 9000

> 原文：<https://hackaday.com/2015/03/03/introducing-the-solder-sucker-9000/>

使用一个常规的柱塞式焊锡膏是乏味的，而且通常不是很有效。如果您试图从 PCB 上回收元件，有时可能会花费比值得做的事情更长的时间——不包括整个电路板的回流！但是，如果你有某种东西可以更快地解除单个组件的焊接，那会怎么样呢？

在受够了他廉价的柱塞式焊钳后，[electro1622]决定尝试一种不同的策略。他一直在重复使用旧电路板上的元件，所以他尝试了一些非正统的方法来移除它们。压缩空气。

现在让我们以**开始，这会很混乱**，所以你可能想设置一个盒子来收集移除的焊料。简单地用你选择的熨斗加热固定你的元件的焊料滴，然后用压缩空气从小喷嘴中喷出。比焊钳快多了。
