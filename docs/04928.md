# 巨型树游戏显示器

![images-mediaFiles-holiday-MegaTreeGame-IMG_4914-640x480](../Images/e98400e8cb2d63d39eeb1d4ce029f11f.png)

圣诞节可能已经过去，但假日黑客仍在不断涌入，就像这个令人敬畏的互动[巨型树游戏展示！](http://www.elazary.com/index.php?option=com_content&view=article&id=57:megatreegame&catid=18:megatree&Itemid=18)

[Lior]喜欢每年举办灯光秀(看看去年令人惊叹的[白色圣诞音乐秀吧！但是把它们拿下来只是…嗯…糟透了！因此，今年他决定制作一个完全可控的非节日专用灯光显示器，可以全年重复使用。](http://www.youtube.com/watch?v=TXnHjVarAHI)

它的特点是 12 x 5 米长的防水 RGB LED 灯条牢固地固定在他房子的前面，形成一个梯形的 12 x 150 像素显示器。它最初是由 Arduino 控制的，但他发现 USB 连接对于他追求的高帧速率来说太慢了——所以他使用了树莓 Pi 和 Teensy 2.0 的组合。

现在，我们必须承认，仅仅是制作一个适合所有假期的可编程灯光秀就很酷，但正如[Lior]所说，一个普通的灯光秀“太过时了”。所以他把整个事情变成了智能手机互动。是的，你实际上可以用你的手机登录，玩一个愚蠢的游戏，向房子和雪人扔礼物。他还有一个非常酷的[光明节展示](http://www.youtube.com/watch?feature=player_embedded&v=oVaJLOykv9E)，上面有一个旋转的陀螺！休息之后，请观看完整的演示视频。
