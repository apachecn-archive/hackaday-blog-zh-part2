# 蒸汽朋克大师的爱德华七世时代笔记本电脑

> 原文：<https://hackaday.com/2012/05/30/edwardian-laptop-from-a-steampunk-master/>

![](img/d091a260ed836f3d9310db8381af7339.png "laptop")

在收到来自阿龙·拉文斯代尔上将的马可尼通知我们一款精致的蒸汽朋克笔记本电脑已经完成后，我们非常高兴。[海军上将]的计算设备，或 Uhlian 计算器，作为首选术语，是设计和功能的非凡结合，适合任何致力于控制快速接近的电气前沿的非凡绅士。

[Ravensdale]的新蒸汽朋克笔记本电脑是基于他的第一台笔记本电脑，一台旧的东芝 Satellite 1100。不是一个速度恶魔，但这个版本的质量是惊人的。当笔记本电脑打开时，铰接键盘向上倾斜到符合人体工程学的位置，露出一组六个 LED 宝石，分别用于电源、电池和硬盘驱动器灯。在屏幕的左侧和右侧，一对微型铜喇叭包含一组立体声扬声器。

这个键盘是对普通键盘的一个令人敬畏的修改，非常让人想起【Ravensdale 上将】以前的[键盘蒸汽发音](http://hackaday.com/2011/10/04/very-impressive-steampunk-keyboard/)。

[海军上将]展示了一个可指示的来检查他为这件精美的工艺品投入的许多小时。还有一个视频显示键盘升降机构和万能钥匙电源开关在休息后可用。
