# Hackaday 奖参赛作品:一个电子果汁机器人

> 原文：<https://hackaday.com/2015/07/01/hackaday-prize-entry-an-e-juice-robot/>

电子烟越来越受欢迎，世界各地的城镇都出现了怪异的潮人店。虽然你可以在加油站和任何出售模拟香烟的地方买到这种电子果汁，但不可避免地会有人想要混合他们自己的丙二醇、甘油、水和尼古丁。对他们来说，[康克林尼克]正在建造一个邪恶产业的终结，一台自动化整个过程的电子果汁打印机。

这种“电子果汁打印机”旨在混合电子烟消耗品的基本成分。这些成分是丙二醇和/或甘油、水、调味剂和尼古丁。[康克林尼克]的项目是使用不同的“工作站”和一个摄像头滑块将这些成分分装到一个小瓶中。它实际上是一个为橡皮泥而不是酒精分配成分的机器人。

这是一个伟大的项目，虽然它不适合每个人——也不应该适合每个人——但它是我们已经拥有的自制技术的一个伟大应用。
