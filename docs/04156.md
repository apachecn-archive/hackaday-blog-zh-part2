# 这个控制项目的第一次竞赛，有奖品

> 原文:[https://hack aday . com/2013/09/26/the controller projects-first-contest-with awards/](https://hackaday.com/2013/09/26/thecontrollerprojects-first-contest-with-prizes/)

![contest-prizes](../Images/72de026264dd1c4be1a6f0b5670b897f.png)

在[Caleb]离开我们之前，他的副业之一是控制器项目，这是一个控制器和控制台改装者与残疾玩家交流的地方。那边一定很热闹，因为[【Caleb】刚刚宣布了他的第一场比赛](http://thecontrollerproject.com/find-solution-win-tools/)，甚至还有奖品。

这场比赛的目标是让 XBox 和 PS3 控制器上的触发按钮能够从控制器的顶部进行控制。这对于有残疾的游戏玩家来说是一个巨大的问题，并且目前没有开放的系统来解决这个问题。如果你能制造出某种机械装置，把肩上的按钮变成顶部安装的致动器，只要把它放在某个地方，就能获奖。

奖品是一个 iFixit 工具包和一个 T2 磁垫。前五名送出肩挂式按钮问题解决方案的人将获得此奖项。最初，[Caleb]想过拆开这些控制器并焊接额外的按钮，但卡扣式机械解决方案更容易安装。

如果你设计了这个问题的解决方案，[在](http://hackaday.com/contact-hack-a-day/)发送(但是[先发送给【迦勒】！](http://thecontrollerproject.com/contact/))我们可能也会在节目中提及。
