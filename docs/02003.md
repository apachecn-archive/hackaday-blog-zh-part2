# Nyan 猫帽将会是派对的焦点。大约一分钟

> 原文:[https://hack aday . com/2012/10/30/nyan-cat-hat-will-the-hit-the-party-on-the-party 约一分钟/](https://hackaday.com/2012/10/30/nyan-cat-hat-will-the-the-hit-of-the-party-for-about-one-minute/)

![](../Images/61b6df1b54b59d43be29515de1fa13b4.png "motorized-nyancat-hat")

说真的，[这顶 Nyan 猫帽](http://build-its.blogspot.com/2012/10/nyan-hat.html)是一套梦幻万圣节服装唯一需要的部分。它在这个静态图像中看起来很好，但是我们敢让你看它在行动中的剪辑(在休息后嵌入),而不要让一个灿烂的笑容爬上你的脸。每个人都会喜欢这个…直到它开始变得很烦人。

当开关打开时，标志性的迷因绕着头饰旋转，当她唱着她的人民的歌时摆动着它的头和尾巴。也有由白色发光二极管制成的星星，在这个过程中非常明亮地闪烁。演示很好，但更好的是看到了构建过程。幸运的是[Ben Katz]在冒险期间发布了一系列详细的文章。负责驱动猫绕着帽子转的机械装置是由丙烯酸树脂激光切割而成的。它包括一个内部带齿的大齿轮，通过一个连续旋转的伺服电机与一个小得多的齿轮连接。头部和尾部的摆动完全是机械的，当猫绕着帽沿行走时，利用旋转运动来转动纺锤。
