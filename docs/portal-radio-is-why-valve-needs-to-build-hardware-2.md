# 门户电台是为什么阀门需要建立硬件

> 原文：<https://hackaday.com/2012/04/16/portal-radio-is-why-valve-needs-to-build-hardware-2/>

[![](img/961d3352c5ae80de8bda1c418145f6c9.png "radio")](http://hackaday.com/wp-content/uploads/2012/04/radio1.png)

我们已经看到了传送门炮的建造，一些可爱的炮塔复制品，甚至是缩小版的 GLaDOS，但是[约翰]的[传送门无线电复制品](http://www.youtube.com/watch?v=YxScP6XKNiU)是这个经常被遗忘的传送门物品的第一个实体版本。

有趣的是，整个收音机是由废料制成的。球体外壳由商用冰箱中的泡沫绝缘材料制成，经过 300 小时的精心雕刻、粘合和喷漆。无线电内脏来自一个升级的无线电，由内部电池或壁式 DC 适配器供电——非常适合携带门炮在试验箱中走动。

现在，收音机里有一个 AM/FM 接收器，还有一个音频输入，这样就可以插入 iPod 之类的东西。虽然我们希望看到一个循环的 [uptempo 版本仍然活着](http://www.youtube.com/watch?v=mD3v1B_aXw0)，但我们猜测【约翰】还没有找到一种简单的方法来处理废弃的零件。
