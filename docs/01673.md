# 从电动订书机中拉出零件的线圈枪

> 原文:[https://hacka day . com/2012/09/21/从电动订书机中取出零件的线圈枪/](https://hackaday.com/2012/09/21/coil-gun-with-parts-pulled-from-an-electric-stapler/)

![](../Images/fd199da9e24c6334dfc633a32099fe02.png "coilgun-from-electric-stapler")

【楼】最新教程详细介绍了[把电动订书机变成线圈枪](http://www.youtube.com/watch?v=ErqIn3BMCuk)的过程。订书机是最贵的部分，但其余部分很简单。他使用了 PVC 管和一些配件，以及一些你可能会在商店里看到的用品。

令人惊讶的是，Bostitch 订书机(零件就是从这款订书机中取出的)对于这个项目来说是如此的完美。将订书钉钉入书页的机械装置使用一个线圈相当大的螺线管。要把它变成一个线圈枪，你只需要用一个金属弹丸替换螺线管的核心。在休息之后的视频中,[Lou]向我们展示了如何制作一个可以安装线圈的桶。在那里，他用一个木制的垫片将一大块光滑的金属放在一个充当抛射物的螺栓上。订书机的原始驱动电路和触发机制完成剩下的工作。
