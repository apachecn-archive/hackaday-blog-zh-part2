# 最大的数控刳刨机是手动控制的

> 原文：<https://hackaday.com/2012/08/09/largest-cnc-router-is-controlled-by-hand/>

今年的 SIGGRAPH 是对麻省理工学院[Alec]、[Ilan]和[Frédo]基于传统 X Y 工作台的数控机床的一个非常有趣的展示。他们创造了一个由计算机控制的 CNC 路由器，理论上尺寸不受限制。这种刳刨机不使用龙门架，而是使用人工在工件上移动刀具，并且仅在相机和步进电机的帮助下对刀具路径进行精细修正。

整个设备是围绕一个手持式刳刨机构建的，其底座包含一个摄像头、电子设备、步进电机和一个非常漂亮的屏幕，用于显示当前刀具路径。在几条受二维码启发的胶带之后，相机向下看着工件，并计算路由器为了做出正确的形状而必须做出的微小变化。用户需要做的只是沿着要切割的零件的轮廓引导刳刨机，误差幅度为半英寸。
