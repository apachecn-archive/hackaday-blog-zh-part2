# 推出 F *手表，一款完全开放的电子表

> 原文：<https://hackaday.com/2014/10/17/introducing-the-fwatch-a-fully-open-electronic-watch/>

当他们的一位同事退休时，几名欧洲核子研究中心的工程师在 4 个月的工作时间后聚在一起开发他的礼物:一只完全打开的电子表。它被称为 F *手表，装有传感器:GPS，气压计，指南针，加速度计和光传感器。所用的微控制器是 32 位 ARM Cortex-M3 SiLabs Giant Gecko，包含 128 kb RAM 和 1MB 闪存。在上图中，你会注意到一个 1.28 英寸 128×128 像素的夏普内存 LCD，但主板还包含一个用于电池充电和连接的微型 USB 连接器，一个微型 SD 卡插槽，一个蜂鸣器和一个振动电机。

这款手表由 500 毫安的 LiPo 电池供电。所有用于构建它的工具都是开源的(FreeCAD、KiCad、GCC、openOCD、GDB ),我们的读者可以通过下载位于[他们的资源库中的所有源文件来制作一个](http://www.ohwr.org/projects/f-watch/repository)。休息之后是一段展示他们冒险经历的视频。
