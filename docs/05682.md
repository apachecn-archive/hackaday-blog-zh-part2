# 冰冻圆周率——一个负担得起的子弹时间记录器

> 原文：<https://hackaday.com/2014/03/31/frozen-pi-an-affordable-bullet-time-recorder/>

把 48 个树莓派摄像头用近半公里的网线绑在一起会怎么样？你得到了自己的[子弹时间捕捉装备。](http://pifacedigital.wordpress.com/2013/12/06/bullet-time-effect-frozen-pi/)

最初是受《黑客帝国》独特的电影效果和一部名为《超自然:动物看不见的力量》的 BBC 老纪录片的启发， [PiFace](http://www.piface.org.uk/) 的所有者决定自己尝试重新创造子弹时间效果。

为了创建这个装置，他们用了 48 个 Raspberry Pis，每个都有一个 PiFace 控制板和标准摄像头。控制板允许在没有键盘或鼠标的情况下使用 Raspberry Pi，因此所有的网络电缆都必须向每个 Pi 发送一个简单的代码，以便拍照。一个简单的激光切割的木头轮廓被用来把它们连接成一个巨大的圆环。

虽然 48 Raspberry Pis 很多，但他们认为这是一个适合课堂环境的合理项目——此外，去学校拍摄自己的子弹时间特技会有多酷？
