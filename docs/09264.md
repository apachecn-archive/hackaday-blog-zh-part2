# Hackaday 奖参赛作品:Flex 模块

> 原文:[https://hack aday . com/2015/06/11/hack aday-prize-entry-flex-modules/](https://hackaday.com/2015/06/11/hackaday-prize-entry-flex-modules/)

(克里斯·汉密尔顿的)黑客日奖的一个参赛作品直接涉及他的工作。他在 [Fyber Labs](https://www.fyberlabs.com/page/homepage) 工作，设计可穿戴的柔性电子产品。虽然任何人都可以出去买一些柔性传感器，每个大型板房都可以制造柔性 PCB，但没有多少人制造柔性产品，更没有人创造工具来制造这些可穿戴电子产品。为了解决这个问题，[【克里斯】正在制造柔性模块](https://hackaday.io/project/2236-flex-modules)，这种电路板将兼容试验板的模块的易用性与可以放置在柔性 PCB 上的东西结合在一起。

这是[Chris]的一个工具包，他已经有了大量已经完成或正在开发的模块。[Flex Sensor ADC 缓冲器和滤波器](https://hackaday.io/project/2240-flex-sensor-adc-buffer-flex-module)旨在读取柔性传感器， [STM32F401 模块](https://hackaday.io/project/2242-stm32f401-flex-module)在这些项目中放置了一个极其强大的微控制器， [12axis 模块](https://hackaday.io/project/4305-12axis-flex-module)为这些项目提供压力、湿度、陀螺仪和温度传感器。[Chris]正在开发二十多个模块，每个模块都与他的柔性电子系统一起工作。

如果你想看看这些模块能做什么的例子，看看[舞蹈工具包](https://hackaday.io/project/3083-dance-kit)【克里斯】制作的。这是一种可佩戴的 LED 灯条，具有运动反馈和生物电监测功能。如果没有灵活性，这个项目将会是一个庞大而笨拙的电路板。有了这些模块，就很容易为这个问题创建一个可穿戴的解决方案。
