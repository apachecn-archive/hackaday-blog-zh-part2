# 使用 Oculus Rift 作为多显示器替代品

> 原文:[https://hack aday . com/2014/12/11/using-the-oculus-rift-as-a-a-multi-monitor-replacement/](https://hackaday.com/2014/12/11/using-the-oculus-rift-as-a-multi-monitor-replacement/)

[Jason]最近一直在玩 Oculus Rift，并想出了一个非常酷的软件演示。这可能以前已经以某种方式或形式完成了，但无论如何我们喜欢这个想法，他实际上已经发布了这个程序，所以你也可以玩它！

它基本上是一个 3D 窗口管理器，被恰当地称为 3D WM——尽管他正在考虑把名字改成一个更酷的名字，也许是**工作空间**或其他什么。

正如他在下面的视频演示中所示，该软件允许你在 Oculus 创建的虚拟球体上设置多个桌面和窗口——本质上是创建一个虚拟的多显示器设置。这个设置有几个明显的缺点，这使得它目前有点不切实际。比如看不到你的键盘(虽然这不应该是一个问题)，看不到你周围的人…当然还有硬件和它缺乏适当的分辨率。但除此之外，还是蛮牛逼的！

在未来的开发中，他希望添加 Kinect 2 和 Leap Motion 控制器的集成，以帮助使其更加直观——也许更多的少数报告风格。
