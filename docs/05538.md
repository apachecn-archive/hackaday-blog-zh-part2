# 开源哨兵枪计划承诺办公室战争的下一个级别

> 原文：<https://hackaday.com/2014/03/14/open-source-sentry-gun-plans-promise-the-next-level-of-office-warfare/>

我们承认，当我们浏览 Reddit 时，我们发现了这个漂亮的拼凑在一起的 Nerf 哨兵炮塔。但我们是否曾经庆幸自己做到了——事实证明，它非常类似于非常棒的开源[项目哨兵枪。](https://sites.google.com/site/projectsentrygun/)

我们实际上已经覆盖了一个在之前使用该系统的[项目，但是看起来从那以后它已经发展得更好了。不分享实在是太酷了。](http://hackaday.com/2013/07/03/sentrifying-a-nerf-gun/)

该系统本身非常简单，易于构建。你需要三个伺服电机，一个 Arduino，一个网络摄像头，以及各种电线，螺母和螺栓等。拿一份代码的[副本，把它们放在一起，你就可以开始工作了！](https://github.com/sentryGun53/Project-Sentry-Gun)

只要看看下面的角斗士 II 彩弹哨兵枪的视频——我们知道你现在就想造一个。
