# 树莓皮门铃功能齐全

> 原文:[https://hack aday . com/2015/02/22/raspberry-pi-门铃-功能齐全/](https://hackaday.com/2015/02/22/raspberry-pi-doorbell-is-fully-featured/)

当你想到门铃时，你通常不会想到任何非常复杂的东西。这是一个按钮，在你的家里响起铃声。大概就是这样。[Ahmad]决定用这个版本把他的[门铃调到 11 个](https://docs.google.com/document/d/1e7U_WM1jSo0wgMX5RdpN0eN6p9TWadGYHqWf6yR2JZM "RasPi Doorbell") (Google Doc)。使用 Raspberry Pi，他能够塞进大量的功能。

当按下门铃按钮时，可以触发许多不同的事件。在演示视频中，[Ahmad]展示了他的手机如何接收短信、电子邮件和推文。该系统甚至可以配置为使用 USB 麦克风通过 Google Hangouts 进行语音呼叫。[Ahmad]演示了这一点，并展示了当按钮被按下时，语音呼叫是如何几乎立即发出的。这可能有点过分，但它确实展示了许多不同的选项，取决于您自己的需求。

在硬件方面，[Ahmad]购买了一个无线门铃。他打开了铃声单元，通过一个电阻将扬声器电线连接到树莓派的几个引脚上。门铃装置本身由 Pi 的 3.3V 电源供电。Pi 还有一个小 LCD 屏幕，显示有用的信息，如互联网连接是否正常。屏幕还会显示最后一次按门铃的时间和日期，以防你不在家应门。

最重要的是，该系统还包括一个树莓 Pi [相机模块](http://hackaday.com/2013/02/06/raspberry-pi-camera-board-incoming/ "RasPi camera")。这使得[艾哈迈德]可以拍下按门铃的人的照片，作为一种安全措施。他甚至可以通过直接传输到 YouTube live 来观看前门的直播视频。[Ahmad]在谷歌文档中提供了他的 Pi 图片的链接，这样其他人可以使用它，并根据他们的需要进行修改。
