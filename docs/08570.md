# 用面包切片机磨刀？

> 原文：<https://hackaday.com/2015/03/18/sharpening-knives-using-a-bread-slicer/>

[Joekutz]来信告诉我们他非常有趣的发明——一台磨刀机，由自动面包切片机改造而成。迷茫？我们读主题的时候也是如此！

厌倦了手工磨刀，[乔]想加快速度。他最近看到了我们关于用面包机制作工具磨刀转盘的帖子，他想，为什么不用面包切片机制作一个呢？我们不知道你们是怎么想出这些的——终于有一些真正的黑客了！

首先，他拆开了面包切片机，回收了电机、齿轮和一些电子设备。他用手头的一些层压板做了一个围栏，并用一台旧录像机做了一个支撑盘的轴。为了控制速度，他使用了一个普通的老式灯光开关调光器；不是最有效的，但确实有用！

它使用你可以从任何五金店买到的磨砂盘，正如你在下面的视频中看到的——根据切纸测试，它的效果非常好！
