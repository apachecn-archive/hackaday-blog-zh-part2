# 用于自动降低游戏桌面的车库门开启器

> 原文：<https://hackaday.com/2012/09/24/garage-door-opener-used-to-automatically-lower-a-game-table-top/>

[Lou]又来了，这次他写信让我们了解他的自动乒乓球台。由于没有合适的地方存放整张多余的桌子,[Lou]决定采用二合一游戏桌的方法，将乒乓球桌的顶部安装到他的台球桌上。乒乓球桌面可不是一个小东西，事实证明，存放它的最好(或者最酷)的地方是在天花板上！一按开关，车库门开启器就会拉开一部分天花板，一个绞盘马达会用两根电缆将桌面降低到位。

该系统使用一些很容易找到的部件，工作非常顺利。[Lou 的]教学视频(休息后嵌入)展示了运行中的系统，并解释了自动化背后的概念。我们不确定绞盘如何停止降低桌子，但天花板部分使用一个灯开关和弹簧组合作为它的限位开关。唯一真正缺少的是闪烁的红灯，工业 klaxon，和雾机需要恭维尖锐的噩梦，绞车马达嚎叫。
