# 3D 打印机上的自动工具更换

> 原文:[https://hack aday . com/2013/09/17/automatic-tool-changing-on-a-3d-printer/](https://hackaday.com/2013/09/17/automatic-tool-changing-on-a-3d-printer/)

![tool](../Images/f3c98a31caacd51e3161d9c30db1b967.png)

[Luis]手上有一个非常有趣的项目。他正在使用 delta 3D 打印机打印一些食物——酸奶、巧克力和其他粘稠液体。因为他打算使用实际的板作为构建表面，校准是关键。解决这个问题的一个办法是对这台打印机生产的所有产品使用相同的、预先测量好的印版。[Luis]然而，解决方案要比这巧妙得多。他给自己的打印机编了程序，自动更换两种工具——一种用于探测建筑表面，另一种用于挤出液体。

这两个工具悬挂在打印机的机身上，通过一点点软件，它们可以被打印机的头部拾取，并用几个磁铁固定住。在软件中[自动调平构建表面后，g 代码命令将工具切换到糊挤压机，用于所有美味的食物。](http://lyulyulyulyu.tumblr.com/post/59283997634/auto-level-z-probe)
