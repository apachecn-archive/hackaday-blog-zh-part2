# 廉价的 ARM 上网本被强加了 Linux

> 原文：<https://hackaday.com/2012/06/20/cheap-arm-netbooks-have-linux-forced-upon-them/>

![](img/a7b6e2d858cf9874c587eaf9f59848b8.png "cheap-ARM-netbooks-have-linux-forced-upon-them")

[Doragasu]以大约 50 欧元(约 63.50 美元)的价格买到了一台 WM8650 上网本。它们预装了一个版本的安卓系统，但他想让它们更像电脑而不是安卓设备。所以他着手在基于 ARM 的上网本上加载 Arch Linux。

这是可能的，因为里面的硬件其实挺好的。800 MHz SoC 配有 256 兆内存。有 2g 的内部存储，7 英寸的显示屏，USB，以太网，WiFi 和音频系统。这与你用 Raspberry Pi(没有视频加速)得到的相当，但也包括所有那些外围设备，一个外壳，一个触摸板和键盘…你明白了。有几个补丁需要应用到内核，让它与硬件一起工作。[Doragasu]在上面链接的帖子中介绍了它们。休息过后，你还可以在视频中听到他的介绍。
