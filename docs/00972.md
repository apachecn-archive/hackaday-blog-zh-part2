# 全地形电动滑板车

> 原文：<https://hackaday.com/2012/07/07/all-terrain-electric-scooter-build/>

[这款全地形电动滑板车](http://build-its.blogspot.com/2012/07/all-terrain-electric-scooter.html)可以在学校区域内破坏限速，甚至不用尝试。[本·卡茨]从头开始建造，并出色地记录了旅程。

他通过重新设计普通老式踏板车的悬挂系统来使用这些大的充气轮。这包括一个悬挂系统，有助于缓冲不平坦驾驶表面的颠簸。增加的甲板高度为机车部件留下了足够的空间。你可以看到安装在后轮附近的三个气缸。这些是马达，连接到一个单独的驱动轴上，有一个[本]制造的齿轮箱。传动轴通过链传动为后轮提供动力。电池装在电机前面的矩形外壳中。

休息之后不要错过视频。[本]采取的东西在道路和越野，平均 15 英里每小时，而最高在 24！
