# 用于干擦绘图仪的 CoreXY

> 原文:[https://hack aday . com/2015/01/25/corexy-for-a-dry-erase-plotter/](https://hackaday.com/2015/01/25/corexy-for-a-dry-erase-plotter/)

在扮演 DnD 多年后，终于轮到[迈克]做 DM 了。当然他不能用他的手*绘制地图*，[所以这意味着建立一个桌面绘图仪](http://suburbnerd.blogspot.com/2015/01/dnd-map-plotter-part-1-introduction.html)。

[迈克]正在以 make langelo 为基础设计他的桌面游戏绘图仪，这是一个极坐标绘图仪，在角落里的两个马达的帮助下在垂直平台上绘制图像。这是一个*桌面*绘图仪，所以通常的垂直排列不起作用，但是有一些项目使用 CoreXY 系统进行类似的*水平*构建。

[桌面 CoreXY 系统](http://suburbnerd.blogspot.com/2015/01/dnd-map-plotter-part-2-plotter-hardware.html)是由刚性铝码棒、3D 打印部件、两个非常便宜的步进电机、一个 Arduino 和一大堆绳子制成的。这是一个非常便宜的建筑，因为[Mike]使用金属尺作为框架，所以它也非常低调——这对于桌面会议来说是一个很好的优势。

到目前为止，[Mike]已经移动了绘图仪的轴，下一步是构建计划中的伺服和笔机制。他有一些关于如何绘制这些地牢地图的好主意，通过矢量化位图图像并将它们发送到 Arduino，我们可能会在即将到来的构建日志中看到。
