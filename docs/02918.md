# 乒乓球“卫星”内的 DIY 空间实验

> 原文：<https://hackaday.com/2013/03/07/diy-space-experiments-within-a-ping-pong-ball-satellite/>

![Pongsat space satellite](img/1c3bd8035d2f837d42dbd35f44fa4cf0.png "Pongsat")

啊哈空间。最后的边疆。虽然许多人梦想有一天成为宇航员(并可能与外星人或赛昂人战斗)，但只有少数人真正实现了梦想。对我们来说幸运的是，有一个中间地带，让大众仍然可以在天上玩得开心。进入[“Pongsat”计划——乒乓球内的太空实验。](http://www.jpaerospace.com/pongsat/index.htm)

由日本航空航天公司(JP Aerospace)创建的这个**免费**项目允许任何人创建自己的迷你实验，并将其送到太空边缘。想象力是极限。好奇一个棉花糖会不会膨胀？对温度感兴趣吗？想知道能不能给太阳能电池充电？把它塞到乒乓球里，找出答案！

查看 [PDF 用户指南](http://jpaerospace.com/pongsat/PongSGuide-n.PDF)开始，然后查看他们的[博客](http://jpaerospace.com/blog/)和[脸书页面](http://www.facebook.com/pages/PongSat/250288635034145)了解更多最新信息。现在出去把你的实验带到火星去吧！(或至少 100，000 英尺)

休息后观看飞行录像。
