# 开源卫星遥测技术

> 原文:[https://hack aday . com/2014/11/25/开源-卫星-遥测/](https://hackaday.com/2014/11/25/open-sourcing-satellite-telemetry/)

国际太阳/地球探索者 3 号于 1978 年发射，其使命是探索地球与太阳的相互作用。几年后，该航天器更名为国际彗星探索者号，被派去探索轨道上的冰球，并于今年早些时候返回地球。与飞船对话是一项艰巨的任务，包括众筹活动、阿雷西博之旅，以及来自全球团队的堆积如山的工作。指挥卫星上的推进器不起作用——储罐中没有压力——但 ICE 任务仍在继续，团队中的一位首席无线电大师已经将为重启项目[制作的遥测解析器/显示器放在 Github](https://github.com/balint256/ice/) 上。

ICE/ISEE 重启项目后台的家伙应该为黑客读者所熟知。他是为廉价的 USB 电视调谐器设计出软件定义的无线电源模块的人，唤醒了所有人对 SDR 游戏的兴趣。[他还通过坐在机场附近用笔记本电脑扮演空中交通管制员](http://hackaday.com/2012/04/16/playing-air-traffic-controller-with-software-defined-radio/)，并在黑帽和 DEFCON 上发表演讲。

ICE/ISEE-3 遥测解析器/显示器允许任何人收听来自卫星的记录遥测帧，检查实际发生了什么，并学习如何在没有计算机的情况下与从数百万英里外快速接近的设备通信。他甚至把一些遥测记录放到互联网上练习。

尽管 ICE/ISEE-3 重启项目还要再等一二十年，直到探测器回到我们的地盘，[巴林特]正在大步向前，在旧金山地区组织了几次软件定义无线电会议。他刚刚举行了第一次会议(视频见下)，会上讨论了在 GNU radio 中创建一个立体声 FM 发射机、SDR 的 DSP 可视化介绍以及来自艾伦望远镜阵列的 SETI 信号。几周后在 Noisbridge 还会有另一场聚会，会上有一些非常酷的主题。
