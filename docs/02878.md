# 激光蚀刻一部 IPhone 5

> 原文:[https://hackaday.com/2013/03/02/laser-etching-an-iphone-5/](https://hackaday.com/2013/03/02/laser-etching-an-iphone-5/)

[![Laser Etched iPhone 5](../Images/3348afd5c7e3b048ca1ef076250c889a.png)T2】](http://hackaday.com/?attachment_id=95357)

里贾纳的 CrashBangLabs 最近得到了一台激光切割机。全光谱切割机是由当地一家公司捐赠的，他们正在升级到一台更大的机器。

由于没有激光切割经验，[Brett]决定他的第一个项目将是[激光雕刻他的 iPhone 5](http://crashbanglabs.org/laser-engraving-an-iphone-5/#/vanilla/discussion/embed/?vanilla_discussion_id=0 "Laser Engraving an iPhone 5")。这是一个有点雄心勃勃的第一个项目，因为功率和速度必须正确设置，以获得良好的对比度，你只有一次尝试获得正确的。此外，使用过多的电力可能会使手机变成激光蚀刻砖。

[Brett]使用一个旧的铝制 iPod 进行测试。一旦激光的速度和功率调好，他就开始为真实的东西加载艺术品。切割机在蚀刻艺术品方面做得很好，但当蚀刻开始时，很明显出现了对齐错误。幸运的是[Brett]决定不打断切割机，最终得到了一部好看的手机，只是有一点对齐问题。

休息之后，检查一下激光切割机工作的时间间隔。
