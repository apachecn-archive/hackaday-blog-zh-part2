# 遥控拖拉机制造

> 原文：<https://hackaday.com/2013/11/08/rc-tractor-build/>

[雷米]是法国的一名工程师，经营着自己的小型等离子切割店，为人们设计和制造定制项目。他的最新项目是这台非常精巧的遥控拖拉机。

它使用两个 350W 的 24V 电机，由两个 12V 铅酸凝胶电池供电。整个框架是用 3D CAD 设计的，然后用等离子切割机切割成型。它的设计目的是牵引小型农业设备，或者将割草变成一种有趣的游戏，可以在舒适的躺椅上一边啜饮冷饮一边进行。

我们分享这个的原因是因为[雷米]制作了一个关于这个项目的制作和建造的优秀视频——所以休息后请留下来，尽情享受吧！但是要注意的是，观看视频可能会引发拥有一台个人等离子切割机的欲望。哦，可能性！
