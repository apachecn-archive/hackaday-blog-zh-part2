# 玩木球的乐趣

> 原文:[https://hackaday.com/2014/01/24/fun-with-wooden-balls/](https://hackaday.com/2014/01/24/fun-with-wooden-balls/)

![DCIM135GOPRO](../Images/fa5797fd67afb6f8a09034f11221cc9c.png)

你有没有发现为了一个项目需要自己做木球？坦白说，我们也没有！但是说真的，你会怎么做呢？幸运的是，哈佛校友杰里米·库克已经尝试了几种不同的方法。

他最初的灵感来自[菲利普·斯蒂芬斯]的视频，他用手工制作的孔锯完全手工制作它们。他不想花几个小时做一个球，所以他想了一些自动化的方法——嗯，算是吧。

他的第一次尝试是使用一个磨和一个由木头夹子组成的基本旋转索引表——固定一个木头销钉，锯一半，在夹子中旋转，重复无限次。最终你会剩下一个木球，它锋利的边缘可以被折断。他对这种方法不太满意，在 Reddit 上发现了一个关于用[一种相当巧妙的方法制作木球的帖子……](http://www.reddit.com/r/DIY/comments/1v3wfa/diy_wooden_ball_without_a_lathe/)休息后留下来看看如何制作。
