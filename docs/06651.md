# 你可能很酷，但你没有燃气轮机摩托车酷

> 原文:[https://hackaday . com/2014/07/25/you-may-be-cool-but-you-not-gas-turbine-motorcycle-cool/](https://hackaday.com/2014/07/25/you-might-be-cool-but-youre-not-gas-turbine-motorcycle-cool/)

在过去的四年半时间里，[Anders]一直致力于一个摩托车项目。这可不是普通的将车库地板涂满油的旧哈雷摩托车—[这是一辆燃气轮机驱动的自行车](http://hackaday.io/project/1350-Gas-turbine-Land-Racing-motorcycle),旨在打破在波恩维尔的陆地速度记录。

[安德斯]自行车内部的发动机是燃气轮机，而不是喷气式发动机。这些发动机的设计真的没有太大的区别，除了涡轮将所有的能量都转储到传动轴上，而真正的喷气式飞机将所有的能量都转储到这辆自行车后面的汽车的前保险杠上。[Anders]从零开始构建这个引擎，在一个巨大的 120 页论坛线程上完整地记录了[。几乎所有的东西都是他加工的，用螺栓固定在他设计和制造的框架上，如果运气好的话，将会打破邦纳维尔盐滩上 349 公里/小时(216 英里/小时)的陆地速度记录。](http://jetandturbineowners.proboards.com/thread/53/landspeed-bike-diy-gas-turbine)
