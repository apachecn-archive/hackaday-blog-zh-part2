# THP 四分之一决赛:低血糖

> 原文：<https://hackaday.com/2014/09/14/thp-quarterfinalist-hypoglycaemia/>

出于某种显而易见的原因，没有多少医学黑客进入 Hackaday 奖的四分之一决赛。一个例外是[【托马斯】“低血糖警报系统](http://hackaday.io/project/2117)”，这是一种蓝牙设备，可以检测睡眠中的糖尿病患者的低血糖并呼叫帮助。

这不是唯一一个进入 Hackaday 奖四分之一决赛的血糖监测仪。[约翰·科斯蒂克] [为他的 1 型儿子](http://hackaday.com/2014/07/13/open-source-glucose-monitoring-an-the-front-lines-of-innovation/)逆向设计了一个连续血糖监测器(我们还为[做了一个关于他的黑客传记](http://hackaday.com/2014/09/09/thp-hacker-bio-john-costik/))。这个项目的范围略有不同，不依赖于预先存在的血糖传感器。事实上，它根本检测不到葡萄糖。相反，它使用湿度和温度传感器来检测低血糖时经常出现的大量出汗。

这种低血糖监测器应该由用户在晚上佩戴。睡眠时血糖水平会下降，如果血糖下降过低，可能会导致死亡。当监视器检测到低血糖症状时，它通过蓝牙链接连接到智能手机，并向联系人列表中的电话号码发送短信提醒。无论谁收到这条信息，都会试图唤醒潜在的无反应糖尿病患者，如果失败，就会在他们的牙龈下放一些蛋糕糖霜(说真的。向警察/急救人员要蛋糕糖霜。好的有一些)。

[Thomas]正在研发一种功能性设备，尽管他的外壳出现了一些问题。现在，他正致力于蓝牙通信部分的建设，我们希望一个完整的，正常工作的设备就在眼前。
