# 宇宙的净熵值如何大幅增加？

![waste](../Images/30695d62c7511fafcfa8a0dd511caad7.png)

人们将会提出的最大也是最后一个问题是，“宇宙的净熵值如何大幅减少”。接下来，最糟糕的——也可能是第一个——被问到的问题是，“宇宙的净熵值怎么会大幅度增加 *？”*前一个问题没有足够的数据来给出有意义的答案，[我们找到了后一个问题的答案](http://www.niklasroy.com/project/137/perpetual_energy_wasting_machine)。这是一种设计用来浪费能量的机器，与永动机完全相反。

这种机器沿着一栋建筑的两层楼设置，用电缆、滑轮和杠杆不断按下电梯按钮。该装置由电梯门的开启提供动力，因此当电梯在一楼开启时，二楼的机器部分呼叫电梯。这种情况不断重复。

等等。越来越好了。在电梯轿厢内，有一个改进的打印计算器，也由电梯门驱动。每次门打开时，它都会计算电梯每个周期消耗的能量。这是一个没有埋头孔的液压电梯，所以向下移动实际上是自由的，但电梯的每一次循环仍然会消耗 11.8 千焦的宇宙能量。为了使构建完全浪费资源，打印计算器干净利落地将其打印带清空到废纸箱中。

我们很想称之为[Rube Goldberg]机器，但这似乎并不适合这台什么都不做的机器。称它为无用的盒子更合适，但这比一个自动关机的盒子更令人印象深刻。不管它是什么，你可以在下面看到它的视频。
