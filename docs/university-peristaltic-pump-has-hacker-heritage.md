# 大学蠕动泵有黑客血统

> 原文：<https://hackaday.com/2015/07/29/university-peristaltic-pump-has-hacker-heritage/>

范德比尔特大学的一个团队已经开始组装他们自己的蠕动泵。蠕动泵用于为研究、医疗和工业应用输送精确量的流体。他们甚至偶尔习惯于[给鱼缸](http://fishtankprojects.com/diy-aquarium-projects/arduino-controlled-dosing-pumps.html)投药。

[![pump](img/f86c5df60466c192e7c73296bbb03f65.png)](https://hackaday.com/wp-content/uploads/2015/07/pump.gif) 它们的工作原理是用一系列滚筒挤压软管中的液体(查看右边[维基百科](https://en.wikipedia.org/wiki/Peristaltic_pump)的精彩 gif)。我们之前见过 [3D 打印的](http://hackaday.com/2014/09/16/a-3d-printed-peristaltic-pump/)蠕动泵，便宜的泵已经出现在[易贝](http://Cheap ebay: https://www.youtube.com/watch?v=9szzHkiefuM)上。但这种构建是为实验室级设计的，虽然廉价的易贝设备可以提供约 20 毫升/分钟，但这种设备可以提供微升/分钟的流速。与通常花费数千美元的商业研究级泵相比，它还具有显著的成本优势，这些泵中的每一个仅花费 50 美元。

该泵具有明显的黑客传统，使用 Arduino Uno、Adafruit 电机护罩和 3D 打印的机械零件。所以特别棒的是，他们还免费提供他们的设计文件和 Arduino 代码！
