# 富巴里诺竞赛:《我的世界》、塞尔达、阿尔杜伊诺斯和哈卡戴

> 原文：<https://hackaday.com/2013/12/22/fubarino-contest-minecraft-zelda-arduinos-and-hackaday/>

![song](img/a0b0892efa0fb66aa325715bde006d10.png)

为了迎合 Fubarino 大赛的玩家群体，[Laurens]巧妙地结合了塞尔达、《我的世界》和一个 Arduino 的传说，创造了一个非常非常酷的东西。

[Laurens]拼凑了一个 Arduino、MIDI 连接器和 LCD 显示器，它将读取 MIDI 键盘并检测何时播放时间/Majora's Mask 中的一首歌曲。然后 Arduino 会更慢更长地播放歌曲，就像在游戏中一样。

事情变得很酷:因为[Laurens]有一个 Arduino，它知道什么时候播放 OoT/MM 的歌曲，他可以让孙松控制灯光，或者风暴之歌打开他的喷水系统。他选择将所有这些命令传送到《我的世界》，在那里,《治疗之歌》给了《我的世界》这个角色一些生命,《风暴之歌》控制了雨水，以及塞尔达和《我的世界》的其他精彩混搭。

这个项目提供了足够多的东西，但[Laurens]还增加了一个复活节彩蛋。当在键盘上播放 ASCII 中的字母时，我们最喜欢的 URL 出现在 Arduino 和《我的世界》内部。

这里有一个[图片库](http://imgur.com/a/PA8QC)和[源代码](https://dl.dropboxusercontent.com/u/43744321/MIDI_OoT.zip) (dropbox，所以不要发垃圾邮件给它)【劳伦斯】的了不起的项目。
