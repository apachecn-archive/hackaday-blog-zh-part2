# 一个水激活手电筒？

> 原文:[https://hackaday.com/2014/09/22/a-water-powered-flashlight/](https://hackaday.com/2014/09/22/a-water-powered-flashlight/)

我们都见过柠檬电池或土豆钟，但你见过水激活电池吗？

【内森·斯塔布菲尔德】是一位美国发明家(生于 1860 年)，他从来没有像那个时代的其他伟大发明家【特斯拉、贝尔、爱迪生等】一样得到如此多的认可——尽管他确实展示了一些非常有趣的无线电话技术。除了涉足不可见的无线电波，【斯塔布菲尔德】[还申请了一项名为“地球电池”的专利](http://www.google.com/patents/US600457)，它利用了浸没在水中(或潮湿的泥土中)的两个不同材料的线圈(一对伏打线圈)。正如你所想象的，它并不是非常有效，无论如何也不是非常高效——但它确实有效。

【Lasersaber】最近一直在摆弄“Stubblefield Coil”，利用该理论设计了一款工作手电筒。他设计了一个 [3D 打印线圈支架](http://www.thingiverse.com/thing:463996)，可以让你轻松地将铜条和镁条缠绕在上面来制作线圈。三个这样的电池串联在一起就可以生产出你的水电池(和手电筒的手柄)。

虽然还没有完全完成，但他计划用一种多孔材料包裹它，这种材料可以很快变干——这样当你需要手电筒时，你可以把它浸泡在一桶水中，希望在需要更多水之前有半个小时的运行时间。它不会永远持续下去。
