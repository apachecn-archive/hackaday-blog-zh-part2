# 《火星人》:科幻小说写作方式的一个里程碑式的变化

> 原文：<https://hackaday.com/2015/08/04/the-martian-a-landmark-change-in-how-sci-fi-is-written/>

除非你在过去的几百年里一直生活在火星的岩石下，否则你很可能听说过[安迪·威尔] 的书[《火星人》。我们在 HAD 并不经常推荐一本书，但是这里有很多很酷的小事情，所以我们必须与你们这些优秀的人分享。我们不会给任何剧透。但请注意，底部的视频是这样的，我们希望鼓励评论不要剧透。](http://www.andyweirauthor.com/)

那么这本书为什么会引起我们的注意呢？首先，它是由一位真正伟大的作家在网上自助出版的，一次一章。随着关注他作品的人越来越多，作者开始收到越来越多关于故事和技术细节的反馈。然后，他会回去根据观众的建议/修改对作品进行修改。这让你想起什么了吗？也许有点像开源运动？当然，作家和他们的观众一起工作，帮助保持一部小说到它的每一部续集的连续性。但这是根本不同的，观众变成了一种创造性的力量，可以穿越时空改写未完成的故事。

引起我们注意的第二件事是，这是一本由另一位极客写的书。看，[安迪]是一个程序员，在写作这本书时，而不仅仅是编造日期和飞船的飞行路线，他实际上写了软件来做真正的轨道力学，所以这本书尽可能准确。如果你喜欢阅读技术细节，同时对精彩的故事很感兴趣(哪个业余读者不喜欢？)，这是给你的书。

如果你的手忙着烙铁，我们也可以全心全意地建议有声书，因为表演者做得非常出色。或者如果你愿意，你可以等到十月份电影上映。我们不能保证好莱坞不会把这个搞砸，所以你最好回避一下，提前看完这本书。
