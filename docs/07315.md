# 用催眠的方式写信息

> 原文：<https://hackaday.com/2014/10/13/writing-a-message-in-hypnotizing-style/>

如果你曾经在机场候机楼遇到过快速旋转的分离式襟翼显示器，你很难不停下来，对它们的动作惊叹几秒钟。因为同样的迷恋，[M1k3y]开始[修复一个旧的一百二十个字符的符号](http://m1k3y.de/0002-split-flap-display-de.html)，他在自己的博客上概述了这个过程。

找到关于这个古老遗迹的文献证明是不可能的；该模型的生产者自己甚至没有保持它的手了。尽管如此，[M1k3y]还是通过研究元件，通过排除过程确定了驱动符号的少量电路的功能。经过近一年的研究，他偶然发现了 trollhle Compute Club 的一个视频，演示了相同显示模型的成功使用。幸运的是，他们很友好地分享了他们的工作源代码。通过对他们例子中的串行协议进行逆向工程，他能够编写自己的软件，最终让标牌动起来。

一旦启动并运行，[M1k3y]了解到只有 80 个符号的字符仍然是可操作的，但这足以做出一个令人着迷的声明！下面是一段骑行信的视频:
