# 谷歌使用的技巧是在液晶显示屏上隐藏秘密信息

> 原文:[https://hack aday . com/2015/07/15/trick-Google-used-hides-secret-messages-on-LCD-screens/](https://hackaday.com/2015/07/15/trick-google-used-hides-secret-messages-on-lcd-screens/)

[Travis]今年没有被选中去 Google I/O，但他确实在事后获得了一些 I/O 启发的乐趣。他的朋友告诉他，谷歌在活动中散布了经过特殊改造的液晶显示屏。当用肉眼观看时，屏幕显示正常的节目信息。不过，当通过 I/O swag 附带的一个特殊的透明徽章查看时，谷歌寻宝游戏的 URL 会神奇地出现。[Travis]对这种效果很感兴趣，[决心要自己复制这种效果](http://www.xodustech.com/projects/google-io-scavenger-hunt-polarized-monitor)。

[ ![dual-lcd-3](../Images/b819571d9a7d16e6598f589bc6ce9874.png) Travis]发现透明徽章实际上是一个偏光滤镜。每个标准液晶显示器都有两个，通常与液晶显示器本身的玻璃粘合在一起。如果你从 LCD 上取下滤光器，你会看到背光的主要部分—[,当然，除非你戴上偏振眼镜](http://hackaday.com/2011/11/25/making-a-privacy-monitor-from-an-old-lcd/)。不过谷歌的显示器没有这种效果。他们展示了一个全彩色显示器，第二个全彩色隐藏显示器只能通过偏振器看到。[Travis]聪明且经验丰富，所以他只需要一点立体思维就能猜出谷歌的诡计。显示器中实际上使用了两个 LCD。第一个是带背光的标准 LCD。诀窍是把第二块 LCD 上的偏光膜撕下来，放在第一块 LCD 的前面。没有偏光镜，第二个 LCD 对任何人都是不可见的。

[Travis]迅速着手使用几个过时的 VGA 液晶显示器复制显示器。他很快发现，最难的部分是从薄玻璃液晶夹层中剥离偏振塑料。几个液晶显示器放弃了他们的努力，但最终[特拉维斯]成功了。他在一个为冷阴极荧光灯(CCFL)显示器设计的盒子中使用了一个薄的 LED 背光，使一切都适合一个盒子。结果看起来就像一个标准的液晶显示器-也就是说，直到通过偏光过滤器观看。点击休息，看看隐藏的消息液晶显示器的行动！
