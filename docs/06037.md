# 热胶注射成型

> 原文：<https://hackaday.com/2014/05/12/injection-molding-with-hot-glue/>

注射成型是简单地将熔化的热塑性塑料压入某种模具中，让它冷却，然后撬开模具得到成品。热胶枪基本上是手持式热塑性塑料挤出机，所以当[斯科奇]挖出他闲置的一些旧注塑模具时，[没多久就把两两组合在一起](http://www.scorchworks.com/Blog/hot-glue-injection-molding/)。

注塑模具不是任何正常人都有的东西，但几年前[斯科奇]发现了金格里出版的两本书，这两本书也是出版了如何用废料建造一个金属车间的说明书的人。[斯科奇]在一台小型数控铣床上——锡格河·X3——制造了他的模具，他最初的注射成型塑料实验相当成功，即使模具是由自铸坯料制成的。

在用他的设备成型了几个热胶乐高零件后，[斯科奇]在网上看了看，发现这并不是什么新鲜事。一家公司甚至[出售基于热胶枪的注射成型套件](http://www.mindsetsonline.co.uk/Catalogue/ProductDetail/injection-moulding-pack?productID=d6c62a67-6148-4fbc-a62d-dece4b4be0c6&catalogueLevelItemID=00000000-0000-0000-0000-000000000000)使用聚乙烯胶棒。他们的演示视频(见下文)似乎比[斯科奇]的努力要复杂得多，所以我们会说他在这一点上领先了。
