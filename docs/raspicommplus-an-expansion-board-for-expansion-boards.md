# RasPiCommPlus，用于扩展板的扩展板

> 原文：<https://hackaday.com/2014/11/17/raspicommplus-an-expansion-board-for-expansion-boards/>

将 GSM 模块连接到 Raspberry Pi 的最简单方法是购买一个分支模块，安装一些软件，然后用 Pi 连接到移动网络。也需要 GPS？这是一个完全不同的模块，使用不同的软件。RasPiCommPlus 背后的家伙正在研究一个更好的解决方案——[一个用于分线板的分线板](https://www.indiegogo.com/projects/raspicommplus-all-in-one-raspberry-pi-extension),它负责将大量模块插入 Pi，并整理内核驱动程序，使与这些模块的接口变得容易。

目前，该团队拥有一个 GPS 和 GSM 模块、数字输入和输出模块、一个模拟输入模块以及 RS-232 和-485 模块。他们正在开发一些很酷的新产品，包括夏普内存显示器的突破，9 轴 IMU，步进电机驱动器和 1 线突破模块。
