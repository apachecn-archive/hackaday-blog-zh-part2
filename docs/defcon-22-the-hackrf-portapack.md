# DEFCON 22:HackRF 便携式包

> 原文：<https://hackaday.com/2014/08/17/defcon-22-the-hackrf-portapack/>

[https://www.youtube.com/embed/koS-RobcKXI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/koS-RobcKXI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

当您将最好的(当然也是性价比最好的)软件定义无线电与 10 年前的 iPod 的用户界面相结合时，您会得到什么？HackRF PortaPack 由[Jared Boone]开发，并在上周末的 DEFCON 上展示。

[Jared]是[hack RF](https://greatscottgadgets.com/hackrf/)的原始开发者之一，这是一种 10MHz 到 6GHz 的软件定义无线电*，也可以半双工传输*。自从 HackRF 的开发(在某种程度上)结束以来，[Jared]一直在从事 porta pack 的[工作，这是 HackRF 的一个附件，可以将其变成一个便携式的、ARM Cortex M4 供电的软件定义无线电。不，它没有运行 GNU Radio 的完整计算机强大，但是它有能力监听数量惊人的无线电信号。](http://www.sharebrained.com/2014/05/28/portapack-h1-imminent/)

因为[Jared]为 PortaPack 使用了一个相当低功耗的微处理器，所以他使用了许多技巧来让一切顺利运行。他在 DEFCON 的无线村做了一个闪电般的演讲，讲述了他正在使用的芯片的优缺点，令人惊讶的是，他在代码中很少使用浮点运算。你可以看看下面的视频。
