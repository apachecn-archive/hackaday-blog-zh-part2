# 使用导电墨水的 3D 印刷电路板

> 原文:[https://hack aday . com/2012/05/22/3d-印刷电路板-使用-导电-墨水/](https://hackaday.com/2012/05/22/3d-printed-circuit-boards-using-conductive-ink/)

![](../Images/005423639e137e21968bc7be61512de1.png "3d-printing-pcb")

使用 3D 打印机制造印刷电路板的想法很诱人，好消息是这是现实。该项目表明，可以使用一种特殊的[打印头将轨迹应用到挤压基材](http://wiesel.ece.utah.edu/redmine/projects/3d-pcb-maker/wiki)上。

这类似于[点对点 3D 打印机电路](http://hackaday.com/2012/05/04/printing-point-to-point-circuits-on-a-3d-printer/)的一大升级。现在，可以使用导电墨水将迹线直接印刷到 ABS 上。该过程从设计文件开始，设计文件用于对每条迹线都有沟槽的衬底进行建模。然后 Makerbot 打印出这个模型。一旦完成，ABS 挤出机头换成一个特殊的油墨头。然后用导电流体填充每条迹线，导电流体被沟槽壁保持在适当的位置，直到它变干。我们认为这改进了[我们之前见过的痕迹印刷技术](http://hackaday.com/2012/05/18/printing-circuitry-on-a-reprap/)，因为它不需要你的打印机加热来使用熔融金属。

上面的电路将印刷走线用于 LED 电路的高端和低端。它的边缘有点粗糙，但它显示了很大的希望。不要错过跳转后嵌入的演示视频。
