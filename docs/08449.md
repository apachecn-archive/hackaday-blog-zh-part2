# 硬盘…扬声器？

> 原文:[https://hackaday.com/2015/03/03/hard-drive-speakers/](https://hackaday.com/2015/03/03/hard-drive-speakers/)

扬声器真的没有那么复杂。事实上，如果你聪明的话，你可以用任何东西来做一个演讲者。[一个女人]正在用他在 2001 年第一次做的黑客攻击来踢它的老学校，但是现在，以视频的形式:[做你自己的硬盘扬声器！](https://www.youtube.com/watch?v=rVPjQou42i8)

你所需要的只是一个你不再关心的旧硬盘，一根软线，和一个外部供电的放大器(不，你的手机不能用！).如果你没有放大器，[Afroman]甚至有一个教程，所以你可以[建立自己的 D 类放大器在试验板上！](http://afrotechmods.com/tutorials/2014/12/20/class-d-amplifier-tutorial/)

首先，你需要打开硬盘外壳。不过，你可能需要一个梅花或六角钥匙来通过制造商的“安全螺丝”。一旦它被打开，你需要找到硬盘驱动器头——这是一个小的金属臂，看起来有点像电唱机的音频臂。它实际上是由一个线圈控制的，你知道，就像一个扬声器…

拿出你的万用表，开始探测！来自硬盘驱动器磁头的带状电缆将有两根导线，电阻在 4 到 40 欧姆之间，这实际上是控制磁头的线圈，因此电阻也是如此。把你的电线焊接在那里，给它一个放大的音频信号，就这样！
