# 机械鸟实际上是通过拍打翅膀来飞行的

> 原文:[https://hack aday . com/2014/10/12/机械鸟实际上是靠拍打翅膀飞行的/](https://hackaday.com/2014/10/12/mechanical-bird-actually-flies-by-flapping-its-wings/)

事实证明，你不必像 Festo 那样成为一家价值数百万美元的公司，就能创造出一个遥控的会扑翼的鸟机器人。[Kazuhiko Kakuta]是一名过敏症医学博士，闲暇时他喜欢和儿子一起制作飞行机械鸟。

它的翼展刚刚超过一米，重 193 克，靠拍动翅膀飞行。它的大部分部件都是 3D 打印的。如果这还不足以让你印象深刻，考虑一下这个。它没有传感器，没有陀螺仪或任何东西——它都是由[Kazuhiko]手动控制的。

这甚至不是他唯一做过的扑翼飞机。他还从一部动漫电影[中创造了一些东西](http://en.wikipedia.org/wiki/Castle_in_the_Sky)。他甚至[出售其中一款的设计](http://homepage2.nifty.com/smark/3DPShop.html)，通过 Shapeways 打印出来。
