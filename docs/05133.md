# 3D 打印吉他

> 原文:[https://hackaday.com/2014/01/25/3d-printed-guitar/](https://hackaday.com/2014/01/25/3d-printed-guitar/)

我们不确定我们怎么会错过这个，但它绝对值得一看。机电学教授[奥拉夫·迪格尔的] 3D 打印机必须去 12，因为他的[打印了这些不可思议的电吉他琴身](http://www.3ders.org/articles/20120616-new-atom-3d-printed-electric-guitar-has-a-more-metallic-sound.html)。然而，你可能不会在你的细丝打印机上制作自己的，因为[Diegel]使用[SLS(选择性激光烧结)](http://en.wikipedia.org/wiki/Selective_laser_sintering)用尼龙制作身体，然后他在两步过程中对最终的作品进行染色。你可以在[的网站](http://www.odd.org.nz/construction.html)上了解更多关于建设细节的信息。

而且，它们不仅仅是赏心悦目:这些吉他发出明亮的金属声。网站上有足够多的图片和视频让你忙个不停，在那里你可以随心所欲地筛选所有八个设计。休息后，您可能会想继续阅读几个视频，其中包括吉他的一些演示和与传统电吉他的比较，以及吉他建造和制作过程的简史。
