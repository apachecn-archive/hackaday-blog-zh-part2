# 100% DIY interval meter 100%牛逼

> 原文:[https://hack aday . com/2014/11/16/100-DIY-interval ometer-is-100-awesome/](https://hackaday.com/2014/11/16/100-diy-intervalometer-is-100-awesome/)

从这个过程纪录片中很容易看出【Nagyizee】不是一个满足于预制一切的人。他本可以买一些现成的 DSLR intervalometer，但这意味着通过冰冷、没有灵魂的塑料与别人的设计进行交互。

[Nagyizee]想要一个独一无二的从头开始建造的工具。除了计时器之外，他还在市场上寻找光传感器和声音探测器。他以节能的名义选择了一款 8MHz 的 STM32F100 ARM Cortex M3，并开始设计 UI 和固件。用于有机发光二极管显示器的定制图形库进一步简化了它。一旦草图最终确定，[Nagyizee]设计了一个时尚且符合人体工程学的木箱，与一个微型的 [Proxxon F70 一起碾磨。](http://www.proxxon.com/us/micromot/37110.php)

随着外壳决定的方式，他蚀刻和钻孔的 PCB 和放置元件。光传感器需要一个透镜和一个棱镜，所以他用一个 10 毫米的 LED 主体做了一个。不放过任何一个细节，[Nagyizee]还转动了一些按钮，手绘，并制作了一个滚轮。他以[一个证明它很有能力的演示](http://youtu.be/E2Czay6RSNM?t=10m)结束了视频。除了标准的线缆释放模式，它还可以处理长曝光时间、连续拍摄以及对光、影或声音的捕捉。但是等等，还有更多:[Nagyizee]的创作将模式与轻松优雅结合在一起。
