# 使用电压调节器作为恒流源

> 原文：<https://hackaday.com/2015/07/04/using-a-voltage-regulator-as-a-constant-current-source/>

[Afroman]联系我们，在 LM317 上分享他的新[视频。不起眼的 LM317 可调稳压器无处不在。从 wifi 路由器到高规格实验室设备。给定一个噪声输入和一个可变负载，电压调节器将给出一个干净、稳定的输出电压。我们已经多次介绍过 lm 317](https://www.youtube.com/watch?v=iuMngik0GR8)的基本操作和使用方法。但是即使是最普通的零件也可以用新的有趣的方式来使用。

[Afroman]在他的视频中描述了如何使用 LM317 来调节电流而不是电压，从而在不同负载下提供恒定的电流源。这对于包括驱动 led 和激光二极管在内的许多应用是有用的。虽然这种电路可能不如 LED 驱动模块或开关解决方案有效，但 LM317 价格便宜且易于获得。[Afroman]还详细描述了电路的工作原理，让我们能够在这个稍微有些不同寻常的应用中享受这一无处不在的器件。
