# 铁人复制品二合一

> 原文：<https://hackaday.com/2012/05/23/ironman-replica-twofer/>

![](img/ea01d9b1e4e4a9c91b889b99ccc8bac9.png "arc-reactor-and-faceplate")

我们认为现在穿上全套服装去参加《复仇者联盟》的试映会有点晚，但是一件 arc reactor T 恤会很酷。[4]T2 制造了一个胸带，看起来很像[托尼·史塔克的]胸装电源。它有一个 3D 打印的外壳，容纳 ATmega8 和 22 个 led，提供脉冲善良。薄薄的手机电池有助于将包裹的尺寸保持在最小~~，黑色 t 恤上一个策略性的洞使外观更加完美~~。它甚至足够明亮，可以透过这件黑色 t 恤的面料。

但是如果你坚持从头到脚的华丽，你会喜欢[詹姆斯·布鲁顿的]钢铁侠套装复制品。他不仅看起来像个角色，而且还试图将尽可能多的功能构建到项目中。最近他完成了头盔。它有一个电动面板和 LED 边缘照明眼板，给黑客和 cosplay 粉丝留下深刻印象。
