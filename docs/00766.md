# 制造一把灭虫光剑

我已经听过好几次这个笑话了，说一把光剑可以让一个[大杀器](http://www.youtube.com/watch?v=HBI1ddZvD70)。然而，当[里奇·桑博迪]在脸书提出要求时，我想“为什么不呢？”。[Ricky]犯了一个常见的错误，他认为灯泡是真正杀死虫子的部分。快速的谷歌搜索显示许多人有同样的想法。我决定，即使构建一个实用的灭虫光剑可能看起来不酷，我还是要去做。

如果你打算按照这些说明去做，请注意这是一个字面上被设计用来电击致死的设备。这很危险。

我碰巧有一把可伸缩的光剑，所以我只需要一个便携式灭虫器。它们很容易找到，而且相当便宜。这些便携式灭虫器的形状像一个网球拍，功能是让虫子在两个屏幕之间短路。为了制作我的光剑，我所需要做的就是让屏幕适应我的玩具。

我拆开 zapper，把精细或“内部”网切成两半，包起来。我把两部分重叠起来，所以没有必要焊接。然后，我剪下一些电线作为间隔，并安装了更大的“外部”屏幕。检查后，以确保没有位接触，我终于有线到控制器和管道胶带它都在一起。

对于如此快速的黑客攻击，它的效果出奇的好。整个过程大概花了 20 分钟。

如果我再做一次，目标是让它变得更好，我会做以下事情。

*   -使用更高质量的实际发光的光剑。音效也不会有坏处。
*   -把电路塞到光剑手柄里。看起来会更好。
*   -找到一些新的屏幕材料，而不是重复使用球拍上的材料。
