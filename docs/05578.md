# 马克杯音乐是好到最后一滴

> 原文：<https://hackaday.com/2014/03/19/mug-music-is-good-to-the-last-drop/>

邦妮在普林斯顿主修计算机科学，辅修 Awesome。她正在上电子音乐课，并且必须为她的期中项目制作一个数字乐器。她和她的朋友[Harvest]想出了[马克杯音乐](http://bonnie-eisenman.tumblr.com/post/79418680574/mug-music-turning-water-into-an-instrument-with)，它将一杯陶瓷水变成了一种乐器。

这个电路很容易用一个 Arduino、一个线圈、几个电阻和电容复制。[Bonnie]想要试验一下[迪士尼研究实验室的 Touché](http://www.disneyresearch.com/project/touche-touch-and-gesture-sensing-for-the-real-world/) 触摸检测方法，而 Mug Music 就是基于[这种用于 Arduino Instructable](http://www.instructables.com/id/Touche-for-Arduino-Advanced-touch-sensing/) 的 Touché。这些输入通过 [ChucK](http://chuck.cs.princeton.edu/) 转换成 MIDI 音符，这是一种在普林斯顿开发的实时声音合成语言。

正如你可能已经猜到的，你将在跳跃后的演示视频中看到的，你并不局限于接触水。整个杯子也会发出声音。[Bonnie]说如果你同时接触水和地面，你会触发一个雷霆一击。

这将是一个与孩子一起探索的伟大项目，尤其是作为自闭症谱系的孩子的音乐治疗工具。它不像这些便携式音乐楼梯那么有形，但可能不会引起律师的注意。
