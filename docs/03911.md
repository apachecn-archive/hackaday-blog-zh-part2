# 真空压力火箭筒

> 原文：<https://hackaday.com/2013/08/24/vacuum-pressure-bazooka/>

这个真空压力炮的设计不同于我们以前见过的任何设计。乍一看，它似乎有你在[土豆枪](http://hackaday.com/2012/03/13/the-spud-gun-to-end-all-spud-guns/)中看到的组件。但是它们使用燃烧过程来发射抛射物。取而代之的是利用真空的突然释放。

大约三分钟后，我们看到了“点火”系统。这是相当可怕的，因为几个非常强大的弹簧正沿着枪管向你的脸拉一个项圈。这实际上是为了去除在桶中保持真空的后面的塞子。压力差导致空气突然涌入，将 1.5 英寸的射弹从火箭筒前部射出。

Teslonian 先生]自己造了一个手动真空泵来装载武器。这是用一对 PVC 管和一个由木头和皮革制成的活塞完成的。聚氯乙烯和木材弹丸密封在桶使用管带制成的裙。在后膛装弹并塞住枪管后部后，他在工作端盖上铝箔并抽高真空。他的试射不是从肩膀开始的，他只有一次机会，因为子弹击中目标的力度太大，目标被摧毁了。这东西真的需要车载！
