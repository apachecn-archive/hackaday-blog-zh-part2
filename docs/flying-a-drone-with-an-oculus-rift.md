# 用 Oculus Rift 驾驶无人机

> 原文：<https://hackaday.com/2014/08/31/flying-a-drone-with-an-oculus-rift/>

![image001](img/2fa797bce91c7681d2ea343746b6193a.png)

使用虚拟现实耳机远程控制自动驾驶汽车似乎是显而易见的下一步。一些无人机公司已经开始尝试这些想法，将 Oculus Rift 开发工具包集成到他们的悬停四轴飞行器和无人机中。Parrot 在他们的博客上发布了一段视频，显示他们为 Bebop 无人机开发了一个[头部跟踪系统，以努力实现 FPV 飞行。它看起来很有趣，我们想尽快尝试其中一个！](http://blog.parrot.com/2014/05/15/parrot-bebop-drone-discover-fpv-with-oculus-rift/)

至于技术规格，可以在下面嵌入的视频的 YouTube 描述中找到。快速浏览一下就会发现，该操作系统基于 Android，并使用 WiFi 将手持平板电脑连接到漂浮在上方的自动驾驶汽车。这个范围长达 2 公里，给了人们充分的探索自由。移动一个人的头部会旋转附带的相机，从而提供更身临其境的飞行体验。

这不是我们看到的第一架 FPV 无人机。之前，我们报道过一个 [Oculus Rift +头部跟踪设置](http://hackaday.com/2014/06/18/oculus-rift-head-tracking-the-ultimate-drone-experience/)和另一个[与黑甲无人机](http://hackaday.com/2013/07/16/fpv-drones-with-an-oculus-rift/)的类似集成。随着开发人员获得像 Oculus developer kit 2 硬件这样的尖端硬件，我们肯定会看到越来越多的虚拟现实设备用于控制无人机。
