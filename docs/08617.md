# 英格玛机器手表

> 原文:[https://hackaday.com/2015/03/23/enigma-machine-wristwatch/](https://hackaday.com/2015/03/23/enigma-machine-wristwatch/)

我们还没有发现智能手表非常好用。这个为有用性下了一个定义。恩尼格玛机当然是二战期间德国人使用的密码程序。[这款英格玛机器腕表](http://www.asciimation.co.nz/bb/2015/03/24/a-three-rotor-enigma-machine-wrist-watch)不仅功能强大，而且外观模仿原厂机器。带斑点的灰/黑色表壳和英格玛徽章,[Asciimation]很好地模仿了最初的感觉。

驾驶机器的是 Arduino Pro Mini。我们过去已经见过 Arduino Enigma 机器，所以在这里再次见到它并不奇怪。用户界面包括一个分辨率为 128×64 的有机发光二极管显示屏，三个按钮，充电端口位于右侧，开关位于左侧。

休息后展示该设备。相当多的按钮被用来设置三个编码器轮中的每一个。但是当你不使用全键盘时，这几乎是不可避免的。考虑到硬件的简单性，我们对[ascii mation]界面的功能印象深刻。

这对那些对工程感兴趣的孩子来说似乎是完美的。他们学习密码、嵌入式编程、机械设计和制作(这是一个手工缝制的皮革腕带)。当然，如果你做了一个，然后开始戴着它去办公室，我们不会评论。
