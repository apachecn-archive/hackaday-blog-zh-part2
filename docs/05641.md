# 用于触觉反馈的压电

> 原文:[https://hackaday.com/2014/03/26/piezos-for-haptic-feedback/](https://hackaday.com/2014/03/26/piezos-for-haptic-feedback/)

![Haptic](../Images/44ec834019ba3cae27122cb7c5e8a0b8.png)

自任天堂 64 控制器的插件 rumble pack 以来，在界面中放置某种触觉反馈的最常见方式没有太大变化——只是在那里放置一个寻呼机电机，并在用户需要感觉到什么时让它旋转。这种方法需要相对较长的时间来旋转，即使是非常酷的带有语音线圈方向垫的蒸汽控制器也无法“坚持”，或保持高或低来通知用户一些事情。

[Tim]的日常工作是研究非常奇特的压电致动器，当有机会参观触觉研讨会时，他抓住了这个机会[将这些致动器变成某种界面](http://tim.cexx.org/?p=1129)。他最终创造了两个设备:一个双压电手机大小的设备，和一个带有左击按钮的鼠标，它可以根据鼠标垫的颜色升降。

该手机设备包含两个 10 克重的环氧树脂压电致动器。一个小的微控制器和[压电驱动器](http://www.ti.com/product/drv8662?)给了这个伪手机你能想象的最平滑的振动功能。更具创新性的颜色感应鼠标有一个单一的驱动器粘在左边的按钮上，底部有一个光电传感器。当鼠标在纸上的黑色方块上滚动时，按钮就会升起。滚动到较亮的区域时，按钮会降低。这都是非常非常酷的技术，我们可能会在几年后看到苹果、微软或索尼的产品。
