# Altium 的电路制造商

Altium 最近宣布进入免费/低成本 PCB 设计工具市场的 [CircuitMaker](http://circuitmaker.com/) 。他们正在进入一个庞大的行业，像 Eagle、KiCad、gEDA 和许多其他工具套件。在世界创客大会的 Altium 展台上，我们与 Max 进行了几分钟的交谈，甚至还与工具本身进行了一点时间的交流。

实际操作上，它绝对具有 Altium Designer 的外观和感觉，就像熟悉的黄色和绿色方框一样，用于原理图和板材零件。中心舞台是 3D 视图，这是 Altium 自 90 年代末以来在其软件中就有的功能。

CircuitMaker 的网站正在推广该软件的协作功能。可以实时审查和评论设计选择。这也表明数据文件将存在 Altium 自己的云存储系统中。

CircuitMaker 仍处于预测试阶段，但他们现在正在寻找测试人员，所以请前往[网站并注册](http://circuitmaker.com/)！
