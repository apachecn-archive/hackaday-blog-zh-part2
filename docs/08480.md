# 通往莫莉亚矿井的秘密之门

> 原文:[https://hack aday . com/2015/03/07/a-通往莫瑞亚矿区的秘密之门/](https://hackaday.com/2015/03/07/a-secret-door-to-the-mines-of-moria/)

什么家庭影院没有秘密入口才算完整？[Eclipse_007]有一个绝妙的主意，制作了这个[指环王主题的暗门](http://imgur.com/a/vOnt5)，类似于莫莉亚矿井的入口。

这是一个定制的门板，底部装有 led 灯。前面板是一大块有机玻璃，一面涂有乙烯基涂层。[Eclipse_007]从乙烯基涂层中煞费苦心地切割出图案，全部手工完成。安装后，门看起来就像墙的另一部分。但当触摸时，门会亮起，然后打开，露出电影院。已经有计划让它在说出口令时可以声控开启。

正如一位 reddit 用户所说:
