# 3D 缩放比例

> 原文:[https://hackaday.com/2012/04/19/a-3d-pantograph/](https://hackaday.com/2012/04/19/a-3d-pantograph/)

[![pantograph](../Images/b70e1ea17152e18739cb3ca7495433d9.png "pantograph")T2】](http://hackaday.com/?attachment_id=72042)

精巧的木工工具似乎永无止境，比如来自[【woodgears . ca】](http://woodgears.ca/ "woodgears.ca")的这个[3d 路由器缩放仪](http://woodgears.ca/pantograph/index.html "3d pantograph")。[缩放仪](http://en.wikipedia.org/wiki/Pantograph "pantograph")，一种使用连杆来追踪和缩放图纸的设计，可能已经在 17 世纪发明了，但是如果我们诚实的话，我们大多数人都没有听说过这个设备。这种特殊的缩放仪能够通过绕另一个轴旋转来三维描绘字母或其他模板。

如果你想知道在哪里可以得到这些模板，他们已经为你提供了在线[模板生成工具](http://woodgears.ca/stencil/index.html "stencil generation tool")。然而，如果你想制作更详细的模板，我们可能会推荐使用免费的绘图工具，如 [Draftsight](http://www.3ds.com/products/draftsight/download-draftsight/ "Draftsight") (这里有一个[评论](http://www.jcopro.net/2010/12/29/draftsight-review-like-autocad-but-free/ "Draftsight review"))，或 [Sketchup](http://sketchup.google.com/download/ "sketchup") 。

[Woodgears]在休息后的视频中很好地解释了它是如何制作的。你也可以购买计划，如果你想模板用于削减一切。
