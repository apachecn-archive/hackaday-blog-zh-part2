# Hackaday 获奖作品的自动评判

> 原文:[https://hack aday . com/2014/08/04/hack aday-prize-entries 自动评判/](https://hackaday.com/2014/08/04/automated-judging-of-hackaday-prize-entries/)

![cock](../Images/6976fd09834ed1a9cc21802d0fa14eeb.png)

我们有一些互联网黑客精英来评判 Hackaday 奖，这意味着他们不能参加任何项目。我们认为这对其他人都好。评委之一，[Sprite_tm]是一个足智多谋的家伙，当涉及到评判 Hackaday 奖的参赛作品时，他将做他自然而然会做的事情:[建造一台机器来自动完成这项任务](http://hackaday.io/project/2245-JudgeTron-9001)。

[Sprite]对法官 9001 的计划是使用嵌入生物样本中的神经网络来为他做判断。老实说，我们真的很感谢他为此付出的努力；生物黑客现在真的很流行，我们确实喜欢人工智能复兴时期的经典回归。他在这个项目中使用树莓 Pi *和 Arduino】当然也没有坏处。*

[Sprite]从他的垃圾抽屉里抓起一个触摸屏 LCD 和一些其他部件，很快就完成了一个项目，将 Hackaday 奖的参赛作品显示到生物嵌入式神经网络中。这些网需要一点鼓励来选择获胜的作品，所以一个“反馈”机制是用激光从丙烯酸中切割出来的，安装在伺服系统上，并填充积极的强化。

运行在 Pi 上的软件浏览[黑客日奖](http://hackaday.io/submissions/prize/list)的参赛作品列表，从每一张中提取图像。该计划是由生物神经网络选择获胜的作品，并通过反馈机制获得奖励。这些神经网络被证明对反馈机制的伺服齿轮的声音非常敏感，而[Sprite]试图用他的创作找到一个获奖作品的尝试迄今为止被证明是不成功的。尽管如此，还是有一段视频，你可以在下面看看。
