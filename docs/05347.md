# 全向机器人挑战糖果工厂

> 原文:[https://hack aday . com/2014/02/19/universal-robot-take-on-a-candy-factory/](https://hackaday.com/2014/02/19/omnidirectional-robot-takes-on-a-candy-factory/)

![OmniRobot](../Images/e5b5f30c2a2535655777e0b5c5aa500d.png)

[AltaPowderDog]正在建造一个竞赛机器人,作为他在俄亥俄州立大学大一工程课程的一部分。该比赛由雀巢公司赞助，所以机器人必须在一个微型糖果厂执行各种任务也就不足为奇了。学生们分成四个小组，正在建造自动机器人来移动托盘、舀取糖果、操作开关和从试管中拔出插销。每个团队都有一个标准的微控制器板和从网上商店购买机器人零件的资金。该工厂还配备了高架红外导航系统，这应该有助于机器人保持在轨道上。

[AltaPowderDog]的灵感来自[【Michal 的】OmniBot，](http://hackaday.com/2012/04/03/who-needs-mecanum-wheels/)它使用了可调节的几何轮。杠杆和齿轮系统允许机器人同步转动所有四个轮子。这有效地允许机器人在其自身的轴内转动。通过一些适当的路径规划和末端执行器的放置，[alta powder dog]的团队应该能够缩短他们穿过糖果工厂的时间。不过，这个团队遇到了一些问题。这种机器人设计只利用了两个动力轮，这导致团队被困在工厂的斜坡上。为了解决这个问题，车队安装了一个简单的悬挂系统，允许非动力车轮在坡道上向上移动。结果看起来很有希望。休息后的视频包括一个简短的剪辑[AltaPowderDog 的]“机器人快速转弯并启动开关。非常好的工作！
