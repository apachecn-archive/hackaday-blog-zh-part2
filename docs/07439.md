# 从 Nerf 枪到 rf 加农炮:建立一个电影道具

> 原文：<https://hackaday.com/2014/10/29/from-nerf-gun-to-rf-cannon-building-a-movie-prop/>

[David windestl]回到了美国，这次他带着武器，非常危险！他建造了一个[难以置信的射频大炮道具](https://www.youtube.com/watch?v=rC0aK4ejsFM) (YouTube 链接)，作为他为[旋翼 DR1 系列](http://hackaday.com/2014/09/17/rotor-dr1-and-collaborative-development/)准备的无人机猎人衣橱的一部分。[大卫]对黑客日并不陌生。我们之前见过他[从太空边缘滑翔遥控飞机](http://hackaday.com/2013/03/10/launching-a-glider-from-space/)和[建造加力燃烧室](http://hackaday.com/2013/03/25/giving-an-electric-rc-plane-an-afterburner/)作为 Flite 测试组的一员。

[大卫的]无人机猎人角色装备了一门讨厌的射频加农炮，用来炸飞无人机。猎人可以收集和出售他们的奥秘颗粒能源。[大卫]从一把非常大的 Nerf 枪开始。他切掉了枪的前半部分，换上了螺旋天线。这是[大卫]在他的视频地面站使用的同一种天线。加上激光切割木框架，线圈看起来非常危险。我们很高兴这只是作秀。

[大卫]给枪增加了一些附件，包括开关，一个旧散热器，一些电线，和非常重要的奥秘反应堆。我们非常喜欢他的射频屏蔽手套，它可以防止猎人的桶手被烧焦。[David]在厚的耐化学手套上加了一层铜网。他将铜焊接在一起，并添加了一根电线来连接手套和枪。[大卫]然后寻求 DR1 导演[查德·卡普]的帮助来油漆和风化枪和盾牌手套。结果简直令人震惊。

我们喜欢看黑客走出他们的元素，像这样建立道具。他们总是添加一些现实的功能，使最未来的科幻道具变得更加可信。
