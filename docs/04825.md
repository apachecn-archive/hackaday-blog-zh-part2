# LEDmas 树

> 原文:[https://hackaday.com/2013/12/15/ledmas-tree/](https://hackaday.com/2013/12/15/ledmas-tree/)

![led christmas tree](../Images/748f8b27636f39e137c2ed8a24a073e5.png)

[Nick]是一个 LED 狂热分子。所以当他的老板要求他帮忙制作一棵 LED 圣诞树时，他欣然接受了这个机会。

这是一个美丽的建筑，利用激光(？)切割有机玻璃盘、用车床制作的木制“树干”和一吨 RGB LEDs。不幸的是——因为结果非常好——公司正考虑明年将它作为一个产品出售，所以[尼克]不被允许透露更多关于这款产品的信息。不管怎样，它看起来棒极了，我们相信你可以自己动手。

他被允许拍摄视频，所以休息后来看看吧！他在 www.hownottoengineer.com 的博客上还有很多其他非常酷的 LED 项目
