# 全自动 CD 发射器看起来非常有趣

> 原文：<https://hackaday.com/2015/07/08/cd-launcher-looks-dangerously-fun/>

当[JZSlenker]被要求找到一种创造性的方法来销毁一堆刻录错误的光盘时，他没有失望。他想出了一个相当简单但有趣的装置，用高速发射 CD。他没有分享这台机器是如何制造的许多细节，但 18 秒的视频让它的工作原理变得非常明显。

CD 枪主要由一块胶合板制成。这提供了一个安装其他部件的平坦底座。一叠光盘被一个看起来像是焊接在一起的金属笼固定住。廉价的角向磨光机被用作推进机构。砂轮垂直安装在 CD 堆的正前方。轮子必须放在胶合板基座上方足够高的地方，这样 CD 才能放在轮子和基座之间。这个设计与我们自己的布莱恩·本考夫正在制造的贴纸枪非常相似。

一些类型的线性致动器被用作发射机构。致动器被连接到一块切成 L 形的薄金属片上。它几乎看起来像一个收割工具。当按下按钮时，致动器立即启动。这将把金属锤推入堆叠底部的 CD 中。CD 被向前压入砂轮，然后砂轮将 CD 射向空中。根据下面的视频，看起来[JZSlenker]可以用这个装备以每秒三张 CD 的速度开火。

这一定是即将上映的电影中的超级反派武器，对吧？也许是 AOL-man？
