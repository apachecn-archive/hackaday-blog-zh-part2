# THP 半决赛:农场机器人

> 原文:[https://hackaday.com/2014/09/19/thp-semifinalist-farmbot/](https://hackaday.com/2014/09/19/thp-semifinalist-farmbot/)

FarmBot 团队一直忙于他们的数控农业和采集机器。这个想法是通过工具的精确部署来自动化耕作过程:犁、种子注射、浇水、传感器等。一个添加了斜坡的 Arduino 处理移动，一个 Raspi 提供互联网连接。他们的原型已经经历了四次大的迭代:第一次修订解决了更大的问题，如框架/轨道稳定性和零件的简化。现在，他们锁定了物联网集成和高级运动功能编码的细节。

最近的升级通过彻底检查工具的实现提供了显著的改进。最初，该团队设想了一种单一的多功能工具头设计，可以随时携带所有东西。问题是，正在使用的工具如果比其他工具低，可能会工作得最好，把它们堆在一起会带来麻烦。解决办法？当然是通用工具安装系统。休息之后，你可以在视频中看到他们在测试他们的设计。

如果 FarmBot 的进展不够令人印象深刻——诚然，我们会说项目负责人[Rory Aronson]试图实现这一目标是疯狂的……但他做到了——farm bot 的工作人员*通过 Kickstarter 启动并成功资助了整个子项目。* [OpenFarm](http://openfarm.cc/) 是一个开源数据库，旨在成为所有农业和园艺方面的权威维基。这是因为[Rory]遇到了大量关于植物种植的普通、拙劣的建议，所以他就众包了一个解决方案。你知道，没问题。
