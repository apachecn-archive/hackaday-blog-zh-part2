# 使用牙签和镊子填充 SMD 板

> 原文:[https://hack aday . com/2013/02/01/populate-SMD-boads-using-a-a-牙签和镊子/](https://hackaday.com/2013/02/01/populate-smd-boads-using-a-toothpick-and-tweezers/)

这里有一个演示，证明了[你真的不需要特殊的工具来填充表面贴装 PCB](https://www.youtube.com/watch?v=h-NG8Vk1aC0) 。我们以前见过这个板，它是由[开发和展示的连接真实世界和《我的世界》](http://hackaday.com/2012/11/16/avr-minecraft-server-lets-you-toggle-pins-from-the-virtual-world/)的玻璃 PCB 服务器。这是一个很小的电路板，我们很高兴有机会看到他在回流焊接前填充零件的方法。

在休息之后的视频中，[Cnlohr]开始分发从其存储容器中粘贴的一团焊料。他提到，只要你把东西放在冰箱里，就很容易处理。因为他的大多数项目都是单板，所以不值得制作焊接模板。取而代之的是，他用牙签挑一点焊料滴，涂在每个焊盘上。

这并不像听起来那么糟糕。精细间距的 TQFP 脚印可以用一点浆糊拖出来。这一过程持续了大约 7 分钟，之后他拿起一些镊子(不是真空型的)开始放置每个元件。如果他漏了一些浆糊，他会在这一步发现，并在必要的地方添加。最后一步是在他的烤箱里走一趟。
