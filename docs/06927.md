# THP 半决赛:激光焊膏

> 原文:[https://hack aday . com/2014/08/27/THP-semi finalst-laser-solder-paste/](https://hackaday.com/2014/08/27/thp-semifinalst-laser-solder-paste/)

![laser](../Images/c01389497f8d4c84c0c70f8c4ff84557.png)

作为 Hackaday 奖的相对后来者，[AltMarcxs]仍然提出了一种非常有趣的制造工具，这种工具以前从未有人见过。这是[一个旋转激光焊膏涂抹器](http://hackaday.io/project/2017)，是数控机床的附件。它是做什么的？目前，它看起来非常酷，同时也是[AltMarcxs]的一个巨大的时间接收器，但潜力远不止于此，从也可以分配锡膏的拾取和放置机器，到最接近碳纤维打印机的东西。

[AltMarcxs]的构建由两个 3W 激光二极管组成，聚焦在注射器尖端的正上方。注射器分配锡膏，旋转二极管，[Alt]能够将熔化的焊料滴放在一块穿孔板上的任何地方。他上传了一个相当聚焦的视频来证明这一点。

随着一些自制的拾取和放置机器进入 Hackaday 奖的半决赛，很容易看到这样的东西的效用:将电路板放入机器，按下按钮，等待一段时间，等待完全填充和焊接的电路板是电子爱好者的梦想，只有在家制作 PCB 的廉价而简单的方法才能与之匹敌。[AltMarxcs]的机器可能是实现这一目标的一个步骤，但他还想先探索一些其他想法。

这个建筑也有送丝机，允许一点铜线焊接到新形成的金属块上。有计划用复合纤维取代它，用 UV 树脂取代注射器中的糊状物，用激光切割纤维并固化树脂，并建造比我们之前见过的其他碳纤维 3D 打印机好得多的东西。
