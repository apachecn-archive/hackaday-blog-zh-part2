# 在自动取款机上玩末日游戏

> 原文:[https://hackaday.com/2014/07/26/playing-doom-on-an-atm/](https://hackaday.com/2014/07/26/playing-doom-on-an-atm/)

关于这个黑客没有太多的细节，但我们仍然认为它足够有趣，值得分享。YouTube 用户[Aussie50]似乎已经想出了一种在自动取款机上安装末日的方法。不仅是系统在运行软件，他们似乎还在使用 ATM 机的内置按钮来控制游戏中的动作。

今天的许多 ATM 机只是运行 Windows 版本的计算机，所以人们会认为在硬件上运行像 DOOM 这样的老游戏应该不会太难。在视频的开头，你可以很快看到一个默认的 Windows XP 背景屏幕。在后面的视频中，您可以看到[Aussie50]进入了一个似乎是 MS-DOS 命令行的地方。按理说，这种特定型号的自动柜员机确实可以在 Windows XP 上运行，但[Aussie50]可能还必须安装 MS-DOS 仿真软件，如 DOSBOX。

在视频的某一点上，摄影师提到他们正在使用一台 [I-PAC2](http://www.ultimarc.com/ipac1.html "I-PAC2") 。一些研究将向您展示这个小 PCB 是为街机游戏设计的 USB 键盘仿真。看起来你只需连接一些简单的瞬时开关，I-PAC2 就会将其转化为 USB 键盘命令。因此，很可能[Aussie50]将 ATM 的按钮直接连接到 I-PAC2 板上，完全绕过了原来的按钮控制器电路。

视频中还提到[Aussie50]能够让收据打印机工作。将这一点融入末日游戏会很有趣。想象一下，收到一张印有你的高分的收据。这也让我们思考在 ATM 硬件上进行游戏的其他可能性。你能把游戏配置成需要押金才能玩吗？如果你赢了高分，你能配置它分发现金吗？如果你修改了多人死亡竞赛模式，让所有玩家都必须支付入场费，而赢家拿走全部，会怎么样？对于在 ATM 硬件上玩游戏，你有什么创造性的想法？
