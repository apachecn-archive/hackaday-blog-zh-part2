# 行星齿轮已经不存在了

> 原文:[https://hackaday . com/2015/01/13/retro tech atacular-planetary-gears-is-out-of-this-world/](https://hackaday.com/2015/01/13/retrotechtacular-planetary-gears-are-out-of-this-world/)

如果你曾经去过创客集会，仔细阅读过 Thingiverse，或者在 3D 打印机 100 英尺范围内，你可能见过一套带小曲柄的塑料行星齿轮，或者一套双曲线行星齿轮。除了看起来很酷和开始对话之外，这些东西没有太大的用途，但如果你打开卷笔刀或汽车变速箱，你会发现它们肯定有。“但是，黑客日，”你会问，“它们是如何工作的？”。再一次，美国陆军信息团在这里用他们无与伦比的严肃、中世纪的黑白风格告诉你一部 1953 年的训练电影。

我们已经可以听到你喊犯规:‘第二部分？！“你当然开玩笑，哈克达。”不，不要叫我们雪莉。要么第一部分永远消失在时间的史册中，要么它是如此壮观，以至于不能向公众公布。甚至[杰夫·奎特尼]也没有发现它。但是第一部分是重要的，除了概述第二部分回顾的行星齿轮系统的五个支配法则之外，可能还解释了什么是行星齿轮。

行星齿轮或称*周转齿轮*是根据古希腊人观察到的太阳系五大行星的原始用途命名的。在其最基本的形式中，行星齿轮系统由位于中心的太阳轮、与太阳轮啮合并围绕太阳轮旋转的行星齿轮以及与行星齿轮啮合的外齿圈或*齿圈*组成。这些组件之间的交互在[这个动画](http://bl.ocks.org/mbostock/1353700)中很容易看到。更改单选按钮会将该部分固定在适当的位置，您可以观察这会如何影响其他组件。

本周的电影解释了不同大小的多组行星齿轮系统被用来为汽车提供不同的前进速度和倒车操作。这种传动是通过将一台或多台机组置于减速模式来实现的。

反向的情况是一个有趣的问题，在概述中解释一次，然后再更仔细地解释一次。为了节省空间和重量，使用了复合齿轮系统。如果你能意识到将系统的一部分牢牢固定在适当的位置会改变其他部分的行为，那么你就很好地理解了复合行星齿轮系统向车轮发送反向运动的方法。

本质上，中间齿轮系统的环形齿轮(即环绕外侧的大齿轮)通过空心轴连接到倒档单元的太阳齿轮。插入空心轴中的第二轴将中间系统的行星架连接到倒档系统的行星架上。在这种布置中，输出轴成为反作用轴。当中间系统的行星齿轮保持反作用时，其系统的齿圈以与太阳齿轮相反的方向运动。这是反向运动开始的地方。

如果倒档系统的齿圈用离合器固定，那么它就成了反作用齿轮。因此，倒档系统的行星齿轮必须跟随其太阳齿轮的运动。这种来自中间系统的反向运动通过倒档齿轮系统传递到车轮上。当车辆从倒档换出时，倒档系统将被取消。一旦前进档接合，倒档系统将作为一个传递系统，将前进运动传递到中间齿轮系统提供的任何档位。

电影的其余部分处理双重输入对系统的影响。也就是说，当齿圈给定恒定的输入速度而太阳轮给定不同的输入速度时，输出端会发生什么。系统的输出将总是在齿圈上的固定速度和给予太阳轮的任何输入之间。它永远不会超过齿圈的速度。如果太阳齿轮以与齿圈相反的方向运转，则行星齿轮和输出轴将跟随齿数最多的部件的方向，该部件始终是齿圈。

但是，如果太阳齿轮的转速增加到与齿圈的转速相匹配，输出轴将完全停止。如果太阳齿轮的转速超过这个临界点，输出轴将再次向相反方向移动。这一原则对于允许坦克原地做油炸圈饼的差动系统至关重要。
