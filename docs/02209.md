# 耳机灯光秀

> 原文:[https://hackaday.com/2012/11/28/headphone-light-show/](https://hackaday.com/2012/11/28/headphone-light-show/)

![](../Images/fbd54ff594b6d3c35607a7a05c66b1b4.png "el-wire-headphones")

说真的，没人说“看着我！”比如这些耳机。【亚德利·多邦】[在他的耳机](http://www.instructables.com/id/LEVEL-HEADED-EL-WIRE-HEADPHONES/)上加了一道彩虹色的电致发光线，让它们随着音乐跳动。休息后的视频显示耳机随着音乐颠簸。这是该项目的两个版本之一，另一个沿着耳机线本身运行 EL 线。我们有点惊讶的是，并行运行的高频并没有给信号注入噪声。我们确实喜欢看到这些活动，但不幸的是，实际上观察者听不到灯光跳动的曲调。

我们花了一点时间才发现(亚德利)没有推出自己的 VU 硬件。驱动 EL 线的逆变器被设计成随着音乐跳动。但是他确实使用了音频线路而不是麦克风。他提到这还有其他用途，比如让精心制作的声音片段精确控制逆变器。
