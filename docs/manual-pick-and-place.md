# 手动拾取和放置

> 原文：<https://hackaday.com/2014/03/13/manual-pick-and-place/>

组装大型表面贴装 PCB 可能需要很长时间。来自 Freeside Atlanta 的[craftycoder]制造了一台非常漂亮的[手动拾取和放置机器](http://blog.freesideatlanta.org/2014/03/manual-pick-and-place-project.html)，不再需要镊子。当你试图把无源器件放在你的 PCB 上时，它们不会再粘在你的镊子上了！

在过去的几年里，我们已经看到了[的](http://hackaday.com/2012/11/20/pick-and-place-that-cant-pick-or-place-but-it-looks-very-promising/)和[的](http://hackaday.com/2012/10/29/diy-pick-and-place-builds-boards-is-awesome/)和[的](http://hackaday.com/2013/07/22/diy-pick-and-place-just-getting-under-way/)批机器。让这款车脱颖而出的是它的简单和实用的构造。这款拾放器建立在 MDF 平台上，使用亚马逊的轴承、标准 12 毫米导轨，并配有一个小型摄像头，可近距离观察您的零件放置情况。当然，这是一种手工方法，但它比用镊子费力地放置每个部分要好。这将是有趣的，看看这整个建设成本；我们预计它不会太贵。休息之后，在视频中看看这个东西的运行情况。

我们希望这个项目已经激发了你去做一些酷的东西！如果有，让我们知道你做了什么！
