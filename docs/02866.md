# 给你的 GQ 服饰添加一些动画效果

> 原文:[https://hackaday . com/2013/02/27/add-some-animated-bling-to-your-GQ-duds/](https://hackaday.com/2013/02/27/add-some-animated-bling-to-your-gq-duds/)

![geeky-tie-uses-animated-leds](../Images/5b014428ee282a7b3a3480755c15528c.png)

这条领带变成了 VU 米让我们不禁要问:在交谈中，有人能看着你的眼睛吗？它使用集成的麦克风和微控制器，使 RGB LEDs 制成的单列显示器移动到环境声音。

不难猜测这个项目是[Becky Stern]的另一个构建。她最近很火，给了[发光的橄榄球头盔](http://hackaday.com/2013/02/02/glowing-super-bowl-helmets/)和[转向灯背包](http://hackaday.com/2013/02/14/brake-light-backpack-overpowered-with-led-pixels/)。它使用与后者相同的元件系列。一个 Flora board 把一只 Arduino 带到了派对上。它驱动 16 个 RGB LED 像素，这些像素使用单线协议寻址。声音通过麦克风和放大器分线板测量。

由于硬件妨碍了全温莎，该项目使用的领带是使用 velcro 的分离版本。但是，因为你需要针和(导电)线来缝合组件，所以改变任何领带来执行这样的操作并不困难。
