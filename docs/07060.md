# Flir 轻子的分支板

> 原文:[https://hack aday . com/2014/09/13/a-breakout-board-for-a-FLIR-lepton/](https://hackaday.com/2014/09/13/a-breakout-board-for-a-flir-lepton/)

热成像相机现在风靡一时，最好的红外相机之一是 Flir 的轻子模块。这是 FLIR ONE 中的传感器，这是 iPhone 的热成像相机附件。有点令人惊讶的是，Flir 允许任何人购买这个模块，这意味着一大堆机器人和其他各种电子项目。这里是 Flir 轻子的分线板。

电子工匠[Mike]最近得到了一台 FLIR ONE，并做了他最擅长的事情，把它拆开，造出了世界上最小的热成像相机。与专业型号相比，分辨率不是很高，但这个模块只需大约 250 美元。试着找一个更便宜的分辨率更高的热像仪。

有了这个分线板，你显然需要一个轻子模块。现在有一个团购正在进行中，每个模块的价格不到 260 美元。

轻子模块由 I2C 控制，但实际捕捉图像的过程是通过 SPI 进行的。这些图像有点太大，除了最强大的 Arduinos 之外，其他的都无法处理，但是如果你想用 Raspi、BeagleBone 或更大的 ARM 板制作 *Predator* 视觉，这就是入场券。
