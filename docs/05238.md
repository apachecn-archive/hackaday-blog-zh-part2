# 卡萨帕:扩充池

> 原文:[https://hackaday.com/2014/02/05/cassapa-augmented-pool/](https://hackaday.com/2014/02/05/cassapa-augmented-pool/)

![cassapa_4](../Images/995822a5d37846c0b4af984e04d162b0.png)

不擅长打台球？别害怕，卡萨帕在这里！[Alex Porto]已经创建了一个增强现实系统来玩台球，这意味着几乎任何人都可以制作那些很酷的特技镜头！

什么？卡萨帕(caapa)是一个葡萄牙语单词，意思是台球桌口袋。该软件通过在台球桌正上方放置一个网络摄像头进行图像识别。专用软件解释图像并识别球洞、边界、球和球杆的位置，然后可以用于计算游戏物理。然后，投影仪会投射出预测的物理效果，并允许您进行实时更新的微小调整，以拍摄出完美的照片。

不幸的是，让一个大投影仪照在你的台球桌上不会让任何人相信你真的擅长台球。不过，如果你能把它和谷歌眼镜或其他增强视力的护目镜结合起来，那就太酷了。好吧，你仍然是一个非常不诚实的骗子——但无论如何，休息后看看视频。
