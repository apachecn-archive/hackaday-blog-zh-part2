# 3D 打印 RGB LED 手环

> 原文:[https://hack aday . com/2014/02/23/3d-printed-RGB-led-手镯/](https://hackaday.com/2014/02/23/3d-printed-rgb-led-bracelet/)

![3dprintedrgbbraclet](../Images/6e53ae5dd416fec52497cb4d51a5f483.png)

[Marcus 的] [3D 打印 LED 手环](http://www.slickstreamer.info/2014/02/led-bracelet-accelerometer-3dprinting.html)最近经历了多次修改，但每次迭代在简单性和功能性方面都令人印象深刻。受到实验他的[nervous system]的[Diagrid 手镯](http://www.thingiverse.com/thing:7045)的启发，[Marcus]借此机会在他的第一个版本中添加了一些 led，它结合了一条 RGB LEDs，一个小电池和一个[Adafruit 小饰品](http://www.adafruit.com/products/1501)微控制器。

随后很快又推出了第二款产品,这款产品彻底改变了手环的设计，使其更加坚固，并通过升级到不同的条带，使 led 数量增加了一倍。这款手环目前处于第三个版本，一次充电可循环使用约 3.5 小时。这个建筑还配备了一个[三轴加速度计](http://www.adafruit.com/products/1247):当佩戴者摇动手镯时，颜色会跳跃。如果摇晃的时间足够长，手镯会闪烁出令人眼花缭乱的颜色。休息后留下来看几个演示视频。如果你想打印你自己的，前往[马库斯的] [Thingiverse 文件](http://www.thingiverse.com/thing:248567)。
