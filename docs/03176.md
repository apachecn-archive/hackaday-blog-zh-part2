# 黑客空间安全系统带来了 RFID、视频反馈和自动门

> 原文:[https://hack aday . com/2013/04/17/hacker space-security-system-brings-RFID-video-feedback-and-automatic-doors/](https://hackaday.com/2013/04/17/hackerspace-security-system-brings-rfid-video-feedback-and-automatic-doors/)

![rfid-hackerspace-door-lock](../Images/e19a8f65a1e7841404efe8b2ff0a3151.png)

[威尔]一直在努力为他的黑客空间的 RFID 门锁开发一个替代系统。最初的处理器已经有几年的历史了，他决定升级到一个更强大的处理器，并在此过程中添加一些附加功能。

上面看到的控制箱是系统的外部组件。这是一个电话服务箱，就像你在美国大多数房子后面看到的那样。他们有一些这样的东西放在周围，它们是一个完美的选择，因为…嗯…它们应该是不怕风吹雨打的锁定外壳。[威尔]从过去三年运行锁的 Arduino 跳到树莓 Pi 板。这给了他很多额外的工作动力，他利用这一点，增加了一个车辆备用液晶显示屏，用于视觉反馈。你可以看到它给出了他在测试期间使用的“访问授权”消息，但休息后的演示视频显示，他们计划做一些图像脚本，以显示每当读取标签时 RFID 标签所有者的头像。

还有其他几个功能也包括在内。每当标签被读取时，该系统就会发出推文，帮助会员跟踪现在谁在该空间逗留。它还可以安装在一个滑动门上，其中一个成员使用车库门开启器马达实现了自动化。
