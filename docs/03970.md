# 又一辆自动平衡独轮车

> 原文:[https://hack aday . com/2013/08/31/又一个-自平衡-独轮车/](https://hackaday.com/2013/08/31/yet-another-self-balancing-unicycle/)

如今，没有人有时间磨练自己的平衡技能，如果建造自己的赛格威还不能让你感到足够的惊喜，那么来自[Scitech]的家伙们的[自平衡独轮车](http://www.instructables.com/id/Self-Balancing-Unicycle/?ALLSTEPS)应该可以了。他们的建筑是链条驱动的，使用容易找到的回收剃刀滑板车零件。再加上一个电机控制器，5 自由度惯性测量单元和一些电池，它几乎已经准备好以潮人的方式烧掉人行道了。

我们看到的一些以前的独轮车有点笨重，但[Scitech]自行车的目标是简化其方管钢架和脚踏板。像往常一样，像这样的独轮车需要骑手付出一些努力:转移你的体重来控制转向和油门。[Scitech]一伙还发现，通常最好不要意外地将电机反向连接到控制器上。我们建议您在休息后找到一个头盔并观看视频。

对于独轮车来说太酷的黑客们可以建造一个[危险快速的电子滑板](http://hackaday.com/2013/08/10/diy-23mph-electric-skateboard/)来代替。
