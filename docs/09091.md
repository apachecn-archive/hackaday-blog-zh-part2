# 准将 64 模块使一个移动计算机

> 原文:[https://hack aday . com/2015/05/22/commodore-64-MODS-make-it-a-mobile-computer/](https://hackaday.com/2015/05/22/commodore-64-mods-make-it-a-mobile-computer/)

一些 C64 准将的拥有者和爱好者一直在修补他们珍贵的单位，不断增加升级。[wpqrek]对他的 C64 的最新[升级使其完全便携](http://wpqrek.blogspot.com/2015/02/my-c64-use-12v-dc-battery-power-source.html)-他增加了 DC-DC 转换器，使其可以使用外部电池电源。

他在外壳内安装了两个独立的 DC-DC 转换器，一个用于 5V，另一个用于 9V。他选择了这些高效转换器，因为他计划使用电池为设备供电，并希望最大限度地提取果汁。他连接了一个桶形插孔插座以接受 12V 输入，另一个 XT60 插座可以连接一个 LiPo 电池。一个普通的 2200mAh RC 电池就足以为他的 C64 供电 1.5 小时。为了确保 LiPo 电池不会完全放电，他添加了一个简单的蜂鸣器电路，在 3.3V 左右开始发出蜂鸣声。

仅仅添加一个外部电池如何帮助它变得便携？嗯，他已经添加了一个小的 [LCD 显示屏和几个其他的模块，我们在之前的帖子](http://wpqrek.blogspot.in/2011/09/my-c64-mods.html)中已经提到过了。这些早期的模型并没有使单位真正便携。添加最新的黑客。休息之后请看视频。
