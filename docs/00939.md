# 数控禅宗园艺

> 原文:[https://hackaday.com/2012/07/06/cnc-zen-gardening/](https://hackaday.com/2012/07/06/cnc-zen-gardening/)

![](../Images/a97cc0dc1680bbcd7bac8010f903cc93.png "electronic-zen-gardening")

马里兰州巴尔的摩的哈佛黑客空间刚刚公开了他们为红牛创作大赛建造的禅宗花园。这是一个数控创造，将有助于缓解你对 DIY 3d 打印机的挫折感，因为你似乎无法正确校准。

在硬件方面，机器的底座是一个沙箱。找到介质的正确颗粒尺寸是构建过程中最困难的部分之一。使用 CNC 构建中常见的龙门架沿三个轴驱动触针。滑轮和一些支架是 3d 打印的，其余的支架是用激光从木头上切割下来的。Bullduino 通过步进电机控制板控制触控笔，并通过一组 MOSFETs 驱动 led。还包括限位开关，以确保错误不会导致设备损坏。

休息后，你可以看到一个构建蒙太奇放在一个最伟大的 8 位游戏配乐的所有时间。我们希望他们已经展示的一件事是负责“擦除”花园的内置水平条。

更新:哈佛黑客空间的成员带来了一段新视频，展示了“擦除”过程。休息过后你会发现的。
