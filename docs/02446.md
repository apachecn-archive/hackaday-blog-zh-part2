# 莫尔斯电码花是想告诉你什么

> 原文:[https://hack aday . com/2013/01/02/morse-code-flower-is-trying-to-tell-you-something/](https://hackaday.com/2013/01/02/morse-code-flower-is-trying-to-tell-you-something/)

![morse-code-flower](../Images/d943982b0e123e9d0285cec967cfa681.png)

对于不经意的观察者来说，这朵花看起来很漂亮，因为它被照亮的中心在渐隐。但是那道光有隐藏的意义。有些眨眼的时间比其他的长；[这朵花正在使用莫尔斯电码](http://renaud.schleck.free.fr/fleur_lumi.php)。

[Renaud Schleck]想用他的 MSP430 微控制器尝试一些不同的东西。他选择了一款看起来像花的 LED 灯，因为它可以作为家居装饰的一部分。为了添加莫尔斯电码信息，他想要比纯粹的数字闪烁更有说服力的东西(而不是分散注意力)。因此，他将硬编码信息中的点和破折号用脉冲宽度调制法转换成衰减信号。

他对电路进行了自由变形，这样它和为它供电的硬币电池就能放入花盆。簧片开关负责打开和关闭果汁。当放在磁铁附近时，花朵开始轻柔地播放。
