# 开源机械臂现在触手可及

> 原文：<https://hackaday.com/2015/03/14/open-source-robotic-arm-now-within-reach/>

对于任何一个寻找工业过程自动化、教育或大型焊接项目的巨大帮手的人来说，大多数可用的选项可以轻松地打破银行。[Mads Hobye]和 FabLab RUC 的其他人已经解决了这个问题，并提出了一个非常强大、廉价、开源的工业手臂机器人，任何人都可以轻松制造。

机器人本身是基于 Arduino 的，并且可以选择连接各种过程可能需要的任何末端执行器。项目网站上提供了所有器件的原理图以及所有 Arduino 源代码。[Mads Hobye]指出，他们在三天的冲刺中制作了这个机器人，所以它应该不会花很长时间来让你自己启动和运行。甚至还有一个虚拟机器人，可以下载并使用常规的机器人代码，可以用于测试或简单地感受机器人，而不必建造它。

这是一个很棒的项目，因为它是开源的，所以对学生、小企业和爱好者来说都很棒。连接任何末端执行器的选项也是一个额外的好处，我们可能会建议尝试一下【耶鲁】的[腱驱动机械手](http://hackaday.com/tag/robotic-hands/)。休息之后，请观看这个令人敬畏的机器人的视频。
