# 蒸汽动力机械车间

> 原文:[https://hackaday.com/2015/08/19/steam-powered-machine-shop/](https://hackaday.com/2015/08/19/steam-powered-machine-shop/)

有时很难相信一百多年前当电还没有普及的时候，东西是如何制造出来的。为工具提供动力的最常见方式之一是通过皮带传动——由水车、蒸汽机甚至马提供动力。[大卫·理查兹]花了很多时间制作他自己的[时期精确的蒸汽动力机器车间](https://www.youtube.com/watch?v=9WXHNBMLZZM)——这太棒了。

它大致代表了 20 世纪 20 年代美国机械车间的样子。没有一个工具比 1925 年更新。整个商店由一个使用蒸汽动力的轴驱动。大约在 1895 年，一个巨大的锅炉为宾夕法尼亚州建造的 5 乘 5 蒸汽机提供蒸汽。使用皮带和离合器，它驱动几台车床、钻床、一台铣床，甚至一台牛头刨床——一台和爱迪生博物馆里的一模一样的机器！

看到过去的东西是如何制作的，看到[大卫]为建造这家商店所付出的努力，真的令人难以置信。他定期在他的频道[下发布视频，我们强烈建议你去看看。](https://www.youtube.com/channel/UCBdj-vOveiEFWe3vnGoJUag)
