# 三个烙铁控制器的故事

> 原文:[https://hack aday . com/2015/09/02/a-tale-of-three-soldering-iron-controllers/](https://hackaday.com/2015/09/02/a-tale-of-three-soldering-iron-controllers/)

[ZL2PD]需要更换一个旧的 Weller 焊台，并决定不使用互联网上随处可见的廉价焊台。他有一个很长的故事，关于为什么他必须设计自己的控制器，但你从来没有向我们解释。他对他的旅程做了详细的记录，最后，在选定一个之前，他制造了三种不同的控制器。

他从使用热敏电阻测量温度的 Hakko 手机开始。控制器的第一次迭代有模拟控制。他对设计中的零件数量和简单的 LED 显示屏不满意。这使他用一个 ATTiny CPU 和一个串行 LCD 代替了控制器。

第三个也是最后一个版本仍然使用 CPU。[ZL2PD]意识到 LCD 显示器不是很有用，所以他回到 led，并创建了一个 PCB。最后满意了，他创造了一个 3D 打印的外壳，一个艺术的刻度盘，并清理了一个工具架。结果看起来很好，用你自己设计的工作站焊接一定会有令人满意的地方。

在我们看来，他所剩下的就是开一个通风橱。或者也许他会造一个[拆焊同伴](http://hackaday.com/2013/11/11/build-your-own-desoldering-station-on-the-cheap/)。如果你想复制他的结果，请注意一些熨斗使用不同类型的温度传感器。或者你可能只是想花 16 美元买一个便宜的电台(见下面的视频),然后从事其他项目。我们不会说的。
