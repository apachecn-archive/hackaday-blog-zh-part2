# 将塑料牛奶壶变成有用的工具

> 原文：<https://hackaday.com/2015/02/21/turning-plastic-milk-jugs-into-a-useful-tool/>

[Peter]显然喜欢去他的木材店工作。他也喜欢把东西变成别的东西。在他的最新项目中，他将塑料牛奶壶变成了塑料木工槌。

[Peter]从收集和“加工”牛奶罐开始。牛奶壶通常由高密度聚乙烯制成。HDPE 是一种石油基塑料，具有很高的强度密度比。它易于回收，非常适合这类项目。过去我们甚至看到过这种材料被回收到 3D 打印机的灯丝中。“加工”程序实际上只是用刀片将罐子切开。[Peter]提到过去他曾使用搅拌机成功地做到了这一点，但不幸的是，他被禁止使用搅拌机。

接下来，所有的塑料片都堆在一个金属上，放入一个小烤箱。它们被熔化成一个相对扁平的固体块。这个过程执行三次。最后一步是将三大块堆叠在一起，将它们融化成一大块塑料。

在等待塑料熔化的同时，[彼得]开始着手处理把手。他很好地利用了自己的木工技术，用一块樱桃木雕刻出了一个漂亮的木柄。手柄被精心塑造，并用各种工具打磨。最后涂上一些亚麻籽油，看起来很专业。

当塑料大部分熔化在一起时，[彼得]必须在塑料还很软的时候迅速开始工作。他把塑料从金属托盘上撬下来，塞进一个用纤维板制成的长方形模具中。当他把塑料全部塞进模具时，他用热风枪根据需要软化塑料。随着模具适当填充，他关闭它，夹紧它关闭。

一旦塑料冷却，[彼得]必须将其切割成正确的形状和大小。他把塑料块带到带锯前，切割出所有合适的角度。然后，他用钻床和凿子来切割手柄的矩形安装孔。然后用砂带磨光机将塑料片加工成最终形状。所有[彼得]剩下要做的就是向上滑动它，只有手柄。手柄和安装孔的形状防止塑料片从手柄顶部飞出。看看下面的视频，看看整个过程。
