# 一架遥控的，摇摆翼 F-14

> 原文:[https://hack aday . com/2014/11/02/a-remote-control-swing-wing-f-14/](https://hackaday.com/2014/11/02/a-remote-control-swing-wing-f-14/)

任何人都可以用一片泡沫、一些胶水和一些伺服系统制造一架遥控飞机。建造一架 F-14，配有摇摆翼装置？[托马斯] [发现建造了一个会带你进入危险区域的](http://hackaday.io/project/1691-f-14-foamie-with-auto-stabilization)。

这是[托马斯]第一次从头开始建造遥控飞机，并希望里面有很多电子设备。他选择的机身是古老的 F-14 雄猫，配有机翼，着陆时可以向外摆动，高速飞行时可以向内摆动。这也不仅仅是将现成的接收器放入花哨的机身中:[托马斯]正在用一个小电子板读取来自接收器的 PWM 信号，将升降压器与他自己的代码混合，并使用加速度计实现自动稳定系统。

机体的大部分工作由 RCGroups 论坛上的[【Maybz】完成。这是一个跨越七年帖子的令人印象深刻的线索。[托马斯]不认为他的 F-14 是一个最终目标，虽然:他用这作为一个垫脚石来学习建立一个更复杂的无人机不稳定的飞机。](http://www.rcgroups.com/forums/showthread.php?t=769451)
