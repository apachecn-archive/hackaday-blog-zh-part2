# 令人印象深刻的国产赛格威是真正的交易

> 原文:[https://hack aday . com/2014/08/25/impressive-manufactured-Segway-is-the-real-deal/](https://hackaday.com/2014/08/25/impressive-homemade-segway-is-the-real-deal/)

[Kristian]刚刚完成了他的全尺寸赛格威[从头开始。](http://blog.tkjelectronics.dk/2014/07/full-size-diy-balancing-robot/)

早在 2012 年，他利用陀螺仪传感器和 PID 控制器制作了一个小型平衡机器人——你可以在这里看到[原文。](http://hackaday.com/2012/03/07/short-and-squat-balancing-bot-is-extremely-stable/)最酷的事情是，他基本上只是扩大了他的原始项目，创造了这个全尺寸的赛格威！

它使用两个 500W 24V DC 电机(MY1929Z2)安装在铝制检查板框架上，其余结构由钢制管道和配件制成。我们真正喜欢的是转向联动装置；类似于真正的赛格威，你把手柄往你想转的方向拉。他通过将另一段平行于车轮的管道通过弯管接头连接到车把上来实现这一点。它由两个轴台轴承支撑，背面是一个固定的电位计——当你向一个方向倾斜手柄时，管道会旋转，旋转电位计。为了让它回到中间位置，他在两边都加了弹簧。
