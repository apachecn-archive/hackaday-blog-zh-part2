# 用电解法制作的镀金凯蒂猫斧头

> 原文:[https://hack aday . com/2012/08/29/gilded-hello-kitty-ax-made-using-electrolysis/](https://hackaday.com/2012/08/29/gilded-hello-kitty-ax-made-using-electrolysis/)

![](../Images/fbe6b26c7ef7944f4a8d1a31006c42fb.png "gilded-hello-kitty-ax")

嗯……可爱？为了那个拥有一切的凯蒂猫粉丝？

是啊，我们真的不知道这是怎么回事。这是一件[丹尼斯]花了很多时间的艺术品。他擦亮它，蚀刻它，画它，应用金叶，并且为一个符咒钻一个洞。我们对那个洞最感兴趣。他花了很长时间才想出如何在坚硬的钢板上钻孔。最终奏效的解决方案是[用电解法钻穿金属](http://hrenovina.net/10488) ( [翻译](http://translate.google.com/translate?sl=auto&tl=en&js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&u=http%3A%2F%2Fhrenovina.net%2F10488))。

[丹尼斯]首先在他想挖洞的地方挖一个小坑。这需要时间，而且很大程度上侵蚀了他正在使用的雕刻钻头。但只够装一滴盐水。他将电源的正极连接到斧头上，然后将负极浸入盐水中。每次液滴变黑的时候，他就把它擦掉，换上新的溶液。就这样过了五个小时，加上一两包棉签，他就完事了。
