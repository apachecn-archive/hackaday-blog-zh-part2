# 简易多点触控桌

> 原文:[https://hackaday.com/2014/01/01/easy-multi-touch-table/](https://hackaday.com/2014/01/01/easy-multi-touch-table/)

[![Screen Shot 2013-12-31 at 1.53.13 PM](../Images/93ca64639d225683fe2cfc78797f66f3.png)T2】](http://hackaday.com/wp-content/uploads/2013/12/screen-shot-2013-12-31-at-1-53-13-pm.png)

[2bigbros]在他的多点触控桌面系统上安装了一个仪器。这是一个很好的设置，使用典型的[受抑全内反射](http://cs.nyu.edu/~jhan/ftirsense/)方法进行触摸感应。 [Tinkerman 的方法](http://nuigroup.com/forums/viewthread/2383/)被用于屏幕本身，包括用油漆滚筒将硅滚动到羊皮纸上以提高粘合性。[2bigbros]然后为整件东西建造了一个漂亮的铝制和木制框架。他对一些细节轻描淡写，但大多数有基本了解的人和谷歌都能够搞清楚。

对于大多数建筑商来说，这是一个非常容易理解的项目。如果你有兴趣进入其中，有大量的项目可以参考。我们之前的[涵盖了基础](http://hackaday.com/2011/04/16/the-basics-of-building-a-multitouch-table/)，以及一个[更复杂的构建](http://hackaday.com/2013/09/29/50-multitouch-table-is-expensive-indestructable/)。我们甚至看到了一款使用多点触控的[互动塔防游戏](http://hackaday.com/2011/08/31/multitouch-tower-defense-uses-physical-towers/)。如果您决定自己构建一个，不要忘记 TUIO 上的[优秀资源，用于查找框架和示例实现。](http://www.tuio.org/?software)
