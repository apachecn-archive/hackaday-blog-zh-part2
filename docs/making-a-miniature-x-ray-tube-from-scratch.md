# 从头开始制作微型 X 射线管

> 原文：<https://hackaday.com/2012/05/17/making-a-miniature-x-ray-tube-from-scratch/>

[https://www.youtube.com/embed/-0G4-JicCIw?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/-0G4-JicCIw?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

我们知道你们中的大多数人没有理由去制作微型 X 射线管。然而，我们也知道你们中的许多人会像我们一样发现这个视频令人着迷。[Glasslinger]出色地解释了制造迷你 x 射线管的整个过程，从获得铀玻璃和钨杆，从零开始制造灯丝，所有的玻璃工作，以及测试。

诚然，在 hackaday，我们大多数人除了欣赏这种工艺之外不会走得更远，尽管我们很想知道[[Adam Munich](http://hackaday.com/2012/01/13/grenadier-wins-the-laser-cutter-for-his-portable-x-ray-project/)看到这个故事后会说些什么。

如果你喜欢视频中的管道结构，一定要看看[Glasslinger 的]其他视频。他在自己的店里制造各种各样的管子，并且通常在整个过程中分享如此多的信息，以至于每一根管子都有超出特定项目的有用信息。另一个疯狂的部分是，他自己制造了大部分工具，包括他的玻璃车床。

我们真的不应该指出 X 射线是危险的。除非你研究过如何安全地做，否则不要乱用它们。
