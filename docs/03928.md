# Retrotechtacular:塑料 10 周年

> 原文:[https://hackaday . com/2013/08/27/retro tech tacular-the-10 周年的塑料/](https://hackaday.com/2013/08/27/retrotechtacular-the-10-year-anniversary-of-plastic/)

![retrotechtacular-plastic-ten-year-anniversary](../Images/b248d803d24a443314a898d1ccb6482c.png)

这段名为*游行中的工业* 的视频是对塑料黄金时代的独特回顾。我们也珍视这些镜头，因为它让我们看到了美国制造业的鼎盛时期。

我们记得一位中学教师回忆起他的父亲——在陶氏化学公司工作的一名研究科学家——带回家一对光盘给他玩。他第一次接触塑料。这里我们看到了塑料制造成为主流十年后的快照。它从参观注塑工厂开始。上面看到的截图来自第二个短片，它参观了海军舰船模型的生产线，这些模型将用于训练海军人员和作为战略规划地图的道具。这部电影以塑料织物的生产结束，从原材料开始，到合成防虫网结束。

为了证明这是来自过去的真实爆炸，当你得到一个反共产主义的 PSA 时，坚持最后两分钟。经典。
