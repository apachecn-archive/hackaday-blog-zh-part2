# 一个小气泡显示闹钟

> 原文：<https://hackaday.com/2014/06/30/a-tiny-bubble-display-alarm-clock/>

出于这样或那样的原因，我们开始看到许多项目采用一些旧的七段惠普气泡显示器。是的，那些曾经被归入古代电子计算器的显示器正在卷土重来，原因我们不明白，除了有人发现了一堆 NOS 显示器的猜测。[Markus]挑选了一些老式的 tymie 显示器，做了一个非常漂亮的气泡显示闹钟。

为了简单起见，[Markus]没有走通常的 ATMega with RTC 路线。相反，他用一个 MSP430，一个 32kHz 的晶体，和几个按钮来构造这个微型闹钟。它由一节 AAA 电池供电，与专业制作的奇特电路板相比，[Markus]在一些 perfboard 上用一点点漆包线制作了这个。

这是一个整洁的小时钟，由于 MSP430，它有扬声器和最有可能的极端电池寿命，这是一个非常好的便携式，经典外观的闹钟。下面是[马库斯]操纵时间的视频。
