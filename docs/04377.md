# 3D 打印剖面喷气发动机听起来很棒

> 原文：<https://hackaday.com/2013/10/22/3d-printed-cutaway-jet-engine-sounds-great/>

多亏了 3D 打印的奇迹，你现在可以用 3D 打印出你自己的喷气发动机。[与我们之前见过的](http://hackaday.com/2013/10/05/please-dont-build-a-jet-engine-from-a-toilet-paper-holder/)喷气式发动机不同，这种发动机没有操作员被烧焦的可能。[Gerry]自称是来自新西兰的“故障汽车修理工”。他用 3D 软件设计了一个相当棒的喷气式发动机，并用他的 UP Plus 打印机打印出来。发动机本身是一个[高涵道比涡扇发动机](http://en.wikipedia.org/wiki/Turbofan)的剖面模型。虽然我们不确定这个剖面图代表的是哪种型号的喷气发动机，但我们仍然印象深刻。

这不仅仅是一个静态展示模型——发动机将在压缩空气的帮助下实际旋转起来。独立的启动管和运行管分别向涡轮机和主风机输送空气。它甚至有独特的涡轮风扇“嗡嗡声”。虽然这种模型相对安全，但[Gerry]确实警告要保持压力，否则它可能会崩溃。为此，我们建议在快速断开前增加一个调节器。

Thingiverse 项目在指令上有点轻。然而【钢锯】、[在 pp3d](http://www.pp3dp.com/forum/viewtopic.php?f=14&t=22669) 上发布了一张图片和建造日志，纠正了这种情况。[钢锯]确实遇到了一些问题，但没有一点点强力胶不能修复。它可能有更少的运动部件，但这绝对让我们的旧[可见 V8 发动机套件](http://www.amazon.com/dp/B00004YUXS/)相形见绌。
