# 一个开源的 Arduino 吉他踏板

> 原文：<https://hackaday.com/2013/12/21/an-opensource-arduino-guitar-pedal/>

如果你是一个吉他手，或者认识一个吉他手，你可能知道有多少种吉他效果——但是如果你可以自己设计效果会怎么样呢？

[J Rodriguez]刚刚发布了他的开源 Arduino 吉他踏板盾，被称为 pedal shield。他将它设计为一个学习数字信号处理、效果和合成器的平台，而不需要深入的电子或编程知识。它允许你用 C/C++设计你自己的效果，或者从他自己的在线库中下载。一些可下载的预置包括一个八度踏板，混响踏板，延迟踏板，甚至失真踏板！

踏板具有三个可编程电位计、两个主开关和脚踏开关。屏蔽直接插入 Arduino Due，你可以在这里找到所有的原理图和零件清单[在这里。](http://www.electrosmash.com/forum/hardware-pedalshield/17-pedalshield-bill-of-materials-and-alternatives)它完全是在[的开源电子 Cad 设计套件 KiCad](http://www.kicad.org/display/KICAD/KiCad+EDA+Software+Suite) 中设计的。

休息后听一下踏板的动作！
