# Pandaphone 是一款 DIY 婴儿玩具

> 原文:[https://hackaday . com/2014/12/26/panda phone-is-a-DIY-baby-toy/](https://hackaday.com/2014/12/26/pandaphone-is-a-diy-baby-toy/)

[泰勒]正在为他朋友一岁的儿子挑选礼物。在玩具店的货架上搜寻时，[泰勒]意识到这个年龄的孩子的大多数玩具只是会闪光和发出声音的塑料盒。他应该可以相对轻松地制作。在商店里呆了一段时间后，[泰勒]想出了这款手机。

这个围栏是由一块 2×4 的木材制成的。他把那块木头切成三块更薄的木头。顶部有两个孔，允许超声波传感器伸出。中间部分有一个用带锯切割出来的空腔。这将留出空间来存放电子产品。底部的一块作为一个盖子来隐藏内部。

该电路使用 ATtiny85。该程序监视超声波 PING 传感器的距离变化。然后，它会通过一个小扬声器发出声音，这个小扬声器会根据探测到的距离改变音调。结果是，当您的手靠近传感器时，音调较低，而当您的手远离传感器时，音调较高。表壳正面绘有熊猫图案，因此得名“Pandaphone”。根据下面的视频，看起来收件人很享受！
