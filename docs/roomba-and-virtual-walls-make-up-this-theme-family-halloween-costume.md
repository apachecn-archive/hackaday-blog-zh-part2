# Roomba 和虚拟墙组成了这个主题家庭万圣节服装

> 原文：<https://hackaday.com/2012/11/15/roomba-and-virtual-walls-make-up-this-theme-family-halloween-costume/>

这是必然的。你花了大量的时间做一套很酷的服装，却不能让你的孩子摆好姿势拍照。不过没关系，我们还是明白了。这套主题服装将小家伙打扮成 Roomba 吸尘机器人，而妈妈和爸爸则装扮成虚拟墙(用于防止机器人从楼梯上摔下来的模块等)。).这很有趣，也很独特，但如果不是因为一些额外的电子设备，这可能会被归入链接帖子。为了安全起见，每套服装都配有一圈发光二极管。作为一项挑战，[灯光被赋予了相互同步图案的能力](http://rhb.me/2012/11/synchronized-led-ring-construction-details/)。

每件服装的顶部都有一个圆形框架，并附有一组 RGB LED 灯串。为了让它们显示同步图案，设计了一个红外发射器/接收器板，并从 OSHPark 订购。每套服装都有四个这样的模块，所以无论穿着者面对哪里，都不会中断交流。休息之后可以看到同步光环的演示
