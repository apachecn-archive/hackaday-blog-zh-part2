# 带有 Oculus Rift 的 FPV 无人机

> 原文：<https://hackaday.com/2013/07/16/fpv-drones-with-an-oculus-rift/>

这只是时间问题，现在终于有人做到了。Oculus Rift 现在被用于[第一人称视角航拍](http://intuitiveaerial.com/home/2013/7/14/oculus-fpv-first-flight)。这是你站在地面上最接近飞行员座位的感觉。

[Torkel]是直觉航空公司的首席执行官，该公司是大型黑甲无人机的制造商，这是一种为航空摄影设计的六轴直升机。借助 Rift FPV 钻机，无人机向空中携带了一个巨大的有效载荷，包括两个摄像头、一台笔记本电脑和一整套电池。来自这对摄像头的视频在笔记本电脑上进行编码，通过 WiFi 发送到基站，并在 Oculus Rift 上显示。

等待时间大约为 120 毫秒，相当长，但仍然非常适合 FPV 飞行。[Torkel]和他的团队正在研究一种新的硬件迭代，他们希望减少有效载荷质量，增加传输范围，并升级相机和镜头。
