# 用于 8 位 CPU 的多目标 IDE

> 原文:[https://hack aday . com/2014/11/15/multi-target-ide-for-8 位 CPU/](https://hackaday.com/2014/11/15/multi-target-ide-for-8-bit-cpus/)

很久以前，[马丁]玩旧的 8 位计算机。最近，他又在磨练自己的组装技能，他想到了为一船旧系统开发 IDE 的想法。经过一年的工作，他宣布了一个适用于 8 位计算机的多目标集成开发环境，可以在你的浏览器中运行。

这个项目被称为 ASM80 ，它包括一个代码编辑器，一个存放所有代码的工作区，8080/8085、Z80、6502、6800 和 6809 CPUs 的编译器，所有这些 CPU 的仿真器，以及一些捷克计算机、ZX Spectrum 和一些[Grant Searle]单板计算机的仿真器。

这个项目的有趣之处在于，所有不同 CPU 的语法都几乎相同。这是一个真正的模块化代码编辑器，支持宏和你所期望的用于古代计算机的代码编辑器的一切。
