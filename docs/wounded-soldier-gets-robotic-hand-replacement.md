# 受伤的士兵得到机器人手替换

> 原文：<https://hackaday.com/2015/04/12/wounded-soldier-gets-robotic-hand-replacement/>

雪松谷制造商 makerspace 的当地成员 Neal Muzzy 刚刚为他的朋友和受伤的退伍军人[Taylor]制作了一款名为 Dextrus v1.2 的机器人假手，并在开放式仿生学上发布了新闻。

在短短两个月内，[Neal]和他的朋友一起制作了这种机器人假肢，目标是使它比[Taylor]目前的假肢功能更强，更容易使用。第一个原型是通过使用[开源 Dextrus](http://www.instructables.com/id/Dextrus-v11-Robotic-Hand/) 设计制作的，以测试适合度，并使用 EMG 传感器进行控制。一旦他们确定它可以工作——定制！

他们称之为 Dextrus V1.2，它比原来的版本更适合[Taylor]——但这是开放手项目的全部意义——从基础设计开始，并使它变得更好。如果你不熟悉[开放手项目](http://www.openhandproject.org/)，它最初是由[在 Indiegogo](https://www.indiegogo.com/projects/the-open-hand-project-a-low-cost-robotic-hand) 上众筹的，现在是一个让截肢者更容易获得机器人假手的组织。我们在[hack let 41——假肢项目中写到了它。](http://hackaday.com/2015/04/03/hacklet-41-prosthetics-projects/)
