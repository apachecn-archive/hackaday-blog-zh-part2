# E-Ink 显示器将屏幕上的控件移到屏幕外

> 原文:[https://hack aday . com/2015/08/26/e-ink-display-moves-on-screen-controls-off-screen/](https://hackaday.com/2015/08/26/e-ink-display-moves-on-screen-controls-off-screen/)

平板电脑已经走过了漫长的道路。他们似乎终于在市场上找到了自己的位置，现在也许他们可以赶上更传统的电脑。微软应用科学部门为一款新的平板电脑[提出了一个很酷的原型设计，这款平板电脑带有辅助电子墨水输入显示器。](http://www.microsoft.com/appliedsciences/content/projects/DisplayCover.aspx)

平板电脑界面利用了键盘上方的电子墨水条。虽然看起来不多，但这释放了大量屏幕空间，允许您在屏幕外使用各种图标和快捷方式。这对数字艺术家来说非常有意义，因为他们可以在屏幕上画画，而且他们的工具包就在他们的下面，就像真正的绘画一样。

这种东西的另一个伟大用途是签名板——随着一切数字化，你最后一次打印、签署和扫描文件给别人是什么时候？他们甚至开发了一个专用的电子邮件应用程序，你可以在电子墨水屏幕上单独使用，让你最大限度地利用主屏幕进行视频聊天等活动。

演示非常酷，我们经常想知道为什么没有更多的手机集成电子墨水显示屏——这仅仅是开始吗？
