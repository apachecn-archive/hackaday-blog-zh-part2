# 古巴:DIY 社会？

> 原文:[https://hackaday.com/2013/10/10/cuba-a-diy-society/](https://hackaday.com/2013/10/10/cuba-a-diy-society/)

![ScreenShot041](../Images/c59fe71b7e5214b1f17c5904685e2798.png)

美国在 60 年代离开古巴后，大多数工程师都跟他们走了，所以[菲德尔·卡斯特罗]告诉公民们要学会自己制作东西。他们被称为国家创新者和合理化者协会(ANIR)，这正是他们所做的。这是古巴后院革新的开始。

快进几十年，90 年代对古巴来说是非常困难的时期。随着 1991 年苏联的解体，一场严重的经济衰退几乎使这个国家陷入瘫痪，因此古巴的 DIY 文化开始更加繁荣——这是绝对必要的。没有钱，没有进口，只有他们已经有的。制造和修理东西成了生活的一部分，你不能出去买一个解决问题的方法，你必须自己动手。这可能是一个全面的制造商/DIY 社会将会是什么样子的最伟大的例子之一——嗯，也许没有共产主义部分。

休息之后的精彩视频是一个关于设计师[Ernesto Oroza]的小故事，他开始在他的艺术项目下收集这种 DIY 文化的例子，这个艺术项目被恰当地称为“技术不服从”。它值这个表，所以看看吧。
