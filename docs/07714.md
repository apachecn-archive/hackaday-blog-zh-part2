# 木制 LED 矩阵茶几

> 原文:[https://hack aday . com/2014/12/02/a-wood-led-matrix-coffee-table/](https://hackaday.com/2014/12/02/a-wooden-led-matrix-coffee-table/)

[johannes]写道，他制作了一个令人印象深刻的 LED 表。该表基于由 Raspberry Pi 控制的 WS2801 串行可寻址 led。Pi 提供了一个由【安德鲁·芒塞尔】开发的 *node.js* 驱动的网络界面，用于[的房间照明设置](https://www.andrewmunsell.com/blog/raspberry-pi-leds)。web 界面控制显示器上显示的模式和动画速度。

[johannes]在 LED 矩阵周围建造了一个木制咖啡桌，其中包括一个哑光玻璃桌面，以帮助漫射灯光。一个用于插入笔记本电脑的插座和两个 USB 充电端口安装在外壳侧面的面板上，这是一个不错的触摸。led 的电源也在外壳内，不需要外部电源模块。

虽然[johannes]还没有编写自己的软件，但他计划为天气和其他数据添加音乐同步和可视化功能。休息之后，请观看视频，了解表格的运行情况。
