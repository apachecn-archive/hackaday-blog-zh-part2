# 向一位老派金属大师学习机械加工

> 原文：<https://hackaday.com/2013/09/20/learn-machining-from-an-old-school-metal-master/>

在 YouTube 的深处，仍有一些珠宝有待发现。[【基思·芬纳】就是其中的一位](http://www.youtube.com/watch?v=oG7Sv99gqQ4)。[Keith]在马萨诸塞州科德角拥有 Turn Wright 机械厂。在他的小店里，他从事从帆船到重型设备的各种工作。

[Keith]称自己是“一个艺术家，一个做了 36 年这行的批发商”。我们认为他可以把教师加到这个列表中，因为我们已经从他的 Youtube 频道中了解了很多关于机械加工的知识。

[基思]有趣的一点是他在镜头前的表现。他让观众感觉像一个学徒机械师和他一起工作。[Keith]并没有用图形仔细地设置镜头，而是一边工作一边讲述。他也毫不犹豫地向我们展示他的[错误](http://www.youtube.com/watch?v=P521WCTiTnA)——[从错误中恢复](http://www.youtube.com/watch?v=9xRy8wGfcIk)，以及他的[胜利](http://www.youtube.com/watch?v=XVY5JGpd8nc)。

特纳·赖特的主要工具是车床和铣床，但[基思]无论如何也不是老古董。他有一套完整的等离子摄像机系统，并且不怕做一些电脑工作。

大多数(基思的)项目被分成几个视频。我们最喜欢的一个是[“所以你在你的洞里分手了”](http://www.youtube.com/watch?v=7-npwLSJz7o)。在这个系列中,[Keith]展示了如何从一个大型柴油涡轮发动机中取出一个损坏的螺丝拔出器(或 EZ Out)。休息之后，感受一下[Keith]的“生活中的一天”视频。
