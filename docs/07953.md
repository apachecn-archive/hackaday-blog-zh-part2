# 参量球形扬声器不是月亮

> 原文：<https://hackaday.com/2015/01/01/parametric-spherical-speakers-are-not-a-moon/>

一个好的音箱不仅仅是用胶合板建造一个盒子并用地毯覆盖，尽管用 1F 电容器玩也很酷。不，对于一个好的音箱来说，你需要合适的内部容积、合适尺寸的低音端口、合适的扬声器，而且它绝对不能是一个月亮。[Rich]发现他可以用一台 3D 打印机完成所有这些工作，[产生了 NOMOON](http://nothinglabs.blogspot.com/2014/12/project-nomoon-parametric-spherical.html):NOMOON Orbital 音乐制作开源软件，由 Openscad 生成的 Nihilator。

这项工作是早期工作的延续[，设计了博格立方体形状的参数化扬声器。现在[Rich]开始使用 Borg scout ships，这个版本拥有你所期望的扬声器设计的一切。](http://hackaday.com/2014/09/30/3d-printing-of-parameterized-speaker-enclosures/)

NOMOON 在 Thingiverse 定制器上提供[,具有内径、外壳容积(升)、壁厚、扬声器孔、低音端口和线孔等变量。当然，对于普通的 OpenSCAD 安装，定制设计也是可能的。](http://www.thingiverse.com/apps/customizer/run?thing_id=590871)

[Rich]已经打印了一些这样的非月亮，即使有一个低音响应很差的扬声器，就 Youtube 视频而言，他的设置听起来相当不错。你可以看看下面。
