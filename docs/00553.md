# 将一台 1942 年的车床变成一件实用的艺术品

> 原文：<https://hackaday.com/2012/05/28/turning-a-1942-lathe-into-a-functional-piece-of-art/>

几年前，[macona] [买了一台 1943 年的 Monarch 10EE 车床](http://www.practicalmachinist.com/vb/monarch-lathes/my-ee-home-last-warning-tool-porn-inside-99151/)。这台巨大的机器不仅是一件令人惊叹的工程，也是一件令人敬畏的艺术品；这台车床不仅能精确地制造零件，它还是机器时代设计的一件杰作。

君主 10EE 车床在当时是非常高科技的，冷却泵上的陆军部底特律法令区标签承载了这台机器的血统:这台车床最有可能被用来制造非常精确的军事装备，如[诺顿投弹瞄准器](http://en.wikipedia.org/wiki/Norden_bombsight)。

经过 60 年的忠实服务，【macona】的车床捡了几层不同颜色的油漆，普遍陷入失修状态。[macona]花了大量的时间检修这台车床，通过[更换弯曲的进给杆](http://www.practicalmachinist.com/vb/monarch-lathes/ee-feed-rod-replacement-100220/)，[排除电机问题](http://www.practicalmachinist.com/vb/monarch-lathes/mg-motor-timing-fa-relay-adjustment-100146/)，最终用现代交流无刷伺服替换了整个电机[。你可以在休息后的视频中查看交流伺服系统的改进。](http://www.practicalmachinist.com/vb/monarch-lathes/retrofit-10ee-drive-ac-brushless-servo-motor-154967/)

当然，没有几张美照的话，关于改造车床的帖子是不完整的。我们非常感谢[macona]不仅修复了这台机器，还与我们分享了它。多亏了[macona]的修复，这台机器有望再存在 60 年。
