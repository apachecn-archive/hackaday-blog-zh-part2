# Arduino 作为一种廉价的业余无线电频率计数器

> 原文:[https://hack aday . com/2013/02/09/arduino-as-an-abundant-ham-radio-frequency-counter/](https://hackaday.com/2013/02/09/arduino-as-an-inexpensive-ham-radio-frequency-counter/)

[托德·哈里森]真的有我们的号码。像他一样，我们不想在不必要的时候花钱，而且无论如何，黑进我们自己的解决方案要有趣得多。这一次，他正在帮助一位业余无线电爱好者朋友。这位朋友的收音机没有频率显示，购买附加设备的成本会比收音机高。所以[托德]已经着手为建伍 TS-520S 高频业余无线电建造一个 Arduino 频率计数器。

这篇文章(以及休息后发现的视频)并没有涵盖整个项目。这相当复杂，只是为了确保[托德]最初的想法是可行的，所以他花了大约 35 分钟解释这个问题，然后测量无线电输出，并测试 Arduino 能否准确读取它们。因为无线电具有非常大的操作范围，[Todd]将需要添加外部组件来实现这一点。额外的电路设计将是下一个项目部分的主题。
