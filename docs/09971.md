# 遥控割草机没有草可割

> 原文：<https://hackaday.com/2015/08/31/rc-lawnmower-has-no-grass-to-cut/>

他们说懒惰和需求是发明的最大动力之一。说这话的人没想过发明家无聊时会发生什么。[随机机械师]决定给自己造一台遥控割草机，尽管他一直在经历可怕的干旱——导致几乎没有草可以割。

为了让割草机可以遥控，他用电动轮椅的残骸拼凑了一台燃气割草机。这最终非常有效。他用一个旧的遥控汽车遥控器和它的两个伺服系统来遥控原来轮椅的操纵杆。简单，但是超级有效。

由于一些铅酸电池，轮椅割草机又快又好又重，前轮是脚轮，非常容易操纵。真遗憾，他没有草可割！
