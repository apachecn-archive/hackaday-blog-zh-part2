# 伯克利三录仪现在是开源的！

> 原文:[https://hack aday . com/2014/01/05/the-Berkeley-tricorder-is-now-open-source/](https://hackaday.com/2014/01/05/the-berkeley-tricorder-is-now-open-source/)

[礼萨·马奈]刚刚发布了他的[柏克莱三录仪](http://rethinkmedical.com/tricorder.html)的设计供公众使用。他从 2007 年开始设计它，作为他的博士论文，现在他已经完成了(恭喜！)，他决定通过开源让它成长！

我们在大约 7 年前报道过它，当时它还处于第一个原型阶段，从那时起它已经走过了很长的路。最新版本的功能包括肌电图(EMG)、心电图(ECG)、生物阻抗光谱仪、脉搏血氧计、加速度计，所有数据都记录到微型 SD 卡或通过蓝牙发送到平板电脑或智能手机进行数据可视化。

他发布了它，希望其他研究人员可以在他们自己的研究中利用该硬件，希望涌现出一个对非侵入式健康监测感兴趣的人的社区。运气好的话，伯克利三录仪的开发将会继续，也许有一天，它会名副其实！

不幸的是，没有新的视频展示最新的迭代，但我们在休息后附上了原始视频，这是由[Reza]自己在设备上进行的很好的叙述。
