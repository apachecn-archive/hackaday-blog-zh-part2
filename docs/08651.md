# 面向所有人的回收机器人手臂！

> 原文：<https://hackaday.com/2015/03/28/a-recycled-robot-arm-for-all/>

我们扔掉的电子垃圾多得令人难以置信。完美的组件，批量生产，价格低廉。在他们生命的最后，直接去垃圾填埋场或一些贫穷的国家熔化。你不希望你能帮忙吗？

谈到电子垃圾，步进电机多如牛毛，你可以用步进电机做很多很酷的项目——在 Hackaday.io 上，【Madivak】刚刚开始一个[机器人手臂的设计，它利用了回收的部件。](http://hackaday.io/project/4181-robotic-arm-project)

它还处于开发的早期，但这意味着现在是在项目站点上开始遵循它的好时机。这个机械臂是为他本科学位的最后一年项目设计的。除了步进器，他还使用学校的 Utilimaker 3D 打印机制造所有其他机械部件，控制来自 DRV8825 步进驱动器和 Freescale Freedom KL25Z 开发套件。休息后检查片段以查看构件的当前状态。
