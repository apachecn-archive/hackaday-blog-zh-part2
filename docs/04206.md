# VU 米舞会礼服

> 原文:[https://hackaday.com/2013/10/03/vu-meter-prom-dress/](https://hackaday.com/2013/10/03/vu-meter-prom-dress/)

![vu meter prom dress](../Images/b8402af747cf78a190df21710df7f151.png)

[凯尔]刚刚完成了这件[VU·米特舞会礼服](http://www.lammieinnovations.com/2013/10/prom-dress-update/)的收尾工作，看起来棒极了！

这件衣服使用了 70 英尺的海蓝宝石 EL 线，2600 毫安时锂电池，重新使用的 DB9 电缆，Arduino 仿制品，Adafruit 麦克风前置放大器，并通过 Sparkfun 的 [EL Escudo Dos 提供 8 个可寻址的 triac 通道。每一圈电线都是用透明线缝在衣服上的。然后用带状电缆将这些独立的部分在背后的拉链附近用菊花链连接在一起。最重要的是，【凯尔】有](https://www.sparkfun.com/products/10878)[一个便宜的热成型设备](http://www.lammieinnovations.com/2013/09/thermoforming-plastic/)，利用一个烤箱，他用它来制作电子产品的丙烯酸外壳。

这条裙子是送给他的幸运朋友[黛安]的，我们认为它会成为一个难忘的舞会！为了看到这个令人敬畏的 VU 米在行动，逗留后，为视频休息。
