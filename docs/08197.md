# 消磨时间做 Flappy Bird，而不是玩它

> 原文：<https://hackaday.com/2015/01/30/kill-time-making-flappy-bird-not-playing-it/>

在游戏意外成功后，所有的 Flappy Bird 克隆体都在天上飘来飘去，有一些比其他的更好。所谓的更好的 T1，我们的意思是，从各种硬件上一起破解。如果你手头已经有了一台 Arduino，那么你就离自己制作的还差一半了:

**《极简主义者》** **版本**

他用一些我们中的一些人可能躺在我们的长凳上的零件创建了自己的 Flappy Bird 游戏。一个加载了适当代码的 Arduino 连接到一个 16×16 的 LED 矩阵，它显然显示了你玩游戏所需的最少量的视觉信息。唯一需要的其他部分是一个单一的按钮和电阻拴在试验板上，以控制你的拍打。通过方便的图表和在同一页面上找到的代码所需的库，这很容易在一个下午就能完成。如果之后你仍然发现自己有太多的时间可以消磨，以至于你无法忍受玩 Flappy Birds，那么总有一种选择，那就是打造一个幽默大小的手机壳，将所有东西都塞进去……这是我们希望看到的。

**《花样年华》** **版本**

如果你想要比纯色 led 更高的分辨率，或者你只是喜欢游戏中著名的可怕的鸟的抽象，你可以把 16×16 矩阵换成诺基亚 LCD 屏幕。在 Hackaday.io 上重现这一构建将为游戏提供一个“更详细”的图形，并且仍然很容易组装。类似地，Ardunio 装载了生成游戏所需的智能，以及一个用于控制的按钮。将 Arduino 连接到屏幕所需的代码和令人生畏的(/讽刺)两步可以在他的项目页面上找到。

如果你一定要玩 Flappy Bird 来打发无聊，那就没有理由不玩你自己做的东西。
