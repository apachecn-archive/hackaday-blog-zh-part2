# Hackaday 奖参赛作品:有用的东西有用的代码

> 原文:[https://hack aday . com/2015/06/08/hack aday-prize-entry-useful-code-for-useful-things/](https://hackaday.com/2015/06/08/hackaday-prize-entry-useful-code-for-useful-things/)

Hackaday 奖并不仅仅是关于建造有助于地球的东西；你也可以建造一些东西，让其他人也能建造一些东西来拯救地球。[Eric]没有用他的 common code library 拯救世界，但它会让其他人更容易构建下一个伟大的东西。

commonCode 背后的思想与任何合理大小的桌面应用程序中的共享库是一样的；它为 AVR 微控制器构建任何东西提供了一个公共库。位操作、定时器接口、数学函数、图形、I/O 和外设驱动程序都可以在 commonCode 库中找到。这使得我们中有发展障碍的人很容易创建他们想要的任何项目。

commonCode 库不仅仅是为了 Hackaday 奖而创建的。[Eric]早在 Arduino 出现之前就已经开始摆弄 AVR 了，他有几十个永久安装的项目。这是回馈社区的好方法，也是允许人们开发自己的东西来解决他们心中任何问题的完美方式。
