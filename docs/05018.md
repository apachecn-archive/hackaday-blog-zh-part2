# 用乐高玩具画画

> 原文:[https://hackaday.com/2014/01/10/drawing-with-legos/](https://hackaday.com/2014/01/10/drawing-with-legos/)

![WritingMachineFrontLarge](../Images/d80ca5412453a0eab57785fbde1e0ec3.png)

有许多精心制作的乐高作品，但是你可能没有见过像[安德鲁·卡罗尔的]乐高绘图机这样的东西。他从电影*雨果*和 19 世纪的自动机中汲取灵感，特别是【杰凯-德罗兹】的制图员，不久前我们在[的一篇回顾性文章中对其进行了特写。](http://hackaday.com/2013/10/29/retrotechtacular-automata/)

(安德鲁的)手摇创作分为三个部分:一个绘图仪，一个“编码笔画程序”——它将信息存储在片段的[链接中——和一个将链接翻译成笔画的阅读器。绘图仪在 Y 轴上移动笔，在 X 轴上移动纸张以在页面上进行标记，并且还具有简单的提升机构，该机构暂时在 Z 轴上提升笔，以中断字母(或绘图)之间的笔画。](http://acarol.woz.org/WritingMachineChainsLarge.jpg)

[Andrew]将链式阅读器比作电影放映机，通过机械装置传递信息。尽管你不会在他的网站上找到详细的操作指南来解释这些设备的内部工作原理，但有一些描述基本组件的线索和几个视频，这两个都嵌入在下面。
