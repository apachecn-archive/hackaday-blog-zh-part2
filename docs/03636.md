# 建造一面闪亮的无限远镜子

> 原文:[https://hack aday . com/2013/07/23/building-a-blinky-infinity-mirror/](https://hackaday.com/2013/07/23/building-a-blinky-infinity-mirror/)

这个[无限镜原型](http://blog.freesideatlanta.org/2013/07/infinity-mirror-prototype-build.html)足够小，可以在投入整个项目之前测试这个理论。[凯文]和[爱德华]心中的整个项目是一个巨大的无限门户，他们将在今年的炼金术节上展示。

它被称为无限镜，因为它有一种比物理外壳深得多的错觉。这个技巧是通过放置两个互相平行的镜子，中间放一个光源来实现的。其中一面镜子是双向的，让一部分光穿过而不是反射给观众。

原型的视频演示显示，它只是一个 2×4 的木制框架，可以容纳两个镜子。中间是一串 LED 灯。这个原型的好处是，在顶部添加一点钢化玻璃将使它成为黑客空间的无限吧台。

你可以在自己的项目中加入的另一个镜子技巧是[在玻璃后面增加一个 LED 显示屏](http://hackaday.com/2011/03/02/bathroom-mirror-hud-displays-time-and-weather/)。
