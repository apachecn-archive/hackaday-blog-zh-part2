# 数蜜蜂

> 原文：<https://hackaday.com/2012/06/14/counting-bees/>

这是[Hydronics]设计的[蜜蜂计数器](http://www.instructables.com/id/Honey-Bee-Counter)。它被安装在蜂箱的开口处，可以计算进出的蜜蜂数量。我们自己也不是经验丰富的养蜂人(事实上我们更倾向于[清除带刺的动物](http://hackaday.com/2010/07/01/wasp-sucker-clears-the-air/))，但我们理解它们在农业和生态系统中的重要作用，所以我们很高兴有人为它们建造了一个美好的家园。

大多数设备是一个排列着反射传感器的电路板。在电路板的顶部有一个双排插脚插座，用于接收监控这些传感器的 Teensy+。蜜蜂每次进入或离开蜂箱都必须经过这个印刷电路板的下面，从而触发传感器。在休息后的视频中,[Hydronics]展示了带有上网本的系统，用于监控输出。但听起来他计划在未来版本的蜜蜂计数器中集成显示系统。
