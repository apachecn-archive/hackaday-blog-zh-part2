# 现在就做一个婴儿等离子切割机！

> 原文：<https://hackaday.com/2015/08/06/build-a-baby-plasma-cutter-right-now/>

哪个黑客不想要等离子切割机？即使你不是 MacGyver，你也可以用手头的东西在几分钟内完成这个。有什么问题吗？你可能无法用它切割锡纸以外的任何东西，它可能更像是一个[碳-空气电弧刨](https://en.wikipedia.org/wiki/Air_carbon_arc_cutting)(使用等离子)而不是一个真正的等离子切割机。尽管如此，正如[物理小店]在视频中展示的那样，它在切割铝箔方面做得很好。

如果你和我们一样，在得到四节 9V 电池、一些锡纸、一根铅笔芯和一些回形针并试用后，你现在回来了。如果你比我们更有自制力，你可能要想想你要把锡纸盖在什么上面。在视频中，他们使用了洗衣篮和橡皮筋，但任何能保持铝箔悬浮的东西都可以达到目的。

虽然这不是一个真正实用的等离子切割机，但我们正在考虑将这样的东西捆绑到 3D 打印机上，并切割箔模板。希望视频上的锯齿状边缘更多的是由手动操作，而不是由锯齿状的迷你闪电蒸发箔片造成的。

视频中反复提到铅，但令人困惑的是铅笔芯并不是由铅制成的。实际上是[石墨](https://en.wikipedia.org/wiki/Graphite)(碳的一种形式)使其成为这种应用的良好(且廉价)电极，即使它与粘土粘合剂混合。

如果你想要更实际的东西，准备至少花几[几百](http://hackaday.com/2009/01/06/make-a-plasma-cutter/)或[更多](http://hackaday.com/2014/06/22/cnc-plasma-cutter-build-presented-in-excruciating-detail/)。你需要一个[气体源(或者几种气体)，一个 MHz 范围的高压电源，和一个强大的恒流 DC 电源](http://www.hypertherm.com/en-us/Training_and_education/Intro_to_plasma/What_is_plasma/what_is_plasma.jsp)。你可能还想加入 LinuxCNC 。
