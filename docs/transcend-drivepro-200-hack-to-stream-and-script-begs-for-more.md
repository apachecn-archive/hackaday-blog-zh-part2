# 创见 DrivePro 200 Hack 到流和脚本；乞求更多

> 原文：<https://hackaday.com/2015/02/01/transcend-drivepro-200-hack-to-stream-and-script-begs-for-more/>

创见营销他们的 DrivePro 200 相机，用作汽车仪表板摄像头。我们对这样一个被降级为相当平凡的任务的东西的质量和明显的功能集感到有点惊讶。但是【小工具迷】[四处打探，发现了一个不错的小金块](https://www.youtube.com/watch?v=EpDLnRCraUc):你可以通过 WiFi 直播视频；帧速率、质量和低延迟令人印象深刻。除此之外，下一个黑客就等着你去解锁了。

就像现在这样，你打开相机的内置 WiFi AP，telnet 到设备上的两个不同端口(将其发送到智能手机连接模式)，你就可以使用 RTSP 将视图实时传输到你的计算机上。太好了，这本身就是一个很好的攻击，我们确信不久之后会有人想出一个自动的方法来触发它。[GA]还发现了如何在开机时进入脚本模式。他实际上没有执行任何代码…这就是你要做的。如果你有一个这样的东西，就把它拿出来，开始黑客攻击吧！把文件放在 SD 存储器上然后重启就行了。精心制作这个文件以支持 shell 访问将会打开整个黑客世界，从延时和动作感应到实时的特殊处理和过滤。我们认为有巨大的潜力，所以[让我们了解最新的](http://hackaday.com/submit-a-tip/)，因为你找到了新的方法来 pwn 这个硬件。
