# [本·克拉斯诺]和他的 8 KJ 红宝石激光器

> 原文：<https://hackaday.com/2014/05/23/ben-krasnow-and-his-ruby-laser/>

[https://www.youtube.com/embed/ZUevWmUViJM?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/ZUevWmUViJM?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

我们再次高兴地发现，另一个人参加 Maker Faire 只是为了炫耀其令人敬畏的东西，而不是兜售一些商品。在我们看来[Ben Krasnow]代表了硬件黑客的最高梯队(显然 Google[x]同意这一点，因为他们刚刚把他挖走了)。但[本]总是强调解释他是如何做的，以便其他人可以学习，并在某一天实现类似的伟大。这一次，它是一个功能性的红宝石激光器，由一个储存 8 千焦耳能量的电容器组支持。这就是激光能穿透钢板的原因。自从一月份[第一次展示这个项目以来，他确实取得了很大的进步。](http://hackaday.com/2014/01/10/lazing-with-a-ruby/)

不幸的是，我们直到周日晚些时候才到达[本的]展位。他之前的演示烧穿了一些封条，留给他一个不起作用的激光器。但他是一个值得信赖的人，所以我们相信他，并期待他发布一个关于激光的视频，希望是关于失败的视频。他还提到，他可能会尝试用这个装置进行月球激光测距；从月球上的反射器反射激光并测量延迟。这可以用来计算到月球的距离。

顺便说一句，当他说“[红宝石棒](https://www.youtube.com/watch?v=OsJKdxPwZdk)”这几个字的时候，不开玩笑是超级困难的。
