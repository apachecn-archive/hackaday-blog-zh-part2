# 更新:你看到的是你激光切割的

> 原文:[https://hackaday . com/2015/08/26/what-you-see-is-what-you-laser-cut-part-2/](https://hackaday.com/2015/08/26/what-you-see-is-what-you-laser-cut-part-2/)

如果激光切割机有一点让它们有点难以使用，那就是如果没有笨重的电脑在中间，一个人很难与它们一对一地互动。诚然，激光有点危险，但如果有一种不用电脑就能使用激光切割机的方法就好了。幸运的是，[Anirudh]和他的团队一直致力于解决这个问题，创造了一种可以直接与其用户互动的[激光切割机。](http://chalkaat.sbalabs.co/)

激光切割机与一个视觉系统相连，该系统可以观察许多线索。正如我们之前所介绍的，[这个特殊的激光切割机可以“看到”笔画](http://hackaday.com/2015/06/28/what-you-see-is-what-you-laser-cut/)，并将指示激光切割机沿着笔画切割(当然，一旦所有手指都离开切割区域)。这个系统的更新是，现在，用户可以从智能手机导入一幅画，并用一组相机可以观看的物理令牌来操纵它。一个标记改变切割的位置，另一个改变比例。这将激光切割机的功能从简单地在笔画位置切割扩展到能够切割任何用户操作的图像，而无需直接与计算机交互。请务必在休息后观看视频，了解这是如何工作的。

该系统不仅适用于激光切割机，也适用于任何通常由计算机界面控制的数控机床。通过直接与项目互动，你不太可能最终得到稍微有点错误的大小或尺寸的东西。如果你是数控新手，你可能想看看[这个指南](http://hackaday.com/2013/12/05/learncnc-game/)，或者也许你只是想把你现有的数控机床变成一个[象棋机器人](http://hackaday.com/2011/05/23/why-build-a-cnc-mill-when-you-can-have-a-chess-robot-instead/)。
