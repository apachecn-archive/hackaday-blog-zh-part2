# 基于 Web 的 TI 图形计算器仿真器

> 原文：<https://hackaday.com/2013/01/03/web-based-ti-graphing-calculator-emulator/>

多亏了这个[基于网络的 TI-83 和 TI-84 仿真器](http://www.cemetech.net/projects/jstified/)，你可以把 TI 绘图计算器留在家里。和几乎所有的仿真器一样，这依赖于来自实际硬件的 ROM 镜像来工作。但是如果你有一个受支持的计算器(TI-83+，TI-83+ SE，TI-84+或 TI-84+SE)，你可以自己转储图像，这应该很管用。

[Christopher Mitchell]称这个项目为 jsTIfied，因为他是用 JavaScript 和 HTML5 编写的(这就是 js 的来源)，它基于德州仪器的硬件系列(因此是大写的 TI)。在同意你不会从他的网站上得到任何 rom 后，你可以选择文件加载到你的浏览器上。计算器的图像有工作按钮，并会显示开机画面，就像真的一样。你可以像平常一样使用它，但是你可以为环境加载程序。休息后请看演示。

这些年我们在网上看到了一些关于 TI 线价格的争论。几十年来，尽管他们生产的硬件几乎一样，但价格并没有下降多少。看到有人知道如何模拟硬件，并在网络界面上启动，这很酷！但我们不禁要问，为什么 TI 不为 iOS 和 Android 销售一款同等的应用程序，或者至少利用每次生产数百万个应用程序来降低零售价格？
