# 玻璃印刷电路板 LED 时钟

> 原文：<https://hackaday.com/2012/07/16/glass-pcb-led-clock/>

这个[钟看起来棒极了，因为建造时使用了玻璃 PCB](http://www.youtube.com/watch?v=-9TS1OC-yqw) 。这个横幅图像可以让您看到所有的走线和组件，但当它离开桌面时，组成 7 段数字的 led 看起来是浮动的。

这个概念并不新鲜，但它比我们以前见过的格式要大得多。当我们第一次看[CNLohr]的[玻璃 PCB 制造](http://hackaday.com/2011/09/12/glass-pcbs/)时，他正在使用显微镜载玻片。这使用了一个大得多的玻璃窗格，但似乎仍然使用铜箔粘贴在玻璃上，然后像正常一样进行墨粉转移蚀刻。

在这里，他正在测试一些 74LV164 芯片作为恒流驱动器。Reddit 帖子上的一位评论者对以这种方式使用芯片表示怀疑，我们也是如此。但正如休息后的视频所示，这似乎是可行的(至少目前如此)。[CNLohr]还提到焊接在显示器上的 AVR 被烧坏，这对他的情况没有帮助。尽管如此，我们还是喜欢他的造型，迫不及待地想看看他将何去何从！
