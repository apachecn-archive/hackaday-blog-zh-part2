# 本周失败:灯泡控制器烧坏

> 原文:[https://hackaday.com/2014/07/03/fail-of-the-week/](https://hackaday.com/2014/07/03/fail-of-the-week/)

![fotw-nyc-resistor-chandelier-driver](../Images/39d77d0872db24fbde706bd03707d8b7.png)

几个星期以来，我们一直想抽出时间来做这件事。很难找到好的失败记录…这就好像黑客害怕承认有时项目会失败。我们希望你摆脱这种观点，因为失败是学习和真正理解的最快途径！

[xymax]正在为一个有 150 个灯泡的吊灯设计控制系统，每个灯泡使用 5 瓦。这个项目是为纽约电阻互动派对[准备的，亚当·法比奥上个月参加了这个派对](https://hackaday.com/2014/06/13/nyc-resistor-interactive-show-2014/)。随着演出截止日期的临近，最后一件作品在深夜才完成……但它是反向连接的。在一个值得拍成闹剧电影的故事[中，相反的极性导致该模块所有七个控制器板上的芯片像上面看到的那样爆炸。但这还不是全部，原型制作时正在使用的笔记本电脑通过 USB 连接，开始冒烟！](http://youtu.be/Oj15gi7x6mg?t=1m30s)

我们所有人都感受到了这种设备故障的痛苦。幸运的是,[xymax]寻找教训，而不是纠结于错误本身。使用保护二极管，键控连接器，并写下你的失败。希望阅读这篇文章能帮助其他人避免类似的破坏设备的错误。
