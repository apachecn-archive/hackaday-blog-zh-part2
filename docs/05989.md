# 使用 Oculus Rift 的运动压缩

> 原文:[https://hack aday . com/2014/05/07/motion-compression-with-oculus-rift/](https://hackaday.com/2014/05/07/motion-compression-with-oculus-rift/)

![motion-compression-with-oculus-rift](../Images/8563ba565223262ac71e472e07a7644b.png)

[AlexPewPew]向我们透露了苏黎世瑞士联邦理工学院正在进行的一些有趣的虚拟现实工作。Oculus Rift 非常擅长映射用户的头部运动，以匹配头戴显示器中显示的图像。但是为了在虚拟环境中自由行走和移动，需要完全不同的硬件。在之前，我们已经看过一些[的巧妙设置，但没有一个像这样高效。](http://hackaday.com/2012/08/31/the-simulacrum-an-innovative-solution-to-walking-in-virtual-reality/)

在上面的视频中，他们在天花板上以网格的形式放置了多张条形码纸。安装在用户头上的一个摄像头会抬头看纸的网格，并获取用户的位置。不过，最精彩的部分是他们如何将一个大的虚拟空间融入一个小房间。当用户走在一条笔直的虚拟路径上时，软件会慢慢地在小房间的曲线上画出实际的路径。最终结果是用户在小房间里绕圈行走，认为他或她正在探索一个更大的空间。很棒的东西！

如果你有一个头戴式显示器，却想不出有什么东西可以参加 Hackaday 奖竞赛，这将是一个很好的想法。你还在等什么…开始黑吧！
