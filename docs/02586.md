# Atari 操纵杆内的游戏系统

> 原文：<https://hackaday.com/2013/01/22/gaming-system-inside-an-atari-joystick/>

![gaming-system-inside-an-atari-controller](img/761cfee641b343333730a086ad271b67.png)

这种原始的 Atari 控制器非常小(看看 RCA 电缆的规模感)。尽管它很小，[凯尔·布林克霍夫]还是设法在控制器中安装了一个完整的游戏系统。这个口袋大小的 Atari 是他做的另一个项目 ArduPong 的后续项目，该项目让他使用操纵杆和 Arduino 玩 Pong。这个呈现从那个构建中取出外部项目框，并将所有内容移动到一个紧凑的小包中。

在休息后的视频中[Kyle]向我们介绍了内部情况。他带去的 Arduino 板是一个很小的板，并不比 ATmega328 的尺寸大，所以可以很容易地安装到一边。操纵杆内部已经被 PlayStation 控制器的模拟操纵杆模块所取代。这也是按钮的由来。只需将它连接到 9V 电池和电视的复合视频输入，您就可以开始玩游戏了！

现在，如果你只是想让你的 Xbox Live 游戏看起来复古，那就看看这款装有 Atari 操纵杆的 Xbox 360 控制器吧。
