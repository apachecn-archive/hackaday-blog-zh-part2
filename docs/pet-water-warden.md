# 宠物水管理员

> 原文：<https://hackaday.com/2013/10/21/pet-water-warden/>

这个周末的 Make 项目对宠物主人来说是一个很棒的项目——一个自动水碗补充装置！

这是一个相当简单的建筑，利用了一个旧水罐、一个 Arduino、一个水族馆泵和一些自制的水传感器。一如既往，MAKE 有非常详尽的指南，预计构建时间只有一两个小时。他们甚至加入了在推特上发布状态的功能，包括水库什么时候空了。

但是我们是不是把这个问题过于复杂化了？使用水壶的重力供水器也可以工作得很好。当然，你不会得到 Twitter 的更新，但是我们希望你和你的宠物在一起的时间足够长，可以知道它们什么时候渴了。

一个更精致的版本可能包括一个接入你房子的电磁水阀，消除了对蓄水池的需要，使这个项目更有用一点。但即使这样可能也有点多，我们真的需要 Arduino 吗？当碗空了的时候，弹簧加载的水碗会断开触点吗？把它连接到一个控制水阀的 5 秒钟定时器继电器上，你就大大简化了这个项目！
