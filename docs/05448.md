# 用 PS3 控制器遥控任何东西

> 原文:[https://hack aday . com/2014/03/03/remote-control-any-with-a-PS3-controller/](https://hackaday.com/2014/03/03/remote-control-anything-with-a-ps3-controller/)

![back](../Images/75a0a0458e17a3bf33efa58e19a68c41.png)

为你的下一个项目寻找遥控器时，你可能想看看你的客厅。Wii 控制器是黑客的最爱，但对于遥控飞机、汽车、坦克和多架直升机来说，挥舞电子棒并不是最棒的。你需要的是双模拟控制，这是 90 年代以来每个 playstation 都包含的东西。

[Marcel] [为此为索尼 DualShock 3 控制器设计了一个替代电路板](http://www.proofofprinciple.nl/index.php/ps3-rc-2)。有了这个板、一个 XBee 和一个旧控制器，很容易使用 XBee 接收器将双模拟控制和大量按钮添加到任何项目中。

替换板基于 ATMega328p uC，包括 Lipo 充电电路和电源，以及模拟棒和 DualShock 控制器内所有按钮板的输入。

是的，在之前，我们已经看过了[[Marcel]的项目的早期版本，但这次他增加了一些新功能——现在隆隆声起作用了，由于许多人不能或不愿意旋转几块板，[Marcel]发起了一场 Indiegogo 活动。](http://hackaday.com/2013/06/19/long-distance-ps3-controller/)
