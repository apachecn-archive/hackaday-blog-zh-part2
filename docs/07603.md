# 使用 MIDI 和磁铁用尖齿产生音调

> 原文：<https://hackaday.com/2014/11/18/using-midi-and-magnets-to-produce-tones-with-tines/>

通常你会认为管风琴的声音来自巨大的物体。[Matthew Steinke]设法将所有丰富的旋律深度压缩到一个烤面包机大小的声学设备中(YouTube 链接)，该设备使用电磁来创建其熟悉的声音。

[Matthew 的]乐器有一系列细的垂直齿，每个齿都与一个小型 MIDI 控制的电磁体相连。当磁体以特定频率调制脉冲时，拉动和释放叉齿使其以特定音调连续共振。*叉风琴*能够产生 20 个以中音 C 开始的全复调半音音符，可以作为智能手机上标准键盘或合成器应用程序的附件。该乐器的经典风格主体由桃花心木和巴宾加木制成，内置共鸣板和微型微控制器，负责接收 MIDI 并调节软件振荡器向磁铁发送电压。
