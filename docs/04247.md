# 定制无线耳机充电站

> 原文:[https://hack aday . com/2013/10/08/custom-wireless-headphone-charging-station/](https://hackaday.com/2013/10/08/custom-wireless-headphone-charging-station/)

![wirelessHeadphoneChargingStation](../Images/53114642b98e4cbddd64942796556f2f.png)

我们已经开始期待我们的技术带来相当多的便利，以至于重复插入设备进行充电似乎很乏味。hack aday regular[Valentin Ameres]决定抛弃插头，自己造了一个无线耳机充电器。我们以前看过[瓦伦丁的]作品，有一点是肯定的:这家伙喜欢他的激光切割机。他应该这么做，考虑到它已经为一个[华丽的电弧反应堆复制品](http://hackaday.com/2013/08/29/laser-cut-arc-reactor-replica/)和他的[气枪炮塔](http://hackaday.com/2013/02/15/airsoft-turret-has-turn-tilt-and-auto-feed-to-keep-those-bbs-flying/)生产了关键部件。[Valentin]再次点燃它，用丙烯酸雕刻出充电支架，然后用一个小火炬和桌子的边缘将支架弯曲成形。

他在网上找到了所需的线圈，并将接收线圈焊接到一个备用的迷你 USB 插头上。这些组件被粘在一个激光切割的丙烯酸附件上，该附件贴合在耳机的侧面，并通过直接插入耳机的 miniUSB 插孔来固定。耳机放在激光切割的充电支架上，一侧有一个丙烯酸挤压物，将发射线圈固定在接收线圈的位置上。[Valentin]还在支架顶部添加了一个简单的瞬时开关，当耳机按下时，可以激活发射器线圈和状态 LED。
