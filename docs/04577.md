# 临近空间环境舱

> 原文:[https://hack aday . com/2013/11/16/near space-environmental-chamber/](https://hackaday.com/2013/11/16/nearspace-environmental-chamber/)

![vacuum](../Images/393b09d3a411219655a5caa8cd3ad8b9.png)

如果你要将一些硬件发送到 100，000 英尺的高度，那里的大气压力是我们在表面上享受的 1%，温度下降到华氏温度和摄氏温度不再重要的程度，你可能需要做一些测试，以确保发射前一切正常。不过，用一些聚氯乙烯，[那是小菜一碟。](http://arkorobotics.com/blog/?p=113)

有几个环境条件需要考虑；高空气球所经历的近乎真空的环境将由冰箱压缩机来模拟，增加的太阳能通量由灯泡来模拟，寒冷的温度由一块干冰来提供。

对于一个合适的高海拔、低温环境舱，测试有效载荷应通过辐射冷却，壁中嵌入充满液氮的管子。这是美国宇航局做事的方式，但对于 200 美元的预算，[arko]的舱模拟高海拔环境刚刚好。
