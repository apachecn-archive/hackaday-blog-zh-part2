# 坦克的诞生(漫长而艰难)

> 原文：<https://hackaday.com/2014/11/18/retrotechtacular-the-long-arduous-birth-of-a-tank/>

在整个 20 世纪 50 年代和 60 年代初，美国陆军通过名为 *The Big Picture* 的半小时电视纪录片向其内部成员和美国公众提供定期状况报告。由于该节目由政府制作，每一集都立即进入公共领域。这篇特别报道向[讲述了 T-48 项目](http://www.youtube.com/watch?v=LQZKu0mmRGk)的故事，该项目以 90 毫米 M48 巴顿坦克而告终。

电影开篇提供了坦克的简史以及从一战到朝鲜战争期间学到的教训。陆军寻求一种更坚固的能适应各种气候和地形的车辆，于是信息收集过程开始了。在五角大楼举行的一系列会议中，所有相关方探讨了各个方面，该项目获得批准，一个马尼拉文件夹被正式指定用于该项目并贴上相应的标签。

然后我们参观研发工厂，在那里开发和测试新的坦克材料和部件。正是在这里，传动齿轮在扭力机上经受考验。空气净化器相互竞争以决定哪一个能过滤掉最细的灰尘和沙子。经过仔细分析后，不同的罐壳材料用各种有据可查的电极试焊在一起，这些面板被拿到外面，这样它们的焊缝就可以直接点火。

许多零件都在一台能以 40 万磅的压力挤压的机器上进行了拉伸测试。炮塔测试者摇动并摇动该装置，以确保当坦克穿越山丘和山谷时，火炮不会掉落。看起来，最终将组成 M48 Patton 生产的 85k 部件中的每一个都经过了严格的测试。就连覆盖驾驶座的皮革也与设计用于老化和磨损的设备相抵触。

此时，一名项目工程师来到现场指导比例模型的制作。一旦获得批准，一个全尺寸的工程原型就被组装起来，并在其分子的一英寸范围内进行测试。这包括将它开到一个寒冷的测试仓库，在那里，穿着西装的人在玻璃后面将它暴露在-65 华氏度下，并验证它在没有缸体加热器的情况下启动。

坦克原型在阿伯丁试验场进行性能测试。模拟的现场条件包括 60%的坡度、波浪状道路和一个大浴缸。车辆必须在各种丘陵、沼泽、湿地和泥沼上证明自己。最后，制作了几个飞行员模型用于战术测试。一旦获得完全批准，就可以开始大规模生产。这部电影估计，一辆新型坦克从构思到交付平均需要三到六年的时间。
