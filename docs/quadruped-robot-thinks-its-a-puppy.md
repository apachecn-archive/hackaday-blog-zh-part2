# 四足机器人认为它是一只小狗

> 原文：<https://hackaday.com/2013/10/18/quadruped-robot-thinks-its-a-puppy/>

回到 2012 年纽约 MakerFaire，我们注意到一个神奇的小蒸汽朋克四足机器人在户外人群中行走。这个机器人执行得非常好，有一种独特的能力，用它小狗般的动画吸引孩子们。原来这是[【德鲁的】小步行机器人(又名小狗机器人)](http://drewsrobots.blogspot.com/2011/09/little-walking-robot-aka-puppybot.html)。

小狗机器人实际上已经存在很长时间了。他是用[德鲁]参加机器人战争和战斗机器人比赛后剩下的零件出生的。这些比赛中的机器人通常由无线电控制飞机或汽车发射器控制。这些系统中的大多数是作为遥控汽车或飞机的包装出售的。除了发射器和接收器，包通常包括一个电池和 3 或 4 个伺服。标准的遥控伺服系统对于战斗机器人来说太弱了，所以它们仍然放在他的零件箱里。

在[德鲁]称之为缓慢的周末，他开始组装伺服系统，最终完成了一个可以在房间里爬行的基本机器人。从那以后，机器人有了自己的生活。[Drew]改进了电池系统，并添加了一个微控制器来自动化各种步态和动画。他带着这个机器人参加了他的一次战斗机器人比赛，它获得了“最酷机器人”奖——尽管它实际上并没有参加比赛！

这个机器人的第一次公开亮相是在 GenCon 2005 年，那里的观众很喜欢它。在此后的几年里，[Drew]不断迭代小狗机器人的设计。他提出了几个具体的目标，并在他的博客中概述了这些目标。其中包括人群友好性、易于维修和简单性。所有这些目标都源于他在维修和升级小狗机器人时的经历。我们认为这些目标和[Drew 的]整个博客对任何有抱负的机器人制造者来说都是一个很好的信息来源。
