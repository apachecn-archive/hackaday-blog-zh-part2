# LED 滑雪板点亮夜晚

> 原文:[https://hack aday . com/2014/05/17/led-snowboards-light-up-the-night/](https://hackaday.com/2014/05/17/led-snowboards-light-up-the-night/)

![night time snowboarding](../Images/2dc3bfb070dce5c4d94db745776ab03d.png)

晚上玩滑雪板很棒——但是除非你在一个光线充足的滑雪斜坡上滑雪，否则你不会有太多的运气在山里自由滑——当然，除非你有一个配有泛光灯的[板！](https://www.youtube.com/watch?v=3rBKUFHd1VQ)

在 [Signal Snowboards](https://www.youtube.com/user/SignalSnowboard) 的人们做了很多很酷的滑雪板模型，比如完全用纸制作滑雪板，制作加热板在你行进时融化雪，制作防弹滑雪板……还有很多。渴望利用日益减少的 2014 年登机季节，他们决定制作泛光灯滑雪板，这是一种四面配备 LED 灯的滑板，可以在夜间进行惊人的骑行——以及非常酷的视频和照片效果！

一家公司捐赠了一吨的 LED 头灯和手电筒给他们，他们开始工作。虽然这在技术上就像把手电筒绑在电路板上一样简单，但因为他们实际上是制造电路板，所以他们已经将灯完全集成到设计中。在他们店里看到全过程还是挺爽的！
