# 气泡显示器的分辨率越来越高

> 原文:[https://hack aday . com/2014/08/14/bubble-displays-is-increasing-in-resolution/](https://hackaday.com/2014/08/14/bubble-displays-are-increasing-in-resolution/)

![PipeDreams 3 bubble display](../Images/e068a49acc369071710b1096edd803cd.png)

[布鲁斯]创造了一个非常酷的气泡显示器,能够显示人们可辨认的照片。这整个艺术装置在三层楼高的地方也毫不逊色！这个位于加拿大多伦多的安大略科学中心。如果你对气泡显示器不熟悉，它们由几个装满液体的透明垂直管组成。安装在每根管子底部的气动电磁阀允许一定量的空气在特定的时间进入管子。因为空气比液体轻，所以气泡在液体管中向上移动。如果这些泡泡出现的时间正确，就可以做出有趣的图案。这种设置使用一台[基于 Linux 的计算机](http://hackaday.io/project/496-A-3-story-tall-96-tube-bubble-%22printer%22)和定制的控制软件来操纵阀门。

[布鲁斯]并不是从制作超级复杂的泡泡显示器开始的。这实际上是他的第三次复飞，有 96 个单独的管子，能够显示光栅图像，这是迄今为止最复杂的一次。他的第一个作品由 16 根管子组成，每根管子的直径都比最近的作品大。直径越大，管数越少，分辨率越低，只能显示简单的形状。版本 2 有两倍多的管子，这次是 32 根。除了将试管数量增加一倍之外，布鲁斯还对试管中的液体进行了着色，不是所有的颜色都一样，而是彩虹的所有颜色，从红色到紫色。不过，这个版本不能显示光栅图像。在我们看来，第三次是最有魅力的一次！休息后的视频…
