# 咖啡桌上的美国建筑大街

> 原文：<https://hackaday.com/2012/05/07/building-main-street-usa-in-a-coffee-table/>

![](img/b10111b0c9f6143f3853653ea4f748d2.png "parade")

[Alex George]一直在收集艺术家[Robert Olszewski]手工制作的迪士尼乐园美国主街微缩模型。这些模型非常精确，但却是静态的。[Alex]有一些主要街道电子游行的彩车，它们在几个 led 的帮助下发光。有一天，[亚历克斯]发现自己希望能看到一个微型游行围绕着他的立体模型，并做了我们任何人都会做的事情:[在他的微型城镇周围做一个微型电子游行。](http://www.bygeorgeithinkyouvegotit.com/miniparade.html)

[Alex]通过在他的微型主街道下设计一个链条和链轮系统开始了他的建造。不展示时，游行花车藏在城镇的下面。不过，到了晚上，游行队伍会浮上水面进行表演。

如果没有任何灯，这就不是电力游行，所以[亚历克斯]抓了几个闪光灯安装在每个花车的底部。这些是小的可编程 RGB 发光二极管，可以在游行可见的整个时间内重复相同的灯光序列。对[亚历克斯]和[罗伯特·奥尔谢夫斯基]来说，这都是非常出色的工作和精湛的技艺。

[Alex]的“制作”视频和彩车的完整演示将在休息后提供。
