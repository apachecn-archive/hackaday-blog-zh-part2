# 智能手机控制的迷宫

> 原文:[https://hack aday . com/2013/02/22/smart phone-controlled-labyrinth/](https://hackaday.com/2013/02/22/smartphone-controlled-labyrinth/)

![smartphone-controlled-labyrinth](../Images/f8d65f32206eb004a693f33778232e6b.png)

这整个项目可以作为一个应用程序来完成，在屏幕上虚拟地绘制迷宫和球。但这远没有马特的成就有趣。他建造了一个小迷宫，对他手机中的加速度计做出反应。

仔细看看手机。它不是安卓系统，不是 iPhone，也不是黑莓。那东西是 Windows phone…不，真的！这款手机可以兼做计时器，我们认为这是一个不错的选择。它与驱动和监控迷宫的 Netduino 通信。

你可能已经注意到，这个迷宫是手工建造的，而不是玩具的修改过的商业版本。他在一对伺服电机上安装了一些硬纸板，然后在表面上建造了迷宫。还有检测金属球何时桥接两个触点的传感硬件。这让我们对牛头怪的复仇形态有了美好的回忆。
