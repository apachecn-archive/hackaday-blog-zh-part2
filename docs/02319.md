# 解锁威瑞森 Galaxy Note II 和银河 S3

> 原文:[https://hack aday . com/2012/12/12/unlocking-Verizon-galaxy-note-ii-and-galaxy-S3/](https://hackaday.com/2012/12/12/unlocking-verizon-galaxy-note-ii-and-galaxy-s3/)

![galaxy-note-II-verizon-unlock](../Images/a0bfbe41da33f3ee1454ca81d134ec59.png)

[Adam Outler]和朋友们一直在努力解锁一些威瑞森 Android 设备的引导加载程序。他最近的冒险包括解锁威瑞森品牌的三星 Galaxy Note II。

您不能在锁定了引导加载程序的设备上运行 Cyanogenmod。这大概就是为什么 XDA 论坛的威瑞森手机用户很快就筹集到足够的钱，把其中一只小狗放在亚当的手里。休息之后，他回顾了他在视频中发现漏洞的过程。我们不是这方面的专家，但显然。闪烁 Odin 时使用的坑文件是利用漏洞的入口点。一点代码被注入其中，这提供了一个开放的闪存替换引导加载程序。

我们在标题中提到了 S3 星系。显然[也解锁了](http://forum.xda-developers.com/showpost.php?p=30274025)，但是有一个大问题。空中更新可能会阻止 S3。为了避免 Galaxy Note II 的这一问题，作为漏洞利用的一部分，最初的引导加载程序被打了补丁并被刷新。
