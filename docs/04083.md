# 梵高团队使用 OpenXC 从您的硬盘中创作艺术品

> 原文:[https://hack aday . com/2013/09/18/team-van-Gogh-uses-openxc-to-create-art-from-your-drive/](https://hackaday.com/2013/09/18/team-van-gogh-uses-openxc-to-create-art-from-your-drive/)

![vangogh](../Images/fe7fee2706fede81a979740f82c1b847.png)

在这个[视频](http://www.youtube.com/watch?v=VZedQH9FHiw)中，【Joe Grand】带我们浏览 [OpenXC](http://openxcplatform.com/) 黑客马拉松活动中[【梵高团队】的参赛作品](http://imgur.com/a/ZvDA9#YuP9ho7)。在可能是历史上最伟大的公路旅行中，[乔·格兰德、本·克拉斯诺、泰克尼加和超级牛逼的西尔维娅]全都挤在一辆车里。他们带来了一大堆开发板、电线、一台 CB 收音机，当然还有西尔维亚的水彩机器人。

顾名思义，[梵高团队]比其他团队采取了更艺术的方式来迎接挑战。OpenXC 转向、换挡、油门和刹车数据通过芯片套件发送到 RS-232 链路，进入[TechNinja]的笔记本电脑。笔记本电脑将数据转化为水彩机器人的指令。有了这个系统，一个简单的周日驾车旅行可以变成抽象艺术。

该团队还展示了如果 OpenXC 扩展到将数据发送回车辆可以做些什么的概念——这是福特不支持的。他们的例子是，当有电话打进来时，使用该系统降低代表汽车蓝牙系统的 CB 收音机的音量。

这一挑战的大部分是通过来自 OpenXC 车辆界面的模拟数据完成的。该团队只有几分钟的时间在一辆真正的汽车上解决问题。然而，他们很好地证明了他们的概念，赢得了大奖。

任何参加过黑客马拉松的人都知道，在活动接近尾声时，人们会变得多么疲惫和易怒。值得注意的是，这里的团队笑容满面。我们认为这是一个很好的例子，四个人聚在一起(其中一些人是第一次)创造了一些伟大的东西。
