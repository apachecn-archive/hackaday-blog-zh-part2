# DOTS 使用颜料控制树莓 Pi 2

> 原文:[https://hack aday . com/2015/04/28/dots-uses-paint-to-control-raps berry-pi-2/](https://hackaday.com/2015/04/28/dots-uses-paint-to-control-rapsberry-pi-2/)

在 SXSW Create Raspberry Pi foundation，离我们两个桌子远的地方，有源源不断的孩子在 Raspberry Pi 上玩《我的世界》，拿起画笔。绘画活动是由他们为活动旋转的电路板驱动的，该电路板使用导电涂料来控制 Raspberry Pi 2。

![rear-of-the-raspberry-pi-2](../Images/269d9abd8959afcf0fb735b2d0394a5f.png)电路板使用[帽子形状](https://github.com/raspberrypi/hats)，这是盾牌的一个有趣的名字(也是一个聪明的名字，因为它代表“附在顶部的硬件”)。在这张图片中，您可以看到电路板的背面。它采用极薄的表面贴装引脚插座。

正面暴露了几个圆形的铜垫，它们组成了一个“连接点”游戏，通过在表面上涂导电墨水来玩。这导致飞机被涂在板上，并显示在电脑上。有一组垫子，允许用户选择在显示器上画什么颜色。

我们喜欢这种不同的教育方式。孩子们更习惯于敲击触摸屏、点击鼠标或敲击键盘。但是导电墨水提供了几个学习机会；油漆只是简单的把内圈和外圈连接起来；这些圆中的一个在每个点(地)上都是相同的；任何将点的这两部分连接在一起的东西都会成为计算机的输入。很棒的东西！

该基金会下个月将把这些板带到 Maker Faire Bay 地区，所以请顺便来看看这些板的使用情况。你可以在 Raspberry Pi 网站上了解圆点板的制作过程。他们给[一些想要为这个项目做贡献的软件开发人员](https://www.raspberrypi.org/dots/)分发了几块电路板。广告之后是我们对马特·理查森的视频采访。
