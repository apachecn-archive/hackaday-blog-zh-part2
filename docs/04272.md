# 气垫船飞机？

> 原文:[https://hackaday.com/2013/10/10/hovercraft-plane/](https://hackaday.com/2013/10/10/hovercraft-plane/)

[鲁迪·希曼]一直在他的车库里研究一种我们认为是新型的车辆。这是一艘气垫船——但它也有翅膀。

没错，你可以在地面、水上驾驶它，甚至可以带着它飞行！然而，这并不是一个新的想法。在做了一些挖掘后，似乎这种交通工具的第一种类型实际上是在 1996 年由[通用气垫船](http://www.hovercraft.com/content/index.php?main_page=index&cPath=216)测试的——快速浏览他们的网站会发现你甚至可以自己购买工具来制作一个！不管它从何而来，也不管是谁首先创造了它，这都是一个非常有趣的概念，而且飞行起来会很刺激。哦，你甚至不需要一个飞行员的执照，它被认为是一艘船，并遵循相同的规则和划船条例。

休息后留下来看看一个在行动！现在我们所需要做的就是找出如何将其中一个与一艘德洛丽安气垫船结合起来！
