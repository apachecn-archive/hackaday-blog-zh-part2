# Kickstarter 项目真的能造出太空电梯吗？

> 原文:[https://hackaday . com/2012/08/30/can-a-kickstarter-project-actually-build-a-space-elevator/](https://hackaday.com/2012/08/30/can-a-kickstarter-project-actually-build-a-space-elevator/)

这是构成科幻小说的材料:一部可以爬升到太空而不需要火箭的电梯。能做到吗？不。但是这个 Kickstarter 项目的目标是[资助最终使太空电梯成为可能的研究](http://www.kickstarter.com/projects/michaellaine/space-elevator-science-climb-to-the-sky-a-tethered)。他们已经远远超过了他们的目标，并计划使用额外的资金来扩大实验的范围。

一次彻底的成功将会是[一条伸入太空的系绳](http://hackaday.com/2008/10/07/space-elevator-a-real-possibility/)，由一个被离心力拉离地球的重物保持牵引。这还没有真正引起人们的注意(上次我们听说人类还没有能力制造一种足够强的物质来防止绳索断裂)。正在研发的是一个气象气球，它支撑着一条机器人可以攀爬的带子。这个团队对此并不陌生，他们已经在大学和几年前关闭的一家初创公司建立并测试了几个模型。现在，他们希望在空中获得一条 3-5 公里长的带子，并制造一种新的机器人来攀爬它。

现在，我们必须满足于 1000 英尺。休息后的攀登视频。但是我们希望在我们有生之年能看到一个类似上图所示的地月货运系统。
