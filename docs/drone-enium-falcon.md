# 无人驾驶飞机-猎鹰

> 原文：<https://hackaday.com/2015/02/05/drone-enium-falcon/>

如果你有一架四轴飞行器，你可能不止拥有一架。这是一种让人上瘾的东西。因此，挖掘出那个损坏的模型，并围绕它建立一些令人敬畏的东西。我们建议[让它看起来完全像千年隼](http://imgur.com/a/CjYA6)。好吧，公平地说，这是围绕一个定制的四轴飞行器建立的，最初设计用于携带相机和 GPS，但在这个项目中被删除。我们不确定库存型号是否有足够的额外重量来举起这样一个花哨的机身(也许你会在评论中加入这一点？).

和任何伟大的建筑一样，这都是从比例图开始的。该图纸被打印出来用作发泡聚苯乙烯的切割模板。让它看起来如此神奇的部分原因是机身不是二维的。重要的地方有深度，这都是因为[奥利维耶]近乎神话般的泡沫切割/成型技巧。

最后一点是前面的发光二极管和模拟尾部的弯曲引擎。你几乎可以看到这个东西在贝斯平的漂浮城市下面拿起一个没有把手的[卢克]。这只猎鹰飞起来像…四轴飞行器(你还指望什么？凯塞尔跑 12 秒差距？)，您可以在休息后的视频中看到。第二个片段显示了从四框架上移除泡沫体是多么容易，这又是一个不错的选择！

当然，如果你不喜欢《星球大战》,你可以试试让[作为飞行体](http://hackaday.com/2012/02/28/human-shaped-planes-troll-nyc/)翱翔天空。
