# 给自己造一架飞机

> 原文:[https://hack aday . com/2014/07/01/retro tech tacular-build-yourself-an-air plain/](https://hackaday.com/2014/07/01/retrotechtacular-build-yourself-an-airplain/)

![retrotechtacular-cub-j-3-piper](../Images/556d964bd23290c2f2f81c6ad0b6149f.png)

如今的飞机超级复杂——想想最近 B787 中燃烧的锂电池问题，这些问题可能已经解决，也可能没有解决——但它并不总是这样。这里有一个很好的例子。[Piper J-3 Cub](https://www.youtube.com/watch?v=Q6q1VKsTeKQ)的制造展示了简单高效的机械设计，通过大量步骤实现，所有步骤均无需自动化。

构建从框架开始。管道被咬入专门的鱼嘴，以便在捆绑到夹具和点焊之前紧密配合。在机身为一个整体的情况下，拆除框架，对每个接头进行完全焊接，随后进行检查。电缆穿过框架，将操纵面连接到驾驶舱。继续到机翼装配，我们特别惊讶地看到手工钉钉子，以确保木材肋骨金属梁。你认为那个工人一生中敲了多少钉子？整架飞机被织物覆盖，增加了一个引擎，它进入了蓝色的天空。

回顾制造技术很有意思——你认为视频中显示的大型模型会在这些日子里建造出来吗，或者他们只是使用 CAD 渲染？
