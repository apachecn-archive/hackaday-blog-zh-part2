# 博物馆应该比主题公园更受欢迎

> 原文：<https://hackaday.com/2014/07/11/museums-should-be-more-popular-than-theme-parks/>

[https://www.youtube.com/embed/YHutbJxDwkI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/YHutbJxDwkI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

作为底特律之旅的一部分，我们安排了一次实地考察，其中一次是去亨利·福特博物馆。在相当令人失望的前半个小时里，我们在抛光汽车的静态展览中闲逛，之后我们来到了博物馆的一部分，它为任何有工程头脑的人开启了血清素泵。有早期工业化的惊人展示，包括工厂用蒸汽机、早期发电机和早期装配线机械的例子。我们将深入讨论这些内容，但是将它们编辑在一起需要一些时间。

现在，我们想让你快速一瞥一个令人愉快的 t 型车展览。每天早上，博物馆的工作人员都会将整辆车拆开，并在一整天的时间里帮助参观博物馆的人完成将它组装起来的过程。

为什么这不是取代游乐园的模式？这种用真实设备进行的实际操作——而不仅仅是一个经得起大众检验的模型——是吸引所有年龄段好奇者的纯金。与博物馆工作人员的互动给这个机构增加了有形的人文元素，在这个过程中，你可能会学到比历史更多的东西！
