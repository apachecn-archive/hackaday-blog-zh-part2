# 复古:发现电子音乐

> 原文:[https://hackaday . com/2013/10/08/retro tech tacular-discovery-electronic-music/](https://hackaday.com/2013/10/08/retrotechtacular-discovering-electronic-music/)

![retrotechtacular-discovering-electronic-music](../Images/7fbac33cc78106ee2d33e329e0b7eb40.png)

我们今天理所当然地认为，我们听到的许多音乐都包括合成的乐器和声音。但是回顾一下 1983 年的《发现电子音乐视频系列》,我们可以看到这项技术最初的雏形。第一部分的前五分钟可能会让你的听觉受到困扰，但这是值得的，因为这是我们使用如上所述的设备进行声音生成的时候。该系列的所有三个部分都嵌入在下面；总共大约二十分钟的视频。

今天使用的混音器板和其他控制接口仍然有很大的面积用于旋钮和调节。但它们也包括大量的软件处理选项，这些选项直到计算机变得负担得起和无处不在时才可用。视频中显示的是一组硬件接口，用于处理来自振荡器的信号或改变录制的声音。我们花了很多时间[惊叹软件定义无线电](http://hackaday.com/2013/08/23/a-comparison-of-hacker-friendly-sdrs/)以及它如何让大众能够接触到射频黑客技术。但是这里有谁没有使用无数的音频软件对电子音乐进行过哪怕一点点的修补呢？如果您需要构建自己的包络和滤波电路，您会这样做吗？
