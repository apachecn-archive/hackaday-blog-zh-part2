# 一支步枪，大约 1700 年手工制作

> 原文：<https://hackaday.com/2013/07/04/a-rifle-handmade-circa-1700/>

[![rifle](img/801b0f1c035d6f8f357c6824d2d4d8f3.png)](http://hackaday.com/wp-content/uploads/2013/07/rifle.jpg)

今天，步枪是用非常精密和复杂的机床制造的，用于铣、磨和镗枪管。长枪出现的时间比这些现代机床要长得多，那么在布里奇波特磨坊和谢林车床出现之前的时代，枪械工匠究竟是如何创造出如此精密的艺术品的呢？

在一个令人惊叹的 10 集视频系列中，威廉斯堡殖民地的[Wallace Gustler]带我们经历了大约 1700 年制作燧发枪的过程。顺便说一句，所有的视频都是在休息后嵌入的。

制造步枪的第一步是制造枪管。这是由一根熟铁制成的，围绕心轴锤成一根管子，然后在熔炉里焊接在一起。在一台原始的手摇车床的帮助下，枪管被钻了出来，并最终在一个切削工具的帮助下被膛线，这个切削工具更多的是由山核桃木而不是工具钢制成的。

桶装好后，[华莱士]走向锁。同样，所有的东西都是手工制作的，几乎完全来自于 1700 年在新世界殖民地当地采购的材料。弹簧钢是这种对当地材料的渴望的例外之一，还有一些从进口来源回收的黄铜。

当然，枪匠必须是金工大师，但他也必须是优秀的木雕师。枪的枪托由一块巨大的糖枫木板制成，经过精心雕刻，以容纳枪管、锁和定制的铸铜件。

结果是一把制作精良的燧发枪，能够在几百码外瞄准目标。[华莱士·古斯特勒]这支枪的几乎所有部件都是手工制造的，对于一个大师来说，这是一次令人印象深刻的技能展示，但对于任何想用双手工作的人来说，这是一个灵感。
