# 城市农业利用水培技术在没有农田的地方创造农田

> 原文：<https://hackaday.com/2012/06/27/urban-farming-uses-aquaponics-to-make-farmland-where-there-is-none/>

[埃里克·芒杜]在奥克兰务农。在这片水泥丛林中没有开阔的田野，即使有，他所在城镇的土壤也受到了污染，不适合种植食物。但是他没有使用古老的耕作方法。事实上，一个世纪前的农民不会认识到他正在做的任何事情。他的技术利用鱼、循环水和砾石在他能找到的任何地方种植蔬菜；一种叫做水培的耕作方法。

休息后的视频很好地展示了他的农场。该系统的两个主要部分是一个鱼生活的大水槽和一个凸起的砾石床，水中的鱼废物在这里被过滤掉并被细菌分解成蔬菜的食物。更多的部件可以被添加到组合中。例如，一旦水被石床过滤，它可以通过重力流入另一个容器，该容器用于种植由浮动泡沫板悬挂的莴苣。但是水最终总是会回到鱼槽中，在那里它可以被重复利用。这最终节省了正常农业用水的 90-98%。

但是[Eric]也对增加一些自动化感兴趣。大约 7 分钟后，我们看到了他在 Arduino 和其他硬件的帮助下工作的控制系统。
