# 达斯·维德魔法 8 号球

> 原文：<https://hackaday.com/2014/11/27/darth-vader-magic-8-ball/>

想象一下，你的妻子喜欢达斯·维达，并希望在做出重要的人生选择时得到帮助。(谁没有呢？)[bithead942]在一个项目中解决了这两个问题，它摧毁了一个乐高时钟，并制作了一个会说话的电子动画达斯·维德 8 球式神谕。现在他的妻子可以简单地按下达斯的头，她的决策由原力的黑暗面处理。

你可以在休息下方的[视频](https://www.youtube.com/watch?v=9hq30UtIk0E)中看到结果。

内部主要由 Arduino Nano、WTV020SD WAV 播放芯片和一些时髦的伺服系统组成。[bithead942]对现有的时钟内部进行了改造，找到了一种让它完全适合的方法。斗篷有所帮助，扬声器非常适合以前的时钟显示。

然后，他使用 IMDB，在《星球大战》电影中寻找达斯·维德的引语，听起来有点像 8 号球的答案。正如[bithead942]提到的，达斯·维德并不太喜欢正面的东西，所以找到他说“是”的例子是一项艰巨的工作。这与比服用百忧解的啦啦队长更光明的前景形成对比，但他们称之为黑暗面是有原因的。

我们真的很喜欢腰部和手臂伺服系统协同工作的方式，让达斯活了起来。增加的橡木底座和拉出式指令卡不仅让达斯看起来很花哨，还能防止他前倾说话时摔倒。总的来说，这是一个非常好的版本，写了很多困难和解决方法。
