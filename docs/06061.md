# 本周失败:投影仪 LED 改造

> 原文:[https://hacka day . com/2014/05/15/本周故障-投影仪-led-翻新/](https://hackaday.com/2014/05/15/fail-of-the-week-projector-led-retrofit/)

![fotw-projector-repair](../Images/26c573e6ea9081adc7b3ea96554604dd.png)

对于一个项目来说这是一笔交易，解决它能有多难？

如果你是一个真正的黑客，我们敢打赌，你以前也曾陷入过这种思维过程。[Luft]大约一年前购买了这台二手夏普 XR-10X-L 投影仪，并计划用 LED 灯泡对其进行改造。他收集了所有的零件并开始工作，成功地测试和安装了修改。但是幸运的是，[这个项目陷入了某种引导循环](http://blog.luftek.si/2014/03/projector-sharp-led-lamp-part-2.html)。

这次失败当然不是因为缺乏准备。记录他的冒险经历的第一个帖子显示，[黑客在](http://hackaday.com/2013/03/28/epson-projector-led-mod/)之前就已经完成了，他获得了这个特定硬件的服务手册，并且在订购部件时做了功课。成功需要避开一些确保外壳和内部部件就位的传感器，并确保镇流器的电子状态被正确报告，尽管 LED 源不需要它。通电后，所有的绿色指示灯都会亮起，但紧接着会出现重启序列。

他试着观察串口，看是否能得到任何状态信息，但是没有成功。根据本专栏的性质，让我们看看是否可以在评论中提供任何建设性的故障排除建议。
