# 双宠物食品分配器加倍方便

> 原文:[https://hack aday . com/2015/05/31/dual-pet-food-dispenser-is-double-benefit/](https://hackaday.com/2015/05/31/dual-pet-food-dispenser-is-doubly-convenient/)

你的狗或猫是否每天早上叫醒你，要求喂食？也许你可以在晚上喂斯巴基。但这不会限制你的自发性吗？这肯定限制了[Jorge]的下班后计划。他有两条狗，吃同一种食物，但数量不同。这是他设计和制作双宠物食品分配器的一个重要因素。

[Jorge]从 3D 建模他的需求开始。当食物从储料斗转移到碗中时，分配分两个阶段进行。一个 12V 的打印机马达转动 3D 打印的螺旋钻，将金块运送到中转区。在这里，一个伺服系统以跷跷板运动的方式控制一个斜坡，将食物滑入一个或另一个碗中。

该分配器是围绕 PIC18F2420 设计的。虽然这台微型电脑是[Jorge]的第二选择，但它符合设计中的所有要求。他的丙烯酸外壳具有四个按钮，通过 16×2 LCD 进行导航和选择。[Jorge]食物在第一阶段卡住了。一个朋友建议他用振动来搅动食物，但是没有用。[Jorge]最终添加了一个带辐条的搅拌轴，有助于保持食物移动。休息后继续参观。

如果你想在计时器上分配单份食物，看看这个由拾荒零件制成的自动猫食器。
