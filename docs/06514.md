# THP 参赛作品:汤姆，便携式 3D 打印机

> 原文:[https://hack aday . com/2014/07/09/THP-entry-tome-the-portable-3d-printer/](https://hackaday.com/2014/07/09/thp-entry-tome-the-portable-3d-printer/)

好吧，3D 打印机是存在的。你只需花几百美元买下 T2，把它们放在你的桌子上，就能让它们开始生产塑料零件。有点行人，不是吗？你可以带一些东西到现场给客户，然后打印出一些新的零件*就在那里*怎么样？给最近的人道主义危机送个打印机怎么样？毕竟，如果没有 3D 打印机，我们听到的 3D 打印机的所有人道主义用途都不会有任何好处。

[TOME](http://hackaday.io/project/1437-TOME---Portable-3D-Printer)是[Philip]试图将 3D 打印机便携化，也是他参加 Hackaday 奖的原因。对 TOME 来说，初步目标是能够在一个电池上打印四个小时，一个自动调平床，以及一个易于更换的可访问热端。

对我来说，这个设计已经相当有趣了。精明的打印机爱好者会注意到在 X 托架上没有步进电机。在 X 轴上移动头部的任务由底座中的步进器负责，方形轴和一组齿轮来回移动一切。

通过这种奇怪而巧妙的电机设置，整个打印机能够自行折叠，从而可以安装在防水塑料盒中。如果你在路上带着打印机，这是你需要的。
