# 具有简单电路的可切换磁体

> 原文:[https://hack aday . com/2015/03/17/带简单电路的可切换磁体/](https://hackaday.com/2015/03/17/switchable-magnet-with-a-simple-circuit/)

磁铁太棒了。电磁铁更酷。但是，如果你能制造出一种半永久的可切换磁铁，它的作用就像一个电磁铁，但不使用任何能量来吸住金属，那会怎么样呢？你会想看看这个[低功率磁性保持和释放机制。](http://www.wattnotions.com/low-power-magnetic-hold-and-release-mechanism/)

这其实是一个很简单的概念。它基本上是一个附着在永磁体上的电磁体——它会像你预期的那样抓住任何金属物体——但如果你让电流通过附着在它上面的感应器，电流产生的磁场会暂时抵消磁场——从而释放你被抓住的物体。由于重力的作用相当快，这种电流脉冲不需要很长，只需要几分之一秒。

现在真正的问题是你能做到多大？我们不久前报道了另一个名为 [Open Grab](http://hackaday.com/2013/07/12/electro-permanent-magnets-for-quadcopters/) 的项目，该项目讨论了在四轴飞行器中使用这种技术的可能性。

对于一个完全不使用电力的解决方案*看看用于焊接的可切换磁铁夹具——它们非常酷——但当然受专利保护。*
