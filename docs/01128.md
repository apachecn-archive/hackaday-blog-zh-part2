# [迪诺]会说话的盒子

> 原文:[https://hackaday.com/2012/07/24/dinos-talking-boxes/](https://hackaday.com/2012/07/24/dinos-talking-boxes/)

![](../Images/4e30b80c8aa7abe7de475326faac5fac.png "dinos-talking-boxes")

[迪诺]已经完成了他的对话盒项目的四分之三。他完成了两个盒子中的一个，并展示了他用来将动作与声音结合起来的技术，以便用盒子顶部模仿嘴唇的颤动。

你可能还记得[Dino's] [先来看看 EMIC2](http://hackaday.com/2012/07/11/emic2-text-to-speech-module/) 。这是一个单板文本到语音模块，为机器提供语音。但是，如果没有一些电子动画，这有什么意思呢？所以[迪诺]开始尝试不同的概念，随着演讲移动盒子顶部。这说起来容易做起来难，但正如你在休息后的视频中看到的，他确实做得很好。他构建了一个电机控制电路，该电路采用 LM386 放大器芯片的音频输出，并将其转换为电机的驱动信号。轴没有直接连接到盒盖上。取而代之的是，它有一根弯曲的金属丝，被一根绳子限制住，这样它就不会旋转太远。它掀起用一块布铰接的盖子。
