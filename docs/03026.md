# NANDputer 主要是布线

> 原文:[https://hackaday.com/2013/03/21/nandputer-is-mostly-wiring/](https://hackaday.com/2013/03/21/nandputer-is-mostly-wiring/)

我们敢打赌，按重量这个项目主要是布线。我们甚至可以打赌，电线的重量超过其余的组件 2 比 1。我们将祈祷永远不会有松散的连接，但目前看来，基于 NAND 的计算机项目已经启动并运行。几乎所有的构建都是基于与非门的，这就是为什么点对点连线如此疯狂。有一个外围板使用一些非 NAND 组件，但他最终计划更换它，使系统…纯？

现在准备好疯狂的部分。这只是程序计数器的一半！还有一块板看起来很像。这两块板加上至少六块大小和复杂程度相似的板，就构成了一台正常工作的计算机。疯了！帖子分享了大量的细节，但你也可以在休息后直接跳过来看它运行程序的视频。

如果你想知道基于 NAND 的计算机是如何工作的，你应该通过[这门在线课程](http://hackaday.com/2012/10/11/programming-tetris-by-first-building-a-logic-gate-then-a-computer-then/)来了解一下。
