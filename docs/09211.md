# 每周赢家:Stickvise

> 原文：<https://hackaday.com/2015/06/04/weekly-winners-stickvise/>

我们正在进行一场史诗般的比赛，为今年夏天的 Hackaday 奖参赛作品颁发 50，000 美元的奖金。本周，我们为 65 个不同的项目分发了一个小尺寸的 PCB 夹具。这实际上是从[开始的，是 Hackaday.io](https://hackaday.io/project/3287-stickvise-low-profile-soldering-vise) 上一个非常棒的项目。

获奖者名单如下，请查看他们的项目；去掉那些你觉得很棒的项目，留下你鼓励的话语作为对这些项目的评论。然后开始工作并[提交你自己的东西](https://hackaday.io/had2015)。在这些每周的赠品中，你的胜算很大。我们对你获胜的最佳机会的建议是润色你正在分享的信息——讲述你到目前为止已经做了什么和你打算做什么的全部故事。张贴一些图片，无论是原型的图片，你正在做的东西的渲染图，还是餐巾背面的手绘图。

通常我们会在这个宣布获胜者的帖子中发布下周的比赛。但是这一次我们改变了一些。在周一寻找一个分享接下来所有细节的帖子！
