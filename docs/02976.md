# 逆向技术:机械瞄准计算机

> 原文:[https://hack aday . com/2013/03/13/retro tech tacular-mechanical-targeting-computers/](https://hackaday.com/2013/03/13/retrotechtacular-mechanical-targeting-computers/)

![retrotechtacular-mechanical-computer](../Images/c5ec47ff20080babf5ada111384fc247.png)

这些海员周围站着的设备是一台美国海军目标计算机。它不使用电力，而是依靠[机械计算来调整舰炮](https://www.youtube.com/watch?v=mpkTHyfr0pM)的弹道。通过转动曲柄和其他输入机制设置多达 25 种不同的属性，让计算机自动计算击中目标所需的枪设置。这些参数包括船只及其目标的速度和航向、风速和方位，以及目标相对于船只的位置。一想到这台计算机的复杂性，就令人难以想象。

这个七集系列的第一集将在广告之后播出。该系列包括轴、齿轮、凸轮和差速器。听起来坐着挺无聊的，是吧？但是，正如我们从这种风格和训练电影的年份中所期待的那样，它在镜头中加入了大量简单的演示。
