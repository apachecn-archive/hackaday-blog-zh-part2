# 为微型直升机铸造新齿轮

> 原文：<https://hackaday.com/2012/10/17/moulding-new-gears-for-a-micro-helicopter/>

[![heli_09](img/5441846a8b982725833f7bf0ca9069f4.png "heli_09")](http://hackaday.com/?attachment_id=88247)

所以你的直升机模型的齿轮坏了，而且手边没有 3d 打印机。如果你需要你的小直升机马上起飞，[詹姆斯]写信告诉我们他的解决方案。你可能已经从标题中猜到了，他制作了一个微型模具，并用它制作了一个他需要的齿轮的复制品。考虑到印刷的复杂性或一些微小的减法，这个小齿轮变得非常漂亮！

广告之后的视频展示了完成这一程序的所有步骤。如果你想直接看结果，可以在 10:00 左右去看看完成的装备，最后看看飞行中的小家伙。如前所述，他不得不在模具加工后在齿轮中间钻一个孔，但这是唯一的加工操作。

直升机齿轮工作得很好，但一定要看看[xrobots] 上其他一些真正有趣的[项目，其中一些我们已经在这里介绍过了！](http://www.xrobots.co.uk/ "xrobots")
