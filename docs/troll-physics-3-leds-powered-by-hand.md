# 巨魔物理学:3 个手动发光二极管

> 原文：<https://hackaday.com/2013/10/09/troll-physics-3-leds-powered-by-hand/>

[Henryk]刚刚发给我们他的简单 LED 电路拼图的最新一集。在摄像机前，他将 3 个发光二极管的一个引脚焊接到不同的开关上。然后，他将三个组件放在手中，扳动每个开关，使相应的 LED 亮起。我们期待你在评论中的解释。

你可能还记得[Henryk]制作的另外两个视频(也是在休息后嵌入的)。第一个视频是一个简单的电路，由一个电阻、三个开关和三个 led 串联而成。当连接电池时，led 不知何故一次打开一个。第二个视频展示了相同的电阻/开关/led，这次是并联电路。打开第一个开关使第一个 LED 点亮，打开第二个开关使第二个 LED 点亮。

这里是我们介绍的其他几个巨魔物理项目:原始 LED 电路贴、[超级豪华版](http://hackaday.com/2012/05/27/ask-hackaday-troll-physics-super-deluxe-edition/)和[神奇的解决方案](http://hackaday.com/2012/01/27/followup-troll-physics-solved/)。
