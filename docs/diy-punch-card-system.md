# DIY 打孔卡系统，无需挂卡

> 原文：<https://hackaday.com/2015/07/30/diy-punch-card-system/>

有时你只是有些零件躺在那里，想用它们做点什么。[Tymkrs]有一个机器人裁纸刀，所以很自然地[他们制作了打孔卡](http://www.instructables.com/id/DIY-Paper-TapePunch-Card-Maker-and-Reader/)。但是，当然，他们需要一个穿孔读卡器，所以他们也做了一个。店里到处都是东西。

[剪影肖像](http://www.silhouetteamerica.com/)切纸刀是用来做剪贴簿的，但是有什么比穿孔卡片更能唤起过去的回忆呢？为了裁剪他们的数据，而不是可爱的小猫或花朵，他们编写了一些自定义代码，将 ASCII 字符转换成一排排的点。卡片完成了——你只需要清理那些没有完全切开的洞。这些都是臭名昭著的挂查德。

读取器由一块木头组成，有一个用于卡片的间隙和垂直钻孔，用于 led 和光敏电阻。这是电缆到一个螺旋桨开发板与一些简单的固件。我们会使用光电二极管或光电晶体管，因为这是我们的垃圾箱里的东西(因为它们的反应时间更快)，但当你有柠檬时，就做柠檬水。

好了，现在你有了一个穿孔卡片阅读器和记录器，你用它做什么呢？想到了密码存储。
