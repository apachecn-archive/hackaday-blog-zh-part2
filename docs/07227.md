# 与虚拟现实的互动让我们更加接近真实的全息甲板

> 原文：<https://hackaday.com/2014/10/03/interacting-with-virtual-reality-brings-us-even-closer-to-a-real-holodeck/>

我们的一位读者最近一直在玩虚拟现实，并提出了一个非常酷的测试版研究——用手进行虚拟互动。

使用 Oculus Rift、Leap Motion 控制器和 Unity 4.6 的测试版，[TomáMarianík]构建了一个物理交互的测试环境。 [Leap Motion](http://hackaday.com/2012/05/21/hackit-leap-motions-new-motion-sensor/) 控制器能够以极高的细节跟踪你的手指，这使他能够在测试环境中创建一双虚拟手，几乎完美地模仿他的动作。这里的黑客是让所有的工作在一起。

在下面的演示中，他展示了与全息菜单的互动，从解剖学上正确的人类身上抓取身体部分(感谢 Unity3D)，并操纵他的环境。

他承认这个软件还没有完全完善，但是他太兴奋了以至于不能展示这个概念。我们当然很高兴看到接下来会发生什么——一些触觉反馈手套和一些允许你四处走动的动作感应凉鞋怎么样？
