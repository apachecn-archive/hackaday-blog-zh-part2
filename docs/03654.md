# Monsterpalooza 的掠夺者套装包括超工程肩炮

> 原文:[https://hack aday . com/2013/07/24/predator-suit-for-monster palooza-include-over-engineered-should-cannon/](https://hackaday.com/2013/07/24/predator-suit-for-monsterpalooza-includes-over-engineered-shoulder-cannon/)

![off-the-hook-predator-suit](../Images/fffa710cb81783743e6b81b4c72d6b2b.png)

这套*掠夺者*套装在今年的 Monsterpalooza 会议上首次亮相。简直不可思议。但是肩炮才是真正吸引我们注意力的。事情是[完全机动化，包括声音和光](http://www.instructables.com/id/Building-a-killer-Predator-costume)射击效果。

大约两年前，我们看到了杰罗姆·凯尔蒂的能力。他展示了一个基于 Arduino 的电子动画平台，这是他为掠夺者肩扛炮组装的，根据掠夺者头盔的指向进行追踪。但是除了视频演示之外，没有太多关于实际构建的信息。这篇文章弥补了这一点。

这种质量的复制品很少是一个人的作品。一队粉丝加入进来促成了这件事。从另一名团队成员[Jerome]那里获得背包和佳能的模制部件后，他开始将支撑结构、电机和控制电子设备安装到可用的空间中。这意味着大量的铣削、切割和成型零件，如上面看到的支撑臂，它将伺服电机集成到矩形轮廓中。所有的控制器都装在背包里，电缆连到头盔和大炮上。
