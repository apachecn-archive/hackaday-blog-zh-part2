# [Ian Lesnet]湾区创客集会指南

可能已经过去了一个星期，但是前 Hackaday 校友和巴士海盗的发明者伊恩·莱斯内特(Ian Lesnet)为湾区制造商集会(Bay Area Maker Faire)做了一个很好的指南。

旧金山地区的 Maker Faire 吸引了 100，000 名制造商、修补匠、黑客和普通极客参加 DIY 和一般很酷的东西的集市。所有的常客都在那里，包括[杰里·埃尔斯沃思]和她的[64 号准将贝斯·基塔尔](http://hackaday.com/2012/05/23/jeri-builds-a-c64-bass-keytar/)以及各种各样的角色扮演者，包括一个[蒸汽朋克波巴·费特](http://blog.makezine.com/2012/05/22/cool-clothes-and-styles-of-maker-faire-attendees-part-2)和一对[太空陆战队员](http://blog.makezine.com/2012/05/21/space-marines-tower-over-maker-faire/)。大楼外面有一艘 40 英尺长的蒸汽船和令人惊叹的马修·里斯的德罗宁气垫船。

在与其他制造商的访谈中，[Ian]得到最多的建议是，“慢慢来。”在创客节期间，湾区有成千上万的建筑商，很容易让人不知所措。

如果你想知道的话，[伊恩]还在创客大会上从所有供应商那里买了一大堆超赞的施瓦格。Radio Shack 有一个装满随机组件的盒子，德州仪器(Texas Instruments)的[Kenneth]给了[Ian]一个 TI Launchpad，一个电容测试升压包，以及所有赠品之王，一个 [Chronos 手表](http://processors.wiki.ti.com/index.php/EZ430-Chronos)。

休息过后，你可以看看[Ian]上传的一些视频项目采访。来自数千个制作者的非常棒的作品。
