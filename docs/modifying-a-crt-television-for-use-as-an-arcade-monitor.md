# 改装阴极射线管电视用作街机监视器

> 原文：<https://hackaday.com/2013/06/17/modifying-a-crt-television-for-use-as-an-arcade-monitor/>

发现自己需要一个街机显示器[埃里克·赖特]转向这个古老的阴极射线管电视。问题是街机显示器和电视的操作方式不同，分辨率和刷新率都不同。[Eric] [改造了电视机，使其像街机显示器一样工作，但只取得了有限的成功。他希望再做一些修改就能找到一个完整的解决方案。](http://www.thetechnickel.com/video-games/crt-tv-as-arcade-monitor)

上图显示他在改造后的夏普电视上测试吃豆人游戏。熟悉游戏的人会立刻注意到有问题。我们看到了《吃豆人》和《幽灵》走过的大部分路线，但是迷宫本身却完全不见了。为了达到这一点，[Eric]查阅了电视和街机原理图，找出如何将复合同步和三个颜色通道直接连接到街机。通过这种方式，CRT 计时被强制符合游戏标准。问题是没有办法调整各个颜色通道的驱动和截止。这是[埃里克]希望在下一轮实验中解决的问题。

如果你已经有一个正常工作的街机显示器，但没有游戏柜[为什么不使用树莓派](http://hackaday.com/2012/12/17/using-arcade-monitors-with-the-raspberry-pi/)？
