【Korben】正在[用相框当蓝牙音箱](http://korben.info/recyclage-denceinte-vibrations.html) ( [译](http://translate.google.com/translate?sl=auto&tl=en&js=n&prev=_t&hl=en&ie=UTF-8&u=http%3A%2F%2Fkorben.info%2Frecyclage-denceinte-vibrations.html))。他为这个项目黑了一个[摇滚](http://www.avconceptproducts.com/en/rock-pocket-vibration-speaker-xml-398_402-944.html)。这是一种带有振动元件的设备，可以用来将任何中空物体做成扬声器。

通过掌握阿波罗导航计算机模拟器来娱乐自己。这是计算机[的 JavaScript 版本，用于阿波罗登月任务](http://hackaday.com/2013/09/10/retrotechtacular-the-apollo-guidance-computer/)的模块中。[感谢格雷戈里和保罗]

这里有一个小镜子附件，可以让你把你的笔记本电脑用作高射投影仪。[伊恩]称之为剪辑图。将其粘贴到网络摄像头上，并将键盘用作绘图表面。由于它只是使用摄像头，因此适用于现场演示和视频会议。我们想不通的是，为什么图像最后没有倒过来？

本指南将让你[把杨桃板变成 AirPlay 扬声器](http://lukse.lt/uzrasai/2013-12-airplay-wifi-speaker/)。

那些不擅长记住撞球游戏规则的人将会喜欢这个产品。它是一些附加硬件，使用颜色传感器来检测球何时被装入袋中。基于 [Raspberry Pi 的系统会自动为每场比赛的](http://www.libertygames.co.uk/blog/pool-table-scoring-tweeting-iphone-app/)打分。

我们花了太多时间坐在电脑前。如果我们有一台跑步机，也许我们会试着做一个【柯克的】[跑步机办公桌附件](http://redetermine.blogspot.com/2013/08/diy-pvc-ikea-treadmill-desk-built-this.html)。它由聚氯乙烯制成，并使用一些经过改造的变径配件来调节高度。看起来你在腰带前部失去了一点空间，但是如果你只是以步行的速度使用它，应该不会有太大影响。

只需几个蛤蜊，你就可以拥有自己的智能镊子。[Tyler]在一些防静电镊子上加了铜带。铜焊盘上焊接有导线，导线的另一端带有一些鳄鱼夹。将它们夹在你的万用表上，你就有了自己的电子镊子。
