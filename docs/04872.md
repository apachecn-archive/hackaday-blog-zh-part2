# 黑掉一些拨片

> 原文:[https://hackaday.com/2013/12/20/hack-some-picks/](https://hackaday.com/2013/12/20/hack-some-picks/)

![lawlpicks](../Images/13a070481e813a39d8294c2e7e44ef03.png)

你知道你可以用廉价的碳钢钢锯刀片制作自己的开锁工具，对吗？那么，如果你在每个网站上被这个黑客绊倒了，这些网站也告诉你阅读并崇拜[麻省理工学院开锁指南](http://www.lysator.liu.se/mit-guide/MITLockGuide.pdf)；现在是 T4 培养新爱好的时候了，因为我们很多人都有了额外的假期空闲时间。不像你为你想打动的女孩买的正宗手工制作的室内悬挂立式番茄花园，钢锯刀片不会削弱你的购买力。还有，信不信由你，[虽然](http://hackaday.com/2013/08/25/making-keys-after-the-apocalypse/) [我们](http://hackaday.com/2013/06/09/ask-hackaday-can-you-steal-a-car-with-a-mini-tesla-coil/) [有](http://hackaday.com/2012/10/01/picking-locks-with-toool/) [十三](http://hackaday.com/2012/07/21/picking-handcuffs-with-laser-cut-keys/) [摘](http://hackaday.com/2011/08/30/knock-lock-with-logic-chips/) [摘](http://hackaday.com/2011/01/19/cracking-a-manipulation-proof-million-combination-safe/) [那个](http://hackaday.com/2010/05/28/2010-sees-new-lock-impressioning-speed-record/) [是](http://hackaday.com/2010/02/15/electronic-key-impressioning/) [坐](http://hackaday.com/2010/02/03/foil-impressioning-defeats-security-locks/) [中的](http://hackaday.com/2009/10/06/how-to-crack-a-master-lock/)[一类](http://hackaday.com/2009/05/22/marc-weber-tobias-vs-medeco/)，这不在其列。

虽然该指南选择使用现有的精选作为模板，但你可以在网上找到大量的。追踪完镐后，下一步是固定钢锯刀片，用旋转工具雕刻出多余的部分，然后打磨边缘，去除任何锋利的部分。我们建议你小心不要让刀片在这里太热，否则你会改变它的晶体结构:也许我们的一位精通铁匠的读者可以更好地解释你在使用碳钢时应该瞄准和避免什么。像往常一样，戴上必要的安全头盔:你的眼睛很重要，休息后你会需要它们来看视频。

你怎么想呢?我们应该努力复兴开锁类别吗？我们知道，如果你想进入黑暗艺术，许多黑客空间都有开锁活动。通过[发送概述这些活动的提示](http://hackaday.com/contact-hack-a-day/)以及任何符合这一主题的其他内容，帮助我们开始工作。
