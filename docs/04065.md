# 解除武装:将枪支变成机械化的音乐乐队

> 原文：<https://hackaday.com/2013/09/16/disarm-turning-guns-into-a-mechanised-musical-orchestra/>

[佩德罗·雷耶斯]是一位居住在墨西哥城的视觉艺术家。他最近的项目叫做“解除武装”，这是一个独特的展览，将拆除的武器与雕塑和技术结合在一起。

墨西哥军队定期收缴枪支，通常要么被掩埋，要么被销毁。但这一次，一位在国家犯罪预防委员会工作的朋友想问问[佩德罗]他是否可以用它们做点什么。几百支不起作用的枪之后，[佩德罗]有了一个全自动的机械管弦乐队。文章没有深入介绍它是如何工作的，但看起来螺线管主要用于驱动各种仪器。整个设置由电脑控制，但没有太多关于界面设计的信息。

当被问及这个项目时，[Pedro]解释了他的理由:“‘解除武装’作为一个项目的美妙之处在于，它将音乐与雕塑结合在一起，同时也与技术结合在一起。这是对本可以夺走你我生命的金属的救赎。所以它们作为乐器更好。”

休息之后，观看由创造者项目制作的关于“解除武装”的 8 分钟纪录片。基于武器的管弦乐听起来相当不错！

这不是他第一次用火器做出不同的东西。库利亚坎是墨西哥西部的一个城市，是全国枪支相关死亡率最高的城市之一。因此在 2008 年，他开展了一个名为 [Palas por Pistolas(铲子换枪支)](http://pedroreyes.net/palasporpistolas.php)的项目，他们从库利亚坎人那里收集了 1500 多件枪支，将其熔化，然后锻造成铲子，以帮助一项植物倡议。交易枪支的人会收到代金券，他们可以用代金券购买电子产品或家用电器。
