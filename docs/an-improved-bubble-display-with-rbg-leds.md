# 一种改进的 RGB LEDs 气泡显示器

> 原文：<https://hackaday.com/2013/09/10/an-improved-bubble-display-with-rbg-leds/>

制作气泡显示器是一项艰巨的任务，但[Jay]利用迭代设计构建了这个令人印象深刻的(60 管，巨大的)气泡显示器。该显示器的功能是当气泡在流体中上升时，在每个管中分配气泡作为被照亮的像素。他的建造日志以对细节的高度关注，尤其是耐心，一步一步地完成了展示的建造。

[Jay]没有一头扎进去，把一些管子拼凑在一起，而是花时间研究其他气泡显示项目，包括几年前我们特别报道的一个项目，该项目衍生自[的另一篇黑客文章](http://hackaday.com/2008/07/29/the-liquid-display/)。他的原型开始时很小，以测试潜在的功能:是否使用水或甘油，气泵的时间和气泡的大小，以及其他一些功能。[Jay]甚至用每个测试的视频填充日志，这样你可以看到问题和解决方案在每个步骤中展开。

成品显示器拥有 60 个 30 英寸高的显像管，宽度为 64 英寸。[Jay]还在灯管交汇的每个边缘安装了 RGB LEDs，以更好地分配光线。您可以观看以下众多视频中的一个。
