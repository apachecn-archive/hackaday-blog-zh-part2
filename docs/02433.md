# 冰唱片在你能听之前需要在深冻中六个小时

> 原文:[https://hackaday . com/2012/12/31/ice-record-single-needs-6-hours-in-deep-freeze-before-you-can-listen/](https://hackaday.com/2012/12/31/ice-record-single-needs-six-hours-in-the-deep-freeze-before-you-can-listen/)

![ice-record](../Images/6ca539bd1636ecb2268960b1400ba2e5.png)

这种令人愉快的营销策略[要求听众凭空捏造自己的唱片](http://www.fastcodesign.com/1671478/a-record-made-of-ice-that-actually-plays)。“大声喊出来”乐队想用他们的最新单曲引起轰动。所以他们想出了如何用冰制作可播放的唱片。这样做的主要问题是，当冰开始融化时，凹槽立即开始退化。所以他们把一个唱片模具和一瓶水运送给了几个精选的听众(总共只有十个)。休息后在视频中听到结果。

如果你想为自己做这样的东西，我们可以帮你一点。这个模具是由硅树脂制成的，不久前我们还看到了[一份给模具制作新手的指南](http://hackaday.com/2012/10/17/reproduce-3d-printed-models-by-making-your-own-molds/)。原材料也不难找到。上面的项目尝试了几种不同的方法，发现用普通的蒸馏水可以得到最好的结果。不，最难的是弄清楚如何做你自己的主人。如果你在家庭实验室中找到了这样做的方法，[请告诉我们](http://hackaday.com/contact-hack-a-day/)！
