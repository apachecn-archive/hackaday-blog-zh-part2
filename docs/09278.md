# Hackaday 奖参赛作品:一部 DIY 智能手机

> 原文：<https://hackaday.com/2015/06/13/hackaday-prize-entry-a-diy-smartphone/>

它可能不会改变世界，但[泰勒]的 [DIY 智能手机](https://hackaday.io/project/5083-diy-smartphone)是一个很好的例子，说明你可以用现成的零件做什么。他用树莓皮、Adafruit 的几个零件和 3D 打印的外壳建造了一个完整的工作手机。

在 Tyfone 里面是一个树莓 Pi 型号 A，[一个阿达果 FONA 细胞模块](https://www.adafruit.com/products/1946)，[一个 PiTFT](https://www.adafruit.com/products/2097) ，除此之外别无其他。里面有一块 1200 毫安的电池，一个 3D 打印的外壳将所有东西放在一起。

对于操作系统，[泰勒]没有运行 Android 那只是针对 Raspi 2 的，Raspberry Pi 2 型号 A 还没出。相反，[泰勒]用 Python 写了自己的非操作系统。它可以发送和接收短信，打电话，拍照，连接 WiFi 网络，以及做 2003 年的诺基亚能做的一切事情。

[Tyler]制作一个视频，介绍他的 Tyfone 的所有功能。你可以看看下面。
