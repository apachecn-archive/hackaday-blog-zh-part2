# 利用 Raspberry Pi 实现无线交流控制

> 原文：<https://hackaday.com/2014/02/26/wireless-ac-control-with-the-raspberry-pi/>

我们以前看过[康纳]的一些作品，看起来他现在正转向产品设计。他为 Raspberry Pi 设计了一个适配器来控制几个无线墙上插座继电器，允许任何人用 Pi 来控制灯、咖啡机、电视和其他属于非常广泛的家庭自动化旗帜下的任何东西。

该系统使用 315MHz 无线电分线板来控制三个墙壁插座适配器之一中的继电器。Pi 上运行着一个脚本，它有一个非常漂亮的 GUI 来打开和关闭灯。

[康纳]的产品非常类似于智能手机控制的某种支持 WiFi 的墙上插头中继系统，为此，他决定将他的设备命名为 Belchin' Emo Switch。这个名字可能还需要改进，但是他卖三个树莓 Pi 控制的墙上插座，价格和品牌 a 的一个墙上插座一样。
