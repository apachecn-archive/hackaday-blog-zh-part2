# 房间的 3D 映射

> 原文:[https://hackaday.com/2013/06/08/3d-mapping-of-rooms-again/](https://hackaday.com/2013/06/08/3d-mapping-of-rooms-again/)

去年，我们看到了 Kinect 有史以来最酷的应用。它被称为 Kintinuous，[它又回来了](http://www.youtube.com/watch?v=D3yYjaLmiqU)，这次是 Kintinuous 2.0，有新的和改进的特性。

当我们第一次听说 Kintinuous 的时候，我们被震惊了。带有 Kinect 的计算机绘制大范围区域地图的能力具有多种多样的应用，如谷歌街景、创建定制反恐精英地图以及考古挖掘。不过，Kintinuous 1.0 有一个问题:扫描一个循环会创建一个不连贯的地图，循环的起点和终点会在不同的地方。

在 Kintinuous 2.0 的视频中，你可以看到一个超过 300 米长的巨大扫描，其中两个循环自动拼接成一个连续扫描。这是一个惊人的壮举，尤其是考虑到计算机在几秒钟内处理七百万个顶点。

不幸的是，看起来 Kintinuous 2.0 不会很快正式发行。这个 Kintinuous 的论文仍在审查中，围绕软件有一些“问题”,不允许回答是否和何时发布的问题。然而，一旦论文发表，任何人都可以自由地重新实现它，我们很乐意把它作为一个开放的挑战留给我们的读者。
