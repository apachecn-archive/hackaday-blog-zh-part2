# 最小的便携式码头

> 原文：<https://hackaday.com/2015/01/03/the-smallest-portable-pi/>

当你把一个非常小的树莓派克隆体塞进一个 Game Boy Advance SP 里，你会得到什么？我们不知道该怎么称呼它，但它可能是有史以来最好的便携式游戏机之一，能够在一个小小的翻盖显示器上运行从 Apple II 到 Quake III 的模拟器。

这不是我们第一次看到[frostedfires]在一个塞进游戏机的小系统上工作。[在](http://hackaday.com/2014/11/19/a-raspberry-pi-in-a-game-boy-advance-sp/) [bacman 论坛](http://www.bacman.co.uk/home/)上关于这个版本的最初帖子只是涵盖了基本内容——让一个 [Odroid W](http://www.hardkernel.com/main/products/prdt_info.php?g_code=g140610189490) 启动并运行，并将 Quake III 放在微型显示器上。现在构建已经完成，我们可以看看如何将一个 Raspberry Pi 克隆变成我们见过的最小的可移植项目之一。

在一个小型便携式仿真站中使用一个 Raspi 克隆体作为唯一的组件是不可能的，所以[frostefires]添加了一些其他的电子元件来使一切工作。这里有一个来自 PSP 的操纵杆作为鼠标，除了普通的 Game Boy 按钮之外还有几个额外的按钮，一个 USB 集线器，WiFi 适配器，扬声器和放大器，电池和相关的充电电子设备，以及一个处理所有输入的 Teensy 3.1。

这是一个非常令人印象深刻的构建，可以运行从 Apple II 到新一代任天堂游戏机和手持设备(包括 Game Boy Advance)的模拟器，但由于 HDMI 连接器位于外壳外部，[frostedfires]也可以将它用作一个小型的便携式媒体中心。看看下面的视频，看看这个游戏男孩的行动，玩马里奥赛车和 1080p 视频。
