# 一个廉价的可以拯救生命的 DIY 烟雾探测器

> 原文：<https://hackaday.com/2014/07/29/a-cheap-diy-smoke-detector-that-can-save-lives/>

一根有故障的电线、一根被丢弃的燃烧着的香烟或一个没用完的卷发棒都可能引发火花，吞噬附近的一切，直到剩下的只是一堆堆闷烧的灰烬。这就是烟雾探测器如此重要的原因。它们是救生设备，可以在无声但致命的一氧化碳开始进入之前唤醒睡在里面的人。但是，如果没有人在家，警报开始鸣响，会发生什么呢？这地方在主人不知情的情况下被烧成了灰烬。

因此，当[马丁]购买了一个电池供电的烟雾探测器，并[将其安装好，以通知他压电警报器何时被激活](http://harizanov.com/2014/07/diy-internet-of-things-fire-alarm/)时，自动火灾报警器的发展继续进入无线互联网连接的领域。

他的家庭自动化系统(一个运行节点为红色的树莓程序接口)连接到一个 T2 时髦的基于 ATTiny84 的传感器，并无线传输数据，将信息重定向到他的手机。可以发送短信、电子邮件和[推送通知](https://www.pushbullet.com/)。一旦压电警报器开始鸣响，系统就会提醒他已经探测到烟雾，他应该尽快检查。

电子设备完美地安装在箱子里，等待房间里的任何烟雾干扰亮起。除了救生功能和即时推送通知之外，让这个项目更好的是，它的价格很便宜。当只需要一个或两个传感器和一个树莓皮就可以黑掉房子里已经有的火警警报时，没有必要购买一个全新的、昂贵的 Nest protect。
