# 在 CD 上录制唱片

> 原文:[https://hackaday.com/2012/07/23/cutting-a-record-on-a-cd/](https://hackaday.com/2012/07/23/cutting-a-record-on-a-cd/)

![](../Images/70f3c75be523ac5e5a58393d35786575.png "CD")

黑胶唱片是一种令人惊讶的简单技术，但令人惊讶的是，我们还没有看到许多利用将音乐录制到黑胶磁盘上的便利的构建。【sering son】[自制~~乙烯基~~聚碳酸酯切割器](http://www.youtube.com/watch?v=y1I3Gi3X3HU&feature=context-vrec)在家录制自己的唱片。令人印象深刻的是，他这样做的时候，零件就放在那里。

就像过去的专业和过时的唱片切割机一样，[Seringson]的构建使用两个以 45 度角安装的扬声器驱动器来再现立体声音轨。这些驱动器中的每一个都通过用一根极其锋利的针将左右声道刻成 CD 的聚碳酸酯来再现它们。从视频来看,[Seringson]的唱片切割机的音频质量相当好——足以重现 1940 年 78 RPM 唱片的声音，但还不足以重现最近制作和掌握的东西。

让我们印象深刻的是[Seringson]能够用他周围的碎料做一个唱片切割器。现在我们将耐心等待，直到组合唱片/CD 发行。
