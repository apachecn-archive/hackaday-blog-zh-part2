# 绘制运动皮层

> 原文:[https://hackaday.com/2012/05/14/mapping-the-motor-cortex/](https://hackaday.com/2012/05/14/mapping-the-motor-cortex/)

![](../Images/a7d8e6bba60c9d53849b570952402241.png "IR_head")

[Bruce]给我们发来了康奈尔大学 ECE4760 班的另一个精彩的期末项目。你在上面看到的是一个由 36 个近红外发光二极管组成的阵列，为了光谱学的目的照射进这个年轻人的大脑。根据大脑活动(血流)的不同，光的反射也不同。在这个项目中，他们正在绘制他们的运动皮层，并使用 java 应用程序在 PC 上显示出来。你可以看到整个钻机，以及休息后的两个视频中的读数。

当这条消息传来时，我们的作家之一杰西·康登也加入了进来。

> 嘿，我曾经在宾夕法尼亚大学实习过。使用两种频率的近红外光，它们都穿透皮肤和骨骼，一种通常被血液反射，另一种被含氧血液反射。因为你的大脑实际上调节血液流向正在使用的部分，所以你可以通过观察血流量来了解大脑的活动，但你也需要了解大脑是否真的在使用血液，所以氧合会给你一个完整的画面。额叶皮层是一个测量的好地方，因为头骨的那个部分没有头发，它给你情绪反应和“啊哈！”你解决问题的时刻。
> 
> 很久以前的一篇文章说，该系统将被用作测谎仪，因为当你撒谎时，你会同时想到真相和谎言，并表现出活动的增加。
> 
> 但是很难对反应进行分类，因为你不能通过关闭大脑来真正建立“基线”活动
