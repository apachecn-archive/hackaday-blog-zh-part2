# 在车库里用压力清洗机液压成形

> 原文：<https://hackaday.com/2014/08/13/hydroforming-in-the-garage-with-a-pressure-washer/>

听说过液压成形吗？这是一种在极高压力下用水将金属板成型的制造工艺。这不是你在家能做的事……【to】除非你是【柯林·福尔泽】。

[液压成形](http://en.wikipedia.org/wiki/Hydroforming)的工作原理是通过水(方便地，不可压缩的)对模具内的金属板均匀地分配压力。许多汽车零件都是以这种方式制造的。典型的系统在 15，000 PSI 左右运行。

在建造了一个巨大的脉冲喷气发动机(配有黄油)[对法国放屁](http://youtu.be/7Ydv9Ef-99I)之后，【科林】从 YouTube 的一个评论中得到想法，试图在家里进行液压成形——弯曲金属板来制造巨大的臀部不是那么容易的。另一方面，液压成形是一个非常简单的过程。将一些金属板焊接在一起，添加一个管件来连接您的廉价压力清洗机和吊杆——液压成型金属部件。

哇——我们是不是漏了一步？典型的液压成形使用模具来推动金属板以形成形状。[科林]的第一次尝试是无模具的，这限制了他能够形成的零件的复杂性——尽管他能够为他的加大马力的机动踏板车制造一个非常好的锥形排气管…

在下一个视频中，他承诺将这个过程变得非常实用，所以我们很期待看到他的成果。用混凝土制作模具是可行的，所以这是可能的。
