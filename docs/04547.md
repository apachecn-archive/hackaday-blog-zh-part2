# 39 Raspberry Pi 3D 扫描仪

> 原文：<https://hackaday.com/2013/11/13/39-raspberry-pi-3d-scanner/>

[Richard]刚刚在他可笑的 39 Pi 3D 扫描仪上发布了一个指令！ 没错。39 个单独的覆盆子馅饼，带摄像头模块。

但是为什么呢？嗯，[理查德]喜欢 3D 打印、Arduinos、覆盆子馅饼和他的孩子。他想为他的孩子制作一些 3D 模型(因为照片都是上世纪的了)，所以他开始研究 3D 扫描仪。不幸的是，他发现的几乎所有设计都要求被摄对象静坐一会儿——这是他 2 岁的儿子不喜欢的。因此，他开始思考一种一次性拍摄所有照片的方法，让他能够即时生成 3D 模型，而无需等待。

他最初打算购买 39 台便宜的数码相机，但不想将所有图像保存在单独的 SD 卡上，因为提取所有图像会非常繁琐。另一方面，使用覆盆子，他可以从网络上抓取所有的覆盆子。因此，他着手建造一个非常棒(也有点贵)的真人大小的 3D 扫描室。完整的细节可以在他的博客上找到，地址是[www.pi3dscan.com](http://www.pi3dscan.com/)

休息之后，请在格罗宁根 2013 年创客大会上留下来观看它的运行！
