# DIY 散热印刷电路板

> 原文:[https://hackaday.com/2013/09/12/diy-heatsinking-pcbs/](https://hackaday.com/2013/09/12/diy-heatsinking-pcbs/)

![pcb heatsink](../Images/1239e31d9ac54774c1e4d69611d22761.png)

我们之前已经介绍过很多自己动手制作 PCB，但是[这个视频指南](https://www.youtube.com/watch?&v=PHNphoskSTQ)增加了一个简单的方法来为大功率设备散热，我们认为你可能会觉得很方便。

这是一个非常简单的过程，[CNLohr]使用它来防止他的小型 RGB LED 项目过热。它首先使用易于获得的硅导热片作为基底，将其应用于铜箔上。然后，他通过让铜片和经过[改进的层压机](http://www.youtube.com/watch?v=ZBM7p78B27k)运行几次，将墨粉转移电路图案应用到铜片上。他注意到，你必须将硅胶片的塑料背面贴在铜箔上，以防止层压机将其咬碎。

在典型的氯化铁蚀刻过程完成后，他使用 220 号砂纸去除墨粉图案。通常使用钢丝绒，但由于硅树脂的敏感性，砂纸可以更好地避免剥落痕迹。

在之前，我们已经[介绍了【CNLohr】，因为他在他的工作室里做了一些顶级的教程——这为视频提供了一个既迷人又分散注意力的背景。这个有点长(大约 20 分钟)，但是非常彻底，从头到尾贯穿了整个过程。休息之后来看看。](http://hackaday.com/2011/09/12/glass-pcbs/)
