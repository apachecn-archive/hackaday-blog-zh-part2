# 又一个 Arduino 闪光灯，实际上很酷

> 原文:[https://hackaday . com/2012/08/06/yet-other-arduino-blinkenlight-this-real-蛮-酷/](https://hackaday.com/2012/08/06/yet-another-arduino-blinkenlight-thing-actually-pretty-cool/)

![](../Images/df835bf40b615d2f18380e3914623d86.png "Tiny")

在塔斯马尼亚 Linux 用户群邮件列表上，[Hoolio]看到有人抱怨他们即将到来的黑客空间最终沦落为 Arduino 粉丝俱乐部。[胡里奥]问 Arduino 怎么了，这个可怕的塔斯马尼亚人回答说，“有太多无聊的闪光灯，没有足够多真正酷的东西。”[胡里奥]将此视为一项挑战，[创造了他自己的 Arduino blinkenlight 项目](http://hooliowobbits.blogspot.com.au/2012/08/tiny-this-is-not-space-invayders.html)，在一个 5×5 的 led 矩阵上模拟*太空入侵者*

电路板只是一个蜂鸣器，25 个发光二极管，10 个晶体管，一个罐子和一个按钮。在游戏开始之前，一个 LED 追踪器在显示器的外围被追踪，它的速度可以由底池控制。当按钮被按下时，游戏开始，允许[胡里奥]用罐子左右移动他的船，并用按钮发射他的激光。

是的，这是一个为 Arduino 的闪光灯阵列编写的游戏。不过，这并没有削弱体型。如果这是一个神话般的米色，回到 1978 年，我们会喜欢 LED 版的*太空入侵者*就像[美泰的](http://www.handheldmuseum.com/Mattel/FB.htm) *[足球](http://www.handheldmuseum.com/Mattel/FB.htm)一样。*

休息之后可以看[Hoolio]的游戏试玩。
