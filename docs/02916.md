# 动感单车大灯以开阔的路面作为展示

> 原文:[https://hackaday . com/2013/03/06/dynamic-cycle-head-use-the-open-road-as-display/](https://hackaday.com/2013/03/06/dynamic-bicycle-headlight-uses-the-open-road-as-a-display/)

这东西太酷了，看起来几乎像假的。但是[马特·理查森]不是一个骗子。他实际上为你的自行车建造了一个可以被称为[的俯视显示器。他把它称为前灯，因为它借用了类似的功能。它安装在车把上，从自行车前部射出光线。但它不仅仅是一个电池和一个灯泡，它使用微型投影仪来赋予光线一些意义。在休息后的视频中，他在纽约的街道上炫耀。](http://www.youtube.com/watch?v=Nfk1-XMASrk)

到目前为止，他只显示与旅行速度有关的信息，但证据就在那里，等待着一个辉煌的新用途。喂投影仪的是树莓 Pi 板。对于这个原型[马特]安装它，随着便携式手机充电器的作用，电源，在一大块硬纸板绑在自行车框架内。

如果你想自己做这个，小心 BOM 的价格标签。他用的那台投影仪售价超过 400 美元。我们想知道你能否用一个旧手机屏幕和这个 diy 电影放映机拼凑出一个基本的替代品？
