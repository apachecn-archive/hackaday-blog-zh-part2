# 逆向技术:制造瓷绝缘体

> 原文:[https://hack aday . com/2015/06/16/retro tech atacular-making-ceramic-insulators/](https://hackaday.com/2015/06/16/retrotechtacular-making-porcelain-insulators/)

这是一部由通用电气公司制作的无声电影，描述了多种电力线用瓷绝缘子的制作过程。在上釉和烧制之前，熟练的工匠手工模制、成型和雕刻这些电网的重要部件。

这一时期的瓷绝缘体由瓷土、球粘土、燧石和长石制成。在干法工艺中，原料被粉碎并筛选成细粉，然后压入模具，通常具有 Play-Doh Fun 工厂式的效果。一旦成型，他们被手工修剪，以消除翅片和防水板。然后，在立式车床上旋转的同时，对工件进行喷涂上釉。

其他类型的绝缘体是通过湿法生产的。粘土在泥浆泵中混合，泥浆泵是一种宽容的机器，它接受各种形状、大小和湿度的废料，并挤出大圆木中湿的、可加工的材料。大块的原木在陶轮上成型或压入模具。一旦它们接近干燥，这些作品就会在大师的手中形成最终的形状。然后它们被上釉并在一个巨大的高温窑中烧制。
