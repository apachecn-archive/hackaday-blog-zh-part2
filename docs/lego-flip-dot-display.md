# 乐高翻转点显示

> 原文：<https://hackaday.com/2015/04/25/lego-flip-dot-display/>

我们不需要提到翻转点显示器是可怕的。除了在转换时，它们不使用任何能量，即使在最明亮的日子里也能看到，并且有一点老派的魅力。因此，理所当然的是，由乐高制成的[翻转点显示屏非常棒。见鬼，它甚至拼出了“棒极了”。](http://www.eurobricks.com/forum/index.php?showtopic=108403&#entry2197352)

[https://www.youtube.com/embed/OyC_Qh6B0hw?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/OyC_Qh6B0hw?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

通过在模板上排列单个单元砖来打开或关闭像素，从而对显示器进行编程。一组手指抬起，新模板滑入，手指下降到模板上以设置显示点盘。听起来很简单，对吧？

单像素机制本身就足够有趣:
