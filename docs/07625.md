# 3DS 自制频道和自定义固件

> 原文：<https://hackaday.com/2014/11/21/3ds-homebrew-channel-and-custom-firmware/>

任天堂一直对允许独立和自制开发商为他们的游戏机制作游戏非常谨慎，3DS 也不例外。它被锁定了，一些 3DS 和控制台黑客已经花了数年时间寻找一种方法，可以轻松地允许任何人运行未签名的代码。[那一天终于来了](http://gbatemp.net/threads/introducing-ninjhax-a-nintendo-3ds-homebrew-exploit.374233/)。该漏洞被称为 NINJHAX，它允许任何人安装家酿频道，这是 3DS 家酿开发世界中所有精彩内容的储存库。

最新的漏洞利用依赖于零售游戏《立方忍者》中的一点代码来运行未签名的代码。这个游戏包括一个关卡编辑器，允许玩家通过二维码和 3DS 的摄像头分享不同的关卡。通过精心制作这些二维码中的一个，3DS 获得了运行自制频道的能力

如果这个漏洞听起来很熟悉，那么你是对的。打开 Wii 进行自制开发的最常见方式[是 Smash Stack](http://wiibrew.org/wiki/Smash_Stack) ，这是在 Super Smash Bros. Brawl 中发现的一个漏洞。这种利用也通过修改定制阶段来实现，并为 Wii 的大量自制开发打开了大门。

在下面的视频中，[smea]通过启动 Cubic Ninja，进入二维码关卡编辑器，然后加载自制游戏，展示了他的漏洞。启用此漏洞的游戏副本 Cubic Ninja 是此漏洞所必需的。上周，你可以在易贝和亚马逊上花几美元买到立方体忍者。今天，价格稳定在 50 美元左右，一些非常愚蠢或非常渴望的人支付高达 300 美元。如果你已经有了这款游戏，你只需要获得家酿初学者工具包，生成一个二维码，然后开始安装未签名的代码。[所有说明均可在【smeal】网站](http://smealum.net/ninjhax/#qrcode)上获得。
