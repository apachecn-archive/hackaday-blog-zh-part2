# 3D 打印假手

> 原文：<https://hackaday.com/2013/09/09/3d-printed-prosthetic-hand/>

![hand](img/679be37c9bd3a897535e8fd269e9fe94.png)

3D 打印——有望实现小规模制造和定制零件——是假肢行业的理想选择，但迄今为止，假手一直是一个非常非常难攻克的难题。[Joel]一直致力于[开放双手项目](http://www.openhandproject.org/)，该项目旨在使机器人假肢能够为制造商、研究人员和截肢者所用。

尽管手内部的机制相当简单——DC 齿轮马达收回钢缆“肌腱”——[Joel]能够将所有这些设备装入一个非常小的体积，比真正的肉基手大不了多少。为了激活手上的机械肌肉，用户只需弯曲前臂的几块肌肉。这些电信号被一套定制的电子设备接收，并告诉张开的手该做什么

在[Joel]的[Indiegogo 视频](http://www.youtube.com/watch?v=6dI-dNE2yQ0)中，他在截肢者[Liam Corbett]的帮助下回顾了是什么让他的机器人手工作的。从美学角度来看，张开的手比[利亚姆]的双叉钩有了很大的改进，加上视频中展示的灵巧，可能会更有能力。
