# 生命计给出了视频游戏健康的真实生活测量

> 原文:[https://hack aday . com/2013/05/01/life-meter-gives-a-real-life-measure-of-video-game-health/](https://hackaday.com/2013/05/01/life-meter-gives-a-real-life-measure-of-video-game-health/)

![water-light-health-bar](../Images/518ed40f14ba49abe1e64d9c59fbbee3.png)

这种黑客通过将你的视频游戏角色的[健康计显示为一列发光的水](http://www.youtube.com/watch?v=ywfZsYApj_M)来实现虚拟现实。

休息之后你会发现嵌入的构建视频真的很棒。柱子是一根透明的管子，一端用手拧紧排水管配件固定。这使得[b 层]可以附加一个他为这个项目建造的柔性气囊。一个致动器将一个铰接板推靠在膀胱上，以升高和降低管中的水位。

光是这一点就已经相当令人印象深刻了，但是[b ayer]还做了更多的工作。他在仪表底部使用了一个四通接头。一个叉连接到膀胱，另一个允许使用水族馆泵注入空气。配件的底部有一个透明的窗口，以便 RGB LED 阵列可以照射到掺有荧光笔墨水的水中，以获得彩色辉光。为了将整个事情整合在一起，他编写了上面看到的自定义控制界面。
