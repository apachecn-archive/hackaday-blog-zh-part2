# Hackaday 奖参赛作品:从 Q 到 NAND

> 原文：<https://hackaday.com/2015/07/24/hackaday-prize-entry-from-q-to-nand/>

末日就要来了，据我所知，没有多少人的车库里有半导体工厂。在世界末日之后，我们将需要计算机，而[matseng]的 Hackaday Prize 项目就是这样——[一个用分立元件构建计算机的框架](https://hackaday.io/project/6668-aytabtu-discrete-computer)。

这个项目的启示录有点夸张，但通过用晶体管、电阻和二极管构建数字设备，人们可以学到很多东西。[matseng]的计算机的构建模块非常简单:他用三个电阻、四个二极管和一个 NPN 晶体管构建了一个与非门。这些与非门可以组装成任何形式的数字逻辑。你再也找不到更好的功能完整性的可视化例子了。

像这样的项目必须自上而下和自下而上地进行。为了从高级到 1 和 0，[matseng] [建造了一个汇编器](https://hackaday.io/project/6668-aytabtu-discrete-computer/log/21214-assembler-for-this-cpu)，[建造了一个仿真器](https://hackaday.io/project/6668-aytabtu-discrete-computer/log/21463-cpu-emulator)。这个项目日志中列出了关于什么是[指令集的一些想法，目前【matseng】正在为一个有八个寄存器的哈佛架构而努力。对于一台受内存容量限制的计算机来说，这是一项繁重的工作，但就雄心而言，Hackaday 奖中没有多少项目可以与这台微型、巨型计算机相媲美。](https://hackaday.io/project/6668-aytabtu-discrete-computer/log/21059-instruction-set)
