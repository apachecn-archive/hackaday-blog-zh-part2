# 好吧，谷歌一下。发动吉普车！

> 原文：<https://hackaday.com/2015/08/21/okay-google-start-the-jeep/>

当[wizardpc]买他的吉普车时，它配有一个 Avital 3100L 汽车报警系统；但在它开始出故障后，他需要更换它。所以他选择了一个相同类型的新报警系统——然后他决定破解它。

在安装新的报警系统(Avital 5103L 组合装置)时，他意识到有一根额外的电线，当接地时，它会启动车辆——Avital 在这种特定型号的软件问世之前就已经包括了硬件升级。得分。

从那时起，这是一个相当容易的黑客。他所需要的只是一个 Raspberry Pi 2、一个中继板和几行非常简单的代码。在移动端的东西是黑客的集合；他在 Android 手机上使用 Tasker 为 Google Now 添加了一个特殊的命令。他告诉谷歌“启动吉普车”，几秒钟后，她打开了。

如果 Google 能公开一些钩子，这样我们就可以在 Google Now 上添加我们自己的功能，而不需要为此使用应用程序杂耍[wizardpc],这不是很好吗？如果你有自己的一套 Google Now 黑客工具，我们很乐意听听。[给我们发个提示](http://hackaday.com/submit-a-tip/)！
