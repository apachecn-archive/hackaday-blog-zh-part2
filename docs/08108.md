# 用 Tron Radio 学习 Python

> 原文:[https://hack aday . com/2015/01/20/learning-python-with-tron-radio/](https://hackaday.com/2015/01/20/learning-python-with-tron-radio/)

[5 伏特迷]已经完成了他的 Arduino 项目，但是从来没有用 Python 做过，当然也没有用 GUI 做过。有一天听了网络电台后，一个项目的新想法诞生了:一个带有小触摸屏的树莓 Pi 用户界面，显示 soma.fm 曲目。[终于完成了](https://www.youtube.com/watch?v=QzvNIHI-k-4)，这是对 Python、Pygame 和用 Pi 驱动小显示器的一个很好的介绍。

播放 soma.fm 流由 [mpd](http://www.musicpd.org/) 和 [mpc](http://linux.die.net/man/1/mpc) 处理，而驱动 2.8″ TFT LCD 的任务由 [fbtft Linux framebuffer 驱动](https://github.com/notro/fbtft/wiki)处理。这给[5v 迷]留下了创建一个 GUI，一些按钮，以及如何播放一些流的任务。这意味着在 Inkscape 中绘制一些按钮，但这些按钮确实很糟糕，所以[5v 迷]放弃了，打开了电视。 *Tron Legacy* 正在播放，给了他灵感来完成他的 Tron 主题音乐播放器。

[5 伏特迷]的工作成果是几百行 Python 和 Pygame 以及一些多色皮肤都被包裹在一个 Tron 主题中。它看起来很棒，工作起来也很棒，是对 Python 和 Pygame 的很好的介绍。
