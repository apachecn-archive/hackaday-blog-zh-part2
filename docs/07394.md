# 多蒂翻转点钟

> 原文:[https://hackaday.com/2014/10/24/dottie-the-flip-dot-clock/](https://hackaday.com/2014/10/24/dottie-the-flip-dot-clock/)

我们为什么如此喜欢用不恰当的技术制造的低效、嘈杂的时钟？通过观看[David Henshaw]发给我们的关于[Dottie(翻转点钟)的视频(如下),自己回答这个问题。](http://www.dhenshaw.com/Art/Dottie/start.html)

但是除了作品本身，我们真的很喜欢构建日志中的进展，从“我该怎么做？”一个装箱完成的项目。

这个建筑的另一个令人惊叹的方面是一个丙烯酸外壳和一系列巧妙编写的软件可以让一个项目看起来有多好。从正面看，你永远也猜不到背面是一个由试验板和以太网线组成的(难以置信的)老鼠窝。那些随机的开关模式会让你忘记所有的线路。

和伺服转向，螺线管驱动的钟声只是甜。我们确信我们很想在现实生活中听到它们。
