# 没电的电池反弹的原因

> 原文:[https://hack aday . com/2014/09/05/the-reason-dead-batteries-bounce/](https://hackaday.com/2014/09/05/the-reason-dead-batteries-bounce/)

在过去的几年里，非常消息灵通的人已经能够简单地通过掉落来判断碱性电池是好是坏。当从坚硬表面上方一两英寸处掉落时，优质电池不会反弹，有时会直立着落地。另一方面，没电的电池会反弹。感谢[李]和他的几个朋友，[我们现在知道为什么会发生这种情况](https://www.youtube.com/watch?v=qrGV7zKEdtU)。

在和他的几个朋友出去玩的时候，[李]能够把所有关于为什么没电的电池会反弹的争论浓缩成两种理论。第一种理论，即“反弹理论”称，没电的电池会增加电池中的除气作用，增加电池中的压力，从而增加电池本身的弹簧常数。第二种理论是“反反弹理论”，认为电解质的凝胶状特性起到了一种质量阻尼器的作用。

[Lee]设计了一个实验来测试弹跳电池的排气“弹跳理论”。一个物体——在这种情况下是一个黄铜棒——被扔在好电池和坏电池上，而不是扔电池。没有区别。即使在钻孔排出电池内部的任何气体后，黄铜块也会以同样的方式从好电池和坏电池上反弹。

这意味着死碱性电池反弹的原因是由于电解质。[Lee]切开几节 AA 电池，发现好电池中的电解液是一团化学物质。在废电池中，同样的电解液硬化成固体。[李]把这比作一个防反弹锤。

最后，在我们大多数人通过[【达夫·琼斯】](https://www.youtube.com/watch?v=PRf9JTg3QwA)的视频了解到如何反弹没电的电池一年多后，我们有了答案。这是电解液中的一种化学变化，它将电解液从一种粘性物质变成一种固体，从而使用完的电池反弹。
