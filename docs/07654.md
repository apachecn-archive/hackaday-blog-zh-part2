# 鸡肉动力钟摆

> 原文：<https://hackaday.com/2014/11/24/chicken-powered-pendulum/>

每隔一段时间，我们都会收到一个非常可爱的链接，我们只需要把它贴出来。比如:[这个视频](http://youtu.be/poQ7L-6XkY4)来自【Ludic Science】。这是一个来回踢钟摆的发条鸡玩具。不多不少。

但在你开始尖叫“不是黑客！”在下面的评论中，请思考一下这里发生了什么。鸟的内部有一个弹簧，一个齿轮，通过鸟脚的运动卡住和释放(一个[擒纵机构](https://en.wikipedia.org/wiki/Escapement))。这使得整个装置非常类似于一个真正的摆钟。

见鬼，小鸡玩具本身很酷。它的鼻子很重，所以在正常情况下它会前倾。但是当它上紧时，向前倾斜会触发擒纵机构并使其跳跃，在此过程中向后倾斜并重置扳机。头重脚轻的鸡就是一个倒立摆！

如果你愿意的话，看看他用非常好的低技术方式创造的枢轴:一根弯曲的金属丝，穿过一根短铝管，由几颗珠子固定住。当然，其他枢轴摩擦更小，但像这样使用杆和套筒的优点是钟摆运动被限制在一个平面内，因此它永远不会错过鸡爪。

我们唯一的遗憾是他错过了(差那么多)视频结尾明显提到的“裸女”。
