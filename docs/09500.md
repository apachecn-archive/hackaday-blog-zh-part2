# Hackaday 奖参赛作品:EyeDrivOMatic

> 原文：<https://hackaday.com/2015/07/08/hackaday-prize-entry-eyedrivomatic/>

Hackaday 奖有很多项目旨在改善我们这些残疾人的生活。一个好的 3D 打印假肢是比赛的自然想法，盲人和聋人的项目也是如此。[帕特里克·乔伊斯]、[史蒂夫·埃文斯]和[大卫·霍普金森]正在帮助一种更加衰弱的疾病:运动神经元疾病，或 ALS。[Steve]和[Patrick]都患有 ALS，他们正在进行一个项目，该项目将利用他们眼睛的运动来移动他们的轮椅。

该项目始于[Patrick]几年前的一个想法——为什么不使用商业眼球追踪技术来驾驶轮椅。眼球追踪技术是一个相当好解决的问题，但是由于一些难以解释的原因，没有明确的方法将这个系统连接到轮椅上。

在过去的几年里，[Patrick]自学了 Arduino 和 Processing 来制作一个设备原型，该设备将连接到运行眼球跟踪工具的计算机，并将其转化为伺服运动。一个小型的 3D 打印装置连接到[Patrick]轮椅的操纵杆上，只需要一点点代码，他就可以通过看屏幕来驾驶他的轮椅。这是对 3D 打印和不起眼的 Arduino 的伟大使用，但这种技术以前并不存在，这绝对令人印象深刻。

因为[Patrick]几乎可以构建他想要的任何硬件，所以他还添加了一些简洁的功能。这种建造的“大脑盒”需要两个伺服输出，但[帕特里克]出于其他目的增加了两个。他将在椅子的一侧安装一个 Nerf blaster，但他也有其他想法，添加一个风扇，一个机器人手臂，甚至是红外或射频发射器；他可以只用眼睛控制电视。
