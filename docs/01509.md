# 蒸汽动力六足机器人

> 原文:[https://hackaday.com/2012/09/03/steam-powered-hexapod/](https://hackaday.com/2012/09/03/steam-powered-hexapod/)

![](../Images/bcac51eb3df9cc21689be3128eed0a5e.png "When I Roll Into The Wild Wild West; When I Stroll Into The Wild Wild West")

这台[全机械六足机器人](http://www.bricobidules.com/index.php?post/2012/08/15/Une-fourmi-a-vapeur) ( [翻译](http://translate.google.com/translate?sl=auto&tl=en&js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&u=http%3A%2F%2Fwww.bricobidules.com%2Findex.php%3Fpost%2F2012%2F08%2F15%2FUne-fourmi-a-vapeur))经过精心策划，建造精美。它不会伸长脖子去看前方。那是推进机器的蒸汽机的烟囱。

从机械角度来说，腿是最难的部分。那只是因为蒸汽机不是从零开始建造的。这是一辆由固体燃料片驱动的 D14。它输出高转速但低功率，因此齿轮比被设置为 286:1，以充分利用其输出。

腿本身是由黄铜棒制成的。这些锚定在一个更大的齿轮的一侧，有一个支点，允许腿垂直滑动。在休息后的剪辑中可以看到最好的结果。当驱动轮旋转时，枢轴点向前移动身体，直到脚被杆的滑动抬起。它最终看起来比一些更灵巧的六足动物更优雅，但它缺乏转弯能力。
