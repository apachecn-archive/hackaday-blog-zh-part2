# 利用公共数据制作激光切割地图

> 原文:[https://hack aday . com/2014/04/21/using-public-data-to-make-laser-cut-maps/](https://hackaday.com/2014/04/21/using-public-data-to-make-laser-cut-maps/)

![laser cut maps](../Images/c3962f841e019d727891c45d4491c02d.png)

如果你有激光切割机，你会想看看这个精彩的教程。[Steven Smethurst]已经想出了如何提取公共地图数据，并将其转换成一个文件[,随时可以用激光切割成你选择的材料。](http://www.instructables.com/id/Laser-cut-wooden-maps-with-public-data/)

在他的例子中，他正在使用温哥华的开放数据目录来构建他的地图。为了做到这一点，他使用了一个名为 [TileMill](https://www.mapbox.com/tilemill/) 的程序，你可以从 MapBox 免费获得，这是一个设计你自己的交互式地图的伟大软件，最棒的是，你可以从各种来源导入数据，例如温哥华的开放数据！

您可以导入形状(。SHP)文件，并将它们作为图层添加到 TileMill 中。从那里您可以操作您的地图，调整细节，然后导入为. SVG 或。DXF 文件准备激光切割。

除了如何做的指导，他还录制了一个深入的视频教程，你可以在休息后看看。
