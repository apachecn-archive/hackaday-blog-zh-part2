# 气动火箭人

> 原文：<https://hackaday.com/2014/05/03/pneumatic-rocket-man/>

这是一个有趣的“五四与你同在”项目……一个气动的波巴·费特喷气背包——发射——人体模型！

因其疯狂的[四旋翼飞行器项目](http://hackaday.com/2014/03/22/its-not-2015-yet-but-marty-and-his-hoverboard-are-already-here/)而闻名的，想要尝试改变气动动力。他设法获得了一个灭火器，并通过自制的 PVC 空气输送系统将其放在他忠实的测试假人的背上——这次他穿上了一套完整的[波巴·费特]服装。

他用一个电磁驱动的喷水阀来释放压力，这是用一个 N-FET 来控制的。一个 AVR 微控制器发送一个 500 毫秒的脉冲来触发它，并处理遥控发射台，你可以驾驶它四处寻找最佳飞行位置。电磁阀是 1”逐步下降到 1/2”连接端口进入[波巴的]脊柱，它有一个 3/4”套管，像一个气动活塞。为了好玩，他甚至用 Boba-CAM 从可怜的人体模特的角度捕捉 25 英尺以上的发射！
