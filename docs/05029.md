# 打印机升级循环数控机床不仅仅使用步进电机

> 原文:[https://hack aday . com/2014/01/11/printer-up-cycled-CNC-machine-uses-than-just-the-stepper-motors/](https://hackaday.com/2014/01/11/printer-up-cycled-cnc-machine-uses-more-than-just-the-stepper-motors/)

有时候你从照片上就知道这将是一个非常酷的项目。当大多数人从旧打印机中回收零件时，他们通常会扔掉剩下的部分。在这种情况下，[Shane] [利用整个打印机来建造他的数控机床](http://justcheckinm8.blogspot.co.nz/)。

他从一台旧的惠普 2500C A3 打印机开始，他计划只回收部分打印机。当他把它拆开的时候，他意识到这个底盘会成为他真正的数控机床的一个很好的框架！考虑到这一点，他很快改变了他的游戏计划，让每个轴都在打印机内部。

他在 X 轴和 Z 轴上都使用了常规的滚珠轴承抽屉滑轨，并覆盖了一个巧妙的铝角设计，以防止任何可能的芯片卡住它们。另一方面，Y 轴利用了打印头滑架的原始轴滑道。每个轴都由螺杆驱动，使用打印机回收的步进电机。

一个 Arduino UNO 位于项目的核心，它带有一个 Protoneer CNC 防护罩来控制步进驱动器。他还包括手动控制的紧急停止、保持、恢复和取消按钮。

这是一个伟大的项目，也是一个利用现有资源进行项目的惊人例子。休息后留下来观看 it 打印演示！
