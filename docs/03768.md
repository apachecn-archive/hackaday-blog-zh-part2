# DIY 23mph+电动滑板

> 原文：<https://hackaday.com/2013/08/10/diy-23mph-electric-skateboard/>

游览纽约的最佳方式是什么？如果你问[papo2110]，他很可能会建议你[自己制作一个高速、长距离的电动滑板。](http://endless-sphere.com/forums/viewtopic.php?f=35&t=37230)如今，浏览任何在线创客社区，你都会被十几个电动汽车项目绊倒。大约 18 个月前，甚至在广受欢迎的[启动 Kickstarter](http://www.kickstarter.com/projects/170315130/boosted-boards-the-worlds-lightest-electric-vehicl) 之前，【papo2110】就开始拼凑一副牌了。他的冲浪板使用一个无刷电动马达、一辆遥控车 [ESC](http://en.wikipedia.org/wiki/Electronic_speed_control) (配有刹车)和一个链传动装置来驱动他在中央公园周围最高时速 23 英里。

然而，这个项目最令人印象深刻的壮举是通过迭代设计进行的不知疲倦的修改。甲板得到了铝和碳纤维升级。丰富的 8S Headway LiFePo4 取代了较小的 6S 配置。甚至添加了灯光。随着建造的进行，板子正以每小时 27 英里的速度推进:只用一个马达。休息之后，带上你的头盔和晕车药，准备看一些视频。

如果四个轮子太多了，你想要更危险的速度，看看几个月前的 E-trike。
