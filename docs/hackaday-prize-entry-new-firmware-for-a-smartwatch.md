# Hackaday 奖参赛作品:智能手表的新固件

> 原文：<https://hackaday.com/2015/07/16/hackaday-prize-entry-new-firmware-for-a-smartwatch/>

智能手表是下一件大事。没人知道我们会用它们做什么，但这从未阻止一种产品成为最热门的商品。WeLoop Tommy 不是 Apple Watch，不是 Moto360，也不是微软的 Band。然而，这是一块漂亮的智能手表，具有清晰的记忆显示屏和持续几天以上的电池。作为他的 Hackaday 奖参赛作品，[Krzysiek] [正在为 WeLoop Tommy](https://hackaday.io/project/4510-open-source-sportsmart-watch) 制作开源固件，这将增加其他智能手表所没有的功能。

这个项目是对 WeLoop Tommy 智能手表的完全逆向工程。[Krzysiek]正在将所有东西拆成最基本的组件，并找出 RAM、闪存、按钮、LCD 和加速度计如何连接到处理器。之后就是定制固件的时候了。

[Krzysiek]已经有了一个测试应用，可以在 Sharp memory 显示屏上显示[OSSW]。不多，但硬件扎实。有了正确的固件，WeLoop Tommy 将能够使用重新调整的硬件和开源固件做 Android、苹果或微软 smartwatche 可以做的一切事情。
