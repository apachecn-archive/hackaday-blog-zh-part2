# 把华彩乐段变成最后一步

> 原文:[https://hack aday . com/2015/02/19/turning-a-cadenza-into-a-finishing-move/](https://hackaday.com/2015/02/19/turning-a-cadenza-into-a-finishing-move/)

当你把决斗钢琴和 2D 格斗游戏混合在一起时，你会得到什么？毫无疑问，一些复杂的控制，但也是一个可怕的表演平台！艺术家[Cyril]和[Eric]制作的“ [*声音战士*](http://www.foobarflies.io/pianette/) ”装置的确切意图是，两个对立的政党可以在街头战士比赛中一决雌雄，他们的钢琴演奏能力映射为攻击组合和龙拳。

为了将一架钢琴变成一个荣耀的街机游戏杆，[Cyril]和[Eric]需要一种方法来记录何时以及弹奏了什么音符，然后将这些数据转化为格斗游戏本身的命令。首先，他们对不同类型钢琴的内部运作做了功课。无论他们要设计什么样的数字增强系统，都必须在不影响钢琴功能的情况下工作。

有许多可能的方法来记录钢琴的使用时间，虽然有几种方法可以达到预期的目的，但在他们做出决定之前，需要写下并讨论每个传感器的利弊。他们考虑的一些选项包括琴键本身的压力传感器，检测钢琴中单个琴锤运动的加速度计，甚至是通过计算分析任一乐器发出的声音的麦克风。最后，他们选择实现小而精确的压电爆震传感器，并将其固定在每个按键的内部机构上。这些乐器在演奏时可以记录微弱和强烈的音符，而不会改变乐器的自然声音。

![fighterPiano_2](../Images/4b559ad6901506add3cf413e1c0799be.png)

在为 PS2 决定一个街头战士迭代来开发项目的其余部分后，他们不得不玩一点实际的游戏来感受动作的命令列表。他们想构思一种方法来将演奏的音符映射到控制器，但不是直接的“按键到按钮”的方式。这个想法是，如果有人擅长弹钢琴，他们也会擅长在游戏中执行动作。所以他们必须弄清楚一组音符和和弦如何转化为移动角色或攻击。

我强烈建议查看他们从头开始构建安装时的详细演示。除了令人着迷之外(他们准备这个项目是为了争取时间重开巴黎的一个历史遗址)，你会发现他们开发的一切都是开源的。完成的装置听起来很棒。你可以在下面的一场真实决斗中看到它的动作:
