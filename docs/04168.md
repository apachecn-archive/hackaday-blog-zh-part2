# DIY 金属探测器

> 原文：<https://hackaday.com/2013/09/28/diy-metal-detector/>

想找个轻项目教年轻黑客一些很基础的电子学？这里有一个快速简单的周末项目，[一个简单的金属探测器](http://www.instructables.com/id/Simple-metal-detector/)！

我们都知道 555 定时器非常有用，并在各种项目中出现，但您知道金属探测器也是其中之一吗？[vonPongrac]偶然发现了这本方便的指南，这是一本关于 [50 555 电路](http://www.talkingelectronics.com/projects/50%20-%20555%20Circuits/50%20-%20555%20Circuits.html)的免费电子书，其中包含许多很酷的项目想法，包括一个简单的金属探测器电路。这是一个非常基本的概念，使用一卷铜线作为自制的[扼流圈](http://en.wikipedia.org/wiki/Choke_(electronics))——当金属或磁铁靠近线圈时，它会改变输出频率，而 555 定时器反过来会改变输出声音，提醒你附近有金属的存在。

休息之后是测试阶段的视频。如果你手头没有 555(tisk tisk ),但仍然想要一些寻宝的乐趣，你也可以[基于 Arduino](http://hackaday.com/2013/07/25/build-your-own-metal-detector/) 建造一个。
