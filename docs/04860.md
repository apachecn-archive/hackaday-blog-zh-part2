# Fubarino 竞赛:LED 矩阵游戏机

> 原文：<https://hackaday.com/2013/12/19/fubarino-contest-led-matrix-game-console/>

![pixels](img/c485788321eab89cf410a71751c17926.png)

我们的 [Fubarino 竞赛](http://hackaday.com/2013/12/03/new-contest-win-one-of-20-microchip-fubarino-boards/)的大量参赛作品终于开始涌入举报热线。好事也是一样， *这是投稿的最后一天。你还在等什么？* 我们刚刚过了参赛截止日期。

最近的一个来自[Vojtak]，他创造了一个看起来很棒的极简主义游戏机，除了纯粹的意志力、令人印象深刻的决心和一个 Arduino。8×8 红色 LED 矩阵由出色的 Max7219 显示驱动器驱动，3 轴加速度计和电池充电电路填补了这一空白。在软件方面，[Vojtak]为他的主机编写了许多应用程序，包括 Snake，一个迷宫游戏，以及许多使用内置加速度计的东西。

作为我们 Fubarino 竞赛的参赛作品，[Vojtak]需要将我们的 URL 实现为复活节彩蛋。通过输入 Konami 代码并进入控制台的图像查看器，你有四个额外的槽来保存你的作品，这些作品最初填充了类似于本文标题的图片。这次提交的最令人印象深刻的复活节彩蛋来自迷宫游戏。乍一看，没有什么奇怪的，但在巨大的迷宫中滚动后，你可以看到“HACKADAY”。COM”用像素写的。当我们建造树篱迷宫时，提醒我们这样做。
