# 已经令人印象深刻的狼獾爪子现在被高压充电

> 原文:[https://hack aday . com/2013/05/31/已经印象深刻-狼獾之爪-现在用高压通电/](https://hackaday.com/2013/05/31/already-impressive-wolverine-claws-now-energized-with-high-voltage/)

![wolverine-claw-high-voltage](../Images/7265fc87cd32e5c7f24b503aa1638024.png)

几年前[詹姆斯]制作了一套非常惊人的狼獾爪子复制品。它们被一根绑在他手指间的杆子绑在一起，所以当他抓住它时，除了爪子你什么也看不见。此外，尾巴的设计让它们看起来像是从他的皮肤里出来的，他做出了一个惊人的复制品。但他们也相当功利，正如你在演示/如何做的视频中看到的那样，当干草球从镜头外扔向他时，他用长矛刺它们。机械商店的粉丝会喜欢学习这些是如何制作的。

最近，在看到[我们自己的雷神之锤产品](http://hackaday.com/2013/05/02/high-voltage-thors-hammer-mjolnir-at-80000-volts/)后，他决定更新这个项目。他从一个旧油炉中抢救出一个巨大的变压器，开始着手工作。他不知道这东西能输出什么样的电压，但这并不妨碍他给这对爪子接上电线，让[放出火花](http://www.thehacksmith.ca/2013/05/electrified-high-voltage-wolverine-claws.html)。他甚至创造了一个雅各布的梯子效果，把爪子放在一个狭窄的角度。
