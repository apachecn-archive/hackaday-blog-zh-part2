# 电子树新娘礼物

> 原文:[https://hackaday.com/2013/09/16/electron-tree-bridal-gifts/](https://hackaday.com/2013/09/16/electron-tree-bridal-gifts/)

[马克]刚刚给我们发来了这个有趣的李奇腾伯格数字的例子，或者更普遍的说法是被捕获的闪电。

他昨天刚刚和他美丽的妻子(查理)结婚，他们想为他们的婚礼做些不同的事情。他们选择捕捉丙烯酸球体内部的闪电。如果我们自己这么说的话，这真是一份令人印象深刻的礼物！

有趣的是，我正在读一本解释这种现象的疯狂科学书。这些 Lichtenberg 人物是通过将一束高能电子束射向一片丙烯酸树脂制成的。许多电子被困在丙烯酸树脂内部，形成一个电荷平面。当丙烯酸物体被接地螺柱击中时，会形成放电路径，所有电子都会逃逸，在其轨迹上留下一条完全独特的闪电路径。

不幸的是，要制造这些，你需要一个直线加速器；一台非常昂贵的机器，[马克]很幸运地在工作中使用了它。然而这对夫妇并没有止步于此，他们还使用 PIC12F1501 微控制器设计了带灯的底座来完成礼物！

休息之后看看他们是怎么做出来的！提个醒，电子发射时视频声音很大！如果你戴着耳机，把音量调低。
