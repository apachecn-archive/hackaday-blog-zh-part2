# 决斗的机械公牛

> 原文:[https://hackaday.com/2012/07/03/dueling-mechanical-bulls/](https://hackaday.com/2012/07/03/dueling-mechanical-bulls/)

![](../Images/8c48d158c104a6579304fbe5ca8a5a88.png "dueling-mechanical-bulls-madagascar-institute-colin-stands")

你能比其他人在机械公牛上呆得更久吗？谁在乎！我们想知道你是否有能力建造自己的决斗机械公牛。在休息后观看视频中的开发阶段后，我们认为您会同意项目的建设部分比结果的“运动”更有趣。但是，我们不能看比赛而不喜形于色。

这个项目是由纽约布鲁克林区外的艺术团体马达加斯加研究所开发的。上面展示的场景是今年谷歌 IO 大会上的装置，两个参赛者可以在同一时间以相同的方式在相同的硬件上一决高下。你可以在背景中辨认出墙上的一个标志。它充当记分牌，有两个红色箭头，其中一个会亮起，以识别从公牛身上被扔出的失败者。

如果这是用 Bullduino 驾驶的话……也许明年吧。
