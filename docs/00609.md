# 为视障人士向现实生活中的对象添加语音标签

> 原文:[https://hackaday . com/2012/06/01/为视障人士在现实生活中的物品上添加语音标签/](https://hackaday.com/2012/06/01/adding-voice-labels-to-real-life-objects-for-the-visually-impaired/)

![](../Images/2ba2b07aa594699afbfcf69c10035feb.png "Touch Sensors-GadgetNews-small")

[Matt Oppenheim]来信与我们分享他的作品。他一直在为视障人士修改电子设备的接口。它从卡式卡座开始。正如[Matt]指出的那样，许多视障人士使用磁带来阅读有声书籍和报纸。[Matt]在按钮上添加了一些触摸传感器，这样他就可以在用户感觉到每个按钮时宣布它们是什么。这使得他们能够快速了解设备的布局。

在发现卡带播放器上简单的界面很快就学会了，从而不再需要他的附加物后，他决定追求稍微复杂一点的东西。[Matt]着手修改一个数字收音机，增加了许多不太友好的按钮。正如你在休息后的视频中所看到的，他很好地完成了这件事。
