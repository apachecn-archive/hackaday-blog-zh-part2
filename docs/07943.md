# 对分引擎使可爱的灯-仍然曲柄

> 原文：<https://hackaday.com/2014/12/31/bisected-engine-makes-cute-lamp-still-cranks/>

作为迈向著名的 Top Gear V8 咖啡桌的初级步骤，[英国茶]将一个[小型单缸发动机改造成了一个台灯](http://www.instructables.com/id/Small-Engine-Lamp/?ALLSTEPS)，它利用活塞的机械驱动来开关。在这个黑客的制作过程中，没有身体健康的引擎受到伤害，因为这个引擎已经是一具尸体了——非常适合(茶叶先生)在家里支撑和展示。

遗憾的是没有光剑，他花了 30 分钟在钢锯上劈开圆筒，然后用喷砂清理所有的铁锈、油漆和内部粘糊糊的东西。一旦清理干净，他就自己重新粉刷。在油漆和清漆之间，他计算出他在金属上增加了 20 层。

接下来，他创造了一些木头部分和湿成型的皮革，后来他把它们染成黑色。他对沃尔玛的新灯比对马达更不感兴趣，他解剖了灯的电子元件并给它接线。

轴上没有曲柄，扭开灯或关灯看起来有点笨拙，但是，只需要足够的压力来推动锁定的~~瞬时~~按钮，它似乎工作得很好。对于任何想自己动手制作的读者来说，报废的压缩机和气体动力工具在垃圾场相当常见，而且几乎免费。基于引擎的项目开始时可能会令人生畏，如果你在结束时还需要一个工作引擎的话。在这样一个项目中，你通常只把发动机作为一种建筑材料来使用，熟悉它们是一个简单的入门方法。
