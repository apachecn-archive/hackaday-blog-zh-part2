# Floreo:电子纺织品和移动服装

> 原文:[https://hack aday . com/2014/05/09/floreo-e-textililes-and-moving-clothing/](https://hackaday.com/2014/05/09/floreo-e-textililes-and-moving-clothing/)

![pedul](../Images/ec0de06255ea4da963265a00d8ad4e56.png)[Alica]和[Jerika]是亚利桑那州立大学数字文化项目的大四学生，他们希望将传统上由男性主导的东西变得对女性更加友好。他们选择了电子纺织品，这是最常见的非常前卫，几乎无法接近的大量发光二极管和滑稽的机制。他们最初的设计反映了这一点，有多个 LED 灯条和巨大的肩垫。然后，他们发现了 Flexinol 形状记忆致动器线[，并发现这可能是一种更“软”的技术与高级时装的整合](http://techdivas.weebly.com/)。

[Jerika]和[Alica]选择创造一种电子花，能够在形状记忆合金的帮助下开花。当电流通过 Flexinol 电线时，它会收缩。女孩们将这些金属丝缝到激光切割的织物花瓣上，在 LiliPad Arduino 的帮助下，创造了一朵繁荣的织物花。

虽然由于电子设备的怪异和过早烧断电线，他们无法完成他们的礼服，但他们确实成功地创造了一个展示预期效果的花别针。
