# 计划 B:基于开源粉末的 3D 打印机

> 原文:[https://hack aday . com/2014/08/22/plan-b-an-open-source-powder-based-3d-printer/](https://hackaday.com/2014/08/22/plan-b-an-open-source-powder-based-3d-printer/)

![Open Source Powder Based 3D Printer](../Images/b18214121c46a29bee3de7f784e03fbe.png)

3D 打印机有各种形状和尺寸。最广为人知的是 FDM(熔融沉积成型)风格，这是最容易适应消费级机器。我们仍在等待一些更先进的 3D 打印技术的广泛应用——所以你可以想象当[Yvo de Haas]告诉我们他的基于开源粉末的 3D 打印机时，我们有多兴奋！

基于粉末的 3D 打印是商业行业中最经济、最易于使用的技术之一，因为它有一个奇妙的地方——不需要支持材料！它们的工作原理是将精细的粉末层铺在一起，然后通过激光烧结或使用类似喷墨头的粘合剂将粉末粘合在一起。正因为如此，周围的粉末充当了你可能需要的任何复杂几何形状的支撑——你可以在这种类型的机器上打印任何东西。

[Yvo]刚刚完成了他自己版本的这种 3D 打印机，称为[计划 B](http://ytec3d.com/plan-b/) 。机械上类似于常规的 3D 打印机，his 能够沉积精细的粉末，然后使用黑客攻击的惠普喷墨墨盒将它们结合在一起。休息之后来看看。
