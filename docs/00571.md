# 用乐高酿造啤酒

> 原文：<https://hackaday.com/2012/05/30/brewing-beer-with-lego/>

![](img/61fea0b7d5c94fe467b207fac4ce7f22.png "coil")

[Matt]发了一组 YouTube 视频，带我们参观了他的乐高思维风暴控制的酿酒厂。

[Matt]正在使用一个 [RIMS 酿造装置](http://www.hbd.org/kroyster/definition.html)，该装置将麦芽浆再循环并加热，以从谷物中提取更多的淀粉。这导致了糖化醪中的[美拉德反应](http://en.wikipedia.org/wiki/Maillard_reaction)，产生了更浓郁的麦芽味。

为了控制他的轮辋设置，[Matt]正在使用一个乐高 Mindstorms 砖，在他的管道上安装了几个乐高温度传感器。乐高为适当的 RIMS 设置提供了所有的温度和泵控制，非常适合不想用 Arduino 或其他微控制器板的自制者。

作为一个小题外话，精明的黑客读者会注意到我们的[啤酒黑客类别](http://hackaday.com/category/beer-hacks/)少得可怜。现在快到夏天了，是开始酿造的最佳时间。如果你有一瓶啤酒，一定要让[送来](http://hackaday.com/contact-hack-a-day/)。

休息过后，你可以看到所有[Matt]的 RIMS/LEGO brewery 视频，或者你可以查看他的 YouTube 频道。
