# 3D 打印注塑模具

> 原文：<https://hackaday.com/2013/09/15/3d-printed-injection-molds/>

布达佩斯大学的一个团队通过使用 Stratasys 3D 打印机成功创建了一个用于原型制作的功能性[注射模具。](http://blog.stratasys.com/2013/08/21/3d-printed-injection-molds-rapid-manufacturing/)

原型注塑模具价格昂贵。它们通常由钢或铝加工而成，由于大多数模具的复杂几何形状，这既昂贵又耗时。【Jozsef Gabor Kovacs 博士】在布达佩斯大学高分子工程系工作，在那里他想出了利用 3D 打印生产原型模具的创新方法。

使用 Objet Connex 3D 打印机在数字 ABS PolyJet 光聚合物塑料中打印模具。所用的注射材料是聚缩醛；其具有相当低的 175℃的熔点。通过使用这种方法，他们能够在不到 24 小时内从原型模具变成测试部件。我们甚至都不愿意去想从一家机械工厂发货会有多贵。
