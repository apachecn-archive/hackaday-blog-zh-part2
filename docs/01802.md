# LED 照明同构键盘看起来和听起来一样好

> 原文:[https://hack aday . com/2012/10/05/led-illuminated-同构-keyboard-looks-as-good-as-it-sounds/](https://hackaday.com/2012/10/05/led-illuminated-isomorphic-keyboard-looks-as-good-as-it-sounds/)

![](../Images/39c6949e87b212d0bc6ec0254924a5e3.png "rainboard")

与钢琴上的传统乌木和象牙不同，同构键盘将按钮排列在网格上。这使得每一个和弦都有相同的形状，要移调一首曲子，只需将手指向左或向右移动几个位置。[Brett Park] [送来了一个他建造的装载了 led 的同构键盘](http://rainboard.shiverware.com/Rainboard_-_DIY_Dynamic_Rainbow_Isomorphic_Keyboard)，我们认为它是音乐家们寻求从玩 MacBook 升级的完美工具。

[Brett]的键盘主体是由一片丙烯酸树脂制成的。在为每个[透明的街机按钮](https://www.adafruit.com/products/471)钻了 64 个孔之后，【Brett】将他的六边形键盘的侧面弯曲成一个看起来非常坚固的外壳。

硬件方面，[Brett]用了一个 [64 按钮 Arduino 盾](http://spikenzielabs.com/SpikenzieLabs/Button64Shield.html)和一个 [Sparkfun MIDI 盾](https://www.sparkfun.com/products/10587?)。每个按钮后面的 RGB LEDs 是通过乐器外部生成的 MIDI sysex 信息控制的，这使得它非常适合从您想要的任何软合成器获得一点视觉反馈。

在休息后的视频中，你可以查看灯光模式以及[Brett]的即兴表演。注意，所有的和弦都是相同的形状，改变音调只需要[Brett]将手稍微移向一侧。
