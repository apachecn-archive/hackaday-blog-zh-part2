# 安装在吉普车上的前视红外照相机

> 原文:[https://hackaday.com/2013/10/26/a-jeep-mounted-flir-camera/](https://hackaday.com/2013/10/26/a-jeep-mounted-flir-camera/)

[![jeep](../Images/bc07d6010d3cc6cabc9347c0adf30601.png)T2】](http://hackaday.com/wp-content/uploads/2013/10/jeep.jpg)

[埃迪·扎里克]又来了，把他的吉普牧马人改装成让我们都嫉妒的东西。这一次，他设法从一辆老式卡迪拉克上获得并安装了前视红外照相机。它真的是一个前视红外热成像相机，而不仅仅是一个近红外黑客。凯迪拉克在 HUD 上使用了这项技术，但[埃迪]决定将其连接到他的仪表板屏幕上。他也没有满足于简单地面对它，而是将它安装到一个[戈莱特](http://www.golight.com/)探照灯底座上。他在屏幕下安装了操纵杆，让他可以控制方向。

[埃迪]在这个项目上花了大约 500 美元，这似乎很多，但当你考虑到一个新的前视红外相机的成本时，就不是了。我们很想知道他是在哪里找到这么多东西的！也许他去了当地的废品收购站？如果你知道这种零件的好来源，请在评论中告诉我们！

之前我们报道过【埃迪的】[加压水龙头](http://hackaday.com/2013/04/12/jeep-wrangler-gets-pressurized-water-right-out-of-the-bumper/)，[防风雨键盘输入](http://hackaday.com/2013/09/05/custom-car-keypad-entry/)，以及[其他各种黑客](http://hackaday.com/2013/07/05/packing-a-jeep-wrangler-full-of-hacks/)。我们期待着看到他接下来会在他的吉普车上增加什么。
