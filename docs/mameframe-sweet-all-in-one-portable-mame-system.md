# game frame–甜蜜的便携式 mame 系统

> 原文：<https://hackaday.com/2015/05/28/mameframe-sweet-all-in-one-portable-mame-system/>

电子游戏爱好者[NEIN]热爱 MAME。他不太喜欢 MAME 的一件事是搬动又大又重的 MAME 橱柜。那么，如果你想在旅途中玩游戏，你会怎么做呢？[NEIN]决定开发一款便携式 MAME 解决方案，将所有的东西都放在一个盒子里，这样几乎没有设置时间就可以开始玩了。他称之为“ [*”的 MAMEFrame*](http://www.instructables.com/id/MAMEFrame-The-battery-powered-MAME-system/?ALLSTEPS) ”。

这可能看起来是一个标准的双人 DIY 控制器，然而，它绝不是。显示器位于机柜内，这是一个视频投影仪，通过 HDMI 电缆连接到 Raspberry Pi。[NEIN]选择使用树莓 Pi 而不是大型 PC 来帮助保持事物的轻便和简洁。这两者就像是天生的一对。投影仪有内置电池和 USB 端口。Raspberry Pi 由投影仪 USB 端口提供的 5 伏电压供电，使该设备完全便携和无线。只要把它放在桌子上，指向一面墙，你就准备好引导吃豆人到 256 级了！

你知道吗，树莓派最早的黑客之一是 MAME 制造的。
