# 复古的:照相排字机，机械奇迹

> 原文：<https://hackaday.com/2013/09/17/retrotechtacular-linotype-machines-mechanical-marvels/>

在本周的回顾中，我们将[关注 Linotype 机器](http://www.youtube.com/watch?v=6wHiddZOfa8)；带来印刷媒体大规模生产的机械奇迹。

1876 年一个寒冷阴沉的日子，住在美国的德国发明家[[ott mar mergen thaler](http://en.wikipedia.org/wiki/Ottmar_Mergenthaler)]找到了[James O. Clephane]，他要求一种更快捷的方式来制作法律辩护状。报纸打字机有各种各样的专利，但是效果不太好，所以[Mergenthaler]开始着手新的设计。传统上，铅字组是在一台机器上铸造，然后在另一台机器上压印来产生文字。在火车上[Mergenthaler]想，为什么不把机器结合起来呢？由此，一种革命性机器的想法诞生了。

Linotype 机器有一个矩阵库，矩阵是创建嵌条的字符模具——铸造 ***型线的名称。*** 操作员用键盘输入一行文字，然后释放出相应字母的矩阵。这些然后被转移到铸造站，在那里[类型的金属](http://en.wikipedia.org/wiki/Type_metal)被铸造成一个叫做[热金属排版](http://en.wikipedia.org/wiki/Hot_metal_typesetting)的过程中的矩阵。然后，矩阵被返回到图书馆，文本的铸造线被冷却、移除，并用于印刷媒体的大规模生产中的压印。这听起来很简单，但现在意识到整个机器是机械自动化的；只要你继续用铅字金属填充它，你就可以简单地通过在键盘上打字来继续制造子弹。

这些机器从 19 世纪晚期一直使用到 60 年代和 70 年代，直到被更有效的平版胶印和计算机排版所取代。

休息之后，看看 1960 年代的精彩纪录片，你会惊叹于机器的机械运作。如果你没有 35 分钟来吹，至少检查一下 1:30 到 6:45 的基本概况。但是你可能会看得停不下来。

如果你想亲眼看看这些神奇的机器，而你又恰好住在英国，那你真是时好时坏！每年九月的第一个星期六，惠廷顿出版社都会举办一年一度的开放日，届时你可以看到这些机器在运转。不幸的是，这意味着你刚刚错过了今年的机会。但是把它放在你明年的日历上，因为这是为数不多的仍然用活版印刷的印刷机之一。

对于我们观众中的美国人来说，在巴尔的摩工业博物馆有一个正在运作的项目，尽管你不太可能看到它在运作。不过，不用担心，你可以去看看 T2 的电影《T3》，它在全国有很多放映活动。其中一些甚至包括一个真正的 Linotype 机器演示！
