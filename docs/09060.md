# 在 30 秒内破解组合锁

> 原文：<https://hackaday.com/2015/05/17/cracking-a-combo-lock-in-under-30-seconds/>

忘记组合锁的密码了吗？如果你手头有一台 3D 打印机、一个 Arduino、一个伺服系统和一个步进电机——你可以很容易地[制作自己的组合断路器](http://samy.pl/combobreaker)。它能够在 8 次或更少的尝试中解开任何主密码锁。

创作者[Samy Kamkar]是一名隐私和安全研究员，他绝对喜欢寻找安全漏洞。事实上，我们刚刚在我们自己的[Hack a Day Prize:Los Angeles event](http://hackaday.com/2015/04/20/hackaday-los-angeles-event-develop-your-hacking-superpowers/)上听到了他的消息，在那里，他谈到了他内置在手机充电器中的无线键盘嗅探器[。](http://hackaday.com/2015/01/14/keystroke-sniffer-hides-as-a-wall-wart-is-scary/)

他之前向我们展示了如何使用他制作的[在线计算器](http://null-byte.wonderhowto.com/how-to/crack-any-master-combination-lock-8-tries-less-using-calculator-0161629/)在 8 次或更少的尝试中[破解组合锁。这个项目只是自动化整个过程的延伸。](https://www.youtube.com/watch?v=09UgmwtL12c)

一如既往，他在他的构建日志视频中对该项目进行了极其详尽的解释——包括设计 3D 打印部件！如果你想建立自己的系统，花费不到 100 美元，你可以从他的 GitHub 中获取所有必要的信息和源文件。
