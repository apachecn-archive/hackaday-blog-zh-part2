# 黑客 21-万圣节黑客第二部分

> 原文:[https://hack aday . com/2014/10/31/hack let-21-Halloween-hacks-part-2/](https://hackaday.com/2014/10/31/hacklet-21-halloween-hacks-part-2/)

我们问了，你听了！上周的 Hacklet 以在 [Hackaday.io](http://hackaday.io/) 上征集更多万圣节主题项目而告终。一些伟大的黑客上传了令人敬畏的项目，本周的 Hacklet 就是关于他们的。我们的每一个特色项目都是在过去 7 天内上传到 [Hackaday.io](http://hackaday.io/) 的。

[![masseffect2](../Images/9574cf4f85170278b3df885dcf7d1033.png)](http://hackaday.io/project/3234) 质量效应在【特维斯特涅科的】 [5 天 SpeedBuild 质量效应装甲](http://hackaday.io/project/3234)中遇上蠢朋克。顾名思义，[TwystNeko]只用了 5 天就造好了盔甲。乙烯-醋酸乙烯酯(EVA)泡沫被用来制作服装的大部分。通常 EVA 泡沫需要密封。为了节省时间，[TwystNeko]跳过了这一步，只是刷上了一些金色的丙烯酸涂料。实际的删减是基于[TwystNeko]找到的一个在线模板。为了完成盔甲，[TwystNeko]使用了定制的[蠢朋克·盖伊·曼努埃尔头盔](http://hackaday.io/project/3247)。不错！

[![rat](../Images/c26bfe1ca16acbc0009a0ee561bdfd08.png)](http://hackaday.io/project/3236) 【格里夫】凭借[鼠鬃毛](http://hackaday.io/project/3236)赢得本周最恐怖项目。从邪恶的疯狂科学家实验室的书中，格里夫基于牙刷和振动传呼机马达制造了一个标准的刚毛机器人。他用派对商店里的一只小橡胶老鼠的尸体完成了这个硬毛机器人。老鼠确实让机器人移动得慢了一点，但对他的儿子来说仍然很有趣。[Griff]计划使用一个 CdS 细胞，让老鼠在房间的灯打开时看起来像是在奔跑。鼠窜肯定会让我们逃之夭夭！

[![pumpkin](../Images/ce6562bed8e32162aa8e42e6446a56b8.png)](http://hackaday.io/project/3242) 【魔法狼菲】被创造[南瓜链](http://hackaday.io/project/3242)来点亮整个房子的万圣节。这件作品的灵感来自 2011 年的[Jeri Ellsworth 的] [运动感应 barbot 礼服](http://hackaday.com/2011/06/01/jeris-dress-lights-up-when-someone-invades-her-personal-space-step-back-nerds/)。Pumpkin-O-Chain 使用类似的 RC 延迟线和 74HC14 逆变器来使 led 按顺序打开。虽然他希望延迟比[Jeri 的]长一点，所以他在这个版本中改用了 100K 欧姆的电阻。结果是当有人经过 PIR 运动传感器时触发一个很好的效果。

[![pumpkinlite](../Images/b5f41e6b36dedef25c253c5f4cdb93ec.png)](http://hackaday.io/project/3283) 【佩特里】厌倦了自己的鬼火蜡烛燃尽，于是自己造了[南瓜灯](http://hackaday.io/project/3283)。该灯去年首次亮相，由 Teensy 2.0++运行。今年，[Petri]决定降低功耗，在 TI 的一个 launchpad 板上改用 MSP430 处理器。Teensy 和 MSP430 上有大量输出可用，[Petri]认为他还不如使用 RGB LED。新改进的鬼火可以运行几个小时而没有火灾的危险。

我们 c [![cuth2](../Images/81a37d4dcc53db8ee3ae71a16884f24a.png)](http://hackaday.io/project/2414) 不能不提到【格里夫】更新[钩针克苏鲁面具](http://hackaday.io/project/2414)就结束这个星期。我们在上周的黑客文章中特别介绍了这个面具，并打电话给【Griff】询问最新消息。好了，期末项目完成了，看起来棒极了！我们确信[格里夫的]儿子今年会发大财！
