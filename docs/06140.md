# Plotterbot Drawing Daleks

> 原文：<https://hackaday.com/2014/05/27/plotterbot-drawing-daleks/>

[https://www.youtube.com/embed/3a59yj-HZsg?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/3a59yj-HZsg?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

两根弦，两个马达，还有一些非常有创意的软件。这就是绘图仪机器人背后的魔力，当我们在 Maker Faire 上穿过它的路径时，它正在吸引达雷克斯。这是马克 2 号，它是在拆卸了 T2 马克一号 T3 之后建造的。不幸的是，我们无法告诉你两者之间的区别。

这台机器本身是一个相当不错的小包装。有一个挂在墙上的盒子，两端各有一个电机/线轴组合。在这两个中间是一个 Arduino Mega，带有定制的驱动程序屏蔽。它需要一个存有绘图文件的 SD 卡。还有一个小触摸屏显示器，可以很容易地选择你想在贴在墙上的纸上画什么。

当我们举办饰品比赛的时候[Jay]用绘图仪机器人画了一个头骨和扳手，这些头骨和扳手是由许多较小的头骨和扳手组成的。他很友好地把那件艺术品带到集市上给我们看。谢谢[杰]！
