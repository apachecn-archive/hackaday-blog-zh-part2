# 物联网为什么会失败(以及如何挽救)

> 原文：<https://hackaday.com/2015/09/03/why-iot-will-fail-and-how-to-save-it/>

时髦的技术有两种可能的命运:失败并消失或者成功并消失。记得“多媒体”和“网络”曾经是流行词汇。他们成功了，现在他们已经消失得无处不在。当然，有很多失败的流行语(比如电信)你可能都不记得了。他们就这样默默无闻地消失了。

除非你一直生活在本地黑客空间的 CNC 工厂下，否则你可能听说过或读过“物联网”(IoT)。大大小小的公司都意识到，在下一个重大事件中提前介入有利于股价，目前，物联网是每个人都试图参与的领域。

不过，我注意到了两件事:首先，物联网并不新鲜。将嵌入式系统连接到互联网已经过时了(早在 2003 年，我甚至写过一本名为《嵌入式互联网设计》的书)。其次，物联网目前的发展方式注定要失败。

为了明白我为什么会这么说，让我们做一个思维实验。互联网早期，没有大资金介入。这只是一个古怪的政府和学术项目。但是假设互联网是在今天的商业环境下发明的。

现在想象一下你当地报纸上的广告:订阅 KoolDNS！比任何其他提供商提供的免费地址都多。调查显示，当你使用 KoolDNS 时，85%的网站都是可用的。另一个广告会让你买一个能让你阅读一半以上最流行网站的浏览器。

什么？当然，典型的黑客阅读器可以直接管理 IP 地址，并且可能为一些非 HTML 格式拼凑一个阅读器。但是更广泛的公众依赖于 DNS 服务和标准的 HTML 格式。如今，很少一部分网站依赖于特定的浏览器功能，这已经够难的了。

然而，这正是我们使用物联网的方向。每个人都有一个开放或半开放的经纪人。许多公司联合起来创建标准，与其他公司的标准竞争。在其他领域(例如，WiFi)，供应商已经提交了 IEEE 标准。IEEE 正在制定物联网标准，可能会在 2017 年发布。与此同时，现在把一些东西推向市场，希望它能够起飞并取代其他所有人，这是一种抢占土地的行为。

如果你不认为这种事情会影响你，你可能会想去看看你的 HD DVD 或 Betamax 播放器上的电影。作为一名消费者，在一场标准战争中站在错误的一方可能会非常昂贵。

怎么挽回？我在避免非开源的物联网基础设施。虽然不能保证最终胜出的标准会善待开源，但至少你(或某人)可能会让它达标。此外，如果您决定坚持使用它，至少您可以自己支持它，如果这在您的应用程序中有意义的话。

事实是，我不认为 IoT 会赢或输。我认为它已经存在了很多年，并将继续在有意义的地方蓬勃发展。对于那些没有意义的地方，这种时尚最终会消失，大公司会去榨取下一个大东西(也许是嗅觉视觉)，而我们其余的人会继续开发满足我们需求的嵌入式系统。

在下面的视频中，杰里米·里夫金表示，物联网将引发一场与工业革命不相上下的新经济革命，到 2030 年将有 100 万亿个联网传感器。然而，我不禁认为，他的零边际成本论点将更多地适用于《星际迷航》式的复制者，而不是物联网。
