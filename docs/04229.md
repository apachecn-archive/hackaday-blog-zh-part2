# (请不要)用卫生纸架造喷气发动机

> 原文:[https://hacka day . com/2013/10/05/请不要用卫生纸架造喷气发动机/](https://hackaday.com/2013/10/05/please-dont-build-a-jet-engine-from-a-toilet-paper-holder/)

涡轮增压喷气发动机一直被认为是后院工程的圣杯之一。这是有充分理由的——它们很难构建，更难运行。许多涡轮由于热启动或缺油而过早结束。[【柯林·福尔泽】然而，让它看起来很容易](https://www.youtube.com/watch?v=EorMrpM6q9Q)。[科林]是疯狂黑客的支持者——我们之前已经介绍过他的[陆地速度记录和](http://hackaday.com/2013/01/21/land-speed-baby-carriage-record-set-at-53-mpg/)[他的脉冲喷气动力茶壶](http://hackaday.com/2012/09/21/pulse-jet-tea-kettle/)。

在他最新的录像中，[科林]从废品场拿了一个马桶刷架，一个卫生纸卷架，一些管道配件，当然还有一个小型涡轮增压器。不知何故，他将所有这些转化成一个工作的喷气发动机。这里值得注意的是没有焊接。一些接头仅仅用胶带粘在一起。

称这是一个工作的喷气发动机并不夸张。正如每个后院喷气式飞机驾驶员都知道的那样，DIY 喷气式飞机的首要目标(除了不伤害自己)是自给自足。涡轮机通过空气软管、真空吸尘器或叶片鼓风机旋转起来。诀窍是打开燃料，去除空气源，让涡轮机依靠自己的动力继续旋转。一旦发生这种情况，你的引擎正在执行 F-18 或 747 使用的相同的“吸、挤、敲、吹”燃烧过程。

所有这些并不意味着发动机除了保持自身运转之外还能做任何有用的事情。这就是后院喷气发动机的第二个目标——推力。你需要一个可用的推力，然后才能把引擎绑在推车上，在附近兜风。

[科林的]喷气式飞机是在两台录像机上制造和测试的。第一张概述了如何建造它，[第二张展示了喷气式飞机的运行](http://www.youtube.com/watch?v=EorMrpM6q9Q)。为了安全起见，[科林]在 YouTube 评论中暗示，该系列将有第三个视频显示他如何发生煤气泄漏事故，并导致他“一周不能行动”。不用说，不要在家里尝试这种特殊的引擎构造。
