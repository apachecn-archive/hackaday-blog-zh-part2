# 一个简单的软盘音乐控制器

> 原文:[https://hack aday . com/2014/10/12/a-simple-floppy-music-controller/](https://hackaday.com/2014/10/12/a-simple-floppy-music-controller/)

虽然用软驱播放音乐已经做了很多次，但让任何带有步进电机的设备播放音乐仍然对我们所有人的黑客有吸引力。[Tyler] [设计了一个 Arduino 盾](http://discspace.org/floppy-music-thing/)和一个库，它可以让你立即启动并运行。[Tyler]的保护罩包括连接 4 个软盘驱动器的插头，这些插头直接插入保护罩。驱动器在使用前不需要任何修改。

虽然您可以简单地用一些跳线将一些软盘驱动器连接到 Arduino，但这种突破性的保护使连接驱动器变得很简单。除了设计盾牌，[泰勒]还发布了一个 Arduino 库，让事情变得更加简单。这个库可以让你简单地设置你想要每个驱动器播放的频率，这样可以节省一些跑腿的工作。

控制软盘的 Arduino 库[在 GitHub](https://github.com/tkbletsc/FloppyMusic/tree/master/FloppyMusic) 上可用，休息后还包括控制器的视频。
