# 阿蒂尼控制的电眼管

> 原文:[https://hack aday . com/2012/09/12/at tiny-controlled-magic-eye-tube/](https://hackaday.com/2012/09/12/attiny-controlled-magic-eye-tube/)

![](../Images/554721d50e46b49ba92190e1731f191d.png "eye")

在早期的广播电台，最昂贵的收音机是非常令人印象深刻的家具。这些无线接收器配有制作精美、抛光的木箱，是任何家庭房间的焦点。一些最昂贵的收音机甚至包括一个显示接收强度的可视指示器，这是马库斯决定用一台 at tiny 85 T1 重新设计的东西。

有问题的显示管是 EM800 [电眼管](http://www.magiceyetubes.com/)，用于收音机、立体声音响和电子测试设备中作为基本的显示指示器。通过施加控制电压(从 0 到-10V)，可以像条形图显示一样控制发光显示。

[Marcus]“显像管显示器是围绕 ATtiny85 微控制器构建的，使用自制的 PCB。一旦向 EM800 的阳极提供 250 伏电压的问题得到解决，这是一个相当简单的构建。

在休息后的视频中，你可以看到[Marcus]的魔眼管的条形图显示慢慢增长和后退，非常适合显示计算机上当前的 CPU 负载或动态条形图显示会使用的任何其他内容。
