# 飞行步兵来找你了(六十年前)

> 原文:[https://hackaday . com/2013/03/20/retro tech tacular-飞行步兵-六十年前来找你/](https://hackaday.com/2013/03/20/retrotechtacular-flying-foot-soldiers-are-coming-for-you-sixty-years-ago/)

![Hiller_VZ-1_Pawnee_(2)](../Images/4a471c2047a198d8d4d27e9051cd61c2.png)

上图是 1950 年代的一项了不起的实验技术，但最终没有任何进展。希勒·VZ-1 波尼是一种单人驾驶的交通工具，被认为可以为美军提供战术优势。海军研究办公室花了几年时间开发这种飞机，它使用安装在平台底部的两个旋翼。它们彼此相反地旋转——这样就不需要像直升机上那样的尾桨——来提升平台离开地面一小段距离。虽然他们中的六个人幸存了下来。但好消息是你可以去美国东海岸或西海岸的博物馆看它们。

既然严肃的事情已经过去了，我们就来谈谈休息后的视频剪辑吧。叙事风格是新闻片时代的瑰宝。我们不知道这种口音是怎么回事，但我们完全相信，在你当地的 hackerspace 上，每年至少有一次大会应该要求所有的演讲者使用他们对这位才华横溢的绅士的声音的最佳印象。
