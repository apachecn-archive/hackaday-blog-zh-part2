# 使用 Kinect 的增强现实沙盒

> 原文:[https://hack aday . com/2015/08/22/augmented-reality-sandbox-using-a-Kinect/](https://hackaday.com/2015/08/22/augmented-reality-sandbox-using-a-kinect/)

想让你 5 岁儿子的朋友都嫉妒吗？如果他告诉他们他可以在沙箱里造出真正的火山呢？这将会是沙盒的未来吗？

这实际上并不难设置！该系统由一台运行 Linux 的好电脑、一台 Kinect、一台投影仪、一个沙箱和沙子组成。就是这样！加州大学(加州大学戴维斯分校)现在已经安装了一些这样的系统来教孩子们地理知识，这是 3D 扫描和投影制图的一个非常酷的演示。正如你在上面的动画 gif 中看到的，Kinect 可以跟踪沙子的地形，然后将其“现实”投影到上面。在这种情况下，一个迷你火山。
