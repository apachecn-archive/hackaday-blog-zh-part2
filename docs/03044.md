# 流氓 Pi:RPi Pentesting Dropbox

> 原文:[https://hack aday . com/2013/03/24/rogue-pi-a-rpi-pentesting-Dropbox/](https://hackaday.com/2013/03/24/rogue-pi-a-rpi-pentesting-dropbox/)

[![Rogue Pi](../Images/d14146c670d0e061d9d5aa74724719a3.png)T2】](http://hackaday.com/?attachment_id=96673)

pentesting dropbox 用于允许 pentester 远程访问和审计网络。设备被放到网络上，然后建立允许远程访问的连接。作为最后一个项目，[Kalen]构建了 [Rogue Pi](http://crushbeercrushcode.org/2013/03/developing-the-rogue-pi/ "Rogue Pi") ，这是一个基于 Raspberry Pi 的 pentesting dropbox。

Rogue Pi 具有一些有助于测试的特性。首先，它有一个通电测试，验证在目标网络上的安装是否成功。由于 dropbox 的安装需要不显眼，这有助于在不被检测到的情况下设置设备。液晶屏允许用户查看安装是否成功，而无需额外的计算机或外部显示器。

开机后，设备会创建一个反向 SSH 隧道，提供对设备的远程访问。使用反向隧道允许设备绕过网络的防火墙。Aircrack-ng 已被包括在设备中，以允许无线攻击，并且如果有线网络出现问题，隐藏的 SSID 允许无线访问。为在 Pi 上运行而构建的测试工具有一长串。
