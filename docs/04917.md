# 果冻铅球打印机

> 原文：<https://hackaday.com/2013/12/26/jello-shot-printer/>

在为朋友即将到来的 25 岁生日检查一些果冻时，[Sprite_tm]有了一个顿悟。如果你能在果冻的里面[打印出设计**会怎么样？**](http://spritesmods.com/?f=ri&art=jello3dprinter)他迅速抓起一个注射器，开始将食用色素注入其中一杯果冻中——成功了。不幸的是，他的朋友指出手工制作每一个果冻要花太多时间，对此[雪碧]回应道:

> 没关系，我会很快做出一台 3D 打印机，可以为你在果冻里做出漂亮的图形。

经典。黑客思维的伟大之处在于，面对问题时，你永远不会说不！

为了实现这种打印机，[Sprite_tm]采用了一些旧的 CD-ROM 驱动器来创建一个三轴移动平台。他正在使用第四驱动器的喷射器组件来按压注射器，该注射器将香蕉酒、绿色食品色素和玉米淀粉的混合物通过医疗管道推至墨头。为了控制它，他只使用了一个只有 2K 内存的 ATTiny2313。找到正确的流程需要一点时间，但是效果出奇的好。休息后留下来看看它的打印能力。
