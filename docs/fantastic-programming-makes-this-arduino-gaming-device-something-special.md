# 奇妙的编程让这款 Arduino 游戏设备与众不同

> 原文：<https://hackaday.com/2012/10/01/fantastic-programming-makes-this-arduino-gaming-device-something-special/>

这个 Arduino 游戏机的硬件很好。但是[制作这款名为《扭曲的蛇》游戏的代码是无与伦比的](http://r0d0t.tumblr.com/post/32510849587)。[Rodot]已经为硬件编写了几个游戏，它使用了一个 Arduino，160×168 TFT 屏幕，一个 3 轴加速度计和两个输入按钮。如果你感兴趣的话，论坛上有一个帖子，他在里面跟[谈了更多关于硬件设计的事情](http://arduino.cc/forum/index.php/topic,119099.0.html)。但是休息之后，你不会想错过两个视频中的任何一个。

第一个片段展示了一个弹跳球平台游戏。加速度计来回移动球，顶部的滚动级别将更多的壁架带入游戏中。这本身就是一个很棒的游戏。但是第二个视频中展示的扭曲的贪吃蛇游戏让我们自己的基于 ARM 的贪吃蛇游戏看起来像是一个 3 岁的孩子编写的。[Rodot]填满了 ATmega328 芯片的所有程序存储器来实现这一点。有一个菜单系统，允许颜色主题和难度选择。游戏本身让蛇随着尾巴以优美的曲线跟在后面去它想去的任何地方。哇！
