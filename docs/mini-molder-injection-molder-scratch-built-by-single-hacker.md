# 迷你成型机:吹塑机刮-由单个黑客建立

> 原文：<https://hackaday.com/2014/05/22/mini-molder-injection-molder-scratch-built-by-single-hacker/>

[https://www.youtube.com/embed/2P7MaZUrSQQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/2P7MaZUrSQQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

我们在创造者大会上找到了[詹姆斯·杜兰德]。他是为数不多的在 Maker Faire 上有展位的制造商之一(没有提到销售或未来的众筹)——他正在展出一台他从零开始建造的吹塑机。

制作过程是 100%[詹姆斯]。每一个定制零件都是他设计和打磨的。所有的组装技术都是他一路学来的。我们没有看到任何不适合生产的东西。我们既感动又羡慕。

大约三年前，当他了解到半个世纪前流行的一种吹模自动售货机——Mold-A-Rama machine([)后，他萌生了制造迷你成型机的想法。在他的博客](http://en.wikipedia.org/wiki/Mold-A-Rama)上，他的旅程被记录为[一个成型类别。在很大程度上，花在 CAD 设计上的 1.5 年似乎真的有了回报。他确实分享了一个需要重新设计的元素。最初的原型有一个问题，模具在合在一起时会被推高。他通过翻转铰链设计来调整机械装置，使其向下关闭。当饮水机和水泵冷却模具和里面的塑料模型时，这似乎可以保持一切正常。](http://ashermade.com/category/mini-molder/)
