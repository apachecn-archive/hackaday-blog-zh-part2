# 禅石花园表使用磁铁和沙子

> 原文：<https://hackaday.com/2012/04/26/zen-rock-garden-table-uses-magnets-and-sand/>

![](img/7cd056f2cb1f24fec0030fe40ec5580a.png "zen")

[Nick]正在制作一个[咖啡桌沙盘绘图仪](http://www.youtube.com/watch?v=R1fC33mAp_E)的原型，它可以在沙盘上画出很像禅宗岩石花园的图案。

[Nick]的 zen rock garden 使用磁铁在沙滩上画出有趣的图案。该建筑使用 3D 打印齿轮和激光切割零件来旋转桌子，并沿着圆的半径移动磁铁。在原型的第一次测试中，滚珠轴承晃动了一下，但是通过在沙子下面加一块泡沫解决了这个问题。电源通过底座中的滑环提供，桌子通过蓝牙控制。

说到带磁铁和滚珠轴承的禅宗咖啡桌，我们偶然看到了一个看起来更专业的原型的视频，这是 T2 成功的 Kickstarter 活动的基础。像[尼克]的原型一样，整个建筑依靠磁铁和滚珠轴承来移动沙子。因为这张禅桌使用 XY 轴而不是[尼克]的极坐标设置，[绘制标识](http://vimeo.com/33818346)在数学上要容易得多，尽管它看起来不像圆形岩石花园那么酷。

休息过后，你可以看到这些禅石花园咖啡桌在工作。
