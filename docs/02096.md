# 从头开始构建热成像传感器

> 原文：<https://hackaday.com/2012/11/12/building-a-thermal-imaging-sensor-from-scratch/>

[https://www.youtube.com/embed/pIb1scnD67o?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/pIb1scnD67o?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[Rob]住在一栋有 100 年历史的房子里，这些古老的板条、灰泥墙和旧窗框带来了大量的通风。解决这种能源效率低下的通常方法是使用热成像相机，这是一种用视频图像覆盖物体温度的设备。这些相机贵得吓人，所以[Rob]做了我们任何人都会做的事情[，自己造了一台](http://www.instructables.com/id/Thermal-Imaging-Phone-Camera/)。

该建筑以一个售价约 80 美元的 Melexis MLX90620 远红外热电堆为中心。基本上，这个传感器是一个非常非常低分辨率的相机(16×4 像素)，它感应的是热量而不是光线。通过将这种传感器粘贴在带有 Arduino Mini 和 wifi 网络适配器的试验板上，[Rob]能够将数据从红外传感器下载到他的 iPhone 上，并覆盖在摄像头的馈送上。

结果，正如在上面的视频中看到的，是一个低分辨率但仍然非常有用的热成像相机，非常适合在老房子中寻找冷气流或像捕食者一样追踪[Arnie]。
