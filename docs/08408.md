# LED 声卡不是你爸爸

> 原文：<https://hackaday.com/2015/02/26/led-sound-board-is-not-your-father/>

谁不喜欢星球大战，发光二极管，和音乐？[Stathack]正在寻找一件独特的艺术品放在他的客厅里……所以他决定制作自己的[维德情商。](http://nerdsrule.co/2015/02/25/art-arduino-a-sound-sensor-leds-and-star-wars/)

EQ 是一个由胶合板和中密度纤维板制成的巨大的 4x 5 的部件。[Stathack]通过使用投影仪将轮廓投影到表面上，将熟悉的头盔描绘到它上面。由于没有超大的计算机数控系统或激光，他于是费力地用一把竖锯将设计中所有的白色部分切割出来——天哪。

这一过程只需要数周的打磨、填充和再次打磨，因为线锯非常精确。

完成这些工作后，他用中密度纤维板制作了背板，为 led 提供结构支撑和安装位置。稍后再喷一点油漆，用 Arduino 做一个简单的电路，就完成了，棒极了。
