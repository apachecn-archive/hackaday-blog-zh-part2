# [本·克拉斯诺]解释千瓦时表

![benkrazpowermeter](../Images/bf10a7c78cb755e6f5051c9fce42342e.png)

[本·克拉斯诺]回来了，这次他是在[拆电度表](http://www.youtube.com/watch?v=xtModjpxfxM)。虽然不像[在家制作气凝胶](http://hackaday.com/2011/11/16/making-aerogel-at-home/)，或者 [DIY 扫描电子显微镜](http://hackaday.com/2011/03/23/diy-scanning-electron-microscope/)那么令人兴奋，但【本】一贯低调的解释方式让一个复杂的话题变得简单易懂。

自从商业力量出现以来，这些老式的机械仪表就一直是家庭和企业的主要设备。我们一直认为电表是一个基本的电动机。根据本的解释，这些仪表是电磁场的复杂舞蹈。三个线圈在铝盘附近产生磁场。这在磁盘中产生涡流，从而产生净扭矩。圆盘旋转，转动发条装置，推动表盘。

为什么是三圈？一个是高匝高规格电压线圈，另外两个是低匝低规格电流线圈。电压线圈必须相移 90 度，以在磁盘上产生适当的扭矩。困惑了吗？看视频！[Ben]在解释场的相互作用方面做得比我们在文本中做得更好。

有趣的一点是，圆盘非常紧密地保持平衡，以至于一个小小的“防蠕动孔”被用来稍微扰乱涡流。当仪表上没有负载时，这个孔防止圆盘缓慢转动。

[本]没有提到停止电表的老故事。我们都记得 PopSci 和其他杂志封底的那些广告。虽然不切实际，但显然一些计价器停止技术确实有效。不过应该注意的是，电力公司会注意到给定客户电力消耗的变化，所以我们不建议在家里尝试这种方法。
