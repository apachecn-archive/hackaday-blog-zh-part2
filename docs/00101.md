# 用戴森的备用零件比赛

> 原文:[https://hack aday . com/2012/04/14/与戴森赛车-备件/](https://hackaday.com/2012/04/14/racing-with-dysons-spare-parts/)

![](../Images/b1bd069f0a6a9bbd1fca81347c3b4312.png "dyson-racer")

当需要在戴森设计工厂放松时，这些工程师知道如何正确地做这件事。最近，该公司向他们的工程师发出挑战，要求他们参加成人版的松林德比，在这场比赛中，他们驾驶由戴森发动机驱动的汽车进行比赛。

休息后的视频展示了一个由翻转的木托盘制成的赛道上的大量计时赛。大多数车辆是由我们不认识的零件制成的。但其中一些非常熟悉，比如我们最喜欢的干手器(见上图)和戴森球真空吸尘器标志性的一枝黄花歧管。

正如你在最后一段视频中看到的，课程突然结束了。有一个条目包括一个人类骑手，他似乎和无人驾驶的马车走得一样快。视频在他撞墙之前被切掉了，但我们无法想象他有时间在设计中加入刹车。
