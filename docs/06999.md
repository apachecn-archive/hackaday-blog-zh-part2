# 两个戒指盒的故事

> 原文:[https://hackaday.com/2014/09/08/a-tale-of-two-ring-boxes/](https://hackaday.com/2014/09/08/a-tale-of-two-ring-boxes/)

这是一个关于两颗心、两次订婚和两个戒指盒的故事。大约几年前，[curtisabrina]向他的女朋友求婚。他没有直接把戒指交给她，而是把它放在一个他定制的[锁紧戒指盒](http://imgur.com/gallery/mskl7)【imgur link】里。乍看之下，这个盒子看起来很正常，但掀开第一个抽屉，就会发现一个复杂而华丽的齿轮系统。齿轮只能由一对互锁的心形钥匙转动——这是几个月前[curtisabrina]送给她的礼物。这种机制简直令人惊叹——行星减速驱动弹簧加载的光圈打开，露出订婚戒指。

[curtisabrina]在他的招牌制作工作之余做了他的戒指盒。这份工作让他接触到了一些难以置信的工具，比如 MultiCam 3000 系列 CNC。这个盒子非常棒，他在 Reddit 的一个帖子中展示了他的作品。

快进两年。[joetemus]正准备向他的女朋友求婚，也想做类似的事情。他没有高级商店工具，但他有一把。以最初的盒子为灵感，[joetemus]开始设计。随着时间的推移，试验和错误，第二个[戒指盒出现了](http://imgur.com/a/3Mu8J)【imgur 链接】。像最初的盒子一样，[joetemus]是从一块粗切的木板开始的。几乎每个部分，包括铝齿轮，都是在 Shapeoko 2 上切割的。[joetemus]也用一个 [Reddit 帖子](http://www.reddit.com/r/DIY/comments/2f9fnu/i_made_one_of_the_most_engineered_ring_boxes_for/)庆祝他的成就。

[joetemus]的戒指盒不像[curtisabrina]的那么复杂或精致，但他使用的机器比[Curtis abrina]使用的设备便宜得多。我们觉得她们两个都很棒，很高兴地报告她们的两个女朋友都说:“是的！”
