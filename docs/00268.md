# 电子桌面弹射器

> 原文:[https://hack aday . com/2012/04/30/electronic-table-top-catapult/](https://hackaday.com/2012/04/30/electronic-table-top-catapult/)

![](../Images/d9571409d71696ceac443d9db5394387.png "catapult")

[不寻常的特拉维斯]想出了这个[相当光滑的电子弹射器](http://www.instructables.com/id/Arduino-Controlled-Catapult/)。这种易于建造且价格适中的钻机有一个 arduino 作为 3 个伺服系统的大脑和控制器。一个是释放，另一个控制张力，第三个控制角度。电路和建设都非常简单，使这成为一个完美的周末项目。他更喜欢小一点的，所以如果你能缩小它，就把你的设计告诉他。

我们一直很喜欢各种复杂程度的弹射器，所以请随意提交你所做的任何变化。
