# Arduino 饮料机将硬币变成液体勇气

> 原文:[https://hack aday . com/2014/10/05/arduino-drink-dispenser-turns-quarters-into-liquid-courage/](https://hackaday.com/2014/10/05/arduino-drink-dispenser-turns-quarters-into-liquid-courage/)

有没有觉得家里需要一台自己的自动售酒机？如果你想的话，[Ben]和[Dan]正好有适合你的基于 Arduino 的机器！

这实际上是布鲁内尔大学产品设计学校作业的一部分——整个设计和制造只用了一个多星期。这台机器接受并计算硬币，在液晶显示屏上显示出总数。当正确的数量被插入，你可以选择你的镜头，机器会给你倒一个硬的。

我们喜欢这台自动售货机的一点是——我们不确定它是否真的有资格成为一台**barbot**——它没有任何花哨的泵。事实上，它只是使用两个便宜的电磁阀和重力来分配饮料，就像一个典型的酒吧瓶分配器。
