# 音速起子控制你的电视，对木头不起作用

> 原文：<https://hackaday.com/2012/07/12/sonic-screwdriver-controls-your-tv-doesnt-work-on-wood/>

![](img/d2cc0cbcacc7e089d85bce00237ea33b.png "sonic")

当然，你可以用一把简单的塑料声波螺丝刀来冒充医生。它甚至可能在顶端有一个 LED，如果你真的为了音效而挥霍一个小扬声器。是的，你可以用医生的加利弗里亚军刀的一个不雅版本来制作，或者[你可以得到这个](http://www.thewandcompany.com/sonic/)非常详细的声波螺丝刀复制品。

这款第 11 代医生声波螺丝刀不是塑料或树脂复制品，而是由压铸、镀铜金属制成，带有翡翠绿色聚碳酸酯尖端。这个复制品有几个锦囊妙计:除了在适当的时候点亮并提供一些声音效果，它还可以作为一个可编程的红外遥控器，借助三轴加速度计提供手势界面。如果这还不够，尖端下方的铜“棒”还可以作为板上微控制器的触摸接口。

对于我们见过的那么多[令人瞠目结舌的光剑版本](http://hackaday.com/2010/08/27/devote-your-life-to-replicating-a-lightsaber/)，我们真的很惊讶没有看到更多的音速起子版本。这真的很酷，这个玩具可以作为一个遥控器，但我们打赌一些黑客读者可以用一个可扩展的珠宝盒复制这个复制品，甚至可能是一个蓝牙手势控制器。

如果你想要自己的声波螺丝刀，你可以在 ThinkGeek(美国)或 Forbidden Planet(英国)买到。休息之后是 ThinkGeek 发布的一段视频，展示了这款真正令人敬畏的遥控器。

向[Zerocool]致敬，感谢他送来了这个。
