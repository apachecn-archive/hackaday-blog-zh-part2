# LED 音乐会礼服

> 原文：<https://hackaday.com/2013/12/07/led-concert-dress/>

[Geri]有一个很棒的哥哥。上周末，他为她做了这件[令人惊艳的 LED 礼服](http://www.pnuke.co.nz/2013/11/led-dress.html)在[泰勒·斯威夫特]演唱会上穿！

你可能知道也可能不知道，她鼓励她的粉丝带着“发光的”东西在她的演唱会上挥舞。快速检查了一下展览场地的进入条件，看起来 led 是允许的，所以帕特里克开始工作。

他用的是一套防水的红色 LED 灯条和一个从中国订购的廉价控制器。他们需要一个相当强大的电池组，所以[Patrick]组装了一个开关模式降压转换器，将 LED 控制器的 19.8V 4.5A/h 电池组降至恒定的 12V。他们的妈妈不想弄乱红色的礼服，就把布条缝好。这件衣服超级亮，看起来很棒——它消耗大约 25 瓦，所以电池组应该可以持续整个音乐会。

不幸的是，在音乐会前一周，他们发现 Vector Arena 不允许 LED 灯进入音乐会，你可以想象，这是非常令人心碎的。

谢天谢地，有人联系了组织者，他们为他们破例了！[Geri]甚至登上了当地报纸的[头版](https://twitter.com/PatrickHerd/status/407076867686952960/photo/1/large)！休息后留下来看看裙子的视频吧！
