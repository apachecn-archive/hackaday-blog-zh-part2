# 使用 OpenSLS 将激光切割机变成 3D 打印机

> 原文:[https://hack aday . com/2014/06/02/将激光切割机变成带 opensls 的 3d 打印机/](https://hackaday.com/2014/06/02/turning-a-laser-cutter-into-a-3d-printer-with-opensls/)

![STL](../Images/da223adfe0f3c6f1c6b16f64c0c29ab5.png)

[Andreas Bastian]一直在研究一种设备，它可以将现成的激光切割机变成能够选择性激光烧结粉末塑料成 3D 物体的东西。他做了很多工作，但是现在他可以看到他的劳动成果了:他已经成功地用蜡和尼龙粉末打印出了一些东西。

与几乎所有其他廉价 3D 打印机不同的是，[Andreas]的设计既不依赖于将塑料喷到床上，也不依赖于用紫外光固化液体树脂。取而代之的是，将一层精细的粉末铺在构建平台上，用激光熔化。熔化的层落下，另一层粉末被施加，并且循环重复直到零件被完成。制造一台这样的机器是一个挑战，但是[Andreas]有一个很好的想法，改装一台现成的激光切割机，使他能够专注于设计火药和活塞系统的困难任务。

这是一个非常有趣的项目，大多数定制零件都是由激光切割丙烯酸制成的:无论你正在改造什么激光切割机，都可以轻松切割成合适的尺寸。Wiki 上有很多信息[，下面还有一些展示烧结过程和粉末分布的视频。](http://reprap.org/wiki/OpenSLS)

哦。最后一点。安德烈亚斯是在莱斯大学(Rice University)乔丹·米勒(Jordan Miller)令人惊叹的实验室研发的。在这个先进制造研究所发生了许多有趣的事情，包括生物打印、DLP 树脂打印机和使用喷墨打印机进行细胞培养。[看看这篇文章](http://hackaday.com/2014/03/18/mrff-3d-bioprinting/)在中西部说唱节上的精彩演讲。
