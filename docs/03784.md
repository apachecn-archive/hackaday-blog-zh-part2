# 1200 小时的工作造就了最小的 V12 发动机

> 原文：<https://hackaday.com/2013/06/09/1200-hours-of-work-results-in-the-smallest-v12-engine/>

[https://www.youtube.com/embed/c1pJIVqCC1E?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/c1pJIVqCC1E?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[José Manuel Hermo Barreiro]花了很多时间手工制作这些微型发动机。每一个零件都是专门为它所搭载的发动机定制的。他发明了飞机引擎、汽车引擎和船用引擎，这些引擎实际上都在运转，而且是世界上同类引擎中最小的。

在这个视频中，他站在一个房间里，几个引擎排成一行，都在平稳运行，他认为在他面前可能有超过 15，000 个小时的工作。
