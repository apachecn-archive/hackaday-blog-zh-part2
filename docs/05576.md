# MRRF: 3D 生物打印

> 原文：<https://hackaday.com/2014/03/18/mrff-3d-bioprinting/>

在今年的中西部 RepRap 节上有一些主题演讲，有点令人惊讶的是，大多数演讲都不是由负责设计你最喜欢的打印机的人做的。[Jordan Miller]、[Andy Ta]和[Steve Kelly]做了一个最有趣的演讲，内容是关于 RepRap 和其他 3D 打印技术在生物技术和组织工程中的应用。是的，50 年后当你需要打印重要器官时，这就是它的来源。

【乔丹】从组织工程和 3D 打印开始，他的工作是[打印三维糖晶格](http://hackaday.com/2012/07/02/printing-organs-with-a-3d-printer/)，这种糖晶格可以嵌入培养基中，然后溶解。糖留下的孔变成了滋养细胞培养物的脉管系统和毛细血管。他的项目和创客文化的惊人成功促使他和其他人创办了[先进制造研究院](http://amrinstitute.org/build_the_future..html)，将年轻创客带入科学界。这是莱斯大学主办的[项目](http://news.rice.edu/2013/09/03/rice-brc-hosts-advanced-manufacturing-research-institute-2/)，在研究和让制造商从事科学研究方面取得了惊人的成功。

这些年轻的创客之一就是【安迪 Ta】。作为一名经济学专业的学生，[Andy]几年前第一次听说了 maker 和 RepRap 社区，并买了一个 MakerBot 纸杯蛋糕。这是一台糟糕的打印机，但它确实让他参与到社区中，主持构建研讨会，并研究围绕 DLP 固化的 UV 树脂的 3D 打印构建。在 AMRI，[安迪]开始研究紫外线固化树脂的特性，找出正确的光、树脂和曝光类型，以创造出具有打印细胞集落所需特性的固化树脂。你可以在[安迪]的网站上查看他的最新作品。

[Steve Kelly]也在 AMRI 做了一些工作，但他没有使用通常的 RepRap 或 DLP 投影仪打印机，而是使用喷墨打印头拍摄细胞培养物。他最初的实验包括简单地用细菌菌落重新填充一个喷墨墨盒，并发现这些细胞实际上在被加热和高速射出喷嘴的过程中存活了下来。大多数喷墨打印机实际上不会在精确的网格上显示不同的颜色，这使得它不能用于培养细胞。[Steve]用一个附着在 RepRap 上的喷墨控制器护罩解决了这个问题。[史蒂夫]的所有工作都记录在他的 Github 上。

这都是令人敬畏的工作，是基于 3D 打印机的生物工程的开端，也是业余科学家和专业制造商集思广益所能做的惊人例子。下面的视频链接。
