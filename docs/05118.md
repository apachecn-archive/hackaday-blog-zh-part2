# 自行车是如何制造的

> 原文：<https://hackaday.com/2014/01/21/retrotechtacular-how-a-bicycle-is-made/>

你的自行车能泰然自若地驾驭木板路和泥潭吗？如果它是由罗利自行车公司制造的，它应该是。本周的回顾展是对 1945 年标准自行车从头到尾的生产过程的回顾。在拍摄的时候，罗利已经生产自行车将近 60 年了。

这部电影的中心是一个男孩和他的父亲在自行车诞生的工厂的制图室讨论购买自行车。为了能够与现代的“安全模型”相比较，我们简单地提到了“小东西”,在现代的“安全模型”中，骑手平衡地坐在两个大小相同的轮子之间。两人正在与首席设计师谈论模型，父亲询问他们的制造过程。

我们得到了从车架到前叉，从轮毂到车把的完整故事。框架由高质量的钢锻造而成，其强度经过了高温和应变的考验，其强度远远大于其在制造或使用过程中所承受的强度。它是由卷成管的长片形成的，在连接处用火焰密封，并切割成一定长度。框架件与由单片钢形成的支架连接。这个过程特别有意思。

支架最初是一片扁平的金属。这件作品被塑造成一个碗，然后这个碗被绘制成一个[柯林斯玻璃](http://en.wikipedia.org/wiki/Collins_glass)的形状。在下一个过程中凸起，最终被刺穿，然后形成一个开放的圆柱体，就像一块陶器在轮子上挖空一样。这样就没有接头，只有一块坚固、轻便、可靠的钢。

为了使它们更坚固，它们被压在夹具中。然后，添加构成框架的管子，并通过支架固定。整个框架进入熔炉进行最终回火，然后冷却和清洗。去除钎焊过程中多余的黄铜，并对框架进行电镀。

自行车的前叉也是类似的，但因为它必须吸收更多来自泥潭的冲击，所以它必须更坚固，并且形成为一个单件。经过炉内强化后，用砂布和砂轮对其和框架进行打磨，使搪瓷能平滑地粘附在和 上。车架和前叉经过防锈处理，并浸漆以增加光泽。 

这孩子对车把结构特别感兴趣。这些是在机器上按照设定的设计进行成型和弯曲，在电镀槽中进行电镀，并抛光至防风雨的光泽。挡泥板——在这里被称为挡泥板——是由钢带一次成型的。接下来是踏板曲柄、链轮和轮毂。踏板曲柄是锻造和切割，各种孔是钻孔和螺纹。之后，一种特殊的乳状液体冷却并润滑它们。链轮是由一块扁钢制成的，经过几次压制，它就变成了精美的细丝。踏板曲柄安装在齿轮上，轮毂上装有润滑脂轴承。

辐条是在一台切割、钩住和穿线的机器上制成的。它们首先安装在轮毂上，然后安装在轮辋上。辐条车轮的真实性进行测试，任何松散的辐条被收紧。之后，轮胎和内胎被安装到轮辋上。

自行车零件终于为装配线、质量检查和最终调整做好了准备。我们再次回到首席设计师的办公室，在那里，他得出结论，优质的材料和整个过程中不断的检查造就了一辆首屈一指的自行车。
