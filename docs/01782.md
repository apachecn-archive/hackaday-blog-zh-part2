# 200 磅重的无线网络部署机器人压到了我的脚

> 原文：<https://hackaday.com/2012/10/03/200-pound-wifi-deploying-robot-ran-over-my-foot/>

![](img/8a79f85b6ed843748cdbec12143eeb9b.png "wibot")

来自马萨诸塞州萨默维尔 Artisan's 疯人院的[Adam Bercu]和[Dan Landers]在今年的 Maker Faire 上带来了一个非常非常酷的玩具。这是一个 200 磅重的无线中继器部署机器人能够漫步在无情的地形和我的脚。

该机器人通过一个网络界面进行控制，并借助一个前置网络摄像头进行平移和倾斜控制。所有信号都通过 WiFi 连接发送到 node.js 网络服务器；这不是与机器人远距离交流的最佳方式，但[亚当]和[丹]有一些锦囊妙计。

在机器人的背面是两个鹈鹕箱，上面装有电池和 Linksys WRT54G 无线路由器。当机器人达到其范围的极限时，它会激活一个螺线管，放下一个 WiFi 中继器。这个中继器有足够的电池电量保持供电大约一天半，这意味着机器人可以多次旅行，通过一些非常不利的地形部署无线网络。非常适合灾难和搜救行动。
