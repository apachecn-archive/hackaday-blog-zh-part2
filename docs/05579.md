# 3D 打印机只能制造小饰品——皮划艇呢？

> 原文:[https://hackaday . com/2014/03/19/3d-printers-only-can-make-trinket-what-about-kayaks/](https://hackaday.com/2014/03/19/3d-printers-can-only-make-trinkets-what-about-kayaks/)

哇哦。草根工程公司的吉姆·史密斯刚刚完成了他的 3D 打印皮划艇的最后加工。它会浮起来。

单个零件是在[Jim 的][大型自制 3D 打印机](http://hackaday.com/2011/09/08/huge-diy-3d-printer-predictably-prints-huge-3d-items/)上打印的，该打印机大致基于 RepRap——除了它的最大构建体积是 403 x 403 x 322.7mm 毫米之外。

皮划艇本身是由 28 个印刷部分组成的，为了将它们固定在一起，他安装了黄铜螺纹热塑插件，这样就可以将各个部分用螺栓固定在一起。组装前使用硅树脂填缝，以确保防水密封。

它最初是基于布莱恩·汉瑟尔(Bryan Hansel)的西斯克维特湾皮划艇，但(Jim)对它进行了大量修改，以适应 3D 打印。它以 0.65 毫米的层高度打印以减少打印时间，最终仍然超过 1000 小时！他甚至根据自己的身高和体重对设计进行了优化，以提高性能。

船体厚 6 毫米，采用定制的肋骨结构来增加强度，您也可以在下图中看到将各部分紧固在一起的方法:

![greng_2031](../Images/ddde0c9a9e8ca846213d7f3a2dc80fd2.png)

它总共重约 65 磅，其中 58 磅是 ABS 塑料——它使用了 7 磅的螺钉和黄铜插件——哇！哦，因为整个东西是 3D 打印的，[吉姆]还增加了一些方便的功能，如船头和船尾的摄像头支架。谈一个大项目！
