# 当你的乐队成员很忙时，鼓手机器人会加入即兴演奏

> 原文:[https://hack aday . com/2012/04/03/drummer bot-joins-the-jam-session-when-your-band mates-busy/](https://hackaday.com/2012/04/03/drummerbot-joins-the-jam-session-when-your-bandmates-are-busy/)

![drummerbot](../Images/6f336cbb8a9b57711591d2e1d8194230.png "drummerbot")

似乎大多数情况下，[Steffest]发现自己在没有打击乐器伴奏的情况下，也能在吉他上尽情摇滚。像任何有事业心的黑客/音乐家通常会做的那样，[他制造了一个机器人鼓手](http://blog.stef.be/drummerbot)当他想玩的时候加入进来。

他的鼓手机器人由 Arduino 驱动，Arduino 负责控制机器人拥有的 8 个伺服电机。机器人的架子鼓由各种物品组成，从风扇马达到锅盖等等。[Steffest]希望能够产生尽可能多的声音，所以大多数伺服电机都是双向驱动的，这样机器人就可以用每个“手臂”击打多个物体。

[Steffest]是一个用网络界面连接物理对象的狂热爱好者，所以他建立了一个简单的基于 HTML 的序列器，允许他从他的手机给机器人编程。一旦音序器被编程，只需简单地按一下按钮，鼓手机器人就可以启动了。

[Steffest]说这个机器人工作得很好，但是如果你现场听的话，声音有点原始。然而，正如你在下面的视频中看到的，一点点 Ableton Live 后期处理对平滑事情大有帮助。
