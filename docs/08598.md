# 救命啊！我的咖啡壶里有个小恶魔！

> 原文：<https://hackaday.com/2015/03/22/help-theres-an-imp-in-my-coffee-pot/>

咖啡。我们社会的命脉。豆子的甜蜜，它的化学成分让我们感觉非常非常好。难道它就不配在物联网中占有一席之地吗？[Matt]和[Don]也这么认为——所以他们把他们的旧咖啡壶和他们的手机连接起来。

在收到他们的电动 Imp 板的开发者版本后，两人开始考虑测试它的小项目；甚至可能有现实世界的应用。因为 Imp 能够通过网络接收输入，所以编写一个应用程序来控制东西是非常容易的——在这个例子中，是一个咖啡壶。

就硬件而言，这其实很简单。咖啡壶控制板为 Imp 提供电源，咖啡机的开/关开关连接到 Imp 的一个输出端。一个简单的应用程序之后，我们就拥有了无线 java 功能。呵。Java。
