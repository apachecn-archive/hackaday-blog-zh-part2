# 世界上最小的漫画刻在人的头发上

> 原文:[https://hack aday . com/2014/05/17/worlds-small-comic-etched-on-a-human-hair/](https://hackaday.com/2014/05/17/worlds-smallest-comic-etched-onto-a-human-hair/)

![world's smallest comic](../Images/db7b5f020f02a10c7978131d5331ed94.png)

[Sébastien Bourdeauducq]有了创造世界上最小的漫画的想法——所以他组建了一个团队，并能够制作一幅漫画，其中每个画面都是 20 微米宽的<,蚀刻在一根人的头发上。

他们使用聚焦离子束(FIB)蚀刻，其中产生一股高速物质射流，并以一根头发为单位喷射，以蚀刻漫画。把它想象成一束超级聚焦的激光束。在他们的 GitHub 上查看它的原始显微图像。

有趣的是，他们这样做只是为了好玩——目的是为在汉堡举行的 [EHSM 会议](http://ehsm.eu/)做广告。异常硬(软件)&软(软件)会议是开源和 DIY 黑客世界的一次大规模聚会。如果你碰巧住在欧洲，我们建议你去看看——看看他们安排的会议演讲就知道了！创客运动遇上专利法，光学&乐高积木光子学，量子密码学基础，质谱原理……不胜枚举！
