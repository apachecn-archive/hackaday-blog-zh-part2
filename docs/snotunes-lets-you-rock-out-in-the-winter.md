# SnoTunes 让你在冬天尽情摇滚

> 原文：<https://hackaday.com/2015/02/03/snotunes-lets-you-rock-out-in-the-winter/>

[Matt Bilsky]是 Hackaday 多年的忠实读者，他终于鼓起勇气向我们提交了一个项目。我们发誓，我们不字节！不管怎样——我们很高兴他这么做了，因为他的项目绝对棒极了。他称之为 SnoTunes，这是一个为户外设计的背包立体声系统。

这是一个巨大的 160 瓦立体声，电池寿命为 7-8 小时，有点防水，可以无线控制。它的大脑是运行 Kodi(以前是 XBMC)的 Raspberry Pi B+背包内部隐藏了一个 7 英寸的显示屏，可以进行更多的微调控制。

它可以获取和下载 YouTube 音乐视频，还可以创建一个可以通过短信操作的播放列表。你可以分享 YouTube 链接，让它下载歌曲并排队，你可以跳过歌曲(但只有四个人提出请求)，它甚至可以自动解析音乐视频标题，提取歌曲名称和乐队。它也适用于 AirPlay——但谁会用它呢？

他有一堆解释建造的视频，如果你想尝试建造一个，他的个人网站上有大量的资源。我们以前见过很多便携式的自制立体声音响，但是从来没有放在背包里！
