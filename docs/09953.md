# 孩子和黑客:盲人机器人

> 原文：<https://hackaday.com/2015/09/10/kids-and-hacking-blind-robotics/>

如果你是一个业余爱好者，很有可能当你还是个孩子的时候，就有一些成年人感染了你对科学、工程、修补或其他任何东西的狂热，让你来到这里。也许是家长或者老师。对我们中的许多人来说，这是一个当地业余无线电操作员。但很可能是某个对这种事情有热情的人，你抓住了它。

提前偿还债务是非常有益的。学校和青年组织总是在寻找与孩子分享激情的人，在合适的年龄和合适的学校，你可以成为推动孩子远离不良道路的人。

真正的问题是你做什么？孩子们对冗长的幻灯片演示和抽象的讲座反应不佳。我总是被教导“要教学，你必须被听到。”让孩子倾听的最好方法是让他们参与一些活动。

多年来，我一直在和孩子们一起做一个小练习，我想和大家分享一个很好的例子，告诉大家如何利用有限的时间和资源。你唯一需要的特殊装备是可以充当眼罩的东西(比如大手帕或睡眠面罩)。其他的一切你都可以即兴创作，你可以根据你手头的资料、学生的年龄和你所拥有的时间来定制具体的细节。

### 用你的思维给机器人编程

我开始问孩子们关于机器人的问题。他们有吗(比如 Roomba)？他们在电影中看到过什么？机器人是怎么移动的？让他们和你交谈，而不只是倾听，你就成功了一半。然后我把孩子们分成至少两队。小组应该有 4-6 个孩子，尽管你的小组越多，你需要的时间就越多。

我给一个虚构的机器人设定了一个任务。我会站在垃圾桶或其他方便的地标前说:“我是这个充电站的机器人。”比如，我会走到一把椅子前，说“我得去饮水机那里拿点水。”然后我会走到其他容易找到的地方，告诉他们这些是需要浇水的植物(两个就足够了)。然后我会回到充电站。

孩子们可以使用笔记本、活动挂图或白板来写一个“程序”，这只是用英语完成工作所需的步骤。你可以从每队挑选一个孩子做机器人，另一个做电脑。来控制机器人。如果我有更大的孩子或其他成年人，我会给每个队一个“教练”，但有时我会这样做，只是在队与队之间游走。但是这里有一个问题:教练应该推动至少一个团队做一套开环步骤，另一个团队做闭环步骤。

### 在开环和闭环中思考

我的意思是。一组学生可能会写下类似“向前迈 4 步，然后转到 3 点钟方向。”这是开环的。另一组可以编程为“向前走，直到你碰到桌子，然后转向 3 点钟方向。”这是闭环。

当队伍准备好时，机器人会得到一个眼罩(我通常不会提前告诉他们)。计算机必须准确读出程序中的内容，机器人必须服从。结果往往令人捧腹。通常(但不总是)闭环团队做得更好。

当它结束的时候，你可以和他们一起复习他们学过的东西:计算机很字面；传感器很重要；开环控制和闭环控制的区别。我可能会展示或提及一些其他的例子。例如，大多数 3D 打印机是开环的(除了家用开关)，因为价格是一个因素，步进电机非常精确。马桶使用闭环控制来操作注水机制(对于正确的年级水平，讨论马桶很有趣)。

### 培养下一代黑客

如果你有一个小时和一些孩子在一起，这是一个很好的锻炼，也很有趣。你可以根据年龄量身定制(例如，高中学生应该能够学习更粗糙的课程，并编写更严谨的程序)。

你可能发誓再也不回学校了。但是，和孩子们一起做这样的活动是令人愉快的，你永远不知道他们中的一个人在完成大学工程学位后什么时候会穿过城镇，停下来说声谢谢。即使他们没有，向前支付的满足感也是值得的。你甚至可以自己学到一些东西。
