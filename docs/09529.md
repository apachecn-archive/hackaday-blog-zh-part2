# 自制相机稳定器

> 原文:[https://hackaday.com/2015/07/13/homemade-camera-stabilizer/](https://hackaday.com/2015/07/13/homemade-camera-stabilizer/)

我们已经在这里展示了相当多的相机万向节和稳定的凸轮，但这一个脱颖而出。首先，[丹尼尔·瑞尤]在大二的时候建造了它。他的 [2 轴相机万向节](http://diyinnovator.com/brushlessgimbal/)使用无刷 DC 电机，由碳纤维制成。

[Daniel]在数控台式铣床和一些手工工具上加工碳纤维零件。他还必须自学扎实的作品来设计它。在他光滑的 DIY 指南中，他首先列出了零件和从哪里获得它们，以及所需的工具。大多数万向节使用伺服轴运动，这限制了范围，不能提供非常平滑的运动。无刷电机克服了这些限制，使一个漂亮，平滑的移动万向节建立了广泛的运动范围。当阿列克谢·莫斯卡伦科(Aleksey Moskalenko)推出 AlexMos 无刷电机控制器(T1)时，[丹尼尔]订购了这款产品，然后一直等到他能找到合适的电机。所有加工零件的 CAD 文件可从[下载](http://diyinnovator.com/wp/wp-content/uploads/2015/07/solidworksGimbalParts.zip)(。zip 文件)。

然后，他继续在博客上写他的构建进度，用大量的照片来描述加工和组装。他在这个过程中做了几个不错的设计选择——比如使用压紧螺母使组装和拆卸变得容易，拆除一个电机并用定制的更长的轴代替它的轴，而不是使用耦合器来延长它。最后，结果不仅是一个好看的，重量轻的钻机，但由于他使用的电机和控制器，工作得非常好。看看下面的视频，看看它的作用。
