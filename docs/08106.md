# 带有内置屏幕的 SEGA Dreamcast 控制器

> 原文：<https://hackaday.com/2015/01/20/a-sega-dreamcast-controller-with-a-built-in-screen/>

[Fibbef]在为一个建筑竞赛项目努力工作时，意外地烧坏了电路板。他不是一个轻易放弃的人，他选择在比赛只剩两天的时候开始一个新项目。他设法在这么短的时间内修改了一个 SEGA Dreamcast 控制器来控制彩色屏幕。

Dreamcast 控制器的形状在某种程度上有助于这种类型的 mod。它已经有一个小窗口，以确保视觉记忆卡的视线不受阻碍。不幸的是[Fibbef 的]屏幕对这个窗口来说有点太大了。这意味着他必须扩展控制器和电路板。

拆开控制器后，他拆下了存储卡连接器。然后，他干净利落地将电路板垂直切成两半。他必须用手将所有的线路重新连接起来。原来最初是他搞砸了什么东西，不小心把控制器的右半部分烧了。为了解决这个问题，他将第二个控制器切成两半，并将两块电路板焊接在一起。

由于 PCB 侧有更多的水平空间，现在[Fibbef]需要扩展控制器的外壳。他把控制器切成几块，确保开始按钮居中美观。然后，他用胶带将冰棒棍固定住，以弥补案件中缺失的部分。然后，所有的棍子都覆盖上一层厚厚的 ABS 水泥，以形成一个更坚固的外壳。所有这些最终都被 Bondo 所掩盖，Bondo 是视频游戏控制台 mods 中的一个[常见伎俩](http://hackaday.com/2012/12/13/a-pair-of-n64-portables/ "N64 Mod")。然后用砂纸打磨光滑，并涂上黑色底漆，以获得令人惊讶的漂亮效果。

屏幕本身仍然需要一种方法来获得电源和视频信号。[Fibbef]制造了一个适配器盒来接收这两种信号，并通过一根电缆将它们传递给控制器。盒子作为电源输入的 USB-A 连接器，以及视频的复合连接器。还有一个 USB-B 连接器用于输出信号。[Fibbef]使用标准打印机 USB 电缆向控制器发送电源和视频信号。最终结果看起来很棒，并有助于使 Dreamcast 更便于携带。看看下面的演示视频，看看它的实际效果。
