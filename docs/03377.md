# 全轮机器人构建使用了所有的东西

> 原文：<https://hackaday.com/2013/06/08/omniwheel-robot-build-uses-a-bit-of-everything/>

机械师、电子工程师、程序员和工厂工人都是你可以运用的技能，如果你承担一个项目，比如[建造这个全轮机器人](http://www.mtahlers.de/index.php/robotik/omnivehicle) ( [翻译](http://translate.google.com/translate?sl=auto&tl=en&js=n&prev=_t&hl=en&ie=UTF-8&eotf=1&u=http%3A%2F%2Fwww.mtahlers.de%2Findex.php%2Frobotik%2Fomnivehicle))。

全向轮在这种三脚架方向上工作，因为它们包括垂直于轮轴转动的滚轮。这避免了由固定位置车轮引起的差异问题。当三个电机被正确驱动时，如下面的视频所示，这种设计使轮式机器人最具机动性。

铝板作为底盘。[Malte]铣削板，为电机切割出带螺纹孔的槽，以接收安装螺钉。几个支架支撑着构成电子部分的大块原型板。大型 DIP 芯片是 ATmega168。它通过三个红色步进电机驱动板驱动电机，这些驱动板是他在易贝捡到的。

到目前为止，该车是拴着的，使用的是仿制的六轴式控制器。但正如我们之前所说的，正确驱动马达是困难的部分，他肯定解决了这个问题。
