# SparkFun 爬楼梯机器人挑战赛

> 原文：<https://hackaday.com/2015/03/26/sparkfun-stair-climbing-robot-challenge/>

如果你错过了，SparkFun 最近举办了 [Actobotics 爬楼梯挑战赛](https://www.sparkfun.com/news/1782)，在那里你可以建造一个能够爬楼梯的机器人，并赢得一些甜蜜的 SparkFun 现金！

比赛现在已经结束，获胜者刚刚宣布——参赛者提出的一些机器人简直太棒了！

一等奖颁给了[耶格家族],他们制造了一个轮式机器人，可以在楼梯上滚动，甚至不眨一下眼睛——看起来很酷。请查看以下内容。

我们真的很喜欢第二名的获胜者——它非常慢，但却是一个非常巧妙的设计。我们认为你会同意的。

[https://www.youtube.com/embed/un_GEB4s_4U?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/un_GEB4s_4U?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[Chad Giddings]机器人并没有上楼梯，但仍然是一个非常酷的机器人平台，他一直在开发。坦克踏板驱动，带有一个车载工具抓手、摄像机和一组轮子(理论上)让它在平衡自己的同时爬楼梯。
