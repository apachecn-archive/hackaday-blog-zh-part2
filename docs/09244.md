# Hackaday 奖参赛作品:有点暗

> 原文:[https://hack aday . com/2015/06/09/hack aday-prize-entry-a-bit-dingus/](https://hackaday.com/2015/06/09/hackaday-prize-entry-a-bit-dingus/)

曾几何时，几乎每台电脑——甚至笔记本电脑——都带有并行端口。这是 25 针的比特碰撞好，可从每一个编程环境，可以控制任何随机堆在桌子上的电子设备。并行端口的时代已经一去不复返了，如果你想用计算机控制一个引脚，你可以通过 USB 或其他方式控制一个微控制器。

【ajlitt】的[微型丁格斯](https://hackaday.io/project/4826-tiny-bit-dingus)就是:一个塞在带有几个引脚头的 USB 插头中的微控制器。使用正确的应用程序，您可以通过 USB 控制这些引脚。这是最接近现代硬件的并行端口。

这位丁格斯并不意味着取代巴士海盗，一个 Arduino，或任何其他东西；这是一种小而简单的方法，用尽可能少的部件将随机电子设备连接到计算机。如果你正在寻找一个零件添加到你的电子修补程序日常携带钻机，这将是它。

在钻头内部有一些有趣的硬件。不久前，[ajltt]遇到了 [Freescale KL27](http://www.freescale.com/webapp/sps/site/prod_summary.jsp?code=KL2x#MKL27Z256VFM4) ，这是一款 Cortex M0+,它没有晶体，有 USB 引导加载程序，根本不需要很多额外的组件。这是该项目的完美尺寸，5x5mm，不会折断，但仍然可以通过 USB 闪存。
