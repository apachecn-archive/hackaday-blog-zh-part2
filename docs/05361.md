# 这个派对点唱机里面更大

> 原文：<https://hackaday.com/2014/02/21/this-party-jukebox-is-bigger-on-the-inside/>

为了纪念最近的《神秘博士》50 周年，[大卫·普劳蒂]决定建造一个 1/3 比例的 Tardis 复制品。他还决定在内部增加一些额外的功能……介绍[回收的 Tardis 点唱机！](http://imgur.com/gallery/oLkDENl/new)T3[T5](http://imgur.com/gallery/oLkDENl/new)

它主要是用回收的纸箱(披萨、联邦快递、UHaul 等)建造的，[大卫]在粉刷和细节方面做得非常出色！

因为它太大了，[大卫]希望它也很实用，所以他在里面添加了蓝牙扬声器、声控灯、迪斯科球，甚至还有一台造雾机。这一切都是由遥控器无线控制的，它肯定会在他决定举办的任何派对上大受欢迎。

请留下来观看展示其运行的视频——当然，还有制作我们最喜欢的声音 VWORRRRRP VWRORRRP VWORRRP！
