# 伏特表时钟也显示温度

> 原文：<https://hackaday.com/2012/08/24/volt-meter-clock-also-displays-the-temperature/>

[IronJungle]着手组装每个修补匠最喜欢的项目:一个以奇怪方式显示时间的时钟。对于他的时钟，[Jungle] [采用了三个电压表](http://ijprojects.blogspot.com/2012/03/voltmeter-clock-w-f-c-and-k-temperature.html)并将它们转换成一个时钟，在定制的纸质表盘上显示当前的小时、分钟和秒钟。

[IronJungle]将 PIC 14M2 微控制器连接到 DS1307 实时时钟，以跟踪当前时间。至于显示，[丛林]采取了三个伏特表，并连线到他的图片上的 PWM 输出。有了这个，他能够精确地控制指针在仪表中的位置，从而显示时间。

除了显示时间，[钢铁丛林]还在他的身体上安装了一个小型温度传感器。通过按下秒显示下方的按钮，时钟能够以华氏度、摄氏度和开尔文显示当前温度。

休息过后，你可以看看[IronJungle]的伏特计时钟一小时一小时的定时录像。
