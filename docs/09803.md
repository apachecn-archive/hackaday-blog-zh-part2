# Arduino 动力橡皮筋哨兵炮塔不是谎言

> 原文:[https://hack aday . com/2015/08/13/arduino-powered-rubber-band-sentry-turtle-is-a-lie/](https://hackaday.com/2015/08/13/arduino-powered-rubber-band-sentry-turret-is-not-a-lie/)

你知道当你不在的时候，隔壁房间的那个家伙会偷偷溜进来，把你桌子上的宇航员冰激凌偷走。多亏了凯文·托马斯，如果你有一台 Arduino 和一台 3D 打印机，你就可以制作一把橡皮筋哨兵枪来保护你讨厌的食物。你还需要一些公制硬件，一个 Arduino Uno，和一些伺服电机。

视频显示[凯文]手动瞄准枪支，但如果你在硬件上添加一些传感器，[软件可以自主操作枪支](http://projectsentrygun.rudolphlabs.com/)。建造细节有点稀疏，但有一份材料清单，结合 3D 打印文件和视频，应该可以让你弄清楚。

我们不禁希望有一个第一人称视角(FPV)的摄像头，并通过手机控制，这样你就可以躲在扫帚间里狙击那些冰淇淋贼。另一方面，如果你让枪工作，增加遥控器一点也不难。你可能在你的四轴飞行器上有一个 WiFi FPV 相机，最终从那棵树出来，有很多方法可以通过蓝牙或 WiFi 进行控制。

不是说你没有选择。但是在 Hackaday 总部，我们有很多橡皮筋，没有多少绿色猪。如果你[宁愿打彩弹](http://hackaday.com/2011/03/22/autonomous-paintball-sentry-gun-2/)，小心不要不小心把你的立方体内部重新粉刷了。
