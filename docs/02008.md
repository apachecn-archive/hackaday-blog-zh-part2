# 可怕的南瓜馅饼

> 原文:[https://hackaday.com/2012/10/31/a-scary-pumpkin-pi/](https://hackaday.com/2012/10/31/a-scary-pumpkin-pi/)

[![headless-scarecrow](../Images/276f335b6a594be78b4629f168014709.png "headless-scarecrow")T2】](http://hackaday.com/?attachment_id=89251)

当你把运动传感器、树莓派和南瓜组合在一起时，你会得到什么？当夜幕降临时，一个无头的稻草人，腿上放着一个[发光的雕刻南瓜](http://www.insentricity.com/a.cl/189/RaspberryPiHalloweenHack "Raspberry Pi Halloween hack")！这种破解的执行真的很棒，正如休息后的视频所示，其结果应该会让任何敲门的孩子感到非常害怕。

这种黑客的一个巧妙效果是，它使用 X10 家庭自动化模块来关闭门廊的灯，以获得额外的可怕效果。在这之后，南瓜灯会亮起并说“救救我！”如果你想复制，或者在此基础上构建，页面上提供了说明以及所有内容的源代码。当你在那里的时候，一定要浏览一下[【不可思议】](http://www.insentricity.com/index.cl "Insentricity")，因为还有很多其他的技巧可供你参考！别忘了一定要把你想到的其他万圣节技巧发给我们。
