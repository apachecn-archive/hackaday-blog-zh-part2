# 爆笑 DARPA 机器人坠落视频

> 原文：<https://hackaday.com/2015/08/23/hilarious-darpa-robots-falling-video/>

如果你担心天网，快速浏览一下 [DARPA 机器人挑战赛](http://hackaday.com/2013/12/22/darpa-robotics-challenge-trials-wrap-up/) (DRC)的视频片段卷轴(YouTube，内嵌在休息时间下方)，你会感觉好很多。分配到 DRC 的 IEEE Spectrum 记者从他们的[严肃报道](http://spectrum.ieee.org/automaton/robotics/humanoids/drc-finals-course)中抽出时间，制作了这个有趣的汇编，汇集了所有价值数百万美元的机器人故障[和这篇文章](http://spectrum.ieee.org/automaton/robotics/humanoids/darpa-robotics-challenge-robots-falling)。

[![robot_falling](img/acd1b36f8cf8e8dd13374ee9a169b428.png)](https://hackaday.com/wp-content/uploads/2015/08/robot_falling.gif) 很难不对机器人产生共鸣，我们承认我们对第一对夫妇的摔倒有一点退缩。(但是三四次之后，开始爆发出疯狂的笑声。)

[https://www.youtube.com/embed/g0TaYhjpOfo?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/g0TaYhjpOfo?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

所以你可能至少在几年内不会受到终结者的威胁。但是在你开始沾沾自喜之前，人类，记住发生在自动驾驶汽车上的 [DARPA 大挑战](https://en.wikipedia.org/wiki/DARPA_Grand_Challenge)；2004 年没有车队完成比赛，但是一年后有五支车队在一场相当激烈的比赛中完成了比赛。DARPA 的挑战和其他事情一样都是关于学习的。

请记住，当你看到这些机器人在试图转动门把手时摔倒时，它们在试验的前一阶段已经坐下来并驾驶过汽车。那真是太酷了。
