# 纸板六足机器人有三个马达

> 原文:[https://hack aday . com/2012/04/23/cardboard-hexapod-gets-around-with-three-motors/](https://hackaday.com/2012/04/23/cardboard-hexapod-gets-around-with-three-motors/)

![](../Images/79e6f1e65c03f9e24590176d8fce5d16.png "cardboard-hexapod")

这是少花钱多办事的一个教训。[Oldrobot] [用纸板](http://electronicpapercraft.blogspot.com.au/2012/04/cardboard-robot-beetle.html)做了这个六足机器人的大部分零件。他仍然从他的真空吸尘器中取出盒子，它只是碰巧有一个大的黑色区域，使得甲虫的顶部看起来像是被油漆过的。

控制板来自一架旧的无线电控制飞机。由于遥控飞机使用伺服系统进行飞行控制，所以很容易就可以连接三个系统。一个控制中间腿的设置，中间腿提升身体并改变哪个推进腿与地面接触。另外两个服务器移动成对的前腿或后腿。它使用了与另一款 RC 控制器 hexapod 相同的概念，但在制作底盘和腿上花费的时间要少得多。

正如你在休息后的视频中看到的，控制方案不是最直观的。但是一旦你掌握了控制杆的方向会影响每条腿的运动，这个机器人就会有相当精确的转向。
