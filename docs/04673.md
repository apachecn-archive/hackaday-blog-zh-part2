# 8 位视频游戏是预算有限的复古游戏的最佳选择

> 原文：<https://hackaday.com/2013/11/28/8-bit-video-game-is-best-of-retro-gaming-on-a-shoestring-budget/>

[Petri]写信来炫耀他和[Antti]开发的 8 位游戏系统和原始平台。现在不要误解我们，令人印象深刻的是二人组能够把看起来非常有趣的游戏放在一起。但是我们已经看到许多行业领先的视频游戏是由一两个人开发的(我们认为[可以追溯到雅达利](http://hackaday.com/2013/06/05/retrotechtacular-how-i-wrote-pitfall-for-the-atari-2600/)的时代)。不，对我们来说最有趣的是游戏机也是他们的发明。我们应该注意到，标题屏幕是他们的朋友[Juho]的作品。

对此不能全信，因为插图右下角的图像明显包含了一个 Arduino。但是，这难道不是开放硬件和通过互联网分享知识的状态的一个证明吗？这甚至在业余爱好的水平上也是可能的。我们称之为“爱好”并不意味着你必须降低期望值。这东西功能齐全。休息后观看该片段，可以看到 ATmega328 驱动一个 104×80 分辨率的屏幕，采用 256 色调色板，同时为 chiptunes 使用四个音频通道。这个东西甚至利用一个原始的 NES 控制器端口供用户输入。

对于那些认为[我们在](http://hackaday.com/2010/10/24/hackvision-is-build-your-own-retro-game/)之前已经见过同样的事情的人来说，我们永远不会厌倦看到那些大量努力显然得到回报的项目！
