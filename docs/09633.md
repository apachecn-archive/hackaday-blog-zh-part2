# 想把东西点着，你知道…为了科学。

> 原文：<https://hackaday.com/2015/07/25/trying-to-set-things-on-fire-you-know-for-science/>

来自[Just Think]的这段视频引起了我们对 6 种不同常用液体燃料的明火测试的注意:Jet-A、柴油、取暖油、煤油、avgas(100 升)和汽油(对我们的国际读者来说是汽油)。我们喜欢他的低技术测试方法——只有一些泥瓦匠罐子和一个后院。[结果相当有趣](https://www.youtube.com/watch?v=7nL10C7FSbE)。

他从测试 Jet-A 燃料开始。是的，和你给波音 737 加满油的东西一样。当他把一根火柴扔进去时，我们退缩了一秒钟——然后什么也没发生。这是一个普遍的误解，认为喷气燃料会自己维持火焰。正如他在视频中展示的那样，它需要雾化才能燃烧。他继续测试柴油和取暖油——注意到除了颜色之外，两者完全相同。取暖油中添加了一种红色染料，标明“非卖品”,因为它不征税。两者都不会保持火焰。

接下来是煤油，它只能勉强维持火焰。煤油通常在极冷的气候下用作柴油的替代品，因为柴油会胶凝并堵塞燃油系统。最后，他测试 avgas 和汽油。两者都能很好地维持火焰。

我们认为这个小实验很有趣，因为结果有点反直觉。所有这些不同的燃料因其不同的属性而用于不同的应用，当然，如果你想了解更多，这背后有一些真正有趣的科学。

我们不需要告诉你在使用燃料时要注意安全。尽管像 Jet-A 燃料这样的东西本身不会在容器中携带火焰，但这并不意味着当它与其他东西(如衣服)结合在一起时，在其他情况下不会剧烈燃烧。所以我们把这个归档在“不要在家里尝试这个”——相反，休息后坐下来享受 YouTube 视频。
