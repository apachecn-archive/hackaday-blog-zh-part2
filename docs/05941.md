# C64 MIDI 和闪光灯推车

> 原文：<https://hackaday.com/2014/04/30/c64-midi-and-flash-cart/>

Commodore 64 和 128 中的 SID 芯片可以说仍然是芯片音乐的黄金标准，C64 本身仍然是 MIDI 排序的不错的计算机。[Frank Buss]意识到 Commodore 计算机的大多数 MIDI 墨盒要么停产，要么严重受限，[所以他着手创建自己的](http://www.startnext.de/en/kerberos)。

与市面上为数不多的 Commodore MIDI 墨盒不同，[Frank]的 Kerberos 有 MIDI 输入、输出和直通，由 6850 ACIA 芯片控制，就像 80 年代的旧接口一样。这允许 Kerberos 与旧的时序电路、Passport 和 Datel 软件接口。他在众筹网站上提供 Kerberos 购物车，所以如果你想自己买一辆，就试试吧。

因为 Kerberos 也是一个 Flash cart，所以它也为 T1 提供了一些软件；[Frank]获得了 Steinberg 的许可，可以在 Kerberos 上安装他们的 Pro 16 软件。 [SID Wizard](http://sourceforge.net/projects/sid-wizard/) 也预装在推车上，还有一些其他难以置信的跟踪器和测序仪。当然，不要求购物车的 Flash 部分只能存放 MIDI 和 synth 软件。你可以通过 MIDI 接口上传一些游戏到购物车。下面是 Kerberos 的视频。
