# 通过黑客技术打造更好的电视

> 原文：<https://hackaday.com/2015/09/04/better-tv-via-hacking/>

智能电视只是有电脑和网络连接的哑电视，对吗？在规则 34 的一个变体中，如果它里面有一台电脑，就会有人入侵它。当[smarttvhacker]购买了一台索尼 48 英寸智能电视时，[他注意到手册中列出的所有软件许可证](https://smarttvhacking.wordpress.com/2015/08/24/hacking-the-sony-kdl-48r510c-smart-tv/)，并意识到这是入侵电视的一大助力。

我们没有可比的索尼模型，但[smarttv hacker]的帖子是他从电视观众到电视统治者的旅程的名副其实的旅行日志。通过分析从网络端口扫描到固件升级转储的一切，他最终能够安装一个 telnet 服务器。

我们不确定他现在有了 shell 提示符后打算做什么，但这真的不是重点，对吗？我们常说，如果你打不开它，你就不拥有它。或许一个推论会是:如果你不能登录它，你就不拥有它。

Blackhat 几年前曾利用三星电视(见下面的视频)。另一方面，何必呢？只要[用树莓酱](http://hackaday.com/2013/11/20/raspberry-pi-smart-tv/)自己做就行了，这已经[做过不止一次](http://hackaday.com/2013/06/08/raspberry-pi-used-to-build-a-smart-television/)。
