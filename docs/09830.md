# Hackaday 奖参赛作品:软矫正器

> 原文：<https://hackaday.com/2015/08/14/hackaday-prize-entry-soft-orthotics/>

美国有近一百万人患有脑瘫，这是一种导致四肢痉挛的神经疾病。对于脑瘫患者来说，生活质量的最大因素之一是使用手臂的能力，这意味着他们的肘部需要一个昂贵而笨重的矫正器。[Matthew]有一个更好的主意:[为什么不做一个软矫正器](https://hackaday.io/project/7174-neucuff-a-soft-orthotic-exoskeleton)？

这不是[马修]的第一个软机器人项目。他是超级释放器公司的首席科学家，该公司负责研发完全柔软的机器人“大西洋绿海龟”和其他柔软的气动机器人“T2”。

这种柔软，灵活的外骨骼矫形是专为慢性运动障碍患者设计的。传统的矫形器价格昂贵，难以移动，并且不舒服，但是通过设计这种矫形器，使其同样坚固但更宽容，这些装置将大多数问题最小化。

Neucuff 由极其简单的材料制成——只有一些氯丁橡胶、一个 velcro 和一个二氧化碳筒。与所有医疗设备一样，将这种产品推向市场的问题是 FDA 的要求和认证。这使得 Hackaday 奖成为[Matthew]和 Super Releaser 的其他人以及任何试图通过监管要求来改变世界的人的一个绝佳机会。
