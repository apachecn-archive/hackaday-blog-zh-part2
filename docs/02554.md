# 现实生活中的马里奥赛车对可发射的物品做出反应

> 原文：<https://hackaday.com/2013/01/17/real-life-mario-karts-react-to-launchable-items/>

![real-life-mario-kart](img/c0f19ef33c6cf5dc5a8b633b3ed3e028.png)

滑铁卢实验室的团队通过这个现实生活中的马里奥赛车项目再次证明了他们的创造力。其中一名团队成员必须在奥斯汀公园工作，因为他们似乎可以自由地改装卡丁车和拍摄休息后发现的令人愉快的视频演示。

穿上戏服去卡丁车场是一回事。修改交通工具以对另一个骑手发射的物品做出反应是另一回事。您可以在每个驱动程序的右侧看到涂有颜色的 PVC 启动器硬件。通过塞入毛绒物品(香蕉、星星、蘑菇等)。)在炮管里，驾驶员可以用压缩空气将它发射相当远的距离。每个物品的内部都有一个 RFID 标签。当卡丁车上的 RFID 阅读器检测到标签时，它可以控制方向盘、刹车踏板或限速器，采取适当的行动。

如果你没有这种内部渠道进入你当地的卡丁车赛道，不要担心。你只需要[建造自己的卡丁车](http://hackaday.com/2012/05/11/the-wiikart-a-wireless-go-kart/)。
