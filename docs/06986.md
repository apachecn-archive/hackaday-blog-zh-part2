# 自制便携式采金滚筒筛

> 原文:[https://hack aday . com/2014/09/02/自制-便携-采金-滚筒筛/](https://hackaday.com/2014/09/02/homemade-portable-gold-mining-trommel/)

[the jog dredger]一直在测试他在家里制作的新黄金洗衣机。将混有岩石的泥土通过这个滚筒筛，黄金和贵重物质就能被过滤出来。这个过程的视频可以在下面看到。

该设备由本田 GX120 发动机驱动，旨在方便从一个地点拖到另一个地点。系统中没有使用塑料部件，以使其更加耐用。但是不使用塑料意味着这个家伙很重。钻机干重约 240 磅，湿重约 265 磅，附带一个水槽。橡胶轮胎使得机器可以毫无困难地从一个地方移动到另一个地方。

虽然网站上描述了这些部件，但在网上找不到这种特定设备的操作说明。这可能是因为,[ the jogdruder]正试图出售他的产品，赚些钱。发布如何建造自己的指令很可能会减少他的设计的潜在利润。不管是哪一种，这是便携式采金滚筒筛，非常适合那些希望加强他们的采金和探矿游戏的人。我们心中真正的问题是:你能以这种方式用更少的努力获得更多的黄金，还是通过电子垃圾采矿获得更多的黄金？
