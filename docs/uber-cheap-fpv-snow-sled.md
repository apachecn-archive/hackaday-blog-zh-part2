# 优步-便宜的 FPV 雪撬

> 原文：<https://hackaday.com/2015/03/20/uber-cheap-fpv-snow-sled/>

冬天的忧郁让你们这些遥控车辆的家伙沮丧了吗？好吧，振作起来！春天就在眼前。不过，与此同时，你可以从[benn builds]的书中借鉴一页，建造一个[FPV 雪撬](https://www.youtube.com/watch?v=mDbBnj_H-Sc)。

那么，在冬天使用这款雪橇与其他遥控车辆有什么不同呢？[BenNBuilds]在他温暖舒适的房子里控制它。车载 FPV 摄像机将视频传回房子里的接收器，视频可以显示在液晶显示屏或 FPV 护目镜上。此外，当遇到棘手的情况时，能够从窗口看到雪橇也不会有什么坏处！

这艘飞船本身是用闲置的零部件廉价制造的。框架由泡沫板制成，由导管电风扇驱动。类似于气垫船或沼泽船的方向舵提供方向。由于[benn builds]发射机有几个额外的频道，他为他的 FPV 摄像机安装了一个云台系统。休息过后，看看这只小狗的视频。
