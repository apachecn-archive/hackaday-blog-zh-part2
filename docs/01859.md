# LED 矩阵吊坠

> 原文：<https://hackaday.com/2012/10/12/led-matrix-pendants/>

如果你想摆弄一些微控制器，但心里没有真正的目的，这个项目非常适合你。它很便宜，容易组装，还有闪烁的发光二极管！[TigerUp]向我们展示了他如何只用五个组件组装[一些 LED 矩阵挂件。](https://sites.google.com/site/tinymatrix/)

他称这个项目为微型矩阵，因为挂件的轮廓只有 0.5 英寸乘 0.7 英寸，所以很合适。在背面，ATtiny2313 芯片被直接焊接到 LED 显示屏的腿上。它们只是碰巧与芯片上的 I/O 引脚对齐，这使得焊接非常简单。电源来自一个硬币电池，它通过一根红黑相间的电线连接到吊坠上，这根电线构成了这个装置的项链。尚未提及的最后两个元件是用于改变模式的瞬时按钮开关和 reset 引脚上的上拉电阻。材料清单以 4 美元的价格响起，他的固件提供了 9 种不同的模式，正如你在休息后的剪辑中看到的那样。

[TigerUp]的灵感来自于[这个 8×8 矩阵项目](http://hackaday.com/2009/03/21/64pixels-are-enough/)。
