# 银河 S4 感应充电黑客保持一切案件内

> 原文:[https://hack aday . com/2013/08/12/galaxy-S4-inductive-charging-hack-keep-everything-inside-the-case/](https://hackaday.com/2013/08/12/galaxy-s4-inductive-charging-hack-keeps-everything-inside-the-case/)

我们已经看过这个黑客很多次了，但是这个做了一个伟大的工作[将所有的手机端感应充电组件](http://hackintech1.wordpress.com/2013/07/30/diy-samsung-galaxy-s4-wireless-charging-for-under-30/)内在化。

它使用 Palm Pixi 无线充电硬件，这似乎是最受欢迎的系统。我们已经看到，你可以将它添加到任何使用 USB 充电的手机中。但我们不喜欢打开手机来焊接 USB 接头的想法。我们也不希望 USB 插头[一直伸出手机底部](http://hackaday.com/2010/07/28/inductive-cellphone-charging-without-voiding-warranty/)。

这个黑客满足了这两个问题，它实际上是感谢制造商。三星 Galaxy S4 恰好在可拆卸背板内有两个触点，这两个触点是为三星自己的感应充电硬件设计的。与手掌充电硬件的接触是通过将铜箔压入到位来实现的。后盖内侧的匹配箔迹线将此修补到 Touchstone 接收器硬件中，这是从 Palm case 直接移植而来的。

这被吹捧为成本低于 30 美元的解决方案。这大大超过了三星正品感应充电套件的当前价格。
