# PebblyPi:智能门铃

> 原文:[https://hack aday . com/2014/05/08/the-pebblypi-a-smart-门铃/](https://hackaday.com/2014/05/08/the-pebblypi-a-smart-doorbell/)

![PebblyPi](../Images/aae7b8287ed12c6a13803e6e6bc7e284.png)

一块 Pebble 智能手表和一个树莓派。他们是完美的一对。这可能是[丹尼尔]在着手他的最新项目时的想法，[一个名为 PebblyPi](http://www.muacksandglomps.com/blog/2014/04/29/pebblypi-smart-doorbell/) 的智能门铃(由[本]提交的提示)。

实际项目很容易实现。你真正需要的只是一个[树莓派](http://www.raspberrypi.org/)，一个开关，一个电阻，一个 [Pebble 智能手表](https://getpebble.com/)(外加一部智能手机)。在 Raspberry Pi 上使用一个简单的 Python 脚本，按钮按压通知被发送到[按钮](https://pushover.net/)，这允许通知到达您的智能手机(从而到达您的 Pebble 智能手表)。Pushover 是一个非常酷的通知服务，适用于 Android 设备、iPhones、iPads 和您的桌面。这个项目背后的概念非常棒，而且它的实现如此简单，这为将您的家用电子设备与 Pebble 智能手表(甚至只是您的智能手机)相连接开辟了许多其他可能性。使用任何联网系统在您的任何设备上创建自定义通知的能力令人惊叹！

你可能会收到来自你的[异常准确的气象站](http://hackaday.com/2013/08/21/ultrasonic-anemometer-for-an-absurdly-accurate-weather-station/)的通知，甚至是你的[土壤湿度监测器](http://hackaday.com/2012/12/03/gadgeteer-plant-monitor-wants-it-wet-and-photogenic/)的通知。你在你的项目中使用过 Pushover 吗？可能性是无限的！
