# 电动三轮车建日志如黑客破解

> 原文：<https://hackaday.com/2013/04/05/electric-tricycle-build-log-is-like-hacker-crack/>

这辆小小的电动三轮车是一辆非常强大的电动三轮车。有些人可能认为测试三轮车不需要安全帽，但看到休息后的视频，其中[Ben Katz]在支撑梁之间飞驰而过，证明这种乘坐方式有一些活力。

浏览上面链接的演示帖子很有趣，但是当我们开始浏览六个构建日志帖子时，我们感觉自己被这个项目吸引住了。每一步都令人欣喜。它始于一个铝箱，将主机的两个后轮，传动系统，电机和电池。[Ben]决定使用 A123 锂电池，并在测试了可用空间后，他开始选择电机和驱动电路。当他最终拿到了这个项目的实际电池时，他开始了构建自己电池的迷人过程。几十个被热粘在一起，然后焊接在一起，然后放入苏打瓶中，用热风枪击打塑料。我们甚至还没有进入自行车轮毂齿轮传动系统，盘式制动器，差速器，链传动和电机…你知道我们说的把你吸进去是什么意思。

哦，如果你想知道这不是[本]的第一个电动汽车制造。去年，他展示了他的全地形滑板车。
