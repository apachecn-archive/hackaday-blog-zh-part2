# 远程呈现机器人让你玩一手牌

> 原文:[https://hack aday . com/2012/05/23/tele presence-robot-let-you-play a hand-of-cards/](https://hackaday.com/2012/05/23/telepresence-robot-lets-you-play-a-hand-of-cards/)

![](../Images/1b403af7acb13e355fb23f863c3a02df.png "card-game-telepresence")

虚拟纸牌游戏使互联网激增，但是这个打牌远程呈现设备是独一无二的。[Patrick]称这个项目为 Vanna，我们打赌这是向电视明星[Vanna White]致敬。就像她翻转空白牌来显示字母一样，这个装置可以翻转牌面朝上或朝下。

六个卡盘中的每一个都连接到一个步进电机。本地玩家发牌，将每张牌放在托盘中，使其面向远程玩家的网络摄像头。该远程玩家有一个屏幕界面，可以通过向前倾斜托盘并将牌放在其正面来弃牌，或者通过向本地玩家倾斜来玩牌，以便他们可以看到其面值。休息过后，剪辑中的一切都变得清晰了。

由于 PIC 18F4585 的控制，硬件可以在 Windows 机器上通过 USB 进行控制。但是让它与您选择的操作系统对话应该很简单。
