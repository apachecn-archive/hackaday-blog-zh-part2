# 字面试验板黑客迫使它接受双引脚接头

> 原文:[https://hack aday . com/2015/08/18/literal-bread board-hack-forces-it-to-accept-dual-pin-headers/](https://hackaday.com/2015/08/18/literal-breadboard-hack-forces-it-to-accept-dual-pin-headers/)

通常当对某样东西有明确需求时，一些企业家会满足这种需求。不幸的是，似乎没有人认为有必要开发一种可以处理双排头电路板的无焊试验板产品。这些器件在两个方向上都有 0.1 英寸的间距，因此虽然它们可以安装在标准试验板上，但触点会使器件上的相邻引脚短路，这使得器件毫无价值。

[Baz]需要将 RF24L01 模块连接到试验板。Baz[没有将导线连接到设备上，也没有设计分线板，而是真的黑掉了他的试验板](http://headfuzz.co.uk/node/143)。为了腾出一个区域来插入双排设备，他把试验板拆开，拉出弹簧触点，切开它们，然后再把它们放回去。

当然，你必须确保切口足够宽，使弹簧的两部分不会接触。看起来像是[Baz]用了一个小螺丝刀来帮助弹簧保持形状，并用简单的斜切器切割它们。

这不是我们看到的第一个[试验板手术](http://hackaday.com/2015/08/14/panel-mounted-breadboard-accessories/)。这可能是一个很好的方法来使用那些积累的小面包板，但似乎永远不会被使用。如果你宁愿黑模块而不是试验板，你可能会从下面的视频中得到一些想法。
