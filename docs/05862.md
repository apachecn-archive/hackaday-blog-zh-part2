# [巴林特]的 GNU 无线电教程

> 原文:[https://hack aday . com/2014/04/22/ba lints-GNU-radio-tutorials/](https://hackaday.com/2014/04/22/balints-gnu-radio-tutorials/)

![Waterfall](../Images/246ec9a6987fd6fe97a14a57a4288941.png)

(巴林特)在处理软件定义的无线电和廉价的 USB 电视调谐器方面有一点历史，这些调谐器在几年前会变成非常昂贵的硬件。现在[巴林特]终于发布了一些非常棒的[GNU 无线电教程](http://files.ettus.com/tutorials/)，旨在让软件定义无线电初学者能够使用当今最酷的硬件进行操作。

[巴林特]在这些地区很出名，因为他是第一个为价格低得令人难以置信的 USB 电视调谐器创建[GNU 射电源块](http://hackaday.com/2012/03/30/working-software-defined-radio-with-a-tv-tuner-card/)的人，允许任何人花 20 美元并有足够的耐心等待来自中国的包，以收听从 22 到 2200 MHz 的*一切*。在那个波段发生了很多有趣的事情，包括客机和交通管制之间的 ACARS 消息，这使得[【巴林特】可以用最少的硬件来扮演空中交通管制员](http://hackaday.com/2012/04/16/playing-air-traffic-controller-with-software-defined-radio/)。

现在的教程是面向绝对初学者的，从 GNU Radio 启动和运行开始。从那里，教程继续接收调频广播，并与[一个小的硬件投资](https://www.ettus.com/product/details/UB210-KIT)，甚至传输多个频率。

毫不夸张地说，软件定义无线电是最通用、最有趣的项目之一。[巴林特]甚至演示了用一个简单的 SDR 项目触发餐馆寻呼机，这个有趣的项目肯定会惹恼他的同事。
