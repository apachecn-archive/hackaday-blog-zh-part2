# 用 Dubstep Drop 灭火

> 原文：<https://hackaday.com/2015/04/13/putting-out-fires-with-a-dubstep-drop/>

乔治梅森大学的两名工程系学生制造了一种相当另类的灭火器。它使用一个低音炮来发送强大到足以扑灭小火的声波。

在概念上类似于一个巨大的烟圈佳能，该设备使用一个低音炮与一个管子，在末端有一个较小的孔径开口。当低音下降(字面上)，这导致一个强烈的声波(嗯，空气)，被驱逐出设备。正如你在下面的视频中看到的，它在扑灭小火方面非常有效。

他们使用一个小型频率发生器和放大器为系统供电，并通过广泛的测试发现 30-60Hz 效果最佳。这实际上不是一次大的空气爆炸，而是一种来回的压力波——搅动空气，并将其与火分离。不过有一个问题。

> “声波的一个问题是它们不能冷却燃料，”伊斯曼说因此，即使你把火扑灭了，如果你不拿走燃料或冷却它，它会重新点燃。

[Robertson]和[Troy]将该项目作为他们的高级研究工作开始，但现在正计划继续开发其商业潜力。他们在 11 月申请了临时专利，并继续在不同类型的火上测试该设备。
