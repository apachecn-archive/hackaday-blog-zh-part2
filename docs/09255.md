# Hackaday 奖参赛作品:教学运算放大器

> 原文：<https://hackaday.com/2015/06/10/hackaday-prize-entry-teaching-opamps/>

TI 制造了一些很棒的芯片，为了销售这些芯片，他们非常愿意将一些很棒的教程、例子和在线课程放在一起，让工程师们开始工作。这不仅限于 5 美元的发射台；TI 为其精密运算放大器提供了出色的视频和实验室系列。这些教程附带了一个评估模块，价格约为 200 美元。是的，几个运算放大器和一个 PCB 需要两个本杰明。当然，没有工程师会支付这个费用；他们的工作会。但是有人想在家学习怎么办？

这就是[SUF]的黑客日奖项目的由来。他正在制作一个价值 200 美元的实验板的复制品，即使没有研究每个组件的最便宜的解决方案，[SUF]估计他可以用大约 50 美元来制作这个工具包。就像我说的，TI 板是商业购买。

TI 提供的完整实验室和教程使用 NI 的虚拟实验室。同样，这也不是一个普通的电子黑客能负担得起的，但是任何想要完成这个教学模块的人都可能会使用他们自己的工具。

就教授电子学的项目而言，[SUF]已经把它赶出了公园。他已经依赖于优秀的教程，但把价格降低到更合理的水平，更容易接受不与公司账户挂钩的支票簿。
