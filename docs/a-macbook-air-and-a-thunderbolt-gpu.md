# 一台 Macbook Air 和一个 Thunderbolt GPU

> 原文：<https://hackaday.com/2013/07/31/a-macbook-air-and-a-thunderbolt-gpu/>

当英特尔和苹果发布 Thunderbolt 时，人们听到了苹果唱诗班的哈利路亚。由于苹果的硬件产品中很少有可以升级的，所以外置显卡是最好的选择。不幸的是，英特尔似乎并不喜欢外部 GPU 的想法。这并没有阻止像拉里·加迪这样的一些有创造力的人独立解决问题。现在，他正在通过 MacBook Air 的 Thunderbolt 端口运行 GTX 570，并在内部 LCD 上显示所有内容。梦想成真。

[Larry]正在用一些相当专业的硬件做这件事。首先是一个[迅雷到 ExpressCard/34 适配器](http://www.bhphotovideo.com/c/product/860811-REG/Sonnet_ECHOPRO_E34_Echo_Pro_ExpressCard_34_Thunderbolt.html)，之后是一个 [ExpressCard 到 PCI-E 适配器](http://www.hwtools.net/Adapter/PE4L%20V2.1.html)。再加上电源、GPU 和大量的软件配置，[Larry]手上就有了一个真正的 Thunderbolt GPU。

当然，通过 Thunderbolt 端口运行 GPU 也有一些缺点。目前的 Thunderbolt 规格相当于 PCI-E 4X 插槽，是发挥高端 GPU 所有马力所需的四分之一。也就是说，这是一种优雅但不完美的方式，可以在 MBA 上获得更好的图形性能，
