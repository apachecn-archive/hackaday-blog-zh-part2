# 呱呱蛋计时器

> 原文：<https://hackaday.com/2012/09/26/quacking-egg-timer/>

![](img/4da84115039cd984c9c908fdca7bef9a.png "quacking-egg-timer")

[汤姆]设法为厨房制造了一个古怪的数字计时器。大多数人会使用一些七段显示器和一些按钮，然后称之为完成，他找到了一种方法，使它变得更有趣。最上面的毛绒玩具是一只黄色的鸭子，长着橙色的嘴。时间用完了鸭子会嘎嘎叫，叫你回厨房。

正如你在休息后的视频中看到的，[汤姆的]和家人在一起忙得不可开交。这个项目对他来说足够快，能在他设法抓住的日益减少的空闲时间里适应它。他使用了 MSP430 Launchpad 附带的一个芯片。由于该系列处理器在睡眠时提供极低功耗模式，因此非常适合这种电池供电的应用。至于鸭子，它是一个玩具，里面有几个手表电池和一个小 PCB。有人四处打探，把他带到了一个垫子上，当他落地时，这个垫子会发出嘎嘎声。
