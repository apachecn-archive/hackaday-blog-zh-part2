# 烦人的机器

> 原文：<https://hackaday.com/2015/02/16/the-annoying-machine/>

无用的机器——一台有拨动开关、机械手臂和一些只会自动关闭的东西的机器——是一种主要的可指示物，由世界各地的随机工作台构建而成。已经是老生常谈了，现在 hackaday.io 有了更好的项目:[烦人的机器](http://hackaday.io/project/4333-the-most-annoying-machine)，纯粹为了烦人而存在的机器。

根据[unigamer]的说法，讨厌的机器是无用机器的邪恶表亲。从外面看，它非常简单:一个标有“开”和“关”的开关，一个 led 孔。打开开关，烦人的机器就会发出烦人的嗡嗡声。关掉机器，嗡嗡声就会消失。然后开关会自动弹回打开状态。阴险。

一个开关和一个蜂鸣器很简单，但是这个建筑的关键部件是一个被驱动的摇臂开关。它基本上是一个普通的拨动开关，带有两个附加的线圈端子，可以电子地来回移动开关。再加上一个 Arduino、蜂鸣器、电池和一个用于开关的升压转换器，这就是全部了。

如何停用烦人的机器？Arduino 上有一个加速度计，在关闭开关后，通过将盒子抛向空中，它将重置。这种烦人的机器已经有了第二版的计划，它将会更加响亮，并且由铝制成。任何能保护它免受不可避免的挫折打击的东西。
