# 机器人最喜欢的笔

> 原文:[https://hackaday.com/2014/06/17/a-robots-favourite-pen/](https://hackaday.com/2014/06/17/a-robots-favourite-pen/)

[![A test of various pens using a robot](../Images/facf007adacee23ad0826689941330d2.png)T2】](http://hackaday.com/2014/06/17/a-robots-favourite-pen/all_pens/)

有些人对他们的笔非常挑剔。整个论坛都在讨论钢笔优越性的话题。但是，归结到个人的选择。有些人喜欢中性笔，而有些人喜欢圆珠笔。

[Jens]造了一个绘图机器人，它能画出像这样的图。它使用几个连接到两个步进电机的联动装置，对笔进行精细控制。随着机器人的工作，[Jens]开始寻找用于机器人绘图的最佳笔。

在机器上测试了七支笔，每支都画出相同的图案。[Jens]发现凝胶笔和滚珠笔在机器人上效果最好，并开始检查每种笔的性能。

这些笔的表现各不相同，但有两个获胜者被选中用在机器上。Pentel Energel Deluxe RTX 和 Pilot G-2 07 击败了竞争对手，因为它们在高速行驶时保持了良好的线条。

如果你想制造一个绘图机器人，[Jens]的研究应该可以帮助你为你的应用选择最好的笔。休息过后，一段机器人行动的视频将为您带来灵感。
