# 点焊机；不要买，要造

> 原文：<https://hackaday.com/2014/07/21/spot-welder-dont-buy-it-build-it/>

点焊机在为你的项目制作金属板外壳时非常方便。问题是，商用的相当贵…好消息是，它们实际上很容易制作！这是【卡伊奥·保罗西】第一次提交一天的工作，这是他和他父亲刚刚完成的一个周末项目。

点焊机的工作原理是在焊接材料的两个电极之间散发大量热量。它利用变压器将电源电压转换为非常低的电压，但高电流能源。这种焊机最酷的一点是，它非常安全地握住电极，因为电压很低，你不会触电。通过在一个小的表面区域流过大电流(通常大于 1000A @ ~1-2V ),可以将大多数材料加热到足以将它们焊接在一起。

它们可以使用微波炉的变压器、一些重型焊丝(通常为 2/0 或更粗)和一些其他零碎物品如木材、电极，也许还有一些螺母和螺栓来制作。在最基本的水平，你基本上是重新包装变压器的次级线圈，以改变比率，以产生一个低电压，高电流的变压器。
