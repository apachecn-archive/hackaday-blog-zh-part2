# 用模拟示波器触发示波器

> 原文：<https://hackaday.com/2014/12/13/tripping-on-oscilloshrooms-with-an-analog-scope/>

这可能是一个老把戏，但看到示波器这样的功能性工具被用于这样一个不相关的目的还是很酷的。[Jerobeam Fenderson]制作了一个视频，解释如何将立体声音频输入到一个旧的数字示波器中，以创建所有的东西，跳舞的蘑菇…因为为什么不呢？

在这种情况下，[Jerobeam]使用 Tektronix D11 5103N 设置在 X Y 模式，并从他的 RME Fireface UC 音频接口连接左右声道。一个通道对应 X，另一个对应 y。从这里，他在软件的帮助下离散地控制波形，如 [Pure Data](http://puredata.info) (Pd)和 [Max](https://cycling74.com/products/max/) (不是免费的，但更强大)，这些软件是可视化编程环境，使音乐家和艺术家能够在不编写代码行的情况下创建软件。他的视频解释了如何用正弦波做一个圆，然后用远远超出我们理解的数学把它打得落花流水。结果非常迷人，让我们想亲自尝试一下。幸运的是，如果你对正弦波的声音感兴趣……[Jerobeam]在他的博客上有更多关于如何[做一些你自己的](http://www.jerobeamfenderson.net/post/101351329308/how-it-works)实验的信息，不管你的硬件是模拟的还是数字的。
