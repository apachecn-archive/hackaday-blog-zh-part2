# 老式打火机中使用铂催化剂

> 原文：<https://hackaday.com/2014/11/01/the-platinum-catalyst-use-in-a-vintage-lighter/>

[本·克拉斯诺]有一种无与伦比的诀窍，为他的实验选择最有趣的概念。我们确信这是基础知识和史诗般的好奇心的结合。这一次，他展示了一个老式点烟器，它的怪癖是不需要“敲打”就能产生火焰。这是一种催化打火机，[用铂点燃甲醇蒸汽](http://benkrasnow.blogspot.com/2014/10/catalytic-lighter.html)。

下面视频中展示的概念是[铂的催化剂特性](http://en.wikipedia.org/wiki/Platinum#Catalyst)与某些类型的可燃气体。上图显示的是打火机的盖子，盖子上有一个保护罩，包裹着一大块铂黑粉末。它被一根铂丝悬挂着，当氢气经过时，反应会使铂黑和铂丝发出炽热的光。

这个简单快速的实验填补了我们自身的知识空白。我们已经熟悉了催化转换器在汽车中的作用；在未燃烧的碳氢化合物离开车辆的排气系统之前消耗掉它们。我们也知道这些设备是小偷寻找里面的铂(和其他金属，如钯和铑)的目标。现在我们确切地知道了催化转换器是如何工作的，以及铂在这个过程中所扮演的重要角色。这都要归功于[本]对这种打火机工作原理的演示。

现在，如果你戴着一枚白金结婚戒指，而你的手经过一股氢气流，你会被灼伤吗？
