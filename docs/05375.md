# 通量 1440:一个非常不切实际但令人敬畏的时钟

> 原文：<https://hackaday.com/2014/02/22/flux-1440-a-highly-impractical-but-awesome-clock/>

我们的一个线人刚刚给我们发来了这个伟大的项目——这是一个我们以前从未见过的独特风格的时钟。它是由 Felix Vorreiter 完成的，我们认为是研究生项目的一部分。这是[通量 1440](http://www.behance.net/gallery/FLUX-1440/2420150) ( [译](http://translate.google.com/translate?hl=en&sl=auto&tl=en&prev=_dd&u=http%3A%2F%2Fwww.behance.net%2Fgallery%2FFLUX-1440%2F2420150) )。

它使用 1200 米长的标记绳索，绳索被送入钟内，并被系在各种滑轮和齿轮之间。每秒钟，绳子移动 1.3 厘米。每隔 57 秒，时间可以通过绳股读取——但只有 3 秒。在那之后，一切都“回到河里”，比喻混乱。

其背后的解释是德语，但我们试图拼凑出一个关于其背后含义的一般性声明。当然，如果我们的德国读者能提供更好的翻译，我们会很高兴！

> FLUX 1440 将时间显示为空间维度，并使用一根长的分段绳来计算一天的长度。每一分钟的长度都是通过身体感觉到的，因为观看者必须等待形状的变化，直到当前时间从混乱的标记中显露出来。
