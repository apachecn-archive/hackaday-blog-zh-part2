# Alert Tube 监控您数字生活的方方面面

> 原文:[https://hack aday . com/2013/05/25/alert-tube-monitors-all-aspects-of-your-digital-life/](https://hackaday.com/2013/05/25/alert-tube-monitors-all-aspects-of-your-digital-life/)

![alert-tube](../Images/3d624233d69f8ed0581f614cbc27c216.png)

这个未来派的设备可以让你知道所有你的社交网络上发生的事情等等。因为它的功能和形状，迈克·沃森称这个设备为警示管。硬件主要依赖于一个 Raspberry Pi 板，它似乎是为这种用途量身定制的。信息收集方面展示了一个名为[的新兴服务的力量。](https://ifttt.com)

我们听说过 IFTTT，只是因为[克里斯·甘梅尔]和[达夫·琼斯]在*Amp Hour*的一集里报道过它。[戴夫]认为它几乎没有实际用途而不予考虑。但是这个项目展示了如何利用它从互联网上快速获取你想要的数据。可以把它想象成许多站点的 API 集合，比如 Twitter、脸书以及本地天气预报等等。这个项目设置 IFTTT 来监控你的账户，用喜欢的颜色，声音，甚至文本到语音来提醒你。

项目解释有几页长，但是你可以通过观看演示视频快速浏览一下。
