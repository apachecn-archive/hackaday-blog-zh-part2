# 走进盒子里

> 原文:[https://hackaday.com/2013/10/03/step-into-the-box/](https://hackaday.com/2013/10/03/step-into-the-box/)

![thebox](../Images/6022a3625f02f30aa5f608010d7880ab.png)

以三个工业机器人、两个 4' x 8 '的画布和几个强大的视频投影仪为例。根据谁在做机器人编程，你可能会得到很多碎玻璃和碎片，或者你可能会得到一个盒子。creators project 发布的最新视频，这个盒子以工业机器人和投影映射为特色。我们最近在同一频道播出了[解除武装](http://hackaday.com/2013/09/16/disarm-turning-guns-into-a-mechanised-musical-orchestra/)。

这个盒子是将多种现有技术整合在一起并带来惊人结果的例子之一。我们在看视频的时候不禁会想到 CastAR 等[系统的可能性。机器人移动两张大画布，同时投影仪在上面显示一系列 3D 图像。第三个机器人移动摄像机。](http://hackaday.com/2013/09/24/castar-comes-to-maker-faire-ny-2013/)

在幕后视频中，创作者透露机器人使用 Maya 插件编程。该插件允许他们将机器人的动作与动画同步。整个视频是一个复杂的编排舞蹈——甚至演员的位置都是预先编程到 Maya 中的。

演员和机器人描述了魔法的几个原理:变形、悬浮、交叉、传送和逃脱。所有这些都符合著名的[阿瑟·C·克拉克]的名言:“任何足够先进的技术都与魔法无异”

虽然我们喜欢这个视频，但我们不禁想到了一些变化。如果我们没有这种可重复的机器人，或者根本没有机器人，会怎么样？早在 2007 年初，【约翰尼·李中】就在展示他被 Wiimote 现场追踪的[可折叠显示器。使用[Johnny 的]系统，即使是人类也能足够准确地处理画布。](http://www.youtube.com/watch?v=nhSR_6-Y5Kg)
