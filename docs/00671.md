# 震动机器人利用惯性运动

> 原文:[https://hack aday . com/2012/06/07/shaker bot-leverages-inertial-for-locomotive/](https://hackaday.com/2012/06/07/shakerbot-leverages-inertial-for-locomotion/)

![](../Images/1a1764d2d84eb662a1b88a65ce9f5d6c.png "shakerbot-uses-inertia-for-locomotion")

这个机器人使用了蛇形板的概念来四处走动。[Sevag Babikian]建立了这个项目，[不厌其烦地解释了它是如何工作的](http://www.depotbassam.com/2012/06/shakerbot-snakeboard-inspired-robot.html)。

我们建议浏览一下他的解释，但正是休息后的视频让我们明白了所有的事情。中心有一个大飞轮。这不是连续旋转，而是在任一方向上摆动几英寸。这种惯性变化会移动整个框架的重心，从而导致运动。通过在适当的时间摆动前后轴，这头野兽可以惊人地直线前进。

它只用三个马达就能做到这一点。[Sevag]推测，随着进一步的测试和完善，这可能是一种非常有效的移动机器人的方式。我们对此有所怀疑，但我们被这种波动以及它们推动事物前进的方式所迷惑。
