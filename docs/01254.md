# ReactionWare 3D 打印药物

格拉斯哥大学发布了一篇化学研究论文，涵盖了打印药物化合物的应用过程。

是的，这是正确的，实际上是打印药物。使用各种化学原料，他们看到了在家里制造药物的未来。使用标准的 3D 打印技术，有可能以这样一种方式组装预填充的“容器”，即发生所需的化学反应来生产所需的药物。这就像你家里有一个小型药物制造工厂。这可能会产生深远的影响。

需要有一个锁定软件等或某些化学限制，以防止这项技术的滥用。提出这篇论文原理的[李·克罗宁]教授称这一过程为“反应件”

克罗宁教授发现，使用这种制造工艺，即使是最复杂的容器也可以在几个小时内相对快速地制造出来。
