# Arduino 动力蒸汽朋克蒸汽表

> 原文:[https://hack aday . com/2013/11/30/arduino-powered-steam punk-steam-gauge/](https://hackaday.com/2013/11/30/arduino-powered-steampunk-steam-gauge/)

![steampunkGauge](../Images/d2c762bc66a956e93bd54afa53ec4c9b.png)

[墨菲的律师]在他厨房的墙上有一些空白的空间，所以他决定用一个呼呼作响的蒸汽朋克小发明来填充它:[一个 Arduino 驱动的蒸汽表](http://www.instructables.com/id/STEAMPUNK-STEAM-GAUGE-POWERED-BY-ARDUINO/)。

建造始于一个来自易贝的旧的 10 英寸阿什克罗夫特压力计，[墨菲的律师]对其进行了解剖，以确定其内部的状态。发现压力计的波登管完好无损，他开始着手构建一种不需要实际蒸汽就能产生运动的方法。解决方案是在管子和外壳之间安装一个连续旋转的伺服机构。伺服系统自身缺乏弯曲管子的力量，所以[墨菲的律师]用黄铜制作了一个简单的杠杆来帮助它前进。

电子设备包括一个 Arduino Uno 和一个附带的自制 PCB。Uno 的代码会生成随机运动来旋转伺服系统，三个内置的 led 会反映速度、暂停和运行时间的值。最后一次升级是以新表盘的形式出现的，它提供了一些更新的文本以及一个方形切口，让你可以看到以前模糊的齿轮在运转。看看下面的视频，然后看看另一个蒸汽朋克革新:[爱德华七世时代的笔记本电脑。](http://hackaday.com/2012/05/30/edwardian-laptop-from-a-steampunk-master/)
