# FPGAs 跟踪你的乒乓球比赛

> 原文：<https://hackaday.com/2015/05/15/fpgas-keep-track-of-your-ping-pong-game/>

现在是毕业时间，你知道这意味着什么！又一轮高级设计项目，做一些通常非常独特的事情。[Bruce Land]从康奈尔大学发来了一个很棒的例子，那里的学生正在进行一个项目，该项目使用[FPGA 和几个摄像机来记录乒乓球比赛](http://people.ece.cornell.edu/land/courses/ece5760/FinalProjects/s2015/ttt/ttt/ttt/index.html)的比分。

该系统的工作原理是处理一场乒乓球比赛的 NTSC 直播。球被涂上特定的颜色以帮助检测，处理视频的 FPGAs 可以跟踪球网的位置，球反弹的次数，以及球是否被球员击中。有了所有这些信息，系统可以跟踪游戏的分数，分数显示在桌子附近的监视器上。现在，玩家可以自由地专注于他们的游戏，而不必担心保持分数！

这是一个令人印象深刻的 FPGAs 和视频处理演示，其应用不仅仅局限于乒乓球。你会用它做什么？看到学生们在做什么总是很有趣；这些实验中的核心概念往往会在以后进入他们的职业生活。也许他们甚至会把这个项目带到下一个层次，建造一个真正的乒乓球机器人来配合他们的得分系统！
