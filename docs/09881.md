# hack let 71–水上项目

> 原文:[https://hack aday . com/2015/08/21/hack let-71-waterborne-projects/](https://hackaday.com/2015/08/21/hacklet-71-waterborne-projects/)

水:没有水，地球上的生命就不会存在。地球的 71%被水覆盖。只剩下 29%的土地可供我们人类居住——而且并不是所有的土地都适合居住。水是如此重要，以至于大多数人类居住区都是从靠近水的地方开始的。水可以喝，或者水可以运输货物。海洋、湖泊和河流中有这么多水，难怪黑客、制造商和工程师们会建造一些难以置信的项目，这些项目可以在水面上和水下工作。

[![airboat1](../Images/bb02a65c114e8e159f76b9274d73fc7c.png)](https://hackaday.io/project/6660) 我们先从【Ossum】和[螺旋桨驱动的遥控船](https://hackaday.io/project/6660)说起。通过互联网的挖掘，[Ossum]发现了一些由飞机引擎和螺旋桨驱动的船只的旧计划。被这种飞艇吸引，他开始建造自己的版本。[Ossum]以前从未建造过船，但他不会让这阻止他！他的设计使用前部安装的牵引螺旋桨来牵引船只，而不是汽艇通常使用的后部马达推进器配置。[Ossum]船体已经接近完成，他的木工技术真的很出色。他用 CAD 设计了一个拉环锁框架。一台计算机控制的机器完成了切割木头的快速工作。其余的过程，虽然，一直是老式的手工工具，和大量的耐心。

[![boat2](../Images/ce38f70b17a8f3bb021c6fe107aece3f.png)](https://hackaday.io/project/4848) 接下来是【乔丹·莱斯-斯普林格特】带着[自主帆船](https://hackaday.io/project/4848)。[Jordan]正在建造一艘 1.2 米长的帆船，能够在开阔的水面上自主航行。他的最终目标是参加横跨大西洋的比赛[微操挑战赛](http://www.microtransat.org/)。【乔丹】号使用的是 MaxiMOOP 船体设计，应该能够应对巨大的海浪。穿越海洋是一个困难的前景。破船浪和撕帆风竞相毁船。电子设备必须完全防水，因为即使是少量的盐水也会在 3 个月的海上航行中损坏电路板。[乔丹]有基本的船体完成，现在正在工作的方向舵。我们已经有几个月没有他的消息了，所以如果你看到[乔丹]，告诉他过来给我们一个关于自主帆船的最新进展！

[![sub](../Images/a9f2fa998206869a771ec97b325a88d3.png)](https://hackaday.io/project/2025)【cjh】创造了 [ArduSub](https://hackaday.io/project/2025) 作为他们 2014 年黑客日奖的参赛作品。ArduSub 是一种自主水下航行器。其想法是创建一个传感器平台，可以测量当地环境，包括 PH 值、溶解氧(DO)、氧化还原电位(ORP)、盐度和温度。这些测量值和 GPS 坐标对于海洋生物学家和其他海洋研究人员来说非常重要。ArduSub 的船体由普通的 PVC 管制成，易于使用，相当坚固。[cjh]使用 Atmel ATxmega128D3 作为他们的主处理器。每个传感器模块还包含一个自己的 ATxmega8E5。各种模块通过 RS-485 串行网络进行通信。数据通过高频波段业余无线电发射机发回陆地。

[![airboat2](../Images/5be48d661eb13441f58c94be5027b5d0.png)](https://hackaday.io/project/2359) 终于有了【CompuCat】带 [DIY 飞艇和控制器](https://hackaday.io/project/2359)。[CompuCat]使这艘飞艇的船体设计变得简单。他们用一块聚苯乙烯泡沫塑料作为基底，将泡沫雕刻成想要的形状。然后用液体塑料覆盖泡沫，给它一个坚固的外壳。这是玻璃纤维和碳纤维复合材料技术的一种更简单的替代方法。这艘船没有使用现成的遥控电子设备。一个以 9600 bps 速度运行的 XBee 从一个定制控制器向船发送命令。飞艇的处理器是一个 Atmel ATmega328p，安装在专为这个项目建造的板上。
