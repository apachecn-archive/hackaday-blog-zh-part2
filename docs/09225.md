# 疯狂的麦克斯启发喷火器尤克里里

> 原文:[https://hack aday . com/2015/06/07/mad-max-inspired-flamelisser-ukulele/](https://hackaday.com/2015/06/07/mad-max-inspired-flamethrower-ukulele/)

疯狂的麦克斯的新电影引起了很多关注，一些人称之为现代经典。电影中有一把喷火器吉他，这意味着是时候进行 cosplay 装备了。我们的老朋友[凯莱布]喜欢喷火和喷火，所以组装[一把由杜夫战士启发的喷火尤克里里对他来说很自然](http://makezine.com/projects/mad-max-doof-warrior-inspired-flamethrower-ukulele/)。

该 uke 的燃料是一罐丁烷，由填缝枪驱动。这个设置实际上非常聪明；通过移除填缝枪上的锁片，当枪的扳机被挤压时，丁烷被释放，但当扳机被释放时，丁烷停止释放。点火器是一个简单的烤架，用来点燃煤气。

[Caleb]以他的喷火作品而闻名。他的真人大小的喷火食人鱼工厂使用类似的装置来发射火焰。
