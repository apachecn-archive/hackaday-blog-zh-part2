# 本周失败:霜杯计划只是有点冷

> 原文：<https://hackaday.com/2014/10/23/fail-of-the-week-project-frosty-mug-is-merely-chilly/>

像我们许多人一样，[C]在编码时喜欢喝冰镇的提神苏打水。在不加冰的情况下无限期保持苏打水冰冷的强烈愿望的驱使下，[C]开始了[Frosty Mug](http://burnt-traces.com/?p=222)项目。

[C]的既定目标是将一瓶 20 盎司的塑料瓶汽水无限期地保存在约 35 华氏度的环境中。他开始用一个热电装置来冷却一个铝盘，就像一个冰冷的杯垫。第一部和第二部让他意识到处理产生的热量是一个大问题:它变得如此之热，以至于变形了 PLA 框架。[C]还意识到仅底部冷却无法完成工作。

这个项目现在在它的第三个建筑中，如上图所示。如你所见，它比过山车更像 T2。那个 3d 打印的枪套衬有铝板。另一个扁平件覆盖开口并连接到冷却元件。一个结实的 CPU 散热器尽其所能，几个 U 形支架将它固定在一起。

[C]用一瓶冷冻到华氏 38 度的减肥防晒霜对其进行了测试。在大约 70 华氏度的环境温度下放置 30 分钟后，苏打水的温度测量值为 45 华氏度。[C]哀叹没有使用对照瓶进行比较，并报告电源变得相当热。C 不会那么容易就放弃的。你对第四部有什么想法吗？
