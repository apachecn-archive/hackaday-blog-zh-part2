# Genezap 使用体罚提高你的视频游戏技能

> 原文：<https://hackaday.com/2012/08/02/genezap-improves-your-video-game-skills-using-corporal-punishment/>

![genezap](img/af4ce1c887bd56a1a6e835a9541c9396.png "genezap")

好像玩电子游戏时被人揍还不够烦人似的，[furtek]认为帮助他提高技能的最好方法是每次他在屏幕上的角色死去时都给他造成身体上的痛苦。

当他在网上寻找一些东西来打破一天的沉闷时，他看到了一个视频，视频中有人决定试一试狗电击项圈，只是为了寻求刺激。这激发了 furrtek 的想象力，他开始认为用同样的设备让视频游戏中的死亡变得更加不愉快会很酷。

在网上订购了一套项圈后，他把它们拆开，看看它们是如何工作的，并测量它们能产生多大的震动。[furtek]然后用一对 ATtiny 25s 改装了两个 Genesis 控制器，让他向项圈发送火灾信号。不幸的是，股票起源游戏不允许你向控制器发送信号，所以[furtek]不得不花一些时间从图像中侵入，以便在玩家受伤或失去生命时触发事件。

我们认为这个项目非常巧妙，如果你不介意摆弄你的旧控制器，你也可以有一个无情的教练绑在你的脖子上。对于那些稍微不喜欢疼痛的人，你可以看着[furtek]和他的朋友[Dyak]遭受糟糕游戏的后果来娱乐。
