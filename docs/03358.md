# 五金店机器人手

> 原文:[https://hackaday.com/2013/06/03/hardware-store-robot-hand/](https://hackaday.com/2013/06/03/hardware-store-robot-hand/)

![hardware-store-robot-hand](../Images/807baa450b5d33f0bd9c6d60e4ccf2f5.png)

这里有一个机器人手[可以使用大部分五金店物品](http://www.youtube.com/watch?v=e4Rtgo6Vmq8)来制造。它没有最强的抓地力，但它有逼真的动作。演示视频显示它捡起像金属螺母一样的小物体。

上图显示了手的无名指和小指开始弯曲。这些由安装在手掌区域的伺服电机控制。每个手指的骨骼结构从自行车链条的链环开始。首先通过移除摩擦配合杆来分离链节。每个杆都被一个螺钉和一个螺母取代，这也允许弹簧(打开手指)固定在每个“关节”上。

[Aaron Thomen]在手完成后并没有停止设计过程。他继续建造了一个控制器，让你用手指拉动一些环来影响运动。这种运动由一组电位计测量，并转化为电信号来定位手的伺服电机。下面嵌入了演示以及两个操作视频。
