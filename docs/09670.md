# Arduino 赋予弹球桌新的生命

> 原文：<https://hackaday.com/2015/07/27/pinball-table-gets-new-lease-of-life-with-arduino/>

忘记所有这些视频游戏的废话:弹球是真正的游戏之王。毕竟，它涉及到大块金属高速飞行。[retronics]同意:他用 Arduino 复活了一张旧的布赖尔伍德阿斯彭弹球桌。

当他买下这张桌子时，他发现电子元件已经被烧坏了:板上的许多分立元件都被烧坏了。因此，他没有更换单个部件，而是将桌子掏空，用 Arduino Mega 取代了逻辑板，Arduino Mega 驱动鳍状肢、显示器和编钟，使弹球成为令人愉快的体验。幸运的是，[这个家用弹球桌被很好地记录了下来](http://www.ipdb.org/machine.cgi?id=3660)，所以他能够很容易地找出如何重新布线剩余的部分，以及如何在软件中重建得分系统。

他在 refurb 上的总花费约为 300 美元，而 junker 一开始只花了 50 美元。现在花 350 美元，你可能就能找到一张能用的弹球桌。但这并不是真正的重点:他这样做是为了体验使用像脚蹼和倾斜开关这样的机电组件的工作。我们对之前用备用零件制作了一个 Android 示波器的家伙的期望不会少于此。
