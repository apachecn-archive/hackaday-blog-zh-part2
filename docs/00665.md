# 用你的声音控制鼠标

> 原文:[https://hack aday . com/2012/06/07/用你的声音控制鼠标/](https://hackaday.com/2012/06/07/controlling-a-mouse-with-your-voice/)

![](../Images/a358eb085ac6de378774f8a3a0a4e396.png "mouse")

完全有可能在没有鼠标或触控板的帮助下使用电脑。Shift 和箭头键会让你走得很远，但那完全太费力了。[Stephen]想出了一个用你的声音控制鼠标的非常好的方法，这个项目肯定会很快进入那些行动不便的人的桌面

声控鼠标与 OS X 内置的语音识别、一点 AppleScript 和一点 Python 结合使用。当用户说“显示网格”时，一个编号为 1 到 100 的 10×10 的网格就会显示在屏幕上。通过说出“35”，光标移动到网格中的第 35 个单元格。从那里，可以通过说出基本方向来控制鼠标，例如南方和西北方。

[Stephen]在广告之后放了一个他的语音鼠标项目的非常聪明的演示。尽管他的 mac 在识别他的一些语音命令方面确实有一点困难，但这比试图只用 shift 和箭头键浏览网页要早得多。
