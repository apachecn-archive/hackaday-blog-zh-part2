# 将闪电放入丙烯酸树脂中

> 原文：<https://hackaday.com/2015/01/08/putting-lightning-in-acrylic/>

i3 底特律黑客空间的一些人有了一个机会，可以让他们用丙烯酸树脂捕捉闪电。在一家塑料管制造商、一些铅板和一堆 1/2 英寸厚的丙烯酸树脂的帮助下，他们制作了一些李奇腾伯格人像。

Lichtenberg 图形是世界各地镇纸中的 3D 电子树。它们是通过绝缘体放电产生的，闪电是人们见过的最令人印象深刻的李奇登伯格图形。这些图形可以在较小的物体上形成，唯一需要的是大量的电子被注入绝缘体。

这是在[水星塑料公司的新束设施](http://www.mercuryplastics.com/neo-beam/neo-beam-capabilities.html)中发现的，这是一台 5MeV 电子加速器，通常用于为 PEX 管道中的分子交联提供能量，以增强耐化学性。有一天，i3Detroit 的一些人接管了生产线，将大约 1000 个丙烯酸树脂零件通过机器，制作出李奇腾伯格图形。

当丙烯酸树脂通过电子加速器时，它们内部会充满电荷。快速的机械震动释放出丙烯酸，在塑料中创造出美丽的树状图形。在画廊有很多完成人物[的照片，但是如果你想看一些*真正*酷的东西，一个铅屏蔽的 GoPro 也通过电子加速器运行。你可以看看下面的视频。](https://plus.google.com/u/1/photos/114012295505053901631/albums/6084341596412030449)
