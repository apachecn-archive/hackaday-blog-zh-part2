# USB Ableton 脚控制器重用旧键盘

> 原文:[https://hack aday . com/2013/09/14/USB-ableton-foot-controller-reuses-old-keyboard/](https://hackaday.com/2013/09/14/usb-ableton-foot-controller-reuses-old-keyboard/)

![footLooper2](../Images/233cee0d7af56033459936c67f4ada32.png)

不，Hackaday 还没有开始做鞋子广告，这是 Ableton Live 的[mat lek]的[脚控器](http://www.instructables.com/id/USB-foot-controllerlooper/)。

马特里克弹吉他，需要一种简单的方法来控制 Ableton Live，他将它用作回路。Ableton 通常期望键盘输入，所以他确实给了它。

一个旧的戴尔键盘被拆成了控制器板。这暴露了键盘用来扫描按键矩阵的引线。从那里，它只是简单地试错连接不同的引脚在一起，并看到哪些关键是打印在电脑屏幕上(一个文本编辑器很好地工作)。控制活套只需要 8 个字符，所以[Matlek]选择了数字 1-8。

由于一些电线将共享引脚，在按钮和键盘控制器之间有一小块条形板就很方便了。[Maltek]在他的迷你按键矩阵中使用了基本的瞬时按钮，尽管我们认为这个盒子看起来足够坚固，可以支持一些更大的踏脚转盘风格的按钮。

所有东西都放在一个坚固的鞋盒里，鞋盒还可以防止暴露在外的键盘 PCB 短路。

该项目的唯一主要缺点是盒子足够轻，当录音或触发循环时，它很容易在地板上滑动。添加一些重的物品(或者我们敢说，一些鞋子)会解决这个问题。盒子底部的自粘橡胶支脚也会有所帮助。
