# Hackaday 奖参赛作品:用于替代时基的时钟

> 原文：<https://hackaday.com/2015/07/10/hackaday-prize-entry-a-clock-for-alternate-timebases/>

安赫-莫波克的贵族维蒂纳里勋爵的等候室里有一座奇怪的钟。虽然这个时钟总体上保持准确的时间，但滴答和滴答不同步，偶尔会完全错过一个滴答。净效应是把一个人的大脑变成一种粥。

是的，[一个 Vetinari 时钟](https://hackaday.io/project/5880-crazy-clock)已经进入了 Hackaday 奖。这不是一个在很长时间跨度内随机但精确的时钟；这是你在任何工艺品商店都能找到的普通时钟机芯的完全替代品。

除了 Vetinari 时钟之外，[Nick Sayer]的疯狂时钟还可以编程为恒星时钟(每天快 3 米 56 秒)、火星时钟(每天慢 39 米 36 秒)和潮汐时钟(每天慢 50 米 28 秒)，以及一些每天仍有 86400 次滴答的“新奇”模式，从微妙到“小丑车”级别的疯狂。

[Nick]正在争取 Hackaday 奖的“最佳产品”类别，为此，他正在设计一种板，直接替代 Quartex Q80 时钟机芯中的板。有了这种新的电路板，[Nick]可以在几分钟内更换机芯中的电子元件。围绕 ATtiny45 建造意味着它是无限可攻击的。带有这种机芯的时钟将是一个伟大的产品，尽管从下面的视频来看，我们不想整天都在它身边。
