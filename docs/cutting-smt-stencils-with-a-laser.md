# 用激光切割 SMT 模板

> 原文：<https://hackaday.com/2015/04/03/cutting-smt-stencils-with-a-laser/>

制作自己的 PCB 原型？不知如何应用你的 SMT 电子产品？嗯——你有激光吗？[Felix]向我们展示了使用激光切割机轻松雕刻 SMT 模板的明确方法。

这里真正的技巧是雕刻，而不是切割。通常情况下，如果你用足够的力量直接切穿塑料，你会得到边缘熔化和燃烧-这对于 SMT 模板来说不会很好。所以[费利克斯]发现以大约 20-30W @ 400-450 毫米/秒的速度雕刻。他使用聚酯薄膜作为材料。

结果非常棒——但是如果你没有激光，他也有一个很好的 DIY 金属 SMT 模板的教程，通过化学蚀刻苏打易拉罐金属！
