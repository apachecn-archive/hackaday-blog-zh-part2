# 气象气球的有效载荷(几乎)会自动回到你身边

> 原文:[https://hack aday . com/2012/10/02/weather-balloon-payload-that-almost-guides-self-back-to-you/](https://hackaday.com/2012/10/02/weather-balloon-payload-that-almost-guides-itself-back-to-you/)

![](../Images/1cc5ac1d0229da693d35c1847f54e2ea.png "weather-balloon-payload")

将昂贵的电子产品送入临近空间的最大问题是试图回收它们。[Lhiggs]开始着手解决这个问题，他的大四项目是机械工程学位。他认为从 100，000 英尺高空落下的有效载荷应该能够滑行回到一些预定的坐标。在这里你可以看到其中一个测试，有效载荷使用翼伞引导其下降。

翼伞只需在两根支撑绳之间转移重量就可以控制方向。在这种情况下，[Lhiggs]设计的有效载荷悬挂在一对伺服电机驱动的臂上。由于有效载荷已经携带了高度和位置硬件(如 GPS，电子罗盘和高度计)，所以在与气象气球分离之前，只需等待目标高度，然后使用伺服系统导航到着陆区。

不幸的是，该项目从未完全完成。但是你可以看到他走了很远。广告之后嵌入了测试镜头，显示该设备从飞机上掉落。
