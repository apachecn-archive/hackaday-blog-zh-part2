# 飞行意大利面怪物展示

> 原文：<https://hackaday.com/2013/12/13/flying-spaghetti-monster-display/>

[Anthony Liekens]，我们最喜欢的来自比利时的黑客之一，最近完成了这个大型(和真棒！)[飞行意大利面怪物 LED 显示屏！](http://anthony.liekens.net/index.php/Misc/FSMDisplay)

十二月有这么多不同的节日，[安东尼]决定让他的家人庆祝一个稍微不那么传统的神。机身是一块巨大的 4 英尺乘 8 英尺的木板，我们认为是[安东尼]手工切割的，共有 300 个 RGB LEDs，由一个 Arduino 驱动。鸡肉丝网为 FSM 的嘴和眼睛中的发光二极管提供支撑。[Anthony]在他自己的后院黑客空间建造了所有的东西，叫做[开放式车库](http://hackaday.com/2013/10/22/hackerspacing-in-europe-garage-space-in-antwerp/)，这是一个非常棒的社区黑客空间(我们应该知道——我们在[欧洲黑客空间之旅期间住在他的地方！](http://hackaday.com/2013/11/06/hackerspacing-in-europe-conclusion/))

[Anthony]在他的个人[YouTube 频道](http://www.youtube.com/user/aliekens/videos)上有一堆展示视频，但在休息后留下来看一个快速样本，它展示了在他家前窗飞行的意大利面条怪物。
