# 使用封闭循环水系统的个人冷却

> 原文:[https://hack aday . com/2012/11/26/personal-cooling-using-a-closed-loop-water-system/](https://hackaday.com/2012/11/26/personal-cooling-using-a-closed-loop-water-system/)

![](../Images/74ba7f3c57872985cea140d176d32439.png "water-based-personal-cooling-system")

这不是一个结肠造口袋，这是第一个原型的[斯蒂芬的] [刮建闭环个人冷却系统](http://www.philippinestuffs.com/personal-water-body-cooler/)。他一定是住在一个热得让人不舒服的公寓里，因为这是我们几个星期以来从他那里看到的第二个冷却系统。[以前提供的是蒸发系统](http://hackaday.com/2012/11/23/quick-and-easy-personal-evaporative-cooler/)。这一次，他泵入冷水来缓解压力。

左边的图像显示了该系统的第一次迭代，它从一个大水壶中抽取冷水，通过一个他戴在脖子上的塑料管环。为了改进设计，他制作了右边的版本。作为一个水库，他抓住了一个防水的 ID 容器，旨在保持你的贵重物品在游泳池或海洋中干燥。里面有一个泵，他用 5V 电池供电。它通过由一根塑料管组成的颈带循环水。

这将在一段时间内有效，但随着冷水吸收你身体的热量，效果将会消失。我们认为他需要在下一次迭代中给储液器增加一个珀耳帖冷却器。这可能有助于改善回路，增加其接触皮肤时的传热能力。

休息之后，会有最新版本的演示。
