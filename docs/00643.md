# 更新:对基于光学传感器的钢琴进行了许多改进

> 原文:[https://hack aday . com/2012/06/05/update-many-improvements-to-optical-sensor-based-piano/](https://hackaday.com/2012/06/05/update-many-improvements-to-optical-sensor-based-piano/)

![](../Images/e101f35503a1f0572977a89d240da00e.png "optical-sensor-piano-update")

[Sebastian]写信告诉我们他几年前开始的光学传感器项目的最新情况。你可以在这里找到[他最近的更新，但是休息之后有四个不同的帖子链接记录了他的进展的各个部分。](http://sebion.wordpress.com/2012/05/31/motherboard-the-next-generation/)

你可能不记得最初的项目了，但是他想给电子键盘的按键增加分辨率和灵敏度。随着传感器的建立，他开始尝试使用力的数据来影响声音的其他部分。他在一月份的帖子显示，随着琴键从玩家那里获得更多的力量，这种情况会改变音调。

三月份，他在一架旧钢琴上安装了传感器阵列。他上传的视频是他弹钢琴的地方，但我们听到的是传感器输入产生的声音。休息之后我们已经嵌入了它。

上周，他发表了两篇博文。它们包括传感器板的重新设计，以及他为降低制造成本所做的面板工作。基本单元被重新设计为使用 AT90USB 微控制器，该微控制器整合了以前版本中使用的独立芯片。
