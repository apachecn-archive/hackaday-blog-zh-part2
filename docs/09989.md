# Hackaday 奖半决赛:弯曲太阳能蓝牙标签

> 原文：<https://hackaday.com/2015/09/01/hackaday-prize-semifinalist-bendy-solar-bluetooth-tags/>

去年一月，[德莱尔吉娜]的妹妹找不到她的猫。这家人在附近搜寻了两个小时，直到猫从床底下出来，再次证明猫拥有人类，而不是人类拥有猫。这个问题的解决方案将以技术的形式出现，具体来说就是[【DrYerzinia】的参赛作品 Hackaday Prize](https://hackaday.io/project/6919-sol-ble-tracker) ，一款太阳能蓝牙追踪设备。是的，你可以在亚马逊或易贝上购买一个 BLE 追踪器，但这个版本有一个方便的包装:它由一个柔性电路板制成，可以适合任何地方，包括猫的项圈。

[DrYerzina]的蓝牙跟踪器是围绕蓝牙 LE 模块构建的，添加了一些无源器件、led 和其他部件，粘合并焊接到双面柔性 PCB 上。为此，他增加了一个柔性太阳能电池和一个柔性脂肪电池。所有这些都被塞在一个用柔性细丝 3D 打印的外壳里。

虽然 Hackaday 奖充满了可穿戴设备，但[DrYerzina]的项目处于 hombrew 可穿戴技术的最前沿。在这个奖项中，我们从未见过有人致力于制造一种可以弯曲的装置。最棒的是，他实际上在建造一个有用的设备；每天只有 15 分钟的阳光(这种情况很可能发生在熟睡的猫身上)，这种蓝牙标签可以工作数周。
