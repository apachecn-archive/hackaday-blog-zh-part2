# 当你能种植家具时，为什么要建造家具呢？

> 原文：<https://hackaday.com/2015/07/02/why-build-furniture-when-you-can-grow-it/>

[Gavin Munro]正在颠覆家具制造的标准模式。不是砍伐树木并把它们切成木板——或者更糟，把它们变成锯屑用于刨花板——【加文】是[字面上种植家具](http://fullgrown.co.uk/)。

柔软的年轻柳树树苗修剪和训练使用电线和塑料模板工作。这些树被鼓励向正确的方向生长，形成腿、胳膊、座位和背部，最终各个部分被嫁接在一起，继续生长成一个坚实的整体。当椅子成熟时，树叶被移除，椅子从地面上被切割下来，稍加调味和整理，你就得到一把独特而实用的椅子。更重要的是，由于它是一块坚固的木头，随着时间的推移，没有接头会松动。

你不得不佩服这些椅子的奉献精神。现在的作物大约有九年了，离收获还有几年。从这样一个项目的组织中可以学到很多东西——规划一条生产线，让第一批成品从陈列室出来已经有十年或更长时间了，这可不是一件容易的事。看来[加文]也考虑到了这一点，开始生产一系列很快就会盈利的灯具。休息后的视频不仅展示了[Gavin 的]椅子和灯具，还展示了他的第一批桌子。
