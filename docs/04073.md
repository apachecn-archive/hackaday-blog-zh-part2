# 令人不安的鬼屋的机器人触角

> 原文:[https://hack aday . com/2013/09/16/机器人触须扰鬼屋/](https://hackaday.com/2013/09/16/robotic-tentacles-for-a-disturbing-haunted-house/)

![testicles](../Images/d89f7de9d63ea9b02150e51d13801f4b.png)

[ivorjawa]今年万圣节要上演一出我们真的不想去的鬼屋。他的机器人触手已经令人毛骨悚然，我们认为一旦它被织物和泡沫橡胶覆盖，只会更加可怕。

由于四根钢缆穿过这个奇怪的盖革式装置，每个触手都可以在两个轴上移动。在触手的底部有两个步进电机驱动的气缸，可以收紧一根电缆的松弛部分，并拉出另一根电缆。其中两个控制箱由步进电机和 Arduino 电机护罩驱动，允许触手向任何方向伸出和抓取。你可以在[【ivor Java】的 flickr](http://www.flickr.com/photos/ivorjawa/sets/72157635075054974/) 上查看构建的机制

在一个半相关的注意事项上，即使我们离万圣节还有一个多月，我们现在应该有更多的万圣节构建在我们的提示行中。如果你正在做一个，不要害怕[在](http://hackaday.com/contact-hack-a-day/)中发送它，即使你只是在炫耀一个正在进行的工作。
