# 装满曲子的手提箱

> 原文:[https://hackaday.com/2012/11/22/a-suitcase-full-of-tunes/](https://hackaday.com/2012/11/22/a-suitcase-full-of-tunes/)

![](../Images/a22cfb758dcc187482aeed3c55c143b1.png "suitcase-full-of-tunes")

通过构建您自己的 boomcase，将聚会带在身边。这是一个内置在行李箱中的放大器和一组扬声器。它使用一个音频插孔来连接你最喜欢的音乐播放器，加上一点额外的保护——比如这些扬声器的格栅——它仍然可以轻轻地用来运输你的衣柜。

一个 20 世纪 60 年代的手提箱被破坏了。[Jay]肯定手头已经有了，因为加上一些用过的零件，他声称总共只花了 50 美元。在一块纸板上尝试了几个不同的扬声器方向后，他用蓝色油漆工胶带覆盖了外壳的外部，并开始切割孔。他选择的放大器有一个漂亮的面板，恰好正好放在表壳的顶部。目前，他用一个 10，000 mAh(即:10 Ah)的便携式设备充电电池为它供电。但正如你在休息后的演示中听到的，为系统提供足够的电力似乎没有问题。
