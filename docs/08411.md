# 展示你的西红柿的最好方式

> 原文：<https://hackaday.com/2015/02/26/the-best-way-to-sport-your-tomatoes/>

谁会在跑的时候吃西红柿？没有人……而日本番茄酱公司，戈薇想要改变这一点。

多年来，都乐一直赞助日本马拉松比赛，为跑步者提供即食香蕉，以快速补充营养。他们今年广告活动的一部分是一种可穿戴的可食用香蕉，跑步者可以不用手就能吃。出于幽默和超越荒谬程度的愿望，戈薇与知名的日本团体 Maywa Denki 合作，创造了 [Tomatan](http://www.otamatone.com/tomatan/) ，这是一个微笑的装置，安装在跑步者的肩膀上，将西红柿从漏斗中直接送到他们的脸上……提醒人们，尽管日本可以吃可穿戴的香蕉，但他们也可以很容易地吃机械分配的西红柿。

这有什么了不起的？当我们谈论日本的愚蠢和商业主义时，前面提到的 Maywa Denki 似乎总是像一个挥舞着精灵的无厘头一样出现。如果你没听过这个名字，[这个公司其实是一个艺人团体](http://www.otamatone.com/maywa-denki/)的身份。他们以发明滑稽可笑的机械小发明而闻名，然后作为 Maywa Denki 品牌的严肃产品进行营销。这个 16 磅重的背负式番茄喂食器旨在帮助提高跑步者的耐力，只是他们众多“无意义机器”中的一个例子。

当我们把自己创造的东西推向市场时，它们通常以某种产品的形式呈现在世人面前。Maywa Denki 将这种常见的产品概念转变为围绕他们的艺术作品创造一种身份。因为为什么不解决一个想象中的问题，而这个问题实际上并不需要解决方案呢？
