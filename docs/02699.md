# Raspberry Pi 相机板到来

> 原文:[https://hack aday . com/2013/02/06/raspberry-pi-camera-board-incoming/](https://hackaday.com/2013/02/06/raspberry-pi-camera-board-incoming/)

![camera](../Images/af5fb655a0721edf944be8f86de6e4f2.png)

你的 Raspberry Pi 有用于相机和显示器的板载连接器，但直到现在还没有硬件半神接受将图像传感器或 LCD 连接到这些端口之一的挑战。似乎每个人都在等待为这些端口设计的官方 Raspi 硬件。等待即将结束，树莓基金会希望在未来几周发布一款相机板。

相机模块基于一个 500 万像素的传感器，允许它在 Raspberry Pi 基金会的一些驱动程序的帮助下捕捉 2560×1920 的图像以及完整的 1080 视频。

考虑到我们看到的 Raspi USB 网络摄像头项目实际上并不那么强大—[OpenCV 在没有任何图像处理的情况下大约以 4 fps 的速度运行](http://www.fanjita.org/serendipity/archives/57-Capturing-webcam-video-with-OpenCV-on-Raspberry-Pi-Arch-Linux.html),在有边缘检测的情况下大约以 1 fps 的速度运行 Raspberry pi 相机板应该不会给 Pi 带来太多负担，可以实现一些非常酷的计算机视觉项目。

相机板应该在一个多月内上市，所以对于我们这些等待现在就拿到它的人来说，我们必须满足于在休息后以 30fps 的速度向网络传输 1080p 视频的演示视频。
