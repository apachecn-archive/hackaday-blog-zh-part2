# 发光的复活节彩蛋比染发更有趣

> 原文:[https://hack aday . com/2013/04/15/发光的复活节彩蛋比染色工作更有趣/](https://hackaday.com/2013/04/15/glowing-easter-eggs-more-fun-than-a-dye-job/)

![glowing-easter-eggs](../Images/d2739bcce8760911f94bd10cb36faac6.png)

这是一个让孩子们在下个复活节记住的项目。它使用[电子设备点亮你的鸡蛋，而不是让它们死亡](http://www.apfelkern.info/index.php/2013/03/show-osternest-mit-arduino-und-lindt-hasen/) ( [翻译](http://translate.google.com/translate?sl=auto&tl=en&js=n&prev=_t&hl=en&ie=UTF-8&eotf=1&u=http%3A%2F%2Fwww.apfelkern.info%2Findex.php%2F2013%2F03%2Fshow-osternest-mit-arduino-und-lindt-hasen%2F))。

该项目仍然有一只脚在旧的传统，因为它开始了吹鸡蛋。底部较大的孔用于排空蛋白轭，大小刚好可以插入 LED。你可以简单地将它们连接到电池和电阻上，但[Rene]决定通过在复活节篮子的假草地中隐藏 Arduino 板来增加一些功能。通过这种方式，RGB LEDs 可以通过不同的颜色发光、闪烁和旋转。铝箔覆盖的巧克力兔子不仅仅是为了展示。他将它们连接到 Arduino 的 I/O 引脚上，用作开关。当它们都放在同一块箔片上时，电路就完成了，灯光秀也开始了。跳完之后在剪辑里自己看。

当然，对于大一点的孩子，你需要更复杂的东西来吸引他们的注意力。
