# 在圣诞灯的随机串上显示文本

> 原文:[https://hack aday . com/2012/12/16/displaying-text-on-random-strings-of-Christmas-lights/](https://hackaday.com/2012/12/16/displaying-text-on-random-strings-of-christmas-lights/)

![lights](../Images/7f3db9ac45a2acc4010d0fbcf5264e03.png)

在微控制器的帮助下，几串 GE 彩色效果圣诞灯可以很容易地变成一个完全可编程的 LED 灯条，或者如果你愿意，一个巨大的 RGB LED 显示屏。[Hubbe]有几串这种彩色效果的圣诞灯，但他不想花时间将他的灯串排列成一个阵列，只是为了得到一个可编程的显示。他对这个问题的解决方案是[混沌显示器](http://fredrik.hubbe.net/chaosdisplay/)——将随机扔在树上的一串串圣诞灯变成完全可编程的显示器，能够显示文本和图像。

【Hubbe】的灵感来自 [QC Co-Lab 的光墙](https://www.qccolab.com/LightWall?page=7)由 GE 色效灯供电。拥有一个巨大的 RGB LED 显示屏非常酷，但需要为每个圣诞灯像素建立一个框架。[Hubbe]有一个不同的想法——只是把灯扔在树上，用网络摄像头找出每个圣诞灯在显示屏上的位置。

实际建筑由六串色彩效果灯组成。在把它们扔到树上之后，[Hubbe]把他的手机放在三脚架上，记录下每一束光线的图像。凭借一定的计算能力，他能够创造出一个由缠绕在一起的圣诞彩灯组成的虚拟显示器。
