# 米奇，会自己充电的机器人

> 原文:[https://hack aday . com/2014/05/19/给自己充电的机器人 Mikey/](https://hackaday.com/2014/05/19/mikey-the-robot-that-charges-itself/)

![mikey-the-robot](../Images/89f5138e3a59024cc5d099f863c34630.png)

米奇是[迈克的]自主机器人。像所有好父亲一样，他给机器人起了名字。Mikey 是一个基于 Arduino 的机器人，它使用一个像素相机进行视觉。

[Mike]从一个普通的四轮驱动机器人平台开始。他添加了一个 Arduino Uno、一个电机控制器和一个 [Pixy](http://hackaday.com/2013/08/21/a-fast-and-easy-to-use-vision-sensor/) 。Pixy 通过串行链路向 Arduino 发送指示。米奇最初的任务是开车四处寻找地板上的青蛙。从那以后，【迈克】为迈克找到了更高的使命:[自我充电](http://www.instructables.com/id/Mikey-Go-Home/)。

生活最基本的特征之一就是吃。在自主机器人的情况下，这意味着自我充电。[Mike]通过训练小精灵探测绿色方块，给了 Mikey 自我充电的能力。绿色方块代表米奇的充电站。安装在 3D 打印支架上的探针固定正极导线，而基站底座上的弹簧与 Mikey 腹部的导电胶带接触。一旦电路完成，Mike 停止移动并开始充电。
