# 3D 球体显示和控制器

> 原文：<https://hackaday.com/2012/05/09/3d-sphere-display-and-controller/>

![](img/4ba18c57f6e4cc0ded77d10fee9eeb47.png "sphere")

[Nirav]已经在球形显示器上工作了大约一年，他刚刚想出了一个与这个屏幕交互的好方法:[一个相邻的现实跟踪器](http://eclecti.cc/computergraphics/snow-globe-and-the-adjacent-reality-tracker)，它可以旋转显示器，以匹配控制器的当前方向。

早些时候，[Nirav]使用几个 3D 打印部件和一个 [Showwx 激光微型投影仪建造了一个 8 英寸的球体显示器。](http://www.microvision.com/showwxplus/)显示屏看起来很棒，但是【Nirav】没有办法与之互动。为了解决这个问题，他在 3d 打印的地球模型中放入了惯性测量装置。IMU 与计算机通信，以读取[Nirav]控制器的旋转和方向，并相应地调整球形显示器。

这是一个非常好的建筑，非常接近未来科幻显示器。至少现在，[Nirav]可以和他创造的[牛逼卫星追踪 app](http://eclecti.cc/computergraphics/satellites-on-a-snow-globe-spherical-display) 和大陆漂移模拟互动了。

有一堆视频[Nirav]展示了他的显示器能做什么。休息之后你可以去看看。
