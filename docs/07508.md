# Echo，第一个有用的家用电脑智能？

> 原文:[https://hack aday . com/2014/11/07/echo-the-first-used-home-computer-intelligence/](https://hackaday.com/2014/11/07/echo-the-first-useful-home-computer-intelligence/)

我们熟悉 Siri 或微软 Cortana 等功能，这些功能从科幻小说中摸索出一个熟悉的概念，但却让我们做一些愚蠢的事情，比如站在公共场合对着手机大喊大叫。亚马逊对人工管家的想法采取了一种新的方法，将人工智能从我们的外围设备中分离出来，使其成为一个独立的单元，像其他任何电器一样在家庭中运行。然而，它可以通过蓝牙与你的设备集成，帮助你写购物清单等任务，或者只是帮助你记住一升有多少夸脱，而不是操纵你的星舰。无论你要求什么， [Echo 都会满足](http://www.amazon.com/oc/echo)。

这个设备只不过是互联网和一个塞在花瓶大小的最小黑色圆柱体中的扬声器，哦，还有六个指向每个方向的远场麦克风，它们听着你说的每一个字……*永远*。如你所料，Echo 只在你说出它的名字引起注意后才处理你说的话。如果你碰巧离得太远，定向麦克风听不到，你也可以从另一台设备上的 Echo 应用程序寻求帮助。对于亚马逊低得出奇的要价来说还不错，Prime 订阅者要价 100 美元。即使你对这个聊天方尖碑的想法垂涎三尺，或者有足够的兴趣购买一个只是为了检查一下(并弹出它的小接缝)，它们目前只能通过邀请购买……据说这类产品将在几周内上市。

互联网在很大程度上充当了一把无形的、无时不在的家庭知识瑞士军刀，这种想法无可否认是非常美好的。它拨动了我们对未来科技的热切心弦。然而，Echo 作为第一个此类产品的成功取决于它内部的人工智能如何无缝(和快速)地执行。如果它能坚持下去，或者在未来的迭代中被证明能坚持下去，那么想想在不久的将来这项技术能与什么样的更大的系统集成在一起是令人兴奋的……我们可能会比我们想象的更早拥有指挥中心意识。

也就是说，邀请一个小小的 WiFi 探头进入你的私人生活空间来监听你所做的一切将需要一些时间来克服…你的想法？
