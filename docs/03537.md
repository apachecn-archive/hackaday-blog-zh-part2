# 一个定制的星际键盘

> 原文:[https://hack aday . com/2013/07/05/a-custom-starcraft-keyboard/](https://hackaday.com/2013/07/05/a-custom-starcraft-keyboard/)

![keybaord](../Images/4938e172512cf2f23cd40e73b6d4b57c.png)

如果你正在寻找一个项目，但不知道该怎么做，你会怎么做？如果你是[gltovar]，就随便看看你的爱好。他玩星际争霸已经有一段时间了，最终决定[打造一个定制的星际争霸键盘](http://gltovar.com/blog/?p=323)。

[gltovar]发现了一种有趣的键盘布局，称为[核心](http://www.teamliquid.net/forum/viewmessage.php?topic_id=341878)。表面上，这种布局允许玩家更快地使用星际争霸中最常用的命令。[gltovar]决定他需要一个专门为核心设计的键盘，并着手制作一个自己设计的键盘。

键盘矩阵围绕一排 1×5 的按键设计，每个按键都连接到一个充当 USB 设备的微型微控制器。每一排都是一个独立的电路板，允许[gltovar]在一个定制的外壳中上下移动这些排。

这是[gltovar]想要的定制键盘的一切。这是否改善了他的星际争霸游戏完全是另一回事，但我们确信他至少玩得很开心。
