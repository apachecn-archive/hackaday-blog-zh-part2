# 星门 LED 时钟有足够的活力

> 原文：<https://hackaday.com/2014/10/26/star-gate-led-clock-has-plenty-of-pizazz/>

【大卫·霍普金斯】终于在 Hackaday.io 上完成了他的 [Star Gate LED 时钟，看起来棒极了。](http://hackaday.io/project/1919-stargate-led-clock)

我们最初在[Hacklet 18-Tick Tock，是时候计时了](http://hackaday.com/2014/10/10/hacklet-18-tick-tock-its-time-for-clocks/)中介绍了他的项目进展，我们很高兴地说它终于完成了。该时钟采用 60 个 WS2812 LEDs 来模拟 Star Gate 的人字形，并显示时间。引擎盖下是一个 RTC，一个 Arduino Nano，一个 LDR，甚至还有一个每小时一次的“钟声”。我们提到过它在晚上也会自动变暗吗？

更让我们印象深刻的是构建质量，而[David]实际上并没有提到他是如何做到的——不管怎样，它看起来很棒！休息后留下来看看它的运行视频，这样你就能真正体会到时钟的功能。
