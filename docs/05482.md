# 数学！3D 打印的 BMO 拥有全方位的情感

> 原文:[https://hack aday . com/2014/03/08/mathematical-3d-printed-BMO-has-a-full-range-of-emotions/](https://hackaday.com/2014/03/08/mathematical-3d-printed-bmo-has-a-full-range-of-emotions/)

![3d_printing_hero-1](../Images/33930beae7e4a854a0a064cbb635d974.png)

冒险时间的忠实粉丝？然后你会爱上这个可爱的 3D 打印 BMO 机器人！

冒险-什么？冒险时间是一个“儿童”电视节目，讲述了芬恩(人类)和杰克(狗)以及他们的个人电脑 BMO，一个古怪的小机器人游戏男孩。这是成年人也能欣赏的节目之一——有时会想，与电视上的其他节目相比,**怎么会是一个儿童节目呢！**

无论如何，[诺]和[佩德罗]是该节目的忠实粉丝，他们决定尝试制作自己的 3D 打印 BMO。它利用一个 8×8 的 LED 矩阵来显示 BMO 的全部情感，这是由一个 [Gemma](http://learn.adafruit.com/introducing-gemma) 控制的——一个直径为 1”的小 Attiny85 平台板，可通过 USB 用 Arduino IDE 编程。他们甚至为 LED 矩阵找到了一个超级方便的[动画工具，可以让你非常容易地添加自己的表情和动画——只需复制输出代码，替换 anim.h，你就大功告成了！](http://www.diyode.com/mini8x8/)

留下来看看它是怎么做的！
