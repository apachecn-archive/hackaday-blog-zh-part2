# 虚拟本拆

> 原文：<https://hackaday.com/2015/08/25/virtualbench-tear-down/>

当你把一个混合信号示波器、一个函数发生器、一个万用表、一个电源和一些可编程数字 I/O 交叉放在一个盒子里，你会得到什么？听起来像是一个极客笑话，但它实际上是国家仪器公司的 VirtualBench 产品。[Shahriar]有一个，想知道里面是什么，所以他[做了一个撕掉](http://thesignalpath.com/blogs/2015/08/23/national-instrument-virtualbench-review-teardown-experiments/)。

由此产生的视频(警告，这是一个很长的视频，超过一个小时)讲述了仪器内部的内容以及使用它的感觉。他做了一些非常有趣的测试，包括观察时钟发生器和分析带通滤波器。2000 美元的价格标签对大多数黑客来说是难以接受的。如果你分别购买每样东西，你可以很容易地花费几乎一样多的钱，但正如[Shahriar]在视频结尾指出的那样，VirtualBench 做得很好，但它可能在任何一项功能上都不如专用乐器。

尽管如此，如果你现金充裕而空间不足，或者你正在装备一个实验室，在那里由一家公司将所有东西放在一起(并提供支持)可能是有价值的，看起来 VirtualBench 确实做了很多事情。即使你不想买这个设备，偷看一下它的内部，看看是什么让它滴答作响，仍然是很有趣的。正如我们之前提到的，这个乐器很容易控制，也有很多可能性。
