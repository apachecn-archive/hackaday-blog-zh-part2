# 3D 打印机播放经典 MIDIs

> 原文：<https://hackaday.com/2015/03/09/3d-printer-plays-classic-midis/>

不管出于什么原因，我们似乎都痴迷于将除了扬声器之外的东西做成扬声器。硬盘驱动器、软盘驱动器、CD 驱动器、传真机、无生命的物体、点阵打印机，现在——好吧，让我们跟上时代——[3D 打印机！](http://sinkhacks.com/playing-the-theme-from-jurassic-park-on-a-3d-printer/)

[安德鲁·辛克]想尝试踏步机音乐(现在这真的是一种音乐类型吗？是的，我们称之为循序渐进。))，所以他找到了 HomeConstructor.de，正好有一个很棒的 [MIDI 到 g 代码转换器](http://en.homeconstructor.de/converter-convert-music-in-g-code.html)专门用来让那些步进器嗡嗡作响。他选择的工具是一个原始的 Printrbot，但不幸的是，它需要几个小时调整 g 代码才能正常工作。
