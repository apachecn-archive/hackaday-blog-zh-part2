# 麻省理工学院的玻璃 3D 打印机

> 原文：<https://hackaday.com/2015/08/22/mits-glass-3d-printer/>

你的 3D 打印机的热端有多热？大多数低成本打印机的温度最高可达 240°C(464°F ),因为它们含有聚醚醚酮，如果温度升得更高，聚醚醚酮就会开始变软。即使是主动冷却的金属热端通常也不会超过 400 C (752 F)。很性感，对吧？[【麻省理工学院】新 G3DP 打印机](http://matter.media.mit.edu/news/article/glass-printing-from-mediated-matter1)温度高达 1900 F(超过 1000 C)，打印光学透明玻璃。

通过改变设计和印刷参数，G3DP 可以限制或控制光的透射、反射和折射。打印机使用双加热室。上室作为 1900 华氏度的窑，而下室用于退火结构。打印头是氧化铝-锆石-二氧化硅喷嘴。

显然，这个过程已经申请了专利，你可以从[约翰·克莱恩的]论文和一些即将出版的出版物中阅读技术细节。关于这篇论文，我们发现了两件有趣的事情:[John]是一名建筑师，打印机使用 GCode。

如果你找不到任何氧化铝-锆石-二氧化硅和 1900 度的热端，也许用数控切割玻璃是一个更容易实现的目标。
