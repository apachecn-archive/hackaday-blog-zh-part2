# 将手动相机镜头转换为使用电动变焦和对焦

> 原文:[https://hackaday . com/2012/05/06/converting-a-manual-camera-lens-to-use-motorized-zoom-and-focus/](https://hackaday.com/2012/05/06/converting-a-manual-camera-lens-to-use-motorized-zoom-and-focus/)

![](../Images/fd11fc9f96ce1033177ca272cf3d2b0d.png "automatinc-lens-focus-and-zoom")

[Guy]来信分享他最近完成的这个机动相机镜头项目。他真的很喜欢变焦镜头，但由于变焦和对焦都是手动控制的，他有时很难及时将两者设置到正确的位置来拍摄。随着现代 DSLR 相机允许视频捕捉，他还想有一个平滑的变焦选项，总是在焦点上。解决方案是在戒指上添加马达，并用 Wii classic 控制器控制它们。

当涉及到附加硬件时，这种方法真的很棒。他有一些制作精美的圆环，环绕着镜头上的焦点和变焦环。然后，它们由同步带固定。这些皮带有齿，齿与一对伺服电机上的齿轮啮合。从那里开始，用 Arduino 驱动电机很容易，用一个分线连接器连接到 Wii 控制器。你可以看到[Guy]在广告之后展示了他的身材。
