# 为 Android 平板电脑添加感应充电

> 原文:[https://hack aday . com/2012/08/16/add-inductive-charging-to-an-Android-table t/](https://hackaday.com/2012/08/16/adding-inductive-charging-to-an-android-tablet/)

![](../Images/18ff3315111de157953a1ef2b3a5cd5c.png "inductive-charging-for-an-android-tablet")

这是一台梅格 G9T，一台运行安卓 4.0 的 9.7 英寸平板电脑。【食肉动物】向我们展示了[如何将其改装成使用感应充电](http://www.slatedroid.com/topic/37248-gemei-g9t-wireless-charging-mod-tutorial/)。感应充电硬件取自掌上设备(这使用了[在其他几个黑客中看到的 Touchstone 充电](http://hackaday.com/2010/06/21/inductive-charging-for-the-htc-evo/)硬件)。与平板电脑的电子设备接口很容易，但实际放置线圈和磁铁就是另一回事了。

休息后的视频将向您展示整个过程。他首先卸下螺丝，撬开平板电脑的外壳。从那里[食肉动物]展示了如何小心翼翼地从手掌背板上取下线圈、电路板、屏蔽和磁铁。磁铁首先被放置在平板电脑的背板上。这种金属太厚了，不能很好地固定住，所以他用一个 Dremel 磨掉足够的材料来实现牢固的连接。不幸的是，金属会屏蔽线圈工作所需的磁场，所以他在外壳上切了一个和线圈一样大小的洞。该区域覆盖有液体绝缘胶带，以防止短路，所有东西都用胶带固定到位。两根跳线从线圈的电路板连接到 5V 充电输入是所有需要完成的黑客。
