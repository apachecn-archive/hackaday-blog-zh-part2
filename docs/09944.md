# 3D 打印迷你 Vectrex

> 原文：<https://hackaday.com/2015/08/29/3d-printed-mini-vectrex/>

随着 3D 打印机越来越普遍，制作复古电脑和视频游戏设备的微型模型是一种让书呆子出去而不是填满房子的方法。[Jason]环顾四周，注意到没有人制作过 Vectrex 视频游戏系统的模型，于是他用独特的 80 年代早期视频游戏系统的 3d 打印微型模型填补了空白。

对于那些不喜欢复古游戏系统的人来说， [Vectrex](https://en.wikipedia.org/wiki/Vectrex) 是一款 1982 年的 8 位游戏机，它的独特之处在于它在控制台中自带单色矢量图形 CRT。[Jasons]型号配有一个带 SPI 接口的 2.2 英寸 LCD。

仿真由一个配备 360Mhz MIPS CPU 和适度的 32 兆 ram 的单板机驱动，这对于处理 8 位数学和线框图形来说绰绰有余。使用的仿真器是 VECX 的端口，显示器被重新路由到 LCD 屏幕，而不是使用标准的 SDL 接口。

该案件是在 Sketchup 建模，整个地段是由 3v3 脂电池供电。休息之后，请加入我们，观看迷你模型运行“地雷风暴”介绍的快速视频，这是机器上的原始游戏。
