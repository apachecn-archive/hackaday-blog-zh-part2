# 复古音乐:高音是电子音乐的基本元素

> 原文：<https://hackaday.com/2015/08/05/retrotechtacular-the-trautonium-was-elemental-to-electronic-music/>

电气工程师兼音乐发烧友[Freidrich Trautwein]不满。他认为钢琴的平均律音阶限制了演奏者表达的空间。因此，在 1930 年，[特劳特魏因]和一位名叫[奥斯卡·萨拉]的有成就的钢琴家开始研究一种机电乐器，这种乐器可以将弦乐组无品指板的滑音带给键盘手。[特劳特魏因]称他的发明为[Trautonium](http://120years.net/the-trautoniumdr-freidrich-trautweingermany1930/)。

[声音由锯齿频率发生器在乐器中产生](http://www.doepfer.de/traut/traut_e.htm)。然后通过过滤器，由基于电阻串的[手册](https://en.wikipedia.org/wiki/Manual_(music))操纵。频率和音调相对于演奏者的手指沿着一段不导电的琴弦的位置以及所施加的压力大小而变化。该电阻串悬挂在一对支柱之间的导电金属条上。在柱子上施加一个小电压，这样当琴弦碰到下面的金属条时，演奏者就会操纵一个压控振荡器。一系列同样不导电的金属舌悬挂在琴弦上方。这些按音阶间隔放置，可以像按键一样使用。

这个早期的合成器能够产生多种声音，从清脆的啁啾声到潮湿的拍击声，以及介于两者之间的任何声音。事实上，阿尔弗雷德·希区柯克的惊悚片中的所有音效都是由该乐器唯一的大师【奥斯卡·萨拉】在改良的 Trautonium 上制作的。他继续为数百部电影配乐，通过指尖上的 Trautonium 观看它们，将他的作品记录并分层到一个怪异的声音墙中。
