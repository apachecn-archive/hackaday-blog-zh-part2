# 建造一座海盗大炮以供娱乐和掠夺

> 原文:[https://hack aday . com/2012/12/04/建造一座海盗大炮以供娱乐和掠夺/](https://hackaday.com/2012/12/04/building-a-pirate-cannon-for-fun-and-plunder/)

今年我们在泳池边的日子已经过去了。但是和一个两岁的孩子玩海盗船听起来确实很有趣。这就是为什么[石驴] [建造了这个带有开火声音的海盗炮道具](http://imgur.com/a/DPqZ0)。

设计的简洁非常出色。三段 PVC 和五个配件组成了一个看起来很真实的桶，当你把它拿出来玩的时候，不会把你的背甩出去。经过一些切割和粘合，整个东西被喷上了哑光黑漆。缠绕在桶上的绳子是一个很好的触摸。底座是用一些废弃的松木制成的，但正是顶部的那个小木箱让它变得非常特别。这是保险丝盒，手指轻轻一点，就会发出保险丝燃烧的声音和视频，然后是惊天动地的巨响。[石头驴]用他的旧 Droid X Android 手机，为它编写了一个应用程序，可以播放声光秀。休息之后来看看视频。
