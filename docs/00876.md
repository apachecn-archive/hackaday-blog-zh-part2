# 使用 3D 打印零件构建拾取和放置

> 原文:[https://hack aday . com/2012/06/28/building-a-pick-and-place-with-3d-printed-parts/](https://hackaday.com/2012/06/28/building-a-pick-and-place-with-3d-printed-parts/)

![](../Images/75fd11646ff3fdc91f0487b394f228bb.png "arm")

在过去的几个月里，[HeliumFrog]一直在[建造一个 SCARA 机器人](http://www.heliumfrog.net63.net/hf06robot/hf06blog.html)，作为一台取放机器的基础。有点令人惊讶的是，这是第一个用 3D 打印机打印的机器人。

SCARA 型机器人是一种铰接臂，非常适合将元件从管子和卷轴转移到 PCB 上。[HeliumFrog]开始建造一个手臂，它的关节处有大齿轮，由步进电机和齿形带驱动。Z 轴最初由丝杠驱动，但经过深思熟虑的重新设计后，改为另一种齿形带。

我们已经看到了自己的 [DIY 拾取和放置机器](http://hackaday.com/2011/06/24/a-pick-and-place-machine-for-under-1k/)，但大多数都是基于传统的 X/Y 笛卡尔坐标系。[HeliumFrog]的 SCARA 机器人应该——至少在理论上——更快、更准确，同时占用车间更小的空间。

[HeliumFrog]或多或少完成了他构建的基础，现在正着手为他的 SCARA 机器人构建一个塑料挤出工具头。非常酷，应该能让这个机器人以低于 400 英镑(约 600 美元)的价格自我复制。

休息之后，你可以看看这个关节臂机器人的视频。
