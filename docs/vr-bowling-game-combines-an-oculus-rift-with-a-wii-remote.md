# 虚拟现实保龄球游戏结合了 Oculus Rift 和 Wii 遥控器

> 原文：<https://hackaday.com/2014/08/13/vr-bowling-game-combines-an-oculus-rift-with-a-wii-remote/>

[marclar83]得到了一个 Oculus Rift，以便为即将到来的会议演示做准备。他开始下载演示，熟悉了 VR 界面，但失望地发现有人还没有开发出良好的虚拟现实保龄球体验。这促使他设计了一款集成了 Wii 遥控器的 [VR 游戏，记录控制器的动作并将加速度计数据发送到他的电脑。](https://developer.oculusvr.com/forums/viewtopic.php?t=6301)

他创造的游戏类似于 Wii Sports Bowling，但有一个额外的好处，就是可以沉浸在 Oculus Rift 的虚拟世界中。Wii 遥控器上的 D-pad 被编程为切换姿势和保龄球方法，允许用户选择他们是想把球扔向中间还是让球曲线前进。按下背部的触发按钮开始摇摆运动，当释放时，保龄球以很高的速度射向球道，撞上终点的球瓶。

因为游戏是在最初的 DK1 上设计的，所以图像的分辨率是一个需要解决的挑战，但是[marclar83]通过在屏幕一侧实现两个用户界面解决了这个问题，这两个用户界面显示重播和描绘还有多少个大头针；证明对游戏玩家来说是更好的体验。这个免费的公开 alpha 版本在官方 VRBowling 网站上提供给 Windows、Mac 和 Linux。描述这个项目的视频可以在下面看到。T3[https://www.youtube.com/embed/Ag5gMPjY0Fg?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/Ag5gMPjY0Fg?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)
