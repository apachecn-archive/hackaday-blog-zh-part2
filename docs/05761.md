# 奥斯卡在巡逻时千万不要丢了你的铅笔

> 原文:[https://hack aday . com/2014/04/10/永远不要丢了你的铅笔和奥斯卡一起巡逻/](https://hackaday.com/2014/04/10/never-lose-your-pencil-with-oskar-on-patrol/)

![OSkAR](../Images/063e927506854e0e3dbd4e9a99651887.png)

[Courtney]一直在努力开发基于 OpenCV 的说话机器人 OSkAR。奥斯卡是美国西弗吉尼亚州谢泼德大学[【库尼】的顶点工程](http://webpages.shepherd.edu/ccrite03/Project%20Website/documents/paper.pdf) (pdf 链接)。奥斯卡的目标是成为一个辅助机器人。OSkAR 将在典型的家庭环境中导航，通过语音合成软件报告它发现的对象。

为了做到这一点，[Courtney]开始用一个小猎犬骨黑和罗技 C920 网络摄像头。这个机器人的身体是用乐高 Mindstorms NXT 零件建造的。这意味着，当 OSkAR 不能自主操作时，可以通过蓝牙从 Android 手机上控制它。在软件方面，[Courtney]从 BBB 的 stock Angstrom Linux 发行版开始。在遇到视频问题后，她将桌面环境切换到了 [Xfce](http://www.xfce.org/) 。OpenCV 提供了机器视觉系统。[Courtney]为 OSkAR 识别的几个物体创建了模型。

现在，奥斯卡的生活就是在房间里四处寻找铅笔和门框。当发现一支铅笔或一扇门时，奥斯卡会宣布这个物体，以及它是在他的左边还是右边。对于机器人来说，这听起来可能是相当无聊的生活，但这学期还没有结束。[Courtney]仍在努力创造更多的对象模型，这将把 OSkAR 的兴趣扩展到新的领域。
