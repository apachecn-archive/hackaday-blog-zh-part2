# 新的智能手机保护套带回了旧的智能手机功能

> 原文：<https://hackaday.com/2015/06/22/new-smartphone-case-brings-back-old-smartphone-features/>

我们都记得智能手机刚刚起步的美好时光。意识到我们可以把一台功能齐全的电脑塞进一部手机大小的东西里，这是一场相当具有革命性的变革。一些早期的手机，如最初的摩托罗拉 Droid，有一些功能在今天并不常见，[liviu]开始通过给他的现代智能手机添加滑动 QWERTY 键盘来解决这种情况。

这款手机的设计始于一部三星 Galaxy Note 4 和两个外壳:一个用于手机，一个用于键盘。[liviu]找到一个手机大小的蓝牙键盘，在把它塞进箱子之前，把所有不必要的部分都去掉了。然后，他用个人电脑电源和两部旧翻盖手机的零件制造了滑动装置，然后将两部分组装在一起。使用两个翻盖手机铰链给这种情况下的额外功能，能够翻转后滑出。其结果是一部现代智能手机，具有梦幻般的经典智能手机风格，看起来非常有用。

我们已经推出了让旧智能手机焕发新生命的项目，但这可能是第一个让新智能手机焕发新生命的项目。我们不介意看到更多配备这些功能的旗舰手机，但[liviu]在弥补制造商的不足方面做得很好！
