# Muse 脑电图耳机拆卸

> 原文：<https://hackaday.com/2015/07/15/muse-eeg-headset-teardown/>

Adafruit 公司的[Lady Ada]刚刚令人愉快地拆除了 Muse EEG 耳机。

Muse 耳机是一款相当昂贵的消费级 EEG 耳机，能够追踪你的脑电波以进入更深的恍惚状态，从而保证更好的冥想。我们不太喜欢在 Hackaday 这里冥想，但是脑电图传感器真的有用。这是非常酷的看到这个内部没有分叉 300 美元，我们自己一个可能打破。

像大多数脑电图耳机一样，它们并不是真的被设计成睡觉时佩戴的。耳朵上方的两个大豆荚一边装有电池和充电电路，另一边装有大脑。它的简洁部分是一个小小的可调节金属片，允许在保持所有电气连接的同时对表带进行调节。柔性电路容纳沿着带的长度延伸的前额电极。

在过去，我们已经看到了在 [Lucid Scribe 项目](http://lsdbase.org/2012/12/11/First-Dream-with-Modified-MindWave/)上所做的工作，使用了[一种改良的 Neurosky Mindwave](http://hackaday.com/2012/12/20/modifying-an-eeg-headset-for-lucid-dreaming/) EEG(售价 99 美元，比黑客便宜得多)。这个想法是能够相应地监控你的睡眠周期，然后给做梦者声音提示，以便在梦里“醒来”。想想《T4》盗梦空间的音乐。

不幸的是，看起来缪斯女神不会更适合做清醒梦。如果你能够将电极从耳机的其他部分分离，那么它可能就能工作了。
