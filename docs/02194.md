# 乐高 LP 播放器

> 原文:[https://hackaday.com/2012/11/26/lego-lp-player/](https://hackaday.com/2012/11/26/lego-lp-player/)

![](../Images/62cf93a4615b92d82917f4ddaf809b60.png "lego-LP-player")

这个 LP 播放器完全由乐高零件制成。它播放每张唱片上编码的歌曲，但不是用凹槽里的唱针。相反，每张唱片底部都有一个颜色代码，由下面的光学传感器解读。

除了功能之外，[夏嫣·武尔祖恩]还确保了建筑看起来像零件。喇叭是一个很好的接触，但你也可以欣赏旋转的迷你数字在基地的前侧。右边有一扇隐藏的门，可以进入驱动系统的 NXT 模块。使用两种不同的工具产生了新的记录。首先，这首歌是用压轴，一个成熟的乐谱程序写的。它被导出并通过第二个程序运行，第二个程序产生应用于记录的彩色光盘设计。休息后，您可以在片段中为自己听到歌曲。

如果 LP 玩具是你的拿手好戏，你会想看看这个费雪玩具的 3D 打印记录。
