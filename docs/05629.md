# 从旧扫描仪构建网络可控 RGB LED 灯

> 原文:[https://hack aday . com/2014/03/26/building-a-network-controlled-lamp-from-an-old-scanner/](https://hackaday.com/2014/03/26/building-a-network-controllable-rgb-led-lamp-from-an-old-scanner/)

![EthernetLamp](../Images/704b4264be60b8783c754dfa28996dab.png)

能够利用你的一个旧项目来做一个更好的新项目是非常令人满意的。来自 Hackshed 的[Steve]就是这样做的:他将一个基于 Arduino 的网络服务器集成到一个新的[网络可控 RGB 灯](http://www.hackshed.co.uk/lanp-a-diy-arduino-network-controllable-rgb-lamp-made-from-scanner-parts/)中。

这款灯的独特之处在于，RGB LED 条来自一台老式爱普生扫描仪。回收旧项目或废弃电子产品的剩余部分是真正的黑客之道。在确定引脚和正确的电压运行的 led，乐趣开始了。随着 LED 栏正常工作，下一步是集成一个基于 Arduino 的 web 服务器。使用 SD 卡托管网站和以太网 Arduino 屏蔽，led 变得可通过网络控制。没有错过任何一个节拍，[史蒂夫]集成了一个[基于 Javascript 的颜色选择器](http://www.daviddurman.com/flexi-color-picker/)，支持多种网络浏览器。这使得界面看起来相当专业。休息之后，请务必观看灯的运行情况！

总的结果是一个惊人的变色灯，完美的工作。剩下要做的就是为它制作一个外壳，或者将其集成到现有的灯具中。这是使用 LED 灯带的好方法，否则会被浪费掉。如果你找不到像这样的彩色棒扫描仪，你总是可以从 RGB 条开始。
