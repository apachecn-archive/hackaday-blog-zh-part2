# 用于轻型取放的压电真空泵

> 原文：<https://hackaday.com/2014/10/30/piezo-vacuum-pump-for-lightweight-pick-and-place/>

如果你正在建造一个拾取和放置机器，甚至只是一个真空笔，你将需要一些方法来拾取微小的部分。这意味着一些糟糕的东西，水族管，以及与之相关的一切。几个月前，[Wayne]发现了一个叫做微型鼓风机的有趣装置，它可以从一个小而轻的装置中吹出少量的空气。几经修改后，[和他有了一个压电真空泵](https://sites.google.com/site/wayneholder/piezoelectric-vacuum-pump-for-pick-an-place-machine)来拾取微小的零件。

[发现的微型吹风机](http://www.murata.com/products/mechatronics/fluid/feature)【Wayne】在 Mouser 上可以买到，价格大约 45 美元，但是这个设备会吹。为了把它变成糟糕的东西，他需要找到一种方法堵住泵的输入端，这样它就能抽真空。最终决定将鼓风机安装在一堆泡沫板内，[叶晚]粘在 20 号针上，能够吸取 0603 SMD 零件。

新的压电吸盘非常轻，功率消耗非常合理:18V 和 20mA。这将是一个很好的装置，可以安装到某个抓放机上，而不必通过运动平台的机械装置运行真空管。下面视频。
