# 矢量激光投影仪是设计过程中的一课

> 原文：<https://hackaday.com/2014/09/20/vector-laser-projector-is-a-lesson-in-design-processes/>

两年的电子工程课程之后，[约书亚·贝特曼]和[亚当·卡特利]正在寻找一个有趣的暑期项目。他们没有一瘸一拐地使用他们自己可以筹集的资源，而是设法让他们的学校——布里斯托尔大学——来支付这笔费用！

现在 Uni 没有无缘无故就资助的习惯，我们认为这就是为什么他们两个在记录他们的工作方面做得如此之好。我们习惯于看到专门针对一个项目的博客，但是这个博客有[一个庞大的作品集，包含了构建过程中的每一项工作](http://www.lazeyprojector.blogspot.co.uk/)。在任何组装开始之前，他们绘制设计图以形成规格，布置电路和电路板插图，甚至制定软件如何运行，以确保硬件满足他们的所有需求。

当零件到达时，手工组装表面贴装电路板的工作就开始了。这反映在他们录制的快速动作视频中，包括这个以[176 引脚 LQFP](https://www.youtube.com/watch?feature=player_embedded&v=YWLqxKhittY) 为特色的剪辑。驱动板是 Raspberry Pi 的屏蔽，它驱动负责镜子 X 和 Y 移动的检流计。

下面的视频展示了他们的成功，这个博客是一个很好的资源，一旦有了新的文凭，在申请工作时可以参考。

你认为下一步应该怎么做？我们提倡像几年前我们看到的这台 RGB 激光投影仪一样去疯狂小镇旅行。当然[我们周四看到的经典矢量游戏](http://hackaday.com/2014/09/18/building-a-vector-monitor-controller/)也同样令人敬畏，根本不用提醒这个硬件。
