# 现实生活中盒子里的小鸟

> 原文：<https://hackaday.com/2014/02/20/real-life-flappy-bird-in-a-box/>

这是 Flappy bird，那是 flappy bird，我们真的不太清楚原版直升机游戏的克隆版是如何变得如此流行的。总之，【Fawn Qiu】——make everything 的创始人——决定加入这股潮流，制作这个令人敬畏的实体版 Flappy Bird！

她在 Tribeca Hacks Hackathon 上把它扔在一起，它使用了一个 Arduino、两个伺服电机、一个簧片开关和一些磁铁。她受到最初的[盒子里的马里奥](http://hackaday.com/2011/09/09/no-quarters-required-for-this-sidescrolling-game-in-a-box/)游戏的启发，这是她的项目[make everything](http://www.projectmakeanything.org/)的一个很好的例子，这是一个技术项目社区，帮助在美国为孩子和成人培养 STEM(科学技术工程数学)的新文化。他们相信“有了正确的工具和指令，我们可以制造任何东西！”。

请观看下面的视频，在视频中[小鹿]带着它走上街头，让随机的陌生人在这个标志性的游戏中一试身手！
