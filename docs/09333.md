# 今年我们将看到的第一个小男孩

> 原文：<https://hackaday.com/2015/06/20/the-first-pipboy-well-see-this-year/>

你听说我们将在 2015 年 11 月 11 日关闭 Hackaday，对吗？那是辐射 4 的发布，相信我:那天我不会让 T2 做任何事。辐射系列中的一个新游戏意味着更多的动力装甲角色扮演，以及来自 Vault-Tec 友好人员的数百种不同的可穿戴电子设备。我说的是 PipBoy，这是*系列*的腕戴电脑，[和【THEMCV】制造了我们今年将看到的第一台](http://imgur.com/a/LFaPP)。这不会是最后一次。

这个小男孩[THEMCV]创造的是 3000a 模型，与《辐射 3》和《T2》中的模型相同。我们之前已经看过一些真人版的 PipBoy[这一个使用了亚马逊独家特别版*辐射 3 附带的小男孩道具*](http://hackaday.com/2012/11/14/a-wearable-pipboy-3000/)。自*辐射 3* 发布以来，事情已经发生了变化，为了打造他的 PipBoy，【THEMCV】[刚刚从 Shapeways](http://www.shapeways.com/product/LYHLHJLGZ/pipboy-3000) 买了一个。

电子设备包括一个 Raspberry Pi 型号 A、3.5 英寸 LCD、一个电池组和一个模拟 PipBoy 3000 软件的优秀软件。它看起来很棒，但[THEMCV]仍然需要找到一些复古未来主义的按钮和转盘来完成 PipBoy 体验。
