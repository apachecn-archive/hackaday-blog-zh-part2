# 焦点法官:乔·格兰德

> 原文:[https://hackaday.com/2014/07/15/judge-spotlight-joe-grand/](https://hackaday.com/2014/07/15/judge-spotlight-joe-grand/)

![judge-spotlight-joe-grand](../Images/4e4b9b870de10d32eaabb73262a83e4a.png)

多年来，我们一直对乔·格兰德着迷。他在 DEFCON 上的早期演讲，以及为其设计徽章的大量工作,有助于让会议引起我们的注意。这些年来，我们已经看到许多硬件来自他的公司[宏大创意工作室](http://www.grandideastudio.com/)，当然还有电视节目 *[的原型！对于某些电视高管来说，这一定是太棒了，以至于不能让它继续下去。](http://en.wikipedia.org/wiki/Prototype_This!)*

我们问了 Hackaday 奖的评委 Joe 几个问题。他发回了[下面嵌入的视频回应](https://www.youtube.com/watch?v=8e8K4L9kTg0)。他谈到了他最近在做的事情，波士顿的黑客社区，展示了他在讲授安全知识时使用的一些硬件，等等。

2.  **4:10**–你是早期 [L0pht](http://en.wikipedia.org/wiki/L0pht) 黑客组织的成员。那是什么，你和那个社区的人还有联系吗？
