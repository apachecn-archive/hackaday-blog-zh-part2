# 使微视图变得冗长

> 原文:[https://hackaday.com/2014/12/12/making-microview-wordy/](https://hackaday.com/2014/12/12/making-microview-wordy/)

尽管 MicroView 推出了大量设备，但我们还没有看到很多项目在项目中使用这种微型 Arduino 和有机发光二极管显示器。不要害怕，因为嵌入式系统工程师、播客和 Hackaday 奖评委【Elecia White】[在这里为这款非常小、非常酷的设备带来了可穿戴版本](http://hackaday.io/project/3577-wordy)。

微视的大小和形状迫切需要做成一个环，为此，[Elicia]正在使用风干的弹性聚合物粘土。为了将粘土附着在显微视图上，[Elecia]将一些母头放在一块试验板上，然后将粘土覆盖在它们上面塑造成一个环形。它的工作，虽然[Elecia]没有做任何太棘手的标题和粘土，有一些有趣的事情，你可以通过粘土运行电线。

这个戒指是做什么的？这是一个神奇的 8 号球，一个由加速度计控制的乒乓球游戏，一个每日一词的东西(有定义)，所有这些都被塞进一个由硅、有机发光二极管和粘土制成的黄铜关节里。下面视频。
