# 一种加热人的新方法

> 原文:[https://hackaday.com/2013/12/16/a-new-way-to-heat-people/](https://hackaday.com/2013/12/16/a-new-way-to-heat-people/)

[Leigh Christie]是麻省理工学院的一名研究人员，他开发了一种有趣的解决方案来为人们供暖，而不是为建筑物供暖。

他的 TEDx 演讲“建筑物供暖是愚蠢的”，展示了麻省理工学院可感知城市实验室解决能源问题的努力。他们的研究集中在寻找一种替代能源，以解决用于加热大型空间的惊人的能源浪费。虽然 TED talk 文章在 Hackaday 很少见，但我们认为这个想法既简单又有用。此外，[利]也是几年前在火人展上带给我们蒙多蜘蛛的人。他是个黑客。

不管怎样，这是什么？他发明的系统非常简单，非常聪明:一个追踪人的红外热聚光灯。使用微软 Kinect，灯会跟随你，让个人而不是整个空间保持温暖。[Leigh]有一个宏伟的计划，在大型建筑中实施他所谓的“局部供暖”以节省能源消耗，但较小规模的实施可能证明对大型车库或车间同样有益。在冬天，你的工作场所需要多少取暖费用？黑客空间似乎是一个拼凑起来的“局部供暖”系统的完美测试环境。如果有人造了一个，我们想听听。
