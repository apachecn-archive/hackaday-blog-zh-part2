# 使用示波器和信号发生器目视调谐您的高频天线

> 原文:[https://hackaday . com/2013/09/20/使用示波器和信号发生器直观调谐您的高频天线/](https://hackaday.com/2013/09/20/visually-tune-your-hf-antenna-using-an-oscilloscope-and-signal-generator/)

许多读者喜欢摆弄射频和业余无线电。人们总是关心的一件事是调谐天线。新设备从来都不便宜，所以每当另一个使用现有测试设备的选项出现时，都会引起我们的注意。[Alan Wolke]又名[w2ew]介绍了他使用信号发生器和示波器调谐高频天线的过程。

该过程更多的是一种教学辅助手段，而不是商业设备的实际替代品，主要是因为合适的信号发生器和示波器是大件物品，有时无法获得或负担不起。也就是说，如果你有这样的测试设备，你只需要建立一个简单的分线板，包含一种形式的[惠斯通电桥](http://en.wikipedia.org/wiki/Wheatstone_bridge)，其中未知的 Rx 是天线。两个示波器探头跨接在电桥平衡节点上。需要特别注意探针电缆长度和示波器的 50 欧姆输入阻抗的匹配。还需要一对 1K 探针耦合电阻，以防止影响连接点的阻抗。一旦选定的信号被注入，您可以调整天线调谐器，直到示波器上的两个电压波形匹配，表明您的天线网络被调谐到 50 欧姆阻抗，没有电抗。

能够在视觉上调谐你的天线真的可以帮助你理解在转弯过程中发生了什么；不仅匹配输入阻抗，还匹配表示电感或容抗的相移。休息之后，请加入我们，观看视频并了解[Alan]演示第二部分的内容。
