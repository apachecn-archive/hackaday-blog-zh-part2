# 复古:手工制作吉尼斯桶

> 原文：<https://hackaday.com/2015/07/21/retrotechtacular-coopering-guinness-barrels-by-hand/>

在将近整整 200 年的时间里，爱尔兰都柏林的吉尼斯啤酒厂雇佣了技艺高超的工匠来手工制作木桶。这些人被称为库珀，从事他们的行业需要几年的学徒期。库伯格是一个封闭的社会，因为许多职位是通过家族世代相传的。随着 20 世纪 50 年代后期铝桶和不锈钢桶的兴起，吉尼斯世界纪录的制桶大师成了濒临灭绝的物种。

本片中展示的合作过程的几乎每一步都没有经过任何精确的测量。像迪克·弗拉纳根这样的库珀大师只需要他的眼睛和经验丰富的判断力。他的酒桶最初是被称为“板条”的橡木板，已经在架子上干燥了至少两年。一个古铜色的人选择符合他心意的五线谱，然后锯掉末端。这似乎是该过程中唯一使用电动工具的部分。

铜匠用斧子和凿子手工制作每根板条，使其两端逐渐变细。一旦他把它们做成足够的形状，可以做成一个桶，他就把它们排列在一个圆筒里，围绕在一个叫做铁环的金属带的里面。捆绑好的板条被蒸半个小时，以使它们足够柔韧，便于成型。

蒸汽处理后，板条张开的一端用钢丝绳捆绑，将它们拉得足够近，这样就可以在它们上面安装一个铁环。然后用燃烧的橡木刨花烧焦木桶的内部，这个过程密封木材并去除其酸性。此后，打磨末端并钻[桶孔](http://www.merriam-webster.com/dictionary/bunghole)。

对于每个桶，库珀工艺定制一套箍。这些是在桶的外部被刮平后安装的。最后，木桶两端的顶盖由更多的橡木板制成，这些橡木板用传力杆固定在一起。这是库珀唯一一次使用工具来测量东西，他这样做是为了在头上获得合适的周长。他把边缘切成斜角，这样头部就能嵌入桶壁上钻出的凹槽。一旦他们坐下，小桶就准备好了黑色的，浓郁的黑啤酒。
