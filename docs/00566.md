# 攀爬机器人把自己的脚粘在墙上

> 原文:[https://hack aday . com/2012/05/29/climbing-robot-glues-its-own-foots-to-the-wall/](https://hackaday.com/2012/05/29/climbing-robot-glues-its-own-feet-to-the-wall/)

![](../Images/62173b1da2e87fbb6822d648dc4454fa.png "hot-glue-climbing-bot")

抓取所有表面的问题一直困扰着攀爬机器人领域。但是如果你不在乎损坏墙壁，为什么不干脆让机器人把它的脚粘在表面上。这正是这个机器人如何做到这一点，使用一对攀爬脚结合热熔胶枪挥舞手臂。它似乎是[的前身，也就是我们上周看到的](http://hackaday.com/2012/05/23/hot-glue-appendages-may-be-predecessor-to-the-flow-metal-of-the-t-1000/)的热熔胶 3D 打印机器人。

查看视频，全面了解攀爬过程，但这里是它的要点。腿和臂能够绕机器人的中心轴旋转，平行于攀爬表面。一旦一条腿粘合到位，机器人就会绕着它旋转，这样身体就在那条腿的正上方。从那里，它伸出手臂在墙上涂一些胶水，然后移开，这样另一只脚就可以压在热粘合剂上。当新附着的脚已经充分冷却时，下面的脚被重新加热以将其从壁上释放。此时，整个过程重复进行。

我们仍然喜欢基于真空的攀登者，比如这个装备降落伞的机器人。但是这个家伙身上的热胶手臂的设计可能有一天会成为我们自己的项目之一。
