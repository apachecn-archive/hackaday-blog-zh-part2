# 由吸管制成的触须和指骨

> 原文:[https://hack aday . com/2013/01/03/用吸管做成的触手指骨/](https://hackaday.com/2013/01/03/tentacles-and-phalanges-made-from-drinking-straws/)

![arnie](../Images/22028dafd2e8241ffd2fb7e51c904ff3.png)

人类的手是你见过的最令人印象深刻的机器之一——生物的、机械的或其他的。手有二十几个自由度，可以轻轻地抚摸最脆弱的花瓣而不会碰伤它，或者用巨大的力量将锤子敲进铁砧。然而，模拟人手是一项相当大的挑战，需要几十个伺服系统和复杂的机械联动装置。Instructables 上的[Tomdf]是[能够使用弹簧吸管和定制的 3d 打印关节创造手、触角和其他怪异的生物装置](http://www.instructables.com/id/Spring-Loaded-Drinking-Straw-Hinges/)。

[Tomdf]在看到一些 3D 打印的饮用吸管连接器后，有了饮用吸管指骨的想法，这些连接器旨在用于用一次性塑料管创建 3D 对象。在为吸管设计了一种新的弹簧接头后，[Tomdf]能够添加一些长度的线作为韧带来控制吸管的各个部分。这与我们去年在 Maker Faire 上看到的可怕的恶魔产卵的设置类似，但对于任何可能突然出现在你脑海中的项目来说，它都具有更大的可扩展性。

休息之后，你可以看看吸管触角的动作。
