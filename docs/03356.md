# LED 迫击炮板以 21W 的功率对抗太阳亮度

> 原文:[https://hack aday . com/2013/05/31/led-凡人-棋盘-战斗-太阳-亮度-5-6a-功率/](https://hackaday.com/2013/05/31/led-mortal-board-battles-suns-brightness-with-5-6a-of-power/)

[杰里米·布鲁姆]的目标是成为今年春天硕士毕业典礼上最聪明的学生。他为他的迫击炮板设计了一个 LED 钻机，它应该使用高达 21 瓦的功率来对抗太阳的强度。但他并没有以抢眼的强度停下来。当他这样做的时候，他还加入了一些互动功能，这样他身后的人就有办法避免失明。

真正吸引我们眼球的是他为这个项目制作的 3D 打印零件。有一个很好的 LED 安装板，还有一个安装在手腕上的树莓板外壳。等等，他为什么需要 RPi 来驱动一些 led？我们已经提到了作为 WiFi 热点的 Pi 所促进的交互性。连接到接入点并选择一种颜色。如果你坐在[杰里米]后面，你会选择黑色！所有这些在他的视频演示中都有解释。
