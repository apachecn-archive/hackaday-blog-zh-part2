# 这个蛋糕不是谎言

> 原文：<https://hackaday.com/2014/12/31/this-cake-is-not-a-lie/>

推出世界首创(？)可食用的**和**互动的 RGB 矩阵蛋糕——[the ArCake。](http://hackaday.io/project/3386-the-arcake)

几年前，我们的一位来自德国的读者[Treibair]受到了我们在 hack aday 制作的[LED 蛋糕的启发。我们使用了天使蛋糕方块，让 LED 灯从蛋糕下面穿过方块，LED 灯放在技术先进的蛋糕托盘中。这种方法效果很好，但是我们并不建议人们遵循我们的饮食步骤。](http://hackaday.com/2011/06/12/hacking-cakes-with-leds-the-sequel/)

然而，这并没有阻止[Treibair],他在蛋糕上想出了自己独特的花样！他有一个更直接的方法，而不是用各种各样的天使蛋糕。

这很容易做到，只需遵循以下步骤:

*   在蛋糕上钻几个孔
*   把你的果冻放进蛋糕里
*   让她打开盒子

这就是你做事的方式。

由此产生的 LED 扩散器让大量的光通过，同时保留了它们最重要的品质——美味。总之，他做了 30 个果冻填充的洞，这样他就可以在蛋糕下面放置一个 5 x 6 的 led 矩阵。现在，当他把蛋糕给妻子时，它会给她读生日快乐的信息，然后让她用 Wii 双节棍控制器玩跳跃游戏！
