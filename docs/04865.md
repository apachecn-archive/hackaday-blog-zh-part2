# DIY 激光切割机

> 原文:[https://hackaday.com/2013/12/20/diy-laser-cutter/](https://hackaday.com/2013/12/20/diy-laser-cutter/)

[Jens]决定尝试建造自己的激光切割机,看看用一个相当低功率的 300mW 激光二极管能切割多少。

他受到今年早些时候的一个[类似项目的启发，选择使用相同的 LPC-826 激光二极管，你可以在网上找到，价格约为 10-30 美元。切割机本身有一个木制框架，并在两个轴上使用抽屉滑轨。螺纹 M6 杆和 NEMA17 步进电机提供驱动，整个事情由 Arduino Nano 和](http://hackaday.com/2013/05/06/dvd-laser-diode-used-to-build-a-laser-engraver/) [Easy Driver](http://www.schmalzhaus.com/EasyDriver/) 步进电机驱动器控制。

那么它能切割什么呢？在他的实验中，他能够切断粘性塑料(贴纸)、EVA 泡沫和黑纸。他还能够雕刻木头和 ABS 塑料，尽管塑料与激光的配合不太好。他还发现它对激光切割模板很有用，然后他用盐酸和过氧化氢制作出生锈的艺术品。

考虑到制作一个这样的东西有多便宜，这是一个不错的工具。休息后留下来看它激光切割鲨鱼！
