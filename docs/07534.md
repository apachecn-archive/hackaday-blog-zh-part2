# RFToy 让无线项目变得更简单

> 原文:[https://hack aday . com/2014/11/10/rftoy-makes-wireless-projects-easy/](https://hackaday.com/2014/11/10/rftoy-makes-wireless-projects-easier/)

[Ray]已经创建了一个简单的小工具 [RFToy](http://rayshobby.net/?p=9938) ，它可以帮助建立带有各种常见无线电模块的无线系统。RFToy 是运行在 ATmega328 上的开源微控制器板。虽然 RFToy 与 Arduino 代码兼容，但[Ray]选择放弃熟悉的 Arduino 屏蔽布局，选择一种更容易安装 RF 模块、更便于手持的布局。

[RFToy]包括流行的 nRF24L01 2.4 GHz 收发器的接头，以及许多低成本无线电子设备中的 433/315 发射器和接收器。128×64 像素的有机发光二极管屏幕和 3 个按钮的界面使得设置简单的用户界面来测试新设计变得容易。

[雷]在这里没有任何新的突破。他所做的是为无线项目创造一个简单的工具。任何在无线系统上工作过的人都会告诉你，像这样的工具对于调试你的电路为什么不说话是无价的。是发报机吗？接收器呢。供电线路还有别的问题吗？

休息之后，请观看[Ray 的]演示视频。在里面，他嗅探、记录并回放来自几个遥控插座的信号。[Ray]还演示了如何使用 nRF24L01 来回发送温度数据。
