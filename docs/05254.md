# 双轮平衡机器人起死回生

> 原文:[https://hack aday . com/2014/02/07/双轮平衡机器人起死回生/](https://hackaday.com/2014/02/07/two-wheel-balancing-robot-revived-from-the-dead/)

![Capture](../Images/eac61b83adb91561ea3a9a02e2566ef7.png)

[Jouni]不久前造了一个非常漂亮的小两轮机器人——但他从来没有把它工作得很好。从这里提到的另一个黑客那里获得灵感和一点开源代码，[他已经完成了这个机器人，而且它工作得很好！](http://www.youtube.com/watch?v=HleYvRYjm-Y)

在看过[【Jose 的】3D 打印的空气曲棍球机器人后，](http://hackaday.com/2014/02/05/robot-dominates-air-hockey-frightens-john-connor-wayne-gretzky/)他在创造者的博客上戳来戳去，发现了 [B 机器人](http://cienciaycacharreo.blogspot.com.es/2013/10/b-robot-un-robot-equilibrista-impreso_28.html)，一个 3D 打印的双轮、步进驱动、平衡机器人。事实证明，它与[Jouni]之前自己制造的机器人极其相似！

[Jouni 的]机器人具有两个 NEMA-17 步进器、12v 2200mAh 电池组、Arduino Pro Mini、MPU6050 陀螺仪和 FrSky 接收器。幸运的是，[Jose 的] B 型机器人使用了相同的步进器和陀螺仪！使用来自 GitHub 的一些【Jose】的[代码，【Jouni】能够给他的小机器人带来新的生命！](https://github.com/JJulio/b-robot)

我们已经包括了原始项目和[Jouni 的]版本的视频。开源项目是不是很牛逼？
