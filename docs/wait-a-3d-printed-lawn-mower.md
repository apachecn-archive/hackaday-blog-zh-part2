# 等等，3D 打印割草机？

> 原文：<https://hackaday.com/2015/01/05/wait-a-3d-printed-lawn-mower/>

好吧，我们不得不承认，我们从未想到会有这一天…[3D 打印割草机？](http://3dprint.com/34873/3d-printed-lawn-mower/)什么？为什么？啊？怎么会？当我们看到这个出现在提示行时，这至少是我们脑海中的一些想法。

[Hans Fouche]有一台巨大的 3D 打印机，占据了他车库的大部分空间，在打印了几个大花瓶、一个公文包、碗，甚至一个风力涡轮机叶片之后，他决定尝试打印一台割草机。一台奇怪的割草机。

为了做到这一点，他逆向工程他的旧生锈的割草机，并重新设计成可打印的。除了钢轴，一些紧固硬件，当然还有马达和叶片，整个东西都是 3D 打印的。看起来效果也不错。

[https://www.youtube.com/embed/bKvRb5LJP9A?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/bKvRb5LJP9A?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

根据这篇文章，在过去的 18 年里，[Fouche]一直在 3D 打印东西——使用热熔胶，糖粉，甚至巧克力。但当他买了一台带有标准 0.5 毫米喷嘴的 3D 打印机时，他对它的速度感到失望，所以他决定自己制造一台。

他称这个巨大的 3D 打印机为猎豹，这是有道理的——它每小时能产出 1 公斤塑料。这相当于约 2.5-3 毫米的层高度，考虑到热端实际上处理塑料颗粒，而不是细丝，这是非常令人印象深刻的。他甚至计划在稍微整理一下设计后，生产和销售它。

大幅面 3D 打印无疑为你认为无法 3D 打印的东西开辟了一些有趣的可能性……毕竟，你还记得那个真人大小的 3D 打印混凝土城堡吗？
