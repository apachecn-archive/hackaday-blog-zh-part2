# DIY 酷雕塑家冷冻脂肪与冷冻脂肪分解

> 原文：<https://hackaday.com/2013/11/19/diy-coolsculptor-freezes-fat-with-cryolipolysis/>

你可能想知道为什么上图中的[Eddy]决定在他的躯干上固定两个 CPU 散热块。我们自己也有点担心。事实证明，[Eddy]已经成功建造了他自己的[冷冻脂肪分解装置](http://en.wikipedia.org/wiki/Cryolipolysis)，能够使用一种更广为人知的技术“酷塑”将目标零度以下的温度传递到身体的不同部位

冷冻脂肪分解是一种控制冷却的非手术方法，它将脂肪细胞暴露在低温下，同时还产生真空以限制血液流向治疗区域。[Eddy 的]挑战是发现治疗表面到底有多冷——这是最初的发明者严守的秘密。在翻遍原始专利并确定了-3C 和 0C 之间的范围后，[Eddy]开始拼凑这个医学杰作并设计一个能够控制它的系统。

他的成品包括一个简单的三按钮界面和随附的 LCD 屏幕，两者都连接到 Arduino，允许用户调节温度并记录会议时间。不幸的是，结果可能需要几个月才能出现，所以[Eddy]不知道他的创作是否可行(尽管遭受了冻伤和一些皮肤变色，哎呀！)你可以在[Eddy 的]项目博客上挑选大量的照片和详细信息，然后留下来观看澳大利亚新闻节目中解释酷雕塑过程的视频。需要一些额外的鼓励来在自己身上做实验吗？你可以随时[将一些电极绑在你的头上](http://hackaday.com/2013/05/28/shocking-your-brain-and-making-yourself-smarter/)并让电流通过它们。你知道，为了科学。
