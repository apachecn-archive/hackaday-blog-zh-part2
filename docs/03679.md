# 迪士尼原型将触觉反馈添加到手势界面

> 原文:[https://hack aday . com/2013/07/29/Disney-prototype-adds-haptic-feedback-to-gesture-interface/](https://hackaday.com/2013/07/29/disney-prototype-adds-haptic-feedback-to-gestural-interface/)

![disney-haptic-feedback](../Images/7bfc5285325e80634fb45c653a904c8b.png)

这个小设备大约是网络摄像头的大小，它以同样的方式栖息在你的电脑显示器上。这是迪士尼对手势输入的触觉反馈的解决方案。也就是说，在空中挥动你的手来控制一台电脑，这会给你一些实际触摸虚拟物体的感觉。

这东西向使用者喷射环形空气。我们认为视频中的足球演示是如何使用它的最好例子。正在玩一种游戏，其中虚拟足球向用户发射。这个装置喷射出一股空气，随着每个球前进。当你把手放在正确的位置时，你会感觉到空气的漩涡，并且知道你已经和虚拟物体接触了。

在硬件方面，这只是乞求在你的地下室重新创建。我们这里有一个有六个面的 3D 打印外壳。其中五个有扬声器元件，当给电信号时会产生压力波。当协调时，它们会产生一个完美的环形涡流(想想雪茄烟雾环)射出灵活的喷嘴，由于万向节的设置，喷嘴可以瞄准。当然，让它互动的元素是一个 3D 摄像头，在家庭作坊里制作时，它可以是 Kinect 或 T2 Leap Motion。
