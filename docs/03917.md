# 世界末日后制造钥匙

> 原文：<https://hackaday.com/2013/08/25/making-keys-after-the-apocalypse/>

制作钥匙是一门神奇的艺术，需要很多技巧和技术。对于那些生活在后启示录世界的人来说，[丹]有一个简单得多的解决方案来解决你的锁和插销的钥匙太少的问题-[只要用太阳的力量把它们从废料中扔出来](http://www.youtube.com/watch?v=iMfkgBvQ8ig)。

为了制作钥匙的模具，[丹]正在使用一个两件式的熟石膏模具。首先，在一个小容器里铺上一层厚厚的石膏，钥匙浮在表面上。干燥后，将浇口与粘土放在一起，并将钥匙嵌入固化石膏块中。几个小时后，一个合适的模具就造好了，可以接受熔化的金属。

铸造材料是锌——不如原来的钢钥匙坚硬，但强度足以打开一把锁。这种锌在一个装有巨大菲涅尔透镜的钢铁和石膏坩埚中熔化。

至于天启之后这种复制钥匙的方法的实用性，我们就不得不怀疑这种方法的实用性了。除非你挨家挨户地寻找投影电视，否则巨型菲涅尔透镜并不是你随机找到的东西，在世界末日后找到一罐脱模剂是难以置信的。也就是说，这是一个很酷的金属铸造演示，可以在家里或任何黑客空间轻松完成。
