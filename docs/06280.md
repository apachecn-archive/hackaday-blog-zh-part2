# 具有力反馈的数字弹球

> 原文：<https://hackaday.com/2014/06/10/digital-pinball-with-force-feedback/>

在 Hackaday 待久了，你就会听说 MAME，以及所有其他在电脑上模拟老式街机的方法。建造通常很棒，有真正的街机按钮、中密度纤维板橱柜和带有恰到好处的复古风格的侧面图形，让任何古代视频游戏的鉴赏家都感到高兴。然而，MAME 只是在模仿旧的视频游戏，而不是像在项目网站上安装的[数字弹球系统【罗尼德】这样的实体系统。](http://hackaday.io/project/714)

[ronnied]在之前的工作中受到现实生活中全尺寸白水弹球机的启发，并决定是时候以某种方式购买一台自己的弹球机了。他有一台备用电脑，一台旧的 16:9 显示器用于主游戏场，还有一台较小的 4:3 显示器用于后玻璃。通过一个 MDF 机柜、PinMAME 和一点点工作，[ronnied]拥有了自己的机器，能够重现数百台经典机器。

这一构建不仅仅停留在几个街机按钮和一个屏幕上；[ronnied]添加了一个用于倾斜机制的 3 轴加速度计，螺线管和一个从真实弹球机上撕下来的柱塞，以获得更真实的界面，以及一个用于非常响亮的触觉反馈的威廉姆斯敲门器。[在](http://hackaday.com/2013/09/22/how-to-make-pinmame-awesome/)之前，我们已经看到了弹球模拟器中的螺线管、蜂鸣器和敲门器，这些机电附件带来的振动和嗡嗡声使一切都变得不同；没有它们，这就像在电脑上玩弹球模拟器一样。有了它们，你很容易相信自己在玩一台真的机器。
