# 事实证明，无钥匙宝马汽车非常容易被盗

> 原文：<https://hackaday.com/2012/07/07/keyless-bmw-cars-prove-to-be-very-easy-to-steal/>

许多高端汽车现在都推出了可以解锁和启动汽车的射频遥控钥匙。不再有插入点火装置的物理钥匙。事实证明，对于宝马来说，这意味着对于一个老练的罪犯来说，偷车极其容易。我们一直喜欢金属钥匙上也有芯片的想法。双层安全系统对我们来说是有意义的，并且可以防止(或者至少减缓)最近在英国发生的一连串宝马盗窃事件。

所以事情是这样的。如上图所示的设备可以连接到车辆的车载诊断(ODB)端口。然后可以用它来编程一个新的遥控钥匙。这当然是更换丢失或损坏的设备的必要功能，但似乎犯罪分子已经找到了自己的方法。现在唯一困难的部分是进入车内而不触动警报。[根据这篇文章](http://jalopnik.com/5923802/watch-hackers-steal-a-bmw-in-three-minutes)里面有超声波传感器，用于探测入侵并使车辆无法移动。但不知何故这被绕过了。
