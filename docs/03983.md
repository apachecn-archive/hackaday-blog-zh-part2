# 尖叫快速遥控气垫船

> 原文：<https://hackaday.com/2013/09/04/screaming-fast-rc-hovercraft/>

在所有漂浮在水上的自制遥控气垫船中，[这款不仅外观更好，速度也出乎意料的快。[scratchbuildaircraft]从当地的一元店购买泡沫板来建造气垫船的主体和一个重型垃圾袋，垃圾袋的中心有一个孔，用来放裙子。空气从气垫船的 EDF(电动管道风扇——背面的大风扇)到达裙部区域，它通过底部的一个矩形孔泵送空气。](http://www.instructables.com/id/Very-Fast-RC-Hovercraft/)

安装在风扇后面的伺服系统控制方向舵，而其余的电子设备和电池则干净地藏在泡沫塑料机身下面。我们不知道 Turnigy 马达能提供什么样的最高速度，但假设它能保持不翻转，这可能会令人印象深刻。在下面的视频中，看着它带着太多的升力升空。

对于稍微慢一点的东西，今年夏天早些时候总会有太阳能气垫船。
