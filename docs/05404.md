# 六足机器人让人类和钱包感到恐惧

> 原文:[https://hack aday . com/2014/02/25/hexapod-robot-terrifies-humans-and-wallet/](https://hackaday.com/2014/02/25/hexapod-robot-terrifies-humans-and-wallets/)

![hexapod](../Images/aa0c336680bb6f448c98be6b59085c2c.png)

[凯文]带给我们[假人，他最新的机器人项目](http://forums.trossenrobotics.com/showthread.php?6725-Golem-MX-64-4dof-Hexapod-project.)。假人不是像与他同名的一样由泥土和石头制成[，而是由 T6 铝和伺服系统制成。我们没有香蕉作为尺度，但是假人很大。不是[杰米·曼策尔的]](http://en.wikipedia.org/wiki/Golem) [巨型机器人项目](http://hackaday.com/2013/08/08/six-years-a-giant-robot-and-a-kickstarter/)大，但是直径 2.5 英尺(76.2 厘米)，16 磅(7.3 公斤)，没有人会说假人是轻量级的。有了这样的质量，标准遥控伺服系统就没什么机会了。[凯文]全力以赴，为假人的腿找到了动力 MX64 伺服系统。光是这些伺服系统就让假人的成本远远超出了普通爱好者的预算。不过凯文还没说完。他添加了英特尔 [NUC](http://en.wikipedia.org/wiki/Next_Unit_of_Computing) 主板，配有第四代 i5 处理器、120 固态硬盘和 8g 内存。传感由陀螺仪、加速度计和机载罗盘模块处理。由于缺少 GPS，我们假设假人将主要在室内使用。我们肯定喜欢安装在假人背上的迷你低音炮。嘿，即使是机器人也有自己的音乐。

假人目前正在通过蓝牙配对的 Dualshock 3 控制器在人类控制下行走。[凯文]的目标是利用假人学习[机器人操作系统(ROS)](http://hackaday.com/2014/01/29/the-robot-operating-system-ros-101/) 。他已经安装了 ubuntu 13.04 并准备好了。[凯文]没有提到视觉系统，但基于[的其他一些机器人使用 Xtion pro live](https://www.youtube.com/watch?v=9iE3Yv2s6n4&)这一事实，我们充满希望。我们迫不及待地想看到假人自主的第一步。
