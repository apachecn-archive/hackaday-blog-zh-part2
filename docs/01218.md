# 机器人远程呈现机器人

> 原文:[https://hack aday . com/2012/08/02/NXT-Android-tele presence-robot/](https://hackaday.com/2012/08/02/nxt-android-telepresence-robot/)

![](../Images/cbccec05f3e814403252c93e32ff9171.png "nxt-android-telepresence-bot")

这是一个使用安卓设备和乐高 NXT 部件的远程呈现机器人。[沃尔夫冈]手头有一部多余的手机，他决定好好利用它。Mindstorm 的部件使得生产一个小型机器人变得非常容易，添加电话真的增加了他可用的计算和连接选项。

Android 设备能够通过蓝牙控制 NXT 机器人。[Wolfgang]没有详细说明这一部分，但你可以从另一个 Android 控制的 Mindstorm 项目中获得一些关于主题[的提示。[Wolfgang]希望在旅行时能够在家办理登机手续。他在 Android 设备上使用 nanohttpd 来提供一个简单的网络界面。它使用 HTML5 从手机的摄像头推送一张快照作为用户反馈，并提供一组方向箭头，让他驾驶机器人到处跑。](http://hackaday.com/2010/08/02/android-controlling-mindstorms-nxt/)

很明显，如果他离开太久，这东西会没电的。为了解决这个问题，他安装了一块电池，为 NXT 部件和手机供电。现在他只需要建立一个感应充电站，他就真的准备好了。
