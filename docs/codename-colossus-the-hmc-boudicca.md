# 代号巨像:HMC·布迪卡

> 原文：<https://hackaday.com/2015/06/25/codename-colossus-the-hmc-boudicca/>

[Machination Studio]的创始人[Michael Sng]想要创造一个与众不同的玩具系列。他最近完成了代号为“巨像”玩具生产线中的第一个生产原型:HMC·布迪卡。蛋形的 HMC·布迪卡像坦克一样，有着明显的合金弹头感觉，但称它为玩具几乎是一种伤害。

HMC·布迪卡身高超过 20 英寸。它由 400 多个零件组成，其中大部分是 3D 打印或激光切割的。内部零件是 FDM，而外部零件是 SLS 打印。这是一个以六足方式行走的运动部件，所以有许多伺服系统，电机，传感器和 led，这些都由 Arduino 控制。这个原型投入了大量的工作和对细节的关注。HMC·布迪卡的设计可以用十字螺丝刀轻松拆卸。电子元件都是插入式设备，因此在更换传感器或伺服系统时不需要焊接。

代号巨像是一个定制的玩具系列，本质上是手工制作的。每件作品都将像 Each 布迪卡一样单独手绘和组装。虽然网站上还没有公布官方价格，但我们认为这些价格不会便宜。事实上，该网站声明每件作品都将比之前的售价上涨[2%](http://machinationstudio.com/faq/),以帮助保持作品的价值并控制成本上涨。这可能会成为潜在买家争夺的焦点。它强调了[迈克尔的]哲学，即代号巨像是一件可收藏的艺术品，是大规模生产的对立面。

不考虑商业策略，我们有兴趣看到这个系列的任何其他设计。看着这些机器人像一支机器人军队一样行进会很有趣！
