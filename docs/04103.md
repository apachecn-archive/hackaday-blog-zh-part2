# 马里奥在树莓派的帮助下弹钢琴

> 原文:[https://hackaday . com/2013/09/20/Mario-plays-piano-with-little-help-from-raspberry-pi/](https://hackaday.com/2013/09/20/mario-plays-piano-with-a-little-help-from-raspberry-pi/)

![mario-piano](../Images/29d37f90668fbe2a779e1d0b831a887d.png)

[大卫]已经创建了他自己的现场机器人乐队,来演奏现场版本的 NES 游戏的音乐和音效。我们大多数在 80 年代和 90 年代长大的人都将任天堂游戏的音乐深深印在了脑海中。虽然多年来已经创作了一些[惊人的](http://ocremix.org/) [混音](http://www.videogameslive.com/)，【大卫】已经设法做了一些真正独特的东西。用一个仿真器、一些软件能力和一对树莓 pi 武装起来，[Dave]创建了一个在模拟仪器上播放游戏音乐和声音效果的系统。一架雅马哈 Disklavier 钢琴通过与树莓派的连接来处理大部分工作。打击乐由第二个 Pi 处理。小军鼓、木块和铃鼓都是由定制的螺线管设置驱动的。

转换过程都是在玩游戏的过程中进行的。[Dave]说这个过程在现场播放时有大约 1 秒的延迟，但我们确信可以通过一些软件调整来解决这个问题。

我们特别喜欢《超级马里奥兄弟 2》的加入，因为介绍音乐似乎是为钢琴而写的。以前也有离线转换，但听到(和看到)在真正的钢琴上“现场”演奏的原始音乐是很特别的。

关于声音的一个音符。如果听起来好像钢琴有时会“踩在自己身上”，那的确如此。钢琴同时处理背景音乐和音效。马里奥的跳跃和滑行声与音乐同在一个八度音阶内。
