# 按需回形针

> 原文:[https://hackaday.com/2015/07/07/on-demand-paper-clips/](https://hackaday.com/2015/07/07/on-demand-paper-clips/)

3D 打印机非常适合打印出你需要的零件或物品，但如果你用完了回形针，它们真的有帮助吗？是的，最重要也是最容易被忽视的弯曲金属紧固件如果不容易得到的话，会严重影响你的生活。这个问题有一个解决方案，它叫做[回形针 Maximizer 1.0](http://congruently25.rssing.com/chan-13367387/all_p3.html#item57) 。使用这种机器的唯一后果可能就是[毁灭人类](http://wiki.lesswrong.com/wiki/Paperclip_maximizer)。

这台机器把一卷金属丝有条不紊地弯成回形针的形状。就像 3D 打印机上的挤出机一样，有一个滚花驱动轮，带弹簧轴承，夹住电线。这个驱动轮是由一个钢筋混凝土伺服，已被修改为连续旋转。在驱动机构之后，金属丝穿过一个坚固的导向块。退出后，电线找到弯曲头，也由伺服驱动。弯曲头的末端有一个轴承，用于弯曲导块周围的焊线。弯曲几次形成回形针后，弯曲头左右摆动，用砂轮切断新制成的回形针。不幸的是，这个过程的这一部分并不好用。切断轮电机由控制整个机器的 Arduino 直接供电，其功率输出不足以轻松切断电线。它还会在切割的金属丝上留下尖锐的毛刺，这对于回形针来说不是一个很好的特征。但我们只是将这些视为黑客会议的未来素材！
