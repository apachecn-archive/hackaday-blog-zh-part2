# 本周败笔:吹风机当电灯开关

> 原文:[https://hacka day . com/2015/03/05/本周失败电吹风灯开关/](https://hackaday.com/2015/03/05/fail-of-the-week-hair-dryer-as-light-switch/)

家用电器，真的没那么难。但是当你处理别人留给你的难题时，事情会变得很奇怪。(丹尼尔的)姐姐和她丈夫最近遇到了这种情况。他们失败的视频演示中有很多过早的笑声，但值得坚持下去…你很快就会明白为什么她无法控制自己的娱乐。

手头的项目是用一个简单的灯具取代浴室风扇。交换完成后，电灯开关就像预期的那样工作了。但是第二个开关用来控制不同的灯，现在表现得很奇怪。它不会激活原来的灯，而是切换新的灯具。更奇怪的是，当第二个开关打开时，原来的开关显然现在充当了一个奇怪的调光器。这很奇怪，但失败的妙招是当他们插上吹风机并打开它点亮灯，但不启动吹风机。

和本周所有失败的部分一样，这里的目标不是批评而是同情。你认为这是什么原因造成的？我们迫不及待地想看看你想出了什么。鼓励张贴简单的图表(你可以在评论中使用 HTML img 标签)。女士们先生们，开始你们的猜想。
