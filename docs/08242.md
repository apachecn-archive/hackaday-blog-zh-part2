# 用复古风格为您的 PS4 增加额外的存储空间

> 原文:[https://hackaday . com/2015/02/05/add-extra-storage-to-your-ps4-with-retro-flair/](https://hackaday.com/2015/02/05/add-extra-storage-to-your-ps4-with-retro-flair/)

[弗兰克]想出了一个聪明的办法来扩展他的 PS4 的存储空间。他设法将他的数字 PS4 游戏存储在经典 NES 盒式磁带形状的存储设备中。从技术角度来看，这是一个相对简单的方法，但结果是一种有趣的存储数字游戏的方式。

他从设计自己的 NES 子弹 3D 模型开始。然后他在他的 [Ultimaker 3D 打印机](http://hackaday.com/2011/10/13/ultimaker-quality-faq-is-like-porn-for-3d-printers/ "Ultimaker")上打印墨盒。最终的印刷品是老式墨盒的高质量复制品。这种构建的诀窍在于，每个盒式磁带实际上都包含一个 2.5 英寸的硬盘驱动器。[Frank]可以将每个游戏存储在单独的驱动器上，将每个游戏放在单独的盒式磁带中。然后，他为这些当代游戏印制了自己的 80 年代风格标签。乍一看，你很难注意到这些游戏不是经典的 NES 游戏。

以盒式磁带的形式存储游戏是一回事，但将它们读入 PS4 又是另一回事。诀窍是使用连接到 PS4 主板的 SATA 连接器。[Frank 的]项目页面听起来好像他能够在不打开 PS4 的情况下插入 SATA 电缆，方法是将连接器连接到冰棒棒上，然后用它来插入连接器并将其插入到位。SATA 电缆的另一端进入定制的 3D 打印外壳，适合假冒的 NES 墨盒。该外壳使用机器螺钉固定在 PS4 的侧面。

现在[弗兰克]只需将他选择的墨盒滑入插槽，PS4 就能立即读取它。在这个时代，我们试图把越来越多的比特塞进越来越小的地方，这可能不是最实用的构建。但是有时候黑客并不实用。有时候只是为了开心。这个项目就是一个完美的例子。
