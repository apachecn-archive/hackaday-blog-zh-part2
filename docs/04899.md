# Fubarino 竞赛:FPGA Pong

> 原文:[https://hackaday.com/2013/12/26/fubarino-contest-fpga-pong/](https://hackaday.com/2013/12/26/fubarino-contest-fpga-pong/)

![pong](../Images/c547c41cd8ffeea181952486e66d3881.png)

为了让[Eric]参加我们的 Fubarino 竞赛，他使用了非常低级的硬件，并在 FPGA 上创建了 Pong。

[Eric]使用了 Basys 2 FPGA 板来创建这个虚拟的逻辑门版本的 Pong。输出是通过 VGA 端口，多人和人工智能播放器的实现，所有必要的 Pong 机制-碰撞检测，按钮和开关输入，以及记分也在这个项目中。

Fubarino 大赛当然需要一个复活节彩蛋，所以当左边玩家的分数达到 13，右边玩家的分数达到 37(懂了吗？1337?)，之前的方形球变成了极其像素化的 Hackaday 徽标版本。由于[Eric]的 [FP(V)GA 模块](https://github.com/MadLittleMods/FP-V-GA-Text)在他的 FPGA 板上显示文本，Hackaday URL 也被显示。

改进的 Pong 球和 URL 只在分数为 13-37 时出现，使这成为一个隐藏得非常好的复活节彩蛋。下面是[Eric]演示他的乒乓球的视频。
