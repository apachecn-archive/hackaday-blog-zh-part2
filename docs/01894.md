# 神奇的手指输入设备是一个摄像头在你的指尖

> 原文:[https://hackaday . com/2012/10/16/magic-finger-input-device-is-a-camera-on-your-finger-tip/](https://hackaday.com/2012/10/16/magic-finger-input-device-is-a-camera-on-your-finger-tip/)

![](../Images/66949753947fc459b1bf3f76917fbb48.png "magic-finger-input-device")

如果我们可以不用鼠标，只戴一个顶针作为控制界面会怎么样？这就是神奇手指背后的概念。[它为您的指尖增加了运动跟踪传感器和 RGB 摄像头](http://labs.blogs.com/its_alive_in_the_lab/2012/10/magic-finger-the-future-of-touch-interaction.html)。

触摸屏很棒，但是如果你想使用任何表面作为输入呢？然后你拿起今天最简单的标准输入设备:电脑鼠标。但是更进一步，考虑除了位置传感器之外，将鼠标用作图形输入设备的可能性。这个概念允许 Magic Finger 区分许多不同的材料。它知道你的桌子和一张纸的区别。此外，它还通过一种他们称之为微矩阵的编码方案打开了数据传输的大门。它就像一个超级小的二维码，由设备中的摄像头读取。

休息后发现的概念视频展示了该设备使用的许多很酷的技巧。我们最喜欢的是通过在设备背面移动手指来控制的平板电脑，而不是通过触摸屏幕来打断视线和留下指纹。
