# 杰纳斯:守门人

> 原文:[https://hackaday.com/2012/09/29/janus-the-gatekeeper/](https://hackaday.com/2012/09/29/janus-the-gatekeeper/)

[![janus-001](../Images/37389064474c589fae43f04569236e8b.png "janus-001")T2】](http://hackaday.com/2012/09/29/janus-the-gatekeeper/janus-001/)

[Piet]来信告诉我们他的黑客技术，允许他的前门在没有钥匙的情况下被打开。不像我们八月份介绍的[这个黑客](http://hackaday.com/2012/08/22/no-secret-knocks-required-at-steves-house-your-subway-pass-will-do/ "subway pass door opener")，你不需要地铁通行证，只需要一个好的记忆。正如在他的文章(和休息后的视频)中解释的那样，如果输入门铃响的正确顺序，大门会自动解锁。

对于硬件，a[【mehduino】](https://wiki.bitlair.nl/Pages/Projects/mehduino "mehduino")用于获取门铃输入，并决定是否实现了“秘密敲门”。门可以通过处理器上的按钮远程解锁。在“远程振铃器”按钮上输入代码时，只需按住编程按钮即可对代码进行重新编程。

请务必在休息后观看视频，了解这种锁定的效果。住房申请可能不完全是你所期望的。同样有趣的是，在真正的黑客时尚中，裸露的处理器被一个钩子挂在他的墙上！
