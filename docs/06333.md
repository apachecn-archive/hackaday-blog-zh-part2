# 用 Arduino 和 Android 触发远程焰火

> 原文:[https://hack aday . com/2014/06/17/trigging-remote-fireworks-with-an-arduino-and-an-Android/](https://hackaday.com/2014/06/17/triggering-remote-fireworks-with-an-arduino-and-an-android/)

![LIGHTING ROCKET](../Images/98775ccbe4e5c3bb9ce6aaecfdf509bf.png)

随着加拿大国庆日和独立日的迅速临近，一些制造商正期待着制作他们自己的烟花来庆祝——当然你可以使用火柴或打火机……或者你可以[拿出你可信赖的 Arduino 和手机！](http://www.sistemasorp.es/2014/06/16/encender-petardos-y-cohetes-con-arduino-y-android/) ( [译](https://translate.google.ca/translate?sl=auto&tl=en&js=y&prev=_t&hl=en&ie=UTF-8&u=http%3A%2F%2Fwww.sistemasorp.es%2F2014%2F06%2F16%2Fencender-petardos-y-cohetes-con-arduino-y-android%2F&edit-text=&act=url))

为了点燃保险丝，[Oscar]使用了一小段镍铬合金线，由 Arduino 通过 Mosfet 控制。为了控制 Arduino，他使用带蓝牙模块的 [ArduDroid](http://www.techbitar.com/ardudroid-simple-bluetooth-control-for-arduino-and-android.html) 。该应用程序允许您触发各种数字和模拟输出，并发送和接收数据。

留下来看看几个不同的演示视频的电路，测试，并发射一些小瓶火箭！
