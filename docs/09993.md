# Hams Talk 数字

> 原文:[https://hackaday.com/2015/09/06/hams-talk-digital/](https://hackaday.com/2015/09/06/hams-talk-digital/)

莫尔斯电码被认为是一种数字模式，尽管有机大脑比电子大脑更善于复制它。业余无线电爱好者做“电话”(业余无线电爱好者对语音的称呼)是从调幅开始的。二战后的某个时候，单边带或单边带被广泛采用。单边带占用的带宽更少，比调幅更可靠。在数字方面，hams 转向了不同的、更复杂的数字传输类型，计算机降低了带宽，提高了可靠性。然而，最近的趋势是通过业余无线电对语音进行编码——一种用无线电代替以太网的 VoIP 使用一种叫做 [freedv](https://freedv.org/tiki-index.php) 的开源程序。

[AA6E]制作了一个非常翔实的视频，他使用 freedv 与一个遥远的电台进行了一次 QSO(对话)。有趣的是，在接近尾声时，两个电视台切换到常规单边带。这种差异是巨大的，并真正说明了即使带宽更低(单边带约为 3 kHz，而单边带为 1.25 kHz)，数字模式也更优越。freedv 软件(可用于 Windows 或 Linux)将音频压缩到每秒 700-1600 比特，并将其分布在 16 个 QPSK 信号上。

即使你没有业余无线电执照(如果没有，为什么没有？)，看到使用数字方法传输语音等复杂信号与旧的模拟传输方式相比是多么有趣。
