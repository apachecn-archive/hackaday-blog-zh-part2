# 这辆机器人汽车不仅仅是看上去那么简单

> 原文:[https://hacka day . com/2012/05/08/这辆机器人汽车比你看到的要多/](https://hackaday.com/2012/05/08/theres-more-than-meets-the-eye-to-this-robot-car/)

![](../Images/c3b11fa0c3dcb2c15f70c5598659e8f5.png "rc-autorobot-video")

这是一个任何变形金刚爱好者都会喜欢的机器人。当然，这辆车看起来有点四四方方，但是当你看到它展开成一个两足机器人 ( [翻译](http://translate.google.com/translate?sl=auto&tl=en&js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&u=http%3A%2F%2Fwww.hobbymedia.it%2F37969%2Fautobot-transformer-il-robot-radiocomandato-in-grado-di-trasformarsi-in-unautomodello))时，你会忘记这一切[。【Zak Sawa】负责创作。他使用 22 个伺服电机完成了建造，让汽车变形，并提供了有点笨拙的行走能力。再把它折起来，汽车就可以开走了。换句话说，这就是你一直想要的变形金刚玩具；无线电控制，它实际上工作！](http://www.hobbymedia.it/37969/autobot-transformer-il-robot-radiocomandato-in-grado-di-trasformarsi-in-unautomodello)

这是大约四年工作的结果。显然，这是第八次迭代，在休息之后，视频上的一个注释预告了第九个版本正在进行中。不仅仅是机器人看起来很棒，看看存放它的手提箱。
