# 带有步进电机分线板的定制仪表

> 原文：<https://hackaday.com/2013/01/19/custom-gauges-with-a-stepper-motor-breakout-board/>

在一个项目上扔一些闪烁的发光二极管，它一定会成为 Hackaday 的头版。不过，我们确实喜欢模拟角色的构建，而[这个模拟规格步进电机分线板](https://tindie.com/shops/TheRengineer/analog-gauge-stepper-breakout-board/)看起来就像是让这些项目成为现实的门票。

这个项目背后的想法很简单:拿一个步进电机，在上面放一根针，然后把它连接到一个 Arduino 上。即时模拟仪表，测量 Arduino 可以计算的任何东西。

建造中使用的电机是 Switec X27.168，与几十种不同品牌和型号的数万辆汽车的仪表板上使用的电机相同。通过 Arduino 的[Guy Carpenter]Switec X25 库来控制电机，允许 Arduino Uno 同时控制多达三个步进电机仪表。

针的移动是惊人的平稳和相当快，正如在休息后的视频中看到的。如果你想要一个比 led 和 LCD 更模拟的显示器，这是一个非常酷的套件。
