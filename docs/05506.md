# 开膛手:一种不同的数控机床

> 原文：<https://hackaday.com/2014/03/13/the-ripper-a-different-kind-of-cnc-machine/>

这里有一个很棒的 CNC 构建，它跨越了一个标准的 CNC 路由器…带有一个能够轻松铣削金属的 CNC 机器。介绍 **[开膛手](http://maxmali.com/ripper-the-different-cnc/)** 。不，不是杰克。

[马克西米兰·马里]从小就开始每天阅读 Hack。几年前，他看到了一个 DIY CNC 构建指南，这启发他在 16 岁时就开始设计开膛手。他现在 19 岁(在奥地利学习机电一体化)，去年夏天筹集了足够的资金，终于制造出了他的第一个原型。它的造价约为 4000 欧元，与这种口径的商用机器相比只是九牛一毛。

该机器的床尺寸略大于一平方米，Z 高度为 225 毫米。它还足够坚硬，可以轻松地以 850 毫米/秒的速度切割铝！看看下面的视频，我们印象深刻。我们最喜欢的部分是当他向自己的 iPhone 屏幕投入一个工具来展示它的准确性和可重复性。
