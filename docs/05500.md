# 建造登山车

> 原文:[https://hackaday.com/2014/03/09/building-the-mountainbeest/](https://hackaday.com/2014/03/09/building-the-mountainbeest/)

![mounainbeest-pic](../Images/6d1df1151cbab74f58b41292aa77ff81.png)

一位朋友问杰出的建筑商、业余校友杰里米(Jeremy)今年是否要为他当地的制造商“做一些真正疯狂的事情”。Makerfaire 的时间不多了，他不想从头开始打造 awesome，于是向他的朋友推荐了几个去年暂时搁置的项目。获奖的未完成作品是 Mountainbeest，这是一款四足机械步行器，灵感来自于 Theo Jansen 的 Strandbeest。

在之前，我们已经看到了[登山运动的开始，从[杰里米]为一条腿建立连接开始。这个体型](http://hackaday.com/2013/06/13/building-a-strandbeest/)[变成了两条腿](http://www.jcopro.net/2013/06/27/progress-on-the-mountain-beest/)，现在它是一个完全的四足动物，理论上能够漫步在【杰瑞米】后院的郁郁葱葱的山上。

现在的计划是让[杰里米]借助一个失败的六足机器人遗留下来的挡风玻璃刮水器马达进行 Beest 行走。他还没有透露所有的细节，但看起来动力系统将由自行车零件制成。下面是项目当前状态的视频。
