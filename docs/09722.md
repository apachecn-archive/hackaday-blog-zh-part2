# 自制高压帽

> 原文：<https://hackaday.com/2015/08/02/homemade-high-voltage-caps/>

你身边有 15000 伏的电容器吗？[路德科学]没有，所以他做了下一个最好的事情。他建造了一些。

如果你理解电容器(由电介质隔开的两个平行导体)背后的物理原理，你就不会觉得它的制作过程非常令人惊讶。[Ludic]使用透明薄膜作为绝缘体，使用铝箔作为导电板。然后他把它们包在一个管子里。他确实提出了一些关于保持床单光滑以及如何将电线连接到箔片上的有趣技巧。棕色的包装纸让我们想起了你可能在古董收音机里找到的旧帽子。

不过，到目前为止，最精彩的部分是演示了在电路中有无电容的情况下，从高压电源拉起电弧。如你所料，玩充入电容器的几千伏电压需要一定的谨慎，所以要小心！

[Ludic]用标准仪表测量了电容值，但不清楚 15000 伏的额定值来自哪里。也许是他在视频中使用的电源和电容器实际上可以更高。

由此产生的自制电容器具有相当低的值(纳法拉)。你可以做很多，或者增加表面积以获得更大的容量。你也可以尝试找到更薄的电介质，尽管如果太薄，你自制的电容可能无法承受高压。如果你想要更大的电容(但可能没有那么大的电压)，你总是可以自制一个超级电容。
