# UTF-8——“最优雅的黑客”

> 原文:[https://hackaday . com/2013/09/27/utf-8-the-most-elegant-hack/](https://hackaday.com/2013/09/27/utf-8-the-most-elegant-hack/)

虽然看起来可能不太像，但上面的图片是原始电子邮件的一部分，其中[Ken Thompson]描述了 UTF-8 的实施情况。在美国计算机时代的初期，当我们还在使用电传打字机时，对英语进行编码是我们所担心的。程序员标准化了 ASCII 字符集，但是没有空间容纳其他语言中使用的所有字符。为了实现实时全球通信，我们需要更好的东西。有许多提议，但由[肯汤普森]和[罗布'指挥官'派克]提交的一个被接受，很可能是因为它是一个多么美丽的黑客。

[汤姆·斯科特]出色地描述了 UTF-8。我们不明白他为什么选择在一个繁忙的咖啡馆里解释这件事，但他的热情绝对能胜任这项任务。在视频中(插播在广告之后),他很快展示了 ASCII 的简单和天才。然后，他解释了支持这么多字符集的挑战，以及为什么 UTF-8 有这么大的意义。

我们考虑过让这成为一个逆向技术，但是共识是理解 UTF-8 是如何产生的对现代黑客和程序员是有用的。如果你有兴趣了解更多，在这个 Reddit 帖子的[中有大量链接，包括一个到](http://www.reddit.com/r/programming/comments/1mx7v5/utf8_the_most_beautiful_hack/)[原始电子邮件](http://www.cl.cam.ac.uk/~mgk25/ucs/utf-8-history.txt)的链接。
