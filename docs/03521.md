# 有吊扇的子弹时间

> 原文:[https://hackaday . com/2013/07/02/bullet-time-with-a-ceiling-fan/](https://hackaday.com/2013/07/02/bullet-time-with-a-ceiling-fan/)

![freaking awesome](../Images/78a80154208f8642cd230e525aa4ff3f.png)

子弹时间至少从第一部 *Matrix* 电影开始就已经存在了(实际上在那之前有一个 Gap 广告)，尽管它是一种经常使用的电影技术，它仍然没有过时。[Jeremiah]想要挖掘子弹时代的美妙之处，而[只用一台 GoPro 和一台吊扇就想出了一个很棒的摄影设备](http://jeremiahwarren.com/blog/2013/06/30/how-to-create-a-diy-matrix-bullet-timetime-slice-rig-using-a-ceiling-fan-and-a-gopro/)。

这个建筑实际上只依赖于两个组件:一台 GoPro 相机和一台吊扇。在[Jeremiah]的视频中，一个吊扇安装在两棵树之间的一块结实的木板上。GoPro 借助一块木头、一个铰链和一小段电缆悬挂在一个风扇叶片上。在[耶利米]把风扇和调光开关连接起来后，他就可以控制风扇的速度了。

这不是 GoPro 第一次被用于子弹时间装备。事实上，我们的朋友[Caleb]也做了一个类似的构建，他将镜头转向一只懒惰的 suzan。肯定喜欢 GoPro 的高帧率吧？

休息后的维迪阿斯。
