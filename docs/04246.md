# LED 引导的钢琴教学

> 原文:[https://hack aday . com/2013/10/08/led-guided-piano-instruction/](https://hackaday.com/2013/10/08/led-guided-piano-instruction/)

![LEDpianoGuide](../Images/8c1f0802bde75490448847789cc8c266.png)

[凯崔]不会弹钢琴。更确切地说，他不能，直到他把他的键盘转换成包含 LED 引导指令的键盘。[Kay]是一名微生物工程研究生，研究生最负担不起的就是私人音乐课。然而，凭借价值 70 美元的零部件和一部手机，他可能找到了一个临时的替代品。

该建筑就像一个瘦身后的现实世界吉他英雄，依次点亮每个音符。在之前，我们已经见过[这样的项目，led 安装在按键上方。然而，凯的设计更容易理解。他将发光二极管直接嵌入按键，包括每个黑键上方的发光二极管，以指示升半音。一个 Android 应用程序获取你选择的 MIDI 文件并解析数据，通过 USB OTG 将结果比特发送到](http://hackaday.com/2010/04/26/midi-piano-instructor/) [IOIO 板](http://hackaday.com/2013/01/25/new-ioio-is-faster-cheaper/ "New IOIO is faster, cheaper")。然后，一组移位寄存器驱动发光二极管。

对于一个完全的新手来说，[凯]似乎受益于这些灯。我们不确定 led 是否给出了预期音符的任何指示，但是，看起来他是在每个音符点亮后按下按键的。看看下面他的视频演示，帮助我们推测红灯意味着什么。如果你是一个电子专家，想在生活中一天都不练习的情况下创作音乐，我们建议你试试[【弗拉基米尔】的机器人吉他](http://hackaday.com/2013/08/26/update-vladimirs-robot-guitar/ "[Update] Vladimir’s Robot Guitar")。
