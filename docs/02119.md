# 告诉你天气预报的伞架

> 原文:[https://hack aday . com/2012/11/15/an-umbrella-stand-that-tell-you-the-weather-forecast/](https://hackaday.com/2012/11/15/an-umbrella-stand-that-tells-you-the-weather-forecast/)

![](../Images/39741eb3b53461aa6fcdd0f6a07f2038.png "umbrella-stand-tells-you-the-weather")

这个项目采用了一个伞架，并让它能够[让你知道当你离开家时是否需要带伞](http://openpicus.blogspot.it/2012/11/diy-umbrella-stand-hack-with-rain-alert.html)。上面的图片是一个概念图，但第一个原型已经建成，似乎工作得很好。休息后在视频里自己看吧。

这个项目是由 openPicus 组织的。他们出售一种叫做 Flyport 的原型模块，可以为你的项目提供 WiFi 连接。该板连接到一组发光二极管，用于照亮半透明的塑料伞架。但是如果 led 一直亮着，您可能不会注意到颜色的变化。该系统还设计了一个 PIR 运动传感器。当你走向门口准备离开一天时，它会切换到合适的颜色——绿色代表晴朗，蓝色代表下雨，红色代表暴风雨——在你经过时及时引起你的注意，拿起雨伞。

你不需要花一大笔钱来完成这样的黑客攻击。你可以在垃圾中寻找一个 PIR 传感器，使用一种颜色的发光二极管来告诉你什么时候会下雨或有暴风雨，而 T2 是一种廉价而简单的以太网替代物。
