# 使用简单的相机黑客技术，永远不要错过路边拍照。

当你在高速公路上开了几天车，你会看到一些有趣的事情。如果你像我一样，你通常没有时间拿出你的相机拍一张照片。尤其是如果它只是一辆看起来很傻的车，或者一棵有趣的树或其他什么东西。这款软件可以让你按下按钮就能拍下公路上的风景，让你轻松拍照。

首先，我采购了一些相机。沃尔玛有一批价格不到 30 美元的相当便宜的数码相机。买了两台 710 万像素的 vivitars。把它们从包装中取出来，拍了几张样片，很明显照片质量不会很好。这些东西的光学元件非常便宜。它将为这个项目工作得很好。

我开始计划各种方法来触发这些。最理想的情况是，它们将永远在线，以获得更快的响应时间。然而，我意识到，对于这个项目来说，我考虑的所有方法都过于复杂。所有真正需要的是延长电源和快门按钮。我可以在 3 秒钟内打开一个并抓拍一张照片。

在焊接了一些扩展器并将其连接到一个接线盒后，我用它们的包装和一些小吸盘为它们制作了简单的线束。这应该能让他们在这次旅行中保持足够好的位置。我在镇上拍了一些照片，结果还可以接受。没什么了不起的，但足以让人看到那辆满载悲伤小丑的校车在他们睡觉时驶过。
