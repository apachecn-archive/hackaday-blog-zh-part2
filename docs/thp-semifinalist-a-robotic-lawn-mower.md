# THP 半决赛选手:机器人割草机

> 原文：<https://hackaday.com/2014/09/27/thp-semifinalist-a-robotic-lawn-mower/>

对于世界上所有的 Roombas，你一定想知道为什么机器人割草机不更普遍。当然，你可以出去买一个，但是割草对机器人来说是小菜一碟；没有楼梯，障碍物相对较少，边界线系统比简单地弹跳到像 iRobot 这样的东西上要简单得多。

[【schu humi】的 autoCut](http://hackaday.io/project/1797) 是唯一进入 Hackaday 奖半决赛的家用机器人。在下面，这个机器人是电动的，具有全自动操作，甚至有一个电机来改变叶片的高度。刀片实际上设计得更像一台无绳除草机；当遇到坚硬的障碍物时，叶片会枢转回来，尽管[这个安全笼](http://hackaday.io/project/1797/log/4710-a-thought-on-safety)确实是个好主意

与 roomba 中的随机“碰撞和转向”算法不同，这个机器人在导航方面花了很多心思。[schuhumi] [正在使用超声波导航](http://hackaday.io/project/1797/log/9058-thoughts-on-the-ultrasonic-navigation-system)对机器人在院子里的位置进行三角测量。这是个好主意。没有必要浪费时间或精力去考虑机器人已经砍掉的部分。

你可以看看[schuhumi]的概述视频和下面的演示。
