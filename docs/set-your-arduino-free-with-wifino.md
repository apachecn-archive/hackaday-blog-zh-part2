# 通过 Wifino 释放您的 Arduino

> 原文：<https://hackaday.com/2012/07/06/set-your-arduino-free-with-wifino/>

![](img/3c9561034bda405cfef4ad04e34ec3ad.png "HeyListen")

[托德]来信告诉我们他最近的项目。它被称为 wifino ，旨在通过提供基于网络的 IDE、Arduino 草图的在线存储，甚至支持 WiFi 的硬件来无线上传草图，从而解放你的 Arduino。

wifino 的构思思路与我们本周早些时候看到的 codebender IDE 的思路相同。wifino 不仅提供基于网络的编程环境，还与 [wifino 硬件](http://www.wifino.com/?page_id=44)协同工作，这意味着你可以通过无线连接上传草图。

在软件方面， [wifino IDE](http://www.wifino.com/?page_id=40) 具有代码编辑、编译和从浏览器上传的功能。github 风格的界面计划正在进行中，允许来自世界各地的制造商轻松共享 Arduino 草图。

[Tod]计划在未来几周内启动 Kickstarter，将 wifino 板投入使用。我们一定会让你随时了解我们的任何信息或规格。

休息过后，您可以从 Windows 和 iOS 客户端查看[Tod]上传代码。
