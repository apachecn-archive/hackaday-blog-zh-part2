# 性感的六轴机械臂是一件艺术品

> 原文:[https://hack aday . com/2012/09/10/sexy-six-axis-robotic-arm-is-a-work-of-art/](https://hackaday.com/2012/09/10/sexy-six-axis-robotic-arm-is-a-work-of-art/)

![](../Images/a82a611986d4b9c5355d36482534a836.png "arm")

我们不知道这是中年危机还是太多焊接烟雾的结果，但 Instructables 公司的[sparten11]正在建造我们见过的最酷的机械臂之一，为此我们感谢他。

建造始于一组有刷 DC 电机，能够在 60 伏、高达 8 安培的电压下运行。这些马达与旋转编码器相连，旋转编码器与齿轮[sparten11]一起使用，每转提供 400，000 步。结合重型电机控制器，【sparten】的手臂对任何工业过程都有足够的动力和控制。

当然，没有骨骼和大脑，肌肉是没有用的，所以[sparten]在他家的机器店里加工他手臂的结构和机械部件。这是一套令人印象深刻的装备；他的机器人底座测试了他的车床的能力，喷水切割的手臂形成了一个异常强大的机器人手臂的优美骨架。用于构建的电子设备包括一台运行 Windows XP 的微型 PC，带有从覆铜板蚀刻而成的伺服控制板。

构建还没有完全完成，但从休息后的视频来看，[sparten]将在短期内拥有一个令人难以置信的机器人手臂在他的工作室里走动。
