# 复古技术:昔日神奇的后院发明

> 原文：<https://hackaday.com/2015/07/28/retrotechtacular-fantastic-backyard-inventions-of-yore/>

新闻集团[British Pathé]在其 60 年的历史中创作了许多新闻短片和纪录片。最近，该公司从他们的档案中发布了几十部电影，并把它们放到了互联网上。这里是他们创作的一组令人愉快的短片,突出了包括交通和通信在内的各个领域的奇怪和奇妙的发明。

最突出的发明之一是 [Dynasphere](https://www.youtube.com/watch?v=rvA7hDh_vhY) ，这是一种单轮车辆，可能值得在复古聚光灯下呆上一周。有几架踏板驱动的飞机可能激发了 Gossamer Condor 的灵感，还有一辆类似空气曲棍球前锋的悬停滑板车，它还可以兼作吹叶机。在另一部电影中，一名男子驾驶一辆 Vespa 来到泰晤士河畔，将车停在那里。他把滑板车两边的鳍拉下来，把它变成了一艘航海船。当他打响指时，一个可爱的女孩从画面外的某个地方出现。她爬到背上，然后他们一起离开水面。

这些电影的平均放映时间大约是两分钟。其中一些更短，引发了许多问题。幸运的是，大多数视频描述都有关于这些神奇发明的更多信息的链接。这些电影中几乎所有的发明家都表现出对安全的完全漠视，但几乎所有参与其中的人似乎都过得很开心。
