# 男人用电弧反应器向女友求婚

> 原文:[https://hack aday . com/2012/12/19/男子用电弧反应器向女友求婚/](https://hackaday.com/2012/12/19/man-proposes-to-girlfriend-with-an-arc-reactor/)

![arc](../Images/875d350038da9230927a455bb5e5e423.png)

经过长时间的努力思考，如何向他的女朋友求婚，[Ed]想到了一个好主意:使用一个电弧反应器作为戒指盒，有足够的 LED 灯来照亮除了他们的爱情之外的一切，并有伺服系统来呈现戒指并拨动心弦。

[Ed]着手从他的 arc 反应堆心脏中给他现在的未婚夫建造一个简单的圆形排列的 adafruit RGB LED 灯和一个 Arduino。这个电弧反应堆有两种模式:一种点亮模式，看起来很棒，一种“戒指模式”，使用两个伺服系统打开前盖，将订婚戒指带入视野。

在[Ed]的未婚夫同意后，电弧反应堆中间的盖子合上，继续作为桌面装饰品使用。休息过后，你可以看看[Ed]的提议装置。
