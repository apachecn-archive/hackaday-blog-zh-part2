# 建造倾转旋翼无人机

> 原文：<https://hackaday.com/2012/10/04/building-a-tilt-rotor-uav/>

我们在这里看到很多四轴飞行器，甚至偶尔会看到八轴飞行器。但是这个建筑只用了两个螺旋桨。这是一个倾斜旋翼的构造，它允许两个向上的螺旋桨向前和向后倾斜。这种无人机设计在现实世界中的对等物是 [V-22 鱼鹰](http://en.wikipedia.org/wiki/V-22_Osprey)。

电机安装在垂直于行驶方向的横梁上。它们中的每一个都安装在一个可以由伺服电机旋转的轴承上。它们彼此独立旋转，这允许偏航。当然，滚动是通过以不同的速度驱动螺旋桨来控制的，俯仰是通过同时倾斜两个支架来调整的。

[Stephen]提到倾转旋翼机的设计比使用两个以上支柱的相关设计有几个优点。这种设计建造成本更低，用电量更少，并且更容易实现自动驾驶。如果你想在空中看到它，不要错过休息后的剪辑。

这只是我们记得看到的第二个倾转旋翼机。另一个是万圣节道具。
