# 新的和改进的钢丝弯曲机

> 原文：<https://hackaday.com/2012/09/29/new-and-improved-diwire-bender/>

[https://www.youtube.com/embed/xrr4peibbNI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/xrr4peibbNI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

我们去年夏天看到的 DIWire bender 今年在 Maker Faire 上推出了新的改进版本，现在能够比以前的铝弯曲版本更准确地弯曲钢铁。

我和[Marco Perry]聊了起来(显然是上图)，他解释了 DIWire bender 的一个新功能，这个功能使得用 CNC 弯曲钢丝制作的锡焊或焊接作品变得更加容易。这台新机器有一个锋利的特点，可以精确地标记电线应该焊接的地方。

DIWire 背后的人们——Pensa——在 Maker Faire 的展位上有一吨非常酷的钢丝雕塑，非常偶然地在 Hackaday 的红牛展位附近。如果有机会，一定要去看看。
