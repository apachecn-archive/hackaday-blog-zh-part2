这款网真机器人绝不会让您的 Skype 来电者偷偷靠近您。[Priit]建立了这个项目，他称之为 Skype Got Legs，这样他的远方朋友就可以在聊天时跟随他在家里四处走动。但正如你在休息后听到的，用来机动化底座的电钻声音非常大。

除了噪音污染，我们喜欢粗糙的外壳。这很实用，但似乎效果很好。使用 Ajax 和 PHP 函数调用的组合通过 web 发送命令。这两个钻头由 Arduino 通过几个汽车继电器控制。这些电钻由原装充电电池组供电。为了不改变这些电池，[Priit]想出了一种方法，使用合成酒瓶软木塞作为连接器。它们被切割成一定的尺寸，并有镀锡的电线穿过其中的孔。现在，当他插入改变后的软木塞时，他们将电线压在电池触点上。[https://www.youtube.com/embed/EKpRiLlYS-g?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/EKpRiLlYS-g?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)T3】
