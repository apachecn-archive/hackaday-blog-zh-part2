# 全球辐射监测网络更新

> 原文：<https://hackaday.com/2014/09/11/global-radiation-monitoring-network-update/>

事情一直忙到[全球辐射监测网络中央指挥部](http://hackaday.io/project/1662)。作为 Hackaday 奖的半决赛选手，项目创建者[【Radu moti San】](http://hackaday.com/2014/07/03/thp-hacker-bio-radu-motisan/)有相当多的工作要做。他也没有偷懒。凭借 33 个项目日志(还在增加)，[Radu]一直让我们了解他的监控网络和 uRADMonitor(实际的监控硬件)的最新进展。

[Radu]的最新消息是，他已经准备好生产型号为 A 的 uRADMonitor。从项目到生产的过程可能会有大量的工作，因为要采购零件、建立装配车间，还要处理过程中出现的各种问题。不过，我们确信[拉杜]能处理好。

城市监测网络也在发展。亚利桑那州的普雷斯科特刚刚安装了一台新的监视器。这是美国的第十个单元。你可以在 [uRADMonitor 网站](http://www.uradmonitor.com/)上实时查看全球辐射的地图、数据和图表。
