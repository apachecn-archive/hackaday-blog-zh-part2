# 肯德鸡感应炉

> 原文：<https://hackaday.com/2014/12/02/kentucky-fried-induction-furnace/>

[约翰]和[马修]建造了一个基于感应加热器的炉子，并用它在厨房里为[制作美味的铝杯子蛋糕](http://www.johndearmond.com/2014/11/09/using-the-induction-heater-aluminum-melting-part-2/)。为什么要感应加热？因为它很节能，不会像燃料炉那样产生烟雾。为什么要在厨房里熔铝？我们猜测他们这么做只是因为他们可以。当然还有一个[的视频](https://www.youtube.com/watch?v=o--6HH8Nxgg)，下面休息一下，记录他们的第一次倒酒。

现在不要被这里采用的部分低技术的方法误导。尽管被铸造在一个大的肯德基桶里，这个迷你铸造厂被很好地组合在一起，并且关于它是如何被建造的写得非常好。DIY 感应加热器也是一项重要的业务，它的温度和气流通过机箱的散热器受到监控。这是一件非常好的事情，因为高电压和高热的结合需要一点尊重。

不管怎样，我们花了很长时间在[约翰]的网站上搜寻。如果你对感应加热器感兴趣的话，可以得到很多有用的信息。尽管如此，我们将在后院进行金属铸造。
