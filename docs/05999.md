# 用锐利度给 3D 打印着色

> 原文：<https://hackaday.com/2014/05/08/coloring-3d-prints-with-sharpies/>

轻松地全彩色打印物体是“喷射塑料”3D 打印机场景的首要目标之一，到目前为止，所有的实验都依赖于多种颜色的细丝，有时还依赖于多个挤出机。当然，这需要一批不同颜色的细丝，但是[马修·毕比]有一个不同的想法:[为什么不在一根天然颜色的细丝](http://3dprint.com/3340/ulimate-filament-colorer/)进入打印机之前给它染色呢？根据他的直觉，[Mathew]正在用普通的夏普记号笔做一些实验，结果打印出来的效果比你想象的要好得多。

这种技术的基本步骤是在尖头的粗端钻一个孔，拉出尖头的毛毡，在进入挤压机之前将一段细丝穿过标记器。细丝被 Sharpie 墨水染色，并且最终的打印保留了标记的颜色。

尽管这项技术很简单，但结果是惊人的。一根米色的“天然”细丝很容易变成夏普斯中的任何一种颜色。

除了常见的 Sharpie，这种 3D 打印机细丝着色技术还有一个稍微有趣的应用；任何在宿舍里用过黑光灯的人都知道，你可以用普通荧光笔里的染料制作一些酷炫的紫外线敏感酒瓶。“夏普技巧”是否适用于荧光笔或其他记号笔还不得而知，但至少值得尝试一下。
