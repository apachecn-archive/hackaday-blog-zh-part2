# 你能想象的任何颜色的 LED 云灯

> 原文:[https://hack aday . com/2013/01/17/led-cloud-lamp-in-any-color-you-can-image/](https://hackaday.com/2013/01/17/led-cloud-lamp-in-any-color-you-can-image/)

![rgb-cloud-lamp](../Images/f85f237fa9d74f48ebaf72a55745321c.png)

这款[Dablondeemu]打造的灯将为你的家居装饰增添一点奇思妙想。这个项目最初是一个数字艺术和装置课程的作业。但是，遥控变色云最终成为了她弟弟的一份漂亮的礼物。

该原型使用 Arduino、试验板和一组 led 来执行其任务。承认下一个版本应该有一个独立的电路板。电子元件被放置在一个透明的塑料容器中，然后用多填充物进行装饰，这种填充物通常可以在装饰枕头中找到。聚酯纤维在过滤和漫射光线方面做得很好。但它们似乎不会干扰遥控器发出的红外信号。

如果你喜欢创意形状的扩散器，你应该看看这个巨大的 LED 灯。它被塑造成看起来像一个通孔封装，导线隐藏着电源线。
