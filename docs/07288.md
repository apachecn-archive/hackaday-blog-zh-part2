# 机器人驱动的推车上的延时摄影

> 原文:[https://hack aday . com/2014/10/11/time lapse-photography-on-an-Android-powered-dolly/](https://hackaday.com/2014/10/11/timelapse-photography-on-an-android-powered-dolly/)

如果你要去阿拉斯加旅行，你需要确保你手头有足够的物资来应对等待你的荒野。如果你是[布莱斯]，那份供应清单包括一些有趣的摄影设备，[包括一个他制作的相机推车，用来拍摄美妙风景的延时视频](http://raspberrywood.com/android-timelapse-dolly/)。

在硬件方面，小车将摄像机安装在一个倾斜的轨道上。摄像机从一侧开始，向上朝另一侧移动，这在延时拍摄中创造了一种独特的效果。该装置由步进电机驱动，安装在一些漂亮的轴承上。布莱斯计划使用的两台相机是佳能 T2i 和 EOS-M，它们位于三脚架的顶部。

软件和电子方面也很有趣。与通常的 Arduino 不同，[Bryce]选择通过 Android 和 IOIO 板控制钻机。这为该项目提供了很多通信选择，包括蓝牙。整个装置由一个 19V 电池组供电。如果你在寻找更简单的东西，你可能会想试试[鸡蛋定时器来计时](http://hackaday.com/2013/06/04/gopro-panning-time-lapse-with-ikea-egg-timer/)！休息之后，请查看[Bryce]的钻机运行视频。
