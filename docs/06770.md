# IPad 在 Mac Classic 中找到新家

> 原文:[https://hack aday . com/2014/08/08/ipad-finds-new-home-in-MAC-classic/](https://hackaday.com/2014/08/08/ipad-finds-new-home-in-mac-classic/)

![](../Images/c106ab1d3e76a494f23c947ee570800b.png)

我们谁没有备用的 iPad 和 Mac Classic 呢？如果你是那些幸运的人之一，那么这个项目是给你的。[site hirac]用 Mac 经典的外壳([翻译](https://translate.google.com/translate?sl=ja&tl=en&js=y&prev=_t&hl=en&ie=UTF-8&u=http%3A%2F%2Fhirac.info%2Fsite%2Farchives%2Fmacintosh-ipad-stand%2F&edit-text=&act=url))为 iPad 做了一个非常整洁的[支架。碰巧 Mac Classic 和 iPad 的屏幕尺寸非常接近。虽然屏幕尺寸差不多，但是分辨率不一样。最初的 Macintosh Classic 有一个分辨率为 512 × 342 像素的黑白屏幕。iPad 的分辨率为 1024 x 768 像素，比最初的 Mac 多 450%的像素。](http://hirac.info/site/archives/macintosh-ipad-stand/)

为了让 iPad 更合适，必须对外壳进行重大修改。首先，Mac 的所有内部部件都被移除，只留下一个空盒子。机箱的前面板被移除，并在左侧制作了一个插槽。这个插槽有助于让 iPad 滑入 Mac。在前面板的内侧，相当多的注塑支架被修剪掉以留出空间。在后半壳的左侧也切了一个槽。当外壳重新组装时，前半部分和后半部分的插槽为 iPad 提供了足够大的孔。奇怪的是，前面板上有一些塑料部件，它们的高度正好可以将 iPad 固定在理想的位置，与外壳中的屏幕切口对齐。

如果你渴望建造一个的话，所有的案例模式细节都有很好的记录。这已经是一个相当不错的项目了，但如果我们必须提出任何建议，那就是增加一个对接连接器和一些集成扬声器。就像这个项目现在的情况一样，iPad 用电池供电，iPad 的扬声器面对着箱子。
