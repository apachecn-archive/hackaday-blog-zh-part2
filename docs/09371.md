# 无人驾驶飞机现在可以载人

> 原文:[https://hack aday . com/2015/06/24/autonomous-drones-now-carry-people/](https://hackaday.com/2015/06/24/autonomous-drones-now-carry-people/)

有几家公司试图制造第一辆自动驾驶汽车，但这个项目让我们认为他们可能都在朝着错误的方向前进。[Thorstin]想用四轴飞行器来运送人，并建造了一个自动四轴飞行器的工作原型，它实际上能够提升一个人。

这个设备实际上不再是四轴飞行器了；那将无法产生足够的升力。它总共有十六个旋翼，使它成为一架十六旋翼直升机(我们假设)。这种设置产生 282 磅的静态推力，正如下面的视频所示，足以将一个普通人连同铝合金框架和所有用于为所有这些电机提供动力的锂离子电池一起抬离地面。

PID 控制系统就位后，设备就可以起飞了！我们喜欢突然变得和真人一样大、可以骑行的业余项目，我们也希望在某个时候看到这个项目完全自主。也许很快我们就会看到人们从一个地点摆渡到另一个地点，而不是开着他们的无人驾驶汽车到处跑。
