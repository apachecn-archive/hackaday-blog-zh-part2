# 牙仙变成了气动的

> 原文:[https://hackaday.com/2013/09/13/tooth-fairy-goes-pneumatic/](https://hackaday.com/2013/09/13/tooth-fairy-goes-pneumatic/)

![pneumaticToothFairy](../Images/581210a35019a55ca2cf94f4f99980da.png)

孩子们的幻想人物早就应该进行一些技术升级了，所以 MAKE[Jeff high Smith]在墙上挖了几个洞，[建造了一个气动运输系统](http://makezine.com/video/making-fun-pneumatic-transport-for-tooth-fairy/)让他的孩子把他们的牙齿送到牙仙子那里。该项目使用一个 1.5 英寸的 PVC 管道系统，在阁楼上有一个中央真空系统，在每个孩子的房间里有两个终端站。交替哪个站具有关闭的阀门和打开的门决定了气流路径，并且将小塑料旅行瓶从一个站穿梭到下一个站。

每个工作站都有自己的 iPhone 界面，可以向 Raspberry Pi 发送数据并传递信息，包括指示牙齿移动路径的模拟地图。除了通过 Pi 的一个 GPIO 控制真空之外，这款手机主要是为了分散孩子们的视觉注意力，而父母则溜到另一个房间，用一些零钱代替牙齿。[Jeff]确保在每个站点增加一个锁定的门以限制进入，并希望保持神秘感。

在下面的视频中，看着他儿子的脸因整个事件而露出纯粹的喜悦，并后悔自己的童年发生在创客革命之前。然后和一个[啤酒取物机器人](http://hackaday.com/2013/09/08/brewster-fetches-your-beer-automatically/)一起庆祝你的成年。
