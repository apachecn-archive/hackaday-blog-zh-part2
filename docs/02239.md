# 将四架较小的直升机变成一架较大的四轴飞行器

> 原文:[https://hack aday . com/2012/12/03/将四架较小的直升机变成一架较大的四轴飞行器/](https://hackaday.com/2012/12/03/turning-four-smaller-helicopters-into-one-larger-quadcopter/)

![copter](../Images/ca9c5a3cc13b3b42dd6f107a1968a71d.png)

过去几年我们看到了一大批四轴飞行器，这是有原因的——任何四轴飞行器的关键部件都是惯性测量单元。从历史上看，这些 IMU 是一个非常复杂和昂贵的套件，几年前价格下降，允许任何口袋里有几美元和几个无刷电机的人在他们的车间里建造一个四叶片无人机。

[Starlino]制造了一些四轴飞行器，但他想避开 IMUs，让他的新直升机的大部分质量位于底盘的中心。他想出了一个他称之为四轴混合动力的设计，可以由四个廉价的三通道直升机玩具组装而成。

[Starlino]四轴混合动力车的大部分升力来自一对来自司马 001 三通道直升机玩具的同轴旋翼。玩过这些玩具直升机的人都知道它们有多稳定；如果尾桨坏了，你只剩下一架只能上下的直升机。

为了让他的四轴混合动力车有几个自由度，他将三通道直升机的四个尾桨连接到几个交叉布置的吊杆上。通过从 4 通道直升机上取下接收器并添加自己的控制板，[Starlino]让每个尾桨控制飞机的俯仰和翻滚。

在休息后的视频中，你可以看到 quadhybrid 即使没有 IMU 也非常稳定，而且非常敏捷。由于[Starlino]的直升机可以由廉价的 3 通道直升机的替换零件制成，我们预计这些尾部马达很快就会在您最喜爱的在线遥控玩具零售商处出现。
