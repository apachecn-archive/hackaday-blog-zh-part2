# 在你扔掉南瓜之前，做一个龙灯

> 原文：<https://hackaday.com/2012/11/06/before-you-throw-out-that-pumpkin-make-a-dragon-o-lantern/>

![](img/7fe881789312b4bb0331ed2898d86b7d.png "dragon")

是的，对于万圣节来说有点晚了，但是在你扔掉放在你前门外的腐烂的葫芦之前，用一个龙灯来计划明年的庆祝活动怎么样？

第一个进入我们提示线的龙灯是【亚伦】为南瓜雕刻比赛制作的[喷火龙杰克灯](http://www.youtube.com/watch?v=YYJfDWBXQtI&feature=youtu.be)。这个想法很简单；只要在南瓜上雕刻一个龙头，插上蜡烛，用一罐 Aquanet 喷洒明火。当然，它很粗糙，但是经过一点工程设计，它会变成一个非常非常酷的建筑。

对于一个稍微复杂一点的构建，[Chris] [构建了一个喷火的杰克灯](http://www.majorcallisto.com/projects/dragon-pumpkin/)，只要有人说“不给糖就捣蛋”就会触发它这一构建的关键部分是微软 Kinect，用于其语音识别功能，而不是其形成 3D 深度云的能力。

[克里斯]建造的火部分是由 Arduino 控制的，它触发螺线管将少量燃料分配到南瓜壳中。[Chris]不介绍他的“poofer”的构造，但是您当地黑客空间的任何成员都应该能够向您展示它是如何完成的。
