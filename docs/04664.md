# 一个模块化的游戏机合成器

> 原文:[https://hack aday . com/2013/11/27/a-modular-game-boy-synthesizer/](https://hackaday.com/2013/11/27/a-modular-game-boy-synthesizer/)

![Euro](../Images/8ee109ab2f041613e809876759e90194.png)

全世界的合成器头和电子音乐爱好者都喜欢一个好的架装式合成器。然而，如今，合成更倾向于小型、数字化和“复古”，而不是六七十年代那种庞大笨重的庞然大物。在这方面，[吉斯]可能走在了前面，因为他为他的 eurorack 合成器构建了[一个 Game Boy 模块。](http://gieskes.nl/undefined/eurorack/?f=euroboy)

在[吉斯]的 Game Boy 上运行的软件是德高望重的[小声音 DJ](http://www.littlesounddj.com/lsd/) (LSDJ)，这是在每个人最喜欢的 8 位掌上电脑上创作 chiptunes 的最后一句话。正如 chiptunes 中使用的任何合适的 Game Boy 一样，对 20 世纪 80 年代的硬件进行了一些修改。[吉斯]用一个“重复”信号接入盒式磁带连接器，为 LSDJ 提供了减速、嘈杂的信号。还有通过 CV 的音高控制，音频输出达到 10Vpp

在下面的视频中，你可以看到[吉斯]的“欧洲男孩”使用了几个 Doepfer synth 模块。里面还有一个用旧硬盘制成的非常酷的脉冲发生器，所以绝对值得一看。
