# 微型有机发光二极管 O-scope 适合安装在试验板上

> 原文：<https://hackaday.com/2012/09/30/tiny-oled-o-scope-fits-on-a-breadboard/>

[https://www.youtube.com/embed/UfAkdd9kXNY?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/UfAkdd9kXNY?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

随着今年 Maker Faire 上 3D 打印机的过剩，看到制造商们谈论最多的工具是一个简单的示波器真是令人惊讶。

[【Gabriel Anzziani】的 Xprotolab](http://www.gabotronics.com/development-boards/xmega-xprotolab.htm) 是一款极小的示波器、函数发生器、逻辑分析仪，通用 128×64 有机发光二极管显示器是您下一个原型项目的完美补充。凭借其试验板友好格式和 USB 输出，它将尽职尽责地充当 200kbps 示波器、8 通道逻辑分析仪，或者如上面的视频所示，作为 Wii 双截棍的完美接口，或者只是一个简单的数字蚀刻草图。

在折叠上方的视频中,[Gabriel]展示了他的微型有机发光二极管示波器的功能。
