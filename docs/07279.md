# 幼儿点唱机不需要硬币或按钮捣碎

> 原文:[https://hack aday . com/2014/10/09/幼儿点唱机不需要硬币或按钮捣碎/](https://hackaday.com/2014/10/09/toddler-jukebox-requires-no-quarters-or-button-mashing/)

啊，学步的孩子。他们既好奇又笨拙。当他们想触摸和接触我们喜欢的东西，想让他们玩的时候，很难说不。[Shawn]对他儿子对家庭 Sonos 系统的兴趣有这样的感觉，并设计了一个优雅的解决方案，他称之为 [Song Blocks](http://shawnrk.github.io/songblocks/) 。

Sonos 放在一个藏着 RasPi B+的梳妆台上。使用编号为 1-12 的裸胡桃木块，他的儿子可以不用实际接触就能使用 Sonos。每个积木都有一块磁铁和一个 NFC 标签。当他的儿子将一块嵌有磁铁和 NFC 控制器板的积木贴在右边抽屉的正面时，B+读取标签并播放歌曲。它还在推特上发布歌曲选择和艺术家。

这些积木本身非常漂亮。[肖恩]用看起来像快递新邮票的东西给它们编号，然后用烙铁把数字烧进去。他的 [Python 脚本](https://github.com/shawnrk/songblocks)在 git 上，他有到他的构建页面上使用的库的链接。歌曲块演示视频等着你跳完。
