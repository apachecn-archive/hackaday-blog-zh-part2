# 从太空发射滑翔机

> 原文：<https://hackaday.com/2013/03/10/launching-a-glider-from-space/>

我们得到了大量关于气象气球发射将电子产品带入太空的提示。但是每隔一段时间就会有一个人脱颖而出。该项目确实将电子有效载荷送入太空，但它也让[David] [使用遥控飞机](http://rcexplorer.se/projects/2013/03/fpv-to-space-and-back/)从近空间飞回他的硬件。

返回舱没有动力，但这应该不成问题，因为从气象气球上发射将为飞行提供足够的高度。因为那部分大气的温度非常低，大卫不得不考虑几个因素。显然，你希望你的电池和控制电子设备与寒冷隔绝。但是通常不会想到的是控制滑翔机襟翼的伺服电机的问题。他们通常在齿轮上有一些白色的油脂。在低至-50C 的温度下，油脂会变硬，使伺服系统停止工作，所以他确保在飞行前彻底清洁齿轮。

不幸的是[大卫]在从地面站捕捉图像和录制视频时遇到了一些问题。但他的文章仍然是一个有趣的阅读，休息后的剪辑从滑翔机的鼻子相机给出了整个项目的概述。
