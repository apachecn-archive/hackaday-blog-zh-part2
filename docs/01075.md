# MIDI 瑞士军刀由 MSP430 制成

> 原文：<https://hackaday.com/2012/07/18/midi-swiss-army-knife-built-from-an-msp430/>

[https://www.youtube.com/embed/0hvQUXcXlEg?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/0hvQUXcXlEg?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

紧随 TI MSP430 的 web 服务器之后，[Rob]分享了 MSP430 LaunchPad 的 MIDI booster pack,这是一种非常便宜且功能越来越强的微控制器，有点像狂热的追随者。

[Rob]的 MIDI 增强包包含一个 MIDI 输入和输出端口，以及他能想到的任何 MIDI 操作应用程序。到目前为止，[Rob]已经有了一个 MIDI 自动琶音器、一个和声器、一个回声、过滤器、一个在诺基亚 LCD 上显示输入信息的 MIDI 监视器，以及一个与光传感器或操纵杆连接以操纵 MIDI 变量的控制器。

早在 80 年代(和 90 年代，甚至到现在)，一个专用的 MIDI arp 或 harmonizer 是一个疯狂昂贵的套件，尤其是考虑到设备是如此简单。那些专用的机架装配盒现在可以用一个 [TI 发射台](http://processors.wiki.ti.com/index.php/MSP430_LaunchPad_(MSP-EXP430G2))和【Rob】的助推器包代替。

休息之后，你可以看看 arp 和 harmonizer 的运行情况。
