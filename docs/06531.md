# BigBoy Advance，大手的巨型 GBA

> 原文:[https://hack aday . com/2014/07/12/big boy-advance-a-giant-GBA-for-big-hands/](https://hackaday.com/2014/07/12/bigboy-advance-a-giant-gba-for-big-hands/)

![big boy advance](../Images/aae58c37c55e98f3127317ef3729c068.png)

曾经希望 Game Boys 的尺寸稍微大一点吗？Bacman 的[John]又名[Bacteria]，决定用这个复古控制台 mod——[big boy 尝试一些不同的东西。](http://www.bacman.co.uk/bacforum/viewtopic.php?f=18&t=4555)

如果你不熟悉 [Bacman 网站](http://bacman.co.uk/)，这是一个致力于复古视频游戏控制台改装的网站——我们的黑客【John】是幕后黑手。我们已经[分享了](http://hackaday.com/2014/07/08/the-homebrew-xbox-360-and-ps3-portable/) [很多](http://hackaday.com/2014/05/07/plug-and-play-portable-console-saves-space-so-you-dont-have-to/)[他们之前的项目](http://hackaday.com/2014/04/30/a-masterpiece-of-3d-printed-case-modding-with-an-ouya/)。

BigBoy 基本上是 Game Boy Advance——配备 8 英寸显示屏。它使用的电子器件来自一个定制盒子里的仿制 RetroBit，这个盒子是[John]在家里用真空吸尘器制作的。他勾勒出了提议的轮廓，用塑料片和热胶水制作了一个模具，并为真空成型器制作了一个混凝土虚拟模具——这意味着如果他想再造这个项目，那简直是小菜一碟！

在把所有的按钮放好之后，在这里和那里做一些小的调整，他决定用一些斑点石漆来油漆整个东西，这给了它一个非常独特和健壮的完成。他的构建日志相当广泛，所以如果你感兴趣，别忘了去论坛上看看。
