# 愚人节项目教同事不要碰你的东西

> 原文:[https://hackaday . com/2013/04/02/April-fools-project-teaches-collaborates-not-touch-your-stuff/](https://hackaday.com/2013/04/02/april-fools-project-teaches-coworkers-not-to-touch-your-stuff/)

![april-fools-siren-box](../Images/a45976f62759df167fa52bd8bc7b3e57.png)

看一眼这个项目盒子上面的标签，很明显你不应该去按红色塑料保护的开关。但是，如果你的同事有点像提米家的人，至少会有一些人会过来，他们的手就是放不下。他建造这个是为了给那些人一个教训。扳动开关，一声响亮的警笛开始鸣响。问题是，107 分贝的警报不能被开关关掉，你必须把开关的钥匙放在旁边。

警报器是一个独立的单元，只需要 6V 到 15V 之间的电源。这使得这个项目非常简单，唯一需要考虑的是如何建立一个机械的 SR 锁(设置-重置)。解决方法是使用机械继电器。拨动开关将常开连接器连接到公共端子以启用继电器。钥匙开关断开继电器的接地连接，允许电磁开关再次打开。

如果你需要一些帮助来理解中继连接是如何工作的，我们在休息后嵌入了一个不相关的视频。
