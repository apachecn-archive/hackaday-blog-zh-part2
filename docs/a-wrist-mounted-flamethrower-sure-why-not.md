# 腕式喷火器？当然，为什么不呢？

> 原文：<https://hackaday.com/2014/09/25/a-wrist-mounted-flamethrower-sure-why-not/>

[https://www.youtube.com/embed/TR53GF12TOQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/TR53GF12TOQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

创客大会上有三种类型的展位。首先是奇怪的企业展台，像百事可乐“革命性的液体摄入”或类似的废话。如果你想知道的话，那是所有摊位中排队最长的。第二种类型的展位是你期望在那里的人——Atmel、TI 和 Makerbot 都全力以赴。

第三种展位有点难找。他们是创造者大会的“展示和讲述”精神，斯蒂芬·霍斯是其中的佼佼者。为什么？腕式喷火器，这就是原因。

火焰喷射器以原本用于野营炉的丙烷瓶为燃料，微控制器和锅设置负责火焰的高度。[斯蒂芬]拇指下面的按钮负责丙烷流量和基于~~tazer~~的点火器。手腕测量传感器可以重新调节火焰的高度，以适应佩戴者可以移动手腕的距离。

总而言之，这是一个很棒的展览项目，尽管我们对指派给斯蒂芬的纽约消防队长感到有点遗憾。顺便说一句，我们为[斯蒂芬]鼓掌，因为*而不是*引用了漫画人物手中射出的火焰。
