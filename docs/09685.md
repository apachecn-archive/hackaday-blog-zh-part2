# 使用 Kinect 进行虚拟物理康复

> 原文:[https://hack aday . com/2015/08/03/virtual-physical-rehab-with-Kinect/](https://hackaday.com/2015/08/03/virtual-physical-rehab-with-kinect/)

网站已经发现将事物“游戏化”可以增加参与度。例如，你可能只是为了得到一个高级贡献者徽章(这甚至不是真正的徽章，而是一张徽章的照片)而在论坛上发布帖子。现在[Yash Soni]把同样的想法带到了物理治疗中。

[Yash]的父亲不得不经历无聊的物理治疗来治疗椎间盘突出，这促使他开发了[运动疗法](http://www.kinectotherapy.in/)，旨在使治疗更像一个视频游戏。他们声称它可以用来帮助许多类型的病人，从中风患者到脑瘫患者。

患者可以看到他们的屏幕化身复制他们的动作，并可以在玩家正确或错误地移动时提供音频和视觉反馈。统计数据也可用于患者的健康护理专业人员。

这不是一个原创的想法(见下面的视频)，但随着[一篇 IEEE 论文](http://theinstitute.ieee.org/technology-focus/technology-topic/microsoft-kinect-could-reinvent-physical-therapy)和显然来自医疗专业人士的大量参与，它看起来像是未来增长的有力竞争者。然而，有趣的趋势是，我们已经看到了 Kinect 设备的许多[独特的](http://hackaday.com/2015/05/30/interactive-fur-mirror-follows-your-every-move/)用途[以及游戏趋势可能在游戏玩家社区之外变得具有商业重要性的想法。](http://hackaday.com/2014/02/01/autonomous-lighting-with-intelligence/)
