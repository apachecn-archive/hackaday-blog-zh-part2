# 用于时钟收音机的压电晶体扬声器非常容易制造

> 原文:[https://hackaday . com/2014/03/20/压电晶体扬声器用于时钟收音机——非常容易制造/](https://hackaday.com/2014/03/20/piezoelectric-crystal-speaker-for-clock-radio-is-alarmingly-easy-to-make/)

让我们面对现实吧:我们大多数人起床都有困难。很多时候是因为警报声不够大，不足以把我们从维京人的梦中唤醒。[RimstarOrg]的房主协会不允许他在后院养一只公鸡，所以[他制作了一个压电晶体扬声器](http://www.youtube.com/watch?v=R7zjfaPKMSE)来提高音量。

[RimstarOrg]的扬声器使用一个绑在豆罐振膜上的[罗谢尔盐](http://en.wikipedia.org/wiki/Potassium_sodium_tartrate)水晶。在他的演示中，他首先将一个旧的时钟收音机直接连接到水晶上。这根本不是很大声，所以他反过来加了一个门铃变压器。这个声音大一点，但还是不会让【RimstarOrg】起床。

进入微波炉变压器。现在它足够响亮了，虽然它不是[火警](http://hackaday.com/2013/11/03/fire-bell-wakes-you-for-work-by-shaving-years-off-your-life/)。他还演示了使用压电点火器和水晶收音机耳机的扬声器。一如既往，视频在跳转之后。[RimstarOrg]在摘要中有很多相关的链接，所以你可以学习如何种植你自己的罗谢尔水晶。
