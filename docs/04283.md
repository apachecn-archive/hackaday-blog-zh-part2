# 点亮耳环

> 原文:[https://hackaday.com/2013/10/11/light-up-earrings/](https://hackaday.com/2013/10/11/light-up-earrings/)

![light up earrings](../Images/1cc704b7bea18d14d64aabf7188cf660.png)

不幸的是[ch00f]最近太忙了，没有时间为 Hack a Day 写东西，但是他完成了一个很棒的小项目——[圣诞 LED 耳环](http://ch00ftech.com/2013/10/08/light-up-earrings/)！

就像他所有的项目一样，有一篇精彩的文章涵盖了一切——甚至是那些没有成功的东西。但是这个项目真正让我们钦佩的是他工作的规模。微型电池夹在两块板之间？只有 19 毫安。这实际上足以为耳环供电几个小时，但只有几个 microSD 卡的大小和厚度！

第二件真正让我们吃惊的事情是主板本身，根本没有编程头的空间！为了解决这个问题，实际上[ch00f]将 PCB 分成了 3 个部分，对其进行了编程，然后删除了编程标题部分！如果这还不够聪明，这个怎么样——他还在车上安装了霍尔效应传感器，以便在充电时关闭它们！更不用说一个复杂的木箱来装它们了…

休息后留下来看看精彩的演示视频，它甚至有一些 1930 年代的经典音乐，这真的很有情调。
