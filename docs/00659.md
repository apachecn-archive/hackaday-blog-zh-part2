# 池塘中的自动尾波滑水

> 原文:[https://hack aday . com/2012/06/06/池塘自动滑水/](https://hackaday.com/2012/06/06/automatic-wakeboarding-in-a-pond/)

![](../Images/c8c749d3a5e47ab0837a3f7678fcf846.png "wake")

[Rick]发来了一个他和他叔叔[Charlie]这个夏天一直在做的很棒的项目。这是一个完全自动的尾波滑水系统，它把查理和他的同伴从后院池塘的一端拖回来。

整个系统由池塘两端的两座塔支撑。串在这些塔之间的是一条 420 英尺长的电缆，在停止前将骑手从一端拉到另一端，并自动向相反方向继续。除了停止和启动连接到电缆的电机，不需要用户输入。这意味着任何一个足够幸运到达[查理]农场的人都可以不间断地滑水一下午。

[Rick]发布了几个简短的视频，介绍了系统的内部情况。从我们可以看到，这些家伙正在使用一个 [LabJack U3-LV](http://labjack.com/catalog/u3-lv) 数据采集单元来获取骑手的速度、方向和位置。这些数据被发送到一台笔记本电脑上，这台电脑控制着悬挂在其中一座塔上的马达。
