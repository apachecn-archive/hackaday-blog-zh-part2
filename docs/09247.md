# 自烤面包片以来最伟大的事情

> 原文:[https://hack aday . com/2015/06/09/自烤面包以来最伟大的事情/](https://hackaday.com/2015/06/09/the-greatest-thing-since-toasted-sliced-bread/)

没错。[柯林·福尔泽]刚刚淘汰了一种家用电器。当你可以一边切面包… **[一边烤面包的时候，谁还需要烤面包机！](http://www.youtube.com/watch?v=kcjGRXTpHGI)T3】**

让[Furze]把《银河系漫游指南》中的某些东西变成现实吧。他的一个订户向他的新系列《Furze 的发明秀》提交了一个想法，他自己动手制作了长期受人尊敬的光剑面包刀。我们一直在等待这一天。

不幸的是，它不完全是一把光剑。事实上，它更像是一个光锯，如果发音正确，很容易被误认为是真的。使用重新包装的微波变压器——很像自制的点焊机装备——【Furze】正在通过钢锯刀片以低压泵送一吨安培，使它变得炽热，准备烤面包。
