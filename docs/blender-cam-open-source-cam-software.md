# 搅拌机凸轮-开源凸轮软件

> 原文：<https://hackaday.com/2013/10/19/blender-cam-open-source-cam-software/>

[Vilem]提交了一个关于他正在为 Blender 开发的插件的提示，这个插件叫做 Blender CAM。它允许直接从 Blender 导出到一个 [G 代码](http://en.wikipedia.org/wiki/G-code)文件。他已经为此工作了几个月，[定期发布更新](http://blendercam.blogspot.com/)，进行各种调整和改进。虽然项目尚未完成，但[Vilem]已经取得了一些令人印象深刻的进展。它目前支持 2D 和 3D 策略，各种刀具类型，模拟 3D 操作，甚至自动桥接。

上面的图像是使用插件制作的，它显示了可能的细节层次。我们迫不及待地想看到他计划添加的 4 轴和 5 轴支持。

休息之后嵌入基本教程视频。就像任何与 Blender 相关的东西一样，它并不是令人难以置信的自动化，但另一个免费工具绝对是一件好事。看起来[Vilem]正在寻找一些可以帮忙的其他开发人员。如果你有知识，你可以考虑贡献。
