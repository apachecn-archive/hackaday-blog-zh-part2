# 根据封面来判断书的好坏

> 原文:[https://hacka day . com/2013/06/15/where-books-is-butters-by-their-cover/](https://hackaday.com/2013/06/15/wherein-books-are-judged-by-their-cover/)

是的，Kindles 很棒，一台装满 pdf 的电脑很棒，而且[蒂姆·伯纳斯·李]可能会比[古腾堡]更重要地被载入史册。这并不意味着书籍没有什么无形的东西能引起如此多藏书家的喜爱。即使是一本充满空白页的书本身也可以是一件艺术品，而且大多数时候这些书都是手工制作的。

这段由 Smith Settle bookbinders 制作的精装书的视频涵盖了从书页上的文字到成品书的整个过程。不，他们不用活字印刷。对开本是用平版印刷术制作的，但是对开本的分类、粘合、缝制和装订的方式与中世纪的方式非常相似。

接下来是 1968 年由爱荷华州立大学制作的一部精彩电影，讲述了在精美的皮革装订书籍上制作黄金工具的故事。如今，你很难找到一本带有金色工具的书，但对于勤劳的业余装订工人来说，这仍然是一种技术。

首先，金箔被涂在一本书的皮革书脊上。然后用定制的工具加热到几百度，压进叶子里。热量将黄金与皮革粘合在一起，并通过定制的工具将设计烙进皮革。之后，多余的叶子被简单地擦掉，一个精美的工具化皮革书就诞生了。

装订真正酷的是几乎任何人都可以做。去工艺品商店，买一些硬纸板和纸，给自己装订一本书。你可以写一本空白日志，或者为外面的书呆子们(包括我自己)，做一本硬皮的 NASA 布线标准。
