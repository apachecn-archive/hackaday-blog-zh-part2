# 时尚的咖啡厅 Battlestation 宠坏顾客

> 原文:[https://hack aday . com/2014/05/07/stylisted-cafe-battle station-flowers-customers/](https://hackaday.com/2014/05/07/stylish-cafe-battlestation-spoils-customers/)

![Cafe Battlestation](../Images/f86f2a46d54cde4837cd168c870dcec1.png)

[Tasos]给我们发来了这个关于他为他的网吧制作的定制战斗站的提示。(希腊语；[译](https://translate.google.com/translate?sl=auto&tl=en&js=y&prev=_t&hl=en&ie=UTF-8&u=http%3A%2F%2Fwww.thelab.gr%2Fcases-psu-and-mods%2Fkataskeyi-grafeiwn-vaseis-othonwn-gia-internet-cafe-134357.html&edit-text=&act=url)。这张桌子从最不起眼的地方开始:一个看起来像是中密度纤维板的简单框架，还有用来固定电脑的柜子。这个建筑的目标是提供一个美观和用户友好的建筑。[Tasos]不满意现成的显示器支架带来的限制，所以他制造了自己的、更可调的替代品，通过它他可以运行必要的电线。

[Tasos]对每个显示器支架进行了彻底的打磨、涂底漆和喷漆，以获得散发出金属般完美的成品，然后他安装了一个用于启动计算机的大按钮和一些 led 以提供柔和的背光。在引擎盖下，[Tasos]将电脑的内部部件安装到各种定制的外壳中。虽然他还没有提供这部分建筑的全部细节，但我们怀疑更多的图像即将出现。你可以在他的论坛帖子里找到更多的细节。
