# 我们有一个问题:家用电器

> 原文：<https://hackaday.com/2015/03/26/we-have-a-problem-household-electrical/>

黑客日，我们有个问题。你家的电接通了。它一直开着。这些千瓦时过得有多快，是什么原因造成的？对于大多数人来说，他们唯一的衡量标准是电表本身(没人看)和电费账单(很少有人真正分析)。人们对手机电池使用的关注远远超过对住宅耗电量的关注，这是不是很愚蠢？我认为是的，因此出现了另一个关于 [Hackaday 奖](http://hackaday.io/prize)参赛作品的好点子。

## 更好的测量方法

这里最棘手的问题是获得可靠的数据。就在昨天，我们看到了[一个令人难以置信的资源监控项目](http://hackaday.com/2015/03/25/resource-monitoring-solution/)，它使用光学传感器来测量电表中的转动或轮子。我们已经看到了类似的项目，有一个闪烁的 LED 仪表，和一些其他的方法。但在许多情况下，电表在室外，这使得便宜、易于安装的传感器难以实现。即使我们做到了，这仍然只提供了一个数据流，整个房子。

或者你可以接入断路器盒。我们已经看到[比尔·波特] [就是这样做的](http://hackaday.com/2010/12/21/clamp-sensor-power-monitoring/)，并且有一些商业上可用的套件包括一个钳形电流传感器的章鱼。这是一点改进，但仍然需要用户打开配电板(不要嘲笑这种说法，你知道大多数人不应该这样做)来安装它们。我确信还有其他我遗漏的方法，我很乐意在下面的评论中听到它们。

## 点

总结一下我在这里得到的东西，想想[的 Kill-A-Watt](http://hackaday.com/2010/05/17/tweet-a-watt-now-speaks-to-google-power-meter/) ，它被证明是一个非常有趣的黑客行为。人们不仅希望看到某样东西消耗了多少能量，还希望扩大数据的可访问范围。我们不记得有任何成功的努力将这个概念向前推进了几代。但是，如果有人能够破解这个难题，它可能会产生一波节能浪潮，因为人们能够更好地与他们家中使用大量电力的设备联系起来。
