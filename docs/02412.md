# 内置一个小灯泡的混凝土桌子

> 原文:[https://hack aday . com/2012/12/24/a-concrete-table-with a little-blinky-built/](https://hackaday.com/2012/12/24/a-concrete-table-with-a-little-blinky-built-in/)

![concrete-table-with-led-inlay](../Images/a3ddb931c49ef107102bdc69b518fe43.png)

曲速地带黑客空间的成员想要一个超乎寻常的咖啡桌。他们最终为玻璃顶浇注了一个混凝土底座。在设计过程中有几件事情需要解决。首先，他们想将 led 集成到混凝土侧面。由于混凝土非常重，所以必须考虑便携性。拼图的最后一块是决定在磨砂玻璃下放置什么样的硬件。

支腿设计有一个大的开口区域，以尽可能减轻重量。横梁上有一组空白，用一些绿色 led 拼出黑客空间的名称。这是通过在浇注混凝土之前在模板中放置每个字母的泡沫切口来实现的。他们用硅胶封住了每个字母，但仍然有一些渗漏，这很可能是在挤压表格以帮助消除气泡时造成的。吸管被放在泡沫中，以便让电缆穿过电子设备。一切就绪后，他们用热熔胶填充空隙，作为扩散器。

磨砂玻璃下的 RGB LEDs 没有太多细节。但你可以看到他们在休息后的剪辑中制作的灯光表演。
