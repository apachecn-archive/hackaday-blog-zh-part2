# 建造一个链锯工厂来制造木板

> 原文:[https://hack aday . com/2014/03/06/build-a-chain-saw-mill-to-make-planks/](https://hackaday.com/2014/03/06/building-a-chainsaw-mill-to-make-planks/)

这里有[一个电锯黑客](http://www.instructables.com/id/Chainsaw-Mill-Build-Use-Tips-n-Tricks)比[我们分享的最后一个更有意义……](http://wp.me/pk3lN-uht)这是一个你可以建立的设置，帮助你将原木切割成可用的木板用于你自己的项目！

我们的这个工具黑客向导是[BongoDrummer]，他是威尔士一个名为 [Flowering Elbow](http://www.floweringelbow.org/) 的团体的联合创始人，致力于通过帮助激发人们的发明创造、鼓励项目合作以及为社区做出贡献来想象和创造更美好的未来。我们认为他只比我们之前的电锯发明者多一点点

 *[BongoDrummer]从适当的安全说明开始，解释事故统计数据，并提供正确使用链锯的复习指南。从那里，他开始着手工厂的设计和建造。他选择使用铝挤压型材，因为它坚固、轻便、易于使用，更不用说易于组装了！视频和更多信息后，跳转。

构建日志非常详细，易于理解。[BongoDrummer]呈现了几个巧妙的功能，比如使用自行车内胎作为握把套，制作辅助加油器以保持刀片平稳运行，以及使用滑板轮和轴承作为导向辊。
