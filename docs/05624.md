# 3D 打印…卷尺？

> 原文:[https://hackaday.com/2014/03/25/3d-printed-measuring-tape/](https://hackaday.com/2014/03/25/3d-printed-measuring-tape/)

![3d printed measuring tape](../Images/38d91f4bb8da45313efddd8582617c6e.png)

这里有一个推信封的新的…一个 [3D 打印的卷尺怎么样？](http://thecornercase.blogspot.ca/2014/03/3d-printed-tape-measure.html)

这种独特的 3D 打印工具是在单一作业中设计和打印的。[愤怒的和尚]最近一直在用这些复杂的设计挑战自己，最近完成了一套完全 3D 打印的卡尺，这本身就令人印象深刻。

为了寻找他的最新挑战，他思考如何制作这种 3D 打印卷尺。当他继续思考这个问题时，他意识到实现它实际上会有多复杂。在设计和打印了一些基本部件来帮助他巩固他的想法后，他开始工作。这把卷尺有 **114 个独立零件。**它包括 52 英寸的卷尺，1 英寸的刻度和标记精确到 1/8 英寸。它甚至有一个手摇曲柄(对不起，没有弹簧复位)来卷起磁带。

现在你可以想象，像这样复杂的组装有点超出了普通 3D 打印机的能力范围——UV 树脂打印机可能可以做到，但[愤怒的和尚]使用了商业 Objet Eden 3D 打印机。尽管如此，这是一个令人印象深刻的设计展示，看看下面的视频，自己看吧。
