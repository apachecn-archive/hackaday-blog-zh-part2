# 带阿特拉斯去散步

> 原文：<https://hackaday.com/2015/08/17/taking-atlas-for-a-walk/>

还记得波士顿动力公司的人形机器人阿特拉斯吗？谷歌收购的公司，呃，Alphabet 旗下的，呃，最有可能成为天网的？嗯——他们刚刚向我们展示了阿特拉斯现在可以在森林里慢跑了。

波士顿动力公司几天前在 YouTube 上发布了一个快速演示，介绍了该公司正在为他们的机器人进行的一些升级。首先是展示大狗的新肢体…看起来像大象鼻子，末端有一只假手。大狗现在可以随时离开测试实验室——门把手不再是障碍。考虑到多年来它已经能够穿越极其崎岖的地形，这不是一个好兆头…

另一方面，Atlas 也取得了长足的进步。从 2013 年[第一次独立站立](http://hackaday.com/2013/07/16/atlas-humanoid-robot-standing-on-his-own/)开始，他很快转移到[被药球击中(没有摔倒！)](http://hackaday.com/2013/10/08/atlas-survives-the-wrecking-ball/)——而现在，他可以在外面跑了。幸运的是，他们还没有完全弄清楚电池组的解决方案…休息后他的户外活动视频。
