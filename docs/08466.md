# 极便宜的电机平衡和振动分析

> 原文：<https://hackaday.com/2015/03/07/dirt-cheap-motor-balancing-and-vibration-analysis/>

作为一名有事业心的黑客和敏锐的工具爱好者，[Chris]知道“感觉”的重要性。一般来说，便宜的工具会在你手中晃动，因为马达不平衡。他想找到一种便宜的方法来量化这种感觉，于是[制作了一个视频](http://www.youtube.com/watch?v=pMxtYRdbjwI)描述他如何使用大多数人身边的几样东西来确定电钻的[阻尼](https://en.wikipedia.org/wiki/Damping):耳塞、钕磁铁、废钢和 Audacity。

他把钻头固定在一个悬臂式的废钢上，用老虎钳固定。粘在钢上的钕磁铁中断了耳塞中的磁场，耳塞用第三只手工具固定住。[克里斯]用带子绑住钻头的扳机，用自耦变压器控制它的速度。首先，[Chris]使用 Audacity 的 plot spectrum 找到系统的固有频率，然后让钻头以相同的速度运行，以诱导不同节点的摆动。正如他解释的那样，人们甚至不需要使用软件来显示[振动节点](http://www.bibliotecapleyades.net/ciencia/ciencia_resonance04.htm)——连接到系统并瞄准磷光目标的激光将绘制正弦波。

只是为了好玩，他让钻头严重失衡，以找到系统自我分裂的频率。休息之后来看看。
