# 本周失败:透明电路设计显然是一个挑战

> 原文:[https://hack aday . com/2014/10/09/本周失败透明电路设计显然是一个挑战/](https://hackaday.com/2014/10/09/fail-of-the-week-transparent-circuit-design-is-clearly-a-challenge/)

[Frank Zhao]想尝试制作透明电路板。他的计划是用激光切割机蚀刻路径，并用导电墨水填充凹槽。凹槽大约 0.1 毫米深 x 大约 0.8 毫米宽。

他用的是镍墨，比银墨稍微便宜一点。不过，墨水是他最小的问题之一。在每英寸数百欧姆的测量电阻下，这已经是一个交易断路器，因为他的电路不能在 0.3V 以上的电压降下工作。更糟糕的是，由于激光切割机的运动，波谷很粗糙，与笔尖的按压-分配特性不匹配。这造成了一些溢出，他无法优雅地处理，因为墨水也碰巧融化丙烯酸。

[弗兰克]将再次尝试使用铜箔和更宽的轨道。你认为如果他使用银墨水和不同的输送方法，比如移液器，他会做得更好吗？更深的凹槽怎么样？
