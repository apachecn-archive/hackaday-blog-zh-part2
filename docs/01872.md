# 来自 7400 逻辑芯片的可配置 RFID 标签

> 原文:[https://hack aday . com/2012/10/12/configurable-RFID-tag-from-7400-logic-chips/](https://hackaday.com/2012/10/12/configurable-rfid-tag-from-7400-logic-chips/)

![](../Images/77ac73d117c134c081e1d6bfa83d518a.png "rfid-from-7400-chips")

这个焊接噩梦是一个可配置的 [RFID 标签，它由 7400 系列逻辑](http://www.vagrearg.org/content/rf74xxid)芯片构建而成。这个项目的成果是一个 iPhone 大小的模块，可以用作使用 125 kHz 标签的安全系统的新门禁卡。最棒的是，一系列开关使得标签可以手动编程，尽管是二进制的。

当然这是今年 7400 逻辑竞赛中[的一个词条。它来自去年的获胜者，他花了很多时间记录这个项目；这是我们喜欢的。我们感到惊讶的是，这么多的芯片可以简单地由读取器线圈中感应的东西供电。这只是 7400 系列多年来如此受欢迎的原因之一。算出数字后，构建一个 64 位移位寄存器，将标签 ID 输入设计的编码部分。在这一过程中有许多问题需要解决，但一旦功能正常，表面贴装设计就被放在一起，最终产品在休息后的视频中展示出来。](http://hackaday.com/2012/09/06/2012-open-7400-logic-competition/)
