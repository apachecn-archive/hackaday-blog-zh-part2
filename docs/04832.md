# 黑进一个加热垫

> 原文:[https://hackaday.com/2013/12/16/hacking-a-heating-pad/](https://hackaday.com/2013/12/16/hacking-a-heating-pad/)

![HeatingPadBreadboardS](../Images/70b6c9b14cfc6b876a911b5ecf39b506.png)

[Bob]和他的妻子使用床加热垫。在冬天，他们通常在睡觉前一个小时左右打开它，这样床就既舒适又温暖。问题是，如果他们不小心让它开着，他们会在几个小时后醒来:过热。他们需要的是一个先进的计时系统。

一个普通的出口定时器不能满足他的需求:一年中的大部分时间，加热垫应该在轻微延迟后关闭，但在冬天，他们更喜欢让加热垫保持在低得多的温度。[Bob]决定创建一个带有微控制器的定制定时器，以提供可调的持续时间和加热水平。

电路很简单。它由一个微控制器、一个 2 位数的 LED 显示屏、两个按钮和两根连接到加热垫原始控制器的电线组成。最终构件允许您设定面板打开、关闭和/或降低几个级别的时间。这是一个了不起的黑客，你可以在休息后的视频中看到界面是如何工作的。
