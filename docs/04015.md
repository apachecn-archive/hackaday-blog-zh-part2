# 布鲁斯特会自动帮你拿啤酒

> 原文:[https://hack aday . com/2013/09/08/Brewster-fetches-your-beer-automatically/](https://hackaday.com/2013/09/08/brewster-fetches-your-beer-automatically/)

![brewster-fetches-from-the-refriderator](../Images/31b64d97e2f05df44fb9aa42c5c304a5.png)

害怕如果你离开房间，你会错过最好的游戏？现在你不需要移动你的屁股去抓下一杯啤酒了。布鲁斯特被开发用来从冰箱里取出冰镇啤酒，并自动把它们送到你面前。

该机器人最初是一个 roomba，但在底盘顶部增加了一个机械臂，从而大大改变了用途。这不仅让布鲁斯特抓住一罐啤酒，但它可以首先打开迷你冰箱，并达到足够远的内部从后面得到一个。这不需要对冰箱进行改造，但 roomba 的低净空要求在地板上放置一个迷你冰箱。休息之后，请观看视频中的演示。我们认为当前版本运行在预编码的路线上；这个项目只是在等待一个具有测绘和机器视觉的副产品。

这种单罐交付的替代方案是[将整个冰箱变成一个机器人](http://hackaday.com/2012/07/09/cool-master-advanced-beer-delivery-system/)。
