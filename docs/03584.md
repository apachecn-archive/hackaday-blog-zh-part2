# 外观简单的古董网络收音机有很多引擎盖下

> 原文：<https://hackaday.com/2013/07/14/simple-looking-antique-internet-radio-has-a-lot-under-the-hood/>

乍看之下，你可能甚至没有注意到这个 1934 年的收音机被改装过。但是仔细研究调谐度盘会提示你已经发生了变化。它仍然像原版一样滚动播放。但它不是一个上面有数字的轮子。旋转运动是由 LCD 屏幕产生的效果。

这是我们第二次看到(弗洛里安·安海因的)网络电台项目。[第一个](http://hackaday.com/2012/11/28/internet-radio-occupies-an-80-year-old-radio-case/)使用与 Arduino 配对的笔记本电脑的内脏将所有东西组合在一起。这一次，他选择挥舞树莓皮板。它为 USB 声卡提供了更好的音质。我们用一个小的放大器板来驱动收音机原格栅后面的一个大扬声器。

查看演示视频，看看无线电拨号的行动。令人高兴的是，他不厌其烦地模仿一个旋转圆盘，以保持该项目的主题。
