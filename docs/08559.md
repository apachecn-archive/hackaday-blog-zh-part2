# 一个马达，一个 Arduino 和一大堆激光切割

> 原文:[https://hack aday . com/2015/03/16/a-motor-an-arduino-and-a-whole-bunch-of-laser-cutting/](https://hackaday.com/2015/03/16/a-motor-an-arduino-and-a-whole-bunch-of-laser-cutting/)

[Guido]最近被委托为一个想要独特东西的客户建造一个动态雕塑。他想出的[真的很牛逼。](http://innovativeengineeringservices.com/orbis_kinetic_sculpture)

它叫做 ORBIS:木制动感和灯光雕塑。它安装在墙上，为房间提供了一个焦点——一个明亮的闪光旋转的焦点！它只是呆在那里做一些随机的事情吗？不，当然不是！[Guido]它有一个独特的控制盒，两个 Arduino 2560 和一个 Xbee 来进行通信。

[![Orbit Kinetic Sculpture](../Images/a83c225ebc4f1fbef0b851017fcc228c.png)T2】](https://hackaday.com/wp-content/uploads/2015/03/orbis-4-146182552_std.jpg)

他被告知要使用新旧技术来设计它，所以他在盒子的侧面有一个旋转电话拨号盘，允许用户通过不同的模式来改变。

顶部的开关还可以让你改变雕塑的颜色和移动的速度。由于是无线的，它可以很容易地放在咖啡桌上，成为一个即时的对话开始。

休息之后看看它的实际效果。
