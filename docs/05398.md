# 埃尔丝数码管就在你伸手可及的地方

> 原文:[https://hack aday . com/2014/02/24/El-wire-谢妮-tube-is-in-you-reach/](https://hackaday.com/2014/02/24/el-wire-nixie-tube-is-in-your-reach/)

![FUQ84C3HS0D05D9.MEDIUM](../Images/d4ef90258c5d19e022bef52f2755b4fe.png)

谢妮管很棒，但有时对一些制造商来说有点遥不可及，无论是获得它们，还是想出如何驾驶它们。H3 实验室的黑客空间决定尝试制造一种有趣的替代品——EL wire 谢妮电子管。

[Marty]在一个非常详细的说明书中带领我们完成构建，它使用了 CoolNeon EL wire。他用一个带酷霓护盾的 Arduino Uno 来控制它。这个构建最棘手的部分是形成数字以最小化重叠——为了解决这个问题，他在 Blender 中建模。他制作了一个测试夹具，在摆弄电线之前，先用衣架上的电线形成数字。

EL 线可以焊接在一起——这只是一种艺术，在另一个详细说明中有所解释。为了涂黑部分号码和尾部电线，[马蒂]使用了黑色塑料蘸。这些数字被安装在一个聚苯乙烯泡沫塑料圆筒上，这个圆筒装在一个大砖石广口瓶的底部。这是一个很好的构建，也是一个有趣的项目，可以进入 Nixies …而不用真正进入 Nixies。
