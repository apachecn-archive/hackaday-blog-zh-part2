# 冷战时钟全是电子管

> 原文:[https://hack aday . com/2014/07/22/cold-war-clock-is-all-tubes/](https://hackaday.com/2014/07/22/cold-war-clock-is-all-tubes/)

[![A clock built from tubes](../Images/6abbfd21fb793a919cd4bde52b178ed9.png)T2】](http://hackaday.com/2014/07/22/cold-war-clock-is-all-tubes/frontrt_small/)

时钟是很好的项目。它们服务于一个真正的目的，有各种各样的方法来实现一个独特的时计。[汉克]的[冷战时钟](http://wv7u.com/cwc/hourglass.html)只用了 1959 年就有的零件和技术。它不含半导体，但有声音警报和合理的时间准确性。

翻看手绘的原理图，你会发现一些[迪卡龙电子管](http://en.wikipedia.org/wiki/Dekatron)。这些老式部件被用作存储和计算时间的寄存器。[Hank]找到了一些便宜的苏联 Dekatrons，但是不得不自己加工插座来连接它们。这些电子管进行计数，但实际显示由[数码管](http://en.wikipedia.org/wiki/Nixie_tube)组成。

1959 年，一项成本估算显示这座钟的价格为 2130 美元，相当于今天的 17040 美元。显然，这超出了大多数业余爱好者的价格范围。实际建造成本大约为 1600 美元。

这座建筑有一些复杂的细节。前面板看起来很真实，手册上有“拆除时钟以防止敌人使用”的说明[汉克]称之为“创造性的时代错误”从某种意义上说，这是一个从未存在过的产品的复制品。
