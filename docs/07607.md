# 激光切割专辑发布

> 原文:[https://hackaday.com/2014/11/19/laser-cut-album-released/](https://hackaday.com/2014/11/19/laser-cut-album-released/)

在另一个世界，激光切割机和留声机比 MP3 播放器更常见，为你乐队的新专辑 ( [翻译](https://translate.google.com/translate?sl=auto&tl=en&js=y&prev=_t&hl=en&ie=UTF-8&u=http%3A%2F%2Fzurich.fablab.ch%2Flaser-cut-record&edit-text=&act=url))发布激光切割机文件是很有意义的。在这个宇宙中，它既古怪又可怕。

ASIC 的新 EP，别名来自 Fablab Zürich 的【Patric】，在以其他形式的数字下载发布之前，以 PDF 格式发布，而[的预告片视频](http://vimeo.com/109938873)(嵌入在广告下方)看起来棒极了。

这张专辑借鉴了阿曼达·加塞伊的指令性作品，将音乐转换成适合激光切割机的 pdf 格式，我们认为她在[唱片公司的发行页面上获得欢呼是很经典的。这张专辑的其他内容都将在知识共享许可下发布。](http://hulahoneys.net/releases/pentaprism-ep/)

[https://player.vimeo.com/video/109938873](https://player.vimeo.com/video/109938873)

刻录一张唱片对激光打印机的设置和校准是一个严峻的考验，而[Patric]仍在努力优化他的 Epilog 75 瓦激光切割机。他承诺一旦完善后会尽快公布细节。当然，你可以像正常人一样等一个星期，但这有什么意思呢？打开你的激光器。

感谢理查德的提示！
