# 百香果获得激光防御

> 原文：<https://hackaday.com/2012/08/21/passion-fruit-acquire-laser-defenses/>

![](img/1867d11c6f13e7c6e4cd66498ae8727b.png "passion-fruit-powered-laser")

显然，如果你住在夏威夷，被成熟的百香果淹没是个问题。赖安·K 解决这种情况的方法是用他多余的水果给激光器提供能量。在一个让[和【沃尔特·怀特】引以为豪的实验中](http://www.youtube.com/watch?v=gLZXC0ZFd08)，【瑞安】收集日常用品，以水果为基础组成电池。

他用一些镀锌螺栓作为锌的来源。它形成每个电池的一极，一根细铜管作为另一极。每个电池都相当脆弱，但当与其他电池结合在一起时，它就成了一个相当不错的电池。我们已经看到用酸性水果为发光二极管供电，但是赖安还想做得更多。他建立了一个电路，可以储存电力，直到他有足够的潜力为 LED 二极管供电。休息后，你可以看到一个四秒钟的水果挥舞着它的新激光防御系统的剪辑。
