# 杰克 DVD 抓取机器人

> 原文：<https://hackaday.com/2013/09/05/jack-the-dvd-ripping-robot/>

[安迪]手上有一个相当大的问题。在过去的 15 年里，他一直在收集 DVD，自从他开始收集以来，他的书架上已经没有空间放这些神奇的塑料光盘了。现在所有的东西都上了云，所以他决定建一个媒体服务器，里面装满了他所有的 DVD。任何试图翻拍电影的人都知道，这可能是一个非常漫长而乏味的过程。他对这个问题的解决方案应该是我们所有人都很熟悉的——[他决定制造一个机器人来自动抓取他所有的 DVD。](http://www.britishideas.com/2013/09/03/jack-the-ripper-bot-i-introduction/)

有了一台全新的 3D 打印机，[安迪]开始着手设计开膛手杰克机器人。该设计有两个安装在标准计算机 DVD 驱动器上的托盘，一个“入”托盘和一个“出”托盘。机器的框架直接固定在驱动器上，整个装置仅由三个标准的业余爱好伺服系统驱动。

机器人由树莓皮驱动，但实际上是在一台旧笔记本电脑上进行的。[Andy]说翻录一张 DVD 大约需要 1 小时 15 分钟，所以一个装满 24 张光盘的托盘意味着大约需要 28 小时的翻录时间。我们认为，每天给机器喂食一次，比每隔一小时左右回到电脑旁要好得多。

所有打印部分的 STL 和 Raspi 和计算机的软件都在[Andy]的 github 上，如果有人想升级到蓝光开膛手的话。
