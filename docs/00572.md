# 物理邮箱—邮件标志和所有

> 原文:[https://hack aday . com/2012/05/29/physical-email-box-mail-flag-and-all/](https://hackaday.com/2012/05/29/physical-email-box-mail-flag-and-all/)

![](../Images/4ae88353e609d75d0158a183f9533285.png "physical-email-box")

我们给你一个侧视图，因为我们真的很喜欢红色的新邮件标志。当然，它的工作方式与美国邮政信箱相反(旗帜告诉邮递员有外发邮件要取)，但这仍然是一种有趣的方式。你在这里看不到的是[这个实体邮箱](http://www.eraclitux.com/en/2011/11/25/comunikino-an-arduino-based-communication-system/)有一个字符 LCD 屏幕来阅读你的消息，顶部有一组按钮来发送回复。

[Eraclitux 的]项目将一个 Arduino、LCD、几个按钮和一个伺服电机放在一个金属项目箱内。它连接到他的电脑，通过 USB 电缆接受命令。Python 脚本是大部分奇迹发生的地方。如果您对使用 POP 和 SMTP 包将 Python 脚本与电子邮件服务器连接起来感兴趣，这是一个很好的参考项目。你的回答非常有限，只能用预编的信息来回答“是”、“否”或“已读”。但重要的是旅程，而不是目的地。
