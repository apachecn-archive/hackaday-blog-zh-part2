# 在不到一天的时间里制作一个纸板书架

> 原文:[https://hackaday . com/2012/07/27/一天之内做一个纸板书架/](https://hackaday.com/2012/07/27/make-a-cardboard-bookshelf-in-less-than-a-day/)

![](../Images/94fdbd544117753e62f8d6c2383b2c1c.png "cardboard-bookshelf")

卢卡斯想出了一个真正的赢家，他将纸板回收用作书架。它的视觉效果令人愉悦，基本上可以用胶水和安装支架的成本来建造，如果你以后决定放弃它，你也不必感到难过。

他节省下来的原材料成本都花在了劳动力上。这个项目中有 23 层不同的纸板，不包括每块纸板之间的间隔方块。休息后，剪辑中的大部分时间显示了他剪切图层的快速视频。这显然花了大约八个小时的切割，我们会想象他在所有这些工作之后有一只爪子。

这是挂在一个单独的 L 型支架上，支架位于方形开口处，用两个钉子保持水平。我们建议在你的设计中加入更好的安装技术。如果你对此有什么想法，请在评论中告诉我们。
