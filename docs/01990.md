# DIY 挑选和放置建立董事会，是真棒

> 原文：<https://hackaday.com/2012/10/29/diy-pick-and-place-builds-boards-is-awesome/>

![](img/ceeb5ae971eb9d000b8997d3be1f3603.png "PNP")

可能是因为放置大量 SMD 元件的痛苦，【gravelrash】建造了自己的[自制拾放机](http://hairyraho.com/home-made-pick-place-smt-machine)。

没有因为镊子、立体显微镜和过度的专注而沮丧，[grave rash]用中国的 CNC 路由器制造了一台拾取和放置机器。这款产品的零件卷轴没有使用自动送料器。相反，[gravelrash]从易贝获得了五个手动进料器，使他的抓放器能够容纳 25 个不同的元件卷轴。

当然，有一个用于吸取 SMD 部件的真空泵和一个能够将元件从卷轴移动到电路板的双轴门架。该软件是 Mach3，一个通常与旋转刀具一起使用的程序，用于铣削木材、金属和塑料。[gravelrash]用一些真空控制的针代替了这个马达，以拾取、移动和放下元件到板上。

虽然构建速度可能没有我们见过的其他一些拾取和放置机器快，但它几乎和手工放置组件一样快，而且不会因为非常小的零件而紧张。

向[亚历山大]致敬，因为他送来了这个。
