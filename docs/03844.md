# Nerf，气枪等的反应目标范围。

> 原文:[https://hack aday . com/2013/08/14/reactive-target-range-for-nerf-airsoft-etc/](https://hackaday.com/2013/08/14/reactive-target-range-for-nerf-airsoft-etc/)

![reactive-target-range](../Images/ff04bed5f6788d25e02b09869f73645e.png)

花时间[建造一个反应目标范围](http://www.superfunadventurequesttime.com/projects/2013/reactive_targets/)确实增加了玩具武器的乐趣。它能让你超越简单的瞄准和射击，进入真正的技能游戏。

该项目由 Arduino 板支持。它连接到每个金属片靶背面的压电元件上。探测投射物何时击中目标的工作方式与曾经流行的撞击块的工作方式几乎完全相同。为了提供互动的乐趣，每个目标都有一个 LED，当它点亮时，表示目标是活动的。从这里开始，只是一个添加不同挑战的编码问题。到目前为止,[维克多标准]已经实现了快速抽签，定时和快速射击模式。休息后的演示展示了一切，包括他想出的使系统便携的巧妙模块化设计。

我们希望看到这些目标安装在机动轨道上。每一轮都有目标以更快的速度向你靠近，让你保持警觉。
