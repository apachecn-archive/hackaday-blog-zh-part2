# 反恐精英气枪机器人

> 原文:[https://hack aday . com/2014/08/24/the-counter-strike-airsoft-robot/](https://hackaday.com/2014/08/24/the-counter-strike-airsoft-robot/)

![F6R9THZHR4SR0HV.MEDIUM](../Images/cac0a36ec77140e1a518aedd961efa2d.png)

[乔恩]和他的兄弟将一辆遥控汽车改装成一个机器人，可以向空中发射气枪子弹。这辆小型机动车辆被拆开，顶部安装了一个手持设备。一个拉动装置被安装到位，并且增加了一个安全程序以确保没有事故发生。

机箱支架是用来固定手柄的。然后在这一点上对设置进行了测试，Raspberry Pi 服务器配置了一个摄像头，作为机器人的眼睛。一旦一切就绪，车轮着地，车辆就能四处移动，定位自己，将伺服系统瞄准指定的目标。镜头通过网络传输，展示了机器人在看什么。

休息之后可以看到遥控反恐精英机器人的视频。你可以把这当成你的玩具军队。那[使这个成为你的玩具空军](http://hackaday.com/2013/10/16/pov-airsoft-turret-on-a-rc-plane/)。
