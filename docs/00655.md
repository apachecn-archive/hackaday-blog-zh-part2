# 气动汽车发射器实际上不仅仅是黑客的幻想

> 原文:[https://hackaday . com/2012/06/06/pneumatic-car-launcher-is-just-a-hackers-fantasy/](https://hackaday.com/2012/06/06/pneumatic-car-launcher-is-actually-more-than-just-a-hackers-fantasy/)

![](../Images/7403adc49334be0bbe23d896fa1ae40c.png "pneumatic-car-launcher")

所以，如果你必须发动一辆车通过这些球门柱，你会怎么做？当然不是用弹弓(虽然我们可能会用蒸汽动力弹射器，就像航空母舰一样)。由于一个非常强大的气动发射器，那辆带有白色条纹的栗色汽车即将进行飞行。

我们通常不参加拆除比赛，所以这种事情对我们来说是新的。我们更惊讶地发现，这并不完全是一种新奇的行为。这个系统实际上是电影中一些更危险的汽车特技是如何完成的。这次发射使汽车在一个优雅的弧线上准确无误，让我们想起了《T2》和《飙风天王》中扣人心弦的特技(电视，不是最近的电影)。但是，如果你是一个寻找可怕车祸的导演，可以改变装置，让汽车在空中盘旋。听起来记录是在空中飞了将近八圈。

我们没有多余的汽车可以推出，所以我们将坚持[让五彩纸屑飞舞](http://hackaday.com/2010/11/08/build-a-confetti-cannon-for-your-next-party/)。
