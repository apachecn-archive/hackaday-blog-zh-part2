# 任何地方都有鼓！

> 原文:[https://hackaday.com/2015/01/16/drums-anywhere/](https://hackaday.com/2015/01/16/drums-anywhere/)

康奈尔大学电子和计算机工程学院的学生在他们的高年级项目中再次努力。这一次，它是由它的创造者[Shiva Rajagopal]和[Richard Quan]设计的被称为“随处鼓”的[。由于还有其他高度便携的乐器，如卷起式钢琴，他们认为应该有一个便携式鼓套件，实际上听起来像鼓，这个 ECE 二重奏击中了隐喻和物理鼓的头部……除了这个项目实际上不使用物理鼓来发声。](http://people.ece.cornell.edu/land/courses/ece4760/FinalProjects/f2014/svr24_rq32/svr24_rq32/index.html)

该项目由两个 3D 打印的盒状传感器组成，传感器带有 velcro 带，可以附着在任何可能躺在周围的鼓槌状物体上。盒子里面是一个柔性传感器和一个微型麦克风，当它们撞击另一个物体时，会向微控制器报告“心跳”。

在软件方面，微控制器中存储了两种采样声音，但他们计划在未来添加更多声音。微控制器向一对扬声器输出声音，传感器对力很敏感，所以音量可以从几乎听不见一直到[约翰·博纳姆]式的轰鸣。理论上，这也可以扩展到包括两个以上的额外声音的“节拍盒”，或者是无线的。选项实际上是无限的，尽管该团队指出，它们受到特定微控制器 ATmega1284 上中断和 ADC 转换器数量的限制。

这是另一个有趣的没有鼓的有鼓组的场景，并且无疑扩大了虚拟鼓组的功能范围。看到来自高级设计班的有趣项目也很棒！休息后请务必观看视频。
