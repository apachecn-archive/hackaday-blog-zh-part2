# 浏览器中的试验板

> 原文:[https://hackaday.com/2015/07/20/a-breadboard-in-a-browser/](https://hackaday.com/2015/07/20/a-breadboard-in-a-browser/)

[Flownez]发送了一个提示，古老的 Falstad 电路模拟器的一个端口现在[可用，不需要 Java](http://lushprojects.com/circuitjs/) (它使用 HTML 5)。这是一个受欢迎的端口，因为一些现代浏览器(尤其是 Chrome)使得运行 Java 小程序变得困难，并且阻止了 Falstad 模拟器的执行。

与最初的模拟器一样，这款模拟器非常适合展示教室电路，并鼓励在浏览器中构建或学习电路。不需要安装额外的软件，这对于即兴演示来说非常方便。另一个很酷的功能是将电流可视化为动画点。这些点沿着电流的方向移动，移动的速度与电流的大小成正比。用移动的点来观察电容器充电非常有说明性。您还可以查看范围格式的数据或将鼠标悬停在事物上以读取它们的值。

您可以打开一个空白电路并添加相当多的元件(使用鼠标右键或菜单来添加元件和电线)。然而，您也可以从许多预定义的电路中挑选，从简单的(例如分压器)到说明性的(想到 PLL 倍频器)。甚至还有一个 AM 收音机(见下文)，你可以通过改变调谐电容器的值来调谐以找到几个“电台”。电路元件包括许多类型的模拟和数字元件。
