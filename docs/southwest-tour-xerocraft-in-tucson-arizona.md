# 西南之旅:亚利桑那州图森的 Xerocraft

> 原文：<https://hackaday.com/2012/08/12/southwest-tour-xerocraft-in-tucson-arizona/>

[https://www.youtube.com/embed/T01tYscq_NM?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/T01tYscq_NM?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

穿过亚利桑那州继续向南旅行，我们在图森的 xero craft 与[康纳]会面。[Connor]立即为商店略显凌乱的状态道歉，但正如我告诉他的那样，我更喜欢用过的商店。Xerocraft 是我们[西南之旅](http://hackaday.com/?s=southwest+tour)中有趣的一站。在前门厅有一个几乎是户外的工作区，这是我在密苏里州不习惯看到的。室内区域包括他们的电子设备、图书馆、3d 打印和厨房区域。一个漂亮的小后院可以让一些更大的工作得以完成，比如你在视频中看到的那辆车目前正在进行的电子转换。

如果你曾经在这个地区，你一定要顺便去看看，至少尝试在《超级粉碎兄弟》中取代[康纳]的位置。
