# 为 3D 打印机框架使用电气导管

> 原文:[https://hack aday . com/2013/06/12/using-electrical-conduit-for-a-3d-printer-frame/](https://hackaday.com/2013/06/12/using-electrical-conduit-for-a-3d-printer-frame/)

![electrical-conduit-for-cnc-frame](../Images/81995abdb3bd870fe48b48d2f086913e.png)

我们一直在寻找可以在当地采购且价格低廉的零件。这个黑客符合这两个标准。【李·米勒】想出了一个办法[用钢电导管作为 3D 打印机的框架](http://boulderhackerspace.com/2013/06/08/introducing-the-uconduit-h-bot-3d-printer-reprap/)。他最近完成了上图中的设备，并在科罗拉多州博尔德的一个黑客空间 Solid State Depot 展示了它，他是那里的成员。

仔细观察这张图片中框架的角落，您会看到组成夹紧装置的 3D 打印零件。每个都有三个螺旋在一起的组件。两个间隙之间各有一个橡胶 O 形圈。当塑料夹拧在一起时，它们会挤压牢牢固定电线导管的环。这也有抑制振动的副作用。

当然很容易找到这种类型的导管，在每个家庭商店(和大多数五金店)都有卖。当你挑选的时候，确保你检查了一块是直的。零件的 SCAD 文件可从[他的 Github repo](https://github.com/iquizzle/uconduit) 中获得。
