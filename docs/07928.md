# 示波器和光电二极管的钟摆音乐

> 原文:[https://hack aday . com/2014/12/28/pendulum-music-for-示波器和光电二极管/](https://hackaday.com/2014/12/28/pendulum-music-for-oscilloscope-and-photodiodes/)

两个唱盘和一个麦克风？试试[两个示波器和一对光电二极管](https://www.youtube.com/watch?v=b0AEcanL7D0)。[dfiction]为更现代的电子产品重新诠释了史蒂夫·赖克的经典反馈作品。视频在休息后嵌入。

最初的[钟摆音乐](http://www.youtube.com/watch?v=fU6qDeJPT-w&t=2m57s)是 1968 年的一首概念音乐作品。基本上，你设置一堆麦克风在扬声器上摆动，当它们经过时发出反馈。随着挥杆的衰减，产生的节奏和音调振荡会随着时间而改变。它要么令人着迷，要么完全无聊，这取决于你的心态。

在[dfiction]版本中，反馈是通过在示波器上传递一个“光麦克风”来产生的。因为他有一对这样的装置，一个麦克风也为另一个“示波器”提供信号。由此产生的声音是这种混乱和粗糙的噪音-隆隆声。我们喜欢。

如果缓慢发展的“过程音乐”突破了你的注意力范围(或者如果它不适合你)，你完全可以跳过这段视频。尝试在 [1:40](http://www.youtube.com/watch?v=b0AEcanL7D0&t=1m40s) 和 [3:45](http://www.youtube.com/watch?v=b0AEcanL7D0&t=3m45s) 左右进入该片段，以了解正在发生的事情。但是一旦你到了那里，你最好让它顺其自然。
