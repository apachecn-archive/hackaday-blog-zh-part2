# 电动割草机转换为遥控控制

> 原文：<https://hackaday.com/2015/06/28/electric-lawn-mower-converted-to-rc-control/>

没有太多人喜欢在院子里推着割草机，但不幸的是，这是一项必要的家务。反推式割草机倡导者[imadethis2014]决定(正如他的绰号所暗示的)为他的推草问题制定一个[解决方案。他是无线电控制设备的粉丝，所以把他的割草机换成遥控是有道理的！](http://www.reddit.com/r/electronics/comments/3bbygt/i_decided_mowing_the_lawn_should_be_more_rc_and/)

割草机本身由现成的电池供电，有 21 英寸的甲板。库存轮和处理被删除，并取代了新的挤压铝框架。后面是一对来自易贝的二手电动轮椅马达。这些驱动电机通过一对铝支架安装在新的框架上，这对铝支架是[imadethis2014]在他的 Shakepoko 数控机床上设计和切割的。由于两个后驱动轮相互独立工作，前端的一对旋转脚轮允许割草机转向。

一个大型汽车大小的电池盒容纳了两个较小的轮椅电池以及电机控制和遥控电子设备。休息后看看视频，割草机似乎做得相当好。[imadethis2014]承认他需要一些新的轮子，因为目前的轮子在草地上没有太大的牵引力。他还在考虑为自动割草添加 GPS，但目前还没有实现。
