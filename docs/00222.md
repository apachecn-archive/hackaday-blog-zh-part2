# 电话很小，是自制的

> 原文:[https://hackaday . com/2012/04/26/% C2 % b5 手机是小而自制的/](https://hackaday.com/2012/04/26/%c2%b5phone-is-small-and-home-made/)

![](../Images/eed69ffba521821d603699ff94c2ee4b.png "obligatory zoolander joke goes here")

[Victor]在昨天的 DIy 手机评论中出现，展示他自己制作的手机，[电话](http://elektronikforumet.com/forum/viewtopic.php?f=3&t=53000) ( [谷歌翻译](http://translate.google.com/translate?sl=sv&tl=en&js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&u=http%3A%2F%2Felektronikforumet.com%2Fforum%2Fviewtopic.php%3Ff%3D3%26t%3D53000))。[Victor]努力让这东西变得非常紧凑。正如你在休息后的视频中看到的那样，为了节省空间，他甚至去掉了数字键盘。相反，你用一个小操纵杆和两个按钮来做任何事情。他声称他真的只给一小部分人打电话，所以一旦他把他们的联系信息编入程序，这种布局就很好。

出于某种原因，没有一张图片出现在这个项目的论坛帖子里。不过，跟着他试着让这个东西运转起来还是很有趣的。
