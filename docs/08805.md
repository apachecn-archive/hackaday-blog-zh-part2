# 用机器人拍球

> 原文:[https://hack aday . com/2015/04/16/用机器人弹跳球/](https://hackaday.com/2015/04/16/bouncing-a-ball-with-a-robot/)

作为 kinetic 项目的粉丝，[Jimmy Demello]组装了一个非常酷的机器人，它可以用手来回拍球。

它是由房子周围的废料制成的，由一些木头、一根钓鱼竿、chop sticks、一个伺服电机和一个 Arduino Uno 组成。他最初的计划是制造[克劳德·香农]的[杂耍机器人](https://www.youtube.com/watch?v=sBHGzRxfeJY)，但这是他目前为止所能做到的——至少到目前为止。

它在接住一个 2 厘米的滚珠并再次传回之前，先将它弹起两次。他认为扩大规模并使用篮球将是一个很棒的项目，我们不得不同意。
