# 与家庭搅拌焊接混在一起

> 原文：<https://hackaday.com/2015/07/03/getting-mixed-up-with-home-stir-welding/>

大多数设计用来将两件现有物品连接在一起的过程都是消耗性的，无论是作为工件和工具的第三方的某种材料，还是工具本身。在搅拌摩擦焊的奇妙世界中，当工具穿过连接路径时，处于结合状态的两个零件的材料通过摩擦旋转在一起。当然，有专业的机器可以相对轻松地完成这项工作，但由于生产线上有大量啤酒，[skookum_choocher]决定[自己制作](http://www.youtube.com/watch?v=L4c5WaYCt8A)。

在第一个视频中，他使用金刚石研磨机将钻头加工成碳化钨按钮，从而加工出一个摩擦焊接工具。一旦他有一个粗糙的肩膀和突起，是时候让她撕裂。尽管夹紧和他的工具的几何形状的问题，焊接最终在尾端成功。

他没有被吓倒，在对工具肩部进行了一些调整，更换了他可怜的旧 Bridgeport 上的皮带，并将夹紧强度增加了四倍后，他又开始了另一次尝试。你夹住 16 吨，你得到了什么？事实证明，这是一个比第一次稍好的对接焊缝。由于担心这种焊接不足以赢得赌注，他选择了搭接焊接，将工件叠放在一起做成三明治。我们更喜欢比萨饼和啤酒，但还是要祝贺他。
