# 收听 Cube Sats 的简单方法

> 原文:[https://hack aday . com/2015/06/29/easy-way-to-listen-cube-sats/](https://hackaday.com/2015/06/29/easy-way-to-listen-to-cube-sats/)

比尔·米拉 发现了一种简单的方法来[使用廉价的 SDR 加密狗收听业余“立方体卫星”卫星](http://soldersmoke.blogspot.com/2015/06/back-with-sats-catching-cubes-with.html)。

DVB-T SDR 加密狗售价高达 13 美元，[高度复杂的天线](https://www.arrl.org/files/file/Technology/tis/info/pdf/ab18-16.pdf) (pdf)由一点铜线制成，并使用铝线作为接地层。

一旦一切就绪，[比尔]就上了网站上的[天堂，看看卫星什么时候会经过他的上空。他锁定了棱镜卫星，还有几个从俄罗斯和伊斯坦布尔发射的立方体卫星。](http://www.heavens-above.com/AmateurSats.aspx)

当在 470 兆赫收听日本 FO-29 卫星时，多普勒频移很明显，这证明他确实调谐到了低地球轨道上的一个设备。请务必查看他的博客了解更多细节，更不用说下面的视频，你可以亲自听到多普勒频移。
