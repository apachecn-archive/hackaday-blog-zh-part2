# 六足直升机

> 原文:[https://hackaday.com/2012/12/18/the-hexapod-hexacopter/](https://hackaday.com/2012/12/18/the-hexapod-hexacopter/)

[![Hexapod Hexacopter](../Images/c8e83b010e1b7fb9823e600ea3a58d1a.png)T2】](http://hackaday.com/?attachment_id=91876)

在 Mad Lab Industries，他们有了建造一架可以行走和飞行的四轴飞行器的想法。通过将一架[六足机器人和一架](http://madlabindustries.com/hackerspace/projects-2/hexapod-quadcopter/ "Hexapod Hexacopter")六足直升机结合起来，他们完成了这项发明。

六足部分从 [PhantomX 六足套件](http://www.trossenrobotics.com/phantomx-ax-hexapod.aspx)开始，但它太重了，无法飞行。为了减轻重量，他们为框架和腿制造了碳纤维部件。即使减轻了重量，他们仍然需要六个转子来保持稳定。

建造的 hexacopter 部分使用更多的定制碳纤维部件来安装马达。吊杆和支架也是由铝制成。他们使用六个电动马达、螺旋桨和电子稳定控制系统来提供升力。

各种控制器被用来运行机器人。两个 Arbotix 设备处理六足控制，一个 Hoverfly 飞行控制器保持它在空中。它是用 Spektrum 控制器远程控制的。

他们有一些雄心勃勃的下一步措施，包括断开和重新连接六角直升机和基地的机制。休息之后，看看这个令人印象深刻的视频。
