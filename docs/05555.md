# 侵入百叶窗与家庭自动化系统接口

> 原文:[https://hack aday . com/2014/03/16/hacking-window-blinds-to-interface-with-home-automation-system/](https://hackaday.com/2014/03/16/hacking-window-blinds-to-interface-with-home-automation-system/)

![dscn0248](../Images/6e51d249e392ba5bb004dbcaf5ca3a2c.png)

家庭自动化很棒，但是当你开始在家里将不同的系统混合在一起时会发生什么呢？跟随[Bithead]的旅程，与他的电动百叶窗互动！

在他原来的百叶窗坏了很多次后，[Bithead]和他的妻子决定投资一些新的电动百叶窗，但[Bithead]想把它添加到他的家庭自动化系统中…不幸的是，商业产品非常昂贵，所以[Bithead]知道他必须想出如何手动连接系统。

问题是，公司通常不会宣传美国黑客喜欢了解的关于产品的深度信息，所以[Bithead]开始查看商店展厅。销售人员不太理解他对控制盒的关注！

在确定了某个特定品牌**应该为他的目的服务后，他咬紧牙关买下了它们——现在没有回头路了！然而，他很幸运，因为实际上有一个销售人员不知道的控制接口，他所要做的就是在他的网络控制中路由。**

为此，他利用了他的另一个项目中剩余的 Arduino Fio、几个 3V 簧片继电器和 Xbee S6B，让 Arduino 可以通过房子的 WiFi 进行通信。他在盲注到达之前就准备好了整个系统。他甚至成功说服公司的安装人员在他们安装百叶窗之前帮他进行改装！
