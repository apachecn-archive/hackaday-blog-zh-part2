# 未来不需要另一个联网的泰迪熊 SXSW Create 的 Hackaday

> 原文:[https://hackaday . com/2014/03/12/the-future-dont-need-another-internet-connected-teddybear-hackaday-at-sxsw-create/](https://hackaday.com/2014/03/12/the-future-doesnt-need-another-internet-connected-teddybear-hackaday-at-sxsw-create/)

[https://www.youtube.com/embed/4HVSDYUV9_0?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/4HVSDYUV9_0?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)T2】

Hackaday 今年碰巧在西南偏南的[参观了](http://sxsw.com/)[SXSW Create](http://www.sxsw.com/interactive/create)——这是致力于黑客、制造者和 DIY 场景的节日的一部分。虽然规模不大，但这一事件与这段时间在纽约其他地方无处不在的互联网炒作形成了鲜明对比。所以我们觉得应该顺道拜访，向他们表达爱意。

Trey German 向我们展示了几个很棒的实时电源控制演示，使用他的 C2000 发射台和他的蓝牙冷却器，无论出于什么原因，这些设备都决定在大秀开始时失败。我们最期待的演示是使用 [Energia 框架](http://energia.nu/)的热电偶控制烧烤，但我们失望地得知该男子禁止在帐篷里烧烤热狗。宇宙告诉我们，我们不是来参加派对的。

[![atx_hackerspace_TARDIS](../Images/eb0ca37a681021c6038daabb298354d2.png) ](http://hackaday.com/wp-content/uploads/2014/03/atx_hackerspace_tardis.jpg) ATX 黑客空间有一个大展台，展示了神秘博士的 TARDIS 的全尺寸复制品(谁不想在其中拍张照呢？)和一个功能齐全的 1930-es 真空管收音机，带有一个强制性的 iPad 基座。我们还了解到，大量工作中的老式真空管已被捐赠给 hackerspace，所以如果你有需要，你知道该找谁。

该活动还有一长串[行业参与者](http://www.sxsw.com/interactive/create)。我们印象最深的产品发布是 Easel by Inventables，这是一款浏览器内应用程序，可以轻松控制他们的 Shapeoko 数控铣床，绝对有可能为更多人带来设计和制造的乐趣。

[![e_coli_ut_austin_experiment1](../Images/011bd820771a1363c22dea91fbff3bf2.png)](http://hackaday.com/wp-content/uploads/2014/03/e_coli_ut_austin_experiment1.jpg) 然而，我们看到的最有趣的东西是那些有点脱离当前科技主流的东西。来自 [UT Austin iGEM 团队](https://www.utexas.edu/research/eureka/project/view?project_id=107)的【Dennis】展示了合成生物学家正在做的一些疯狂的工作。他们改造了大肠杆菌，使其对咖啡因上瘾，用细胞生长来衡量特定饮料中的咖啡因含量，并以此对奥斯汀当地的咖啡店进行排名！我们还与几个从事自动化花园和土壤传感器工作的人进行了交谈，他们正在教育与会者增加环境数据聚合对我们种植食物的方式可能产生的巨大潜力。

引用 Growerbot 公司的一句话:“我们肯定有足够多的联网泰迪熊。我们需要更多联网的番茄植株”。

跳跃之后，请继续关注我们，在 2014 SXSW 上观看我们所有冒险的画廊。
