# 黑客日奖参赛作品:终极电路弯曲

> 原文：<https://hackaday.com/2015/06/24/hackaday-prize-entry-ultimate-circuitbending/>

电路弯曲是这样一个过程，拿一个小的电子玩具或乐器，将电线焊接到 PCB 上的焊盘上，并希望它产生的声音很酷。无论如何，这都不是一门科学，你从 Speak 'N 咒语或老式 MIDI 键盘中发出的任何美妙、怪异的声音都完全是偶然的，或者是长时间的实验造成的。

[阿尔法·查理]的黑客日奖参赛作品[是你见过的技术最先进的电路弯曲](https://hackaday.io/project/6389-the-rythm-composter)。他使用的是一个老式的数字节拍盒，Roland TR-626，在 PCB 上的随机焊盘之间有计算机控制的电线。

到目前为止，你可以简单地通过弯曲电路的仪器上有多少开关来判断弯曲电路的技术熟练程度。[阿尔法查理]不需要开关。相反，他使用[几个交叉点开关 IC](http://www.intersil.com/en/products/audiovideo/video-switching/unbuffered-crosspoint-switches/CD22M3494.html)将 TR-626 的 PCB 上的不同引脚和焊盘与 Arduino 连接起来。所有这些[都由一个触摸屏显示器](https://hackaday.io/project/6389-the-rythm-composter/log/19810-patch-a-tron-walkthrough)控制，实验电路就像按几个按钮一样简单。每个“弯”都是由电脑控制的，可以随意保存和调用。

当然，如果听起来像废话，电路弯曲对任何人都没有好处。[阿尔法查理]不必担心那里。在下面的视频中，他听到了一些非常独特的声音，对于像我这样听任天堂音乐的傻瓜来说，听起来像是天使合唱团。
