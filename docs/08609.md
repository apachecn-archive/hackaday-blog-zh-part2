# 用你的肌肉控制东西

> 原文：<https://hackaday.com/2015/03/23/control-stuff-with-your-muscles/>

[David Nghiem]多年来一直致力于设计用于读取肌肉信号的电路。在一家初创公司遭遇厄运后，他没有放弃，继续研究自己的想法。他决定以一种能够读取肌肉信号的可穿戴套装的形式与黑客社区分享他的创新。

事实证明，当你弯曲肌肉时，它会发出一种叫做表面肌电信号的信号，简称 SEMG。[David]正在使用 Arduino、数字电位计和一系列运算放大器来读取 SEMG 信号。发光二极管用于显示信号电平。

[大卫的]项目背后的历史可以追溯到二十世纪末，他雄辩地指出——“天哪，那是很久以前的事了”。他与麻省理工学院航空航天实验室和波士顿大学神经肌肉研究中心合作，在那里他为宇航员研制机械臂。这个想法是在手臂上施加一个相反的力来帮助防止肌肉退化。

请务必查看[David]的大量有据可查的工作，以及显示他的项目处于不同完成阶段的几个视频。如果这给了你肌电图的 bug，看看这个关于检测信号的[指南](http://hackaday.com/2011/06/29/detecting-muscles-with-electromyography/)和机器人假肢概念的[应用。](http://hackaday.com/2010/05/04/emg-controlled-prosthesis/)
