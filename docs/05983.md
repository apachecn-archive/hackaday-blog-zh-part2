# 圆锯切球…木球，不用担心

> 原文:[https://hack aday . com/2014/05/06/circular-saw-cuts-balls-wood-balls-dont-worry/](https://hackaday.com/2014/05/06/circular-saw-cuts-balls-wooden-balls-dont-worry/)

想知道木头球是怎么做的吗？通常情况下，它们是在木制车床上用一些花哨的夹具和固定装置制成的。[伊西·斯旺]设计了一种廉价的机器，使用标准圆锯将一块木头雕刻成球体，开始将木质球体制造带给大众。

它是这样工作的:一块木头被固定在一个木头夹具上，用手钻旋转。夹具和钻头安装在一个绕垂直轴旋转的木环上。当旋转钻头时，使用者手动地将整个组件绕第二轴线来回移动。与此同时，圆锯越来越靠近即将形成的球体，一点一点地蚕食。从木块上切下大部分材料后，将木块从夹具上取下，旋转 90 度，切割剩下的两个小块。最终的结果是一个非常漂亮的球体。

![Wood Sphere Finished](../Images/4f619348e9d798e70cd108eae90b3baa.png)

这台机器除了一般 DIY 者在商店里随意摆弄的东西之外，没有什么特别之处。如果你想做一个木制球体，但没有圆锯或不想失去手指，没问题，试试这个使用轨道打磨机的[方法。](http://hackaday.com/2014/01/24/fun-with-wooden-balls/)
