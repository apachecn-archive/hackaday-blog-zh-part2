# 遥控割草机让你坐下来享受表演

> 原文：<https://hackaday.com/2014/02/22/remote-controlled-lawn-mower-lets-you-sit-back-and-enjoy-the-show-2/>

"很难找到真正想修剪草坪的人。"从来没有比这更真实的说法了。[Kurt]的项目将一台旧割草机变成了一台遥控割草机。

这种构建的第一步是用内置齿轮齿链轮的迷你自行车轮胎取代前驱动轮。后轮随后被大脚轮所取代。使用的 12-24V DC 电机和齿轮箱来自 National Power Chair。虽然我们以前见过更复杂的遥控割草机，但是这个项目是一个很好的开始。库尔特想要的只是让割草变得更有趣，我们相信他已经成功了。这东西机动性很强，一毛钱就能转弯。休息后请观看演示视频。

下一步是什么？添加一个全球定位系统，一个树莓皮，和其他一些零碎的东西。将它与一些聪明的程序结合在一起，你将拥有自己的自动割草机。你已经造出一台完全自主的割草机了吗？[让我们知道](http://hackaday.com/contact-hack-a-day/)！
