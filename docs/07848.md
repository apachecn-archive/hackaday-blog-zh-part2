# 更有趣的即时相机

> 原文:[https://hack aday . com/2014/12/18/朝向更有趣的即时摄像机/](https://hackaday.com/2014/12/18/towards-more-interesting-instant-cameras/)

当[Ch00f]在 Nordstroms 买牛仔裤时，他注意到热敏收据打印机在纸上打印图像的速度有多快。这种观察并不是[Ch00f]经常光顾的圈子所独有的——有一些小型收据纸打印机可以连接到互联网、iPhones 和其他一大堆 Kickstarter 友好的关键字设备。

然而，一种可以快速廉价地制作图像硬拷贝的设备并不是你不再考虑的东西。这个概念在他脑子里转了几年后，[Ch00f]终于想出了一个完美的构造—[一台相机](http://ch00ftech.com/2014/12/17/printsnap-instant-camera/)。

构建的硬件基于 STM32F4 发现板。对于这种应用程序来说有点太强大了，这是[Ch00f]在 ARM-land 的第一次冒险之一。其余的硬件包括一台热敏收据打印机和一台 JPEG 相机，后者替换了一个移动中丢失的手机 CMOS 相机模块。

一台定制的相机需要一个定制的外壳，为此[Ch00f] [做出了非凡的](http://ch00ftech.com/2014/12/17/printsnap-instant-camera/4/)。整个外壳由一块漂亮的印花胡桃木数控铣削而成。最终结果看起来对原型来说太好了，但是用一点亚麻籽油就可以很好地润色。

现在[Ch00f]有了一种即时照相机，它可以将宝丽来的想法转化为以十分之一美分的价格打印出来的东西。有一个延时功能——只是快门按钮上的一个拉链——在荧光笔的帮助下过滤，以及以动画书格式录制电影的能力。

这是一个伟大的项目，也是一个伟大的众筹活动。[Ch00f]已经开始了这方面的工作。他已经有了[一个时尚现代的网站](http://ch00ftech.com/printsnap/),它需要过多的滚动，这是赢得 Kickstarter 的第一步。[Ch00f]还学习了很多关于 arm、DMA、抖动、伽马校正和 JPEG 格式的知识，但这不会让任何人打开他们的钱包。你知道什么会吗？流畅的视频。你会在下面找到。
