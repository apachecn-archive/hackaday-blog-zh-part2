# 交通摄像对抗

> 原文：<https://hackaday.com/2012/10/23/traffic-camera-countermeasure/>

不要误解我们，我们开车非常小心，因为这是我们经常做的最危险的事情。但是即使是一个小心的司机也可能被糟糕的交通和闯红灯的摄像机抓住。这些是监控十字路口的设备。如果红灯亮的时候你被夹在中间，他们会给你拍照，你会收到一张罚单。嗯，以前就是这样。这个交通摄像头对策就把它给终结了。如你所见，[no photo 使用自己的闪光灯过度曝光交通摄像头图像](http://www.nophoto.com/pages/how-it-works)。

上图显示的是原型。使用闪光传感器作为触发器，箔片将两边的闪光反射到牌照上。根据休息后的演示视频，该系统甚至可以击败预闪光和双照片类型的相机。

在你的车上使用灯光有非常严格的限制(颜色，位置等等。).我们想知道这是否符合要求？
