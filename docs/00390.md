# 黑客空间简介:荷兰恩斯赫德的 TkkrLab

> 原文:[https://hack aday . com/2012/05/11/hacker space-intro-tkkrlab-in-enschede-Netherlands/](https://hackaday.com/2012/05/11/hackerspace-intro-tkkrlab-in-enschede-netherlands/)

![](../Images/7a479134427af27d3c1480eefcbd898b.png "hackerspace-intro-tkkrlab")

[Dave Borghuis]组织了一次在荷兰的 [TkkrLab 黑客空间](http://www.tkkrlab.nl/)的快速视频游览。

参观从入场系统开始。去年年底，我们调查了这个项目。空间总面积约 150 平方米(约 1600 平方英尺)，分为三个房间。有一个“脏”房间，里面放着一些木工和一般制作工具。另一个房间用于电子产品原型制作，包括 3D 打印机和数控铣床。最后，还有一个大会议室，它可以作为一个休闲室，让您在项目中工作、看电影或举办活动。

周二晚上有每周例会，对所有人开放。还提供了一系列课程，涵盖诸如学习 Python 或 Arduino 介绍等主题。
