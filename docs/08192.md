# 数字控制电路弯曲

> 原文：<https://hackaday.com/2015/01/30/digitally-controlled-circuit-bending/>

在我们经常使用的互联网的某些地方，电路弯曲并没有得到太多的尊重，但它确实是一种艺术。典型的例子[是我们见过的最不可思议的电路弯曲](https://www.youtube.com/watch?v=sA46_QxmvSE)。是的，它是将电线焊接到 PCB 上看似随机的点上，但这些弯曲点是数字控制的，只需按几下触摸屏控制器，就可以让鼓机在弯曲脆脆和经典的 20 世纪 80 年代鼓机之间转换。

所有的电路弯曲都必须从一件有趣的设备开始，对于这个项目，这部电路弯曲杰作的创造者[Charles]正在使用一台 [Roland TR-626](http://www.vintagesynth.com/roland/626.php) ，这是 TR-606 的一个稍微更现代的版本，是臭名昭著的 TB-303 的打击对手。电路以经典方式弯曲，将 PCB 上的信号连接到地、VCC 或板上的其它信号。[Charles]然后通过将这些电线连接到由 Arduino Mega 控制的 384 个模拟开关上，其他人都可以做到。Arduino 上还有一个触摸屏，通过一个光滑的用户界面，这台旧鼓机可以数字化弯曲，不需要大量的拨动开关。

[Charles]发布了几个视频，介绍了这款触摸屏电路弯鼓机的结构、功能和声音。这是一件了不起的作品，从这一点上提高了每个弯音模的标准。
