# 树莓 Pi 上的 Android 4.0

> 原文:[https://hack aday . com/2012/08/01/Android-4-0-on-raspberry-pi/](https://hackaday.com/2012/08/01/android-4-0-on-raspberry-pi/)

![](../Images/635a9c20037e327248977aef0636c851.png "android-4-ics-on-rpi")

树莓基金会的人正在展示他们的最新产品。这一次是运行在树莓平台上的 Android 4.0。虽然这是一个预发布的公告，但听起来这项工作还很遥远。视频播放的硬件加速已经就绪，但仍有一些音频问题需要解决，才能准备就绪。

我们知道基金会不是第一个完成这种事情的。环顾四周，你可以看到[其他一些概念验证视频](http://www.youtube.com/watch?v=t0MrtlyJOq0)，这些视频显示冰淇淋三明治在一个阶段或另一个阶段在板上运行。但是休息后视频中的演示给了我们希望，坚如磐石的支持就在眼前。毕竟，网飞运行的是安卓系统，[也是，XBMC](http://hackaday.com/2012/07/14/xbmc-hits-android/) 也是。两者都运行在 RPi 上，这使得这款设备离极其便宜且基本开放的机顶盒的圣杯又近了一步。我们还没有看到的一件事是这个设备的致命控制方法。如果你有锦囊妙计，你应该发布一些细节，然后[把链接](http://hackaday.com/contact-hack-a-day/)发给我们。
