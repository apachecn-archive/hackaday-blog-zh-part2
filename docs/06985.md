# THP 半决赛选手:莫特诺

> 原文：<https://hackaday.com/2014/09/03/thp-semifinalist-the-moteino/>

Hackaday 奖的一个明显的非官方主题是物联网和家庭自动化。虽然有很多项目在寻找新的有趣的方法从互联网上打开电灯开关，但很少有人认真研究这样做所需的硬件。[【费利克斯】的 Moteino 就是这些项目中的一个](http://hackaday.io/project/2197)。

Moteino 以 Arduino 为基础，增加了一个低成本的无线电模块，可以与世界其他地方通话。该模块为 HopeRF RFM12B 或 RFM69。这两款无线电均在 ISM 频段 434、868 或 915 MHz 下工作。它与背部绑有无线电模块的 Arduino 非常相似，编程很容易，应该能够完成 ATMega328 已经完成的任何事情。

[Felix]提供 mote ino[已经有一段时间了，现在已经有一些很棒的项目在使用这个平台。事实上，其他一些 Hackaday 奖项的参赛作品也在设计中加入了 Moteino 元素；](http://lowpowerlab.com/moteino)[植物朋友](http://hackaday.io/project/1982-Plant-Friends)用它做了一个传感器节点，[这个项目](http://hackaday.io/project/2573-433MHz-Texting-and-Controller)用它发短信和用手机遥控。
