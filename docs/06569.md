# 红牛创造:宏盟:这一次，他们不是邪恶的

> 原文：<https://hackaday.com/2014/07/15/red-bull-creation-omnicorp-this-time-theyre-not-evil/>

[https://www.youtube.com/embed/ijKtcspPWmQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/ijKtcspPWmQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

虽然红牛创作的大部分建筑发生在底特律的一个回收中心/艺术空间，但*评委*却在底特律的东部市场，这是一个巨大的农贸市场，几乎什么都有。Omnicorp hackerspace 就在东方市场之外，所以这是他们的地盘:他们知道什么行得通。对于他们今年的加拿大皇家银行的参赛作品，他们去了当地:一个轮式信息亭，也是一个制作思慕雪和烧烤一些蔬菜和狗的好地方。

虽然团队的信息亭值得称赞，但给所有参加红牛创造活动的游客一顿像样的午餐是一个好主意:所有的材料都已经在那里了，所以只需要一根延长线和一点点木炭。

红牛创造活动是这个项目真正闪耀的地方:数百人穿过至少六个桶，烟火，一辆该死的“龙”倾卸卡车，还有一个声音大到半英里外都能听到的 DJ。我们将在明天的文章中讨论这个问题。只能说我们的主编很开心。
