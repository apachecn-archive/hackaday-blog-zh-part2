# 热电偶真空计拆卸

> 原文：<https://hackaday.com/2013/07/16/thermocouple-vacuum-gauge-teardown/>

我们不知道[本·克拉斯诺]是怎么弄到这么多酷的硬件的。这一次使用了一些老式技术:[一个热电偶真空计](http://benkrasnow.blogspot.com/2013/07/thermocouple-vacuum-gauge-teardown-and.html)。

上图所示的部分是物体的传感器部分。这本身就足够有趣了。它有一个带有电加热元件的空气室。当存在空气时，它会散发热量，当处于真空状态时，热量积聚并导致热电偶在其连接处产生一些电压。

继续观看他的演示，事情会变得有趣得多。用于测量传感器的原始单位是对所有东西都有尖角的日子的倒退，如果你拿着剪刀跑，你最终会教自己为什么这不是一个好主意。设计师对电源电压的存在相当漫不经心，因为它几乎没有与外壳本身的接地连接分开。
