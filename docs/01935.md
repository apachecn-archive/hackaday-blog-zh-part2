# 万圣节灯光秀升级

> 原文:[https://hack aday . com/2012/10/22/Halloween-light-show-gets-an-upgrade/](https://hackaday.com/2012/10/22/halloween-light-show-gets-an-upgrade/)

[![](../Images/cbcbf7d926a6e91b71d76b95c95c81d3.png "Halloween Light Show")T2】](http://hackaday.com/?attachment_id=88583)

这是[戴夫]第二年上演[万圣节灯光秀](http://dl.dropbox.com/u/755737/lights_2012/index.htm "Halloween Light Show") ( [缓存](http://dl.dropbox.com.nyud.net/u/755737/lights_2012/index.htm))，与去年的[圣诞秀相比，他的最新作品得到了一些升级。他把圣诞风格的灯泡灯换成了高亮度的发光二极管，并升级到了 48 个控制通道。](http://dl.dropbox.com/u/755737/Lights/index.htm "Christmas Light Show")

控制器来自 [Light-O-Rama](http://www.lightorama.com "Light-O-Rama") ，每个控制器提供 16 个输出通道。他们通过 [RS-485](http://en.wikipedia.org/wiki/RS-485 "RS-485") 通讯；与使用 [DMX512](http://en.wikipedia.org/wiki/DMX512 "DMX512") 协议控制专业剧院灯光的网络类型相同。整个系统由某个中国零售商提供的 20 安 DC 供电。

[戴夫]的表演特色是点亮安装在他房子上的南瓜、墓碑和人脸。灯光与他通过调频发射机播放的歌曲列表相协调，允许汽车调谐到与灯光同步的音乐。

如果您碰巧在 Estacada 或 OR，您可能想去[Dave]家，亲自看看这个节目。否则，休息后会有两场灯光表演的视频。
