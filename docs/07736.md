# 电子垃圾打印机看起来不错，打印出来非常非常小

> 原文：<https://hackaday.com/2014/12/05/e-waste-printer-looks-nice-prints-really-really-small/>

在过去的几年里，3D 打印机的价格确实下降了不少。即便如此，至少也要花几百美元才能开始这项爱好。[mikelllc]认为这将是一个有趣的挑战，看看他能否以低于 100 美元的价格建造一台实用的[3D 打印机。](http://www.instructables.com/id/eWaste-60-3DPrinter/?ALLSTEPS)

为了保持在预算之内，[mikelllc]采取了合理的路线，并决定尽可能多地使用回收部件。在每个 DVD 和软盘驱动器中，都有一个步进电机、丝杠和托架，用于移动驱动器的读/写磁头。这些组件将用于驱动打印机的 3 个轴。两个 DVD 驱动器和一个软驱被拆卸，以访问所需的组件。

幸运的是[mikelllc]有激光切割机。他用 5 毫米厚的丙烯酸板材做了这个框架。所有部件都有插槽和标签，以方便组装，并保持一切都是直的和方形的。DVD 和软盘驱动器的电机和框架安装在预先设计好的亚克力框架上。Y 轴负责来回移动印刷床。它安装在螺丝上，这样可以调整它，以确保一个水平的床。

一个小小的 DVD 驱动器步进电机的功率不足以用作挤出机电机，因此为此购买了一个标准的 NEMA17 电机。该电机是 MK7/MK8 型直接驱动挤出机的一部分，该挤出机主要由 3D 打印零件制成。挤压机安装在框架上，鲍登管将细丝引导至安装在打印机移动托架上的热端。远程安装挤出机电机使其质量远离轴，在这种情况下，对于小型扫气驱动步进电机来说可能太重。

电子设备为标准 RepRap 型，与 hotend 的相同。回收电机与 RepRap 电子设备配合良好。经过这么多的努力，可打印区域只有 37 毫米 x37 毫米 x18 毫米，但这不是这个项目的重点！[mikelllc]实现了他的目标:用回收零件制造一台超级便宜的打印机。他还制作了挤出机和激光切割框架文件供下载，这样任何人都可以跟随他的脚步。如果你正在挖掘这台电子垃圾 3D 打印机，但想要更大的打印量，请查看这台打印机。
