# 为你的卡车制作太阳能充电器的原型

> 原文：<https://hackaday.com/2012/05/16/prototyping-a-solar-charger-for-your-truck/>

[Bryan]他得到了一块太阳能电池板，并决定把它带到路上，而不是扔在屋顶上。在阳光明媚的日子里，它会给汽车电池充电，让他在偏僻的地方使用他的立体声音响，而不需要保持发动机运转。他没有购买现成的解决方案，而是选择自己设计和制造充电电路。

充电器使用 Arduino，它通过调节器从面板获取自己的电力。它会感应电池的电压水平和面板的可用电量，并在必要时将其与电气系统连接或断开。该系统包括一组 LED 指示器，他将其安装在点烟器附近的仪表板上。这也给了他一个借口来安装一个电压表，它使用一个 2.5 位的七段显示器来读出电池电压。休息后你可以看到一个简短的概述。
