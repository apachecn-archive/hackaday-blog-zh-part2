# 3D 打印机排气

> 原文：<https://hackaday.com/2013/12/06/3d-printer-exhaust/>

[马尔科姆]终于受够了他的 3D 打印机产生的烟雾，所以他决定安装这个相当大的排气系统。

他已经有了一个非常棒的装置，他的 A 型 3D 打印机放在一个文件柜里，上面有一个塑料手提袋，防止他的细丝吸收太多的水。但是你知道，印刷 ABS 时释放的烟雾实际上对你非常有害。考虑到这一点，他 3D 打印了适配器环，并在他的文件柜外面安装了一个从空间加热器中回收的风扇。一个调光器开关提供可变的风扇速度，一些干燥机通风管将烟雾重新引导到中央真空管道，然后直接排到外面。当系统不使用时，可以堵塞管道以防止冷空气进入室内。这是一个相当干净的建筑，但[马尔科姆]想为风扇和速度控制电路做一个更好的外壳。

我们看到这样做的主要问题是从构建室中移除过多的热量，这总是会影响打印质量。你的 3D 打印机排气了吗？
