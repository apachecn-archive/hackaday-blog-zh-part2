# 黑客日奖参赛作品:生物手

> 原文：<https://hackaday.com/2015/06/25/hackaday-prize-entry-biohand/>

我们见过的 3D 打印的最大用途之一是修复术；即使在今天，一个专业制造的假肢也要花费成千上万美元。为了获得 Hackaday 奖，[Martin] [正在建造一只低成本的 3D 打印手](https://hackaday.io/project/6411-biohand-low-cost-3d-printed-hand-prosthesis)，它的工作方式就像一只自然的手，但用马达代替了肌肉和肌腱。

有很多 3D 打印的手指机构使用绳子和电线来移动手指。这有它的优势:它与正常手的肌腱排列极其相似，但[马丁]想看看是否有更好的方法。他用一个四连杆机构代替了琴弦，并用一个螺纹杆和伺服电机驱动每个手指。相对来说比较强；仅电机和驱动螺杆系统就能够提升 1 千克，这种机械安排还有一个额外的好处，即使用伺服的电位计向驱动电子设备提供手指位置的反馈。

这远不是角逐 Hackaday 奖的唯一一个假手项目。在他们的项目中，[OpenBionics]正在研究一种非常新颖的机制来模拟人手的功能，[Amadon Faul]正在全力以赴，在他的新木块项目中用铝[铸造掌骨和指骨。它们都是令人惊叹的项目，它们都极大地利用了 3D 打印技术，而且绝不是有太多的假肢项目进入了 Hackaday 奖。](https://hackaday.io/project/2854-neolimb)
