# 失灵的全球定位系统/GLONASS/盖革计数器

> 原文：<https://hackaday.com/2015/04/26/a-deadbugged-gpsglonassgeiger-counter/>

你认为你很擅长把非常小的零件焊接到 PCB 上吗？你可能不如[Shibata] [用微小的 0402 尺寸的零件制作 GPS/GLONASS 和盖革计数器混搭的](https://www.youtube.com/watch?v=xo8qckXgNNw)deadbug 风格。

该设备使用一个非常小的 GPS/GLONASS 接收器，一个 AVR ATxmega128D3 微控制器，一个标准的诺基亚电话显示屏和一个有趣的盖格管，该管带有云母窗口，可以跟踪其位置和当前的辐射水平。这个项目背后的想法并没有那么了不起；令人惊讶的是这个项目的组合方式。它是通过技巧或祈祷结合在一起的，微小的磁线取代了通常的 PCB 走线，单个元件组成了整个电路。

虽然没有太多关于焊料、热熔胶和电线的细节，但电路确实很有趣。不知何故，[柴田]正在为盖革管产生高电压，并想出了一个在显示器上显示所有相关信息的真正伟大的方法。这是一个伟大的项目，用一些疯狂的焊接技巧接近杰作的领域。

谢谢[丹尼]送来这个。
