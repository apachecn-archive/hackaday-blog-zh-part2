# 持续的视觉会成为一个伟大的平视显示器

> 原文:[https://hack aday . com/2014/02/27/视觉暂留-成就伟大-hud/](https://hackaday.com/2014/02/27/persistence-of-vision-would-make-a-great-hud/)

![povtyp](../Images/3bb1a032642525aae2ab482fbb7adad5.png)

[Eduardo Zola]刚刚对这个令人敬畏的实时视觉暂留显示器[进行了最后的润色，它可以在您键入时显示文本！](http://www.youtube.com/watch?v=afvmLPOEfR4)

它看起来像是安装在一个小型直流风扇上,[爱德华多]使用台式电源供电。这使他可以手动微调速度，而无需调整实际的 POV 控制器。显示器通过一个小型 USB 射频接收器从键盘接收字符，它的响应时间非常快。

关于这个项目没有太多的信息，但它肯定给了我们一个想法——视觉暂留是否可以用来在车辆中创建一种平视显示器？你怎么想呢?
