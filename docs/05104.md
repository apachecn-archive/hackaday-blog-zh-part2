# 达拉斯创客空间之旅

> 原文:[https://hackaday.com/2014/01/19/dallas-makerspace-tour/](https://hackaday.com/2014/01/19/dallas-makerspace-tour/)

[Paul]给我们发来了这个由会员[Andrew Floyd]制作的达拉斯创客空间的视频参观，他带我们参观了这个令人印象深刻的空间，并提供了解说。一进入[6000 平方英尺的设施](http://dallasmakerspace.org/)，他就带我们穿过门厅，进入电子设备室，这里的电子设备储存空间比可见的墙壁空间还要多，然后他沿着大厅向我们展示一些激光切割和 3d 打印的作品。

每个创客空间都有其特色，达拉斯帮展示了他们令人敬畏的暗室(配有令人毛骨悚然的、潜伏的笼子)和铁匠/锻造工作区。他们甚至有当地铁匠师傅每两周一次的锻造课。空间已经扩大，征服了他们的隔壁邻居，以扩大项目存储，增加一个生物实验室，第二个教室，会议室，等等。

休息后欣赏视频，然后前往他们的网站了解更多信息:[dallasmakerspace.org](http://dallasmakerspace.org)。
