# 高压、木材和树脂产生了分形艺术

> 原文：<https://hackaday.com/2015/07/04/high-voltage-wood-and-resin-result-in-fractal-art/>

燃烧木材，当感觉奇特时，被称为烙画，已经成为一种艺术形式有几个世纪了。[PapaJ06]通过使用微波炉变压器在木头中生成[分形图案，对其进行了新的扭曲。我们以前见过这些](http://imgur.com/a/Y0rbC/layout/blog) [Lichtenberg 图](https://en.wikipedia.org/wiki/Lichtenberg_figure)，但通常是使用百万电子伏加速器在丙烯酸板或[水晶球](http://hackaday.com/2015/01/08/putting-lightning-in-acrylic/)中放电。[PapaJ06]的技术要简单得多，大多数想要成为分形艺术家的人都可以做到，因为它依赖于从 20 美元的 Craigslist 微波炉中回收的变压器。

但真正给木头上的分形图案带来惊喜的是添加了一些磷光树脂来填充放电雕刻的山谷。【PapaJ06】精心准备木材，用混有环氧树脂的发光粉填充烧伤处，再用少许砂光、亚麻籽油、聚氨酯进行饰面。烧焦的木头和完整的木头之间的对比，以及树脂填充空隙的方式，真正展现了利希滕贝格图形的分形性质。

[PapaJ06]并没有向我们展示太多他的过程，但幸运的是[TheBackyardScientist]最近发布了一段他骑闪电过程的视频。休息之后来看看。
