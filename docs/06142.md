# 向 3D 打印添加铜线

> 原文：<https://hackaday.com/2014/05/25/adding-copper-wire-to-a-3d-print/>

导电细丝和印刷焊料是一回事，但如果你可以通过将电线嵌入细丝来增加 3D 打印的味道，会怎么样呢？[这就是【Bas】正在做的](http://basdebruijn.com/2014/05/additive-wire-laying/)，为可印刷电子产品、PCB、线圈和异形天线铺平道路。

[Bas]将细铜线嵌入一层印刷层的技术的总体思想是将电线放在喷嘴前面，有效地将裸线变成任何形状的绝缘线。然而，诀窍是弄清楚如何把电线放在喷嘴前面。[Baz]用一个旋转环实现了这一点，旋转环由连接到控制板上第五个轴的步进电机转动。

这个原型有几件事情没有涉及到——切断电线、将电线连接到元件、微调以及许多其他阻止[Bas]机器构建真正功能电路的事情。尽管有这些限制，这台机器可能现在就能为特斯拉线圈*制作次级线圈*，除非你有一台车床，否则制作起来真的很烦人。
