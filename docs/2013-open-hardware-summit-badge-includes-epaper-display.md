# 2013 年开放硬件峰会徽章包括 EPaper 显示屏

> 原文：<https://hackaday.com/2013/07/29/2013-open-hardware-summit-badge-includes-epaper-display/>

看看这个开放硬件的性感作品。这是你将在今年的开放硬件峰会上[戴在脖子上](http://wyolum.com/projects/BADGEr/)的东西。WyoLum 与 Repaper 合作进行展示，Seeed 工作室进行展示。

它被称为獾，它既是 Arduino *和*又是 Arduino 盾。有几种不同的电源选项；如果你想把电源挂在脖子上，可以使用硬币电池、微型 USB、无人使用的桶形插孔或挂绳终端。你可以看到上面的五个瞬时按钮，但在背面，你会发现 microSD 卡插槽以及一个电源开关，用于保存硬币电池。

看看下面的视频，快速浏览一下。除了作为您的凭证之外，会议日程也是预先安排好的。当然，这是一个开源的设计，所以你可以在上面链接的页面上浏览原理图、电路板插图和代码。哦，第一次攻击已经成功了。这里的[徽章上写着](http://wyolum.com/open-hardware-mini-ereader/) *[罪与罚](http://wyolum.com/open-hardware-mini-ereader/)。*

说到会议徽章，DEF CON 本周开始。黑客日作家【Eric even chick】[将会出席](http://hackaday.com/2013/07/27/hackaday-is-going-to-def-con/)，我们希望他有机会在活动中看到一些徽章黑客活动。
