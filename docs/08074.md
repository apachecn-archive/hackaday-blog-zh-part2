# 拼接动画在其页面上栩栩如生

> 原文：<https://hackaday.com/2015/01/16/spliced-animations-come-to-life-on-their-pages/>

还记得你在等午餐铃的时候在历史课本上乱画的那些动画书吗？[Maric]将“动画书”的一般原理颠倒过来，让我们的大脑在这个过程中旋转。通过将多帧图像拼接成一幅图像，他可以在一页纸上赋予动画以生命。

这项技术很简单，但能产生令人印象深刻的结果。通过在他的图像上叠加一个垂直的黑条图案，在任何给定点只有一小部分图像是可见的。图案中的间隙属于动画中的单个帧。当[Maric]在图像上滑动图案时，随后的帧展现在我们的眼前，而我们的大脑填充其余的帧。

仔细观察可以发现这些动画所受约束的更多细节。在这种情况下，每个动画循环的帧数由透明图案中的宽度给出。具体来说，它是可以并排放入相邻黑色矩形中的透明狭缝的数量。

让这个演示如此完美的诀窍是动画剪辑在它们开始的地方结束，从而产生一个清晰、连续的错觉。
