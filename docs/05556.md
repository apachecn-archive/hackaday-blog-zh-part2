# MRRF: 3D 打印树脂模具

> 原文:[https://hack aday . com/2014/03/16/mrrf-3d-printed-resin-molds/](https://hackaday.com/2014/03/16/mrrf-3d-printed-resin-molds/)

参观中西部 RepRap 节，你当然会发现大量 3D 打印的小玩意和小饰品。在今年的 MRRF 上，一个稍微有趣的发现是来自[马克·凡迪彭博斯]的大量树脂铸件。他是 RotoMAAK 背后的家伙[，这是一个旋转的，“这是在电影*接触*中出现的”，类似于为树脂旋转铸造而设计的设备。在电影节上，他展示了他的最新项目，](http://www.rotomaak.com/) [3D 打印树脂模具。](https://www.youtube.com/watch?v=37pthxP51NQ&feature=youtu.be)

有了合适的模具，任何有两部分树脂的人都可以在一小时内复制几十个相同的部分。唯一的问题是你需要一个模具来铸造零件。你可以打印一个塑料零件，然后制作一个硅胶模具来铸造你的零件。更聪明的解决方案是直接打印模具并用树脂填充。

[Mark]用 ABS 打印出上面看到的两部分兔子模具，用聚氨酯树脂填充，然后将其放入他的 RotoMAAK 旋转浇铸机中。六分钟后，零件弹出来了，模具准备好再做一只兔子。
