# 自制谢妮电子管

> 原文:[https://hackaday.com/2013/11/07/homemade-nixie-tubes/](https://hackaday.com/2013/11/07/homemade-nixie-tubes/)

你喜欢谢妮电子管吗？不高兴它们不再真正生产，旧的成本正在上升？为什么不自己做呢？捷克的【达利博】就是这么干的，包括自己吹玻璃管！

他选择了 Z568 谢妮电子管进行复制，因为这是他最喜欢的谢妮风格。为了制作显示屏，他在 0.3 毫米的不锈钢板上蚀刻出了数字和外壳——这可能意味着如果他掌握了制造电子管的诀窍，他就可以实际生产出来出售了！为了进行玻璃吹制，他从易贝弄来了一台[希斯威玻璃吹制车床——但不幸的是，他没有记录下多少关于制造玻璃管的东西，这太糟糕了，因为我们认为这将和谢妮展示自己一样令人着迷。在他第一次尝试使用密封良好的电子管时，谢妮成功了，他甚至记录到了与工业电子管非常相似的惊人电压值——对于后院小屋里制造的东西来说，这还不错！](http://dalibor.farny.cz/743/#more-743)

从那以后，他继续完善这种艺术，并带着一个被称为[上海时间谢妮钟的漂亮作品参加了一个名为“当布拉格遇到上海”的玻璃艺术竞赛。](http://dalibor.farny.cz/introducing-shanghaitime-nixie-clock/)

如果这篇文章看起来有点熟悉，那是因为这不是我们第一次发布关于自制谢妮管的文章，但是我们认为达利博的是迄今为止最优雅的！休息后留下来看看他的第一个测试视频——你甚至可能会认为他在作弊，这些管子看起来太专业了！
