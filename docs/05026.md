# 通过无数次的冒险教马里奥玩乒乓球和蛇

> 原文：<https://hackaday.com/2014/01/10/teaching-mario-to-play-pong-and-snake-through-innumerable-exploits/>

这是我们很久以来见过的最酷的经典超级任天堂娱乐系统(SNES)黑客。你现在看到的是【Masterjun】的[超级马里奥世界(完全控制)](http://tasvideos.org/4156S.html)。我们的第一个建议是，你[看完视频](http://www.youtube.com/watch?v=OPcV9uIY5i4)，然后回来这里做一个解释。类似于我们在 Gameboy 上看到的[口袋妖怪黄，【Masterjun】在《超级马里奥世界》中创造了整个 Pong 和 Snake 克隆体。他还创建了一个菜单和结束屏幕，以及他的商标笑脸图形。更令人惊讶的是，这是在一个运行未经修改的游戏卡带的真实 SNES 上公开的。[Masterjun]实际上使用了双多抽头电缆，有效地将 8 个控制器连接到一个 SNES。这给了他足够的带宽来通过控制器端口快速下载新的二进制文件。](http://hackaday.com/2012/11/24/programming-a-game-boy-while-playing-pokemon/)

欢迎来到工具辅助高速奔跑(TAS)的世界，在这里模拟器和脚本被用来通过视频游戏创建高速奔跑。跑步者经常一帧一帧地工作，费力地输入命令来创造完美的跑步。在这些速度竞赛中，游戏的错误和故障经常被利用。事实上，在像这样的跑步中，速度跑排在第二位，以此来炫耀其优势。速度运行创建的输出是控制输入的脚本文件，可以在仿真器上执行，以便随时“重新运行”TAS。这个脚本也可以保存到 PC 或 Raspberry Pi 中，并在真实游戏系统的控制器端口中播放。[基于 PIC 的硬件转换器](http://truecontrol.org/)用于将数据转换为 NES 或 SNES 控制器格式。正如所料，这些脚本是开环运行的。由于没有来自正在运行的游戏的反馈，它们可能并且确实会由于控制台硬件的差异而变得不同步，例如振荡器晶体的公差。当一切都同步并且正常工作时，结果是令人敬畏的。
