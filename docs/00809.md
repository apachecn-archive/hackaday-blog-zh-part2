# 醋栗是最新的 ARM Linux 板

> 原文：<https://hackaday.com/2012/06/22/gooseberry-is-the-latest-arm-linux-board/>

![](img/2ced4fd8ccad3e2b8e2fe97c2b62e0c2.png "goose")

看起来我们终于看到了单板计算机的激增。最新的一部叫做《醋栗 T1》。虽然它在发布时将花费大约 40/62 美元，但它大大优于目前的微型 Linux 主板。

As the latest in what will be a long line of these ARM-ified single board computers, the Gooseberry hugely outperforms the [VIA APC](http://hackaday.com/2012/05/23/apc-android-computer-isnt-a-raspberry-pi/#comments) and Raspberry Pi with a [Allwinner A10](http://rhombus-tech.net/allwinner_a10/) CPU running at 1.2 GHz, 512 MB of RAM, and built-in WiFi. Basically, the Gooseberry has all the features you would expect from an Android tablet; the reason for this is because the Gooseberry actually *is *the board found in a few Chinese tablets.

有几个非常好的功能，如 LVDS 输出，以添加 LCD 显示器而不占用 HDMI 输出，并且外形非常薄，非常适合构建小型便携式设备。
