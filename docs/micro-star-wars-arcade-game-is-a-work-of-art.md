# 微型星球大战街机游戏是一件艺术品

> 原文：<https://hackaday.com/2012/04/26/micro-star-wars-arcade-game-is-a-work-of-art/>

不，这不仅仅是另一个 MAME 内阁的构建。arcadecontrols.com 论坛上的[Le Chuck]建造了一个功能齐全的 1/6 比例的[复制品](http://forum.arcadecontrols.com/index.php?PHPSESSID=2d952e06a2eab10701882357069368ee&topic=117874.0;all)，这是 1983 年经典的雅达利街机游戏*星球大战*。

硬件是一个基于 Linux 的便携式媒体播放器，运行一个模拟 1983 年星球大战的视频游戏。当[Le Chuck]打开他的柜子时，MAME4All 启动并直接进入游戏。

因为没有 1/6 比例的拱廊部件，[Le Chuck]需要从头开始建造他的大部分建筑。该案件是椴木，随着非常准确的照明硬币槽门。游戏的操纵杆有点棘手；《星球大战》游戏使用的 X-Z 操纵杆是仿照 X-wing 驾驶舱的轭架设计的。为了制造这款 joysitck，[Le Chuck]拆开了几个罐子，用薄金属片制作了操纵杆。控制器的操作与原件完全一样，只是比例为 1/6。

休息之后，你可以看到这个令人难以置信的视频。
