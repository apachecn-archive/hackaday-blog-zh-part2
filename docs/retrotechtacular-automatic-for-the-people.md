# Retrotechtacular:自动为人民服务

> 原文：<https://hackaday.com/2015/06/09/retrotechtacular-automatic-for-the-people/>

纵观其悠久的历史，美国机器和铸造(AMF)已经涉足自动化的许多领域。随着 20 世纪 50 年代和 60 年代美国文化景观向快速、廉价和方便的食品转变，这些食品可以在宽敞、有鳍的汽车内消费，AMF 有了 [AMFare](http://www.youtube.com/watch?v=FmXLqImT1wE) ，这是一个(几乎)完全自动化的接受订单、准备食物和计算账单的系统。

AMF 将这个系统命名为“ORBIS ”,因为它有两个主要功能:订购和记账。但是奥比斯并不是完全自主的。一名操作员从餐厅内的桌边电话和免下车顾客使用的对讲机接收订单，并在一个巨大的控制台上输入它们。订单被发送到几台机器上，以各种方式准备、烹饪和包装食物。我们见证了汉堡的完整细节，从冲压出完美的肉饼到它们最终的塑料包装形式。

令人惊讶的是，AMFare 的选择不仅限于美味的汉堡、薯条和奶昔。它可以制作出 16 种不同的菜单项，而且速度非常快。在一个小时的时间里，AMFare 可以生产 400 多个汉堡，350 多份薯条，或者大约 700 份奶昔。即便如此，整理订单需要人工干预。我们可以想象，清洁那些[昂贵的](http://archives.chicagotribune.com/1964/05/28/page/81/article/a-m-f-automates-drive-in-chef)鲁布·戈德堡式机器的可怕任务也是如此。
