# 一个硬件视觉效果合成器

> 原文:[https://hack aday . com/2012/09/12/visualit-a-hardware-visual-effects-synthesizer/](https://hackaday.com/2012/09/12/visualist-a-hardware-visual-effects-synthesizer/)

![](../Images/76cca46cdb4e00589263afef0fc67e2d.png "visual-effects-synthesizer")

[Berto]来信告诉我们他制作的视觉效果合成器。它作为视频信号的通道，将清晰的图像渲染成更具迷幻味道的图像，如上图所示。一方面，这让我们如今随身携带的高质量视频设备蒙羞。另一方面，它可以在聚会或当地的舞蹈俱乐部制作一些非常有趣的背景视频。

这不是用于 PC 或基于 FPGA 的处理系统的滤波器。一组模拟部件改变输入的复合视频(NTSC 或 PAL 格式),并将结果传输到电视机或投影仪。[Berto]包括改变效果的控制。它们安装在一个面板上，所有东西都放在一个方便的手提箱里。休息后看看视频剪辑，更好地了解视频操作。
