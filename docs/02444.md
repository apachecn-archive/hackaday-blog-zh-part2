# 变色包与衣服颜色相配，告诉你里面是什么

> 原文:[https://hackaday . com/2013/01/02/color-changing-bag-matches-clothing-color-tell-you-whats-inside/](https://hackaday.com/2013/01/02/color-changing-bag-matches-clothing-color-tells-you-whats-inside/)

![color-changing-shoulder-bag](../Images/d7128770946a57847bb3dad11c1b7f38.png)

给你的日常用品添加一些灯光，肯定会让你的风格偏向银翼杀手的世界。但是如果你能增加控制闪光组件的功能，你实际上已经得到了一些东西。一个很好的例子就是凯瑟琳·麦克罗伊的变色龙包。这是一个有亮片的单肩包。它可以与你的衣服颜色搭配，但她也建立了一些功能，让你知道包里是什么。

这个项目从使用一个纸板模板开始，模板的大小和形状与包的翻盖一样。在添加了一个 Arduino 来控制 led 和一个交互式元素的 RFID 阅读器后，她缝了一个替代的翻盖，它也充当了扩散器。在休息后的视频中，她通过阅读缝在围巾末端的标签来演示如何搭配围巾的颜色。然后，她开始装载离家一天所需的所有物品。当钥匙、手机和电脑被放入包中时，它们的标签被读取，产生不同的颜色组合。一旦她需要的东西都在里面了，翻盖就会变成绿色，她就出门了。

这将与你的[带照明的雨伞](http://hackaday.com/2009/11/12/raid-the-dollar-store-for-an-illuminated-umbrella/)完美搭配。
