# Hackaday 奖参赛作品:一款开源工业相机

> 原文：<https://hackaday.com/2015/06/14/hackaday-prize-entry-an-open-source-industrial-camera/>

在过去的几年里，将相机连接到互联网变得越来越便宜。使这成为可能的进步并不是来自安全摄像头，而是微小的手机摄像头模块、ARM 板和嵌入式计算。现在，如果你想要一个后院的直播，你可能会得到一个树莓 Pi 和相机模块。这将适用于 90%的情况，但如果你想在稍微恶劣的环境中直播呢？如果你想直接在相机上进行图像处理呢？想让这个相机有个环保的评级怎么办？

[Apodiant]参加 2015 年 Hackaday 奖正在解决后一个问题。这是一款开源工业智能相机,具有以太网、USB 和串行输出，一个用于图像处理的 ARM CPU，所有这些都隐藏在一个坚固的铝制外壳中。

这款相机的初步 BOM 是 imx 6——一款非常强大的微控制器，可以运行 Linux 和 OpenCV。图像传感器是一个 120 万像素的单元[Apodiant]已经有了经验，外壳是现成的交易，任何人都想建立自己的。

如果这种设置听起来很熟悉，那么你是对的:已经有一些项目采用了相机模块，添加了强大的微控制器，并在其上运行图像处理。这些项目中最新的一个是 OpenMV。这让[有了一个成功的 Kickstarter](https://www.kickstarter.com/projects/botthoughts/openmv-cam-embedded-machine-vision/description) ，既然【Apodiant】要参加 Hackaday 奖最佳产品竞赛，这看起来是一个很好的选择。
