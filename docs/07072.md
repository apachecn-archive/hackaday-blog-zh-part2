# DIY 数控除尘

> 原文:[https://hackaday.com/2014/09/14/diy-cnc-dust-collection/](https://hackaday.com/2014/09/14/diy-cnc-dust-collection/)

数控机床擅长生产定制零件，但它们往往会在生产过程中搞得一团糟。[Darcy]建造了他自己的 [CNC 灰尘收集装置](http://inventorartist.com/cnc-dust-control/)来收集灰尘并保持他的工作空间清洁。

为了捕获灰尘，定制的防尘裙包围切割工具并引导真空。这是通过将丙烯酸部件粘合在一起制成的，创造了一个包含灰尘的盒子，并为真空系统提供了一个连接。

为了 1 美元，[达西]建造了一个旋风除尘器。这使得空气旋转，导致灰尘落到容器底部。结果是更少的灰尘进入真空吸尘器，花费在真空袋上的钱也更少。

因为吸尘器会发出很大的噪音，所以需要一个消声器。这只是一个简单的木箱来装机器。它还可用于将废气排出室外，以进一步防止污染工作空间。

虽然我们在过去已经看到过一些类似的版本，但是 Darcy 的设计对那些想要建立自己的系统的人来说是有帮助的。他还给了我们一个视频，展示了防尘裙的有效性，休息后你可以找到它。
