# 嘀嗒嘀嗒钟

> 原文:[https://hackaday.com/2014/06/22/tic-tac-clock/](https://hackaday.com/2014/06/22/tic-tac-clock/)

![tic-tac-clock](../Images/046bdcd6c28f6a8c9e98bda0fc831e1f.png)

这里有一个吃一堆井字游戏糖果的借口:一旦容器空了，它就为你的下一个项目做了一个漂亮的小围栏。这个特别的产品介绍了[一个非常有趣的点对点时钟项目](https://sites.google.com/site/dannychouinard/Home/atmel-avr-stuff/tictac)。

[Danny Chouinard]用很少的资源做了很多。只要看看上面的照片，你就能了解电路的要点。它使用 3×5 Charlieplexed LED 显示屏(这是因为电路板上只有几个电阻。有点难以看到，但在电阻和 ATtiny84 之间有一个时钟晶体，背面是一个小压电蜂鸣器。唯一不太清楚的是电源。从一个 9V 电池中回收的两个 AAAA 电池，能够保持该单元在大约 2 年的适度使用中运行。

休息之后的视频值得一看。它显示可以在 LED 矩阵上闪烁的各种字符和信息。起初，很难判断单用户输入按钮是由(丹尼的)拇指按下的。

如果你不想造一个钟，还是有足够的理由吃下一整容器的薄荷糖。你可以用 PIC 编程器或 T2 相机来代替它们。
