# 器官捐赠者放弃了莱斯利扬声器

> 原文：<https://hackaday.com/2015/01/14/organ-donor-gives-up-a-leslie-speaker/>

大约十年前，理查德收到了一架旧风琴。搬到一个新房子意味着带着他移动器官会很麻烦，所以他选择收获一些有趣的组件。具体来说，他保留了莱斯利扬声器。

莱斯利扬声器是一种特殊的扬声器装置，可以产生颤音效果。你可以在下面的[理查德的]视频中听到这种效果。像这样简单的效果在现在的电脑上很容易实现，但几十年前却不是这样。在数字电子技术出现之前，音乐效果通常是通过模拟手段来实现的。[Richard 的] Leslie speaker 是一个装在泡沫聚苯乙烯挡板后面的小型扬声器。挡板围绕扬声器旋转，改变声音的反射角度，产生音乐效果。

[Richard]尝试将该扬声器连接到其他乐器，但发现关闭电机会在扬声器上产生可听见的爆裂声。为了解决这个问题，他建造了一个简单“缓冲”电路。该电路只是一个简单的 240 欧姆电阻和一个 0.05 微法电容。这些元件为瞬变提供了一条接地路径，防止电机通电时发出爆裂声和咔哒声。现在[Richard]可以将这款经典的音频设备用于更新的项目。
