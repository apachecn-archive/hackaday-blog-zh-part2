# 自制 PCB 制造的终极工具

> 原文：<https://hackaday.com/2015/04/26/the-ultimate-tool-for-homebrew-pcb-manufacturing/>

虽然 OSHPark、Seeed Studio 和 DirtyPCB 已经把在家制作自己的电路板的大部分乐趣和紧迫性都拿走了，但仍然有一些小众案例和喜欢单干的古怪人。对他们来说，[jarzbski]创造了[终极自制 PCB 制造解决方案](http://www.jarzebski.pl/projekty/uvpcb.html)(。pl，这里是 [Google translatrix](https://translate.google.com/translate?sl=pl&tl=en&js=y&prev=_t&hl=en&ie=UTF-8&u=http%3A%2F%2Fwww.jarzebski.pl%2Fprojekty%2Fuvpcb.html&edit-text=&act=url) 。

[jarzbski]正在为他的 PCB 使用紫外线敏感的光掩模，但是他没有做像黑光灯这样简单的事情来显影他的电路板。他用 30 瓦的紫外发光二极管来曝光他的电路板。这当然会产生大量热量，为了缓解这种情况，他添加了帕尔贴冷却器、温度传感器和风扇来冷却这种燃烧视网膜的 LED。考虑到[jarzbski]在建造之前使用了四个 4.5W 的 led，30 瓦就可以完成这项工作。

开发 PCB 只是等式的一部分；你也需要蚀刻它。为此，[jarzbski]正在使用一个 1.6 升的小型水族箱和四个水族箱加热器将 120 毫米 x 120mm 毫米的多氯联苯浸泡在水箱中。没有提到[jarzbski]使用的化学物质——氯化铁、氯化铜或其他——但加热器和曝气器应该会使蚀刻非常顺利。
