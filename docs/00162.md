# 一个普通读者在本·赫克秀上的经历

> 原文:[https://hackaday . com/2012/04/20/one-hackaday-readers-experience-going-on-the-Ben-heck-show/](https://hackaday.com/2012/04/20/one-hackaday-readers-experience-going-on-the-ben-heck-show/)

![](../Images/8a55f39b7e8c88f9e04bd45c797f836d.png "Jesse N Ben")

在 Hackaday 论坛上，[Colecago] [分享了他与[Ben Heck]](http://FORUMS.hackaday.com/viewtopic.php?f=3&t=2141) 合作的经历。他们正在建造一些“机器人行李”，你可以在下面的休息后看到这一集。他们的想法是制造一件行李，它会跟着你穿过机场，而不是被拖着走。

[Colecago]分享了一点关于“幕后过程”的内容。在很少的视频中包含了惊人的工作量。据我们所知，这个视频花了一个多月的时间来制作。[Jesse/Colecago]非常尴尬地遇到了一个他称之为“UART Dislexia”的错误，他反复错误地连接了电路。我们说，不要紧张，每个人都会这样。

他提到的另一点是 Arduino 在这个过程中有多痛苦。虽然评论中的人们经常争论 arduino 的使用，但[Jesse]解释了如果没有 arduino，这个特定的案例会变得容易得多。
