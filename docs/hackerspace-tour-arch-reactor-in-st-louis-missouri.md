# 黑客空间之旅:密苏里州圣路易斯的拱形反应堆

> 原文：<https://hackaday.com/2013/04/02/hackerspace-tour-arch-reactor-in-st-louis-missouri/>

[https://www.youtube.com/embed/iH7osNm5raw?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/iH7osNm5raw?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

上个周末，我很幸运地来到了密苏里州的圣路易斯。宇宙中我最喜欢的一些地方都在那里，城市博物馆是我最常想到的地方之一。我意识到我从来没有参观过圣路易斯的黑客空间！

一个打给拱形反应堆的电话弥补了这一点。尽管那天是复活节，他们还是下来带我参观了一番。空间相当不错，有休息区、电子工作站、制造工具和一个完整的木材店。除了 hackerspace 的宜人氛围，这栋建筑还包括一个有趣的小型艺术自行车团体，名为[香蕉自行车队](http://bananabikebrigade.blogspot.com/)，甚至还有一个由回收材料制成的屋顶酒吧。对于那些对汽车感兴趣的人来说，自行车店里有几辆不错的 corvairs 和一辆 80 年代中期的保时捷 911。

如果你有机会顺道拜访，你一定要去参观 Arch Reactor。
