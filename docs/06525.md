# [汤姆·萨克斯]建立他自己的太空计划

> 原文:[https://hack aday . com/2014/07/10/Tom-Sachs-builds-his-own-space-program/](https://hackaday.com/2014/07/10/tom-sachs-builds-his-own-space-program/)

![sachslunarcapsuleinterior](../Images/a247986091ddb019301230f1b15ae1c5.png)

出生于 60 年代中期的[汤姆·萨克斯]一直对太空着迷，尤其是阿波罗计划。就像他那一代的每个孩子一样，[汤姆]想象自己穿着尼尔·阿姆斯特朗和巴兹·奥德林的靴子，凝视着月球表面。他从未放弃这个梦想，几年后，作为一名成功的现代艺术家，他建立了自己的太空计划。

[【汤姆·萨克斯】](http://en.wikipedia.org/wiki/Tom_Sachs_(artist))是一位[金砖](http://en.wikipedia.org/wiki/Bricolage)的大师。取自法语单词修补，维基百科将 bricolage 定义为“……从碰巧可用的各种各样的事物中构建或创造一个作品，或者通过这样的过程创造一个作品。”这个术语也可以描述我们在自己的项目中使用的垃圾箱采购方法。

汤姆的 2007 年月球计划和他的 2012 年火星计划都展示了他惊人的月球着陆器。着陆器由胶合板、找到的物品和垃圾制成，当我们第一次看到它时，它真的让我们看了两遍。对细节的关注令人难以置信。乍一看，人们可能会误以为这是美国宇航局自己建造的模拟器。几秒钟后，自定义触摸开始跳出，如快餐店的“谢谢”垃圾门，或备有龙舌兰酒和伏特加的酒吧。着陆器的工具也不仅仅是为了展示，因为画廊以模拟太空任务开幕，这可以被最好地描述为艺术、即兴表演和成人虚构的史诗游戏的混合。

[汤姆的]装置还包括任务控制，在他的火星作品中，包括一系列令人眼花缭乱的屏幕、控件和一个 80 年代的音箱。穿着白衬衫，打着细领带，戴着我们已经习惯了的 60 年代 NASA 工程师的角质框架眼镜，这就是[汤姆]工作的地方。他真的是这项任务的工程师。
