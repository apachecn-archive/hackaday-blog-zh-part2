# 黑客狗项圈，以节省电池钱。

> 原文：<https://hackaday.com/2012/09/06/hacking-dog-collars-to-save-money-on-batteries/>

[Lou]发送了这个非常简单的黑客程序，它为他的狗项圈节省了购买特殊电池的费用。他使用一种隐形的栅栏系统，如果狗越过他院子里的某些标记，就会受到电击。项圈使用特殊的电池，所以你不用把多个全尺寸的电池绑在你的狗脖子上。这是特别必要的，当你有一个更小的狗，没有太多的重量开始。

(娄)发现，8 美元的替换电池只是一个塑料外壳，他可以用更低的价格买到。所有需要做的就是在塑料外壳上割几个小口，以释放旧电池，然后他将新电池压入到位。这个微小的改动将会为他节省一笔可观的费用，因为这个装置在几周内就会烧光一块电池。
