# 初学者的 Android/Arduino 示例显示了 App Inventor 的强大功能

> 原文:[https://hack aday . com/2013/02/15/beginners-androidarduino-example-shows-the-power of-app-inventor/](https://hackaday.com/2013/02/15/beginners-androidarduino-example-shows-the-power-of-app-inventor/)

这是一个简单的项目。它使用 Android 设备来切换由 Arduino 驱动的 LED。连接由插入试验板的蓝牙模块提供。但是看一眼 Android 设备上的用户界面，你可能会认为这一点都不简单。事实是[Kerimil]并没有永远学习 Java 和编写应用程序。相反，他在炫耀[App Inventor 的力量，让你的 Android 控件快速启动并运行](http://arduino.cc/forum/index.php/topic,148853.msg1118346.html#msg1118346)。

检查第三个按钮；您上次在项目中添加语音命令是什么时候？值得点击查看这一部分是多么简单。App Inventor 是一个图形工具，它将 Android 手机的力量释放给那些对编程有最基本理解的人。例如，中断后显示关闭的声音控制由一个括号提供，该括号使用条件语句来“听”单词开、关和闪烁。休息之后你会看到语音识别图。

你可以尝试这个项目完全图形化。还有一个选项是[以类似的方式对项目的 Arduino 端进行编程](http://hackaday.com/2011/11/30/graphical-programming-with-the-arduino-ide/)。
