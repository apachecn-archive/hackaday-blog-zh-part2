# 用 Arduino 升级您的警报系统

> 原文：<https://hackaday.com/2015/09/05/upgrading-your-alarm-system-with-an-arduino/>

当[道格]搬进他的新房子时，他发现一个旧的警报面板被安装好了——但是它不再有监控服务了。他不想每月支付费用来设置它，他决定尝试将 Arduino 与系统连接，以便将事件推送到网上！

最酷的是，他实际上受到了几年前我们在 Hackaday 上分享的另一个类似项目的启发，该项目名为[根据你的意愿弯曲家庭安全控制面板](http://hackaday.com/2012/07/27/bending-a-home-security-control-panel-to-your-will)。但该项目仅向您展示了如何将 Arduino 与键盘连接起来— [Doug]更进一步，直接与控制板连接以获得更多功能。

他使用 Arduino Uno 和以太网分线板将其连接到网络。这使得他可以在警报系统启动、解除或触发时向自己发送文本消息。如果你也有一个 DSC 1550 报警系统，所有的代码[都可以在 GitHub](https://github.com/dougkpowers/pc1550-interface) 上找到。

这是一个非常巧妙的技巧，所以不要忘记休息后看看视频。
