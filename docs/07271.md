# 黑客日十周年:演示场景和眨眼(1)

> 原文：<https://hackaday.com/2014/10/08/hackaday-10th-anniversary-demoscenes-and-blink1/>

在我们的 10 周年聚会上有两个 LA hackerspaces 的代表，他们的成员都能够就他们实验室的项目发表演讲。来自 null space labs 的[Arko]带着他的一些发明出现了，包括他的高空气球载荷和他一直在工作的演示板。来自 Crashspace 的[Tod]和其他的坠毁人员一起出现了，并帮助完成了早上的组装和实验室工作。

#### 演示板

[https://www.youtube.com/embed/eeVOy_6byuI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/eeVOy_6byuI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

出于这样或那样的原因，Demoscenes 在美国并不是很受欢迎。在欧洲，你可以发现团队整年都在为程序生成的音乐视频工作，为 Commodore 64s、Amigas、更奇怪的计算机和 x86 汇编编码。这是一门艺术，但是对于我们这些在池塘这一边的人来说，没有多少地方可以展示完美的图形编程技能。

[Arko]想要改变这一切。他围绕 PIC micro 设计了一个演示板，带有硬件图形加速、USB OTG、640×480 的 VGA 输出和一个音频输出端口。这是一个创建演示的平台，并且【Arko】~~已经将【lft】~~的著名工艺演示[移植到他的平台上。编辑:Craft demo 在老版本的 ATmega88 板上播放。PIC 板能力强一点。](https://www.youtube.com/watch?v=mHHQq6tULJo)

因为在美国很少有演示聚会，仅仅建造一个板来玩演示是有点短视的。[Arko]做这个演讲的主要原因是告诉大家明年的 layer one demo party,它离 Hackaday Hackaspace 只有几英里远。它恰逢 LayerOne 大会，并且该板本身将很快在 Hackaday 商店中出售。

#### 眨眼(1)和如何 Kickstarter

[https://www.youtube.com/embed/xkS_sdrOcCc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/xkS_sdrOcCc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

当谈到电子和科技 Kickstarters 时， [Blink(1)](http://blink1.thingm.com/) 定义了拥有一个最低限度的可行产品意味着什么。它是一个 USB 插头，一个小型微控制器和一个 RGB LED。就是这样。[Tod]想接手这个简单的项目，并学习如何将其转化为产品。[Tod]强调了他计划中的“学习”部分；这个演讲的另一个标题是，“如何在多次失败后仍能卖出 20，000 台。”

闪烁(1)从我的第一个 Arduino 草图开始，闪烁三个 LED，很快转移到 USB LED 设备。这个相当大的 USB 加密狗在那里呆了几年，直到他决定把它变成一个产品。事实证明，制造一个产品比制造一个套件要复杂得多，要考虑外壳、包装和不可避免的 CNC 模具故障。组装——以及他的第一个 Kickstarter 的成功——也是一个问题。[托德]的朋友们最终组装了大部分工具。

尽管存在这些问题，[Tod]仍然能够发运几千件产品，现在正与 SeeedStudio 一起进行另一次生产。这是一个非凡的故事，谷歌、迪士尼、微软、脸书和其他一大堆大公司都使用了眨眼。Blink(1)也存在于主流 Linux 内核中，这是很多 Kickstarters 不能说的。
