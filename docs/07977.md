# 回顾:不伦瑞克对轮胎有偏见

> 原文：<https://hackaday.com/2015/01/06/retrotechtacular-brunswick-shows-a-bias-for-tires/>

介于车轮匠锻造的早期轮胎和现代钢带子午线轮胎之间，每个人的无马马车都骑在斜交轮胎上。本周的电影是对不伦瑞克轮胎公司大约在 1934 年的工厂的令人眼花缭乱的参观，在那里，轮胎是在看起来相当危险的条件下手工制造和测试的。

它打开了一个看起来像是来自巴西的场景:形成帘布层的帘线是从成千上万个独立的线轴上抽出来的，这些线轴以欢快的角度从杆子上伸出来。大约 1800 根这样的帘线会聚合在一起，并涂上一层具有高抗摩擦性能的橡胶化合物。由此产生的薄板被斜切成板层，每一层都被放在一个鼓上，然后被迅速送往轮胎室。

胎面胶由生橡胶和七种秘密原料制成，然后在班伯里密炼机中搅拌。一旦每批 800 磅制成，它就被送往工厂进行更多的添加剂和重型揉捏。原料被压成扁平的宽带，在 300 英尺的滚动传送带上冷却。在这一点上，粘合带应用于下侧，这将有助于在轮胎的寿命期间保持胎面与胎体的连接。

每个轮胎都是围绕一个宽鼓手工制作的。涂上几层并抹平，用辊子将边缘压紧，同时挤出残留的空气。接下来添加珠子，接着是保持侧壁附着的鳍状带。这些条带被冲压在胎圈上，然后其他帘布层被磨平。

在添加胎面之前，Brunswick 轮胎公司希望暂时中断轮胎装配，以获得关于其专利双减震技术的教育信息。在这些精致轮胎的胎面下，你会发现两层额外的帘线，它们经过特殊的缓冲处理。这种帘线和应用于胎面胶下侧的粘合剂层一起工作，以保护轮胎本体免受高速道路震动。任何装有 Brunswick Super Service 轮胎的老爷车肯定能从容驾驭木板路和泥潭。

你可能会注意到生产线上下来的东西几乎不像轮胎。那是因为它必须分两个阶段形成。首先，在成型箱内为硫化模具准备轮胎。安装了一个空水袋，轮胎在它周围模糊地形成。现在它准备好了硫化机，热量和压力被施加到轮胎的内部和外部。在硫化机内部，300 华氏度的水以 240 磅/平方英寸的压力喷入水袋，而外部则沐浴在蒸汽中。难怪那家伙赤膊上阵。

如果没有对检验标准和严格的测试程序的吹嘘，任何一部宣传性的“看看我们的工厂”的电影都是不完整的，但不伦瑞克在这两个部门都做了一些有趣的事情。也就是说，研究实验室:常规和实验性轮胎配方是在工厂的缩小版中创建和测试的。小批量的橡胶在小型班伯里密炼机中混合，然后在小型烤箱中烘烤。不幸的是，当时它们没有被制成微型轮胎，但这些材料经过了拉伸强度和耐热性的彻底测试。

全尺寸轮胎在刑讯室接受各种测试。例如，一些被过度充气到爆炸的程度，另一些被反复放在夹板上以模拟路边检查。轮胎在测功机上进行了数千英里的运行时间测试，然后与竞争轮胎进行比较。但是最好的利用空间和时间的方法是打桩机测试。在这里，一个钢楔沿着轨道上升了七层楼，落在毫无防备的轮胎上。测试后，轮胎被分阶段切割下来进行逐层分析。

顺便说一下，这是自 1845 年以来一直生产台球桌和台球桌的同一个不伦瑞克。在禁酒令颁布之前，他们也生产装饰华丽的酒吧固定装置，但被迫多样化，所以他们扩展到汽车轮胎和橡胶马桶座圈。到拍摄这部电影时，他已经把轮胎部门卖给了 B.F .古德里奇，保留了布伦瑞克的名字。
