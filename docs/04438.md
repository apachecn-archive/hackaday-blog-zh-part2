# 万圣节的黑仔拱廊橱柜

> 原文:[https://hack aday . com/2013/10/31/a-killer-arcade-cabinet-for-Halloween/](https://hackaday.com/2013/10/31/a-killer-arcade-cabinet-for-halloween/)

![slaughterhouseArcade](../Images/623ae02223c06d970f2b87766f6ff010.png)

[Clay]与他人共同拥有一个游戏厅已经很酷了，但他真正给我们留下深刻印象的是他定制的[*飞溅屋* 橱柜](http://www.multigame.com/splatterhouse.html)，它的建造是为了让他的顾客感受到万圣节的精神！来自 1988 年的 Namco 格斗冠军， *Splatterhouse* 出现在一个朴实无华且容易被人遗忘的柜子里。[Clay]抢救出一辆老威廉姆斯*Defender，*用干墙混合物、沙子和油漆涂在侧面，以获得石头纹理。然后，他将一些粉红色的绝缘泡沫切割成一个破烂的“木制”框架，并将其用作显示器边框。为了突出重点，他把橡胶条做成撕裂的肉的样子，放在木板中间。控制面板是另一件艺术品:[Clay] 3D 打印了真人大小的人类股骨作为游戏的操纵杆，并将按钮转换成眼球的样子。

[Clay]决定超越令人惊叹的化妆品，并利用一个定制的子板连接到游戏的 CPU，该子板可以检测游戏中的不同事件和状态变化，如玩家的健康状况。ATMega165 使用四个 PWM 输出，连接到机柜内和监视器挡板周围的多个 led，以对不同的事件做出反应。如果一个玩家受到伤害，监视器周围会有红灯闪烁。在游戏中插入硬币或死亡会导致字幕后面的一组不同的发光二极管变得疯狂。
