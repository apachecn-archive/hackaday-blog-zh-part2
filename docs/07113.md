# THP 半决赛:廉价卫星转发器

> 原文:[https://hack aday . com/2014/09/20/THP-半决赛选手-廉价卫星转发器/](https://hackaday.com/2014/09/20/thp-semifinalist-cheap-satellite-transponder/)

2016 年，一颗通信卫星将被发射到中东某处的地球静止轨道上。通常，这是相当平常的事情。然而，这颗卫星将携带两个业余无线电转发器，用于全欧洲、非洲、中东和印度的业余无线电爱好者。[【2FTG】正在建造一个卫星转发器与这颗卫星](http://hackaday.io/project/1233)通话，他正在用他工作台周围的垃圾做这件事。

这颗卫星的上行链路频率将在 2.4 GHz 附近，[2FTG]需要一种方法来处理这部分频谱中的带外干扰。做到这一点的简单而廉价的方法是使用为 WiFi 波段制作的过滤器。相反，[2FTG] [在他的垃圾箱](http://hackaday.io/project/1233/log/7868-image-reject-filters)里有几个空腔过滤器，并决定走这条路。这意味着他必须重新调整过滤器，这一过程将会非常困难。[2FTG]三十分钟就搞定了。

天线是另一回事，但由于[2FTG]有金属咖啡罐供应，这部分的建设只是焊接一点电线到 SMA 连接器，钻一个孔(使用日志作为钻孔停止，不多不少)，然后焊接连接器到罐。
