# 秘密之门现在不那么秘密了

> 原文：<https://hackaday.com/2014/09/19/secret-door-is-now-not-so-secret/>

你很难找到一个不认为暗门很酷的人。它们可以有多种不同的形式，内置书柜、假壁炉，甚至是地毯下的简易活板门。[oggfaba]创造了一个[甜蜜的暗门](https://www.youtube.com/watch?v=2da1DRLuo9o)进入他的房子。它做得非常好，没有必要用建筑细节来隐藏它，它就在那里，显而易见。

对于不明真相的旁观者来说，房子的后面看起来和任何一个有窗户和水龙头的房子一样。那个水龙头实际上不起作用，就像一个门把手。门-这个暗门的一部分只是一个标准的玻璃纤维外门，配有一个电子插销，覆盖着外部壁板，与房子的其他部分相匹配。

有两种锁门和开锁的方法。有一个遥控器可以远程解锁已安装的锁舌。还有一个键盘隐藏在它自己的伪装成房屋壁板材料的迷你暗门下。门锁、键盘或遥控器都没有遭到黑客攻击。晨光工业 QF-01SN 锁销可现货供应，具有两种解锁选项。

自从他在 YouTube 上上传了一段门的运行视频后，比起担心保守秘密，他更为自己的门感到自豪。他不会把密码给视频里的电子锁看…..他会吗？想要更多的秘密之门？看看这款[书柜](http://hackaday.com/2013/02/22/door-hidden-by-bookcase-is-a-marvel-of-diy-engineering/)，配有翻转书致动器。
