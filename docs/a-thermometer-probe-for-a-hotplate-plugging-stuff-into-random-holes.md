# 一个电热板的温度计探针，把东西塞进随机的孔里

> 原文：<https://hackaday.com/2015/06/22/a-thermometer-probe-for-a-hotplate-plugging-stuff-into-random-holes/>

YouTube 上声音变调的最著名的化学家[NurdRage]回来了，带着我们最喜欢的消遣之一回来了:购买便宜的设备和工具，阅读翻译得很差的手册，[以及弄清楚如何在没有任何说明的情况下做一些事情](https://www.youtube.com/watch?v=ORBgemwMLS4)。

[NurdRage]最近拿起了磁力搅拌器和电热板。到目前为止，它一直工作得很好，但它缺少一个温度计探针。[NurdRage]当他订购的时候以为他得到了一个，他只是从来没有得到一个。联系卖家没有得到回应，阅读翻译糟糕的手册甚至没有透露制造商是谁。考虑到这是一个仿制品，更多的研究显示这个热板是 SCILOGEX 热板的复制品。SCILOGEX 温度探头的价格为 161 美元。这可不酷。

温度探头在手册中被列为 PT1000 传感器；0°c 时电阻为 1000ω的铂基 RTD。如果这一假设正确，可以通过在连接器中插入一个 1kω电阻来确定温度探头连接器的引脚排列。当热板读数为 0°C 时，这是温度探头连接的导线。

找到合适的插针连接器后，[NurdRage]在易贝花几美元买了一台 PT1000，从一个 20 年前的键盘上拿了一个 DIN-5 连接器，把所有的东西连接在一起。传感器被装在一个移液管里，电线束沿着乙烯管蜿蜒而下。

对于 20 美元的零件，[NurdRage]设法避免支付 161 美元的真品。它的效果和股票、商业单位一样好，是一个很棒的视频。看看下面。
