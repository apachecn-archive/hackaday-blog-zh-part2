# 多个弹珠机混乱

> 原文：<https://hackaday.com/2012/04/18/multiple-marble-machine-mayhem/>

如果你喜欢大理石机器，或者你只是喜欢头韵，准备好大吃一惊吧。[Denha]显然这些年他有很多空闲时间，因为他收集的大理石机器非常令人难以置信。追溯到 2009 年，[集合](http://www.youtube.com/watch?feature=player_embedded&v=3NJ7Fr6VrPU# "Marble machines")包括相对简单的机器，如开始时的图片，到包括大约 5:30 的物理逻辑门的机器。

有趣的是，即使是由两个提升弹珠的机械装置和一个滑梯组成的“简单”装置也有锦囊妙计。该幻灯片实际上是模块化的，因此您可以对不同的幻灯片设计使用相同的“泵送”机制。这并不是唯一的“泵”设计，最后一台机器的特点是一个大理石提升机构，带有一个巧妙的连杆组件，可以将电机的运动转化为一种提升手。

如果这还不够疯狂，在 4:35 左右的休息之后，视频中还有另一个令人惊讶的弹珠表演在等着你！
