# 我们见过的最好的 LED 立方体建筑

> 原文：<https://hackaday.com/2012/04/06/the-best-led-cube-build-weve-seen/>

[Nick]在信中向我们讲述了他在六个月的时间里构建的 LED 立方体。他称 LED 立方体为“做死”，但(尼克)可能太谦虚了。他的 [8x8x8 RGB LED 立方体](http://www.hownottoengineer.com/projects/rgb-led-cube.html)是我们见过的最好的。

作为构建的开始，[Nick]构建了一个简单的 4x4x4 立方体作为概念验证。婴儿魔方起了作用，但制作过程让他开始思考。他需要从左到右建造柱子，而不是从下到上一层层地建造他的怪物 LED 立方体。在建造了一个夹具，焊接了八个由 64 个 led 组成的面板，并购买了一个新的烙铁头之后，[Nick]有了一个漂亮的组装好的 LED 立方体。唯一缺少的是电子设备。

我们见过的大多数 LED 立方体使用 TLC5940 LED 驱动器进行硬件 PWM，[Nick]决定使用更简单但更熟悉的 [STP16 芯片](http://au.element14.com/stmicroelectronics/stp16cp05mtr/ic-sm-driver-led-sink-16-bit/dp/1542235)。在将他巨大的 LED 驱动板连接到一个[芯片套件 Uno](http://www.digilentinc.com/Products/Detail.cfm?NavPath=2,892,893&Prod=CHIPKIT-UNO32) 后，80 小时的编程开始了。

最终，[Nick]建造了我们见过的最好的 LED 立方体(尽管它不是最大的)，并汇集了近年来最好的建造日志之一。因为没有视频的 LED 立方体是不完整的，休息后会有一个很棒的演示。
