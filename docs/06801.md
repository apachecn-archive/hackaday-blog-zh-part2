# 复古时间科技:[弗兰]和怀表

> 原文：<https://hackaday.com/2014/08/15/retro-time-tech-fran-and-pocket-watches/>

无论你是拥有一只怀表，还是想拥有一只，或者只是觉得它们很酷，[Fran 的] [关于设置和调节怀表的视频](http://www.youtube.com/watch?v=qMQdXAZm044)对这些美丽的机械艺术作品进行了全面的概述。在阐述了杆、杠杆和按键手表之间的优缺点后，[Fran]打开了她的 1928 Illinois，揭示了内部结构，并演示了如何调节调节器。

虽然她没有完全拆解，但在整个过程中有很多对零件的识别和解释。为了让她的手表慢一点，[Fran]需要逆时针转动一个非常小的固定螺丝大约四分之一圈，从而减慢周期:这需要一把精致的珠宝螺丝刀、细致的触摸和极大的耐心。结果也无法立即看出。需要一两天的时间来观察手表现在是否走得准。

跳伞后请留下来观看视频，其中还包括对 1904 年埃尔金手表、其调节器和其他关键部件的深入研究。
