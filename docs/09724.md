# Hackaday 奖参赛作品:可穿戴式皮肤电活动监测仪

> 原文:[https://hack aday . com/2015/08/03/hack aday-prize-entry-wearable-electro dermal-activity-monitor/](https://hackaday.com/2015/08/03/hackaday-prize-entry-wearable-electrodermal-activity-monitor/)

皮肤电活动或皮肤电反应有许多实际应用。从对情绪状态的研究到像电子测量仪这样的非常规的应用，一切都使用皮肤电活动。作为他的 Hackaday 奖参赛作品，[qquuinn]正在制作一款可穿戴生物反馈腕带，它可以测量皮肤电反应，是一款简单方便的可穿戴设备，非常适合治疗焦虑或压力障碍。

一百多年来，任何人都可以用欧姆表来检测皮肤电活动。[quin ]正在做一些比最原始的现代方法[更复杂的事情，测量皮肤电反应](https://www.cs.cmu.edu/~dst/E-Meter/)并使用双运算放大器来感应皮肤电阻的微小变化。这些数据被输入 ATMega328，atmega 328 将数据发送到一个 x 形的微型 LED 显示屏上。

读取皮肤电活动很容易，但在可穿戴设备上可靠地读取却不容易。然而，皮肤接触的问题，放大器的问题，以及将整个东西放在一个方便的包装中的问题。[qquuiinn]在黑客频道的小组讨论[中向社区询问了这些问题，得到了很多非常好的建议。这是 hackaday.io 上的项目可以做什么的一个很好的例子，也是 hackaday 奖的一个很好的项目。](https://hackaday.io/project/5373-hacker-channel)
