# 激光点烟器让吸烟变得更加危险

> 原文:[https://hack aday . com/2013/08/01/激光点烟器让吸烟变得更加危险/](https://hackaday.com/2013/08/01/laser-cigarette-lighter-makes-smoking-even-more-dangerous/)

使用这个激光点烟器开始吸烟的危险包括射瞎你的眼睛，给你自己留下一个类似《T2》哈利波特《T3》的前额伤疤。这东西有一个两瓦的激光二极管，可以毫不费力地烧掉所有接触到它的东西。

[Masterjoa3000]在休息后的视频中向您展示它是如何构建的。你需要获得二极管和支持硬件作为散热器。在电线连接到正极和负极导线之前，它们被压配合在一起。对于打火机上的挡风玻璃来说，外壳有点太宽了，但可以通过在挡风玻璃的中心切掉一个“V”来解决这个问题。接下来是一个微小的驱动板，焊接到二极管和瞬时按钮开关上。开关代替了打火石，所以按下点火轮就能激活激光器。整个东西仍然适合在未改变的外壳。

这是对同一想法的另一种理解，激光指向不同的方向。
