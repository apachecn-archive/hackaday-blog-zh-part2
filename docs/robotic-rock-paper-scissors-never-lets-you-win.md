# 机器人石头剪子布永远不会让你赢

> 原文：<https://hackaday.com/2012/06/27/robotic-rock-paper-scissors-never-lets-you-win/>

所以机器人在井字游戏、国际象棋、危险游戏中打败了我们，现在它们也是石头剪刀布的主导物种。这个[机器人手臂在游戏中每次都会胜过你](http://spectrum.ieee.org/automaton/robotics/artificial-intelligence/robot-hand-beats-you-at-rock-paper-scissors-100-of-the-time)。它不仅速度快得跟不上，还能知道你打算做什么，并据此做出反应。所有这一切发生得太快了，你很难当场发现。

东京大学的研究人员提出了将高速视觉与高速手结合起来的想法。显然，分析你的选择只需要一毫秒。手形成征服位置所花的时间只比这稍微长一点。正如你在休息后的剪辑中看到的那样，它已经知道 1-2-3 射击的协议，不需要任何操作员干预就可以开始新的游戏，或者反复教你尝试与机器竞争。

在之前，我们已经在游戏中被一台机器打败了[。这是人类玩家第一次不需要穿特殊的装备，机器也从虚拟手变成了实体手。](http://hackaday.com/2010/03/10/game-glove-learns-your-weakness/)
