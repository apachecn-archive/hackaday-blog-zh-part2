# 每天享受日出

> 原文:[https://hack aday . com/2014/07/27/享受每一天的日出/](https://hackaday.com/2014/07/27/enjoying-the-sunrise-every-single-day/)

![Exif_JPEG_PICTURE](../Images/210ceea8e16865b3c8f6ed4778f5b364.png)

[Andy]想在日出时照几张，但是日出前醒来有明显的相关问题。相反，[他造了一个设备](http://hackaday.io/project/2126)，可以计算当地日出时间，拍一张照片，然后一直睡到第二天早上。

这个项目使用的相机是一台老式的佳能傻瓜相机，选择它是因为它能够加载 [CHDK 固件](http://chdk.wikia.com/wiki/CHDK)。其他电子设备包括 Arduino pro mini、LiPo 电池和充电器板、实时时钟和用于用户界面的旧诺基亚 LCD。

有相当多的代码用于计算每天太阳何时升起，但一旦计算出来，所有[安迪]要做的就是把相机带到一个漂亮的地方，对准东方，记录几天的日出。当放入“游戏相机”外壳时，它足够坚固，可以抵抗除小偷之外的任何东西，并且有足够的电池电量供几周的日出使用。
