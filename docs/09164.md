# 从废料到剑:铸造锡器

> 原文：<https://hackaday.com/2015/05/31/from-scrap-to-sword-casting-pewter/>

[TheBackyardScientist]一直名副其实，这一次他在自己的院子里铸造了一把锡剑。[白镴](https://en.wikipedia.org/wiki/Pewter)是一种软合金，主要成分(85–99%)是锡，还有铜、锑和铋。旧的锡合金铸件也经常使用铅。白镴的伟大之处在于它的低熔点(170–230°c)。在如此低的温度下，白镴可以在普通的电热板上熔化。把它想象成一种进入金属铸造世界的简单方法——不需要锻造。当然，任何被溅过焊料的人都会告诉你，滚烫的熔融金属总是值得很多尊重的。

[古代科学家]通过搜寻当地的旧货店获得他的金属。他使用“消失模”铸造法，用泡沫塑料雕刻出一把剑。这把剑被嵌入一个 5 加仑的沙桶中，沙桶带有一个冒口，可以让模具被填满。白镴在一个便宜的电炉上熔化，然后倒入模具。热金属在接触时熔化泡沫，同时填满砂型中留下的空腔。[BackyardScientist]留下了一把坚固的白镴剑。它不会保持优势，但它是这种技术的一个很好的例证。
