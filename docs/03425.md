# 让蓝牙去掉耳机里的电线

> 原文:[https://hack aday . com/2013/06/17/let-bluetooth-take-the-wires-out-of-your-耳机/](https://hackaday.com/2013/06/17/letting-bluetooth-take-the-wires-out-of-your-headphones/)

![bt-wireless-headphones](../Images/e616409a585177c356162e6dbd911ff9.png)

这张图片显示了[艾伦]将他的有线耳机转变为[内置蓝牙音频接收器](http://www.moddingstudio.com/content.php?1529-Guida-Costruire-cuffia-wireless-b)([翻译为](http://translate.google.com/translate?sl=auto&tl=en&js=n&prev=_t&hl=en&ie=UTF-8&u=http%3A%2F%2Fwww.moddingstudio.com%2Fcontent.php%3F1529-Guida-Costruire-cuffia-wireless-b))的要点。

他从一副有线“耳罩”式耳机和他一直使用的配件市场蓝牙音频适配器开始。但是如果你不打算把它们插到音源上，为什么还要有六英尺长的额外电线呢？[Alan]扔掉蓝牙硬件周围的塑料外壳，打开耳机为它寻找空间。它很紧，但是有足够的空间。

不幸的是，耳机设计中没有隐藏在连接耳机的弧形中的有线交叉。艾伦自己串了一些红线来连接两个扬声器。安装电路板时，USB 端口位于电线进入塑料外壳的位置。当他们需要充电时，插上电源就变得轻而易举。

您可以通过尝试在他的构建日志中发现树莓 Pi 来玩一点“Waldo 在哪里”的游戏。
