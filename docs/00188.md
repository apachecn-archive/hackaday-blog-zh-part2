# 坦克大战:一款控制实体机器人的视频游戏

> 原文:[https://hack aday . com/2012/04/23/tank-wars-a-video-game-controlling-a-physical-bot/](https://hackaday.com/2012/04/23/tank-wars-a-video-game-controlling-a-physical-bot/)

![](../Images/e5bad03dbfe076e5246b532d0ffb0ad7.png "TankWars-A-Physical-Video-Game")

[Joe]给我们发了一封电子邮件来展示他的最新作品。《坦克大战》是电子游戏/机器人混合体的开始。你可以通过 iPad 控制坦克，告诉它去哪里以及如何开火。你有真实的生活目标，在这种情况下，另一个机器人。当你击中目标时，界面会更新游戏数据。目前，这距离成为 wifi 控制的机器人只有一步之遥。[Joe]正在努力使游戏更具互动性。

坦克和目标都由 Arduinos 和 RN-XV WiFly 模块运行。该坦克具有向前和向后的避障功能，从视频来看，这似乎有时会使导航有点挑战性。iPad 界面只是一个网页，所以它真的可以在任何设备上使用。这太酷了，我们迫不及待地想看看他如何从这里开始。
