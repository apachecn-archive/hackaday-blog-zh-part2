# 机器人钟琴嘎吱作响的“爆米花”

> 原文:[https://hack aday . com/2015/04/12/robotic-glockenspiel-crunches-popcorn/](https://hackaday.com/2015/04/12/robotic-glockenspiel-crunches-popcorn/)

[詹姆斯]给我们发了一段他最新创作的视频:[一个机器人钟琴](https://www.youtube.com/watch?v=69pGYTYSEug)，它目前被设置用来演奏“爆米花”。它使用八个伺服系统来驱动木槌，以快速、清脆的动作击打音条。伺服系统由 16 通道 I C 伺服驱动器和 MIDI shield 驱动，后者又由 Arduino Uno 控制。他的自动锁定望远镜的前一个化身使用了螺线管、销钉和橡皮筋。

[格申·金斯利]1969 年为合成器“爆米花”创作的歌曲多年来被许多艺术家翻唱，尽管最受欢迎的可能是[热黄油]1972 年发行的。休息后请查看，并挖掘可爱的电缆管理。接下来我们很想看[詹姆斯]的自动驾驶直升机表演《大黄蜂的飞行》。

如果你渴望更多的电声创作，看看[Aaron Sherwood]的[磁性电话](http://hackaday.com/2013/12/28/the-magnetophone/)。
