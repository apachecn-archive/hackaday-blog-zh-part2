# 制作精美的 N64 便携式电脑

> 原文:[https://hack aday . com/2015/07/19/a-制作精美-n64-便携/](https://hackaday.com/2015/07/19/a-beautifully-crafted-n64-portable/)

随着几十台功能强大的单板 Linux 计算机的出现，你可能会认为经过时间考验的改造老式视频游戏控制台的做法将成为一门失传的艺术。仿真器可用于一切，这些微小的 Linux 盒子比这些旧控制台中发现的原始电路更小。[Chris]，最好的控制台改装者之一，[仍然在推出项目](http://forums.modretro.com/viewtopic.php?f=3&t=14218)。他的最新产品是便携式 N64，这正是我们对这位业界大师的期待。

多年来，我们已经看到几十款任天堂 64 被改装成电池供电的手持设备，而[克里斯]的最新项目遵循了人们熟悉的格式:从控制台上拆下 PCB，添加屏幕、一些按钮和电池，并将所有东西包装在一个漂亮的盒子里。这是构建的最后一部分——案例——这里很有趣。这种外壳是用 3D 打印和数控加工相结合的方法制造的。

这个项目的外壳最初是用 PLA 印刷的，各部分粘在一起，最后填充以获得漂亮光滑的表面。[Chris]说 PLA 是一个错误的选择——低熔点意味着研磨面板时产生的热量会使牙胶粘在牙片上。在未来，他仍将使用外壳的印刷部件，但对于精密工作，他将转向铣削聚苯乙烯板。

随着案件的完成，几个散热器被添加到板上最大的芯片，新的按钮分线板铣，和一个定制的音频放大器布局。其结果是一个制作精美的便携式 N64，比任何模拟器都要高级和充实。
