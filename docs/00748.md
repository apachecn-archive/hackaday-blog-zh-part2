# [Bob]使用 Haxlr8r 的经验

> 原文:[https://hack aday . com/2012/06/19/bobs-experience-with-hax lr 8 r/](https://hackaday.com/2012/06/19/bobs-experience-with-haxlr8r/)

![](../Images/47dc75b2a315df40f7cd0f91e3b9d0b7.png "IMG_20120612_160756")

[Mike Szczys]最近与[Bob Baddeley]聚在一起，谈论他将爱好项目推向市场的经历。他还没到那一步，但[鲍勃]决定走一条我们觉得很有趣的路线。他一直在参加一个名为 Haxlr8r 的为期 111 天的加速器项目。这个想法是，当他的产品真正上市时，这种经历将给他所需的制造能力，以满足需求。他在中国[度过了相当长的一段时间，他在博客上详细描述了这段经历。该计划还提供发展现金，以换取一定比例的公司。](http://hackaday.com/2012/06/07/getting-your-stuff-built-how-to-shop-conduct-business-stay-alive-and-eat-your-way-through-shenzhen/)

他正在开发的产品是一个 LED 记分牌，这也解释了为什么他称该公司为[便携式计分器](http://portablescores.com/)。这是一个业余爱好项目，他带来了一个早期的原型，在休息后的视频中展示。这是一个木制框架，背面和基底是泡沫板，正面是透明的丙烯酸树脂，以保护 led。所有的焊接都是点对点的(休息后也有一张图片)。他的重新设计已经从 Arduino 转移到 PIC 控制器，将显示器的整个表面做成一个 PCB，并增加了蓝牙控制。很高兴看到有人真的为一个精心设计的项目而努力。我们祝他一切顺利！
