# 音阶用计算尺

> 原文:[https://hack aday . com/2012/04/02/slide-rule-for-music-scales/](https://hackaday.com/2012/04/02/slide-rule-for-musical-scales/)

![](../Images/bbba91b9f44e78331d5839abb771365a.png "music-scale-slide-ruler")

对于所有涉足音乐的工程师来说,[Magnetovore]是你的后盾。音乐家必须知道他们的音阶，他想出了一个适用于大调和小调音阶的纸艺计算尺。

这个系统很容易使用。他上传了 PDF 文件，让你打印出顶层的蒙版和底层的条形图和方向。顶层的布局就像钢琴键盘，每个键都有一个窗口，还有几个窗口用来标识正在显示的大调和小调音阶。滑动遮罩，直到每个键都是纯色。色码显示每个调的主音、三度音和属音，这样你就知道从哪里开始了。在休息后的视频中，你可以看到它是如何按顺序弹奏所有非黑键的。但是等等，如果你现在订购，你会以同样低的价格得到大提琴音阶的计算尺；免费！
