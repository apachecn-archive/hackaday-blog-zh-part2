# 面部跟踪糖果大炮！

> 原文:[https://hack aday . com/2014/01/13/the-face-tracking-density-cannon/](https://hackaday.com/2014/01/13/the-face-tracking-confectionery-cannon/)

奥林学院的一个机电工程学生团队想出了一个非常有趣的学期项目——[一个气动棉花糖大炮，可以跟踪人脸，并瞄准嘴巴](http://confectionerycannon.com/)！

这个被称为糖果佳能的设备是一个令人印象深刻的机械构造，它需要奥林学院的许多制造资源，如激光切割机、研磨机和车床。该设备的大部分是由丙烯酸制成的，这是因为它易于激光切割，而且价格实惠。特殊的铝片提供了强度，主要是用商店里的废料制成的。

四个伺服系统，一个网络摄像头，一个螺线管和一个 Arduino Uno 组成了电气系统，该系统使用 Python 和 OpenCV 来跟踪人脸( [GitHub](https://github.com/fbourke/confectionerycannon/tree/master) )。一个 PVC 罐被用作气压罐，装有一个压力为 30 磅/平方英寸的安全释放阀。为了发射大炮，喷水阀由一个结实的螺线管控制。它目前只有 4 个大棉花糖的弹匣容量，但该团队计划很快升级。

他们建立了一个很棒的网站，上面有大量关于这个项目的信息，休息之后是他们为这个项目制作的一个有趣的宣传视频——他们甚至让学院的副总裁来尝试一下！
