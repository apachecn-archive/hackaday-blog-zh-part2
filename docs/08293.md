# 用倾斜的管子建造东西

> 原文：<https://hackaday.com/2015/02/12/building-things-with-lean-pipe/>

有时候你只是想快速简单地构建一些东西。也许你只是需要一个实际项目的基本结构，或者你希望能够很容易地修改设计。也许你没有机会使用许多花哨的工具来建造一个坚固、轻便的结构。另一种可能性是，你希望能够分解你的结构，并在以后转移它。在这种情况下，[你可能会考虑使用贫油管道](http://www.instructables.com/id/Lean-Pipe-Computer-Desks/?ALLSTEPS)。

瘦管有点像成年人的 K'NEX。它由金属管和专用配件组成。如果你以前用过 PVC 管，这可能听起来很熟悉。不同的是，瘦管更坚固，专为建造坚固的结构而设计。设计用于贫油管道的夹具比 PVC 管道更容易使用。对于 PVC 管，似乎永远没有合适的配件，必须自己制造适配器，这迅速增加了设计成本。

典型的倾斜管道配件要么滑过一段管道的末端，要么在中间的某个地方缠绕它。然后拧紧调节螺钉，将管件夹紧在管段周围的适当位置。下面的视频很好地展示了配件的不同可能性。这种材料的主要问题是你可能无法在当地的五金店找到它。幸运的是，在网上快速搜索会找到很多网上购物的选择。

那么你能用这些东西造出什么呢？科迪一直在为自己打造具有工业外观的电脑桌。他首先从框架设计开始。这是由瘦管子制成的部分。一旦框架完成，他只需要在木材表面上工作。他真正需要做的就是把木头切割成型，然后把它修整得好看一点。然后把它放在适当的位置，为了更加安全，可以用螺栓固定。
