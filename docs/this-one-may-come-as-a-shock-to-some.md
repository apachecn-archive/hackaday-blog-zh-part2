# 这可能会让一些人感到震惊

> 原文：<https://hackaday.com/2014/12/31/this-one-may-come-as-a-shock-to-some/>

[克里斯]似乎已经征用了妻子的缝纫室的一个体面的部分为他的电子冒险。由于这仍然是她的主张，她明确表示他的区域需要一些组织和一张新桌子。由于对他们购买的替代宜家桌子的外观和感觉不满意，他决定通过添加铜层压板来提升风格和价值。

他的决定并非纯粹基于美学。如果你跟着，这意味着他的新电子工作表面是导电的。是的，它在墙上接地。虽然他不在乎防静电垫的臭味或它们容易褪色和开裂，但他确实打算用一小块来让他的硅高兴。

[Chris]使用了一张 20 号铜片，他切割并刻划下来，以适合他的瑞典三明治木质底座，并留有足够的悬垂余量。在刮开铜片的一面和接收基底的一面后，他挤下一些足以满足橡胶手套协议要求的粘合剂，并将其全部夹在一起几个小时。留在原地铜做了，但留平它没有。锤平突出部分后，[Chris]用 Scotch Brite pad 在小漩涡中手工打磨铜，从视觉上打破了轻微波动的表面。休息后的指导和欢闹的播放。
