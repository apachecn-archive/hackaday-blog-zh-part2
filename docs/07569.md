# 视频游戏文化

> [https://hacked ay . com/2014/11/14/视频游戏文化/](https://hackaday.com/2014/11/14/video-spiel-kultur/)

不知何故，不要问我们如何，我们为 Hackaday 颁奖晚会选择的场地非常适合与 Hackaday 相关的恶作剧。拐角处有一个黑客空间，附近的一个仓库里有一个计算机历史博物馆，离我们的场地不到 100 米的地方有一个惊人的视频游戏档案馆。

[视频游戏存档](http://videospielkultur.org/)是一个惊人的视频游戏集合，来自视频游戏被装在*盒子*和*真实手册*的时代，你需要确保你购买的游戏与你的系统兼容。在里面，一面墙是旧的纸板电脑盒子，部分是根据系统，部分是根据它们看起来有多酷来索引的，而另一面墙则是以前五代游戏机的游戏。
