# 口袋妖怪蓝成为 IDE

> 原文:[https://hack aday . com/2013/12/02/pokemon-blue-becoms-an-ide/](https://hackaday.com/2013/12/02/pokemon-blue-becomes-an-ide/)

有了 WiFi，奇迹交易，和新的神奇宝贝，你会认为最新一代的每个人都有理由拥有一台任天堂便携式电脑是所有行动的地方，对不对？显然不是，因为[口袋妖怪蓝刚刚成为游戏少年](http://forums.glitchcity.info/index.php/topic,6638.0.html)的开发工具。

尽管所有的观念都是理智的，但这并不是我们第一次看到有人从第一代口袋妖怪游戏中为游戏男孩编程。大约在去年的这个时候，【bortreb】[发布了一个工具辅助运行](http://hackaday.com/2012/11/24/programming-a-game-boy-while-playing-pokemon/)，它存储和丢弃游戏中的项目，以写入游戏男孩的 RAM。使用这种方法，[bortreb]能够在《口袋妖怪黄》中制作出一个 chiptune 版本的*我的小马*主题。

一年后，[zzazglitch]已经超越了[bortreb]所做的。代替工具辅助运行，[ZZAZZ]的黑可以在真正的 Game Boy 上手动完成。这个戏法通过一个下溢故障来获得玩家库存中的物品“8F”。这里有一个很好的教程来教你如何做。使用这个 8F 项目，可以投掷一些项目，并激活“编程”模式，其中可以通过走到地图上的 X-Y 位置，使用 8F 项目，并逐字节编写程序，将代码写入 RAM。

可以写入 Game Boy RAM 的最大代码量是 254 字节，刚好足够[thezzazglitch]编写一个非常非常简单的 Akranoid、Breakout 或单人 Pong 版本。不多，但非常，非常令人印象深刻。
