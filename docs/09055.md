# 二冲程发动机太美了

> 原文：<https://hackaday.com/2015/05/16/2-stroke-engine-too-beautiful-to-behold/>

这座建筑的美丽令人目眩。我们喜欢保持简约的家居风格——不太像[乔·麦克米伦]的水平，但能达到那里——然而这将是我们选择作为装饰的东西。这是一台手工制造的二冲程发动机，专门设计用于使燃烧过程可见，而不是将其锁在一块金属内。

如果你有一种挥之不去的感觉，那就是你以前见过这个，因为这个惊人的工艺令人难忘。几年前，我们研究过同样由 Huib Visser 制造的四冲程发动机。这款新产品去掉了皮带，留下了一个几乎完全由玻璃和金属抛光而成的高光泽结构。玻璃圆筒包含燃烧，推动石墨活塞驱动飞轮。一个经过的磁铁触发火花塞点燃白色气体燃料，所有这些在休息后的视频中都有很好的说明。

这是非卖品，这并不奇怪。放弃如此美丽的东西有多难，尤其是知道是你创造了这种美丽？不过不用担心，你绝对可以自己造。只是要确保在你的前六次尝试中把门槛设得低一些。我们甚至见过使用五金店零件制造[引擎。](http://hackaday.com/2013/07/26/building-an-internal-combustion-engine-from-hardware-store-parts/)
