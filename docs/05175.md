# 特斯拉线圈自动绕线机

> 原文:[https://hackaday.com/2014/01/28/tesla-coil-auto-winder/](https://hackaday.com/2014/01/28/tesla-coil-auto-winder/)

![tesla winder](../Images/5061a5e0d7b4108abb7ff07b4e2025c0.png)

特斯拉线圈很牛逼。但是如果你曾经做过，你就会知道缠绕次级线圈是多么乏味。所以[克鲁克斯]决定制造一台机器来帮他做这件事。

他目前正在制作他的第一个特斯拉线圈——代号为[Project Icarus](https://krux.org/article/Project_Icarus)——他还没有解决所有的后勤问题，但他已经在慢慢收集组件。他所知道的是，他想使用 4.5”次级线圈，使用 22AWG 磁线，这意味着很多匝！由于他也是当地[黑客空间](https://synshop.org/)的成员，他决定将它做成一台模块化机器，可以为不同规模的项目缠绕不同大小的线圈。

本质上，他建立了自己的数控车床来完成这个，嗯，缺少一个轴。有一个主旋转轴，和一个沿着它移动的导线，确保线圈被紧紧缠绕，没有间隙。这是一个令人印象深刻的构建，你可以看出他在设计上花了很多心思——他甚至在导线轴上安装了半灵活的 3D 打印电机耦合器，以帮助减轻快速加速！主旋转轴也由 3D 打印的人字形齿轮驱动，类似于 Printrbot 挤出机上使用的类型。建筑的其余部分由胶合板和钉板制成，允许通过在组件周围移动来缠绕更大的线圈。他甚至有一个全功能的命令控制台，带有手动/自动控制和一个液晶显示器，可以对正在缠绕的线圈进行反馈！

休息后留下来看看[Krux]解释迷人的建设，并看到一个 814 圈特斯拉线圈绕组的有趣延时！
