# 回顾:Shakey 展示了 1972 年的机器人技术创新

> 原文:[https://hackaday . com/2012/05/17/retro tech tacular-shakey-shows-off-robotics-innovation-from-1972/](https://hackaday.com/2012/05/17/retrotechtacular-shakey-shows-off-robotics-innovation-from-1972/)

![](../Images/97d5a41a76c5ae8ba54989f4d4a3a3d0.png "shakey-1972-robotics")

在这个 retro techular[装置中，我们来看看 Shakey](http://www.youtube.com/watch?v=qXdn6ynwpiI) ，一个在 1966 年到 1972 年间在斯坦福研究实验室开发的机器人。这是一个辉煌的时代，学生们留着长发，但仍然穿着长袖和领带做他们的研究。

机器人实际上正在与负责处理过程的 PDP-10 计算机进行无线通信。没有计算机显示器用于与机器人互动。取而代之的是，电传打字机让你在纸上打出你的命令，而机器的回应也印在同一张纸上。有一个用于图像识别的摄像头，以及当身体接触到障碍物时提供反馈的传感器。

我们很想知道这个项目有什么样的预算，但可惜我们找不到任何相关信息。如果你想的话，你可以亲自去看莎琪。这个信息页面提到这台机器在加州山景城的计算机历史博物馆展出。
