# Pwdr，开源粉末打印机

> 原文:[https://hack aday . com/2012/08/08/pwdr-the-open-source-powder-printer/](https://hackaday.com/2012/08/08/pwdr-the-open-source-powder-printer/)

![](../Images/1bc55212857e23d4f34537358686c897.png "pwdr")

[认识一下 pwdr](http://pwdr.github.com/) ，这是一款开源 3D 打印机，它完全不同于我们已经爱上的 RepRaps 和 Makerbots。

pwdr 不是将塑料喷到构建表面上，而是像工业环境中使用的非常非常昂贵的粉末打印机一样工作。Pwdr 使用石膏、陶瓷和混凝土作为原材料，并用喷墨墨盒中沉积的水将这些粉末颗粒结合在一起。

pwdr 内部有两个箱子，一个用于存储原材料，另一个用于制造零件。要打印的部分一次构建一层，就像普通的桌面打印机一样。每一层完成后，反向旋转的滚筒将原材料刮过构建区域，然后印刷另一层。

与 Makerbot 中使用的熔化塑料印刷方法相比，pwdr 有很多优点；因为每一个构建都是自支撑的，所以可以打印出无法用基于挤压机的打印机制造的物体。Pwdr 还支持激光烧结，这意味着 pwdr 有可能用 ABS、尼龙甚至金属制造物体。

现在，pwdr 仍然处于开发的早期阶段，但是你可以从 Thingiverse 上的[文件中构建你自己的粉末打印机。休息之后看看 pwdr 打印的视频。](http://www.thingiverse.com/thing:27794)
