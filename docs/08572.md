# RadioShack 的消亡可能标志着夫妻店的崛起

> 原文：<https://hackaday.com/2015/04/16/radioshack-demise-could-signal-the-rise-of-mom-and-pop/>

不管你对 RadioShack 的感觉如何，对许多黑客来说，这是一个可以在本地获得组件的地方。当听说[商店被关闭](http://hackaday.com/2015/02/05/ive-come-here-to-bury-radio-shack-not-praise-it/)(至少对那些寻求非手机商品的人来说)时，我们想知道是否有人会站起来迎接创客市场。答案可能实际上是夫妻店——由热衷于黑客和制作的人拥有的独立商店。

在三月份的 SXSW Create 上，Hackaday 展台就在这样一家公司的隔壁。[马丁·博戈莫尔尼]正在努力工作，开一家名为[修补匠&扭曲](https://www.facebook.com/groups/tinkerandtwist/)的实体店。在下面的视频中，他简要介绍了商店的概念，重点是展示最好的产品和工具，并在当地储存。

该商店将位于德克萨斯州奥斯汀的一个购物中心。但考虑到许可和扩建，开设一个店面需要大约 100 天。[Martin]决定通过在 SXSW Create 展览(和销售产品)将商店带给黑客。除了搭建一个店面作为你的摊位，你还能做什么？商店的标志是由硬质泡沫塑料加工而成的，并与一组柱子和店面橱窗结合在一起。我们在活动的最后一天很晚的时候顺道拜访了他们，他们度过了一个愉快的周末。一开始库存非常充足的货架看起来几乎空无一物。

Tinker & Twist 只是专注于黑客和制造商的独立商店不断增长的趋势中的最新一例。我们在加州帕萨迪纳 Deezmaker 的朋友去年带我们参观了一下。他们将黑客空间、小型制造商(以 3D 打印机的形式)和一体化零售店的概念结合在一起。这些类型的例子让我们非常高兴 RadioShack 多年来一直密切关注那些实际建造东西的人。我们希望看到更多像廷克&这样的商店出现并运营，以支持和增强各地的黑客社区。
