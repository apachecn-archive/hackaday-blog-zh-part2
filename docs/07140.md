# 飞思卡尔和德州仪器商品和世界制造商大会

> 原文：<https://hackaday.com/2014/09/22/freescale-and-texas-instruments-goodies-and-world-maker-faire/>

飞思卡尔在本周末的 Maker Faire 上对 Hackaday 非常友好，展示了一些电路板，并回答了一些关于为什么旧摩托罗拉应用笔记不能在互联网上使用的问题。

SolidRun 的鹰嘴豆板对于任何一个用过树莓派的人来说都是一种奇怪而熟悉的外形。它还有一个有趣的特点:CPU 在一个小模块上，允许任何人将芯片组升级到更强大的东西。在顶级配置中，它有一个双核 iMX6 CPU，带有 1gb RAM、LVDS 输出和千兆以太网。该板的所有复杂部分都在一个模块上，任何人都可以把这个模块放到另一个项目中，就像 Intel Edison 一样。

飞思卡尔展台[中还有 pcDuino](http://www.pcduino.com/pcduino-v3/) ，一款双核 ARM Cortex A7，带以太网、WiFi 和 SATA，带 Arduino 外形规格引脚排列。这是一个有点小众的产品，但能够在类似 Raspi 或 BeagleBone 的东西上堆叠盾牌是一个很好的功能。

[https://www.youtube.com/embed/Acrzg0wVP1A?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/Acrzg0wVP1A?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

德州仪器的[Trey German]展示了一些非常酷的东西，包括用于 Launchpad 微控制器的四轴飞行器板。这不是一个有 IMU 和几个伺服输出的板；这是一个完整的框架，马达和道具。框架是从一些通常用于路标的奇怪复合材料上切下来的，尽管它没有在集会上飞行(顺便说一下，没有东西在飞行)，但对于一个在板房里制作的四轴飞行器来说，它相当轻。

TI 的 CC3200 开发板也是如此。这是一个带有 ARM Cortex M4 和 WiFi 无线电的单芯片[，我们之前已经见过](http://hackaday.com/2014/07/04/tis-new-family-of-wifi-chips/)。CC3200 运行 TI 的 Wiring/Arduino 启发的开发环境 [Energia](http://hackaday.com/2014/09/13/energia-on-the-cc3200/) ，CC3200 Launchpad 板大约 30 美元，这是建立物联网的一种简单而廉价的方式。
