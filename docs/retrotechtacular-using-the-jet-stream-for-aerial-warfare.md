# 逆向技术:利用喷射流进行空战

> 原文：<https://hackaday.com/2015/04/14/retrotechtacular-using-the-jet-stream-for-aerial-warfare/>

无人驾驶飞行器最近风靡一时。但是，虽然今天的战斗无人机技术尽可能现代化，但这个想法本身并不新鲜。19 世纪中叶，奥地利在威尼斯放飞装满炸弹的气球。大约一百年后的二战期间，日本人利用他们新发现的关于急流的知识向美国和加拿大发送气球。

每个气球花了大约四天时间到达北美西海岸。他们携带燃烧和杀伤装置作为有效载荷，包括自毁装置。气球的“商业端”是电池、爆破块和一个装有四个无液气压计的盒子，用来监测高度。为了让气球保持在 8000 英尺以内。垂直范围的喷射流，他们被设计成下降压载沙袋开始一天的飞行使用一个吹塞和保险丝系统。理论上，气球已经在第四天到达北美领空，除了燃烧弹和中央杀伤人员有效载荷之外，什么也没留下。

尽管这个项目很短暂，但日本人在 1944 年 11 月至 1945 年 4 月间发射了大约 9300 个这样的热气球。他们中的几个人没能着陆。其他的被击落或降落在偏远地区。有几个人一路顺风，有两个甚至一路漂到了密歇根。对于一个宣纸气袋来说还不错。
