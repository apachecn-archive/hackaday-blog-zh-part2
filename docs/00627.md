# 一个控制台来管理它们

> 原文:[https://hackaday.com/2012/06/03/76101/](https://hackaday.com/2012/06/03/76101/)

![](../Images/45f45b15756dce078be16386216ac2f9.png "alphaomega")

【细菌】复古控制台改装师[带着一个相当大的项目](http://www.made-by-bacteria.com/viewtopic.php?f=18&t=308)回来了。“Unity”(最初称为 Alpha Omega)，这将是一个可以从 20 个不同的主机系统玩游戏的单一单元。它将由一个电源供电，有一个视频输出，奇怪的是，只有一个控制器。

[Chris Downing]非常友好地向我们提供了 Unity 控制器的运行视频。该控制器并不像我们想象的那样庞大，因为它必须支持大量的控制台，但这可能在一定程度上是因为您实际上为了每个系统的兼容性而将大脑换成了控制器。

除非我们在大范围的讨论中遗漏了什么，否则下面的游戏系统已经被同化了:

任天堂 NES-完成，尚未添加到系统中
美泰智能视觉
任天堂 GameCube(通过 GBA 播放器和 GBA)
阿米加 CD32
飞利浦 CD-i
Playstation 2(也玩 Playstation 1 游戏)
