# 自然召唤的天然材料

> 原文：<https://hackaday.com/2015/07/11/natural-materials-for-natures-call/>

当谈到为你的浴室改造挑选高端设备和预约时，这些天有很多选择。当然，双拖鞋浴缸或 2500 美元的不锈钢马桶也能彰显个性，甚至当地的大盒子家居商店也有一些非常独特的东西。但是对于一个独一无二的水槽，跟随[武士木匠]的领导，在一块巨石上雕刻出一个水槽。

[手绘轮廓内的几个垂直切口给他足够的空间用冷凿子和锤子挖出大部分材料。角磨机上的金刚石砂轮和冲击钻上的凿子钻头使他获得了最终的光滑表面。休息之后，有一个视频展示了最终的安装，包括钻出排水孔和将水槽安装到梳妆台上，这是一个漂亮的粗切板，似乎是当地采购的木材。整个装置看起来棒极了，似乎运行良好；我们唯一的吹毛求疵是盆里没有溢出来的水，但很难看出他如何能提供一个而不使项目明显复杂化和潜在地破坏美观。](https://hackaday.com/wp-content/uploads/2015/07/capture2.png)

虽然它们可能不符合他改造的自然氛围，但无论是[这个](https://hackaday.com/2015/02/12/slick-bathroom-mirror-is-all-tricked-out/)还是[这个](https://hackaday.com/2011/03/02/bathroom-mirror-hud-displays-time-and-weather/)精心设计的镜子都可以完成高端浴室改造。
