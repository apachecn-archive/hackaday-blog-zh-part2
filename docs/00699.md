# 使用 3D 打印齿轮进行计算

> 原文:[https://hack aday . com/2012/06/11/用 3d 打印齿轮计算/](https://hackaday.com/2012/06/11/calculating-with-3d-printed-gears/)

这里有一台由纽约电阻器公司的克里斯·芬顿制造的 3D 打印机电计算机。它使用印刷在 Makerbot 上的塑料寄存器、一堆弹簧针和名片大小的穿孔卡，能够存储 32 位指令和数据。

如果你想知道，这个[不是我们第一次看到](http://hackaday.com/2012/05/03/a-3d-printed-electromechanical-computer/)[Chris]“FIBIAC”。自从上次更新以来，[Chris]设法得到了一个遍历 Fibonacci 数列前三位的程序。理论上，FIBIAC*可以做的事情没有限制，但是只有三个寄存器，他只能计算圆周率的前三位。*

有了更多的注册表，[Chris]的计算机就可以扩展，但是每个注册表需要大约 8 个小时才能打印出来。我们确信[Chris]会很乐意接受任何额外的 3D 打印寄存器的捐赠，所以如果你想制作一些这样的齿轮寄存器，你可以在 Thingiverse 上获得文件[。](http://www.thingiverse.com/thing:16068)

作为概念的证明，[Chris]' FIBIAC 很神奇，但它没有达到预期的设计。[Chris]创建的穿孔卡格式能够存储 8 个寄存器，并且寄存器本身可以扩展到远远超过其当前的 3 位宽度。尽管如此，这仍然是一个令人难以置信的构建，并且由于非常聪明的设计，它还具有易于扩展的优势。
