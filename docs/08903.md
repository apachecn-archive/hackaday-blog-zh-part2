# 用一盏灯来预见未来

> 原文：<https://hackaday.com/2015/04/29/use-a-lamp-to-see-into-the-future/>

我们以前听说过神灯，但这个是真的。【Alex】创造了一个[壁挂灯，可以告诉你未来会是什么样子](https://hackaday.io/project/5491-project-aladdin-weather-forecast-lamp)；至少就天气而言。它被恰如其分地命名为“阿拉丁工程”，当你走出家门时，它能让你一眼就知道大量的天气信息。

该灯由 12 个垂直排列的 LED 灯条组成。底部的长条代表当前小时，上面的每个长条代表未来的另一个小时。每个条带的颜色指示温度，每个条带内的 led 的各种动画指示风速和降雨量。

该系统使用内置 Java 的天气预报后端，可以在项目页面上找到。led 由 C 语言编写的应用程序控制，整套 led 封装在半透明的外壳中，使其具有非常专业的外观。休息之后，请务必观看演示视频。一定要看看其他一些使用普通台灯而不是复杂的 LED 灯的天气灯。
