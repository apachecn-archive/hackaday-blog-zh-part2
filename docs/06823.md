# DIY 定制成型耳塞综述

耳机如今已经变得无处不在。多亏了 iPod 和智能手机，人们戴着耳塞式耳机已经变得司空见惯。然而耳塞并不总是舒适的。有些人穿太松了。对其他人来说，太紧了会导致疼痛。为此，我们找到了一些解决这个问题的好办法。

[cptnpicard]在 Imgur 画廊中记录了他定制的 Sugru 耳塞。他把一副标准耳塞模制到了他的耳朵模型中。跳伞时，他用它们来保护听力和听音乐。 [Sugru 的常见问题解答](http://sugru.com/blog/the-official-sugru-faq-any-questions-we-ve-got-answers)指出，虽然固化材料对皮肤接触(和耳朵使用)是安全的，但一些人对未固化材料很敏感。

在 Reddit 上讨论他的项目时，一些用户插话说他们已经使用[弧度定制耳塞](http://www.radians.com/radsite/index.php/industrial/industrial-safety-products/item/radians-custom-molded-earplugs)套件制作了定制成型耳塞。弧度材料在 10 分钟内硬化，这比等待 Sugru 一个小时要好。

食物链的绝对顶端必须是[在耳麦中构建自己的三重驱动器](http://www.instructables.com/id/DIY-In-Ear-Monitors/?ALLSTEPS)，这正是【marozie】所做的。专业定制成型显示器的价格可能超过 1000 美元，这使它们成为专业音乐家和音响发烧友的天堂。[marozie]发现 mouser 从 [Knowles](http://www.knowles.com/) 那里储备了不少传感器。然而，这些微型扬声器并不便宜；光是一个司机，你就可以花 70 多美元。

[marozie]用耳模印模工具给他的耳朵做了个模型。他用这个模子做了一个模子。从那以后，就是在他精心制作的驱动电路和音频管上浇树脂了。由此产生的显示器看起来和听起来都令人难以置信。

不言而喻，定制耳麦需要将化学物质放入耳朵。定制耳模套件带有微小的挡板，以防止模具材料进入过多而造成损坏。这是我们建议遵循说明的少数几个地方之一。点击休息时间观看耳朵成型过程的演示视频。
