# Wiimote 控制灭绝:Dalek 风格

> 原文:[https://hack aday . com/2014/05/31/Wii mote-controlled-extinction-dalek-style/](https://hackaday.com/2014/05/31/wiimote-controlled-extermination-dalek-style/)

![Dalek Build](../Images/796146a2af6c00a6b88b39e270d0844d.png)

参加会议的人可能已经走过了许多达雷克斯:外星人在活动场地周围开车，大声威胁要灭绝，偶尔用旋转的眼柄拍打人们。在 [SoMakeIt Hackerspace](http://www.somakeit.org.uk/) (你可能还记得[我们今年早些时候的报道](http://hackaday.com/2014/01/20/people-of-southampton-unite-theres-a-new-makerspace-in-town/))，詹姆斯·布鲁顿(James Bruton)和他的黑客伙伴们一直在努力工作[建造这个 Wii 遥控 Dalek](http://www.youtube.com/watch?v=ab5n4LWvvAA) 。

大多数 Dalek 在人体内建造了一个驾驶座，驾驶着一辆打捞上来的电动轮椅，在那里，他们使用操纵杆控制周围，并在越来越强烈的香气中煨煮。这个版本和大多数版本一样，从源于[项目 Dalek](http://www.projectdalek.co.uk/) 的计划的木质结构开始。然而，在里面，[詹姆斯]和他的工作人员已经接入轮椅的电机控制器，从一个与 Raspi 相连的 [Arduino 虾](http://hackaday.com/2012/11/10/the-ultimate-low-cost-dev-board/)那里输入 PWM 信号。Pi 从 Wiimote 接收蓝牙信号，并通过他们的[定制 Python 脚本](http://github.com/somakeit/dalek)，轻松指挥 Dalek。

他们仍在努力完成 Dalek 的身体，但他们正在使用一些聪明的策略向前推进:使用 3D 打印机来解决一些微妙的造型选择。他们上传了一个图库，里面有脸书的其他照片[，休息过后，你可以在视频中看到他们在创作中游手好闲(失去平衡，差点自杀)。](http://www.facebook.com/media/set/?set=a.471455232984895.1073741844.269112773219143)
