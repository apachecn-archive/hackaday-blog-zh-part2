# 具有触觉反馈的触摸灯

> 原文：<https://hackaday.com/2014/12/18/touching-light-with-haptic-feedback/>

我们中的许多人都曾在某个虚拟或增强的场景中，带着当今野外为数不多的耳机之一，进行过静止的嬉戏。虽然观看令人信服的现实虚构物的体验本身就是一种令人兴奋的感觉，但[李屏宾]和[王聪]正在研究如何将其他感官结合起来。

来自康奈尔大学的两人组已经建造了一个能够通过触觉反馈对光线做出反应的机械外骨骼。这意味着佩戴者可以触摸光源周围的光球，就像它是一个固体一样。光敏电阻像天线一样安装在每个手指的尖端，它们沿着边缘向下排列，以接收更多的漫射光。当该设备的佩戴者将他们的手朝着光源移动时，传感器触发安装在手背上的伺服电机，以启动和缩回一系列 3D 打印的肌腱，这些肌腱向上拱起并连接到佩戴者的各个手指。这样，当电阻器接收到不同量的光时，它们可以独立地做出反应来模拟物理轮廓。

该项目的目标之一是用价值不超过 100 美元的材料制作出一个可行的概念证明，这是[Mark]和[Kevin]用一些多余的现金实现的。他们的部件清单可以在他们的博客上找到，还有更多关于这个项目的细节。
