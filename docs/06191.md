# 机器人军队 IRL 加上一个巨大的建造日志

> 原文：<https://hackaday.com/2014/05/30/robot-army-irl-plus-a-massive-build-log/>

我们又一次去了制造者集会的“黑屋子”采访机器人军队的莎拉。她和负责该项目的软件开发的马克展示了 30 个会跳舞的三角洲机器人。具体来说，他们在和另一个集会者的动作一起跳舞。Kinect 传感器监控这些运动，并将它们转化为来自 deltabots 的匹配运动。

你应该记得[在 11 月](http://hackaday.com/2013/11/29/build-and-control-your-own-robot-army/)看到这个项目。既然这个模型的标准已经制定出来了，那就只需要花大约三周的时间来组装军队。我们很高兴看到 Kickstarter 的[在三月初完成了目标的](https://www.kickstarter.com/projects/1984252088/robot-army-starter-kit)250 %,而且还有更大的计划。[Sarah]说目标仍然是用机器人填满一个房间，我们甚至有一天会看到一个更大的版本。

采访有点短，因为机器人军队展台就在 Arc Attack 旁边(因此有了降噪耳机)，我们不得不在他们震耳欲聋的干扰中进进出出。但是你可以在 Hackaday.io 上的这个巨大的构建日志中看到更多关于这个项目的信息。也可以看看[的机器人军队网页](http://robot-army.com/)。他们在 MFBA 和[的冒险经历有一个很好的插图，泡沫快乐扳手](http://hackaday.io/project/1159-Foam-Jolly-Wrencher)被写进了作品！
