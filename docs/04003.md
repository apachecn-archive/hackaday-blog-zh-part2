# 家用产品中的氯酸钾

> 原文：<https://hackaday.com/2013/09/07/potassium-chlorate-from-household-products/>

右上方是非常纯的氯酸钾，以至于当它与糖混合并被一些硫酸催化时会爆发出火焰。[家庭科学家先生] [使用家用化学品](http://www.youtube.com/watch?v=7MsHq_dUfnY)和一些从易贝采购的基本设备生产了 KClO3。

实验从含有 8.25%次氯酸钠的浓缩漂白剂开始。他需要氯酸钠，所以用一个热板来煮漂白剂，直到晶体开始形成。实现这种反应的更有效的方法是使用电解(查看[我们最近看到的 HHO 发电机](http://hackaday.com/2013/08/17/hho-generator-makes-bubbles-that-go-boom/)自制的钻机)。下一步是添加氯化钾，它来自杂货店，作为无钠盐的替代品。在与漂白反应的过滤残余物混合后，将两者合并。这并没有什么突然性——这表明并不是所有事情都是应该的。但是在冰箱里过夜会产生如上图所示的氯酸钾晶体。

激烈的测试(见下文)让他知道实验成功了。从这里开始，产品可以用于制造固体火箭发动机。
