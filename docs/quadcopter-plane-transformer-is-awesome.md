# 四轴飞行器平面变形金刚超赞

> 原文：<https://hackaday.com/2015/02/15/quadcopter-plane-transformer-is-awesome/>

是四轴飞行器吗？一架飞机？不，两者都是！丹尼尔·鲁布里又开始了他的垂直起降变形金刚之旅，他称之为“空中徘徊者”。

SkyProwler 使用开关叶片型机构从四轴飞行器模式移动到平面模式。机翼可以拆卸，使它成为一个普通的四轴飞行器，拥有所有典型的铃铛和哨子。它可以通过 GPS 跟随你，通过航路点自主飞行，并具有这种很酷的万向节机制，可以在无人机飞行时保持 GoPro 稳定，从而提供更好的视频体验。

[丹]的最终目标是名为 [SkyCruiser](http://www.krossblade.com/) 的全尺寸乘客模型，它使用与他小得多的 SkyProwler 相同的弹簧刀变形机制。如果你还没有看过下面的视频，请一定要看看，并且[让我们知道](http://hackaday.com/submit-a-tip/)任何你自己的四轴飞行器/飞机混合飞行器。

**更正:**我们之前将[Daniel Lubrich]与 ATMOS 计划相关联。这是错误的，已从文章中删除。[ATMOS UAV](http://www.teamatmos.nl/)是一个单独的项目，我们之前在的[中已经介绍过了。](http://hackaday.com/2012/09/04/autonomous-plane-quadrotor-both-meet-the-atmos/)
