# 巧妙的化学反应导致更快的 3D 打印

> 原文：<https://hackaday.com/2015/03/19/clever-chemistry-leads-to-much-faster-3d-printing/>

树脂印刷，它可能会很乱，但你会得到很高的分辨率，这要归功于用投影仪发出的光固化粘性物质的光学特性。很快，它的腰带上又多了几个刻痕来统治它的沉积表亲:速度和缺乏层次。[树脂印刷的突破](http://www.nature.com/news/chemical-trick-speeds-up-3d-printing-1.17122)使其速度比以往任何时候都快，并且几乎消除了印刷结构的分层。

这个概念在树脂池的底部使用了透氧层。这抑制了固化，显然是突破的来源。树脂就在这一层的边界上固化，并允许所谓的连续生长过程，而不是基于层的方法。所描述的好处之一是在提取零件时不需要树脂流入，但我们对这种说法持怀疑态度(树脂仍然需要从某个地方流出)。尽管如此，对我们来说，需要使用昂贵的、可能脏乱的、有有效期的树脂(至少与塑料丝相比)使得沉积成为一个竞争者。与基于层的技术相比，速度的提高和强度优势的声明可能就是那个杀手级的特性。

这项技术来自一家名为 [Carbon3D](http://carbon3d.com/) 的公司。他们将其命名为 CLIP，即连续液体界面生产。休息之后，你可以看到这个概念的视频说明(对我们的口味来说有点太简单了)，以及该公司首席执行官约瑟夫·德西蒙(Joseph Desimone)本月的 TED 演讲。当然，还有强制性的延时打印演示。
