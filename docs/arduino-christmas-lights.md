# Arduino 圣诞灯

> 原文：<https://hackaday.com/2013/12/01/arduino-christmas-lights/>

这里有一个让你沉浸在 12 月节日气氛中的妙招！Arduino 控制的圣诞灯！

这一切都是因为[Anx2k]在他的另一个项目中有一些剩余的 LED，所以他决定将它们用作永久安装的圣诞灯。他将它们安装在他的瓦屋顶下，并将所有电线接入他的阁楼，在那里他有一个作为主要控制中心的电箱。他使用 Arduino Uno 来控制它们，并使用 460W 的电脑电源来提供电力。LED 模块本身是 Adafruit RGB 像素串。实际上每个瓷砖有三个 LED 模块——两个向上发光照亮瓷砖，一个向外发光。

他设置了一大堆不同的模式来运行，它们非常棒！休息之后请看视频。
