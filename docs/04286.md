# 树莓皮车库门自动化

> 原文：<https://hackaday.com/2013/10/13/raspberry-pi-garage-door-automation/>

像我们许多人一样，[费利克斯]有时怀疑他是否真的关上了车库门。我知道我总是反复检查我的车锁！所以，他决定将他的车库门自动化，让他安心一些！

他一直在做一个相当大的 [Raspberry Pi 家庭自动化系列](http://lowpowerlab.com/blog/2013/10/02/raspberrypi-home-automation-gateway/)，在这个最后的片段中，他展示了他的新 GarageMote 板，正如你所猜测的，它可以让他无线控制门。这是一个非常简单的板，配有一个小继电器、一个二极管和两个电阻。8 针接头连接两个霍尔效应传感器，用于检测车库门和原始开门器的状态。然后他将这个连接到他自己设计的开源无线 Arduino 克隆体，被称为 [Moteino](http://lowpowerlab.com/moteino/#whatisit) 。其中一对与 Raspberry Pi 通信，Raspberry Pi 充当他的安全家庭自动化网关服务器。

整个项目非常巧妙，并且有很好的文档记录——所以如果你正在考虑让你的家自动化，那么[Felix]有丰富的知识可以分享——至少如果你想用的话是 Raspberry Pi！

休息后留下来看看网络服务器控制的车库门。
