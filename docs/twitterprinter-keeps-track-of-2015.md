# TwitterPrinter 追踪 2015 年每日黑客奖

> 原文：<https://hackaday.com/2015/03/11/twitterprinter-keeps-track-of-2015/>

[Mastro Gippo]正在成为一个传奇人物。他没有赢得 2014 年的 Hackaday 奖，但参加了在慕尼黑举行的活动，为了确保他跟上今年的奖项，他建造了这个[老式打印机，打印 Hackaday 奖 Twitter 帐户的所有更新](http://www.mastrogippo.it/2015/03/the-hackaday-prize-2015/)。

该设备使用现在著名的 ESP8266 模块将打印机连接到互联网。不过，它并不直接从 Twitter 上抓取数据，而是查看[Mastro Gippo]自己的服务器，以避免一次被过多的推文淹没。该程序将推文分割成适合打印机的格式(纯文本)，然后打印机可以将数据解析到纸上。设计的其余部分包括一个 3.3V 电源调节器和一些晶体管来打开和关闭打印机。休息之后，请务必查看该设备的运行视频！

[Mastro Gippo]指出，这消除了为了跟上 2015 年 Hackaday 奖而拥有智能手机的需要，这很讽刺，因为他参加的[小饰品日常携带竞赛](http://hackaday.com/2015/01/07/trinket-edc-contest-winners/)是一部比一般人更智能的[手机](http://hackaday.io/project/3472-1337-3310-tool)。不过，我们希望他的正式参赛作品不会浪费太多纸张！
