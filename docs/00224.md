# 用内部 IMU 对物体建模

> 原文:[https://hack aday . com/2012/04/26/modeling-an-object-with-internal-imus/](https://hackaday.com/2012/04/26/modeling-an-object-with-internal-imus/)

[约瑟夫·马洛赫]发来了一个非常酷的视频，他在 3D 空间中为一块扭曲转动的泡沫塑料建模。

为了转换他的泡沫片的扭曲、弯曲和转弯，[Joseph]使用了几个惯性测量单元(IMU)来跟踪可变形物体的形状。这些 IMU 由一个 3 轴加速度计、3 轴陀螺仪和一个 3 轴磁力计组成，用于跟踪它们在 3D 空间中的运动。当这些 IMU 沿着可变形物体放置时，可以从计算机下载数据，并且可以在虚拟空间中重建物体。

这个项目来自蒙特利尔麦吉尔大学[输入设备和音乐交互实验室](http://www.idmil.org/home)富有成果的头脑。虽然我们不太确定如何在用户界面中使用模型化的可变形物体，但是新生婴儿有什么用呢？如果你知道这个可以用来做什么，请在评论中留言。也许[的动力手套](http://en.wikipedia.org/wiki/Power_Glove)需要更新了——一种支持 IMU 的连体衣，会让 Kinect 相形见绌。
