# 用 Arduino 泡茶很完美

> 原文：<https://hackaday.com/2015/05/21/steeping-tea-perfectly-with-an-arduino/>

虽然 Hack a Day 的大多数人都离不开我们的日常 java，但我们理解并尊重那些喝茶的黑客，比如 24 小时工程师[Brian McEvoy]。像任何自尊的黑客一样，[Brian]寻求提高日常任务的效率，以便将时间花在真正重要的事情上——因此他决定自动化他的茶杯。

他的 3D 打印了一个小型茶叶装袋机制，一个小小的 RC 伺服电机可以启动，这使他可以控制茶叶袋在杯子里浸泡的时间。另一个 3D 打印的外壳包括 Arduino、几个按钮和一个 LCD 屏幕，允许你选择你最喜欢的草药的浸泡时间。事实上，这个项目的大部分是 3D 打印的，这意味着大部分成本来自所需的最小电子设备——你可能已经有了这些东西。他还包括了所有你需要的设计文件，以便制作你自己的。

这个项目已经进行了一段时间，但他最终完成了，而且效果很好。如果你渴望了解一些基本的构建日志和一路的故障排除，他有一大堆贯穿整个过程的博客文章。

它在功能上与我们今年早些时候分享的自动泡茶机非常相似，但我们喜欢他们如何采取完全不同的方法来解决完美定时的问题。请注意，他们都使用 Arduino——但你真的能责怪他们吗？
