# 希望这是真的:用砖块 3D 打印房子

> 原文：<https://hackaday.com/2015/06/29/hope-its-real-3d-printing-houses-with-bricks/>

你必须接受这一次的大肆宣传，因为它显然还没有准备好进入黄金时间。但几天前，网上开始流传这样的谣言:一位澳大利亚发明家开发出了一种机器人，能够自己用砖块建造复杂的结构。

在你发疯之前……我们绝对不认为这是真的。这充其量只是一个概念证明，但这并不妨碍我们兴奋起来。你等待能代表我们建造整个建筑的机器人有多久了？我们对[挤压混凝土墙](http://hackaday.com/2014/05/29/man-builds-concrete-3d-printer-in-his-garage/)的前景感到兴奋。但这更像现实世界中的乐高建筑。野兽将砖块切割成一定长度，沿着伸缩臂传送每块砖块，并在放置砖块时涂上黄油。至少休息后的渲染视频是这么显示的。

我们现在听到这个消息是因为[马克·皮瓦茨]创立并花了十年时间开发哈德良项目的 FastBrick 机器人公司刚刚被卖给了一家名为 DMY 资本有限公司的公司。他们当然会想从销售中得到一些新闻。

有一个现有挖掘机上的砖块进料器的图像，坦率地说，看起来像 PS 过的。还有一些真实的图像，比如这里看到的和另一个“打印头”拿着一些砖块。但足以认为这里有潜力。

这个想法是，机器人的底座是固定的，手臂足够长，可以到达正在建造的结构的任何部分。精确定位是通过在不同于机器人的位置上的固定标记来实现的。头部使用激光测距和标记对其位置进行三角测量(已经说过，我们现在假设需要不止一个标记)。

你觉得怎么样？我们会不会在我们的邻居家看到这种令人难以置信的复杂的可怕的生产结构，而大坏狼根本无法吹倒它？
