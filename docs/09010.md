# 真人大小的乐高飞船零件

> 原文：<https://hackaday.com/2015/05/11/life-sized-lego-spaceship-parts/>

啊，80 年代的太空乐高玩具。你可能认为海盗船和城堡更酷，但你错了，因为宇宙飞船。宇宙飞船。宇宙飞船。

这些太空乐高套装有一些非常有趣的部件，微小的两英寸斜面部件印有乐高模拟的计算机、显示器、电话、对讲机、扬声器、控制面板和其他一切使体素飞船飞向月球的东西。现在，这些作品已经可以使用了，而且几乎和实物一样大。

[Love Hultén]将这些奇妙的部件进行建模，并将其放大到正常乐高尺寸的六倍。然后，这些砖块被安装上按钮、显示器、一部旧电话的内部部件以及所有其他使这些砖块具有功能的装备。两个电脑积木可以连接在一起，它会用一个乐高形状的控制器玩视频游戏。对讲机工作正常，控制面板上的按钮可以用来开灯。

应该注意的是，乐高家族不仅仅是当你踩在上面时会感到疼痛的小砖块。Duplo 是为那些想把乐高塞进自己喉咙里的孩子制作的积木，是乐高的两倍大。Quatro 是为幼儿制作的积木，是 Duplo 的两倍，Lego 的四倍。由于[Love]制作的积木是普通乐高积木的六倍大，我们将让评论来决定这类积木应该如何命名。
