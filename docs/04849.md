# 电影胶片:过去是记录在胶片上的

> 原文:[https://hackaday . com/2013/12/17/retro tech atacular-films-used-to-be-record-on-film/](https://hackaday.com/2013/12/17/retrotechtacular-films-used-to-be-recorded-on-film/)

![retrotechtacular-movie-film-photo-finishing](../Images/34a9d2c07b58be7ff15cd12f626b58b1.png)

我们肯定这个标题会让一些读者发痒，因为仍然有许多备受尊敬的导演坚持用胶片而不是数字拍摄，但本周的主题展示了电影行业在过去几十年中肯定被降级为生命支持的一部分。照片冲印，曾经是所有人用来冲洗照片的化学过程的大本营，现在已经几乎不存在了。这是一个关于电影和照片洗印如何在电影的大部分时间里推动电影发展的故事。

上面看到的卷轴是负片和正片。底片进入照相机并捕捉图像。在显影和固定底片之后，重复该过程。光线穿过固定的底片，以便曝光一卷新的胶卷。该胶片被完成并固定以产生可用于投影仪的卷轴。这个简单的过程包含在下面的剪辑的开始部分。1940 年的报告继续深入讨论该过程中使用的化学技术。但是，你真的会在看到一半的时候开始享受旧的手工方法，这些方法已经被“现代实验室”所取代。我们喜欢那些巨大的模拟表盘！视频最后展示了电影发展过程的真正工业化。
