# 一张展开的木头桌子

> 原文:[https://hackaday.com/2014/12/07/an-expanding-wooden-table/](https://hackaday.com/2014/12/07/an-expanding-wooden-table/)

几年前，精细木工领域推出了[弗莱彻绞盘桌](http://fletchertables.com/)。这是一张圆桌，只需旋转桌面就能扩大直径。一个精心设计的机械装置将桌子的叶子向外移动，同时将桌子的内部抬高。这是一张非常酷的桌子，非常昂贵，100 年后可能会在博物馆里找到。

[Scott Rumschlag]认为他的木工技术足以完成制作这种扩展桌的任务[，并设法在他的工作室里制作了一个](https://www.youtube.com/watch?v=7JFJSIJIe9A)。像弗莱彻绞盘表，这是一个表，增加其直径简单地通过旋转桌面。与商业产品不同，这款产品的价格不像一辆汽车那么高，你可以看到桌子内部的内部结构。

[Scott]的桌面由三部分组成。四分之一圆块是唯一显示桌子处于最小位置的东西，排列在“叶叠”的顶部。当桌子展开时，在木头制成的线性轴承和沿着这个机械装置底座滑动的滚轮的帮助下，另外四片叶子从下面向上移动。

桌子的中心——星星——有点难以设计。虽然树叶在斜坡的帮助下沿着桌面向上移动，但对于如此靠近桌子中心的东西来说，这是一种不切实际的解决方案。[Scott]没有使用斜坡，而是使用了提升杆和金属铰链，将桌子的星形提升到正确的高度。尽管做好每件事都要做大量的木工活和微调，但让你的头脑清醒过来并不太难。
