# 3D 打印肌肉在打印过程中膨胀

> 原文:[https://hack aday . com/2015/07/15/artial-muscles-3d-printed-on-inflatable-surface/](https://hackaday.com/2015/07/15/artifical-muscles-3d-printed-on-inflatable-surface/)

就在你认为你已经看到了这一切的时候……[费戈尔·库尔特]在 RepRap 论坛上提出了一种在充气结构上进行 3D 打印的方法——等等什么？

该过程使用一台定制的 3D 打印机，带有一个牙膏挤出头和一个第四轴——带有气动空气供应。使用喷射沉积方法，形成硅树脂管，然后使用红外线固化每一层，红外线也内置于系统中。一旦硅胶足够厚，就通过透气心轴加压膨胀。然后，激光扫描膨胀硅胶的形状，让计算机为表面生成工具路径。然后你点击打印。简单吧？

我们甚至不确定如何开始解释他是如何想到这个的。基本上，这种 3D 打印的方法允许制造一些非常独特的柔性组件，也许最有趣的应用是“电介质弹性体人工肌肉”，这是一种真正先进的气动人工肌肉(你可以用五金店的零件来制造)。这也使得制造一个柔软的机器人比我们上周看到的手工铸造硅树脂肌肉更容易[。](http://hackaday.com/2015/07/08/soft-and-squishy-silicone-robotics/)

这是 3D 打印的一个迷人的新方向，我们很高兴看到这项技术继续发展。在[弗戈尔公司] [主网站](http://fergalcoulter.eu/)上有更多信息。
