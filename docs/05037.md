# USB 充电器解决蒙古用电问题

> 原文:[https://hack aday . com/2014/01/11/USB-charger-solves-Mongolia-electricity-problem/](https://hackaday.com/2014/01/11/usb-charger-solves-mongolia-electricity-problem/)

![thermoelectric](../Images/633f2b09106604ee37d8668f4856d237.png)

生活在蒙古偏远地区的人们没有电和煤气，他们的家庭依靠传统的柴炉，几乎所有时间都在使用。许多人使用太阳能电池板为小型工具发电，但不幸的是，经常会连续几天阴天。【Chingun 已经】看到了这个问题，创造了自己聪明的解决方案——[一个小型的热电 USB 充电器](https://www.youtube.com/watch?v=TGY0xNx9HPE)。

[Chingun 的]设备的特点是在一个铝罩内有一系列珀尔帖板。这种装置被设计成可以放在炉子上，或者绑在炉子的管子上。当珀尔帖板两侧的温差足够大时，就会感应出电荷。他用一个小风扇来帮助冷却珀尔帖板的另一面。一个小控制箱内有一个电压调节器电路，通过 USB 提供 5V 电压。

这个项目很酷的一点是，它在一定程度上是麻省理工学院教授[Tony Kim]的成果，他大约一年前去蒙古给学生们讲授 edX 电路课程。[Chingun]是他的一个学生，这是一个解决现实世界问题的很好的例子。

休息之后的一段精彩视频对这个项目进行了完整的解释，由[Chingun]亲自详述——非常值得一看！
