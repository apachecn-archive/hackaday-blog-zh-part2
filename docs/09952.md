# 了解表面贴装

> 原文：<https://hackaday.com/2015/09/07/understanding-surface-mount/>

你知道什么是 MELF 吗(是的，在工作中谷歌一下是安全的)？QFP 包裹、LCC 包裹和 PLCC 包裹有什么区别？需要 0603 电阻还是 1206 电阻？

如果你是表面贴装器件(SMD)的老手，你可能知道这些问题的答案。但是如果你已经用通孔完成了大部分工作，那就是一堆混乱的首字母缩写和数字。当然，你可以谷歌一下，发现 at 0603 电阻是 0.06 英寸乘以 0.03 英寸。[TopLine]有一本很棒的小册子，它将许多常见的定义集中在一个地方，可供下载，可以帮助您理解不同的 SMD 术语。

这本小册子不仅仅是一个图表或表格。例如，它定义了经常混淆的术语，如扫描和镊子。我们确实注意到它缺少一些像 CCGA 这样的新软件包，但它仍然是一个很好的入门指南。

如果你仍然害怕用 SMD 构建，你可以看看之前[介绍过的](http://hackaday.com/2014/12/31/3d-printed-smd-tool/)[一些工具](http://hackaday.com/2015/02/05/diy-smd-twofer-manual-pick-and-place-and-the-beak/)。如果你不想要特殊的工具，你可以试试[牙签](http://hackaday.com/2013/02/01/populate-smd-boads-using-a-toothpick-and-tweezers/)(或者试试视频中的简单方法)。
