# 装满了出租车的吉普牧马人

> 原文:[https://hack aday . com/2013/07/05/packing-a-jeep-wrangler-full-of-hacks/](https://hackaday.com/2013/07/05/packing-a-jeep-wrangler-full-of-hacks/)

只挑选一张图片来展示这辆吉普牧马人的所有设计是一项艰巨的任务。我们决定使用这种定制的天花板控制台，因为它的特点是在有限的区域内完成最多的工作。

在你决定视频不适合你之前，给它一点时间。[Eddie Zarick]首先在贴花、徽章、刺绣座椅、拉链头等方面宣传他的“Oakley”品牌。但是之后你会看到我们之前看到的加压水系统。拉开后门，有一个他建造的漂亮的货物盖，包括一个小房间，当他想把顶部拿掉时，可以储存柔软的侧面。还有其他一些有趣的东西，比如他用来吓唬超速者的警用雷达欺骗器。哈！

我们之前提到的天花板控制台完全是定制的。它包括一个 CB，扫描仪，火腿和七英寸的 Android 平板电脑。还有一组控制各种铃铛和哨子的按钮；实际上是聚光灯和逆变器。只要加一个马桶，他就可以住在他的车里了。
