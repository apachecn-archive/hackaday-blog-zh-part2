# 将 3D 打印机变成注塑机

> 原文：<https://hackaday.com/2015/02/16/turning-a-3d-printer-into-an-injection-molding-machine/>

注射成型机能够形成非常详细的塑料零件，只需将塑料喷入模具。3D 打印机喷射塑料。为什么没有人想到使用 3D 打印机挤压机将塑料推入模具，这是我们永远也不会知道的。

[bfk]已经研究了一段时间来生产非常小、非常详细的零件，并实现了 3D 打印机的挤出机服务于注塑机的大部分功能。它将塑料熔化，并迫使其通过一个小孔。塑料是进入构建平台还是进入模具并不重要；但是通过一个简单的硅胶模具，任何人都可以用每个黑客空间都有的工具复制极小的零件。

所需的工具是 RTV 橡胶，这是最受欢迎的模具材料。除此之外，它只是硅胶润滑剂，销钉和乐高来制作浇口，当然还有一些制作模具的材料。一旦模具制作完成，只需将模具举到打印机的喷嘴前，挤出一点塑料即可。

最终的“打印”像树脂打印机打印出的最好的照片一样细致。它非常适合制造像[bfk]当前项目这样的非常小的模型的零件，但这种技术可以扩展到任何需要大量具有严格公差的小塑料零件的地方。
