# PVC 密码筒保护你的东西安全

> 原文：<https://hackaday.com/2014/02/01/pvc-cryptex-keeps-your-stuff-safe/>

显然我们以前从未共用过密码筒！因《达芬奇密码》而流行(并被创造)的密码筒是一种组合式锁，最初用于存储秘密信息。你可以用几根 PVC 管、一些环氧树脂、螺母和螺栓制作自己的[！](http://www.instructables.com/id/Cryptex-5/)

地窖有一个巧妙的设计，如果不简单地破坏它，就无法真正撬开锁。在[丹·布朗]的小说中，据说一些地下室储存了一瓶易碎的醋，上面写着纸莎草纸——这样，如果强行打开，瓶子就会破裂，溶解薄薄的纸莎草纸。

它们的工作原理类似于自行车的密码锁，如果猜中了正确的密码，里面的制动栓就会对齐，让锁芯的两半分开。它实际上是一个非常简单的内部设计，整个机制可以很容易地用最少的工具自己制作。

使用聚氯乙烯是一个相对简单的项目，尽管你也可以用金属制作一个更令人印象深刻的项目，就像这个用铜管制作的令人惊叹的项目。
