# 盒子制作的艺术

> 原文:[https://hackaday.com/2013/12/31/the-art-of-box-making/](https://hackaday.com/2013/12/31/the-art-of-box-making/)

![IMG_2507-1000](../Images/4f71782a444721ab879cf9e508e06471.png)

[达西]有点喜欢纸板。最初只是通过定制尺寸的盒子来降低邮寄成本的简单方法，现在已经变成了成熟的盒子制作艺术。

他最初是通过手工制作盒子开始的，但是在他变得相当熟练之后，他很快通过加入一些技术改进了他的工艺。他现在在 SketchUp 中设计盒子，然后使用自制的 CNC 路由器将纸板切割和刻划成更别致的风格。他的博客上有很多他的纸箱设计，看到他的设计真的很酷。他也有一堆自己制作的技巧，所以如果你是那些幸运的黑客之一，可以出售他们制作的东西，那绝对值得一看！如果你不卖任何东西，也许纸板灯罩更适合你。

要查看他的一个 CNC 切割盒的视频示例，请在休息后留下来。现在他需要做的就是设计一台自动折盒机！
