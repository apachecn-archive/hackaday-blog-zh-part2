# 配有强力鞋带的耐克鞋，明年刚刚上市

> 原文:[https://hackaday . com/2014/10/29/nikes-with-power-laces-just-in-time for the next-year/](https://hackaday.com/2014/10/29/nikes-with-power-laces-just-in-time-for-next-year/)

随着世界上第一个悬浮滑板在几天前展示，我们正处在从*回到未来*的神话世界的边缘。悬浮滑板很酷，但还有很多来自遥远的 2015 年的其他酷技术:融合先生、充气披萨、防尘罩，当然，还有带电动鞋带的耐克。[【猎人】刚刚造好了它们](http://www.hscott.net/shoes/)，并配上合适的鞋子，重新开机。

[Hunter]使用了受 *BttF* 启发的 Nike Air Mag 鞋，以及一些电子设备——一个 Arduino pro mini，一个力感测电阻器和一个电机。开始制作时，先在鞋的背面刻出一个凹槽，用于安装电子设备。一根小钓鱼线绕在鞋子上，为动力鞋带提供动力。

鞋垫后跟下的力敏电阻告诉微控制器一只脚何时进入鞋内，电机轴上的旋转编码器确保所有的动力鞋带周期都相同。这与屏幕上看到的鞋子不太一样——较低的鞋带无法复制，当然也没有 *BttF* 鞋快，但它确实有用，就鞋带而言，它们很好用。
