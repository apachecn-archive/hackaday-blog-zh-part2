# 购物手推车为露营装了电线

> 原文:[https://hack aday . com/2013/12/17/shopping-trolley-is-wired-for-camp/](https://hackaday.com/2013/12/17/shopping-trolley-is-wired-for-camp/)

[詹姆斯]为即将到来的新西兰复活节营地需要一些很酷的交通工具，所以他创造了一个[定制的电动购物手推车](http://jct250.wordpress.com/2013/10/08/finished-trolley/)，它一定会吸引眼球。这个项目的基础是一个标准的移动踏板车，它方便地采用模块化设计。所有的电子设备都有快速维修的连接器，整个后轮轴和电机组件只需拉动一个杠杆就能弹出。

[詹姆斯]必须做一些焊接和底盘返工，以实现他的目标，安装一个购物车顶部的滑板车的框架。一旦完成，虽然，设置看起来很棒。实际上坐在里面很舒服，因为[詹姆斯]做了一个切口让车手的脚穿过。真正的乐趣来自电子产品。手推车是我们见过的最有线的代步车模型。大多数电子设备都包含在座位下的一个项目箱中，有几个 Arduinos 控制着各种系统:与最初的踏板车电子设备，GPS 接收器和 GSM 无线电连接。[詹姆斯]还增加了 RGB LED 头灯、喇叭和 Jaycar 的多音警报器。

驾驶手推车很简单。街机操纵杆选择速度，滑板车的标准手动控制用于前进、后退和转向。[詹姆斯]做的一个更有趣的模块是一个定制的 Windows 应用程序，通过 USB 无线电模块控制手推车。整个系统可以得到保护，安全代码存储在 NVRAM 中，以防止电源循环解锁系统。[詹姆斯]甚至可以从他的触摸屏上命令电车前进或后退。我们很乐意看到他增加一个转向伺服系统，使它成为一个完全远程控制的解决方案，尽管这一步需要某种手动控制的离合器。

最终设计效果非常好。[詹姆斯]可能不会通过保留 scooter 的原始速度控制和相关电子设备赢得任何短程加速赛，但他确实用更大的电池扩大了范围，所以我们确信手推车会在整个营地受到欢迎。使用电动轮椅的底座已经建造了类似的项目。如果你有一个你想要控制而不需要对硬件进行侵入性的改变，看看这个使用连接器接口的[可访问性黑客。](http://hackaday.com/2013/08/29/wheelchair-hack-lets-two-year-old-explore-on-his-own/)
