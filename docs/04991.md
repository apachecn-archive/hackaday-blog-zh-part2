# 复古追逐者标志点亮你的生活

> 原文:[https://hack aday . com/2014/01/05/retro-chaser-sign-lights-up-your-life/](https://hackaday.com/2014/01/05/retro-chaser-sign-lights-up-your-life/)

![lighted-sign](../Images/21b799207f5bef249c0b1dd6381e03ea.png)

[Gnsart]制作电影行业常用的道具。他创造了一个惊人的[复古拉斯维加斯风格的追光标志](https://imgur.com/a/9lb8b)。这个标志是几年前作为一项工作开始的。虽然[Gnsart]可以处理物理装配，但机械追光器的成本使项目超出了预算。标志工程当时被取消了，但他从未忘记。

快进到几周前。发生在 Arduino 社区。他意识到，有了 Arduino Uno 和一个普通的中继板，他终于可以制作这个标志了。他从一些剩下的雪松篱笆桩开始。纠察队被粘起来，然后切成箭头形状。然后用桨状钻头设计并钻好灯孔。[Gnsart]想让木头看起来有点老，所以他创造了一个 ebonizing 染色。0000 钢丝绒，浸在醋里，让它生锈几天，制成了一种非常适合这项任务的液体。溶液被刷上去，就像污渍一样被去除，从而形成一个老化的木材。我们以前见过这种技术用于茶叶、染色剂和其他材料来达到预期的效果。

[Gnsart]然后建立他的边缘。22 号钢板被弯曲以适应弯曲制动器的轮廓。钢板被钉在木头上，然后点焊成一个连续的整体。最后，安装了灯插座，并连接到 Arduino。[Gnsart]首先试验了机械继电器，虽然我们喜欢这种声音，但我们不确定它们能持续多久。他明智地决定最终采用固态继电器。结果不言自明。led 很棒——但是低瓦数的白炽灯有一种温暖的光芒。
