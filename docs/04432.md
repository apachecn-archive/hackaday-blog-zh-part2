# 无限极 HMD 带来 210 度 FOV 到党

> 原文:[https://hack aday . com/2013/10/29/infinite ye-HMD-brings-210-degree-FOV-to-the-party/](https://hackaday.com/2013/10/29/infiniteye-hmd-brings-210-degree-fov-to-the-party/)

今年，头戴式显示器又热又重。InfinitEye 还没有官方网页，所以我们链接到由 TheRoadToVR 完成的[评论。请注意，这是显示的第二个版本。InfinitEye](http://www.roadtovr.com/infiniteeye-worlds-first-ever-hands-210-fov-hmd-video/) [在二月](http://www.mtbs3d.com/phpbb/viewtopic.php?f=138&t=16434)发布了他们的 V1 HMD 计划。InfinitEye 原型看起来与早期的 [Oculus Rift](http://hackaday.com/2012/08/17/were-getting-an-oculus-rift-dev-kit/) 原型惊人地相似。钩带和看起来像是面罩的框架将光学系统连接在一起。有趣的是这个光学系统。InfinitEye 已经决定采用头戴式液晶显示屏，类似于 rift，而不像 [castAR 的投影系统](http://hackaday.com/2013/10/15/castar-goes-live-on-kickstarter/)。

InfinitEye 团队决定采用两块屏幕，每只眼睛的分辨率高达 1280×800。光学系统也很简单—[菲涅耳透镜](http://en.wikipedia.org/wiki/Fresnel_lens)。这与第一个版本的护目镜非常相似，但是 InfinitEye 团队声称这个新版本提供了 210 度的视野。我们不知道的是他们到底改变了什么。我们很好奇更宽的视野是否会减少我们一些人对裂缝的 [Sim 病](http://www.siggraph.org/education/materials/HyperVis/virtual.env/percept.iss/simulate.htm)——尽管公平地说，几乎任何头戴显示器都需要一些时间来调整。我们可以肯定的是，虚拟(和增强)现实的未来是光明的。
