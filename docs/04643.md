# 使用 SSRelay 进行 DIY 家居控制

> 原文:[https://hack aday . com/2013/11/26/DIY-home-control-using-a-ss relay/](https://hackaday.com/2013/11/26/diy-home-control-using-a-ssrelay/)

![ssrelay](../Images/228c22f06d3580d8d946c55271d02697.png)

[Ben Jones]刚刚创建了一个名为 Maker-Guide 的新网站，他在那里制作了一些非常翔实和制作精良的 DIY 视频，内容涉及从自制摄影技巧到使用固态继电器控制插座的任何东西。

这不是我们第一次看到[继电器控制的插座](http://hackaday.com/2010/01/31/120v-switching/)，或者[这个……](http://hackaday.com/2011/10/31/arduino-controlled-four-socket-outlet/)，但这肯定是我们见过的最干净的工作之一。在他的视频指南中，[Ben]向我们展示了如何将标准固态继电器安装到常规插座盒中，并使用 Arduino Uno 轻松控制它。

它甚至看起来似乎有足够的空间在盒子里安装一个小型的无线设备——甚至可以使用一个小饰品？用电力线网络通过局域网控制每个盒子怎么样？可能是迄今为止最简单的家庭自动化实现！嗯，从某些 NEC(国家电气规范)方面考虑，高低压在同一个盒子里运行…

不管怎样，如果这让你扫兴的话，休息之后看看那些很棒的视频指南吧！
