# 商业产品中的 ESP8266

> 原文:[https://hack aday . com/2015/08/25/esp8266-in-commercial-products/](https://hackaday.com/2015/08/25/esp8266-in-commercial-products/)

业余爱好者的电子市场仍然很小，即使随机的公司推出了一些非常有趣的硬件，这些零部件并不完全是为我们准备的。ESP 8266 WiFi 模块与这一趋势略有偏差，数百种不同的 ESP 开发板四处浮动，怪人成袋购买。

【4 ndreas】[终于在一款产品](http://chaozlabs.blogspot.de/2015/08/esp8266-in-wild-wifi-led-controller-hack.html)中找到了 ESP8266 这不是一个非常值得注意的观察，直到您意识到为 ESP 开发开源工具链已经做了多少工作。

【4ndreas】在阿里快递上发现一个 [RGB LED 条，可以通过 WiFi 控制。在里面，他找到了每个人都喜欢的 WiFi 模块，通过短路两个引脚，他以引导模式启动了控制器。](http://es.aliexpress.com/item/rgb-WiFi-controller-1-port-control-15-rgb-lights-communicate-with-Android-phone-to-dim-output/32301423622.html)

由于围绕 ESP8266 的大量开源开发，有许多工具可用于编程这种廉价的 LED 控制器。[4ndreas]尝试着为控制器编写自己的固件，[想出了这个项目](https://github.com/4ndreas/WiFI-RGBWW-module)。

这不是一个杀手级项目，但它确实展示了廉价 WiFi 模块开源工具链的威力。这只是发现的第一个内置 ESP8266 的产品，但毫无疑问，还有其他产品正等着被拆开，并以更先进的方式控制。
