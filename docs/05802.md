# 瑞典音乐节上演奏的冷冻乐器

> 原文：<https://hackaday.com/2014/04/15/sweden-music-festival-plays-frozen-instruments/>

[蒂姆·林哈特]想为这个瑞典音乐节做些不一样的事情——所以他决定用冰手工雕刻所有的乐器。

音乐节由七支演奏不同音乐风格的乐队组成，在音乐节期间有超过 40 场音乐会。从小提琴到大提琴，蒂姆·林哈特(Tim Linhart)在一张张冰面上精心雕刻了每一件乐器。他添加琴弦和音板来完成每一件作品，如果温度超过零度，游戏就结束了。音乐会在一个用冰建造的建筑中举行，以确保这种情况不会发生。

由于它们是由多层构成的——他还加入了一些 RGB 发光二极管，让这些乐器更有活力。他们实际上听起来也很不错！
