# 背光 PCB 面板作为墙面艺术

> 原文：<https://hackaday.com/2013/09/24/backlit-pcb-panel-as-wall-art/>

在他朋友的婚礼上[萨尔·德瑞默]想要一件独一无二的礼物，还有什么比一件艺术品更独特的呢？他开始设计一些东西，可以和他喜欢游戏的极客朋友说话。这个全面板 PCB 就是他想出来的。这是一款[壁挂式灯具，使用可寻址 led 和 PCB](http://boldport.blogspot.co.uk/2013/09/the-lifegame.html) 作为外壳和扩散器。

在右边，你可以看到交付给他的面板。他使用布线槽将外壳主体与四个侧部件和安装支架分开。这种设计让他可以折断零件，然后将它们焊接到位。你唯一需要加进去的就是一对螺丝(嗯，还有让它亮起来的组件)。

我们对 PCB 作为扩散器的出色表现感到震惊。当没有覆盖丝网或铜层时，衬底是半透明的。字母的轮廓利用了这一点，以及沿侧边的圆形区域。字母本身是铜质填充物，上面去除了艺术图案。当作品由 42 个 WS2812B LEDs 照明时，这确实增加了视觉吸引力。下面的视频展示了这件作品。这真的将 PCB 作为艺术提升到了一个新的高度
