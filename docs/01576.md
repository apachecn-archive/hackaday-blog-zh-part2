# 控制精神的蟑螂

> 原文：<https://hackaday.com/2012/09/10/mind-controlling-cockroaches/>

生产微型机器人还不容易，也不划算，但是当我们可以控制蟑螂的思想时，为什么我们还需要这么做呢？北卡罗来纳州立大学的一个团队或研究人员在他们最新的研究论文中称这种增强的马达加斯加发声蟑螂为昆虫生物机器人[。这个话题已经不是第一次被提起了。已经有了研究中的](http://ibionics.ece.ncsu.edu/assets/EMBC_12.pdf)[证明](http://hackaday.com/2010/06/11/cockroach-pimps-a-sweet-ride/)和更多的[业余努力](http://hackaday.com/2011/03/08/coming-soon-to-a-store-near-you-remote-control-cockroaches/)。但是休息后视频里看到的准确度和控制力是无与伦比的。

蟑螂被控制成完美地沿着地板上的一条线行走。让这种迭代如此有效的一个原因是，微控制器包括一种新型的基于 ADC 的反馈环路，用于刺激昆虫的大脑。这有助于确保蟑螂不会习惯于刺激而停止反应。由于这种昆虫可以活大约两年，这一突破使它成为一种可重复使用的工具。我们不确定这个工具会被用来做什么，但是也许下一次昆虫瘟疫会被人类控制，而不是大自然。
