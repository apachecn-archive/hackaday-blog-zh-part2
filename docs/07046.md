# 基于速度和速率更改兴趣点颜色

> 原文:[https://hack aday . com/2014/09/11/changing-poi-colors-based-on-speed-and-velocity/](https://hackaday.com/2014/09/11/changing-poi-colors-based-on-speed-and-velocity/)

LED 玩具已经成为地下狂欢文化的代名词，因为参加派对的人会盯着被称为 poi 的旋转光漩涡。大多数这些对象都是预编程的，但也有一些可以定制编码。然而，只要轻轻一点加速度计，就会根据它们在空间中的移动速度来改变彩色的能量圈。一个令人震惊的例子是这个被称为“中心逃离”的 LED 设备[，它将加速度计数据](http://c4ss4ndr4.stellr.me/center-flee-two/)转换为交替的 RGB 颜色序列。

LED 值被“打印”到旋转弧中特定点处的系留物体上。这些设备由 Arduino 控制，XBee 无线模块将数据传输到附近的计算机，无需在每次旋转后手动移除 SD 卡。

当旋转时，poi 就像表演者的彩色旋转延伸，产生迷人的，充满活力的效果。很高兴看到绑在鞋带上的荧光棒发展成为能够吸引周围观众的重力感应设备。

类似想法的其他例子包括用数控机床切割的这个[加速度计 poi](http://hackaday.com/2013/09/12/accelerometer-poi/) 和这些 [LED 五线谱](http://hackaday.com/2014/08/28/led-light-staffs-for-the-ultimate-portable-rave/)用于最终的便携式 rave。
