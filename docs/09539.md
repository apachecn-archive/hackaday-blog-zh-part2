# 汉姆希尔把你的 Arduino 放在无线电上

> 原文:[https://hack aday . com/2015/07/13/hamshield-puts-your-arduino-on-the-radio/](https://hackaday.com/2015/07/13/hamshield-puts-your-arduino-on-the-radio/)

任何人都可以抓住一个 USB 电视调谐器卡，并开始监测电波，但要进入真正的收音机，你需要你的业余无线电执照。一旦你有了这些，带宽就真的打开了…如果你买得起设备的话。然而，[spaceneedle]和他的朋友们通过为 Arduino 制造这种火腿无线电防护罩，大大降低了成本，同时增加了拥有无线电的可能性。

HamShield 是适用于任何标准 Arduino 的多功能屏蔽，使其能够像现成的收音机一样工作，但功能几乎是无限的。任何可以想象的东西都可以被编程到 Arduino 中，供无线使用，包括语音和分组应用。该项目的沙盒已经包括了诸如建立网状网络、通过 APRS 通信、建立中继器或信标、监测气象站以及一系列其他业余无线电应用。

HamShield 在很宽的频率范围内工作，并且仅使用 250 mW 的放大器。能量消耗非常小，以至于 HamShield 团队[通过一个小型太阳能电池板](https://www.youtube.com/watch?v=YKwKY8bz9rc)来操作它，这使得它非常适合偏远地区的人们。该项目目前正在筹集资金，并且已经超过了他们在 Kickstarter 上的目标，恰如其分地将自己标榜为瑞士业余无线电大军。收发器似乎非常健壮，这意味着使用这个工具的唯一障碍就是简单地为你想做的任何项目编写 Arduino 代码，无论是作为[警察扫描仪](http://hackaday.com/2013/02/28/hacking-a-ham-radio/)还是仅仅作为[频率计数器](http://hackaday.com/2013/02/09/arduino-as-an-inexpensive-ham-radio-frequency-counter/)。如果你想继续关注 hackaday.io，[这个项目可以在这里找到](https://hackaday.io/project/6749-hamshield-vhfuhf-transceiver-for-arduino)。
