# 黑客空间简介:密歇根州大急流城的极客群体

> 原文：<https://hackaday.com/2012/04/27/hackerspace-intro-the-geek-group-in-grand-rapids-michigan/>

![](img/1eaa0648cc520b4fa9c8678b1a445b47.png "geek-group-hackerspace")

我们很高兴地看到一些链接从[我们呼吁黑客空间介绍](http://hackaday.com/2012/04/25/hackerspaces-and-hackaday/)开始滚滚而来。这是对极客群体[的一种重新介绍。他们已经存在了一段时间，我们已经展示了几个来自集体的](http://thegeekgroup.org/)[有趣的项目](http://hackaday.com/2009/11/21/build-a-camera-boom-at-less-than-a-grand/)(查看这个[脉冲电容器尸检](http://thegeekgroup.org/2012/04/11/equipment-autopsy-giant-pulse-capacitor/))。你可能还记得[他们在大本营位于卡拉马祖时遇到的一些税务问题](http://hackaday.com/2010/03/16/tax-exempt-geek-group-hit-with-huge-tax-bill/)，但他们已经解决了这个问题，搬出了小镇。这是他们在密歇根州大急流城的新位置，休息后你可以在视频中看到大约 4:45 开始的巡演。请注意，视频中有一些以“S”字形式出现的轻度粗俗语言(不客气，隔间居民)。

我们希望你能看看视频，知道我们不需要你花一周的时间在后期制作上。通过视频带我们参观，就好像我们身临其境一样。告诉我们你在做什么，我们将永远感激你。
