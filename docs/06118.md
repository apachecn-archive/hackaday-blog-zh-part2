# 维修尼康 D3

> 原文:[https://hackaday.com/2014/05/23/repairing-a-nikon-d3/](https://hackaday.com/2014/05/23/repairing-a-nikon-d3/)

![headless](../Images/1e973456f0e603cfe6bdc3482c040a36.png)

很少有产品像现代 DSLR 一样电子和机械复杂。在传感器、快门、各种液晶显示屏和到处运行的柔性印刷电路板(FPC)之间，这足以让即使是最有条理的 DIY 修理者在恐惧中颤抖。EEVblog 论坛的 TiN 并没有被吓跑，他在易贝买了一台坏了的尼康 D3，希望能修好它。

[D3](http://en.wikipedia.org/wiki/Nikon_D3) 是尼康在 2008 年的顶级专业相机。由于拥有 1200 万像素的全画幅传感器和一系列其他功能，二手模型仍然占据了原来 5000 美元价格的很大一部分。[TiN 的]照相机被描述为被掉落，并且在到达时已经死亡，正如在易贝上所描述的那样。电池门被破坏了，所以[TiN]连接了外部电源。照相机还是坏了，所以是时候开始拍摄了。多亏了互联网，[TiN]才得以找到该相机的维修手册。他决定接下来检查电源板。前面有一个明显的孔的 TO225 封装晶体管是一个很好的起点。

替换了晶体管，照相机就开始工作了。主液晶显示器显示实时传感器视图，它会拍照。然而，一切并不完美，因为两个辅助液晶显示器仍然是死的，D3 的镜子每隔一次拍摄就会被卡住，导致错误显示。

点击休息时间，观看[TiN]故事的其余部分。

废弃的液晶显示器被证明是另一个简单的解决方法。D3 有许多交织的柔性印刷电路遍布全身。跌落一定会使其中一个移位，因为移除并重新连接所有部件会使辅助液晶显示器恢复原状。

为了解决镜子的问题，[TiN]不得不把相机拆成裸露的框架。在这样做的时候，他发现了一个相机所有者最可怕的噩梦——镜头支架周围的镁框出现裂缝。论坛成员建议使用胶水、支撑和其他修复方法。然而，[锡]把他们留了下来。他计划将相机用作高分辨率的延时拍摄设备，因此镜头支架不会看到大量滥用或沉重的镜头。

随着整个快门组件被掏空，[锡]发现一个标有 262 的小弹簧被摔松了。这个弹簧阻止了镜子弹回。一旦弹簧重新定位，[锡]能够重新组装相机。它工作得很好，足以拍下他留下的所有螺丝的照片。相机现在工作得很好，并被用来拍摄惊人的 4K 延时视频，如下图所示。
