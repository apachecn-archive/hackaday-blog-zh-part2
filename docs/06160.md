# 在两台路由器上运行《我的世界》

> 原文：<https://hackaday.com/2014/05/27/running-minecraft-on-two-routers/>

![router](img/bd0d24bb6917310d604e6ce1b7c23b30.png)

在一些奇怪的硬件上运行《我的世界》并不陌生。早些时候，[他在《我的世界》制作了这个 Linux 驱动的显微镜载玻片……东西](http://hackaday.com/2013/02/18/cnlohrs-microscope-slide-linux-avr-minecraft-thing/)用红石杠杆拨动发光二极管。考虑到《我的世界》是否可以在 AVR 上运行，他[决定在路由器](http://www.youtube.com/watch?v=G5eo5d8F5DU)上尝试同样的事情，具体来说是 TP-LINK TL-WR841N。就像运行 Linux 的显微镜载玻片一样，这被证明是一项简单的任务。他有另一个路由器可以运行《我的世界》，这个也可以打木头。他真的只有一件事可做。

像显微镜载玻片和无线路由器一样，[CNLohr]的 CNC 路由器现在也运行《我的世界》服务器。“因为它在那里”这句话浮现在脑海里。当连接到 CNC 服务器时，玩家用胡萝卜控制一个雪傀儡(一个有杰克灯笼头的雪人)。无论雪傀儡走到哪里，工具头都会跟随，让他可以在世界上雕刻物体，并在数控机器中固定的一张 MDF 上雕刻。

这当然是一个奇怪的构建，但[CNLohr]能够用雪傀儡控制的数控机器雕刻出像素般的块状 Hackaday 标志。代码[此处](https://cnlohr.net/pubsvn/projects/bigmill/)，视频如下。
