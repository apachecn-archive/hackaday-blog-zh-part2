# 眩目的墨镜遮住了佩戴者的四只眼睛

> 原文：<https://hackaday.com/2014/05/28/blinding-shades-hide-wearers-four-eyes/>

我们在创造者大会上碰到了加勒特·梅斯。他没有展出，但为了保持乐趣，他做了一些东西来炫耀。这对 RGB LED 灯罩是在的前一天晚上组装的。他们可能是一起被黑的，但他们绝不是黑客。我们特别感兴趣的是铰链设计，它由 PCB 基板和几个机器螺钉制成。

我们上面的视频很好地展示了他编码的闪烁模式。令我们惊讶的是，佩戴者几乎看不到眼镜发出的光。插槽也不难看到，它们很好地隐藏了(加勒特的)处方眼镜。这双鞋是从几年前我们看到的单色版升级而来的。那套也展出了，但你真的需要仔细看看新的设计。幸运的是，我们花了很长时间编辑这个视频，所以[mace tech 博客现在有了完整的细节](http://www.macetech.com/blog/)。
