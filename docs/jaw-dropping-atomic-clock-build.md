# 令人瞠目结舌原子钟建筑

> 原文：<https://hackaday.com/2014/11/27/jaw-dropping-atomic-clock-build/>

你可以在互联网集市上寻找一个会说话的时钟，但你永远也找不到比这更棒的了。只要看看它…即使它不工作，这将是可怕的。

[Art]果然名不虚传。他的[铷标准原子实时时钟](http://forums.hackaday.com/viewtopic.php?f=3&t=5087&p=20346)无疑是硬件艺术的一个例子。基板是点对点焊接的 perfboard 模块的集合。每一个都精心设计。这样的布局需要什么？一个华丽的外壳，不会遮挡您对组件的视线。为此，他用了一个铜管框架和一个定制的铝制底盘锅。

对于电路本身，[艺术]告诉我们，他想建立一个类似于旧的惠普谢妮频率计数器，所以他用逻辑芯片。这些图片和一些视频注释是我们了解这一过程的唯一线索。希望你在评论中的鼓励能帮助他分享更多。

我们之前提到的会说话的时钟部分呢？PIC 使用[Roman Black 的] [BTc 声音压缩算法](http://www.romanblack.com/BTc_alg.htm)回放音频，每分钟都能读出时间。
