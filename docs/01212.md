# 用激光笔和扬声器显示推文

> 原文:[https://hack aday . com/2012/08/01/用激光指示器和扬声器显示推文/](https://hackaday.com/2012/08/01/displaying-tweets-with-a-laser-pointer-and-speakers/)

[![](../Images/8b36bb41b83935deffd77e9893638f91.png "hack")T2】](http://hackaday.com/wp-content/uploads/2012/08/hack1.jpeg)

今年在 Toorcamp 上，[Rich]将展示他的[基于激光的矢量显示器](http://nothinglabs.blogspot.com/2012/07/lasertweet-twitter-projecting-laser-show.html)，能够仅使用一个激光笔、一对镜子、扬声器和一个 Arduino 来投影推文。稳定的手和诅咒缺乏一个光具座不包括在内。

[Rich]的[指令](http://www.instructables.com/id/LaserTweet-Twitter-Projecting-Laser-Show/?ALLSTEPS#step8)检查了构建的细节；一个 Python 脚本在他的电脑上运行，获取所有带有特定标签的最新推文。这些推文被发送到一个“duino ”,在那里一点代码将文本翻译成滚动矢量显示。该项目的代码基于[Rich]以前的[的一个版本](http://www.instructables.com/id/Arduino-Laser-Show-with-Full-XY-Control/?ALLSTEPS)，用相同的扬声器/激光设置绘制形状。

理论上，使用一对扬声器在墙上画文本与在示波器上画画没什么不同。当然，当 Toorcamp 结束时，[Rich]总是可以选择将他的激光推特 *[变成](http://hackaday.com/2012/03/26/dinos-one-year-extravaganza-is-a-laser-oscillograph/)* [示波器](http://hackaday.com/2012/03/26/dinos-one-year-extravaganza-is-a-laser-oscillograph/)。
