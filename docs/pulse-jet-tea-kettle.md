# 脉冲喷射茶壶

> 原文：<https://hackaday.com/2012/09/21/pulse-jet-tea-kettle/>

[柯林·福尔泽]只是在这张照片中炫耀。他的脉冲喷射茶壶造得很好，不需要喷灯的帮助也能通过，但谁能责怪他的表演技巧呢？事实上，一旦它开始运转，就看不到火焰了。这是因为燃烧发生在管道的早期阶段，加热浸没在水中的一段，这样你就可以马上喝茶了。

一旦这东西调好了，它就会像机器狮子一样咆哮。[科林]对着镜头喊他的评论，但它被当成一个扭曲的信号。加压的丙烷和空气都被送入喷射器。它们由装置底部的两个旋钮调节(那个外壳实际上只是一个馅饼罐)。还有一个 9V 电池供电的点火器内置在底座上。休息后，您可以在视频中看到该单元是如何构建的。
