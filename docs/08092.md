# 大笑运动夹克实际上是一件运动夹克

> 原文:[https://hack aday . com/2015/01/18/laugh-track-jacket-is-actually-a-blazer/](https://hackaday.com/2015/01/18/laugh-track-jacket-is-actually-a-blazer/)

想象一下:你在 Larry's Laugh Lounge 的第一个开放式麦克风之夜。你排在第二位。这比第一个去好，但是人群远没有达到两杯的最低限度，你爸爸的笑话就像一个铅气球一样。现在怎么办？你的秘密武器的时间到了。当你漫不经心地把手伸进人字纹外套的口袋时，你抛出了“强硬的人群”这一说法。你按下一个按钮，蟋蟀的声音就会传到麦克风。有人在后面咯咯地笑。你微笑着，想起 Reddit 上那个芭比的笑话，你的行为转变为亵渎，你第一次真正大笑的甜美声音将永远留在你的记忆中。

这款来自 Adafruit 的[Becky Stern]的笑料夹克基于他们自己的[音频 FX 板](https://www.adafruit.com/product/2210)，这是一个独立的单元，可以存储和播放 WAV 和 OGG 文件。该板还配有 [16MB 的闪存](https://www.adafruit.com/product/2217)，用于扩展预先录制的 Foley artistry。这是一个简单的焊接和缝纫项目，有很多可穿戴的应用程序，所有的组件都可以在 Adafruit 商店买到。有很多地方可以获得已经是 WAV 格式的[免费音效](https://www.freesound.org/browse/tags/sound-effects/)，因为主板不支持 MP3。一如既往，[Becky]提供了清晰透彻的指南，并配有大量图片和介绍视频，您可以在休息后观看。
