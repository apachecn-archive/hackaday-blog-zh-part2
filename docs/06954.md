# THP 半决赛:FNIR 大脑成像仪

> 原文:[https://hack aday . com/2014/08/31/THP-半决赛选手-fnir-brain-imager/](https://hackaday.com/2014/08/31/thp-semifinalist-fnir-brain-imager/)

当前神经科学和心理学领域的研究工具是功能性磁共振成像。它基本上与任何设备良好的医院中的 MRI 机器相同，但有一个关键的区别:它可以检测血氧水平的非常小的变化，从而检测大脑中的活动区域。为什么这很重要？对于研究人员来说，发现大脑的哪个区域对某些刺激有反应是通往终身教职镇的门票，在 Publicationton 和 Grantville 停留。

fMRI 实验室很昂贵，而[Jeremy]提交 Hackaday 奖的目的是以更低的成本做同样的事情，并以某种方式极大地增加使用这种技术进行的研究数量。他是怎么做到的？使用与高科技静脉探测仪相同的技术:红外线。

[Jeremy]的想法与光电容积描记器非常相似，更广为人知的是脉搏血氧计。在康奈尔大学和德雷克塞尔大学的几篇论文的指导下，[杰里米]正在使用近红外发光二极管，而不是相对常见的发光二极管，这些论文证明这项技术可以用于观察大脑中的血氧浓度。

基于光，这种设备不会深入大脑。对于许多用例来说，这很好:运动皮层就在你的头骨旁边，从一只耳朵延伸到另一只耳朵，视觉在你的后脑勺，记忆就在你的前额。能够用一个可以佩戴的设备非侵入性地扫描这些区域有着令人难以置信的应用，从让截肢者控制假肢到只需想想就能控制视频游戏角色。

[杰里米]的设备很小，大约有一部手机大小，使用一系列发光二极管和光电二极管来组合一个人脑袋里正在发生什么的图像。图像会有些粗糙，分辨率低，不会像 fMRI 那样覆盖整个大脑。它也没有花费数百万美元，这使得它成为我们 Hackaday 奖最具科学颠覆性的参赛作品之一。
