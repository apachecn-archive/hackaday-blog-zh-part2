# 格申菲尔德教授谈晶圆厂实验室和全数字化

> 原文：<https://hackaday.com/2015/06/04/prof-gershenfeld-speaks-on-fab-labs-and-all-things-digital/>

Fab Labs 与我们今天看到的非常熟悉的黑客空间携手发展。如果你想更多地了解它们的过去和未来，fab 实验室的创始人兼麻省理工学院比特和原子中心主任 Gershenfeld 教授]将为我们带来关于这些 Fab 实验室和我们生活的数字世界的全新视角。

在 Edge 上一个小时的非正式聊天中，[格申菲尔德教授]深入探讨了我们世界中的*数字*概念。我们可能会认为*数字*是一个二进制信号，一个离散成 0 和 1 的模拟波形，今天所有的计算机架构都建立在这个基础上。*然而，数字*不仅仅存在于计算的意义上；这是一个已经被应用于通信、计算以及如今的个人制造的概念。

(格申菲尔德教授的演讲)可能会强调未来即将发生的变化，但变化今天已经在发生。如今，fab labs 和 hackerspaces 以一种非常特殊的方式为他们的社区服务。他们把“该领域的专家”从大学和孤立的实验室中带走，并把他们分散到世界各地。随着这种转变，任何人都可以走进他们的大门，通过与这些当地专家交谈，在嵌入式编程和计算机辅助制造等领域建立坚实的基础。简而言之，这两个地方都找到了一种发展专业知识的文化，这种文化比大学里的同行更容易为国际社会所接受。

如果你能抽出时间，戴上耳机，收听并继续你的 CAD 工作、PCB 布局或 Arduino 库。你可能会发现，你的工作是建立在一些数字原则之上的，你的贡献推动了开发链中的其他部分向更好的方向发展。

最后，如果你有兴趣记录建造自己的工厂实验室，看看 CBA 网站上的[库存](https://docs.google.com/spreadsheets/d/1U-jcBWOJEjBT5A0N84IUubtcHKMEMtndQPLCkZCkVsU/pub?single=true&gid=0&output=html)、[布局和指南](http://www.fabfoundation.org/fab-labs/setting-up-a-fab-lab/)。
