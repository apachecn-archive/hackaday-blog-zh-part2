# 机器人变色龙教小女孩伪装

> 原文:[https://hack aday . com/2014/02/07/机器人-变色龙-教导-小女孩-关于-伪装/](https://hackaday.com/2014/02/07/robot-chameleon-teaches-little-girl-about-camouflage/)

![IMG_8407](../Images/b67786d401ddfb0a7c441d48eb1f5edd.png)

[Markus]一直在用一本古老的动物百科全书教他的女儿关于动物的知识。几天前，他们偶然发现了变色龙，当他试图解释它的伪装能力时，她不太明白。于是他决定[给她做一个宠物变色变色龙机器人](http://letsmakerobots.com/node/40254)。最棒的是他在她睡觉的时候做的！

这是一个相当简单的电路，由一个 Arduino Uno、一个带分线板的 TCS3200 颜色传感器、一个乒乓球、一些电阻和一个 RGB LED 组成。他计划以后增加温度感应和电容式触摸传感器。到目前为止，他的女儿很喜欢它，一直和它玩。她开始学习一些变色龙如何改变皮肤颜色来伪装自己——她也在学习一些新颜色的名字！

和往常一样，休息之后会有一段演示视频。
