# 奥斯曼帝国

> 原文:[https://hackaday.com/2014/02/11/the-ottoman-empire/](https://hackaday.com/2014/02/11/the-ottoman-empire/)

![ottomon empire](../Images/1fe3e55fb9117b78aeae0c7be954a064.png)

还记得 60 年代和 70 年代的美好时光吗？那时立体声系统被直接安装在家具中。没有吗？嗯，这就是这个项目的灵感来源。介绍奥斯曼帝国(T1)，这个双关语太糟糕了，我们甚至不打算在这里重复。

[Alec]的灵感来自于 [Blaupunkt](http://en.wikipedia.org/wiki/Blaupunkt) ，这是一家德国电子产品制造商，曾经生产一系列非常好的橱柜音响([例子](https://www.google.com/search?site=&tbm=isch&source=hp&biw=1920&bih=916&q=blaupunkt+cabinet+stereo&oq=blaupunkt+cabinet+stereo&gs_l=img.3...329.2879.0.2979.24.9.0.13.3.0.153.914.5j4.9.0....0...1ac.1.34.img..14.10.753.RkM8oLdPC7A))，将家具和电子产品非常精致地融合在一起。

他最近完成了一个相当酷的 8 声道数据备份系统，剩下一个备用的 T2 BSR 唱机，或者他喜欢称之为“唱机的福特品托”他决定把它变成有用的东西，将它整合到一个从当地古董店买来的瑙格海德搁脚凳上。这种旧家具的问题是什么？没有结构元素——它实际上只是装满了碎木头！他把它全部清理干净，然后开始制作自己的木制框架来支撑 BSR——他做了很好的修改，甚至隐藏了所有的电子设备，让它看起来非常像样。

现在他需要做的就是在顶部添加一个压力开关，这样当他放松时，它就会开始播放肖邦的音乐。
