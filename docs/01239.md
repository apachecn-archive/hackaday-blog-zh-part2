# 电动势示波器探头

> 原文:[https://hackaday.com/2012/08/03/emf-oscilloscope-probe/](https://hackaday.com/2012/08/03/emf-oscilloscope-probe/)

![](../Images/0197cf8b4904087765e3e01be5b8afe0.png "OLYMPUS DIGITAL CAMERA")

[Tuomas Nylund]想要一种方法来可视化他周围的电磁场(EMF)。他认为示波器是最适合这项任务的工具，但他需要一种方法来拾取电场并将它们馈入示波器的一个探头。他最终[制造了这个 EFM 探针加密狗](http://tuomasnylund.fi/drupal6/content/simple-emf-probe)来完成这项任务。

他承认这只不过是一个连接到探针的感应器，不应该用于严肃的测量。但我们认为他低估了自己。这可能不是他所认为的精度，但他在设备中集成的放大电路和滤波组件似乎可以提供非常可靠的输入信号。我们也很欣赏使用 BNC 连接器的简单接口。休息后，观看演示视频，了解焊台控制器、扫描液晶屏和开关模式电源产生的电动势。
