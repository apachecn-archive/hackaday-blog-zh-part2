# 用于 3D 打印零件的超声波雾化蒸汽抛光机

> 原文：<https://hackaday.com/2015/05/16/ultrasonic-misting-vapor-polisher-for-3d-printed-parts/>

如果你曾经见过挤压式打印机的 3D 打印零件，你首先会注意到的是它的纹理。这是由打印机一层又一层地放下塑料造成的。这种表面纹理并不总是令人满意的，所以人们已经找到了一些方法来平滑 3D 打印的零件。例如，如果您使用 ABS，您可以用丙酮冲洗或“涂抹”零件。另一种平滑的方法是在容器中加热一些丙酮，让丙酮蒸汽做功来平滑完成的零件。

来自 engineerdog.com 的[Mike]认为他可能找到了一个更优雅的解决方案，使用一个便宜的超声波加湿器，你可以在网上花 40 美元买到。这种室内加湿器使用压电传感器，能够以高频率振动液体以产生薄雾。[Mike]从加湿器中取出传感器和电子设备，并将其安装到油漆罐中。这是丙酮储存的地方，通过传感器转化为蒸汽。水族箱泵用于将高浓度的蒸汽转移到抛光室(一个特大的泡菜坛子。)他增加了一个弹簧加载的电动定时器(你可以在办公楼的浴室里找到的那种)，让他的蒸汽抛光机像微波炉一样容易使用。

[Mike]以丙酮处理前后的零件强度测试结束了他的帖子，并惊讶地发现零件在处理后变得更弱了。你可以在他的博客上阅读更多关于这个的内容，休息后还可以看到蒸汽抛光机的视频。
