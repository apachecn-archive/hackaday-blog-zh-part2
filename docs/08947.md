# 打印小狗假肢

> 原文:[https://hackaday.com/2015/05/04/printing-puppy-prosthetics/](https://hackaday.com/2015/05/04/printing-puppy-prosthetics/)

当[aimzzz]遇到这只天生没有手臂的小狗时，[显然需要一些辅助硬件](http://www.instructables.com/id/Adaptable-Wheelchair-for-handicapped-Dogs)。我们喜欢快速原型技术变得如此容易，以至于像为小狗造轮椅这样的事情不仅负担得起，而且很有趣！

该项目的主要部分是一个摇篮，这将是舒适的狗。3D 打印在这里是一个很好的选择，因为它可以定制，以适应特定狗的需求。我们记得见过另一只名叫 [Derby 的狗，它有 3D 打印的腿](http://hackaday.com/2014/12/17/derbys-got-legs-he-knows-how-to-use-them/)，为功能不正常的生物腿腾出空间。在这种轮椅的情况下，摇篮可以改变，为腿腾出空间。

建筑的其余部分完全是机械的。铝管、管连接器和车轮与印刷摇篮(和一些填充材料)相结合，使一个甜蜜的乘坐。这需要一点训练来适应，但正如你在休息后看到的，这对小狗来说非常容易和直观。
