# 自我平衡器的工作方式与我们习惯看到的不同

> 原文:[https://hack aday . com/2012/06/29/self-balancer-do-it-different-than-used-to-seeing/](https://hackaday.com/2012/06/29/self-balancer-does-it-differently-than-were-used-to-seeing/)

![](../Images/373d9be67ccf3cf62374db69eb1c4b7c.png "self-balancer-does-it-differently")

这个自平衡机器人仍然只使用两个轮子，但是它的平衡与我们过去看到的非常不同。大多数项目使用[一种类似赛格威](http://hackaday.com/2012/06/25/printing-and-programming-a-self-balancer/)的外形，这就像自行车一样工作。但是它不需要保持前后轮旋转来保持直立。事实上，休息后的视频显示，它在完全静止的情况下保持了完美的平衡。[青木 2001]的创作不是停留在一个地方。他在前面和后面安装了距离传感器，用来像推斥力一样移动自行车。

驾乘者所在的大轮子确保车辆不会翻倒。它就像一个倒立摆，通过旋转它所连接的马达来克服大轮子的惯性。同样的概念在三月份的一辆全尺寸自行车上出现过。但是当你只需要一个轮子的时候，为什么要用两个轮子呢？休息后还可以看到他的独轮车版本嵌入。

也值得看看(青木的)YouTube 上的其他内容。他有一个在球上保持平衡的小机器人。这是这种黑客技术的桌面版。
