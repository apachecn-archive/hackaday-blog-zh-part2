# 在麦克唐纳 F-101 的巫术中，不要回避俯仰

> 原文：<https://hackaday.com/2015/06/23/retrotechtacular-dont-balk-at-pitch-up-in-the-mcdonnell-f-101-voodoo/>

麦克唐纳飞机公司的 F-101“巫毒”是一种短小精悍的超音速飞机，能在两分钟内从停机坪飞到 40000 英尺的高空。但是对于所有的创新和工程，伏都教有一个共同的问题，音高。也就是说，伏都教的后掠翼造成了飞机非常急剧地向上翘起的趋势，使飞行员失去了控制。

麦克唐奈向巫毒飞行员保证，这个问题很容易克服冷静的头脑和坚实的基础知识的问题。本培训影片旨在提供这一基础，探索上仰的原因以及在阻力伞展开和不展开的情况下的规定改出方法。

拖曳滑道始终是帮助校正飞行器的推荐路线。对于全尺寸上仰情况尤其如此。然而，没有牵引槽也可以回收。改出时的高度损失与上仰时的高度成正比。也就是说，当发生上仰时，飞行器的高度越低，在回到直线和水平飞行时损失的高度就越少。
