# 用麻省理工学院的开放式课程制作一个 360 美元的合成孔径雷达

> 原文：<https://hackaday.com/2012/12/18/build-a-360-synthetic-aperture-radar-with-mits-opencourseware/>

来自麻省理工学院林肯实验室的几位教授在寒假期间给那些可怜的麻省理工本科生提供了一些事情做:他们教授一门为期三周的课程，内容是构建一个笔记本电脑供电的雷达系统，能够进行雷达测距、多普勒和合成孔径成像。有趣的是，团队将为班级建造的雷达系统的 BOM 总计为 360 美元，如果你想继续建造自己的系统，他们还会将整个班级放到网上。

根据课程的[讲义，无线电系统由现成的 LNA、振荡器和分路器组成。通过连接两个咖啡罐“cantennas”，可以从来自雷达的信号中录制一个. WAV 文件，并使用 MATLAB 将该音频信号转换为多普勒雷达。](http://ocw.mit.edu/resources/res-ll-003-build-a-small-radar-system-capable-of-sensing-range-doppler-and-synthetic-aperture-radar-imaging-january-iap-2011/projects/MITRES_LL_003IAP11_proj_in.pdf)

这是一个非常雄心勃勃的项目，深入 RF 和模拟设计的兔子洞。其中一位讲师在 YouTube 上演示了雷达的测距模式；休息之后你可以去看看。
