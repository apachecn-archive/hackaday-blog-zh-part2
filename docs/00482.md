# 轻型三轮车允许你为你的生活踩踏板

> 原文:[https://hack aday . com/2012/05/21/light-trikes-allow-you-to-pedal-for-you-life/](https://hackaday.com/2012/05/21/light-trikes-allow-you-to-pedal-for-your-life/)

![](../Images/d11046fc0be9f385d6bdd6a9ec4c83f7.png "tron trike")

不幸的是，今年我们的作家都不在加州。这意味着我们不能去海湾地区的 MakerFaire 亲眼看看那些很酷的东西。我们一直在关注网络，一些项目引起了我们的注意。你在上面看到的装备是一款游戏的[物理控制器，灵感来自经典的创光周期](http://blog.makezine.com/2012/05/17/see-light-trikes-and-crowd-connect-at-maker-faire-bay-area/)。游戏的玩法很明显，你踩着三轮车去转动车把来转动你的轻便三轮车。硬件似乎相当简单，他们使用 arduino 从自行车上收集信息，然后通过 NetLab hub 发送信息，这是一个跨平台的工具包，用于从微控制器获取数据并将其馈送到 flash。我们认为他们在演示上做得非常好，这看起来很有趣。

出于某种原因，我们想让他们建一个看起来像斯坦利酒店的关卡。
