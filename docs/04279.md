RGB LED 灯条，LED 数字钟，七段 LED 显示屏，有什么是一个 Arduino 做不到的吗？

外面有一百万个 Arduino 克隆体，他们都做着几乎相同的事情。你知道什么最棒吗？一百万个。这是 Hackaday 自己的 Arduino 克隆产品 HaDuino 背后的设计理念。

也是开瓶器。说真的。创意推介以“你知道什么是真正酷的想法吗？”

我们给自己这个:它像 Arduino 克隆一样工作。开瓶器部分是另一个故事。我们的原型板在打开一瓶后就坏了。不过，这只是一个原型，如果你想尝试制作一个真正的 Arduino 兼容开瓶器，所有的设计文件都在 GitHub 上。

还有一件事:我们不卖这个。你可以把它理解为两件事之一。它是如此的独特和酷，获得它的唯一方法是制作一个主板，或者我们只是不想因为支持问题在 Arduino 社区引起轰动。你知道，因为这是一个 Arduino 开瓶器。

它是开源的，开放硬件，随你喜欢。
