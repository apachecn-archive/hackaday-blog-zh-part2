# 摇动手机启动滑板车

> 原文:[https://hack aday . com/2012/06/18/shake-phone-to-start-scooter/](https://hackaday.com/2012/06/18/shake-phone-to-start-scooter/)

![](../Images/7e0ca468a6ac46e1449acbc1bea98465.png "bluetooth-scooter-ignition")

摇一摇你的安卓设备，这个滑板车[就启动了。这种摇动必须从前到后进行，因为左右摇动是为了解锁鞍座([布拉德]将他的头盔存放在其中)。](http://bradsprojects.wordpress.com/2012/06/14/bluition/)

连接通过蓝牙实现，左把手附近有一个摇臂开关，可以禁用接收器，这样就不会耗尽电池。你可以看到滑板车前部的锁板敞开着。他在车内安装了驱动板，驱动板嵌入点火系统，并驱动座椅插销的螺线管。听起来像是闩锁装置使用了鲍登线缆，其手柄在锁定面板内。通过增加一个螺线管和大量润滑电缆，他设法让它从驱动板工作。

休息之后，请观看视频，观看适当的演示。手机通过 SL4A 运行 Python 脚本，负责用户界面。
