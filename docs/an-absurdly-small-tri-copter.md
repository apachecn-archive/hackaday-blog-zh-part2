# 一架小得离谱的三人直升机

> 原文：<https://hackaday.com/2013/08/24/an-absurdly-small-tri-copter/>

Femtoduino 背后的团队 Arduino 的一个非常小的重新包装——从 YouTuber [phineasIV]，又名[Eric]发送了几个视频，显示了我们见过的最小的多直升机之一。

因为这不是传统的 quad 或 hexcopter，控制系统有点奇怪。其中两个马达和支柱沿着垂直轴固定，而后支柱连接到一个小伺服系统，从一边旋转到另一边。尽管如此，电子设备对于任何多旋翼飞行器来说都是相当标准的——一个三轴陀螺仪通过 [MultiWii](http://www.multiwii.com/) 提供了飞行器的稳定性，而一个令人惊讶的小[伺服接收器](http://www.deltang.co.uk/rx31b.htm)、蓝牙模块、Femtoduino 和三个无刷 ESC 将一切联系在一起。

最终结果是一架三轴飞行器，重量与 Crazyflie Nano Quadcopter 差不多，但略小一点。尽管它在视频中令人印象深刻(见下文)，但我们很想亲眼看看这只微小的机器人蜂鸟。
