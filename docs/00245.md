# 科学展机器人 Ca。1984

> 原文:[https://hack aday . com/2012/04/28/retro tech tacular-science-fair-robot-ca-1984/](https://hackaday.com/2012/04/28/retrotechtacular-science-fair-robot-ca-1984/)

![](../Images/c7cb8fcf70b3fdb7edb191f2ce588677.png "7th-grade-robot-project")

[迈克尔·苏兰]刚刚看到了他七年级科学竞赛项目的视频。他真的很喜欢建造和展示机器人，我们很高兴这些镜头没有被载入史册。休息之后看一下片段。这不是一个机器人撒尿，这是“消防部队”，泵出液体扑灭非常近距离的火灾。前面有一个太阳能风扇，如果你知道如何激活它，爪子就会旋转。肩膀和底座也是电动的，看起来像早期计算器的键盘在机器人的背部提供了一些额外的控制。

我们不能看这个而不笑，这激励我们在 Hackaday 上推出一个重复的功能。我们想让你把你的旧录像找出来数字化。任何与黑客相关的东西都是公平的游戏…不要害羞！也不一定非得是你。例如，我们喜欢[格蕾丝·赫柏的]纳秒讲座的视频。如果你有一些过去的片段[，给我们发一个链接](http://hackaday.com/contact-hack-a-day/)！
