# 脚踝手镯让你成为帕尔帕庭皇帝

> 原文:[https://hack aday . com/2012/06/27/ankle-手镯-让你成为皇帝-帕尔帕廷/](https://hackaday.com/2012/06/27/ankle-bracelet-lets-you-become-emperor-palpatine/)

![](../Images/d9ea4d86cc9588198b9fdf7447bfb72f.png "shooting-sparks-from-your-fingers")

想从你手中射出闪电来惩罚你的敌人吗？你可以(几乎)通过[这个静电发生器黑客](http://www.youtube.com/watch?v=EvspL4kMY54)做到这一点。上面你可以看到电荷从这个家伙的指关节上跳下来，通过发光二极管汹涌而来。但这并不是当脚踝上只戴一点点硬件时，你能耍的唯一花招。休息后的视频显示，当带电的手在沙粒上挥动时，沙粒四处跳跃。

诀窍是用 9V 电池给负离子发生器供电。这不能直接完成，因为离子发生器正在寻找交流电源。但是转换就像废弃一个被设计用来插入汽车点烟器的逆变器一样简单。所有东西都被塞进一个眼镜盒，然后可以绑在你的脚踝上。为什么对脚踝部位如此着迷？答案的一部分是，这提供了一种将离子发生器输出与您的皮肤进行交互的简单方法。答案的另一部分是，你需要确保系统接地(但你没有)，并且构建包括围绕你的鞋的环来实现这一点。

休息之后，请查看视频中的演示和完整构建说明。
