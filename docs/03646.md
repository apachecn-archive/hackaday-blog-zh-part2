# Cyclone PCB 工厂:3d 可打印电路板工厂

> 原文:[https://hack aday . com/2013/07/24/cyclone-PCB-factory-3d-printable-circuit-board-mill/](https://hackaday.com/2013/07/24/cyclone-pcb-factory-3d-printable-circuit-board-mill/)

![printable-cyclone-pcb-factory](../Images/01404f3266ec85fa96bac8b6f7492d21.png)

如果你能为另一台 3D 打印机 3D 打印大部分零件，为什么不能为一家 PCB 工厂 3D 打印呢？那是[旋风 PCB 厂](https://github.com/carlosgs/Cyclone-PCB-Factory)回答的问题。它将帮助你吻别那些碳粉转移或光阻天。

自制电路板往往相当小，这真的有助于保持该项目的成本和范围下降。大多数安装部件和齿轮都是 3D 打印的。当然，还有通常的机床项目，你几乎必须购买:滚珠丝杠，精密杆，步进电机，电机旋转路由工具。

看看下面的视频，看看这个项目现在在哪里。PCB 铣削的一个重要方面是要有一个水平的构建表。刀头往往呈“V”形，所以切得稍微深一点就会把你想隔离的痕迹吹掉。演示表明，这可以自动校准软件，以解决覆铜层高度的任何变化。

我们记得看到过[一个快速组装的 PCB 工厂](http://hackaday.com/2011/06/02/snap-together-pcb-mill/)。但我们非常确定，那台机器使用的是由高密度聚乙烯铣削而成的零件，而不是 3D 打印的部件。
