# 一个 Gamecube 和 N64 主机混搭

> 原文:[https://hack aday . com/2013/01/02/a-game cube-and-n64-console-mashup/](https://hackaday.com/2013/01/02/a-gamecube-and-n64-console-mashup/)

![gc](../Images/59ae60543affb1e0dcc24b6a5a37815b.png)

[Hailrazer]在细菌制造论坛上，他有点厌倦了他所有的控制台把他的电视下面的空间弄得乱七八糟。不过不用担心，因为将 Gamecube 和 N64 结合成一个看起来非常专业的系统实际上是相当容易的。

虽然[Hailrazer]的 Gamecube 相当完整，不包括添加 mod 芯片和 SD 卡来保存 Gamecube 磁盘映像，但版本的 N64 部分需要相当多的硬件入侵。在为 Gamecube 找到一个 Game Boy Advance 玩家后——这是一个整洁的硬件附件，允许你在 Game Cube 上玩 GBA 游戏——[ hail Razer]认为他找到了 N64 case mod 的完美外壳。

GBA 球员的内脏被扔掉了，N64 的内脏被小心翼翼地锉平，以适合他们的新家。一个 [Everdrive 64](http://krikzz.com/index.php?route=product/product&product_id=54) 在 SD 卡上保存了几乎每一个美国 N64 版本，使得访问盒式磁带端口变得不必要。

Gamecube 侧面的开关在 Gamecube 和 N64 之间切换视频和音频输出。这是一个整洁的小设置，将两个控制台打包到小小的 Gamecube 空间中。
