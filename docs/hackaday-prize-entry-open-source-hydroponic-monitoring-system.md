# Hackaday 奖参赛作品:开源水培监控系统

> 原文：<https://hackaday.com/2015/05/17/hackaday-prize-entry-open-source-hydroponic-monitoring-system/>

几个月前，[亚当]正在为他在地下室建立的一个小型水培系统建造一个控制器系统。从那以后，Hackaday 奖公布了，并给出了主题——一次拯救世界一株植物——他正在将他的花园控制和监控系统重新设计为 Hackaday 奖参赛作品。

虽然建筑的机械和绿色部分正是你从五金店零件设计的东西中所期待的，但电子部分相当有趣。无论是水培还是无土栽培，所有植物的湿度和 PH 值都由一组电子设备监控，这些电子设备将数据上传到云端。

当前的硬件设置包括一个 [DyIO](http://www.neuronrobotics.com/store/dyio) ，一个非常酷的开发平台，具有 24 个数字 I/o 和 24 个伺服输出，一个 Raspberry Pi，以及一些加载了 ARM 微控制器和 ESP8266 的模块板。[Adam]正在检查这个建筑的所有硬件。

到目前为止，[Adam]已经向电路板制造厂发送了一些电路板，包括模拟传感器模块和数字传感器模块。一个 WiFi 模块集线器和一些零碎的东西，使集成到现有的花园或水培设置更容易。对于今年的 Hackaday 奖来说，这是一个很好的项目，证明了你不需要想出一个新的构建来提交东西。
