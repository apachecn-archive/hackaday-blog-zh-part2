# 一个开源的 DIY 数字化仪

> 原文:[https://hack aday . com/2015/05/19/an-open-source-DIY-digitizer/](https://hackaday.com/2015/05/19/an-open-source-diy-digitizer/)

当你看目前可用的扫描 2D 和 3D 物体的方法时，你基本上是在看一个成像过程。要么你拍一张 2D 物体的照片，要么你用 3D 扫描仪抓取一团点云，然后用这种方式制作一个 3D 物体。事情并不总是这样——真正的硬件 3D 数字化仪早在 70 年代就开始使用，接触式测头是高端数控机床的标准设备。

【Nikolaj Mobius】需要一种方法来记录物理空间中的点，并且不想处理图像的问题，[他做了一个开源的 DIY 数字化仪](http://fablab.ruc.dk/diy-digitizer/)。它基本上是一个激光切割臂，每个关节都有旋转编码器。通过使用 Arduino 读取旋转编码器，[Nikolaj]可以数字化工件上的几个点——仅够制作一个支架，或找到零件的关键尺寸。

当你需要的信息比一套卡尺能提供的多一点时，这是一个很好的工具，也是一些古老技术再次变得有用的一个很好的例子。
