# Hackaday 奖参赛作品:从零开始，用泡沫制造汽车

> 原文：<https://hackaday.com/2015/04/30/hackaday-prize-entry-building-a-car-from-scratch-out-of-foam/>

想要一个令人印象深刻的例子，展示几个人在车库里能做些什么？从零开始，从一大块泡沫开始，制造一辆电动汽车怎么样？

[MW Motors]的 Luka EV 有几个项目目标:自然，它应该是全电动的，最高时速为 130 公里或 80 英里。应该有 300km 以上的射程，而且要好看。最后一项很棘手；制造一辆电动汽车并不太难，但要让一辆看起来不错却是一个挑战。

汽车的设计实际上是从数字文件开始的。获得一大块泡沫，并小心地雕刻成想要的形状。这种泡沫被玻璃纤维覆盖，零件从玻璃纤维模具中取出。这是一种小批量生产的好方法——一旦模具完成，为第二辆 Luka EV 制造另一个车身就相对简单了。

安装好所有的灯、配件、窗户和装饰后，[是时候把车身放到底盘上了](https://hackaday.io/project/5066-luka-ev/log/16007-building-the-component-testing-chassis)。这是焊接出来的方管，并作为一个试验台，可以独立于玻璃纤维的混乱。在底盘上有电池、悬架、电机控制器和装有轮毂电机的车轮。它运转良好，[即使只有一个马达](https://www.youtube.com/watch?v=dvDoH3b6cKc)。

这个项目还有更多的内容，包括[一个关于在英国建造公路合法汽车的伟大指南](https://hackaday.io/project/5066-luka-ev/log/16658-iva-individual-vehicle-approval)。车队不在英国，但对“小系列”车辆来说，这里的环境要友好得多。这些要求很容易满足——比如“有一个喇叭”——但它们有很多。

[这辆车已经很漂亮了](https://hackaday.io/project/5066-luka-ev/log/17062-first-paint)，这还只是它在拖车上的样子。我们迫不及待地想看到这东西上路。
