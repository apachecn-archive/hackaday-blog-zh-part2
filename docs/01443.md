# 像指挥交通一样移动你的手臂来控制你的房子

> 原文:[https://hackaday . com/2012/08/27/像指挥交通一样动动胳膊控制你的房子/](https://hackaday.com/2012/08/27/control-your-house-by-moving-your-arms-like-youre-directing-traffic/)

![](../Images/8e9437d9ecabac40998544254a4c26d8.png "laser-gesture-light-switch")

这个家庭自动化项目可以让你挥动手臂来开关东西。[图恩]和[江]一直在研究这个概念，作为他们大学硕士论文的一部分。它使用一个 3D 摄像头和一些定制软件来采集你的手势。我们真正喜欢的是提供反馈的激光笔。你可以在墙上看到一个红点，他指向哪里，红点就跟到哪里。每个可控制的设备都有一个特殊的区域，当用户指向它时，圆点会吸附到这个区域。通过抬起他的另一只手臂，可以打开或关闭选定的对象。

休息之后，请观看两个视频，以全面了解这一概念。我们希望看到使用某种类型的激光投影仪，而不仅仅是一个点。这样你就可以拥有一个弹出菜单系统。想象一下，墙上挂着一个虚拟遥控器，可以用来跳到下一个音轨、调节音量或更换电视频道。
