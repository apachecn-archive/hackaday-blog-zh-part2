# 虚拟现实通过 3 个 Kinect 摄像头变得真实

> 原文：<https://hackaday.com/2014/05/14/virtual-reality-gets-real-with-3-kinect-cameras/>

不，这不是恐怖电影中的场景，这是 3D 办公环境中的[奥利弗·克雷洛斯]头像。如果他看起来有点奇怪，那是因为他戴着 Oculus Rift，他的图像是由 3 个微软 Kinect 摄像头拼接而成的。

[奥利弗]创造了一个难以置信的逼真的 3D 环境，至少对佩戴者来说是如此。他认为秘密在于整个系统的低延迟。当加上一个良好的 3D 环境，就像上面显示的办公室，大脑会被骗去相信它真的在房间里。[Oliver]提到他发现自己下意识地移动以避免撞到他知道不存在的桌腿。用[奥利弗]的话说，“它绕过了[恐怖谷](http://en.wikipedia.org/wiki/Uncanny_valley)”。

[Oliver]不是从 3 个 Kinect 摄像头中提取骨架数据，而是使用视频和深度数据。他正在一台 i7 Linux 机器上拼接和处理这些数据，这台机器配有 Nvidia Geforce GTX 770 显卡。强大的硬件肯定，但不是尖端的怪物钻机可能会预期。[Oliver]还记录了他的软件堆栈。他正在使用 [Vrui VR 工具包](http://idav.ucdavis.edu/~okreylos/ResDev/Vrui/)、 [Kinect 3D 视频捕捉项目](http://idav.ucdavis.edu/~okreylos/ResDev/Kinect/)和[协作基础设施。](http://idav.ucdavis.edu/~okreylos/ResDev/Collaboration/)

我们迫不及待地想看看[奥利弗]拿到 Kinect One 后会做些什么(以及一些好的 Linux 驱动程序)。
