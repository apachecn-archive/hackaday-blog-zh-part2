# 立体显示看起来像达芬奇素描

> 原文：<https://hackaday.com/2012/06/14/volumetric-display-looks-like-a-davinci-sketch/>

[这个立体显示器](http://thecriticaltechnologist.wordpress.com/audiovisual/)确实显示了很大的潜力。它恰到好处地平衡了简单性和独创性。这个装置是由[米歇尔·大卫]开发的，包括一个螺旋状的屏幕和一个通过光学过滤盘发光的激光。

该设备的移动部件安装在同一根轴上，该轴由连接到 Dremel 工具的皮带系统旋转。由于屏幕和信息盘安装在同一根轴上，同步过程的一部分已经完成。你可以看到投影仪的独立支架，在这种情况下是一个激光器，但也可以与其他光源一起工作。由于投影表面朝向和远离投影仪移动，由于聚焦问题，激光是优选的。不幸的是[Michel]在以足够高的频率切换二极管时遇到了一些问题。尽管如此，我们还是认为广告之后的结果非常棒。

就硬件而言，这比[试图旋转一个大的 LED 矩阵](http://hackaday.com/2010/08/09/spin-peggy-get-3d-pov/)要简单得多。
