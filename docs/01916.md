# 苹果手机黑客不是 Adafruit 卖的

> 原文:[https://hack aday . com/2012/10/19/iphone-hack-not-sell-by-adafruit/](https://hackaday.com/2012/10/19/iphone-hack-isnt-sold-by-adafruit/)

[![light-up-iphone](../Images/0a8bfc8cba8657300edaab984622ce5f.png "light-up-iphone")T2】](http://hackaday.com/?attachment_id=88386)

[Becky]来信告诉我们她最近为 Adafruit Industries 所做的工作。这种黑客技术可以让你点亮手机背面的苹果标志，正如她所说，这肯定会让你的保修失效。从视频的外观来看，它需要一些非常好的焊接技能，以及可选的中级胶带技能。如果你准备好迎接挑战，并且有胆量在手术台上看到你的 iPhone，结果是相当不错的。

进行这种黑客攻击的另一个挑战是，它使用了 Adafruit 不打算出售的套件，因为它使用了山寨苹果零件。它们存在于某个地方，但在哪里寻找它们将是一个挑战。另一方面，从纯实验/工程的角度来看，休息后的视频还是挺有意思的。我们中的许多人都很犹豫要不要拆开一部几百美元的手机，更不要说焊接上电源了！
