# 增强型咖啡冲泡机知道你有多沉迷于咖啡因

> 原文:[https://hack aday . com/2014/11/25/enhanced-coffee-brewer-know-how-do-of-a-咖啡因上瘾者你是谁/](https://hackaday.com/2014/11/25/enhanced-coffee-brewer-knows-how-much-of-a-caffeine-addict-you-are/)

在一些办公室里，谁应该出资最多来补充社区的咖啡供应是一个常见的争论点。[奥利佛·克罗恩设计的这款名为 Juraduino 的 RFID 咖啡冲泡器通过记录每个人的消耗量和频率，彻底解决了他工作场所的问题；相当的外交技巧。](https://www.youtube.com/watch?v=hGvdymS_08c)

[Oliver]将他的旧 Jura 咖啡机捐赠给他的办公室，并在机器的面板下巧妙地隐藏了一些附加硬件。安装在内部的 Arduino 通过迷你 USB 从咖啡机本身的逻辑单元供电。一旦员工在机器前面刷 RFID 卡，一个实时时钟模块就会显示咖啡需求时间，然后将此人选择的数量记录在一个迷你 SD 卡上。存储的数据通过一个额外的蓝牙模块发送到一个定制的应用程序[Oliver],该应用程序是由麻省理工学院的应用程序 Inventor 为他的手机创建的，用于显示信息。这些细节可以在周末以电子邮件的形式发送给办公室里的每个人，明确宣布谁可以重新储备大量的社区用品。
