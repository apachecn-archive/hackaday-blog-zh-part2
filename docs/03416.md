# 吉他秀和讲述

> 原文:[https://hackaday.com/2013/06/14/guitarduino-show-and-tell/](https://hackaday.com/2013/06/14/guitarduino-show-and-tell/)

![guitarduino-show-and-tell](../Images/3ceae1496d8bb4110a22ea2662de1efa.png)

[Igor Stolarsky]在一个叫 3s & Sevens 的乐队演奏。我们会说他是吉他手，但是因为他在玩这把被砍的斧子，我们可能应该叫他乐队的吉他手。向下滚动，听一听他可以用硬件附件做什么的快速演示剪辑，然后查看[他对硬件](http://www.youtube.com/watch?v=ZdB_bCEMHb8)的视频解释。

吉他本身有几个附加的输入。最明显的是一组彩色按钮，它们是骑在 Arduino 板上的盾牌。它通过 USB 电缆连接到他的电脑上，控制他的 MaxMSP 补丁。它们不碍事，就像一个样本线圈，他可以一起玩。但是看看他弹奏的手下面的吉他琴体，你也会看到一些灰色的补丁。这些，连同脖子后面的一个长条，是他在演奏时启动的压力传感器。其结果是一种我们以前从未见过的无缝集成水平。现在，他只需要将原型转移到无线系统中，就万事俱备了。

如果你没有像[伊戈尔]那样的粉碎技能，也许[一个自动和弦装置](http://hackaday.com/2012/04/24/dexterity-enhancing-guitar/)会帮你一把。
