# 用树莓皮做一个更好的捕鼠器

> 原文：<https://hackaday.com/2015/08/31/building-a-better-mousetrap-with-the-raspberry-pi/>

造一个更好的捕鼠器，世界就会找到你，但是一个更智能的捕鼠器呢？[Alain]决定通过制作一个由[Raspberry Pi 供电的联网智能陷阱](https://hackaday.io/project/7463-raspitrap-v10)来找出答案。操作的大脑是一个运行 Arch Linux 的 Pi。与之相连的是一个红外触发器，一个打开滑动门的伺服系统，以及一个可以看到你新朋友的摄像头。

捕捉器的外壳是 CNC 切割的泡沫 PVC 板，易于切割和清洁。一旦红外光束被切断，Pi 就会转动伺服系统，拉动前门上的一个插销。一旦你的新朋友安顿下来，led 灯就会打开，照亮拍摄对象，然后拍照并通过电子邮件发送出去。

有了自动警报，你不必手动检查陷阱，你也不必担心被困动物在里面太久。休息之后，请加入我们，观看展示所有功能的演示视频和真实示例。
