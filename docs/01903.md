# 坦克履带机器人建造目标是平稳行驶

> 原文:[https://hack aday . com/2012/10/17/tank-tread-robot-build-aims-for-a-smooth-ride/](https://hackaday.com/2012/10/17/tank-tread-robot-build-aims-for-a-smooth-ride/)

这款坦克机器人有各种各样有趣的东西，但那个漂亮的悬挂系统立刻吸引了我们的眼球。它有助于保护机器人的身体在粗糙的表面上行驶时不被震裂。请务必查看构建日志的四个部分，它们可以在上面链接的文章的左边栏中找到。

这是一个硕士论文项目，并已建立从共同的部分。踏板的马达来自一对无绳电钻，并添加了一些电容器，以帮助防止启动时的牵引力。每个踏板都是由一对自行车链条和许多 PVC 管段连接而成。每个 PVC 块的弯曲部分朝向链条，边缘朝向地面以获得更大的牵引力。支撑踏板中间的树轮每个都有一个铰链和弹簧，以吸收全速跑进混凝土人行道拐角的震动，就像我们在休息后的视频中看到的那样。
