# Arduino 大班！克隆忠实于原作

> 原文:[https://hack aday . com/2012/08/01/arduino-taipan-clone-stays-true-to-the-original/](https://hackaday.com/2012/08/01/arduino-taipan-clone-stays-true-to-the-original/)

![taipan](../Images/7e77f5120db8ee691b4e4d736c843ffc.png "taipan")

复古苹果[和 TRS-80 游戏]的粉丝无疑会很快认出上面的图片。大班！在当时是一款流行的游戏，并且[Simon]认为尝试用 Arduino 重新制作这款游戏是一个不错的选择。

他的目标是使用标准的 Arduino Duemilanove 来重现游戏，而不是选择 Mega 或类似 Raspberry Pi 的东西。看到这两个选项“太容易”，他冒险加入了这个项目，并给自己施加了一些限制，使它成为一个更有成效的冒险。

最终，[西蒙]不得不使用两个 Arduinos 一个控制游戏，另一个运行显示器。西蒙将两块电路板、一个键盘和一个液晶显示屏塞进了第一版的大班电脑中，这一举动肯定会让苹果迷和艾勒比迷们高兴。

[西蒙]确保在移植过程中没有任何细节被忽略，确保包括每一行文本以及在原始游戏中发现的每个 bug。
