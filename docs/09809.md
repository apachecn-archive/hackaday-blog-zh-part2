# Amazon Dash:破解它来运行你自己的代码

> 原文：<https://hackaday.com/2015/08/12/amazon-dash-hack-it-to-run-your-own-code/>

亚马逊 Dash 是一个 5 美元的购买猫砂按钮，具有很好的再利用潜力，但你需要先知道里面发生了什么。[托尼·迪科拉]在这本[出色的 Dash](https://learn.adafruit.com/dash-hacking-bare-metal-stm32-programming/connections) 裸机指南中有详细介绍。在这本书里，他讲述了如何进入仪表板并重新编程，让它做一些比买猫砂更有趣的事情。

他首先打开设备，连接一个程序员，然后建立一个工具链来编译程序运行。这不适合胆小的人，因为你是在为一个并非真正为它而构建的设备直接编程，但是[Tony]已经发布了一些例子，并且很少有工具可以在这个过程中帮助你。有一个安全网，[托尼]提供了如何重置亚马逊破折号按钮的细节，如果你设法砖它。

我们已经看到一些有趣的黑客重新利用 Dash[通过拦截连接到 WiFi](http://hackaday.com/2015/08/10/hacking-the-amazon-dash-button-to-record-whatever-you-want/) 的设备来捕捉您孩子的排便，但本指南更进了一步。它允许你运行自己的代码，这使它成为一个真正低成本和精心设计的一体化 WiFi 设备。缺少的部分是运行内部 WiFi 模块的概念验证代码。如果你正在制作那个[，我们很想听听它的故事](http://hackaday.com/submit-a-tip/)！
