# 适用于所有 3D 打印机的自动打印弹出器

> 原文：<https://hackaday.com/2015/06/06/automatic-print-ejector-for-all-3d-printers/>

早在 2010 年，Makerbot 发布了自动化构建平台，这是一个用于纸杯蛋糕或自动装置的整洁的加热传送带，当打印完成时，它会吐出部分。如果你需要生产 20 件东西，这是一个伟大的发明；如果你想坐在专利上不创新，这是一个完美的发明。

你不需要等到 2030 年才能得到一种能自动从打印床上移除印刷品的设备。MatterHackers 公司的人想出了一种[自动指纹排出器](https://hackaday.io/project/5023-automatic-print-ejector)，它能以最像鲁比·戈德堡(Rube Goldberg)的方式移除指纹:用拳击手套。

自动打印排出器很大程度上是直接从[勃斯特·基顿]电影中截取的。这是一系列剪刀机构，末端有一个 3D 打印的拳击手套，由步进电机驱动。当打印完成时，拳击手套简单地从打印机床上打出一个指纹。
