# le-go kart；一辆完全用乐高建造的赛车

> 原文：<https://hackaday.com/2012/09/08/le-gokart-a-gokart-built-entirely-from-lego/>

[![](img/3446e579e73e82c1cfd95c8991cd1082.png "LEGO-gokart")](http://hackaday.com/wp-content/uploads/2012/09/lego-gokart.jpg)

来自荷兰的[Eric Steenstra]决定[完全用乐高 Mindstorm 零件建造一辆赛车](http://meglug.tumblr.com)。经过测试，一个 16 英石(224 磅)的人可以负重 100 多公斤。这种小车可以很容易地载着一个孩子并推动他前进。Eric 使用了 48 个减速的 stock Mindstorm 马达和 16 个电池组来提供扭矩和速度之间的平衡。

这辆车不指望在速度部门赢得任何比赛。从与众不同的角度来看，这轻而易举地赢得了胜利。卡丁车的第一次试驾是在两周前，所以驾驶性能和耐用性在这个阶段仍在开发中。在这个巨大的思维风暴创作之后，请观看视频。
