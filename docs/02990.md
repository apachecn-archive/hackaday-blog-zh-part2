# AC Vs DC 人体疼痛测试

> 原文:[https://hackaday.com/2013/03/16/ac-vs-dc-human-pain-test/](https://hackaday.com/2013/03/16/ac-vs-dc-human-pain-test/)

![AC-vs-DC-human-pain-test](../Images/d9b80f7698beca033b313789b78a208e.png)

有没有想过被电击会有多痛？好奇 AC 比 DC 差吗？想知道人体能承受多少伏特吗？虽然许多人可能会对这种想法感到害怕，但[Mehdi Sadaghdar]是一名电气工程师，他决定将自己变成人类的小白鼠，并找出答案。

[Mehdi]测量了他干皮肤，湿皮肤，最后是舌头的电阻。他发现他的舌头电阻最小，所以它能感觉到低得多的电流。然后，他用一个台式电源，把自己的舌头作为试验场——慢慢地把电压调高，直到他再也无法忍受疼痛。他测试了以下几个阶段:他第一次能感觉到电流，什么时候开始感到讨厌，什么时候感觉像是折磨，什么时候他再也无法忍受疼痛。他试过 AC 和 DC，报告说 AC 更差。

休息之后，看看这个信息丰富，但不可否认有时很搞笑的视频。[Mehdi]看起来是个很棒的工程师！记住——不要在家里尝试。
