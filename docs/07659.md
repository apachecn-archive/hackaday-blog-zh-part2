# Frappé咖啡机器人提供自动采摘功能

> 原文：<https://hackaday.com/2014/11/29/frappe-coffee-robot-offers-automated-pic-me-up/>

虽然[Giorgos Lazaridis]已经几次光临 Hackaday，但我们从来没有报道过读者[Jim]在搜索关于 PIC16F1937 的信息后遇到的他的 frappé机器的构造。他的网站显示构建正在进行中，但他在这里肯定有一个工作原型，而且肯定很棒。

Frappé咖啡的制作方法是将喷雾干燥的速溶咖啡、少量冷水和糖混合，在使用单一搅拌器的手动摇壶或奶昔机中品尝。根据需要添加更多的水和/或牛奶来注满玻璃杯。这种方法是在希腊的塞萨洛尼基偶然发明的，现在已经相当流行了。

除了 16 页的详细构建日志之外，[Giorgos]还拍摄了视频，展示了组成机器的每个模块。操作者将一个玻璃杯放在与转盘相连的支架上。它首先移动到咖啡和糖分配器，通过同一个容易拆卸的漏斗落下。下一站完美地结合了加水和拍子的步骤。绑在搅拌器外壳上的一段软管分配最初的冷水基。然后，搅拌器自动下降，搅拌混合物。混合后，搅拌器被拉高一英寸左右，然后加入更多的水冲洗干净。然后，搅拌器完全撤回，玻璃是充满了其余的方式。frappé的最后一站对整个过程至关重要:必须添加一根可弯曲的吸管。这是至关重要的,[Giorgos]令人钦佩地用一根刺将吸管插入玻璃。

[Giorgos]的咖啡机器人是围绕一个 PIC16F1937 打造的。他为主板和机器的每个模块卷了自己的 PCB。在这个项目中有很多逻辑上的独创性,[Giorgos]的构建日志很好地传达了这一切。休息之后，请务必查看[Giorgos]'机器的运行情况。全套八个视频展示了每个模块，并在下面的视频中达到高潮，非常值得您花时间观看。
