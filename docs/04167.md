# 机械打字机打你的推文！

> 原文:[https://hack aday . com/2013/09/27/mechanical-types-types-your-tweets/](https://hackaday.com/2013/09/27/mechanical-typewriter-types-your-tweets/)

![ScreenShot027](../Images/b1c4626af3ae3069be23badbff4b64de.png)

虽然我们没能去参观上周末的多伦多创客大会，但是一个朋友让我们知道了这个伟大的黑客技术。一台机械化打字机打出针对制造者的推文，@mschwanzer！

[Michael Schwanzer]有一些博客文章概述了这一构建，但是这篇[新闻文章的第一部分和附带的视频](http://www.thestar.com/news/gta/2013/09/21/toronto_mini_maker_faire_featuring_robots_and_3d_printing_a_family_affair.html)很好地解释了它。打印机-打字机的特点是由 Arduino 使用移位寄存器控制的螺线管阵列。Raspberry Pi 从 Twitter 收集信息，然后将数据解析到 Arduino 进行输入。一个简单的概念，但一个复杂和相对昂贵的建设。

在展会期间，人们可以在这个网站上打印和发布自己的推文。你仍然可以看到它的作用，休息后看看视频就知道了！
