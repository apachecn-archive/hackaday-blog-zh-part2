# 一对 N64 便携式电脑

> 原文:[https://hackaday.com/2012/12/13/a-pair-of-n64-portables/](https://hackaday.com/2012/12/13/a-pair-of-n64-portables/)

![portables](../Images/8b98dad4c5263ad2dd86915284fbf628.png)

Casemodding 已经远远超越了仅仅几年前的老式便携式 Ataris 和 NESes。现在，新的热点是更现代的游戏机，包括 GameCube，Dreamcast 和古老的 N64。在过去的几天里，我们收到了两个 N64 机箱改装，我们想不出比这两个更好的机箱制造和控制台改装展示了。

首先是[【特拉维斯】的 N64 掌机。](http://made-by-bacteria.com/viewtopic.php?f=18&t=2239)表壳由一片 ABS 塑料制成，使用 Bondo 使一切变得圆滑光滑。这款掌上电脑有一个 7 英寸的显示屏，以及两个锂离子电池，能够提供长达三个小时的游戏时间。这座建筑的装修非常壮观，证明了(特拉维斯)的耐心和邦多技巧。

接下来是[一个非常非常小的构建](http://forums.modretro.com/viewtopic.php?f=3&t=10901)声称是最小的 N64 便携式。这是[巴德]的作品，比 N64 弹药筒大不了多少。里面是一个 3.5 英寸的屏幕和足够的 LiPos 提供大约 2 小时的游戏时间。与其他(更大的)构建不同，[芽]将墨盒插槽放在外壳的外部，允许墨盒以 90 度角伸出。

这两个非常棒的作品都展示了大量打磨和填充的效果。休息之后你可以看看每个案例的视频。
