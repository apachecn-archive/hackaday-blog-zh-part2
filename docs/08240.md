# 吃豆人时钟吃的是时间，不是颗粒

> 原文:[https://hack aday . com/2015/02/05/PAC-man-clock-eats-time-not-pellets/](https://hackaday.com/2015/02/05/pac-man-clock-eats-time-not-pellets/)

吃豆人时钟肯定会吸引我们所有人内心的复古怪胎。通过一个微小的时间展示，很明显这个项目更多的是关于艺术作品，而不是保持时间。“吃豆人”周期性地随机开合嘴巴。EL 线也增加了一个漂亮的发光触摸。

这个项目是在一个非常小的 2.0 版本上运行的。这是一个小型廉价的微控制器，与 Arduino 兼容。Teensy 使用外部实时时钟模块来保持准确的时间。它还通过串行接口连接到七段显示板。这使得布线简单，显示器易于安装。最后一个主要部件是伺服系统。它只是一个标准的伺服系统，安装在定制的 3D 打印安装支架上。当伺服机构朝一个方向旋转时，嘴张开，反之亦然。框架还用蓝色 EL 线勾勒出轮廓，给经典的吃豆人增添了一点额外的东西。

物理时钟本身几乎完全由木头制成。[Bob]显然是一个熟练的[木工](http://hackaday.com/2014/11/21/a-modern-woodworking-workbench/ "Wood working bench")，正如下面的制作视频所证明的。《吃豆人》和《幽灵》都是在卷轴锯上切割的，尽管[鲍勃]提到如果他的打印机足够大，他会 3D 打印出来。许多组件被热粘合在一起。电子元件也被热粘合在适当的位置。这通常是一种方便的安装解决方案，因为它相对坚固，但只是半永久性的。

[Bob]提到他不能同时运行 EL 线和伺服系统。如果他尝试这样做，几分钟后，这个少年就会“失控”。他在寻找建议，所以如果你有建议，一定要留下评论。
