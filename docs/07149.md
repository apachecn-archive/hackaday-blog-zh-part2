# 43oh.com 不在德州仪器展台旁边

> 原文：<https://hackaday.com/2014/09/23/43oh-com-wasnt-next-to-the-texas-instruments-booth/>

[https://www.youtube.com/embed/H4JIIeihgZw?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/H4JIIeihgZw?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

除了在 Maker Faire 的德州仪器展台上发现的所有酷炫的主板和升压包，来自[43oh.com](http://43oh.com/)的人们也进行了展示，但不是在 TI 展台旁边。公平地说，TI 展台就在 NASA 的正对面。43oh 很酷，但不是 NASA 酷。

在 43oh 论坛上被称为[spirilis]的[Eric]展示了一些在论坛上开发的整洁的小东西，包括一个闪电探测器、一个 VFD 时钟、一个机器人和一大堆闪闪发光的东西。展台上有一系列令人惊讶的项目和电路板，涵盖了从 OLEDs 到电机驱动器的所有内容。

43oh 是一个围绕 TI 微控制器的有趣社区，就像围绕 Atmel 产品建立的 AVRfreaks 论坛一样。43oh 在[有一个非常活跃的论坛](http://forum.43oh.com/)，IRC，在[有一个展示会员项目的商店](http://www.store.43oh.com/)。很高兴在展会上看到这些人，我们希望更多的非官方社区能在未来的展会上有更多的展示。

很抱歉上面视频中的麦克风中断了。展台附近有一片射频喷涌的海洋。如果有人对*数字*无线话筒系统有任何建议，我们洗耳恭听。[这个](http://www.azden.com/products/wireless-systems/pro-series-dslr-wireless/)是目前的钻机。
