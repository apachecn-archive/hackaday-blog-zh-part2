# 不要把它贴在任何地方:用 USB 包装保护它

> 原文：<https://hackaday.com/2014/03/21/dont-just-go-sticking-that-anywhere-protect-the-precious-with-a-usb-wrapper/>

哦，看，一个公共充电站。你应该相信它吗？你可能会被抢劫。哦，等等，你有一个由设计的 [USB 包装器来拒绝访问你的数据。](http://dangerousprototypes.com/forum/viewtopic.php?f=56&t=6240)

这个项目的灵感来自 USB 避孕套，但这些的问题是，它们完全切断了数据线，并将充电速率限制在 USB 2.0 (500mA)。数据线用于向设备传递关于充电器供电能力的信息。许多制造商将 D+和 D-短接在一起，但苹果对这些线路施加特定的电压。

[Scasagrande]的 USB 包装器为您提供了多种选择。可以设置为专用充电口，索尼，开路，或者苹果。这个黑客的超级酷的部分是给你苹果粉丝的。底部的滑块可以让你模仿任何苹果充电器，使用任何 USB 立方体([包括一个你可能已经做好的](http://hackaday.com/2014/03/12/usb-ify-your-old-cell-phone-chargers/))，只要你的斜挎包里有那根有趣的电缆。硬件是开源的，可以在[【Scasagrande】的回购](https://github.com/Galvant/usb_wrapper-pcb)处获得。
