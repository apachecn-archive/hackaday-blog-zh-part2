嗯，他们可能会生气。

头足类动物有一个巧妙的技巧，它们可以通过改变色素泡或“有色细胞团”的大小和形状来改变颜色。像大多数细胞一样，这些染色体对电流有不同的反应。主要是，扩张和收缩。

后院大脑是一个在家中进行神经科学研究的组织，他们决定进行一项实验，将西普里斯·希尔的歌曲《膜中的疯狂》中的信号直接注入鱿鱼鳍上的神经，以观察色素细胞的反应。不出所料，它们会随着节拍跳动。

仅仅因为它不是一个惊喜，并不意味着它不好玩。
