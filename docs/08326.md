# Retrotechtacular:电传打字机之旅，拆卸

> 原文:[https://hackaday . com/2015/02/17/retro tech tacular-tele printer-tour-tear down/](https://hackaday.com/2015/02/17/retrotechtacular-teleprinter-tour-teardown/)

本周，我们将乘坐时光倒流机回到 1940 年，对电传打字机进行一次信息丰富的快节奏观察。在电报局的柜台前，[玛丽]向职员背诵她的祝福。他填好一张表格，把它塞进一个小罐子里，然后嗖的一声通过一根管子送到仪器室。在这里，一名操作员在一台被称为电传打字机的迷人的机电设备上打出电报，[玛丽]的祝贺通过电线传输到数百英里外她朋友的当地电报局。

我们看到电传打字机是一种收发器，它将操作员的按键机械地转换成 5 位二进制代码。例如，' y' = 10101。然后，这种代码以电脉冲的形式传送到远处办公室的电传打字机上，在那里它们被翻译成字母数字数据。这部电影很好地解释了所有这些发生的方法，并且用一个抽象的、彩色编码的电传打字机内部模型来解释。

首先解释从操作者输入到二进制输出的转换，然后在接收端机械翻译回文本。在这里，它被上面显示的打印轮打印在一条薄纸带上。这个时代的收报局的电报员以意味深长的散文形式，将磁带剪贴在空白电报上。最后，它被一个骑着摩托车的厚颜无耻的小伙子送到了预定的收件人手中。
