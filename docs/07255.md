# 四旋翼吊舱赛车

> 原文:[https://hackaday.com/2014/10/06/quadrotor-pod-racing/](https://hackaday.com/2014/10/06/quadrotor-pod-racing/)

一群来自法国阿尔戈奈的多翼飞机爱好者封锁了一条穿过森林的道路，并花了一天时间进行比赛。由此产生的视频让它看起来像是一段美好的时光。

20 名“无人机”飞行员都使用第一人称视角(FPV)相机设置进行完全沉浸，以每小时 50 公里的速度穿过 150 米的树林，这是为了最大限度地刺激和刺激而选择的。赛道基本上沿着一条人行道，但飞行员仍然必须非常警惕，以避免自然障碍(我们称之为“树木”)。叙述者补充说，几乎随机的灯光和相机文物增加了事件的额外难度。

为了完整地绕着赛道练习了几次后，他们开始在预赛中互相比赛。在最后的预赛中，视频中的 3 点 40 分，五架直升机开始并驾齐驱，冲进了树林。其中，只有两个冲过了终点线。

FPV 无人机坠毁的场景仍然让我们有点畏缩。我们想知道有多少参与者在接下来的几个晚上呆在修理间里。
