# 永远不要接触电子产品？

> 原文:[https://hackaday.com/2013/12/26/neverwet-on-electronics/](https://hackaday.com/2013/12/26/neverwet-on-electronics/)

![adafruit-neverwet-04](../Images/58eb218e7f8f0d3993818e56f73c1cfe.png)

NeverWet 是做电子产品的吗？在 [Adafruit 的团队只需要找出](http://learn.adafruit.com/neverwet-electronics/overview)——[在某种程度上，它确实有效！](http://www.youtube.com/watch?v=P42MzjuEPig)

但是等等，什么是永不湿？是 [Rust-Oleum 的神奇拒水涂层](http://www.neverwet.com/)超级疏水。它实际上是有效的，我们有点惊讶我们还没有看到它被用于黑客攻击！无论如何，让我们从一个简短的免责声明开始吧。never wet**不是**为防水电子产品设计的。

但是这什么时候停止过对科学的追求呢！？

实验者选择了一些电子豚鼠来测试 NeverWet 的能力。一个 Arduino Micro，一个 FLORA LED broach，一个 Raspberry Pi。使用适当的应用方法，他们在不幸的电子产品上涂上几层厚厚的产品。他们用普通的纽约自来水测试了每一种成分。FLORA LED broach(如上图所示)在水下持续了大约 4 个小时才死亡。Arduino Micro 的情况类似，但是 Wet Raspberry 在失去与 SD 卡的连接之前只启动了一次。
