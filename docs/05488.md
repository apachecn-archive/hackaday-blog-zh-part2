# 真的还是假的？神奇的乒乓球机器人

> 原文:[https://hack aday . com/2014/03/08/the-amazing-ping-pong-robot/](https://hackaday.com/2014/03/08/the-amazing-ping-pong-robot/)

你愿意在乒乓球 ( [译](http://translate.google.com/translate?sl=auto&tl=en&js=n&prev=_t&hl=en&ie=UTF-8&u=http%3A%2F%2Fulfhoffmannroboter.wordpress.com%2F2014%2F03%2F05%2Fes-ist-vollbracht%2F))中扮演一个[机器人吗？我们肯定会的。受到即将到来的](http://ulfhoffmannroboter.wordpress.com/2014/03/05/es-ist-vollbracht/)[人机对抗](http://www.youtube.com/watch?v=_mbdtupCbc4)的启发，【雅各布】写信告诉我们关于【乌尔夫·霍夫曼的】乒乓球比赛机器人。如果你曾经想在没有人的时候打乒乓球，或者厌倦了和同样的对手打乒乓球，这个项目就是为你准备的。好家伙这东西太神奇了；休息之后，你必须在视频中看到机器人的动作。

虽然机器人的建造并没有记录在一篇帖子中，但[Ulf Hoffmann]的博客有许多视频和迷你帖子，讲述了他如何建造桨挥舞奇迹。构建的范围从最初的想法，到手绘草图，再到上面看到的技术图纸。从这些部分的手臂被建成，但机械装配只是该项目的一部分。它还需要软件来跟踪球，并计算如何正确地返回它。一定要浏览他过去的帖子，那里有丰富的信息。

另外，请务必在 3 月 11 日入住，看看谁会赢得这场[史诗般的人机对决](http://www.kuka-timoboll.com/en/home/)。休息后看预告片(第二个嵌入视频)。

**更新:**许多评论家称这是一个赝品。想到这真让人难过，因为这是一个*真正*酷的项目。但是我们已经改变了标题，并要求你权衡一下你认为它是真的还是假的。我们也联系了[Ulf]，询问它是真实的硬件，还是 CGI 增强的视频。如果有他的消息，我们会通知你。
