# 构建自己的函数生成器

> 原文:[https://hack aday . com/2015/08/01/build-your-own-function-generator/](https://hackaday.com/2015/08/01/build-your-own-function-generator/)

[Scullcom]已经发布了他的[函数生成器构建教程](https://www.youtube.com/watch?v=kbkBmhBM0wI)的第二部分。[Scullcom]之前发布了这个版本的第一部分，它涵盖了他的设计所基于的 XR2206 单片函数发生器 ic。在这一部分中,[Scullcom]涵盖了输出阶段和最终装配。

我们之前已经讨论过[数字](http://hackaday.com/2012/02/08/attiny25-based-function-generator-causes-a-wave/)和[模拟](http://hackaday.com/2011/03/27/function-generator-built-and-mounted-inside-electronics-bench/)函数发生器的构建。[Scullcom]的设计通过提供他所使用的设计的详细描述很好地补充了这些，并提供了该项目中使用的 Arduino Nano 的完整原理图和代码。该设计涵盖音频频率(~40Hz 至 30KHz)，具有方波、正弦波和三角波输出。虽然 XR2206 无法与现代 DDS 函数发生器竞争，但如果你是一名预算有限的黑客，并且正在寻找一个有趣的项目，这可能正是你所需要的。即使你不决定建造一个，你可能会发现[Scullcom]对输出阶段的描述很有趣。

伟大的项目[Scullcom]，我们期待你的下一个版本！
