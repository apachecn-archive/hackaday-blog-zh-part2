# 水下滑翔机原型

> 原文:[https://hackaday.com/2012/08/17/water-glider-prototype/](https://hackaday.com/2012/08/17/water-glider-prototype/)

![](../Images/bcec3596c987359ae7f94613a377b072.png "water-glider-prototype")

[Byrel Mitchell]来信分享了一些关于这架水下滑翔机的细节，他一直在密歇根理工大学非线性自主系统实验室和他的同学们一起工作。顾名思义，它在水中滑行，而不是使用水下遥控潜水器上常见的[推进系统。身体两侧的翅膀固定在适当的位置，将压载的变化转化为前进的动力。](http://hackaday.com/2012/06/28/diy-rov-explores-the-watery-depths/)

滑翔机的前部在上面图像的右下方。仔细看，你会看到三个注射器指向鼻子。这些充当压载舱。齿轮马达移动连接到注射器柱塞的小齿轮，允许驱动设备的 Arduino 用水填充和排空水箱。当满的时候，机头下沉，滑翔机向前运动，当空的时候，机头上升到水面，这也导致向前运动。

休息之后，你可以找到两个视频，第一个展示了功能，并在游泳池中演示了该设备。第二部分介绍控制系统的细节。
