# 用车床和相机制作朝鲜蓟动画

> 原文:[https://hack aday . com/2015/01/14/用车床和相机制作你的洋蓟动画/](https://hackaday.com/2015/01/14/animate-your-artichoke-with-a-lathe-and-camera/)

螺旋，分形，甚至骨骼的长度比例都在诉说着编织进宇宙的一致比例。数学隐藏在事物的结构中，当在艺术中观察到这一事实时，奇迹就发生了。教授、艺术家和发明家[John Edmark]从自然中发现的几何图案中汲取灵感，并使用黄金比例作为设计标准来建造[雕塑。在这个项目中，他通过生动的生物形态的蜥蜴来表达这些特征。](http://web.stanford.edu/~edmark/)

![goldenratio2](../Images/c0c535b2c2ed15f0ae47ddffa78cb3b6.png)【John】用 Rhino 制作了几个 3D 雕塑，包含了与松果和棕榈树叶相似的几何属性。当这些线段从自然界中的那些物体中生长出来时，它们以大约 137.5 度的间隔生长。这种间距产生了一种特殊的螺旋外观，这正是[约翰]想要重现的。为了做到这一点，他使用了 Python 脚本来计算球体表面上伸展的四边形网。从每一个部分，茎状突起从顶部中心向外延伸。一旦这些图形被 3D 打印出来，它们就被一次一个地安装到一个旋转底座的中心，并被设置为以 550 RPM 的速度旋转。然后，摄像机以 1/2000 秒的帧速率拍摄运动中的形状，在正确的位置捕捉物体的静止图像，以产生卷须从顶部盛开并从两侧倾泻而下的错觉。用闪光灯代替照相机也可以达到同样的效果。

[John]在他的[指令页面](http://instructables.com/id/Blooming-Zoetrope-Sculptures/)上有更多信息。他还提供了一个用实际的朝鲜蓟表演这一把戏的视频；这个项目的灵感来源于黄金比例的一个活生生的例子。感谢[Charlie Nordstrom]帮助他记录这些令人敬畏的雕塑，并告诉我们关于它们的事情！
