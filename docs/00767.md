# 运动洒水器赶走排便的狗

> 原文:[https://hack aday . com/2012/06/18/motion-spreader-chases-away-排便狗/](https://hackaday.com/2012/06/18/motion-sprinkler-chases-away-defecating-dogs/)

不想让狗在前面的草坪上拉屎？你可以竖起一个标志，你可以像一个古怪的老守财奴一样赶走它们，或者你可以[建造一个运动探测洒水系统](http://www.unmaintained.com/index.php/spaghetti-ip-cam-arduino-motion-detect-sprinkler/)。当你被冷水喷到脸上(或者更糟)时，排队等便便是相当困难的。

设置非常简单。上图显示的是从网络摄像头看到的景象。监控视频的服务器正在运行检测一帧和下一帧之间运动的软件。当它在正确的位置看到一些东西时，它会向 Arduino 发出信号，触发一直阻止水的螺线管。休息之后，看看电影，里面显示[菲尔·塔克]走过草地触发陷阱。

洒水车黑客总是很有趣。这款可变射程洒水喷头仍然是我们的最爱之一。
