# 在推特上发布啤酒自动售货机需要员工批准

> 原文:[https://hack aday . com/2013/02/01/tweeting-beer-dispenser-requires-co-worker-approval/](https://hackaday.com/2013/02/01/tweeting-beer-dispenser-requires-co-worker-approval/)

![tweeting-beer-dispenser](../Images/f27182814f414bed90ba24c02ed09c0c.png)

面对空啤酒冰箱的次数太多了，金属制品公司的团队想出了一个发放麦芽饮料的审批系统。诀窍是移除罐装分配器上的物理控制。你能得到一杯冷饮的唯一方法是通过它的推特账户问机器。如果里面有啤酒，它会等待你的同事批准。

这种机器有两种型号。第一个是被黑客攻击的冰箱，门上有一个分配器孔。这驻留在他们的悉尼办公室，显然不是那么好，只有在休息后的视频显示。

上图是位于他们新加坡分公司的 2.0 版本。这是一个小得多的设备，但工作非常好，因为它开始是一个商业上可用的罐头分配器。你可以看到组成驱动电路的 Arduino Leonardo 和试验板。

这里没有太多的细节，但是不难找到一百万个 Arduino 使用 Twitter 的例子。这里有一个将莫尔斯电码作为输入，[将消息作为 Tweet](http://hackaday.com/2012/01/31/telegraph-key-makes-for-a-fantastic-twitter-input/) 发布。
