# 做她的 C64 低音吉他

> 原文：<https://hackaday.com/2012/07/09/jeri-ellsworth-on-making-her-c64-bass-keytar/>

![](img/9a267a3f968276bc4de1df5e3ab3dfa9.png "jeri-on-making-her-c64-bass-keytar")

[Jeri Ellsworth]终于腾出一些时间来谈论她的 64 bass keytar 准将的建造过程。我们认为，把一把带锯带到吉他琴身上，最终成为一种奇妙的新乐器。

当她在 Maker Faire 展示这个项目时，我们只是粗略地看了一下它能做什么。她最近的视频涵盖了完成这个项目的所有内容。一旦吉他主体的大部分不见了，她就从一个死去的 c64 中取出内脏，以便将外壳与吉他琴颈配对。作为一名工匠，她改变了电脑的徽章以保持其标志性的外观，然后开始使用压电传感器为每根琴弦添加拾音器。这样做是考虑到 Maker Faire，因为在活动中可能发现的所有特斯拉线圈周围的磁性拾音器都不可靠。在通过连接到原始 c64 SID 6581 芯片的 FPGA 进行处理之前，这些信号被放大和滤波。
