# 使用激光促进毛发生长

> 原文：<https://hackaday.com/2014/12/15/using-lasers-for-hair-growth/>

HowToLou 带着一个相当有趣的构建回来了:[一百个用于毛发生长的激光二极管](https://sites.google.com/site/howtolou//laserhat)。

在你嘲笑激光使男性秃顶脱发的头发再生的想法之前，有[数量惊人的 FDA 文件](http://google2.fda.gov/search?q=laser+hair+growth&proxystylesheet=FDAgov)涉及激光二极管和红色发光二极管用于头发生长，还有[一项有趣的研究](http://www.sciencefriday.com/segment/05/30/2014/laser-blast-can-regrow-teeth-in-rats.html)涉及激光使牙齿再生。是的，伙计们，这是一件真实的事情，但出于显而易见的原因，它永远不会得到双盲研究。

[Lou]正在用 100 个激光二极管制作他的帽子，其中大部分来自亚马逊。这些二极管被植入一块泡沫地板中，这是一个相当有趣的解决方案，将几十个二极管放在一个灵活的模块中，非常适合制作可穿戴设备。

这些激光器由三节 AA 电池供电，电池被塞进一个四槽电池座，电池座经过改造后可以容纳一个电源开关。[楼]已经戴了一个月的九二极管帽子了，如果照片可信的话，他看到了一点点头发的生长。至少，这是一个有趣的伪医学构建，似乎正在产生结果。

像这样的帽子大约 700 美元就可以买到。[Lou]花了大约 60 美元建造了他的。我们称之为胜利，即使最终的结果并不令卢满意。只是不要用剩下的那只眼睛看激光。
