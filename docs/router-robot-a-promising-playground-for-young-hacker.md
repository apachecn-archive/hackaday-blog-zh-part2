# 路由器机器人是年轻黑客的乐园

> 原文：<https://hackaday.com/2014/02/07/router-robot-a-promising-playground-for-young-hacker/>

![router-robot-project](img/6981867b952a486d66fd53a490261753.png)

[Stephen Downward]组装了一个令人印象深刻的互联网控制机器人。关于他的视频演示(也在下面嵌入)有很多我们觉得令人愉快的东西。值得注意的是，当讨论从为项目选择的电子设备、机械装配及其当前状态的问题，以及让他控制火星车的软件后端等一切问题时，他显然知道自己在说什么。

漫游者的大部分是他在旧货店买的 Linksys WRT-54G 路由器。很长一段时间以来，这一直是建造火星车的流行模型。[Stephen]没有直接从路由器的串行端口开车，但这对于他来说可能是一次冒险。目前，他使用 Arduino Mega 和以太网屏蔽将电机连接到他的网络。前面的 IP 摄像头为他提供了视频，让他完全通过互联网使用自己用 C#编写的程序进行操作。他提到，他的 CD 车轮并不理想，因为它们的胎面面积较薄(覆盖有遮蔽胶带)，并且安装不准确，导致其中一个车轮倾斜。他希望自己设计并印刷。他计划在当地大学的 3D 打印服务重新上线时租一台 3D 打印机。

我们认为制造机器人最困难的部分是建立并运行你的第一个平台。既然他已经明白这是一个改进和添加的问题。既然他已经掌握了 Mega 的 I/O，我们希望看到他实现一系列不同的传感器:[线跟随](http://hackaday.com/2010/08/13/a-collection-of-quick-line-followers/)，碰撞传感器，距离传感器，见鬼…也许有一天他会[为它清理一些激光雷达](http://hackaday.com/2010/11/17/newest-hardware-bounty-the-open-lidar-project/)！
