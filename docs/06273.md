# 为 CNC 路由器设计的机械钟使用 3D 打印机获得新生

> 原文:[https://hackaday . com/2014/06/10/mechanical-clock-designed-for-a-CNC-router-gets-new-life-using-a-3d-printer/](https://hackaday.com/2014/06/10/mechanical-clock-designed-for-a-cnc-router-gets-new-life-using-a-3d-printer/)

[Madis Kaasik]不久前使用 Solid Edge (3D CAD)设计了一个时钟——但从未有机会制作它——直到他成为挪威一所大学的交换学生，有机会使用一台大型工业 3D 打印机！

他原本打算用数控切割机或激光切割机切割出来，但当发现他可以使用大学的 3D 打印机时，他决定尝试一下——这实际上是他打印的第一件东西！这些设计需要为 3D 打印做一点点修改，但是现在已经完成了，他还把它们上传到 Thingiverse 供任何人使用。

这需要对钟摆、重量和齿轮进行相当多的微调，才能让它正常运转，但[Madis]最喜欢这个项目的是认识到 3D 打印的可能性有多大——他很高兴开始他的下一个大的 3D 打印努力！
