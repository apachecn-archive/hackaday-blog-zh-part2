# 复古:手表是如何工作的

> 原文:[https://hackaday . com/2013/01/23/retro tech tacular-how-a-watch-works/](https://hackaday.com/2013/01/23/retrotechtacular-how-a-watch-works/)

![how-a-mechanical-watch-works](../Images/91f413598610bdaf312b754daac90a6e.png)

任何一个曾经试图用电子项目来记录时间的人都会对一个能在几个月或更长时间内保持准确的钟表产生敬意。说到机械表，我们认为它更值得尊敬。这段视频由汉密尔顿钟表公司于 1949 年 [制作，旨在解释精密机械表](http://www.youtube.com/watch?v=cZwq1KL4SD0) 背后的基本流程。

这需要几分钟的时间来抓住演示的实质，但是我们认为你会发现介绍和解释本身一样有趣。当需要观察手表内部时，会使用一组大部件来帮助说明每个部件的工作原理。该剪辑(也是在休息后嵌入的)在这些演示中做得很好，但几乎马上你就会意识到包装在一个令人难以置信的小包装中的复杂性。它还解释了宝石轴承带来的低摩擦特性。尽情享受吧！
