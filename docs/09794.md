# 3D 打印虎钳将开发板放在试验板旁边

> 原文：<https://hackaday.com/2015/08/14/3d-printed-vice-holds-your-breadboards/>

一段时间以来，Stickvise 一直是 Hackaday 社区的主要内容。如果你需要焊接用的东西，没有比这更好的低成本帮手了。但是如果你只是使用试验板和某种类型的开发板，那么另一种老虎钳即将出现，它使用类似的弹簧夹紧来固定一切，同时你可以构建一些令人惊叹的东西。

虽然[Pat]的灵感来自前面提到的老虎钳，但新的 3d 打印老虎钳正是你在准备焊接之前需要的。老虎钳是用橡皮筋弹簧加载的。底座的大小适合在中心安装标准试验板，两侧带有夹持臂，以固定 Arduino 等开发板。这种创新而简单的装置可以很好地夹住电路板，这样你就不会在桌子周围追逐它们，把电线撞得乱七八糟了。

该板有一些细微的差别，所以请务必查看下面的视频，看看它的实际效果。正如我们提到的，它使用橡皮筋而不是弹簧来保持简单，并且它有一些容易 3d 打印的形状，如三角形轨道。如果你想自己 3d 打印，你需要的文件可以在项目网站上找到。如果你想变得更简单，我们在这里也看到了一些其他的恶习。

老虎钳[在 Hackaday 商店](http://store.hackaday.com/products/stickvise)有售。
