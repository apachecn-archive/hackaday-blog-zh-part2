# 将 Belkin WeMo 变成死亡陷阱

> 原文：<https://hackaday.com/2013/01/31/turning-the-belkin-wemo-into-a-deathtrap/>

Belkin WeMo 是一个小型的 WiFi 连接插座，由移动设备控制，可以将互联网控制添加到台灯、咖啡机，或者，如果你觉得大胆，你的家庭服务器。这是一个有趣的设备，但当然将你的电水壶连接到互联网会有一些安全隐患。[丹尼尔] [能够在他的 Belkin WeMo](http://www.youtube.com/watch?v=BcW2q0aHOFo) 上扎根，并完全控制他的互联网连接渠道，能够把它变成一个死亡陷阱。

[Daniel]说他的漏洞可能被开发成一种病毒，可以扫描 WeMo 设备。一旦找到这些连接互联网的设备，很容易快速打开和关闭这些设备；对台灯来说不太危险，但是如果把它插在空间加热器上就可能致命。

在休息后的视频中，你可以看到[丹尼尔]利用其 UPnP 实现中的一个缺陷来利用 WeMo。有他的终端黑客和他的台灯快速开关的镜头，这对高电流设备来说是非常危险的。
