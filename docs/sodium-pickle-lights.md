# 钠泡菜灯

> 原文：<https://hackaday.com/2015/03/02/sodium-pickle-lights/>

几周前，23b 黑客空间的人们举行了 [Sparklecon](http://www.sparklecon.org/wiki/index.php?title=Main_Page) ，这个活动充满了常见的信息安全产品、锁和开锁工具以及硬件。当然，骗局需要一些很酷的演示。[他们选择把泡菜放进电弧焊机](http://blog.shop.23b.org/2015/02/angry-electric-pickle.html)，结果令人印象深刻。

这种构建始于几年前，当时 23B 的一个成员的父亲在万圣节搞了一个巧妙的把戏。用一根切开并剥去外皮的延长线，将两根导线插入泡菜中，并连接到主电源上。泡菜中的钠开始发出明亮的橙黄色光，每个人都被深深打动了。几年后，23b 发现自己有一堆无用的碳刨棒，一个 200 安培的焊机，一个泡菜，还有一群想看些酷东西的人。

制作比太阳还要亮的泡菜的诀窍是将弧度设置得恰到好处；电极之间四分之一英寸似乎是最佳的，但即使这样，泡菜照明似乎对失败的夹具非常有弹性，夹具由牛奶箱、胶带和 PVC 制成。下面的视频(至少是第一个 Sparklecon 的视频)。
