# 音乐灯光秀远没有你想象的那么复杂

> 原文：<https://hackaday.com/2012/06/21/musical-light-show-is-far-less-complex-than-you-might-think/>

![color-changing-light-tube](img/9270c1a21dc5b45d7faca334a584543d.png "color-changing-light-tube")

[Matt 和 Jason Tardy]组成了名为 AudioBody 的音乐表演二人组，他们最近在 make:解释他们如何穿上他们的一个商标片段。他们表演中最受欢迎的部分是变色灯管，两人旋转并投掷这些灯管，就像高科技版的蓝人乐团。虽然视觉效果非常流畅，但背后的技术远比大多数人最初想象的简单。

正如你在下面的视频中看到的，这些管子看起来只不过是简单的白光。然而，当兄弟俩完成他们的表演时，灯管从白色切换到蓝色，然后再切换回来，颜色之间有一种液体般的过渡。

[迟到者]说，大多数人认为微控制器或其他复杂的电子设备是他们光魔法的来源，但真正的答案要简单得多。每根管子的末端都嵌有一个明亮的 LED 手电筒。位于灯管内的滑动蓝色滤光器提供了颜色之间的丝般平滑过渡——不需要任何花哨的电子设备。

如果你想看看他们是如何建造的，一定要去 AudioBody 的网站看一下[迟到者]自己的演示。
