# 逆向技术:在全息图上发光

> 原文:[https://hack aday . com/2014/03/11/retro tech atacular-sheding-light-on-holograms/](https://hackaday.com/2014/03/11/retrotechtacular-shedding-light-on-holograms/)

本周的回顾是 1972 年由大英百科全书的精英们制作的全息摄影介绍。它相当令人钦佩地详细说明了什么是全息图以及它们是如何制作的。

全息图与照片大不相同，尽管两者都记录在胶片上。全息摄影是基于波的叠加效应:两个振幅相等的波峰产生一个更大的波峰，而振幅相等的波峰和波谷相互抵消，产生干涉效应。该视频很好地展示了水波纹的概念，并解释了声波和光波也会产生同样的效果。

激光是全息摄影所需的高强度、高光谱纯度光线的关键。白炽灯光由太多的波长组成，不能有效地分成两个相同的光波源。为了制作全息图，用光学设备将激光分成两束。一束光直接聚焦在被记录的物体上，称为物光束。第二束光线通过一系列镜子从场景中射出，直接照射到胶片感光乳剂上。

这部电影记录了两束光波之间的干涉。显影后它看起来是空白的，但仔细观察就会发现明暗相间的条纹。当曝光的胶片只放在参考光束的路径上时，记录在胶片上的干涉图案将光束一分为二，再现了场景。借助于投影屏幕，可以看到显示 2D 原始物体的全息图。照片和全息图的另一个很大的区别是，即使全息图的一小部分也能再现整个场景，但照片的一部分就是这样。
