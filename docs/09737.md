# 赛格威建立你可以自己做

> 原文：<https://hackaday.com/2015/08/11/segway-build-you-can-do-yourself/>

我们在 Hackaday 上看到了很多 Segway 的变体，但它们并不总是有关于它们是如何制作的详细说明…我们很幸运，[制作材料]的[Bob]刚刚完成了他的广泛的 Segway 项目，有大量的制作视频！

受到他在网上看到的其他自动平衡滑板车项目的启发，[Bob]想尝试制作一个。因此，他从自己喜欢的设计中提取一些零碎的东西，想出了自己看起来很结实的赛格威克隆设计。他使用谷歌 Sketchup 绘制了框架，从那里开始，所有人都在甲板上。一点钨极氩弧焊后，这是时间的组成部分。

除了一些微小的机械故障，该项目的最长部分是电子和软件——让它像真正的赛格威一样工作。在编写自己的代码后，他遇到了一些障碍，但幸运的是，他能够从当地创客聚会上的某个人那里获得一些帮助，这节省了他的许多故障排除工作。
