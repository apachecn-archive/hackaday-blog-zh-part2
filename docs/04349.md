# 烟火启动机

> 原文:[https://hackaday.com/2013/10/19/pyrotechnic-star-machine/](https://hackaday.com/2013/10/19/pyrotechnic-star-machine/)

![Star_Pump_Cutaway](../Images/5d2220c0273dd0bf99b60afb0a42862b.png)

[Robert McMullen]设计并建造了这个[迷人的，全自动的，烟火明星泵](http://www.pyrotechnicmachines.com/star_pump.html) ( [谷歌缓存](http://webcache.googleusercontent.com/search?q=cache:http://www.pyrotechnicmachines.com/star_pump.html))。这是他的工程项目——灵感——的项目要求。他也是奥林学院消防艺术俱乐部的成员。

让我们回顾一下。烟火明星到底是什么？它们是构成所有抛射式烟花核心的[烟火颗粒](http://en.wikipedia.org/wiki/Pyrotechnic_star)。通常是金属粉末、化合物或盐，它们构成了美丽的颜色和烟花的“星星”。唯一的问题？如果做得不好，它们会相当危险。幸运的是，[罗伯特]选择了一个最安全的方法:抽水。基本概念是，初始混合物在被压成圆柱形时是潮湿的，然后被切割和分配，在干燥一段时间后即可使用。

这台机器是一个非常光滑的机械装置，看起来甚至有点令人着迷。在他的博客上有很多信息，甚至技术图纸和 Solidworks 文件都是可用的！如果这一切看起来很熟悉的话，[罗伯特]制造了一台类似的机器来制造防水保险丝。

休息后留下来看看它的运行吧！不幸的是，我们很遗憾地通知你…星星的爆炸不包括在内。
