# 变压器电感耦合仿真是 SFW

> 原文：<https://hackaday.com/2014/10/02/transformer-inductive-coupling-simulation-is-sfw/>

[詹姆斯]有一个朋友在当地的社区大学教书。当这位朋友请他制作一个[变压器耦合模拟](http://www.youtube.com/watch?v=az_WPAsTA-o)时，他非常乐意。对我们来说幸运的是，他还制作了一个视频，解释了在“示波器”上显示输出时发生的情况。

为了进行模拟，[James]使用 PVC 管建造了初级和次级线圈。初级线圈由 11 匝 14AWG 绞合线组成，施加 4V ~~电压。他演示的第一个次级线圈构造类似，但有 13 匝。正如您将看到的，第一个线圈在第二个线圈中感应出约 1.5V 的电压。[James]首先将它与两个方向相同的绕组耦合，从而产生两个同相的 2Mhz 波形。当他以另一种方式插入次级线圈时，它的波形与初级线圈的波形不同相。~~

他的第二个次级线圈具有相同直径的 PVC 芯，但缠绕了大约 60 匝更细的线，准确地说是 28AWG 贝尔线。这种匹配从他加在初级线圈上的 4V 感应出次级线圈上的 10V。[詹姆斯]演示包括一个简短的李沙育模式接近尾声。如果你对这些还不够了解，[这里有一个很好的基础演示](http://www.youtube.com/watch?v=t6nGiBzGLD8)以及对它们背后机制的解释。
