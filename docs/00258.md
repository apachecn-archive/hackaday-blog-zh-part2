# 多点触控桌面使用 Kinect 进行 3D 显示

> 原文:[https://hack aday . com/2012/04/30/multi touch-table-uses-a-Kinect-for-a-3d-display/](https://hackaday.com/2012/04/30/multitouch-table-uses-a-kinect-for-a-3d-display/)

[Bastian]送来了一张他自己做的咖啡桌。不过，这不是放饮料和 Make 副本的地方:这是一个带 3D 显示器的多点触控桌子。由于没有描述可以做到这一表正义，[看看视频](http://www.youtube.com/watch?v=2CbiOikirrg&feature=player_embedded)。

这一设计的灵感来自于这篇黑客文章的[主题，在这篇文章中,【programming4fun】使用一台普通的 2D 投影仪和一台 Kinect 构建了一个“全息显示器”。这两个构建都是基于相对于用户头部重新绘制 3D 空间的原理——当[Bastian]在咖啡桌上移动他的头部时，Kinect 会跟踪他的位置，并向相反方向移动盒子的三维网格。它非常聪明，看起来是一个有前途的用户界面。](http://hackaday.com/2011/07/02/3d-display-using-a-kinect/)

除了 Kinect，咖啡桌还使用了类似微软 Surface 的显示屏；四个红外激光器放置在角落，并通过底座中投影仪旁边的摄像机进行检测。

休息之后，你可以看到演示视频和 NUI 论坛上张贴的图片画廊。
