# 一个非常小的紫外线传感器

> 原文:[https://hack aday . com/2015/05/02/an-exceptually-small-uv-sensor/](https://hackaday.com/2015/05/02/an-exceptionally-small-uv-sensor/)

我们看到的大多数攻击都是这样的——用热熔胶和胶带将硬件项目粘在一起。[【x-labs】“2015 年黑客日奖的参赛作品，UV 徽章](https://hackaday.io/project/4706-uv-badge)，当然不在这些项目之列。这是一个专业的一次性产品，能够在一个小小的外壳中显示紫外线指数、温度、湿度和压力。

紫外线徽章设计用于户外。这意味着任何从诺基亚手机上撕下来的旧显示屏都不行；在阳光下会褪色的。相反，[x-labs]正在使用一种非常阳光可读的夏普内存液晶显示器。这是一个不错的选择，因为它是一个非常低功耗的设备。

在 54 x 34 x 7.1 毫米 3D 印刷外壳内是一个非常薄的 PCB，以及所有表面贴装元件。该设备由一个硬币电池供电，应该可以运行几个月。

由于产品设计得如此之好，我们想知道 UV 徽章是否会参加今年 Hackaday 奖最佳产品类别的竞选。没有太多的项目在运行，获胜者得到足够的资金，机械和经验来把他们的项目变成产品。
