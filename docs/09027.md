# Hackaday 奖品:一台价值 100 美元的 CT 扫描仪

> 原文：<https://hackaday.com/2015/05/14/hackaday-prize-entry-a-100-ct-scanner/>

当你的爸爸是一名兽医，把一台旧 x 光机扔在你的车库里，而你正在寻找 Hackaday 奖的参赛作品时，你会怎么做？[造一台 CT 扫描仪，当然是](https://hackaday.io/project/4740-100-ct-scanner-xray-desktop-3d-scanner)。至少[movax]是这么说的。

[movax]的父亲在车库的 x 光机里放了一些其他的好东西。有包括闪烁体的胶卷暗盒。通过将相机对准这些 x 射线到可见光的转换板，[movax]可以用 x 射线拍摄数字照片。从那以后，它只是建造一个围绕一个物体旋转的装置和许多–*许多–*数学。

有趣的是，这并不是 DIY CT 扫描仪第一次登上 Hackaday 的版面。[Peter Jansen] [从辐射检查源、CMOS 图像传感器和激光切割胶合板的美丽排列中建造了一台机器](http://hackaday.com/2013/10/23/towards-a-low-cost-desktop-ct-scanner/)。这没有使用合适的 x 射线管；相反，[彼得]使用的是最强的合法检查源(钡 133)。蔬菜和水果的扫描时间仍然以天或小时计算，他继续在[制造核磁共振成像机](https://hackaday.io/project/5030-low-field-mri)。

有了真正的 x 射线源，[movax]的机器将比钡基建造的任何东西都好得多，加上正确的代码和图像分析，这可以用作真正有用的 CT 扫描仪。
