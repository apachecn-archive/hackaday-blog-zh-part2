# 把 300 瓦的发光二极管放在遥控飞机上

> 原文:[https://hackaday . com/2012/09/07/将 300 瓦的发光二极管放在遥控飞机上/](https://hackaday.com/2012/09/07/putting-300-watts-of-leds-on-an-rc-plane/)

[![](../Images/28f9f6d9fc8f2c9bcfd7a4c7238e38f4.png "UFO")T2】](http://hackaday.com/wp-content/uploads/2012/09/ufo.jpg)

作为 FPVlab 论坛的一员，[HugeOne]真的很喜欢把摄像机绑在遥控飞机上，凭感觉飞来飞去。他还喜欢在夜间驾驶飞机。将这两种兴趣结合起来，你就有了 [300 瓦的 led 灯在夜晚四处飞舞](http://fpvlab.com/forums/showthread.php?258-Night-flying-TSII-with-300W-leds-headlight)，这很可能会导致【休戈内】的家乡魁北克不明飞行物报告的激增。

将 16 个 CREE XM-L led 放在如此有限的空间中的主要问题是热量问题；尽管这些发光二极管效率惊人，但它们仍然会产生大量的热量。[HugeOne]通过将这些 led 焊接到一根铜管上，并将两个散热器连接到他的飞机上进行液体冷却，解决了这个问题。

其结果是一个小而轻的 LED 阵列，能够产生超过 20，000 流明的光，在魁北克的野外飞行。这极大地提高了[HugeOne]的夜间飞行能力(休息后的视频)，而且 UFO 报告的增加肯定惹恼了当地警察局。

有人知道单引擎客机上的导航灯和着陆灯有多亮吗？
