# 自动售货机原型

> 原文:[https://hack aday . com/2012/09/17/vending-machine-prototyping/](https://hackaday.com/2012/09/17/vending-machine-prototyping/)

![](../Images/8cdaa4672e9afc4866f79642720e58b9.png "a-vending-machine-protytpe")

[Vending Mexico]计划设计、制造和销售他们的自动售货机。你必须从某个地方开始，所以他们建造了这个原型。它提供了一系列的自动售货功能，但它是由我们在业余爱好项目中常见的部件组成的。

他们没有接受的一个挑战是识别硬币和找零的能力。你可以看到他们选择使用 Coinco Guardian 6000 转换器。但定制电路接入设备，识别投入投币口的钱数，并控制自动售货机进行兑换。现在只有一个项目可供选择；一些口香糖储存在纸板隔板中，用典型的金属开瓶器——由伺服电机驱动——来分发产品。在该隔板的正下方有一排红外发光二极管，它们带有一组互补的红外光电晶体管。机器使用这些来检测产品何时掉落。这样，如果你的糖果卡住了，你就可以拿回你的钱。

休息之后，视频中展示了用户界面。它使用一套七段显示器进行反馈。arcade 按钮用于选择所需的产品。视频对话是西班牙语，但我们毫不费力地说出正在展示的内容，尽管我们不会说这种语言。

我们不记得见过其他临时搭建的自动售货机。似乎所有这些都是对旧的商业自动售货硬件的攻击。
