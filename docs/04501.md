# 经典视频游戏模拟的飞跃

> 原文：<https://hackaday.com/2013/11/08/linux-on-a-leapster-for-classic-video-game-emulation/>

圣诞节就要到了，如果你有侄女、侄子或自己的宠物在你家游荡，你可能会想今年你将如何补贴圣诞老人。看起来玩具反斗城将在今年的黑色星期五以 30 美元的价格出售 LeapsterGS。这是一个运行在 550 MHz ARM 9 上的 Linux 设备，有 128 MB 内存和 2 GB 闪存。对于儿童玩具来说过于强大，但对于一个月后忘记它的孩子来说是完美的，因为现在你可以用正确的 Linux 安装来替换固件并运行经典模拟器。

在这些为儿童制造的廉价掌上电脑上安装 Linux 并不是什么新鲜事；我们已经看到用[蛙跳 DIDJ](http://hackaday.com/2010/03/28/leapfrog-didj/) 和[蛙跳探险者](http://hackaday.com/2010/07/08/leapster-explorer-continues-the-didj-tradition/)完成的。然而，这些游戏机的 CPU 相当缺乏，RAM 也不多。摩尔定律似乎终于开始适用于袜子，Leapster GS 的功能足够强大，可以玩所有这些任天堂、Game Boy 甚至 MAME 游戏。

刷新新固件所需要的只是在 LeapsterGS 的电路板上焊接几根电线，用于串行连接。新的 LeapsterGS 固件甚至有 MP3 和电影播放器，所以即使这些机器的接收者在一周内厌倦了它，它仍然有很多生命。

LeapsterGS 玩下面最棒的街机游戏的视频。
