# 使用 LCD、键盘和 SD 的 Arduino BASIC 解释器

> 原文:[https://hack aday . com/2012/05/01/arduino-basic-interpreter-using-LCD-keyboard-and-SD/](https://hackaday.com/2012/05/01/arduino-basic-interpreter-using-lcd-keyboard-and-sd/)

![](../Images/4b35c954fdf53a6c9b4764a2602a4260.png "Arduino-BASIC-interpreter")

如果你手头已经有了这些部件，这个 Arduino 基本解释器将会成为一个非常有趣的一日项目。[Usmar A. Padow]组装一个 Arduino Uno、SD 卡、四行字符 LCD 和 PS/2 键盘。但他也包括了替代选项，即通过使用计算机终端而不用 LCD 屏幕，或者通过仅使用 Uno 的 RAM 而不用 SD 卡。正如你在休息后的演示中看到的，这个简单的输入/输出就是你实验一些古老计算所需要的。

我们很难看到这个画面而不回想起橙色或绿色的单色显示器。就像几十年前一样，BASIC 的这种实现让您用行号开始每一行代码，一旦输入了行，就不允许进行字符编辑。[Usmar]展示的示例程序很容易理解，但是如果您以前从未使用过 BASIC，它足以帮助您入门。

去年八月，我们看到了另一个黑客将 Tiny BASIC 移植到 Arduino 上。你可能也想看看那个。
