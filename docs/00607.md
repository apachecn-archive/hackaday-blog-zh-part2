# 用 Kinect 绘制大面积的 3D 地图

> 原文:[https://hack aday . com/2012/06/01/3d-mapping-of-a-Kinect/](https://hackaday.com/2012/06/01/3d-mapping-of-huge-areas-with-a-kinect/)

![](../Images/fab480a629c3c111abd37e72aa3b7e85.png "kinect")

你在上面看到的图片不是玩偶之家，没有夹子的视频游戏，或者任何其他人工创造的虚拟环境。这个浴室存在于现实生活中，但被 Kinect 和 [Kintinuous](http://www.cs.nuim.ie/research/vision/data/rgbd2012/) 数字化成了一个 3D 物体，这是一个很棒的软件，可以实时创建巨大的 3D 环境。

Kintinuous 是 [Kinect Fusion](http://hackaday.com/2012/03/03/very-easy-3d-scanning-software-with-reconstructme/) 和 [ReconstructMe](http://hackaday.com/2012/03/03/very-easy-3d-scanning-software-with-reconstructme/) 项目的延伸。Fusion 和 ReconstructMe 仅限于在 3D 中映射小区域，例如桌面，Kintinuous 允许 Kinect 从一个房间移动到另一个房间，在 3D 中映射整个环境。

kincontinuous[的论文](http://www.cs.nuim.ie/research/vision/data/rgbd2012/kintinuous.pdf)介绍了作者如何能够捕捉点云数据并覆盖彩色视频以创建纹理化的 3D 网格。休息之后是两个展示 Kintinuous 能力的视频。令人瞠目结舌，寓意惊人。我们找不到 Kintinuous 的二进制文件或源代码，但如果有人找到了链接，请给我们留言，我们会更新这篇文章。
