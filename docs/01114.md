# 用混合动力驱动您的家用电器

> 原文:[https://hack aday . com/2012/07/22/驾驶混合动力家用电器/](https://hackaday.com/2012/07/22/driving-your-home-appliances-with-hybrid-power/)

![](../Images/d6aec0137bdaf050af425b53bff33a43.png "hybrid-power-for-your-home")

这个混合你家电器的系统是对太阳能的有趣利用。它专注于尽可能频繁地将电器从电网无缝切换到储存的太阳能。休息后有一个宣传片解释了设置，但这里是它的要点。

跟着上面的象形图走。我们从左边的太阳能电池板开始。这给一个 12V 电池的充电器充电。当电池充满时，充电器向逆变器供电，逆变器将 12V DC 转换为 110V 交流电。这被馈送到电器(在这种情况下是灯)和壁装电源插座之间的通道。直通将在来自插座的市电和来自逆变器的 110 之间切换。房主不会知道，也不会关心，哪个电源正在被使用。但是阳光充足的月份应该会降低能源费用。真正的问题是需要多长时间来弥补系统节省的电力成本。
