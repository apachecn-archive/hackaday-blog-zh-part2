# 用 CD 和卷尺测量激光波长

> 原文：<https://hackaday.com/2015/03/20/measure-laser-wavelength-with-a-cd-and-a-tape-measure/>

显然，激光的波长不能用像木工卷尺那样大的刻度来测量。至少不是直接的，这就是光盘出现的原因。【Styropyro】[使用 CD 作为衍射光栅](https://www.youtube.com/watch?v=q0kETJwL9mY)，产生足够大的光学图案以进行测量。

[衍射光栅](http://en.wikipedia.org/wiki/Diffraction_grating)将一束光分成多束，其位置由光的波长和光栅的特性决定。由于我们不知道光栅(CD)的属性，因此[Styropyro]使用绿色激光作为参考。这有几个原因:绿色激光的特性不会随着热量而改变，它的波长也是已知的。

一切都与三角形有关。嗯，实际上这都是关于数学的，而数学都是关于三角形的。对于那些不喜欢特殊字符的人来说,[Styropyro]不仅很好地解释了每个符号代表什么，还将其应用于控制实验(在下面的视频中的摄像机上)。测量三角形的边长，然后使用简单的三角学来确定 CD 的狭缝距离。这是他用来测量和确定未知激光波长的一个丢失的数据。
