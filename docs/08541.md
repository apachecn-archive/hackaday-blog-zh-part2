# 夜间太阳能灯泡拯救你的肺

> 原文：<https://hackaday.com/2015/03/14/nocturnal-solar-light-bulb-saves-your-lungs/>

印度在电力方面有点问题。事实上，超过 7400 万农村家庭完全没有电力供应。取而代之的是，他们依靠燃烧燃料来照明——巧合的是，他们吸入了有害的烟雾。更不用说燃料并不便宜。[Debasish Dutta]想要改变这种情况——所以他想出了另一种低成本的太阳能灯。

这是一个非常简单的灯，由廉价的特百惠容器，2V 太阳能电池板，白色 LED，可充电的 AA 或 AAA 电池，光电二极管和焦耳小偷(升压 IC)制成。一天的充电可以为整个晚上的操作提供大约 20-22 流明。虽然看起来不多，但一盏典型的煤油灯发出的光还不到这个亮度的一半。

有了光电二极管，它会在晚上自动打开，白天自动关闭。衣架既可用作充电支架，也可用作晚上悬挂的挂钩。
