# 你肯定认识铝制饮料罐

> 原文：<https://hackaday.com/2015/04/17/you-betta-recognize-the-aluminum-beverage-can/>

他回来了，[比尔·哈马克]又名*工程师家伙*。他有一个习惯，通过一丝不苟地揭示一件事物的所有工程设计，来揭示平凡是多么的不平凡。这一次是[铝制饮料罐](https://www.youtube.com/watch?v=hUhisi2FBuw)。你可能知道它是一个汽水罐、一个啤酒罐，或者一个无盐的[空气罐。但是我们希望你能找到一个不熟悉这些容器的人。](http://i.imgur.com/VsVTz.jpg)

我们知道你在想什么:你已经在*的一集*中看到了这些是如何形成的。你错了。我们也看了那集。但只要给[比尔]几分钟时间，他就会在剩下的一集里把你吸引过去。现在底座和侧壁的模具成型，我们会告诉你你知道那是怎么回事。但后来[比尔]闯入这些容器的历史，引用铝通过减少顶部直径可以节省。他以庆祝现代“停留”标签的独创性来结束这篇文章，这应该会让你兴奋得掉下眼镜。

如果这是你第一次听到工程师的名字，你会有一个愉快的周末。狂欢看他的整个背部分类！我们最喜欢的包括对[的分析、机械傅立叶计算机](http://hackaday.com/2014/11/18/harmonic-analyzer-mechanical-fourier-computer/)和[彩色阳极氧化](http://hackaday.com/2012/05/29/how-anodization-is-used-to-make-pretty-ipod-colors/)中涉及的概念。我们甚至[读过他的书](http://hackaday.com/2012/07/06/book-review-eight-amazing-engineering-stories/)。
