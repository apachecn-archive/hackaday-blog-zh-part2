# 给我的摩托车拉皮条

> 原文:[https://hackaday.com/2012/05/30/pimp-my-scooter/](https://hackaday.com/2012/05/30/pimp-my-scooter/)

![](../Images/8ceb81a0b891bf420a4cdca1752f3943.png "pimp-my-scooter")

[格伦]用一辆看起来很悲伤的小摩托车造出了这台闪亮的派对机器。我们敢打赌，你一定想知道为什么我们认为这是一个党的机器，当它看起来如此普遍？在这张照片中唯一真正的赠品是定制排气，但隐藏在野兽的身体是 720 瓦的党权力加上一大堆额外的。

当他到达他要去的地方时，[格伦]停好车，掀起座椅展开娱乐活动。附在车座下面的是一个 720 瓦的音频放大器。它驱动座位下的一个大扬声器，以及安装在前控制台的两个高音扬声器和两个中音扬声器。但如今，没有视频的派对就不算派对了，这就是为什么你还会发现直立座椅上悬挂着一个 7 英寸的液晶屏幕。音乐和视频由预先安装的 iPod touch 或他安装在车上的个人电脑提供。就差一个游戏机了！
