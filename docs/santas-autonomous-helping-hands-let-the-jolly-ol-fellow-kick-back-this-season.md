# 圣诞老人自动伸出援助之手，让这个快乐的家伙在这个季节尽情放松

> 原文：<https://hackaday.com/2014/12/20/santas-autonomous-helping-hands-let-the-jolly-ol-fellow-kick-back-this-season/>

对于那些怀疑圣诞老人年度送货时间表可行性的人来说，这里有一个难题会让最铁石心肠的非信徒感到困惑。

德国卓越认知互动技术中心(CITEC)的工作人员制作了一个奇妙的演示，展示了圣诞老人的一双援助之手。在这个 2.5 分钟的视频中，机器人执行了一系列令人印象深刻的自主填袜子动作:从识别袜子上的开孔，到从一堆可用的糖果中抓取特定的糖果。

在硬件方面，手臂似乎是一个 [KUKA 变体](http://www.kuka-robotics.com/en/ "KUKA-variant")，而在软件方面，可视化是由开源机器人软件[ROS](http://www.ros.org/ "ROS'")RVIZ 工具处理的。

如果视频中的一些道具[看起来很熟悉](http://justai.org/interactive-disambiguation-of-object-references-for-grasping-tasks/ "look familiar")，你会发现 CITEC 的研究人员已经探索出了一些恒星的感知、分类，以及对相关研究课题的把握。谁知道这双手会如此快乐地在这个假期加班呢？整个视频被设置成一个清晰的计算机配音的顺口溜，作为他们对这个项目方法的一个偷偷摸摸的总结。

现在，要是我们能让这双手去做我们的其他脏活就好了。
