# 用多米诺骨牌学习逻辑门

> 原文:[https://hack aday . com/2015/05/07/learning-logic-gates-with-dominos/](https://hackaday.com/2015/05/07/learning-logic-gates-with-dominos/)

尽管我们大多数人对逻辑门了如指掌，但我们刚刚发现了这个很棒的解释视频，你可以用一种非常有趣的方式来教孩子——只用多米诺骨牌。

由[Numberphile]制作，我们的主机使用多米诺骨牌设置各种“电路”来解释所有各种逻辑门。有些模式设置起来有点棘手，因为你实际上必须根据多米诺骨牌的间距来设置时间——这让我们想知道到底有多少失误！

但不要相信我们的话，休息后很值得一看。
