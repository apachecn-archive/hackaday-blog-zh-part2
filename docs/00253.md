# 比特邦超级粉碎兄弟。

> 原文:[https://hack aday . com/2012/04/29/bit banking-super-smash-bros/](https://hackaday.com/2012/04/29/bitbanging-super-smash-bros/)

![](../Images/fb85dabbae07d3bf16b7f9f112cb6c5c.png "smash")

[Kyle]和他的几个同学正在总结普渡大学的微控制器接口课程，他们认为最好提交他们努力的结果。这是超级粉碎兄弟的一个版本，只需在微控制器上敲打引脚。

该项目的硬件基于飞思卡尔 9S12c32，这是 30 岁的 M68HC11 微控制器的更新版本。对于控制，这些人使用了装在 Altoids 盒子里的 Playstation 2 操纵杆和按钮，实际的控制台是由钉在一起的木条制成的，看起来像 Super Smash Bros 的板条箱。

有九个可玩的角色:皮卡丘、猎鹰队长、耀西、大金刚、马里奥、路易吉、林克、科比和福克斯。尽管这些角色只有四个像素高，但游戏看起来极具可玩性(至少当两个玩家不选择相同的角色时)。休息之后是超级粉碎兄弟:Bitbang 版的视频演示，以及展示主机和游戏的图片画廊。所有的代码都在 GitHub 上供你阅读。
