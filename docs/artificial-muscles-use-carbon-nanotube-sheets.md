# 人造肌肉使用碳纳米管薄片

> 原文：<https://hackaday.com/2014/05/17/artificial-muscles-use-carbon-nanotube-sheets/>

像空气一样轻，比钢更坚固，比橡胶更柔韧。听起来像是钢铁侠系列的下一部吗？[托尼·斯塔克]肯定会注意到这项迷人的技术。幸运的是，它不是来自好莱坞的工作室，而是来自德克萨斯大学纳米技术研究所的大厅。

纳米技术研究所的[Ray Baughman]教授和他的科学家团队通过在森林般的结构中生长碳纳米管来制造气凝胶片，从而开发出一种人造肌肉。想象一个垂直的竹林，每个竹茎代表一个碳纳米管。现在想象一下，单个的竹茎通过更小的水平线连接在一起。因此，如果你取出竹子并开始拉，线会拉动其他的，你会得到这个片状结构。

这些碳纳米管气凝胶片具有一些真正科幻小说般的特性。它们可以在 1600 摄氏度到接近绝对零度的温度下工作。如果你注入电荷，每个纳米管会相互排斥，使薄片的尺寸扩大到原来的 220%。你的肌肉大约有 20-40%会这样做。休息后留下来观看这些碳纳米管气凝胶片的制作和演示视频。
