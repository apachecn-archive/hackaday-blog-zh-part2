# 机器人剧场与其说是演员的，不如说是舞台工作人员的

> 原文:[https://hackaday . com/2013/07/17/robot-theater-not so-for-actors-as-stage hands/](https://hackaday.com/2013/07/17/robot-theater-isnt-so-much-for-the-actors-as-the-stagehands/)

![robot-theater](../Images/cc80724cde4bdaa3a88ebc18ccb66d15.png)

[Chris Rybitski]开发了这个低姿态机器人来帮助移动舞台上的布景。测试镜头显示，它动作敏捷，能够移动数百磅的货物。演示显示增加了一个木制平台，长度约为金属底盘的两倍，两端都有脚轮，以支撑额外的重量。它似乎可以承受几个乘客的重量而自由移动。

更换大型布景的巧妙系统长期以来一直使剧院成为黑客的天然滋生地。平衡转台、轨道系统等都是常见的地方。但是我们认为这有很大的潜力。现在电子设备看起来很复杂，因为有一个 Arduino 运行电机，它使用以太网屏蔽和 Linksys 无线路由器连接到 LAN。

我们认为他应该直接接入路由器的串口。如果他加载 DD-WRT 或 OpenWRT，他可以很容易地使遥控器成为一个网络界面。我们还想知道是否有可能让它成为一个线跟踪者，能够利用地板上的图案自动精确定位。
