# LED 标志照亮了天黑后的海滩

> 原文：<https://hackaday.com/2015/06/06/led-sign-brightens-up-the-beach-after-dark/>

[Warrior_Rocker's]一家为他们的海滨别墅买了一个别致的新标志。这个标志上有垂直拼写的“海滩”一词。它最初带有蓝色发光二极管来照亮每个字母。问题是 led 的光束很窄，会让房间另一边的人看不见。此外，没有办法改变发光二极管的颜色，这将增加有趣的因素。这就是为什么[勇士]决定用多色发光二极管升级标志。

在移除标志的纸板衬垫后，[Warrior]用锤子轻轻敲击棍子，移除了原来的 led。他决定用 WS2811 LED 像素取代原来的 LED。这些像素模块支持多种颜色，并且可单独寻址。这将允许各种各样的颜色和动画。像素被一种防风雨的树脂材料覆盖。【战士】用热风枪烘烤树脂，直到变脆。然后他用钳子和美工刀将它完全取出。最后，用一些热胶水将像素固定住。

[Warrior]找到了一个不到 10 美元的兼容射频遥控器，而不是从头开始制作一个遥控器。将 LED 控制器从其外壳中取出并焊接到 LED 串上。然后，它被热粘在一块纸板上，并被放入标志原来的电池盒中。请观看下面的视频进行演示。
