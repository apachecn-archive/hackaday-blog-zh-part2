# 麻省理工学院的学生将俄罗斯方块玩到了极致

> 原文:[https://hack aday . com/2012/04/23/MIT-students-take-Tetris-to-a-grand-scale/](https://hackaday.com/2012/04/23/mit-students-take-tetris-to-a-grand-scale/)

![](../Images/6b46dbf5c7c510124e29f03cbadf9b3e.png "mit-green-building-tetris")

小心，这个黑客可能会让人怀疑你在自己的计算机科学部门的乐趣。上周末，麻省理工学院的一群学生完成了一项大规模的黑客行动，他们将一栋建筑变成了俄罗斯方块游戏板。

这个建筑是麻省理工学院校园里的绿色建筑。它容纳了地球、大气和行星科学部门，但选择它是基于一侧窗户形成的网格的大小和规则性。该组织尚未提供太多细节，但休息后的视频显示了游戏的玩法和启动屏幕。建筑的中间部分被用作滚动字幕，在游戏碎片开始掉落之前显示单词“俄罗斯方块”。我们只是猜测(我们希望你能在评论区添加你的猜测)，但我们打赌他们组装了一套无线 RGB LED 灯，并在每个窗户的窗台上放一个。看起来确实有一些“死”像素，但这并没有减少整体效果的乐趣。

如果你没有自己的建筑可以玩，你应该小规模地玩，在角色显示器上玩俄罗斯方块。
