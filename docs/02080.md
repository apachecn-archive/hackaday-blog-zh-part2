# 激光充电发光显示器

> 原文:[https://hack aday . com/2012/11/09/laser-charged-gluming-display/](https://hackaday.com/2012/11/09/laser-charged-glowing-display/)

![](../Images/e5fe001c284abcf007b72986e7e1feb5.png "laser-phosphor-display")

这是我们见过的最好的发光显示器之一。目前[H]正把他的发明作为一个模糊的时钟，但它肯定能够显示任何信息。

该项目使用一轮发光纸作为展示面。这有一种在黑暗中发光的特性，可以使用明亮的光源充电。在这种情况下，使用紫外激光二极管。这也许是最好的可能来源，因为它的强度将允许非常快速的充电。这里的创新是使用第二个圆盘作为模板。仔细观察上图，你会发现激光二极管垂直安装在显示屏表面。一面镜子反射——我们相信会稍微扩散——激光点。然后，它通过黑色轮上的一个切口，该切口被成形为所需的字符。正如你在休息后的视频中看到的，这产生了一封清晰明亮的信。

将这个项目与像绘图仪一样移动二极管的[项目进行比较，我们认为你会同意这是一个更简单的实现，看起来仍然很棒！](http://howiem.com/wordpress/index.php/2012/11/09/hs-laser-dymo-clock/)
