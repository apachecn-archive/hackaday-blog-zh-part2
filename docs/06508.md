# 派对就绪迷你 LED 音量塔

> 原文:[https://hack aday . com/2014/07/10/party-ready-mini-led-volume-tower/](https://hackaday.com/2014/07/10/party-ready-mini-led-volume-tower/)

![Audio LED Light Tower](../Images/da846f8f1e1eb73a200c7caaca8d0814.png)

有许多非常酷的音乐视觉效果，但最好的是你自己制作的那种。[Ben's] [迷你 LED 音量塔](http://www.instructables.com/id/Mini-LED-Volume-Towers-VU-meters/)为您的音乐增添一些美丽的光彩。

[Ben]在 YouTube 上看到各种令人敬畏的立体声 LED 塔(也称为 VU 米)时，受到启发，创建了这个项目。我们甚至[推出了](http://hackaday.com/2013/10/03/vu-meter-prom-dress/)几个 VU 米，[一个最近推出的](http://hackaday.com/2014/07/07/vu-meter-record-player-lights-it-up/)。[Ben]讲述每个细节，包括如何测试您的电路(这是任何项目中非常重要的一部分)。示意图看似简单。它基于 LM3914 显示驱动芯片，采用简单的链式比较电路来控制音量条显示。你真正需要的是一台 3D 打印机来制作底座，你就可以建造这座令人敬畏的塔。

休息后，观看已完成的塔楼。接下来呢？如果能看到一个更大的显示频率量级的塔会很酷！
