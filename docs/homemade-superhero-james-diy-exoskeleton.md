# 自制超级英雄:[詹姆斯的] DIY 外骨骼

> 原文：<https://hackaday.com/2014/08/30/homemade-superhero-james-diy-exoskeleton/>

我们不仅仅是 Hackaday 的一群拿着打字机的猴子；只要时间允许，我们都会戴上黑客的帽子。或者，在 Hackaday 自己的[James Hobson]——又名[The hack Smith]——他穿上了这个光滑的外骨骼原型[，而不是](http://hackaday.io/project/1240-Pneumatic-Exoskeleton)，将自己变成了一个超级英雄。受电影 [*极乐世界*](http://en.wikipedia.org/wiki/Elysium_(film)) 外骨骼的启发，这个项目让【詹姆斯】向创造钢铁侠风格服装的更大目标更近了一步。

不过，就目前而言，外骨骼本身已经足够令人印象深刻了。该建筑由定制的穿孔钢管和气压缸组合而成，连接在某种背撑上。在演示视频中，[詹姆斯]凝视着固定在杠铃上的 170 磅的煤渣砖，尽管他并不轻量级，但当[詹姆斯]一次又一次地弯曲临时重量时，你可以从他的反应中立即看出外骨骼提供了多大的帮助。这还只是一半的压力。【詹姆斯】认为他可以打破 300 磅的举重纪录，如果他不先摔断腿的话。

有大量构建过程的幕后镜头，所以请确保在跳转后留下来观看大量视频，并查看[The hack Smith]网站了解更多他的项目。
