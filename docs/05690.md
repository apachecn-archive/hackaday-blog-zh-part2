# FRAM 发射台的 Web 界面

> 原文:[https://hack aday . com/2014/04/05/we B- interface-for-the-fram-launch pad/](https://hackaday.com/2014/04/05/web-interface-for-the-fram-launchpad/)

![webUILaunchpad](../Images/4a71c803160a3e439c7b0326b60901bc.png)物联网正在全力发展。加入物联网的第一步是显而易见的，在你的项目中加入一个网络界面。[Jaspreet]来信告诉我们他的项目，该项目为他基于 MSP430 的项目增加了一个[网络界面，使得向物联网添加任何项目变得容易。](https://github.com/jaspreetsingh009/webui_msp430)

如果你以前从未做过，创建一个网页界面可能会有点困难。这个项目通过使用一台运行 Linux 的专用计算机来处理所有与 web 相关的任务，使它变得很容易。LaunchPad 只需使用 USB 和 Python 与计算机接口，计算机托管网页并使用 [Node.js](http://nodejs.org/) 实时更新。结果是一个非常专业的界面，具有令人印象深刻的响应显示器，可以控制板载 led，从集成 ADC 读取模拟值，并传输加速度计数据。请务必在休息后观看它的运行！

我们可以看到这个项目被扩展到带有大量传感器的 Raspberry Pi 上运行。下一步你会添加什么样的网络界面？[家庭自动化](http://hackaday.com/2014/03/16/hacking-window-blinds-to-interface-with-home-automation-system/)？一个[气象站](http://hackaday.com/2013/08/21/ultrasonic-anemometer-for-an-absurdly-accurate-weather-station/)？[让我们知道](http://hackaday.com/contact-hack-a-day/)！
