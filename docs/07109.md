# 甜蜜的踏步机[杰里米]的石头和它的盒子

> 原文:[https://hack aday . com/2014/09/18/sweet-stepper-of-jeremys-rocks-with-its-box-out/](https://hackaday.com/2014/09/18/sweet-stepper-of-jeremys-rocks-out-with-its-box-out/)

受其他软驱管弦乐队的启发，[Jeremy]用步进电机谐振器构建了一个 Pi 驱动的 MIDI 音乐盒，[在 hackaday.io](https://hackaday.io/project/3033) 上概述了这一构建。

电机的控制来自 Iteaduino Mega 2560。音乐以 MIDI 文件开始，被处理成文本文件，并通过 Raspberry Pi 串行播放。他使用 K'NEX 乐器和 9g 伺服系统添加了打击乐器，我们认为这是一个不错的选择。它可以通过脂肪或从墙上获得动力，并且[杰里米]在保护中烘烤以防电池爆炸。正如他在休息后的巡演视频中解释的那样，盒子被夹在一张木桌上，以提供更丰富的声音。

[Jeremy]最喜欢的部分是封装，因为这是他第一次使用面板安装组件。留下来看一遍胆量和展示其音乐实力的第二个视频。
