# 孤独？给自己造一个象棋机器人！

> 原文:[https://hack aday . com/2015/03/15/lonely-build-yourself-a-chess-robot/](https://hackaday.com/2015/03/15/lonely-build-yourself-a-chess-robot/)

[Oriol Galceran]为他的期末项目构建了一个有趣的机器人象棋选手。它被称为 [ChessM8](http://hackaday.io/project/4709-chessm8) ，考虑到[Oriol]只有 17 岁，这是一个令人印象深刻的壮举！他使用的 Arduino Mega 通过 Python 脚本连接到主机。

人工智能可以是任何使用[通用国际象棋接口](http://en.wikipedia.org/wiki/Universal_Chess_Interface)协议的国际象棋引擎，[Oriol]指出他们中的大多数都是这样做的。我们以前在这里见过其他的[象棋机器人](http://hackaday.com/2013/05/09/voice-controlled-chess-robot/)，还有其他你可以[在你的墙上玩的](http://hackaday.com/2014/05/16/wall-mounted-chess-set-lets-you-stand-and-play/)和使用[谢妮管的](http://hackaday.com/2012/06/27/how-about-a-nice-game-of-nixie-chess/)。但是奥里奥尔的体型是最大的。

他说，棋盘下有一个簧片开关网络，用来检测棋子是否存在。了解他如何处理去抖问题，以及霍尔效应传感器是否是更好的选择，将会很有意思。请在评论中告诉我们你将如何检测棋子。
