# THP 四分之一决赛:低成本固态宇宙射线天文台

> 原文：<https://hackaday.com/2014/09/11/thp-quarterfinalist-low-cost-solid-state-cosmic-ray-observatory/>

有许多众包项目将来自世界各地的数据放到互联网上，追踪从[闪电](http://hackaday.com/2014/06/29/a-cloud-of-lightning-detectors/)到[飞机转发器](http://hackaday.com/2014/08/25/piaware-automated-airliner-tracking-on-the-raspberry-pi/)的一切。[aelias36]的 Hackaday 奖的参赛作品略有不同。[他正在追踪宇宙射线](http://hackaday.io/project/2418)，并希望将他的低成本硬件变成世界上最大的天文台。

宇宙射线是起源于太阳系之外的质子和其他原子核。它们以光速的很小一部分撞击地球大气层的最顶端，地球表面经常被宇宙射线产生的粒子喷射。探测这种粒子喷雾是所有地面宇宙射线观测站的基础，而且[aelias]已经想出了一种廉价的方法，将探测器放置在地球的每个角落。

解决方案是一个简单的 [PIN 二极管](http://en.wikipedia.org/wiki/PIN_diode)。运算放大器将二极管中产生的微小信号放大成微控制器可以使用的信号。添加一个 GPS 模块和一个以太网连接，这个简单的探测器可以向服务器发送时间，位置和粒子数，用众包数据创建一个巨大的天文台。

就宇宙射线探测器而言，aelias 正在研究的探测器并不出色；这里的重点是让大量数据进入现场，并将大量数据转化为高质量的数据。这是一个有趣的项目，也是我们在 Hackaday 奖中看到的唯一一个众包规模如此之大的项目。
