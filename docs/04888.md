# Fubarino 竞赛:一个用发光二极管雕刻的房间

> 原文：<https://hackaday.com/2013/12/24/fubarino-contest-a-sculpted-room-with-leds/>

[Sisam]和[Mclien]是一对父子团队，他们建造了这个雕塑般的房间，里面有看起来很有机的内置座位区和雕塑灯罩。当你有一个看起来很酷的房间时，你唯一的选择就是用 RGB LEDs 灯来填充它，碰巧他们的灯光控制器有一个很棒的 Hackaday 复活节彩蛋。

房间照明由一个[变化多端的 VU 屏蔽罩](http://macetech.com/store/index.php?main_page=product_info&products_id=11)、 [OctoBar LED 控制器](http://macetech.com/store/index.php?main_page=product_info&cPath=1&products_id=35)和几个[这些 RGB LED 模块](http://macetech.com/store/index.php?main_page=product_info&cPath=1&products_id=16)提供。对于一个 RGB LED 项目来说，这些都是非常标准的，但这次竞赛提交的作品真正闪耀的地方是所有房间灯光的控制器。它有三个红色、绿色和蓝色通道的滑块，每个灯光位置的结实的拨动开关，一个显示程序模式的 LCD，一个旋转开关和用于在存储的设置中循环的按钮。

每当灯光的颜色值设置为 Hackaday green， #00c100 时，该项目的复活节彩蛋就开始发挥作用。当这种情况发生时，黑客日 URL 显示在控制器的显示器上。

令人敬畏的工作，和一个看起来很酷的房间。我们不介意一个关于你如何雕刻它的教程，[Sisam]。
