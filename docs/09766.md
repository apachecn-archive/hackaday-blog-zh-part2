# 新版 Raspberry Pi 瘦客户端

> 原文:[https://hack aday . com/2015/08/16/new-version-of-raspberry-pi-thin-client/](https://hackaday.com/2015/08/16/new-version-of-raspberry-pi-thin-client/)

有趣的是，你有多少次把你的成熟的个人电脑作为另一台电脑的终端(这台电脑的功能通常不如终端电脑)。如果你需要的只是一个远程显示器和键盘，一个树莓派就足够了。一个新的 Pi 2 板会更好。

你可以推出自己的一套远程访问软件，但你不必这样做。[Gibbio]已经创建了一个名为 RPiTC 的瘦客户机映像，最近[发布了 1.4 版本](https://github.com/Gibbio/RPiTC)。该版本支持多种远程协议，包括 Microsoft Remote Desktop、Citrix、VMWare，甚至 X3270。

它支持 WiFi 和 VPN。我们有点失望，它似乎没有任何串行通信程序(以防我们想在旧的电传打字机中建立一个)。当然，它只是一个 Linux 系统，所以你可以安装任何你想要或需要的东西。

如果你不认为把一台电脑变成瘦客户机有什么大不了的，我们以前也讨论过另一个方向的。
