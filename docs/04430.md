# LED 魔术棒正好赶上万圣节！

> 原文:[https://hack aday . com/2013/10/29/led-magic-staff-just-in-time-for-Halloween/](https://hackaday.com/2013/10/29/led-magic-staff-just-in-time-for-halloween/)

[Dave]在过去的几个月里一直在非常努力地研究他的 Arduino 驱动、LED 照明的魔法杖，现在终于走到了一起。

他使用了 6 个 LED 灯带，每个灯带包含 55 个 LED——在全亮度下，工作人员可以在 5V 下吸收令人印象深刻的 20A 电流！为了给它供电，他给员工配备了 8 节镍氢 C 号电池(5000 毫安时，1.5V)。这相当于满功率运行约 15-20 分钟(255，255，255，LED 值)——为了应对这种情况，他通常运行一种持续时间更长的闪亮 LED 算法。此外，在最大功率下，它真的很刺眼。

五线谱由 Arduino Uno 控制，目前只有两种不同的模式:随机和全亮度。不过不用担心，他正计划添加一个声音传感器，将它变成一个均衡器，一个震动传感器，让它在行走时产生凉爽的涟漪效应，也许还有一些其他有趣的图案！
