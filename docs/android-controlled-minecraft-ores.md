# 安卓控制的《我的世界》矿石

> 原文：<https://hackaday.com/2013/03/27/android-controlled-minecraft-ores/>

[Ryan]有一个朋友的生日快到了，受到 12 岁孩子最喜欢的游戏的启发，他决定用 RGB LEDs 制作一个《我的世界》矿石积木。通过 Android 手机发出的命令，钻石块可以从钻石变成翡翠。

在几次失败的开始后，[Ryan]最终在谢菲尔德的一个类似黑客空间的组织 Acess Space 进行了矿石立方体激光切割。盒子由 3 毫米的 MDF 制成，窗户由磨砂丙烯酸激光切割而成，而立方体上的石头图案是一个巨大的定制贴纸。

随着繁琐的部分建设的方式，[瑞安]开始工作的电子。他使用了一个连接到几个非常大的 RGB LEDs 的 PIC，以及一个允许他将手机连接到矿石块的蓝牙模块。拨出正确的颜色需要一些工作，但最终，[Ryan]有了一个机器人控制的《我的世界》矿石块，能够在黄金、铁、钻石、祖母绿、青金石和红石之间变形。

休息之后，你可以看看[Ryan]的矿石块的视频。
