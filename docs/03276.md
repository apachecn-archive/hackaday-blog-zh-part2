# 磁性数控大理石迷宫

> 原文:[https://hackaday.com/2013/05/16/magnetic-cnc-marble-maze/](https://hackaday.com/2013/05/16/magnetic-cnc-marble-maze/)

![magnetic-cnc-marble-maze](../Images/71d45f27d2addb47c2d7deba13df72a5.png)

[Martin Raynsford]想出了一个方法，将一些知识偷偷放入一个有趣的包中。他做得非常好，测试对象甚至不知道他们只是在自学一点点数控编程。

上面的装置是一个大理石迷宫，但[马丁]只是在运动场上蚀刻了一个图案，而不是建造墙壁。大理石是一个滚珠轴承，[使用隐藏在下面的磁性数控龙门](http://msraynsford.blogspot.co.uk/2013/05/project-24-mazing-2.html)在迷宫中移动。哪里可以买到这种尺寸的滚珠轴承？如果你是马丁，你可以从 T2 的激光切割大金刚游戏中找到它们。

他在制造者集会上展示了装备。它以简单的命令作为运动的基本方向和单位。“玩家”(请记住，他们正在秘密学习一些东西，而不仅仅是玩游戏)输入一系列动作，如“N10，E10”，然后通过串行连接推送到 Arduino。它遵循这些命令，移动隐藏的磁铁，从而拖动滚珠轴承。这很简单，但是请在休息后观看该剪辑，我们认为您会同意步进电机的声音和球的运动对年轻人来说就像裂缝一样。
