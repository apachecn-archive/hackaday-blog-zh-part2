# 幻影 2.0 用 4024 个发光二极管照亮了沙漠

> 原文:[https://hack aday . com/2012/10/15/mirage-2-0-lights-up-the-desert-with-4024-LEDs/](https://hackaday.com/2012/10/15/mirage-2-0-lights-up-the-desert-with-4024-leds/)

![](../Images/99dac8d8dcd5485701e0636ec146039f.png "mirage2")

在变异车辆的燃烧人部门(DMV)登记一辆变异车辆是粗略的。要获准在夜间运行，古怪的滚动创作必须有一定程度的照明存在。这可以防止车辆融入到风景中。不幸的是，幻影 1.0 是专门为这一点而建造的，使用反射表面将货车变成半隐形的闪亮平板。甚至连电线、灯光舞池和聚光灯都无法安抚车管所。解决办法？将整个 friggen 车辆包裹在由 4000 个 led 组成的网[中](http://dustandmirrors.tumblr.com/about)！抓住那个官员！

硬件的大部分[是 Phillips display 的东西，数字 LED 灯具控制器用于解释 HDMI 数据，然后将颜色数据传输到寻址链。所有这些映射和寻址意味着整个设置就像一个 168×24 像素的监视器。发光二极管的分离链也允许机组人员操作车门和进出车辆。](http://dustandmirrors.tumblr.com/)

基本的汽车是基于隐藏天行者陆地飞车轮子的相同原理建造的，只是在这种情况下，想法是用聚酯薄膜和镜子覆盖整个汽车。这种镜面包裹的一个有趣的副作用是，沙漠灰尘的光泽有助于很好地反射周围的 LED 光，模糊像素颜色。这让我们不禁想拿起一桶聚酯薄膜来做一些间隔展示。
