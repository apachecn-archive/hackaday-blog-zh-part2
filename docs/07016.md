# 用磁场追踪足球

> 原文：<https://hackaday.com/2014/09/08/tracking-footballs-with-invisible-magnetic-fields/>

美国国家橄榄球联盟的官方足球是由芝加哥一家名为威尔逊体育用品的公司手工制作的。那里生产的足球长度通常在 11 到 11.5 英寸之间，平均重量在 14 到 15 盎司之间。最初，动物膀胱排列在外面，偶尔从猪的里面，给传统的美式足球一个长久以来的绰号“猪皮”现在，它们由牛皮或硫化橡胶制成，上面缝有鞋带，增加了重量。这导致椭圆形球体自然倾斜。这是通过在足球的另一边插入额外的重量来平衡它来解决的。了解了这一点，聪明的黑客会意识到平衡点是巧妙地添加一个像这个一样的[运动跟踪发射器的完美位置。这样做不仅可以跟踪球在球场上的位置，还可以跟踪它在 3D 空间中的精确位置！](http://rickettslab.org/research/magneto-track/)

因为每个足球都是独一无二的，所以每个足球之间都存在差异。这意味着在足球中嵌入电路只会略微修改设备，这是一件好事，因为体育狂热分子往往非常固执于技术是否应该影响比赛。只要加在气囊上的发射器和环形天线的重量差不超过一盎司左右，足球本身就不会受到太大影响。

该项目的研究是在迪士尼(ESPN 的所有者)的资助下，在北卡罗来纳州立大学和卡耐基梅隆大学开发和测试的。选择使用磁场而不是其他基于摄像头的球跟踪系统是因为它允许计算机在发生堆积时识别足球。与足球不同，NFL 中的足球通常是隐藏的。

现在的问题是，NFL 会接受这种类型的系统吗？“他们已经将即时回放整合到游戏中；截至 2014 赛季，各队都在场外配备了微软平板电脑，供教练和裁判用于比赛中间分析。然而，这款游戏仍然使用[1907 年](http://www.nytimes.com/2009/01/01/sports/football/01chains.html?pagewanted=all&_r=0)开始的同样古老的“棍子和链条”方法来测量下降。在高中或大学实施它将作为一个原型。从那里，研究人员可以在尝试全国整合之前尝试进入季前赛。但是粉丝会喜欢吗？它会从游戏中消失吗？由你决定。
