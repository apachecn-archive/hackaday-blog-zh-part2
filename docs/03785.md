# 令人难以置信的弓车床工作在摩洛哥

> 原文：<https://hackaday.com/2013/06/10/incredible-bow-lathe-work-in-morocco/>

[https://www.youtube.com/embed/wnv0DAR_gWA?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/wnv0DAR_gWA?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

弓形车床是一种相当古老而简单的装置。弓是用来来回旋转一块木头，而工具可以用来塑造它，就像现代车床。尽管木头在振荡而不是朝一个方向旋转，但结果非常平滑。

看着这个街头小贩展示他的弓车技术。我发现他的脚用得非常好，令人印象深刻。你可以看出他这样做已经很久了。当那枚戒指自由弹出时，我也感到惊喜，这是我没有预料到的。
