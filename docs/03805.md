# 自动机和木制齿轮

> 原文:[https://hackaday.com/2013/07/08/automata-and-wooden-gears/](https://hackaday.com/2013/07/08/automata-and-wooden-gears/)

[![mechanism](../Images/caf62262ebb62897d41567b7914c4cdd.png)T2】](http://hackaday.com/wp-content/uploads/2013/07/mechanism.jpg)

虽然我们每天接触的大多数动画机器——从钟表到汽车到电脑——都是由金属制成的，但用木头创造自动化物体是一门艺术。【掘北】[是这类发明](http://dugnorth.com/)的创造者，用木制齿轮、嵌齿和凸轮制作自动机。

[Dug]的发明与涡轮发动机相比很简单，但它们仍然保留了自己的艺术性。只需简单的木工工具，他就能创作出日常场景的动人片段，从狗对着鸟叫，到圣诞老人优雅地翱翔在平安夜的房子上。

下面，你会发现一个视频，是[Dug]的作品，“一个不受欢迎的晚餐客人”——一只自动化的狗对着一只木鸟吠叫。还有一段他在去年的世界创客大会上接受 Tested 的牛人采访的视频。
