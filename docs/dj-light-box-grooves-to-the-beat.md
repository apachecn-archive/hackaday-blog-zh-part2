# DJ 灯箱音乐节拍

> 原文：<https://hackaday.com/2015/02/04/dj-light-box-grooves-to-the-beat/>

在一次产品设计课上，[Oscar de la Hera]设计并制作了一个 LED 灯箱，[可以对音乐](http://www.oscardelahera.com/lightboxdj.html)做出反应——而且做起来很好看！

他小心翼翼地用橡木制作了一个单向镜面的盒子，里面是一个 6×6 的新像素矩阵。面板后面是一个 Arduino Uno，它使用一个 MSGEQ7 芯片和两个音频插孔来接收音频信号并创建一个灯光表演。当灯熄灭时，它看起来像一面别致的小镜子——但当你打开音乐时，它就变得活了。

如果你对它是如何制作的感到好奇，或者如果你想自己制作，在 Instructables 上有一个关于如何制作自己的[的完整教程——别忘了在休息后看看它的运行情况！](http://www.instructables.com/id/The-Light-Box/)
