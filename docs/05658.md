# 神奇的乒乓球机器人是假的

> 原文:[https://hack aday . com/2014/03/28/the-amazing-ping-pong-robot-was-fake/](https://hackaday.com/2014/03/28/the-amazing-ping-pong-robot-was-fake/)

嗯——你们是对的。事实证明，实际上是一对动画师愚弄了互联网。

不确定我们在说什么？上个月，[库卡机器人集团]发布了一段非常精美的视频，展示了一个工业机器人与这项运动的明显世界冠军打乒乓球——做得非常好，看起来很有趣，但不幸的是……也是假的。第一个[Kuka]视频出来几周后，一个叫[Ulf Hoffmann]的人发布了另一个视频，一个小型乒乓球机器人，[看起来**几乎**可行。](https://www.youtube.com/watch?v=imVNg9j7rvU)

正如我们的一些读者指出的那样:

> 对于伺服系统的大小和手臂结构来说，这些动作似乎不自然。~詹姆斯
> 
> CGI。正如其他人所指出的，当机器人从侧面展示时，手臂的影子消失了，即使它们是在其他镜头中添加的。~布兰登
> 
> 我的 cgi 密报是桌子下面的电缆。它会伸展而不是滑动。~ Aj
> 
> 注意它运行的是 Outlook Express 和 Internet Explorer——没有哪个有自尊的黑客/制造商会运行这些应用程序——哈哈。~冯斯基皮

还有一张显示 CGI 打嗝的 GIF 图 …真令人失望！不管怎样，正如 c 't Hacks 的菲利普·斯特芬所报道的那样，真相已经大白。事实证明，甚至[乌尔夫·霍夫曼]都不是真实的。这个精心制作的赝品是由一对动画师[Tobias Becker]和[Steffen Tron]炮制的——你知道吗，我们对此印象深刻。

两人计划今年成立一家代理公司，进行病毒式广告宣传——[Ulf]是一次试验，看看他们能做些什么。为了让它尽可能真实，他们创造了制造者，甚至从去年开始记录这个项目，给它增加一些真实性。不幸的是，当[库卡]机器人的视频出来时，他们不得不赶紧发布一些东西，借着成功的东风。

对于那些想知道他们实际上是如何做到的人来说，你是对的——完全 CGI。[Steffan]站在桌子后面击球——他们只是删除了他，并在机器人中制作了动画。至于讨厌的“伺服”噪音？它们实际上是通过手动转动 rklin 模型火车引擎制成的。
