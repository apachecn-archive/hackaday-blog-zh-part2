# 黑客日奖参赛作品:开放仿生学

> 原文：<https://hackaday.com/2015/06/17/hackaday-prize-entry-openbionics/>

在过去的几年里，3D 打印社区一直在寻找一种突破性的应用程序，用于生产塑料喷射品。社区偶然发现的最有趣的应用之一是修复术。

最近几年有很多人预热他们的 3D 打印机和激光切割机来制作假肢。对于[OpenBionics]的 Hackaday 奖参赛作品来说，他们正在建造一只成本不到 200 美元、重量不到 300 克、可以用 3D 打印机和激光切割机轻松制造的假手。

人手是地球上最复杂的末端执行器，模拟它的运动范围是一项艰巨的任务。尽管如此,[OpenBionics]团队仍在努力模仿具有三个自由度的拇指，在手上放置 144 种不同的抓握方式，并使他们的手具有柔软的指尖。

即使具备所有这些能力，[OpenBionic]的机器人手——包括马达和所有部件——也和普通人手差不多大。这太不可思议了，尤其是当你想到你的手的马达——肌肉——都在你的手臂上。

这个团队制作了一个视频来展示他们的手的能力。这有点不可思议，可以做任何事情，从举起咖啡杯到握笔。你可以看看下面的视频。
