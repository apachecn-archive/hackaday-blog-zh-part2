# 微型遥控 P-51D 野马提示秤重 3 克

> 原文:[https://hack aday . com/2014/11/11/tiny-RC-p-51d-mustang-tips-the-scales-at-3-grams/](https://hackaday.com/2014/11/11/tiny-rc-p-51d-mustang-tips-the-scales-at-3-grams/)

商用遥控飞机逐年变得越来越小。在 21 世纪初，手掌大小的飞机或直升机是许多遥控爱好者的梦想。今天，你可以在当地商场花大约 20 美元买到它们。然而，最小的模型仍然是由一群精英建模师建造的。重量只有 3 克，[[马丁·纽维尔的] P-51D 野马型号](https://www.youtube.com/watch?v=JSgRTPGYEQA)当然把他放在那一组。虽然 P-51 的 11.6 厘米翼展可能不会使它成为世界上最小的飞机，但它的许多功能使它令人难以置信。

野马是一个 8 频道的事情，有电梯，油门，方向舵，副翼，襟翼，导航灯，工作收缩，闪烁的加农炮灯。这是赖特，我们说收回，如在一个 3 克模型的可收回起落架。

野马的所有飞行表面都采用完全比例控制。然而，没有闭环伺服涉及。飞行表面使用磁致动器，由一个被一圈磁线包围的微小钕磁铁组成。我们不确定这些致动器的信号是直接 PWM 还是[Martin]在改变频率，但系统工作正常。收回使用热敏镍钛诺“肌肉线”和一个曲拐系统，以确保起飞后起落架收起并锁定，并在着陆前再次放下。

我们没有野马的任何飞行视频，但我们有一个更小的 1.2 克飞机(马丁)最近飞行的镜头。点击休息时间查看！
