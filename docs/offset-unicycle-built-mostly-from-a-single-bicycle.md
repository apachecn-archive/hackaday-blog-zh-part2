# 偏置独轮车主要由一辆自行车制成

> 原文：<https://hackaday.com/2014/12/28/offset-unicycle-built-mostly-from-a-single-bicycle/>

[娄的]朋友们都说，这将是不可能建立一个独轮车有偏移踏板。将踏板移到独轮车的前面会失去平衡，使使用者无法骑上它。[Lou] [证明他们错了](https://www.youtube.com/watch?v=_r9PlN2r_fc "offset unicycle")使用的大部分部件来自一辆捐赠的自行车。

捐赠的自行车被切割成一个更小的版本。踏板保持在原来的位置，并最终在骑手的前面。座位是向后移动的，这是这个造型的关键。让骑手的腿在前面，需要在后面有一个平衡。向后移动座椅可以相对轻松地完成这项工作。

为了防止轮毂空转，[卢]使用一些打包钢丝将链轮直接固定在轮辐上。他还必须移除脱轨器并缩短链条。所有这些使踏板直接连接到车轮上，允许更多的控制。视频很好地解释了快速高效的构建。它让任何人看起来都很容易尝试。当然，实际上骑独轮车是另一回事。
