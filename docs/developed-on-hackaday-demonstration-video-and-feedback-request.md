# 在 Hackaday 上开发:演示视频和反馈请求

几个月来，我们亲爱的 Hackaday 读者一直在关注 Mooltipass 密码管理员的冒险，今天我们终于发布了第一个视频。这是许多贡献者的劳动成果，一个原型，只是因为我们对开放硬件的动机和我们愿意花很多(所有！)把我们的业余时间花在一个很棒的项目上，这个项目可能足够好，可以被其他人收购。自从去年 12 月开始这个项目以来，我们已经取得了很大进展。

在上面嵌入的视频中，我们展示了我们平台的一些计划功能，而其他功能正在等待实施(我们的首要任务:输入 PIN 码……)。快速浏览一下我们的官方 GitHub 库,你会发现是什么成就了我们现在的地位。下一步是什么？

![](img/b9a7360793230ad449c3184f9c2d1d48.png)我们需要你的投入，这样我们才能想出最好的方法把 mool time 送到读者手中，因为我们的目标不是赚钱。测试版刚刚投入生产，两周后我将去深圳见我们的组装人员。如果将材料和制造考虑在内，我们预计每台设备的成本约为 80 美元，因此请花 3 秒钟时间回答下面的投票:(投票已结束)
