# 太阳能电池激光通信系统

> 原文：<https://hackaday.com/2015/07/02/solar-cell-laser-communication-system/>

忘了用一根绳子连起来的汤罐吧。现在有一种不依赖物理连接或无线电的无线通信方式。这是一个通信平台，[使用激光发送数据](http://rimstar.org/science_electronics_projects/laser_communicator_diy_homemade.htm)，它是以一种几乎任何人都可以建造的方式完成的。

这种发送信息的方法[并不完全是新的](https://en.m.wikipedia.org/wiki/Laser_communication_in_space)，但是这个项目是我们见过的最好的项目之一，使得它对于普通的修补者来说是可行的。一个标准的麦克风和音频放大器用于将信号发送到发射器，这只是一个典型的普通激光器，任何人都可以花几美元找到。几个发光二极管防止激光接收太多的功率，接收端的太阳能电池解码信息，并通过另一个放大器和扬声器输出。

当然，你需要视线来启动和运行这个通信系统，但只要你能做到这一点，就没有限制。如果你想尝试增加通信距离，你可以在周围找到[难以置信的强大激光，并且购买 1W 或更高的其他产品的限制出奇的少。你可以很容易地增加范围，但要小心不要让你的接收站(或任何动物，植物，建筑物等)着火！](http://hackaday.com/2013/05/24/3w-handheld-laser-raises-hope-for-a-real-lightsaber-someday/)
