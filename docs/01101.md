# 机器人垃圾桶可以接住你扔在它附近的任何东西

> 原文:[https://hack aday . com/2012/07/20/robot-trash-can-catches-any-you-throw-near-it/](https://hackaday.com/2012/07/20/robot-trash-can-catches-anything-you-throw-near-it/)

![](../Images/6b2e835c911d142c98aed1be101822fe.png "waste-basket-can-catch-better-than-anyone")

这个家伙要把蓝色的球扔到书架和废纸篓中间。当它到达那里[时，废物筐将已经移动到位以完美地接住球](http://www.youtube.com/watch?v=NqDTE6dHpJw)。它对你扔的任何东西都一样。

我们无法阅读字幕，但看起来这可能是作为广告的一部分制作的，广告在休息后的几秒钟内播放。从那里我们看到了一个适合箱底的移动装置的发展。它从一个旋转轮子开始，但是很快变得更加复杂。一旦低姿态的三轮汽车被铣削和组装，就该开始编写代码来翻译来自 Kinect 3D 摄像头的输入，并推断出捕捉垃圾的位置。最终的结果似乎完美地做到了这一点。
