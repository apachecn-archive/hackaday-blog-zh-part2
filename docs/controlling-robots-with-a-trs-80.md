# 用 TRS-80 控制机器人

> 原文：<https://hackaday.com/2012/05/15/controlling-robots-with-a-trs-80/>

![](img/7b1f3f66697135e735329677345c723c.png "TRS")

EZ-B 蓝牙机器人控制器的幕后策划者[DJ Sures]发来了一个非常有趣的作品，他用 1983 年的 TRS-80 电脑控制了一个机器人。

这个机器人是我们以前见过的可爱的瓦力。 WALL-E 通过蓝牙连接到装有 [EZ-Builder](http://www.ez-robot.com/EZ-Builder/) 硬件和软件包的台式电脑进行控制。

为了让 Trash-80 与 WALL-E 通话，[Sures]将一个微型蓝牙模块连接到 6402 UART 的 TX 引脚上。这是一个非常非常简单的修改，在第一批笔记本电脑中添加了蓝牙串行连接。在将 TRS-80 和瓦力-E 与运行 EZ-Builder 的电脑同步后，让机器人对 30 年前的键盘发出的铿锵声做出反应简直易如反掌。

有一个视频显示[DJ Sures]在休息后检查了他的构建，并展示了 WALL-E 随着一点 dubstep 而疯狂的精彩演示。休息之后请继续关注。
