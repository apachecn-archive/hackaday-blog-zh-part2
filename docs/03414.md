# 3D 打印手摇数字电脑

> 原文：<https://hackaday.com/2013/06/11/3d-printed-hand-cranked-digital-computer/>

看到这台数字计算机工作是多么美好的事情啊。[克里斯·芬顿]在设计和打印这台机械数字计算机方面做得非常出色。如果你对一个叫做你自己的[的感兴趣，看看他本周发布的源文件](http://www.thingiverse.com/thing:101068)。

[Chris']设计灵感来自对维多利亚时代机械织布机的一些研究。他调整了概念，制造了一个穿孔卡片阅读器，从三个孔的容量开始，发展到可以读取十个孔的设计。它提供的位数刚好够寻址上图所示的所有三个计数器。通过插入一张名片大小的穿孔卡给计算机编程，然后开动机器。下面的视频从远处展示了这一过程…希望他会发布一个后续视频，更近距离地观察每一件作品。

这不是他的第一台基本计算机器。看看去年的机电版。
