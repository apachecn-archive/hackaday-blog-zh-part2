# 爱普生投影仪 LED 模块

> 原文：<https://hackaday.com/2013/03/28/epson-projector-led-mod/>

更换投影仪灯泡会非常昂贵。有时候买一个全新的投影仪比买一个新灯泡更划算。[Shawn]最近在易贝的爱普生 EMP-S4 上发现了一个不错的交易，并决定冒险一试。他认为，除了卖家提到的那盏丢失的灯之外，它可能还能用。他的怀疑是正确的，[一个定制的 LED 模块之后](http://diytronics.blogspot.com/2013/03/epson-emp-s4-projector-led-mod.html)，他的投影仪启动了。

如果没有安装库存灯，投影仪会给出错误信息并自动关闭。所以，第一步是建立一个小的旁路。一旦解决了这个问题，[Shawn]安装了一个 30W 2000 流明的 led，并定制了一个旧的奔腾 CPU 散热器来降低 LED 的温度。他还连线了散热风扇与股票排气扇平行的好措施。光学透镜有助于聚焦光线，一些定制的线路可以像库存灯一样打开和关闭 LED。

最终，他的第一个实验成功了，但是 Shawn 想尝试一个 8000 流明 100 瓦的 LED 灯，让它和普通的灯一样亮。休息之后，请观看一小段视频。
