# GuruBrew 的 15 分钟 Windows 8 生存指南

> 原文:[https://hack aday . com/2013/09/30/gurubrews-15 分钟-windows-8-survival-guide/](https://hackaday.com/2013/09/30/gurubrews-15-minute-windows-8-survival-guide/)

[![GuruBrew Windows 8 Survival Guide](../Images/cd810370c24c51fe521e7a632fb910a9.png)T2】](http://hackaday.com/wp-content/uploads/2013/09/gurubrew-windows-8-survival-guide.jpg)

我们所有人都有一个共同点，那就是我们在业余生活中对操作系统的依赖。如果操作系统是 Windows，那么 Windows 8 可能会让你大吃一惊。许多读者是 Linux 或苹果用户，他们不在乎微软是否正在发布桌面导航的整个范式转变。然而，你可能会发现自己面对这个新的操作系统，如果你现在不熟悉的话，你会看起来像是在训练车轮上，考虑到 Windows 8 发布的计算机数量，这一天不可避免地会很快到来。

因此，如果你还没有用过 Windows 8，那么就来看看来自大师 Brew 技术展的 Windows 8 生存指南。这是对 Windows 8 桌面中新的触摸屏导航方法的出色概述，包括热点、魅力和磁贴等等。您还将学习使用鼠标和键盘的技巧。这不是一个关于使用 Windows 8 的完整教程，但你至少会知道如何导航，搜索应用程序，使用多个应用程序，并找到像任务管理器，控制面板，文件浏览器以及你熟悉的桌面这样的工具。
