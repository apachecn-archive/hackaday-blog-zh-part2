# TIQ 探测器比大多数更符合逻辑

> 原文：<https://hackaday.com/2014/05/29/tiq-probe-is-more-logical-than-most/>

[https://www.youtube.com/embed/XZz1W-LurlU?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/XZz1W-LurlU?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

我们的“舞蹈卡”上有[马克],人们可以在创客大会上找到。但在我们找到他之前，他撞上了我们，一手拿着 TIQ 探测器，另一手拿着测试盒。TIQ 是以传统逻辑探针的形式构思的，但由于内部的 Cypress PSoC 5LP，它比几十年前的工作台工具要聪明得多。当然，它可以告诉你 uC 引脚是 1 还是 0，但它也可以检测它正在探测的信号类型，并具有内置过压保护。

该工具的目的是在用数字万用表测量的事物和用合适的示波器测量的事物之间架起一座桥梁。我们认为他做了很好的工作，包括一些没有昂贵的工作台设备的人可能想要的东西。例如，您可以将其设置为在 i2c 等常见数据协议上触发，并将探头本身用作基本的脉冲发生器。

探针[的大部分细节可以在它的 Kickstarter 页面](https://www.kickstarter.com/projects/1866698905/tiq-probe-a-new-way-to-debug-electronics-projects)上找到(只剩下几天了)。你可能也会对[他的公司页面](http://innavatus.com/)感兴趣。我们对他搬运的试验台内部很好奇。[马克]是一个经常阅读的人，所以希望他能在下面留下一个评论，详细介绍那个黑盒。
