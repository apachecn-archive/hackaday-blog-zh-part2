# 胳肢我吧 Elmo…冻结在碳化物里

> 原文:[https://hack aday . com/2013/02/23/tickle-me-elmo-frozen-in-carbonite/](https://hackaday.com/2013/02/23/tickle-me-elmo-frozen-in-carbonite/)

[![elmo-in-carbonite](../Images/94524ebc8996ad4e18389dc9aecd6513.png)](http://hackaday.com/?attachment_id=95056) 我们在【HAD】喜欢任何把儿童玩具和科幻技术结合起来的黑客，所以在看到[咯吱-我-Elmo《冰雪奇缘》在我们的书中绝对是一个胜利。这也是加入你当地黑客空间的一个很好的理由，或者只是和一些志同道合的朋友聚一聚。这个想法来自黑客空间](http://toddblatt.blogspot.com/2013/02/tickle-me-elmo-in-carbonite.html "Tickle-Me-Elmo in Carbonite")[【巴尔的摩节点】](http://baltimorenode.org/ "Baltimore Node Hackerspace")的一次即兴头脑风暴，或“谈论疯狂想法会议”。

幸运的是,[托德]可以获得所有必要的工具来实现这个“疯狂的想法”。一个[商店机器人]被用来切割盒子，侧板在 Thingiverse 上的这些文件的帮助下被 3D 打印出来。为了进行处理，使用 Arduino 编程的[ATtiny85]被用于为该项目供电。

没有提到[托德]是否愿意放弃他的作品，但是，我们猜测他不会讨价还价。他不会轻易放弃他最喜欢的装饰。
