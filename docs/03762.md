# 黑客感知器，智能袜子

> 原文:[https://hack aday . com/2013/08/10/hacking-sensoria-the-smart-sock/](https://hackaday.com/2013/08/10/hacking-sensoria-the-smart-sock/)

![sensorSock](../Images/c104c162c779f44b4163adc45e56192f.png)

传感器与袜子相结合的产品“Sensoria”暂时不会上架，但[Andrew]设法搞到了一个 Sensoria 游戏软件开发工具包，并已经黑进智能袜子来控制一架廉价的玩具直升机。我们很少看到如此粗糙却又明显有效的项目。这只袜子通过它的配套设备——蓝牙脚链——向 LabVIEW 发送数据。LabVIEW 随后与一个连接的 Arduino 对话来操纵一个伺服系统，这个伺服系统[Andrew]只是…用胶带粘在直升机的控制器上。结果:一个快速和肮脏的黑客证明了惊人的直觉，提供加速器式的脚控制来驱动油门。休息之后，看看[安德鲁]惩罚他的直升机的视频。

这是我们看到的第一个针对 Sensoria 的黑客攻击，它仍然在 Indigogo 的众包阶段[。他们已经达到了他们的筹资目标，但仍有一些 SDK 无人认领。你可以在下面观看袜子传感器点亮实时热图的官方视频。](http://www.indiegogo.com/projects/sensoria-smart-sock-fitness-tracker)

如果你错过了，看看 AsTeRICS 项目的由颈部肌肉控制的直升机。
