# 乐高外骨骼控制环太平洋机器人

> 原文:[https://hack aday . com/2015/08/09/Lego-exoskeleton-controls-Pacific-rim-robot/](https://hackaday.com/2015/08/09/lego-exoskeleton-controls-pacific-rim-robot/)

尽管《环太平洋》令人捧腹，但它仍然是一个令人敬畏的概念。巨型机器人战斗服，与外星人一决雌雄。好吧，看起来这并不像我们最初想象的那样牵强。制造商[Danny Benedettelli]刚刚发布了一个他自己的乐高外骨骼套装的视频，穿上它可以用来控制桌面大小的独眼巨人机器人。你可能记得【丹尼】是[乐高头脑风暴 EV3 图书馆](http://makezine.com/review/review-the-lego-mindstorms-ev3-laboratory/)的作者，

独眼巨人机器人(也是他的设计)最初是在四年前使用乐高 Mindstorms NXT 系统和运行定制应用程序的安卓手机制造的。独眼巨人在这次演示中升级了一点。现在，它通过蓝牙和 Arduino 与(丹尼的)遥测服进行通信。

相对来说，这个系统相当简单。乐高外骨骼在每个关节上都有电位计，可以映射到机器人的自由度。当一个电位计旋转时，相关的机器人关节会模仿它。简单吧？
