# Fubarino 竞赛:VFD 时钟

> 原文：<https://hackaday.com/2013/12/13/fubarino-contest-vfd-clock/>

![hackad](img/8a498efcb1ce79c9200da46ab01c45d7.png)

我们的 Fubarino 竞赛的参赛作品正在缓慢而稳定地到来。[James]已经有了一个很棒的 VFD 时钟，并认为在他的项目中添加一个 Hackaday 复活节彩蛋会很简单。

[James'] clock 基于 TI Stellaris LaunchPad，带有六个漂亮的七段 VFD 显示管。时钟的时间由 DS1307 RTC 芯片控制，Stellaris 控制的小型开关模式电源将电子管的功率从 5 伏提高到 50 伏。显像管由 Max6921 VFD 驱动芯片控制。

这个项目的复活节彩蛋——显示黑客日网址——只有当你打开时钟，秒针显示 37 时才会出现。对于复活节彩蛋来说，这是非常微妙的，也是我们喜欢的方式。

[James]项目的所有代码和电子管钟外壳的设计都在 GitHub 上。这真是一个了不起的项目，也是[詹姆斯]为自己赢得奖金的一个好方法。
