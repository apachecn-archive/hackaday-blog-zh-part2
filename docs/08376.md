# 一个更小的自制 Amstrad

> 原文:[https://hackaday.com/2015/02/22/a-smaller-homebrew-amstrad/](https://hackaday.com/2015/02/22/a-smaller-homebrew-amstrad/)

虽然它们在美国不是很受欢迎，但 Amstrad CPC 464 和 CPC 6128 在欧洲非常受欢迎。[Zaxon]喜欢他的' 464，并且为了一点学习经验-以及 Amstrad 占用大量桌面空间的事实-[决定克隆他最喜欢的计算机](http://www.elektroda.pl/rtvforum/topic2992332.html)(。pl， [Google translatrix](http://translate.google.com/translate?hl=en&sl=pl&u=http://www.elektroda.pl/rtvforum/topic2992332.html) 。

克隆体最初是最初 Amstrad CPC 464 的简单示意图，但最初使用的部件需要一些现代的等价物。尽管如此，大部分旧芯片仍保留在克隆体中；原来的日立 HD46505 CRT 控制器依然存在，原来的 DRAM 芯片和老式 Z80 CPU 也是如此。

添加了一些现代设施，包括 PS/2 键盘的接口和一个磁盘，它比原始的盒式驱动器或奇怪的 3.5 英寸磁盘有很大改进:一个模块上的磁盘，或者基本上是一个奇怪形状的 CompactFlash 卡，可以直接插入主板的 IDE 插座。它们通常在拆卸旧的瘦客户机时出现，但是在逆向计算项目中使用它们是一个伟大的想法。
