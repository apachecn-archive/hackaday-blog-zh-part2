# 西瓜空气船

> 原文:[https://hackaday.com/2012/12/17/watermelon-air-boat/](https://hackaday.com/2012/12/17/watermelon-air-boat/)

![watermelon-air-boat](../Images/dd5328b14cd86a4c07af184ff0617d73.png)

我们认为，当其他人都在驾驶帆船时，如果你开着西瓜，你会在中央公园吸引一些目光。由于[启动电子设备]投入的工作，这个[西瓜既适合航海，又可以无线电控制。](http://startingelectronics.com/articles/radio-controlled-watermelon/)

我们使用了这张图片，因为它向您展示了船体内部的情况，但您不会想错过休息后在剪辑中绕着地上游泳池行驶的事情。中空的外壳很有浮力，在加上螺旋桨的情况下保持漂浮和直立没有问题。遥控飞机套件的零件被安装在一个木制脚手架上。这提供了大量的推力与伺服电机移动转动方向控制的道具。没有匕首板，所以飞船对转弯的反应有点慢。但是你期望一个浮瓜能有多灵敏呢？
