# MSP430F5529 LaunchPad 上的多任务处理

> 原文：<https://hackaday.com/2014/06/23/multitasking-on-the-msp430f5529-launchpad/>

到底什么是多任务、调度和上下文切换？对于那些有兴趣了解操作系统如何工作的人来说，这是一个很好的问题，即使是小型实时操作系统(RTOS)。[Jeffrey]也有同样的问题，所以他为 MSP430F5529 LaunchPad 构建了一个[多任务调度器。](http://jeffrey.co.in/blog/2014/06/multitask-scheduler-for-msp430f5529-launchpad/)

在嵌入式世界中，这些主题是最难理解的。选择一个有助于你理解棘手话题的项目是一种很好的学习方式，而且会非常有益。在他的文章中，[Jeffrey]讲述了所有这些东西是如何工作的，以及如何在 MSP430 上实现它们。总的来说，这是一个伟大的阅读和非常有益的。想了解更多关于 RTOS 的信息，请查阅《自由人》一书中的几个章节。休息之后，请务必观看他的代码。

[Jeffery]很好地将他的所有代码作为开源代码发布，所以一定要在 GitHub 上查看他的[库。“请随意使用并了解更多信息。我已经使代码不言自明。好好享受！”](https://github.com/jeffreyantony/Scheduler-MSP430F5529)
