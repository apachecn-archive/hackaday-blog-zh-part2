# 使用 3D 打印吹雪机自动化冬季

> 原文：<https://hackaday.com/2015/02/14/automate-winter-with-a-3d-printed-snowblower/>

遥控车不仅仅是孩子们的。事实上，你可以得到一些非常酷的迷你多功能车。事实上，你甚至可以买一辆迷你履带式吹雪车！但是[The_Great_Moo]对他的阔嘴猴腮暴雪 SR 的性能相当失望，所以他做了任何有自尊的黑客都会做的事情——他重新设计了整个该死的东西，并 3D 打印了它。

从头开始重新设计的美妙之处在于，你可以专门为 3D 打印设计它(当然，除非你想批量生产它！)，所以【The_Great_Moo】不慌不忙，脑子里装的都是图层强度的零件。大的部分以 0.4mm 的分辨率打印，更精细的部分；像齿轮和轴，以 0.2 毫米的分辨率打印。他用他的达芬奇 1.0 打印机打印了所有的东西，而且显然只用了 40 个小时！

除了螺栓和螺母，一切都是 3D 打印的——甚至是正时皮带齿轮和变速箱！但真正的问题是…真的能吹雪吗？你得看视频才能知道。
