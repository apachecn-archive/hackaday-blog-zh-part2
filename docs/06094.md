# 创客集市上的青春毁灭

谁小时候拆东西没惹过麻烦？2014 年创客大会上的 TakeItApart 展位是我最喜欢的展位之一。它让任何人(尤其是孩子)拿起一件要回收的电子产品，打开它，看看里面有什么。好消息是，你不需要能够把它重新组合起来，因为它无论如何都会被碾碎作为它的组成材料。

看着一个 7 岁的孩子用一字螺丝刀摆弄随身听收音机，有一种宣泄的感觉(那些塑料盒比你想象的更结实)。我问是否有人在这个过程中成功切开了他们的手[回到未来的风格](http://youtu.be/QzklMXES1BU?t=1m36s)，谢天谢地，答案是否定的。但是至少有一个“免费日托”的例子，父母们离开了——在 MF 有很多分心的事情——这让他们的后代非常懊恼。

看到这让我想起了最近对[Bunnie Huang] 的采访，他在采访中提到了他小时候从 Apple II 的插座中取出芯片的事情。他会把它们拉回来，再放回去，看看会有什么效果。哈！如果你有类似的童年经历可以分享，我们很乐意在评论中听到。如果你只是想看一堆东西的内脏，那就去[武藏部](http://takeitapart.com)。
