# 基于 Arduino 的模块化红外温度计

> 原文:[https://hack aday . com/2014/03/20/modular-arduino-based-infrared-thermometer/](https://hackaday.com/2014/03/20/modular-arduino-based-infrared-thermometer/)

![IRTemperature](../Images/7dff191938aba7ca781c83801e95b7f3.png)

[Brian]一开始就有一个清晰简洁的目标，“让一个普通人将一个可听见的音调与一个红外无接触温度计的温度联系起来。”带着他最新的项目，埃斯佩里。IRBud ，他达到了这个目的。

[Brian 的]帖子中我们最喜欢的部分之一是他的 BOM。能够很容易地看到红外温度传感器在 [DigiKey](http://www.digikey.com/product-search/en?x=0&y=0&lang=en&site=us&KeyWords=MLX90614ESF-ACF-000-TU) 的价格是 26 美元，这对读者有难以置信的帮助。选择这种特定的传感器是因为其他人已经[成功地将其与 Arduino](http://bildr.org/2011/02/mlx90614-arduino/) 接口。不必重新发明轮子是件好事！为了制作，[Brian]决定将红外温度传感器连接到一根重新设计的柔性 iPhone 耳机电线上。使用耳机插座连接传感器和扬声器后，实际设备相当模块化。在行动中听到这个东西很酷，听起来几乎像老派的 GameBoy 音乐！休息之后来看看。

你在你的项目中使用过红外温度传感器吗？请告诉我们。
