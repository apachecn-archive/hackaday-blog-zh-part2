# 四轴飞行器的奇特遥测控制显示器

> 原文:[https://hack aday . com/2012/05/07/fancy-telemetry-control-display-for-a-quad copter/](https://hackaday.com/2012/05/07/fancy-telemetry-control-display-for-a-quadcopter/)

![](../Images/01cada1c644e96f0441dadbcf1ed6d8c.png "fancy-telemetry-system-for-quadcopter")

我们见过的大多数四轴飞行器项目都使用基于操纵杆的控制系统。这可以让你像遥控飞行器一样飞来飞去。但是,[Saulius]正在通过[提取和显示遥测数据](http://lukse.lt/uzrasai/2012-05-carambola-powered-quadcoper/)来增强他的控制系统。它并没有真正改变飞行器的控制方式，但它让飞行器漫游得更远，因为操作员可以观看计算机屏幕，并放弃四轴飞行器在视线范围内的需要。

一个杨桃板(也是在这个气象站项目中使用的[)用于提供连接。这是基于 WiFi 的，这有助于我们了解它可以行驶的范围。四轴飞行器携带一个摄像头，显示在上图的右下角方框中。还有一个人工地平线，以及显示遥测数据的反馈刻度盘。](http://hackaday.com/2011/10/21/extend-your-personal-weather-stations-reporting-capabilities/)

看起来在这两个仪表板小部件之间有一个卫星视图。我们现在没有看到任何东西出现，但有可能这意味着覆盖一个基于 GPS 数据的飞机位置虚拟标记。最后一部分其实只是猜测。赶上跳伞后的 80 秒试飞。
