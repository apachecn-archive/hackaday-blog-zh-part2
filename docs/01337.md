# 没有系绳的乐高遥控潜水器

> 原文:[https://hackaday.com/2012/08/14/lego-rov-without-a-tether/](https://hackaday.com/2012/08/14/lego-rov-without-a-tether/)

![](../Images/de6f096f0f89d7a9f163216e3c6fe7a4.png "lego-rov-without-a-tether")

[Brane] [用乐高 mindstorm 零件](http://dexterindustries.com/blog/2012/07/05/underwater-robot-with-the-nxtbee_lego_mindstorms_nxt/)造了一个水下机器人。仔细观察这张图片，您应该会发现少了一些东西。通常将水下机器人的电源和控制线连接到水面的系绳不见了。这是一个无线解决方案，让他使用 Xbox 控制器控制设备。

休息后的视频显示了大约五分钟的试驾镜头。[Brane]有一个大鱼缸，他可以在里面测试这个东西。因为他把它作为他在大学的高级工程项目放在一起，这很可能是学校的一个测试设施。下面是我们对硬件的一点了解:它使用 NXT Mindstorm 部件来控制电机，带有一个密封的电池室。连接由 XBee 模块和 NXT 适配器板提供，称为 NXTBee。自带 XBee 模块的笔记本电脑构成了通信的另一端。现在[Brane]使用一个连接到笔记本电脑的 Xbox 控制器，但通过直接将 XBee 和控制器组装在一起，可以很容易地构建一个独立的设备。
