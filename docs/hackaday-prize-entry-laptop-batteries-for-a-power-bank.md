# Hackaday 奖参赛作品:一个电源库的笔记本电池

> 原文：<https://hackaday.com/2015/06/30/hackaday-prize-entry-laptop-batteries-for-a-power-bank/>

USB 电源组——可以给你的手机或平板电脑充电的巨大电池——如今无处不在。你可以在加油站或者你最喜欢的网上零售商那里买到任何你想要的东西。这些电源组中的大多数都有一个巨大的缺点；他们需要通过 USB 充电。对于 10，000 毫安的电池，这需要一段时间。

我们已经有了容量巨大的电池，能够快速充电，而且从一些易贝的拍卖来看，可以以很低的价格买到。[Kumar] [正在开发一种利用这些电池及其内部电子设备的设备](https://hackaday.io/project/5770-smart-battery-makes-a-smarter-power-pack),以构建一个更智能的电源库。

现在，[Kumar]正在使用戴尔 Latitude D5xx/D6xx 替换电池，他可以轻松地拿起这些电池。这些电池有一个 SMBus 接口，通过一个低功耗 ARM 微控制器和一个 [TI BQ24725a](http://www.ti.com/product/bq24725A) ，他拥有高效安全地为这些电池充电所需的一切。

[Kumar]说他正在为他的项目寻找一些社区建议和功能请求。如果你有，一定要把它们放在他的项目页面上。
