# 与[Veronical Belmont]共度一天的事实或虚构时光

与来自 Hackaday 的客人[我]一起欣赏[Veronica Belmont]的一集有趣的“事实或虚构”节目。“事实还是虚构”节目通常会选择一些流行的话题，并与能够阐明该话题的专家进行交谈。他们有各种各样的聪明人，我也是。如果你看几集，你会看到她倾向于让人们谈论一些科学，但不可避免地转向“我们真的能做到吗？”，这往往会引起被采访者尴尬而又有点幽默的回应，因为他们谈论的大多数事情都很古怪，比如传送门枪。我喜欢关于火星生活的那部，尤其是当她问这位先生电影中火星人的刻画有多准确时，就在他解释说我们在看微观事物之后。

另一方面，这是一个非常非常小的世界。几年前，我在 CES 上偶遇[Veronica],我们发现她和她的丈夫都在[Phil Torrone]刚刚开始的时候在同一个办公室工作。
