# Nebulophone 微控制器合成器项目听起来很棒

> 原文:[https://hack aday . com/2012/10/17/nebulo phone-微控制器-合成器-项目-声音-棒极了/](https://hackaday.com/2012/10/17/nebulophone-microcontroller-synthesizer-project-sounds-great/)

![](../Images/b9d884c5a476e2a4af9bf0b20a4d853d.png "nebulophone-buy-it-or-build-it")

从视频判断(休息后发现), Nebulophone 是我们见过的最好的 DIY 合成器之一。尤其是考虑到硬件设计的简单性。它使用 AVR 芯片和运算放大器。其余的零件只是一些廉价的组件。

该设备是由 Bleep 实验室开发的，他们出售左边的合成器套件。但由于这是一个开源项目，你可以按照他们的设计来制作你自己的，这就是[BlinkyBlinky]对右边的[所做的。](http://www.instructables.com/id/DIY-Arduino-Nebulophone-Synth)

ATmega328 驱动该设备，这是 Arduino Duemilanove 中经常使用的芯片。键盘是一组连接到微控制器的迹线。这些是套件 PCB 上的镀锡焊盘，但 DIY 版本只是使用一些粘合铜箔和焊接到其上的跳线。琴键由一个探针演奏，探针产生电连接，这是这些[唱针式设计](http://hackaday.com/2010/09/23/analog-stylophone/)的常见做法。有可能你手头上有让这一切发生的一切，所以在下一个让每个人都有点疯狂的寒冷的冬天的周末记住它。
