# 这张 LED 餐桌将整个房间紧密联系在一起

> 原文:[https://hack aday . com/2012/04/13/this-led-table-really-ties-the-room-together/](https://hackaday.com/2012/04/13/this-led-table-really-ties-the-room-together/)

![led-matrix-table](../Images/264a45c8324194c4eeeeb14fcb8e3d1d.png "led-matrix-table")

除了四旋翼飞行器和便携式游戏机，我们永远不会厌倦看到的黑客之一是 LED 矩阵表。[Christian Enchelmaier]来信分享了他对曾经流行的像素化家具的看法，[我们认为它表现得非常好](http://chriative.com/blog/projects/led-table/) ( [翻译](http://translate.google.com/translate?sl=de&tl=en&js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&u=http%3A%2F%2Fchriative.com%2Fblog%2Fprojects%2Fled-table%2F))。

[Christian]第一次外出时没有选择全尺寸的咖啡桌，而是选择了小尺寸的搁脚凳/床头柜。他使用 RGB LEDs 构建了一个 16×16 的矩阵，将每个 led 封装在自己的“像素”中，这在这些构建中很常见。[Christian]使用 Atmega 128 运行节目，显示当前时间和日期、温度、音乐可视化、游戏、图像以及短视频。他还在桌子上安装了红外接收器，这样他就可以从远处控制桌子的显示。

到目前为止，[Christian]还没有任何关于牌桌的视频，但是在他的构建日志中有大量的图片可以让你忙起来。
