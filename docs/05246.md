# Pi 供电的 IP 无线电

> 原文:[https://hackaday.com/2014/02/06/pi-powered-radio-over-ip/](https://hackaday.com/2014/02/06/pi-powered-radio-over-ip/)

![Pi](../Images/143ab4d4332d7544c4abeb8e3a19c697.png)

将一个树莓皮连接到一个小而便宜的手持收音机上，让任何人在他家几英里之内都可以连接到世界各地的业余无线电操作员。

对于硬件，[KP4TR]正在使用一个 Raspi，一个宝丰 BF-888s 400 MHz–470 MHz 对讲机无线电，一个 USB 声卡，以及一对用于 5V 和 3.7V 线路的变压器。所有这些都藏在一个令人印象深刻的复古外观的盘子和支架外壳中，配有橡子螺母和一个连接到 Raspi GPIO 的 RGB LED，以指示无线电是在发射还是在接收。

使用的软件是[SVXLink](http://sourceforge.net/apps/trac/svxlink/wiki/SvxLink),[echo link](http://www.echolink.org/)软件的 Linux 端口。这个应用程序允许全世界的火腿通过互联网连接到非常远的收音机。
