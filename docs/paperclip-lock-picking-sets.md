# 回形针锁撬锁套装

> 原文：<https://hackaday.com/2014/08/06/paperclip-lock-picking-sets/>

开锁已经成为全世界黑客的标志性技能，并定期在黑客空间和创客集市上教授。但是很多时候，这些布景已经在网上的某个地方制作或购买好了。然而，[Sean]已经在本文结尾嵌入的视频中演示了如何用普通回形针创建一个开锁工具。Wikihow 也有关于如何建造它们的令人敬畏的说明。

最棒的是，这些拨片的材料很容易找到。还有其他方法可以一起打造一套。例如，可以使用街道清扫车的刷毛。电工金属带也是一种很好的材料，但这些回形针组是迄今为止最容易使用的。几乎任何有办公文具用品的地方都会有成堆的这些小金属夹子。

但是它们的效果如何呢？你以前做过回形针开锁套装吗？
