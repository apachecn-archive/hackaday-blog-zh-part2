# Android CNC 控制器

> 原文:[https://hackaday.com/2012/06/08/android-cnc-controller/](https://hackaday.com/2012/06/08/android-cnc-controller/)

![](../Images/bc4bbc5e353a988adcc561cc1fcd6532.png "android-cnc-controller")

[Matt]自豪地拥有一台基于 JGRO 的 CNC 路由器，他一直在研究一种不用电脑就能控制它的方法。他想出的是用这个安卓平板电脑驱动数控机床的方法。

黑客的一大部分是他使用的数控控制器。 [TinyG](http://www.synthetos.com/wiki/index.php?title=Projects:TinyG) 是一种可以通过 USB 接收命令并将其转换为多达六个轴的指令的板。在休息后的视频中[Matt]展示了一种直接 USB 连接作为控制方法。这是我们最感兴趣的部分，但是该系统也可以在计算机的帮助下通过网络运行，向 TinyG 提供命令。第二种方法意味着 Android 控制器将是无线的。

三个库托管[Matt]正在使用的代码。从演示来看，Android 应用程序并不缺少功能。
