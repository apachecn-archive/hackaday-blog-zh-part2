# 外在动机:一个开放的、模块化的效果踏板

> 原文：<https://hackaday.com/2014/09/04/extrinsic-motivation-an-open-modular-effects-pedal/>

微控制器和 Arduinos 很酷，但只在数字领域交易确实有其局限性。事实上，大多数你心目中的电子英雄并不是通过数字输出上闪烁的引脚开始他们的电子职业生涯的；他们只在模拟领域使用收音机，是的，还有吉他效果踏板。

[Josh]的 Hackaday 奖参赛作品是整个竞赛中最相似的项目。这是一个开源的效果踏板，它利用了历史上最流行的踏板的模块化设计。

将电吉他转变为听觉实验的许多著名电路都是基于小电路模块，封装并重新封装，直到获得所需的音调。[Josh]想把这些模块分别打包在不同的板子上，具体来说就是 shields，虽然没有使用 Arduino，所以可以创造任何声音。

[Josh]已经做了一些研究来决定克隆什么电路和电路模块。任何一个有踏板的人都应该对这个列表相当熟悉——电子管尖叫者、绒毛脸、大手套和 90 年代阶段在列表的最前面。他可能不会得到复杂的数字效果，如音高移位器和数字延迟，但它仍然是一个伟大的实验项目。
