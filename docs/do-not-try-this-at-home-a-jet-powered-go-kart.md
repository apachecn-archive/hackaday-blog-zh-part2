# 不要在家里尝试这个:一辆喷气式卡丁车

> 原文：<https://hackaday.com/2015/05/17/do-not-try-this-at-home-a-jet-powered-go-kart/>

[柯林·福尔泽]又开始了。这一次，他建造了一辆由喷气发动机驱动的怪异卡丁车。

如果你不熟悉 Furze 先生]，他对制造高速车辆并不陌生，比如世界上最快的婴儿车。他还对脉冲喷气发动机有点着迷。他甚至用卫生纸架做了一个[。](http://hackaday.com/2013/10/05/please-dont-build-a-jet-engine-from-a-toilet-paper-holder/)他曾是一名水管工——但现在他是 YouTube 上最好的疯狂科学家创作者之一。我们只是希望他不要太快死于他的极限项目，因为他的安全领带可能救不了他！

这个月的项目也不例外——他把他曾经在法国放屁的巨型脉冲喷气发动机绑在一辆微型卡丁车的框架上。“正如你所见，喷气机与卡丁车的比例相当不错”。不是开玩笑——引擎的长度必须是卡丁车车架的 2.5 倍！

休息后留下来看他冒险自娱自乐。

他的项目最好的部分是，他也做了很棒的建造日志来配合它们，看着它们总是一种享受。他已经找到了用捷径和高能音乐来缩短注意力持续时间的关键。这使得建造任何东西都很有趣，包括像金属支架这样无聊的东西！
