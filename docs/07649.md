# 中欧计算机收藏

> 原文:[https://hack aday . com/2014/11/24/central-European-computer-collecting/](https://hackaday.com/2014/11/24/central-european-computer-collecting/)

在 Hackaday 短暂的捷克之旅中，我们有幸遇到了一个人，他最近在 Hackaday 上展示了他的一个项目。这是[Martin]为 8 位 CPU 开发的多目标 IDE，完全用 JavaScript 编写，是适用于 6502、6800、6809、Z80、8080 和 8085 的完整开发套件。[马丁]很友好地坐下来，给我们独家新闻，为什么他对旧电脑感兴趣，为什么他开发了他的 8 位 IDE 项目， [ASM80](http://www.uelectronics.info/2014/11/15/an-introduction-to-asm80/) 。

[马丁]成长在计算机杂志的时代，最初想建造自己的计算机。这个计划没有成功，但他的父母在 1986 年给他买了一个 Speccy，但他对旧硬件的热爱仍然存在。多年来，这演变成计算机收集，与旧 ZX 光谱，准将 64，ORICs，橡子圆了他的收藏。正如我们在电脑展上了解到的，欧洲中部的电脑在英语互联网上是看不到的，[马丁]的收藏也不例外。

除了为一些非常旧的电脑做一些非常酷的东西，[马丁]还向[Hackaday Hackaspace](https://hackaday.io/project/80-hackaspace)捐赠了一些东西。这是一台 [PMI-80](http://en.wikipedia.org/wiki/PMI-80) ，一台为大学计算机科学学生制造的单板计算机，基本上是 KIM-1，但基于由 [Tesla](http://en.wikipedia.org/wiki/Tesla_(company)) 制造的英特尔 8080 的捷克斯洛伐克克隆版。该板上有 1k 的 RAM 和 1k 的 ROM、一个计算器键盘和几个七段显示器。当时，这是一台很棒的“学生”电脑，在欧洲并不罕见，但这是我在大西洋彼岸见到的第一台。

你可以在下面看到一些 PMI-80 的图片和[Martin]的采访。[马丁]还承诺写一部经典中欧计算机的简史，这是一个在英语圈里很少被提及的主题。当他完成后，我们会发布一个链接。
