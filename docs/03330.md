# 3W 手持激光器为有一天真的光剑带来了希望

> 原文:[https://hackaday . com/2013/05/24/3w-hand-held-laser-raises-hope-for-a-real-lightsabre-day/](https://hackaday.com/2013/05/24/3w-handheld-laser-raises-hope-for-a-real-lightsaber-someday/)

该横幅图像可能看起来有点夸张，但它很好地展示了[这款 3W 手持式激光器真正能做的事情](http://www.youtube.com/watch?v=96E0RfyL2Gc)。在一个烟雾弥漫的房间里打开它，它看起来就像一把细光束光剑。

你认为这东西会表演什么样的把戏？也许它会爆开一些黑色的气球？准备好被震惊吧，因为它的威力比这大得多。下面的视频显示，它几乎瞬间燃烧并点燃了一系列物品。[Styropyro]通过点燃纸张、纸板、闪光纸、闪光粉，烧穿一根木头，并在视频结束时点燃一种未公开的物质来测试他的创作。但是我们最喜欢的一个是当他开着一辆太阳能玩具车，带着强烈的光束。

他从 DLP 投影仪上取下二极管，用笔记本电脑中常见的一对 18650 锂离子电池驱动它。他自己做的围栏。它看起来很棒，但我们不禁想知道这些组件是否适合一个精心制作的复制品。
