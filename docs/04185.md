# DIY 汽水罐电池

> 原文:[https://hackaday.com/2013/09/30/diy-soda-can-battery/](https://hackaday.com/2013/09/30/diy-soda-can-battery/)

![sodaCanBattery](../Images/ff103d693c743165a297f34ad3575df3.png)

用汽水和汽水罐制造一些临时电池可能不是特别有用，但这是对电极和电解质的一个很好的介绍，也是对柠檬和土豆的一个很好的突破。[Go-Repairs]的那帮人砍掉了罐盖，暂时把苏打水放在一边，然后用钢丝绒将罐内部的保护性塑料涂层去掉。这个过程可以通过抓住你的钻头，把钢丝绒塞进铲形钻头的末端来加速，尽管压得太紧可能会撕裂罐子。

当苏打水倒回去的时候，你可以通过将一根导线夹在罐子上，另一根导线夹在浸在苏打水里的铜币上来获得一些电压。串联更多的罐子可以增加果汁，但你需要整整六包才能让 LED 工作——而且只是刚刚好。说明书建议把苏打换成不同的电解质:排水清洁剂，它可以从六个一组的系列中泵出令人印象深刻的 12 伏电压。然而，你要小心，因为它很可能会把罐头吃穿，而且只差一个盖子就危险了。
