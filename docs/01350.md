# 将[M. C .埃舍尔]版画变成实物

> 原文:[https://hack aday . com/2012/08/17/turning-m-c-escher-prints-into-real-objects/](https://hackaday.com/2012/08/17/turning-m-c-escher-prints-into-real-objects/)

![](../Images/620cfd26ae923cbf07ffeab64570141b.png "escher")

九月就要到了，很快世界各地的大学新生将会用月亮的阴暗面海报和 Moon 埃舍尔版画来装饰他们宿舍的墙壁。任何人都可以出去简单地买一个棱镜，但是如果你想要一个来自[埃舍尔]宇宙中的物体和建筑的真实版本呢？以色列技术学院的教授[Gershon Elber]决定[将[Escher]的照片变成现实](http://www.cs.technion.ac.il/~gershon/EscherForReal/)。

首先从简单的形状开始，如彭罗斯三角形和内克尔立方体，[埃尔伯]决定扩展到更不可能的形状，如[埃舍尔]的*瀑布，【贝尔维迪尔】，*和*相对论。*这些建筑很难在任何传统的计算机设计程序中可视化，因此【埃尔伯】为他的 [IRIT 计算机建模程序](http://www.cs.technion.ac.il/~irit/)编写了一个插件，在将它们提交给 3D 打印机之前设计这些建筑。

在休息后的视频中，你可以看到一些由此产生的[埃舍尔]建筑的旋转视图。当然，他们只从一个角度工作——即使这样，也只是闭上一只眼睛——但看到这些著名的建筑研究被带入现实世界，这是令人惊讶的。
