# 来自激光切割机的 3D 对象

> 原文：<https://hackaday.com/2015/09/02/3d-objects-from-a-laser-cutter/>

演员想当歌手，歌手想当演员。与此相对应的黑客可能是，3D 打印机希望成为激光切割机或数控机床，而激光切割机希望成为 3D 打印机。当[Kurt]和[Lawrence]发现他们的技术商店获得了一台 120 瓦的 Epilog Fusion 激光切割机时，他们开始考虑是否可以让它切割出 3D 形状。这个问题引导他们进行了几次最终成功的实验。

这个想法是切掉材料，旋转工件，并以类似于一些激光切割机处理雕刻圆柱形物体的方式切割更多的材料。与 3D 打印的加法不同，这一过程与传统加工过程一样是减法。开发者使用木材作为基础材料。他们想用丙烯酸树脂，但是发现切下来的部分容易粘在一起，所以他们继续用木头。然而，木材在切割时往往会烧焦。

最后，他们不仅要建造特殊的夹具和电子设备，还要移植一些第三方控制软件来解决 Epilog Fusion cutter 内置软件的一些问题。最后的改进是使用激光的光栅模式在零件上绘制表面细节。

结果比你预期的要好，看起来相当独特。我们已经介绍了一个类似的过程，用丙烯酸树脂制作小棋子。这似乎是同一个想法的自然延伸。当然，也有非常复杂的工业机器进行三维激光切割(见下面的视频)，但它们与典型的桌面切割机不在一个类别中。
