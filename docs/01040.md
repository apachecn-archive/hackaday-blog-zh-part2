# XBMC 推出安卓系统

> 原文:[https://hackaday.com/2012/07/14/xbmc-hits-android/](https://hackaday.com/2012/07/14/xbmc-hits-android/)

![](../Images/f73e34bf5a448b0323e5f1f45d7e2ee9.png "OLYMPUS DIGITAL CAMERA")

XBMC 刚刚发布了我们期待已久的公告。[Android 平台现已获得官方支持](http://xbmc.org/theuni/2012/07/13/xbmc-for-android/)。看到这种开源媒体中心软件的流行，以及一些人愿意花数百美元在小型计算机上使用它作为他们选择的机顶盒界面，我们知道硬件制造商采取行动只是时间问题。在这种情况下，它是一家名为 Pivos Technology Group 的公司，该公司帮助资助了将 XBMC 引入 Android 的努力。

好消息是，安卓版的 XBMC 应该可以在非常多的设备上运行。坏消息是，这种广泛的设备支持硬件视频加速还需要一点时间。目前，唯一拥有所有视频格式硬件加速的平台是上面看到的 Pivos XIOS DS 设备。看起来有点像白色版的苹果电视。这被证明是苹果硬件的一个很好的替代品，苹果硬件需要越狱来运行 XBMC，而目前这一代硬件没有越狱可用。你可以花大约 100-120 美元买到 XIOS DS，正如你在休息后看到的那样，它可以流畅地运行 XBMC，显示在播放 1080p 视频时导航菜单的能力，并演示工作视频插件。

此外，这是一款 Android 设备，可以像其他设备一样访问应用程序。我们环顾四周，似乎网飞的应用程序将工作，但目前有一个问题，视频驱动程序的单位已升级到 ICS。你可以在这个论坛帖子中查看[该设备的开箱情况，这就是我们得到上面看到的图像的地方。](http://www.pivosforums.com/viewtopic.php?f=11&t=685)
