# 世界上最大的泡泡机器诞生于太空计划

> 原文：<https://hackaday.com/2015/07/06/worlds-greatest-bubble-machine-born-of-space-program/>

[戈登·柯克伍德]需要肥皂泡。大肥皂泡。他为什么需要肥皂泡？因为——肥皂泡！

实际上，[戈登]是一名摄影师，他想捕捉到偷拍的瞬间和稍纵即逝的表情。有什么比根据需求制造一个巨大的肥皂泡更能激发惊奇和惊叹的方式呢？还有什么比用复杂的、电脑控制的巨型泡泡机更好的方法呢？

【戈登】气泡产生机制的灵感来自于[加拿大臂](https://en.wikipedia.org/wiki/Canadarm)机器人航天飞机臂的末端执行器，它使用缆绳抓钩设计来捕获和固定有效载荷。[Gordon]使用类似的原理将泡汁浸泡过的线交织在一起，在一个平面内将它们拉开，形成一层肥皂膜。一股风或风扇的快速喷射使气泡膨胀并发射出去，该装置可以夹住气泡以精确控制其大小。

[戈登]投入机器的工作量令人印象深刻。他的 Instructables 帖子非常详细，不仅涉及到他的构建，还涉及到他的设计过程和原型，肥皂泡仪器的科学，甚至是对其他先锋泡泡爱好者的工作的认可。他还贴心地附上了专业级泡泡果汁的配方，其中的秘密成分可能会让你大吃一惊。
