# 聪明的杂耍球

> 原文:[https://hackaday.com/2015/04/28/smart-juggling-balls/](https://hackaday.com/2015/04/28/smart-juggling-balls/)

为了获得 2015 年的 Hack a Day 奖，[Arkadi]正在开发一个教育平台，使用 Arduino 开发智能杂耍球。

他的目标是创建一个开源平台，除了杂耍之外，还可以用来以互动和有趣的方式教授物理。杂耍球配有一个 RGB LED，一个 Arduino pro mini 和一些 MPU。它们可以通过编程根据加速度、重力、向心力、旋转来改变颜色，甚至可以远程控制。随着项目的进一步发展，他还计划在项目的同时创建讲座内容，这将使它成为高中(甚至大学)技术或物理课的一个优秀的交互式项目。

虽然还没有完全完成，但他已经在 GitHub 上发布了制作自己的 set [的所有源代码和说明。](https://github.com/arkadiraf/Juggling_Balls_Public)休息后留下来看看原型球的运行，别忘了让[破解你自己的黑客大奖入口](http://hackaday.io/had2015)！
