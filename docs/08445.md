# X-Carve，Shapeoko 的逻辑升级

> 原文:[https://hack aday . com/2015/03/03/x-carve-the-logical-upgrade-to-a-shape oko/](https://hackaday.com/2015/03/03/x-carve-the-logical-upgrade-to-a-shapeoko/)

当谈到用于黑客空间和设备极其完善的车库的小型数控雕刻机时，Shapeoko 或类似的机器一直是默认的机器。它非常简单——一个连接到线性轨道上的 Dremel 并且对于从铣削 PCB 到安排复杂的木工项目到用笔绘制设计都很有用。现在，[巴特·德林]，Buildlog.net 激光和发明背后的家伙[已经合作创造了下一代雕刻机](https://www.inventables.com/technologies/x-carve)。它被称为 X-Carve，虽然它与 Shapeoko 2 完全兼容，但它增加了一些改进，使机器变得更好。

X-Carve 取消了基于 Dremel 的主轴，代之以可以产生扭矩的东西。在库存安排中有一个 24VDC 主轴，它将通过 Gcode 为您提供速度控制。当然，有适配器可以安装在这类机器中最常用的 Dewalt 和 Bosch 路由器。

就机架而言，X 和 Y 轴是 makerslide 没有变化。Z 轴丝杠可以升级到 Acme 螺纹杆，这是对几乎所有其他 DIY 机器套件中的 M8 螺纹杆的改进。整个机器基本上是一个 Shapeoko 应该拥有的所有升级，更强的角落，NEMA 23 电机，增加了刚度。

X-Carve 有几个版本，从升级套件到 Shapeoko 2，再到机器空间为一平方米的满载套件。大型高端套件的价格约为 1250 美元，但带有 500mm 导轨、NEMA 17 和螺纹杆丝杠的较小套件的价格约为 800 美元。
