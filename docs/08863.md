# Robottermilk Pancakes

> 原文:[https://hackaday.com/2015/04/23/robottermilk-pancakes/](https://hackaday.com/2015/04/23/robottermilk-pancakes/)

有这样一个名字，我们怎么可能错过这个呢？这个制作煎饼的机器人在不到 24 小时的时间里就完成了，真是一个黑客。[Carter Hurd]、[Ryan Niemo]和[David Frank]凭借该项目获得了 2015 年俄亥俄州立大学 Makethon 奖。

机架使用 RepRap 的皮带在抽屉滑轨上运行。电机本身是带编码器的 DC。[Carter]告诉我们，由于大多数 3D 打印机都是基于步进电机构建的，这意味着他们必须从头开始构建控制软件，但幸运的是，我们能够在其余部分重复使用 PID 软件。听着，驱动煎饼面糊的泵是从一个 Keurig 中抽出来的，一个伺服电机用来扭结管道，停止流动。我们被用 Sriracha 瓶子作为喷嘴逗乐了。

不仅仅是打印机一起被黑。该团队还开发了一个 iPhone 应用程序，让你画出你想要的图案，并通过 WiFi 将其推送到机器上。

灵感来了吗？我们是！如果你在纽约附近，你需要在 5 月 2 日到 3 日的黑客马拉松中带上这种游戏。一个晚上，很多乐趣，很多食物，很多硬件。你能完成什么？
