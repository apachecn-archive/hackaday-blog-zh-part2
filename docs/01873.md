# 模拟器 101:如何编写一个像 NES CPU 一样运行的程序

> 原文:[https://hackaday . com/2012/10/12/emulators-101-how-to-write-a-program-that-functions-like-an-nes-CPU/](https://hackaday.com/2012/10/12/emulators-101-how-to-write-a-program-that-functions-like-an-nes-cpu/)

![](../Images/64f91b23bdd235e6365ef6b8fdb9bd52.png "how-to-write-an-NES-emulator")

我们打赌阅读这篇文章的每个人都曾经在模拟器上玩过游戏。您可能对这些模拟器的工作原理有一个基本的概念。但是我们敢打赌，你们中的大多数人对游戏模拟器的实际实现一无所知(我们知道我们是)。但是在看了这个[演示[Bisqwit]如何编写他自己的 NES 模拟器](http://www.youtube.com/watch?v=QIUVSD3yqqE#t=262s)之后，这一切都改变了。该描述只涵盖了编写模拟 NES CPU 硬件本身的代码的基础知识。但是它以这样一种方式呈现，使得任何具有编程基础知识的人都很容易理解。他从处理处理器操作码的 switch 语句开始，然后一段一段地展示了他是如何精炼他的代码，使其在保持可读性的同时工作的。我们认为这是一个伟大的教学方法，并感谢他投入时间制作这个教程。

解说大约在 4:22 开始，视频在休息后嵌入。您还可以在那里找到前两个演示视频。这些主要涉及模拟器编码过程的快动作文本编辑，在第二个视频的结尾有一些游戏测试。
