# 天体摄影和数据分析感知系外行星

> 原文：<https://hackaday.com/2014/12/01/astrophotography-and-data-analysis-sense-exoplanets/>

[大卫·施奈德]正在阅读最近发现的系外行星。简而言之，这些行星围绕着除太阳以外的恒星运行。研究科学家使用的装备包括大型望远镜，但他们使用 CCD 传感器的事实让[大卫]想知道是否可以在后院便宜地完成这一版本。答案是肯定的。通过捕捉和处理来自谷仓门跟踪器的数据，他能够验证一颗已知的系外行星。

谷仓门跟踪器是用来移动摄像机以补偿地球转动的设备。这在夜间拍摄图像时是必要的，因为没有它，星星将不会在相机的框架上保持“静止”。好消息是它们很容易建造，[这些年来我们已经见过](http://hackaday.com/2014/01/23/avr-barn-door-tracker-for-astrophotography/)几个。

除了必须等到他所在的地球部分在一次系外行星凌日的同时被指向正确的方向之外，[大卫]已经准备好收集他需要的所有数据。这部分很快就变得有趣了。相机需要捕捉行星穿过地球和它所围绕的恒星之间(称为凌日)。证明这种情况发生的数据非常微妙。为了揭示这一点，大卫需要通过参考其他几颗恒星来控制大气变化的数据集。在那里，他专注于凌日目标的数据，并比较了整个捕捉图像集的点。结果是亮度下降，符合最初发现的规格。

[David]在休息后的片段中解释了整个过程。
