# PortableSDR 成功登陆 Kickstarter

> 原文：<https://hackaday.com/2015/01/14/portablesdr-makes-it-to-kickstarter/>

去年的 Hackaday Prize 见证了许多被称为 Kickstarter 运动的项目，但没有人看到人们像[[迈克尔]的 PortableSDR](http://hackaday.io/project/1538-portablesdr)一样向屏幕扔钱。这是一款小型、手持式、电池供电的短波软件定义收发器，可以做任何事情，覆盖范围高达 30MHz。这是终极启示录电台，现在由“我的第一电台”宝丰争夺王位，现在，终于，[在 Kickstarter](https://www.kickstarter.com/projects/1703258614/psdr-pocket-hf-sdr-transceiver-with-vna-and-gps) 上发起了一场运动。

PortableSDR(现在叫做 PSDR)是迈克尔理想的收音机。恰好 Hackaday 奖给了他设计、开发和制造无线电的动力，最终使他在 Hackaday 奖中获得第三名。

无线电本身是完全独立的，电池供电，在 STM32F4 处理器上实现软件定义的无线电。该设计包括用于瀑布显示的 LCD、矢量网络分析和接收 GPS 的能力。

为了保持其火腿传统，[Michael]将 PSDR 作为一个套件提供，包括 PCB、外壳和所有你在 Digikey 上买不到的部件，价格为 250 美元。让那些烤面包机回流焊炉变暖，因为在这个建筑里有许多 SMD 零件。
