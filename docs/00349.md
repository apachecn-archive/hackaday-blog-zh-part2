# 遥控汽车控制器和接收器的更换

> 原文:[https://hack aday . com/2012/05/08/RC-car-controller-and-receiver-replacement/](https://hackaday.com/2012/05/08/rc-car-controller-and-receiver-replacement/)

![](../Images/2cd97c13564f6aa9eb272fdf7e8ad434.png "replacing-the control-cirtuitry-on-an-rc-car")

这种[无线电控制的汽车控制器替代品](http://www.elecfreaks.com/3044.html)是一个伟大的项目，可以用你身边的花哨硬件尝试一些新事物。黑客分为两部分，接收器和发射器。它们通过蓝牙通信，所以如果你只想构建硬件的一面，你应该能够让大多数蓝牙设备作为另一面工作。例如，为汽车制造接收器，并用 Wii 遥控器控制它。或者使用控制器在 Android 手机上玩模拟器游戏。

这两件作品都是基于 Arduino 的。控制器使用了一个带有蓝牙屏蔽的 Freeduino 以及一对模拟棒分线板。汽车方面使用蓝牙 Bee(与 Xbee 插座引脚兼容的蓝牙模块)和 Arduino pro。一旦硬件组装完成，只需很少的代码就可以启动并运行系统。休息之后，请加入我们，快速观看该车的运行片段。
