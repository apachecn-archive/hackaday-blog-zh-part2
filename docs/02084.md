# 尼康 WU-1a WiFi 加密狗黑客

> 原文:[https://hackaday . com/2012/11/10/Nikon-Wu-1a-wifi-dongle-hacking/](https://hackaday.com/2012/11/10/nikon-wu-1a-wifi-dongle-hacking/)

![](../Images/66bd09087aaea3d12813bfc728614e71.png "nikon-wifi-dongle-hacking")

这里有一个相当棘手的消费电子逆向工程。[Joe Fitz]看到了尼康 WU-1a。这是一个可以插入尼康 D3200 相机的加密狗，可以产生 WiFi 连接，可以通过智能手机接收和控制。该应用程序从取景器中显示当前图像，允许您拍摄照片，然后下拉照片。问题是，他的 D800 相机的相同功能将花费他 1200 美元，而这个加密狗只需 60 美元。这是一个强大的激励，以找到一种方法来使用吴-1a 与他的相机模型。这不仅仅是重新布线。它包括嗅探 USB 流量和深入查看硬件中所用芯片的数据手册。我们不确定，但他可能已经为加密狗推出了新的固件。

目前细节还不太清楚。你最好的选择是在休息后观看嵌入的视频。还有[一组幻灯片](http://goo.gl/5530H)，是【乔】在本周末的 [BsidesPDX](http://www.securitybsides.com/w/page/40113672/BsidesPDX) 上为一次演讲而放在一起的。它会让你对他所经历的过程有一个大致的了解。但是他也[发起了一个论坛主题](http://nikonhacker.com/viewtopic.php?f=2&t=218)，随着对话的进行，我们希望从中了解更多。
