# 为天文摄影建造一个谷仓门跟踪器

> 原文:[https://hack aday . com/2013/08/08/building-a-barn-door-tracker-for-astronomical-photography/](https://hackaday.com/2013/08/08/building-a-barn-door-tracker-for-astronomical-photography/)

从你的后阳台往外看，这是一个非常惊人的图像。右边是用来记录这种宝石的钻机。它叫做谷仓门跟踪器，由 DCH972 制造。这个版本的细节到处都是，有[的一个视频](http://www.youtube.com/watch?v=_mPij7eHOmo)(也在下面找到)，[的另一个相册](http://imgur.com/a/GbMaj)的一些最好的图片，以及 Reddit 线程中大量的[背景信息。](http://www.reddit.com/r/astrophotography/comments/1jk0h5/equipment_homebuilt_tracking_mount_up_and_running/)

这种设计也被称为黑格或苏格兰山。当我们到处删除链接时，请查看主题为的维基百科页面[。该系统的要点是以这样一种方式移动相机，即使地球在移动，星星也似乎保持在同一位置。有一个 ATmega32u4 分线板骑在试验板的顶部。它正在做一些非常复杂的数学运算，以便计算步进电机的时间。这是因为支架就像一本相册，一边用铰链连接，另一边用滚珠丝杠打开。这种线性驱动需要与安装平台的角度变化相配合，最终需要与地球的运动同步。但是一旦一系列图像被正确捕捉，它们就可以被处理成上面显示的合成照片。](http://en.wikipedia.org/wiki/Barn_door_tracker)

如果错过了去年五月的 SDR 星系旋转探测器，你会发现它同样引人注目。
