# 看看去封装的 IC

> 原文：<https://hackaday.com/2012/08/13/taking-a-look-at-decapped-ics/>

除了想玩玩硝酸，[Ben]真的没有理由[解封几个 74xx 和 4000 系列逻辑芯片](http://www.youtube.com/watch?v=mT1FStxAVz4&feature=em-uploademail)。我们并不介意，因为他提供了一个很好的教程来查看一个没有覆盖环氧树脂和树脂的裸 IC。

大多数 IC 封装在坚硬的环氧树脂外壳中，很难看到内部电路。[Ben]试图用 Dremel 工具磨掉这种环氧树脂，但运气不太好，直到他转移到 CNC 研磨机上，在不破坏焊线的情况下去除 0.040-0.050”的环氧树脂。

在芯片上方雕刻出一个漂亮的口袋后，[本]在芯片上滴几滴硝酸来溶解环氧树脂涂层。这在室温下进行得非常慢，但是在将芯片放在一个热的盘子上之后，酸能够露出下面的芯片。

在成功去除所有环氧树脂并给它们一个丙酮浴后，[Ben]把他的芯片拿到显微镜前，能够检查出潜在的电路。他不知道他可以用这些解封的逻辑芯片做什么，但焊线仍然完好无损，所以他仍然可以在构建中使用这些芯片。

我们希望看到一些解封的 MEMS 器件，但是如果你对 Ben 的解封芯片有什么建议，请在评论中留言。
