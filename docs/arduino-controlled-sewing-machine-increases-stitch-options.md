# Arduino 控制的缝纫机增加了线迹选择

> 原文：<https://hackaday.com/2015/01/07/arduino-controlled-sewing-machine-increases-stitch-options/>

[Andrew]正在通过增加一个 Arduino 控制模块将他的旧机械缝纫机带入 21 世纪。最初，他的阿尔法缝纫机只能做直缝或不同宽度的锯齿形。因为这是一台旧缝纫机，所有的控制都是旋钮和杠杆。RC car 伺服系统安装在缝纫机上，现在只负责实时控制针的水平运动和送布牙(负责推进织物通过缝纫机的金属部件)的行程量。针杆上还有一个开关，当针处于满针位置时，它会反馈给 Arduino。

通过完全控制针迹宽度*和*织物前进，有可能产生一些以前在这台机器上不可能出现的令人惊叹的针迹图案。每个针脚图案都在 Arduino 中预先编程。现在可以通过 Arduino 的串行 USB 连接来控制缝纫机，但这种操作的工作流程还处于初级阶段。[Andrew]计划把这台缝纫机做成全自动的，这样他就可以绣字母和数字了。

尽管该项目仍在进行中，但[Andrew]已经为那些想进一步完善他的成就的人们提供了他的初步 Arduino 代码。要继续阅读关于被黑的缝纫机，看看这台被改造成[绣花机](http://hackaday.com/2014/06/10/diy-embroidery-machine-sews-your-name-in-your-undies/)。
