# 激情项目将比格伯恩变成独立的超级 NES

> 原文：<https://hackaday.com/2015/01/31/passion-project-turns-beaglebone-into-standalone-super-nes/>

所以你想在你的 BeagleBone 上玩一些复古游戏，只需加载 Linux 并启动你最喜欢的模拟器，对吗？如果你是认真的就不会。[Andrew Henderson]从 beagle board-xM(beagle bone Black 的前身)开始走上这条道路，并发现 Snes9X 的性能并不完全符合他的想法。他跃跃欲试，[创建了一个名为](http://hackaday.io/project/3913-beaglesnes) [BeagleSNES](http://beaglesnes.sourceforge.net/) 的成熟发行版,其中包括用于提高性能的引导加载程序和内核 hacks，一个自定义 GUI，并且正在为嵌入式游戏装备开发硬件。查看[项目](http://beaglesnes.sourceforge.net/BeagleSNES_Guide.pdf) (PDF)附带的文档；这是开源项目指南应该如何呈现的蓝图！

他目前正在开发的硬件是一个 Cape(BBB 的附加板被称为 Cape ),它为原始任天堂和超级任天堂控制器添加了连接器。它还包括一个 RTC，将代替一些墨盒(口袋妖怪黄)中包含的实时时钟功能。此外，3D 打印外壳也在工作中，这将使它成为一个便携式设备，就像另一个 BBB 便携式黑客。
