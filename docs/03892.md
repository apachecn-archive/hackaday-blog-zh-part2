# 3D 打印橡皮筋动力扑翼机的惊人飞行

> 原文：<https://hackaday.com/2013/08/21/amazing-flight-of-a-3d-printed-rubber-band-powered-ornithopter/>

实际上，我们将链接到二月份的一个旧帖子，因为我们认为它和最近的作品一样令人印象深刻。这是由橡皮筋驱动的 3D 打印扑翼机 ( [翻译](http://translate.google.com/translate?sl=auto&tl=en&js=n&prev=_t&hl=en&ie=UTF-8&u=http%3A%2F%2Fblog.goo.ne.jp%2Fflappingwing%2Fe%2Fbeaba77dd84dbed71002fb2a85dd9f57))。框架很像传统的橡皮筋飞机。不同的是，上紧发条后，它不会转动螺旋桨。四个塑料薄膜翅膀的拍打使它像变魔术一样飞起来。说真的，看看下面的演示…我们几乎把这个贴成了“真的还是假的？”如果我们几年前没有看到[类似的产品](http://hackaday.com/2010/11/11/japanese-micro-planes/)。

考虑到发射前的快速上链是飞行的全部动力，这次飞行持续了相对长的时间。但是该网站最近提供的 ( [翻译](http://translate.google.com/translate?sl=auto&tl=en&js=n&prev=_t&hl=en&ie=UTF-8&u=http%3A%2F%2Fblog.goo.ne.jp%2Fflappingwing%2Fe%2F55e13c8b33ae4e5f83ce50c8ed1ee5ac))包括上面看到的机动扑翼机设计。它携带了一个小型锂电池用于连续飞行。这些设计有一个 3D 打印齿轮系统，这使它们有点复杂，但给派对带来了转向和远程控制。如果你想要一个你自己的，他们正在做一个小批量的套件。我们认为自己制作原型并打印出来会有趣得多。当然，这是重新发明轮子。但这是一个非常酷的轮子！
