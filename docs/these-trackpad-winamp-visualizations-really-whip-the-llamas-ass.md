# 这些触控板 Winamp 可视化真的让美洲驼大吃一惊

> 原文：<https://hackaday.com/2014/05/17/these-trackpad-winamp-visualizations-really-whip-the-llamas-ass/>

尽管我们喜欢干净整洁的 M 型电脑，但这些卷曲的 clicky 庞然大物在某种游戏风格上有些欠缺。没有任何空白键可以覆盖，也没有一个全彩色的触控板可以让你编写应用程序。[Gus]有这样一个键盘:Razer Deathstalker Ultimate 具有弹簧刀用户界面。他为自己制作了这个甜美的音频可视化工具，因为[将 Winamp 可视化工具扩展到了 Switchblade UI](https://github.com/gguuss/razerviz) 。

[Gus]的 hack 建立在 [Tiny3D](https://github.com/icebreaker/tiny3d) 可视化框架之上。它会做你可能会想到的事情——读取可视化值，将它们设置为在触控板上显示，并将其渲染到所述触控板上。[Gus]使用 10 个可编程键中的一些来动态改变颜色，结果非常棒。正如[Gus]指出的，这仅仅是插件所能做的事情的开始。你需要 Razer SDK 来开始，你可以从[【格斯】的回购](https://github.com/gguuss/razerviz)中获得其他成分。一旦你完成了这个，你可以尝试[自动调暗你的键盘背光](http://hackaday.com/2013/05/11/auto-dimmer-hacked-into-keyboard-backlight/)。

当然跳完之后还有演示视频。走吧。
