# 把 IBM 电动打字机变成打印机。

> 原文：<https://hackaday.com/2012/06/13/turning-an-ibm-selectric-into-a-printer/>

![](img/d26dc654692a9de8b7f0d34791e93fc2.png "selectric")

在计算机占据整个房间的时代，CRT 显示器是一种奢侈品。大多数时候，输入和输出是由电传打字机处理的——一种直接与计算机相连的打字机。[乔希]想要自己的打字机终端，所以他[拆开了一台 IBM Selectric II](http://www.robotdialogs.com/2012/05/started-working-on-selectric-ii.html)开始工作。

IBM Selectric II 不使用电子键盘，而是使用机电键盘来倾斜和旋转 Selectric 的打字球。在正常操作中，键盘下面的一系列轴是接合的。[Josh] [在这些杠杆上增加了一个安装器](http://www.robotdialogs.com/2012/05/typomatic-part-2.html)的部件，并将每个部件连接到 16 个螺线管中的一个。

借助 Arduino，一组螺线管可以打印任何按键，[Josh]从 20 世纪 70 年代初就有了一台全自动打字机。[Josh 最近一直在打印大量 ASCII 艺术作品，为本月晚些时候的堪萨斯城创客节做准备。休息之后，您可以查看构建视频。
