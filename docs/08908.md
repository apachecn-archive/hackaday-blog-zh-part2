# 黑客日奖参赛作品:探地雷达

> 原文:[https://hack aday . com/2015/04/29/hack aday-prize-entry-ground-penetrating-radar/](https://hackaday.com/2015/04/29/hackaday-prize-entry-ground-penetrating-radar/)

今年的 Hackaday 奖正在升温，现在有相当多的项目正在进行中，涉及的领域很少出现在车库或研讨会上。其中最有趣的是[【格伦·鲍尔斯】‘开放式探地雷达](https://hackaday.io/project/4440-open-ground-penetrating-radar)。正如标题所言:一个不到 500 美元就能透视地球的开源雷达系统。

虽然探地雷达对于考古和人们搜索埋在农田中间的宝藏非常有用，但最大的应用是安全。你只需要谷歌一下“佛罗里达天坑”就能明白窥视地球的价值。

[格伦]正在用最少的零件建造他的探地雷达。宝丰 VHF/UHF My First Radio 用作信号发生器，控制器只是一个光隔离器，开关控制器是一个 7404 hex 反相器。这简直是再简单不过了。

当然，这些组件只能组装成一个简单的雷达，探地雷达的真正价值是绘制区域地图的能力。为此，[格伦]正在拿出一个 Pi 和一个 GPS 加密狗来控制整个事情。可视化是由美国海军提供的。如果它适用于潜艇，它应该适用于金属推车，对不对？

这是一个伟大的项目，不仅因为它可以帮助一大群人，而且是一个不用数万美元的测试设备就能做这么多事情的最好例子。
