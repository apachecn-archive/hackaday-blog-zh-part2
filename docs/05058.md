# 水力:低成本水射流切割机

> 原文：<https://hackaday.com/2014/01/14/hydro-the-low-cost-waterjet-cutter/>

喷水切割机通常是巨大的机器，价格也同样昂贵。但是，如果有一个业余爱好者水平的喷水切割机，其实是负担得起的呢？嗯，在宾夕法尼亚大学的高级设计项目中，[Adam Libert]和他的团队[做了一个零售价不到 5000 美元的产品。](http://www.adamlibert.com/Waterjet)

【Adam】是这个令人惊叹的项目的首席机械设计师，他设计了完全防水的 XY 机架，能够承受来自切割机的水和磨料。整个机器只有 2 英尺 x 2 英尺高，大约 5 英尺高，这使得它非常便携，很容易在门口移动——它依靠普通的 120 伏交流电和商店的空气。它能够切割高达 1/4 英寸的铝和 1/8 英寸的钢，工作面积为 12 英寸×14 英寸，公差为 0.005 英寸。

不足为奇的是，该项目在 2012 年赢得了机械工程高级设计竞赛，并获得了杰出创造力的赞誉。我们无法找到这个项目未来计划的任何信息，但我们希望他们将它开源，甚至为它举办一场众筹活动。

我们的目标是创造出第一台低成本、小规模、易于使用的喷水切割机，从视频来看，他们似乎做到了——休息后留下来自己看看。
