为什么不发布一篇链接来结束我们为期两周的湾区创客集会报道呢？这次是视频链接。我们把一些没有自己视频的有趣片段混合在一起，也感受了一下整个周末在露天广场漫步的感觉。休息之后加入我们的链接。

*   我们认为自由形式的时钟没有被记录，但是 [SparkysWidgets](http://www.sparkyswidgets.com/) 是寻找它的地方。
*   不幸的是，我们没有得到任何有 BLE 在内建的 Femtoduino 的音频。我们所能做的就是向你展示它们是如何被装在一个小罐子里的。
*   Mojo FPGA 板正在驱动一个漂亮的颜色跟踪六足机器人。
*   [Paul Stoffregen]展示了他为 Teensy 3 工作了几个月的音频库。
*   一瞥[阿杜博伊](http://www.arduboy.com/)；一个 PCB 薄的手持游戏系统。
*   [PicassoBot](http://educatedrobot.com/) 通过网络摄像头为你拍照，并使用简单的机械臂用钢笔画出肖像。
*   在【阿尔瓦罗的】[激光射击馆](http://alvarop.com/2014/05/maker-faire-2014/)与一个激光机器人炮塔比赛。
*   机器人罗伊向人群伸出双臂。

这就是我们 2014 年[湾区创客大会的报道](http://hackaday.com/tag/bay-area-maker-faire-2014/)。
