# 用工具撬锁

> 原文：<https://hackaday.com/2012/10/01/picking-locks-with-toool/>

![](img/df2adcc31d888c014aec702f1cff26d3.png "toool")

没有教会孩子们摇动和扭转锁直到它们打开的快乐，哪一个制造者集会是完整的？Toool，[开锁人的开放组织](http://toool.us/)本周末来到纽约，展示他们的撬锁技术，并让孩子们对操纵小型机械装置感兴趣。

Toool 的工作人员有一个非常酷的设备——只有一些桌子和椅子，还有一些镐和扭力扳手。桌子上有几把经典的万能锁，但也有一系列六把不倒翁锁，每把都标有数字 1 到 6，表示锁里有多少个插销。这个想法是让一些人开始使用单销锁，并最终让他们使用全部六个销。

在休息后的视频中，Toool 的一个更有活力的家伙解释了他们为什么在那里，还展示了在 30 秒内两次撬开主锁。说真的，人们:在买锁之前，先学习一下锁。
