# 构建一个简单的音频放大器

> 原文：<https://hackaday.com/2014/01/04/build-a-simple-audio-amp/>

[Ynze]已经建立了一个看起来和听起来都很棒的音频放大器。他的放大器使用的是美国国家仪器公司(现 TI) LM3886 Overture 系列 68 瓦功率放大器。LM3886 将[Ynze]放大器完全归入“增益克隆”类别。 [Gainclone 或 Chipamp](http://en.wikipedia.org/wiki/Gainclone) 是 DIY 社区长期使用的术语，用来描述基于高度集成半导体放大器的音频放大器。Gainclone 的名字来源于 47 实验室出售的原始 Gaincard 音频放大器。Gaincard 在 1999 年推出时使用了不到 100 美元的零件。它听起来很好，足以在音响发烧友市场上获得 3300 美元的价格。较低的零件数量和简单的结构催生了音频 DIY 社区来构建他们自己版本的 Gaincard。现在有数百种变体，涉水通过不同的版本可能是一个令人生畏的任务。[Ynze]找到了一个可行的基本设计，并在此基础上进行构建。

关于[Ynze 的]放大器以及许多 Gainclones 的一个有趣的事情是，它们不使用电路板。所有布线都是点对点完成的。电阻直接焊接到放大器芯片的引脚上。对于初学者来说，这可能是一些棘手的焊接，但有几个 PCB 套件可用。[Ynze]在两种情况下制造他的放大器。一个盒子里装着电源，另一个盒子里装着放大器。[Ynze]正在使用一个大型环形变压器将他当地的 230V 电源降低到+25V 和-25V。放大器电路本身很简单，一些分立元件围绕着 LM3886 及其散热器。[Ynze]还在他的木质底盘上做了一些非常好的木工活。由此产生的放大器看起来就像是 20 世纪 60 年代的产品，但里面隐藏着 20 世纪 90 年代的电子产品。
