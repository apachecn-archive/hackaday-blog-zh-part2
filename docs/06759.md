# 打造自制 IPhone

> 原文：<https://hackaday.com/2014/08/06/building-a-home-made-iphone/>

几年前，[Michele]制造了一款带有触摸屏、相对强大的处理器和一大堆传感器的移动设备。老实说，从来没有人问过他为什么要建造这个，因为这是一个令人印象深刻的电子设计和制造展示。[[米歇尔]称之为伊格鲁皮奥。](http://hackaday.io/project/2116)虽然这不是一部功能齐全的手机，但它仍然是一个令人印象深刻的项目，有其自身的优点。

iGruppio 内部是 Pic32mx 微控制器、240×320 TFT 触摸屏和足够多的传感器来实现 10 自由度 IMU。为 iGruppio 编写的软件在很大程度上受到了 iPhone 和一个完全自制项目的启发——所有软件都是由[Michele]自己编写的。虽然第一版的 iGruppio 有点笨重，但第二版(见上图)使用了一个旧的 iPhone 外壳，将一堆电路板和插头变成了一个惊人的紧凑设备。

不，最新版本中没有蜂窝调制解调器，但[Michele]已经将所有资源放在了 Github 上，任何想自制手机的人都可以看看他的作品。下面是视频演示。
