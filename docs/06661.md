# 逆向技术:设计和制造 RCA 电视

> 原文：<https://hackaday.com/2014/07/29/retrotechtacular-designing-and-building-rca-televisions/>

虽然说他们不再像过去那样制造东西几乎是老生常谈，但本周的回顾展提供了相当确凿的证据，证明他们过去至少在制造消费电子产品上投入了更多的时间和精力。集合你自制的俏皮机器人，坐在这部 1959 年的电影[前，这部电影名为《T2》，是为 RCA Victor 电视部门的新员工制作的一部叫好又叫座的电影。](http://www.youtube.com/watch?v=2si4pop4kDE)

它可能以叮当声开始，但事情很快变得严重。对于那些致力于为您带来物有所值的最佳电视机的男男女女来说，质量可不是闹着玩的。这种无与伦比的卓越始于孜孜不倦地改善声音和图像质量的 R&D。每个变压器都在五倍额定电压下测试，每个电容器都在两倍额定值下测试。每个开关都经过了一系列的机械测试，包括高压蒸汽浴，以确保它们能够承受住，即使你在南方八月炎热难忍的夜晚把你的电视机拖到门廊上。

![hot august nights](img/37c8f7d3404aa292229472bdf0d7d2b2.png)

机箱设计也同样重要——把一个可以用 60 年的机箱和显像管放在一个便宜的盒子里有什么用呢？来自世界各地的木材因其美观和耐用而被仔细考虑。毕竟，电视机是美国家庭室内家具的核心。这些精心挑选的木材在一系列的烤箱中烘烤，以证明它们可以承受数小时的连续使用。

这种严格的消费者报告式测试在每台电视机的每个部件上进行。每个真空管都用软木塞敲打，每个荫罩都用显微镜检查。一个接一个，每一组的音量都在一个特殊的房间里用一个移动的麦克风进行测试。一旦一套最终完成，他们就把它放在棚子里的转盘上旋转，以测试与其他套发生干扰的可能性。这些只是美国制造业毫无疑问昂贵的几个原因。
