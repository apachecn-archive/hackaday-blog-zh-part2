# 愤怒的小鸟哨兵枪有猪飞

> 原文：<https://hackaday.com/2014/07/04/angry-birds-sentry-gun-has-pigs-flying/>

【Farlei Heinen】在学校有 C 编程课。不想做另一个教科书之外的无聊的千篇一律的项目，他决定做一些不同的事情——他已经建造了一个 Arduino 控制的哨兵枪！

它由两部分组成:哨兵枪本身和一个声纳探测塔，它可以告诉你是否成功地击倒了猪。他在项目的核心使用 Arduino Mega，它控制伺服系统并读取声纳传感器的信息。

哨兵枪使用两个伺服电机来控制上下，左右。装填机制是手动的，用弹力带发射弹丸。然而，发射机制是一个微型(9g)伺服系统，它可以释放弹性和发射弹丸。目标是为孩子们制作的愤怒的小鸟玩具。

这是一个非常酷的项目，如果你有兴趣构建自己的项目，那么[Farlei]甚至已经发布了它的源代码——休息之后，看看它的运行情况吧！
