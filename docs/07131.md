# 纽约制造商博览会上的 Flite 测试

> 原文：<https://hackaday.com/2014/09/21/flite-test-at-nyc-maker-faire/>

[https://www.youtube.com/embed/mXUUuaFOF4M?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/mXUUuaFOF4M?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

今年的纽约市制造者集会可能是一个禁飞区，但这并没有挫伤 [Flite 测试](http://flitetest.com/)工作人员的精神。我们在 Maker Faire 的 R/C 和无人机区域的帐篷外遇到了 Flite Test。[乔希·比克斯勒和亚历克斯·兹瓦达]是广受欢迎的 YouTube 频道的两位主持人。[乔希]也因他的遥控飞机设计而闻名，如[血腥奇迹](http://store.flitetest.com/bloody-wonder-speed-build-kit/)和[简单故事](http://store.flitetest.com/ft-simple-storch-speed-build-kit/)。除了主持，【Alex】就是平面设计师 Flite 测试。他也是一名王牌四轴飞行器飞行员，在很多集里都能看到他驾驶 250 码的四轴飞行器。

大多数 Flite Test 的设计使用 Readi-Board，在美国和加拿大的 Dollar-Tree 商店可以买到。Flite 测试团队正直接与这种泡沫的制造商合作制造这种泡沫，他们的设计在世界各地都有售。

该团队还接受了令人难以置信的挑战，例如一个实际工作的飞行烤面包机，在它存档时制作烤面包片。他们的[飞行煤渣砖](https://www.youtube.com/watch?v=bns5x4pvl5Y)也不容错过。迄今为止，他们最大的挑战是[谷歌项目 Wing +亚马逊 Prime Air mashup](https://www.youtube.com/watch?v=rnSw4KT0AT4) 。该视频展示了一架无人机，它运送了一架无人机，一架无人机运送了一架运送口香糖的无人机。俄罗斯套娃没有这些家伙的把柄！
