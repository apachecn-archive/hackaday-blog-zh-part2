# 电钻驱动的滑板车用腿行走

> 原文:[https://hack aday . com/2015/07/10/drill-powered-scooter-walks-with-legs/](https://hackaday.com/2015/07/10/drill-powered-scooter-walks-with-legs/)

[伊西·斯旺]是一个受欢迎的木工尤图伯，他最近爱上了[西奥·詹森的]动态艺术 Strandbeest。自然，他不得不[做出自己的](https://www.youtube.com/watch?v=mGOled86Sjo)；当然是用他自己的闪光。

如果你不熟悉的话，西奥·詹森的 Strandbeest 是一个由风力驱动的行走动力雕塑。它启发了一个[仓鼠球驱动的 Strandbeest](http://hackaday.com/2011/02/07/hamster-powered-strandbeest/) ，一些漂亮的 [3D 打印的](http://hackaday.com/2011/05/02/3d-printed-strandbeests-made-to-order/)，甚至还有一个[纸工艺版本！从机械上来说，这是一个奇迹——他的 TED 演讲太棒了。](http://hackaday.com/2011/05/01/papercraft-strandbeest-is-a-great-rainy-day-project/)

当[伊西]看到所有这些腿在动时，他知道他必须再造它——所以他想出了这个两条腿的版本，推着他到处走——有点像三轮车，但后轮是……腿？这当然很奇怪，但仍然是一个令人印象深刻的壮举。更不用说他用一个小小的无绳电钻给整个装置供电…

尽管它看起来像加工过的铝，但实际上它是由木头制成的，尽管它确实有一个金属齿轮箱，使用蜗轮来传递钻机的扭矩。我们想看这个的赛格威版本…我们可能要在办公室里用激光切割机…
