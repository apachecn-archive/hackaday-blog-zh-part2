# 复杂线圈背后的简单概念

> 原文:[https://hack aday . com/2012/09/13/simple-concepts-behind-complex-coil guns/](https://hackaday.com/2012/09/13/simple-concepts-behind-complex-coilguns/)

![](../Images/b9ed7d7650889c1ccebda49e014534ee.png "coil-gun-extravaganza")

线圈枪使用电磁线圈来推动金属射弹。从表面上看，它们可能相当复杂。但是当你分解这些概念的时候，就很容易学会了。如果你曾经想过涉足这一领域，这本冗长的线装入门书会有很大的帮助。

coilgun 的基本概念由三部分组成:线圈、电压源和结合两者的开关。在上面的构建中，你可以看到枪的透明枪管上有两个线轴。这些组成了一对加速器，连接到那些提供电压的巨大黑色电容器。他们使用的开关实际上看不出来，但从文章中我们知道它是一个[晶闸管](http://en.wikipedia.org/wiki/Thyristor)；可控硅整流器(2N6504)。

在休息后的视频中，你可以看到这三个部分在一起进行试射。这是漫长旅程的第一步。为了获得更高的抛射速度，你必须增加线圈，如上图所示。但是间隔和时间很快使这个简单的概念变得复杂。但是如果你能解决所有的问题，你最终会得到一些非常棒的硬件。
