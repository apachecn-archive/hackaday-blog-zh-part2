# UMotio:兼容 Arduino 的 3D 手势控制器

> 原文:[https://hack aday . com/2014/04/30/umotio-an-arduino-compatible-3d-gesture-controller/](https://hackaday.com/2014/04/30/umotio-an-arduino-compatible-3d-gesture-controller/)

[![uMotio](../Images/150e30a74d221b9fad902df2e569f1f2.png)T2】](http://hackaday.com/wp-content/uploads/2014/04/gestic.png)

Mooltipass 项目 USB 代码贡献者[Tom]和他的朋友[Ignatius]最近发起了他们的 Indiegogo 活动:认识 3D 手势控制器 [uMotio](https://www.indiegogo.com/projects/umotio-arduino-compatible-3d-gesture-controller#home) (Indiegogo 链接)。由于[Tom]花费了大量的个人时间来帮助 Mooltipass 社区，我们认为感谢他的一个好方法是尝试让他们伟大的开放项目更接近一个传播产品。

正如你在休息后嵌入的视频中看到的，uMotio 是一个即插即用系统(检测为 USB HID 操纵杆和带 CDC 端口的键盘)，可用于许多不同的场景:游戏、计算机控制、运动、音乐等。该平台基于 ATMega32u4 和广泛讨论的 [MGC3130](http://www.microchip.com/wwwproducts/Devices.aspx?product=MGC3130) 3D 跟踪和手势控制器。这允许 0 至 15cm 的检测范围，分辨率高达 150dpi。uMotio 与 Arduino 兼容，因此可以很快适应您的特定项目，尤其是使用其专用的扩展头和库。uMotio blue 甚至集成了内部锂离子电池和蓝牙低能耗模块。
