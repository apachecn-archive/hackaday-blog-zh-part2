# 向日葵热机

> 原文：<https://hackaday.com/2015/08/28/a-sunflower-heat-engine/>

想和孩子们一起做一个很酷的科学实验吗？为什么不造一种类似向日葵的热机呢？Rimstar.org 向我们展示了这一切！

这实际上是一个非常巧妙的小项目。利用咖啡袋中的金属箔碎片，[史蒂夫]创造了一个完美平衡的轮子，看起来有点像向日葵。当箔片花瓣受热时，它们会试图膨胀，但就像双金属条一样，构成包装的材料会以不同的速度膨胀，以拉直受热的花瓣。这会使车轮的重心偏离轴线，从而导致旋转。随着轮子的旋转，箔片花瓣冷却，另一个花瓣被加热，产生连续的运动——至少直到热量供应被带走！

要获得恰到好处的平衡需要做很多工作，但由于巧妙的轮轴设计，调整非常容易。轮子实际上漂浮在一个钉子上，钉子的尖端粘在一块磁铁上，另一端由一系列磁铁悬挂着。这是你能得到的最接近无摩擦轴的东西了！
