# 3D 打印外骨骼帮助这个小女孩发展更正常的身体功能

> 原文：<https://hackaday.com/2012/08/12/3d-printed-exoskeleton-helps-this-little-girl-develop-more-normal-body-function/>

这个 2 岁的女孩患有关节弯曲症，导致她无法移动手臂。但是在一点帮助下，她的肌肉可以得到加强，以实现更正常地使用她的四肢。这已经不是第一次使用外骨骼了，但是[3D 打印部件的出现让骨骼的工作效果好了很多](http://www.youtube.com/watch?v=WoZ2BgPVtA0)。

以前的外骨骼是金属制成的，相当重。当你谈论一个 25 磅的孩子时，每多一盎司都很重要。转向塑料零件减轻了负担。现在，这个结构可以安装在她的躯干上，用橡皮筋来帮助她的运动，直到她的肌肉足够强壮，可以自己做这件事。

当然，对[艾玛]来说，这不是外骨骼。这是她的神奇手臂。
