# 完整的 Siri 家庭自动化控制除了厨房水槽以外的一切

> 原文:[https://hackaday . com/2013/02/06/complete-Siri-home-automation-controls-everything-but-the-kitchen-sink/](https://hackaday.com/2013/02/06/complete-siri-home-automation-controls-everything-but-the-kitchen-sink/)

![complete-siri-home-automation](../Images/a1e90c7c72f52e012b748b6cfadfd48e.png)

[猫王模仿者]花了整整三天时间，但在这段时间里，他设法把家里的一切控制权交给 Siri。所使用的技术是人们熟悉的。运行 SiriProxy 的 Raspberry Pi [监听来自 iPhone](http://hackaday.com/2011/11/21/siri-proxy-adds-tons-of-functionality-doesnt-require-a-jailbreak/) 的命令，并根据【Elvis’】预定义的配置对其进行操作。这里的区别是，它不只是一个单一的设备(阅读:灯)被控制来证明这个概念。他的视频(休息后嵌入)显示他在家中操作一系列设备。

演示从他的车库门被打开和关闭开始。从 YouTube 视频描述中，我们知道他使用了 Trendnet IP 摄像头，看起来其中一个摄像头让他看到他是否记得关闭车库。接下来，他解除了家庭安全系统，如上图所示。从那里，他调整巢恒温器，关掉客厅的灯，并改变电视频道。

我们认为发出语音命令的需求很快就会过时。但除此之外，我们赞扬他的工作，把一切都整合到一个单一的界面。
