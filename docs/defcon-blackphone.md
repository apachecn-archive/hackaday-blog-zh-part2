# 防御状态:黑色电话

> 原文：<https://hackaday.com/2014/08/27/defcon-blackphone/>

[https://www.youtube.com/embed/h1Q4FtHL-RU?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/h1Q4FtHL-RU?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

尽管到处都是技术人员和用便携式设备做有趣事情的人，但你不会想在 DEFCON 的四分之一英里范围内有一个活动的无线电。你手机上的应用程序一直在向互联网泄露个人数据，DEFCON 的绵羊墙的人非常成功地获得了几千个电子邮件帐户的用户名和密码。

Blackphone 旨在解决这个问题，所以当我们在 DEFCON 上遇到 Blackphone 的一些成员时，我们很有兴趣快速瞥一眼他们的设备。

Blackphone 的核心功能来自其名为 PrivatOS 的操作系统。这是 Android 4.4.2 的一个分支，据说可以密封其他手机中的后门。还有一个来自 [Silent Circle](https://silentcircle.com/) 的应用包，让 Blackphone 能够打加密电话、发短信(带文件共享)以及加密和密码保护的联系人列表。

Blackphone 的硬件令人印象深刻；四核 Nvidia Tegra 为您的应用程序、视频和播放 2048 提供了所有需要的电力，2000 毫安时的电池应该可以提供足够的电力让您度过一两天(特别是因为您可以关闭内核)，常见的前置/后置摄像头、GPS、802.11bgn 和 GSM 以及 HSPA+/WCDA 无线电意味着这款手机可以在大多数网络上使用。
