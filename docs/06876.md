# 超好的数控路由器建设留下了一点希望

> 原文:[https://hack aday . com/2014/08/21/super-nice-CNC-router-build-leaves-little-to-desired/](https://hackaday.com/2014/08/21/super-nice-cnc-router-build-leaves-little-to-be-desired/)

![Aluminum CNC Router](../Images/d6074a74802ad6aff5339e66251ebcd3.png)
【恩佐】来信告诉我们他最近完成的[数控路由器](http://www.angelfire.com/electronic2/b94/cncrouter/cncrouter.htm) ( [翻译](https://translate.google.com/translate?hl=en&sl=it&tl=en&u=http%3A%2F%2Fwww.angelfire.com%2Felectronic2%2Fb94%2Fcncrouter%2Fcncrouter.htm))。这是一个优秀的高品质，全铝建设，没有偷工减料。工作范围是相当大的 340×420 毫米，Z 方向 80 毫米。线性滚珠轴承有助于平稳移动，带有轴向和径向轴承的丝杠为精确和可重复的运动提供了坚实的基础。

![Aluminum CNC Router](../Images/295486e00904c0a28431f236d793e166.png)

过去我们在 Hackaday 上有一堆 [CNC 路由器项目](http://hackaday.com/2013/07/26/cnc-router-built-with-8020-rail/)，包括其他制作精美的[铝制项目](http://hackaday.com/2013/10/24/a-very-professional-homemade-cnc-router/)，但是【Enzo】是唯一一个在他的计算机和机器控制系统上花了和在 CNC 机器本身上一样多精力的人。这款电脑运行 Windows 和 Mach3，是一款一体式产品，从一台破旧笔记本电脑的旧液晶显示屏开始。随着重复使用的屏幕，一个非常小的 ETX 形状因子主板被塞在一个定制的有机玻璃外壳内。紧凑型闪存卡满足存储要求。

在显示器下面是另一个漂亮的定制外壳，里面装有步进电机驱动器。前面板上有 3 个开关，用于将 main 的电源输出到 PC、主轴和一个 AUX，以备将来使用。在后面板上有用于每个步进电机、限位开关和 PC 连接的 [D-sub](http://en.wikipedia.org/wiki/D-subminiature) 连接器。哦，对了，顺便说一下【恩佐】设计了他自己的[双极电机驱动器](http://www.angelfire.com/electronic2/b94/stepperdriver/stepperdriver.htm) ( [翻译](https://translate.google.com/translate?sl=auto&tl=en&js=y&prev=_t&hl=en&ie=UTF-8&u=http%3A%2F%2Fwww.angelfire.com%2Felectronic2%2Fb94%2Fstepperdriver%2Fstepperdriver.htm&edit-text=))并把设计发送出去制造。这些板使用 A4989 IC 和 mosfets 来控制电机。图表在他的网站上，以防你想自己做一些。
