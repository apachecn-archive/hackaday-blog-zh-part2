# [Furze]用火焰手套点燃一切

> 原文:[https://hack aday . com/2014/05/30/furze-sets-fire-to-everything with-pyro-gloves/](https://hackaday.com/2014/05/30/furze-sets-fire-to-everything-with-pyro-gloves/)

![Pyro Build](../Images/4b2e2de70e988370070183b2a4e0315c.png)

克雷齐曼[柯林·福尔泽]回来了，这一次他用他的火焰手套点燃了一切。虽然 Hackaday 的读者已经是一群有眼光的人了，但这是一个我们一直收到提交的版本，并且经常以[为特色。这将是一个非常令人印象深刻的建设，以超越其他火灾隐患。但是，就像他的](http://hackaday.com/2013/08/18/wrist-mounted-flamethrower-on-the-cheap/)[气动狼獾爪](http://hackaday.com/2014/05/16/pneumatic-wolverine-claws-are-quite-possibly-the-best-thing-ever/)和他的[电磁靴](http://hackaday.com/2014/05/23/electromagnetic-boots-for-all-your-upside-down-needs/)，【Furze】知道如何制造疯狂，然后上演一出好戏。

Pyro build 是[Furze]的[3 集庆祝 x 战警盛典的一部分，【T1]是对超级英雄领域的致敬，与新 x 战警电影的上映相吻合。[Furze]从一个装有两个电磁阀的定制储气瓶开始:一个用于指示灯，另一个用于大爆炸。他还贴上了压电元件和 AA 电池，放在一个舒适的小容器里。组件中体积较大的部分放在背包中，如预期的那样，连接到腕戴设备上。然而，这个火焰炮是我们在这里遇到的唯一一个。](http://www.colinfurze.com/x-men.html)

我们在 Hackaday 上展示的建筑通常在同一个手腕上结合了指示灯和燃料库。危险，是的。傻？大概吧。[Furze]在他的版本中把它们分开了，然而，指示灯在一个手腕上，燃料在另一个手腕上。向后轻推一个手腕来点燃指示灯，然后瞄准第二个手腕来释放一个火热的嗖嗖声。

然而，这种设置可能并不安全，尤其是在你无法瞄准的情况下。不用说，不要在家里尝试这个。但是，请务必观看下面的视频，包括幕后的构建分解。
