# SNES 耳机大声疾呼蓝牙控制

> 原文：<https://hackaday.com/2013/04/01/snes-headphones-scream-out-for-bluetooth-control/>

这是一个散发着书呆子气息的建筑。这是[一个改装成耳机](http://www.youtube.com/watch?v=SIVYzjTu5wE)的 SNES 控制器，直接来自【lyberty5】的车间。

制作开始时，他从朋友 SNES 那里偷了一个控制器，小心翼翼地将按钮和 d-pad 从塑料框架中取出。PCB 被切成两半，剩下的塑料在一些环氧树脂的帮助下被精心制作成圆形扬声器外壳。热熔胶，可能还有几片苯乙烯。

结果是一副完美成形的 SNES 耳机，其制造质量达到了我们所见过的最好水平。不幸的是，虽然按钮仍然附着在 PCB 上，但它们什么也不做。我们认为一个小的蓝牙适配器——或者甚至是一套带音量和播放控制的蓝牙耳机——将会是这个 20 年前的糖果按钮的一个很好的用途。

尽管如此，一个令人敬畏的构建，[lyberty5]通过构建这些美妙的耳机真正展示了他的手艺。休息后你可以看到构建的延时。
