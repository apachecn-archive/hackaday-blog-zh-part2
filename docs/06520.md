# 最不实用的档位指示器

> 原文:[https://hack aday . com/2014/07/09/a-最不切实际档位指示器/](https://hackaday.com/2014/07/09/a-most-impractical-gear-position-indicator/)

几年前，[Pat]为他的铃木 V-Strom 送去了[一个非常好的档位指示器](http://hackaday.com/2012/02/09/suzuki-v-strom-current-gear-indicator/)。凭借一个七段显示器、一个小型微控制器和直接连接到自行车 ECM 的电线，它足以完成工作，而且比售后齿轮指示器便宜得多。一个简单、优雅的解决方案，可以很好地完成一项工作。这还能更好吗？

“更好”是一个相对的术语，根据你优化的目标，一个更复杂的解决方案可能更好。[Pat]认为让他的摩托车价值翻三倍是一个有价值的目标，所以[他用示波器](http://whiskeytangohotel.com/mdo3000)取代了七段显示器。这是世界上唯一的基于示波器的摩托车档位指示器，现在[Pat]需要一根非常非常长的延长线。

像早期更实用的版本一样，这个版本读取自行车 ECM 的电压来确定自行车的档位。然后，当前档位显示在 Tek MDO3000 上，微控制器上有两个 PWM 引脚。实用？不，但它看起来很酷。下面视频。
