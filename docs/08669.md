# 本周失败:简单奶酪？打印机说没有

> 原文:[https://hack aday . com/2015/03/31/本周失败-轻松奶酪打印机-说不/](https://hackaday.com/2015/03/31/fail-of-the-week-easy-cheese-printer-says-no/)

嗯，[这是及时的](https://www.youtube.com/watch?v=uCy0NEbJf4s)。我们在今年的中西部 RepRap 节上看到了许多东西，包括打印机和材料。我们向您介绍了通过远程挤压机设置、注入铜和其他金属的强韧细丝以及由 K'NEX 制造的打印机而实现的美味产品。然而，没有人用罐装奶酪印刷，这也许是有原因的。

[Andrew]这里创造了一个 3D 打印的手臂，它可以将一罐气溶胶奶酪状物质固定在适当的位置。一个马达使支架将喷嘴移动到一边，分配粘性物质。首先，他把它卷成一团喷在饼干上。这很好，直到是时候离开饼干了。[安德鲁]后来试图建立四个俗气的墙让我们欢呼。休息之后你就能明白我们的意思了。

有几个问题在起作用。有时附加物会直接从壶嘴的末端掉下来。其他时候，罐中的空气会中断流动，就像手动操作时一样。每隔一段时间，喷口似乎离基底太近了。

你怎么看待奶酪印花的可行性？如果挤压是远程进行的，奶酪通过一个更薄的尖端挤压，效果会更好吗？冷却打印床会有帮助吗？让我们知道。
