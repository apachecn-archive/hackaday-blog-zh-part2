# 为你的 IPad 制作一个电容衣夹键盘

> 原文：<https://hackaday.com/2014/11/07/make-a-capacitive-clothespin-keyboard-for-your-ipad/>

即使所有的优化和新技术的风格，键盘是一个很难取代的东西。触摸屏只是不提供将我们与媒介联系起来的触觉反馈。[Adam Kumpf]通过构建自己的键盘界面来与 iPad 钢琴应用程序配合工作来解决这一问题，所有这些都来自你可能在厨房里找到的工艺材料。

要自己制作，你首先需要[一堆衣夹，它们最终会成为你的钥匙](http://www.instructables.com/id/Clothespin-Piano-for-iPad/)。[Adam]展示了如何用一些基本的橡皮筋将分开的两半晒衣夹缝合到一块纸板上。这些拉紧键，这样它们可以在放在它们下面的钢笔或铅笔上来回摇动。当你按下一端时，另一端会抬起，导致另一端的插销按下 iPad 的相应按键，就像钢琴里的锤子一样。用一点导电铝箔包裹在接触的一侧，你就可以快速解决摇滚肖邦的渴望。
