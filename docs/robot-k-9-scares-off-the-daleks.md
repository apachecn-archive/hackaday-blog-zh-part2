# 机器人 K-9 吓跑了达雷克斯

> 原文：<https://hackaday.com/2015/05/03/robot-k-9-scares-off-the-daleks/>

[bithead 942]对广受欢迎的《神秘博士》系列的热爱让她开发了第四代博士机器人伴侣的[复制品。它的名字叫 K-9，仅用了 4 个月就从零开始建造。它的外壳由 HPDE 制成——一种轻质且可弯曲的塑料。一个](https://bithead942.wordpress.com/2015/05/02/k-9/)[定制的塑料弯曲器](https://www.youtube.com/watch?v=wVEf7PfuKxo)被制造出来以获得恰到好处的角度，定制的激光切割部件被用在不同的地方。

它的框架由[铝槽](https://www.servocity.com/html/actobotics1.html)组成，里面装满了有趣的电子设备。一个带 XBee 盾的 arduino 控制着远程语音、*激光和眼睛传感器。另一个 arduino 与电机护罩配对，以控制颈部运动的线性致动器。一个树莓皮可以让液晶屏幕保持有序。*

 *伙计们，我们还没完。因为这只小狗是无线电控制的，所以需要一个定制的控制器。Sparkfun 的 Fio 与另一个 XBee 配对，与一个 16×2 的 LCD 和各种其他电子设备一起使用，以保持机器人在一个无形的皮带上。

一定要去看看博客网站，因为它详细介绍了用于构建这个复杂但令人敬畏的项目的各个部分。
