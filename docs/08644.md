# 3D 打印台灯跟随特斯拉走向无线

> 原文：<https://hackaday.com/2015/03/28/cordless-desklamp/>

如果没有尼古拉·特斯拉，我们在电力方面会落后很多。因此，为了向最伟大的发明家之一致敬，[David Choi]决定制作他自己的无线[特斯拉台灯！](http://3dprint.com/46711/tesla-desk-lamp/)

不出所料，[大卫]是[尼古拉]的忠实粉丝，一直受到他的生活和实验的启发——尤其是他热爱无线电源。自从他从远处看到一个特斯拉线圈点亮了一个灯泡，他就被迷住了。他甚至给自己的猫取名特斯拉。

有趣的是，[大卫]实际上高中物理不及格，但几年后在卫斯理大学上学时决定以此为职业。这并没有在他毕业时停止，他还在业余时间学习了电子设计——这是他了解共振的地方。

为了应用他所学到的知识，他创造了一个非常独特的无线台灯。不要让图片欺骗你；其实是 3D 打印的！它使用一种复古的“老式”灯泡，通过 6.5 兆赫的信号将电力无线传输给它。做好无线部分相对容易，因为一旦他计算出所需的线圈数量，他所要做的就是 3D 建模铜线的轨道。
