# 重复使用塑料瓶的超级简单方法

> 原文：<https://hackaday.com/2014/06/20/super-simple-way-to-re-use-plastic-bottles/>

我们的一个线人刚刚发来一个非常简单的小技巧，可以重复利用塑料瓶，把它们变成有用的塑料线。它是俄语的，但是在翻译的英语字幕和视觉演示之间，它很容易理解。

YouTube DIY survival ist[адвокатегоров]制作了许多非常酷的教程视频，从制作刀具，用加热的 PVC 管成型零件，用硬币制作戒指，甚至如何增加你的土豆农场的产量(什么？).在这篇文章中，他向我们展示了如何使用一小块铝型材制作一个非常简单的夹具，它可以将塑料瓶切成长条状，然后可以用于许多不同的事情。

夹具是可调的，你可以轻松地生产不同宽度的丝带。你为什么想要塑料丝带？他用它作为捆东西的细绳(非常结实)，以及他的工具的把手——只需将它紧紧地缠绕在某个东西的手柄上，然后使用热风枪，你就可以在适当的位置形成一个塑料手柄。

再说一次，视频是俄语的，但是你可以翻译字幕——这有点帮助。
