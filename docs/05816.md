# 自制重力灯，持续时间不长却证明了概念！

> 原文:[https://hackaday . com/2014/04/16/自制-重力-光-不持久-但证明-概念/](https://hackaday.com/2014/04/16/homemade-gravity-light-doesnt-last-long-but-proves-the-concept/)

![gravity light](../Images/0d3ef80fbc10cd80f8e5977d3897d9b6.png)

受到十分之一瓦重力灯的启发后，[史蒂夫·杜弗兰]决定尝试自己制作一个来验证这个概念。

Deciwatt 的重力灯是一种创新设备，专为第三世界国家设计，帮助消除煤油灯等昂贵的照明设备。它在滑轮上的重量很小，可以在 3 秒钟内被提起。在缓慢下降的过程中，重物提供 25 分钟的照明！它经济实惠、可持续、可靠。这也是令人印象深刻的机械，这正是为什么[史蒂夫]决定尝试自己制作。

他用了一个 LED，一个小型 DC 发动机，几块木头，一个旧自行车车轮，一些自行车链条和几壶水。水与链条相连，链条套在自行车上最小的齿轮上。发电机由环绕在轮圈外侧的皮带驱动。这使得电机有足够的速度来为 LED 发电。他目前的设计只持续了大约 3 分钟，但他已经在进行第二次迭代了。像这样的测试系统真的会让你对真正的重力光所付出的努力有所欣赏。
