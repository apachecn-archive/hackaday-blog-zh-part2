# 模块化多翼核心在多个方向飞行

> 原文:[https://hack aday . com/2014/11/27/modular-multi copter-core-flys-in-multi-orientations/](https://hackaday.com/2014/11/27/modular-multicopter-core-flies-in-multiple-orientations/)

[Ioannis Kedros]声称对建造多旋翼无人机的游戏相当陌生。看看他的最新作品，你永远也不会知道。是的，我们正在谈论这里看到的四轴飞行器，但它是机器的核心，非常有趣。他想出了一个 PCB 集线器，允许多个方向用于同一块电路板。这些包括三轴飞行器和四轴飞行器，对于不同的应用具有不同的支柱角度。

![multicopter-hub-pcb](../Images/13db0c5e3970b41baf3b9297da024b1b.png)PCB 的丝网印刷有虚线，显示一对电机支架可能的不同角度。对于传统的四轴飞行器飞行来说，其中一组就是一个完美的“X”。另一组减小了前后支柱之间的角度，用于更高性能的四轴飞行，而最后一组用于三翼机设置。

我们建议您看看[Ioannis']项目报告，看看您是否对这个特定的应用程序感兴趣。在下 PCB 订单之前，他的设计技术通过了所有可能的检查方式。如果你想避免被愚蠢的错误烧伤，这个过程是无可替代的。
