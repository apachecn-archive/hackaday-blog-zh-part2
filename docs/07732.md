# 【神奇科学】简易电动火车

> 原文：<https://hackaday.com/2014/12/04/amazing-sciences-simple-electric-train/>

制作电磁铁很简单，只需在钉子上缠绕一些电线，然后将电线绑在电池的两端。完成后，你可以拿起一些回形针——它很好地展示了这个概念，但它可能需要更多的活力。【神奇科学】已经做到了，[制作“电动火车”](https://www.youtube.com/watch?v=J9b0J29OzAU) (YouTube 链接)。所有需要的是一些盘绕的铜线，一个电池和足够薄的磁铁穿过线圈。磁铁卡在电池的两端。把电池放在线圈里面看热闹！流经线圈的电流产生的电磁力推动附着在电池上的磁铁，推动电池前进。

[惊人的科学]玩了一点设置。将线圈的两端连接在一起，电池将循环运行，直到耗尽。添加一个小山丘，甚至另一个电池/磁铁组合，看他们去！我们甚至可以自己制作一个版本，在这个假期带着它去家庭聚会——它简单、有趣，并且可以在我们大口喝蛋奶酒的时候教给年轻人科学知识。
