# 原子钟是如何工作的？

> 原文：<https://hackaday.com/2012/06/12/how-do-atomic-clocks-work/>

![](img/e9ba6e6cc988a7d254d4a083fd2347fa.png "Screen Shot 2012-06-12 at 7.53.34 AM")

[Bill Hammack]又名[Engineerguy]再次带来了另一个精彩的信息视频。这一次，他准确地描述了原子钟是如何工作的。他从一个很好的比喻开始，果冻被戳时会抖动。他解释了这是如何类似于时钟内的石英晶体由于我们给它的电“戳”而振荡。他继续解释 GPS 卫星在确定地面上的物理位置时是如何依赖这种精确度的。

像往常一样，[Bill]在快速传递信息和提供完整细节方面做得非常出色，同时还保持了足够的简单性，即使是那些不熟悉技术的人也能理解。
