# 虚拟电池可以让你使用交流适配器

> 原文:[https://hack aday . com/2013/06/13/dummy-batteries-let-you-use-a-AC-adapter/](https://hackaday.com/2013/06/13/dummy-batteries-let-you-use-an-ac-adapter/)

当电池供电的消费电子产品不包括连接外部电源的方式时，我们会感到沮丧。如果可以避免的话，我们尽量不要购买一次性碱性电池，这个虚拟电池交流适配器黑客将有助于我们的任务。

上图所示的电池盒用于电动婴儿秋千。它接受 C 号电池(谁有那些只是躺在那里？)并且缺少连接壁式电源适配器的筒式插孔。[詹森·史密斯]提到您可以通过将正极和地线直接连接到导体弹簧来解决这个问题。但是，如果您确实想使用电池电源，使用虚拟电池会更容易移除适配器。

每个橙色的假人都是一个木制销钉，两端各有一个螺钉。螺丝用一根跳线连接，使两个端子短路。这完成了电池盒中的电路，并允许他从底部的适配器电池为一切供电。该适配器使用 LM317 可调电压线性调节器。他使用固定的电阻值来设定他的目标电压。该设备应该相当宽容，因为电池电压开始高于印刷值，并随着电池耗尽而下降。

这种技术已经存在很长时间了。我们最喜欢的一个黑客是将苹果 Magic Trackpad 转换成 USB 电源。
