# THP 半决赛:太阳能系统

> 原文：<https://hackaday.com/2014/09/28/thp-semifinalist-solar-energy-system/>

建造一个太阳能装置并不像买几块电池板，把它们连接到电池上，然后把一个逆变器放进去那么简单。为了最大限度地利用你昂贵的电池板，你会想看看所谓的最大功率点跟踪。太阳能电池板有一个 IV 曲线，这随着他们得到多少阳光而变化。为了最大限度地利用一组电池，你需要确保从你的电池中获取最大的能量。

[【纳撒尼尔】的太阳能发电机](http://hackaday.io/project/1981)就是这么做的。它可以处理高达 500 瓦的功率，从一组太阳能电池中吸取能量，并将其释放到电池中。那不是一切；该项目还有一个用于测量和显示所有相关信息的微控制器，以及一些插入几个 DC 负载的终端。

虽然太阳能发电机是为脱离电网的应用而设计的，但这可以很容易地以低廉的价格扩大家庭安装。如果你想要超过 500 瓦左右，你会希望看到一个更大的控制器，但对于任何低于这个功率，[内森]已经涵盖了你。
