# 基于计时器的冷却帮助您的网络度过夏季

> 原文:[https://hackaday . com/2012/07/02/timer-based-cooling-helps-your-network-survive-the-summer/](https://hackaday.com/2012/07/02/timer-based-cooling-helps-your-network-survive-the-summer/)

![](../Images/d57e62be8a2f0e0eb28de9a6f049241a.png "network-cooling-rack")

感谢[萨米]制作的视频，微笑着开始你的一周。它展示了他为自己的网络设备制作的冷却架。该项目是出于必要而开发的，因为夏季天气导致他的调制解调器和路由器变热，在某个时候其中一个会关闭并拒绝再次工作几个小时。我们自己还没有遇到过这种情况，但知道设备内置了过热保护装置是件好事。

他的解决方案是建造一个机架，在两个设备的上方和下方提供风扇冷却。正如他的大多数项目一样，我们认为制作视频(休息后嵌入)是一半的乐趣。除了用转盘玩一些额外的特殊相机效果，他给了我们一个很好的整体构建视图。底座包括垫片和 velcro 条，用于将设备固定在一对排气扇上方。机架每个角落的支架将第二对风扇悬挂在设备上方。但如果没有一些定制的 LED 效果，它看起来就不会那么好。

这纯粹是定时驱动。这是一个插件模块，使用机械定时来切换电源。但是一些创造性的电路(或小型微控制器)可以实现基于温度的开关。
