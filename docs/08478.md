# 动画 ASCII 流体动力学模拟器是复古酷

> 原文:[https://hack aday . com/2015/03/07/animated-ascii-fluid-dynamics-simulator-is-retro-cool/](https://hackaday.com/2015/03/07/animated-ascii-fluid-dynamics-simulator-is-retro-cool/)

ASCII 艺术在我们心中占有一席之地。如果你足够幸运，在图形用户界面(GUI)出现之前就开始接触计算机，那么你会记得花了几个小时来巧妙利用 ASCII 字符，使屏幕上的图形尽可能逼真。

虽然这个[动画 ASCII 流体动力学模拟器](http://www.youtube.com/watch?v=QMYfkOtYYlg)可以追溯到 2012 年，但不分享实在是太酷了。它是[国际混淆 C 代码竞赛](http://www.ioccc.org/) (IOCCC)的产物。一年一度的竞赛，目标是尽可能写出最令人困惑的 C 代码——利用 C 编程语言中的漏洞和歧义来混淆(隐藏)程序的目的。基本上，做学校里教你不要做的事情。你可以在这里看一下[的源代码](http://www.ioccc.org/2012/endoh1/)。

我们肯定程序员(远藤佑介)会是第一个承认，这样一个低分辨率的模拟器没有实际用途，但我们给它一个 A+在复古酷部门反正。(更不用说，源代码太混乱了，甚至无法评论)休息之后，看看视频中的动画 ASCII 图形。
