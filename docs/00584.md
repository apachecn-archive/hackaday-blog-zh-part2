# 加州理工学院装备机械手的机器人

> 原文:[https://hack aday . com/2012/05/30/cal techs-manipulator-arm-equired-robot/](https://hackaday.com/2012/05/30/caltechs-manipulator-arm-equipped-robot/)

![](../Images/ab57cb43f6e0eb721efc2e0190714223.png "caltech-rover-arm")

[Justin]写信告诉我们关于[漫游者](http://crt.caltech.edu/index.htm)的情况，他的加州理工学院团队已经参加了[美国宇航局的探索机器人操作比赛](http://www.nianet.org/RoboOps-2012/index.aspx)。他们闪亮的时刻在本周晚些时候，但休息后你可以看到一些测试片段。

上图中的操作员正在使用一个控制器，这是一个机械臂的比例模型，有两个摄像头提供反馈。其中一个监视器显示了手臂本身的反馈，提供了抓取器的视图。另一种是从机器人身上拍摄工作区域的广角镜头。该臂具有由伺服电机驱动的六个自由度。控制器是从丙烯酸树脂上切割下来的手臂激光的复制品。在每个关节处都有一个电位计，其值用于确定框架的位置。

起初，我们认为这比使用游戏控制器更容易疲劳，也不方便。但是当我们看到手臂的灵活性时，很明显操纵杆和按钮只会让事情变得更困难。
