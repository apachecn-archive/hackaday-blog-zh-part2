# Hackaday 奖参赛作品:合成孔径多光谱成像

> 原文：<https://hackaday.com/2015/07/22/hackaday-prize-entry-synthetic-aperture-multispectral-imaging/>

他的教授给【康拉德】布置了[建造合成孔径多光谱成像设备的任务。这是一个有趣的挑战，涉及编程、图形和一点电气工程。](https://hackaday.io/project/6730-synthetic-aperture-multispectral-imager)

在一个看起来像哑巴机器人的花哨的黄色盒子里，塞着五个树莓 Pi，一个 TP-Link 以太网交换机，三个树莓 Pi NOIR 相机和一个 Flir 轻子热感相机。有了三个摄像机，可以使用不同的技术来改变被记录的任何东西的焦距——这是构建的合成孔径部分。通过添加不同的过滤器——IR pass、UV、visual 和 thermal，这款相机可以记录大范围波长的图像。

[康拉德]已经提出了一个完全模块化的工具箱，允许大量的成像实验。通过移除滤镜，他可以追踪 3D 物体。所有的过滤器就位后，他可以缩小他能记录的光谱范围。这是一个完全模块化的移动实验室，我们迫不及待地想看看这个小盒子到底能做什么。
