#### 不要将激光瞄准剩余的眼睛

![laser](../Images/d32f6e16a6c188802a06be80eab65b79.png)

在 reddits 上，[CarbonGod] [认为他的激光笔](http://www.reddit.com/r/electronics/comments/15w0zv/)有点过了。他的红色激光笔有一个标签，上面写着输出功率小于 5 毫瓦。唯一的问题是，它融化了黑色塑料，并将热电偶加热到 140 华氏度。[碳神]正在向一位工程师朋友乞讨、借或偷电表，但在此之前，[我们已经从[酸葡萄]](http://www.reddit.com/r/electronics/comments/15z6bp/follow_up_post_about_the_incorrectly_labeled/)那里得到了测量结果。他的激光器输出 105 毫瓦(红色)、56 毫瓦(蓝色)和 53 毫瓦(绿色)。

而这些被标为输出< 5 mW 的激光器却不是。现在，如果我们知道在哪里可以买到这些功率过大的激光器…

#### 不可能找到这个高清的视频

![rickroll](../Images/1153a308e0831dea450e1da870c3de07.png)

[扎克]创造了[一个物理 rickroll 设备](http://zachrattner.com/rroll.php)。这是一个 Arduino 和一个 MP3 防护罩连接到一个超声波传感器。当有人走到设备六英尺以内时，Arduino 就会开始播放*永远不会放弃你。*当那个人走开时，歌曲暂停，只有当超声波传感器检测到其他东西时才重新开始。有一个[令人捧腹的视频](http://www.youtube.com/watch?v=dQw4w9WgXcQ)是关于【扎克】触发他的物理 rickroll 设备，或者你可以在构建页面查看。

#### 嘿，你！写点代码！

![react](../Images/48270fb61d6493719b6d8a75613bf086.png)

[William]来信告诉我们一个叫做[ReactOS](http://www.reactos.org/en/index.html)的项目。该项目的目标是创建一个免费的开源操作系统，它可以与 Windows XP 进行二进制比较。是的，这个项目已经存在很长时间了，但是随着[微软放弃对 XP](http://www.microsoft.com/en-us/windows/endofsupport.aspx) 的支持，ReactOS 团队可以*真的*使用一些开发人员很快就能推出测试版。如果你知道一些底层的 Windows 东西，但从未参与过开源项目，那么[去看看开发者的 wiki](http://www.reactos.org/wiki/Welcome_to_the_ReactOS_Development_Wiki) 。
