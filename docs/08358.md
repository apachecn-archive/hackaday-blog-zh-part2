# SNES 控制器修改为完全无线

> 原文：<https://hackaday.com/2015/02/20/snes-controller-modified-to-be-completely-wireless/>

[Pat]正在寻找一种无线控制消防电视装置的方法。他本可以选择众多可能的消费产品中的一种，但他决定更进一步。他修改了一个装置，使之适合最初的 SNES 控制器 T1。所有的按钮都是功能性的，控制器甚至配有无线充电器。

[Pat]最初推出了一款蓝牙视频游戏控制器，主要用于在平板电脑上玩视频游戏。最初的控制器看起来有点像 XBox 控制器。[帕特]撕开这个控制器，设法把内脏塞进一个原装的 SNES 控制器。他甚至不需要拆除原来的 SNES 印刷电路板。[Pat]提到重新连接原来控制器上的所有按钮相当繁琐，但最终并不太困难。对原始控制器的唯一外部可见的修改是为电源按钮做的一个小孔。

为了让这个单元完全无线化，[Pat]还安装了 Qi 无线充电模块。现在，将控制器放在充电板上，只需大约 45 分钟就可以给小型 LiPo 电池充电。这个控制器将会是一个 [RetroPi](http://hackaday.com/2014/06/01/original-nes-guts-upgraded-with-retropie/ "RetroPi NES mod") 或其他类似项目的完美补充。如果你不喜欢蓝牙，你可以试着用一个[罗技接收器](http://hackaday.com/2013/11/19/wireless-snes-controller-for-logitech-receiver/ "Wireless SNES controller")来代替。
