# moostar——Zoltar 的算命驼鹿仿制品

> 原文:[https://hack aday . com/2013/05/27/moostar-算命-驼鹿-敲落-of-zoltar/](https://hackaday.com/2013/05/27/moostar-fortune-telling-moose-knock-off-of-zoltar/)

![zoltare-the-fortune-telling-moose](../Images/9b74cb4ec44eec155807c2ee28556f26.png)

来认识一下 Moostar，[受 Zoltar 启发的算命驼鹿](http://www.instructables.com/id/Moosetar-Zoltar-Fortune-Teller)。你还记得佐尔塔吗，那个让汤姆·汉克斯成为电影明星的投币算命师？也许你没有看过[的电影](http://en.wikipedia.org/wiki/Big_(film))，但是【Sketchsk3tch】看了，他相对轻松地将这部电影拼凑成一个全公司会议。

如果你擅长为你的项目选择零件，事情会简单很多。他从一只唱歌的圣诞驼鹿开始，一个迷你等离子球充当水晶球，不知何故发现了一个收藏家的篮球盒，它的大小正好适合围栏。

电子设备也组装得非常好。他用一台热敏打印机吐出财富——这实际上是给员工的安全提示，因为那是他工作的部门。硬币接收器是一个有趣的部件，他尝试了两种现成的方法让驼鹿说话。第一个如下所示，使用 Arduino Wave shield 播放的预先录制的消息。这是通过使用[一种电子媒体 2](http://hackaday.com/2012/07/11/emic2-text-to-speech-module/) 文本到语音模块改进的，它真正打开了驼鹿的聊天范围。
