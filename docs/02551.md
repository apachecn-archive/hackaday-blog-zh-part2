# 建立和安装您自己的高端循环电表

> 原文：<https://hackaday.com/2013/01/17/build-and-install-your-own-high-end-cycling-power-meter/>

骑自行车的功率表会花掉你一大笔钱。[Keith]报价从 1500 美元到 4000 美元不等。我们认识几个认真的骑自行车的人，他们会三思而后行，甚至不会考虑把这种投资放在自行车的配件上。但是如果你有时间，[Keith]会向你展示如何制作和安装你自己的自行车功率表。

上面的链接是[Keith]一路走来所有帖子和视频的综述。我们在休息后嵌入了他的介绍视频，在视频中他讨论了项目的目标。该系统允许独立测量每条腿的功率。这是通过使用曲柄上的应变仪来监控扭矩来实现的。该数据与节奏测量(骑车人转动曲柄的速度)相结合，这是计算骑车人功率输出所必需的。

零件清单的价格约为 350 美元。这还不包括他用来测试和校准他的计算的设备。
