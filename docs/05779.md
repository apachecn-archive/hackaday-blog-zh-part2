# 平行缝导电线

> 原文：<https://hackaday.com/2014/04/12/sewing-conductive-thread-in-parallel-lines/>

[Cynthia]分享了一个很棒的视频，视频中的[机器使用一个通常是大多数机器标配的捆绑脚将平行的导电线](http://www.youtube.com/watch?v=P-wFOYC6cQY "Part 2 Creating 3 parallel lines of conductive thread on a machine")缝合到丝带上。当使用 NeoPixel 等可寻址 led 来相当精确地排列接地、数据和正极时，这种技术可能特别有用。将导电线缝在丝带上也使它更容易连接到许多衣服或纺织品上，也更容易更换或重复使用。

这种方法非常简单，主要是利用绳索脚上的凹槽来引导导电踏板，并确保间距均匀。使用之字形针脚将两条线缝在一起，相距约 3 mm。第三条线是分开缝的，以确保缝线不会打断前两条线。在视频中，使用了带条纹的丝带，它有轻微的槽，另外有助于线留在原位，下水道留在目标上。
