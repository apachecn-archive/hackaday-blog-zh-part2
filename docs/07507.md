# 如何使用电传打字机

> 原文:[https://hack aday . com/2014/11/11/retro tech atacular-how-to-tele typewriter/](https://hackaday.com/2014/11/11/retrotechtacular-how-to-teletypewriter/)

本周，你将通过这部 1963 年的军事训练电影了解 AN/GRC-46 的来龙去脉。你会问，AN/GRC-46 是什么？为什么它是一个完整的移动战术遮蔽无线电电传打字机钻机能够连续波，语音和电传传输。

影片涵盖了组成 [AN/GRC-46](http://www.rattrig.com/AN-GRC-46/46.htm) 的组件、它们的功能、系统的能力以及正确的操作程序。小小的 1400 磅里发生了很多事情。钢棚，所以每一块都将从地面上推出。

你将熟悉电压分配系统和 AN/GRC-46 的附件。接下来是 RF 信号传输和调制器执行的[频移键控](http://en.wikipedia.org/wiki/Frequency-shift_keying) (FSK)的简短课程。讨论了发射机和接收机的范围，以及前面提到的能力:使用键控器的连续波操作、声音操作、电传操作和电传纸带的再穿孔。

最后，你会观察到一个经验丰富的操作员联系并发送电传信息，其动作如此小心谨慎，以至于接近催眠状态。当他不发送信息或在海滩上散步时，通常可以发现他在清洁和/或润滑发射机过滤器。
