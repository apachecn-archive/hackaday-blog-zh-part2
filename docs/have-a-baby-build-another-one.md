# 生个孩子？再建一个！

> 原文：<https://hackaday.com/2012/10/14/have-a-baby-build-another-one/>

![](img/551b00f01493a9cafdba650171660e8b.png "bot")

自从他的女儿出生后，[马库斯]就一直记录着对人类行为的观察。不管听起来如何，这种情况并不奇怪；发展心理学领域充满了这类研究。正是[Markus]对这些数据的处理使他的项目独一无二。他正试图使用[随机学习来模拟他女儿](http://letsmakerobots.com/node/34383)的行为，并把她的思想放进一个机器人里。基本上，[马库斯]正在建造他新生女儿的机器人版本。

随机学习的[基础](http://www.ece.stevens-tech.edu/~mouli/node5.html) ( [PDF 了解更多信息](http://scholar.lib.vt.edu/theses/available/etd-5414132139711101/unrestricted/ch3.pdf))是控制系统是在现有系统的基础上建模的——在这种情况下，是一个婴儿——通过告诉机器人它做得是好还是坏。可以把它想象成只能对 1 或 0 做出反应的自动机的经典条件反射。

[Markus]基于 Arduino Mega 和一些超声波距离传感器构建了一个机器人平台。通过观察周围的环境，机器人可以判断下一步该做什么。在休息后的视频中，[Markus]展示了他的机器人在障碍跑道上寻找道路——实际上只是一对沙发垫。

离四肢着地爬来爬去，注意闪亮的东西，把一切都搞得一团糟还有很长的路要走，但我们喜欢[马库斯]创造一个初级人工智能的分析方法。
