# 反技术:作用在翼型上的力

> 原文:[https://hack aday . com/2014/08/19/retro tech atacular-forces-action-on-an-翼型/](https://hackaday.com/2014/08/19/retrotechtacular-forces-acting-on-an-airfoil/)

我们可能都做过一个非常清晰的升力基本原理的演示实验:如果你坐在车里，从不同的角度把你扁平的手伸出窗外，你的手会像飞机的机翼一样上下起伏。本周的《技术回顾》精确地解释了飞行是如何通过升力和阻力原理实现的。这是一部 1941 年的陆军训练纪录片，名为“[空气动力学:作用在翼片上的力](http://www.youtube.com/watch?v=EoN-xHiS0fo)”。

什么是翼型？从上下文来说，这是飞机机翼的形状。面对作用在其表面上的压力差，翼型产生有用的空气动力学反应，例如使飞行成为可能的升力。正如电影解释的那样，升力和阻力的概念是根据相对风的尺度来衡量的。风对机翼的作用力根据机翼和作用在机翼上的气流方向之间形成的锐角而变化。正如你可能已经知道的，升力是与相对风向成直角测量的，而阻力是与相对风向平行测量的。升力受到箔片重量的阻碍，阻力受到张力的影响。

![wind tunnel testing](../Images/525a32e98d795d38a5fca5207cb9e78f.png)

翼型有几种不同的厚度和曲率，影片展示了弦是如何从每种形状中衍生出来的。这些弦用于测量和描述相对于相对风的攻角。

作用在翼型上的力是在风洞中测量的，风洞提供直线和可预测的气流。模型飞机是由通向天平的电线支撑的。这些天平测量阻力以及前后升力。

在迎角试验中，升力和阻力朝着所谓的失速角增加。过了这一点，升力突然减小，阻力取而代之。升力和阻力与机翼面积、相对风速的平方和空气密度成正比。当飞机在空中时，阻力是一种减速力，它等于飞机的推力或推进力。

翼型模型也在风洞中进行单元测试。它们是由沿着位于表面下的箔片的许多点延伸的小管构成的。这些管子在单点离开模型，并与一组压力计管相连。这些管道将作用在翼型模型上的压力与大气压力的参考点进行比较。压力计管中不同的液位清楚地证明了翼型上的压力值。这些水平被拍照并绘制成压力曲线。现在，可以画一张图来表示相对于迎角的正压力和负压力。

最后，当飞机接近并达到极限速度时，我们看到了俯冲对升力的影响，而这种升力是通过缓慢地拉离俯冲而获得的。下次你把你的手摇飞机飞出窗外时，请记住这一点。
