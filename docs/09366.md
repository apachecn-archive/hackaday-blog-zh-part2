# 用 PCB 技术蚀刻的黄铜钟面

> 原文:[https://hack aday . com/2015/06/24/用 pcb 技术蚀刻时钟面/](https://hackaday.com/2015/06/24/etching-a-clock-face-with-pcb-techniques/)

在过去的几个月里，[克里斯]一直在用黄铜加工一个计时器，并在他的 YouTube 频道上记录了整个过程。本周，他完成了钟面。他正在复制的时钟来自 CNC 出现之前的时代，根据[Chris]的说法，在一块黄铜上雕刻罗马数字的工作将被发送给雕刻师。他不再用传统方式做事，而是用氯化铁蚀刻黄铜。这是真正的工匠工作，也为蚀刻印刷电路板提供了一个伟大的教程。

[Chris]正在使用光致抗蚀剂工艺来雕刻他的时钟表盘，就像制造 PCB 一样，这项任务从用丙酮彻底擦洗和清洁一些黄铜开始。将光致抗蚀剂放在黄铜上，印上一张透明片，整个东西暴露在四个黑光灯下。之后，未曝光的光刻胶用碳酸钠溶液溶解，蚀刻时间到了。

时钟表面被氯化铁蚀刻的时间比任何印刷电路板都长；[克里斯]正在用虫胶蜡填充这些蚀刻画，以在镀银黄铜之间形成良好的对比，并需要深的、轮廓分明的空隙。
