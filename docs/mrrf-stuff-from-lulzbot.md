# MRRF:来自 Lulzbot 的东西

> 原文：<https://hackaday.com/2014/03/17/mrrf-stuff-from-lulzbot/>

许多 3D 打印机的大牌都在中西部 RepRap 节上展示他们的产品，其中最大的一个是 Lulzbot，他们展示了神话般的 Taz 3 打印机。今年，他们展示了一种新的灯丝，一种新的挤出机，并向我们透露了他们正在进行的一个非常酷的项目。

Lulzbot 携带的新产品是 [Ninjaflex 细丝](https://www.lulzbot.com/products/ninjaflex-tpe-thermoplastic-elastomer-3mm-075kg)和与之配套的[挤压机](https://www.lulzbot.com/products/flexystruder-tool-head)。Ninjaflex 是我们见过的最有弹性的细丝，摸起来有点硬的硅橡胶。从卷轴上直接下来，细丝将拉伸到略小于其原始长度的两倍，在固态打印形式下，它是一种坚硬但柔软的材料，非常适合用于遥控坦克踏板、玩具和 [3D 打印树脂模具](http://hackaday.com/2014/03/16/mrrf-3d-printed-resin-molds/)。由于样品零件在周末受到的所有虐待，我们将称之为 Ninjaflex 有效的坚不可摧，只要你不试图把层拉开。

同样来自 Lulzbot 的消息是关于他们正在研发的新型 3D 扫描仪。硬件尚未最终确定，但未来的设备将使用网络摄像头，激光和转盘来扫描物体，并将其直接转换为。STL 文件。是的，这意味着不会有任何点云或与 Meshlab 乱搞。lulz person【Aeva】[正在开发软件](http://forum.lulzbot.com/viewtopic.php?f=41&t=589&start=10)，该软件从背景中减去一个物体，并将其转化为体素。该扫描仪将是低成本和开源的，这意味着无论扫描仪的体积如何，最终都会有人用相同的软件建造一个人大小的 3D 扫描仪。
