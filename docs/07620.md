# 开源激光射击模拟器

> 原文:[https://hack aday . com/2014/11/20/开源激光射击模拟器/](https://hackaday.com/2014/11/20/open-source-laser-shooting-simulator/)

想在家练习枪法吗？看看无家可归者(家用激光射击模拟器)，这是一个用于练习枪法的开源工具。[Laabicz]开发了这个系统，作为商业激光射击模拟器的廉价替代品，商业激光射击模拟器同样简单，但非常昂贵。

[Laabicz]的模拟器主要使用改装的气枪，装有电池(安装在弹匣中)和一个激光。只要你能想出如何连接激光器和触发开关，任何枪都可以与该系统一起使用。为了给激光器供电，当触发开关关闭时，一个小电容器由电池充电。一旦按下扳机，[电容器通过激光器](http://homeless-eng.webnode.com/labr-gun/control-electronics/)放电，并产生一个短脉冲光。

模拟器是在处理中编写的，需要一个投影仪和一个网络摄像头。加工草图将可配置的移动目标投射到屏幕或墙壁上，当激光在任何目标上触发时，网络摄像头会进行检测。该软件支持多种目标类型(包括移动目标),并且非常易于配置。休息之后，请观看视频，了解使用中的系统。
