# DIY 泡沫切割机让它太容易了

> 原文：<https://hackaday.com/2014/01/26/diy-foam-cutter-makes-it-too-easy/>

没有热钢丝剪，切割泡沫是相当棘手的。没有吗？嗯，你很幸运，[达西·威特]有一个关于如何做一个的指南。它只需要一个多小时就能建成，而且几乎不需要任何补给！

[达西]正在使用他放在周围的一个旧的 9V 电源，但你可能可以使用任何 DC 电源。他在 SketchUp 中设计了框架，并用他的 CNC 路由器将其切割出来，尽管锯子对 MDF 也同样适用。在两个 1/4-20 螺栓之间拉紧一根 40 号镍铬线，其中一个用弹簧固定。弹簧起到了保护作用，防止在过度用力切割时折断金属丝。这可能是一个简单的构建，但它确实是一个方便的工具。

[Darcy]在广告之后的视频中展示了他的作品切割泡沫。我们认为他也可以用它切割薄塑料——修改你的 3D 打印，有人知道吗？尽管他可能需要把电压调高一点。
