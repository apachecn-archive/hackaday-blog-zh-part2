# 七十进制可编程电阻器

> 原文:[https://hack aday . com/2013/09/18/70 年可编程电阻/](https://hackaday.com/2013/09/18/seven-decade-programmable-resistor/)

![124](../Images/57a5165a4d53b983ae2185a2b72116dc.png)

[Gerry Sweeney]厌倦了在制作项目原型时摆弄电阻，所以他决定要一个电阻十进制盒子。问题是它们通常都很大，而且相当昂贵(100 美元以上)。由于对可供购买的选择不满意，他决定自己设计。

在附带的视频中，他展示了他的第一次尝试，这是受到我们之前报道的一篇帖子的启发，该帖子称[使用了电阻和拇指轮十进制开关](http://hackaday.com/2012/08/21/building-a-resistor-substitution-decade-box/)的组合。他稍微修改了一下设计，改用表面贴装电阻，这使得 70 进制电阻盒相当紧凑和方便。但是他仍然不满意。

他决定自己设计印刷电路板。简单的设计利用表面贴装电阻来节省空间，并利用跳线来选择电阻。没有笨重的开关碍事，而且生产成本相当低。

休息之后，请查看该项目的精彩视频解释。他还展示了一个商用电阻十进制盒的内部！
