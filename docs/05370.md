# 具有单一步进机的双色挤出机

> 原文:[https://hack aday . com/2014/02/21/双色单步进挤出机/](https://hackaday.com/2014/02/21/dual-color-extruder-with-a-single-stepper/)

![extruder](../Images/a89d9189ba877ffca212a5dd32bc358f.png)

一旦你有了一台 3D 打印机，能够用一种颜色制造一些物体，下一个合乎逻辑的升级就是一台双挤出机。双挤出机允许多种颜色的打印，并通过添加可溶解的细丝，打印物体的能力，否则是不可能的。在现有的 3D 打印机上安装双挤出机存在一个问题:只需使用第二个步进电机，就可以显著减少打印机的打印面积。这就是 Dglass 3D [的挤压机](https://www.kickstarter.com/projects/dglass3d/next-generation-3d-printer-extruders-the-rugged-hp)要解决的问题。这是一种双螺杆挤出机，仅使用一个步进电机，比其他单丝挤出机占用的空间更少。

这不是 Dglass 3D 的人第一次尝试启动双丝挤出机；去年我们看到了[一个非常相似的机械装置](http://hackaday.com/2013/08/10/dual-extruders-in-the-space-of-one-stepper-motor)，它使用一个步进电机来驱动两根灯丝。然而，这种旧型号缺乏回缩，这意味着两种彩色打印将介于混乱、不准确和不可能之间。

新的挤出机使用伺服系统来“锁定”细丝并将其驱动到热端。这意味着收回长丝是可能的，从用这种挤出机打印的样品来看，结果看起来相当不错。

下面你会看到一些双色/收缩挤出机同时用黑白细丝打印物体的视频演示。这非常酷，加上可溶解的细丝意味着非常复杂的物体可以非常容易地打印出来。
