# 智能滑板盒为您的特技添加音效

> 原文:[https://hackaday . com/2014/07/04/smart-skateboard-box-adds-sound-effects-to-your-tricks/](https://hackaday.com/2014/07/04/smart-skateboard-box-adds-sound-effects-to-your-tricks/)

这里有一个相当有趣的旨在制作音乐的项目——使用滑板。它叫做 [SkateHack](http://www.skatehack.org/) ，是一个混合了定制硬件、电子和滑冰的开源项目。

他们一直在从事两个不同的项目，这两个项目都利用压电传感器和接触式麦克风。第一个是 2012 年 7 月在瑞典建造的，名为 Augmented Ramp，它将滑板半管变成了一种乐器。压电传感器和接触式麦克风转换来自 ramp intro 数字触发器的振动，然后由软件处理以创建音乐。结果是一个独特的混合泳，随着每一个技巧的变化。

第二个项目被称为铝土矿，它的制作方法大同小异，但被设计成任何人都可以轻松建造。这是一个滑板特技盒，它还可以将摩擦和特技转换成很酷的音效和音乐。他们称之为滑板驱动的音乐采样器——事实上，的确如此。
