# 真实世界 AdBlock

> 原文:[https://hackaday.com/2015/01/23/real-world-adblock/](https://hackaday.com/2015/01/23/real-world-adblock/)

每天，你的眼球都会被麦片盒上的广告、广告牌、t 恤衫、杂志、牛奶盒上的广告、贴在公共汽车、建筑物、香蕉上的广告以及写在天上的广告所攻击。[Reed]、[Jonathan]、[Tom]和[Alex]想出了一个解决方案:[一个品牌黑仔](http://jonathandub.in/cognizance/)，它会审查你每天每分钟看到的所有广告和品牌。这是一个你现在就可以构建的真实世界的 adblock。

该团队的系统使用了一个定制的头戴式显示器，由纸板、护目镜、网络摄像头和一个 7 英寸的显示器制成。该系统的软件使用 Python 和 OpenCV 来监控来自网络摄像头的图像，将它们与一系列品牌和标志进行比较，并以不引人注目的模糊方式将其过滤掉。

现在这个系统只有几个品牌和标志，包括胡椒博士，好时，麦当劳，脸书，星巴克，并且有明显的证据表明这是在宾夕法尼亚大学，Wawa 和 Tastykake 建立的。在下面的视频中，对这些不同品牌的检测和跟踪非常好。该系统也是立体的，这意味着它可以全天、每天佩戴，而不会损失深度感知。
