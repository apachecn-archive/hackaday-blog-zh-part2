# 我们在招人

> 原文:[https://hackaday.com/2014/03/06/were-hiring-2/](https://hackaday.com/2014/03/06/were-hiring-2/)

标题说明了一切。我们需要更多的作家来保持新鲜的黑客来了，现在是你申请这份工作的机会。

贡献者被雇佣为私人承包商，并为每个职位支付报酬。作家应该有专业的技术知识来理解他们所写的项目，并对我们报道的广泛话题充满热情。如果您有兴趣，[请发送电子邮件至我们的招聘热线](mailto:jobs@hackaday.com)并附上:

*   关于你的背景(教育、工作等)的细节。)让你成为团队中有价值的一员
*   链接到你的博客/项目帖子等。已经在互联网上公布了
*   一个用 Hack a Day 之声写的例子。包括一个横幅图像，150 个字，该项目的链接，以及任何相关的黑客一天功能的链接。

#### 的鼓励话语

首先，我们不会公开讨论薪酬。想知道我们付多少钱吗？提交一份成功的申请，我们将讨论它。

其次，不要错过这个机会。我看着这些帖子中的一个过去，又等了一年，才看到下一个并申请。现在我在经营这个地方。我们的团队由热心的读者组成。如果你对这里的东西充满热情，并且每周有几个小时来写作，你现在就需要申请！

#### 为什么我们要雇佣更多的作家？

您可能已经注意到，我们开始派人参加活动，并继续努力开发我们自己独特的原创内容。这两者都需要时间，我们需要更多的团队成员来完成发布时间表，这样 Hackaday 社区才能得到它应该得到的帖子。

你还在等什么？女士们先生们，开始申请吧！
