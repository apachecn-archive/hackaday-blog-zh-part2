# 让马里奥赛车成为现实

> 原文:[https://hackaday.com/2015/07/11/making-mario-kart-real/](https://hackaday.com/2015/07/11/making-mario-kart-real/)

如果你曾经有过休闲卡丁车的经历，你可能会和[HowToLou]产生共鸣。他注意到，每当他试图比赛时，同样的情况总是不可避免地发生。一名选手最终会排在队伍的前面，其他任何人都无法超过他们。结果更像是一个卡丁车车队，而不是一场真正的比赛。这时，他意识到像马里奥赛车这样的视频游戏早就知道如何解决这个问题了。[Lou]从这些游戏中汲取灵感，并将其应用到现实生活的卡丁车中，以改善体验。其结果就是他所谓的[快闪卡丁车](https://www.youtube.com/watch?v=oZEIXFYea1A)。

改善体验的关键是添加更多的功能，这些功能通常不会在真正的卡丁车体验中出现。闪光卡丁车使用由计算机控制的电子驱动系统。这个设置允许计算机限制卡丁车的速度，这样它们都是一样的。该系统包括一个带有内置控制按钮的罗技游戏方向盘。还有一个彩色液晶显示屏安装作为抬头显示器。屏幕显示赛车的速度，以英里每小时为单位，以及多个 MP3 音乐曲目可供选择。系统为用户提供有限数量的加速令牌，列在抬头显示器上。用户还可以查看他们当前的排名，他们在赛道上的位置，甚至可以直接看到他们身后的情况。

卡丁车的后面包括一个 23 英寸的液晶显示屏，向其他玩家显示你是谁，你是哪个队的。为了增加乐趣，骑手可以使用这个屏幕向其他参赛者显示嘲讽信息。卡丁车的前部包括一个用于射击其他卡丁车的激光炮以及一个“令牌勺”传感器。这使得玩家可以获得虚拟物品，如激光炮弹药、盾牌或额外的加速令牌。

为了实现所有这些附加功能，[Lou]从一个典型的卡丁车底盘开始。在那里，他为后端构建了一个定制的玻璃纤维外壳。这里装有大多数敏感的电子设备。该系统由三节 12V 深循环电池供电。一台 15 马力的电动机驱动后轮。油门是由一个油门踏板控制的，油门踏板只需向一个与控制计算机相连的传感器输入信号。该系统的核心是一台运行在 2.6Ghz 小尺寸 Zotac 主板和 Windows XP 上的计算机。该软件是用 C#自定义编写的。计算机插入 miniLAB 1008 接口板。这就是它与各种传感器通信的方式。接口板也用于控制许多继电器，这些继电器反过来控制赛车的速度。

不幸的是[Lou]几年前就造了这辆卡丁车，而且没有包括他使用什么传感器，或者软件如何工作的许多细节。尽管如此，这是一个很酷的想法，我们不得不分享它。请务必观看下面的[Lou]视频，了解卡丁车的运行情况。
