# 使用 Diy 射线荧光镜制作图像和视频

> 原文：<https://hackaday.com/2012/08/27/making-images-and-videos-using-a-diy-fluoroscopic-x-ray/>

[约瑟夫]一直在摆弄 x 光片。具体来说，他一直在使用自己的装置制作荧光图像，这是一种 x 射线摄影，可以制作视频图像。如果你曾经看过那些人吞咽的 x 光片，那就是荧光透视法(我们是其他怪异事物的粉丝，比如[这个骨架吹小号的视频](http://www.youtube.com/watch?v=pjSBQl38ksk))。

上图是【Jozef】自己的手。他曝光了大约一秒钟，从 Curix Ortho 普通屏幕的反面拍摄了这一事件。当被他在易贝上获得的 x 射线管粒子击中时，屏幕会发出荧光。这一特殊事件使他的手中毒约 10 拉德。我们不知道什么水平是安全的(快速搜索并没有启发我们)，所以请在评论区讨论。

当然【Jozef】并没有止步于静止图像，他在显像管和屏幕之间放了一个转盘，拍了[一堆旋转电子](http://boginjr.com/electronics/xrays/3d/)的 x 光视频。休息之后你会看到视频。
