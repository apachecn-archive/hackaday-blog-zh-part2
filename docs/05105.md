# DIY 管道冷冻套件

> 原文：<https://hackaday.com/2014/01/19/diy-pipe-freezing-kit/>

你有没有在没有关闭所有水源的情况下，需要冻结水管来快速做管道工作？对于承包商来说，这实际上是一种相当常见的做法，只不过他们使用了一种相当昂贵的工具。事实证明，实际上有一种[相当便宜且简单的 DIY 解决方案](http://go-repairs.blogspot.co.uk/2014/01/diy-freezing-kit.html)你可以用最少的材料或经验来做。

[Go Repairs]向我们展示了你所需要的只是一些管道保温材料(或一块大海绵)、一个塑料袋、两条拉链和一个空气除尘罐。用绝缘材料和塑料袋包裹管道，并用拉链固定住。将空气除尘器倒置，将大部分内容物释放到绝缘材料中。恭喜你，你刚刚冻住了你的烟斗。

它的工作原理是利用压缩气体的特性——在这种情况下是四氟乙烷——空气除尘器中的“空气”。

当这种气体处于压力下时，它在罐中以液体和气体两种形式存在。当你正常喷洒易拉罐时，你只是将气体排出——但如果你将易拉罐倒置，液体反而会流出。因为液态四氟乙烷在大气压下不能以液体形式存在，所以它一离开罐子就立刻沸腾了。由于吸热反应的发生，这会导致周围环境的温度急剧下降。
