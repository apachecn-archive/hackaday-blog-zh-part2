# 3Doodler，一种 3D 绘图笔

> 原文：<https://hackaday.com/2013/02/20/3doodler-a-3d-drawing-pen/>

这是我们的“为什么我们没有想到这个”列表中的第一件事。它被称为 [3Doodler](http://www.kickstarter.com/projects/1351910088/3doodler-the-worlds-first-3d-printing-pen) ，这是一种基于 3D 打印机中发现的塑料挤出技术的设备，被塞进一只适合你手掌的钢笔中。

如果你熟悉 3D 打印机，3Doodler 的设计应该会成为你的第二天性。这种电子塑料熔化笔内部有一个小马达，它迫使 3 毫米 ABS 或 PLA 细丝通过一个加热喷嘴。使用 3Doodler，您只需将 3Doodler 的尖端举到空中，就可以绘制三维图像。

虽然 3Doodler 显然旨在手工制作塑料物体，但我们想知道这种设备是否可以成功地适用于 3D 打印机。3Doodler 团队在 3Doodler 内部放置了一个非常小、非常便宜的挤出机和热端，他们手上有一些东西，我们很想拆开看看它是如何工作的。
