# 构建一个声音反应型 EL 面板，并在此过程中学习一些东西

> 原文：<https://hackaday.com/2012/05/15/building-a-sound-reactive-el-panel-and-learning-something-in-the-process/>

![](img/2221084eb914f1f11cf8e59a159419ec.png "EL")

我们已经看到很多使用电致发光线的建筑，通常是在服装和道具领域。不幸的是，大多数电气工程师不处理闪烁和变暗的 EL 电线和面板，任何试图控制电致发光的修补程序都没有很多关于如何控制 EL 材料的资源。[ch00f]想要填补这个知识空白，所以他构建了一个[声音反应式 EL 面板驱动器](http://ch00ftech.com/2012/05/14/dimming-an-el-panel-is-harder-than-it-looks/)，并在这个过程中学到了很多东西。

没有人真正知道电致发光电线和面板如何在分子水平上工作，但[ch00f]知道改变电场的方向会导致 el 材料发光。改变电场的频率将改变 EL 材料的亮度，因此[ch00f]所要做的就是制造一个可变频率的 EL 驱动器——这比听起来要困难得多。

我们不会用细节来烦你，因为我们不能做任何公正的记录。我们将跳到结尾，告诉你[ch00f]经过一个月的工作，包括制作自己的变压器和做一大堆数学运算，已经能够制作出声音反应式 EL 面板。你可以在休息后看看[ch00f]的[托尼·斯塔克]风格的 EL 面板的视频。
