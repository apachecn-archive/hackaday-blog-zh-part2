# 一个非常非常小的电子管放大器

> 原文:[https://hacka day . com/2013/01/11/a-真-真-微-管-amp/](https://hackaday.com/2013/01/11/a-really-really-tiny-tube-amp/)

在用套件制作了他的第一个电子管放大器后，他开始着手下一个放大器的制作。由于电子管放大器比固态放大器更具实验性，[Jarek]决定用军用剩余超小型电子管制作他的下一款放大器，在这个过程中[创造了我们见过的最小的电子管放大器](http://soniktech.com/tubes.php)。

[Jarek]无意中发现了超小型双三极管 6021 电子管，而不是通常在电子管放大器中发现的笨重的 12AX7s 和 EL34s 电子管，这种电子管最初是为弹道导弹、军用航空电子设备以及一些至今仍被列为机密的设备设计的。这些管子不仅缩小了电路的尺寸；与较大的放大器相比，这个微小的放大器消耗功率。

灯管工作所需的 100+伏电压由一个开关模式电源提供，同样保持了最终项目的规模。结果是令人敬畏的，正如在休息之后的视频中听到的。仍然有一点嗡嗡声来自放大器，但这真的是一个神话般的作品，通过使用非常微小的管更令人敬畏。
