# 外在动机:P = NP 如果你有时间机器

> 原文：<https://hackaday.com/2014/09/02/extrinsic-motivation-p-np-if-you-have-a-time-machine/>

并非所有 Hackaday 奖的参赛作品都是严肃的——至少我们希望不是——而这一张是所有参赛作品中最有趣的。[Eduardo]想在 CPU 管道中放一个通量电容器。再读一遍最后一句话，喝杯咖啡，仔细考虑一下，然后回来。这个帖子还会在。

假设《BTTF》中描绘的事件可能在另一个历史或宇宙中是真实的，考虑一下德罗宁时间机器的属性:它需要 1.21 吉瓦(我们假设从现在起这是十亿瓦)，有大约 3000 磅的总重，包括核反应堆和/或气垫车转换，能够在时间中旅行 30 年。如果穿越时间所需的能量与质量成比例，将一个 CPU 寄存器送回过去只需要一瓦特左右。是的，老布朗博士想把一辆车送回过去是错误的——发回信息要容易得多。现在，你用它做什么？

[Eduardo]正在用它来加速流水线 CPU。在 CPU 流水线中，指令是并行执行的，但是如果一条指令依赖于另一条指令的输出，就会发生不好的事情，CPU 设计者已经花费了漫长的不眠之夜来找出如何防止这种情况。基本上，MEMS 通量电容器解决了 CPU 设计中的所有突出问题。这太棒了，太疯狂了，我们很高兴看到它成为 Hackaday 奖的参赛作品。

[爱德华多]，虽然，不是只见树木不见森林。如果您的 CPU 中有一个通量电容器，*为什么还要费心优化 CPU* ？只需一个普通的 CPU，添加一个通量电容寄存器，并将一个漫长而复杂的计算的输出写入时间旅行寄存器。所有的计算立即发生，你的 Ps 和 NPs 是不可区分的。所有算法都在 O(1)中运行，整个努力对于整个家庭来说是一次轻松的嬉戏。
