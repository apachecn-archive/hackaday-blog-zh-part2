# 如何用电脑控制你的廉价遥控汽车

> 原文:[https://hackaday . com/2012/08/29/how-to-control-your ' s cheap-RC-car-with-a computer/](https://hackaday.com/2012/08/29/how-to-control-your-cheap-rc-car-with-a-computer/)

[Jon]来信告诉我们他的可编程遥控汽车，以及他制作的指南。根据他的说法，这个项目可以用价值 9 美元的零件加上一个 Arduino 和一个小型玩具车来建造。所以如果你是从零开始，大约 50 美元。

在它的核心，这个项目是关于使用 Arduino 允许你的计算机向玩具车发送信号。为此，[Jon]包含了应该能够在 Mac、Linux 和 PC 操作系统上运行的 JAVA 代码。Arduino 代码也包括在内。

大多数小型遥控汽车，如本项目中使用的那些，全速打开或关闭，但本项目允许 PC/Arduino 给汽车 PWM 信号来控制速度。正如在休息后的视频中指出的那样，这在慢速下可能有点不平稳，但仍然是一个整洁的效果。一个体面的焊接量是需要得到这个项目的工作，但它可能是一个好项目，特别是如果你有一些零件已经可用！
