# 未来科技和升级你的大脑

![hackingNPhilDec](../Images/2d0c3d8f44757d3b27ce9183d9c13c64.png)

几周前[Jacob Merz]给我发了一封关于他的感官扩展项目的电子邮件，该项目允许佩戴者通过将红外光映射到特定的音调来“听到”红外光。虽然只是一个粗略的原型，但[雅各布的]设备反映了一个更大的技术可能性领域:为人体开发一种类型的“外围设备”。*编辑:更新了图库，增加了新照片和雅各布新网站的链接。*

你会想听听[(大卫·伊格尔曼的)TEDx Alamo 演讲](http://www.youtube.com/watch?v=wpSBdA0Dc14)，特别是在大约 10 分钟的时间里，他谈到了[声波眼镜](http://zabonne.co.nz/?action=product&id=10685&category=10062)。[Eagleman]声称，如果给人脑一个与真实世界相对应的持续输入，它可以将信号翻译成有用的信息。声波眼镜，为盲人佩戴者提供一种声纳，最终只是…工作。你的大脑可以“学习”自己的输入设备驱动程序。

如果你认为声波眼镜听起来比 20 世纪 70 年代的发明更熟悉，你可能会想到[ [尼尔·哈比森](http://en.wikipedia.org/wiki/Neil_Harbisson) ]，他建造了一个类似的设备，让他能够“听到”颜色。严格地说，虽然[哈比森]声称是第一个英国电子人，但他肯定会遇到来自[唐娜·哈拉威的] *[《电子人宣言》](http://en.wikipedia.org/wiki/A_Cyborg_Manifesto)，*的抵制，该宣言认为“电子人”的概念对我们的时代来说并不新鲜；人类总是使用工具来扩展他们的能力，即使是最简单的工具也应该算作电子人。

本周的*黑客和哲学*更加开放。我邀请你推测这些技术以及它们是如何融入人体的:从试图替换缺失肢体的假肢到热切的工程学生或技术爱好者[将钕磁铁植入他们的手指。](http://www.iamdann.com/2012/03/21/my-magnet-implant-body-modification)

1.这一切将走向何方？人类会拥有基本上即插即用的设备吗？

2.还有其他植入技术的例子吗？

3.伊格尔曼正在研究其他一些“感官替代”的例子你能像这样拼凑出一些有用的东西吗？还是你已经有了？
