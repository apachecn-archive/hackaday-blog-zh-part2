# 酒鬼端上所有的鸡尾酒

> 原文:[https://hack aday . com/2012/08/28/inebriator-servers-up-all-the-cocktails/](https://hackaday.com/2012/08/28/inebriator-servers-up-all-the-cocktails/)

机器人酒保，被亲切地命名为酒鬼，是一个精通的作品。我们认为您会对其饮料配送系统的简单和优雅感到惊讶。

最明显的部分是九个酒瓶排列在顶部，带有 LED 背光，非常时尚。每一个上面都有一个阀门，可以被玻璃杯的边缘压住，以便分配它的有效载荷。为了给玻璃杯注入酒精，醉酒者沿着一条轴线在瓶子下面驱动一辆小车。当它就位时，它有一个致动臂，可以升起和压下瓶子的阀门机构。一旦所有的酒都在杯子里，它就移动到左边，用搅拌器把它装满。这些被储存在桌子下面的冷藏箱里的瓶子里。它们用氮气加压，一个电子驱动的阀门让液体流动。饮料在字符显示器上选择，手推车上有一个重量传感器，以确保没有容器接收饮料时不会混合。

休息之后，你不会想错过观看这个的。
