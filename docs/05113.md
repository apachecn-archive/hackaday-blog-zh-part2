# 皮男孩

> 原文:[https://hackaday.com/2014/01/20/the-piboy/](https://hackaday.com/2014/01/20/the-piboy/)

![piboy](../Images/bc6b2b3b219bd887998c198fe8bf693b.png)

你会怎么处理一个坏掉的游戏机、一个 3 英寸的液晶显示器、一堆电线、一个 USB SNES 控制器、一个 32gb 的 SD 卡和一个树莓派？你做一个[口袋模拟器，当然！](http://retrogamepunk.blogspot.co.uk/2014/01/making-pi-boy.html)

[Anton]不久前他决定要建立一个模拟器。他心中有几个具体的目标:它必须是手持的、便携的、对儿童安全的，并且不需要键盘就可以使用。他首先将坏掉的 Gameboy 剥去外壳，然后用热烙铁移除所有内部塑料安装部件。接下来的挑战是把所有东西都装进箱子里并给它供电。因为他的 3 英寸液晶显示器需要 12V 电源，所以[Anton]需要一种方法将 5V 电源提供给 Pi。他很幸运，原来他的 LCD 控制板有一个 5V 的测试点/扩展引脚！

从那里它只是一个重复使用原来的 Gameboy 的扬声器，关闭外壳，并加载模拟器的问题！和往常一样，休息之后会有一段演示视频。
