# retro tech tacular:Blit 让我接触到了多道程序的力量！

> 原文:[https://hackaday . com/2012/05/04/retro tech tacular-the-blit-given-me-access-of-power of-multi programming/](https://hackaday.com/2012/05/04/retrotechtacular-the-blit-has-given-me-access-to-the-power-of-multiprogramming/)

![](../Images/3ecfba0aa55192bb9ee8f245a1767bea.png "the-blit-retrotechtacular")

我们通常试图描述我们的标题。但当这句话带着明显的兴奋从叙述中蹦出来时，我们不禁窃笑。这一期 Retrotechtacular 是为 Blit 制作的宣传视频。这是一个基于图形的硬件终端，用于 Unix 系统。它最大的优点是能够同时运行(并在屏幕上显示)几个不同的程序——这种活动叫做多道程序设计。但是还有“数字化鼠标”。板上有一个 68000 微处理器 256k 内存(他们称之为四分之一兆字节)，通过 RS232 连接。屏幕是 800 乘 1024；没错，是人像定向。

这一集中值得注意的是一些经典的眼镜框，以及用于场景转换的 rad 合成器声音。使用一个愚蠢的同伴向无所不知的叙述者问愚蠢的问题的摄像技术到底发生了什么？
