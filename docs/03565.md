# 用液态金属进行 3D 打印

> 原文：<https://hackaday.com/2013/07/09/3d-printing-with-liquid-metals/>

![Gallium](img/fd96e4428ffc93303c5dd09a825c2566.png)

虽然今天的 3D 打印机基本上仅限于塑料和树脂，但桌面制造的圣杯是用金属打印。虽然我们现在还不能在台式打印机上打印出钢铁物体，但是来自北卡罗来纳州立大学的[Collin Ladd]、[Ju-Hee So]、[John Muth]和[Michael D. Dickey]正在通过用液态金属的微小球体打印物体来慢慢实现这一目标。

该团队用于金属 3D 打印的介质是 75%镓和 25%铟的合金。这种合金在室温下是液体，但当暴露在氧气环境中时，从注射器中挤出的小金属珠上会形成一层非常薄的氧化物。微小的金属球通过微小的金属球，团队可以用这种合金建造金属物体，将珠子堆叠成几乎任何可以想象的形状。

除了小金属球，[Collin]和他的团队还能够制造能够连接电子元件的独立电线。是的，与取放机相结合，配备这种技术的打印机可以制造真正的印刷电路板。

尽管该团队只在非常小的规模上研究镓，但他们相信这项技术可以扩大到打印铝。这是一项具有挑战性的努力，但它将把今天的塑料挤压 3D 打印机变成更像明天的《星际迷航》复制器的东西。
