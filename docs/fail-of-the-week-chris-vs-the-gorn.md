# 本周败笔:[克里斯] Vs .高恩

> 原文：<https://hackaday.com/2014/10/16/fail-of-the-week-chris-vs-the-gorn/>

本周，[克里斯]打破了平衡，但最终失败了。他在旅途中，穿越北方大地，在一个光线昏暗的回音室里即兴创作，而这个回音室恰好有一个老虎钳。

对枪械一无所知(你相信吗？)，[他决定用纯废料造一个 BB 炮](http://www.youtube.com/watch?v=N7yFDS6h7vE)。有几种火花会飞起来，从日立的钻头车床开始，到一个小小的纸杯蛋糕火花结束。[Chris]继续通过目测找出一些热地杆，并提供有用的提示，如果你也尝试这样做的话。在空腔的中心，他钻了一个小孔作为保险丝。

他的第一个导火索是碾碎的火柴头糊。它燃烧的有点慢，不能发射 BB。很自然地，B 计划是制作凝固汽油弹胶水，将 Pyrodex 手枪火药粘在纸上。正如你可能想象的那样，它工作得很好。填料烧焦了，但仍然没有快乐。在装满推进剂后，它仍然没有爆炸，只是烧坏了气孔。那么，怎么回事？枪管长度不够？应该用竹子而不是热地杆？还不够想要吗？在下面给我们你的饲料。
