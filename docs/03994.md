# 自制 LED 头盔

> 原文:[https://hackaday.com/2013/09/06/homemade-led-helmet/](https://hackaday.com/2013/09/06/homemade-led-helmet/)

![LEDHelmet](../Images/094c5eda80c2835a56c1e7a6a8838f93.png)

我们都看过蠢朋克头盔的制作，但[乔治的]项目是一个自制的 LED 头盔，它没有捷径，并且在面罩上装满了数百个独立的灯。他从一个原型开始，该原型使用 PIC 18F4580 微控制器连接到 MAX7221 LED 驱动器，这使他能够控制一些点阵显示器，以测试布线和样本脚本。然后他用这个原型开发了一个滚动文本功能。

测试完成后，[George]用纸板夹具将数百个发光二极管连接到 8×8 块区域，使一切保持直线。他本可以就此打住，但[乔治]做得更进一步，在头盔内部增加了一个 LCD 显示屏和一个 7 段时钟模块，以方便佩戴者查看。时钟显示头盔的当前每分钟心跳率，而 LCD 显示正在显示的内容(图案、文本/ Pacman、条纹)。在显示器的底部和头盔的下巴之间可以看到外面。如果你需要更好的能见度，我们建议[一个不那么密集的自行车头盔矩阵](http://hackaday.com/2011/05/04/led-matrix-helmits-inspiried-by-you-know-who/)。
