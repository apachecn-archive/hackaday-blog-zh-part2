# 轻松实现网真升级

> 原文:[https://hacka day . com/2013/01/28/网真-轻松升级/](https://hackaday.com/2013/01/28/telepresence-upgrade-with-a-minimum-of-effort/)

![telepresence-upgrade-with-very-simple-parts](../Images/4b83251395bbfca2a7866e9b6e02861c.png)

这种网真升级让员工可以从 4000 多公里外的地方参与到办公室中来。这是他们之前设置的升级，之前使用[旋转平台上的笔记本电脑](http://hackaday.com/2011/11/17/simple-telepresence-hack-lets-remote-user-rotate-this-laptop/)来增加视频会议体验的控制。但最初的版本只能旋转，这个可以让你在两次充电之间驾驶你的虚拟自我 15 个小时。

真正的工作是在机器人的底部，因为音频和视频是由平板电脑处理的，与运动无关。这个团队花了大约 400 美元来组装这个东西。它始于一大块胶合板。两个 3A 马达与机器人前部的割草机轮子相匹配。拖在底座后面的是几个脚轮，使它可以在不打滑的情况下转动。Arduino 的电机屏蔽和 WiFi 屏蔽使得通过互联网控制它成为可能。他们甚至在客户端添加了一些功能，以使用 PlayStation 3 控制器。休息后，查看剪辑中的完整机器。
