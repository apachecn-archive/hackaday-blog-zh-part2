# 史上最大的游戏机？

> 原文：<https://hackaday.com/2015/05/25/the-biggest-game-boy-ever/>

感觉怀旧？怀念最初的任天堂游戏机的坚实感吗？你可以用它打碎窗户，然后一直玩口袋妖怪！嗯，[拉兹]是，他建立了可能是有史以来最大的游戏机。 **[Gameboy XXL:德克萨斯版](https://hackaday.io/project/5929-game-boy-xxl)。**

实际上，它是受比利时一个名为“任天堂”( Nintendoom)的音乐节的委托制作的——图片视频游戏 music + rave。无论如何，组织者认为有一个巨型功能游戏男孩会很酷，所以[Raz]开始工作。他用 10 平方米 3 毫米厚的中密度纤维板制作了它，并在布鲁塞尔私人实验室用激光将其切割成型。里面的电子设备包括一个 19 英寸的液晶显示器、一个树莓皮和几个超大按钮。

这真是太棒了。它运行 [Retropie](http://blog.petrockblock.com/retropie/) 让你几乎可以玩任何你想玩的游戏。休息之后来看看。
