# 机器人画家像照相亭一样工作

> 原文:[https://hack aday . com/2013/11/26/robot-painter-works-like-a-photo booth/](https://hackaday.com/2013/11/26/robot-painter-works-like-a-photobooth/)

![robot-painter-photo-booth](../Images/3826c78bdc795fad61ef5fde7c260fff.png)

[Ben]、[David]、[Drew]、[Kayla]和[Peter]建造了一个机器人艺术家作为他们的高级设计项目。这融合了一堆不同的项目想法，但我们最喜欢的是它的工作方式很像一个摄影亭，产生一幅画。Raspberry Pi 使用网络摄像头拍摄图片，将图像转换为三种颜色(加上画布的白色背景)，并启动机器人。该团队感叹道，完成项目的初始测试(见下面的剪辑)进行得很好，但花了几个小时来制作这幅画。他们期望什么？这是艺术！

这与水彩机器人([的制造过程](http://hackaday.com/2013/11/25/the-making-of-the-watercolorbot/)我们昨天刚刚看过)有很大的不同。水彩机器人使用平面画布和龙门系统。这个被称为 PICASSAU 的产品使用直立的画布，画笔的安装方式与[绘图仪机器人](http://hackaday.com/2013/11/06/plotterbot-hangs-on-your-wall-to-work/)非常相似。最大的区别是，有能力枢轴画笔，以拿起更多的油漆，并为清洁之间的颜色变化。
