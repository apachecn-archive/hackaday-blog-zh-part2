# 在古驰上空移动；激光切割手袋是一种东西

> 原文:[https://hacka day . com/2015/07/13/move-over-Gucci-laser-cut-pagets-are-a-thing/](https://hackaday.com/2015/07/13/move-over-gucci-laser-cut-handbags-are-a-thing/)

当你想制作一个带有一些便捷技术功能的定制手袋，并拥有一台漂亮的激光切割机时，会发生什么？你最终会做[克里斯蒂安]做过的事情:设计并组装一个由激光切割皮革制成的女式手提包，配有 iPhone 充电器和 LED 灯。

该包的设计是在 Adobe Illustrator 中完成的，并被发送到位于维也纳[Christian]附近的 hackerspace 中的 Epilog Legend 36EXT 激光切割机。一旦零件被精确切割，传统的皮革缝制方法被用来组装手袋(在鞋匠的帮助下)。

包的内部衬着旧的蓝色牛仔裤和一个白色的 LED，LED 用导线连接并固定在适当的位置。该灯由硬币电池供电，由您选择的按钮或滑动开关控制，有助于定位深袋中的物品。

将一个标准的 USB 电池组放入旧牛仔裤的一个口袋里，你就可以在镇上度过一个夜晚了。休息之后，请加入我们，观看展示这个实际项目的设计、施工和特点的视频。
