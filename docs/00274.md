# 放下一切，马上建造这辆卡丁车！

> 原文:[https://hack aday . com/2012/05/01/drop-everything-and-build-this-go-kart-right-now/](https://hackaday.com/2012/05/01/drop-everything-and-build-this-go-kart-right-now/)

![](../Images/78568b6c24c0db911452dbc54cc313a1.png "hub-drive-go-kart")

有牛逼的项目，然后有让我们在键盘上流口水的东西。在看到这辆使用四个轮毂电机作为直接驱动轮的卡丁车后，我们刚刚收拾完我们的烂摊子。我们承认，这更像是艺术品，而不是黑客，因为这些人是机械工程师，知道他们在做什么。但是我们怎么能放弃分享这样的东西呢？

这个设计比我们记得见过的其他卡丁车都要小。低靠背的飞行员座椅是最大的部分，下面有一个小孔，用于放置电池和控制硬件。每个轮毂电机都是手动缠绕的，阅读相关的博客帖子，似乎这是构建中一个巨大而痛苦的部分。

所以看着这些家伙在麻省理工学院的一个工程大楼的走廊里捣乱是非常有趣的。但是，在午夜时分，两辆卡丁车在螺旋式停车场上比赛的镜头绝对令人愉快。休息之后你会发现两个视频都被嵌入了。
