# 激光蚀刻的 LED 非共沸器看起来像一个圆形单色屏幕

> 原文:[https://hack aday . com/2012/10/03/laser-etched-led-zeotophe-looks-like-a-circular-monochrome-screen/](https://hackaday.com/2012/10/03/laser-etched-led-zeotrope-looks-like-a-circular-monochrome-screen/)

这个动画展示的静止图像真的不公平。但是你可以想象一下这看起来真的像一个老式的单色显示器。它实际上是由发光二极管和蚀刻丙烯酸制成的非共沸物。发光二极管闪烁的速度与旋转的丙烯酸树脂同步，产生动画图像。

你可能已经知道[非共沸者使用移动的物理模型](http://hackaday.com/2011/03/19/building-a-zoetrope-using-kinect-processing-and-a-laser-cutter/)来欺骗眼睛看动画。在这种情况下，模型被蚀刻在一块丙烯酸树脂上，这样当材料被边缘照亮时，它们的轮廓就会发光。Inkscape 设计了 12 个饼形窗格，看起来像 Linux 游戏第四次世界大战中的场景。一个步进电机旋转戒指，让完美的同步在剪辑后看到休息。
