# DIY 热成像相机

> 原文：<https://hackaday.com/2013/12/01/diy-thermal-imaging-camera/>

热成像相机是非常有用的设备，可以让你看到掠夺者。如果你想通过窗户让热量从你的房子里散发出去，或者只是想知道你的电子项目下一步会在哪里爆发，它们是无价的，尽管很贵的工具。[卡贝坦·QK]想出了一个简单而便宜的方法来制作你自己的热成像相机，只需要几美元的零件。

[Kaptein]他的相机是基于一个非接触式红外测温枪。这种设备对于检测温度很有用，但是不能产生像 1000 美元的表兄弟那样的红外图像。通过从这个温度枪中取出热电堆，添加运算放大器、A/D 转换器，并将其连接到具有平移和倾斜伺服的 Arduino Nano，[Kaptein]能够在场景中缓慢扫描热电堆并生成图像。

在下面的视频中，你可以看到[Kaptein]的扫描相机正在读取环境温度，并为他的电脑创建一个成像程序。它运行得非常好，而且这个系统还有很多可以改进的地方。摆脱伺服系统并转向镜面有望加速一切，用彩色显示器取代 8 位灰度显示器将大大提高动态范围。
