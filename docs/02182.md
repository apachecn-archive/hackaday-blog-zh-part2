# Raspberry Pi 相机板将于明年年初推出

> 原文：<https://hackaday.com/2012/11/24/raspberry-pi-camera-board-coming-early-next-year/>

![](img/c8c756ff6cac281e08a2f0dbe58b3928.png "pilogo")

我们已经看到数百个版本在 Raspberry Pi 上修补 GPIO 引脚。它们非常适合弥合物理传感器和虚拟世界之间的差距，但树莓 Pi 上还有一些未使用和被忽略的引脚。Raspberry Pi 基金会的人终于给这些未使用的引脚赋予了自己的生命，为我们最喜爱的单板计算机配备了新的摄像头模块。

这款相机的规格相当令人印象深刻——它可以录制 1080p 和每秒 30 帧的 H.264 视频。最棒的是，它只需 25 美元。

在 Raspi 基金会将这种板发送给制造商之前，还有一些障碍需要克服。他们仍然需要确保带状电缆不会发出任何干扰，但如果一切顺利，相机模块应该可以在明年年初上市。

休息之后，你可以在视频中看到摄像机的动作。如果你仔细听，你会听到 Raspi 基金会的[Rob Bishop]说他们也在为 pi 上的其他*未使用的*引脚开发显示器，这是我们迫不及待想看到的。
