# Nin10do 复古游戏机傲视群雄

> 原文:[https://hackaday . com/2015/07/09/nin 10 do-retro-game-console-stands-above-all-others/](https://hackaday.com/2015/07/09/nin10do-retro-game-console-stands-above-all-others/)

如果你的客厅娱乐区没有树莓派复古游戏机，你再也没有任何借口了。拿出你的烙铁和伏特/欧姆表，预热 3d 打印机，因为你将无法抗拒制作我们见过的最好的复古游戏机之一—[nin 10 do](https://hackaday.io/project/4387-raspberry-pi-nin10do)。

它的创造者是[TheDanielSpies]。他不仅从零开始做这个东西，他还做了一项非凡的工作，记录了所有的构建细节，使得跟随他的脚步做一个你自己的变得前所未有的容易。他在 Autodesk 设计了这个盒子，并用 [XT 共聚酯](http://www.3ders.org/articles/20140904-colorfabb-launches-xt-3d-printing-filament-made-with-eastman-amphora-3d-polymer.html)细丝打印出来。当然，他使用了 Raspi，以及来自低功率实验室的 [ATX Raspi 板](http://lowpowerlab.com/atxraspi/)，以使功率循环更容易。甚至还有一个小步进器，可以打开和关闭隐藏控制器四个 USB 端口的盖子。一切都与 Python 捆绑在一起，使项目非常容易修改和定制成你喜欢的样子。

所有代码，图表和。stl 文件可以在他的 [github](https://github.com/TheDanielSpies/Nin10do-Project) 上获得。它甚至有自己的[脸书页面](https://www.facebook.com/raspberrynin10do)！一定要看看大量的视频来帮助你建立。
