# 电脑出现之前绿屏是如何工作的

> 原文：<https://hackaday.com/2015/02/07/how-green-screen-worked-before-computers/>

如果你知道电影是如何制作的，那么你可能听说过“绿幕”。这项技术也被称为色度键合成，通常用于根据色调将两幅图像或视频合并在一起。通常你会看到一个演员在绿色背景前拍摄。使用视频编辑软件，编辑可以用另一个视频剪辑替换那个特定的绿色。这让演员看起来像是处在一个完全不同的环境中。

这并不奇怪，有了电脑，这是一个非常简单的任务。任何基本的视频编辑软件都会包含一个色度键功能，但是你有没有想过在计算机把它变得如此简单之前，这是如何实现的？[汤姆·斯科特] [发布了一个视频](https://www.youtube.com/watch?v=msPCQgRPPjI "Green Screen Video")来解释这一点。

在电影的早期，工作室可以在全黑的背景下拍摄演员。然后，他们会使用越来越高的对比度一遍又一遍地复制电影，直到他们最终得到一个黑色的背景和一个演员的白色轮廓。这种薄膜可以用作遮片。用一台[光学打印机](http://en.wikipedia.org/wiki/Optical_printer "Optical Printer Wikipedia")工作，工作室可以进行双重曝光，将背景胶片和演员的胶片结合起来。你可以想象这是一个比点击几下鼠标更麻烦的过程。

对于绿屏效果，工作室实际上可以使用专门的滤光器。他们可以使用一种可以忽略绿色特定波长的滤光器。然后他们可以用这个滤镜拍摄演员。然后用光学打印机将合成的蒙版与演员的镜头和背景胶片结合起来。它非常类似于黑色背景的老式款式。

电子模拟视频有一些其他有趣的技巧来实现相同的基本效果。[Tom]解释说，模拟信号包含需要在屏幕上显示的各种颜色的信息。电子电路可以观察特定的颜色(绿色)并用背景视频中的信号代替。电影公司甚至使用两个机械连接在一起的摄像机来做相同的动作，同时记录演员和模特。然后信号可以通过这个特殊的电路，合成的图像被同时记录下来。

视频中还有一些其他的例子，汤姆用来描述这些老技术的效果对理解这些概念大有帮助。想想这个过程会有多复杂真是疯狂，现在我们可以用家里已经有的电脑在几分钟内完成。
