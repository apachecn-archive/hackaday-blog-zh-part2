# 黑镜，黑洞:关掉你的电视

> 原文:[https://hack aday . com/2014/04/23/black-mirror-black-hole-kill-your-television/](https://hackaday.com/2014/04/23/black-mirror-black-hole-kill-your-television/)

你相信有人认为互联网是浪费时间的吗？嗯，当然不是在这个特定的地址，但是我们可以想到其他一些绝对是兔子洞的地方，甚至在底部没有一个激动人心的槌球游戏。如果你需要帮助实现蒂姆·费里斯所谓的低信息饮食，有一些浏览器扩展会阻止你访问阻止你做事的网站。[伊凡的女朋友]认真对待这个时间管理策略，甚至创建了一个简单的网页，上面写着“不要浪费你的时间！”如果她想去脸书，那就能看出来。

所有这些都有一个小问题，它已经存在很长时间了。【伊凡的女朋友】还在看电视。出于对她目标的爱和尊重，他决定恶作剧阻止她看电视。令人愉快的是，30 秒后，电视会向她显示她自己的网页。

他们有数字电视和模拟电视，所以他必须把两者都安装好，以覆盖他的基地。数字电视是一种显示器，由带有 HDMI 输出的机顶盒供电。由于机顶盒只能通过红外遥控器控制，[Ivan]使用 HDMI 开关从 STB 输入转换到 Raspi，Raspi 将显示谴责网页并播放平克·弗洛伊德的“[时间](https://www.youtube.com/watch?v=JwYX52BP2Sk)”。

模拟电视做得稍微多一点。他在 AV 输入上放了一个 Raspi，但从内部连接，所以没有什么看起来可疑。Raspi 每秒检查一次电视状态，一旦电视打开就切换到 pi。同样的交易:判断网页，平克·弗洛伊德。美妙的是，[伊凡]的两个场景都记录了她的反应；数字电视使用 dash 摄像头，模拟电视使用 Android 手机。休息之后，请查看[Ivan]的模拟电视 Pi 之旅。

如果你或(伊万的女朋友)需要更多的时间管理帮助，总会有[滚动自己的番茄定时器](http://hackaday.com/2014/02/09/hack-all-the-things-in-the-time-you-save-with-this-led-pomodoro-timer/)。
