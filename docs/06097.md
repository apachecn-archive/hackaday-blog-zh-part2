# 电弧攻击展示了泰斯拉背包，这当然不是武器

> 原文:[https://hacka day . com/2014/05/19/arc-attack-shows-off-Tesla-backpack-that-is-that-are-not-a-a-weapons/](https://hackaday.com/2014/05/19/arc-attack-shows-off-tesla-backpack-which-is-certainly-not-a-weapon/)

[https://www.youtube.com/embed/tc5SFMxTzyI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/tc5SFMxTzyI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

[史蒂夫·沃德]向我们展示了他的特斯拉线圈项目，它看起来非常像一个视频游戏武器。这个手工制作的杰作真的很有冲击力，能够抛出脂肪白色电弧或威胁性稍低的紫色等离子体。上面的视频又吵又暗，但是看看背包里的电子设备(神圣的巨大电容器蝙蝠侠！)和快速功能演示都非常值得张贴。不幸的是，我们没有得到枪在行动的音频，所以你必须想象那些声音效果。

该钻机由 5Ah 锂电池供电，该电池提供 60V 电压，然后升压至 400V。已经提到的巨大电容储存了大约 2000 焦耳的能量(我们喜欢史蒂夫说 2000 焦耳这个数字时不喜笑颜开的样子)。这个项目是几年前我们在堪萨斯州看到的原型的最先进版本。如果你想了解更多这些家伙在做什么[，请访问他们的主页](http://arcattack.com/)。
