# 为光纤环形灯构建 LED 光源

> 原文:[https://hack aday . com/2013/11/24/building-an-led-source-for-a-fiber-optic-ring-light/](https://hackaday.com/2013/11/24/building-an-led-source-for-a-fiber-optic-ring-light/)

【彼得】已经完成了他的[光纤显微镜光源](http://www.instructables.com/id/10W-LED-fiber-light-source/)。当我们上次拜访[彼得]时，他为一个 10 瓦的 LED 创造了一个[调光电路。](http://hackaday.com/2013/11/20/simple-10-watt-led-driver-is-hot-stuff/)那个 LED 驱动器现在已经在[Peter 的]“Franken-ebay scope”中找到了它的最终归宿，这是一台立体显微镜，由他几年来获得的零件组装而成。像这样的立体显微镜对于处理表面贴装零件或检查 PCB 问题非常有价值。[Peter]有光纤环和鞭状物，但没有光源。最初的光源是 150 瓦的卤素灯。10 瓦 led 和驱动电路是一个很好的替代品，但他需要一种方法来连接 LED 和光纤鞭。让整个系统保持凉爽也是一个好主意。

这对[Peter]来说不成问题，因为他可以使用铣床。他用一个旧的 CPU 散热器从他的垃圾箱作为光源的基础。散热器被钻孔和攻丝用于 led。下一个问题是实际的光纤鞭接口。为此，[Peter]从铝棒料上磨出一个定制块。完成的组件包含 LED、驱动器和光纤鞭。金属片支架允许整个组件安装在显微镜的立柱上。我们必须承认，如果我们处在彼得的位置，我们会选择一个便宜的 LED 环形灯。然而，最终的结果是一个非常干净的设置，将大量的光线投射到任何[彼得]需要放大的地方。
