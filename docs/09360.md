# 爬行空间仓库包括小型叉车

> 原文:[https://hack aday . com/2015/06/24/crawl space-warehouse-includes-midget-叉车/](https://hackaday.com/2015/06/24/crawlspace-warehouse-includes-midget-forklift/)

认识一下[Cliff Stoll]，ACME Klein 瓶的杰出制作人和企业家。几年前，他帮助一个朋友设计了一个玻璃烤箱，以获得玻璃吹制的正确温度曲线。当被问及他希望如何获得帮助时，他问他们是否可以吹一个克莱因瓶——他们这样做了。

他对自己创作的前景感到兴奋，他把它展示给一些朋友看，并惊讶地发现人们想买它！他创造的那个花了 3 天的汗水和泪水来建造，所以他不打算自己开始制造它们。相反，他决定打电话给一些玻璃制造商，看看他能以多少钱买到克莱因瓶……这时他发现了大量购买的力量。

那么，你如何处理 1000 多个克莱因瓶呢？你在你的爬行空间里建了一个迷你仓库——就是这样。

是的，这家伙利用他的低天花板的爬行空间，把它变成了存放克莱因瓶的仓库。他甚至制造了一台迷你遥控铲车来搬运材料。
