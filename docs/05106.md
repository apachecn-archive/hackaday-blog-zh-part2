# DIY 卷轴锯

> 原文:[https://hackaday.com/2014/01/19/diy-scroll-saw/](https://hackaday.com/2014/01/19/diy-scroll-saw/)

卷轴锯是非常方便的工具，但有时很难证明为一个项目购买一个卷轴锯是合理的。那么为什么不把**变成**你的项目呢？[SDX42]向我们展示如何使用…等一下，[那是一台旧缝纫机吗？](http://www.instructables.com/id/The-Sewingmachine-Scrollsaw/)

首先是一个小背景故事。他拿起一台旧缝纫机，这台旧缝纫机被它的前主人扔掉了:他们说它运转不正常。他还是拿走了，并决定把它修好。他遇到了两个问题。首先，这比他最初想象的要难解决得多。第二，他意识到他不需要缝纫机。然而，他需要的是一把卷轴锯。

缝纫机实际上相当类似于卷轴锯。它们都是通过将旋转运动转化为线性往复运动来工作的。唯一不同的是布局。[SDX42]将机械装置颠倒过来，在拆下来的缝纫机上建造了一个卷轴锯框架。比那要复杂一点，但是休息之后我们会让他在视频里给你解释。
