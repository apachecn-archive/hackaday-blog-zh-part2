# 周四在城里喝酒

> 原文：<https://hackaday.com/2015/04/27/thursday-drink-up-in-the-city/>

哈卡戴将于本周前往纽约。拿起您的项目，[和我们一起品尝美味的饮料](http://nycmeetup.hackaday.io/)。周四晚上 7 点左右，我们会在鹿角啤酒&酒药店闲逛。参加 Hackaday 在纽约举办的首次社交活动！

这是我们周六下午开始的黑客马拉松的预赛。到目前为止，我们已经确定了今晚的八位 Hackaday 成员:[布莱恩·班乔夫]、[亚当·法比奥]、[比尔·赫德]、[索菲·克拉维茨]、[亚历山大·布拉迪克]、[马特·博格伦]、[贾斯敏·布雷克特]和[罗布·文森特]。但是，嘿，没有你也不会有任何乐趣！我们想用硬件黑客挤满这个地方，所以抓住你的朋友，使用顶部的链接进行回复。

这是我们 2015 年 Hackaday 奖全球巡展的一部分。[现在就开始吧](https://hackaday.io/had2015)对未来的构建提出一些快速的想法，我们可以在周四讨论。回头见！
