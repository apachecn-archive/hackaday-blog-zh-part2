# 建造你自己的水火箭发射器

> 原文:[https://hack aday . com/2012/10/10/build-your-own-water-rocket-launcher/](https://hackaday.com/2012/10/10/build-your-own-water-rocket-launcher/)

![](../Images/f6d6b656403067f4ffee92f9aeb0adf5.png "how-to-build-a-water-rocket")

我们觉得你想在水里玩的日子已经一去不复返了。但是如果你还能在这里或那里找到一个温暖的下午，建造这个水上火箭发射器是一件有趣的事情。我们估计花在这个项目上的大部分时间将用于购买零件。它们都很常见，一旦你手头有了它们，不到一个小时就能组装好。

这个概念很简单，但这并不妨碍人们建造相当复杂的水火箭装置。Lou 设计的这个非常简单，但是它提供了软管和空气压缩机的连接(另一种方法是提前在瓶子里装满水，然后用自行车打气筒来加压)。PVC 用于通过一对阀门将两个输入连接到瓶子。当施加水和空气时，瓶子保持在适当的位置。当拉动绳子释放瓶子时，发射就发生了。

休息后查看构建过程和瓶子发布。我们认为火箭需要几个鳍。
