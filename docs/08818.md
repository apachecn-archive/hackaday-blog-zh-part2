# 黑客日奖品入口:地下定位系统

> 原文:[https://hack aday . com/2015/05/07/hack aday-prize-entry-underground-positioning-system-subps/](https://hackaday.com/2015/05/07/hackaday-prize-entry-subterranean-positioning-system-subpos/)

有很多情况下，我们需要知道自己的位置，但由于 GPS / GSM 信号在我们的智能手机上不可用和/或不可达，我们无法做到这一点。[Blecky]正致力于解决这个问题。这是一个基于 WiFi 的定位系统，可以用在 GPS 不能用的地方。

SubPos 不需要昂贵的许可、专门的硬件、费力的区域分析或对数据连接的依赖(连接到数据库/手机覆盖)。它独立于或与 GPS/Wi-Fi 定位系统(WPS)/室内定位系统(IPS)一起工作，通过利用常见的硬件作为额外的定位数据源。

只要安装了 SubPos 节点，用户想要确定他们在地下或室内的位置，只需使用 Wi-Fi 接收器。这在地铁线、购物中心、停车场、美术馆或会议中心等地方非常有用——基本上是 GPS 无法穿透的任何地方。SubPos 通过利用 Wi-Fi 的所有功能，定义了一种在不同环境中进行地下定位的精确方法。SubPos 节点或现有的 Wi-Fi 接入点用于在标准 Wi-Fi 信标帧中传输编码信息，然后用于位置三角测量。

sub PS 节点的运行方式与 GPS 卫星非常相似，只是 sub PS 不是使用精确的时间来计算发射器和接收器之间的距离，而是使用编码的发射器信息以及客户端的接收信号强度。休息后观看演示视频。
