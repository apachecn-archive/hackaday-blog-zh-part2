# 7 英尺长的飞龙喷火

> 原文:[https://hackaday.com/2012/05/01/dragon_fire/](https://hackaday.com/2012/05/01/dragon_fire/)

![](../Images/9ba86498d33979924bd7ea9cb87a95ef.png "dragon-flying-rc-7-501x331-custom")

要想在托莱多的 R/C 活动中赢得最佳表演，您需要做些什么？当然是造一个 7 英尺长的喷火无线电遥控龙。[里克·哈默尔]把他的电子产品、涡轮发动机、煤油罐和眩晕枪塞进了一个形状像龙的自制身体里。你可以看到一些显示他如何驾驶的建筑照片。它看起来就像任何遥控飞机一样飞行。然而，这一次可能会烧毁一个村庄并持续下去。休息后看看视频，看看它飞行和测试它的喷火机制。
