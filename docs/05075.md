# AxCut:一款开源激光切割机

> 原文:[https://hack aday . com/2014/01/16/ax cut-an-open-source-laser-cutter/](https://hackaday.com/2014/01/16/axcut-an-open-source-laser-cutter/)

如果你一直想要一台激光切割机，并且手头有 1500 台(大约。从今天起 2500 美元)——并且你可以使用 3D 打印机——那么你会想看看[Damian 的] [开源激光切割机:axCut。](http://axrap.blogspot.com/)在过去的几个月里，该项目已经从 OpenSCAD 中的一些模型发展成为一个工作原型。

你会想挖掘他的博客帖子以及他的 YouTube 频道，寻找所有有趣的细节，但从我们收集的信息来看，[达米安]已经到了最后关头。目前的实施包括一个 40W 的 CO2 激光器，具有功能性激光控制和令人印象深刻的安静水冷系统。虽然构建的布线仍然有点混乱，但原型(几乎)如预期的那样切割。他的下一个障碍是消除空气辅助，这应该可以防止一些火灾问题，并保持镜头没有碎片。

休息过后，看看几个视频，如果你对激光切割感兴趣，但想从小处着手，[看看几个月前的显微切片](http://hackaday.com/2013/10/27/microslice-the-tiny-arduino-laser-cutter/)。
