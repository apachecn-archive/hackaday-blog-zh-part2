# 由微波变压器制成的专业点焊机

> 原文：<https://hackaday.com/2014/09/21/a-professional-spot-welder-made-out-of-a-microwave-transformer/>

点焊机是为数不多的金属加工设备之一，实际上自己组装比购买便宜得多。事实上，在从旧微波炉中回收变压器和购买一些其他组件之间，在大多数情况下不到 100 美元是可行的。

我们之前已经[分享过](http://hackaday.com/2012/10/31/diy-spot-welder-makes-metalwork-easy/)这个[黑客](http://hackaday.com/2011/11/06/diy-spot-welder-can-join-anything-together-even-copper/)相当于[几次](http://hackaday.com/2009/06/23/how-to-build-your-own-spot-welder/)，但是【阿尔伯特·范·达伦】真的采取了行动，创建了一个非常详细和广泛的指南，不仅用于构建他的，而且用于各种目的。

[Albert]在设计上允许它配置在相对和串联电极位置，这意味着除了能够将金属板点焊在一起，您还可以在电池上点焊电池片！

他还使用 Arduino 来实现电流应用的精确计时，并创建了一个可调节的力导向器来在焊接过程中提供压力——这是一个通常只能在商业设备上看到的功能。他的博客上有大量的信息，所以如果你有兴趣建立自己的博客，可以去看看！
