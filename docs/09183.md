# 用 OTH-B 雷达覆盖你的圆锥

> 原文:[https://hackaday . com/2015/06/02/retro tech tual-cover-your-conus-with-oth-b-radar/](https://hackaday.com/2015/06/02/retrotechtacular-cover-your-conus-with-oth-b-radar/)

如果你是一个业余爱好者，你已经知道电离层是一个很好的反射全球高频信号的背板。它对[超视距反向散射](http://www.youtube.com/watch?v=2GVPdJzfiJE) ( [OTH-B](http://www.dtic.mil/dtic/tr/fulltext/u2/a261727.pdf) (PDF))雷达应用也很有用，这是美国空军罗马实验室在冷战期间进行的实验。

在试验项目中，发射和接收站点被设置在缅因州内相距 90 英里的地方。1/2 英里长的发射天线由四组 12 个偶极子单元组成，工作功率为 1MW。天线后屏和接地屏进一步扩大了信号的范围。传输通常由传输大楼内的计算机控制，但也可以手动供电和调节。

接收站点有 50 英尺。天线元件延伸 3900 英尺，一个巨大的地面屏幕覆盖近 8 英亩。从发射点的偶极子阵列发射的信号被电离层反射，向下到达接收点。由于步进扫描，该系统能够覆盖 180°弧。美国大陆的 OTH-B 雷达系统在冷战结束时被搁置，但如果有足够的时间和资金，可以重新投入使用。
