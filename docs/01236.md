# 另一个机器人控制的漫游机器人

> 原文：<https://hackaday.com/2012/08/03/another-android-controlled-roving-robot/>

[Sam]从事手机控制机器人已经有一段时间了，随着几个类似的 Kickstarter 活动的启动，他认为分享他迄今为止的进展是一件好事。

在 [IOIO 开发板](http://ytai-mer.blogspot.com/2011/04/meet-ioio-io-for-android.html)的帮助下，【Sam】的机器人由 Android 设备控制。这种设置提供了足够多的计算能力来让机器人快乐地行走，并且允许[Sam]连接更多的传感器。

机箱的设计是为了将接头放在外部的 IOIO 板上，就在一个非常适合放置一两个试验板的小搁板上方。通过正确的硬件和软件设置，[Sam]的机器人可以在附近漫游，收集数据并实时发送到服务器。

如果你想知道为什么一个微小的 Android/IOIO 驱动听起来如此熟悉，这可能是因为我们几天前发布的机器人。与 Botiful 不同，[Sam]只能通过蓝牙控制他的脚踏机器人，因为整个“编写网络界面”有点超出他的能力。希望[Sam]今晚带着他的 Arduino 坦克去波士顿的 Dorkbots 时，能遇到一个热情的程序员。

休息过后，你可以看看[Sam]的机器人原型。
