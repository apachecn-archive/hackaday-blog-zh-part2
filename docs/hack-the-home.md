# 黑进家里

> 原文：<https://hackaday.com/2015/04/07/hack-the-home/>

住在肯塔基州？想为厨房发明下一个大东西吗，比如自动制冰机？你很幸运。通用电气在他们位于肯塔基州路易斯维尔的名为 FirstBuild 的微型工厂举办了一场大型黑客马拉松。不幸的是，预注册似乎已经结束。

首次构建的概念非常酷。通用电气创建了一个他们喜欢称之为微型工厂的东西，用于在小批量生产中创新新产品，以发掘好的创意。我们称它更像是一个有大量资金支持的企业黑客空间。不管他们的动机如何，这仍然是一个很酷的概念。他们有一些很棒的~~玩具，~~呃，工具。

这项活动得到了英特尔、Atmel、TI、Freescale、MakerBot 甚至 Autodesk 等许多知名公司的赞助，而且奖品也不算太差！虽然对于真正的大奖[你需要坚持和我们在一起](http://hackaday.io/prize)。

我们不禁要问——当通用电气看到他们喜欢的想法时会发生什么？希望这能带来一份好工作。
