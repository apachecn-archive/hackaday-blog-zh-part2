# 电脑显示器流光溢彩克隆显示卓越的性能

> 原文:[https://hack aday . com/2013/07/13/computer-monitor-ambi light-clone-shows-approval-performance/](https://hackaday.com/2013/07/13/computer-monitor-ambilight-clone-shows-remarkable-performance/)

![ambilight-project-discreet-led-boards](../Images/b34fd5e1f7a42f8eea23bcb940ad5c40.png)

看看这个神奇的[流光溢彩的电脑显示器克隆](http://www.youtube.com/watch?v=huj5bD-x-2c)，Brafilus 已经研究了几年了。这实际上是第三次修订，看下面的演示视频让我们瞠目结舌。

详细信息只能在 YouTube 页面上以评论的形式提供。但是他给了我们足够的满足。他的自蚀刻电路板上有一个 PIC 18F14K50 微控制器。它正在与 28 个 LED 像素中的每一个进行对话，这些像素本身也存在于 diy PCB 的小块上。他用 C#编写了自己的 PC 软件来捕捉屏幕边缘周围的颜色。他还努力确保在显示器和你看到的从墙上反弹回来的东西之间有足够的调整来实现真正的颜色匹配。

如果你正在电视机上寻找这样的东西，回到几天前，看看那个独立的单元。
