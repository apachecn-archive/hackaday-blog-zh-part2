# 自制液氮

> 原文：<https://hackaday.com/2014/05/23/homemade-liquid-nitrogen/>

就 DIY 低温学而言，干冰是简易模式。你可以在沃尔玛买到冰冻的二氧化碳，或者从拧在二氧化碳罐上的喷嘴里买到。这一切都很普通，一点也不特别。想知道什么是酷吗？[自制液氮](http://www.instructables.com/id/Homemade-liquid-nitrogen-generator/?ALLSTEPS)。

[imsmooth]从标准罐中获取氮气，将气体通过二氧化碳和 H2O 洗涤器，压缩气体，将压缩气体放入冰浴中，并将压缩、冷却的气体慢慢扩散到真空储罐中。当冷的压缩气体被释放到容器中时，玻意耳定律发生，液氮在烧瓶中冷凝。

就材料和设备而言，[imsmooth]正在使用一个填充有沸石的 PVC 塔来过滤二氧化碳和 H2O，一个水肺压缩机(无油)，以及用于预冷器和再生冷却塔的几乎荒谬数量的不锈钢管。除了几个昂贵的阀门、杜瓦瓶和潜水压缩机之外，所有的东西你都可以很容易地从通常的家装商店里找到。

[imsmooth]每小时生产大约 350 毫升的液氮，对于那些不在车库里运行工业流程的人来说绰绰有余。看看下面的视频。
