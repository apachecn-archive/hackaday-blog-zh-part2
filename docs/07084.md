# THP 黑客生物:太空技术

> [https://hackaday . com/2014/09/18/THP 黑客生物空间技术](https://hackaday.com/2014/09/18/thp-hacker-bio-iamteknik/)

[IamTeknik]让他的[家庭自动化助手](http://hackaday.io/project/1214-Project-Jarvis---A.I-Home-Automation-%26-Assistant)参加 Hackaday 奖的原因很简单；我们有智能手机、电视，甚至智能汽车。为什么不是智能屋？

就像《钢铁侠》中的同名电影一样,“贾维斯计划”是一个智能助手，混合了一点家庭自动化。硬件包括通常的继电器和门锁，但这只是开始。还有一个个人数字助理，位于硬件模块和[IamTeknik]智能手机之间。在这里，语音识别、语音合成和类似 Siri 的功能是游戏的名字。Jarvis 能够回答问题、编写报告、阅读社交网络消息，并通过互联网自动化连接到主基站的一切。

[IamTeknic]最近一直在忙着学习计算机系统工程，当然也在忙着他的 Hackaday 奖项目，但他还是能坐下来回答我们 THP 黑客传记的几个问题。你可以看看下面，以及他的个人贾维斯可以做的一些演示。

![](img/b36bf05c59a5770a2c567b59107c46e0.png)

我是一名专业学生，在当地一所大学学习计算机系统工程。它基本上是编程和电子学的结合，我计划在几年内获得硕士学位。

**![](img/c4d9d090a3f9b6720f8c94bffa417268.png)**

我对节能和我的工程职业充满热情。这就是为什么我的许多主要项目都致力于节约能源和保护我们生活的世界。

![](img/2950b3a70998217c82ed88bec26f9e07.png)

在视频中，他们摧毁了一台打印机。对我个人来说，我有时会对我的 3D 打印机有这种感觉。对我来说，这是最有问题的设备，有时你可能会浪费几天的时间来尝试对它进行必要的构建，然而，当它工作时，它非常令人满意。

![](img/456fe19f156fdd97e4a0e05e5ae85a33.png)

Hackaday 上的很多人会因此恨我，但我最喜欢的操作系统必须是 Windows。它可能不是“黑客”操作系统，但作为一名工程师，许多软件都是在 Windows 上运行的，尤其是行业专用软件。它基本上有我需要的所有东西，从 PLC 软件到我的 3D 打印机使用的 3D 建模程序

![](img/86cb90b2c0b6af0a653ce112a7f83607.png)

我最喜欢的台式设备应该是我的桌面电源。我父亲最近把它传给了我，这是他多年前买的第一个电源。依然好用！

![](img/308a7648f6e9856896236bbb41dc80a5.png)

我倾向于支持面向对象的编程语言，因为我就是从这种语言开始的。我最喜欢的是 Java，我对它有很好的了解，可以用它做很多很酷的东西，但 C++紧随其后。

![](img/b695bbab69a146f980245f1ec391f5f7.png)

在比赛开始之前，我已经做了一段时间我的项目。我记得在 EEVBlog 上看过[Dave]宣布这个项目的视频，所以我马上赶了过去，竞赛描述非常适合我的项目，建造任何东西，并使它连接起来。

![](img/d015faab202bc036580668d46d45464b.png)

*   我很想完成贾维斯项目，我的 THP 参赛作品。主要是因为节能，我绝对认为这是未来家庭的巅峰之作。
*   我真的想遥控一辆配有手动变速箱的全尺寸汽车，以应对额外的挑战。
*   然而，更重要的是，在我死之前，我想创造一个系统，它将极大地有助于以高效和环保的方式产生能量。

![](img/696713e81e03f54f23cda21f65a63da1.png)

大约有 800 个参赛作品，所以我不能全部看完，但是我看到的都很棒。如果让我选择，我希望看到更多与 DNA 相关的项目。

![](img/1f48babecd6b5224565c529d357d07be.png)

到目前为止，这个项目进展得很顺利。我刚刚制作了一个运行 Linux 的 7 英寸触摸屏平板电脑，它将被安装在墙上。它将是一个控制器，可以控制媒体、与人工智能互动、订购食品等等。我还在努力改进节能，这样，随着一些更多的功能应该很快为法官准备好。有几个问题，一个是失去了对语音 API 的支持，但是我已经通过使用我的移动设备来处理这个问题。这应该足够了，直到我把语音识别重新启动并运行。最大的延迟是我的 3D 打印机，大量的重印，配置问题，模型问题和许多其他问题，占用了开发一段时间。
