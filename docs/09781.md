# Hackaday 奖参赛作品:垂直气雾栽培

> 原文:[https://hack aday . com/2015/08/09/hack aday-prize-entry-vertical-aero ponics/](https://hackaday.com/2015/08/09/hackaday-prize-entry-vertical-aeroponics/)

为了他的 Hackaday 奖参赛作品，[MIPS ARMSTRONG]正在研究一种开源的玻璃容器，这将是种植食物或其他可食用绿色植物的最快方法之一。他称之为伊甸园项目，它将成为有史以来最先进的自制园艺设备之一。

有几件事使这个室内温室与众不同。最明显的是数量惊人的发光二极管被用作生长灯。[MIPS]使用的是 900 瓦的皇家蓝和深红色发光二极管。为了给这些植物浇水，[MIPS]从美国宇航局得到启示，建造了一个高压雾培系统——一个将直径只有 50 微米的水滴直接喷射到植物根部的设备。

伊甸园更有趣的一面是二氧化碳系统。像人类一样，大部分植物生物量来自碳，植物从大气中获取碳。研究表明，增加培养室中二氧化碳的浓度可以促进植物生长。二氧化碳对植物有毒是有限度的，所以[MIPS]必须用气体传感器密切关注二氧化碳的水平。

通过高压浇水、二氧化碳系统和令人惊叹的 led 阵列，这是地球上最先进的家酿园艺项目之一。这也非常符合今年 Hackaday 奖的主题“建造有意义的东西”，我们迫不及待地想看到[MIPS]对他令人敬畏的气雾栽培植物的未来开发。
