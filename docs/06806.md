# 因此由(动态)砂制成的铸件。。。事实上，结果相当不错

> 原文:[https://hack aday . com/2014/08/12/and-so-castings-made-of-kinetic-sand-turn-out-pretty-well-actually/](https://hackaday.com/2014/08/12/and-so-castings-made-of-kinetic-sand-turn-out-pretty-well-actually/)

那种动态沙的东西很酷。它很柔软，可以培养运动技能，除了洒在地毯上，它一点也不脏。如果你不知道，那是 98%的沙子和 2%的聚二甲基硅氧烷，这是橡皮泥的主要成分，而且据某个黄红小丑说，放进鸡块就足够安全了。[克里斯]的妻子给他买了一些，可能是因为她想看他玩一些不会致命的东西。在研究它的神奇特性的过程中，他发现它本身并没有真正的热崩溃点。在足够高的温度下，它会玻璃化，就像蘑菇云里的沙堡。在这种特性和它的延展性之间，[克里斯]认为他有一个合理的铸造砂替代品。正如你在他最近的实验中看到的，他是对的。另外，他成功地将良性转变为危险。

[Chris]以前从未铸造过铝，所以他决定从小处着手，为旋转拉刀制作偏置凸轮。他在一个蜡纸杯里装了一些神奇的沙子，然后把照相机推进去制作底片。然后他砍下一些铝棒，放入石墨坩埚中。他把他的 DC 弧焊机的电极伸进坩埚，转动到 50 安培。这还不够，所以他去了 110。坩埚很快就发出橙色的光。他小心翼翼地把熔化的铝倒进模子里。跳过去看看结果如何。

剧透:这次没有诅咒！
