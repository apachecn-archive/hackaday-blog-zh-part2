# 高成本的 Arduino 计算器是笨拙的，但仍然很酷

> 原文：<https://hackaday.com/2015/01/19/high-cost-arduino-calculator-is-unwieldy-still-cool-though/>

[Kale_3D]的一位老师制作了一台 Arduino 驱动的计算器。它不够健壮，在教室环境中也没有持续太长时间。在这个不起作用的计算器在班上坐了一会儿后，[Kale_3D]决定试着修理一下。在这个过程中，这个项目不仅仅得到了修复，还得到了全面的重建。

这个计算器使用一个完整的 16 按钮矩阵键盘。Arduino deciphers 按钮在键盘库的帮助下按下，此时适当的字符显示在 2×8 LCD 屏幕上。选择功能与正常略有不同，因为这个项目仅限于 16 个按钮。其中两个按钮不仅允许滚动标准算术函数，还允许滚动三角函数。这是以前版本所不具备的特性之一。

为了保护组件，外壳由 1/4”激光切割木材制成。这两件衣服的边缘有缺口，非常合身。尽管如此，还是增加了角块，使箱子更加坚固。

是的，这个计算器不实用，但这不是重点。最后[Kale_3D]觉得这个项目绝对值得做。他学到了很多关于 Arduino 的东西，尤其是代码调试！最重要的是，他在建造它的时候玩得很开心。休息之后有一段视频展示它是如何工作的。代码和接线图可从项目的可指导页面下载。
