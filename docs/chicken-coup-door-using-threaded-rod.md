# 使用螺杆的鸡舍门

> 原文：<https://hackaday.com/2012/05/01/chicken-coup-door-using-threaded-rod/>

![](img/4ed5039286acaa77f2608232e238df2a.png "threaded-rod-coup-door")

没有公鸡叫醒他们，史蒂夫和他的妻子对此很满意。他们不满意的是无论如何都要早起，以便把鸡从鸡笼里放出来。像许多小规模的蛋农一样，他们寻找一种自动解决方案，在早上打开门。

[史蒂夫]已经看到了一堆不同的自动车门黑客踢在互联网上。但是所有他能找到的都使用了垂直门和滑轮。他的装置有一扇水平打开的门，他意识到他需要建造某种线性致动器。他想出的是一个用五金店零件搭建的系统。他用的是一根普通的旧螺杆和一个连接螺母(它们通常有 3/4 英寸长)。使用导管安装支架将螺母牢牢固定在门上，而安装在门框上的电动螺丝刀可转动螺杆。两个限位开关由磁性传感器组成，通常用于在你进入商店时敲响大门的门铃。Arduino 负责调度和控制开门和关门的电机。休息之后，请观看高制作价值视频。

值得一提的是，我们已经见过至少一个[水平滑动的绳索和滑轮门](http://hackaday.com/2010/11/27/automated-chicken-coop-door-is-solar-powered/)。
