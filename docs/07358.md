# 反科技:蓝领，红色噩梦

> 原文:[https://hackaday . com/2014/10/21/retro tech tacular-blue-collar-red-nightman/](https://hackaday.com/2014/10/21/retrotechtacular-blue-collar-red-nightmare/)

本周的演讲是[一个精心制作的反共产主义宣传](http://www.youtube.com/watch?v=of_eApe5VlU)由国防部的一个部门制作，你可能从未听说过:武装部队信息与教育局。

讲述了因天罗地网和亚当-12 成名的杰克·韦伯。他告诉我们一个位于铁幕后面的假美国小镇。到处都是年轻的同志，他们在研究和表演美国经济和生活方式中的资产阶级活动时蹦蹦跳跳。介绍完这些之后，韦伯拉开了舒适的天鹅绒窗帘，描绘了一个由一个人(杰里·多纳文)领导的典型的美国家庭。

[杰瑞]拥有一切:由珍妮·库珀(Jeanne Cooper)饰演的睿智的妻子(最著名的是《年轻人与不安分者》)、对战争游戏有着浓厚兴趣的儿子、有着漂亮金发的年轻女儿，以及将在衬裙路口成名的美丽大女儿。在餐桌上听到这个女儿令人不安的消息后，杰瑞早早上床睡觉，打了几个盹。

杰克·韦伯否认[杰瑞]有任何关于甜李子的幻想，而是把他丢在苏联的法克斯维尔，让他做一个由共产主义重建的美国的生动噩梦。看着他弄明白发生了什么，以及新政权对他和他漂亮的家庭意味着什么。
