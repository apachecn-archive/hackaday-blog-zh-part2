# LVBots CES 开放日:桌面挑战和衣服机器人

> 原文:[https://hacka day . com/2015/01/23/LV bots-ces-open-house-desktop-challenge-and-ways-bot/](https://hackaday.com/2015/01/23/lvbots-ces-open-house-tabletop-challenge-and-clothes-bot/)

拉斯维加斯的机器人制造爱好者俱乐部 LVBots 在 CES 周举办了一次开放日。这是我和索菲唯一一次离开长街会议大厅的旅行，感觉棒极了！这个小组每月在 [Pololu](https://www.pololu.com/) 提供的地方召开两次会议，Pololu 是一家著名的机器人和电子产品制造商，总部位于麦卡伦国际机场以南。

在聚会的正式部分开始之前，有几个构建被展示出来。[Claire]和[Brian]最近参加了美国电话电报公司赞助的黑客马拉松。他们的发明是一个机器人壁橱。该系统包括移动由智能手机应用程序跟踪的衣架。该软件讨论的有趣功能包括监控每件衣服最后一次穿、最后一次洗的时间，以及它是否适合当前的天气条件。挖掘他们的回购中的代码[。](https://github.com/brianlu365/atthack2015)

在房间的其他地方，一对循线机器人在做他们的事情，几个相扑机器人在竞争将对方推出拳击场。一大群人聚集在投影仪周围，观看各种类型的机器人视频，就困难的部分、如何克服它们以及如何将这些方法应用到他们自己的构建中进行头脑风暴。我可以证明，与一群试图展示最令人惊叹的机器人演示的人在一起，会产生令人惊叹的观赏效果！

当会议的组织部分开始时，我很高兴听到 LVbots 集团的一个长期挑战。桌面挑战有多个阶段，旨在鼓励建造者适度开始，然后迭代以实现新的目标:

阶段 0:将一个机器人带到 LVBots
阶段 1:来回移动而不会掉下来
阶段 2:找到一个物体并将其推下
阶段 3:将物体推入一个目标

[Nathan Bryant]是今晚尝试挑战的两个机器人制造者之一。他用轻木和三个伺服电机建造了这个六足机器人，并正在测试第一阶段。该机器人包括一个悬挂在机器人前面的传感器，用于检测桌面是否不再下方。这时，它后退几步，原地转向，向相反的方向前进。[Nathan]提到他在电子表格中计算出了所有的动作，未来的固件升级将大大提高机器人的移动速度。我们喜欢机器人的声音节奏，这在上面的视频中很容易观察到。在某一点上，一条腿悬在边缘上，看起来像是(内森)把机器人推了回去，但我不记得他真的碰过它，所以我把这称为相机角度的恶作剧。

桌面挑战的下一个阶段是[乔·卡森]。他展示了一个他一直在研究的轮式机器人，它的前部有一个抓臂。机器人在桌子周围寻找预定义的颜色，在这种情况下由高亮标记提供。当发现机器人接近时，抓住，然后继续移动标记到它消失的地方；至少从机器人的角度来看是这样。
