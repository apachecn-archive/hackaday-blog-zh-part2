# 未来如此光明，我们需要光致变色的窗玻璃

> 原文:[https://hackaday . com/2014/10/07/retro tech tacular-the-futures-so-bright-want-need-to-need-colorsized-window panes/](https://hackaday.com/2014/10/07/retrotechtacular-the-futures-so-bright-were-gonna-need-photochromic-windowpanes/)

这是 1999 年夏天肖家人生活中的一天，这是菲尔科-福特公司从 1967 年太空时代的乐观主义中想象出来的。故事从凯伦·肖和她的儿子詹姆斯开始。他们在海滩上，建造他们的模块化六角形房子的沙堡模型，并讨论生活。不祥的音乐在他们的汽车里响起，一辆[福特 Seatte-ite XXI](http://www.rocketfin.com/ford_seattleite.cfm) 车的门漫不经心地开着。你可能会认出凯伦就是马季·杜萨伊，她后来会通过光束登上*号航空母舰*并移除斯波克的大脑。

父亲迈克·肖是一名天体物理学家，他在业余时间致力于开拓火星并培育巨大耐寒的桃子。他由标志性的美国游戏节目主持人温克·马丁代尔扮演。奇怪的是，温克的第一场演出是主持孟菲斯的儿童节目，名为火星巡逻。他凭借《T2》《井字游戏》《T3》《T4》《老千》《T5》《T6》《密码加 T7》《T8》《琐碎的追求》《T9》等经典之作声名鹊起。

迈克在与家庭电脑相连的屏幕上调出了一些他正在使用的母树的照片。虽然今天的许多家庭都有这样的设备，但这种动物几乎是有知觉的。我们从整部电影中了解到，它通过监控他们的生理、活动、财务事项，以及詹姆斯的教育，对他们的生活进行微观管理。

电脑一周给他上两天家教，主要是通过一个巨大的平板显示器。单独的控制台给他录制讲座，并测试他对材料的回忆，这是非常糟糕的。与此同时，他的母亲使用厨房控制台，根据他们的饮食需求和健康记录，为家人的午餐变出定制的菜单。她代表迈克讨价还价了一下，然后电脑开始工作。它选择冷冻的部分，然后用微波炉加热。凯伦唯一的工作就是把食物转移到盘子里，并从自动售货机里取出饮料。

午饭后，Karen 在网上购物，Mike 在办公室里抱怨她买的东西和家里的其他开销。他可以查看他们的银行余额、预算和剩余的汽车付款。延续了每件事都有专用显示器的主题，办公室有三台显示器，每一台看起来都只有一个用途。还有一个电子邮局，他可以用手写笔在平板电脑上给任何人写信。

![fuel cell hvac](../Images/af6ddba42436992b7ef2335627bea1f5.png)

肖家的房子由装有闪光灯的燃料电池供电。它还提供纯净水，燃烧他们的废物，加热和冷却空气，控制湿度，并消除花粉，灰尘和细菌。它肯定不会周期性地释放蒸汽，让他们保持平静和温顺，偶尔会忘记这是哪一年，或者是谁发射了第一架望远镜。

我们不想破坏整个事情。看着 Karen 用完了她大量的空闲时间扔罐子，Mike 出于某种原因穿着高领毛衣完成了电脑为他设定的锻炼方案。尽管这部电影可能很离奇，但菲利普-福特公司的许多梦想都实现了。模拟控制和看起来像接线板的开关很有趣，但化学蒸汽清洁厕所的想法很可怕。额外收获:这里有一段沃尔特·克朗凯特展示同一个办公室的视频短片。
