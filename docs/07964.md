# 3D 可打印 LED 扩散片

> 原文:[https://hackaday.com/2015/01/02/3d-printable-led-diffusors/](https://hackaday.com/2015/01/02/3d-printable-led-diffusors/)

虽然你可以得到任何尺寸或形状的 LED 矩阵，但看起来非常酷、非常适合低分辨率显示器的 LED 矩阵都有扩散片。当它们来自一家不知名的中国工厂时，这些扩散片是放入挤压塑料框架中的塑料薄片。由于[Jana]有一台 3D 打印机，她认为定制的扩散器[只是一些细丝和 SCAD 文件。](https://github.com/Jana-Marie/LOL-Display-Frame-Diffusor)

这种定制 LED 扩散器的基础是在最近的 31C3 会议上创作者送给[Jana]的一个 [LoL 盾](http://jimmieprodgers.com/kits/lolshield/)。这个防护罩实际上只有 126 个 led，多路复用，采用 Arduino 形式，并且许多 led 太亮，彼此之间模糊不清。一个 3D 打印扩散器的计划诞生了。

在进行了一些测量之后，一对 OpenSCAD 文件被快速生成并打印出来。组装包括将 126 个微小的白色扩散片压入一个框架，但一旦所有东西都附着在矩阵上，结果就值得了。
