# 微型盒式录音机成为磁带延迟

> 原文：<https://hackaday.com/2015/06/17/microcassette-recorders-become-a-tape-delay/>

早在音频工程师有花哨的数字延迟，甚至蹩脚的模拟延迟之前，就有磁带延迟了。使用录音和播放头循环播放磁带是 Echoplex 和 Space Echo 的基础，这两种机器都是令人难以置信的工程产品。

总的来说，微型盒式录音机并不是不可思议的工程作品。然而，它们有一条磁带、一个录音磁头和一个播放磁头。把两个放在一起，你就可以建造自己的磁带延迟器。

磁带延迟的基本原理非常简单——只需将一圈磁带绕一圈，通过录音和回放头，录下一些音频，然后将输出发送到放大器。实际操作起来，没那么简单。[dogenigt]不得不用微型卡式磁带制造自己的磁带环，这个过程耗时太长，而且过于挑剔。

对于控制电路，[dogenigt]使用四个音频电位计和一个线性电位计进行速度控制。音频电位计负责输入增益、反馈、干净信号的幅度以及信号经过延迟后的输出。

除了是那些非常依赖于建造者的机械技能的建筑之一，它是一个非常简单的延迟单元，所有的电子设备都已经为条形板布局设计。你可以听到下面的例子。
