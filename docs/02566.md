# 一艘 Arduino 氢气球飞艇…哦，人类！

> 原文:[https://hack aday . com/2013/01/19/an-arduino-hydrogen-blimp-oh-the-humanity/](https://hackaday.com/2013/01/19/an-arduino-hydrogen-blimp-oh-the-humanity/)

![arduino-hydrogen-balloon](../Images/b431d5a27a24f58900c84eeb7b427d4b.png)

这种飞行装置似乎更适合室内使用。嗯，除了[建造机器人控制的氢气飞艇](http://www.instructables.com/id/Arduino-Radio-Controlled-Hydrogen-Blimp)带来的火灾危险。我们经常看到的四轴飞行器的问题是，当一根电机线松了，东西朝一个随机的方向飞去。相比之下，失去飞艇上的一个马达没什么大不了的。

因为建造依赖于气体的浮力，所以重量轻的组件是游戏的名字。底盘的框架由轻木制成。它支持两个微型 DC 发动机，在上图中几乎无法区分。Arduino nano 和无线接收器监控来自发射器的命令，并相应地驱动螺旋桨。

你可能已经注意到了，我们把这个归类为化学黑客。这是因为[Btimar]自己产生了氢气。他用一个带喷嘴的锥形烧瓶进行化学反应。在烧瓶中放置几个散热片和其他固体铝碎片后，他倒上碱液。这会产生 H2，但你需要用冰来保持冷却，以防止反应失控。我们将暂时坚持使用[充氦软式飞艇](http://hackaday.com/2011/12/09/rc-blimp-uses-a-party-balloon-for-lift/)！

在休息后的剪辑中看到这只野兽在[Btimar]的客厅里飞来飞去。
