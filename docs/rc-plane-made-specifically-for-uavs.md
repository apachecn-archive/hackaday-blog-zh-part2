# 专门为无人机制造的遥控飞机

> 原文：<https://hackaday.com/2012/09/01/rc-plane-made-specifically-for-uavs/>

![](img/5c7106628ff4ddebc80d4428cf1767ce.png "UAV")

我们已经看到了相当一部分遥控飞机变成了无人机和 FPV 平台，但[tech pod](http://www.kickstarter.com/projects/841500112/techpod-the-hobby-uav)是我们看到的第一架专门设计用作配备摄像头的机器人飞机。

Techpod 是韦恩·加里斯的创意。他驾驶装有摄像头的 FPV 飞机已经有一段时间了，但最近他意识到目前提供的遥控飞机不能满足他的需求。[Wayne]决定设计他自己的飞机，特别设计在机头安装一个云台摄像机。

[Wayne]的原型是用一些非常花哨的航空设计软件包设计的，并且是用泡沫磨出来的。从休息后的视频中，我们可以看到 Techpod 飞得很漂亮，但需要 Kickstarter 社区将他的模型带给大众。

Techpod 的规格把它与其他高性能的 FPV 和无人机模型放在一起；凭借其 102 英寸(2590 毫米)的翼展和一对并联的电池，Techpod 可以在高空传输视频长达一小时。
