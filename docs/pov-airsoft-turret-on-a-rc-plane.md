# 遥控飞机上的气枪炮塔

> 原文：<https://hackaday.com/2013/10/16/pov-airsoft-turret-on-a-rc-plane/>

老实说，我们很惊讶以前没有人尝试过这种方法——不过，也许有人尝试过！但我们知道的是，Flite Test 的人已经在一架大型遥控飞机上安装了一把电动气枪，被恰当地称为[克拉肯。](http://flitetest.com/articles/airsoft-gunship-the-kraken)

这些飞机是他们自己的 [FT Versa Wing](http://flitetest.com/articles/ft-versa-wing) 的放大版本，具有两个支柱，赋予他们不同推力的能力。他们不仅能够在顶部绑上半自动气枪，他们还有两个 GoPros 拍摄动作，并给炮手一个击落敌机的 POV！不过不要担心，敌机有自己的武器——永久标记！这似乎不太公平，因为标记越近，击落飞机就越容易！
