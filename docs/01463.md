# 伞状风车

> 原文:[https://hackaday.com/2012/08/29/umbrella-based-windmills/](https://hackaday.com/2012/08/29/umbrella-based-windmills/)

[尼克拉斯·罗伊]又来了。他正在利用雨伞将风力应用到他的项目中。他的灵感来自风速计的形状，雨伞被证明是一个很好的选择，因为它们便宜又容易找到。

风速计通过用蛋形帆捕捉风速来测量风速(事实上，我们以前见过用塑料复活节彩蛋制成的[)。伞的面积更大，可以吸收更多的风。然而，从测量风速到产生能量，这是一个巨大的飞跃。这就是为什么他不试图发电，而是直接利用机械力。他从去年的一个项目](http://hackaday.com/2009/05/22/weather-canvas/)中借鉴了[的经验，用双伞装置给音乐盒供电，从而彻底改造了风铃。上面看到的三伞装置就像一个泡泡机，驱动一系列塑料环穿过肥皂溶液，让风来完成剩下的工作。休息之后我们嵌入了他的演示视频。](http://hackaday.com/2011/08/24/water-powered-multi-channel-audio/)
