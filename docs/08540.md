# 圆周率日快乐。3.14159265359……

> 原文:[https://hackaday.com/2015/03/14/happy-pi-day-3-14159265359/](https://hackaday.com/2015/03/14/happy-pi-day-3-14159265359/)

除非你生活在 1592 年，否则这是最接近完美的“圆周率日”了。3/14/15.如果你想在 9:26:53(上午或下午)暂停一会儿？时区？国家？)我们马上就来陪你。嗯，算是吧。

现在，在你大笑，认为圆周率只是一个数字之前，请三思。本质上几乎是神奇的。它对我们日常生活的影响无处不在。从生物学到天体物理学。从历史的角度来看，它可以作为一个里程碑，标志着许多不同文化的数学和科学的进步。简而言之，圆周率是现代世界的基石。没有它，我们永远也不会登上月球。我们可能还在马背上。忘掉电子产品吧。那些有趣的事情永远不会发生。

(作为旁注，3/14 这一天，恰好是阿尔伯特·爱因斯坦的生日。生日快乐！)

我们希望你有一个可怕的圆周率一天。我们也希望在评论中看到你。例如，我们希望听到和看到计算圆周率的算法的帖子。在树莓圆周率上你能计算圆周率多远？或者甚至是你第一次遇到那个神奇数字的童年故事。发帖吧，让我们知道你的想法。
