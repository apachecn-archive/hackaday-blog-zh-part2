# 用于家庭局域网的远程无线传感器

> 原文:[https://hack aday . com/2014/07/19/远程无线传感器家庭局域网/](https://hackaday.com/2014/07/19/long-range-wireless-sensors-for-the-home-area-network/)

在不久的将来，我们都将居住在拥有数百个小型设备的家庭中，这些设备与一个易于连接和控制的传感器网络交织在一起。多年来，世界各地都出现了类似这个无线传感器系统的项目，任何人都可以建造这个系统。

[Eric]希望他的工作将有助于通过让开发人员建立廉价的电池供电的远程无线传感器来实现真正广阔的家庭局域网(HAN)。他的方法集成了可插拔的 OSGI 架构和家庭自动化平台，同时使用 Arduino 作为低功耗的传感器节点，能够利用在线找到的多种廉价传感器。

[Eric]的教程描述了这些开源平台的可能性的几个例子。例如，他展示了他所谓的“邮箱哨兵”，这是一种电池供电的邮件监控设备，使用树莓皮播放臭名昭著的，古老的美国在线原声“你有邮件。”一旦邮递员来了，它还会发送一封电子邮件。

此外，他还列举了其他一些想法，比如婴儿监控哨、洗衣机/烘干机通知系统、漏水探测器，以及在有人试图闯入时发出响亮警报的安全措施。所有这些潜力只需 20 美元左右。

正如[Eric]所说，这个项目成功的关键是将 Ardiuno 和 openHAB 平台联系在一起的 MQTT 绑定。这允许通过物联网设备中常见的以太网连接发送简单的消息。

所以，所有在外面的开发人员都回家去，开始思考下一步可以连接什么！因为有了这个系统，你只需要几个十点和一个互联网插头，你就有了一个强大的基础来建立自己。剩下的就看你的了。
