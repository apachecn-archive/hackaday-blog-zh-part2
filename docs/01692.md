# 立体显示每秒投射 2 亿个体素

> 原文：<https://hackaday.com/2012/09/23/volumetric-display-projects-200-million-voxels-per-second/>

![](img/1d0496ed2fba7017b21fd8fd4faef396.png "3d")

在过去的四年里，[威尔]和[加文]花时间创造了一个巨大的，高分辨率的 3D 显示器。他们刚刚完成他们的构建，所以他们决定将它提供给互联网，希望人们为他们的显示器创造新的 3D 内容。他们称他们的项目为全息穹顶，这是我们见过的分辨率最高的立体显示器。

全息穹顶通过以每秒 20 转的速度围绕其垂直轴旋转半透明螺旋来运行。螺旋上方的微型投影仪每秒能够投影 1440 帧([一个惊人的设备本身](http://www.digikey.com/product-highlights/us/en/texas-instruments-dlp-technologies/688))每秒 60 帧 x 和 y 帧中的每一帧显示 72 帧 z 轴帧。结果是具有 480 * 320 * 72 体素分辨率的 3D 显示器能够每秒显示 20 帧。

这不是我们第一次看到[扫掠螺旋被用作体积显示器](http://hackaday.com/2012/06/14/volumetric-display-looks-like-a-davinci-sketch/)，但它是近年来同类显示器中分辨率最高的。[Gav]和[Will]已经把他们的全息穹顶放在了澳大利亚众筹网站 Pozible 上，如果你想买自己的，但谢天谢地，这些人在主网站上提供了足够的细节来重建这个项目。
