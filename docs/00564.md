# 在天空中建造你自己的眼睛

> 原文:[https://hacka day . com/2012/05/28/打造你自己的天空之眼/](https://hackaday.com/2012/05/28/building-your-own-eye-in-the-sky/)

![](../Images/82a7bf0f02b9e4293107ec54c150037f.png "building-an-eye-in-the-sky")

他一年内每周发一篇文章的目标已经过去了，但是[Dino]继续在新项目中运用他的技能。这一次，他在遥控直升机上安装了无线摄像头。

这些无线电控制的飞行器(有便宜的使用红外控制的，不太可靠)可以在 30-60 美元左右找到。[Dino]已经有一个无线摄像头可以使用，但是添加它和 9V 电池实在是太重了。经过一些测试后，他确定 2oz 的有效载荷是上限。他开始从直升机上拆卸零件，以获得足够的节省来抬起相机和电池。在这个过程中，他发现去掉飞杆上的重量增加了很多机动性，代价是稳定性损失很小。

休息后看看他嵌入的项目视频。虽然比不上[专业多转子相机支架](http://hackaday.com/2011/10/12/sweet-octocopter-takes-the-red-epic-to-new-heights/)的效果，但是便宜又好玩！
