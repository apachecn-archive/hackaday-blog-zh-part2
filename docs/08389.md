# 本周失败:畏缩既是名词也是动词

> 原文:[https://hack aday . com/2015/02/24/fail-of-the-week-wince-is-a-noun-and-a-verb/](https://hackaday.com/2015/02/24/fail-of-the-week-wince-is-a-noun-and-a-verb/)

几年前，[localroger]发现一些令人难以置信的硬件在出售:一台非常小的笔记本电脑，有 7 英寸的屏幕，全键盘，触控板，以太网，WiFi，USB(支持许多 HID 设备)，以及可以持续几个小时的电池。它们以 30 美元的价格出售，[localroger]买了四个。你说，很多？这些机器运行 Windows CE。不，拥有一台 WinCE 设备并不是本周的败笔。

考虑到他应该用这些机器做些什么，[罗杰]想，“一个时钟就可以了”，并开始考虑如何为这些东西编写程序或编写应用程序。这些微型上网本确实配备了内置 IE6 网络浏览器形式的编程语言 JavaScript。这实际上是一个非常非常好的解决方案——为纵向显示格式化的 WinCE 应用程序无法在“宽屏”笔记本电脑上运行，手工编码的 HTML 表格可能是任何人都希望的最佳解决方案。

这些机器—[ Roger]多年来将其中的三台用作闹钟——很好地完成了任务，即使 NTP 已经被排除在操作系统映像之外。这里真正的失败来自于购买一台 30 美元的 WinCE 上网本，并把它用于像闹钟这样的关键任务。显示器烧坏了，电池开始膨胀，一个单元不知何故不允许 IE 运行(可能是坏的闪存芯片)，另一个单元的触控板让光标随机游走。你得到你所支付的。

这些 WinCE 上网本终于被放了出来，[希望同样的激光打印机去](https://www.youtube.com/watch?v=dLk-3HPS12Q)。不过，这都是出于好意；[罗杰]用 Nixies 做了一个更好的闹钟[。](http://hackaday.com/2014/01/02/once-twice-three-times-a-nixie/)
