# 3D 打印时钟的时代即将到来

> 原文：<https://hackaday.com/2014/05/24/the-hour-of-the-3d-printed-clock-draws-nigh/>

![clock_3d_printed_display_large](img/04c74d15e4c28eaf814aae74794b6622.png)

许多人都尝试过，但[Christoph Laimer]已经成功地设计出一个几乎完全由 3D 打印部件制成的(相对)精确的时钟。[Christoph]时钟的每一个齿轮、滑轮、轮子和指针都印有图案。只有几个螺钉、轴、一个砝码和一根绳子没有打印出来。甚至给时钟上弦的曲柄也是 3D 打印的零件。

[Christoph]在搅拌机里设计了他的钟。这需要相当多的设计工作来创造可以工作和可打印的部分。更多的工作是打印 100 多个失败的原型零件。

有人可能会认为[Christoph]正在使用 Makerbot 或 Utimaker 等公司的最新打印机来实现这一壮举。原来他用的是停产的 Rapman 3.2 打印机。这进一步证明了即使是“老”打印机也能做大事！[Christoph]他的打印机确实运行得很慢。用 0.125 毫米的层和 0.4 毫米的喷嘴打印一个单一的齿轮需要他 2 或 3 个小时。

在机械方面，该钟由重力驱动，带有一个[锚式擒纵机构](http://en.wikipedia.org/wiki/Anchor_escapement)。[Christoph]选择使用[摆轮](http://en.wikipedia.org/wiki/Balance_wheel)和[游丝](http://en.wikipedia.org/wiki/Balance_spring) 组件来控制擒纵机构，而不是钟摆。甚至弹簧也是由标准 PLA 印刷而成。重物悬挂在滑轮组上。这个钟不是特别有效率。70 厘米的身高只能跑 2 个小时。

[Christoph]时钟已经被证明精确到每小时 1/4 秒以内。他没有提供温度稳定性数据，但作为一名解放军，我们建议不要太热！
