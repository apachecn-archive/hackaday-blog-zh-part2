# 打个手机，找个帐篷

> 原文：<https://hackaday.com/2014/06/12/call-a-cell-phone-find-a-tent/>

在音乐节上找到你的帐篷是一件棘手的事情——不仅有数英里长的帐篷和你的一模一样，你很可能会有一些啤酒或其他影响你的空间感和/或时间感的物质。想出了一个找到你的帐篷的好办法，用圣诞灯和手机照亮它。

[詹姆斯]建筑的基本理念是每当他打电话时，圣诞灯光就会闪烁。几乎每部手机都有一个可以用来完成这项任务的功能，那就是当接到电话时，背景灯就会打开。添加一个光电晶体管，一点点电路，和一些圣诞灯，你就有了一个功能齐全的帐篷探测器。

电路是一个简单的继电器驱动圣诞灯，由光电晶体管上方的电话触发。这是一个简单的电路，可以建立在一块 veroboard 上，用几块塑料形成外壳，提供了一个合理的防弹设备，可以在音乐节的严酷条件下生存。另外，不需要修改手机来触发远程电路。下面是该设备的运行视频。
