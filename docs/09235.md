# “吉格计数器”让辐射探测变得超现实

> 原文:[https://hack aday . com/2015/06/08/giger-counter-makes-radiation-detection-超现实/](https://hackaday.com/2015/06/08/giger-counter-makes-radiation-detection-surreal/)

这里有一个小问题:盖革和吉格(H.R. Giger，外星人的设计师)发音相同吗？答案是否定的。然而，这位已故艺术家的名字多年来被很多人念错了(根据记录，是酥油)。(史蒂夫·德格罗夫的)朋友发了一条愚蠢的推文，给了他灵感，最终给这件事加上了一个骨架盖子，即 [Giger Counter](http://www.instructables.com/id/Giger-Counter/) 。

内部是一个[可能的盖革计数器套件](http://www.adafruit.com/products/483)。外壳是这个项目中真正的黑客所在，由 1:2 比例的塑料骨架模型、柔性导管和深色金属喷漆制成。模型中仅使用了胸腔、一些椎骨和部分头骨。它们以一种令人愉快的非人类方式组装在一起，一些导管缠绕在它的周围，并进入胸腔的底部，以便更好地测量。经过一些粘合和喷漆后，盖革计数器套件的 LED 通过头骨上的钻孔放置，而电路板位于胸腔内。将电路板放入和取出可能有点棘手，但看起来电池可以更换，而不必将整个电路板取出。
