# 当你不注意的时候，床头柜上的开关会隐藏起来

> 原文：<https://hackaday.com/2014/09/08/end-table-kegerator-hides-the-tap-when-youre-not-looking/>

有什么比普通的床头柜好？一个可以为你提供啤酒的茶几怎么样？[Sam]有这个确切的想法，并利用他的技能使之成为现实。构建的第一步是获得一个足够大的端桌，以容纳一个功能性 kegerator 的所有组件。这被证明是有点棘手，但[山姆]很幸运，从一个车库拍卖中只花了 5.00 美元就买到了一个合适的茶几。

接下来，[Sam]用浴室密封胶封住床头柜上的所有裂缝。这一步对保持内部低温很重要。良好的保温可以保持啤酒更冷，同时使用更少的电力。接下来，在桌子的顶部挖了一个洞作为通风塔。

通风塔安装在几个抽屉滑轨上。这使得塔可以上升和下降，当你不想要它时，让它看不见。这座塔用一个简单的滑轮系统升降。一根细而高强度的绳子系在塔上。另一端连着一个线轴和一个小马达。电机可以卷绕或松开卷轴，以升高或降低塔架。

桌子上有一个 Arduino，它通过自制的 H 桥控制电机。Arduino 连接到一个温度传感器和一个小液晶屏上。这样，用户在喝啤酒之前就可以知道啤酒有多凉。

为了冷藏啤酒，[山姆]拆了一个迷你冰箱。他把压缩机和冷凝器盘管移到新桌子上。他不得不弯曲线圈，小心不要扭结。最后，他扔进了小桶，二氧化碳罐和调节器。最终产品是一个客厅宝石，按需提供啤酒。
