# 按下按钮，接受熏肉。

> 原文:[https://hackaday.com/2014/11/19/push-button-receive-bacon/](https://hackaday.com/2014/11/19/push-button-receive-bacon/)

兔子洞黑客空间的成员上周末[参加了解构](http://thedeconstruction.org/team/rabbit-hole/)，这是一场 48 小时的黑客马拉松比赛。黑客空间的主题是“点亮它！”所以成员们创造了一些与光有关的很棒的项目。展览的明星是他们的熏肉烹饪机。兔子洞制造了“按钮。收到培根”的迷因真实。

一台损坏的激光打印机的传动系统和定影器组件被拆下。激光打印机定影器本质上是热辊。当墨粉通过打印机时，辊子熔化墨粉并将其与纸张融合在一起。在这种情况下，热量来自滚筒内的灯。这盏灯也发出大量的光，这与团队的主题完全吻合。

然而兔子洞成员还没有完成，他们还用一个空的 Altoids 罐子做了一个口袋大小的无限镜。罐头的底部被切开，然后粘上一面镜子。一个坏投影仪的过滤器做了一个完美的半银镜，一些 led 完成了这个项目。

成员们还建造了一个粉丝艺术作品，由 25 个粉丝以头骨形状连接在一起组成。眼扇和鼻扇都亮着。当风扇插上电源时，它们踢了几秒钟才开始旋转。然而一旦它们真的旋转起来——兔子洞里有一股强大的风。
