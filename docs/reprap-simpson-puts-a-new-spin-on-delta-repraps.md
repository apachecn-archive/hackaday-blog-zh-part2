# RepRap Simpson 对 Delta RepRaps 进行了新的诠释

> 原文：<https://hackaday.com/2013/09/18/reprap-simpson-puts-a-new-spin-on-delta-repraps/>

就在你认为你已经在 3D 打印机世界看到了一切的时候，一些新的东西突然出现了！[尼古拉斯·苏厄德]发布了一段重新说唱辛普森的视频，这是他的最新作品。辛普森是一个三角洲机器人——但不同于我们之前见过的任何一个[三角洲。以前的产品臂在垂直轨道上运行。正如你所看到的，这种设计将三个铰接臂直接安装到打印机的底座上，使用钢缆作为关节机构的一部分。](http://hackaday.com/2012/07/13/3d-printing-with-a-delta-robot-that-seems-to-simplify-the-concept/)

从[Nicholas]在[RepRap 论坛](http://forums.reprap.org/read.php?178,206458)上的帖子来看，辛普森的接地三角形设计已经经过了一些修改。但是基本的几何形状保持不变。[尼古拉斯]称这个版本为“比例齿轮传动接头辛普森”。这个名字可能不会脱口而出，但动作非常流畅、自然、快速。

与任何 delta 设计一样，反向运动学在软件中发挥着巨大的作用。[Nicholas]正试图用光学校准系统简化这一过程。对于喜欢冒险的人来说，这些方程被发布在[论坛](http://forums.reprap.org/read.php?178,206458,207216#msg-207216)上，一个 python Gcode 预处理器被发布在 [Thingiverse](http://www.thingiverse.com/thing:104117) 上。

甚至辛普森的基地也受到了特别的关注。它是由喷水切割的玄武岩建成的。我们喜欢在大关节上使用相对的斜齿轮，以及用于张紧线缆驱动的吉他机头。我们不确定的一点是系统的寿命——电缆拉伸会是一个问题吗？打印的零件会受到线缆的磨损吗？只有时间能证明一切。
