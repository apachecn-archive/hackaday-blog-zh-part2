# 一个你可以引以为豪的 50 美元的投影屏幕

> 原文:[https://hack aday . com/2012/09/08/a-fifty-dollar-project-screen-you-can-proud-of/](https://hackaday.com/2012/09/08/a-fifty-dollar-project-screen-you-can-be-proud-of/)

![](../Images/138cb28ce3152d2264ae1130abc117a9.png "fifty-dollar-projection-screen")

[Lou]来信分享了他在自己家里制作的价值 50 美元的投影屏幕。我们最近已经看到了几个这样的项目。不像[用在湖边小屋](http://hackaday.com/2012/09/04/movie-night-at-the-lake-cabin/)，或者[填充一个尴尬的壁龛](http://hackaday.com/2012/08/21/128-inch-silver-screen-for-your-viewing-room/)，这个版本没有使用织物做屏幕。他真的把它画在了墙上。

获得优秀最终产品的关键是确保你的墙壁是平的。[Lou 的]教学视频(休息后嵌入)显示如何修补墙上的洞，以及修复高点。在开始这个过程之前，他使用投影仪的网格功能绘制出将被用作观看区域的墙壁部分(就是上面屏幕上看到的网格)。一旦该区域用遮蔽胶带做了标记并被仔细修复，他就用亮白色或银色的油漆进行油漆。你也可以考虑使用油漆添加剂来达到更好的效果。我们已经看到了用于此目的的[喷砂珠](http://hackaday.com/2012/02/22/projection-screen-using-latex-paint-and-sand-blasting-beads/)。

一个框架被添加到该区域，使它看起来像一个适当的屏幕。这只不过是覆盖着黑色织物的模塑件。[Lou]拉伸模塑周围的织物，用胶带将其固定住，直到它可以被钉住。
