# Retrotechtacular:自由的火力

> 原文：<https://hackaday.com/2015/05/19/retrotechtacular-firepower-for-freedom/>

随着美国的安定，其领导人发现他们需要火力来维护自由。这在那个时代的军事交战中变得尤为明显，因此[建立了许多专门的设施](http://www.youtube.com/watch?v=EgJCUpncoY4)来管理不同类型弹药的研究、开发、制造和传播。

新泽西州的皮卡汀尼兵工厂是核武器和常规武器研发的地方。在这个设施中工作的男人和女人创造了反人员装置，包括[一种叫做 Flex-X 的柔性粘性装药](http://www.youtube.com/watch?v=HUJd_jKeStY)，它可以固定在几乎任何东西上。这种爆破炸药可以分层以增加威力，甚至可以在水下引爆。皮卡汀尼还开发了新的火箭发动机、推进剂和射弹的液体推进装置。

在宾夕法尼亚州，一家名为弗兰克福特兵工厂的小型武器弹药厂开发了一种复式步枪弹夹。也就是说，一个铅弹射向目标，另一个坐在它后面的子弹以一个角度射出，落在距离铅弹一英寸左右的地方。弗兰克福特的工人们还研磨了用于目标瞄准和对中的精密光学器件，并开发了一种无壳弹药筒。弗兰克福特公司还生产了适合各种用途的推进剂。这些推进剂被用来运送神经毒剂解毒剂，快速给救生筏充气，并在危险情况下弹射飞行员。

巴尔的摩的埃奇伍德兵工厂专门从事化学武器的研发、制造和供应。他们特别擅长灭火。埃奇伍德研究公司也提供了民用好处，比如炭疽疫苗。此外，马里兰州德特里克堡有一个生物 R&D 翼，在那里开发重要的解毒剂和疫苗。

所有这些 R&D 和制造都是由位于伊利诺伊州乔利埃特附近的弹药采购和供应局(APSA)策划的。除了对所有承包商的投标一视同仁之外，APSA 还控制着分销，在大型计算机上维护库存，这些计算机可以像处理任何事情一样处理数据。
