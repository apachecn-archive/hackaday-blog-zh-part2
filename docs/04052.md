# 问问 Hackaday:我们能比 Phonebloks 做得更好吗？

> 原文：<https://hackaday.com/2013/09/13/ask-hackaday-can-we-do-better-than-phonebloks/>

我们的提示热线正在进行中，建议我们报道名为 phone bloks T3 的 T2 模块化手机概念。这款手机的设计者这样描述这个问题:

> 一部手机只能用几年就坏了或者过时了。虽然通常只是一个部件导致了它的死亡，但是我们把所有东西都扔掉了，因为几乎不可能修复或升级。

他的解决方案是上图中的手机，每个功能都有模块化组件:wifi、摄像头、电池等。不要升级整个手机，只升级你需要的部分。一波追随者支持这个概念，也许他们的心是正确的，希望减少浪费和成本。然而，在 Hackaday 的幕后，大家的反应是一致的。主要的反对理由(除了设计上的不可行性)应该是显而易见的:将手机分成可互换的部分并不能从本质上减少浪费。这些碎片必须放在某个地方。

现在，不要急着去评论区找出额外的问题；有一个有趣的 Reddit 主题可以解决这个问题。相反，我们想走捷径:我们能做得更好吗？我们能不能为未来制造一款生产更少浪费、更容易回收、可能还能升级的手机？它的特性包括什么，我们如何去做？观看概念手机的视频，并在休息后告诉我们您的替代方案。
