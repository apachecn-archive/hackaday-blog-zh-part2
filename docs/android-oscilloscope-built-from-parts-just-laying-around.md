# Android 示波器是由随处可见的零件组装而成的

> 原文：<https://hackaday.com/2012/07/14/android-oscilloscope-built-from-parts-just-laying-around/>

所以你需要调试一个电路，你没有示波器。这不是一个问题，多亏了零美元的安卓示波器，它是由他碰巧放在身边的零件制成的。

每台现代示波器的核心都是 ADC——接收模拟输入并输出数字信号的芯片。每个 Android 设备都有一个这样的转换器连接到麦克风端口。所有[retronics]需要做的就是焊接一个 3 毫米的耳机插孔，连接几个电阻，并附上一对鳄鱼夹。在安装了一个[示波器应用程序](https://play.google.com/store/apps/details?id=ch.serverbox.android.osciprime)之后，【retronics】有了一个还算不错的“示波器”。

是的，这确实是一个穷人的示波器，而且[retronics]可能不会很快用他的 Android 麦克风插孔调试高频 RF 电路。低频的东西，如音频是这个范围真正擅长的地方；如果[中加入一个小前置放大器](http://www.instructables.com/id/A-Preamplifier-for-Smartphone-Oscilloscopes/)，情况就更是如此。

您可以在休息后查看[retronics]的构建。当然，这不是精确和校准的测量，但有时你只需要一个工具就可以了。
