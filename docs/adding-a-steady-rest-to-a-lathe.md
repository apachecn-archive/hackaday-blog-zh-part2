# 给车床增加一个稳定的支架

> 原文：<https://hackaday.com/2014/11/28/adding-a-steady-rest-to-a-lathe/>

稳定刀架是车床的一种工具，它能使机械师在细长的坯料上进行深切削，钻出薄金属片，并且通常能使薄材料保持平直。与跟随刀具的刀具不同，稳定的刀架牢牢地固定在车床的床身上。[Josh]的车床没有稳定的支架，他不能只为它获取零件。没问题，然后:他已经有一台车床、一台磨和一些金属，为什么不从头开始做一个底座呢？

[Josh]能够从网上经销商那里找到真正的稳定支架，但这不是为他的车床制造的。当把它安装到他的机器上时，出现了一个问题:因为每个稳定的支架必须适合车床的床身，他需要一个定制的支架。在一个相当大的磨的帮助下，[Josh]磨掉了一块钢的所有侧面，并切割出 45 度的凹槽。为了使这个底座水平，[Josh]将底座的一侧放在车床上，在刀架上放一个千分尺，并得到一个准确的读数，即需要从未切割的一侧去除多少金属。

随着稳定的休息螺栓上车床，[乔希]转动一杆，发现他偏离了约 0.002 英寸。对于机械师来说，这不是很好，但对于一个快速的项目来说，这是梦幻般的。无论如何，[乔什]真的需要一个稳定的休息，如果它的工作，你真的不能抱怨。
