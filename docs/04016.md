# 我们在招人

> 原文:[https://hackaday.com/2013/09/08/were-hiring/](https://hackaday.com/2013/09/08/were-hiring/)

![HaD_help_wanted](../Images/627f4497105494c73699a2a9acae3cf9.png)

标题说明了一切。我们需要更多的作家来保持新鲜的黑客来了，现在是你申请这份工作的机会。

**更新:**感谢所有的申请。我已经向几个精选的人发出了第一轮邀请。将范围缩小到少数人是很困难的，因为有如此多的高素质、热心的读者提交了申请。我们仍然欢迎您提交申请，我们会将申请存档，但我预计目前不会再有任何报价。

投稿人被聘为私人承包商，每个职位都有报酬。作家应该有专业的技术知识来理解他们所写的项目，并对我们报道的广泛话题充满热情。如果您感兴趣，[请发送电子邮件至我们的招聘热线](mailto:jobs@hackaday.com)，并附上:

*   关于你的背景(教育、工作等)的细节。)让你成为团队中有价值的一员
*   链接到你的博客/项目帖子等。已经在互联网上公布了
*   《黑客之声》中的一个例子。包括一个横幅图像，150 个字，该项目的链接，以及任何相关的黑客一天功能的链接。

### 的鼓励话语

首先，我们不会公开讨论薪酬。想知道我们付多少钱吗？提交一份成功的申请，我们将讨论它。

其次，不要错过这个机会。我看着这些帖子中的一个过去，又等了一年，才看到下一个并申请。现在我在经营这个地方。我们的团队由热心的读者组成。如果你对这里的东西充满热情，而且你每周有几个小时来写作，你需要现在就申请！

### 为什么我们要雇佣更多的作家？

我们的口号说明了一切。我们希望每天都有新黑客以一致的速度出现。现在我们正在实现这个目标。但是，当我们添加更多原创内容时，比如精彩的 3D 打印系列【布莱恩】开始，或者[会议报道](http://hackaday.com/2013/09/07/open-hardware-summit-2013-part-1-demos/)【埃里克】一直在做的，我们需要帮助，以确保我们不会错过我们特色的令人敬畏的项目。随着新作家发现他们对日常特征的适应，如果他们愿意，他们有机会写自己的原创内容。

你还在等什么？女士们先生们，开始申请吧！
