# 一种 3 轴切纸微型激光器

> 原文:[https://hack aday . com/2014/03/16/a-3 轴-剪纸-迷你激光/](https://hackaday.com/2014/03/16/a-3-axis-paper-cutting-mini-laser/)

![LaserCutter](../Images/f04a31da4cd9bdd4651fe15b6b74558e.png)

激光很棒，使用激光的项目也很棒。[kokpat]最近的一份说明书概述了如何使用 CDROM 步进电机和 Arduino 创建一个全功能的[激光切纸机](http://www.instructables.com/id/Arduino-3-axis-Mini-Lazer-Paper-Cutter/?ALLSTEPS)。

这个构建的特别之处在于，它展示了构建用于激光切割机、CNC 机器和 3D 打印机的 3 轴机械系统是多么容易。使用由电机螺杆和基于螺母滑块的托架组成的步进台，机械系统可以非常容易和成本有效地组装在一起。幸运的是，从电子和软件的角度来看，随着 [RepRap](http://reprap.org/wiki/RepRap) 和类似机器的激增，一切都很标准化。只需选择任意三个步进驱动器，找到最相关的固件，瞧！你完了！嗯，差不多了。别忘了一个 100mW 的紫激光！

我们以前见过一辆[吨](http://hackaday.com/2013/04/03/homebuilt-laser-cutter-ideas/)的真正[酷](http://hackaday.com/2013/05/06/dvd-laser-diode-used-to-build-a-laser-engraver/)激光[切割机](http://hackaday.com/2013/12/20/diy-laser-cutter/)，但这必须是最便宜的。休息后观看激光切割机的运行。
