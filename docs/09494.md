# 德瓦尔特无线电修理

> 原文：<https://hackaday.com/2015/07/10/dewalt-radio-repair/>

我们喜欢维修视频，来自 Hackaday 校友[Todd Harrison]的这个 [Dewalt 工地无线电维修](http://www.toddfun.com/2015/06/27/repair-dewalt-radio-part-2-of-2/) (YouTube 链接)也不例外。就像侦探故事一样，我们总是试图猜测是谁干的。

在他的第一个视频中，【托德】追踪到问题出在一个有故障的 6 伏稳压器上，它产生了 8 伏电压。他通过在电路中接入一个 LM317 来取代原来的不可调节部分，从而解决了这个问题。这很有帮助，但是几天后收音机又坏了。所以他在这里追踪电压来找到第二个罪犯。一路上，我们看到了他的福禄克 87 米和 289 米的一些更好的特点。以及在确定最终问题之前，对无线电构造中的一些设计决策感到困惑。

我们不会破坏这个惊喜，但在下面的视频中，我们会发现托德是如何解决这个包裹在谜团中的谜语的！
