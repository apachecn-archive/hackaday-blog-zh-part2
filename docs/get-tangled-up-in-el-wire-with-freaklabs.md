# 和弗雷实验室的电线纠缠在一起

> 原文：<https://hackaday.com/2013/12/10/get-tangled-up-in-el-wire-with-freaklabs/>

弗雷克实验室的 Akiba 一直在研究电致发光线。整个舞蹈团的价值！我们知道【秋叶】是因为他在海啸[后在东京黑客空间](http://hackaday.com/2011/03/23/geiger-counter-ad-conversion-for-radiation-level-crowdsourcing/)的辐射监测工作。如今，他是“救援队管弦乐团”的工程师之一，该舞蹈团去年上演了《T4》、《创舞》(T5)。救援队管弦乐队最近刚刚上演了一部名为《宇宙节拍》的新作。宇宙节拍通过添加舞台投影映射和强大的激光，以及钢铁侠反重力式手持 led，将抢险队的表演提升到一个全新的水平。

正如人们所料，EL wire 服装是由电脑控制的，它能让所有演员的灯光效果保持完美的时间。这就是[秋叶]的用武之地。现代剧院被淹没在射频噪音的海洋中。几千瓦的照明由三端双向可控硅开关控制，发出巨大的噪音。闪光灯和相机闪光灯，以及携带手机和无线设备的所有观众只会增加这一点。不管有没有射频噪音，演出必须继续，EL 服装和 led 必须工作。为此，[Akiba]他还为该集团创造了新的发射机。他还将安装在照明亭上的发射天线从全向鞭状天线换成了定向天线[八木](http://en.wikipedia.org/wiki/Yagi-Uda_antenna)。

EL 电线本身也是一个问题。这根电线不够亮。由于舞者除了控制电子设备之外还穿着 25 米长的电线，所以在电线上折叠起来会很困难。有时候最好的工程实践必须让位于艺术，所以[Akiba]不得不加快速度。这意味着电线经常烧坏。舞蹈团非常擅长在演出期间和演出间隙更换电线。如果你想仔细看看，在[【秋叶的】flickr 流](http://www.flickr.com/photos/22874071@N05)上有很多图片。
