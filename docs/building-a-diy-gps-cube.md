# 构建 DIY GPS 立方体

> 原文：<https://hackaday.com/2012/06/06/building-a-diy-gps-cube/>

![](img/73c8dda1b45e54f1cd871d0b117380a3.png "gps")

最初，[卡门]想为他的自行车制造一个速度计。特征蠕变愚弄了我们所有人，所以在一个月的工作后，[卡门]有了一个 [GPS 功能的立方体](http://karman.homelinux.net/blog/archives/746)，它告诉他当前的纬度和经度，当前的时间，路线，方向和速度。

[Karman]的 GPS cube 使用了一个便宜的 GPS 模块，Arduino Mini Pro，一个华丽的有机发光二极管显示屏，和一个从第一代 iPod nano 回收的 LiPo 电池。令人惊讶的是，建筑非常干净——没有电线、接头或随机的环氧树脂滴到处突出。整个建筑略大于一立方英寸，允许[卡门]在口袋里携带 GPS 设备的能量。

[Karman]的 GPS 立方体代码使用 Arduino 的 TinyGPS 库，它有几个很棒的函数，可以跟踪可见卫星的数量并报告当前时间。现在剩下要做的就是为这个令人敬畏的小项目编造一个案例。一如既往，休息后的视频演示。
