# 3 野外的伐木工

> 原文:[https://hackaday.com/2013/10/20/3doodler-in-the-wild/](https://hackaday.com/2013/10/20/3doodler-in-the-wild/)

还记得 3Doodler 的 Kickstarter 吗？他们刚刚开始发货，比利时一个叫做 Open Garage 的黑客空间刚刚收到他们的！就像任何优秀的黑客空间一样，在玩了之后，他们把它拆开并贴上了照片。

这很棒，因为在我们关于 3Doodler 的最初报道中，我们想知道它的内部是什么样的，以及它是否可以适用于 CNC 机器来制造一台巨大的 3D 打印机。车库实验室都送来了。

休息后留下来看看内部结构，以及使用数控铣床的第一次测试打印！

![13 - 1](../Images/c5b8c29a137eadcff1f5cbf84d4b78dc.png)

内部没有什么太令人惊讶的，但是正如[Anthony]在他们的博客上指出的，它有 3 个外部联系人，但尚未记录在案。经过进一步调查，一个是接地，另外两个控制挤压的快和慢设置，但有 3 秒的延迟。

对拆除感到满意后，他们做了下一件最合乎逻辑的事情——他们把它绑到他们常驻的 CNC 工厂，并尝试 3D 打印！他们目前只做了一层，但这是一个好的开始。
