# 为 Scratch 编程语言构建传感器

> 原文：<https://hackaday.com/2012/05/17/building-sensors-for-the-scratch-programming-language/>

[Kevin Osborn]让年轻程序员编写与物理世界互动的程序变得更加容易。照片中他拿着的设备是一个基于 [Arduino 的加速度计和距离传感器，用于 Scratch 语言](http://baldwisdom.com/itchyduino/)。

Scratch 是麻省理工学院开发的一种编程语言。它以孩子为中心，专注于图形构建模块。这使得向年轻人介绍编程概念变得更加容易，而没有学习语法时遇到的障碍和陷阱。

正如你在休息后的剪辑中看到的，[Kevin 的] Arduino 草图包括自动将加速度计和距离数据拉入 Scratch 环境的挂钩。我们认为，他的例子提供了启动和运行任何类型的传感器所需的一切，无论是磁力计还是 LDR(这两者都将成为一个很好的防盗报警类型的项目)。用您自己的硬件尝试一下，看看您能完成什么。
