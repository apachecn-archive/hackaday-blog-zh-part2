# 修复老式 PDP-11/04 计算机

> 原文：<https://hackaday.com/2015/04/08/restoring-a-vintage-pdp-1104-computer/>

[MattisLind]花了一年半的时间完成了对数字设备公司(DEC)PDP-11/04 的恢复，包括 TU60 磁带机和 LA30P Decwriter 打印终端等外围设备。计算机现在能够运行非常简单的操作系统 CAPS-11 和 CAPS-11/BASIC。就像项目本身一样，他的博客帖子很长，充满了有趣的细节。对于一个 tl 来说；版本博士，休息后查看视频。

该系统最初属于爱立信，[MattisLind]从爱立信电脑俱乐部 EDKX 获得。他很幸运能够访问在线资源，这使得任务变得更容易。但考虑到他必须解决的硬件故障数量和软件挑战，这仍然不容易。第一项任务显然是查看电源。他更换了大电解电容器，电源似乎在他的虚拟负载下工作得很好，但当连接到计算机背板时就失败了。一些更多的挖掘和更换晶闸管后，他把它修好了。晶闸管是短路电路的一部分，用于在一个主开关晶体管出现故障时保护系统免受过电压影响。

电源修好了，CPU 还是启动不了。一些侦探四处寻找，他找到了出故障的总线接收器芯片。他通过中国易趣卖家订购的设备在慢速船上，所以他只是从另一个电路板上拆下一个设备，这稍微改善了一些，但仍然卡在一个循环中。更换的通信板和系统现已通过诊断检查，但未通过内存测试。这被证明是由一个有故障的 DIP 开关引起的。接下来，他攻克了让 CPU 板加速的所有软件难题。

磁带机使用的是普通盒式录音带格式的变体。电机驱动联轴器似乎少了几个非公制螺钉，但原理图上写着“乐泰”，所以问题得到了解决。随着磁带的滚动，他发现设备无法检测到磁带的结尾/开头。这似乎是一个 DEC8881 驱动芯片坏了，所以他用一个 7439 驱动程序替换了它-dec 881 原来是完全相同的芯片-它下面有 7439 标记。他寻找零部件的经历让他走遍了世界。一个失败的 N8271 移位寄存器来自保加利亚，卡带通过易贝从德国，一个 7427 从一个瑞典同胞，另一个瑞典人帮助了一个橡胶轮的驱动轴，和一些更多的卡带从摩洛哥。

修复项目的最后一部分是 LA30 dec writer——一种早期的点阵式打印机。打印头需要一些工作。而且主键盘编码芯片，SMC 的 KR2376 坏了。它最初被一个 Arduino 放在 perf 板上取代，但后来被一个漂亮的 PCB 取代。他很幸运地找到了系统附带的打印机色带和打印纸的密封包。测试运行显示，一些打印头螺线管不能启动，这需要更换更多损坏的集成电路。这种特殊的打印机有一个并行接口，PDP-11 没有必要的 M7910 或 M7731/M7389 板来帮助他与打印机接口。幸运的是，他也设法买了下来，并且还能用，最终完成了这整个漫长的翻新。PDP-11/04 最终运行 CAPS-11 / BASIC，准备打印“HELLO WORLD”。
