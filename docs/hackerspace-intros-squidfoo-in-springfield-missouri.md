# 黑客空间简介:密苏里州斯普林菲尔德的 Squidfoo

我特别高兴地宣布 [SquidFoo，一个位于密苏里州斯普林菲尔德的艺术画廊/工作室和黑客空间](http://squidfoo.com/)([hackerspaces.org 链接](http://hackerspaces.org/wiki/SquidFoo_Art_Gallery_Studio_and_Hackerspace))。对于那些不知道的人，这个小镇就是我所在的地方(布拉德·皮特也是从这里来的！).我很想把这个黑客空间归功于自己，但我不能。[Scott Sauer]和[Phil Broussard]创建了它，当他们听说我在城里时就联系了我。我会帮他们组织起来，并可能策划一些活动，以此来弥补我来晚了。你会在这里看到更多的 SquidFoo，因为这可能是我将来做黑客项目的地方！

这个空间被分成 4 个物理区域。有艺术画廊、艺术工作室/休息室、黑客空间或“合作实验室”以及小店。现在，正如你在照片中看到的，黑客空间和商店区相当混乱。他们刚刚清理完一个相当大的地下室“辐射掩蔽所”来放所有这些东西，所以工作站在不久的将来应该会更加可用。现在，这里有绘画、缝纫、电子站、3d 打印机等全套设备。近期的一些计划包括添加一个 RFID 系统，自动化照明，一个业余无线电工作室，替代媒体工作室，还有一个即将完成的 RepRap Mendel。

如果你在这个地区，请找个时间过来看看。就像我说的，他们刚刚开始运作，所以成员很少。那几个成员踢了一些屁股。我遇到了[瑞安]，又名[[铂菌](https://www.facebook.com/Platinumfungi)，当时他正在为一部即将上映的视频制作一个银河战士头盔道具。他在这个短片中解释了它的制作过程。你可能以前听说过他，他做定制的经典游戏系统。[我知道我在](http://www.youtube.com/watch?v=7F2raBFDTtw)之前看过他的塞尔达主题 NES。
