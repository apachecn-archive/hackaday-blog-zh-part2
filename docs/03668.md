# Hackit:研究人员等了 69 年才看到焦油移动

> 原文:[https://hackaday . com/2013/07/25/hackit-researchers-wait-69-years-to-see-tar-move/](https://hackaday.com/2013/07/25/hackit-researchers-wait-69-years-to-see-tar-move/)

![69-year-science-experiement](../Images/ee942c19fa7ffc3cd4f87b128598315c.png)

这个实验早在 1944 年就在都柏林三一学院开始了。它的目的是证明焦油是流动的，事实上它确实每隔十年释放一滴。事情是，没有人见过这种情况发生，提出了“[如果一棵树倒在森林](http://en.wikipedia.org/wiki/If_a_tree_falls_in_a_forest)的场景。[关于这一事件的 *Nature* 文章](http://www.nature.com/news/world-s-slowest-moving-drop-caught-on-camera-at-last-1.13418)甚至提到了另一个实验，该实验的最后一滴被错过了，因为监控它的摄像机离线了。这一次，他们确实得到了一些(不)重大事件的镜头，你可以在下面看到。

因此，聪明的黑客面临的挑战是:你能想到的最简单的装置是什么，它不仅能连续拍摄实验，还能确保当一滴水滴落下时，你能把货物录下来？我们看到各种各样的高速快门触发器— [这里有一个最新的](http://hackaday.com/2013/05/31/high-speed-photography-with-friggin-lasers/)。但是我们不记得看到过同样的极其缓慢的版本。请留下评论，让我们知道你的想法。
