# Game Boy Pocket Raspi Mod 让所有其他人都相形见绌

> 原文:[https://hack aday . com/2014/06/05/the-game-boy-pocket-rapi-mod-puts-all-others-shame/](https://hackaday.com/2014/06/05/the-game-boy-pocket-rapi-mod-puts-all-others-to-shame/)

互联网上有成百上千个版本将树莓 Pi 放在一个带有按钮的外壳中，这些按钮只用于运行旧游戏的模拟器。这个不同于其他任何一个。是的，它基本上仍然是一个 RetroPi 模拟器，但这个 Game Boy Pocket casemod 超越了任何远程可比的构建。

Game Boy 的口袋小得令人难以置信，但在打磨了外壳内部的凸起，粘住电池门，并在墨盒插槽上安装了一点塑料后，[WarriorRocker]能够在里面安装一个 Raspi。按钮使用与股票游戏男孩相同的 PCB，连接到模拟 USB 键盘的 Teensy 2.0 板。

解决了两个最大的组件后，[战士]将注意力转向了声音、视频和电源。显示屏是一个 2.5 英寸的复合 LCD，实际上非常适合放在屏幕边框后面。音频由一个 3 美元的音频放大器、一个新的更小的扬声器和一个从原版 Game Boy guts 偷来的侧装锅负责。用最初 Game Boy Pocket 所用的 2 节 AAA 电池是不可能的，所以[Warrior]设法找到了 2600mAh 锂电池、升压调节器和充电电路的空间。

结果是一个全彩色的 RetroPi 构建能够在需要充电之前运行三个小时。所有经典的 Game Boy 游戏都与其他系统的精选游戏一起加载到 SD 卡中。结果是我们见过的最好的便携式 Raspi 构件之一。下面视频。
