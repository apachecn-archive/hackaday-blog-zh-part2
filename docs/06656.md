# 带有防晒提醒的帽子很容易制作

> 原文:[https://hack aday . com/2014/07/26/hats-with-sun block-reminders-is-easy-to-make/](https://hackaday.com/2014/07/26/hats-with-sunblock-reminders-are-easy-to-make/)

![light_sunscreen-reminder-hat-00](../Images/e31aa1bfdb8fa295261d5e1fc7ac3b76.png)

几乎任何人都可以制造这种[紫外线指数传感可穿戴设备](https://learn.adafruit.com/sunscreen-reminder-hat)，它可以检测来自太阳的热射线，并提醒用户涂抹防晒霜。没有焊接要求，这使得这是一个很好的初学者项目，适合那些不熟悉如何连接电子传感器的人。

所有需要的是一个 [FLORA 主板](http://www.adafruit.com/product/659)，一个紫外线指数传感器，一个压电蜂鸣器，一个 500mAh 的脂肪电池，2 股导电线，几个家用工具，和你最喜欢的夏天的帽子。

一旦收集了材料，剩下的过程就相对简单了。将菌群穿入到位并连接压电元件只需几分钟。然后添加紫外线传感器，让帽子开始收集数据。再后来一点点编码，整个系统就准备在太阳底下磨破了。

![light_sunscreen-reminder-hat-08](../Images/8ee2f6e8f245c3ff5092bc7420dba517.png)

这个项目的伟大之处在于，帽子可以被编程为在该涂更多防晒霜的时候播放一首歌。每个人，从海滩流浪者到沐浴阳光的美女，再到音乐节的参加者，都会发现这顶帽子很有用。而且，它既便宜又容易制造。

Adafruit 教程页面上的视频展示了安装系统是多么简单。
