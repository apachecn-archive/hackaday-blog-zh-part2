# 啜吸式 Ipod Dock 突出了辅助技术

> 原文:[https://hack aday . com/2012/04/26/sip-and-puff-ipod-dock-highlights-assistive-technology/](https://hackaday.com/2012/04/26/sip-and-puff-ipod-dock-highlights-assistive-technology/)

![](../Images/1a54d651007e7b0db7009e227e1632fe.png "sip-n-puff-ipod-dock")

看看这个，[一个吸一口的 Arduino 盾牌](http://www.tmientp.com/ipoddock.htm)。这是一种辅助技术，允许身体残疾的人使用塑料气管控制东西。吸气(Sip)或吹气(Puff)的不同组合区分了控制命令。

在这种情况下，该设备用于控制 iPod dock，但[鲍勃·约翰逊的] [Kickstarter 项目](http://www.kickstarter.com/projects/1864258471/sipn-puff-arduino-shield)寻求将 Sip-and-Puff 功能放在那里，以便无论需要什么，它都可以弥合控制差距。他在视频中提到的一个例子是莫尔斯电码键盘。

该屏蔽使用压力传感器接收来自塑料管的输入。但是我们也见过使用机械压力开关来完成[。这种技术就是在](http://hackaday.com/2011/03/04/arduino-sip-n-puff-switch/)[建造](http://hackaday.com/2010/01/15/a-day-at-the-lake-for-the-disabled/)小型皮艇所使用的。
