# 那些黑客如何完全控制了那辆吉普车

> 原文:[https://hack aday . com/2015/08/22/how-than-hacker-take-complete-control-of-the-jeep/](https://hackaday.com/2015/08/22/how-those-hacker-took-complete-control-of-that-jeep/)

这是一个多云的日子，气温在 70 度左右——这是开着你的全新吉普切诺基享受轻松驾驶的完美一天。几分钟后，你和你的搭档系好安全带，发现自己正驶入高速公路。当你的伴侣拨弄中央触摸屏时，你闻到了新车的味道。

“看看它有没有 XM 收音机，”你一边玩着前灯的控制一边问。

几秒钟后，一首泰勒·斯威夫特的歌曲开始播放。当窗户落下时，你们一起唱歌。“没有比这更好的生活了，”你想。不幸的是，这种乐趣是短暂的。它始于挡风玻璃刮水器的开启——干燥的玻璃上的橡胶发出可怕的尖锐声音。

“喂，你在干什么！”

“不是我干的……”

确认挡风玻璃刮水器开关处于关闭位置。你开关它几次，但没有效果。突然，收音机关闭了。一个骷髅头和扳手标志的图像出现在触摸屏上。里克·阿斯特利的《永远不会放弃你》开始从扬声器中响起，四扇门完美地同步锁上。空调风扇以最大设置打开，同时，你会感觉到座椅变得更热，因为它们也被设置为最大。发动机关闭，车辆换到空档。你踩下油门，但什么也没发生。你崭新的吉普车在高速公路边停下来，完全不受你的控制。

听起来像好莱坞电影里的情节吗？再想想。

Twitter 的安全工程师[查理·米勒]和 IOActive 的车辆安全研究主管[克里斯·瓦拉塞克]能够从他们的地下室通过无线车载娱乐系统*黑进一辆 2014 款吉普切诺基。一项名为 UConnect 的功能允许车辆通过蜂窝连接连接到互联网，它有一个你可能以前听说过的东西——IP 地址。一旦这两个黑客有了这个地址，他们就有了吉普车的“数字钥匙”。从那里，[查理]和[克里斯]开始修补各种固件，直到他们能够访问车辆的 CAN 总线。这使他们能够控制汽车的许多功能，包括(在正确的条件下)取消刹车和转动方向盘的能力。你可能已经听说过[事件，克莱斯勒为了应对这一漏洞发布了](http://media.chrysler.com/newsrelease.do;jsessionid=E6177B162AD3B811DD035C3BF03B5B97?&id=16849&mid=1)大规模召回。*

但是直到这个周末我们都不知道具体是怎么做到的。[查理]和[克里斯]在一份 90 页的白皮书(PDF)中记录了他们的利用，而[在拉斯维加斯的 DEF CON 演讲中做了详细的发言](https://www.youtube.com/watch?v=OobLb1McxnI)。那个视频是昨晚刚发布的，嵌在下面。看一看，你就会意识到他们为实现这一切做了多少工作。相当惊人。

如果你的车库里有一辆吉普切诺基，不用担心。这两名黑客一直与克莱斯勒密切合作，后者反过来[发布了更新](http://media.fcanorthamerica.com/newsrelease.do?&id=16827&mid=1)以防止这种黑客行为发生。只要确保你[更新你的固件](http://www.driveuconnect.com/software-update/)。
