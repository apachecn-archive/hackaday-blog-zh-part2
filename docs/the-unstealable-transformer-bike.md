# “无法密封”的变形金刚自行车

> 原文：<https://hackaday.com/2014/09/12/the-unstealable-transformer-bike/>

一组智利工程学生设计了一辆自行车，它带有可拆卸的部件，可以重新定位以锁定车辆。他们称之为“耶卡计划”,并将其作为世界上第一辆不可密封的自行车进行销售。

它的天才之处在于框架本身实际上充当了锁定机制。这意味着，如果一个小偷想打破锁，他们将不得不打破实际的自行车，留下一点希望。这也消除了外出购买独立的自行车锁的需要，无论如何可以相对容易地打开自行车锁。

Yerka 的工作原理是将自行车的下管分成两半，并将其向外延伸到附近的物体，如树、灯柱或指定的自行车架。鞍座和座杆然后被移除并插入到下管中钻出的孔中。在那之后，末端的锁是安全的，骑车人可以知道他们的自行车是安全的。

然而，聪明的黑客可能仍然会找到解锁这辆自行车的方法。不管有多难，总会有人想出办法的。与此同时，它给那些希望阻止普通自行车窃贼企图盗窃自行车的人一种很好的安全感。
