# THP 半决赛:医疗周期

> 原文：<https://hackaday.com/2014/09/01/thp-semifinalist-the-medicycle/>

尽管 Hackaday 奖似乎缺乏运输项目，但有几个项目通过了大筛选，进入了半决赛。[Nick]、[XenonJohn]和[DaveW]的项目[是医疗周期](http://hackaday.io/project/1156)。这是一辆肯定会回头率的车，但这些家伙心里有比在路上看起来更好的东西。他认为这种双轮独轮车将有助于派遣急救人员和其他急救人员，穿梭于车流中，快速到达需要他们的地方。

重要的事情先来。独轮摩托车实际上是可行的。和自平衡滑板车基本一样；骑手身体前倾前进，身体后倾刹车，两个轮胎帮助转向。它是全电动的，由一个 450 瓦的马达驱动。它可以在小巷、停车场、[甚至碎石路](http://hackaday.io/project/1156/log/9285-discovery-channel-daily-planet)中狂奔。

这个循环的第 *medi~* 部分来自一个藏在自行车鼻子下的移动分类单元。有用于测量血压和氧气、心率和心电图的传感器。这些数据通过藏在头盔中的单目显示器发送给医疗自行车骑手，并通过 3G 模块传递给场外的医生。

医疗自行车是否对医务人员有用还有待观察，但这些家伙已经创造了一种有趣的交通工具，至少像水上摩托一样酷。令人印象深刻，这辆自行车本身的总制造成本相当低。
