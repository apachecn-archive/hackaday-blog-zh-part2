# 将一台旧的 CRT 电脑显示器重新用作高压科学项目电源

> 原文：<https://hackaday.com/2014/06/14/repurpose-an-old-crt-computer-monitor-as-a-high-voltage-science-project-power-supply/>

终于有人发现了这些旧 CRT 电脑显示器的好用途，它们正被运往垃圾填埋场。来自 Rimstar.org 的 Steven Dufresne 带领我们完成了一个非常简单的将 CRT 电脑显示器转换成高压电源的过程。当然，你可以制造一些小火花，但这种转换对许多科学项目也很有用。[Steve]在他的视频中使用显示器电源来演示如何为离子船供电，这是一个使用高压的经典科学实验。

转换就像你想的那样简单。你需要安全地给电视显像管放电，从高压阳极电缆上切下杯子，并将其重新连接到显示器外的安装支架上。该系统需要接地，因此[史蒂夫]连接了几根接地电缆。一根用于项目的接地电缆，一根用于安全放电棒。它真的很简单，一旦连接到一个科学项目，只需打开监视器，你就有 25 千伏的电压供你使用。你不希望这种转换产生大量的大火花，因为它会损坏显示器内部的零件。添加的 24 万欧姆 2 瓦电阻器[Steve]将有助于将这些放电保持在最低水平，并保护显示器免受损坏。

是的，这是危险的，但当你在进行高压科学实验时，你必须正确处理危险。这不是获得高压的最安全的方式，但如果你不得不为一个项目拼凑一些东西，这将让你达到目的，[史蒂夫]非常谨慎，包括警告人们危险以及如何在每次使用后安全地给你的实验和电源放电。这不是[史蒂夫]建造的第一个高压电源；我们在过去展示了他自制的 30kV 电源，这是一种使用倍压或三倍压电路构建高压电源的更传统的方式。休息后加入我们观看视频。
