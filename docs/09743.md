# 用商店工具烹饪:最危险的早餐

> 原文:[https://hack aday . com/2015/08/04/用工具烹饪-最危险的早餐/](https://hackaday.com/2015/08/04/cooking-with-shop-tools-the-most-dangerous-breakfast/)

在一段颇为滑稽的视频中,[ExplosiveDischarge]的[Dom]和[Chris]向我们展示了如何在不使用厨房的情况下制作一份完整的英式早餐。

我们说的是鸡蛋、培根、火腿、土豆煎饼和烘豆。而不使用单个烹饪元件。其中一些方法是意料之中的，比如用熨斗作为便携式烤架——顺便说一下，这是一个很棒的生活帮，尤其是当你在酒店里，碰巧有一包熏肉而又无处烹饪的时候……或者用喷灯用火焰烧烤一根完美的香肠——巧妙使用电钻供电的烤肉店，用可变电源调节速度！

我们认为我们最喜欢的是用灯泡煮鸡蛋——是的，他们确实戴了安全眼镜。
