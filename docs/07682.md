# 半臭虫

> 原文：<https://hackaday.com/2014/11/28/the-halfbug/>

[亚历克斯]在 Tinkerlog 上发布了他的机器人 [Halfbot](http://tinkerlog.com/2014/11/25/halfbug/) 的建造细节。自从他的[同步萤火虫](http://www.instructables.com/id/Synchronizing-Fireflies/)指示性的方式回到过去，我们一直是他的工作的大粉丝。[Alex]的作品通常将一个非常规的想法与极简主义的设计和精确的执行结合在一起，Halfbot 也不例外。

你必须观看[视频](https://www.youtube.com/watch?v=bO703DJq65Q)(嵌入在休息下面)来充分欣赏它移动的方式。两个大前腿和小前脚垫交替使用，形成一个始终稳定的三脚架，脚轮在后面。它把自己抬起来，向前移动一点，然后再次靠在垫子上，同时腿为下一步做好准备。它不会赢得任何速度锦标赛，但它是一个非常好看的步态。

头部单元也有两个自由度，允许它用超声波测距仪单元扫描周围，并为整个事件添加更多的个性。

[Alex]提到他最近买了一台车床和一台数控铣床。所以毫不奇怪，他从零开始制造所有部件，只是为了给机器一个测试。我们认为他在整体美学方面做得很好，尤其是电池组。

我们很高兴看到[Alex]在开发固件时如何添加新的行为。没有压力！
