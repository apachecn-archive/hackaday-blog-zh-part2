# 然而，另一个 RepRap 主持人看起来很酷

> 原文:[https://hack aday . com/2012/07/25/yeth-another-reprap-host-looks-pretty-cool/](https://hackaday.com/2012/07/25/yet-another-reprap-host-looks-pretty-cool/)

[![](../Images/be0b66f3923014031b5a4df8abd5b998.png "ss")T2】](http://hackaday.com/wp-content/uploads/2012/07/ss.png)

加入其他 RepRap 主机软件包(如 ReplicatorG、RepSnapper 和 Skeinforge)的是另一个 RepRap 主机，这是由[Arkadiusz]开发的一个项目，它将许多简洁的功能结合到一个非常酷的软件包中。

我们真的要感谢[Arkadiusz]的一件事是[一个虚拟表](http://www.youtube.com/watch?v=oA21ulT2ZyU&feature=relmfu)，它允许你导入几个。STL 文件，将它们放在虚拟构建平台上，然后一次性打印出来。以前，我们知道如何做到这一点的唯一方法是创建一个单一的。STL 文件，所有需要的部分已经到位，或排列几个对象，以增加产量。虚拟表格功能允许任何人绕过这些步骤，一次打印出许多对象。

YARRH 还允许您[以 3D 方式查看 GCode。](http://www.youtube.com/watch?v=ayTQ40QOoqo)这个功能目前还有点不完整，但是【Arkadiusz】说它很实用，而且可以运行 3D 打印机。

现在，YARRH 只适用于 Windows，但是 Ubuntu 的软件包(希望 OS X 也可以)即将推出。休息之后，你可以看看 YARRH 的一些视频。
