# 防御战诡计:黑掉黑客帽

> 原文：<https://hackaday.com/2014/08/01/defcon-shenanigans-hack-the-hackaday-hat/>

[https://www.youtube.com/embed/jmHCgFVNx0Q?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/jmHCgFVNx0Q?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

我们不想称之为挑战，因为我们担心 DEFCON 上的常客会把我们的硬件变成一堆冒烟的渣，但我们计划给我们带来一点乐趣。我将戴上这款优雅的头盔，并邀请您[黑进支持 WiFi 的黑客帽](https://www.youtube.com/watch?v=jmHCgFVNx0Q)。

8 月 7 日至 10 日在拉斯维加斯举行的 DEFCON 22 上，整个周末我都会戴着多种颜色的帽子。你可能还会发现[Brian Benchoff]有时会炫耀这款配件。不管怎样，上来打个招呼。我们想看你要展示的任何硬件，我们会给你一些赠品。

不要让它就此结束。拿出你最喜欢的笔测试发行版，黑进帽子的接入点。从那里，路由器将提供更多关于如何侵入其中一个 shell 帐户的信息。拥有一个帐户，您可以在记分牌上留下您的别名，也可以将您自己的自定义消息推送到帽子的 32×7 RGB LED 字幕上。

你可以在这个项目页面上了解更多关于帽子硬件的信息。但是像往常一样，我在很紧的期限内完成了这个项目，并且还在尝试填充项目的所有细节。
