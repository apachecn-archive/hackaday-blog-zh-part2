# Android 控制的显示器支架使用弹珠作为滚珠轴承

> 原文:[https://hack aday . com/2012/07/06/Android-controlled-monitor-stand-uses-marbles-as-ball-bearing/](https://hackaday.com/2012/07/06/android-controlled-monitor-stand-uses-marbles-as-ball-bearings/)

![](../Images/3439e8a7235526ac7d317e848cbdf515.png "stand")

[mobile_earth_explorer]发来了一份他整理的说明书，记录了他建造的一个机器人控制的旋转显示器支架。

支架本身是由 18 毫米厚的木板在车床上加工而成的三个圆盘。转动这些圆盘后，[mobile_earth_explorer]跳到他的台式磨床上，做了一个半圆形工具来为滚珠轴承划出轨道。

一旦磁盘可以在彼此之间自由旋转，剩下的唯一事情就是为伺服系统、Arduino、一对罐子和电线留出空间。pots 控制显示器支架的最大速度，并在您的手机穿过房间时提供手动旋转控制。

[mobile_earth_explorer]编写了 Android 应用程序，这样一个 Arduino 就可以控制多达五个旋转显示器；是的，对于家庭或办公室使用来说，这可能有点过了，但我们确信它对于某种形式的演示非常有用。
