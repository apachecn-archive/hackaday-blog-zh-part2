# 开源的 3D 打印火箭引擎

> 原文：<https://hackaday.com/2015/03/10/open-source-3d-printed-rocket-engines/>

液体燃料火箭发动机几乎是任何人能造出的最难的东西。有热力学、机械加工、电子学、材料科学和软件方面的考虑，只是为了有一些不会在测试台上爆炸的东西。建造液体引擎的数据也不容易找到:许多有用的信息被归类或锁在[埃隆]的一个文件柜里。

新泽西州的 Fubar 实验室正在努力改变这种状况。[他正在开发开源的 3D 打印液体燃料火箭发动机](http://wiki.fubarlabs.org/FubarWiki/Small-Liquid-Fueled-Rocket-Engines.ashx)。现在，它不会飞，但这不是重点:开发成功火箭的第一步是开发成功的发动机，而[格雷厄姆]正在努力使这成为现实。

这个引擎由气态氧和乙醇提供动力，是为 3D 打印而设计的。这实际上是对技术的一个很好的利用。SpaceX 和 NASA 已经使用 DMLS 打印机生产了 3D 打印的发动机部件，但[Graham]正在使用便宜得多的(Shapeways 也有)金属 SLS 打印机来生产他的发动机。用传统方法制造火箭发动机非常困难，这使得 3D 打印成为建造火箭发动机的完美过程。

到目前为止，[格雷厄姆]已经打印出了发动机、喷射器和点火器，都是为了把氧气和乙醇推进燃烧室，点燃它，惊叹马赫锥。你可以在下面看到一个视频，但是在 [GitHub](https://github.com/gNSortino/OSREngines) 、 [Fubar Labs wiki](http://wiki.fubarlabs.org/FubarWiki/Small-Liquid-Fueled-Rocket-Engines.ashx) 和[上也有一些令人难以置信的资源，这里有一堆图片和测试结果。](http://wiki.fubarlabs.org/FubarWiki/3D-Printed-Engine.ashx)
