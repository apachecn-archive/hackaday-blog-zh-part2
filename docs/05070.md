# 这一切都是为了(完美的)饼干

> 原文：<https://hackaday.com/2014/01/15/ben-krasnow-did-it-all-for-the-perfect-cookie/>

[本·克拉斯诺]在执行一项任务。他在寻找完美的巧克力饼干。为了帮助他完成这项崇高的事业，他发明了[饼干完美机](https://www.youtube.com/watch?v=8YEdHjGMeho)。从[用等离子](http://hackaday.com/2013/12/27/cleaning-slides-with-plasma/)清洁，到 [DIY CT 扫描仪](http://hackaday.com/2013/01/09/ben-krasnow-builds-a-ct-scanner/)，到[红宝石激光器](http://hackaday.com/2014/01/10/lazing-with-a-ruby/)，到 [LED 隐形眼镜](http://hackaday.com/2011/12/20/ben-krasnow-sticks-leds-in-his-contacts-just-for-kicks/)，【本】一定是当今最多产、最全能的黑客之一。在一天的辛苦工作后，还有什么比享受一杯牛奶和一块完美的巧克力饼干更好的放松方式呢？

这实际上是我们在 2012 年第一次看到的机器的更新。[本]已经把他的机器装满了原料，一切都在电脑控制之下。现在，机器将按照计算机指定的配料量分配配料，配料量由秤测量。每件事都是一次一块饼干。唯一不足的是机器还没有搅拌机。[Ben]必须为每个数据点混合一块饼干的生面团。他的实验得出了一些令人惊讶的结果。面粉太少实际上会导致饼干更脆，因为湿面团会展开成更薄的一层。[本]还发现，添加额外的红糖也不会使饼干更耐嚼。尽管他仍处于早期实验阶段，但[Ben]提到，因为很难做出一个糟糕的巧克力饼干，所以即使他失败了，味道也很好。
