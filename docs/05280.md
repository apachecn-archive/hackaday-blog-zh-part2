# retro tech tacular:Linux/UNIX TTY 从何而来

![retrotechtacular-tty-demystified](../Images/7a57cbe17ba017a8aed67d87a8f07a0e.png)

我们时常意识到，对我们有意义的话对后代可能没有意义。我想到的两个例子是“挂断电话”或者在汽车里你可以“摇下车窗”。今天的复古技术也是如此。Linux 用户肯定知道 TTY，但是如果你查找这个术语，你实际上得到的是“电传打字机”的参考。这是怎么回事？

莱纳斯·阿克森就这个主题写了一篇精彩的文章，名为《揭开 TTY 神秘面纱的 T2 T3》。我们经常以旧视频作为本专栏的主题，但我们认为你会同意[Linus 的]文章在电影中是值得的(如果可能的话)。Linux 中的 TTY 系统是计算机刚出现时的倒退，因为它是实时交互的。它们被连接到当时的打字机——一种被称为电传打字机的变种上，基本上是通过一根电线把你的击键发送到终端控制的计算机上。

这条通向处理器的铜管道基本上仍然是今天终端仿真器的工作方式。除了显示器和键盘，它们不需要更多的硬件。我们认为自己是相当高级的 Linux 用户，但是 noob 和专家一样会发现一些有价值的东西，这些东西肯定会让你茅塞顿开。
