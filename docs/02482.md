# 一辆带 IP 摄像头的无线遥控汽车

> 原文：<https://hackaday.com/2013/01/08/a-wifi-controlled-rc-car-with-an-ip-camera/>

通过 WiFi 控制你的车很好，但在车上安装一个网络摄像头，这样你就可以实际看到你要去哪里，这更好。[Michael]讲述了他如何制造他的无线汽车,并在帖子中提供了一些很棒的视频。

使用的汽车是一个看似标准的钢筋混凝土单元，它带有一个速度控制器，可回收用于网络。[Michael]去掉了标准无线电，但有了这个控制器，他就不必设计 H 桥电路了。收音机随后被 Sparkfun 的 [WiFi 模块取代。](https://www.sparkfun.com/products/11048 "Sparkfun WiFi module")

IP 摄像机一开始就有一些问题，因为滞后是无法忍受的。在经历了一些足以称得上是一次出色的黑客攻击后，这款相机最终能够在可接受的水平上运行，并向他用来控制一切的 [FLTK](http://www.fltk.org/ "FLTK") 应用程序输出数据。看看他在下面的一个视频中这辆车的运行情况。
