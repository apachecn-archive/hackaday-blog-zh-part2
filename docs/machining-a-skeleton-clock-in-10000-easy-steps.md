# 用 10，000 个简单的步骤加工一个骨架钟

> 原文：<https://hackaday.com/2015/04/03/machining-a-skeleton-clock-in-10000-easy-steps/>

又是一天，又是一个有趣的 YouTube 频道。[Chris]'[click spring channel](https://www.youtube.com/channel/UCworsKCR-Sx6R6-BnIjS2MA/featured)博客是你现在很少看到的东西:用车床、铣床加工零件，没有 CNC。[克里斯]正在做的项目是一个基于[约翰·威尔丁]设计的时钟。它非常大，所有的部件都是由黄铜和钢原料制成的。

当然，制作一个时钟不仅仅是在车床上切出一些零件，然后在铣床上加工。不，你需要制造零件来制造那些零件。【克里斯】已经为他的车床做好了[一个尾座模座](http://clickspring.weebly.com/tailstock-die-holder-for-the-sherline-lathe.html)，[一个在杆上钻孔的夹紧工具](http://clickspring.weebly.com/finger-plate-clamping-tool.html)，[一个漂亮的车床支架](http://clickspring.weebly.com/small-antique-style-lathe-carrier.html)用来固定小零件。

所有这一切都是一流的工作，用定制的锡研磨工具对零件进行镜面抛光，比达到绝对完美所需的努力要多得多。这个时钟项目非常棒，尽管还有几个月它才能达到第一秒。
