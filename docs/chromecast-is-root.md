# Chromecast 是根

> 原文：<https://hackaday.com/2014/08/25/chromecast-is-root/>

Image from [psouza4] on the xda-developers forum

Chromecast 就像一台完美的设备，只要把它插在电视机后面，你就可以在家里最大的显示器上立即看到网飞、Hulu、Pandora 和一个网络浏览器。这是一个比运行 XBMC 的 Raspi 简单得多的设备，我们已经看到一些 Chromecast 黑客从手机和 [rickroll 你周围的每个人](http://hackaday.com/2014/07/28/hijacking-cromecast-with-the-rickmote-controller/)传输视频。

[现在 Chromecast 已经扎根](http://forum.xda-developers.com/hardware-hacking/chromecast/root-hubcap-chromecast-root-release-t2855893)，允许任何人更改 DNS 设置(网飞和 Hulu 用户希望观看他们国家没有的内容),并为 Chromecast 加载定制应用。

对于 Hackaday 的普通读者来说，Chromecast 的根过程应该相当简单。它需要一个 Teensy 2 或 2++开发板、一根 USB OTG 线和一个 u 盘。将 Teensy 插入 Chromecast 并等待一分钟。取出盒子，插上 u 盘，再等几分钟。成功就是你，你的 Chromecast 现在已经扎根。
