# 用现代业余硬件制造的复古手持设备

> 原文:[https://hackaday.com/2014/04/04/118883/](https://hackaday.com/2014/04/04/118883/)

![magpi-gaming-handheld](../Images/cf5a0d761b598471777637bc6dafce4a.png)

还记得那些在 90 年代风靡玩具店的极其恐怖的掌上液晶游戏吗？你知道，那些有定制的液晶显示屏来制作假动画的。这里有一个[的例子，说明那些*应该*已经](http://www.instructables.com/id/Magpi-The-Micro-Arduino-Gaming-Platform-Interface/)了。这是一款基于 LCD 的手持设备，有一些灵魂。

整个东西大约有一个电视遥控器那么大，3D 打印的外壳使它非常像样。但是看看隐藏在里面的线路就能证明这是独一无二的。与现在相比，Arduino Pro Mini 可能是过去最大的技术差异。它有足够的空间来放置下面剪辑中展示的所有不同的设置和游戏。不过用户界面本身绝对是一种倒退。诺基亚 3310 屏幕拥有 84×48 像素的超大单色区域。有四个按钮作为 d-pad，两个作为操作按钮。也许最大的特点(除了我们已经提到的印刷外壳)是能够通过 USB 为内部电池充电。

[Zippy314]和他的儿子一起建造了这个。更有趣的是:学习游戏编程，还是掌握它们并发现你一路错过的 bug？
