# Arduino 圣诞树盾牌

> 原文:[https://hackaday.com/2013/12/18/arduino-xmas-tree-shield/](https://hackaday.com/2013/12/18/arduino-xmas-tree-shield/)

![christmasshield](../Images/728042b6c898000ab1835a25805f98cf.png)

没有什么比 LED 圣诞树更能提醒我们这是一个假期了。这显然不是同类中的第一个，但[Jose]确实提供了一种使用可寻址 RGB LEDs 的新技术。

[Jose]决定使用 20 个 WS2812B RGB LEDs，如果您以前没有见过，这是带有集成控制器的 RGB led。是的，没错，只需要电源/地和 1 条数据线就可以控制数百个 RGB LEDs。这种 LED 树的设计很简单:一个定制蚀刻的 PCB 将其切割成圣诞树的形状。WS2812B LEDs 有助于保持一切清洁，因此圣诞树没有任何“丑陋”的装饰，除了这里和那里需要的旁路帽。额外的奖励是，树上的发光二极管通过压电蜂鸣器与 Arduino 产生的音乐同步。为什么是盾？嗯，整个树可以很好地插入标准的 Arduino 接口。这看起来像是一个完美的入门项目，让你熟悉可寻址 led，或者至少让你在[建立自己的 infinity 门户之前热身。](http://hackaday.com/2013/11/08/freesides-infinity-portal/)
