# 费雪牌电唱机播放《通往天堂的阶梯》

> 原文:[https://hack aday . com/2012/05/24/fisher-price-record-player-plays-stairway-to-heaven/](https://hackaday.com/2012/05/24/fisher-price-record-player-plays-stairway-to-heaven/)

![](../Images/3d4d197ea17e3b68e8d478abb4b84eff.png "zeppelin")

[弗雷德·墨菲]有一个旧的费雪牌音乐盒/唱机，在过去的 40 年里丢失了许多光盘。这是一个非常简单的装置——塑料圆盘上的同心凹槽有塑料突起，可以被唱机“唱筒”的尖齿拾取。看到这个玩具基本上是一个音乐盒，[弗雷德]认为自己制作唱片将是他力所能及的；他做了一件合理的事情，为一个玩具音乐盒制作了一张《通往天堂的阶梯》光盘。

为了找出在哪里放置音乐齿的“凸起”，[Fred]在 Visual C# 2010 中构建了一个小工具，允许他在音阶上放置音符并为磁盘生成必要的 GCode。将这个文件发送到他的 CNC 工厂后，[Fred]有了一个在儿童音乐盒上播放齐柏林飞船的丙烯酸盘。

当然，如果没有这个电唱机播放的楼梯爆破的视频演示，这个指令是不完整的。休息之后你可以去看看。
