# 轻松丝网印刷所有的东西

> 原文:[https://hack aday . com/2014/03/15/easy-silk screen-all-the-things/](https://hackaday.com/2014/03/15/easily-silkscreen-all-the-things/)

丝网印刷并不像看起来那么难。例如，它比雕版印刷更容易，因为你不必反转图像。[Jimmy DiResta]展示了将丝印设备组合在一起并在木头、金属、当然还有 t 恤上印刷是多么容易[。](http://makezine.com/video/diresta-simple-silkscreening/)

一旦你决定了一个设计，就把它打印在一张醋酸纤维纸上，这张纸可以用普通的家用打印机打印出来。你可以购买现成的网状框架，甚至整个套件，但[吉米]告诉你如何建立一个简单的框架和订书针筛网。密封边缘后，混合一些感光乳剂，覆盖网格，让它在黑暗的房间里干燥。

干了以后，把醋酸盐放在屏幕上，用任何可用的光线曝光。[吉米]建立了一个牛奶箱塔高达他的荧光工作灯，并暴露了大约四分钟。现在你差不多准备好留下你的印记了。剥去醋酸盐，用喷射瓶和压缩空气去除多余的乳液。用吹风机把整个东西吹干就大功告成了。用丝网印刷油墨装满橡胶滚轴，用均匀的压力从上往下画，你就得到一个丝网印刷的东西。
