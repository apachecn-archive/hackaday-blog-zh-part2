# 有态度的咖啡机

> 原文:[https://hack aday . com/2013/09/02/a-有态度的咖啡机/](https://hackaday.com/2013/09/02/a-coffee-machine-with-an-attitude/)

![coffee](../Images/2adb6d730c64284edf40a171c6980cf9.png)

海牙的传播博物馆联系了阿明和他在 BerkelaarMRT 的团队，看他们能否制造一台仅通过声音操作的咖啡机。这听起来像是一个有趣的挑战，所以[他们接下了这项工作](http://www.berkelaarmrt.com/index.php/projecten/23-pratende-koffiemachine)并设计出了一个非常棒的互动咖啡机。

互动咖啡机基于 Animo 公司生产的咖啡机，可以制作卡布奇诺、浓缩咖啡或纯咖啡。电子设备基于摩托罗拉 Xoom 10”平板电脑，带有来自 Sparkfun 的[电动绵羊](https://www.sparkfun.com/products/retired/10745)模块。这与带有光耦合器的定制 PCB 一起，允许咖啡机上的按钮被电子激活。

当任何人走近咖啡机时，Maxbotix 声纳模块会检测到他们的存在，一个小的动画咖啡杯会询问他们想喝什么。可爱的小杯子准备好饮料，等待下一个人。
