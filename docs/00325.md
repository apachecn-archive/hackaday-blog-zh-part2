# 使用 Kinect 教授正确的手部动作

> 原文：<https://hackaday.com/2012/05/05/teaching-correct-hand-movements-with-a-kinect/>

![](img/a0edc3c6d24538dd3cf6555e6b7ac3db.png "hand")

如果你正在学习如何演奏一种乐器，投篮，或者甚至学习让一个高尔夫球飞起来到底有多难，这里有一个适合你。这是一个[手运动引导](http://www.youtube.com/watch?v=vNaw9GpuVLQ)的投影可视化。

UIUC 和微软研究院的[Rajinder]、[Hrvoje]和[Andrew]想出了一个很好的方法来提高运动和乐器所需的精细动作技能。基本上，他们拿着 Kinect，在 3D 空间中跟踪用户的手。像箭头和彩色像素这样的小视觉提示被投射到手上，以提供视觉反馈。这些人就他们的工作写了一篇很棒的论文，从结果来看，受试者的手部动作在经过一点训练后有了很大的提高。

虽然这种技术的用途很明显是用于演奏大提琴或重量训练，但没有提到将这个项目用于物理治疗。我们当然可以看到这个项目用于康复的好处。
