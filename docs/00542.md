# Retrotechtacular 功能遥控彩色电视

> 原文：<https://hackaday.com/2012/05/25/retrotechtacular-7-function-remote-control-color-television/>

啊，那时候电视是一件坚固的家具。说到感动，不能说怀念。但是看着这个带 7 功能遥控器储物空间的[木质电视柜](http://www.youtube.com/watch?v=NF2Xp3P6-m0),我们不禁会想，如今的电视在客厅布局方面似乎更加……琐碎……

这款车型的宣传视频是不同时代的瑰宝。你可能已经听说了，本周[【尤金·波利】去世](http://gizmodo.com/5912204/eugene-polley-inventor-of-the-first-wireless-tv-remote-control-has-died)。他被认为发明了第一个无线电视遥控器。休息之后的视频显示，这一点都没有进步。取而代之的是，它在阴极射线管屏幕的每个角落使用可见光和四个光传感器。你可以打开和关闭电视，切换频道，但有时在合适的光线条件下，电视会自动改变。我们想知道你是否曾经回家发现电视神秘地自己打开了？
