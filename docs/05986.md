# Retrotechtacular: ROTOPARK 是一个 40 年前的未来派停车建筑

> 原文:[https://hackaday . com/2014/05/06/retro tech tacular-roto park-is-a-future istic-parking-structure-from-40 年前/](https://hackaday.com/2014/05/06/retrotechtacular-rotopark-is-a-futuristic-parking-structure-from-40-years-ago/)

![retrotechtacular-rotopark](../Images/06dcef109ca1efa0d741be4a526abead.png)

上图是一个已经建成并投入使用的自动化地下停车场的功能模型，但显然它从未广泛流行。这让我们有点难过，因为它消除了每次使用车库时寻找空停车位的需要；让一个机器人帮你停车似乎很有前途。

ROTOPARK 系统的要点是[一个用于停放汽车的转盘和电梯系统](https://www.youtube.com/watch?v=jh1MjyYSPRo)。只要把车开进一层的一个单间车库，拿着你的票，走出人孔。车库隔间地板是一个雪橇，它沿着电梯向下移动(在图像的左半部分显示为蓝色隔间),被储存在汽车的旋转传送带中。

显然，机械故障在这里是一个大问题。电梯坏了怎么办？此外，在交通繁忙的时候，我们认为将您的车辆从系统中取回会比我们习惯的“静态”停车场慢得多。哦，好吧，也许有一天。休息之后，请观看经典的营销视频，展示该系统的概念、构造和使用。

工作版本实际上是在日内瓦机场建造的。我们在 YouTube 评论中发现了一些额外信息，但找不到这些信息的来源，因此请认真对待:
