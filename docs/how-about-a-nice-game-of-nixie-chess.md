# 来一盘精彩的谢妮象棋怎么样？

> 原文：<https://hackaday.com/2012/06/27/how-about-a-nice-game-of-nixie-chess/>

[托尼]送来了他一直在做的一套数码管国际象棋，我们对这一制作的质量非常满意。棋子在没有任何可见电线的情况下发光，棋盘非常优雅，带有镀金和黄铜的触感，并且使用(主要)与旧俄罗斯谢妮电子管同时代的材料和组件进行了非常好的设计。

[Tony]没有选择数字 Nixies，而是选择了 IN-7 和 IN-7A 显像管，这些显像管原本是用来显示 A、V 和~等科学符号的。为了给这些电子管供电，[托尼]在棋盘上的每个方格下使用了 64 个空心变压器，让这些谢妮电子管就像感应充电器一样被供电。

尽管他的博客帖子在细节上有点单薄，但我们不得不称赞[托尼]的惊人构建。他说[将会有一个包括一个巨大印刷电路板的工具包](http://www.lasermad.com/?p=266)，但是我们不会冒险猜测它的价格。
