# 胶带风笛

> 原文:[https://hackaday.com/2013/01/29/duct-tape-bagpipes/](https://hackaday.com/2013/01/29/duct-tape-bagpipes/)

[![Duct Tape Bagpipes](../Images/b8ecf91a316ec0d1b78d46070c8ab499.png)T2】](http://hackaday.com/?attachment_id=93659)

想用管道和胶带制作自己的乐器吗？[斯科特]用 PVC 管、塑料袋和胶带制作了自己的一套薄膜风笛。

风笛是由几部分组成的。无人机是被调成演奏单个音符的管道。它们由固定长度的管子来调音。chanter 是一个有手指孔的管子。这让您可以根据您覆盖的孔弹奏各种音符。吹管用于向袋子中充入空气，空气将穿过无人机和 chanter 上的薄膜。最后，还有储存空气的袋子。

[Scott]的建造使用 PVC 制作无人机和唱机。这些薄膜是由塑料袋碎片制成的。一些狡猾的胶带制作了袋子，并将其密封在各个部分。一个止回阀用来阻止温暖的胶带味的空气吹回你的嘴里。

人们用几卷胶带能做的事情真是太神奇了。这些管子并不完全合拍，但它们肯定能工作。休息之后看看他们的行动视频。
