# SPARC:一个声控机器人在 DTMF 中甜美地歌唱

> 原文:[https://hack aday . com/2015/05/07/SPARC-a-voice-controlled-robot/](https://hackaday.com/2015/05/07/sparc-a-voice-controlled-robot/)

科幻小说中反复出现的主题之一是机器人。从《星球大战》中的 C-3PO 和 R2D2，到《瓦力》和《夏娃》,机器人总是有一定的吸引力。像 Arduino 这样便宜的微控制器已经为更多的人打开了机器人世界。[约翰芬]已经做到了这一点。通过连接两个 Arduinos 作为大脑，他创造了一个语音控制的机器人，他称之为 [S.P.A.R.C .(哨兵/项目助理/机器人伴侣)](https://sites.google.com/site/techntires/home)。

事情开始于他收到一个机器人手臂作为圣诞礼物，并对此感到失望。他没有简单地制造一个更好的手臂，而是“忘乎所以”地制造了一个完整的机器人。整个项目耗时三个月，其中大部分时间他都在学习编程。

SPARC 有三个声纳传感器，用于检测障碍物和运动，一只手臂和几只可互换的手用于握住物体，还有一个 EasyVR Arduino 盾用于语音控制。机器人的“眼睛”是一个 LED“KITT”扫描仪和一个 6884 VU 仪表芯片，当机器人说话时，“眼睛”会闪烁。它携带一部机载智能手机来查看天气，播放手机 SD 卡中的音乐，以及 GPS 功能。

SPARC 可以响应一系列命令和游戏，包括“跟我来”和“唱歌”【JohnFin】还增加了“音序器”功能，可以记录和回放一系列命令。休息之后可以找到这个功能的视频。
