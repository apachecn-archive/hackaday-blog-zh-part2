# Hackaday 奖参赛作品:可穿戴机器人工具包

> 原文：<https://hackaday.com/2015/07/27/hackaday-prize-entry-wearable-robotics-toolkit/>

互联网上充斥着假肢项目，在很大程度上这是可以理解的。假肢设备最终是为每个用户定制的，3D 打印机试图找到一个目的。把两个和两个放在一起，你会得到几个塑料四肢。

高级假肢所需的电子设备比 3D 扫描仪和打印机要难一些。如果你正在设计一条机器人腿，你将需要通过一个致动器泵出几百瓦的能量来推动一个人前进。在过去的几年里，[Jean-Fran ois Duval]一直在麻省理工学院媒体实验室生物机电小组研究这个问题，并提出了他的 Hackaday 奖参赛作品。这是一个用于可穿戴机器人的电机和电机控制系统,解决了其他项目还没有想到的问题。

FlexSEA 的目标不是制造假肢和可穿戴机器人，而是制造驱动这些可穿戴设备的电子设备。这意味着要做从驱动电机、调节功耗、运行控制环路到与传感器通信的所有事情。为了实现这一点，[Jean-Fran ois]使用了 BeagleBone Black、Cypress PSoC 和 STM32F4，这些都是非常强大的硬件。

到目前为止，[Jean-Fran ois]已经记录了当前控制器的硬件和软件，并有一些硬件运行的演示视频。你可以看看下面。
