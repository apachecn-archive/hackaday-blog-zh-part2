# Plex Hack 无需越狱即可运行苹果电视

> 原文：<https://hackaday.com/2013/06/05/plex-hack-runs-apple-tv-without-a-jailbreak/>

苹果公司一直对他们的第三代苹果电视硬件保持严格的控制。它不仅经受住了每一次越狱的尝试，而且他们不允许第三方应用程序像其他 iOS 硬件一样通过 app store。第二代硬件是 XBMC 的流行平台。它需要越狱来加载家庭媒体软件，这就是为什么它不能在最新的盒子上完成。但是这里有一个选择。这个黑客使用 DNS 重定向[使苹果电视 3 成为一个复杂的前端](https://github.com/iBaa/PlexConnect)。

苹果电视没有安装新软件，而是使用一台独立的电脑来拦截网络流量。您将 Apple TV DNS 对准运行脚本的电脑，然后使用原生电影预告片应用程序，该应用程序将填充来自 Plex 后端的电影和电视节目。所有其他功能都将保留，形成一个近乎完美的解决方案。关于系统的更多信息[请查看这个论坛](http://forums.plexapp.com/index.php/forum/136-appletv-plexconnect/)。
