# Hackaday 大奖:20，000 个气象站

> 原文:[https://hack aday . com/2015/08/04/hack aday-prize-entry-20000-weather-stations/](https://hackaday.com/2015/08/04/hackaday-prize-entry-20000-weather-stations/)

Tahmo 团队计划在撒哈拉以南非洲建立一个由 20，000 个气象站组成的网络。这是一个令人印象深刻的目标，他们已经在塞内加尔、乍得、尼日利亚、乌干达和南非建立了试点站。对于他们的 Hackaday 奖参赛作品，他们认为给他们的气象站添加更先进的传感器是有意义的，并提出了 GPS，闪电和大规模土壤湿度传感器。

已经部署的传感器具有通常的气象设备——温度计、风速计、气压计和雨量计。这些站与学校的互联网连接，学生可以监控当地的天气模式并上传数据。Tahmo 团队正在使用 AS3935 Franklin 闪电传感器和 GPS 模块为他们的奖品条目构建一个小型附加板。

为了加快设计周期，该团队使用现成的模块作为闪电探测器和 GPS 模块。他们获得了 hack aday Prize[Collabratorium](https://hackaday.io/event/7030-hackaday-prize-collaboratorium-6)提供的一些 PCB 设计建议，多亏了一些乐于助人的黑客，他们几乎把一切都搞定了。

对于有史以来最雄心勃勃的众包数据收集项目之一来说，这是一个伟大的项目，将极大地改善整个非洲大陆的天气预测。即使他们的参赛作品只是监测雷击，这仍然是一个令人钦佩的目标，也是今年 Hackaday 奖最有用的项目之一。
