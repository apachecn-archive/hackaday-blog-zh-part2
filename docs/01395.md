# 将诺基亚 N82 TFT 面板与 Arduino 配合使用

> 原文:[https://hackaday . com/2012/08/21/use-a-Nokia-n82-tft-panel-with-your-arduino/](https://hackaday.com/2012/08/21/use-a-nokia-n82-tft-panel-with-your-arduino/)

[Andy]一直在努力对诺基亚 N82 的 2.4 英寸手机显示屏[进行逆向工程，以便与 Arduino](http://andybrown.me.uk/wk/2012/08/19/nokia-n82-arduino/ "Nokia N82 with Arduino")配合使用。正如文章中所指出的，这种同样的 2.4 英寸显示器至少可以在其他七种诺基亚产品中找到，因此它们很容易获得。在易趣上，这种电池板的价格低至 3 英镑(或略低于 5 美元)。

结果相当好，可以在休息后的视频中看到。第一个演示显示模拟的天气报告，第二个显示一些 JPEG 图像。虽然本演示中使用了 Arduino Mega，但也可以使用标准 Arduino。文章中包含了原理图和材料清单，但是如果你宁愿只买一块板，他会按照先到先得的原则出售他所制作的其余部分。但没有消息说他有多少库存！
