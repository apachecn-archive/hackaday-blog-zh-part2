# AquaTop:一款游戏触摸显示屏，看起来像是被恶魔附身的水

> 原文：<https://hackaday.com/2013/08/18/aquatop-is-a-touch-display-that-projects-images-on-water/>

你准备好把一个公用水槽大小的水池作为你下一个客厅游戏机的位置了吗？这个演示很吸引人，但是可能还没有准备好被广泛采用。 [AquaTop 是一个交互式显示器](https://googledrive.com/host/0B_E3lRLf4DoZdm84VVozV09Qcms/index.html)，它结合了水、投影仪和深度相机。

水中加入了浴盐，使其变成乳白色。这有双重作用，使其成为投影仪的合理反射表面，并在表面以下隐藏您的手。下面的视频展示了几种不同的游戏。但最引人注目的演示涉及到当你的手指露出水面时的个人手指追踪(见上图右侧)。

还有一个新颖的反馈系统。研究人员黑掉了一些扬声器，这样它们就可以浸没在水箱中，以同样的方式添加了一个带 led 的大扬声器。当输入一个 50 赫兹的信号时，它们会让游泳池的水面跳舞。
