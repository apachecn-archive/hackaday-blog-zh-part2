# 经典大众 Bug 立体声获得蓝牙

> 原文:[https://hack aday . com/2012/05/25/classic-VW-bug-stereo-gets-bluetooth/](https://hackaday.com/2012/05/25/classic-vw-bug-stereo-gets-bluetooth/)

![](../Images/10414217699bbd6de7335efa0c066171.png "bluetooth cassette")

如果你们中的任何人与老汽车狂热者打交道，你就会知道原来的 dash 音响是一件令人垂涎的设备。如果这些年来它们没有被移除并被更现代的东西所取代，它们通常是没有功能的。[Hadrien]费了一番周折才在他的 69 年大众甲壳虫里保留了他原来的音响。首先，他做了一个我们在现代汽车上也能看到的普通黑客。他增加了一个辅助输入。

这暂时满足了他对现代音频的渴望，但他真的发现电缆附件的必要性令人讨厌。他决定他要[添加蓝牙](http://dahzz.skynetblogs.be/archive/2011/09/23/bt2bt-aka-bluetooth-to-beetle.html) ( [谷歌翻译自法语](http://translate.google.com/translate?sl=auto&tl=en&js=n&prev=_t&hl=en&ie=UTF-8&layout=2&eotf=1&u=http%3A%2F%2Fdahzz.skynetblogs.be%2Farchive%2F2011%2F09%2F23%2Fbt2bt-aka-bluetooth-to-beetle.html))。为此，他拿了一副蓝牙耳机，把它们拆开。他用一盒旧磁带作为外壳，把它们改造成可以直接连接到他的仪表盘立体声音响上的东西。他甚至保留了麦克风，以防万一他想把它用作免提设备。

他的汽车音响是 1969 年制造的，不用盒式磁带，所以他就把它放在手套箱里。这并不是最佳选择，但正如他所说，至少他不用插上手机。
