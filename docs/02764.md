# 环形绕组作弊

> 原文:[https://hackaday.com/2013/02/13/toroid-winding-cheat/](https://hackaday.com/2013/02/13/toroid-winding-cheat/)

当你需要一个圆环时，最简单的方法就是自己上弦。问题是，在一个铁氧体磁环上缠绕几百个线圈是一件非常麻烦的事情，尤其是当你不得不制作几个线圈的时候。[Jim W.]开发的这个秘籍会节省很多时间。他将戒指切成两半用于缠绕，然后重新组装。

在这里，你可以看到他有一半的核心安装在钻夹头。为了达到这一点，他先在铁氧体上刻痕，然后用老虎钳夹住一半，用木块和锤子敲打挤出的一半。他还没有找到刻划材料的完美解决方案(美工刀或三角锉都可以，但都有缺点)。如果你有什么好主意，请留下评论。

一旦核心是在两个部分，他用一些铜管的一端变平，弯曲成环段的形状。当它热粘到位后，他带着它转一圈(在休息后的剪辑中显示)。一旦缠绕完成，一点强力胶会重新组合两半。这种事情对于监控电力使用非常有用。
