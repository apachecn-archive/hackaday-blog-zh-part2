# 用五金店的零件制造内燃机

> 原文：<https://hackaday.com/2013/07/26/building-an-internal-combustion-engine-from-hardware-store-parts/>

~~【MacGyver】~~【卢·沃兹尼亚克】正在执行一项任务，只用五金店的零件为[制造一台内燃机。你在上面看到的是他的第三次尝试。根据你的五金店，这可能会冒险超出他们的销售，因为[楼]切换到使用汽油。但是前两次尝试是由丙烷火炬燃料罐驱动的](http://howtolou.com/engine)[。](http://hackaday.com/2012/12/14/almost-building-an-engine-from-hardware-store-parts/)

不幸的是，它仍然没有运行。但是下面的演示让我们觉得他真的很接近了。时机总是很敏感，这似乎是导致问题的原因。他使用了许多卫生设备。在右边，你可以看到零件(包括一个花生酱罐)使他的汽化器有一个阀门直接指向作为阻气门。燃油和空气混合物通过管道向下移动到气缸和气门总成，在此被黑色格栅点火器模块点燃。他的定制切割胶合板齿轮随着飞轮移动。它通过用一根电线拉动叶片开关来触发他的临时火花塞。

我们觉得他就快成功了。如果你对他可能出错的地方有任何建议，[Lou]欢迎你的意见。
