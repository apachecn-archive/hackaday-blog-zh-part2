# 3D 声学操纵:使用声波的看似不真实的悬浮

> 原文:[https://hack aday . com/2014/01/03/3d-声学-操纵-看似真实-虚幻-悬浮-使用-声波/](https://hackaday.com/2014/01/03/3d-acoustic-manipulation-seeminly-unreal-levitation-using-soundwaves/)

哇哦。[Yoichi Ochiai]、[Takayuki Hoshi]和[Jun Rekimoto]是来自东京大学和名古屋工业大学的研究人员，他们刚刚学会了如何御气。

使用一系列驻波超声波，可以在声压节点悬浮小颗粒。超声波束的声轴平行于重力，这也允许通过改变声音的相位或频率来沿着固定轴操纵物体。通过增加垂直于第一超声波束的第二超声波束，可以定位压力节点或焦点，并使小物体悬浮在 2D 平面周围。

在他们的演示中，他们让泡沫颗粒、电阻器、LED 漂浮起来，他们用一块干冰展示波浪，甚至设法让一个小螺丝钉漂浮起来。
