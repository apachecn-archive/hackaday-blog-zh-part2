# 赤壁-米库万，或者一个带福特聚变电池的动力轮

> 原文：<https://hackaday.com/2014/09/30/the-chibi-mikuvan-or-a-power-wheels-with-a-ford-fusion-battery/>

在所有大型制造商博览会上，动力赛车系列都会亮相，将旧的动力轮变成赛车，以每小时几十英里的速度在赛道上疾驰。[查尔斯]在比赛中有些名气——在官方规则中甚至有一个条款[是以他的名字命名的——所以当然他在比赛日带来的任何东西都将是惊人的。](http://www.powerracingseries.org/rules)[那是](http://www.etotheipiplusone.net/?page_id=3434)。它使用了来自福特 Fusion plugin hybrid 的电池组、定制车身和来自废弃 Mac G5 的水冷装置。

几个月前，我们看到[查尔斯]拆开了他花 300 美元买的电池组。这种设备会在你知道你犯了错误之前杀死你，但[查尔斯]能够把包拆开，制作几个电池组——28.8 伏和 16 安培——足以让他在赛道上跑几圈。

Chibi-Mikuvan 的底盘由钢制成，车身由机械加工的粉红色泡沫制成，采用玻璃纤维，并使用从[Burt Rutan]的书《无模复合夹层飞机构造》中收集的一些技巧【Charles】完成[。马达？那是](http://www.etotheipiplusone.net/?p=3250)[一个巨大的无刷电机](http://www.etotheipiplusone.net/?p=3128)用于 1/5 比例的遥控船。传动装置来自角磨机，[电子设备是一件艺术品](http://www.etotheipiplusone.net/?p=3321)。

结果呢？一辆近乎完美的电动轮赛车，整备重量为 110 磅，最高时速为 25 英里。它也处理得很好:在下面的视频中，它在纽约 Maker Faire 的动力轮比赛中超越了整个 hacky racers 领域，之后仍然有足够的果汁在 Faire 周围撕裂。
