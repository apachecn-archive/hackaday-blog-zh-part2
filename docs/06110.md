# 波普斯金——一支机电单人乐队

> 原文:[https://hack aday . com/2014/05/21/popmaskinen-an-electro mechanical-one-man-band/](https://hackaday.com/2014/05/21/popmaskinen-an-electromechanical-one-man-band/)

![one man band](../Images/9a3d22ce4d5c6b6bd354965ba72f0b6e.png)

2014 年斯德哥尔摩迷你创客节刚刚结束，[Johnny Eriksson]凭借其令人印象深刻的[机电单人乐队获得了年度创客奖。](http://jenslabs.com/2014/05/19/stockholm-mini-maker-faire-2014/)

作为一名音乐家/电工/家具木工，[Johnny]拥有多项技能，他想尝试将它们整合到一个项目中。他称之为 Popmaskinen(流行机器)。

Popmaskinen 使用 MIDI 键盘、按钮和旋钮，将数字输出转换为由各种机电组件控制的物理乐器。我们最喜欢的部分之一是吉他，它使用螺线管来弹奏，甚至更多的螺线管来挤压吉他上的各种弦。

也许这种建筑的另一个最有趣的部分是“taktverket”或步伐看守。这是一个带有可调突起的大金属圆筒，实际上触发了单个乐器——有点像音乐盒的工作原理。通过使用一些可调开关，他可以打开和关闭单个乐器，这些乐器将根据 taktverket 的模式播放音乐。
