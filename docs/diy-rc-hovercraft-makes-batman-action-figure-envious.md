# DIY 遥控气垫船让蝙蝠侠动作人偶羡慕不已

> 原文：<https://hackaday.com/2015/01/28/diy-rc-hovercraft-makes-batman-action-figure-envious/>

[Bauwser]有一些备用的遥控直升机零件，拼凑了一个遥控气垫船。它起作用了，但是不合他的意。不过没关系，他知道这只是未来的一个原型；一艘完全由零开始建造的气垫船，它的零件是按照鲍泽的要求特制的。

他首先勾画出一些很酷的多面体形状，看起来既好看又容易建造。然后将轻质但坚硬的泡沫板切割成合适的形状，粘在一起，制成一个三维实体。然后在泡沫上覆盖一层玻璃纤维和树脂来增加强度。车身上开了一个孔，用于安装一个 55 毫米的管道风扇，该风扇提供所需的空气，以填充裙部并提升车辆。另一个管道风扇安装在飞行器的后部，指向后方。这个涵道风扇提供了向前的推力和一个伺服矢量风扇，以使转弯。

[鲍瑟]自己缝的裙子。它是由一个旧的海滩帐篷制成的。这种布料非常轻而且柔韧，非常适合做气垫船。在试运行过程中，灰尘和碎屑被卡在裙管中。快速回到缝纫机添加一些纱布网解决了这个问题，并保持碎片收集到最低限度。最后，[Bauwser]展示了一个伟大的 DIY RC build 通过一点规划和实验可以是什么样子。
