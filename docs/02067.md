# Arduino MP3 点唱机

> 原文:[https://hackaday.com/2012/11/07/arduino-mp3-jukebox/](https://hackaday.com/2012/11/07/arduino-mp3-jukebox/)

![](../Images/cf6aef651c21956f2f1c40b7f5baa32c.png "arduino-mp3-jukebox")

这里有一个便宜的基于 Arduino 的 MP3 点唱机 ( [翻译](http://translate.google.com/translate?hl=en&sl=es&tl=en&u=http%3A%2F%2Farduino-guay.blogspot.com.es%2F2012%2F10%2Freproductor-mp3-con-arduino-i.html))由【何塞·丹尼尔·埃雷拉】组装而成。

他花了一些时间来确保它和他的其他音频设备一起放在架子上看起来很棒。这从一个木箱开始，它是一些重复使用的包装。我们不熟悉“输液”产品；也许是茶或烟草？无论如何，他添加了一个定制的面板来容纳字符 LCD，旋转编码器，两个按钮，并作为两个扬声器的格栅。

扬声器及其附带的放大器电路来自便携式扬声器设备。他将它们与 VS1002d MP3 解码器模块、SD 卡分线板和 Arduino 本身结合在一起。除了上面链接的概述帖子，还有一组组装照片和一篇讨论[如何为控制系统](http://arduino-guay.blogspot.com.es/2012/10/reproductor-mp3-con-arduino-ii.html) ( [翻译](http://translate.google.com/translate?hl=en&sl=es&tl=en&u=http%3A%2F%2Farduino-guay.blogspot.com.es%2F2012%2F10%2Freproductor-mp3-con-arduino-ii.html))安排代码的帖子。休息后，观看和聆听片段中该单元的动作。
