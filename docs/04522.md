# 一晚无预算数控机床

> 原文：<https://hackaday.com/2013/11/10/one-night-no-budget-cnc-machine/>

波兰的黑客空间 Kraków 每周五都会举办一场名为 [NightHack](http://www.meetup.com/HackerspaceKrakow/) 的活动。这个想法很简单。现在是周五深夜，所有的商店都关门了——有东西需要被黑掉。

就在刚刚过去的周五晚上，他们决定试着只用他们现有的设备来制造一台数控机床。天哪，该死的，他们成功了吗！这个建筑使用了 Arduino Mega，破损的 Playstation 3 驱动器，一些备用的 L293D ICs，一些热熔胶和木材。由此产生的数控机床是一个了不起的例子，说明了只要一群合适的人在一起工作，一个晚上就能完成什么。

它的功能可能不足以进行铣削，但作为一个小型的 CNC 绘图绘图仪工作得很好，具有巨大的 5x5cm 工作区，分辨率为 0.16mm。下周他们希望对它进行修改，以允许 PCB 钻孔，这在正确的进给速度下可能是可行的！

我们认为这个聚会的概念很棒——每个黑客空间都堆积了大量的旧垃圾，也许永远不会被使用。这是一个迫使项目完成和旧组件用完的好方法。如果你计划开始安排类似的聚会，请在评论中告诉我们——我们喜欢分享这类项目！
