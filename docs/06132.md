# 发现支持 Wifi 的 10MHz 示波器

> 原文：<https://hackaday.com/2014/05/23/discovering-a-wifi-enabled-10mhz-oscilloscope/>

[https://www.youtube.com/embed/qEqWtKGJhFQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/qEqWtKGJhFQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

正如我们的大多数读者所知，[迈克]上周末背着一个巨大的快乐扳手参观了湾区创客集市。在他的旅行中[，他遇到了上图](https://www.youtube.com/watch?v=qEqWtKGJhFQ)视频中显示的由比利时 Velleman 公司制造的简洁示波器。尽管它只有 10MS/s [的采样率](http://en.wikipedia.org/wiki/Sampling_rate)和 10MHz 的带宽，但我们猜测它可能对一些爱好者仍然有用，因为它可以通过 Wifi 接口与任何 PC/智能手机/平板电脑通信。

黑匣子内是一个 3.7V 1800mAh 锂离子电池，带有一个 USB 端口，可以给它充电或更新示波器的固件。正如视频中所见，平板电脑的触摸屏可以实现更自然的用户界面交互。用于导出采集的样本的协议是开放的，允许用户创建自己的分析程序。示波器使用 8 位模数转换器和 4K 采样缓冲器。
