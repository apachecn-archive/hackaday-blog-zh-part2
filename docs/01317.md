# 图坎普:MC 霍金机器人轮椅

> 原文:[https://hack aday . com/2012/08/14/toor camp-MC-hawking-robotic-轮椅/](https://hackaday.com/2012/08/14/toorcamp-mc-hawking-robotic-wheelchair/)

[![](../Images/6e096b2fd0de4c6e1df6ae5df6587780.png "7773318890_0ca8eabcc6_b")T2】](http://hackaday.com/?attachment_id=82499)

这是旧金山[噪音桥](https://www.noisebridge.net "Noisebridge")黑客空间制造的 [MC Hawking](https://www.noisebridge.net/wiki/Noise-Bot "MC Hawking") 机器人。这是一个机器人电动轮椅，配有一台电脑、一台 XBox Kinect 和一台 Arduino。在软件方面，它使用了 Ubuntu 和开源的 [ROS](http://www.ros.org "ROS") 平台。Noisebridge 的几个人正在 Toorcamp 对机器人进行黑客攻击，以添加机械臂和其他升级。

该项目的一个目标是建立一个硬件平台，让软件黑客可以在自主应用程序上工作，而不必钻研硬件的复杂性。由于自主轮椅可能会变得危险，它显然吹嘘自己不遵守阿西莫夫的三大定律。

MC Hawking 的自主应用的一个例子是面部跟踪。它使用 Kinect 的传感器跟随周围的人。该平台现在正被用于开发 DORA 开源机器人助手项目，该项目希望在 51 天后使用机械臂从冰箱中取出一瓶苏打水。

Noisebridge 的[Jake]指出，他们正在寻找对项目的软件方面感兴趣的人。如果你在湾区，还没有去过 Noisebridge，你需要去。查看他们的网站，了解该组织的大量信息。
