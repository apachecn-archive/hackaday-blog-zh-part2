# Defcon 主持人预览黑客，使普锐斯脱离司机的控制

> 原文:[https://hack aday . com/2013/07/26/defcon-presenters-preview-hack-that-take-Prius-out-of-drivers-control/](https://hackaday.com/2013/07/26/defcon-presenters-preview-hack-that-takes-prius-out-of-drivers-control/)

![hacking-control-of-a-prius](../Images/fffe8f05737759737c6cbed2cbc86624.png)

这是一个 CAN 总线黑客的宝库，会吓得不知情的司机屁滚尿流——或者更糟。[查理·米勒]和[克里斯·瓦拉塞克]准备在今年的 Defcon 上展示他们的发现，这些发现是由 DARPA 资助的。为了炫耀，他们让一名《福布斯》记者在驾驶座上转了一圈。

你一定要看看这个的视频。自从滑铁卢实验室决定在老爷车上冲浪以来，我们还没有在潜在致命的汽车黑客攻击方面获得如此多的乐趣。这些黑客一开始只是看似无害的数据调整，比如谎报你的燃油油位，或者在汽车静止不动时在速度计上显示 199 英里/小时。但当他们将速度读数从 199 立即降至 0 时，事情开始变得有趣起来，这实际上是告诉汽车你发生了车祸(别担心，安全气囊不会启动)。其他邪恶的伎俩包括通过发出命令告诉汽车在路上行驶时自动停车来将方向盘拉到一边。最糟糕的是在车辆行驶过程中不能刹车。哦，踏板还在动，但是刹车钳没有反应。

这项工作的目的是强调汽车制造商需要加强安全的领域。这无疑给了我们一个在下一部邦德电影中会看到什么的想法。
