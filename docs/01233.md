# 超级忍者咖啡。

> 原文:[https://hackaday.com/2012/08/03/super-nincoffee-jr/](https://hackaday.com/2012/08/03/super-nincoffee-jr/)

![](../Images/c0236b6a8d89a4c996f6ed94310d23e9.png "super-nincoffee-junior")

你早上的例行公事没有包括足够的老派游戏。在超级 Nincoffee Jr 的帮助下，打破咖啡因的习惯，同时让你的马里奥继续工作。

[Luigifreakout123]在休息后分享视频剪辑中的构建细节。他首先透露这是他制作的第二个版本。第一个不是 Jr，而是使用了全尺寸的咖啡先生单元。既不煮咖啡，而是作为游戏硬件的外壳。开/关开关和原装电源线用于控制控制台的供电。在顶部和前面切割了开口，用于游戏盒和两个控制器端口。一根复合视频和立体声音频电缆从机器背面电源线旁边伸出。

是的，这非常简单，但是有时候一个像这样令人愉快的项目所需要的就是。
