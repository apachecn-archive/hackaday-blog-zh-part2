# 俄罗斯方块与树莓派决斗

> 原文:[https://hack aday . com/2014/06/29/Tetris-duel-with-the-raspberry-pi/](https://hackaday.com/2014/06/29/tetris-duel-with-the-raspberry-pi/)

![Tetris Duel](../Images/e759453381ebf965ac93019132927dbd.png)

用多个树莓玩家构建一个多人网络游戏可能非常困难。在大会上做这件事简直是疯了！这正是伦敦帝国学院一群一年级学生所做的；他们为 Raspberry Pi 开发了一款基于网络的多人俄罗斯方块游戏。

[Han]、[Piotr]、[Michal]和[Utsav]从裸机组装中创建了整个游戏，而它只包含 4000 行代码！代码有很好的文档记录，所以一定要查看他们的 [Github 库](https://github.com/Tetris-Duel-Team/Tetris-Duel)。这个项目对于那些希望学习裸机组装和网络的人来说是一个很好的参考。他们甚至选择使用老 NES 控制器，一个非常好的触摸。虽然我们有[精选](http://hackaday.com/2014/03/18/handheld-tetris-is-retro-and-we-love-it/)什么[似乎](http://hackaday.com/2013/06/25/led-tie-plays-tetris/)像[百万](http://hackaday.com/2012/04/23/mit-students-take-tetris-to-a-grand-scale/)不同[俄罗斯方块](http://hackaday.com/2012/10/29/pumpkin-tetris-inspired-by-our-own-led-jack-o-lantern/)游戏在[过去](http://hackaday.com/2014/02/26/giant-tetris-adds-some-retro-to-your-room/)，这是第一个多人游戏版本。休息后，在视频中观看俄罗斯方块决斗！

这是对你们所有学生的呼喊。花点时间为你的课程项目创建高质量的文档，并上传到网上。这不仅是一个很好的简历提升，而且很可能在 Hackaday 结束！
