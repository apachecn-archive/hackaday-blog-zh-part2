# 黑客空间:像素艺术竞赛

> 原文:[https://hack aday . com/2014/05/01/hack aday-space-pixel-art-contest/](https://hackaday.com/2014/05/01/hackaday-space-pixel-art-contest/)

在最后的传输过程中——我将在明天发布——我们决定在《我的世界》服务器上开辟一个创意区，供人们建造他们想要的任何东西，作为像素艺术竞赛的一部分。今天，我们宣布艺术挑战的获胜者，并分配他们的分数，以便我们可以得出最终传输的总获胜者。每位获胜者的分数都会增加额外的分数。这些都是由 Hackaday 校友 Caleb Kraft 来评判的，因为他到目前为止还没有参与任何恶作剧，而且被认为是不偏不倚的，他自己也有一套成熟的艺术印章。所以，现在开始…

#### 第三名:入侵 Tardis

[![2014-04-30_09.57.12](../Images/3f79c750b7a41753e30e67af491de855.png)](http://hackaday.com/wp-content/uploads/2014/04/2014-04-30_09-57-12.png) [![2014-04-30_09.57.55](../Images/ce03e2bd557df5ecc6ffa4c5b59a7467.png)](http://hackaday.com/wp-content/uploads/2014/04/2014-04-30_09-57-55.png) [![2014-04-30_09.58.50](../Images/e9f8b8a9c32e8a8e754729f44228f575.png)](http://hackaday.com/wp-content/uploads/2014/04/2014-04-30_09-58-50.png)

第三名由[Marcus1297]的神秘博士获得，题目是“黑掉一个 Tardis”。这是 tardis 的一个很好的再现，虽然它只有二维，但它有很好的细节，从顶部出来的信标光束是一个很好的点睛之笔。干得好！[Marcus1297]的分数增加了 2.5k 分。

#### 第二名:尼古拉·特斯拉纪念馆

[![2014-04-30_10.00.27](../Images/d954e20603e247cb2b622e3df961988b.png)](http://hackaday.com/wp-content/uploads/2014/04/2014-04-30_10-00-27.png) [![2014-04-30_10.00.41](../Images/92c97e56d7655d85276f2183258559f1.png)](http://hackaday.com/wp-content/uploads/2014/04/2014-04-30_10-00-41.png) [![2014-04-30_10.33.04](../Images/364251059119b8a26d0d8ac352c67245.png)](http://hackaday.com/wp-content/uploads/2014/04/2014-04-30_10-33-04.png) [![2014-04-30_09.59.38](../Images/30515bb9fc5c5d99edbd4bd86080cec2.png)](http://hackaday.com/wp-content/uploads/2014/04/2014-04-30_09-59-38.png) [![2014-04-30_09.59.56](../Images/846c402fc889358d751caf1be2ce13ca.png)](http://hackaday.com/wp-content/uploads/2014/04/2014-04-30_09-59-56.png) [![2014-04-30_10.00.11](../Images/e8f10047cb64c6231e52e244e6613e34.png)](http://hackaday.com/wp-content/uploads/2014/04/2014-04-30_10-00-11.png)

第二名是[st3al2012]，他将令人惊叹的特斯拉线圈献给了[尼古拉·特斯拉](https://www.google.com/url?sa=t&rct=j&q=&esrc=s&source=web&cd=1&cad=rja&uact=8&ved=0CCgQFjAA&url=http%3A%2F%2Fen.wikipedia.org%2Fwiki%2FNikola_Tesla&ei=X1FhU4nINJaIuASDyIK4Bg&usg=AFQjCNEbw7bSyCfvzPXUiSS_AVDoFsA9Ug)。之所以选择这个，是因为“这件艺术品是独一无二的”。这里有很多细节，他不仅用环形圈完成了线圈的主要结构，还展示了核心部件。火花隙发生器、电容器甚至交流电源插座。有很多细节，晚上看起来很惊艳。干得好，[st3al2012]你的记分牌多了 5 千分。

#### 第一名:传送门立方体

[![2014-04-30_09.55.46](../Images/a8fb4593720cb530ed26529c3fbf7823.png)](http://hackaday.com/wp-content/uploads/2014/04/2014-04-30_09-55-46.png) [![2014-04-30_09.55.57](../Images/2e9c7b7d47966d562b2e2516be14c586.png)](http://hackaday.com/wp-content/uploads/2014/04/2014-04-30_09-55-57.png) [![2014-04-30_09.56.12](../Images/f9de389cc32ad2b4bbc7e27e43f727fc.png)](http://hackaday.com/wp-content/uploads/2014/04/2014-04-30_09-56-12.png)

第一名由[XDjackieXD]获得，他的传送门立方体令人惊叹。[凯勒]宣布这是赢家说，“艺术和执行是模范”。我们不得不承认，这个立方体看起来很棒，但最棒的是，他费尽周折用放置在立方体内的音符块创作了“[Want you away](https://www.youtube.com/watch?v=dVVZaZ8yO6o)”(游戏的片尾曲)。干得漂亮，他完全值得为此获得 10k 分。

祝贺获奖者，感谢所有做出贡献的人。《我的世界》的服务器还在运行，所以如果你想亲自看看所有的艺术作品，请登录“minecraft.hackaday.com”查看。我们已经在这里建立了[世界和所有用于构建它的插件](http://hackaday.s3.amazonaws.com/HackadaySpaceMinecraft.tar.gz)。等我有机会完成的时候，我会发布我为传送点开发的 MatrixMiner 插件的源代码。
