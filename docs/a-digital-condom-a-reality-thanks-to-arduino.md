# Arduino 让数字避孕套成为现实

> 原文：<https://hackaday.com/2014/02/19/a-digital-condom-a-reality-thanks-to-arduino/>

[比尔·盖茨]基金会目前正在为任何能够改进避孕套的人提供大量的奖励。这是一个值得称赞的目标，也有些困难；男用避孕套没有被经常使用的一个主要原因是，“从男性的角度来看……与没有避孕套相比，避孕套降低了快感。”

虽然大多数受[盖茨]基金会启发的工作是研究避孕套几何形状的变化，但佐治亚理工学院的[Firaz Peer]和[Andrew Quitmeyer]设法用 Arduino 解决了这个问题。

电鳗的基本概念——是的，就是这个名字——是传递短电脉冲，“沿着轴的下侧增加刺激”。这些脉冲是响应不同的传感器输入而传递的——在[视频示例](http://www.youtube.com/watch?v=6oxDsEVqnyY)(令人惊讶的安全工作)中，他们使用缠绕在胸部的力电阻器对每次呼吸进行电刺激。

虽然这只是一个原型，但人们希望避孕套中的导体最终能够在制造过程中沿着避孕套的内表面植入。
