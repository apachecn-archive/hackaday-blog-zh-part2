# 史上最便携的 GameCube

> 原文:[https://hack aday . com/2012/05/22/the-most-portable-game cube-ever/](https://hackaday.com/2012/05/22/the-most-portable-gamecube-ever/)

![](../Images/b0664b11c72d81bbc6670d4d7cf127e9.png "GC")

这是我们见过的最小的 GameCube，直接来自 ModRetro 论坛上的[lyberty5]的工作台。尽管我们在之前已经看到过[的无盘游戏立方体，但是【lyberty5】将这个项目整合得如此之好，它在任天堂的产品阵容中不会显得格格不入。](http://hackaday.com/2011/07/29/sleek-disc-less-gamecube-handheld/)

与我们见过的大多数其他便携式游戏立方体不同，[lyberty5]的版本没有磁盘驱动器。游戏在 Wiikey Fusion 的帮助下从 SD 卡上下载，Wii key Fusion 是一种小型 FPGA 设备，用 SD 卡取代了 GameCubes 和 Wii 中的 CD 驱动器。

外壳由真空成型塑料制成，采用了一直流行的“dremeling and bondoing a controller for accurate button placement”方法。外壳内部是经过改装的 GameCube，这是一个 3.5 英寸的屏幕，能够以 640×480 的分辨率显示 NTSC 视频，电池电量足够一次充电玩两三个小时。

休息之后，你可以看看[lyberty5]快节奏的演示视频，它真正为便携式控制台演示设置了标杆。
