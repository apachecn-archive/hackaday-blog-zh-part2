# Hackaday 奖改变了意大利法律

> 原文:[https://hack aday . com/2015/04/27/Italian-law-changed-by-the-hack aday-prize/](https://hackaday.com/2015/04/27/italian-law-changed-by-the-hackaday-prize/)

最近意大利法律的一个变化是由 Hackaday 奖引起的。旧法限制非意大利公司在该国举办比赛。更新后，现在欢迎意大利公民参加 2015 年[hack aday 奖](http://hackaday.io/prize)的角逐，奖金为 50 万美元。

我们很少听到对黑客日奖的抱怨。当我们这样做时，几乎总是因为有一些国家被排除在参与之外。我们已经非常努力地尽可能多地包括全球，一些国家只是必须排除在外，因为当地的法律关于比赛。来自意大利制造的人们看到了去年 196418 美元的太空之旅，T2 开始着手改变当地的法律。幸运的是他们成功了！

意大利制造基金会成立的目的是鼓励和支持意大利的私人实验室。在看到两个主要的面向黑客和创客的竞赛——2014 年黑客日奖和英特尔 Make it Wearable 竞赛——拒绝意大利公民参加后。他们双管齐下，找到了法律顾问，并在 Change.org 发起了[请愿活动](https://www.change.org/p/ministro-guidi-maipi%C3%B9-not4italians)，约有 18000 名支持者签名。

我们一直在推迟发布，因为我们需要自己对这一变化的法律意见(没有一些帮助，我们不太擅长理解[意大利法律 pdf 文件](http://www.mise.gov.it/images/stories/normativa/applicazione%20art.6%20comma1.pdf))。但是今天，我们已经把意大利从被排除在外的国家名单中删除了。今天就提交你的参赛作品，只要写下你的想法，它将解决许多人面临的问题。制造一些有意义的东西，你就有可能赢得一次太空之旅，10 万美元的“最佳产品”，或者数百种其他奖品。但我们不会等到最后，在接下来的 17 周内，我们将向数百名参赛选手发放[5 万美元的奖金](http://hackaday.com/2015/04/24/50k-in-play-thirty-projects-will-win-custom-pcbs-this-week/)。
