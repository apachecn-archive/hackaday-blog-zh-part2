# 投币天气

> 原文:[https://hack aday . com/2015/07/08/CNC-machined-mini-arcade-game-houses-weather-station/](https://hackaday.com/2015/07/08/cnc-machined-mini-arcade-game-houses-weather-station/)

当你拥有自己的数控机床时，你倾向于先制造东西，然后再问问题。这听起来像是[Rui Cabral]如何将一个气象站塞进一个微型街机游戏柜中。

这个橱柜只有 16 厘米高，非常精致，类似于我们熟悉的街机外形，已经占据了无数个硬币。它甚至看起来像大男孩一样由碎料板制成。屏幕切口由 84×48 单色诺基亚显示器填充，机柜内部的其余部分填充有数控铣削的 PCB、温度和湿度传感器、RTC 和用于将数据上传到手机的蓝牙模块。[Rui]甚至设法用一个乒乓闪屏向所有街机游戏的曾祖父致敬。

这个项目的另一个好看的展示可能是这个制作成气象站的电子名片。我们还展示了[甚至更小的街机橱柜](https://hackaday.com/2013/07/11/the-tiniest-arcade-cabinets-youve-ever-seen/)，尽管可能没有那么实用。

[Rui]休息之后带我们参观他的建筑。
