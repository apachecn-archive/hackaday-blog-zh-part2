# 极其精确的任务控制面板

> 原文：<https://hackaday.com/2014/02/27/ridiculously-accurate-mission-control-panel/>

杰夫·海史密斯(Jeff Highsmith)为他的儿子制作的这个[令人惊叹的任务控制面板](http://makezine.com/video/making-fun-mission-control-desk/)的细节水平让我们震惊不已。

他的孩子刚开始上学，需要一张桌子做作业。他们最近参观了肯尼迪航天中心，他的儿子对太空的所有事物产生了新的兴趣——所以【杰夫】趁机把桌子做成这个令人难以置信的控制面板。

我们最近看到了一个[类似的项目](http://hackaday.com/2014/02/17/dad-built-rocket-control-module/)，但这个项目似乎更上一层楼。桌子本身由中密度纤维板手工制成，结构构件由橡木板构成。他裁剪了十几个单独的控制面板，添加了开关、led 和电位计，并将标签印在幻灯片上，使整个事情看起来非常专业和完美。中间放着一台 iPad，播放着精选的太空视频。

据我们所知，所有的按钮实际上都有作用。他使用带有四个 I/O 扩展器(MCP23017)的 Arduino 来处理所有的输入和输出，并使用 Raspberry Pi 来为模拟添加声音效果。

现在他所需要的是一艘飞船来监控——这显然也在计划之中！
