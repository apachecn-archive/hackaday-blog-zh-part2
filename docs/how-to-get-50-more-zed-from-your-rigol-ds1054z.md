# 如何从您的 Rigol DS1054Z 获得 50 倍的 Zed

> 原文：<https://hackaday.com/2014/11/12/how-to-get-50-more-zed-from-your-rigol-ds1054z/>

[Chris]最近花了很多时间在妻子的缝纫室，昨晚事情变得相当严重，因为他侵入了他闪亮的新 Rigol DS1054Z，以释放隐藏在其中的 1104Z 功能。

谣言是真的，解除对该软件的控制就像查找您的序列号并知道生成有效许可证的正确 URL 一样简单。[克里斯]跑进了一个无用的网站，但这就是在互联网阴暗的停车场地下室做生意的代价。然而，一旦他敲对了门，说出了暗号，他就自豪地拥有了 50MHz 的额外带宽，SPI、I C 和 RS-232 解码器，两倍的存储深度，以及 1104Z 附带的所有触发器。
