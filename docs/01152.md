# 显然，暗黑破坏神 3 是一款街机游戏

> 原文:[https://hack aday . com/2012/07/26/Diablo-3-is-a-arcade-game-apparely/](https://hackaday.com/2012/07/26/diablo-3-is-an-arcade-game-apparently/)

[![](../Images/df6cff98f02f85bab33c1874c7b14cea.png "diablo")T2】](http://hackaday.com/wp-content/uploads/2012/07/diablo.png)

MAME 橱柜简直太棒了。这是重温旧拱廊上又脏又粘的荧光地毯、巨大的噪音、霓虹灯和黑光灯的好方法。如果说 MAME 橱柜有什么问题的话，那就是游戏已经从过去的四分之一大橱柜中走了出来。将《星际争霸》、《TF2》或其他流行游戏放在一个街机柜中是没有意义的。

[Dave]在街机上玩 Gauntlet 长大，但各种控制台端口从未辜负用操纵杆和按钮玩它的体验。暗黑破坏神 3 出来的时候，[戴夫]知道自己要做什么。他建造了一个暗黑破坏神 3 游戏柜，完全可玩，忠实于昔日的地牢爬虫。

令人欣慰的是，旧内阁没有被掏空这一建设；游戏上市前一个月，[戴夫]捡了几块胶合板，给自己做了一个街机柜。在应用了一些非常漂亮的图形，安装了按钮和操纵杆之后，[戴夫]有了一个全功能的暗黑破坏神街机游戏，甚至不需要 25 美分硬币。

最近，我们已经看到我们的版本[颠覆了传统的游戏控制](http://hackaday.com/2012/07/10/custom-starcraft-controller/)，我们希望这一趋势继续下去。休息之后你可以看看[Dave]的演示视频。
