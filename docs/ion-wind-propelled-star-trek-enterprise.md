# 离子风推动的星际旅行企业

> 原文：<https://hackaday.com/2013/12/09/ion-wind-propelled-star-trek-enterprise/>

[Steven Dufresne]最近演示了使用高电压的离子推进，有人指出这有点像《星际迷航企业号》上的曲速驱动器…所以他出去买了一个模型企业号并重建了他的演示！

他关于离子风的原始视频很好地总结了他实验的开始。事实上，这实际上是他在 1996 年新能源国际研讨会上看到的一个设计的再创作，这个设计是 1960 年获得专利的 [*电动装置*](http://www.google.ca/patents/US2949550) 。

通过产生高压电弧，似乎由此产生的“离子风”相对于固定的测试设备推动了企业。我们有没有提到他自己制造了[高压电源？](http://www.youtube.com/watch?v=GEuK1OdYxHk)

[Steven]还指出，推进可能是由于介电泳而发生的，但还没有发现任何确凿的证据来证明这一点。就连这项专利也对其工作原理含糊其辞:

> 本发明利用了我发现的迄今未知的电动现象；也就是说，当一对适当形式的电极以一定的相互固定的间隔关系被保持，并被浸入电介质中，然后带适当程度的相反电荷时，会产生一个力，该力倾向于移动该对电极穿过该介质。

不管它如何或为什么工作，这是一个有趣的视频观看，所以请确保你在休息后留下来看它！
