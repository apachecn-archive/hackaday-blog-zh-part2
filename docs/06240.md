# 制造者如何提议

> 原文：<https://hackaday.com/2014/06/06/how-a-maker-proposes/>

![proposal](img/c8d8e2fcd96362a26849d72a267378f4.png)

[Sefi Attias]刚刚给我们发了一个温馨的小视频，视频中他是如何向他的女朋友[Tania]求婚的——利用了来自科技和其他制造商的一点帮助。

作为一名创客，[Sefi]总是在建造给[Tania]留下深刻印象的东西，所以他认为让这个提议成为独一无二的创客体验才是合适的。

他开始自己设计订婚戒指，要 3D 打印。这是一个非常复杂的小东西，由重复出现的“我将永远把你许配给我”这句话组成。对于印刷来说，这几乎太复杂了——但他们成功地用蜡做到了这一点，这使他们能够创建一个模具，然后用白金铸造最终部件。一旦完成，他们就在适当的位置放上一颗钻石来结束这一切。

第二步是提案，使用四旋翼飞行器、一条 RGB LEDs 和长时间相机曝光使之成为可能。为了向[塔尼亚]实时展示，他们在街道一侧安装了投影仪和屏幕，为他们身后的公园提供了一个超现实的窗口。这一切都是在来自 XLN 太空公司和四旋翼飞行器公司的 20 多人的帮助下实现的。

哦，是的，她同意了。
