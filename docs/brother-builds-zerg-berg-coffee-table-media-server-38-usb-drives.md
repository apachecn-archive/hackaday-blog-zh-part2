# 兄弟建立“Zerg-Berg”咖啡桌媒体服务器–38(！)USB 驱动器

> 原文：<https://hackaday.com/2014/12/20/brother-builds-zerg-berg-coffee-table-media-server-38-usb-drives/>

几个月前[Travis]的媒体服务器死亡后，他的兄弟[Nick]秘密策划在圣诞节替换它。承认这是一个“愚蠢的鲁布·戈德堡式”的安排，尼克想要为他的兄弟姐妹定制一些与众不同的东西。38 个 USB 热插拔笔记本电脑驱动器没有走正常的 SATA 路线，而是聚集在一个定制的皮革外壳中，顶部是古铜色的玻璃。

[Nick]只花了 350 美元就买到了 45 块 500GB 的硬盘，并围绕这些硬盘设计了这个项目。他花了 1000 美元为它们每个都配了金属底座，由价值 800 美元的 PCIe quad 独立 USB 控制器供电——没有集线器。一个 550 美元的 Xeon 主板，带有 14 个 USB 端口，16GB 内存，一个基本的显卡和 1000W 电源，完善了电子设备。

在 Windows 8.1 下，所有的驱动器都排列在存储空间下的单个巨型阵列中，没有 [raid](http://hackaday.com/2009/03/09/24-solid-state-drives-in-raid/) 。

所有的东西都被放在一个木质框架的咖啡桌上，用高档皮革包裹着，这是[尼克]花了 65 个小时手工缝制的。别致的黄铜角撑将框架撑成方形。所有的电线都在桌子下面，所以看得见的表面是干净和清晰的。桌子结构由半英寸见方的木条制成的腿支撑，木条弯成发夹状，用螺栓固定在底部。

总的来说，Travis 的 Zerg-Berg 媒体服务器的价格在 4500 美元左右。[尼克]打算让它持续很长时间。
