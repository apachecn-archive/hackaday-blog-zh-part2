# 电动摩托车进入赛车场

> 原文：<https://hackaday.com/2013/05/21/electric-motorcycle-hits-the-racing-circuit/>

![electric-motorcycle-hits-the-race-track](img/241377ea130b19bfbc8afb98630a5c95.png)

检查一下悬挂在摆臂与摩托车车身连接处的那个结实的电动马达。正是肌肉让这辆[最近完成的电动摩托车准备好参加](http://farfleselectrics.blogspot.com/2013/05/the-first-track-day-at-pats-acres-was.html)比赛。

[杰克逊·爱德华兹]一直在努力从头开始建造这个。他的目标是让它与生产线上的摩托车竞争，他最近的试运行指向成功。这部电影展示了后悬架的几个问题。这实际上导致他在转弯时倾倒了自行车。他没有受伤，但不幸的是，车把上的控制面板被砸碎了。稍加修理后，他可以放心地骑行了。在视频的最开始，当它滑过镜头时，我们被它的安静震惊了。

当然，我们以前见过其他电动摩托车。这些都是从天然气转化而来的。从头开始设计真的提供了很多选择，而翻新是不可能的。一定要仔细阅读他博客上的所有帖子，以了解全貌。
