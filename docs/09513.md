# 成为一个疯狂的科学家，建立一个配电板

> 原文：<https://hackaday.com/2015/07/14/become-a-mad-scientist-build-a-power-distribution-panel/>

大型开关和指示灯的一个实际用途是制作一个配电板，当您想要控制和监控许多设备(如电子工作台或业余无线电台)的功耗时，它会非常有用。老式的外观和使用军用剩余电子设备，这个配电板允许控制背面的插座。我有没有提到过[我 16 岁的时候造的](http://glcharvat.com/Dr._Gregory_L._Charvat_Projects/A_Really_Big_Switch_Panel.html)？

建造它很容易，120 伏交流电通过主断路器进入。它通过交流电流表(带内置分流器)馈入线路滤波器。从线路滤波器到线路电压表和灯丝变压器，为指示灯供电。然后，该交流线路通过总线连接到断路器。每个断路器控制后面板上的一个插座。随着器件的开启或关闭，可以测量电流消耗。休息后的视频概述很好地证明了这一点。

要有创意。使用军用剩余开关、指示器和其他独特的硬件。定制给你喜欢的疯狂科学家外观，同时也提供有价值的功能。

将测得的电流乘以电压来计算瓦特数。注意线路电压。

我发现当你住在出租的公寓或房子外，并且你的实验室装置必须不时移动时，这个配电盘非常有用。
