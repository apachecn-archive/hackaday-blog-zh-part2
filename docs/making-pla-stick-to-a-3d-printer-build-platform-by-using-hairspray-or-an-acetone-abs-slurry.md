# 通过使用发胶或丙酮 ABS 浆料使 PLA 粘在 3D 打印机构建平台上

> 原文：<https://hackaday.com/2013/04/06/making-pla-stick-to-a-3d-printer-build-platform-by-using-hairspray-or-an-acetone-abs-slurry/>

[Chris]在让 PLA 坚持他的 Printrbot 的构建平台方面遇到了一些真正的问题。这当然不仅限于这种品牌的打印机，而且影响到所有使用 PLA 作为原材料的基于挤出机的硬件。他想出了几个解决问题的方法。

第一个是我们非常熟悉的东西。上图显示[克里斯]在平台上涂了一层薄薄的发胶。这是我们自己的 3D 打印机使用的技术。这些纸被用作一个面具，以帮助保持螺纹杆上的粘性物质。想了解更多关于发胶技巧的信息，克里斯建议你[阅读这篇文章](http://www.thingiverse.com/thing:28787)。

第二种技术使用一种用 ABS 下脚料浸透一瓶丙酮制成的浆液。在休息后的视频中，他展示了一个装有溶剂的玻璃罐，里面挂着过去打印工作的废料。这样过几天就可以用了。他拿了一张纸巾，用溶液浸湿，擦了很少一点。他确实提到这最终会侵蚀 Kapton 胶带，所以尽量少用。
