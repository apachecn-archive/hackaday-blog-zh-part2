# 一种低成本的太阳能沼泽冷却器

> 原文：<https://hackaday.com/2014/05/05/a-low-cost-solar-powered-swamp-cooler/>

一个迫在眉睫的、折磨人的夏天正准备向我们中的许多人袭来，这使得[这个极其便宜的沼泽冷却器建筑](http://www.thegoodsurvivalist.com/how-to-make-a-454-homemade-air-conditioner-for-about-15-this-is-totally-awesome-and-so-easy-to-make/)成为一个有吸引力的抵御炎热的工具。

虽然这是一个非常标准的蒸发冷却器，但这种设计是一个整洁且可运输的成品。底座是一个从当地五金店买来的价值 3 . 5 美元的桶，附带泡沫聚苯乙烯衬垫。三个 2 1/8 英寸的孔刻在桶和内衬的侧面，可以很好地安装一些一英寸半的 PVC 管，不需要胶水。

盖子上的最后一个切口是为了放置一个小台扇，这使这个造型更加完美——如果你喜欢，你也可以切进泡沫聚苯乙烯衬里的盖子。视频演示了使用 15W 的太阳能电池板来运行风扇，我们不得不承认，冷却器似乎是一个出色的低成本构建。然而，它需要一个冷冻的加仑罐来抽取冷空气，每个罐大约 5-6 个小时。也许我们的节俭和数学爱好者之一可以对一天几次用一罐冷冻水储存水桶的成本做出一些猜测？跳跃后的视频。
