# 简单的黑客播放任何声音，因为你的门铃

> 原文:[https://hack aday . com/2012/11/23/simple-hack-plays-any-sound-as-your-door-chime/](https://hackaday.com/2012/11/23/simple-hack-plays-any-sound-as-your-door-chime/)

![](../Images/421e352efb7df09834b5107f1d9e682d.png "simple-custom-doorbell")

[Jim]将无线门铃变成定制铃声播放器的技术非常简单。他仅用一把螺丝刀和一把指甲钳作为工具，就成功地完成了整件事。但是，如果你想长期使用这个，我们建议焊接连接，而不是只依靠扭曲的电线。

上面你可以看到项目中使用的所有组件。无线门铃单元与当今市场上大多数电池供电的单元没有什么不同。Radio Shack 盒子里有一个录音模块，可以播放长达 20 秒的音频。这是由右边的 9V 电池供电。[Jim]从门铃上拆下扬声器，并剪断连接到板上的一根电线。这被重新用作记录模块的接地连接。另一条扬声器线连接到模块 PCB 上的“播放”按钮。就是这样，只要录下你自定义的声音，把所有东西装回门铃的箱子里。你可以看到整个破解过程，并在休息后听到演示。
