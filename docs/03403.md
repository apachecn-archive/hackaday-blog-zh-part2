# 开源 PLC

> 原文：<https://hackaday.com/2013/06/09/open-source-plc/>

在工业应用中，控制继电器、伺服系统、螺线管等不仅仅是在 Arduino 中布线和插入一些代码的问题。不，为了可靠运行，您需要一个 PLC(可编程逻辑控制器)来自动化您的所有硬件。PLC 通常是非常昂贵的硬件，这使得[Warwick]想出了他自己的。他制造了两个版本，一个大型的 T1 和一个小型的 T3，这两个版本可以处理任何交给他们的任务。

这两款设备都由运行频率为 48 MHz 的 ATMEL SAM7S ARM 芯片供电。两个器件中较小的一个具有 10 个数字输入、4 个模拟输入和 8 个数字输出，每个能够吸收 200 mA 电流。较大的 PLC 有 22 个数字 ins、6 个模拟 ins 和 16 个数字输出。这两款设备都有大量的 USB、RS-232 和 RS-485 端口连接

下面你可以看到大型 PLC 被用作条形码扫描仪和一个奇怪的设备，使用压缩空气悬浮乒乓球。还有一个较小的 PLC 点亮一些 led 的演示。
