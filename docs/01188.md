# [迪诺]为这个月的制作制造一个自动取货机

> 原文：<https://hackaday.com/2012/07/30/dino-builds-a-fetch-o-matic-for-this-months-make/>

![](img/21c7b19c55c6d8c6cd139ce1be8b89cb.png "dog")

这个月的 Make 杂志上有一篇文章。这是一个自动球发射器，它可以让你在不磨损手臂的情况下和你的狗玩捡球游戏。

[迪诺]的球发射器内的发电站是一个由 18 伏无绳电钻电池组供电的挡风玻璃刮水器电机。当一个球落入漏斗时，它就会打开一个开关，给马达提供一些动力。

实际发射球的摆臂用弹簧固定在球发射器的框架上。这为电机旋转的一半储存能量，直到臂在盒子内旋转一半。然后，手臂快速加速，将球发射过院子。

[迪诺]说他正在训练他的狗在找回球后把球扔进滑槽，创造了一个永久的捡球游戏。至少在电池耗尽之前。休息后的视频。
