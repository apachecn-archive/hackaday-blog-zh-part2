# SMD 焊接在…热沙上？

> 原文：<https://hackaday.com/2014/02/15/smd-soldering-on-hot-sand/>

需要做一些 SMD 焊接？没有工具？没问题！这里有一个创造性的方法，可以作为一个方便的工具添加到你的腰带上:[使用热沙进行 SMD 焊接。](http://www.youtube.com/watch?v=OtrdsLLYAIM)

[奥利弗·克罗恩]最近发布了这个小视频，演示如何使用热沙进行回流焊接。他用本生灯加热一个陶瓷砂锅，作为一种加热板。它似乎工作得很好，这是一种非常独特的方式——如果你想变得更专业一点，你也可以在沙子里扔一个温度探头，以获得更好的热量控制！

当然，还有很多其他的方式来进行回流焊接，比如[使用一个重新设计的烤箱](http://hackaday.com/2012/03/08/toaster-oven-reflow-soldering-roundup/)，[在你吃完熏肉后在煎锅上煎一些电路](http://hackaday.com/2013/07/28/electric-skillet-reflow-soldering-guide/)，或者如果你想变得新奇，你甚至可以[为它制作你自己的工具包！](http://hackaday.com/2013/11/11/build-your-own-desoldering-station-on-the-cheap/)

无论如何，留下来观看 SMD 在热沙子上焊接的史诗视频。
