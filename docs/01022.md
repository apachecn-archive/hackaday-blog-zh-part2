# 猫教练会让他们远离柜台

> 原文:[https://hack aday . com/2012/07/12/cat-trainer-will-keep them-off-the-counters/](https://hackaday.com/2012/07/12/cat-trainer-will-keep-them-off-the-counters/)

![](../Images/0aa3fdcc15eb4aec9a3f72f0e6144b5a.png "cat-training-will-keep-this-from-happening")

我们的猫是不允许上厨房台面的，[钢铁丛林]也有同样的规定。但是他在他的炉灶上方的引擎盖上发现了一些脚印，另外一个安全摄像头拍下了这张[凯尔索]违反规则的照片。因为他不能总是让这个毛球乖乖听话，所以他做了一个电子猫训练器来帮它做这件事。

所需的功能并不复杂。你需要监控猫什么时候在不该在的地方，然后把它赶走。对于后者，他抓起一个红外测距仪。当猫经过传感器前面时，它会触发系统的第二部分:一个非常响亮的高音蜂鸣器。任何微控制器驱动系统都不会有问题。在这种情况下，它是一个 PICAXE 28X1。

休息之后，你可以看到训练员的动作。这绝对有效，因为仅仅播放视频就把我们自己睡觉的小猫赶出了房间。
