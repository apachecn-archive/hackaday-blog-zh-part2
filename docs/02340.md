# 用乐高和冰棒棍解决拉比克魔方

> 原文:[https://hack aday . com/2012/12/15/solving-a-rubicks-cube with-Lego-and-棒冰-sticks/](https://hackaday.com/2012/12/15/solving-a-rubicks-cube-with-lego-and-popsicle-sticks/)

![solver](../Images/cce8479bce85afcca1e3e068a64d1cbb.png)

我们以前见过自动化的拉比克魔方解算器，但从来没有一个像[詹姆斯]'[冰棒棍和乐高拉比克魔方解算器](http://creativetronics.blogspot.co.nz/p/rubiks-cube-solver.html)一样获得如此多的奖项。\

为了降低项目的复杂性，[詹姆斯]选择不使用网络摄像头来检测立方体每个面上的颜色模式。相反，他编写了一个小 Python 应用程序，在让他的算法在立方体上松散并计算如何解决它之前，手动输入每个面上的图案。

到目前为止，[詹姆斯]已经从他的项目中看到了巨大的成功。他参加了新西兰的 Brightsparks 比赛，净赚了 1000 美元。这次竞赛使他赢得了[实现梦想科学竞赛](http://www.realisethedream.org.nz/)，在那里他赢得了一项大奖，并进入了明年初举行的国际台湾科学博览会。一个初露头角的黑客取得了令人敬畏的成就，我们迫不及待地想看看他接下来会想出什么。
