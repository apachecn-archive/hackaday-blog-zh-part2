# 3D 打印双耳麦克风

> 原文:[https://hack aday . com/2015/06/28/3d-printing-双耳-麦克风/](https://hackaday.com/2015/06/28/3d-printing-binaural-microphones/)

双耳音频可能是你可以用一副耳机听的最酷的东西。双耳录音不是只有一个麦克风，而是使用两个麦克风，设置在模拟人脑的内部，以准确复制您在现场听到的声音。

录制双耳音频的唯一方法是将假的塑料耳朵附在仿真头上。大多数著名的麦克风制造商[都有类似这样的东西](https://www.neumann.com/?lang=en&id=current_microphones&cid=ku100_description)，但有了 3D 打印机，一切皆有可能。【卡洛斯】[用 3D 打印机创造了他自己的双耳麦克风](https://github.com/carlosgs/OpenBinaural)，并经历了创建一些音频演示的麻烦。结果很奇怪，就像[卡洛斯]在你耳边低语。

在这个麦克风设置中使用的耳朵来自于[乔纳森·马奇]的一个 Thingiverse 项目。这个模型没有对耳道进行适当的建模，也没有任何方法来屏蔽从一只耳朵到另一只耳朵的声音；这就是为什么专业模型也包括一个头。[Carlos]修复了这些缺点，并创建了一些 3D 模型，精确地模拟了人类的耳朵和头部。

这个项目还有一个简单的立体声麦克风放大器，它被设计成正好放在两耳之间。该放大器是在 KiCAD 中设计的，PCB 是单面的。在一块条板上组装并不简单，但[卡洛斯]确实在他的工厂里用覆铜板制造了它。
