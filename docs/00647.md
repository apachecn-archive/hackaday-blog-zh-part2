# 大规模 Arduino 控制的温室做一些严重的农业

> 原文:[https://hack aday . com/2012/06/05/大规模-arduino-controlled-green house-do-some-serious-farming/](https://hackaday.com/2012/06/05/large-scale-arduino-controlled-greenhouse-does-some-serious-farming/)

![](../Images/4abdc45e905f82b7c9fd017c0146f474.png "arduino-greenhouse-controller")

[Instrument Tek]不是在摆弄一个业余爱好大小的温室。事实上，如果再大一点，我们会称之为商业运作。但是让我们感兴趣的是他构建的专业质量的温室自动化和 Arduino 板。

除了占地面积大约为 10'x10 '之外，这个温室和你在托儿所看到的差不多。这是一个用棍子建造的框架，墙壁是用聚乙烯制成的。专业的温室监控和调节温度和湿度，这个就是这样做的。休息后的视频以显示控制器盒开始。它有温度、湿度和光线传感器，使 Arduino 能够判断生长条件。如果太热，一些板条被打开，一个风扇从结构中排出空气。如果天气变冷，一系列照明设备就会通电。它们包含加热灯，因为这种设置是在加拿大阿尔伯塔省北部，有些晚上会变得相当冷。滴水系统也是自动化的，有一个螺线管来开关水。

除了 3:26 的展示和讲述，我们还嵌入了一个 27 分钟的视频，展示[如何构建控制器盒](http://www.youtube.com/watch?v=ERLkFat_FTA)。所以你可以[在室内的架子上](http://hackaday.com/2012/06/01/scratch-build-garden-nursery-uses-arduino-monitoring/)开始种植植物，然后当它们长到足够大的时候再移植到温室里。
