# 构建清晰的白板

> 原文:[https://hack aday . com/2013/11/22/building-a-crystal-clear-white board/](https://hackaday.com/2013/11/22/building-a-crystal-clear-whiteboard/)

[迈克尔·佩什金]在西北大学教机械工程。他喜欢用图表来说明他的观点，但他也喜欢在这样做的时候面对他的学生。[他的解决方案是开发这种清晰的白板](https://sites.google.com/site/northwesternlightboard/home),最终得到的不仅仅是一些手绘示意图。

上图中很难看出他在黑板上写了什么，但是眯着眼睛看看你是否能发现这种教学方式有什么问题？他写的东西都是反着写的。在这种情况下，这实际上不是问题，因为[迈克尔]使用了[翻转教学](http://en.wikipedia.org/wiki/Flip_teaching)。他录制并在网上发布他所有的讲座。然后，课堂时间用于就讲座主题进行问答。为了让文本以正确的方式阅读，他把摄像机从一面镜子上弹开。

板子本身是一块巨大的钢化玻璃板，用螺栓穿过玻璃板上的孔固定在金属框架上。这使得边缘是自由的。他在顶部和底部添加了挤压成型的轨道，以嵌入 led 灯条。它们照亮玻璃的内部，并激发荧光干擦标记墨水，使其更加明显。[迈克尔]没有停止与董事会，他还装配了一个照明系统，给他很多选择，并使用一个显示器来处理数字叠加。他可以在电脑上画一张图，观察监视器，看他的记号笔在哪里做注解。所有这一切都是实时发生的，这意味着没有后期制作！休息之后，请观看这些功能的演示。

这个[完全可以在没有眼镜的情况下完成](http://hackaday.com/2012/04/26/3d-whiteboard-without-the-whiteboard/)，但是那会让写作的人更加困难。
