# 3D 打印:激光切割 3D 对象

> 原文：<https://hackaday.com/2015/04/30/3d-printering-laser-cutting-3d-objects/>

3D 打印几乎可以创造出任何可以想象的形状，但是问问任何一个在打印机前呆了几个小时的人，他们都会告诉你 3D 打印最大的问题:打印一张照片要花很长时间。波茨坦大学的人机交互实验室利用黑客空间中第二常见的工具找到了解决这个问题的方法。他们正在使用激光切割机将零件生产速度提高 20 倍或更多。

Platener 系统不是直接打印 3D 文件，而是将模型分解成各个组成部分。然后，这些部件可以从丙烯酸树脂或胶合板上激光切割下来，组装起来，并以更快的速度重复进行。

你可能会认为激光切割部件只适合平面，但通过切口弯曲和一层一层堆叠材料等技术，几乎任何可以用 3D 打印机生产的东西也可以用 Platener 生产。

为了测试他们的理论，即 Platener 比 3D 打印更快，Platener 背后的团队从 Thingiverse 下载了超过 2000 个对象。对于传统的 3D 打印和 Platener 系统来说，这些对象的打印时间可以很容易地计算出来，事实证明 Platener 比打印快 20 倍以上，时间超过 30%。
