# 建立一个自制的柴油 ECU

> 原文：<https://hackaday.com/2012/07/10/building-a-homebrew-diesel-ecu/>

![arduino-diesel-ecu](img/2c9d84a5444c70d130f1ff0c75066499.png "arduino-diesel-ecu")

多年来，汽车发动机已经变得越来越复杂，并且随着这种复杂性的增加，越来越依赖复杂的计算机系统来运行它们。这些控制系统通常是许多小时研究和开发的成果，受到创造它们的汽车制造商的精心保护。一位名为[synkooppi]的芬兰黑客决定完全抛弃他的柴油发动机 ECU，用一个比 Arduino 稍大一点的东西建造一个自己的 ECU，而不是依靠一个封闭的系统来为他的汽车提供动力。

正如你从他的网站上看到的，[synkooppi]已经使用 Arduino Mega 创建了他的 DIY ECU，它能够控制采用博世 VP37 或其他直列柴油泵的柴油发动机。到目前为止，他已经掌握了所有的基本工作原理，这使得他能够驾驶和控制奥迪柴油发动机。

虽然关于他的自制 ECU 的许多细节很难获得，但他确实有一系列的开发视频发布在 YouTube 上，这应该有助于满足好奇的头脑。对于那些有备用柴油发动机的人来说，[synkooppi]已经发布了他的第一个代码供大家试用。

留下来看看视频的 ECU 的行动。
