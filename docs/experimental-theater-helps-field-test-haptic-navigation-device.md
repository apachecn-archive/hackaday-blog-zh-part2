# 实验剧场帮助现场测试触觉导航设备

> 原文：<https://hackaday.com/2015/09/04/experimental-theater-helps-field-test-haptic-navigation-device/>

“全神贯注的设备用户”迷因，就像有人在智能手机上密切关注谷歌地图，以至于他们走到外面的车流中，正变得越来越普遍。一个需要面对面交流的界面不仅会在交通中对你的健康造成危害，而且对视力受损的人来说也不是特别有用。触觉界面可以帮助视力正常的人和视力受损的人，但智能手机实际上只有一个触觉技巧——振动。但是耶鲁大学的一名工程师已经开发出了一种 3D 打印变形导航工具，它可能会成为触觉游戏的改变者。

发明者[Ad Spiers]将其命名为 Animotus，该设备是一个分成两层的手持立方体。上层可以向左或向右旋转，伸展或收缩，给用户提供触觉和视觉线索，以确定向哪个方向行走以及离目标有多远。为了对该设备进行实地测试，[Ad]与伦敦的一个戏剧团体合作，制作了一部名为“平地”的互动剧，该剧的大部分是在一个完全黑暗的老教堂里上演的。正如你在休息后的夜视视频中看到的，佩戴追踪设备的观众每人都被给予了一个敌意，允许他们在互动场景中导航。追踪数据显示，用户在使用 Animotus 时很快适应了黑暗中的导航，一些人变得如此依恋他们的设备，以至于他们对这部戏的结局感到不安，其中涉及模拟没收和销毁。

除了表演艺术应用，拥有一个以上自由度的触觉技术还有很大的潜力。想象一下上面提到的谷歌地图的蓝牙接口，或者使用 Animotus 和摄像头引导用户绕过障碍物的电子导盲犬。然而，正如这个 [Hackaday 奖半决赛选手](http://hackaday.com/2015/08/26/hackaday-prize-semifinalist-haptic-navigation/)所展示的，传统触觉仍然有很多用途。
