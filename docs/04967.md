# 神秘盒子

> 原文:[https://hackaday.com/2014/01/03/mystery-box/](https://hackaday.com/2014/01/03/mystery-box/)

有人记得 Myst 这个游戏吗？嗯，[迈克尔]和他的女朋友最近经常玩这个游戏，所以在她生日的时候，他决定给她做一些受这个游戏启发的东西。

对于那些不知道经典的[神秘系列](http://en.wikipedia.org/wiki/Myst_series)的人来说，这是一套始于 1993 年的游戏，在这里你扮演一个陌生人的角色，探索其他星球(称为 age)以解决各种逻辑和机械难题。

无论如何，[迈克尔]让他的女朋友买了参观 [GC319QK](http://www.geocaching.com/geocache/GC319QK_lost-caches-lost-in-der-ostsee) (一个需要潜水的地质宝藏网站)——因为礼物是一个相对较小的象征，所以[迈克尔]为它制作一个别致的盒子是合乎逻辑的——这正是他所做的。这是一个奇特的小木箱，里面有发光二极管、一个按钮、一个插销、一根拔掉的电线、不同的杆和奇怪的传感器——这是一个非常聪明的小拼图。

我们可以向你解释它是如何工作的(使用 Arduino、光电晶体管，也许还有[源代码](https://github.com/michael-buschbeck/mystbox))，但我们认为你会喜欢看(迈克尔的)视频。
