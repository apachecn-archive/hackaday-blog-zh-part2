# 机器人有节奏；携带鼓棒

> 原文：<https://hackaday.com/2013/01/08/robot-has-rhythm-carries-drum-sticks/>

这个小机器人需要和蓝人乐团一起上路。脚踏漫游车携带自己的鼓棒，[使用随机物体作为其鼓组](http://letsmakerobots.com/node/112)。

我们承认这不是一个新鲜的黑客。这听起来像是 2008 年的事，但这是我们第一次记得看到这个小家伙。在观看了跳转后嵌入的视频后，我们认为您会同意该项目值得尽可能多的有志硬件黑客观看。

在那堆东西的顶部是一个扬声器，第二个藏在黄色踏板之间。下面的单元让 PICAX 28 微控制器产生哔哔声和爆裂声，而上面的单元为击鼓提供背景音乐。在超声波测距仪上方延伸的两根杆与几个马达和鼓相连，第三根杆看起来像尾巴。就连左右扫测距仪的伺服也保持着节拍。同步的魔力都在代码中，你可以在[这篇更长的构建教程](http://makeprojects.com/Project/Yellow+Drum+Machine/1077/1#.UOxVTWI1EWm)的步骤 11 中找到。
