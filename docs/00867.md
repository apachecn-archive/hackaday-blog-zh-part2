# 滚动你自己的备用电池系统

> 原文：<https://hackaday.com/2012/06/28/roll-your-own-battery-backup-system/>

[Chris]和[Dom]希望以低廉的价格构建他们自己的电池备份系统。他们在寻找零件方面很有创意，最终以大约 80 美元的价格组装了一套电池供电的闭路电视系统。

由于短期停电在该地区相当普遍，这种电池备份是有意义的。我们已经看到了一些非常粗糙的全屋系统，但这更是一种新奇。这是一件好事，因为黑客二人组决定重复使用被送往废品场的电池。它们连接到一个涓流充电器，确保它们在主电源通电时持续充满。但是当停电时，继电器会将出线盒切换到逆变器(也是一个用过的部件)。

休息之后，娱乐视频中概述了该系统。你会看到他们展示了完整的构建，随后是他们设计的电路及其工作原理。
