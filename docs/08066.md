# 全面编织

> 原文:[https://hackaday.com/2015/01/15/knitting-in-the-round/](https://hackaday.com/2015/01/15/knitting-in-the-round/)

这些年来，Hackaday 上有一些涉及针织的帖子，要么通过修改一台旧的兄弟针织机来融入现代硬件，要么通过建立一台 3D 打印针织机。所有这些针法都是平织的例子，不能制成无缝管。[圆型针织](http://www.knitic.com/2014/12/27/circular-knitic/)通过使用 3D 打印和激光切割机创建开源圆型针织机，打破了这一趋势。

Circular Knitic 是在 70 年代的老式针织机的基础上发展而来的。这种构造远远超出了简单的电子操作，呈现了一个专为圆型针织设计的完整针织机。这是完全自动化的，所以一旦机器安装好，一个巨大的针织纱管就自动生成了，无需任何人工干预。

这不是第一台完全开源的针织机； [OpenKnit](http://hackaday.com/2014/02/20/openknit-the-open-source-knitting-machine/) 可以用铝挤压、一些电子设备和一些 3D 打印零件制成。然而，圆形针织机是我们见过的第一台圆形针织机，根据 Github 的说法[是完全开源的。](https://github.com/var-mar/circular_knitic)

在西班牙萨拉戈萨的一个名为“实干家”的展览上，圆形针织衫的创造者[Varvara]和[Mar]展示了他们的机器，他们将在那里编织六个月。你可以在下面看到一些视频。
