# 奥林匹克铅球机为每一枚美国奖牌倒出一个

> 原文:[https://hack aday . com/2012/08/07/Olympic-shot-machine-pouts-one-out-for-every-us-medal/](https://hackaday.com/2012/08/07/olympic-shot-machine-pours-one-out-for-every-us-medal/)

![](../Images/8537b6c2f139012e69802bf61fc0ce48.png "shot")

还有什么比每次美国赢得奖牌都让机器人给你倒一杯更好的观看奥运会的方式呢？SmartThings ，他们创造了一种为每一个美国奥运冠军倒酒的机器。

从幕后视频中，我们看到整个建筑是由一个带 XBee 盾的 Arduino 控制的。XBee 连接到一个简单的 iPhone 应用程序，当前观看奥运会的用户可以选择美国获得的奖牌。铜牌代表杰克·丹尼尔，银牌代表乔斯·奎尔沃，金牌代表房间里的某个人代表戈尔德施莱格。

尽管构建围绕着 SmartThings 框架，但我们并不十分确定这个框架是什么。从脸书页面(SmartThings 的最佳信息来源，至少在它们发布之前)来看，它看起来像是一个硬件，作为 XBee 桥的互联网，以及一个轻松构建移动应用程序的框架。

不管 SmartThings 是什么，它仍然是一个非常酷的构建。
