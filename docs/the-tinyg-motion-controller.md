# TinyG 运动控制器

> 原文：<https://hackaday.com/2014/09/24/the-tinyg-motion-controller/>

[https://www.youtube.com/embed/2HQi6RzazSc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/2HQi6RzazSc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

当你去参加任何形式的商展时，你都希望展位上有很酷的演示。在 Maker Faire 上，有很多，但最催眠的是一个围绕 Synthetos 的 TinyG 运动控制器建造的机器人。

演示只是一个大型数控龙门移动一个球轴承在字符串。台架在 X 和 Y 轴上移动，微型救援球在 Z 轴上缠绕和展开。球在空间中移动，没有任何晃动地完全停止。甚至有几个透明的塑料管，球落入其中，然后弹出，不需要提升或降低绳子。这是运动控制器最酷的地方，这一切都是由 TinyG 实现的。

TinyG 是在 faire 上发现的少数运动控制和 CNC 板之一。在其基本配置中，它有 6 个运动控制轴，RS485 将几个电路板联网以实现疯狂的机器配置，以及一个功能强大的处理器来正确完成所有工作。
