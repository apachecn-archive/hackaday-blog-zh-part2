# RiotNAS:街头摄影的移动存储

> 原文:[https://hack aday . com/2014/03/27/riot nas-mobile-storage-for-street-photography/](https://hackaday.com/2014/03/27/riotnas-mobile-storage-for-street-photography/)

![riotnas1](../Images/92070b8af32271a1e72c35e0ec47e9a3.png)

你可能知道过去几个月委内瑞拉各地发生的抗议和示威活动，正如最近记忆中类似的公开抗议一样，技术可以为街上的人提供独特的启示。[阿尔弗雷多]向我们发送了这个提示，让我们了解 riotNAS: [抗议者拍摄的照片和视频的便携式存储设备](http://geckotronic.wordpress.com/2014/03/05/riot-nas-almacenamiento-movil-para-fotografia-de-calle/) ( [翻译](http://translate.google.com/translate?sl=es&tl=en&js=n&prev=_t&hl=es&ie=UTF-8&u=http%3A%2F%2Fgeckotronic.wordpress.com%2F2014%2F03%2F05%2Friot-nas-almacenamiento-movil-para-fotografia-de-calle%2F&edit-text=&act=url))。

前提很简单:在这些情况下，社交媒体是地面抗议者的盟友，但手机和相机很容易被认出来并被没收。riotNAS 通过运行 OpenWRT 和 Samba 的路由器提供便携式备份存储。[Alfredo]然后连接了一些外部存储的 USB 存储器和一块电池，可以提供大约 4 小时的工作时间。

现在，他把设备放在一个柔软的，看起来像化妆品的袋子里，这样就不会引起注意，也不会影响信号。查看他的网站，了解未来的设计计划——包括将设备藏在一个挖空的书中——以及存储在 riotNAS 系统上的一些样本照片。如果你对委内瑞拉正在发生的事情感到好奇，[点击维基百科页面](http://en.wikipedia.org/wiki/2014_Venezuelan_protests)或者访问[阿尔弗雷多]网站底部的一些资源。
