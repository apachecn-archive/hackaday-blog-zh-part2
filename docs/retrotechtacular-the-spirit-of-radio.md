# 反技术:无线电的精神

> 原文：<https://hackaday.com/2015/05/05/retrotechtacular-the-spirit-of-radio/>

出于这样或那样的原因，我们中的许多人仍然收听地面广播，无论是棒球比赛、脱口秀还是经典摇滚。但是你知道声音是如何传到你的接收器的吗？本周，我们的焦点聚焦在 KYW 电台制作的一部短片上，这部短片愉快地介绍了 1940 年调幅(AM)无线电传输的奥秘。

声音振动进入麦克风，并被转换成电流或音频波形。电波被放大后发送到几英里外的发射台。在这个过程中，信号会失去能量，因此在发射台被分几个阶段放大。然而，这种声波不能自己传播。它需要搭乘高频载波。这种波是由一个巨大的晶体振荡器在现场产生的，然后在广播之前经过一系列的放大。

最后一步是调幅本身。这里，原始音频波的变化幅度被用来调制高频载波的幅度。现在信号可以发送到塔台了。任何调到载波频率并在信号范围内的接收器都会捕捉到载波。在接收器内，这些电流被转换回我们的耳朵熟悉和喜爱的振动。
