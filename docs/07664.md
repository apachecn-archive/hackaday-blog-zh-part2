# 远程呈现机器人演示装置摆脱了束缚

> 原文:[https://hack aday . com/2014/11/26/tele presence-robot-demo-unit-breaks-free-of-its-confinement/](https://hackaday.com/2014/11/26/telepresence-robot-demo-unit-breaks-free-of-its-confinement/)

如果你把一个网真机器人放到网上让全世界免费试用，会发生什么？当然是欢乐。Double Robotics 是一家制造远程呈现机器人的公司。这款机器人有点像赛格威(Segway)，顶部是一台平板电脑。这个想法是，你可以控制它与自己的平板电脑从一个遥远的地方。这个机器人会让你的脸出现在屏幕上，在你不能(或不想)亲自到某个地方时，它几乎可以陪你到处转。

Double Robotics 决定将其中一个装置接入互联网，作为公开演示。当然，他们不能让这些东西无拘无束地在他们的设施里四处游荡。他们最后把它锁在了办公室里。这让用户能够驾驶它四处转转，感受一下机器人。当然，没过多久，用户就开始思考如何才能摆脱束缚。

一天，一名工人离开办公室时，办公室的门稍微开了一点点。一名用户注意到了这一点，并在足够的耐心和决心后，设法使用机器人把门打开。当时办公室似乎已经关门了，所以没有人在场见证这一事件。接下来是一次欢乐之旅，机器人通过将自己锁在房间里并对接充电站来隐藏自己的踪迹。

虽然这不是典型意义上的黑客，但这是黑客思维的一个完美例子。你被给予一些新技术，并探索到你应该达到的程度。在那之后，许多人会把它扔到一边，不再考虑它。然而，那些有黑客思维的人是不同的。我们的下一个想法通常是，“我还能用它做什么呢？”这个视频以一种有趣和幽默的方式展示了这一点。希望该公司吸取教训，对那件事严加管束。
