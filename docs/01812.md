# Raspi 的 Web IDE

> 原文：<https://hackaday.com/2012/10/06/web-ide-for-the-raspi/>

![](img/3cc69389becae60949ea615d9446622b.png "ide")

对于本月 Adafruit 的 Raspberry Pi Linux 发行版，[Limor]等人决定为 Raspberry Pi 构建一个基于 web 的 IDE。

Raspberry Pi WebIDE 是一个运行在 Raspi 上的 web 服务器。通过在 web 浏览器中连接到您的 raspi，您可以创建自己的 Python 程序，能够与 GPIO 引脚进行交互。所有的代码都在 [bitbucket](https://bitbucket.org/) 的帮助下存储在云端。

WebIDE 目前正处于早期的 Alpha 阶段；有一些错误和小问题，但在休息后的视频中，[Limor]向我们展示了通过互联网将代码推送到 Raspi 并在基于 web 的串行终端中查看结果的可能性。

因为害怕发表评论，我们不得不指出 Adafruit 的 web IDE——以及其他 Arduino web IDEs，如 [Codebender](http://hackaday.com/2012/07/01/codebender-an-online-arduino-ide/) 和[wifi no](http://hackaday.com/2012/07/06/set-your-arduino-free-with-wifino/)——都是在云上工作的。如果你正在计划一个依赖于基于 web 的 IDE 的长期项目，你可能会因为不能在个人服务器上托管云而受到伤害。我们希望看到一个软件包，它允许我们在个人服务器上拥有与 bitbucket 相同的功能。如果你能找到一个做类似事情的项目，或者已经写了你自己的项目，[在](http://hackaday.com/contact-hack-a-day/)发送它，我们会传播这个消息。
