# 当树莓派和 Arduino 合作时你能做什么

> 原文:[https://hackaday . com/2012/08/31/what-you-can-do when-a-raspberry-pi-teams-up-an-arduino/](https://hackaday.com/2012/08/31/what-you-can-do-when-a-raspberry-pi-teams-up-with-an-arduino/)

我们认为将 Arduino 连接到树莓派是多余的，但有一件事引起了我们的注意。[Jan Stevens]提到 RPi 比以太网屏蔽便宜。有意思。随着我们进一步研究他的文章，我们开始认为他有所发现。使用 PHP 串行类在 RPi 和 Arduino 之间通信(死链接；[互联网档案馆](https://web.archive.org/web/20130912065739/http://www.fritz-hut.com/arduinopi-web-based-controller-for-arduino)。这最终成为一种非常廉价的方式，为您的硬件设备带来一些更强大的 web 编程选项。

当然，他只是驱动三个 RGB LEDs。但是休息之后的演示视频让我们看到了一些当嵌入式 Linux 机器运行时可用的界面选项。他使用 jQuery、AJAX、PHP 和 JSON 等等。如果你想亲自尝试一下，你可以从他的 Github repo 中获取代码。当然[我们会想听听你从这个起点开发的任何项目](http://hackaday.com/contact-hack-a-day/)！
