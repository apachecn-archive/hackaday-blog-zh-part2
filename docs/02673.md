# 发光的超级碗头盔

> 原文：<https://hackaday.com/2013/02/02/glowing-super-bowl-helmets/>

![glowing-football-helmets-adafruit](img/88a3152096b6419680858b2e837b5f93.png)

这些极客超级碗装饰发光多亏了[贝基斯特恩]创造的 EL 面板黑客。在这个图像中几乎不可能辨认出来，但是 EL 面板已经被应用到头盔的表面。在旧金山头盔上，你只能分辨出 f 底部的黑色连接器和电线。

El 面板很像 EL 线(但它们是扁平的),因为当连接到高压交流电源时，磷光体被激发。你可以毫无问题地将面板切割成各种形状。这里使用的技术是创建一个黑色乙烯面具去面板的顶部。这使得切割面板更加容易。

面膜贴是在乙烯基切割机上做的。[Becky]是使用矢量工具的大师，你可以在休息后的视频中看到。她用路径勾勒出每个团队徽标，以创建裁剪者可以使用的文件。由于头盔的曲线稍微扭曲了标识，所以试了几次才把标签贴好。一旦拨进来，她就把乙烯粘在 El 面板上，并在周边切开。

Adafruit 团队当然喜欢使用电致发光的口音。
