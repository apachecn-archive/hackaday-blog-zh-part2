# 平流层行动

> 原文:[https://hackaday.com/2013/01/18/operation-stratosphere/](https://hackaday.com/2013/01/18/operation-stratosphere/)

[![12secbeforehit-thumb](../Images/e4213810f35c3eaaef58a123bb6bbac8.png)T2】](http://hackaday.com/2013/01/18/operation-stratosphere/12secbeforehit-thumb/)

全景照片很好，但是完整的 360 度 x 180 度或球形全景照片会更好。[Caleb Anderson]决定进一步发展这一概念，试图使用一个高空气球和六个 GoPro 相机从 100，000 英尺的视频中提取全景照片。

这个项目的[概述可以在这里](http://robotrising.org/2012/09/operation-stratosphere-overview-and-history/ "Operation StratoSphere overview")找到，并给出了一些背景。第一个任务是开始原型化一些有效载荷容器，这对于一个一旦脱离你的手就几乎无法控制的设备来说是非常关键的。除了一些背景，在这个页面上还有一个很酷的第一次测试结果的互动全景，所以一定要去看看。

然而，这项实验中的“真正”黑客行为并不是将一个气球放入平流层或回收它。将这些图像链接成图片是一个巨大的挑战，涉及多种多样的技能和软件知识，我们的大多数读者都会为拥有这些技能和知识而自豪。解释中有几个视频，但我们嵌入了一个摄像机从天上掉下来的视频。请务必至少观看到(或跳到)刚过 1:05，那里所有的摄像机都令人印象深刻地幸免于撞击！
