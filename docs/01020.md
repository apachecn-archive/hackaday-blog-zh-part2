# 为 F250 制造前灯

> 原文:[https://hack aday . com/2012/07/13/fabring-heads-for-an-f250/](https://hackaday.com/2012/07/13/fabricating-headlights-for-an-f250/)

![](../Images/30c3a654bf6a8ced77d091d60f157b1d.png "custom-headlight-build")

这些定制头灯所耗费的时间是惊人的。[Mcole254]正在修理他兄弟的卡车，[用高强度放电(HID)灯](http://www.hidplanet.com/forums/showthread.php?43370-2012-F250-HID-Retro-In-Progress)替换普通的前灯，并在修理时加入一些漂亮的 LED 功能。

构建从移除和拆卸库存大灯总成开始。为了把外壳拆开，他把它放在烤箱里加热，直到胶水变软，部件可以被撬开。目标是用适合新灯具和 led 的组件替换反射器。上面你可以看到白色的碎片，它们是由木头和 PVC 制成的模具真空成型的。他用自制的真空成型器试了几次，但还是没有得到他真正想要的清晰度。他最近的帖子显示了一些将被取代的大规模 3D 打印部件。

在车内，他增加了一排琥珀色的发光二极管作为转向灯。你可以看到它们安装在上下反射镜之间的银带上。休息之后，我们将播放这些超级亮点的演示。
