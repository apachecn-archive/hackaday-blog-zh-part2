# 不寻常的谢妮电子管导致独特的时钟

> 原文:[https://hack aday . com/2015/07/14/不寻常-谢妮-tubes-lead-to-unique-clock/](https://hackaday.com/2015/07/14/unusual-nixie-tubes-lead-to-unique-clock/)

谢妮地铁的吸引力毋庸置疑。橙色等离子体在阴极周围穿过网状阳极并到达玻璃外壳，这种效果非常迷人，管中符号的堆叠给显示器带来了不同于其他任何技术的深度。因此，当[伊恩]在易贝以大甩卖的价格发现一套六个电子管时，他忍不住拿起它们，将它们组合成一个独特但难以读懂的谢妮钟。

原来[Ian]订购的那套电子管更有可能是用于测试仪器，而不是时钟，显示的符号有“Hz”、“V”和“ω”之类。起初对他看似无用的购买感到失望，[伊恩]把买家的悔恨放在一边，不管怎样还是造了他的钟。激光切割丙烯酸树脂，灯管下的蓝色 led 产生发光效果，电池供电的 RTC 与 ATmega328 交谈，以及适当的高压部分，这些都导致了一个好看而实用的时钟，即使[Ian]自己需要一个交叉引用图表来读取时间。休息过后，看完视频，你就能猜出整个字符集；剧透:很明显，ω映射到 0。

我们以前见过很多谢妮的项目，但很少有像伊恩的时钟这样独特的。
