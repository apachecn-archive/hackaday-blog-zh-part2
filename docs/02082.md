# 脚本自动化你的无脑 Android 游戏

> 原文:[https://hack aday . com/2012/11/09/scripting-to-automated-your-mind-Android-games/](https://hackaday.com/2012/11/09/scripting-to-automate-your-mindless-android-games/)

![](../Images/7a313de03ad30663fb28d1929bfafbce.png "scripting-android-games")

有一个愚蠢的小机器人游戏在互联网上制造了一些噪音。它叫做好奇号，是一个里面有东西的三维立方体。问题是，立方体上的每一个像素都必须被点击才能通过一层。当然有多层，而且…嗯，你明白了。[Stephen]认为这是一个完美的脚本，并着手[寻找一种自动玩 Android 游戏的方法](http://spamtech.co.uk/software/curiosity-game-bot/)。

正如你在上面看到的，他有一个很好的开始。为了使用当前形式的脚本，他找到了立方体的一部分，大部分是纯绿色。Android 设备使用 USB 电缆插入计算机，Android 调试桥运行脚本。这非常简单，因为它使用了 SDK 附带的 monkeyrunner 包。证据就在那里，问题只是他是否愿意花时间来完全自动化地玩[游戏](http://spamtech.co.uk/software/curiosity-game-bot/)。休息之后，您可以看到嵌入的脚本演示。

[Stephen]对自动化他自己不想做的事情并不陌生。这里有一个他的代码击败 PlayThru 验证码的例子。
