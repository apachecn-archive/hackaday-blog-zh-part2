# 基于 Arduino 的 Beme 遥控快门

> 原文:[https://hack aday . com/2015/08/10/arduino-based-remote-shutter-for-beme/](https://hackaday.com/2015/08/10/arduino-based-remote-shutter-for-beme/)

衣冠楚楚的黑客肖恩·哈金斯(Sean Hodgins)设计了一个精巧的小项目:一个电池供电的遥控快门(T1)。他设计它是为了和最新的 Snapchat 克隆版 [Beme](http://beme.com/) 一起使用，所有酷小孩现在都在用它。

这项服务旨在摆脱自拍文化，当你将手机贴在胸前时开始录制，这样你就看到了正在录制的东西，而不是你的手机。[Sean]想要更多的控制，所以他制作了一个遥控器，通过在接近传感器上移动伺服臂来开始记录。

他用一个 Arduino Pro Mini、一个电池、一个小型伺服系统、几个电源控制板和 SeedStudio 的一个便宜的 RF 链接制作了这个小巧的设备，所有这些都粘在一个 iPhone 外壳上。这有点粗糙(首先，伺服系统会产生一些噪音，这些噪音会在录音中被拾取)，但这是一个很好的例子，说明了如何快速组装一个原型来测试一个项目。
