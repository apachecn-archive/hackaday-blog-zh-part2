# 廉价投影仪告诉时间，入侵空间

> 原文：<https://hackaday.com/2015/08/12/cheap-projector-tells-time-invades-space/>

建造一个视频投影仪不是大多数人随便做的事情，但[多米尼克·布赫斯塔勒]不是大多数人。作为正在进行的街头艺术项目的一部分，他建造了一个相当整洁的废弃视频投影仪/床头灯/时钟设备，他称之为[大火球](http://2dom.github.io/GreatBallsOfFire/)。它是由一个诺基亚手机屏幕和一个安装在磨砂玻璃光球内的小型投影仪组成的。

构建中最有趣的部分之一是投影仪机制。Dominic 没有从头开始做一个或者拆开一个昂贵的微型投影仪，而是找到了另一个来源:一个来自易贝的便宜的汽车标志投影仪。这些设计是为了当你打开车门时，在地面上显示一个汽车制造商的标志。它配有他需要的所有部件，包括 LED 光源和光学元件。他把它拆开，用手机屏幕替换了汽车标志，创造了一个非常便宜的投影仪。它没有那么亮，但足够亮，当他把它安装在玻璃球内部时，它可以投射时间和奇怪的空间入侵者。这是一个很好的例子，说明有时寻找一个便宜的解决方案而不是免费的解决方案是有意义的:购买汽车标志投影仪为他节省了许多构建光学系统的麻烦。[Dominic]还负责这个令人敬畏的[老式电子管收音机黑客](http://hackaday.com/2013/04/16/tube-radio-husk-gets-a-web-radio-transplant/)，在那里他用互联网收音机播放器取代了旧收音机的内脏。
