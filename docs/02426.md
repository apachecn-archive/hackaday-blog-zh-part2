# 采访了一大群记者

> 原文：<https://hackaday.com/2012/12/30/prusa-interviews-a-whole-bunch-of-reprappers/>

![prusa](img/df4b04f7c315852600da1a2947ab8ce9.png)

任何时候媒体或会议想要一个 3D 打印专家，都有两个人可以求助。最著名的是 MakerBot 的[Bre Pettis]。另一个是一个叫[Prusa]的了不起的家伙，他是最受欢迎的 RepRap 的设计师，也可能是有史以来最受欢迎的 3D 打印机的设计师。他一直在利用自己的名气，采访 3D 打印领域的所有名人，并把他们都放在自己的 YouTube 账户上。

[首先是【k liment】](http://www.youtube.com/watch?v=sr-ASAqxiJg)，RepRap core dev，Sprinter 固件和 Pronterface 主机的创建者。

RepRap 社区大师[Ruben lub bes 向[Prusa]讲述了他从著名 RepRap 家](http://www.youtube.com/watch?v=HcCO_aSkhJI)那里收集的 RepRap 零件。这听起来是一个非常有趣的收藏，从历史角度来看可能非常有价值。谁知道呢，几年后它会不会像古腾堡的第一台印刷机或者蒂姆·伯纳斯·李的下一个立方体一样有趣。

[接下来是【Tonokip】](http://www.youtube.com/watch?v=issEJV9PD6M)。他开发了原始的 Tonokip 固件，所有主要 RepRap 固件都基于该固件。

[Prusa]还采访了费城黑客空间 Hive76 和宾夕法尼亚大学的[Jordan Miller]又名[jmil]。他在 RepRap 社区因 [3D 生物打印](http://hackaday.com/2012/07/02/printing-organs-with-a-3d-printer/)而出名。

总结一下最有趣的人，还有对 Slic3r 固件开发者[Sound] 的采访。在之前，我们已经看过[对【声音】的采访，这次采访继续之前的采访，谈论多台挤出机以及 RepRap 社区有多棒。](http://hackaday.com/2012/12/11/an-interview-with-the-creator-of-slic3r/)

最后，也是很不幸的，是去年九月*在世界创客大会上【Prusa】[采访](http://www.youtube.com/watch?v=YHv31QaT5Ig) *[me](http://www.youtube.com/watch?v=YHv31QaT5Ig)* 的一小段视频。*【Prusa】是 Hackaday 的超级粉丝，所以这次采访只是两个家伙的明星对决。

事实上，2012 年世界创客大会在一个地方聚集了有史以来最多的现任和前任 Hackaday 校友。对[Ian](现在是《危险原型》)，[Phillip Torrone] (Adafruit)，[Phil Burgess] (Adafruit)，现在的 Hackaday 老板[Caleb]和我的集体采访会很棒。明年我们会更加努力。
