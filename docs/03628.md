# 四个后空翻并坚持落地

> 原文:[https://hackaday . com/2013/07/21/四个后空翻和落地棒/](https://hackaday.com/2013/07/21/quadruple-backflip-and-sticks-the-landing/)

这一定是一场编码噩梦，我们甚至不要提及机械构造的细节。这张图片中心附近的模糊球是一个机器人在落地前做了一个四个后空翻。

右边是一根由木柱和一些拉线支撑的高杆。在下面的视频开始时，Hinamitetu 将机器人悬挂在吧台上，开始表演，没有任何实际动作。伺服电机在准备就绪时发出呜呜声；快速达到绕杆旋转的速度。哦，我们多么希望有更多的硬件背景信息！但我们很高兴能浏览(Hinamitetu 的)视频收藏，其中包括自由体操等其他体操项目。他甚至贴出了自己的搞笑视频，展示了高门槛并不总是美好的经历。

如果你渴望更多精彩的表演，你不会对这种走钢丝的表演失望的。
