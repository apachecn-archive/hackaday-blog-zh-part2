# 将经典 NES 控制器变成蓝牙控制器

> 原文:[https://hack aday . com/2015/01/23/turning-a-classic-nes-controller-into-a-bluetooth-controller/](https://hackaday.com/2015/01/23/turning-a-classic-nes-controller-into-a-bluetooth-controller/)

[Pietronet]和我们很多人一样，喜欢时不时地玩一些经典的游戏机游戏。他通常用 Wiimote 作为控制器在电脑上玩游戏。Wiimote 在舒适的配置中拥有大多数经典按钮。此外，它还内置了蓝牙功能，可以轻松地与您的电脑配对。[Pietronet]决定更进一步。他设法将 Wiimote 的所有内容塞进了最初的 NES 控制器中，以获得更真实的感受。

第一步是撬开[遥控器](http://hackaday.com/2007/12/09/wiimote-projector-whiteboard/ "Wiimote projector whiteboard")并找到每个按钮的衬垫。一旦确定位置，[Pietronet]就用 Dremel 把木板切割成更小的尺寸。他切掉了包含方向垫和双截棍连接器的部分电路板。接下来，他必须将非常细的电线焊接到他之前找到的每个按钮垫上。

最初的 [NES 控制器](http://hackaday.com/2013/01/28/diy-nes-controller-makes-no-changes-to-the-design/ "DIY NES Controller")的按钮数量非常有限，[Pietronet]希望尽可能少地修改最初的控制器。因此，他在 Wiimote 的同步按钮上安装了一个磁簧开关。这样，如果他需要将 Wiimote 同步到新的主机上，他可以通过将磁铁放在正确的位置来完成。这是一个不常使用的功能，因此不便之处可以忽略不计。

下一步是将原来的 NES 控制器上的按钮连接到添加到 Wiimote 的电线上。[Pietronet]留下了大部分完整的原始电路板。为了给两节 AAA 电池腾出空间，他不得不切掉一小块，但这并不影响控制器的功能。

NES 控制器的内部必须清除各种支架和塑料碎片，以便为所有额外的组件腾出空间。Wiimote 有一个 LED 指示灯，指示控制器连接正确。[Pietronet]将一个红色 SMD LED 焊接在两根细线末端的适当位置。然后将该 LED 放置在方向垫的底部左侧。透过半透明的滤光器可以看到它。这使得[Pietronet]可以看到 NES 控制器何时正确同步。

箱子又装回了一起，所有的东西都固定好了。结果是什么看起来和感觉像一个经典的 NES 控制器，只有这一个有蓝牙连接和振动电机。看看下面的视频演示，了解一下它的使用效果。
