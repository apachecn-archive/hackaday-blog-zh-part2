# Tumblemill:自制数控铣床

> 原文：<https://hackaday.com/2014/11/13/the-tumblemill-homemade-cnc-milling/>

[Jens]又名[Tumblebeer]对他自制的数控铣床 Tumblemill 进行了令人印象深刻的概述。得知[Tumblebeer]受到 Hackaday 上的专题的启发而从事电子学，这让我们感到很温暖，即使这意味着他从医学院辍学去从事电气工程。不过，我们很高兴他在追寻自己的激情，阅读他的博客可以揭示他已经走了多远:从他第一个项目中的大灾难到逐渐痴迷于制造数控设备，[Tumblebeer]一路上犯了很多错误，但这是应该的。

他的第一个迭代是使用橡胶轮作为线性轴承的 CNC 路由器。它起作用了…几乎不起作用。他的最新作品源于 Solidworks 一丝不苟的建模，主要由铝制成的移动门架设计，并升级了线性运动:这一次有点大材小用，使用了 HIWIN HGH20CA 块。[Tumblebeer]没有采购传统的主轴支架，而是选择了 LM50UU 轴承的外壳，这为他的 2.2kw 主轴提供了完美的配合和更坚固的外壳。

访问他的项目博客，了解工厂建设背后的细节，包括[漫长的升级](http://tumblebeer.com/upgrayedd/)，并在下面的演示视频中逗留，以及通过污损 Arduino 强制(也总是受到赞赏)包含 Jolly Wrencher。
