# 为智能手机制作手套

> 原文：<https://hackaday.com/2015/01/23/making-mittens-for-a-smartphone/>

对于我们这些生活在北半球略显荒凉的地区的人来说，外面冷得吓人。春天不能来得更早，如果你戴着正常的手套，智能手机就不会工作。如果你在手套里缝几根导电线，智能手机就能工作，但如果你更喜欢连指手套，那你就不走运了。没关系，因为 Adafruit 的 Becky 有很好的指南教你如何编织自己的智能手机手套。

从智力上来说，编织织物的概念相当简单——它只是交织在一起的线，形成一个灵活的床单。缝纫也相当简单。另一方面，编织很奇怪。这是一个单一的字符串绑在自己身上，形成一个三维外壳。如果你曾经拿起一对织针，你很快就会意识到发明针织的人可能是人类历史上最伟大的被遗忘的天才。幸运的是，[Becky] [有很多链接](http://www.knittinghelp.com/videos/learn-to-knit)，讲述如何编织，以及如何将纱线变成一副带有这种图案的手套。

为了让这种手套能够与智能手机配合使用，[Becky]正在使用一种不锈钢导电纱线缝合到手套的拇指和指尖。这很有效，现在无论天气有多冷，你都可以使用你的触摸屏设备。
