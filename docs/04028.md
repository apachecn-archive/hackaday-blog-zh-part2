# 梅洛曼二世

> 原文:[https://hackaday.com/2013/09/10/the-melloman-mk-ii/](https://hackaday.com/2013/09/10/the-melloman-mk-ii/)

![mello](../Images/cf85e8d17d53f31e3a034659efafff69.png)

早在 60 年代，奇怪的电子仪器风靡一时。其中最著名的是合成器时代之前的 Mellotron，它是 musique concrète 的化身。这种乐器有着极其复杂的磁带排列，允许演奏者弹奏键盘，并让任何乐器的声音从扬声器中传出。这个系统很容易出故障，在过去的五十年里，磁带技术有了很大的改进，导致[迈克·沃尔特斯]制造了他著名的基于随身听的 Mello tron 的新版本，Melloman。

这个版本是对 2009 年 Melloman 的升级。像原来一样，这个版本使用了 14 个便携式磁带播放器，每个播放器都为每个音符加载了连续的磁带。像带包含相同音符的两个八度音程，每个通道一个八度音程，每当按下一个键时，这些八度音程就被发送到输出。

旧的梅乐曼有一些改进。[Mike]没有使用晶体管，而是使用光耦合器将录制的声音发送到输出端。这个版本也干净了很多，布线看起来非常专业。至于声音演示，你可以看看下面的视频。
