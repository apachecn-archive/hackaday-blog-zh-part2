# 用真空蒸煮器酿造啤酒

> 原文:[https://hack aday . com/2015/02/07/brewing-beer-with-a-sous-vide-cooker/](https://hackaday.com/2015/02/07/brewing-beer-with-a-sous-vide-cooker/)

[Ken]为他的真空烹饪器找到了一个有趣的用途。他一直用它来帮助他的家庭酿酒。制造商不太可能打算以这种方式使用它，但作为黑客，我们并不真正关心保修。

啤酒酿造既是一门科学，也是一门艺术。这个过程中有很多变量，调整其中任何一个都会导致你的啤酒味道不同。酿造过程中有一个过程叫做糖化。糖化是将发芽的谷物浸泡在热水中以提取糖分。糖的提取量很大程度上取决于谷物浸泡的时间和水温。如果你想让你的啤酒尝起来有某种特定的味道，那么你就要确保水保持恒定的、可重复的温度。

作为一名家庭酿酒师，[肯]一直在用他的炉灶面烧水。这使得水变暖，但是为了保持温度一致，他必须不断地监控温度并相应地调节旋钮。谁想整天无所事事地做那件事？他需要能自动控制温度的东西。进入 sous vide 炊具。

[Sous vide](http://hackaday.com/2011/04/05/pid-sous-vide-slow-cooker-bon-appetit/ "sous vide cooker") 是一种烹饪方法，将食物放入密封袋中，然后浸入温度控制非常严格的水浴中。这个过程需要很长时间来烹饪食物，但结果应该是肉被完美烹饪，同时还保留了所有的水分和汁液。[Ken]认为他可以使用真空烹饪器来控制麦芽浆的温度，而不是水浴。

他的实验非常成功。他用灶台帮助将土豆泥加热到接近的温度，然后用真空烹饪器在那里对食物进行微调。[Ken]说他的 mash 能够实现 75%的效率，这正是他想要的。
