# 低温加工:定制橡胶零件

> 原文：<https://hackaday.com/2014/09/22/cryogenic-machining-custom-rubber-parts/>

为你的项目制作一个定制的一次性橡胶部件通常不是一个选择，但是[Ben Krasnow]有一个替代注射成型和铸造的方法:[加工冷冻橡胶](http://benkrasnow.blogspot.com/2014/09/crogenic-machining-custom-rubber-parts.html)。

正如[Ben]指出的，你不可能在你的研磨机上弹出一片橡胶，然后数控加工出需要的形状；钻头将推动材料而不是切割它。然而，首先冷冻橡胶可以让你雕刻到现在已经变硬的材料上。

他的初始设置包括一片上面淋有水的铝板，一个正方形的氯丁橡胶放在水面上，一股稳定的-60 到-80 摄氏度的酒精直接流到橡胶上。下面的水结冰，将氯丁橡胶固定在适当的位置。这被证明是有问题的，因为冰刀在研磨完成之前就断裂了。[Ben]后来添加了一些螺栓来夹紧工件，使铣削过程按计划完成。

这个组件下面有一个小塑料盘，用来收集流出的酒精，用一些管子把它送回去。[Ben]建议不要使用潜水泵——他最初的选择——因为潜水泵在冰冷的酒精中浸泡几分钟后就停止工作了。一个外部的磁力驱动泵解决了这个问题，尽管它需要手动灌注。
