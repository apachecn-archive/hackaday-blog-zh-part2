# 给一台老式雅达利电脑进行急需的升级

> 原文:[https://hacka day . com/2012/04/04/giving-an-old-atari-computer-a-急需升级/](https://hackaday.com/2012/04/04/giving-an-old-atari-computer-a-much-needed-upgrade/)

孩提时，[博伊西]就在 TRS-80 彩色电脑上崭露头角。在当时，这是一台非常棒的机器，拥有相对强大的摩托罗拉 6809 CPU。虽然他的 CoCo 在理论上比 Commodore 和苹果的同时代产品更强大，但与他的朋友们基于 6502 的机器相比,[ boysy]的初恋的图形和声音功能相形见绌。有点嫉妒，30 年过去了，因为现在[boysy]正在为雅达利生产的基于 6502 的机器添加 6809 微处理器。

[博伊西]他的 Liber809 项目的目标很简单:把一个 6809 CPU 放在一台[雅达利 XEGS](http://en.wikipedia.org/wiki/Atari_XEGS)上，然后让[NitrOS-9](http://sourceforge.net/apps/mediawiki/nitros9/index.php?title=Main_Page)在他的弗兰克电脑上运行，这是一个用于 TRS-80 CoCo 的类 Unix 操作系统。经过几个月的工作，【博伊西】[完成了他的目标](http://liber809.blogspot.com/2012/03/dual-cpu-mission-accomplished.html)并且更进一步:Liber809 也可以在 [Atari 1200XL](http://liber809.blogspot.com/2012/03/1200xl-success.html) 上运行。

把[博伊西]的工作放在一个角度来看，就像他从 1993 年拿了一台麦金塔电脑，让它在英特尔 486 上运行。虽然这不是一个非常准确的类比，但我们希望我们的读者能够理解让计算机运行在完全不同的处理器上所需的毅力。

休息之后，你可以看看由 AtariAge 论坛的[SLOR]编写的一个整洁的演示应用程序，展示了在为 6502 设计的机器上运行的 6809。所有相关人员的出色工作
