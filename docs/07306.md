# 加州大学戴维斯分校的研究人员利用光来消除转基因老鼠的记忆

> 原文：<https://hackaday.com/2014/10/13/uc-davis-researchers-use-light-to-erase-memories-in-genetically-altered-mice/>

就像用紫外光擦除 EPROM 中的数据一样，加州大学戴维斯分校的研究人员已经用光擦除了老鼠的特定记忆。[Kazumasa Tanaka、[Brian wilt gen](http://www.wiltgenlab.com/Projects.html)和他的同事]使用光遗传技术来测试当前关于记忆提取的想法。[光遗传学](http://en.wikipedia.org/wiki/Optogenetics)之前已经[在 Hackaday](http://hackaday.com/2014/07/18/backyard-brains-controlling-cockroaches-fruit-flys-and-people/) 上出现过。它是利用光来控制特定的神经元(神经细胞)，这些神经元在遗传上对光敏感。通过这样做，可以实时看到效果。

在他们的研究中，[Kazumasa Tanaka、[Brian wilt gen](http://www.wiltgenlab.com/Projects.html)和同事]创造了基因改变的小鼠，其激活的神经元表达[GFP](http://en.wikipedia.org/wiki/Green_fluorescent_protein)，这是一种发绿色荧光的蛋白质。这使得神经元很容易被定位，并跟踪哪些神经元对学习和记忆刺激做出反应。神经元产生了一种额外的蛋白质，使其有可能在对光做出反应时“关闭它们”。这使得研究人员能够确定哪些特定的神经元参与学习和记忆路径，以及研究当某些神经元活跃或不活跃时小鼠的行为。

动物爱好者可能不想听下面这段话。老鼠被放在笼子里后，受到轻度电击。它们被训练成这样，当它们再次被放入笼子时，它们会想起先前的电击，并会因恐惧而僵住。然而，当[海马](http://en.wikipedia.org/wiki/Brain)(大脑中的一种结构)中的特定神经元暴露在通过光纤传输的光线下时(很可能是通过每只老鼠头骨上的一个洞)，老鼠们高兴地在笼子里跑来跑去，对之前惊吓它们的电击没有任何记忆。曝光后，储存电击记忆的神经元被“关闭”。

研究人员可以从这项研究中看出，老鼠大脑的皮层、海马体和杏仁核都是相连的，并参与学习、创造和检索记忆。这一点长期以来一直受到怀疑，但光遗传学清楚地阐明了这一点。在这项研究中，我们没有看到任何关于“关闭”的神经元是否再次“打开”的明确信息；当这种情况发生时，最终观察老鼠是否恢复了任何恐惧反应将是有趣的。

尽管这项研究令人兴奋，但我们距离那些美好心灵的永恒阳光、T2 神经分析仪的出现、或者利用光遗传学治疗人类记忆障碍的场景还有很长的路要走。尽管如此，我们对结果印象深刻，并希望在未来看到更多关于记忆科学的研究。
