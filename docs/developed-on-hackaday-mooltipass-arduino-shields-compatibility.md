# 在 Hackaday 上开发:Mooltipass Arduino Shields 兼容性

> 原文：<https://hackaday.com/2014/09/16/developed-on-hackaday-mooltipass-arduino-shields-compatibility/>

一些亲爱的读者可能已经有了一个可靠的系统来记住他们每天访问的不同网站的不同复杂密码。这就是为什么他们可能没有关注黑客社区正在建立的[离线密码管理员](https://github.com/limpkin/mooltipass)。

Mooltipass 有一个特性可能会重新引起他们的兴趣:它可以连接 Arduino 护盾。在下面嵌入的视频中，您可以看到几个月前我们的开发团队设想的 Arduino 转换过程。该操作简单地包括使用小刀去除标准 Arduino 接头顶部的塑料块。我们还嵌入了一些用例及其各自的草图，这些草图可以从我们的官方 [GitHub 库](https://github.com/limpkin/mooltipass/tree/master/arduino)下载。

与堆叠几个盾牌一样，可能需要稍微调整一下，以保持 Mooltipass 和连接的盾牌的功能。因此，我们非常欢迎 Arduino 爱好者让我们知道他们对我们的设置的看法。

与此同时，你可能想订阅我们的官方谷歌群,以了解 Mooltipass 的发布日期。
