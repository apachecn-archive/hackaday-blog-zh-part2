# 爆竹弩

> 原文:[https://hackaday.com/2013/02/12/the-firecracker-crossbow/](https://hackaday.com/2013/02/12/the-firecracker-crossbow/)

[![firecracker-crossbow](../Images/7ffe2ae381ebad53b59703c7afa4d4e9.png "if firecracker-crossbows are outlawed, only outlaws will have firecracker-crossbows!")T2】](http://hackaday.com/2013/02/12/the-firecracker-crossbow/firecracker-crossbow/)

如果有两样东西是我们在[HAD]最喜欢的，那就是巨大的德国鞭炮和中世纪的武器。这个聪明的黑客给了我们两个一把玩具弩，既能点燃鞭炮，又能将鞭炮发射到安全的距离。我们没有看到一个大失败的卷轴，但是准备好运行以防故障可能也是一个好主意。

帖子中有一些该机制的图片，但在其核心，这种黑客行为包括撕毁一个烤架点火器，并将触点放入一个缩短库存的玩具弩中。这种类型的设备当然鼓励安全性。这一点在这里尤其重要，因为显然“鞭炮”在德语中大致翻译为“高爆炸药的小棍”，或者可能是“道路照明弹”。
