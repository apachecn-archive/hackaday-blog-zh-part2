# Retrotechtacular:构建 BART

> 原文:[https://hack aday . com/2013/10/01/retro tech tacular-building-Bart/](https://hackaday.com/2013/10/01/retrotechtacular-building-bart/)

有时候，从正常的电子主题后退一步，展示一个了不起的工程项目是很有趣的。本周的 Retrotechtacular 关注了两个报道海湾地区快速交通系统进展的视频。任何去过旧金山的人都会熟悉服务于该地区的 BART 火车系统。让我们来看看半个世纪前是如何建立这个系统的。

啊，打开这第一个视频的民间音乐让我们立刻会心一笑。今天的公共工程视频真正缺少的是一点民间元素。

[https://www.youtube.com/embed/QrzLrYJOnKQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/QrzLrYJOnKQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)T2】

了解如何选择 BART 线路来保护风景的美丽，并作为一种替代“奢侈”的水泥路面，用一辆车运送一名乘客。不要误解我们，我们都支持公共交通，我们自己也经常使用它。大约四分钟后，我们开始学习建造这个系统的一些工程问题，比如一个仅低于街道水平面 20 英尺的地下水位。答案是高压工作环境、特殊设备和水泥。很多很多水泥。另一个挑战是横穿海湾的海底管线部分。这些不是隧道，而是被放入一条浅挖渠道的钢管。
