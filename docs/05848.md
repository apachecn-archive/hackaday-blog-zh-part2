# 覆盆子眼看到了一切

> 原文:[https://hackaday.com/2014/04/20/the-raspberry-eye-sees-all/](https://hackaday.com/2014/04/20/the-raspberry-eye-sees-all/)

[Roman Rolinsky]想用他的树莓皮和他放在身边的 2.4 英寸液晶显示器做一些有趣的事情……所以他做了一个相当笨重的谷歌眼镜版本。

我们以前见过一些自制谷歌眼镜(T1)的例子，或者甚至是用于即时翻译的(T2)现实生活字幕眼镜(T3)，但是我们真正喜欢(罗曼的)项目的是(除了他在我们非常棒的(T4)项目托管网站(T5)上托管它的事实之外)他自己用基本组件组装了投影系统。

为了创建 HUD，他使用了一个半透明的镜子，这是他从一个名为 Horus beam splitter 之眼的 [Khet 2.0](http://www.khet.com/) 游戏中取出的——这是一个非常酷的现实生活中的益智棋盘游戏，就像那些你必须反射激光才能解决难题的游戏一样。然后，他使用一个 3 倍菲涅尔放大镜头，放置在 3D 打印外壳中的 2.4 英寸 LCD 顶部。这将图像放大并反射到直接放置在他眼睛上方的镜子上，从而实现透视显示。

休息后留下来看看树莓眼的视频吧！
