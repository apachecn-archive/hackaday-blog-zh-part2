# Roomba 4000 拆卸准备为您的 Doomba 建设

> 原文：<https://hackaday.com/2012/08/13/roomba-4000-teardown-ready-for-your-doomba-build/>

[![](img/67ee28f4360ce95e7ec5d543dea791cd.png "not a doomba")](http://hackaday.com/wp-content/uploads/2012/08/not-a-doomba.png)

除了理发，[迪诺]还花了一周的时间剪辑他拆卸 Roomba 4000 的旧视频。这些机器人只需几美元就可以在易贝买到，这使它们成为最便宜的机器人开发平台之一。

在[Dino]讲述了如何拧开盖子并拆卸 Roomba 4000 后，他又讲述了主板的布局，并查看了传感器。Roomba 上的轮子实际上是非常巧妙的技术，带有非常酷的行星齿轮系统，是下一个机器人制造的完美驱动系统。

对于一些有趣的机器人项目，有很多方法可以使用 Roombas 中的电子设备。[Dino] [用 Roomba 建造了 2/3 的全地形摇臂转向架机器人](http://hackaday.com/2011/06/22/converting-a-roomba-into-a-mars-rover/)——就像好奇号火星车一样——以及一个使用视差推进器的小型两轮室内机器人。如果你是一名 redditor，总有可能[建造一个 Doomba](http://www.quickmeme.com/meme/3qgflu/) ，但是我们认为【帕特里克】[有一个比绑在吸尘器上的刀更好的主意](http://www.quickmeme.com/meme/3qgg2q/)。

一如既往，休息后的维迪娅。
