# 使用 Jasper 创建您自己的 J.A.R.V.I.S

> 原文:[https://hack aday . com/2014/04/09/create-your-own-j-a-r-v-I-s-using-jasper/](https://hackaday.com/2014/04/09/create-your-own-j-a-r-v-i-s-using-jasper/)

托尼·斯塔克的 J.A.R.V.I.S 无需介绍。随着[舒布罗的]和[查理的]最近发布的 [Jasper，一个永远在线的树莓 Pi 语音控制开发平台](http://jasperproject.github.io/)，你也可以开始制作你自己的 J.A.R.V.I..

[Shubhro]和[Charlie]都是普林斯顿大学的本科生，他们决定将他们的语音控制项目开源(代码可在 [GitHub](https://github.com/jasperproject) 上获得)。Jasper 是建立在便宜的现成硬件上的，这使得它非常容易上手。你真正需要的是一个带麦克风和扬声器的互联网连接[树莓派](http://www.raspberrypi.org/)。只需安装 Jasper，并开始使用内置功能，允许您与 Spotify、脸书、Gmail、knock knock 笑话等进行交互。休息后请务必观看演示视频！

使用易于使用的开发人员 API，您可以毫不费力地将 Jasper 集成到任何现有的 Raspberry Pi 项目中。我们可以看到 Jasper 集成了无线麦克风和扬声器，可以在家里的任何地方进行高级语音控制。多么伟大的工程！感谢[Shubhro]和[Charlie]让这个开源。
