# 3D 打印的尤克里里

> 原文:[https://hackaday.com/2013/12/11/the-3d-printed-ukulele/](https://hackaday.com/2013/12/11/the-3d-printed-ukulele/)

![uke](../Images/45dcca13b5ab070c198e75784130c242.png)

每个人都喜欢的 slic3r 的创造者——[亚历山德罗·拉内鲁西]——一直在努力工作，测试他的 3D 建模技能。他创造了一把几乎完全 3D 打印的尤克里里琴。除了琴弦和调音栓，uke 上的所有东西都来自 MendelMax 3D 打印机，完全没有任何支撑材料。

在视频中，【Alessandro】和 uke virtuoso【Jontom】展示了这种乐器是如何组装在一起的，以及它能发出多好的声音。uke 的主体由两部分组成，琴颈由三部分组成，包括琴头和指板，所有这些都用令人惊讶的传统方法组装在一起。燕尾榫接头将琴颈连接到琴身，舌榫状接头将琴头固定到琴颈上。

[Allessandro]认为所有 uke 部件的打印时间约为 120 小时，塑料价值约为 20 欧元。就四弦琴而言，这听起来和普通乐器一样好，但[Jontom]说动作有点高。我们猜测，这就是文件被发明的原因。
