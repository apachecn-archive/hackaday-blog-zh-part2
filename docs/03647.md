# 树莓派接管了家庭音响和灯光

> 原文:[https://hackaday . com/2013/07/24/home-audio-and-lighting-take-over-by-the-raspberry-pi/](https://hackaday.com/2013/07/24/home-audio-and-lighting-taken-over-by-the-raspberry-pi/)

![687474703a2f2f7777772e6265616b61626c652e636f6d2f696d672f707265766965772d312e706e67](../Images/e6f8cdbf217a77e57f42125d197fa8b8.png)

我们开始看到使用 Raspberry Pi 板作为家庭自动化基础的势头越来越大。这款来自[Iain Hamilton]的最新产品通过一个单一的网络界面结合了[灯光和音频控制。他的前端是作为 RPi 板上的网页运行的。它甚至包括移动设备和计算机的独立布局，以便最大限度地利用屏幕空间。](https://github.com/beakable/BeakPi)

界面顶部的三个按钮允许他配置设置，并在灯光和音频控制之间切换。这个音频控制屏幕向 Pi 上运行的 Spotify 客户端发出命令。Mopidy 包负责几乎所有的事情(就像我们在其他单板计算机 Spotify 服务器上看到的一样)。未来的迭代将提供其他流媒体服务，如 SoundCloud。[Iain 的]家庭照明系统使用 X10 模块进行控制。他用 USB 加密狗来控制那个系统。
