# APRS 追踪系统让你的气球飞起来

> 原文:[https://hack aday . com/2015/03/24/aprs-tracking-system-flys-your-balloons/](https://hackaday.com/2015/03/24/aprs-tracking-system-flies-your-balloons/)

正在寻找一种方法来跟踪您的高空气球，但不想通过蜂窝网络发送数据？[Zack Clobes]和 Project Traveler 的其他人可能正好有适合你的东西:[一个位置报告板，它使用自动分组报告系统(APRS)网络](http://projecttraveler.org/arduinotrack-controller/)来报告位置数据，并以屏蔽的形式轻松安装在 Arduino 上。

该项目基于 Atmel 328P，它报告位置数据所需要的只是一个小天线和一块电池。对于那些不熟悉 APRS 的人来说，它使用业余无线电频率来发送数据包，而不是像 GSM 网络那样的东西。APRS 非常强大，使用它的设备可以发送 GPS 信息以及短信、电子邮件、天气报告、无线电遥测数据和无线电测向信息，以防 GPS 不可用。

如果这种位置报告能力对你来说还不够，该项目也可以作为一个盾牌，这意味着更多的数据线可用于其他事情，如监控传感器和驱动伺服系统。所有这些都在一个不依赖于蜂窝网络的小型轻量级封装中。如果你想尝试一下，可以在项目网站上找到所有的原理图和其他信息，但如果你确实需要蜂窝网络，[这可能更符合你的风格](http://hackaday.com/2015/02/25/spark-goes-cellular-with-the-electron/)。请务必在休息后观看视频！
