# 复古:建立哈蒙德管风琴音调

> 原文：<https://hackaday.com/2015/06/30/retrotechtacular-building-hammond-organ-tones/>

这是一部由哈蒙德风琴公司制作的短片，旨在教育和说服潜在消费者。现在我们确信哈蒙德风琴是最棒的，原因很简单:音调发生器赋予了它们独特的哈蒙德声音，并且在制造的每一步都非常小心。

哈蒙德风琴有 91 个独立的电磁音轮组件。每一个都根据旋转磁盘边缘的波纹和它在电磁体前旋转的速度产生特定的频率。通过使用拉杆来叠加和声，一个管风琴手可以建造出丰富的声音之墙。

哈蒙德不遗余力地追求卓越。所有变压器都在内部缠绕，然后用蜡密封，以防止受潮。每个音轮都经过严格的公差切割、交叉检查，并由音频专家进行验证。声音发生器的组装和微调非常仔细，哈蒙德声称它们再也不需要调整了。

这种程度的关注并不局限于乐器的内部。不，橱柜部门同样一丝不苟。只有最优质的木材经过手工精心干燥、切割、打磨和上漆，然后打磨至高光泽。在出厂之前，每个哈蒙德风琴都要经过严格的检查，并在隔音室进行性能测试。
