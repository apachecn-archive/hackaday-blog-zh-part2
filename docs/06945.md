# 3D 打印凹凸键

> 原文：<https://hackaday.com/2014/08/29/3d-printed-bump-keys/>

如果你有合适的工具，通过一扇锁着的门是很容易的。这只是一个知道如何调整内部的引脚到一个均匀的水平，同时转动机械装置，当一切都完美到位。这就是万能钥匙的好处。你永远不必看到真正的钥匙或它看起来像什么。只需在钥匙背面轻轻一击，用力撞击，锁就能神奇地被打开。

像这样的开锁工具可以花几美元在网上订购，或者像[Jos Weyers]和[Christian Holler]在最近的一篇连线文章中展示的那样，或者你可以在家里打印自己的。这些 3D 打印钥匙的视频(可以在下面观看)试图证明一个人可以用塑料打开一扇门，这让我们有点惊讶，因为边缘似乎会马上断裂。但事实证明，一个薄的塑料凸起钥匙是可以制造的，而且确实可以使用。虽然不确定这些钥匙能使用多久，但有时你真正需要的是在试图打开一个特定的、棘手的锁时一次性使用。

正如文章所述，“韦耶斯和霍勒并没有试图教小偷和间谍一种闯入高度安全设施的新技巧；相反，他们想警告制锁者 3d 打印撞击钥匙的可能性，以便他们可以防范它。“虽然这些信息是针对制锁者的，但我们发现我们的黑客读者也发现这些数据很有用。定期举行开锁活动的黑客空间的组织者可能想打印他们自己的钥匙，并教授以安全为中心的课程。在教育公众锁是如何真正工作的方面，它的用途是无限的。

我们通过 Shapeways、i . Materialise 和 Thingiverse 搜索 3D 凹凸键 STL 文件，结果人手不足。因此，如果你决定创建自己的版本，一定要把模型上传到某个地方，这样其他人就可以从中学习了！
