# 如何自制压电扬声器

> 原文：<https://hackaday.com/2012/09/15/how-to-make-your-own-piezoelectric-speaker/>

从音乐贺卡到汽车音响系统中的高音喇叭，压电扬声器无处不在。制作自己的压电扬声器实际上非常容易，[正如[Steven]在更换了时钟收音机的扬声器后向我们展示的](http://rimstar.org/equip/piezoelectric_crystal_speaker_rochelle_salt.htm)。

压电扬声器需要一个具有压电特性的小晶体，所以这是[Steven]制作罗谢尔盐晶体教程的完美后续。在他的罗谢尔盐晶体上贴上两条铝箔后，[史蒂文]拿起之前连接到时钟收音机扬声器的电线，将它们连接到晶体上，打开了收音机。当连接到锡罐上时，新制造的压电扬声器产生了一点声音，但结果并不令人印象深刻。

为了提高自制扬声器的声音输出，[Steven]需要增加压电扬声器的电压。起初，他尝试了门铃变压器，效果稍好，但当他使用从微波炉中取出的变压器时，产生了更多的声音。

在试验了他的微波变压器和罗谢尔盐之后，[Steven]转而研究烧烤炉和打火机中的压电元件。这些自制的扬声器比他以前使用的大块 Rochelle salt 清晰得多，并且令人惊讶地产生了与 Radio Shack 上的商用压电扬声器[Steven]相同的音频质量。
