# NFC 标签控制你家的照明

> 原文:[https://hack aday . com/2013/02/22/NFC-tags-control-your-home-lighting/](https://hackaday.com/2013/02/22/nfc-tags-control-your-homes-lighting/)

![nfc-controlled-home-lighting](../Images/cdcb2db3acde1c3310a48ca8ba2db90f.png)

这里有一个不需要你在初始设置后考虑的家庭照明技巧。它不需要用户启动应用程序并选择照明状态，而是使用 NFC 标签来选择照明配置。标签可以放在房子的不同地方，这样穿上外套的时候把手机放在门边的桌子上，就可以关掉房子里的所有东西。当然，你需要先爬后走，所以现在这个概念验证只开关台灯中的菲利普斯灯泡。

这种灯泡与 Ninja Blocks 系统兼容，但 Ninja Block 或带以太网屏蔽的 Arduino 可以用来切换任何你想要的东西。忍者客户端代码是系统不可分割的一部分，这就是为什么硬件方面需要与平台相关联。还使用了{X}服务上的[，它在您的 Android 手机和家庭自动化硬件之间架起了一座桥梁。一旦这一切就绪，就只需要按照你的意愿对 NFC 标签进行编程了。不要错过跳跃后剪辑中的演示。](https://www.onx.ms)
