# Fubarino 竞赛:拿着复活节彩蛋的变形人

> 原文:[https://hack aday . com/2013/12/15/fuba Rino-contest-a-shifter-with a-Easter-egg/](https://hackaday.com/2013/12/15/fubarino-contest-a-shifter-with-an-easter-egg/)

![shifter](../Images/5e6de9d8a6e0fae6d3283dec955b0df1.png)

几个月~~几年~~前【韦斯】在黑客论坛上分享了他的[的一个项目(是的，我们确实有一个论坛，你应该去看看)。他为一些模拟赛车创造了一个移位器，极大地改善了任何使用键盘的系统。](http://forums.hackaday.com/viewtopic.php?f=3&t=327)

移位器是由一些废木头、一块切割成 H 门形状的砧板和几个拱廊微型开关制成的。一个巨大的螺栓锁在每个档位的几个橱柜扣上，一组连接到 USB 游戏手柄的微型开关告诉[韦斯的]虚拟汽车他应该挂什么档。

这是一个很棒的构建，但是因为这是我们的 [Fubarino 竞赛](http://hackaday.com/2013/12/03/new-contest-win-one-of-20-microchip-fubarino-boards/)的参赛作品，【Wes】需要在某个地方放一个带有 Hackaday URL 的复活节彩蛋。为了解决这个问题，【韦斯】[用 Teensy 2.0 升级了电子设备](http://forums.hackaday.com/viewtopic.php?f=3&t=327&p=16959#p16959)。当档位换到第一档、第三档、第三档和第七档时，青少年会闪烁莫尔斯电码的网址，并打开一个网页浏览器，加载黑客日。

这不仅是一个伟大的建设，它也是一个非常，非常微妙的复活节彩蛋为我们最喜爱的网站。下面彩蛋的演示。
