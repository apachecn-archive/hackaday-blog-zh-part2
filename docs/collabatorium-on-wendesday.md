# 周三的 Collabatorium

> 原文：<https://hackaday.com/2015/07/07/collabatorium-on-wendesday/>

我们的下一个协作馆就在眼前。周三，我们将开放黑客频道，就您正在构建的内容展开另一场精彩的讨论。展示你的作品，看看别人在做什么，寻求帮助，提供你的技能，等等。一切从欧洲中部时间 19:00(UTC+2)开始。

欢迎所有人，但你需要被添加为 [Hackaday Prize 黑客频道](https://hackaday.io/project/5373-hackaday-prize-hacker-channel)的合作者。现在就去那里，点击左边栏底部的“请求加入这个项目”链接。

上周我们遇到了一场爆炸，一大群黑客在我们的虚拟黑客空间里闲逛。本周我们改变了时区，这样世界不同地方的黑客都可以参与进来。如果你在欧洲，这应该是一天工作结束的足够晚了。请务必加入并代表您所在地区的黑客社区。周三见！
