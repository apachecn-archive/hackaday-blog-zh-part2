# 激光切割时钟将您的 CAD 工具踢到路边，选择 Python

> 原文：<https://hackaday.com/2015/03/24/laser-cut-clock-kicks-your-cad-tools-to-the-curb-and-opts-for-python/>

在一个除了#6-32 螺栓、薄片和丙烯酸薄片之外没有其他库存硬件的世界里，[Lawrence Kesteloot]承担起自己的责任，设计并制造了一个激光切割摆钟。没有昂贵的 CAD 程序？没问题。在一个许多花哨的 CAD 工具可以自动生成齿轮模型的世界里，[Lawrence]回到基本原则，编写脚本来自动生成齿轮轮廓。此外，这些脚本不仅可以导出整个模型的 SVG 文件以方便激光切割，它们还可以使用 Javascript 在浏览器中渲染 3D 模型[。](http://www.teamten.com/lawrence/projects/acrylic-pendulum-clock/visualization.html)

鉴于选择的材料很少，整个项目是一个爱的劳动。甚至视频(休息后)也掩饰了对轴承、螺栓孔间距和时间敏感齿轮比的精心选择，其中每一项在其他 CAD 程序中可能都是一个简单的宏，在这种情况下，[劳伦斯]需要自己添加。

最后，整个项目是开源的，可以在 Githubs 上下载。不是每天我们都可以用简单的命令行咒语为自己建造一个摆钟

```
make cut
```
