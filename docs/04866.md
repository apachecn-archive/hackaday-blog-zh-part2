# 全尺寸乐高汽车可以达到 30km/h！

> 原文:[https://hack aday . com/2013/12/21/full-size-Lego-car-can-hit-30k MH/](https://hackaday.com/2013/12/21/full-size-lego-car-can-hit-30kmh/)

[Steve Sammartino]是墨尔本的一名企业家，他有一个想法:是否有可能设计并制造一辆实用的全尺寸乐高汽车？

他发了一条推特试图为这个项目筹集资金:

> 任何人有兴趣在一个令人敬畏的项目上投资 500-1000 美元&一个世界第一的 tweet me。大约需要 20 名参与者…[#启动](https://twitter.com/search?q=%23startup&src=hash)

不是一个，不是两个，而是四十个澳大利亚人承诺出资启动这个被称为[的疯狂想法。筹集到资金后，[史蒂夫]和[劳尔·瓦伊达]购买了超过 50 万块乐高积木，并在[劳尔]居住的罗马尼亚开始建造。](http://www.superawesomemicroproject.com/)

现在，在你太兴奋之前，这辆车并不是“完全”由乐高制成的。它的特点是真正的轮胎和一些精选的承重元素。也就是说，**整个引擎**完全由乐高制成。它配备了四个轨道发动机，总共使用了 256 个活塞。他们测试的最高速度约为 20-30 公里/小时——可能会更快，但他们不想冒灾难性失败的风险。

自完成以来(花了近 18 个月建造)，它被运回墨尔本的一个秘密地点，但该团队制作了一个展示该项目的优秀视频。休息后留下来看看你童年的梦想变成现实。
