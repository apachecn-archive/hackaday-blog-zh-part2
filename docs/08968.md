# 让开波士顿动力公司，ATRIAS 来了！

> 原文:[https://hack aday . com/2015/05/06/move-side-Boston-dynamics-atrias-is-coming/](https://hackaday.com/2015/05/06/move-aside-boston-dynamics-atrias-is-coming/)

ATRIAS 刚刚在草地上迈出了第一步，标志着这个大学机器人项目取得了令人印象深刻的成就。

ATRIAS 由俄勒冈州立大学建造，是一个两足机器人，其名称在 jest 中代表“假设机器人是一个球体”。这是一个古老的物理学笑话，描述了如何将复杂的科学模型简化为最简单的形式来进行计算——但是~~有时~~(总是)使其在现实中的应用成为一个挑战…

我们相信你们都记得 BD 的大狗和它令人印象深刻的扔煤渣砖的能力，但是记住，它有四条腿和一条尾巴，或者它是一个躯干，一只手臂？另一方面，心房是对人类和我们用两条腿走路的独特能力的真正威胁。他们为它制造的机制非常聪明。

简而言之，它使用弹簧加载的倒立摆来平衡和行走。

[![2013-IllustrationTemplate](../Images/d9537c07a0c2858dfb7c795f931028cf.png)T2】](https://hackaday.com/wp-content/uploads/2015/05/12385_sm_walking-rob.jpg)

这样一个系统的美妙之处在于效率。就像再生制动对电动汽车效率的影响一样，弹簧减少了运动过程中所需的能量，就像我们自己的肌肉通过弹性使用肌腱和组织来保存能量一样。几年前，机器设计团队对此有一个完整的解释。

总之，这是 ATRIAS 在户外迈出的第一步。
