# 手持式数控刳刨机

> 原文:[https://hackaday.com/2015/07/26/a-handheld-cnc-router/](https://hackaday.com/2015/07/26/a-handheld-cnc-router/)

在过去的几年里，手持式路由器的技术水平已经被我们遗忘了。那是 2012 年在 SIGGRAPH 的，我们在去年的 Makerfair】赶上了它[。现在，它正准备生产。](http://hackaday.com/2014/05/21/live-look-at-taktia-augmented-power-tool-and-carbide-3d-mill/)

最初被称为 Taktia 的[整形路由器](http://shapertools.com/)看起来很像一个普通的手持路由器。然而，这款路由器很智能，它能够查看用计算机视觉设计的胶带标记的工件，并根据用户的移动方式*轻微*重新定位刀具。一个简单的描述并不能公正地描述这个工具，所以[来看看 Shaper 团队最近上传的视频](https://www.youtube.com/watch?v=jxQ_NH4bj9o)。

随着用户在工件上移动牛头刨床和电机移动刀头，该工具能够快速、轻松、准确地进行精确切割——木制齿轮和美国轮廓。切割*任何*形状就像将一个文件加载到成形器中，在触摸屏显示器上调用该文件，然后打开切割机一样简单。在桌子周围移动路由器，整形器会处理剩下的事情。

精确度，至少在早期版本中，据说是百分之一英寸的数量级。这对于木材来说已经足够好了，就像这件非常有趣的细木工作品一样，用传统工具很难做到。下面视频。
