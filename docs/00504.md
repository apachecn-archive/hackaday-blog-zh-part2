# 热熔胶附件可能是 T-1000 流动金属的前身

> 原文:[https://hackaday . com/2012/05/23/hot-glue-attachments-may-be-predecessor-to-the-flow-metal-of-the-t-1000/](https://hackaday.com/2012/05/23/hot-glue-appendages-may-be-predecessor-to-the-flow-metal-of-the-t-1000/)

![](../Images/5806bee2c7d93c2e2b6ff616f9e8c80c.png "hot-glue-arms")

T-1000 是来自 T2(第二部《终结者》电影)的变形机器人。它如此神奇，因为它可以呈现任何东西的形状和纹理；人类，穿刺武器，无生命物体。这个机器人连比都比不上，除了一个小特质。当它需要一个工具时，[它可以把它作为自己的附属品](http://spectrum.ieee.org/automaton/robotics/diy/this-robot-makes-its-own-custom-tools-out-of-glue)来构建。这实际上只不过是用 3D 打印机制作工具。然而，正常的四四方方的基础设施不见了。

打印头安装在单个机器人臂上，使用热熔胶来打印工具，以便粘在构成机器人臂商业端的板上。在这种情况下，机器人需要运输一些水。它放下盘子，使用热熔挤出机在盘子上打印一个杯子，然后再次拿起它，用它将水从一个碗转移到另一个碗。你可以在折叠下方的视频剪辑中看到这一切。

当然，这只是一小步。但是热熔胶棒重量轻，不需要太多的能量就能熔化。这是作为便携式工具商店的完美组合。
