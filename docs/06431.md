# 一个微小的机器人家庭

> 原文:[https://hackaday.com/2014/06/29/a-tiny-robot-family/](https://hackaday.com/2014/06/29/a-tiny-robot-family/)

早在 80 年代末 90 年代初，许多年轻的电子爱好者就开始接触光束机器人——由逻辑芯片和回收的随身听制成的小机器人，它们从关于神经网络和 80 年代人工智能复兴的论文中撕下了一页。20 年后，第二次人工智能复兴从未发生，因为一代天才程序员认为，最好地利用他们的智力是在互联网上销售广告。不过，我们有了 Arduino，这个小小的机器人家族是老式光束机器人数字生活的精神继承者。

微型机器人家族是[shlonkin]不断增长的小型自动驾驶汽车的集合，它们通过传感器感知世界，并以不同的行为行动。它们都包含一个 ATtiny85、一个小电池、两个电机、至少一个光电晶体管和一个 LED。一个机器人的左右眼朝下，可以充当线随从。另一个机器人的身体周围有一组发光二极管，允许它向各个方向的其他机器人发出信号。该项目的目标是创造出一系列能够与环境互动和相互互动的微型机器人。下面这条线的跟随者视频。
