# [弗兰]和[汽车]的恐龙呢

> 原文:[https://hackaday.com/2014/07/02/fran-bils-dinosaur-den/](https://hackaday.com/2014/07/02/fran-bils-dinosaur-den/)

我想是我把超级棒的[弗兰·布兰奇]介绍给哈卡戴的古怪老头和难民准将[比尔·赫德]的。因此，我全权负责[【Fran】和【Bil】的恐龙巢穴](http://www.youtube.com/watch?v=JCuRoVYb0dE)，他们正在制作的新的 YouTube 系列。

本周节目的亮点是一个非常复古的 Rubicon 镜式电流计。这是精确测量电压的第一种方法，工作原理有点像类固醇上的普通面板表。在你的骨存量面板米，一个小线圈移动指针显示你正在测量的任何东西。在镜式电流计中，线圈扭曲了连接到镜子的导线。通过在这个镜子上照射光线，并让反射光束在其他几个镜子周围反弹，由线圈控制的镜子角度被大大放大，从而实现非常非常精确的测量。它非常敏感，一个柠檬电池的输出超出了规模，这一切都发生在两个恐龙展示这项技术之前。很棒的东西。
