# DIY 品客罐音箱

> 原文：<https://hackaday.com/2013/12/01/diy-pringles-can-speaker/>

寻找一个有趣的周末项目？如何[用一些非常基础的材料从头开始制作自己的扬声器](http://www.instructables.com/id/Pringles-Can-Speaker-1/)？

[Go Repairs]为 [Instructables](http://www.instructables.com/member/Go+Repairs/) 制作了一系列操作视频，其风格非常令人想起 90 年代的经典艺术攻击——非常清晰、简洁，他甚至还有口音！

该项目只需要你在上面的照片中看到的东西。盖子形成了扬声器的一个简单的塑料锥体，磁铁是核心，使用一些纸、胶带和漆包线就可以构建一个非常基本的音圈。不要期望它会有惊人的音质，但是对于初级黑客来说，它看起来肯定是一个有趣的项目，因为它不需要任何花哨的工具或设备！

休息后留下来看看视频——它是否也让你想起了艺术攻击？
