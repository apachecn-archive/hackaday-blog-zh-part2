# 今晚是黑客与黑客日写作团队的聊天

> 原文：<https://hackaday.com/2015/08/12/tonight-is-hacker-chat-with-the-hackaday-writing-crew/>

太平洋时间今晚 6 点(UTC-7)是 2015 年黑客日奖报名截止日期前最后一次黑客对话。加入我们到[谈论所有硬件](https://hackaday.io/project/5373-hacker-channel)。那些需要最后建议的人，或者正在为史诗般的周末黑客马拉松寻找团队成员来引导你的获胜参赛作品的人，这就是找到它的地方。值得参赛…我们给每个参赛选手一件限量版衬衫..一个执行良好的想法可能会让你进入下一轮！

今晚和我一起参加庆祝活动的有理查德·巴古里、凯文·戴迪、比尔·赫德、克里斯蒂娜·帕诺斯和阿尔·威廉姆斯。我们经营这些东西有点像西部拓荒。只有一点结构，但大多数情况下一切正常。至于结构，[如果你想让你的项目成为讨论的话题之一，就把它加到这张纸上](https://docs.google.com/spreadsheets/d/1cE_a50wf0Ja_5_10qttUY2UyH7fXeIEj3UK71FyLIao/edit#gid=0)。除此之外，互相优秀的同时分享自己的知识和观点。今晚见！
