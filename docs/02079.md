# 相机机架安装在车库门轨道上

> 原文:[https://hack aday . com/2012/11/09/camera-gantry-rides-on-garage-door-tracks/](https://hackaday.com/2012/11/09/camera-gantry-rides-on-garage-door-tracks/)

![](../Images/29daa6011099964e8813cfeacfb7911e.png "camera")

对于我们在 Hackaday 上的许多车库和车间视频，我们很惊讶我们没有早点看到这一点。

[托德]在他的车库商店制作了一堆视频，但使用三脚架是一种痛苦；他总是被他的照相机装置绊倒，这通常总是碍事。他的解决方案是[将他的摄像机安装在高架门架上](http://www.toddfun.com/2012/11/07/camera-gantry/)，利用车库大门未使用的轨道在车间内移动摄像机。

建造始于[托德]拿起他的三脚架，为它制作一个悬在半空中的安装板。这会让相机上下颠倒，所以[托德]还用 1/4-20 的螺栓做了一个 90 度的支架来固定相机。

建造的实际门架部分相当巧妙。首先，[托德]找了一根方管，长度与他的两条车库门轨道之间的距离相同。他为这种管子制作了一辆装在六个脚轮上的卡车，然后将这种管子安装在车库门的轮子上。

结果是[Todd]可以在车库门轨迹范围内的任何地方移动他的摄像机，包括他的工作台和焊接区域。一个巧妙有用的建筑，肯定会为他的 vlog 类型的东西提供一个稳定的平台。
