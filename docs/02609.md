# 用非常旧的喷墨墨盒制作一台生物打印机

> 原文：<https://hackaday.com/2013/01/25/build-a-bioprinter-from-very-old-inkjet-cartridges/>

这一栏信息是用大肠杆菌打印的。这种细菌更广为人知的名字是大肠杆菌，如果它进入我们的食物，会引起很多问题。但相对大小和这种菌株被设计成在黑暗中发光的事实使它成为生物打印的完美候选人。我们发现更有趣的是[它是用黑客入侵的喷墨打印机和计算机部件](http://www.instructables.com/id/DIY-BioPrinter)打印出来的。

这种技术有合法的用途。但是这个项目更多的是为了让人们了解这个方法以及它有多简单。对我们来说，近距离观察现代喷墨打印头是最有趣的。事实证明，普通墨盒的分辨率太高，无法正常工作。为了在如此小的区域内获得如此多的点，喷嘴开口最终太小，大多数生物材料无法通过。内置硅胶技术的过滤器也有问题。

解决方案是使用[墨盾](http://hackaday.com/2011/09/12/inkjet-print-head-driver-shield/)来驱动非常旧的打印机的墨盒。这使得团队可以用 Arduino 来控制墨盒，使调整材料的沉积方式变得非常简单。他们将墨盒支架(使用 HP Deskjet 500 墨盒形式的数十年老技术)安装在光驱的滑轨上，并从那里开始。
