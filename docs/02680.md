# 元素展示也是一面 LED 墙

> 原文:[https://hack aday . com/2013/02/04/elemental-display-is-also-a-led-wall/](https://hackaday.com/2013/02/04/elemental-display-is-also-a-led-wall/)

![elements](../Images/d95f53943027cd3196bbdcd5ad42855c.png)

[丹]是一个元素收集者，他的袜子被铋晶体和溴的橙色蒸汽弄掉了。当然，每个元素收藏家都需要一个合适的展示柜，由于元素周期表的想法是陈词滥调，[丹]决定建立一个元素展示，也是一个真正令人敬畏的 LED 墙。

和大多数人一样，这个建筑开始时用了几张胶合板和 T2 的 120 个丙烯酸搁板来摆放丹收藏的每一件物品。当[丹]的好友[比尔]被叫进来让展示[变得更有趣](http://www.billporter.info/2013/02/02/the-elemental-illuminator/)时，真正的奇迹发生了。

在每个亚克力货架后面是一个由三个 LED 组成的 LED 条，元素周期表的每个部分都有不同的颜色。120 个独立的搁架单元分为 16 个搁架组，每个搁架组由定制的 LED 驱动板驱动。这些驱动板通过电话线连接到主 Arduino，并很好地利用了[一个非常简洁的 TCL5940 Arduino 库](http://code.google.com/p/tlc5940arduino/)。

元素显示有几个选项；全开，闪烁，苹果的“呼吸”模式，和一个图形情商，如休息后的视频所示。
