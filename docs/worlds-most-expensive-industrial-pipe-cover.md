# 世界上最昂贵的工业管道盖

> 原文：<https://hackaday.com/2014/08/14/worlds-most-expensive-industrial-pipe-cover/>

当你将 1000 个 RGB LEDs 塞进一个项目中时，并不难获得 HaD 的关注。事实上，[这种时髦的水晶管](http://www.designboom.com/design/asylum-crystal-led-pipe-light-installation-08-11-2014/)有超过 9000 个！

这个相当独特的项目是私人委托来掩盖一个新建筑中暴露的管道。对我们来说，这似乎有点矫枉过正，但精神病院的工程师们非常兴奋地交付了这个项目。管道覆盖物具有 2912 个 RGB LEDs 控制模块，由安装在装置中的专用 Linux PC 控制。创建了一个网站，允许客户从任何计算机或移动设备控制灯光。

每个水晶碎片都单独粘在表面(大约有 3000 个！)使用 UV 硬化胶。这是一个痛苦而缓慢的过程，但结果是值得的，因为它看起来像是从超人的孤独堡垒中走出来的！
