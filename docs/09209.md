# 这些是你一直在找的机器人控制器

> 原文:[https://hack aday . com/2015/06/08/这些是你一直在寻找的机器人控制器/](https://hackaday.com/2015/06/08/these-are-the-droid-controllers-youve-been-looking-for/)

当我四年级的时候，我们的老师宣布我们有一位特殊的客人从“很远很远的地方”来拜访我们。当我们涌出教室，来到院子里时，我的下巴撞到了地板上——是 R2D2！机器人开始四处移动，发出像电影里一样的噪音。我不敢相信我所看到的。R2D2 是真实的，他就在我面前！(我幼小的心灵得出结论，如果 R2D2 是真实的，那么所有的《星球大战》都是真实的。)我只好转过身去看朋友们的反应。不幸的是，就在那个时候，我看到一个中年人，拿着一个遥控发射器，天线伸出来，站在后台操作控制器。叹气。R2 不是真的——它只是一个遥控机器人。我成为绝地的梦想永远破灭了。

R2 建筑者俱乐部的克里斯·詹姆斯一直在努力制作一对“隐形遥控”控制器，以保持 R2D2 的魅力不被破坏。这些双操纵杆、3D 打印的手持设备可以在您操作时轻松隐藏在您的手掌或宽松夹克的前口袋中。这些微型控制器装载了各种功能，使用 XBee 无线电与 droid 内部的接收器和定制 PCB 通信，进而可以控制几十个伺服系统、电机、声音回放等。因为一些 R2D2 版本会有很多很多功能，而不是每个功能都有一个按钮，[Chris]已经在单元中编程了手势控制，这样两个控制器和可以控制几十个预编程的动作。[Chris]还没有最终完成设计——他仍然称之为“beta”版本，但是到目前为止他的[文档非常出色](http://artoo-detoo.net/stealth-rc/CJ_Stealth_RC_Reference_Guide_v1.1.pdf)(PDF)——是我们见过的最好的文档之一。
