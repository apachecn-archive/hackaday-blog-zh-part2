# 高级设计项目供应无限饮料

> 原文：<https://hackaday.com/2015/04/26/senior-design-project-serves-infinite-drinks/>

如果你有创造力，你可以让你的激情项目计入大学学分。不知何故,[InfinityTable]设法将这个名为 BarT 的 infinity 酒保建筑用作高级设计项目。

这里有很多东西，首先是 30 英寸 x30 英寸的橱柜，因为前面板中间有一个方形切口，所以需要一些定制的镜面玻璃。两个镜子相互面对，中间是一条发光二极管，构成了建筑的“无限”部分。这很受欢迎，但通常只有镜子和灯光。在这种情况下，特殊切口是用于玻璃的小房间。把它放在那里，剩下的部分会为你调制出美味的饮料。

后视镜后面的外壳中有第二个腔室。这里存放着混合饮料的成分。原料从 1.25L 塑料瓶分装。该构建的额外特殊部分是，由于它是一个高级项目，所有驱动电路都使用自卷板。
