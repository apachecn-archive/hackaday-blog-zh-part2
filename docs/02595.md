# 死主板墙隐藏了一个 LED 字幕

> 原文:[https://hack aday . com/2013/01/23/dead-motherboard-wall-are-hides-an-led-marquee/](https://hackaday.com/2013/01/23/dead-motherboard-wall-are-hides-an-led-marquee/)

![dead-motherboard-led-marquee](../Images/dadb5515dfa1b4a9635323e5af0c2d07.png)

[Jared]是一名计算机技术人员，所以他可以毫不费力地拿到坏掉的主板。看起来他倾向于保存更有趣的颜色，并最终找到了废物的用途。他建造了这个墙壁艺术，同时也是一个 LED 天棚。

他想出了这个尺寸和形状——18 英寸乘 48 英寸——因为它和一张中密度纤维板很相配。该轮廓允许由排列成 7 高 21 宽的 2”平方像素组成的网格。顶部和底部的行将作为灯光的框架，这仍然留下了显示字符所需的五个像素。从那里，他开始连接 LED 阵列，这在休息后的剪辑中的测试阶段显示。每个像素都由[Jared]在台锯上制作的椴木框架隔离。这个项目是通过将主板切割成合适的尺寸，然后用螺杆和螺母将它们安装上来完成的。电路板块是不透明的，但它们比网格小，所以 led 会使边缘发光。

这让我们想起了几年前用来模仿彩色玻璃的主板。
