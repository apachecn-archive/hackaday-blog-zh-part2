# 欺骗你的方式通过原来的波斯王子

> 原文:[https://hack aday . com/2012/06/28/骗过你的原波斯王子/](https://hackaday.com/2012/06/28/cheat-your-way-through-the-original-prince-of-persia/)

![](../Images/44a668edf4d61b5d1fcec8471749f4a8.png "cheating-on-prince-of-persia")

如果你没有耐心玩完原版的波斯王子，也许你应该作弊？[模糊]通过将[波斯王子作弊软件构建到 Java 苹果电脑模拟器 JACE](https://sites.google.com/site/brendanrobert/projects/jace)中，使这变得简单了。

他展示了模拟器和他在休息后添加到视频中的欺骗。我们看到了传送到任何地方，立即杀死敌人，打开大门和出口的能力。所有这些都是通过点击鼠标来实现的。但也有一个配置屏幕，用来启用作弊，提供了一些其他作弊选项，不是原来的游戏。在对原始源代码进行了一番深思熟虑的研究后，[Jordan Mechner]设法将这些作弊手段输入到仿真器中，原始源代码是在存储古代数字宝石的曾经丢失的软盘被发现后[Jordan Mechner]最近发布的。

你知道，我们总是看到人们在各种类型的硬件上运行毁灭战士。也许我们应该开始使用流行音乐作为我们的新奇游戏？
