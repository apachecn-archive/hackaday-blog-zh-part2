# HHH:台达数控加工厂

> 原文:[https://hackaday.com/2013/11/09/hhh-delta-cnc-mill/](https://hackaday.com/2013/11/09/hhh-delta-cnc-mill/)

![hhh-delta-mill](../Images/a1667fe983437458d00a91a4ce739caf.png)

[Dan]来信展示了他和几个朋友在大约两周的时间里安装并运行的 delta-bot 数控铣床。来自 Mad Fellows 的团队——亚利桑那州普雷斯科特的一个黑客空间——集思广益，成功地从大部分现有部件中构建了这个东西。你能相信他们在新材料上只自掏腰包 100 美元吗？

经过一些建模工作后，他们开始寻找零件，从报废的液晶显示器中回收大部分丙烯酸树脂。但是，有许多部件，如步进电机、精密杆、轴承、皮带和滑轮，不能或不应该被回收，以便最终得到一个相当坚固的机床。我们喜欢[Dan]的建议，即零件应该拧在一起，因为在更换损坏的部件时，将它们粘合在一起会有问题。

你可能想知道三角洲机器人铣削的强度。建造的目的是为熔模铸造制造模具。丢失的材料(我们不知道是蜡还是其他东西)很容易加工，你可以在跳转后的剪辑中看到工厂做得很好。但是他们也对铝做了一些测试，显然这不是问题。

HHH 的 [CNC 版已经结束了，那我们为什么现在贴这个？我们搞砸了。[丹]在截止日期前提交了一份合格的参赛作品，但不知何故我们错过了它。对不起[丹]！迟到总比不到好——我们会马上收到一件 T 恤衫。](http://hackaday.com/2013/11/04/hhh-cnc-winners/)
