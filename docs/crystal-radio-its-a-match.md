# 水晶电台:匹配！

> 原文：<https://hackaday.com/2015/09/05/crystal-radio-its-a-match/>

水晶收音机通常是孩子第一次接触电子产品。[Billy Cheung]是一名水晶收音机制造商，他希望“让水晶收音机像普通收音机一样易于使用。”他已经制作了许多套，但他最新的一套不仅适合火柴盒，还将火柴盒用作可变调谐感应器。

这种设计没有麦片盒，只有几个组件。火柴盒包含一些铁氧体棒和两种不同的绕组。通过移动火柴盒的内部，你可以调谐不同的电台。虽然这个设计需要两个固定的电容器，但[Billy]发现他有足够的自谐振(大概来自杂散电容),省略它们并不会影响他对强信号的接收。

[Billy 的]网站上有各种各样的水晶收音机版本。你可以找到内置在笔中的收音机，甚至有些能够接收调频电台。因为它们的简单，许多人用他们在 T3 附近找到的[垃圾建造了它们。](http://hackaday.com/2005/11/16/crystal-radio-built-from-modern-junk/)
