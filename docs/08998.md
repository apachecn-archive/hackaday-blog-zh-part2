# 罗斯威尔吃了你的心

> 原文:[https://hackaday.com/2015/05/10/roswell-eat-your-heart-out/](https://hackaday.com/2015/05/10/roswell-eat-your-heart-out/)

当[伊恩·伍德]不小心弄坏了他的 FPV 四旋翼飞行器上的相机时，他有点沮丧。但是在我们打破的所有东西中，我们又创造了新的东西。[Ian]决定在无人机上绑上一些 RGB 发光二极管[把它变成 UFO 来吓唬他的邻居](https://www.youtube.com/watch?v=1QzSaJm08UE)！

现在我们知道你在想什么了:RGB LEDs？这很难构成黑客攻击！你说得对——但是[Ian]并不只是简单地系上一些发光二极管，然后就此收工。哦，不。他正在使用一个微型控制器和 nazancan 解码器来监听 CAN 总线的遥控驾驶杆位置、飞行模式、高度、电池数据等。这意味着发光二极管实际上是对他驾驶无人机的方式做出反应。由于他的双叶 RX 控制器上有一个备用频道，他还可以从地面控制动画模式，做他想做的任何事情。他还去掉了 quad 上的标准指示灯 led，并将它们连接到他的新设置中。它们都被一个位于青少年区的图书馆控制着。休息后在视频里看看。
