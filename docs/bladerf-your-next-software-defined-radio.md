# BladeRF，您的下一代软件定义无线电

> 原文：<https://hackaday.com/2013/02/02/bladerf-your-next-software-defined-radio/>

到现在为止，您可能已经有点厌倦了小型廉价的电视调谐器加密狗软件定义的无线电。是的，使用 USB 电视加密狗是对 SDR 的一个很好的介绍，但它的带宽有限，频率范围有限，不能传输。[bladeRF](http://www.kickstarter.com/projects/1085541682/bladerf-usb-30-software-defined-radio)登场，这款 SDR 弥补了 USB 加密狗的所有缺点，同时也是一款出色的无线开发平台。

bladeRF 能够在 300 MHz 和 3.8 GHz 之间的任何频率上接收和发送信号。再加上强大的 FPGA、ARM CPU 和非常出色的 ADC 和 DAC，您可以构建自己的软件定义的 WiFi 适配器、蓝牙模块、ZigBee 无线电、GPS 接收机或 GSM 和 4G LTE 调制解调器。

这是一个令人印象深刻的工具，但它并不便宜；bladeRF 在 Kickstarter 上售价 400 美元。bladeRF 背后的人似乎在做正确的事情，他们正在用 Kickstarter 的横财做所有正确的事情，比如 USB 供应商 ID。

有一个视频显示两个 bladeRFs 被用作全双工调制解调器。休息之后你可以去看看。
