# 通过复古游戏和手表来怀旧

> 原文：<https://hackaday.com/2015/06/15/give-in-to-nostalgia-with-a-diy-game-and-watch/>

任天堂最早获得人气的产品之一是游戏和手表产品线。任天堂在 1980 年至 1991 年间生产了这些游戏，它们是许多 80 或 90 年代孩子怀旧的源泉。这些是电子手持游戏，有预先绘制的单色图像，可以点亮来制作非常基本的动画。[Andrew]小时候喜欢他以前的“害虫”游戏，但最终他卖掉了它。想要重温童年的记忆，他决定开发自己的[游戏和观看模拟器](http://www.instructables.com/id/DIY-Game-Watch/)。

(Andrew 的)构建核心是他在易贝找到的 PIC18F4550 USB 演示板。该板允许您使用一些简单的前端软件通过 USB 直接上传十六进制文件。[Andrew]用 MPLAB 用 C 写了他的游戏代码。他的设备使用诺基亚 5110 液晶显示屏，由小型锂离子电池供电。

对于房屋，[Andrew]从另一个大小合适的旧掌上游戏开始。他把所有的旧零件都取出来，然后把新零件放回原处。他还用喷漆给了外壳一种金属拉丝的外观。最终结果是一个非常好的原始事物的近似，正如下面的视频所证明的。
